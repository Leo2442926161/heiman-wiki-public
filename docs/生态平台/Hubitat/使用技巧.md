# 使用技巧

## 自定义驱动

海曼设备在 Hubitat 中可通过以下驱动获得最佳兼容性：

| 设备类型 | 推荐驱动 |
|----------|----------|
| 烟雾/CO 报警器 | Generic Zigbee Smoke/CO Sensor |
| 人体红外传感器 | Heiman Motion Sensor (第三方) |
| 门磁传感器 | Generic Zigbee Contact Sensor |
| 水浸探测器 | Generic Zigbee Water Leak Sensor |
| 温湿度传感器 | Heiman Temperature/Humidity Sensor |

## Zigbee 优化

- Hubitat C-8 以上型号使用 EFR32 芯片，兼容性最佳
- 在 Settings → Zigbee 中查看并选择干扰最少的信道
- 使用 Zigbee 嗅探工具（如 Ubisys）诊断掉线问题

## 规则提示

- Rule Machine 中建议添加"防误报延迟"条件（0.5-2 秒）
- 对关键规则启用"健康检查"通知，确保规则持续运行
- 以"子规则"模式组织复杂自动化，便于调试和维护

## Z-Wave 共存

如需同时使用 Z-Wave 设备，注意保持 Zigbee 与 Z-Wave 信道之间的频率间距。
