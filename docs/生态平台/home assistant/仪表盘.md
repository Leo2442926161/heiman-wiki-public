# 仪表盘配置

## 安全总览面板

使用 Section 视图布局，将海曼设备按区域组织：

```yaml
type: section
title: 烟雾与 CO 报警
cards:
  - type: entity
    entity: binary_sensor.heiman_smoke_alarm
    name: 烟雾报警器
  - type: entity
    entity: binary_sensor.heiman_co_alarm
    name: CO 报警器
```

## Mushroom 卡片样式

```yaml
type: custom:mushroom-alarm-control-panel-card
entity: alarm_control_panel.home_alarm
name: 安防系统

type: custom:mushroom-entity-card
entity: binary_sensor.heiman_smoke_alarm
name: 烟雾报警器
icon: mdi:smoke-detector
primary_info: state
secondary_info: last-changed
fill_container: true
```

## 环境传感器面板

```yaml
type: section
title: 环境监测
cards:
  - type: gauge
    entity: sensor.heiman_temperature
    name: 温度
    unit: "°C"
    min: 0
    max: 50
    severity:
      green: 0
      yellow: 30
      red: 40
  - type: gauge
    entity: sensor.heiman_humidity
    name: 湿度
    unit: "%"
    min: 0
    max: 100
    severity:
      green: 30
      yellow: 60
      red: 80
```

## 报警历史面板

```yaml
type: history-graph
title: 传感器历史
entities:
  - binary_sensor.heiman_smoke_alarm
  - binary_sensor.heiman_co_alarm
  - binary_sensor.heiman_water_leak
hours_to_show: 24
```

## 建议布局

- 顶部：安防系统状态 + 报警设备状态卡片
- 中部：环境监测（温度、湿度、水质）
- 底部：传感器报警历史趋势图
- 侧边栏快捷操作：布防/撤防、场景切换
