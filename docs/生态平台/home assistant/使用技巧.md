# 使用技巧

## 设备配置

- 海曼 Matter 设备首次配对时，确保边界路由器（如 HomePod、Nest Hub、Home Assistant Green）处于同一 Thread 网络
- Zigbee 设备通过 ZHA 或 Zigbee2MQTT 集成时，建议先使用 USB 协调器（如 Sonoff ZBDongle-E、Conbee II）的推荐固件

## 自动化优化

- 烟雾报警自动化中添加 `for:` 参数（例如 3 秒）以防止瞬时误报
- 使用 `condition: state` 配合安防模式，避免布防状态下不必要的通知
- 利用 Home Assistant 的 `group` 功能将海曼传感器按区域分组，便于批量操作

## 故障排查

- 设备离线时，首先检查 Zigbee 网络信道是否与 WiFi 2.4GHz 冲突
- Matter 设备无法发现时，重启边界路由器并重新添加设备
