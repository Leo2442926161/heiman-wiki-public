# 自动化示例

## 烟雾报警联动

```yaml
alias: "ZHA 烟雾报警联动"
triggers:
  - trigger: state
    entity_id: binary_sensor.heiman_smoke_zha
    to: "on"
actions:
  - action: light.turn_on
    target:
      area_id: living_room
  - action: tts.speak
    target:
      entity_id: media_player.speaker
    data:
      message: "烟雾报警器触发！"
```

## 设备配对自动化

```yaml
alias: "新设备配对提醒"
triggers:
  - trigger: zha.event
    subtype: heiman
    command: device_joined
actions:
  - action: notify.persistent_notification
    data:
      message: "新海曼设备已加入 ZHA 网络，请前往配置名称和区域"
```

## 水浸检测

```yaml
alias: "ZHA 水浸检测"
triggers:
  - trigger: state
    entity_id: binary_sensor.heiman_water_leak_zha
    to: "on"
actions:
  - action: switch.turn_off
    entity_id: switch.main_water_valve
  - action: notify.mobile_app_phone
    data:
      message: "💧 检测到水浸！"
```
