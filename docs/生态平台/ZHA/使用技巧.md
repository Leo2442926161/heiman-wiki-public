# 使用技巧

## 设备配对

1. 在 Home Assistant 中进入"设置 → 设备与服务 → ZHA → 配置"
2. 点击"添加设备"，然后启动海曼设备的配对模式
3. 若配对失败，将设备靠近 USB 协调器并在 3 米范围内重试

## 首荐的 USB 协调器

| 型号 | 芯片 | 推荐指数 |
|------|------|----------|
| Sonoff ZBDongle-E | EFR32MG21 | ⭐⭐⭐⭐⭐ |
| Conbee II | 2652 | ⭐⭐⭐⭐ |
| Elelabs Zigbee USB | 2652P | ⭐⭐⭐⭐ |

## 信道协调

ZHA 默认信道为 15。若与 WiFi 2.4GHz 干扰严重：
- 使用 `Silicon Labs Flasher Utility` 修改信道
- 常用推荐信道：15、20、25（避开 WiFi 1/6/11）

## 网络优化

- 每隔 3-4 个海曼设备添加一个 Zigbee 路由设备（如智能插座）以扩展网络
- 通过 ZHA 网络可视化工具检查设备连接质量
- 定期检查设备固件更新以保持最佳兼容性
