# 使用技巧

## Thing 配置

通过 MQTT binding 添加海曼设备：

```java
Thing mqtt:topic:heiman_smoke "海曼烟雾报警器" @ "客厅" {
    Channels:
        Type binary : smoke "烟雾状态" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.smoke" ]
        Type number : battery "电池电量" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.battery" ]
}
```

## Item 定义

```java
Contact Smoke_Alarm "烟雾报警 [MAP(smoke.map):%s]" { channel="mqtt:topic:heiman_smoke:smoke" }
Number Smoke_Battery "烟雾报警器电池 [%d%%]" { channel="mqtt:topic:heiman_smoke:battery" }
```

## 坐标选择

推荐使用 **CC2531** 或 **CC2652** 芯片的 USB 协调器，确保固件版本为最新。

## 调试技巧

- 在 Paper UI 中查看 Thing 的 `online/offline` 状态
- 使用 `openhab-cli info` 检查绑定是否正确加载
- 日志中搜索 `heiman` 关键词排查设备通信问题
