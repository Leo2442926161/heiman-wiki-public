# 自动化示例

## 烟雾报警场景

**场景名称：** "烟雾警报联动"

**触发条件：** 海曼烟雾报警器 → 检测到烟雾

**执行动作：**
- 智能插座 → 打开（连接排风扇）
- 灯光 → 打开并闪烁
- 推送通知到手机
- 扬声器 → 播放警报音

## 离家一键场景（Tap-to-Run）

**场景名称：** "离家布防"

**手动执行：** 点按或语音触发

**执行动作：**
- 海曼门磁 → 开启报警
- 海曼人体红外 → 开启检测
- 海曼水浸探测器 → 开启检测
- 所有灯光 → 关闭
- 智能锁 → 上锁

## 温湿度自动控制

**自动化类型：** 自动条件触发

**触发条件：** 海曼温湿度传感器 湿度 > 75%

**执行动作：**
- 智能插座（除湿机）→ 开启
- 推送通知："湿度过高，已开启除湿机"

**触发条件：** 海曼温湿度传感器 湿度 < 50%

**执行动作：**
- 智能插座（除湿机）→ 关闭
