# 使用技巧

## Zigbee 信道干扰

Homey 内置 Zigbee 芯片使用特定信道。若海曼设备频繁掉线：

1. 在 Homey Settings → Zigbee 中查看当前信道
2. 使用 Homey Insight App 扫描环境干扰
3. 切换到干扰较少的信道（远离 WiFi 1/6/11）

## 设备配网

- 海曼设备进入配对模式的方式：长按按键 5 秒（部分型号需快速按 3 次）
- 在 Homey App 中添加设备时，选择"添加设备 → Zigbee → 搜索"
- 配对失败时，将设备靠近 Homey（3 米以内）再试

## Flow 优化建议

- 在 Flow 中添加"延迟"条件（例如 2 秒）以避免瞬态误报
- 使用"标签"功能为海曼设备分类管理

## 信号范围扩展

如海曼设备与 Homey 距离较远，可添加 Zigbee 中继设备（如智能插座）扩展网络覆盖。
