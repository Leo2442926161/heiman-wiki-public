# 自动化示例

## 烟雾报警 - 闪烁灯光并通知

**Flow 配置：**

1. **触发器：** 海曼烟雾报警器 → 状态变为 `alert`
2. **条件：** （无）
3. **动作：**
   - 客厅灯光 → 闪烁（红色，每秒一次）
   - 推送通知 → 手机："⚠️ 烟雾报警器触发！"
   - 语音播报 → Homey Speaker："检测到烟雾，请注意安全！"

## CO 报警 - 关闭锅炉

**Flow 配置：**

1. **触发器：** 海曼 CO 报警器 → 状态变为 `alert`
2. **条件：** （无）
3. **动作：**
   - 锅炉开关 → 关闭
   - 智能插座（新风系统）→ 开启
   - 推送通知 → "一氧化碳报警，已关闭锅炉"

## 温湿度超限告警

1. **触发器：** 海曼温湿度传感器 → 温度高于 40°C
2. **动作：**
   - 空调 → 开启制冷模式
   - 推送通知 → "室温过高，已自动开启空调"
