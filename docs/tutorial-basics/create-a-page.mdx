# 产品中心

海曼科技提供四大核心产品线，覆盖智慧消防、智慧燃气、智慧养老和智能传感领域。

## 智慧消防

包含烟雾报警器、一氧化碳报警器、温度探测器等产品，支持蜂窝网络（4G/NB-IoT）连接，可轻松对接第三方物联网平台。

关键特性：

- 支持全网通，多平台对接
- 符合 EN54-7、UL 217 等国际标准
- 支持 Matter 协议，兼容 Alexa、Google Home、Apple HomeKit

## 智慧燃气

工业及商业级智慧燃气报警系统，专为多种环境提供全方位智能燃气安全解决方案。

关键特性：

- 高精度气体检测传感器
- 实时远程监控与报警
- 自动联动燃气紧急切断阀

## 智慧养老

含烟雾报警器、气体报警器、水浸探测器、人体摔倒传感器、门禁防盗设备及紧急报警器。

关键特性：

- 跌倒检测 AI 算法
- 一键紧急呼叫
- 多终端联动通知家人与服务机构

## 智能传感

覆盖红外探测、门窗磁感、温湿度监测、空气质量检测等全场景传感器。

关键特性：

- 超低功耗设计
- 支持 Zigbee / Matter / WiFi 多协议
- 可接入主流智能家居平台
