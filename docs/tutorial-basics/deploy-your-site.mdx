# 方案部署

海曼科技为您提供端到端的智慧安防解决方案部署指导。

## 方案集成流程

海曼产品支持多种部署方式，包括本地独立部署和云端平台对接。

### 本地部署

```bash
# 配置海曼智慧消防网关
heiman-cli configure --gateway HS1GW --mode local

# 注册探测器设备
heiman-cli register --device HS1SA --gateway HS1GW
```

### 云平台对接

海曼产品已接入主流物联网平台：

- 中国电信天翼
- 中国移动 OneNET
- 华为 HiLink
- 阿里云 IoT
- 涂鸦智能

## Matter 协议支持

海曼 Matter 系列产品可兼容：

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

实现跨品牌、跨生态的互联互通。

## 部署验证

部署完成后，可通过海曼智慧消防 APP 或管理平台进行功能验证和实时监控。
