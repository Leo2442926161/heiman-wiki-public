# 海曼科技企业介绍

**深圳市海曼科技股份有限公司**（Shenzhen Heiman Technology Co., Ltd.）成立于 2005 年，是一家集自主研发、生产、销售、服务为一体的 **国家高新技术企业** 及 **国家专精特新"小巨人"企业**。

## 关于海曼

海曼科技专注于智能传感产品的研发与创新，核心产品覆盖：

- **智慧消防** — 烟雾报警器、一氧化碳报警器等消防传感产品
- **智慧燃气** — 燃气泄漏报警器、燃气紧急切断阀等
- **智慧养老** — 紧急呼叫器、人体摔倒探测器、水浸探测器等
- **智能传感** — 红外探测器、门窗磁感应器、温湿度传感器等
