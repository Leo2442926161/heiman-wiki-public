# HS1CA-M 智能一氧化碳报警器（Matter）

HS1CA-M 是全球 **首批 Matter 认证的一氧化碳报警器**，采用电化学传感器技术，支持 Matter over Thread 跨生态互联。

## 产品概述

- **产品类型**：智能一氧化碳报警器
- **通信协议**：Matter over Thread
- **认证标准**：EN 50291
- **典型场景**：家庭、公寓等场所的 CO 泄漏预警

## 技术参数

| 参数 | 值 |
|------|-----|
| 传感器类型 | 电化学式 CO 传感器 |
| 通信协议 | Matter over Thread（IEEE 802.15.4） |
| 工作频段 | 2.4 GHz |
| 检测范围 | 0~1000 ppm |
| 报警声压 | ≥85 dB（3 米） |
| 供电方式 | DC 3V（CR123A 锂电池） |
| 电池寿命 | 约 3 年 |
| 传感器寿命 | 典型 7 年 |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH（无凝结） |

## 功能特性

- 高精度电化学 CO 传感器
- 浓度分级报警（声光提示）
- 85 dB 高声光报警
- 低电量提醒
- 传感器寿命到期提醒
- 自检功能
- Matter 多生态跨平台互操作

## 认证

- EN 50291 认证
- Matter 认证
- Works with Home Assistant 认证

需 Thread 边界路由器（Border Router），如 Apple HomePod mini、Google Nest Hub 2代、Amazon Echo 4代 等。
