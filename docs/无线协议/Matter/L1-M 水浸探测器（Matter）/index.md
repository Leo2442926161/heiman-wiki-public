# L1-M 水浸探测器（Matter）

L1-M 是海曼推出的 **Matter over Thread** 水浸探测器，通过探针检测漏水或积水，可有效预防水管爆裂带来的财产损失。

## 产品概述

- **产品类型**：水浸探测传感器
- **通信协议**：Matter over Thread
- **检测方式**：探针接触式检测
- **典型场景**：厨房、卫生间、洗衣房、地下室漏水检测

## 技术参数

| 参数 | 值 |
|------|-----|
| 检测方式 | 不锈钢探针接触式 |
| 通信协议 | Matter over Thread（IEEE 802.15.4） |
| 工作频段 | 2.4 GHz |
| 供电方式 | 电池供电 |
| 报警方式 | 蜂鸣器 + App 推送 |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH（无凝结） |

## 功能特性

- 漏水/积水实时检测
- 蜂鸣器现场报警
- App 远程告警推送
- 低电量提醒
- 探针防腐蚀设计
- Matter 多生态跨平台互操作

需 Thread 边界路由器（Border Router），如 Apple HomePod mini、Google Nest Hub 2代、Amazon Echo 4代 等。
