# HS1DS-M - 智能门磁传感器（Matter over Thread）

## 概述

HS1DS-M 是海曼智能门磁传感器的 Matter over Thread 版本，支持原生接入 Apple Home、Google Home、Amazon Alexa、Home Assistant 等所有 Matter 兼容平台，无需专有网关。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Matter over Thread |
| 供电方式 | CR1632 纽扣电池 |
| 电池寿命 | 约 2-3 年 |
| 安装间距 | ≤10 mm |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH |

## 功能特性

- 门窗开/关状态实时检测
- 防拆报警（Tamper）
- 低电量提醒
- 跨平台兼容：Apple Home / Google Home / Alexa / SmartThings / Home Assistant
- Thread Mesh 网络，低延迟本地控制
- 无需专用网关，需 Thread 边界路由器
