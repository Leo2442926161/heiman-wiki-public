# H1-M 温湿度传感器（Matter）

H1-M 是海曼推出的 **Matter over Thread** 温湿度传感器，可实时监测环境温度与相对湿度，支持智能家居自动化联动。

## 产品概述

- **产品类型**：温湿度传感器
- **通信协议**：Matter over Thread
- **检测参数**：温度、相对湿度
- **典型场景**：家庭、办公室、酒窖等环境监测

## 技术参数

| 参数 | 值 |
|------|-----|
| 温度精度 | ±0.3°C |
| 湿度精度 | ±3% RH |
| 通信协议 | Matter over Thread（IEEE 802.15.4） |
| 工作频段 | 2.4 GHz |
| 供电方式 | CR2 锂电池 |
| 电池寿命 | 约 2~3 年 |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH（无凝结） |

## 功能特性

- 实时温湿度监测
- 高精度传感（温度 ±0.3°C，湿度 ±3% RH）
- 低电量提醒
- Matter 多生态跨平台互操作

## 兼容平台

| 平台 | 支持情况 |
|------|----------|
| Apple Home | ✅ 需 Thread 边界路由器 |
| Google Home | ✅ 需 Thread 边界路由器 |
| Amazon Alexa | ✅ 需 Thread 边界路由器 |
| Home Assistant | ✅ |
| SmartThings | ✅ 需 Thread 边界路由器 |

## 边界路由要求

需 Thread 边界路由器（Border Router），如 Apple HomePod mini、Google Nest Hub 2代、Amazon Echo 4代 等。
