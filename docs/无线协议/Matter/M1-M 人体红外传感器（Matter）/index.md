# M1-M 人体红外传感器（Matter）

M1-M 是海曼推出的 **Matter over Thread** 人体红外传感器，集成 PIR 检测与光照度检测功能，适用于智能家居自动化场景。

## 产品概述

- **产品类型**：人体红外传感器
- **通信协议**：Matter over Thread
- **检测方式**：被动红外（PIR）+ 光照度检测
- **典型场景**：人来灯亮、安防联动、环境自动化

## 技术参数

| 参数 | 值 |
|------|-----|
| 检测角度 | 110°（壁挂）/ 360°（吸顶） |
| 检测距离 | ≤7 m（25°C） |
| 光照度范围 | 0~1200 Lux |
| 通信协议 | Matter over Thread（IEEE 802.15.4） |
| 工作频段 | 2.4 GHz |
| 供电方式 | DC 3V 锂电池 |
| 电池寿命 | 约 1 年（Matter 模式） |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH（无凝结） |

## 功能特性

- 移动人体检测
- 环境光照度检测
- 检测距离 ≤7m
- 支持壁挂或吸顶安装
- Matter 多生态跨平台互操作
