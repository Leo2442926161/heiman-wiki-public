# 技术参数

以下为海曼 WiFi 设备的通用技术参数。

## 无线参数

| 参数 | 值 |
|------|-----|
| 协议标准 | IEEE 802.11 b/g/n |
| 工作频段 | 2.4 GHz（不兼容 5 GHz） |
| 传输速率 | 最高 150 Mbps（802.11n） |
| 网络拓扑 | 星型拓扑（直连路由器） |
| 通信距离 | 室内约 30~50 米（视路由器性能） |
| 加密方式 | WPA2-PSK / WPA3 |

## 配网方式

| 方式 | 说明 |
|------|------|
| SmartConfig | 手机 App 一键配网（传统 WiFi 设备） |
| 蓝牙辅助配网 | 通过蓝牙临时通道配置 WiFi 凭据（部分新型号） |
| AP 模式 | 设备开启热点，手机连接后配置 |

## 电气与功耗

| 参数 | 值 |
|------|-----|
| 供电方式 | 市电供电（插座类 / 网关类）或电池供电（报警器类） |
| 功耗水平 | 相对 Zigbee / Thread 较高，因需维持 TCP/IP 连接 |
| 待机功耗 | 插座类约 0.5W ~ 1W |

## 环境参数

| 参数 | 值 |
|------|-----|
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤ 90% RH（无凝结） |
