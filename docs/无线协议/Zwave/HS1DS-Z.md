# HS1DS-Z 门磁传感器（Z-Wave）

HS1DS-Z 是海曼推出的 **Z-Wave** 门磁传感器，通过磁簧开关检测门窗的开关状态，兼容 Z-Wave Plus 标准。

## 产品概述

- **产品类型**：门磁传感器
- **通信协议**：Z-Wave Plus
- **检测方式**：磁簧开关（干簧管）
- **典型场景**：门窗状态监测、安防布防、智能联动

## 技术参数

| 参数 | 值 |
|------|-----|
| 检测方式 | 磁簧开关（干簧管） |
| 通信协议 | Z-Wave Plus（ITU-T G.9959） |
| 工作频段 | 868.42 MHz（欧盟）/ 908.42 MHz（美国） |
| 供电方式 | CR1632 纽扣电池 |
| 电池寿命 | 约 2~3 年 |
| 安装间隙 | ≤10 mm |
| 加密方式 | S2 Authenticated / S2 Unauthenticated |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤90% RH（无凝结） |

## 功能特性

- 实时检测门窗开/关状态
- 防拆报警（Tamper）
- 低电量提醒
- 支持场景联动（开门开灯、离家布防）
- 安装简便（3M 胶贴或螺丝固定）
- Z-Wave S2 安全加密
- 支持 OTA 固件升级
