# HS8MIS - 移动传感器（Zigbee）

## 概述

HS8MIS 是海曼推出的多功能 Zigbee 移动传感器，集成了 PIR 人体检测与光照度检测功能，适用于对光线和移动双重监测的智能场景。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 传感器类型 | 被动红外（PIR）+ 光照度 |
| 检测角度 | 110° |
| 检测距离 | ≤7 m |
| 供电方式 | DC 3V 电池 |

## 功能特性

- 移动人体检测
- 光照度检测
- 低功耗设计
- 支持场景联动
