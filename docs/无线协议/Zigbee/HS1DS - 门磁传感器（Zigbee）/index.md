# HS1DS - 门磁传感器（Zigbee）

## 概述

HS1DS 是海曼推出的 Zigbee 门磁传感器，用于检测门窗的开合状态，是智能安防系统的基础组件。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 传感器类型 | 磁簧开关 |
| 供电方式 | CR1632 纽扣电池 |
| 电池寿命 | 约 2 ~ 3 年 |
| 安装间隙 | ≤10 mm |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH |

## 功能特性

- 实时检测门窗开 / 关状态
- 防拆报警（Tamper）
- 低电量提醒
- 支持联动（开门开灯、离家布防等）
- 安装简便（3M 胶贴或螺丝固定）

## 适用配件

- CR1632 纽扣电池
- 3M 双面胶 / 螺丝
