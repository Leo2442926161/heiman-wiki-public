# HS8MLS - 微动传感器（Zigbee）

## 概述

HS8MLS 是海曼推出的长距离 Zigbee 人体红外传感器（PIR），专为需要远距离检测的场景设计，适用于走廊、大厅等开阔空间。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 传感器类型 | 被动红外（PIR） |
| 检测距离 | 长距离检测 |
| 供电方式 | DC 3V 电池 |

## 功能特性

- 长距离移动检测
- 低功耗设计
- 支持场景联动
