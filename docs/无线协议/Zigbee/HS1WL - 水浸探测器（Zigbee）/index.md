# HS1WL - 水浸探测器（Zigbee）

## 概述

HS1WL 是海曼推出的第一代 Zigbee 水浸探测器，采用探针检测方式，可有效检测漏水或积水情况，是基础型水浸监测设备。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 检测方式 | 探针检测 |
| 供电方式 | DC 3V 电池 |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH |

## 功能特性

- 漏水 / 积水实时检测
- 蜂鸣器报警
- 低电量提醒
- 探针防腐蚀设计
