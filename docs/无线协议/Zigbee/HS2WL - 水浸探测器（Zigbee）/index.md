# HS2WL - 水浸探测器（Zigbee）

## 概述

HS2WL 是海曼推出的第二代 Zigbee 水浸探测器，支持探针和线缆双通道检测，可用于检测水管爆裂、水槽溢水等漏水事件。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 检测方式 | 探针检测 + 线缆检测（双通道） |
| 供电方式 | DC 3V 电池 |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH |

## 功能特性

- 漏水 / 积水实时检测
- 探针 + 线缆双通道检测
- 蜂鸣器报警
- 低电量提醒
- 探针防腐蚀设计

## 适用配件

- 延长检测线缆
