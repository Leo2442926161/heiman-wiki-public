# JT 系列 - 燃气紧急切断阀（Zigbee）

## 概述

JT 系列燃气紧急切断阀是与海曼 Zigbee 可燃气体探测器配合使用的执行设备，检测到燃气泄漏时可远程或自动关闭燃气管道阀门，防止事故发生。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 类型 | 独立式 + Zigbee 联动 |
| 检测气体 | 天然气 / 液化石油气 |
| 认证标准 | GB 15322 |
| 工作温度 | -10°C ~ +50°C |

## 功能特性

- 与 HS1CG / HS3CG 联动自动关阀
- 支持远程手动关阀
- 泄漏报警推送
- 电磁阀驱动

## 认证标准

- GB 15322

## 适用配件

- HS1CG / HS3CG 可燃气体探测器
