# HS8OS - 人体存在传感器（Zigbee）

## 概述

HS8OS 是海曼推出的 Zigbee 人体存在传感器，采用毫米波雷达技术，能够检测静止状态下的人体存在（如躺卧、静坐），弥补了传统 PIR 传感器只能检测移动的不足。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 传感器类型 | 毫米波雷达（mmWave） |
| 检测范围 | 半径 3 ~ 5 m |
| 检测能力 | 静止人体存在（含微动呼吸检测） |
| 供电方式 | AC 供电 |
| 安装方式 | 吸顶安装 |

## 功能特性

- 静止人体检测（呼吸微动）
- 覆盖半径 3~5 米
- 吸顶安装，最佳检测角度
- 可与 PIR 传感器互补使用

## 适用场景

- 智能照明：有人常亮，无人自动熄灯
- 空调节能：人走自动关空调
- 智慧养老：老人长时间静止预警
