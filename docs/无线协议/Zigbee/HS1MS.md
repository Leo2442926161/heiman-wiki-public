# HS1MS - 人体红外传感器（Zigbee）

## 概述

HS1MS 是海曼推出的标准型 Zigbee 人体红外传感器（PIR），采用被动红外检测技术，可感知移动人体发出的红外热辐射，广泛应用于智能照明和安防场景。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 传感器类型 | 被动红外（PIR） |
| 检测角度 | 110° |
| 检测距离 | ≤7 m（25°C） |
| 供电方式 | DC 3V 锂电池 |
| 电池寿命 | 约 3 年 |
| 工作温度 | -10°C ~ +50°C |
| 安装方式 | 壁挂安装 |

## 功能特性

- 移动人体检测
- 110° 广角检测
- 低电量提醒
- 防拆报警
- 支持场景联动
