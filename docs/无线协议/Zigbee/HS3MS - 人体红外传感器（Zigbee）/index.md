# HS3MS - 人体红外传感器（Zigbee）

## 概述

HS3MS 是海曼第三代 Zigbee 人体红外传感器，采用 360° 吸顶安装设计，检测灵敏度和覆盖范围均较上一代显著提升。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 传感器类型 | 被动红外（PIR） |
| 检测角度 | 360°（吸顶安装） |
| 检测距离 | ≤7 m（25°C） |
| 供电方式 | DC 3V 锂电池 |
| 电池寿命 | 约 3 年 |
| 工作温度 | -10°C ~ +50°C |
| 安装方式 | 吸顶安装 |

## 功能特性

- 移动人体检测
- 360° 全向覆盖
- 低电量提醒
- 防拆报警
- 检测灵敏度提升
- 支持场景联动
