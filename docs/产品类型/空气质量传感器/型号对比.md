# 型号对比

## 各型号对比

| 型号 | 协议 | 检测参数 | 供电 | 特点 | 玩法建议 |
|------|------|----------|------|------|----------|
| 空气质量传感器 | Zigbee | PM2.5 / PM10 / CO₂ / TVOC / 温湿度 | 市电 | 多功能合一 | 监测室内空气质量，超标自动联动新风/净化器 + 推送提醒 |

## 场景玩法推荐

| 场景 | 推荐型号 | 联动方案 |
|------|----------|----------|
| 卧室 CO₂ 监控 | 空气质量传感器 | CO₂ > 1000ppm → 开窗提醒 / 联动新风系统换气，睡眠质量更好 |
| 新装修房 | 空气质量传感器 | TVOC/甲醛持续监测，超标联动排风扇，达标后推送安心通知 |
| 儿童房 | 空气质量传感器 | PM2.5 监测，超标自动开启空气净化器，保障呼吸健康 |
| 办公室 | 空气质量传感器 | CO₂ + 温湿度联动空调新风，提高工作效率 |
