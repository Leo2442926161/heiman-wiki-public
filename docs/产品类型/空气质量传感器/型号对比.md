# 型号对比

## 各型号对比

海曼空气质量传感器具体型号请参考海曼官方网站获取最新信息。

| 检测参数 | 说明 | 健康标准 |
|----------|------|----------|
| PM2.5 | 细颗粒物 | ≤ 35 µg/m³（24h 平均） |
| PM10 | 可吸入颗粒物 | ≤ 50 µg/m³（24h 平均） |
| CO₂ | 二氧化碳 | ≤ 1000 ppm |
| TVOC | 总挥发性有机物 | ≤ 0.6 mg/m³ |
| 温度 | 环境温度 | 18-26°C |
| 湿度 | 相对湿度 | 40-60% RH |

## 选型建议

- 根据装修需求选择合适的检测参数组合
- 新装修房建议选择支持 TVOC 检测的型号
- 办公室/教室建议关注 CO₂ 检测功能

更多技术详情请参考 [Zigbee 协议](../../无线协议/Zigbee/支持型号列表.md)。
