# 型号对比

## 各型号对比

| 参数 | 规格 |
|------|------|
| 通信协议 | Zigbee / Wi-Fi |
| 视频 | 支持（Wi-Fi 版） |
| 双向语音 | 支持 |
| 夜视 | 支持 |
| 供电 | 电池或有线 |

海曼智能门铃具体型号请参考海曼官方网站获取最新信息。

## 选型建议

- **需要远程视频**：选择 Wi-Fi 版门铃
- **仅门铃呼叫 + 安防联动**：Zigbee 版即可
- 如需更多型号信息请咨询海曼官方

更多技术详情请参考 [Zigbee 协议](../../无线协议/Zigbee/支持型号列表.md)。
