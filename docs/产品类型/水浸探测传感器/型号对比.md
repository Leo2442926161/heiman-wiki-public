# 型号对比

## 各型号对比

| 型号 | 通信协议 | 检测方式 | 特点 |
|------|----------|----------|------|
| HS1WL | Zigbee | 探针检测 | 基础款，单通道 |
| HS2WL | Zigbee | 探针 + 线缆 | 双通道检测，扩展线缆 |
| HS3WL | Zigbee | 探针检测 | 增强版 |
| L1-Z | Zigbee | 探针检测 | Zigbee 版 |
| L1-M | Matter over Thread | 探针检测 | Matter 版，WWHA 认证 |

## 选型建议

- **基础需求**：HS1WL
- **大范围监测**：HS2WL（可接延长线缆）
- **Matter 跨平台**：L1-M
- **Zigbee 生态**：L1-Z 或 HS 系列

更多技术详情请参考 [Matter 协议](../../无线协议/Matter/支持型号列表.md) 或 [Zigbee 协议](../../无线协议/Zigbee/支持型号列表.md)。
