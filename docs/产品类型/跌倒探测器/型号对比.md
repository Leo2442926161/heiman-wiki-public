# 型号对比

## 各型号对比

| 型号 | 协议 | 传感器 | 供电 | 特点 | 玩法建议 |
|------|------|--------|------|------|----------|
| [HS2FD](../../无线协议/Zigbee/HS2FD - 跌倒探测器（Zigbee）) | Zigbee | 加速度计 + 陀螺仪 | 电池 | AI 算法跌倒检测 | 穿戴式设计，自动检测跌倒 + 一键求救，适合独居老人和养老机构 |

## 场景玩法推荐

| 场景 | 推荐型号 | 联动方案 |
|------|----------|----------|
| 独居老人监护 | HS2FD | 检测到跌倒 → 自动推送子女手机 → 超时未响应自动呼叫 120 |
| 养老机构 | HS2FD | 集中管理平台 + 楼层声光报警 + 护工手持机同步告警 |
| 术后康复 | HS2FD | 患者下床活动检测到跌倒风险 → 护工站提前预警 |
| 浴室/夜间 | HS2FD | IP 防护设计，洗澡可佩戴 + 夜间起夜跌倒自动告警 |

## 选型建议

- **居家养老**：[HS2FD](../../无线协议/Zigbee/HS2FD - 跌倒探测器（Zigbee）)（穿戴式，AI 检测）
- **与紧急按钮搭配**：HS2FD 自动跌倒检测 + 紧急按钮手动求救，双重保障
