# 安装指南

## 佩戴方式

- **腰带夹**：夹在腰带或裤腰上
- **挂绳**：挂在脖子上，垂在胸前

## 使用说明

1. 装入电池
2. 选择佩戴方式（腰带夹或挂绳）
3. 佩戴在身上
4. 通过网关配对
5. 模拟测试：
   - 站立后快速蹲下或躺倒在地
   - 传感器应检测到跌倒并在数秒内触发报警
   - 按下求救按钮测试手动报警

## 注意事项

- 建议**全天佩戴**（洗澡时可取下充电/换电池）
- 采用 AI 算法区分跌倒与日常活动（坐下、弯腰），误报率低
- 检测到跌倒后自动报警，也可手动按键求救
- 可与紧急按钮联动实现双重保障
