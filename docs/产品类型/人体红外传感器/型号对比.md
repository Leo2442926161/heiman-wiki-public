# 型号对比

## 各型号对比

| 型号 | 通信协议 | 检测角度 | 安装方式 | 特点 |
|------|----------|----------|----------|------|
| HS1MS | Zigbee | 110° | 壁挂 | 基础款，经济实用 |
| HS3MS | Zigbee | 360° | 吸顶 | 升级款，全覆盖 |
| M1-Z | Zigbee | 110° | 壁挂 | 支持光照度检测 |
| M1-M | Matter over Thread | 110° | 壁挂 | Matter 版，跨平台 |
| HS8MLS | Zigbee | 110° | 壁挂 | 长距离检测 |
| HS8MIS | Zigbee | 110° | 壁挂 | 人体 + 光照度复合 |

## 选型建议

- **走廊/房间壁挂**：HS1MS 或 HS8MLS（长距离）
- **吸顶全屋覆盖**：HS3MS（360°）
- **需要光照度联动**：M1 系列或 HS8MIS
- **Matter 平台**：M1-M

更多技术详情请参考 [Zigbee 协议](../../无线协议/Zigbee/支持型号列表.md)。
