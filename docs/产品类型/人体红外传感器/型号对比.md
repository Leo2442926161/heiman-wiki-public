# 型号对比

## 各型号对比

| 型号 | 协议 | 检测角度 | 安装 | 特点 | 玩法建议 |
|------|------|----------|------|------|----------|
| [HS1MS](../../无线协议/Zigbee/HS1MS - 人体红外传感器（Zigbee）) | Zigbee | 110° | 壁挂 | 基础款，经济实用 | 人来灯亮、人走灯灭，安防联动检测移动推送报警 |
| [HS3MS](../../无线协议/Zigbee/HS3MS - 人体红外传感器（Zigbee）) | Zigbee | 360° | 吸顶 | 全覆盖 | 吸顶全覆盖无死角，适合客厅/大堂区域 |
| [M1-Z](../../无线协议/Zigbee/M1-Z - 人体红外传感器（Zigbee）) | Zigbee | 110° | 壁挂 | 光照度检测 | PIR + 光照传感器：白天不触发关灯，夜晚自动亮灯 |
| [M1-M](../../无线协议/Matter/M1-M 人体红外传感器（Matter）) | Matter over Thread | 110° | 壁挂 | Matter 版 | 原生 HA + Apple Home，光照度联动窗帘自动调节 |
| [HS8MLS](../../无线协议/Zigbee/HS8MLS - 微动传感器（Zigbee）) | Zigbee | 110° | 壁挂 | 长距离 | 走廊/车库长直区域，检测距离更远 |
| [HS8MIS](../../无线协议/Zigbee/HS8MIS - 移动传感器（Zigbee）) | Zigbee | 110° | 壁挂 | 复合传感 | PIR + 光照度二合一，联动智能灯根据光线自动调节亮度 |

## 场景玩法推荐

| 场景 | 推荐型号 | 联动方案 |
|------|----------|----------|
| 人来灯亮节能 | M1-Z / HS8MIS | PIR 检测有人 → 开灯 + 读取光照度决定亮度 → 无人超时关灯 |
| 安防布防联动 | HS1MS | 离家模式检测到移动 → 推送报警 + 声光警号 + 摄像头录制 |
| 吸顶全覆盖 | HS3MS | 室内中间吸顶，360° 无死角，适合开放空间 |
| Matter 跨生态 | M1-M | Apple Home 自动化 + Google/Alexa 语音，光照联动窗帘 |
| 长走廊/车库 | HS8MLS | 长距离检测，车辆/人员进出自动照明 |

## 选型建议

- **走廊/房间壁挂**：[HS1MS](../../无线协议/Zigbee/HS1MS - 人体红外传感器（Zigbee）) 或 [HS8MLS](../../无线协议/Zigbee/HS8MLS - 微动传感器（Zigbee）)（长距离）
- **吸顶全屋覆盖**：[HS3MS](../../无线协议/Zigbee/HS3MS - 人体红外传感器（Zigbee）)（360°）
- **需要光照度联动**：[M1 系列](../../无线协议/Zigbee/M1-Z - 人体红外传感器（Zigbee）) 或 [HS8MIS](../../无线协议/Zigbee/HS8MIS - 移动传感器（Zigbee）)
- **Matter 平台**：[M1-M](../../无线协议/Matter/M1-M 人体红外传感器（Matter）)
