# 安装指南

## 安装位置要求

- **天花板安装**，位于房间中央
- **最佳使用场景**：厨房、车库、锅炉房、浴室——这些地方烟雾传感器易误报
- 距离墙壁至少 **30 cm**
- 避免安装在通风口、空调出风口附近

## 安装步骤

1. 选定安装位置（推荐厨房或车库天花板）
2. 将安装底座固定在天花板上
3. 装入电池，传感器自动开机
4. 将主体旋转卡入底座
5. 自检：按下测试键，指示灯亮起并发出声音
6. 通过网关配对（Zigbee 版）

## 注意事项

- 温感传感器**不是烟雾传感器**的替代品，而是补充
- 烟雾传感器响应更快，温感传感器抗干扰更强
- 建议同一房间同时安装烟雾和温感传感器
- 每月测试一次，电池低电量时及时更换
