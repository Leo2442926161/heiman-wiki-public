# 型号对比

## 各型号对比

声光报警器属于通用型产品，海曼提供标准型号适用于各类安防场景。

| 参数 | 规格 |
|------|------|
| 通信协议 | Zigbee |
| 报警声压 | ≥ 110 dB |
| 灯光 | 红色 / 彩色频闪 |
| 防拆 | 支持 |
| 供电 | 电池或市电（视型号） |

## 选型建议

- 所有声光报警器均支持 Zigbee 协议
- 选择时关注供电方式（电池供电更灵活，市电供电无需换电池）
- 如需更多型号信息请咨询海曼官方

更多技术详情请参考 [Zigbee 协议](../../无线协议/Zigbee/支持型号列表.md)。
