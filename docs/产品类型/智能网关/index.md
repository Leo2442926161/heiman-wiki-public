# 智能网关

智能网关（Smart Gateway）是海曼智能家居系统的核心中枢，负责连接各类 Zigbee 传感器并实现设备间的联动控制。

## 产品系列

| 系列 | 通信方式 | 特点 |
|------|----------|------|
| HS1GW 系列 | Zigbee + Wi-Fi / Ethernet | 基础型网关 |
| HS3GW 系列 | Zigbee + Wi-Fi / Ethernet | 增强型 |
| HS5GW 系列 | Zigbee + Wi-Fi / Ethernet | 高性能网关 |
| HS6GW 系列 | Zigbee + Wi-Fi / Ethernet + Thread | 最新款，同时支持 Zigbee 和 Matter over Thread |

## 核心功能

- Zigbee 网络协调器
- 支持 Wi-Fi / 以太网连接互联网
- 设备自动发现和配对
- 本地场景联动（部分型号）
- 远程访问（通过海曼云平台）
- 最多支持 128 个子设备（视型号）

## 适用场景

- 海曼全屋智能的核心连接点
- 配合海曼 APP 实现远程监控
- 中小型商业项目的本地集中管理

## 与其他网关的区别

海曼网关是官方生态的入口。对于希望使用第三方平台的用户，建议使用通用 Zigbee 协调器配合 Zigbee2MQTT / ZHA 使用。
