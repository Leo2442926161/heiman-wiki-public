# 安装指南

## 安装位置要求

- **全屋中央位置**，确保 Zigbee 信号覆盖所有设备
- 尽量避免放置在全金属机柜内（金属会屏蔽无线信号）
- **以太网优先**：建议使用有线网络连接，更稳定
- 距离其他 Zigbee 设备不超过 20-30 m（室内）

## 安装步骤

1. 选定安装位置（客厅电视柜、弱电箱旁等）
2. 连接电源适配器
3. 连接以太网线（推荐）或连接 Wi-Fi
4. 等待指示灯变绿/常亮
5. 打开海曼 APP，扫描网关底部二维码添加
6. 添加完成后即可配对子设备

## 注意事项

- 网关是智能家居的核心，建议使用 UPS 备用电源供电
- 以太网连接比 Wi-Fi 更稳定，推荐优先使用
- 如果设备信号弱，可添加 Zigbee 中继设备（如智能插座）
- 不同型号支持的最大子设备数不同，根据规模选择
