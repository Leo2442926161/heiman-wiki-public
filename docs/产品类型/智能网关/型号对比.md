# 型号对比

## 各型号对比

| 型号 | Zigbee 子设备上限 | 联网方式 | 特点 | 玩法建议 |
|------|------------------|----------|------|----------|
| [HS6GW](../../无线协议/Zigbee/HS6GW.md) | 128 | WiFi + Ethernet | 最新款，同时支持未来 Thread 升级 | 海曼自有生态核心，配合海曼 APP 远程管理，适合不折腾的用户 |
| HS5GW | 64 | WiFi + Ethernet | 高性能 | 中大型项目，稳定可靠 |
| HS3GW | 32 | WiFi | 增强型 | 中小户型够用 |
| HS1GW | 16 | WiFi | 基础型 | 入门级，适合少量设备 |

## 场景玩法推荐

| 场景 | 推荐型号 | 说明 |
|------|----------|------|
| 全屋海曼生态 | HS6GW | 搭配海曼 Zigbee 传感器使用，配合海曼 APP 远程管理，无需自己搭建服务器 |
| 第三方平台用户 | — | 建议直接购买 Zigbee 协调器（如 SLZB-06）配合 Zigbee2MQTT / ZHA 使用，灵活性更高 |
| 中小户型 | HS3GW | 32 设备上限，满足基础需求 |

## 选型建议

- **海曼自有生态**：[HS6GW](../../无线协议/Zigbee/HS6GW.md)（128 设备，WiFi + Ethernet）
- **第三方平台**：推荐通用 Zigbee 协调器 + Zigbee2MQTT / ZHA
