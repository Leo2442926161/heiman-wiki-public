# 型号对比

## 各型号对比

| 型号 | 通信协议 | 联网方式 | 最大子设备数 | 特点 |
|------|----------|----------|-------------|------|
| HS1GW | Zigbee | Wi-Fi / 以太网 | 64 | 基础型网关 |
| HS3GW | Zigbee | Wi-Fi / 以太网 | 128 | 增强型 |
| HS5GW | Zigbee | Wi-Fi / 以太网 | 128 | 高性能网关 |
| HS6GW | Zigbee + Thread | Wi-Fi / 以太网 | 128+ | 最新款，支持 Zigbee 和 Matter |

## 选型建议

- **基础需求**（< 50 设备）：HS1GW
- **大户型**（50-128 设备）：HS3GW 或 HS5GW
- **同时支持 Matter**：HS6GW（最新款，支持 Matter over Thread）
- **有线连接稳定**：以上均支持以太网有线连接

更多技术详情请参考 [Zigbee 协议](../../无线协议/Zigbee/支持型号列表.md) 或 [Matter 协议](../../无线协议/Matter/支持型号列表.md)。
