# 安装指南

## 安装位置要求

- 安装在需要监测振动的物体表面
- 常见安装位置：**门窗内侧**、**保险柜**、**管道**、**墙面**、**围栏**
- 确保安装表面平整、清洁

## 安装步骤

1. 选择要监测的物体（如保险柜门内侧）
2. 清洁安装表面
3. 用 3M 胶贴或螺丝将传感器固定
4. 装入 CR2450 电池
5. 通过网关配对
6. 模拟测试：敲击安装表面，传感器应触发报警

## 注意事项

- 对于门窗防撬场景，安装在门/窗内侧靠锁位置效果更佳
- 对于管道监测，直接粘贴在管道表面
- 电池寿命约 2 年
- 可调节灵敏度以适应不同场景
