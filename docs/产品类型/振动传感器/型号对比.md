# 型号对比

## 各型号对比

| 参数 | 规格 |
|------|------|
| 通信协议 | Zigbee |
| 传感器类型 | 3 轴加速度计 |
| 检测维度 | 振动 + 倾斜 |
| 供电 | CR2450 纽扣电池 |
| 电池寿命 | 约 2 年 |

振动传感器为通用型号，支持检测振动强度和倾斜角度变化。

## 选型建议

- 海曼振动传感器为通用 Zigbee 型号
- 如需更多型号信息请咨询海曼官方

更多技术详情请参考 [Zigbee 协议](../../无线协议/Zigbee/支持型号列表.md)。
