# 其他认证

## Q-Label 质量标志

Q-Label 是欧洲智能家居行业的质量认证标志，由 Q-Lab 联盟（现隶属于 Zigbee Alliance / CSA 连接标准联盟）推出。

| 项目 | 说明 |
|------|------|
| 适用范围 | Zigbee 设备互操作性认证 |
| 认证内容 | 设备与其他品牌 Zigbee 网关/控制器的兼容性 |
| 意义 | 确保跨品牌互操作，减少"买回家不能用"的风险 |

海曼 Zigbee 产品通过 Q-Label 认证，确保与以下平台兼容：
- 亚马逊 Echo Plus / Echo Show（内置 Zigbee）
- 飞利浦 Hue 桥接器
- SmartThings Hub
- 其他 Q-Label 认证网关

## Red Dot 设计奖

海曼 HS1SA 系列烟雾报警器荣获 **Red Dot Design Award**（红点设计奖），体现了海曼在产品设计方面的实力。

## ISO 9001 质量管理体系

海曼已通过 ISO 9001:2015 质量管理体系认证，确保从研发、生产到售后的全流程质量控制。
