# CE 欧盟符合性

CE（Conformité Européenne，欧盟符合性）标志是产品进入欧盟市场的强制性安全标志。

## CE 标志的含义

CE 标志表明产品符合欧盟相关指令的基本要求，可在欧盟成员国及欧洲经济区（EEA）自由流通。

## 海曼产品的 CE 认证路径

海曼烟雾/CO 报警器通过以下途径满足 CE 要求：

### CPR 法规（EU No. 305/2011）

建筑产品法规（Construction Products Regulation）要求火灾报警产品进行性能稳定性评估：
- 依据标准：EN 14604（烟雾）、EN 50291（CO）
- 评估机构：TÜV Rheinland 等公告机构
- 认证等级：DoP（性能声明）+ CE 标志

### RED 指令（2014/53/EU）

无线电设备指令（Radio Equipment Directive）要求无线设备：
- 涵盖 Zigbee、Z-Wave、Wi-Fi、Thread 等无线模块
- 测试项目：电磁兼容（EMC）、射频（RF）、SAR、安全

### EMC 指令（2014/30/EU）

电磁兼容指令确保设备不会对其他电子设备产生干扰。

## CE 标志产品

海曼所有销往欧盟市场的产品均带有 CE 标志，包括：
- HS1SA-E / HS3SA（Zigbee 烟雾报警器）
- HS1CA-E / C1 系列（Zigbee CO 报警器）
- HS1SA-M / HS1CA-M（Matter over Thread）
- HS1MS / HS1DS 等全系列 Zigbee 传感器
