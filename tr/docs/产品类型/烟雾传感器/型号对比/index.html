场景玩法推荐​

场景推荐型号联动方案
智能家居深度整合HS1SA-E / HS1SA-MHA 自动化：烟雾报警 → 开全屋灯 → 解锁门锁 → 关燃气阀门 → 推送手机
苹果 HomeKit 用户HS1SA-M原生 Matter 接入，HomePod 作为 Thread 边界路由器，自动场景触发

选型建议​

  • 欧盟市场：[HS1SA-E](../../无线协议/Zigbee/HS1SA-E - 智能烟雾报警器（Zigbee）)（Zigbee）或 [HS3SA](../../无线协议/Zigbee/HS3SA - 智能烟雾报警器（Zigbee）)（Zigbee），如需 Z-Wave 选择 [HS1SA-Z](../../无线协议/Zwave/HS1SA-Z 智能烟雾报警器（Z-Wave）)
  • 美国市场：[HS8SA](../../无线协议/Zigbee/HS8SA - 智能烟雾报警器（Zigbee）)（UL 217 认证）

  • 独立式场景：[S1 系列](../../无线协议/Sub1G(long distance range)/S1 系列 - 独立式烟雾报警器（Sub-1G）)（无需网关）
