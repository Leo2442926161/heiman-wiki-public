# Dokumentversionshantering

Heiman Technology Wiki stöder flerversionsdokumenthantering, vilket säkerställer tillgång till teknisk dokumentation som matchar din produktens firmware-version.

## Versionsjämförelse

| Doc-version | Kompatibel firmware | Releasedatum |
|------------|-------------------|-------------|
| 2.0        | Matter Series      | 2025-06     |
| 1.0        | Zigbee Series      | 2024-01     |

## Visa olika versioner

Använd versionsmenyn i det övre navigeringsfältet för att växla mellan olika dokumentversioner.

## Versionsuppdateringar

Vi uppdaterar kontinuerligt dokumentationen för att matcha produktiterationer. Kontrollera regelbundet för den senaste versionen.
