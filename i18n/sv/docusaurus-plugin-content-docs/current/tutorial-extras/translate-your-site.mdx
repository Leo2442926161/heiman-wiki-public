# Flerspråkigt stöd

Heiman Technology Wiki stöder flerspråkig dokumentation för globala partners och kunder.

- **简体中文 (zh-Hans)** — Standard
- **English (en)**

Heiman-produkter stöder flerspråkiga APP-gränssnitt och enhetens röstprompter, inklusive kinesiska, engelska, japanska, koreanska, spanska och fler.
