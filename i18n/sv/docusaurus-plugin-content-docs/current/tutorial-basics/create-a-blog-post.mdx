# Skapa blogginlägg

Heiman Technologys blogg publicerar företagsnyheter, branschtrender, produktrecensioner och tekniska delningar.

## Hur man bidrar

Heiman-teammedlemmar och partners är välkomna att bidra med artiklar inom följande områden:

- Branschtrendanalys för smart brand/gas/äldreomsorg
- Användningsfall och recensioner av Heiman-produkter
- Djupgående tekniska whitepapers och lösningsanalyser
- Företagsnyheter och evenemangsrapportering

Kontakta marknadsavdelningen eller maila oss för inskickning. Artiklar publiceras på bloggen efter granskning.

## Tidigare inlägg

Besök [Blogghemsidan](/blog) för att se alla publicerade artiklar.
