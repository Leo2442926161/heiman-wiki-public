# Lösningsdistribution

Heiman Technology tillhandahåller vägledning för end-to-end-distribution av smarta säkerhetslösningar.

## Integrationsprocess

Heiman-produkter stöder flera distributionsmetoder, inklusive lokal fristående distribution och molnplattformsintegration.

### Lokal distribution

```bash
# Configure Heiman Smart Fire Gateway
heiman-cli configure --gateway HS1GW --mode local

# Register detector device
heiman-cli register --device HS1SA --gateway HS1GW
```

### Molnplattformsintegration

Heiman-produkter är integrerade med ledande IoT-plattformar:

- China Telecom Tianyi
- China Mobile OneNET
- Huawei HiLink
- Alibaba Cloud IoT
- Tuya Smart

## Matter-protokollstöd

Heiman Matter-seriens produkter är kompatibla med:

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

Vilket möjliggör tvärvarumärkes- och tvärekosystem-anslutbarhet.

## Distributionsverifiering

Efter distribution, använd Heiman Smart Fire-appen eller hanteringsplattformen för funktionsverifiering och realtidsövervakning.
