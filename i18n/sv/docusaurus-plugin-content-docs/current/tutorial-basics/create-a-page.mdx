# Produkter

Heiman Technology erbjuder fyra centrala produktlinjer som omfattar Smart brandskydd, Smart gassäkerhet, Smart äldreomsorg och Smarta sensorer.

## Smart brandskydd

Inkluderar rökdetektorer, kolmonoxidlarm, värmdetektorer, etc., med stöd för mobilnät (4G/NB-IoT) för enkel integration med tredjeparts IoT-plattformar.

Nyckelfunktioner:

- Full nätverkskompatibilitet, integration med flera plattformar
- Uppfyller EN54-7, UL 217 och andra internationella standarder
- Stöder Matter-protokollet, kompatibel med Alexa, Google Home, Apple HomeKit

## Smart gassäkerhet

Industriella och kommersiella smarta gaslarmsystem som ger omfattande intelligenta gassäkerhetslösningar för olika miljöer.

Nyckelfunktioner:

- Högprecisa gasdetekteringssensorer
- Realtidsövervakning och larm på distans
- Automatisk samverkan med nödavstängningsventiler för gas

## Smart äldreomsorg

Inkluderar rökdetektorer, gaslarm, vattenläckagedetektorer, fallsensorer, dörrsäkerhetsenheter och nödlarm.

Nyckelfunktioner:

- AI-baserad algoritm för fall-detektering
- Enknapps nödsamtal
- Meddelanden till flera enheter för familj och serviceorganisationer

## Smarta sensorer

Täcker sensorer för alla scenarier inklusive infraröd detektering, dörr-/fönster magnetisk avkänning, temperatur-/fuktighetsövervakning och luftkvalitetsdetektering.

Nyckelfunktioner:

- Ultra-låg strömförbrukning
- Stöder Zigbee / Matter / WiFi multi-protokoll
- Kompatibel med större smarta hem-plattformar
