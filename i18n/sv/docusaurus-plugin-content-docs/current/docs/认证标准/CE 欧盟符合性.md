# CE EU-överensstämmelse

CE-märket (Conformité Européenne, EU Conformity) är ett obligatoriskt säkerhetsmärke för produkter som kommer in på EU-marknaden.

## Betydelsen av CE-märkning

CE-märkningen indikerar att produkten uppfyller de grundläggande kraven i relevanta EU-direktiv och kan cirkulera fritt i EU:s medlemsländer och Europeiska ekonomiska samarbetsområdet (EES).

## CE-certifieringsväg för Haiman-produkter

Hyman rök-/CO-larm uppfyller CE-kraven genom:

### HLR-förordning (EU nr 305/2011)

Byggproduktförordningen kräver att brandlarmprodukter ska genomgå prestandastabilitetsbedömning:
- Enligt standarder: EN 14604 (rök), EN 50291 (CO)
- Bedömningsbyrå: TÜV Rheinland och andra anmälda byråer
- Certifieringsnivå: DoP (Declaration of Performance) + CE-märkning

### RÖTT direktiv (2014/53/EU)

Radioutrustningsdirektivet kräver att trådlös utrustning:
- Täcker Zigbee, Z-Wave, Wi-Fi, Thread och andra trådlösa moduler
- Testobjekt: elektromagnetisk kompatibilitet (EMC), radiofrekvens (RF), SAR, säkerhet

### EMC-direktiv (2014/30/EU)

Direktivet om elektromagnetisk kompatibilitet säkerställer att utrustning inte orsakar störningar med annan elektronisk utrustning.

## CE-märkta produkter

Alla Heymann-produkter som säljs till EU-marknaden är CE-märkta, inklusive:
- HS1SA-E / HS3SA (Zigbee Smoke Alarm)
- HS1CA-E/C1-serien (Zigbee CO-larm)
- HS1SA-M/HS1CA-M (Matter over Thread)
- Helt utbud av Zigbee-sensorer som HS1MS / HS1DS
