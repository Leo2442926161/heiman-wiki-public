# HS1DS-M - Intelligent dörrsensor (Matter over Thread)

## Översikt

HS1DS-M är Matter over Thread-versionen av Heimans smarta dörrsensor. Den stöder inbyggd åtkomst till alla Matter-kompatibla plattformar som Apple Home, Google Home, Amazon Alexa, Home Assistant, etc. utan behov av en egen gateway.

## Tekniska specifikationer

| Parameter | Värde |
|------|-----|
| Kommunikationsprotokoll | Matter över tråd |
| Strömförsörjningsmetod | CR1632 knappbatteri |
| Batteritid | Ca 2-3 år |
| Installationsavstånd | ≤10 mm |
| Drifttemperatur | -10°C ~ +50°C |
| Arbetsfuktighet | ≤95 % RH |

## Funktioner

- Realtidsdetektering av dörr- och fönsteröppnings-/stängningsstatus
- Sabotagelarm (Sabotage)
- Påminnelse om lågt batteri
- Kompatibel med flera plattformar: Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Thread Mesh-nätverk, lokal kontroll med låg latens
- Ingen dedikerad gateway krävs, trådgränsrouter krävs
