# HS1SA-M Intelligent Brandvarnare (Matte)

HS1SA-M är en **Matter over Thread** smart brandvarnare lanserad av Hyman. Den använder fotoelektrisk rökavkänningsteknik och stöder tvärekologisk sammankoppling av smarta hemplattformar.

## Produktöversikt

- **Produkttyp**: Smart Brandvarnare
- **Kommunikationsprotokoll**: Matter over Thread
- **Certifieringsstandard**: EN 14604
- **Typiska scenarier**: Brandvarning i bostäder, lägenheter, hotell etc.

## Tekniska parametrar

| Parameter | Värde |
|------|-----|
| Sensortyp | Infraröd fotoelektrisk |
| Kommunikationsprotokoll | Matter over Thread (IEEE 802.15.4) |
| Driftsfrekvensband | 2,4 GHz |
| Larmljudtryck | ≥85 dB (3 meter) |
| Strömförsörjningsmetod | DC 3V (CR123A litiumbatteri) |
| Batteritid | Cirka 3 år |
| Drifttemperatur | -10°C ~ +50°C |
| Arbetsfuktighet | ≤95 % RH (ingen kondensation) |

## Funktioner

- Fotoelektrisk rökdetektion för att upptäcka långsam bränning och rök med öppen låga
- 85 dB högt ljud- och ljuslarm
- Påminnelse om lågt batteri
- Självtestfunktion (ett-klickstest)
- Anti-manipuleringslarm
- Fjärravstängning
- Har betydelse för multi-ekologisk interoperabilitet över plattformar

## Certifiering

- EN 14604 certifiering
- Ärendecertifiering
