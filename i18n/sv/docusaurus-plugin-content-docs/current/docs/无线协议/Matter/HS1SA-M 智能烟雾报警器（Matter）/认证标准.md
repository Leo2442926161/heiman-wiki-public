# Certifieringsstandarder

HS1SA-M har godkänt följande certifieringar:

- **EN 14604** — Europeisk standard för brandvarnare, se [EN European Standard](../../../../ApprovalStandards/EN European Standard.md)
- **CE-märkning** — EU-försäkran om överensstämmelse, se [CE EU-överensstämmelse](../../../../certification-standards/CE EU-Conformity.md)
- **Matter Certification** — Godkände CSA Alliance Matter-certifieringen för att säkerställa plattformsoberoende sammankoppling av smarta hem
- **Red Dot Design Award** — Red Dot Product Design Award
