# Tekniska parametrar

Följande är allmänna specifikationer för Hyman Matter-utrustning.

## Trådlösa parametrar

| Parameter | Värde |
|------|-----|
| Underliggande protokoll | Tråd (IEEE 802.15.4) |
| Driftsfrekvensband | 2,4 GHz (ISM-band) |
| Standarder för applikationslager | Matter 1.x |
| Nätverkstopologi | Mesh självläkande nätverk |
| Krypteringsmetod | AES-128-CCM |

## Hårdvarukrav

| Projekt | Beskrivning |
|------|------|
| Gränsrouter | Krävs, som Apple HomePod mini, Apple TV 4K, Google Nest Hub 2:a generationen, etc. |
| Stöd för mobilterminal | iOS / Android (nätverksdistribution genom olika ekologiska appar) |
| Tvärekologisk interoperabilitet | Apple HomeKit, Google Home, Amazon Alexa, Home Assistant, SmartThings |

## El- och strömförbrukning

| Parameter | Värde |
|------|-----|
| Strömförsörjningsmetod | Batteridriven (CR123A / CR2450, etc.) |
| Batteritid | Cirka 1~3 år (beroende på enhetstyp och användningsfrekvens) |
| Standby strömförbrukning | Extremt låg, stöder sömnväckningsmekanism |

## Miljöparametrar

| Parameter | Värde |
|------|-----|
| Drifttemperatur | -10°C ~ +50°C |
| Driftfuktighet | ≤ 95 % RH (icke-kondensation) |
