# H1-M temperatur- och luftfuktighetssensor (fråga)

H1-M är en **Matter over Thread** temperatur- och luftfuktighetssensor lanserad av Hyman, som kan övervaka omgivningstemperatur och relativ luftfuktighet i realtid och stöder smart hemautomatiseringskoppling.

## Produktöversikt

- **Produkttyp**: Temperatur- och luftfuktighetssensor
- **Kommunikationsprotokoll**: Matter over Thread
- **Detektionsparametrar**: temperatur, relativ fuktighet
- **Typiska scenarier**: Miljöövervakning i hem, kontor, vinkällare, etc.

## Tekniska parametrar

| Parameter | Värde |
|------|-----|
| Temperaturnoggrannhet | ±0,3°C |
| Fuktighetsnoggrannhet | ±3 % RH |
| Kommunikationsprotokoll | Matter over Thread (IEEE 802.15.4) |
| Driftsfrekvensband | 2,4 GHz |
| Strömförsörjningsmetod | CR2 litiumbatteri |
| Batteritid | Cirka 2~3 år |
| Drifttemperatur | -10°C ~ +50°C |
| Arbetsfuktighet | ≤95 % RH (ingen kondensation) |

## Funktioner

- Temperatur- och luftfuktighetsövervakning i realtid
- Högprecisionsavkänning (temperatur ±0,3°C, luftfuktighet ±3% RH)
- Påminnelse om lågt batteri
- Har betydelse för multi-ekologisk interoperabilitet över plattformar

## Kompatibla plattformar

| Plattform | Support |

| Google Home | ✅ Trådkantsrouter krävs |
