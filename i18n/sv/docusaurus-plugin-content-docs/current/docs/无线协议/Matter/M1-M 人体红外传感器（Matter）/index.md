# M1-M Människokroppens infraröda sensor (Materia)

M1-M är en **Matter over Thread** infraröd sensor för människokroppen lanserad av Hyman. Den integrerar PIR-detektering och belysningsdetektering och är lämplig för scenarier för smarta hemautomatiseringar.

## Produktöversikt

- **Produkttyp**: Människokroppens infraröda sensor
- **Kommunikationsprotokoll**: Matter over Thread
- **Detekteringsmetod**: Passiv infraröd (PIR) + belysningsdetektering
- **Typiskt scenario**: Ljus tänds när folk kommer, säkerhetskoppling, miljöautomatisering

## Tekniska parametrar

| Parameter | Värde |
|------|-----|
| Detekteringsvinkel | 110° (väggmonterad) / 360° (tak) |
| Detektionsavstånd | ≤7 m (25°C) |
| Belysningsområde | 0~1200 Lux |
| Kommunikationsprotokoll | Matter over Thread (IEEE 802.15.4) |
| Driftsfrekvensband | 2,4 GHz |
| Strömförsörjningsmetod | DC 3V litiumbatteri |
| Batteritid | Ungefär 1 år (Matter mode) |
| Drifttemperatur | -10°C ~ +50°C |
| Arbetsfuktighet | ≤95 % RH (ingen kondensation) |

## Funktioner

- Mobil mänsklig upptäckt
- Detektering av omgivande belysning
- Detekteringsavstånd ≤7m
- Stödvägg eller takinstallation
- Har betydelse för multi-ekologisk interoperabilitet över plattformar
