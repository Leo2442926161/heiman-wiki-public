#Materia

Matter (tidigare känd som Project CHIP) är en standard för sammankoppling av smarta hem som lanserats av CSA Connection Standards Alliance, som syftar till att bryta ned varumärkesbarriärer och uppnå tvärekologisk interoperabilitet.

## Heyman och materia

Hyman var en tidig användare av Matter-protokollet och lanserade **världens första Matter-certifierade kolmonoxidlarm**. Hymans Matter-enheter använder alla **Matter over Thread**-lösningen.

## Tekniska funktioner

| Funktioner | Beskrivning |
|------|------|
| Underliggande protokoll | Tråd (mesh-nätverk med låg effekt) |
| Nätverkstopologi | Mesh självläkande nätverk, ju fler enheter, desto bredare täckning |
| Strömförbrukning | Extremt låg, kan köras i 1-3 år på batteri |
| Säkerhet | Stöder AES-128-CCM-kryptering, enhetsautentisering krävs |
| Interoperabilitet | Över Apple HomeKit, Google Home, Amazon Alexa, Home Assistant och mer |

## Gränsrouterkrav

Att använda Matter over Thread-enheter kräver en trådgränsrouter:

- Apple HomePod mini / HomePod 2:a generationen / Apple TV 4K
- Google Nest Hub 2:a generationen / Nest Hub Max
-Amazon Echo 4:e generationen
- Home Assistant Green + Connect ZBT-2
- Hemtrevlig Pro
