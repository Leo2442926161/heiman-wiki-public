# HS1CA-M Intelligent kolmonoxidlarm (fråga)

HS1CA-M är världens första Matter-certifierade kolmonoxidlarm**, som använder elektrokemisk sensorteknologi och stödjer Matter over Thread korsekologisk sammankoppling.

## Produktöversikt

- **Produkttyp**: Intelligent kolmonoxidlarm
- **Kommunikationsprotokoll**: Matter over Thread
- **Certifieringsstandard**: EN 50291
- **Typiskt scenario**: Varning för koldioxidläckage i bostäder, lägenheter och andra platser

## Tekniska parametrar

| Parameter | Värde |
|------|-----|
| Sensortyp | Elektrokemisk CO-sensor |
| Kommunikationsprotokoll | Matter over Thread (IEEE 802.15.4) |
| Driftsfrekvensband | 2,4 GHz |
| Detektionsområde | 0~1000 ppm |
| Larmljudtryck | ≥85 dB (3 meter) |
| Strömförsörjningsmetod | DC 3V (CR123A litiumbatteri) |
| Batteritid | Cirka 3 år |
| Sensorlivslängd | Typiskt 7 år |
| Drifttemperatur | -10°C ~ +50°C |
| Arbetsfuktighet | ≤95 % RH (ingen kondensation) |

## Funktioner

- Elektrokemisk CO-sensor med hög precision
- Koncentrationsgraderat larm (ljud- och ljusmeddelande)
- 85 dB högt ljud- och ljuslarm
- Påminnelse om lågt batteri
- Påminnelse om sensorns livslängd
- Självtestfunktion
- Har betydelse för multi-ekologisk interoperabilitet över plattformar

## Certifiering

- EN 50291 certifierad
- Ärendecertifiering
- Fungerar med Home Assistant-certifiering
