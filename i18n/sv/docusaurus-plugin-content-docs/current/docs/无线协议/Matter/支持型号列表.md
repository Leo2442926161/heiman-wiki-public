# Lista över modeller som stöds

Hyman Matter-enheter använder alla Matter over Thread-lösningen. Följande är de släppta modellerna som stöds.

## Säkerhetslarmkategori

| Modell | Produktnamn | Beskrivning |
|------|----------------|------|
| HS1SA-M | Intelligent brandvarnare (Matter) | USA-version / EU-version, världens första sats av Matter-certifierade brandvarnare |
| HS1CA-M | Intelligent kolmonoxidlarm (Matter) | USA-version / EU-version, världens första parti Matter-certifierade CO-larm |

## Sensorklass

| Modell | Produktnamn | Beskrivning |
|------|----------------|------|
| M1-M | Människokroppens infraröda sensor (Materia) | Passiv infraröd detektering, stöder Apple HomeKit, Google Home, etc. |
| L1-M | Vattennedsänkningsdetektor (Materia) | Detektering av vattenläckage, larmtryck i realtid |
| H1-M | Temperatur- och fuktighetssensor (Materia) | Temperatur, fuktighetsdetektering, miljöövervakning |
| HS1DS-M | Dörrsensor (Materia) | Detektering av öppning och stängning av dörrar och fönster, anti-manipuleringslarm |

## Anteckningar

- Matter-enheter kräver en **Thread border router** (Border Router) för att användas, som Apple HomePod mini, Google Nest Hub 2:a generationen, etc.
– Den nuvarande produktlinjen Hyman Matter utökas fortfarande, och fler modeller kommer att släppas en efter en.
