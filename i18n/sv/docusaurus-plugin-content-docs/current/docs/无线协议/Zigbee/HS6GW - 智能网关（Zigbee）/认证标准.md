# Certifieringsstandarder

HS6GW (Zigbee version) har godkänt följande certifieringar:

- **CE/RED** — Europeiska unionens direktiv om trådlös utrustning
- **CCC** — China Compulsory Product Certification, se [CCC China Compulsory Certification](../../../../Certification Standards/CCC China Compulsory Certification.md)
