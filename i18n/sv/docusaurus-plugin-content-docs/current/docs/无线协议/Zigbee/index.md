# Zigbee

Zigbee 是一种低功耗、低数据速率的无线 Mesh 网络协议，基于 IEEE 802.15.4 标准，是智能家居领域最成熟和普及的协议之一。

## 海曼与 Zigbee

海曼是最早采用 Zigbee 协议的中国厂商之一，拥有庞大的 Zigbee 产品线。海曼 Zigbee 设备兼容 **Zigbee 3.0** 标准，可与主流 Zigbee 网关和协调器互通。

## 技术规格

| 特性 | 说明 |
|------|------|
| 频段 | 2.4 GHz（全球通用） |
| 传输速率 | 最高 250 kbps |
| 网络拓扑 | Mesh 自愈网络 |
| 节点数量 | 单网络支持多达 65535 个节点 |
| 通信距离 | 室内约 10-30 米（视障碍物） |
| 功耗 | 极低，传感器电池续航可达 3 年 |

## 兼容平台

海曼 Zigbee 设备可接入以下平台：

- **Zigbee2MQTT** — 开源 Zigbee 到 MQTT 桥接
- **ZHA** — Home Assistant 内置 Zigbee 集成
- **Hubitat** — 本地化智能中枢
- **Homey** — 多协议智能中枢
- **SmartThings Hub** — 三星智能家居中枢
- **Tuya Zigbee 网关** — 涂鸦生态
- **海曼 HS6GW 网关** — 官方网关
