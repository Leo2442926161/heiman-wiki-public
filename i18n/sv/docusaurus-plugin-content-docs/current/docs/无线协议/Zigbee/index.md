#Zigbee

Zigbee är ett trådlöst nätverksprotokoll med låg effekt och låg datahastighet baserat på IEEE 802.15.4-standarden. Det är ett av de mest mogna och populära protokollen inom det smarta hemområdet.

## Hyman och Zigbee

Hyman är en av de första kinesiska tillverkarna att anta Zigbee-protokollet och har en enorm Zigbee-produktlinje. Hyman Zigbee-enheter är kompatibla med **Zigbee 3.0**-standarden och kan samverka med vanliga Zigbee-gateways och koordinatorer.

## Tekniska specifikationer

| Funktioner | Beskrivning |
|------|------|
| Frekvensband | 2,4 GHz (över hela världen) |
| Överföringshastighet | Upp till 250 kbps |
| Nätverkstopologi | Mesh självläkande nätverk |
| Antal noder | Enstaka nätverk stöder upp till 65535 noder |
| Kommunikationsavstånd | Ca 10-30 meter inomhus (synliga hinder) |
| Strömförbrukning | Extremt låg, sensorbatterilivslängd upp till 3 år |

## Kompatibla plattformar

Hyman Zigbee-enheter kan anslutas till följande plattformar:

- **Zigbee2MQTT** — Öppen källkod Zigbee till MQTT-brygga
- **ZHA** — Home Assistant med inbyggd Zigbee-integration
- **Hubitat** — Lokaliserat intelligent nav
- **Hemlig** — Smart nav med flera protokoll
- **SmartThings Hub** — Samsungs smarta hemhub
- **Tuya Zigbee Gateway** — Tuya Ecosystem
- **Hyman HS6GW Gateway** — Officiell gateway
