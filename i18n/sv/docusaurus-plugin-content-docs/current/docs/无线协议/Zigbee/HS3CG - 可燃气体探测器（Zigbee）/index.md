# HS3CG - 可燃气体探测器（Zigbee）

## 概述

HS3CG 是海曼第三代 Zigbee 可燃气体探测器，在第一代基础上提升了检测精度和响应速度，适用于天然气和液化石油气泄漏检测。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 传感器类型 | 半导体 |
| 检测气体 | 天然气（甲烷）/ 液化石油气（LPG） |
| 报警声压 | ≥85 dB（3 米） |
| 供电方式 | AC 220V / DC 12V |
| 认证标准 | GB 15322 |
| 工作温度 | -10°C ~ +50°C |

## 功能特性

- 天然气 / 液化气泄漏检测
- 浓度分级报警
- 自动切断联动（配合电磁阀）
- 85 dB 声光报警
- 自检功能
- 传感器寿命到期提醒
- 增强型无线性能

## 认证标准

- GB 15322

## 适用配件

- JT 系列燃气紧急切断阀
