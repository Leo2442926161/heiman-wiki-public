# Certifieringsstandarder

HS3CG har godkänt följande certifieringar:

- **GB 15322/CCC** — China Compulsory Product Certification, se [CCC China Compulsory Certification](../../../../Certification Standards/CCC China Compulsory Certification.md)
- **CE** — Europeiska unionens deklaration om överensstämmelse
