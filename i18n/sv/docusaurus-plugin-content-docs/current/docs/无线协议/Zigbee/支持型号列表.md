# Lista över modeller som stöds

Heyman har den bredaste Zigbee-produktlinjen i branschen, och följande är de släppta modellerna som stöds.

| Modell | Produktnamn | Beskrivning |
|------|----------------|------|
| HS1SA-E | Smart brandvarnare (Zigbee) | EU-version |
| HS1SA-Z | Smart Brandvarnare (Zigbee) | Kina version |
| HS3SA | Intelligent brandvarnare (Zigbee) | Tredje generationens brandvarnare |
| HS8SA | Intelligent brandvarnare (Zigbee) | Typ av högtemperaturlänk |
| HS1CA-E | Intelligent kolmonoxidlarm (Zigbee) | EU-version |
| HS1CA-Z | Intelligent kolmonoxidlarm (Zigbee) | Kinesisk version |
| HS1CG | Brännbar gasdetektor (Zigbee) | Naturgas / flytande gas |

## Instruktioner för modellnamn

| Namnregler | Betydelse |
|--------|------|
| Suffix -E | EU-version (868MHz / EN-standard) |
| Suffix -Z | Kina version (2,4 GHz / kinesisk standard) |
| HS1/2/3/8 | Produktgenerationsnummer, ju större antal, desto nyare generation |
