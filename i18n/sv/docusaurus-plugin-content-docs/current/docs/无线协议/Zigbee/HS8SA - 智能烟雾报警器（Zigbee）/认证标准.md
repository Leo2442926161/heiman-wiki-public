# Certifieringsstandarder

HS8SA har godkänt följande certifieringar:

- **EN 14604** — Europeisk standard för brandvarnare (EU-typ), se [EN European Standard](../../../../ApprovalStandards/EN European Standard.md)
- **UL 217** — amerikansk säkerhetsstandard för brandvarnare (amerikansk modell), se [UL Underwriters Laboratories](../../../../certification-standards/ULunderwriters Laboratories.md)
- **CE/RED/EMC** — Europeiska unionens direktiv om trådlös utrustning och elektromagnetisk kompatibilitet
- **Zigbee 3.0 / Q-Label** — Zigbee Alliance-certifiering, se [Other Certifications](../../../Certification Standards/Other Certifications.md)
