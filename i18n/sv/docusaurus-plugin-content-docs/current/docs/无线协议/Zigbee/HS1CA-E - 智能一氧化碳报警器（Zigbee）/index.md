# HS1CA-E - Smart kolmonoxidlarm (Zigbee)

## Översikt

HS1CA-E är Hymans Zigbee smarta kolmonoxidlarm (CO) för EU-marknaden. Den använder en elektrokemisk sensor och kan noggrant detektera kolmonoxidgas med en koncentration på 0~1000 ppm. Den har godkänt EN 50291-certifieringen.

## Tekniska specifikationer

| Parameter | Värde |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Elektrokemisk |
| Detektionsområde | 0 ~ 1000 ppm |
| Larmljudtryck | ≥85 dB (3 meter) |
| Strömförsörjningsmetod | DC 3V (CR123A) |
| Sensorlivslängd | Ungefär 7 år |
| Drifttemperatur | -10°C ~ +50°C |
| Arbetsfuktighet | ≤95 % RH |
| Installationsmetod | Väggmontering |

## Funktioner

- Elektrokemisk CO-sensor med hög precision
- Koncentrationsgraderat larm (ljud- och ljusmeddelande)
- 85 dB larmljud
- Påminnelse om lågt batteri
- Påminnelse om sensorns livslängd
- Självtestfunktion
- OTA-firmwareuppgradering

## Certifieringsstandarder

- EN 50291 (europeisk standard för kolmonoxidlarm)
-CE/RED

## Tillämpliga tillbehör

- CR123A batteri
