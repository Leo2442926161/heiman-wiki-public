# C1 Series - Carbon Monoxide Alarm (Zigbee)

## Översikt

C1-serien är ett grundläggande Zigbee kolmonoxidlarm som lanserats av Heiman. Den använder elektrokemiska sensorer och är lämplig för scenarier för upptäckt av kolmonoxidläckage. It has passed EN 50291 certification.

## Tekniska specifikationer

| Parameter | Värde |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Elektrokemisk |
| Detektionsområde | 0 ~ 1000 ppm |
| Larmljudtryck | ≥85 dB (3 meter) |
| Strömförsörjningsmetod | Batteridriven |
| Sensorlivslängd | Ungefär 7 år |
| Drifttemperatur | -10°C ~ +50°C |

## Funktioner

- Electrochemical CO detection
- Concentration classification alarm
- 85 dB audible and visual alarm
- Påminnelse om lågt batteri
- Självtestfunktion

## Certifieringsstandarder

-EN 50291
