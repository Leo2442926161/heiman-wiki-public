# Tekniska parametrar

Följande är allmänna tekniska parametrar för Heiman Zigbee-enheter.

## Trådlösa parametrar

| Parameter | Värde |
|------|-----|
| Protokollstandarder | Zigbee 3.0 (IEEE 802.15.4) |
| Arbetsfrekvensband | 2,4 GHz (globalt gemensamt ISM-frekvensband) |
| Överföringshastighet | Upp till 250 kbps |
| Nätverkstopologi | Mesh självläkande nätverk |
| Nodkapacitet | Maximalt 65535 noder i ett enda nätverk |
| Kommunikationsavstånd | Cirka 10~30 meter inomhus (se vägghinder) |
| Krypteringsmetod | AES-128-CCM |

## Kompatibilitet

| Projekt | Beskrivning |
|------|------|
| Zigbee-koordinator | Kompatibel med Zigbee 3.0 standardkoordinator |
| Plattformar som stöds | Zigbee2MQTT, ZHA (Home Assistant), Hubitat, Homey, SmartThings, Tuya Zigbee gateway, Haiman HS6GW gateway |
| OTA-uppgradering | Stöd över-the-air uppgradering av firmware (vissa modeller) |

## El- och strömförbrukning

| Parameter | Värde |
|------|-----|
| Strömförsörjningsmetod | Batteridriven (CR123A / CR2450 / AA) eller USB / elnät (uttagstyp) |
| Batteritid | Cirka 2~3 år för sensorer (beroende på rapporteringsfrekvens) |
| Standby strömförbrukning | Mikroamp-nivå, stöder sömnavstämningsmekanism |

## Miljöparametrar

| Parameter | Värde |
|------|-----|
| Drifttemperatur | -10°C ~ +55°C |
| Driftfuktighet | ≤ 95 % RH (icke-kondensation) |
| IP-skyddsnivå | Beror på den specifika modellen (sensortyp vanligtvis IP30) |
