# Tekniska parametrar

Följande är allmänna tekniska specifikationer för Heimann WiFi-enheter.

## Trådlösa parametrar

| Parameter | Värde |
|------|-----|
| Protokollstandarder | IEEE 802.11 b/g/n |
| Driftsfrekvensband | 2,4 GHz (inte kompatibel med 5 GHz) |
| Överföringshastighet | Upp till 150 Mbps (802.11n) |
| Nätverkstopologi | Stjärntopologi (direkt ansluten till routern) |
| Kommunikationsavstånd | Cirka 30~50 meter inomhus (beroende på routerns prestanda) |
| Krypteringsmetod | WPA2-PSK / WPA3 |

## Nätverksdistributionsmetod

| Metod | Beskrivning |
|------|------|
| SmartConfig | Mobilapp Nätverkskonfiguration med ett klick (traditionell WiFi-utrustning) |
| Bluetooth-assisterad provisionering | Konfigurera WiFi-uppgifter via tillfälliga Bluetooth-kanaler (vissa nya modeller) |
| AP-läge | Slå på hotspot på enheten och konfigurera den efter att telefonen är ansluten |

## El- och strömförbrukning

| Parameter | Värde |
|------|-----|
| Strömförsörjningsmetod | Nätströmförsörjning (uttagstyp/gatewaytyp) eller batteriströmförsörjning (larmtyp) |
| Strömförbrukningsnivå | Högre än Zigbee/tråd på grund av behovet av att upprätthålla TCP/IP-anslutning |
| Standby strömförbrukning | Sockeltyp ca 0,5W ~ 1W |

## Miljöparametrar

| Parameter | Värde |
|------|-----|
| Drifttemperatur | -10°C ~ +50°C |
| Driftfuktighet | ≤ 90 % RH (icke-kondensation) |
