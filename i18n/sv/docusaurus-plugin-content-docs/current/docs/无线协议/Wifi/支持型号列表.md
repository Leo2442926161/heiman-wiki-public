# Lista över modeller som stöds

Heiman WiFi-utrustning är huvudsakligen designad för direktanslutningsscenarier och kan anslutas till hemnätverket utan en dedikerad gateway.

## Smart uttag

| Modell | Produktnamn | Beskrivning |
|------|----------------|------|
| HS2SK-WiFi | Smart uttag (WiFi) | Direkt ansluten till 2,4 GHz WiFi, stöder fjärrkontroll och timing |

## Säkerhetslarmkategori

| Modell | Produktnamn | Beskrivning |
|------|----------------|------|
| HS8SA-WiFi | Smart brandvarnare (WiFi USA-version) | WiFi version brandvarnare för den amerikanska marknaden |

## Gateway-klass

| Modell | Produktnamn | Beskrivning |
|------|----------------|------|
| HS6GW | Smart Home Gateway (WiFi + Zigbee) | Anslut till Internet via WiFi och hantera Zigbee-underenheter |
| HS6GW-2 | Smart Home Gateway (WiFi + Zigbee) | Andra generationens gateway, prestandauppgradering |

## Anteckningar

- Alla WiFi-enheter stöder endast **2,4 GHz**-bandet och är inte kompatibla med 5 GHz
- Det rekommenderas att använda WPA2-PSK-krypterade nätverk
- Nätverk kräver en router för att korrekt ansluta till Internet
