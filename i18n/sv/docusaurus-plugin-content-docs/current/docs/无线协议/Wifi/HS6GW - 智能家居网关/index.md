#HS6GW - Smart Home Gateway

## Översikt

HS6GW är en smart hemgateway lanserad av Heiman, som stöder WiFi + Ethernet-upplänk, inbyggd Zigbee-koordinator och kan hantera upp till 128 Zigbee-underenheter. Att realisera fjärråtkomst och molntjänster genom att ansluta till Internet är kärnan i Haimans smarta hemsystem.

## Tekniska specifikationer

| Parameter | Värde |
|------|-----|
| Upplänkskommunikation | IEEE 802.11 b/g/n (2,4 GHz WiFi) + 10/100M Ethernet |
| Nedlänkskommunikation | Zigbee 3.0 (koordinatorläge) |
| Maximalt antal underenheter | 128 |
| Strömförsörjningsmetod | DC 5V / 1A (Micro USB) |
| Standby strömförbrukning | ≤ 1W |
| Drifttemperatur | -10°C ~ +50°C |
| Arbetsfuktighet | ≤ 90 % RH |
| Indikatorlampa | WiFi Status/Ström/Zigbee Nätverksstatus |
| Återställ metod | Återställ knapp + lång tryckning för att återställa fabriksinställningarna |

## Funktioner

- Dubbla upplänkar: WiFi eller Ethernet trådbunden anslutning, flexibelt val beroende på nätverksmiljö
-Stöd fjärråtkomst: kontrollera enhetens status när som helst och var som helst via molnappen
- Lokal scenkoppling: stöder automatisk drift av utrustning inom LAN (fortfarande tillgänglig när den är frånkopplad)
- OTA-firmwareuppgradering
- Stöd Tuya / Smart Life-plattformen
- Hantera upp till 128 Zigbee-underenheter
-Stöd för självläkning av Zigbee-nätverk och routingoptimering
