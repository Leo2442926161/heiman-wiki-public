#HS2SK-WiFi - Smart mätt uttag (WiFi)

## Översikt

HS2SK-WiFi är ett WiFi-smart mätuttag lanserat av Heiman. Den stöder 2,4 GHz 802.11 b/g/n direktanslutning till hemroutrar och kan uppnå fjärrkontroll och övervakning av energiförbrukningen utan behov av ytterligare gateways. Lämplig för intelligent hantering av elektriska apparater i hem, kontor och andra scenarier.

## Tekniska specifikationer

| Parameter | Värde |
|------|-----|
| Kommunikationsprotokoll | IEEE 802.11 b/g/n (2,4 GHz) |
| Maximal belastning | 16A |
| Maximal effekt | 3680W (230V) / 1800W (110V) |
| Mätnoggrannhet | ±2 % |
| Strömförsörjningsmetod | Nätströmförsörjning |
| Standby strömförbrukning | ≤ 0,5W |
| Drifttemperatur | -10°C ~ +50°C |
| Arbetsfuktighet | ≤ 90 % RH |
| Skalmaterial | PC V0 flamskyddsmedel |
| Jack specifikationer | Europeisk standard / amerikansk standard (beroende på marknadsversion) |

## Funktioner

-Stöd Smart Life / Tuya smart App fjärrkontroll
- Effektövervakning och energiförbrukningsstatistik i realtid
- Fjärrkontroll
- Timing switch och nedräkning
- Scenkoppling och automatisering
- Röststyrning (kompatibel med Alexa, Google Home)
