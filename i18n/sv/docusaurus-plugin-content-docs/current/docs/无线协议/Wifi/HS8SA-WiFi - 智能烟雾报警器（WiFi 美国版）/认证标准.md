# Certifieringsstandarder

HS8SA-WiFi har godkänt följande certifieringar:

- **UL 217** — Amerikansk säkerhetsstandard för brandvarnare, se [UL Underwriters Laboratories](../../../../certification standards/ULunderwriters Laboratories.md)
- **FCC** — Federal Communications Commission-certifiering
- **CE** — Europeiska unionens deklaration om överensstämmelse
