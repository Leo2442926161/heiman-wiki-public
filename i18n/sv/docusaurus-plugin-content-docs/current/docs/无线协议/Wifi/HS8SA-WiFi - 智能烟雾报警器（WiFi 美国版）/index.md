# HS8SA-WiFi - Smart Smoke Alarm (WiFi US-version)

## Översikt

HS8SA-WiFi är en WiFi smart brandvarnare lanserad av Hyman för den amerikanska marknaden. Den använder fotoelektrisk rökavkänningsteknik, stöder WiFi-nätverk och oberoende fungerande dubbla lägen och har godkänt UL 217-certifieringen. Lämplig för brandskydd i bostäder, lägenheter och andra bostadsplatser.

## Tekniska specifikationer

| Parameter | Värde |
|------|-----|
| Kommunikationsprotokoll | IEEE 802.11 b/g/n (2,4 GHz) |
| Sensortyp | Infraröd fotoelektrisk |
| Larmljudtryck | ≥85 dB (3 meter) |
| Strömförsörjningsmetod | Batteridriven |
| Drifttemperatur | -10°C ~ +50°C |
| Arbetsfuktighet | ≤ 90 % RH |
| Installationsmetod | Takinstallation |

## Funktioner

- Fotoelektrisk rökdetektion för att upptäcka rök från långsamma brännskador och öppna lågor
- WiFi-nätverk fjärrlarm push
- Oberoende driftläge (lokalt ljud- och ljuslarm kan fortfarande uppstå när nätverket är frånkopplat)
- 85 dB högt ljud- och ljuslarm
- Påminnelse om lågt batteri
- Självtestfunktion (ett-klickstest)
- Anti-manipuleringslarm
-Stöd för fjärravstängning

## Certifieringsstandarder

- UL 217 (amerikansk standard för brandvarnare)
-FCC
- Efterlevnad av NFPA 72
