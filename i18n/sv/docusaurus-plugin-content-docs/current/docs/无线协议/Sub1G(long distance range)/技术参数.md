# Tekniska parametrar

Följande är de allmänna tekniska parametrarna för Hyman Sub-1G långdistansprodukter.

## Trådlösa parametrar

| Parameter | Värde |
|------|-----|
| Driftsfrekvensband | 433 MHz / 868 MHz / 915 MHz (beroende på marknadsområde) |
| Moduleringsmetod | FSK / OOK |
| Överföringshastighet | Cirka 10~100 kbps (videosegment och konfiguration) |
| Nätverkstopologi | Stjärna / punkt-till-punkt / Internet-nätverk |
| Kommunikationsavstånd | Cirka 1~3 kilometer i öppen miljö (videosegment och kraft) |
| Nätverkskapacitet | En enda grupp stöder upp till 50+ noder |

## Regional frekvensbandsallokering

| Region | Vanliga frekvensband |
|------|----------------|
| Kina | 433 MHz |
| Europa | 868 MHz |
| Nordamerika | 915 MHz |

## El- och strömförbrukning

| Parameter | Värde |
|------|-----|
| Strömförsörjningsmetod | Batteridriven (CR123A / AA litiumbatteri) |
| Batteritid | Cirka 3~5 år (beroende på rapporteringsfrekvens) |
| Sändningseffekt | ≤ 10 mW (+10 dBm), i enlighet med SRD-föreskrifter |
| Standby strömförbrukning | Mikroamperenivå, extremt låg |
| Mottagningskänslighet | Cirka -110 dBm ~ -120 dBm |

## Miljöparametrar

| Parameter | Värde |
|------|-----|
| Drifttemperatur | -10°C ~ +55°C |
| Driftfuktighet | ≤ 95 % RH (icke-kondensation) |
