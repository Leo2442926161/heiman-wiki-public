# S1-serien – Fristående brandvarnare (Sub-1G)

## Översikt

S1-serien är en fristående Sub-1G brandvarnare lanserad av Hyman. Den fungerar i frekvensbandet 433 MHz / 868 MHz, använder fotoelektrisk rökavkänningsteknik och har godkänt EN 14604-certifieringen. Ingen gateway eller internet krävs. Enheter är direkt sammankopplade via Sub-1G radiofrekvens. När någon enhet utlöser ett larm kommer alla enheter i samma grupp att larma samtidigt. Den är lämplig för storskaliga brandscenarier som byggnader, köpcentra och hotell.

## Tekniska specifikationer

| Parameter | Värde |
|------|-----|
| Kommunikationsprotokoll | 433 MHz / 868 MHz Sub-1G (beroende på marknadsområde) |
| Sensortyp | Infraröd fotoelektrisk |
| Larmljudtryck | ≥85 dB (3 meter) |
| Strömförsörjningsmetod | Batteridriven (CR123A / AA litiumbatteri) |
| Batteritid | ≥ 5 år |
| Kommunikationsavstånd | Cirka 1~3 kilometer i öppen miljö |
| Nätverkskapacitet | En enda grupp stöder upp till 50+ noder |
| Sammankopplingsmetod | Trådlös radiofrekvenssammankoppling (ingen gateway krävs) |
| Drifttemperatur | -10°C ~ +55°C |
| Arbetsfuktighet | ≤ 95 % RH |
| Installationsmetod | Takinstallation |

## Seriemodell

| Modell | Beskrivning |
|------|------|
| S1-SA | Grundläggande typ, stöder trådlös sammankoppling |
| S1-SA-P | Version med kamerafunktion |
| S1-SA-T | Högtemperaturlänkversion |
| S1-SA-RF | RF-sammankopplingstyp |

## Funktioner

- Fotoelektrisk rökdetektion för att upptäcka rök från långsamma brännskador och öppna lågor
- Trådlöst sammankopplingslarm: utlöses av vilken enhet som helst i samma nätverk, alla enheter kommer att ljuda och tända larm samtidigt
- Oberoende arbetsläge: ingen gateway, inget internet krävs, direkt kommunikation mellan enheter
- 85 dB högt ljud- och ljuslarm
- Påminnelse om lågt batteri
- Självtestfunktion (ett-klickstest)
- Anti-manipuleringslarm
-Stöd för fjärravstängning
- Design med ultralåg strömförbrukning, batteritid på mer än 5 år

## Certifieringsstandarder

- EN 14604 (europeisk standard för brandvarnare)
-CE/RED
- CPR (EU Construction Products Regulation)

## Tillämpliga tillbehör

- S1-RM trådlös repeater (förlängt kommunikationsavstånd)
- S1-RC fjärrkontroll (test/mute/reset)
