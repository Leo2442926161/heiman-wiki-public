# Certifieringsstandarder

HS1SA-Z har godkänt följande certifieringar:

- **EN 14604** — Europeisk standard för brandvarnare, se [EN European Standard](../../../../ApprovalStandards/EN European Standard.md)
- **Z-Wave S2-certifiering** — Stöder S2-säkerhetsramverk, se [Övriga certifieringar](../../../Certification Standards/Other Certifications.md)
- **CE** — Europeiska unionens deklaration om överensstämmelse
