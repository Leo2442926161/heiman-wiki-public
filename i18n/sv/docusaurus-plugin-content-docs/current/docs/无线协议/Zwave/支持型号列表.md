# Lista över modeller som stöds

Hyman Z-Waves produkter är främst inriktade på de europeiska och amerikanska smarta hemmarknaderna och är kompatibla med Z-Wave Plus-standarden.

## Säkerhetslarmkategori

| Modell | Produktnamn | Beskrivning |
|------|----------------|------|
| HS1SA-Z | Smart brandvarnare (Z-Wave) | Z-Wave version brandvarnare |
| HS1CA-Z | Intelligent kolmonoxidlarm (Z-Wave) | Z-Wave version CO-larm |
| HS1CG-Z | Detektor för brännbar gas (Z-Wave) | Z-Wave version gasdetektor |

## Sensorklass

| Modell | Produktnamn | Beskrivning |
|------|----------------|------|
| HS1DS-Z | Dörrsensor (Z-Wave) | Z-Wave version dörr- och fönstersensor |
| HS1MS-Z | Människokroppens infraröda sensor (Z-Wave) | Z-Wave version av mänsklig kroppsavkänning |
| HS1WL-Z | Vattennedsänkningsdetektor (Z-Wave) | Z-Wave version vattenläckagedetektion |

## Kompatibilitet

| Projekt | Beskrivning |
|------|------|
| Standard | Z-Wave Plus (500 / 700-serien chips) |
| Kompatibel Hub | Homey, Hubitat, SmartThings, Vera, Fibaro, Home Assistant (Z-Wave JS) |
| S2 Säkerhet | Stöd för S2-säkerhetsprotokoll (Okrypterat / S2 Oautentiserat / S2 Autentiserat) |
| SmartStart | Support (700-seriens chipmodeller) |
| OTA-uppgradering | Stöd FW Update CC |
