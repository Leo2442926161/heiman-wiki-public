# Tekniska parametrar

Följande är allmänna tekniska specifikationer för Hyman Z-Wave-enheter.

## Trådlösa parametrar

| Parameter | Värde |
|------|-----|
| Protokollstandarder | Z-Wave Plus (ITU-T G.9959) |
| Arbetsfrekvensband | 800~900 MHz (beroende på region: europeisk reglering 868,42 MHz, amerikansk reglering 908,42 MHz) |
| Överföringshastighet | Upp till 100 kbps |
| Nätverkstopologi | Mesh självläkande nätverk (källa routing) |
| Nodkapacitet | Maximalt 232 noder i ett enda nätverk |
| Kommunikationsavstånd | Cirka 30 meter inomhus (mellan noder), hela huset Mesh-täckning |
| Krypteringsmetod | AES-128 (S2 Authenticated / S2 Unauthenticated) |

## Regionalt frekvensband

| Region | Centerfrekvens |
|------|----------------|
| Europa | 868,42 MHz |
| Nordamerika | 908,42 MHz |
| Andra områden | Med förbehåll för lokala bestämmelser |

## El- och strömförbrukning

| Parameter | Värde |
|------|-----|
| Strömförsörjningsmetod | Batteriströmförsörjning (CR123A/AA) eller nätaggregat |
| Batteritid | Cirka 2~3 år för sensorer |
| Sändningseffekt | ≤ 1 mW (överensstämmer med SRD-bestämmelser) |
| Ta emot känslighet | Cirka -100 dBm |

## Miljöparametrar

| Parameter | Värde |
|------|-----|
| Drifttemperatur | -10°C ~ +50°C |
| Driftfuktighet | ≤ 90 % RH (icke-kondensation) |

## Nätverksegenskaper

| Funktioner | Beskrivning |
|------|------|
| SmartStart | Support (700-serien), skanna QR-koden för att gå med i nätverket |
| S2 Säkerhet | Stöder tre säkerhetsnivåer: Okrypterad, S2 Oautentiserad, S2 Autentiserad |
| Innehåller CC | Security CC, Security 2 CC, Firmware Update CC, Basic CC, Battery CC, Notification CC och mer |
