# Gassamlare - trådlös gasmätare datasamlare

## Översikt

Gassamlaren är en trådlös wM-Bus datainsamlingsenhet som lanserats av Hyman för det smarta gasfältet. Den arbetar i frekvensbandet 868 MHz (EN 13757 standard) och används för att fjärravläsa gasmätarmätdata och ladda upp den till det centraliserade mätaravläsningssystemet. Drivs av industriella litiumbatterier med en batteritid på 6 till 10 år, den är lämplig för centraliserade mätaravläsningsscenarier som lägenheter, bostadsområden och kommersiella byggnader.

## Tekniska specifikationer

| Parameter | Värde |
|------|-----|
| Kommunikationsprotokoll | wM-Bus (EN 13757) |
| Driftsfrekvensband | 868 MHz (europeiskt SRD-band) |
| Överföringshastighet | 4,8 kbps / 16,384 kbps / 100 kbps (T/S/R-läge) |
| Kommunikationsavstånd | Cirka 100~300 meter inomhus, ca 500~1000 meter i det öppna utrymmet |
| Strömförsörjningsmetod | Industriellt litiumbatteri (ER18505 / ER26500) |
| Batteritid | 6~10 år (rapporteras vanligtvis 1 gång/dag) |
| Drifttemperatur | -20°C ~ +60°C |
| Arbetsfuktighet | ≤ 95 % RH |
| Skyddsnivå | IP65 ~ IP68 (beroende på installationsmiljön) |
| Datakryptering | AES-128 (valfritt) |

## Funktioner

-Stöd wM-Bus S/T/R tre arbetslägen
- Samla automatiskt in gasmätaranvändningsdata och rapportera det regelbundet
- Låg strömförbrukning, djup sömn, lång batteritid
- AES-128 datakryptering för att säkerställa kommunikationssäkerhet
-Stöd uppgradering av fjärrprogramvara
- Mekanism för återanslutning av frånkoppling och vidaresändning av data
- Stöder EN 13757-3 applikationslagerprotokoll
- Kan användas med HS1-CM kollektor / HS1-GW gateway för centraliserad uppladdning

## Certifieringsstandarder

- EN 13757-3 (M-Bus applikationsskikt)
