# Tekniska parametrar

Följande är de allmänna tekniska parametrarna för Heimann Wireless M-Bus (wM-Bus) utrustning.

## Trådlösa parametrar

| Parameter | Värde |
|------|-----|
| Protokollstandarder | EN 13757-4 (Trådlöst M-Bus fysiskt lager) |
| Arbetsfrekvensband | 868 MHz (europeiskt SRD-frekvensband, 868,0~868,6 MHz) |
| Moduleringsmetod | FSK |
| Överföringshastighet | 4,8 kbps / 16,384 kbps / 100 kbps (T-läge / S-läge / R-läge) |
| Nätverkstopologi | Stjärna (instrument → samlare/port) |
| Kommunikationsavstånd | Cirka 100~300 meter inomhus, ca 500~1000 meter i det öppna utrymmet |
| Sändningseffekt | ≤ 25 mW (+14 dBm) |

## Arbetsläge

| Läge | Betygsätt | Beskrivning |
|------|------|------|
| S-läge | 16.384 kbps | Standardläge, låg strömförbrukning, lämplig för batteridrivna instrument |
| T-läge | 100 kbps | Höghastighetsläge, lämpligt för stora trafikdatascenarier |
| R-läge | 4,8 kbps | Smalbandsläge, långdistansöverföring |

## El- och strömförbrukning

| Parameter | Värde |
|------|-----|
| Strömförsörjningsmetod | Strömförsörjning för litiumbatteri (ER18505 / ER26500 och andra industriella litiumbatterier) |
| Batteritid | 6~10 år (beroende på rapporteringsfrekvens är typiskt värde 1 gång/dag) |
| Standby strömförbrukning | Mikroamp nivå, djup sömn |
| Ta emot känslighet | Ca. -105 dBm (S-läge) |

## Miljöparametrar

| Parameter | Värde |
|------|-----|
| Drifttemperatur | -20°C ~ +60°C (industriell kvalitet) |
| Driftfuktighet | ≤ 95 % RH (icke-kondensation) |
| IP-skyddsnivå | IP65 ~ IP68 (beroende på den specifika modellen, tillämplig för utomhusinstrument) |

## Dataprotokoll

| Lager | Standard |
|----|------|
| Fysiskt lager | EN 13757-4 |
| Datalänklager | EN 13757-4 (trådlöst M-Bus ramformat) |
| Appliceringsskikt | EN 13757-3 (M-Bus appliceringsskikt) |
| Krypteringsmetod | AES-128 (valfri kryptering) |
