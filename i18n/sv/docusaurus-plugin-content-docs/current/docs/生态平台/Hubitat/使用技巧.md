# Användningstips

## Anpassad drivrutin

Heimann-enheter är optimerade för kompatibilitet i Hubitat med följande drivrutiner:

| Enhetstyp | Rekommenderad drivrutin |
|--------|--------|
| Rök/CO-larm | Generisk Zigbee Rök/CO-sensor |
| Människokroppens infraröda sensor | Heiman rörelsesensor (tredje part) |
| Dörrmagnetsensor | Generisk Zigbee kontaktsensor |
| Vattenläckagesensor | Generisk Zigbee vattenläckagesensor |
| Temperatur- och fuktighetssensor | Heiman temperatur/fuktighetssensor |

## Zigbee-optimering

- Hubitat C-8 och högre modeller använder EFR32-chip, som har den bästa kompatibiliteten
- Visa och välj kanalen med minst störningar i Inställningar → Zigbee
- Använd Zigbee sniffningsverktyg som Ubisys för att diagnostisera avbrutna samtal

## Regeluppmaningar

- Det rekommenderas att lägga till villkoret "Anti-falskt larmfördröjning" (0,5-2 sekunder) i Rule Machine
- Aktivera "hälsokontroll"-meddelanden för nyckelregler för att säkerställa att de fortsätter att köras
- Organisera komplex automation i "underregel"-läge för enkel felsökning och underhåll

## Z-Wave samexistens

Om du vill använda Z-Wave-enheter samtidigt, se till att behålla frekvensavståndet mellan Zigbee- och Z-Wave-kanalerna.
