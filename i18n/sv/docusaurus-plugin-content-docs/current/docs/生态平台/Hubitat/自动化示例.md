# 自动化示例

## Rule Machine 烟雾报警

**规则名称：** 烟雾报警安全联动

**配置：**
```
触发器: 海曼烟雾报警器 变为 active
条件:  模式 != 已解除布防
动作:
  - 开启所有灯光
  - 发送推送通知 "⚠️ 烟雾报警！"
  - 开启警笛（如有）
  - 设置模式为 "紧急"
```

## Rule Machine 温度监控

**规则名称：** 高温保护

**配置：**
```
触发器: 海曼温度传感器 > 38°C
条件:  空调状态 = 关闭
动作:
  - 开启空调，制冷模式，24°C
  - 发送通知 "室温过高，已启动空调"
```

## Rule Machine 漏水检测

**规则名称：** 漏水紧急处理

**配置：**
```
触发器: 海曼水浸探测器 变为 wet
条件:  水阀状态 = 开启
动作:
  - 关闭总水阀
  - 发送推送通知到所有手机
  - 发送短信通知（可选）
```
