# Automationsexempel

## Rule Machine Smoke Alarm

**Regelnamn:** Säkerhetskoppling för brandvarnare

**Konfiguration:**
```
Trigger: Hyman Smoke Alarm blir aktivt
Villkor: Läge != Avaktiverad
Åtgärd:
  - Tänd alla lampor
  - Skicka push-meddelande "⚠️ Brandvarnare!"
  - Slå på sirenen (om tillgänglig)
  - Ställ in läget på "Nödläge"
```

## Regel Maskintemperaturövervakning

**Regelnamn:** Högtemperaturskydd

**Konfiguration:**
```
Trigger: Heimann Temperatursensor > 38°C
Skick: Luftkonditioneringens status = av
Åtgärd:
  - Slå på luftkonditioneringen, kylläge, 24°C
  - Skicka meddelande "Rumstemperaturen är för hög och luftkonditioneringen har startats"
```

## Regel Maskinens vattenläckagedetektering

**Regelnamn:** Nödbehandling av vattenläckage

**Konfiguration:**
```
Utlösare: Hyman vattensänkningsdetektor blir blöt
Tillstånd: Vattenventilens status = öppen
Åtgärd:
  - Stäng huvudvattenventilen
  - Skicka push-meddelanden till alla mobiltelefoner
  - Skicka SMS-avisering (valfritt)
```
