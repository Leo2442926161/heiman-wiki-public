# Användningstips

## Enhetsparning

1. Ange "Inställningar → Enheter och tjänster → ZHA → Konfiguration" i Home Assistant
2. Klicka på "Lägg till enhet" och starta sedan ihopparningsläget för Heiman-enheten
3. Om ihopparningen misslyckas, för enheten nära USB-koordinatorn och försök igen inom 3 meter

## Föredragen USB-koordinator

| Modell | Chip | Rekommendationsindex |
|------|------|---------|
| Sonoff ZBDongle-E | EFR32MG21 | ⭐⭐⭐⭐⭐ |
| Conbee II | 2652 | ⭐⭐⭐⭐ |
| Elelabs Zigbee USB | 2652P | ⭐⭐⭐⭐ |

## Kanalkoordinering

ZHA standardkanal är 15. Om det finns allvarliga störningar med WiFi 2.4GHz:
- Använd `Silicon Labs Flasher Utility` för att ändra kanalen
- Vanligtvis rekommenderade kanaler: 15, 20, 25 (undvik WiFi 1/6/11)

## Nätverksoptimering

- Lägg till en Zigbee-routingenhet (som ett smart uttag) var 3-4 Heiman-enhet för att utöka nätverket
- Kontrollera enhetens anslutningskvalitet med ZHA nätverksvisualiseringsverktyg
- Kontrollera regelbundet efter uppdateringar av enhetens firmware för att bibehålla optimal kompatibilitet
