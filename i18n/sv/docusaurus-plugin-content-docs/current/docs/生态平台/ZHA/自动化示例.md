# Automationsexempel

## Brandvarnarkoppling

``` jaml
alias: "ZHA brandvarnarkoppling"
utlösare:
  - trigger: tillstånd
    entity_id: binary_sensor.heiman_smoke_zha
    till: "på"
åtgärder:
  - åtgärd: light.turn_on
    mål:
      area_id: living_room
  - action: tts.speak
    mål:
      entity_id: media_player.speaker
    data:
      meddelande: "Röklarm utlöst!"
```

## Automatisering av enhetsparning

``` jaml
alias: "Ny enhetsparningspåminnelse"
utlösare:
  - trigger: zha.event
    undertyp: heiman
    kommando: device_joined
åtgärder:
  - åtgärd: notify.persistent_notification
    data:
      meddelande: "Den nya Haiman-enheten har gått med i ZHA-nätverket, gå till konfigurationsnamnet och området"
```

## Detektering av vattennedsänkning

``` jaml
alias: "ZHA Flood Test"
utlösare:
  - trigger: tillstånd
    entity_id: binary_sensor.heiman_water_leak_zha
    till: "på"
åtgärder:
  - åtgärd: switch.turn_off
    enhets-id: switch.main_water_valve
  - åtgärd: notify.mobile_app_phone
    data:
      meddelande: "💧 Översvämning upptäckt!"
```
