# 自动化示例

## "Good Night" 就寝模式

**触发条件：** 海曼门磁传感器 → 关闭（表示已关好卧室门）

**执行动作：**
- 关闭客厅所有灯光
- 智能锁 → 上锁
- 安防系统 → 夜间布防
- 推送通知："晚安，系统已布防"

## "Away" 离家模式

**触发条件：** 海曼人体红外传感器 → 15 分钟无移动

**执行动作：**
- 所有灯光关闭
- 恒温器 → 节能模式
- 安防系统 → 离家布防
- 门磁传感器 → 开启报警
- 推送通知："离家模式已启动，门窗传感器已启用"

## 烟雾报警紧急模式

**触发条件：** 海曼烟雾报警器 → 检测到烟雾

**执行动作：**
- 智能扬声器 → 播放报警音
- 所有智能灯 → 红色闪烁
- 推送通知至所有家庭成员
- 若连接 SmartThings Hub → 触发警笛
