# Automationsexempel

## "God natt" läggdagsläge

**Utlösningsvillkor:** Haiman dörrmagnetisk sensor → Stäng (indikerar att sovrumsdörren har stängts)

**Utför åtgärd:**
- Släck allt ljus i vardagsrummet
- Smart lås → Lås
- Säkerhetssystem → arm på natten
- Push-meddelande: "God natt, systemet är tillkopplat"

## "Borta"-läge

**Triggertillstånd:** Hymans infraröda sensor för människokroppen → 15 minuter utan rörelse

**Utför åtgärd:**
- Alla lampor släcktes
- Termostat → Energisparläge
- Säkerhetssystem → tillkoppling borta från hemmet
- Dörrsensor → slå på larm
- Push-meddelande: "Away from home-läget är aktiverat, dörr- och fönstersensorer är aktiverade"

## Brandvarnare nödläge

**Utlösande tillstånd:** Hymans röklarm → Rök upptäckt

**Utför åtgärd:**
- Smart högtalare → spela larmljud
- Alla smarta lampor → blinkar rött
- Push-meddelanden till alla familjemedlemmar
- Om den är ansluten till SmartThings Hub → utlös siren
