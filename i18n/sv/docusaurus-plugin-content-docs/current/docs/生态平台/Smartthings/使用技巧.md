# Användningstips

## Hubkompatibilitet

Hyman Zigbee-enheter är kompatibla med **SmartThings Hub v2** och **Hub v3 (Aeotec)**.

| Navmodell | Kompatibilitet | Beskrivning |
|--------|--------|------|
| Hub v2 (STH-ETH-250) | ✅ | Kräver uppdatering av firmware till den senaste |
| Hub v3 (Aeotec) | ✅ | Native support, rekommenderas |
| Samsung SmartThings Station | ⚠️ | Stöder endast vissa Zigbee-enheter |

## Uppmaningar om enhetsparning

- Klicka på "+" → "Enhet" → "Varumärke" → "Hyman" i SmartThings-appen
- Enheter som inte känns igen automatiskt kan sökas manuellt genom "Scan Zigbee"
- Det rekommenderas att hålla avståndet mellan enheten och hubben inom 5 meter för första parning

## Batterioptimering

Standardrapporteringsintervallet för Heiman-sensorn är längre. Om realtidsprestanda krävs kan "detektionsintervallet" justeras till 5 minuter på enhetssidan.

## Bästa metoder för automation

- Använd SmartThings "Modes"-funktion för att skilja mellan "hemma/borta hemifrån" tillstånd
- Skapa **två uppsättningar** av automatisering för nyckelsensorer (rök, CO) för redundans
