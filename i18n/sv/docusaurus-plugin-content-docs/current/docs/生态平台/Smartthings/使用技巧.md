# 使用技巧

## Hub 兼容性

海曼 Zigbee 设备兼容 **SmartThings Hub v2** 和 **Hub v3 (Aeotec)**。

| Hub 型号 | 兼容性 | 说明 |
|---------|--------|------|
| Hub v2 (STH-ETH-250) | ✅ | 需固件更新至最新 |
| Hub v3 (Aeotec) | ✅ | 原生支持，推荐使用 |
| Samsung SmartThings Station | ⚠️ | 仅支持部分 Zigbee 设备 |

## 设备配对提示

- 在 SmartThings App 中点击 "+" → "设备" → "品牌" → "海曼"
- 未被自动识别的设备可通过 "Scan Zigbee" 手动搜索
- 建议将设备与 Hub 的距离控制在 5 米以内进行首次配对

## 电池优化

海曼传感器默认上报间隔较长。如需实时性，可在设备页面调整"检测间隔"为 5 分钟。

## 自动化最佳实践

- 使用 SmartThings 的"模式"功能区分"在家/离家"状态
- 为关键传感器（烟雾、CO）创建**两组**自动化以冗余保障
