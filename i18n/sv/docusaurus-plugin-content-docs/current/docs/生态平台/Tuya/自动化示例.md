# Automationsexempel

## Brandvarnarscen

**Scennamn:** "Smoke Alarm Linkage"

**Utlösande tillstånd:** Hymans röklarm → Rök upptäckt

**Utför åtgärd:**
- Smart uttag → öppen (anslut frånluftsfläkt)
- Lampor → tänds och blinkar
- Push-meddelanden till mobiltelefonen
- Högtalare → Spela upp alarmljud

## Ett klick scen för att lämna hemmet (Tap-to-Run)

**Scennamn:** "Arrangemang efter att ha lämnat hemmet"

**Manuell körning:** Tryck eller röstutlösare

**Utför åtgärd:**
- Haiman dörrsensor → slå på larm
- Hyman Human Body Infrared → Slå på detektion
- Hyman vattennedsänkningsdetektor → slå på detektering
- Alla lampor → släckta
- Smart lås → Lås

## Automatisk temperatur- och luftfuktighetskontroll

**Automatisk typ:** Automatisk tillståndsutlösare

**Triggertillstånd:** Hyman temperatur- och fuktighetssensor Fuktighet > 75 %

**Utför åtgärd:**
- Smart uttag (avfuktare) → På
- Push-meddelande: "Fuktigheten är för hög, avfuktaren har slagits på"

**Triggertillstånd:** Hyman temperatur- och fuktighetssensor Fuktighet < 50 %

**Utför åtgärd:**
- Smart uttag (avfuktare) → Stäng
