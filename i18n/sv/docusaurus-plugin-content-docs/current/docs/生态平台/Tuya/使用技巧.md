# Användningstips

## Tap-to-Run vs Automation

| Skriv | Triggermetod | Tillämpliga scenarier |
|------|--------|--------|
| **Tap-to-Run** | Manuellt klick/röst | Att lämna hemmet, komma hem, filmvisningsläge, etc. |
| **Automatisering** | Automatisk sensorutlösning | Brandvarnare, detektering av vattenläckage, etc. |

Utrustning av Heiman-sensortyp bör använda "Automation", och scentypkrav bör använda "Tap-to-Run".

## Moln till lokalt

- Slå på "LAN-kommunikation" i Smart Life-appen för att minska molnberoendet
– Vissa Heiman-enheter stöder lokal exekvering och kan länkas även om nätverket är frånkopplat.
- Inställningsväg: Scen → Inställningar i det övre högra hörnet → "Aktivera lokal körning"

## Enhetsdelning

- Dela Heiman-enheter med familjemedlemmar i Smart Life-appen
- Delade användare kan ta emot larmmeddelanden men kan inte ändra enhetskonfigurationer
- Dela med upp till 20 familjemedlemmar

## Tredjepartsplattformsåtkomst

Tuya-konton kan bindas till Alexa och Google Home för att uppnå röststyrning över plattformar. Bindande väg: Smart Life → Inställningar → Tredjepartskontroll.
