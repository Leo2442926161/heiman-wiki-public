# Automationsexempel

## Regler för brandvarnare

```javascript
regel "Röklarm utlöst"
när
    Objekt heiman_smoke_alarm ändrat till PÅ
sedan
    sendCommand(Vardagsrum_Ljus, PÅ)
    sendNotification("admin", "⚠️ Brandvarnare utlöst!")
    postUpdate(Alarm_State, "smoke_alert")
slut
```

## Regler för temperatursensor

```javascript
regel "Temperatur och fuktighetsrapport"
när
    Objekt heiman_temperature mottagen uppdatering
sedan
    val temp = (heiman_temperatur.tillstånd som Number).floatValue
    val hum = (heiman_humidity.state as Number).floatValue
    if (temp > 40) {
        sendCommand(Luftkonditionering, PÅ)
        sendNotification("admin", "Temperaturen är för hög (" + temp + "°C), luftkonditioneringen har slagits på")
    }
    if (brum > 80) {
        sendNotification("admin", "Luftfuktigheten är för hög (" + hum + "%), var uppmärksam på avfuktning")
    }
slut
```

## Regler för upptäckt av översvämningar

```javascript
regel "Länkning för detektering av vattennedsänkning"
när
    Objekt heiman_water_leak ändras till PÅ
sedan
    sendCommand(Water_Valve, OFF)
    sendBroadcastNotification("💧 Vattenläckage upptäckt! Vattenventilen har stängts!")
slut
```
