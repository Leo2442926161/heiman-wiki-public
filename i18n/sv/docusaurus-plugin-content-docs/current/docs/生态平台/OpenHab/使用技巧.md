# Användningstips

## Sakkonfiguration

Lägg till Heiman-enhet via MQTT-bindning:

``` java
Sak mqtt:topic:heiman_smoke "Heiman brandvarnare" @ "vardagsrum" {
    Kanaler:
        Skriv binärt: smoke "smoke state" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.smoke" ]
        Typnummer : batteri "Batteri" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.battery" ]
}
```

## Objektdefinition

``` java
Kontakta Smoke_Alarm "Röklarm [MAP(smoke.map):%s]" { channel="mqtt:topic:heiman_smoke:smoke" }
Number Smoke_Battery "Rökvarnarbatteri [%d%%]" { channel="mqtt:topic:heiman_smoke:battery" }
```

## Koordinatval

Det rekommenderas att använda en USB-koordinator med **CC2531** eller **CC2652**-chip och se till att firmwareversionen är den senaste.

## Felsökningstips

- Se Things "online/offline"-status i Paper UI
- Använd `openhab-cli info` för att kontrollera om bindningarna är korrekt laddade
- Sök efter nyckelordet "heiman" i loggen för att felsöka problem med enhetens kommunikation
