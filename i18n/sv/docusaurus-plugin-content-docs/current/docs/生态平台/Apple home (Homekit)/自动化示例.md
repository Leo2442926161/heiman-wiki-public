# Automationsexempel

## Säkerhetslarm för rökdetektering

**HomeKit automatisk konfiguration:**

1. **Utlösande tillstånd:** Hyman brandvarnare → rök upptäckt
2. **Säkerhetsinställningar:** Ändra säkerhetssystemets status till "Alarm"
3. **Utför åtgärd:**
   - HomePod → Spela anpassat ljud "Rök upptäckt, var uppmärksam på säkerheten"
   - Alla smarta lampor → lyser och blinkar
   - Push-meddelanden till alla familjemedlemmar

## Hemma/borta hemifrån

**Kommando för genvägar: "Lämna säkerhetskontroll i hemmet"**

```
när den sista personen lämnar hemmet
  Konfigurera säkerhetssystem → aktivera när du lämnar hemmet
  Konfigurera Heiman Dörrsensor → Slå på aviseringar
  Ställ in Heiman kroppssensor → slå på larm
  Släck alla lampor
  Inställningar Termostat → Energisparläge
  Skicka meddelande "🏠 Away from home-läge aktiverat"
```

## Påminnelse om temperaturavvikelse

**Triggertillstånd:** Hyman temperatur- och fuktighetssensor → Temperatur > 35°C

**Utför åtgärd:**
- HomePod → Röstsändningsrumstemperaturen överskrider gränsen
- Push-meddelanden till iPhone
- Slå på luftkonditioneringen automatiskt (om HomeBridge är konfigurerad)
