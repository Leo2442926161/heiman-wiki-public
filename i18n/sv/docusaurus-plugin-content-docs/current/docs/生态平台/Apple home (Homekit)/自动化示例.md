# 自动化示例

## 烟雾检测安防报警

**HomeKit 自动化配置：**

1. **触发条件：** 海曼烟雾报警器 → 检测到烟雾
2. **安全设置：** 将安防系统状态切换为"警报"
3. **执行动作：**
   - HomePod → 播放自定义音频 "检测到烟雾，请注意安全"
   - 所有智能灯 → 亮起并闪烁
   - 推送通知到所有家庭成员

## 在家/离家场景

**Shortcuts 快捷指令："离家安检"**

```
当 最后一个人离开家
  设置 安防系统 → 离家布防
  设置 海曼门磁 → 开启通知
  设置 海曼人体传感器 → 开启报警
  关闭 所有灯光
  设置 恒温器 → 节能模式
  发送通知 "🏠 离家模式已启动"
```

## 温度异常提醒

**触发条件：** 海曼温湿度传感器 → 温度 > 35°C

**执行动作：**
- HomePod → 语音播报室温超限
- 推送通知到 iPhone
- 自动开启空调（如已配置 HomeBridge）
