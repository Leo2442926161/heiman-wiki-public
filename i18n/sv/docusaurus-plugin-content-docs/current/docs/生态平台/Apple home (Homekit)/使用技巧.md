# Användningstips

## Homebridge vs Matter

Heiman-enheter kan ansluta till Apple Home på två sätt:

| Metod | Fördelar | Nackdelar |
|------|------|------|
| **Materia över tråd** | Native support, ingen brygga krävs, låg latens | Kräver trådkantsrouter |
| **Homebridge-plugin** | Kompatibel med alla Zigbee-enheter | Kräver ytterligare drift av Homebridge-tjänsten |

## Trådnätverkskonstruktion

1. Minst en trådkantsrouter krävs (Apple HomePod mini, Apple TV 4K 3:e generationen)
2. Hyman Matter-enheter ansluter automatiskt till trådnätverket inom 10 meter
3. Använd **Nam* eller **Eve App** för att se trådnätverkstopologin

## Automatisk optimering

- Apple Homes automatiseringskategori "Security" kan kopplas direkt till säkerhetsstatus
- Använd kommandot "Transformera" för att få lamporna att ändras smidigt istället för att växla direkt
- Skapa plattformsoberoende länkar via Shorcuts App (som brandvarnare → skicka SMS)

## Vanliga frågor

- När Matter-enheten visar "Reagerar inte", starta om Apple TV/HomePod för att återställa.
