# 自动化示例

## Alexa Guard 集成（美国区）

**前提：** 订阅 Alexa Guard Plus（美国区可用）

**配置流程：**
1. 在 Alexa App 中开启 Guard Plus
2. 添加海曼烟雾/CO 报警器为"智能家居设备"
3. 启用"烟雾警报检测"智能响应

**自动响应：**
- 烟雾报警触发 → Alexa 自动拨打紧急联系人
- CO 报警触发 → Alexa 全屋语音警告，自动通风
- 门窗异常开启 → 推送警报通知

## 烟雾检测例行程序

**触发：** 海曼烟雾报警器 → 检测到烟雾

**执行动作：**
- Alexa 语音播报："⚠️ 检测到烟雾，请注意安全"
- 所有 Echo 设备 → 播放警报音
- 智能灯组 → 全部亮起
- 推送通知到手机

## "离家" 例行程序

**触发：** 语音命令 "Alexa, I'm leaving"

**执行动作：**
- 海曼门磁传感器 → 开启布防
- 海曼人体传感器 → 开启移动检测
- 海曼水浸探测器 → 开启检测
- 所有灯 → 关闭
- Alexa 回复："已开启安防系统，一路平安"
