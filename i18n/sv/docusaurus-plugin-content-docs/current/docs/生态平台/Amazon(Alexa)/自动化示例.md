# Automationsexempel

## Alexa Guard integration (USA)

**Förutsättning:** Prenumerera på Alexa Guard Plus (tillgängligt i USA)

**Konfigurationsprocess:**
1. Slå på Guard Plus i Alexa-appen
2. Lägg till Hyman Smoke/CO Alarm som en "Smart Home Device"
3. Aktivera "Smoke Alarm Detection" smart svar

**Automatiskt svar:**
- Brandvarnare utlöst → Alexa ringer automatiskt nödkontakter
- CO-larmutlösare → Alexa röstvarning för hela huset, automatisk ventilation
- Dörrar och fönster öppnas onormalt → Tryck larmmeddelande

## Rutin för rökdetektion

**Trigger:** Hymans röklarm → Rök upptäckt

**Utför åtgärd:**
- Alexa röstmeddelande: "⚠️ Rök upptäckt, var uppmärksam på säkerheten"
- Alla Echo-enheter → Spela larmton
- Smart ljusset → allt lyser
- Push-meddelanden till mobiltelefonen

## Rutin "Leaving Home".

**Trigger:** Röstkommando "Alexa, jag åker"

**Utför åtgärd:**
- Heiman dörrmagnetsensor → slå på armering
- Heiman Kroppssensor → Slå på rörelsedetektering
- Hyman vattennedsänkningsdetektor → slå på detektering
- Alla lampor → släckta
- Alexa svarade: "Säkerhetssystemet är påslaget, ha en säker resa"
