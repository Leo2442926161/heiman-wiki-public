# Användningstips

## Enhetsupptäckt

- När Heiman Matter-enheten är påslagen kommer den automatiskt att upptäckas av Alexa-appen, vanligtvis inom 30 sekunder
- Om det inte upptäcks automatiskt: Gå in i Alexa App → Enhet → "+" → "Lägg till enhet" → "Matter"
- Använd Matter-konfigurationskoden eller QR-koden på enhetens undersida för att slutföra ihopparningen

## Zigbee-enhetsåtkomst (Echo Plus / Echo Show)

- Echo-enheter med Zigbee Hub kan paras direkt med Heiman Zigbee-sensorer
- Parningsläge: Alexa App → Enhet → "+" → "Zigbee" → "Sök enhet"
- Heimann-enheter är Alexa-kompatibla med inbyggd Zigbee-koordinator

## Rutintips

- Använd "anpassade åtgärder" för att sända mer detaljerade röstmeddelanden
- Lägg till "vänta 5 sekunder" i nödrutinen innan du utför efterföljande åtgärder för att undvika röstöverlappning
- Dra nytta av funktionen "Sound Detection" för att ge Alexa ytterligare svar på Heyman Alarms högljudda larm

## Alexa Guard OBS

Alexa Guard Plus är för närvarande endast tillgängligt i USA. Användare i Kina kan simulera vissa funktioner genom Rutin.
