# 使用技巧

## 设备发现

- 海曼 Matter 设备上电后，Alexa App 通常会在 30 秒内自动发现
- 若未自动发现：进入 Alexa App → 设备 → "+" → "添加设备" → "Matter"
- 使用设备底部的 Matter 设置码或二维码完成配对

## Zigbee 设备接入（Echo Plus / Echo Show）

- 带 Zigbee Hub 的 Echo 设备可直接配对海曼 Zigbee 传感器
- 配对模式：Alexa App → 设备 → "+" → "Zigbee" → "搜索设备"
- 海曼设备兼容 Alexa 内置 Zigbee 协调器

## Routine 技巧

- 使用"自定义动作"播报更详细的语音提示内容
- 突发事件例行程序中添加"等待 5 秒"后再执行后续动作，避免语音重叠
- 利用"声音检测"功能，使 Alexa 对海曼报警器的高音警报做出额外响应

## Alexa Guard 注意

Alexa Guard Plus 目前仅在美国区可用，中国区用户可通过 Routine 模拟部分功能。
