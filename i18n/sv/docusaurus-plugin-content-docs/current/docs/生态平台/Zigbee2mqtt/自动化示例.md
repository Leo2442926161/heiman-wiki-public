# 自动化示例

## MQTT 订阅设备状态

```bash
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
```

输出示例：
```
zigbee2mqtt/heiman_smoke  {"smoke":true,"battery":100,"linkquality":120}
```

## Node-RED 烟雾报警流

[{"id":"1","type":"mqtt in","topic":"zigbee2mqtt/heiman_smoke","name":"烟雾报警","qos":1,"broker":"localhost","x":200,"y":200,"wires":[["2"]]},{"id":"2","type":"switch","name":"检测烟雾","property":"payload.smoke","conditions":[{"t":"eq","v":"true"}],"x":350,"y":200,"wires":[["3"]]},{"id":"3","type":"notification","name":"推送通知","title":"烟雾报警","message":"检测到烟雾！","x":500,"y":200,"wires":[]}]

## 设备在线监控

```yaml
# configuration.yaml
binary_sensor:
  - platform: mqtt
    name: "烟雾报警器在线"
    state_topic: "zigbee2mqtt/heiman_smoke/availability"
    payload_on: "online"
    payload_off: "offline"
    device_class: connectivity
```
