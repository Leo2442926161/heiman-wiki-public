# Användningstips

## Kanalval

Hyman Zigbee-enheter använder kanal 15 som standard. Det rekommenderas att ange manuellt i `configuration.yaml`:

``` jaml
avancerad:
  kanal: 15
```

Undvik överlappning med WiFi 2,4GHz kanal 1/6/11. Använd en WiFi-analysator för att skanna och välja Zigbee-kanalen med minsta störningar.

## Säkerhetskopiering av nätverksnyckel

**Se till att säkerhetskopiera `coordinator_backup.txt` och `secret.yaml`: **

``` bash
cp data/coordinator_backup.txt data/coordinator_backup.bak
cp data/secret.yaml data/secret.yaml.bak
```

Alla ihopparade enheter måste gå med igen efter att ha tappat bort nyckeln.

## Enhetsspecifika optimeringar

- Heiman-enheter rapporterar batteriström var 60:e minut som standard, och intervallet kan förkortas genom "rapporterings"-konfiguration
- Lade till "homeassistant: true"-konfiguration för att automatiskt generera entiteter utan att manuellt skriva MQTT-upptäcktskonfiguration
- Använd `friendly_name` för att ställa in ett läsbart namn för Heiman-enheten

## Uppdatera firmware

Vissa Heiman Zigbee-enheter stöder OTA-uppdateringar, och alternativet `ota` måste vara aktiverat i `configuration.yaml`.
