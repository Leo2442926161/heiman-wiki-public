# 使用技巧

## 信道选择

海曼 Zigbee 设备默认使用信道 15。建议在 `configuration.yaml` 中手动指定：

```yaml
advanced:
  channel: 15
```

避免与 WiFi 2.4GHz 信道 1/6/11 重叠。使用 WiFi 分析仪扫描后选择干扰最少的 Zigbee 信道。

## 网络密钥备份

**务必备份 `coordinator_backup.txt` 和 `secret.yaml`：**

```bash
cp data/coordinator_backup.txt data/coordinator_backup.bak
cp data/secret.yaml data/secret.yaml.bak
```

丢失密钥后所有已配对的设备需重新加入。

## 设备特定优化

- 海曼设备默认每 60 分钟上报一次电池电量，可通过 `reporting` 配置缩短间隔
- 添加 `homeassistant: true` 配置可自动生成实体，无需手动编写 MQTT 发现配置
- 使用 `friendly_name` 为海曼设备设置可读名称

## 更新固件

部分海曼 Zigbee 设备支持 OTA 更新，需在 `configuration.yaml` 中开启 `ota` 选项。
