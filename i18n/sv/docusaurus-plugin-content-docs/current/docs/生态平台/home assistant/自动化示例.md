# Automationsexempel

## Brandvarnare utlöser lampor i hela huset

``` jaml
alias: "Rökvarnare - lampor och aviseringar i hela huset"
utlösare:
  - trigger: tillstånd
    entity_id: binary_sensor.heiman_smoke_alarm
    till: "på"
åtgärder:
  - åtgärd: light.turn_on
    mål:
      entity_id: alla
  - åtgärd: notify.mobile_app_phone
    data:
      meddelande: "⚠️ Brandvarnare utlöst! Kontrollera omedelbart!"
      titel: "Rökvarnare"
```

## CO-larmkoppling

``` jaml
alias: "CO-larm - stäng av gasen och öppna fönster"
utlösare:
  - trigger: tillstånd
    entity_id: binary_sensor.heiman_co_alarm
    till: "på"
åtgärder:
  - åtgärd: switch.turn_off
    enhets-id: switch.gas_fireplace
  - åtgärd: cover.open
    enhets-id: cover.living_room_window
  - åtgärd: notify.persistent_notification
    data:
      meddelande: "Kolmonoxidlarmet har utlösts, gasventilen har stängts och fönstren har öppnats"
```

## Länk för vattensänkningsdetektor

``` jaml
alias: "Flood Detection - Close Water Valve"

  - trigger: tillstånd

```

## Tillkoppling borta från hemmet - människokroppssensor utlöser larm

``` jaml
alias: "Lämna hemläget - larm för mänsklig kroppssensor"
utlösare:
  - trigger: tillstånd
    entity_id: binary_sensor.heiman_motion_sensor
    till: "på"
villkor:
  - tillstånd: tillstånd
    entity_id: alarm_control_panel.home_alarm
    tillstånd: beväpnad_bort
åtgärder:
  - åtgärd: alarm_control_panel.trigger
    entity_id: alarm_control_panel.home_alarm
```
