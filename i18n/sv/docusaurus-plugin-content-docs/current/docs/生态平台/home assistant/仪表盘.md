# Konfiguration av instrumentpanelen

## Säkerhetsöversiktspanel

Använd layouten för sektionsvy för att organisera Heimann-utrustning i områden:

``` jaml
typ: avsnitt
titel: Rök- och kolmonoxidlarm
kort:
  - typ: enhet
    enhet: binary_sensor.heiman_smoke_alarm
    namn: brandvarnare
  - typ: enhet
    enhet: binary_sensor.heiman_co_alarm
    namn: CO-larm
```

## Svampkortstil

``` jaml
typ: anpassad:svamp-larm-kontrollpanel-kort
enhet: alarm_control_panel.home_alarm
namn: säkerhetssystem

typ: custom: mushroom-entity-card
enhet: binary_sensor.heiman_smoke_alarm
namn: brandvarnare
icon: mdi:rökdetektor
primär_info: tillstånd
sekundär_info: senast ändrad
fill_container: sant
```

## Miljösensorpanel

``` jaml
typ: avsnitt
titel: Miljöövervakning
kort:
  - typ: mätare
    enhet: sensor.heiman_temperatur
    namn: temperatur
    enhet: "°C"
    min: 0
    max: 50
    svårighetsgrad:
      grönt: 0
      gul: 30
      Röd: 40
  - typ: mätare
    enhet: sensor.heiman_humidity
    namn: fuktighet
    enhet: "%"
    min: 0
    max: 100
    svårighetsgrad:
      grönt: 30
      Gul: 60
      Röd: 80
```

## Larmhistorikpanel

``` jaml
typ: historia-graf
titel: Sensorhistorik
enheter:
  - binary_sensor.heiman_smoke_alarm
  - binary_sensor.heiman_co_alarm
  - binary_sensor.heiman_water_leak
timmar_att_visa: 24
```

## Föreslagen layout

- Överst: Säkerhetssystemets status + larmenhetens statuskort
- Central: Miljöövervakning (temperatur, luftfuktighet, vattenkvalitet)
- Nederst: Sensorlarm historiskt trenddiagram
- Genvägar i sidofältet: aktivera/avaktivera, scenbyte
