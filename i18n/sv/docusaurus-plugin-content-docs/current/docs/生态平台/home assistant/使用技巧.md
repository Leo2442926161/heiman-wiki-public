# Användningstips

## Enhetskonfiguration

- När du parkopplar din Heiman Matter-enhet för första gången, se till att din gränsrouter (t.ex. HomePod, Nest Hub, Home Assistant Green) är på samma trådnätverk
- När du integrerar Zigbee-enheter via ZHA eller Zigbee2MQTT, rekommenderas att du först använder USB-koordinatorns rekommenderade firmware (som Sonoff ZBDongle-E, Conbee II)

## Automatisk optimering

- Lade till parametern "för:" (t.ex. 3 sekunder) för röklarmsautomatisering för att förhindra omedelbara falsklarm
- Använd `villkor: tillstånd` för att matcha säkerhetsläget för att undvika onödiga meddelanden i aktiverat tillstånd
- Använd "grupp"-funktionen i Home Assistant för att gruppera Heiman-sensorer efter område för enkel batch-drift

## Felsökning

- När enheten är offline, kontrollera först om Zigbees nätverkskanal är i konflikt med WiFi 2.4GHz
- När Matter-enheten inte kan upptäckas, starta om gränsroutern och lägg till enheten igen
