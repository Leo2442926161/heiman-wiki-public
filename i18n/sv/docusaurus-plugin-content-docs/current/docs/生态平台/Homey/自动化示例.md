# Automationsexempel

## Brandvarnare - blixtljus och meddela

**Flödeskonfiguration:**

1. **Utlösare:** Hymans röklarm → Status ändras till `larm`
2. **Villkor:** (Inga)
3. **Åtgärd:**
   - Vardagsrumsljus → blinkar (röd, en gång per sekund)
   - Push Notification → Mobil: "⚠️ Brandvarnare utlöst!"
   - Röstmeddelande → Hemtrevlig högtalare: "Rök upptäckt, var uppmärksam på säkerheten!"

## CO-larm - Stäng av pannan

**Flödeskonfiguration:**

1. **Trigger:** Hyman CO-larm → Status ändras till `larm`
2. **Villkor:** (Inga)
3. **Åtgärd:**
   - Pannbrytare → AV
   - Smart uttag (friskluftssystem) → På
   - Push-meddelande → "Kolmonoxidlarm, pannan har stängts av"

## Temperatur- och luftfuktighetsövergränslarm

1. **Trigger:** Hyman temperatur- och fuktighetssensor → Temperatur över 40°C
2. **Åtgärd:**
   - Luftkonditionering → Slå på kylläget
   - Push-meddelande → "Rumstemperaturen är för hög, luftkonditioneringen har slagits på automatiskt"
