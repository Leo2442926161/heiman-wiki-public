# Användningstips

## Zigbee-kanalstörningar

Homeys inbyggda Zigbee-chip använder specifika kanaler. Om Heiman-enheten ofta faller offline:

1. Visa den aktuella kanalen i Homey Settings → Zigbee
2. Använd Homey Insight-appen för att skanna efter miljöstörningar
3. Byt till en kanal med mindre störningar (bort från WiFi 1/6/11)

## Distributionsnätverk för utrustning

- Hur man går in i parningsläge för Heiman-enheter: tryck och håll ned knappen i 5 sekunder (vissa modeller måste trycka 3 gånger snabbt)
- När du lägger till en enhet i Homey-appen, välj "Lägg till enhet → Zigbee → Sök"
- När ihopparningen misslyckas, för enheten närmare Homey (inom 3 meter) och försök igen

## Förslag på flödesoptimering

- Lägg till "fördröjning" i flödet (t.ex. 2 sekunder) för att undvika övergående falska positiva
- Använd "tag"-funktionen för att klassificera och hantera Heimans utrustning

## Förlängning av signalintervall

Om Heiman-enheten är långt borta från Homey kan en Zigbee-reläenhet (som ett smart uttag) läggas till för att utöka nätverkstäckningen.
