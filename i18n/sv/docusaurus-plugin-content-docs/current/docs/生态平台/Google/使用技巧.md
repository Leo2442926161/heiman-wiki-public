# Användningstips

## Enhetssynkronisering

- När Heiman-enheten är ansluten till Google Home via Matter synkroniseras den automatiskt med Google Home-appen
- Om enheten inte visas klickar du på "Inställningar" → "Enheter" → "Lägg till enhet" → "Matter-kompatibla enheter" i Google Home-appen
- Om Matter QR-koden tappas bort kan den läggas till manuellt med den 11-siffriga inställningskoden på din enhet

## Rutinmässig optimering

- Lade till "Meddela väder och bekräfta tillkopplingsstatus" till rutinen "Leave Home".
- Använd funktionen "Voice Broadcast" för att meddela hela huset när en brandvarnare uppstår
- Använd "Start/Stopp"-åtgärden för att styra aktivering och inaktivering av Heiman-sensorns detekteringsstatus

## Hantering av enhetsplats

- Tilldela specifika rum till Heimann-sensorer (som "Köksröklarm") för att stödja mer exakt röststyrning
- Inkludera flera Heiman-enheter i "Home Device Group" för att aktivera och stänga av helt med ett klick

## Nätverkskrav

- Matter over Thread-enheter kräver Google Nest Hub (2:a generationen) eller Nest Hub Max som gränsrouter
- Zigbee-enheter ansluts via Google Nest Connect eller kompatibel Hub
