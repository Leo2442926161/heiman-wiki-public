# Automationsexempel

## "Jag är ute" rutin

**Trigger:** Säg "Jag är ute" till Google Home

**Utför åtgärd:**
- Heiman dörrmagnetsensor → arm
- Heiman människokroppens infraröda sensor → Slå på hemavgångsdetektering
- Stäng av alla lampor och apparater
- Termostat → Ställ in på energibesparande temperatur
- Google Nest Hub → Visa bekräftelsekort för att lämna hemmet
- Röstsvar: "Lämna hemläget har aktiverats, ha en säker resa"

## "Smoke Detection"-sändning

**Trigger:** Hymans röklarm → Rök upptäckt

**Utför åtgärd:**
- Google Nest-högtalare → Hela husets sändning "⚠️ Rök upptäckt! Kontrollera omedelbart!"
- Alla smarta lampor → På (via Google Home smart light set)
- Push-aviseringar till familjemedlemmars mobiltelefoner
- Kamera → Börja spela in

## Sängdagsläge

**Trigger:** Säg "Godnatt" till Google Home

**Utför åtgärd:**
- Heiman dörrsensor → Slå på dörröppningslarmet
- Heiman människokroppen infraröd → slå på nattdetektering
- Ljus → släcks gradvis
- Röstsvar: "God natt, säkerhetssystemet är på"
