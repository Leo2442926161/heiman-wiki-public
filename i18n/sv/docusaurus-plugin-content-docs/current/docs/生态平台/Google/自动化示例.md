# 自动化示例

## "我出门了" 例行程序

**触发：** 对 Google Home 说 "我出门了"

**执行动作：**
- 海曼门磁传感器 → 布防
- 海曼人体红外传感器 → 开启离家检测
- 关闭所有灯光和电器
- 恒温器 → 调至节能温度
- Google Nest Hub → 显示离家确认卡片
- 语音回复："已开启离家模式，一路平安"

## "烟雾检测" 广播

**触发：** 海曼烟雾报警器 → 检测到烟雾

**执行动作：**
- Google Nest 音箱 → 全屋广播 "⚠️ 检测到烟雾！请立即检查！"
- 所有智能灯 → 亮起（通过 Google Home 智能灯组）
- 推送通知到家庭成员手机
- 摄像头 → 开始录制

## 就寝模式

**触发：** 对 Google Home 说 "晚安"

**执行动作：**
- 海曼门磁 → 开启门开报警
- 海曼人体红外 → 开启夜间检测
- 灯光 → 逐渐关闭
- 语音回复："晚安，安防系统已开启"
