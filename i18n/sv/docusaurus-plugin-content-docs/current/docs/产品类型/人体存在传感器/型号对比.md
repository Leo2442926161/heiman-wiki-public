# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Detektionsteknik | Installation | Funktioner | Spelförslag |
|------|------|--------|------|------|----------|
| [HS8OS](../../Wireless Protocol/Zigbee/HS8OS - Mänsklig närvarosensor (Zigbee)) | Zigbee | Millimetervågsradar | Tak | Stationär mänsklig upptäckt | Mer exakt än PIR, badrummet/sovrummet står stilla under lång tid, vilket löser problemet med att PIR släcker lamporna utan någon |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Håll lamporna tända i badrummet | HS8OS | Lamporna är tända när människor är där och släckta när människor går (upptäck mikroandning för att lösa smärtpunkten för PIR stationära lampor av) |
| Sovrumsövervakning | HS8OS | Upptäck om det finns någon på sängen och koppla ihop luftkonditioneringen för att automatiskt justera temperaturen |
| Smart äldreomsorg | HS8OS | Tidsgräns för toaletter för äldre → Systemvarning meddelar familjemedlemmar för att förhindra olyckor |
| Konferensrum energibesparing | HS8OS | Ingen når N minuter → Stäng automatiskt av ljus och luftkonditionering för att spara energi och minska förbrukningen |

## Urvalsförslag

- **Lösning för PIR lyser ut smärtpunkter**: [HS8OS](../../Wireless Protocol/Zigbee/HS8OS - Människokroppsnärvarosensor (Zigbee)) (millimetervåg, stillhetsdetektering)
- **Parad med PIR**: PIR detekterar rörelse + HS8OS detekterar stationär närvaro, kompletterande användning för bästa resultat
