# Installationsguide

## Krav på installationsplats

- Installera dörrklockan på en höjd av cirka **1,4 ~ 1,6 m** (i ögonhöjd)
- Vänd dig mot dörröppningen för att säkerställa att du kan fånga besökarens ansikte
- Undvik direkt solljus till kameran

## Installationssteg

1. Välj installationsplats (bredvid dörren eller på dörrkarmen)
2. Fäst monteringsbasen med skruvar
3. Anslut strömförsörjningen (anslut den trådbundna versionen, installera batteriet i batteriversionen)
4. Montera dörrklockans kropp på basen
5. Para ihop och anslut till Wi-Fi via APP (Wi-Fi-version)
6. Justera kameravinkeln
7. Testa dörrklockans knappar, video och röstfunktioner

## Anteckningar

- Wi-Fi-versionen kräver god Wi-Fi-signaltäckning vid dörren
- Var uppmärksam på laddningscykeln för batteriversionen, och det rekommenderas att använda den med dörrsensorn (för att upptäcka och vakna efter att dörren har öppnats)
- Rengör kameralinsen regelbundet
- Se till att dörrklockan är inom giltigt Zigbee-signalområde
