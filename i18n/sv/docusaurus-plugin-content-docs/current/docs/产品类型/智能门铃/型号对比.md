# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Video | Strömförsörjning | Funktioner | Spelförslag |
|------|------|------|------|------|----------|
| Smart dörrklocka | Zigbee + Wi-Fi | 1080P tillval | Batteri/nätverk | Dörrklocka + säkerhet två-i-ett | Besökare som ringer mobiltelefonavisering, visuell fjärrintercom, inspelning av rörelsedetektering |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Expressleverans/avhämtningshantering | Smart dörrklocka | Ring på klockan → mobiltelefonavisering → fjärrintercombekräftelse → fjärrupplåsning (länkat smart lås) |
| Säkerhetsövervakning | Smart dörrklocka | Detektering av dörrstag → Video + Push-larm, länkad dörrsensor/belysning |
| Besökskoppling | Smart dörrklocka | Ring på klockan → Inomhusskärm/högtalardisplay → Slå på lampor → Lås upp länkage |
