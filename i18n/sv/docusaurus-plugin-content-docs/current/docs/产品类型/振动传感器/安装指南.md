# Installationsguide

## Krav på installationsplats

- Installerad på ytan av föremål som behöver övervaka vibrationer
- Vanliga installationsplatser: **Innerdörrar och fönster**, **skåp**, **rör**, **väggar**, **staket**
- Se till att installationsytan är plan och ren

## Installationssteg

1. Välj objektet som ska övervakas (t.ex. insidan av kassaskåpsdörren)
2. Rengör monteringsytan
3. Använd 3M tejp eller skruvar för att fixera sensorn
4. Installera CR2450-batteriet
5. Parning via gateway
6. Simuleringstest: knacka på installationsytan, sensorn ska utlösa ett larm

## Anteckningar

- För dörr- och fönsterskyddsscenarier är det mer effektivt att installera det på insidan av dörren/fönstret nära låset.
- För rörledningsövervakning, fäst direkt på rörledningens yta
- Batteritid cirka 2 år
- Justerbar känslighet för att passa olika scenarier
