# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Sensor | Strömförsörjning | Funktioner | Spelförslag |
|------|------|--------|------|------|----------|
| Vibrationssensor | Zigbee | 3-axlig accelerometer | CR2450 | Vibration + lutningsdetektering | Dörr- och fönsterskydd, värdeskåp/låda olagligt öppningsvibrationslarm, onormal rörvibrationsövervakning |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Antibändning av dörrar och fönster | Vibrationssensor | Detektering av våldsamma stötar och vibrationer av dörr/fönster → trycklarm + ljud- och ljuslarm + kamerainspelning |
| Säker övervakning | Vibrationssensor | Skåpet flyttas eller vibreras → Omedelbar larmmeddelande |
| Rörledningsövervakning | Vibrationssensor | Övervaka onormala vibrationer av vattenledningar (vattenpumpfel/vattenslag), ge tidig varning för underhåll |
| Staket/Vägg | Vibrationssensor | Vägg-/staketklättringsdetektering, lämplig för gårdssäkerhet |
