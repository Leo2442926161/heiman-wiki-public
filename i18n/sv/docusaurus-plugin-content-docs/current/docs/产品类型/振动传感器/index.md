# Vibrationssensor

Vibrationssensor utlöser ett larm genom att detektera vibration eller lutning av ett föremål. Den är lämplig för olika scenarier som dörrar, fönster, kassaskåp och rör.

## Kärnfunktioner

- Tredimensionell vibrationsdetektering
- Detektering av lutningsvinkel
- Påminnelse om lågt batteri

## Tillämpliga scenarier

- **Dörr- och fönsterskyddsskydd**: Upptäck vibrationer orsakade av våldsamt inbrott
- **Säkerskåp/lådamonitor**: Upptäck onormala rörelser
- **Rörledningsövervakning**: Upptäck vattenrörsvibrationer (indikerar avvikelser i vattenpumpen eller vattenslag)
- **Vägg-/staketövervakning**: Upptäck klättring eller kollision

## Tekniska parametrar

- Sensortyp: 3-axlig accelerometer
- Detektionsmått: vibration + lutning
- Strömförsörjningsmetod: CR2450 knappbatteri
- Batteritid: ca. 2 år
