# Installationsguide

## Krav på installationsplats

- **Placera på marken**, se till att sonden vidrör marken
- **Nära vattenkälla**: under handfatet i badrummet, under diskbänken, bredvid tvättmaskinen, under varmvattenberedaren
– Den lägsta punkten där vattenläckage behöver upptäckas

## Installationssteg

1. Bestäm installationsplatsen (marken i riskområdet för vattenläckage)
2. Rengör golvytan
3. Placera sensorn på marken med sonden vidrör marken
4. Om du använder kabelversionen (HS2WL), lägg förlängningskabeln längs det område som ska övervakas
5. Sätt i batteriet och sensorn slås på automatiskt.
6. Simuleringstest: rör vid sonden med våta fingrar, larmlampan tänds
7. Parning via gateway

## Anteckningar

- Sonden kan korrodera om den utsätts för en fuktig miljö under lång tid, så kontrollera den regelbundet
- Det rekommenderas att samarbeta med magnetventilen för att uppnå automatisk avstängning
-Batteritiden är ca 2-3 år
- Flera sensorer för att täcka olika områden
