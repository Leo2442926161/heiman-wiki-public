# Sensor för nedsänkning av vatten

Vattenläckagesensor används för att upptäcka vattenläckage eller vattenansamling, vilket effektivt kan förhindra egendomsförluster orsakade av vattenrörsprängningar och vattenöversvämningar.

## Produktserie

| Serie | Kommunikationsmetoder | Funktioner |
|------|----------------|------|
| HS1WL | Zigbee | Grundmodell, probdetektion |
| HS2WL-serien | Zigbee | Sond + kabeldetektering, dubbelkanal |
| HS3WL-serien | Zigbee | Förbättrad version |
| L1-serien | Matter / Zigbee | Matter over Thread-version, WWHA-certifierad |

## Kärnfunktioner

- Realtidsdetektering av vattenläckage/stilleståndsvatten
- Summerlarm
- Påminnelse om lågt batteri
- Korrosionsskyddande probdesign

## Installationsplats

- Under handfatet i badrummet
- Under diskbänken
- Bredvid tvättmaskin/diskmaskin
- Under varmvattenberedaren
- Källare/garage
- Nära kylaren

## Scenkoppling

- **Detektering av vattenläckage + magnetventil**: Stäng automatiskt huvudventilen när vattenläckage upptäcks
- **Vattenläckagelarm + push**: Meddela fastigheten eller familjemedlemmar på din mobiltelefon när du inte är hemma
- **Flood + Smart Shutoff**: Länkad gasvattenberedare eller smart vattenventil
