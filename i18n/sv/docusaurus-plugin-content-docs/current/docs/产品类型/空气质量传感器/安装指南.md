# Installationsguide

## Krav på installationsplats

- Installerad på **1,2 ~ 1,8 m** höjd (andningszon)
- Välj en plats som representerar den genomsnittliga luftkvaliteten i rummet
- **Håll undan**: fönster, dörrar, ventiler, luftkonditioneringsuttag
- **Håll dig borta**: köksångor, luftfuktare, aromaterapi och andra lokala föroreningskällor

## Installationssteg

1. Välj installationsplats (vardagsrum, sovrum eller kontorsvägg)
2. Fäst monteringsbasen med skruvar
3. Anslut till ström (USB- eller adapterdriven)
4. Sensorn är påslagen och displayen (om någon) visar data efter självtest.
5. Parning via gateway
6. Bekräfta att data är normala i APP

## Anteckningar

- Se till att sensorns luftintag och utlopp inte är blockerade
- Rengör sensorhuset regelbundet för att förhindra igensättning av damm
- Placera den i barnrummet för att övervaka CO₂-nivån och uppmana dig att öppna fönstret för ventilation
- Rekommenderas att kombineras med friskluftssystem eller luftrenare
