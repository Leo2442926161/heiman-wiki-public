# luftkvalitetssensor

Luftkvalitetssensorer (Air Quality Monitor) används för att upptäcka olika föroreningar i inomhusluften för att bibehålla en hälsosam livsmiljö.

## Detektionsparametrar

Hymans luftkvalitetssensorer känner av följande indikatorer:

| Parametrar | Beskrivning | Hälsonormer |
|------|------|---------|
| PM2.5 | Fina partiklar | ≤ 35 µg/m³ (24-timmarsgenomsnitt) |
| PM10 | Andningsbara partiklar | ≤ 50 µg/m³ (24-timmarsgenomsnitt) |
| CO₂ | Koldioxid | ≤ 1000 ppm |
| TVOC | Totalt antal flyktiga organiska föreningar | ≤ 0,6 mg/m³ |
| Temperatur | Omgivningstemperatur | 18-26°C |
| Fuktighet | Relativ luftfuktighet | 40-60 % RH |

## Kärnfunktioner

- Detektering av luftkvalitet med flera parametrar
- Färgdisplay (beroende på modell)
- Ljud- och ljuslarm för överskridande av standarden
- Historiska dataposter
- Länkad friskluft/luftrenare

## Tillämpliga scenarier

- Sovrum: Övervaka CO₂-nivåer och uppmana till att öppna fönstren för ventilation
- Barnrum: skydda barns andningshälsa
- Kontor: Öka produktiviteten och komforten
- Nyrenoverade hus: övervaka utsläpp av formaldehyd/TVOC
