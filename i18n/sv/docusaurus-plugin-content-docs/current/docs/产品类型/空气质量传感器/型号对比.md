# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Detektionsparametrar | Strömförsörjning | Funktioner | Spelförslag |
|------|------|--------|------|------|----------|
| Luftkvalitetssensor | Zigbee | PM2,5 / PM10 / CO₂ / TVOC / Temperatur och luftfuktighet | Nätström | Multifunktion i ett | Övervaka inomhusluftens kvalitet, koppla automatiskt friskluft/renare + tryckpåminnelse om den överskrider standarden |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Bedroom CO₂ monitoring | Luftkvalitetssensor | CO₂ > 1000ppm → Påminnelse om fönsteröppning/kopplat friskluftssystem för ventilation, bättre sömnkvalitet |
| Nyrenoverat hus | Luftkvalitetssensor | Kontinuerlig övervakning av TVOC/formaldehyd, kopplad frånluftsfläkt när standarden överskrids, och ett lugnande meddelande kommer att skickas efter att standarden är uppfylld |
| Barnrum | Luftkvalitetssensor | PM2.5-övervakning, automatiskt sätt på luftrenaren när standarden överskrids för att säkerställa andningshälsa |
| Kontor | Luftkvalitetssensor | CO₂ + temperatur och luftfuktighet kopplad luftkonditionering frisk luft för att förbättra arbetseffektiviteten |
