# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Sensor | Strömförsörjning | Funktioner | Spelförslag |
|------|------|--------|------|------|----------|
| [HS2FD](../../Wireless Protocol/Zigbee/HS2FD - Falldetektor (Zigbee)) | Zigbee | Accelerometer + Gyroskop | Batteri | AI-algoritm falldetektering | Bärbar design, automatisk falldetektering + hjälp med ett klick, lämplig för äldre personer som bor ensamma och vårdhem |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Övervakning av ensamboende äldre | HS2FD | Detektering av fall → Automatisk push till barns mobiltelefoner → Automatiskt ring 120 efter timeout och inget svar |
| Äldreomsorgsinstitution | HS2FD | Centraliserad hanteringsplattform + golvljud och ljuslarm + handhållen vårdenhet Synkront larm |
| Postoperativ rehabilitering | HS2FD | Fallrisken upptäcks när patienten stiger ur sängen → Tidig varning på vårdstationen |
| Badrum/natt | HS2FD | IP-skyddsdesign, bärbar i badet + automatiskt larm om du går upp och faller på natten |
