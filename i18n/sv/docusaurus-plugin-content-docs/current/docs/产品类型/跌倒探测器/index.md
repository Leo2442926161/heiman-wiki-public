# falldetektor

Fall Down Detector är speciellt designad för äldre. Den kan automatiskt upptäcka fallhändelser och skicka ut en nödsignal i tid. Det är en nyckelutrustning för smarta äldreomsorgslösningar.

## Produktserie

| Serie | Kommunikationsmetoder | Funktioner |
|------|----------------|------|
| HS2FD | Zigbee | Intelligent falldetektering, automatiskt larm |

## Kärnteknik

Den använder **multi-sensor fusionsalgoritm**, kombinerar accelerometer- och gyroskopdata och använder AI-algoritm för att bestämma fallhållning, effektivt skilja fall från normala aktiviteter (som att sitta ner, böja sig ner) och minska antalet falska larm.

## Kärnfunktioner

- Automatisk falldetektering
- Ring efter hjälp manuellt med ett klick
- Bärbar design (bältesklämma eller rem)
- Påminnelse om lågt batteri

## Tillämpliga scenarier

- Hemtjänst för äldre: dygnet-runt-omsorg för ensamboende äldre
- Vårdhem: centraliserad vårdnad och förvaltning
- Rehabiliteringscenter: Övervakning av fallrisken hos postoperativa patienter

## Länkningsplan

- Upptäck ett fall → Meddela automatiskt familjemedlemmar på sina mobiltelefoner
- Länkad nödknapp → dubbelt räddningsskydd
- Linkage Haiman Smart Elderly Care Platform → Centraliserad förvaltning av institutioner
