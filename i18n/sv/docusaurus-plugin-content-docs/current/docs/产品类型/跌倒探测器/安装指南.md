# Installationsguide

## Bärmetod

- **Bältesklämma**: Klämma fast bälte eller midjeband
- **Lanyard**: häng runt halsen och häng på bröstet

## Bruksanvisning

1. Installera batteriet
2. Välj bärmetod (bältesklämma eller snöre)
3. Bär den på kroppen
4. Parning via gateway
5. Simuleringstest:
   - Stå och sätt dig sedan snabbt på huk eller lägg dig på marken
   - Sensorn ska upptäcka ett fall och utlösa larmet inom några sekunder
   - Tryck på nödknappen för att testa det manuella larmet

## Anteckningar

- Rekommenderas att bäras hela dagen (kan tas bort för att ladda/byta ut batteriet när du duschar)
- AI-algoritm används för att skilja fall från dagliga aktiviteter (sitta ner, böja sig ner), med låg falsklarmfrekvens
- Automatiskt larm efter att ha upptäckt ett fall, eller så kan du trycka på knappen manuellt för att ringa efter hjälp
- Kan kopplas ihop med nödknappen för att uppnå dubbelt skydd
