# Ljud- och ljuslarm (varningssignal)

Den smarta sirenen används för att ge högt ljud och ljuslarm. Den används med olika sensorer och kan användas som utgångsställdon för säkerhetssystemet.

## Kärnfunktioner

- Larmljud med hög decibel (≥110 dB)
- Röda/färgade blixtljus
-Stöder flera larmtoner
- Detektering av sabotage
-Stöd för fjärraktivering och avaktivering

## Tillämpliga scenarier

- Kopplad med rök/CO-sensor: ljuder i hela huset när fara upptäcks
- Säkerhetslarm: utlöses när dörrar och fönster öppnas olagligt
- Nödutrymning: vägleda människor att evakuera i händelse av brand

## Scenkoppling

- **Röklarm + ljud- och ljussiren**: Hela huset kommer att larma samtidigt, vilket varnar alla människor i rummet
- **Dörrsensorlarm + ljud- och ljuslarm**: avskräcka illegala inkräktare
- **Samarbete med gateway**: Länka till andra enheter via Zigbee/Matter-nätverket
