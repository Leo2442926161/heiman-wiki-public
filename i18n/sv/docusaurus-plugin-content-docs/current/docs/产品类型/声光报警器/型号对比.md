# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Larmljudtryck | Strömförsörjning | Funktioner | Spelförslag |
|------|------|--------|------|------|----------|
| Ljud- och ljuslarm (siren) | Zigbee | ≥110 dB | Nätström | Hög ljusstyrka stroboskop + hög decibel | Rök/CO/dörrmagnet utlöser länkning hela huset ljud- och ljuslarm, avskräcker inkräktare när de lämnar hemmet |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Brandkoppling för hela huset | Ljud- och ljuslarm | Alla rök-/CO-sensorlarm → Ljud- och ljuslarm ljuder i hela huset för att väcka sovande familjemedlemmar |
| Säkerhet intrång avskräckning | Ljud- och ljuslarm | Olaglig öppning av dörrsensor + lämnar hemläge → sirenljud + fullt ljus för att avskräcka inkräktare |
| Vägledning vid nödutrymning | Ljud- och ljuslarm | Brandlarmets utlösare → Sirenen styr evakueringsriktningen och samarbetar med dörrsensorn för att detektera om evakuering har skett |
