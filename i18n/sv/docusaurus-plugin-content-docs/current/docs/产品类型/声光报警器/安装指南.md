# Installationsguide

## Krav på installationsplats

- **Installera i mitten av huset** för att säkerställa att det hörs i alla rum
- Hall, vardagsrumstak eller hög vägg
- **2m eller mer** från marken (undvik att bli vidrörd av människor)
- När den används tillsammans med rök-/CO-sensorer, installeras i sovrumsgångar

## Installationssteg

1. Välj installationsplats (korridor eller vardagsrum rekommenderas)
2. Använd skruvar för att fästa monteringsbasen på väggen eller taket
3. Anslut strömförsörjningen (nätutförande) eller sätt i batteriet (batteriversion)
4. Installera larmkroppen på basen
5. Parning via gateway
6. Länktest: utlös valfri sensor, ljud- och ljuslarmet ska svara

## Anteckningar

- Installera på en höjd utom räckhåll för barn
- Testa regelbundet om larmvolymen och belysningen är normala
- För batteriversionen, var noga med att byta ut den när batteriet är lågt.
- Kan ställa in kopplingsregler med andra sensorer
