# Människokroppens infraröda sensor

Människokroppens infraröda sensor (PIR) känner av rörelse genom att detektera infraröd värmestrålning som sänds ut av människokroppen och är den mest använda lösningen för närvarodetektering i smarta hem.

## Produktserie

| Serie | Kommunikationsmetoder | Funktioner |
|------|----------------|------|
| HS1MS | Zigbee | Grundmodell, 110° detekteringsvinkel |
| HS3MS | Zigbee | Uppgraderad modell, 360° takinstallation |
| M1-serien | Matter / Zigbee | Stöder Matter over Thread, valfritt 110° eller 360° |
| HS8MLS | Zigbee | Långdistansmodell |
| HS8MIS | Zigbee | Människokropp + belysning kompositsensor |

## Kärnfunktioner

- Mobil mänsklig upptäckt
- Belysningsdetektering (M1/HS8MIS-serien)
- Detektionsavstånd ≤7m (25°C)
- Installationsmetod: vägg- eller takmonterad
- Batteritid: 3 år (Zigbee) / 1 år (Matter)

## Tekniska parametrar (M1-serien)

- Detekteringsvinkel: 110° / 360°
- Detektionsavstånd: ≤7m
- Belysningsstyrka: 0~1200 Lux
- Strömförsörjning: DC 3V (litiumbatteri)
- Batteritid: 3 år (Zigbee) / 1 år (Matter)

## Scenarioexempel

– Ljuset tänds när någon kommer och släcks när någon går
- Säkerhetskoppling: tryck larm när onormala rörelser upptäcks
- Tillkoppling när du lämnar hemmet: stäng automatiskt av luftkonditioneringen/värmen för att spara energi när ingen är i närheten
