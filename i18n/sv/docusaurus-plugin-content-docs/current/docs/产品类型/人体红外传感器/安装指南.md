# Installationsguide

## Krav på installationsplats

- Väggmonteringshöjd: **1,8 ~ 2,4 m**
- Takmontering: Monteras i taket i mitten av rummet
- **Undvik**: Direkt solljus, värme-/luftkonditioneringsventiler, nära värmekällor
- Undvik hinder (möbler, glaspartier) inom detektionsområdet

## Installationssteg

1. Välj installationsplats (vägg- eller takmonterad)
2. Fäst monteringsbasen med skruvar
3. Sätt i batteriet, indikatorlampan blinkar
4. Installera sensorkroppen på basen
5. Justera detekteringsvinkeln (inom 110° horisontellt område)
6. Gångtest: Efter att ha gått genom detektionsområdet bör indikatorlampan tändas
7. Parning via gateway

## Anteckningar

- Infraröda sensorer är känsliga för temperaturförändringar, och detektionsavståndet kan förkortas i högtemperaturmiljöer på sommaren.
- Undvik vända fönster (förändringar i utomhustemperaturen orsakar falsklarm)
- Batteritid cirka 3 år (Zigbee) / 1 år (Matter)
