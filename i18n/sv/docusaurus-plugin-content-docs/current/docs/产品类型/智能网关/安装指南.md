# Installationsguide

## Krav på installationsplats

- **Centralt läge i hela huset** säkerställer Zigbee-signaltäckning för alla enheter
- Försök att undvika att placera den i ett skåp helt i metall (metall blockerar trådlösa signaler)
- **Ethernet först**: Det rekommenderas att använda en trådbunden nätverksanslutning, som är mer stabil
- Inte mer än 20-30 m från andra Zigbee-enheter (inomhus)

## Installationssteg

1. Välj installationsplats (TV-skåp i vardagsrummet, bredvid svagströmsboxen etc.)
2. Anslut strömadaptern
3. Anslut en Ethernet-kabel (rekommenderas) eller anslut till Wi-Fi
4. Vänta tills indikatorlampan lyser grönt/fast lyser
5. Öppna Haiman-appen och skanna QR-koden längst ned på gatewayen för att lägga till
6. När tillägget är klart kan du para ihop underenheten

## Anteckningar

– Gatewayen är kärnan i det smarta hemmet. Det rekommenderas att använda UPS reservströmförsörjning.
- Ethernet-anslutning är mer stabil än Wi-Fi och rekommenderas att användas först
- Om enhetens signal är svag, lägg till en Zigbee-reläenhet (som ett smart uttag)
- Det maximala antalet underenheter som stöds av olika modeller är olika och bör väljas utifrån skalan.
