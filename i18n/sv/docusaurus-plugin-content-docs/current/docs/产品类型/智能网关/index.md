# Smart Gateway

Smart Gateway är kärnan i Haimans smarta hemsystem, ansvarig för att ansluta olika Zigbee-sensorer och förverkliga kopplingskontroll mellan enheter.

## Produktserie

| Serie | Kommunikationsmetoder | Funktioner |
|------|----------------|------|
| HS1GW-serien | Zigbee + Wi-Fi / Ethernet | Basic Gateway |
| HS3GW-serien | Zigbee + Wi-Fi / Ethernet | Förbättrad |
| HS5GW-serien | Zigbee + Wi-Fi / Ethernet | High Performance Gateway |
| HS6GW-serien | Zigbee + Wi-Fi / Ethernet + tråd | Den senaste modellen stöder både Zigbee och Matter over Thread |

## Kärnfunktioner

- Zigbee Network Coordinator
- Stöd Wi-Fi / Ethernet för att ansluta till Internet
- Automatisk enhetsupptäckt och ihopparning
- Lokal scenkoppling (vissa modeller)
- Fjärråtkomst (via Heiman Cloud Platform)
- Stöder upp till 128 underenheter (beroende på modell)

## Tillämpliga scenarier

- Kärnan i Heiman hela husets intelligens
- Samarbeta med Haiman APP för att realisera fjärrövervakning
- Lokal centraliserad förvaltning av små och medelstora kommersiella projekt

## Skillnader från andra gateways

Haiman Gateway är ingången till det officiella ekosystemet. För användare som vill använda en tredjepartsplattform, rekommenderas att använda den universella Zigbee-koordinatorn med Zigbee2MQTT/ZHA.
