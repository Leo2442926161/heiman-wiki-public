# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Maximal gräns för Zigbee-underenhet | Nätverksmetod | Funktioner | Spelförslag |
|------|----------------|--------|------|----------|
| [HS6GW](../../Wireless Protocol/Zigbee/HS6GW - Intelligent Gateway (Zigbee)) | 128 | WiFi + Ethernet | Den senaste modellen och stöder framtida tråduppgraderingar | Haimans egen ekologiska kärna, i kombination med Haiman APP fjärrhantering, lämplig för användare som inte bryr sig om det |
| HS5GW | 64 | WiFi + Ethernet | Hög prestanda | Medelstora och stora projekt, stabila och pålitliga |
| HS3GW | 32 | WiFi | Förbättrad | Räcker för små och medelstora bostäder |
| HS1GW | 16 | WiFi | Grundläggande | Instegsnivå, lämplig för ett litet antal enheter |

## Rekommenderad scenspel

| Scenario | Rekommenderad modell | Beskrivning |
|------|----------------|------|
| Hela huset Heiman Ekologi | HS6GW | Används med Heiman Zigbee-sensorer och fjärrhantering med Heiman APP, inget behov av att bygga din egen server |
| Tredjeparts plattformsanvändare | — | Det rekommenderas att köpa en Zigbee-koordinator (som SLZB-06) direkt för användning med Zigbee2MQTT/ZHA för högre flexibilitet |
| Små och medelstora lägenheter | HS3GW | 32 enhetsgräns för att möta grundläggande behov |

## Urvalsförslag

- **Haimans egen ekologi**: [HS6GW](../../Wireless Protocol/Zigbee/HS6GW - Smart Gateway (Zigbee)) (128 enheter, WiFi + Ethernet)
- **Tredjepartsplattform**: Rekommenderad Universal Zigbee-koordinator + Zigbee2MQTT/ZHA
