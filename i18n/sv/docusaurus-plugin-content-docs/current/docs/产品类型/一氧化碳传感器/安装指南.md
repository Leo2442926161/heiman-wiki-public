# Installationsguide

## Krav på installationsplats

- **Installerad i andningshöjd** - ca. **1,5 m** från marken
- Nära sovrummet för att säkerställa att larmet kan höras när du sover
- **Installera inte i hörn**, håll luftcirkulationen
- Håll dig borta från ventiler, fönster, fläktar och andra platser som påverkar luftflödet

## Installationssteg

1. Bestäm installationsplatsen (rekommenderad sovrumskorridor eller vardagsrum)
2. Fäst monteringsbasen på väggen (skruvar eller 3M tejp)
3. Sätt i batteriet och sensorn slås på automatiskt.
4. Installera huvuddelen på basen
5. Självtest: Tryck på testknappen, larmlampan blinkar och larmet ljuder.
6. Parning via gateway

## Anteckningar

- Den typiska livslängden för sensorn är **7 år**, varefter hela enheten måste bytas ut
- Testa funktionalitet en gång i månaden
- Blockera inte sensorventilen
- Om det finns ett larm, öppna först fönstren för ventilation och kontrollera sedan om det finns något läckage av CO-källor.
