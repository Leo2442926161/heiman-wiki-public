# kolmonoxidsensor

Kolmonoxid (CO) är en färglös, luktfri, icke-irriterande giftig gas, känd som den "osynliga mördaren". Hymans kolmonoxidsensorer använder **elektrokemisk sensorteknologi** för att exakt detektera CO-koncentrationen i luften.

## Certifierade prestationer

Hyman lanserar **världens första Matter-certifierade kolmonoxidlarm** och är certifierad av Works with Home Assistant.

## Kärnfunktioner

- Elektrokemisk CO-sensor med hög precision
- Detektionsområde: 0~1000 ppm
- Koncentrationsgraderat larm (ljud- och ljusmeddelande)
- 85 dB larmljud
- Påminnelse om lågt batteri
- Påminnelse om sensorns livslängd (vanligtvis 7 år)
- Självtestfunktion

## Säkerhetstips

CO-koncentration och mänsklig kroppsrespons:

| CO-koncentration (ppm) | Människokroppens reaktion |
|---------------------|--------|
| 50 | Friska vuxna tål 8 timmar |
| 200 | Mild huvudvärk efter 2-3 timmar |
| 400 | Huvudvärk och illamående efter 1-2 timmar |
| 800 | Yrsel och kramper inom 45 minuter |
| 1600+ | Dödlig inom 2 timmar |
