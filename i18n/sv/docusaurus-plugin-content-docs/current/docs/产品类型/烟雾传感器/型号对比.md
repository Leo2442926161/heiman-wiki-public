# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Certifiering | Strömförsörjning | Marknad | Spelförslag |
|------|------|------|------|------|----------|
| [HS1SA-E](../../Wireless Protocol/Zigbee/HS1SA-E - Intelligent röklarm (Zigbee)) | Zigbee | EN 14604 | CR123A | EU | Anslut till Zigbee2MQTT/ZHA, länka till Home Assistant, tänd automatiskt lamporna när rök upptäcks + push mobilaviseringar |
| [HS1SA-Z](../../Wireless Protocol/Zwave/HS1SA-Z Smart Brandvarnare (Z-Wave)) | Z-Wave | EN 14604 | CR123A | EU | Anslut till SmartThings/Hubitat, samarbeta med Z-Wave dörrmagnetisk armering och avisera automatiskt brandvarnaren efter att ha lämnat hemmet |
| [HS1SA-M](../../Wireless Protocol/Matter/HS1SA-M Smart Smoke Alarm (Matter)) | Matter över tråd | EN 14604 | CR123A | Global | Inbyggd åtkomst till Apple Home/Google Home, ingen dedikerad gateway krävs, Tråd med låg latens lokaliserad larmkoppling |
| [HS3SA](../../Wireless Protocol/Zigbee/HS3SA - Intelligent röklarm (Zigbee)) | Zigbee | EN 14604 | CR123A | EU | Används med Haiman HS6GW-gateway, länkning med flera enheter, lämplig för Haimans egna ekologiska användare |
| [HS8SA](../../Wireless Protocol/Zigbee/HS8SA - Smart Smoke Alarm (Zigbee)) | Zigbee / Wi-Fi | UL 217 | Batteri/nätverk | USA | Exklusivt för den amerikanska marknaden, UL-certifierad, uppfyller lokala brandföreskrifter, stöder WiFi direktanslutning eller Zigbee-gateway |
| [S1-serien](../../Wireless Protocol/Sub1G(långdistans)/S1-serien - Fristående brandvarnare (Sub-1G)) | Sub-1G | EN 14604 | Batteri | Universal | Oberoende sammankoppling, ingen gateway och Internet krävs, Sub-1G trådlös sammankoppling mellan flera S1-enheter, lämplig för enkla hyreshus |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Djup integration av smarta hem | HS1SA-E / HS1SA-M | HA automation: brandvarnare → tända ljus i hela huset → låsa upp dörrlås → stäng av gasventil → tryck till mobiltelefon |
| Apple HomeKit-användare | HS1SA-M | Native Matter-åtkomst, HomePod som trådgränsrouter, automatisk scenutlösning |
| Europeiska och amerikanska standardkrav | HS1SA-E (CE) / HS8SA (UL) | Uppfyll lokal brandcertifiering, försäkringskrav |
| Ingen gateway krävs/låg kostnad | S1-serien | Klara att använda efter köp, flera enheter är sammankopplade via Sub-1G, utlöser varandra när ett larm uppstår |
| Z-Wave befintligt system | HS1SA-Z | Fungerar med befintliga Z-Wave-gateways som Hubitat/SmartThings |

## Urvalsförslag

- **EU Market**: [HS1SA-E](../../Wireless Protocol/Zigbee/HS1SA-E - Smart Smoke Alarm (Zigbee)) (Zigbee) eller [HS3SA](../../Wireless Protocol/Zigbee/HS3SA) - Val av Smart Smoke Alarm (ZigbeWavee) (ZigbeWavee) krävs) [HS1SA-Z](../../Wireless Protocol/Zwave/HS1SA-Z Intelligent Brandvarnare (Z-Wave))
- **USA-marknaden**: [HS8SA](../../Wireless Protocol/Zigbee/HS8SA - Smart Smoke Alarm (Zigbee)) (UL 217-certifierad)
- **Materia över plattformar**: [HS1SA-M](../../Wireless Protocol/Matter/HS1SA-M Smart Smoke Alarm (Matter)) (Matter over Thread)
- **Fristående scen**: [S1-serien](../../Wireless Protocol/Sub1G(långdistans)/S1-serien - Fristående brandvarnare (Sub-1G)) (ingen gateway krävs)
