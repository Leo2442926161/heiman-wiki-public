# röksensor

Hyman röksensor använder **fotoelektrisk rökavkänningsteknik**, som känsligt kan detektera synliga rökpartiklar som genereras i de tidiga stadierna av en brand och uppnå mycket tidig brandvarning.

## Produktserie

| Serie | Kommunikationsmetoder | Certifieringsstandarder | Tillämpliga marknader |
|------|--------|--------|---------|
| HS1SA-serien | Zigbee / Z-Wave | EN 14604 | Europeiska unionen |
| HS3SA-serien | Zigbee | EN 14604 | Europeiska unionen |
| HS8SA | Zigbee / Wi-Fi | UL 217 | USA |
| S1-serien | Sub-1G Fristående | EN 14604 | Allmänt syfte |
| HS1SA-M | Matter över tråd | EN 14604 | Global |

## Kärnfunktioner

- Fotoelektrisk rökdetektion för att upptäcka rök från långsamma brännskador och öppna lågor
- 85 dB högt ljud- och ljuslarm
- Påminnelse om lågt batteri
- Självtestfunktion (ett-klickstest)
- Anti-manipuleringslarm
-Stöd för fjärravstängning

## Tekniska parametrar (med HS1SA-M som exempel)

- Sensortyp: infraröd fotoelektrisk
- Larmljudtryck: ≥85 dB (3 meter)
- Strömförsörjning: CR123A batteri
- Batteritid: ca. 3 år
- Driftstemperatur: -10°C ~ +50°C
- Arbetsfuktighet: ≤95% RH
