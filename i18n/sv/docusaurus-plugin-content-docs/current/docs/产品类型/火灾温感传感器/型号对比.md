# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Detektionsprincip | Strömförsörjning | Funktioner | Spelförslag |
|------|------|--------|------|------|----------|
| Brandtemperaturgivare | Zigbee / Sub-1G | Termistor (fast temperatur + differentialtemperatur) | Batteri | Antirökstörningar | Speciellt för kök/garage, larm för onormal temperaturökning, kopplingsmagnetventil för att stänga av gas |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Kökssäkerhet | Temperatursensor | Onormal temperaturökning → Stäng av gasventilen → Ljud- och ljuslarm → Push-meddelande från mobiltelefon för att undvika falsklarm om matrök |
| Garage/pannrum | Temperatursensor | Övervaka temperaturen i utrustningsrummet för att förhindra spontan förbränning och koppla frånluftsfläkten för att kyla ner |
| Komplettera med röksensor | Temperatursensor + rök | Röksensorn täcker hela huset, temperatursensorn täcker köket/garaget, komplementlarmet är mer tillförlitligt |

## Urvalsförslag

- **Endast kök**: Temperatursensor (för att undvika falska larm om matlagningsrök)
- **Fungerar med röksensor**: röksensor täcker sovrum/vardagsrum, temperatursensor täcker kök/garage
