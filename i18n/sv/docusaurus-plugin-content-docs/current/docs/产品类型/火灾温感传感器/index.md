# Brandtemperaturgivare

Brandtemperatursensorn (Heat Detector) ger ett brandlarm genom att upptäcka onormala ökningar av omgivningstemperaturen. Den är lämplig för scener som kök och garage där röksensorer är utsatta för falsklarm.

## Arbetsprincip

Temperatursensorn använder ett termiskt element för att upptäcka temperaturförändringar:

- **Larm för fast temperatur**: Utlöser ett larm när temperaturen överstiger en inställd tröskel (vanligtvis 57°C / 135°F)
- ** Temperaturskillnadslarm**: utlöser ett larm när temperaturstegringshastigheten överstiger det inställda värdet (som 8,3°C/min)

## Tillämpningsomfång

- Kök (röksensorer är benägna att få falsklarm på grund av matlagning i denna scen)
- Garage/verkstad (avgaser och damm kan störa röksensorerna)
- Pannrum/värmeutrustningsrum
- Badrum/bastu

## Kärnfunktioner

- Detektering av onormal temperaturökning
- Anti-manipuleringslarm
- Påminnelse om lågt batteri

> **Obs**: Temperatursensorn och röksensorn används komplementärt. Röksensorn reagerar snabbare och temperatursensorn har starkare anti-interferensförmåga.
