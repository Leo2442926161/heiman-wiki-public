# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Märkeffekt | Mätnoggrannhet | Funktioner | Spelförslag |
|------|------|----------------|--------|------|----------------|
| [HS2SK](../../Wireless Protocol/Zigbee/HS2SK - Smart Socket (Zigbee)) | Zigbee | 16A / 3680W | ±2 % | Mättyp | Övervakning av strömförbrukning i realtid + fjärromkopplare, strömavstängning med ett klick i läget utanför hemmet, statistik över luftkonditioneringens/vattenberedarens strömförbrukning |
| [HS2SK-WiFi](../../Wireless Protocol/Wifi/HS2SK-WiFi - Smart Meter Socket (WiFi)) | WiFi | 16A / 3680W | ±2 % | Direktanslutning utan gateway | Ingen extra gateway krävs, Smart Life App direktkontroll, lämplig för användare som inte vill köpa en gateway |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Energiförbrukningsstatistik | HS2SK | Zigbee är ansluten till HA och länkad till InfluxDB + Grafana för att visa daglig/månatlig strömförbrukning och upptäcka högenergikrävande apparater |
| Stäng av strömmen när du lämnar hemmet | HS2SK | När du lämnar hemmet, stäng av icke-nödvändiga apparater som TV/luftkonditionering/asktanklampor med ett klick för att spara energi på ett säkert sätt |
| Tidskontroll | HS2SK / HS2SK-WiFi | Timing switch varmvattenberedare/luftfuktare/laddningshög, kombinerat med topp- och dalpriser för el för att spara elräkningar |
| Gateway-lösa användare | HS2SK-WiFi | Direkt ansluten till routern, styrd av Smart Life App, noll tröskelvärde |

## Urvalsförslag

- **Befintlig Zigbee-gateway**: [HS2SK](../../WirelessProtocol/Zigbee/HS2SK - Smart Socket (Zigbee))
- **Vill inte köpa en gateway**: [HS2SK-WiFi](../../Wireless Protocol/Wifi/HS2SK-WiFi - Smart Metering Socket (WiFi))
