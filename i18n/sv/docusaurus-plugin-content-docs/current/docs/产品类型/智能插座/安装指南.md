# Installationsguide

## Krav på installationsplats

- Anslut direkt till vägguttaget, **Anslut inte i serie** (anslut inte till en remsa eller förlängningssladd)
- Se till att uttagets märkström är ≥ 16A
- Installera inte i fuktiga eller höga temperaturer

## Installationssteg

1. Anslut det smarta uttaget direkt till ett 220V (eller 110V) vägguttag
2. Indikatorlampan tänds efter att strömmen anslutits.
3. Tryck och håll ned strömbrytaren för att gå in i parningsläge
4. Lägg till enhet i Heiman APP
5. Efter lyckad parning kan du fjärrstyra switchen

## Säkerhetsföreskrifter

- **Maximal belastning**: 16A/3680W (europeisk standard), 15A/1800W (amerikansk standard)
- **Överbelasta inte**: Var uppmärksam på effektgränsen när du använder elektriska apparater med hög effekt (luftkonditionering, värmare)
- **ANSLUT INTE I SERIEN**: Sätt inte i smarta uttag i grenuttag
- Täck eller dölj inte uttagshuset
- Kontrollera kontakten regelbundet för att se om den är överhettad

## Typiska applikationer

- Timingstyrning av utrustning med låg effekt som varmvattenberedare, luftfuktare och fisktanklampor
- Samarbeta med sensorkopplingen: ljuset tänds när någon kommer och släcks när någon går.
- Effektstatistik: övervaka hushållsapparaters energiförbrukning
