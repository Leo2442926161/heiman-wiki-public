# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Triggermetod | Strömförsörjning | Funktioner | Spelförslag |
|------|------|--------|------|------|----------|
| Nödknapp | Zigbee | Långt tryck på avtryckaren | Batteri | Bärbar + väggmonterad dubbelfunktion | Äldre kan bära den med sig, enklickshjälp i nödsituationer, kopplat till mobil push + ljud och ljuslarm |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Hemtjänst för äldre | Nödknapp | Plötsligt obehag för äldre → Långt tryck → Tryck till alla familjemedlemmars mobiltelefoner → Länkat ljud- och ljuslarm → Fastighetsanmälan |
| Säkerhet för barnrum | Nödknapp | Om ett barn råkar ut för en nödsituation hemma ensam → Tryck på knappen för att utlösa larmet på förälderns mobiltelefon |
| Butik/Kassadisk | Panikknapp | Dold installation, utlöser tyst larm vid rån, kopplad till övervakningsvideo + tyst larm till säkerhet |
| Toalett/badrum | Nödknapp | Om äldre halkar och ramlar i duschen finns den vattentäta nödknappen till hands för enkel hjälp |
