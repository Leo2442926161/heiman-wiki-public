# Installationsguide

## Krav på installationsplats

- **Väggmontering**: Badrum, säng i sovrummet, bredvid soffan i vardagsrummet - var som helst inom räckhåll
- **Bär den var som helst**: lägg den i fickan eller häng den på nyckelringen
- **För äldre**: Installerad i badrummet och sovrummets säng, höjden är ca 1,2 m

## Installationssteg

1. Bestäm installationsplatsen eller om du ska bära den med dig
2. Väggmontering: Använd skruvar eller 3M-lim för att fixera monteringsbasen
3. Installera batteriet
4. Placera nödknappen på basen eller bär den med dig
5. Parning via gateway
6. Testa: Tryck länge på knappen för att utlösa larmet och bekräfta att appen tar emot aviseringen

## Anteckningar

- Använd **lång tryck mot oavsiktlig beröring** design, som kräver kontinuerlig tryckning i 1-2 sekunder för att utlösa
– Det rekommenderas att länka till ljud- och ljuslarmet
- Testa funktionalitet regelbundet
– För äldre kan den användas med falldetektor för dubbelt skydd.
