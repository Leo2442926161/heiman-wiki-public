# panikknapp

Nödknappen (Emergency Button/Panic Button) används för att snabbt utlösa en nödsignal i en nödsituation och är en viktig del av det smarta äldreomsorgs- och trygghetssystemet.

## Produktserie

| Serie | Kommunikationsmetoder | Funktioner |
|------|----------------|------|
| Nödknapp | Zigbee | Bärbar, en-knapps ring på hjälp |

## Kärnfunktioner

- Utlösa nödlarm med ett klick
- Bärbar design (kan bäras runt eller väggmonteras)
- Påminnelse om lågt batteri
- Anti-oavsiktlig beröringsdesign (lång tryck på avtryckaren)

## Tillämpliga scenarier

- **Smart äldreomsorg**: Äldre kan ringa efter hjälp med ett klick när de plötsligt blir sjuka eller ramlar.
- **Barnrum**: Används i nödsituationer när barn är ensamma hemma
- **Sjukvård**: Sängsamtal på avdelning/vårdhem
- **Butik/Reception**: Utlös ett hemligt larm vid rån eller nödsituation

## Utlösa länkning

- Push-meddelanden från mobiltelefoner till familjemedlemmar/vårdnadshavare
- Larmkoppling för ljud och ljus
- Ring fastigheten automatiskt eller 120
