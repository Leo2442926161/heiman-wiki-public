# Installationsguide

## Krav på installationsplats

- Installerad på en höjd av **1,2 ~ 1,5 m** (representerar den genomsnittliga miljönivån)
- **Håll undan**: direkt solljus, dörrar och fönster, luftkonditioneringsuttag, radiatorer
- **Håll undan**: luftfuktare, köksspisar och andra lokala källor till temperatur- och luftfuktighetsförändringar
- Välj rum som används ofta (sovrum, vardagsrum, babyrum)

## Installationssteg

1. Välj installationsplats (antingen vägg eller skrivbord)
2. Väggmontering: fäst basen med skruvar eller 3M lim
3. Installera CR2-batteriet (observera polariteten)
4. Sensorn slås på automatiskt och skärmen (om någon) visar temperatur och luftfuktighet.
5. Parning via gateway
6. Kontrollera data i APP för att bekräfta att det är normalt.

## Anteckningar

- När du installerar mot väggen, se till att sensorventilen inte är blockerad
-Batteritiden är ca 2-3 år
- Temperatur- och luftfuktighetsdata i olika rum kan samlas in genom flera sensorer
- Kan användas för kopplade luftkonditioneringsapparater, avfuktare, luftfuktare och annan utrustning
