# Modelljämförelse

## Jämförelse av olika modeller

| Modell | Protokoll | Noggrannhet | Strömförsörjning | Funktioner | Spelförslag |
|------|------|------|------|------|----------|
| [H1-M](../../Wireless Protocol/Matter/H1-M Temperatur- och fuktighetssensor (Matter)) | Matter över tråd | ±0,3°C / ±3% RH | CR2 | Matter version, plattformsoberoende | Native Apple Home, kopplat till luftkonditionering/golvvärme för att automatiskt justera rumstemperaturen och automatiskt slå på frånluftsfläkten när luftfuktigheten är hög |
| H1-Z | Zigbee | ±0,3°C / ±3% RH | CR2 | Zigbee version | Anslut till HA/Z2M, larm för temperaturavvikelse (varning för för låg/hög temperatur för äldre) |

## Rekommenderad scenspel

| Scenarier | Rekommenderade modeller | Länklösningar |
|------|--------|--------|
| Apples hemekologi | H1-M | Native Matter access, HomeKit automation: temperatur > 28°C, luftkonditionering på, luftfuktighet > 70%, frånluftsventilation |
| HA fördjupad användare | H1-Z eller H1-M | HA-automation: temperatur- och luftfuktighetskoppling golvvärme/luftfuktare/avfuktare, historiska data InfluxDB + Grafana |
| Vinkällare/cigarrskåp | H1-Z | Temperatur- och luftfuktighetsövergränslarm + push-meddelande från mobiltelefon för att skydda samlingar |
| Babyrumsövervakning | H1-M | Push-meddelanden när temperaturen är för hög/låg, och länka luftkonditioneringen för att anpassa sig till komfortzonen |
| Växthus | H1-Z | Temperatur och luftfuktighet utlöser automatisk vattning/ventilation/solskydd |

## Urvalsförslag

- **Materia plattformsoberoende**: [H1-M](../../Wireless Protocol/Matter/H1-M temperatur- och fuktighetssensor (Matter))
- **Zigbee befintligt system**: H1-Z
