# Om Heiman Technology

**Shenzhen Heiman Technology Co., Ltd.** grundades 2005. Det är ett **nationellt högteknologiskt företag** och ett **nationellt specialiserat och nytt "Little Giant"-företag** som integrerar egen FoU, produktion, försäljning och service.

## Om Heiman

Heiman Technology fokuserar på forskning, utveckling och innovation av smarta sensorprodukter, som omfattar:

- **Smart brandskydd** — Rökdetektorer, CO-larm och brandskyddssensorer
- **Smart gassäkerhet** — Gasläckagelarm, nödavstängningsventiler för gas
- **Smart äldreomsorg** — Nödsignalknappar, fall-detektorer, vattenläckagedetektorer
- **Smarta sensorer** — Infraröda detektorer, dörr-/fönstersensorer, temperatur-/fuktighetssensorer
