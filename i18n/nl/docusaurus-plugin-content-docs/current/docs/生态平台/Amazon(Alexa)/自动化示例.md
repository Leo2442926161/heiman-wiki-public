# Automatiseringsvoorbeeld

## Alexa Guard-integratie (VS)

**Voorwaarde:** Abonneer je op Alexa Guard Plus (beschikbaar in de VS)

**Configuratieproces:**
1. Schakel Guard Plus in in de Alexa-app
2. Voeg de Hyman Rook-/CO-melder toe als een "Smart Home-apparaat"
3. Schakel de slimme reactie 'Rookalarmdetectie' in

**Automatisch antwoord:**
- Rookalarm geactiveerd → Alexa belt automatisch noodcontacten
- CO-alarmtrigger → Alexa stemwaarschuwing voor het hele huis, automatische ventilatie
- Deuren en ramen gaan abnormaal open → Push-alarmmelding

## Rookdetectieroutine

**Trigger:** Hyman-rookalarm → Rook gedetecteerd

**Actie uitvoeren:**
- Alexa-stemaankondiging: "⚠️ Rook gedetecteerd, let op de veiligheid"
- Alle Echo-apparaten → Alarmtoon afspelen
- Slimme lichtset → alles verlicht
- Pushmeldingen naar mobiele telefoon

## "Het huis verlaten"-routine

**Trigger:** Spraakopdracht "Alexa, ik ga weg"

**Actie uitvoeren:**
- Heiman magnetische deursensor → inschakelen inschakelen
- Heiman Lichaamssensor → Bewegingsdetectie inschakelen
- Hyman-waterdompeldetector → schakel detectie in
- Alle lichten → uit
- Alexa antwoordde: "Het beveiligingssysteem is ingeschakeld, een veilige reis gewenst"
