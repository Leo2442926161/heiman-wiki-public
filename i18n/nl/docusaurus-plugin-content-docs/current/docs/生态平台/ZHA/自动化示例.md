# Automatiseringsvoorbeeld

## Rookmelderkoppeling

```jaml
alias: "ZHA rookmelderkoppeling"
triggers:
  - trigger: staat
    entiteit_id: binaire_sensor.heiman_smoke_zha
    naar: "aan"
acties:
  - actie: licht.turn_on
    doel:
      area_id: woonkamer
  - actie: tts.speak
    doel:
      entiteit_id: media_player.speaker
    gegevens:
      bericht: "Rookalarm geactiveerd!"
```

## Automatisering van het koppelen van apparaten

```jaml
alias: "Herinnering voor koppelen van nieuw apparaat"
triggers:
  - trigger: zha.event
    subtype: Heiman
    commando: device_joined
acties:
  - actie: notificatie.persistent_notification
    gegevens:
      bericht: "Het nieuwe Haiman-apparaat is lid geworden van het ZHA-netwerk, ga naar de configuratienaam en het gebied"
```

## Detectie van onderdompeling in water

```jaml
alias: "ZHA Flood Test"
triggers:
  - trigger: staat
    entiteit_id: binaire_sensor.heiman_water_leak_zha
    naar: "aan"
acties:
  - actie: schakelaar.turn_off
    entiteit_id: switch.main_water_valve
  - actie: notificeer.mobile_app_phone
    gegevens:
      bericht: "💧 Overstroming gedetecteerd!"
```
