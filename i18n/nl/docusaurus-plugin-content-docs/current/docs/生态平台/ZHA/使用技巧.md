# Gebruikstips

## Apparaat koppelen

1. Ga naar "Instellingen → Apparaten en diensten → ZHA → Configuratie" in Home Assistant
2. Klik op "Apparaat toevoegen" en start vervolgens de koppelmodus van het Heiman-apparaat
3. Als het koppelen mislukt, breng het apparaat dan dicht bij de USB-coördinator en probeer het binnen 3 meter opnieuw

## Voorkeur USB-coördinator

## Kanaalcoördinatie

Het standaardkanaal van ZHA is 15. Als er ernstige interferentie is met WiFi 2,4GHz:
- Gebruik 'Silicon Labs Flasher Utility' om het kanaal te wijzigen
- Algemeen aanbevolen kanalen: 15, 20, 25 (vermijd WiFi 1/6/11)

## Netwerkoptimalisatie

- Voeg elke 3-4 Heiman-apparaten een Zigbee-routeringsapparaat (zoals een smart socket) toe om het netwerk uit te breiden
- Controleer de verbindingskwaliteit van het apparaat met de ZHA-netwerkvisualisatietool
- Controleer regelmatig op apparaatfirmware-updates om optimale compatibiliteit te behouden
