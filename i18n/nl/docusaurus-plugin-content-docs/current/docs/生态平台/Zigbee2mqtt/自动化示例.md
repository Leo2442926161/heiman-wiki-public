# Automatiseringsvoorbeeld

## Apparaatstatus MQTT-abonnement

``` bash
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
```

Uitvoervoorbeeld:
```
zigbee2mqtt/heiman_smoke {"smoke":true,"battery":100,"linkquality":120}
```

## Online monitoring van apparatuur

```jaml
# configuratie.yaml
binaire_sensor:
  - platform: mqtt
    naam: "Rookmelder online"
    state_topic: "zigbee2mqtt/heiman_smoke/beschikbaarheid"
    payload_on: "online"
    payload_off: "offline"
    device_class: connectiviteit
```
