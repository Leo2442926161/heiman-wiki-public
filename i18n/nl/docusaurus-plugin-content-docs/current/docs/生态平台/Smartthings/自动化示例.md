# Automatiseringsvoorbeeld

## "Goede nacht" bedtijdmodus

**Triggerconditie:** Magnetische Haiman-deursensor → Sluiten (geeft aan dat de slaapkamerdeur gesloten is)

**Actie uitvoeren:**
- Doe alle lichten in de woonkamer uit
- Slim slot → Vergrendelen
- Beveiligingssysteem → 's nachts inschakelen
- Pushmelding: "Welterusten, het systeem is ingeschakeld"

## "Afwezig"-modus

**Triggerconditie:** Hyman infraroodsensor voor menselijk lichaam → 15 minuten zonder beweging

**Actie uitvoeren:**
- Alle lichten uitgeschakeld
- Thermostaat → Energiebesparende modus
- Beveiligingssysteem → buitenshuis inschakelen
- Deursensor → alarm inschakelen
- Pushmelding: "Weg van huis-modus is geactiveerd, deur- en raamsensoren zijn ingeschakeld"

## Noodmodus rookalarm

**Triggerconditie:** Hyman-rookalarm → Rook gedetecteerd

**Actie uitvoeren:**
- Slimme luidspreker → alarmgeluid afspelen
- Alle slimme lampen → knipperen rood
- Pushmeldingen voor alle gezinsleden
- Indien verbonden met SmartThings Hub → activeer sirene
