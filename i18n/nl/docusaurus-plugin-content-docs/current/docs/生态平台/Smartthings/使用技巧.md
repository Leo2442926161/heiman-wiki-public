# Gebruikstips

## Hub-compatibiliteit

Hyman Zigbee-apparaten zijn compatibel met **SmartThings Hub v2** en **Hub v3 (Aeotec)**.

| Naafmodel | Compatibiliteit | Beschrijving |
|---------|--------|------|
| Hub v2 (STH-ETH-250) | ✅ | Vereist firmware-update naar de nieuwste versie |
| Hub v3 (Aeotec) | ✅ | Native ondersteuning, aanbevolen |
| Samsung SmartThings-station | ⚠️ | Ondersteunt slechts enkele Zigbee-apparaten |

## Meldingen voor het koppelen van apparaten

- Klik op "+" → "Apparaat" → "Merk" → "Hyman" in de SmartThings-app
- Apparaten die niet automatisch worden herkend, kunnen handmatig worden doorzocht via "Scan Zigbee"
- Het wordt aanbevolen om de afstand tussen het apparaat en de Hub binnen 5 meter te houden voor de eerste koppeling

## Batterij-optimalisatie

Het standaard rapportage-interval van de Heiman-sensor is langer. Als real-time prestaties vereist zijn, kan het "detectie-interval" op de apparaatpagina worden aangepast naar 5 minuten.

## Beste praktijken op het gebied van automatisering

- Gebruik de "Modi"-functie van SmartThings om onderscheid te maken tussen "thuis/weg van huis"-statussen
- Creëer **twee sets** automatisering voor belangrijke sensoren (rook, CO) voor redundantie
