# Automatiseringsvoorbeeld

## Rookalarmscène

**Scènenaam:** "Rookalarmkoppeling"

**Triggerconditie:** Hyman-rookalarm → Rook gedetecteerd

**Actie uitvoeren:**
- Smart socket → open (afzuigventilator aansluiten)
- Lichten → gaan aan en knipperen
- Pushmeldingen naar mobiele telefoon
- Luidspreker → Alarmgeluid afspelen

## Scène van het verlaten van huis met één klik (Tap-to-Run)

**Scènenaam:** "Arrangement na het verlaten van huis"

**Handmatige uitvoering:** Tik of stemtrigger

**Actie uitvoeren:**
- Haiman deursensor → alarm inschakelen
- Hyman Menselijk lichaam Infrarood → Schakel detectie in
- Hyman-waterdompeldetector → schakel detectie in
- Alle lichten → uit
- Slim slot → Vergrendelen

## Automatische temperatuur- en vochtigheidsregeling

**Automatiseringstype:** Automatische conditietrigger

**Triggerconditie:** Hyman temperatuur- en vochtigheidssensor Vochtigheid > 75%

**Actie uitvoeren:**
- Smart socket (ontvochtiger) → Aan
- Pushmelding: "Vochtigheid is te hoog, luchtontvochtiger is ingeschakeld"

**Triggerconditie:** Hyman temperatuur- en vochtigheidssensor Vochtigheid < 50%

**Actie uitvoeren:**
- Slimme stekkerdoos (ontvochtiger) → Sluiten
