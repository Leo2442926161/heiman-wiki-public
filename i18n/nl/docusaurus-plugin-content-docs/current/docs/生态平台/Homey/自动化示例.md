# Automatiseringsvoorbeeld

## Rookmelder - flitslichten en waarschuwen

**Stroomconfiguratie:**

1. **Trigger:** Hyman-rookalarm → Status verandert in 'waarschuwing'
2. **Voorwaarden:** (Geen)
3. **Actie:**
   - Woonkamerverlichting → knipperend (rood, één keer per seconde)
   - Pushmelding → Mobiel: "⚠️ Rookalarm geactiveerd!"
   - Gesproken aankondiging → Homey Speaker: "Rook gedetecteerd, let op de veiligheid!"

## CO-alarm - Schakel de ketel uit

**Stroomconfiguratie:**

1. **Trigger:** Hyman CO-alarm → Status verandert in 'waarschuwing'
2. **Voorwaarden:** (Geen)
3. **Actie:**
   - Ketelschakelaar → UIT
   - Smart stopcontact (frisseluchtsysteem) → Aan
   - Pushmelding → "Koolmonoxidealarm, ketel is uitgeschakeld"

## Alarm voor overschrijding van temperatuur en vochtigheid

1. **Trigger:** Hyman temperatuur- en vochtigheidssensor → Temperatuur boven 40°C
2. **Actie:**
   - Airconditioner → Schakel de koelmodus in
   - Pushmelding → "De kamertemperatuur is te hoog, de airconditioner is automatisch ingeschakeld"
