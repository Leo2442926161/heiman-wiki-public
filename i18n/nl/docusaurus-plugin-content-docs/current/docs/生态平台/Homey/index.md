#Huiselijk

Homey is een slimme thuishub gelanceerd door het Nederlandse bedrijf Athom, die Zigbee, Z-Wave, Wi-Fi, Matter, Thread en andere protocollen ondersteunt.

## Integratiemethode

Homey Zigbee- en Z-Wave-apparaten koppelen rechtstreeks via de ingebouwde draadloze chip van Homey, waardoor er geen extra gateway nodig is. Matter-apparaten zijn toegankelijk via de ingebouwde Thread-borderrouter van Homey Pro.

## Voordelen

- **Meerdere protocollen in één**: Homey Pro ondersteunt Zigbee, Z-Wave, Thread en Wi-Fi tegelijkertijd
- **Lokaal uitvoeren**: alle automatisering wordt lokaal uitgevoerd
- **Flow Automation**: programmering via slepen en neerzetten om complexe koppelingsscenario's te creëren
