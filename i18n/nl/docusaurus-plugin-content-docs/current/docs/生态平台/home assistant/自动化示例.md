# Automatiseringsvoorbeeld

## Rookmelder activeert verlichting door het hele huis

```jaml
alias: "Rookmelder - verlichting en meldingen door het hele huis"
triggers:
  - trigger: staat
    entiteit_id: binaire_sensor.heiman_rookalarm
    naar: "aan"
acties:
  - actie: licht.turn_on
    doel:
      entiteit_id: alles
  - actie: notificeer.mobile_app_phone
    gegevens:
      bericht: "⚠️ Rookalarm geactiveerd! Controleer dit onmiddellijk!"
      titel: "Rookmelder"
```

## CO-alarmkoppeling

```jaml
alias: "CO-alarm - gas uitzetten en ramen openen"
triggers:
  - trigger: staat
    entiteit_id: binaire_sensor.heiman_co_alarm
    naar: "aan"
acties:
  - actie: schakelaar.turn_off
    entiteit_id: switch.gas_fireplace
  - actie: deksel.open
    entiteit_id: cover.living_room_window
  - actie: notificatie.persistent_notification
    gegevens:
      melding: "Het koolmonoxidealarm is afgegaan, de gaskraan is dicht en de ramen zijn geopend"
```

## Koppeling van detector voor wateronderdompeling

```jaml
alias: "Overstromingsdetectie - Sluit waterklep"
triggers:
  - trigger: staat
    entiteit_id: binaire_sensor.heiman_water_leak
    naar: "aan"
acties:
  - actie: schakelaar.turn_off
    entiteit_id: switch.main_water_valve
  - actie: notificeer.mobile_app_phone
    gegevens:
      bericht: "💧 Waterlekkage gedetecteerd! De hoofdwaterklep is gesloten!"
```

## Buiten huis inschakelen - sensor van het menselijk lichaam activeert een alarm

```jaml
alias: "Thuismodus verlaten - alarm menselijk lichaamssensor"
triggers:
  - trigger: staat
    entiteit_id: binaire_sensor.heiman_motion_sensor
    naar: "aan"
voorwaarden:
  - staat: staat
    entiteit_id: alarm_control_panel.home_alarm
    staat: gewapend_weg
acties:
  - actie: alarm_control_panel.trigger
    entiteit_id: alarm_control_panel.home_alarm
```
