# Dashboardconfiguratie

## Beveiligingsoverzichtpaneel

Gebruik de lay-out Doorsnedeweergave om Heimann-apparatuur in gebieden te ordenen:

```jaml
soort: sectie
titel: Rook- en koolmonoxidemelder
kaarten:
  - type: entiteit
    entiteit: binary_sensor.heiman_smoke_alarm
    naam: rookmelder
  - type: entiteit
    entiteit: binary_sensor.heiman_co_alarm
    naam: CO-melder
```

## Paddestoelkaartstijl

```jaml
type: aangepast: paddenstoel-alarm-bedieningspaneel-kaart
entiteit: alarm_control_panel.home_alarm
naam: beveiligingssysteem

type: aangepast: paddenstoel-entiteitskaart
entiteit: binary_sensor.heiman_smoke_alarm
naam: rookmelder
icoon: mdi:rookmelder
primaire_info: staat
secundaire_info: laatst gewijzigd
fill_container: waar
```

## Omgevingssensorpaneel

```jaml
soort: sectie
titel: Milieumonitoring
kaarten:
  - type: meter
    entiteit: sensor.heiman_temperatuur
    naam: temperatuur
    eenheid: "°C"
    min: 0
    maximaal: 50
    ernst:
      groen: 0
      geel: 30
      Rood: 40
  - type: meter
    entiteit: sensor.heiman_humidity
    naam: vochtigheid
    eenheid: "%"
    min: 0
    maximaal: 100
    ernst:
      groen: 30
      Geel: 60
      Rood: 80
```

## Alarmgeschiedenispaneel

```jaml
type: geschiedenisgrafiek
titel: Sensorgeschiedenis
entiteiten:
  - binaire_sensor.heiman_rookalarm
  - binaire_sensor.heiman_co_alarm
  - binaire_sensor.heiman_water_lek
hours_to_show: 24
```

## Voorgestelde lay-out

- Boven: status beveiligingssysteem + statuskaart alarmapparaat
- Centraal: Omgevingsmonitoring (temperatuur, vochtigheid, waterkwaliteit)
- Onder: historische trendgrafiek van sensoralarmen
- Snelkoppelingen in de zijbalk: in-/uitschakelen, schakelen tussen scènes
