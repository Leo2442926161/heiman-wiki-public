# Automatiseringsvoorbeeld

## Regelmachine-rookmelder

**Regelnaam:** Veiligheidskoppeling voor rookmelders

**Configuratie:**
```
Trigger: Hyman-rookalarm wordt actief
Condities: Modus != Uitgeschakeld
Actie:
  - Doe alle lichten aan
  - Stuur een pushmelding "⚠️ Rookmelder!"
  - Zet de sirene aan (indien beschikbaar)
  - Stel de modus in op "Noodgeval"
```

## Regelmachinetemperatuurbewaking

**Regelnaam:** Bescherming tegen hoge temperaturen

**Configuratie:**
```
Trigger: Heimann-temperatuursensor > 38°C
Conditie: Airconditionerstatus = uit
Actie:
  - Zet airconditioning aan, koelmodus, 24°C
  - Melding verzenden "De kamertemperatuur is te hoog en de airconditioner is gestart"
```

## Regelmachine Waterlekdetectie

**Regelnaam:** Noodbehandeling van waterlekkage

**Configuratie:**
```
Trigger: Hyman-waterdompeldetector wordt nat
Conditie: Status waterklep = open
Actie:
  - Sluit de hoofdwaterkraan
  - Stuur pushmeldingen naar alle mobiele telefoons
  - SMS-melding verzenden (optioneel)
```
