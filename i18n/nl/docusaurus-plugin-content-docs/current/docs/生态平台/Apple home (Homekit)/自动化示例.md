# Automatiseringsvoorbeeld

## Rookdetectie beveiligingsalarm

**Geautomatiseerde HomeKit-configuratie:**

1. **Triggerconditie:** Hyman-rookalarm → rook gedetecteerd
2. **Beveiligingsinstellingen:** Schakel de status van het beveiligingssysteem naar "Alarm"
3. **Actie uitvoeren:**
   - HomePod → Speel aangepaste audio af "Rook gedetecteerd, let op de veiligheid"
   - Alle slimme lampen → lichten op en knipperen
   - Pushmeldingen voor alle gezinsleden

## Thuis/weg van huis scène

**Snelkoppelingsopdracht: "Verlaat de beveiligingscontrole"**

```
wanneer de laatste persoon het huis verlaat
  Beveiligingssysteem instellen → inschakelen bij het verlaten van huis
  Heiman Deursensor instellen → Meldingen inschakelen
  Heiman lichaamssensor instellen → alarm inschakelen
  Doe alle lichten uit
  Instellingen Thermostaat → Energiebesparende modus
  Melding verzenden "🏠 Weg van huis-modus geactiveerd"
```

## Herinnering temperatuurafwijking

**Triggerconditie:** Hyman temperatuur- en vochtigheidssensor → Temperatuur > 35°C

**Actie uitvoeren:**
- HomePod → Kamertemperatuur voor spraakuitzending overschrijdt de limiet
- Pushmeldingen naar iPhone
- Airconditioning automatisch inschakelen (als HomeBridge is geconfigureerd)
