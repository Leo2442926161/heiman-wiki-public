#OpenHAB

OpenHAB is een open source domoticaplatform gericht op leveranciersneutrale en protocolneutrale integratie.

## Integratiemethode

### Zigbee-binding

Met de Zigbee-binding van OpenHAB kunnen Heymann Zigbee-sensoren direct worden aangesloten. Moet worden gebruikt met Zigbee-coördinator (zoals Conbee II, SLZB-06).

### Materie bindend

OpenHAB 4.x en hoger ondersteunen het Matter-protocol. Hyman Matter-apparaten kunnen worden geïntegreerd via Matter-binding, waarvoor een Thread-borderrouter vereist is.

### MQTT-methode

Wordt gebruikt met Zigbee2MQTT om sensorgegevens te ontvangen via MQTT-binding.

> **Referentie**: [OpenHAB Zigbee bindend document](https://www.openhab.org/addons/bindings/zigbee/)
