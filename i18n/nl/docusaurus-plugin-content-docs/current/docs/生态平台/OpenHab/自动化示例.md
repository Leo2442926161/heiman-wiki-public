# Automatiseringsvoorbeeld

## Regels voor rookmelders

```javascript
regel "Rookalarm geactiveerd"
wanneer
    Item heiman_smoke_alarm gewijzigd in AAN
dan
    sendCommand(Woonkamerlicht, AAN)
    sendNotification("admin", "⚠️ Rookalarm geactiveerd!")
    postUpdate(Alarm_State, "rookwaarschuwing")
einde
```

## Regels voor temperatuursensoren

```javascript
regel "Temperatuur- en vochtigheidsrapport"
wanneer
    Item heiman_temperature heeft update ontvangen
dan
    val temp = (heiman_temperature.state als getal).floatValue
    val hum = (heiman_humidity.state als getal).floatValue
    als (temperatuur> 40) {
        sendCommand(Airconditioner, AAN)
        sendNotification("admin", "De temperatuur is te hoog (" + temp + "°C), de airconditioner is ingeschakeld")
    }
    als (brom > 80) {
        sendNotification("admin", "De luchtvochtigheid is te hoog (" + brom + "%), let op ontvochtiging")
    }
einde
```
