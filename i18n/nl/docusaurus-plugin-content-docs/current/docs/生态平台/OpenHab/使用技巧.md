# Gebruikstips

## Ding-configuratie

Heiman-apparaat toevoegen via MQTT-binding:

```java
Ding mqtt:topic:heiman_smoke "Heiman rookmelder" @ "woonkamer" {
    Kanalen:
        Typ binair: rook "rookstatus" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.smoke" ]
        Typenummer: batterij "Batterijvermogen" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.battery" ]
}
```

## Artikeldefinitie

```java
Neem contact op met Smoke_Alarm "Rookalarm [MAP(smoke.map):%s]" { channel="mqtt:topic:heiman_smoke:smoke" }
Nummer Smoke_Battery "Rookmelderbatterij [%d%%]" { channel="mqtt:topic:heiman_smoke:battery" }
```

## Coördinatieselectie

Het wordt aanbevolen om een USB-coördinator met **CC2531** of **CC2652** chip te gebruiken en ervoor te zorgen dat de firmwareversie de nieuwste is.

## Tips voor foutopsporing

- Bekijk de 'online/offline'-status van Thing in de Paper UI
- Gebruik `openhab-cli info` om te controleren of de bindingen correct zijn geladen
- Zoek het trefwoord 'heiman' in het logboek om communicatieproblemen met het apparaat op te lossen
