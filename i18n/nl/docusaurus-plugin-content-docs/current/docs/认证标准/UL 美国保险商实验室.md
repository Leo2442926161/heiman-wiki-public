# UL Underwriters Laboratoria

UL (Underwriters Laboratories) is de meest gezaghebbende organisatie voor de ontwikkeling en certificering van productveiligheidsstandaarden in de Verenigde Staten. De belangrijkste UL-normen voor Heiman-producten zijn als volgt:

## UL 217 — Rookmelderstandaard (VS)

| Project | Beschrijving |
|------|------|
| Toepassingsgebied | Rookmelders voor woningen (Amerikaanse markt) |
| Detectievereisten | Foto-elektrische/ionisatierookdetectie, geluids- en lichtalarm, boven 85 dB |
| Toepasselijke markten | Verenigde Staten, Canada |
| Heiman-producten | HS8SA (Amerikaanse versie) |
| Betekenis van certificering | Het voldoen aan de brandvoorschriften van verschillende staten in de Verenigde Staten is een noodzakelijke voorwaarde voor markttoegang in Noord-Amerika |

**Verschillen met andere normen**: De testscenario's van UL 217 omvatten typische Amerikaanse woningbrandscenario's zoals flitsbranden en langzaam brandende branden, en wijken enigszins af van de testmethoden van EN 14604.

## UL 2034 — Koolmonoxidemelderstandaard (VS)

| Project | Beschrijving |
|------|------|
| Toepassingsgebied | CO-alarm voor huishoudens (Amerikaanse markt) |
| Detectievereisten | Elektrochemische sensor, algoritme voor concentratie × tijdintegratie |
| Toepasselijke markten | Verenigde Staten, Canada |
| Hayman-producten | HS-720ES-M (Amerikaanse versie) |
| Betekenis van certificering | De meeste staten in de Verenigde Staten verplichten de installatie van koolmonoxidemelders |

## De belangrijkste verschillen tussen UL-certificering en EN-certificering

| Aspecten | UL | NL |
|------|-----|-----|
| Normen voor rookmelders | UL 217 | EN 14604 |
| CO-alarmnormen | UL 2034 | EN 50291 |
| Detectiemethode | Specifieke brandcurvetest | Verschillende brandtypetests |
| Toepasselijke spanning | 120V/batterij | 230V/accu |
