# Overige certificeringen

## Q-Label keurmerk

Q-Label is een kwaliteitskeurmerk voor de Europese smart home-industrie, gelanceerd door de Q-Lab Alliance (nu onderdeel van de Zigbee Alliance / CSA Connectivity Standards Alliance).

| Project | Beschrijving |
|------|------|
| Toepassingsgebied | Certificering van interoperabiliteit van Zigbee-apparaten |
| Certificeringsinhoud | Apparaatcompatibiliteit met andere merken Zigbee-gateways/controllers |
| Betekenis | Zorg voor interoperabiliteit tussen merken en verklein het risico dat je het 'thuis koopt en niet gebruikt' |

Hyman Zigbee-producten zijn Q-Label-gecertificeerd, wat compatibiliteit met de volgende platforms garandeert:
- Amazon Echo Plus / Echo Show (Zigbee ingebouwd)
- Philips Hue-bridge
- SmartThings-hub
- Andere Q-Label authenticatiegateways

## Red Dot-ontwerpprijs

De rookmelder uit de HS1SA-serie van Hyman heeft de **Red Dot Design Award** (Red Dot Design Award) gewonnen, wat de kracht van Hyman op het gebied van productontwerp weerspiegelt.

## ISO 9001 kwaliteitsmanagementsysteem

Haiman is geslaagd voor de ISO 9001:2015-certificering van het kwaliteitsmanagementsysteem om de volledige kwaliteitscontrole van het proces te garanderen, van R&D, productie tot after-sales.
