# EN Europese norm

EN (European Norm, European Standard) is een technische norm ontwikkeld door het Europees Comité voor Normalisatie (CEN) en het Europees Comité voor Elektrotechnische Normalisatie (CENELEC). De belangrijkste EN-normen voor Heiman-producten zijn als volgt:

## EN 14604 — Standaard voor rookmelders

| Project | Beschrijving |
|------|------|
| Toepassingsgebied | Onafhankelijke rookmelder |
| Detectievereisten | Optische rookdetectie, geluids- en lichtalarm, melding batterij bijna leeg |
| Toepasselijke markten | EU-lidstaten, Groot-Brittannië, Zwitserland, Noorwegen, enz. |
| Heiman-producten | HS1SA-E, HS3SA, HS1SA-M, S1-serie, enz. |
| Betekenis van certificering | Het voldoen aan de eisen van de EU-bouwbrandregelgeving is een noodzakelijke voorwaarde voor toegang tot de Europese markt |

**Testitems**: responstijd, geluidsdrukniveau (≥85 dB/3m), mute-functie, levensduur van de batterij, temperatuurcyclus, duurzaamheid van de vochtigheid, weerstand tegen radiofrequentie-interferentie, enz.

## EN 50291 — Standaard voor koolmonoxidemelders

| Project | Beschrijving |
|------|------|
| Toepassingsgebied | Koolmonoxide (CO)-alarm voor huishoudelijk gebruik |
| Testvereisten | Elektrochemische CO-sensor, alarm voor concentratieclassificatie |
| Toepasselijke markten | EU-lidstaten, VK |
| Heiman-producten | HS1CA-E, C1-serie, HS1CA-M, enz. |
| Betekenis van certificering | Om de betrouwbaarheid van CO-lekdetectie te garanderen, is installatie in sommige landen verplicht |
