# slim stopcontact

Haiman Smart Metering Plug ondersteunt bediening op afstand en realtime meting van het energieverbruik, waardoor gebruikers intelligent beheer van huishoudelijke apparaten kunnen realiseren.

## Productserie

| Serie | Communicatiemethoden | Kenmerken |
|------|----------|------|
| HS2SK-serie | Zigbee | Meettype, ondersteunt stroombewaking |

## Kernfuncties

- Afstandsbediening met schakelaar
- Realtime stroom-/gasmeting
- Overbelastingsbeveiliging
- Tijdschakelaar
- Vermogensstatistieken

## Technische parameters

- Nominale spanning: AC 220V (Chinese standaard) / AC 110V (Amerikaanse standaard)
- Maximale belasting: 16A/3680W
-Meetnauwkeurigheid: ±2%

## Scenariovoorbeeld

- Schakel vergeten apparaten op afstand uit
- Statistieken van het energieverbruik van de airconditioner/boiler
- Gekoppelde verlaat-thuismodus: schakel alle niet-essentiële apparaten met één klik uit
- Timingcontrole van aquariumverlichting, luchtbevochtigers, enz.
