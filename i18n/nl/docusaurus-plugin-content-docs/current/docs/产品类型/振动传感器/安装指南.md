# Installatiehandleiding

## Vereisten voor installatielocatie

- Geïnstalleerd op het oppervlak van objecten die trillingen moeten monitoren
- Veel voorkomende installatielocaties: **Binnendeuren en ramen**, **kluizen**, **leidingen**, **muren**, **hekken**
- Zorg ervoor dat het installatieoppervlak vlak en schoon is

## Installatiestappen

1. Selecteer het te bewaken object (zoals de binnenkant van de kluisdeur)
2. Reinig het montageoppervlak
3. Gebruik 3M plakband of schroeven om de sensor te bevestigen
4. Installeer de CR2450-batterij
5. Koppelen via gateway
6. Simulatietest: tik op het installatieoppervlak, de sensor zou een alarm moeten activeren

## Opmerkingen

- Voor anti-wrikscenario's voor deuren en ramen is het effectiever om deze aan de binnenkant van de deur/raam, vlakbij het slot, te installeren.
- Voor pijpleidingbewaking direct op het pijpleidingoppervlak plakken
- Levensduur batterij ongeveer 2 jaar
- Instelbare gevoeligheid voor verschillende scenario's
