# Trillingssensor

Trillingssensor activeert een alarm door de trilling of kanteling van een object te detecteren. Het is geschikt voor verschillende scenario's zoals deuren, ramen, kluizen en leidingen.

## Kernfuncties

- Driedimensionale trillingsdetectie
- Kantelhoekdetectie
- Herinnering batterij bijna leeg

## Toepasselijke scenario's

- **Deur- en raambeveiliging**: Detecteer trillingen veroorzaakt door gewelddadige inbraak
- **Kluis/lademonitor**: Detecteer abnormale bewegingen
- **Pijpleidingbewaking**: Detecteer trillingen van waterleidingen (wat wijst op afwijkingen aan de waterpomp of waterslag)
- **Muur-/hekbewaking**: Detecteer klimmen of stoten

## Technische parameters

- Sensortype: accelerometer met 3 assen
- Detectieafmetingen: trillingen + kantelen
- Voedingsmethode: CR2450-knoopcelbatterij
- Levensduur batterij: ca. 2 jaar
