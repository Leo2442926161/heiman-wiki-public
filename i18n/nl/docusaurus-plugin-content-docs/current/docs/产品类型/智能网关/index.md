# Slimme gateway

Smart Gateway is de kernhub van het Haiman smart home-systeem, verantwoordelijk voor het verbinden van verschillende Zigbee-sensoren en het realiseren van koppelingscontrole tussen apparaten.

## Productserie

| Serie | Communicatiemethoden | Kenmerken |
|------|----------|------|
| HS1GW-serie | Zigbee + Wi-Fi/Ethernet | Basisgateway |
| HS3GW-serie | Zigbee + Wi-Fi/Ethernet | Verbeterd |
| HS5GW-serie | Zigbee + Wi-Fi/Ethernet | Gateway met hoge prestaties |
| HS6GW-serie | Zigbee + Wi-Fi / Ethernet + Draad | Het nieuwste model ondersteunt zowel Zigbee als Matter over Thread |

## Kernfuncties

- Zigbee Netwerkcoördinator
- Ondersteuning van Wi-Fi / Ethernet om verbinding te maken met internet
- Automatische apparaatdetectie en koppeling
- Lokale scènekoppeling (sommige modellen)
- Toegang op afstand (via Heiman Cloud Platform)
- Ondersteunt maximaal 128 subapparaten (afhankelijk van model)

## Toepasselijke scenario's

- Het kernverbindingspunt van de Heiman-intelligentie voor het hele huis
- Werk samen met Haiman APP om monitoring op afstand te realiseren
- Lokaal gecentraliseerd beheer van kleine en middelgrote commerciële projecten

## Verschillen met andere gateways

Haiman Gateway is de toegang tot het officiële ecosysteem. Voor gebruikers die een platform van derden willen gebruiken, wordt aanbevolen om de universele Zigbee-coördinator met Zigbee2MQTT/ZHA te gebruiken.
