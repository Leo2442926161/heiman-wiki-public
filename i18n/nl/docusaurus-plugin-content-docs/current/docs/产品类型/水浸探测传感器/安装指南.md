# Installatiehandleiding

## Vereisten voor installatielocatie

- **Plaats op de grond**, zorg ervoor dat de sonde de grond raakt
- **In de buurt van waterbron**: onder de wastafel in de badkamer, onder de gootsteen, naast de wasmachine, onder de boiler
- Het laagste punt waar waterlekkage moet worden gedetecteerd

## Installatiestappen

1. Bepaal de installatielocatie (de grond in het risicogebied voor waterlekkage)
2. Reinig het vloeroppervlak
3. Plaats de sensor op de grond, waarbij de sonde de grond raakt
4. Als u de kabelversie (HS2WL) gebruikt, leg dan de verlengkabel langs het te bewaken gebied
5. Plaats de batterij en de sensor wordt automatisch ingeschakeld.
6. Simulatietest: raak de sonde aan met natte vingers, het alarmlampje gaat branden
7. Koppelen via gateway

## Opmerkingen

- De sonde kan corroderen als deze gedurende lange tijd wordt blootgesteld aan een vochtige omgeving, dus controleer deze regelmatig
- Het wordt aanbevolen om samen te werken met de magneetklep om automatische uitschakeling te bereiken
- Levensduur van de batterij is ongeveer 2-3 jaar
- Meerdere sensoren om verschillende gebieden te bestrijken
