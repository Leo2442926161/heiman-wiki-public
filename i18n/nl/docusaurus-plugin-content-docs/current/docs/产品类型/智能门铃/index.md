# slimme deurbel

Haiman Smart Doorbell integreert deurbel- en beveiligingsbewakingsfuncties en ondersteunt video-intercom op afstand.

## Kernfuncties

- Deurbelknoptrigger
- Live video op afstand (Wi-Fi-versie)
- Tweerichtingsgesprekken
- Bewegingsdetectievideo
- Nachtzichtfunctie

## Toepasselijke scenario's

- Bezoekersoproepen: u kunt ze thuis of onderweg beantwoorden
- Expresslevering/afhaalmelding: bevestiging op afstand van het openen van de deur
- Beveiligingsmonitoring: registreer verdachte activiteiten voor de deur

> **Opmerking**: De Heiman deurbelproductlijn is voornamelijk gebaseerd op Zigbee/Wi-Fi-oplossingen. Raadpleeg de ambtenaar voor specifieke modelparameters.
