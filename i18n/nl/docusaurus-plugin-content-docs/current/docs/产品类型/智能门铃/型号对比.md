# Modelvergelijking

| Model | Protocol | Video | Voeding | Kenmerken | Gameplay-suggesties |
|------|------|------|------|------|----------|
| Slimme deurbel | Zigbee + Wi-Fi | 1080P optioneel | Batterij/netvoeding | Deurbel + beveiliging twee-in-één | Bezoeker belt melding mobiele telefoon, visuele intercom op afstand, opname van bewegingsdetectie |

## Aanbevolen scènegameplay

| Scenario's | Aanbevolen modellen | Koppelingsoplossingen |
|-----|----------|----------|
| Expreslevering/afhaalbeheer | Slimme deurbel | Aanbellen → melding mobiele telefoon → intercombevestiging op afstand → ontgrendelen op afstand (gekoppeld smart lock) |
