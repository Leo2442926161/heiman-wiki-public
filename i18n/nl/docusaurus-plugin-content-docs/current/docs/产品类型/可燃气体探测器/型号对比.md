|------|------|----------|------|-----|-----|----------|

## Aanbevolen scènegameplay

| Scenario's | Aanbevolen modellen | Koppelingsoplossingen |
|-----|----------|----------|
| Automatisering van keukenveiligheid | HS1CG / HS3CG | Detecteer lekken → Sluit automatisch de magneetklep → Zet de afzuigventilator aan → Pushmeldingen naar mobiele telefoon → Gekoppeld geluids- en lichtalarm |
| Huurhuis/appartement | JT-G1 | Onafhankelijk alarm vereist geen netwerk, lage kosten, voldoet aan de brandinspectie-eisen |
| Vereisten voor toezicht op afstand | HS8CG | Garantie voor dubbele communicatie + status voor bekijken op afstand van het cloudplatform |
| Commerciële keuken | HS3CG | Hoge gevoeligheid + gekoppeld rookafvoersysteem + gecentraliseerd beheer van vastgoedplatform |

## Selectiesuggesties

- **Stand-alone gebruik**: JT-G1
- **Zigbee-netwerken**: [HS1CG](../../Wireless Protocol/Zigbee/HS1CG - Detector voor brandbaar gas (Zigbee)) of [HS3CG](../../Wireless Protocol/Zigbee/HS3CG - Detector voor brandbaar gas (Zigbee))
- **Dubbele communicatiegarantie**: JT-HS8CG
