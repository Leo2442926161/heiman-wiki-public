# luchtkwaliteitsensor

Luchtkwaliteitssensoren (Air Quality Monitor) worden gebruikt om verschillende verontreinigende stoffen in de binnenlucht te detecteren en zo een gezonde leefomgeving te helpen behouden.

## Detectieparameters

Hyman-luchtkwaliteitsensoren detecteren de volgende indicatoren:

| Parameters | Beschrijving | Gezondheidsnormen |
|------|------|----------|
| PM2,5 | Fijn stof | ≤ 35 µg/m³ (daggemiddelde) |
| PM10 | Inadembare deeltjes | ≤ 50 µg/m³ (daggemiddelde) |
| CO₂ | Kooldioxide | ≤ 1000 ppm |
| TVOC | Totaal vluchtige organische stoffen | ≤ 0,6 mg/m³ |
| Temperatuur | Omgevingstemperatuur | 18-26°C |
| Vochtigheid | Relatieve vochtigheid | 40-60% RV |

## Kernfuncties

- Detectie van luchtkwaliteit met meerdere parameters
- Kleurendisplay (afhankelijk van model)
- Geluids- en lichtalarm bij overschrijding van de norm
- Historische gegevensrecords
- Gekoppelde frisse lucht/luchtreiniger

## Toepasselijke scenario's

- Slaapkamer: controleer het CO₂-niveau en vraag om ramen te openen voor ventilatie
- Kinderkamer: bescherm de ademhalingsgezondheid van kinderen
- Kantoor: verhoog de productiviteit en het comfort
- Onlangs gerenoveerde huizen: monitor de uitstoot van formaldehyde/TVOC
