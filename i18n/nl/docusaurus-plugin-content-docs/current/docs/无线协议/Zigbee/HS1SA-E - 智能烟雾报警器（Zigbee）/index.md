# HS1SA-E - Slimme rookmelder (Zigbee)

## Overzicht

HS1SA-E is een standaard slimme Zigbee-rookmelder die door Hyman voor de EU-markt is gelanceerd. Het maakt gebruik van foto-elektrische rookdetectietechnologie, is geslaagd voor de EN 14604-certificering en heeft de Red Dot Design Award gewonnen.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Infrarood foto-elektrisch |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | Gelijkstroom 3V (CR123A) |
| Levensduur batterij | Ongeveer 3 jaar |
| Communicatieafstand | ≤70 m (open ruimte) |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RV |
| Installatiemethode | Plafondmontage |

## Kenmerken

- Foto-elektrische rookdetectie om rook van langzame brandwonden en open vuur te detecteren
- 85 dB hoog geluids- en lichtalarm
- Herinnering batterij bijna leeg
- Zelftestfunctie (test met één klik)
- Anti-sabotagealarm
-Ondersteuning op afstand dempen
- OTA-firmware-upgrade

## Certificeringsnormen

- EN 14604 (Europese norm voor rookmelders)
- Red Dot Designprijs
-CE/ROOD

## Toepasselijke accessoires

- CR123A-batterij
- Bevestigingsschroeven/expansiebuis
