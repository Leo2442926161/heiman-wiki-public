#HS2FD - Valdetector (Zigbee)

## Overzicht

HS2FD is een slimme valdetector van Zigbee, gelanceerd door Hyman. Het is speciaal ontworpen voor ouderen. Het heeft een ingebouwde versnellingsmeter en gyroscoop. Het maakt gebruik van een AI-algoritme om de valhouding te identificeren en kan automatisch een noodsignaal uitzenden.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Accelerometer + Gyroscoop |
| Detectietechnologie | AI Valdetectie-algoritme |
| Voedingsmethode | DC 3V-batterij |
| Draagmethode | Riemclip/koord |

## Kenmerken

- Automatische valdetectie
- Roep handmatig om hulp met één klik
- AI-algoritme maakt onderscheid tussen vallen en normale activiteiten (zitten, bukken) om vals alarm te verminderen
- Draagbaar ontwerp (riemclip of sleutelkoord)
- Herinnering batterij bijna leeg

## Toepasselijke scenario's

- Thuiszorg voor ouderen: 24-uurszorg voor alleenwonende ouderen
- Verpleeghuizen: gecentraliseerde bewaring en beheer
- Revalidatiecentrum: monitoring van het risico op vallen bij postoperatieve patiënten
