# HS6GW - Slimme gateway (Zigbee)

## Overzicht

HS6GW is een slimme Zigbee-gateway gelanceerd door Heiman. Als coördinator van Heimans Zigbee-netwerk maakt het verbinding met internet via Wi-Fi of Ethernet-uplink en ondersteunt het de toegang tot maximaal 128 subapparaten.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 (coördinator) |
| Uplink-verbinding | Wi-Fi/Ethernet |
| Max. knooppunten | 128 subapparaten |
| Voedingsmethode | DC 5V USB-voeding |

## Kenmerken

- Zigbee Netwerkcoördinator
- Wi-Fi / Ethernet-uplink-verbinding
- Automatische apparaatdetectie en koppeling
- Lokale scènekoppeling
- Toegang op afstand (via Heiman Cloud Platform)
- Ondersteunt maximaal 128 subapparaten
- OTA-firmware-upgrade

## Toepasselijke scenario's

- Het kernverbindingspunt van de Heiman-intelligentie voor het hele huis
- Werk samen met Haiman APP om monitoring op afstand te realiseren
