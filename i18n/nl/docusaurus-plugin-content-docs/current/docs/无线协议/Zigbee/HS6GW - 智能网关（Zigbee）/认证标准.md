# Certificatienormen

HS6GW (Zigbee-versie) heeft de volgende certificeringen doorstaan:

- **CE/RED** — Richtlijn draadloze apparatuur van de Europese Unie
- **CCC** — China verplichte productcertificering, zie [CCC China Compulsory Certification](../../../../Certification Standards/CCC China Compulsory Certification.md)
