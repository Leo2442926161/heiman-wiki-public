# HS3WL - Waterlekdetector (Zigbee)

## Overzicht

HS3WL is Hyman's Zigbee-waterdompeldetector van de derde generatie. Het heeft verbeterde detectiegevoeligheid en draadloze prestaties op basis van de vorige generatie, en is geschikt voor monitoringscenario's voor waterlekkage.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Detectiemethode | Sondedetectie |
| Voedingsmethode | DC 3V-batterij |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RV |

## Kenmerken

- Realtime detectie van waterlekkage/waterophoping
- Verbeterde detectiegevoeligheid
- Zoemeralarm
- Herinnering batterij bijna leeg
- Sonde anti-corrosie ontwerp
