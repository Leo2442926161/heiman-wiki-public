# Certificatienormen

HS8SA heeft de volgende certificeringen behaald:

- **EN 14604** — Europese norm voor rookmelders (EU-type), zie [EN Europese norm](../../../../ApprovalStandards/EN European Standard.md)
- **UL 217** — Amerikaanse veiligheidsnorm voor rookmelders (VS-model), zie [UL Underwriters Laboratories](../../../../certification-standards/ULUnderwriters Laboratories.md)
- **CE/RED/EMC** — Richtlijn draadloze apparatuur van de Europese Unie en elektromagnetische compatibiliteit
- **Zigbee 3.0 / Q-Label** — Zigbee Alliance-certificering, zie [Overige certificeringen](../../../Certification Standards/Other Certifications.md)
