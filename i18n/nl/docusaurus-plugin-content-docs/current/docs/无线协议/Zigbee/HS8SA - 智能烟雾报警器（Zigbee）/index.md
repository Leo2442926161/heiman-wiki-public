#HS8SA - Slimme rookmelder (Zigbee)

## Overzicht

HS8SA is Hyman's Zigbee + Wi-Fi slimme rookmelder voor de Amerikaanse markt. Het heeft een koppelingsfunctie bij hoge temperaturen en is geslaagd voor de UL 217-certificering.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 / Wi-Fi |
| Sensortype | Infrarood foto-elektrisch |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | Gelijkstroom 3V (CR123A) |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Certificeringsnormen | UL 217 |
| Toepasselijke markten | Verenigde Staten |

## Kenmerken

- Foto-elektrische rookdetectie
- Koppelingsalarm voor hoge temperaturen
- 85 dB hoog geluids- en lichtalarm
- Herinnering batterij bijna leeg
- Zelftestfunctie
- Anti-sabotagealarm
-Ondersteuning op afstand dempen
- Dubbele protocolcommunicatie (Zigbee + Wi-Fi)

## Certificeringsnormen

- UL 217 (Amerikaanse rookmelderstandaard)
