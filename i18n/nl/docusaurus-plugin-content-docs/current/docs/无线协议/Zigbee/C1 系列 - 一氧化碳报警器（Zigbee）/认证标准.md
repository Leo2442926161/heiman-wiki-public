# Certificatienormen

De C1-serie heeft de volgende certificeringen behaald:

- **EN 50291** — Europese norm voor koolmonoxidemelders, zie [EN Europese norm](../../../../Certification Standards/EN European Standard.md)
- **CE/RED/EMC** — Richtlijn draadloze apparatuur van de Europese Unie en elektromagnetische compatibiliteit
