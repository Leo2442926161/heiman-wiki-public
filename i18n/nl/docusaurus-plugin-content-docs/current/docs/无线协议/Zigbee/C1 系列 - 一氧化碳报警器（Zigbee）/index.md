# C1-serie - Koolmonoxidemelder (Zigbee)

## Overzicht

De C1-serie is een basis Zigbee koolmonoxidemelder gelanceerd door Heiman. Het maakt gebruik van elektrochemische sensoren en is geschikt voor scenario's voor het detecteren van koolmonoxidelekken. Het is geslaagd voor de EN 50291-certificering.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Elektrochemisch |
| Detectiebereik | 0 ~ 1000 ppm |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | Werkt op batterijen |
| Levensduur sensor | Ongeveer 7 jaar |
| Bedrijfstemperatuur | -10°C ~ +50°C |

## Kenmerken

- Elektrochemische CO-detectie
- Concentratieclassificatie-alarm
- 85 dB hoorbaar en visueel alarm
- Herinnering batterij bijna leeg
- Zelftestfunctie

## Certificeringsnormen

-EN 50291
