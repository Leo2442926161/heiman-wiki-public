#HS3MS - Infraroodsensor voor het menselijk lichaam (Zigbee)

## Overzicht

HS3MS is de derde generatie Zigbee-infraroodsensor voor het menselijk lichaam van Hyman. Het heeft een 360°-plafondgemonteerd ontwerp en de detectiegevoeligheid en dekking zijn aanzienlijk verbeterd in vergelijking met de vorige generatie.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Passief infrarood (PIR) |
| Detectiehoek | 360° (plafondmontage) |
| Detectieafstand | ≤7m (25°C) |
| Voedingsmethode | DC 3V lithiumbatterij |
| Levensduur batterij | Ongeveer 3 jaar |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Installatiemethode | Plafondmontage |

## Kenmerken

- Mobiele menselijke detectie
- 360° omnidirectionele dekking
- Herinnering batterij bijna leeg
- Anti-sabotagealarm
- Verbeterde detectiegevoeligheid
- Ondersteuning van scènekoppeling
