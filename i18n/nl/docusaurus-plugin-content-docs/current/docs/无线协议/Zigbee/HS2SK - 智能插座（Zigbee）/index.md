#HS2SK - Slimme contactdoos (Zigbee)

## Overzicht

HS2SK is een Zigbee slimme meteraansluiting gelanceerd door Heiman. Het ondersteunt schakelaarbediening op afstand en realtime meting van het energieverbruik. Het is geschikt voor intelligent beheer van huishoudelijke apparaten en heeft een nauwkeurigheid van ±2%.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Nominale spanning | AC 220V (Chinese standaard) / AC 110V (Amerikaanse standaard) |
| Maximale belasting | 16A / 3680W |
| Meetnauwkeurigheid | ±2% |
| Voedingsmethode | Netvoeding |

## Kenmerken

- Afstandsbediening met schakelaar
- Realtime vermogens-/brandstofmeting
- Overbelastingsbeveiliging
- Tijdschakelaar
- Vermogensstatistieken
- Ondersteuning van scènekoppeling

## Toepasselijke scenario's

- Schakel vergeten apparaten op afstand uit
- Statistieken van het energieverbruik van de airconditioner/boiler
- Verlaat de thuismodus om niet-essentiële apparaten met één klik uit te schakelen
