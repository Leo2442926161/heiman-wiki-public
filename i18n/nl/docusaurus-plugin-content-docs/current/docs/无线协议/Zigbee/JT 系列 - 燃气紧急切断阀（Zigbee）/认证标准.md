# Certificatienormen

De JT-serie heeft de volgende certificeringen behaald:

- **GB 15322/CCC** — China verplichte productcertificering, zie [CCC China verplichte certificering](../../../../Certification Standards/CCC China verplichte certificering.md)
- **CE** — Conformiteitsverklaring van de Europese Unie
