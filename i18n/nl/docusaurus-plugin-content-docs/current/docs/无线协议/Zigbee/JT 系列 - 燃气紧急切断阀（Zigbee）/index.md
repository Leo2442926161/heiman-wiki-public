# JT-serie - Gasnoodafsluiter (Zigbee)

## Overzicht

De gasnoodafsluiter uit de JT-serie is een uitvoeringsapparaat dat wordt gebruikt in combinatie met de Hyman Zigbee-detector voor brandbaar gas. Wanneer een gaslek wordt gedetecteerd, kan de klep van de gasleiding op afstand of automatisch worden gesloten om ongelukken te voorkomen.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Typ | Stand-alone + Zigbee-koppeling |
| Testgas | Aardgas / Vloeibaar petroleumgas |
| Certificeringsnormen | GB 15322 |
| Bedrijfstemperatuur | -10°C ~ +50°C |

## Kenmerken

- Gekoppeld aan HS1CG / HS3CG om de klep automatisch te sluiten
-Ondersteuning voor het op afstand handmatig sluiten van de klep
- Druk op het lekalarm
- Magneetventielaandrijving
