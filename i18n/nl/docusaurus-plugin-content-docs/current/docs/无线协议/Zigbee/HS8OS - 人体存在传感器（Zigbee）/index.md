#HS8OS - Menselijke aanwezigheidssensor (Zigbee)

## Overzicht

HS8OS is een Zigbee-sensor voor menselijke aanwezigheid, gelanceerd door Hyman. Het maakt gebruik van millimetergolfradartechnologie en kan de aanwezigheid van een menselijk lichaam in een statische toestand detecteren (zoals liggen, rustig zitten), waardoor de tekortkomingen worden gecompenseerd van traditionele PIR-sensoren die alleen beweging kunnen detecteren.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Millimetergolfradar (mmWave) |
| Detectiebereik | Straal 3 ~ 5 m |
| Detectievermogen | Aanwezigheid van een stilstaand menselijk lichaam (inclusief detectie van micro-ademhaling) |
| Voedingsmethode | AC-voeding |
| Installatiemethode | Plafondmontage |

## Kenmerken

- Stationaire detectie van menselijk lichaam (microbewegingen van ademhaling)
- Dekkingsradius 3~5 meter
- Plafondmontage, optimale detectiehoek
- Kan complementair gebruikt worden met PIR-sensoren

## Toepasselijke scenario's

- Intelligente verlichting: altijd aan als er iemand is, automatisch uit als er niemand in de buurt is
- Energiebesparende airconditioning: schakelt de airconditioner automatisch uit als er mensen lopen
- Slimme ouderenzorg: Vroegtijdige waarschuwing voor ouderen als ze langere tijd stil blijven staan
