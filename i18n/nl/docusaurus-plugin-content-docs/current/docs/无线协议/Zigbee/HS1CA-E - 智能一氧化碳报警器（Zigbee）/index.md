# HS1CA-E - Slimme koolmonoxidemelder (Zigbee)

## Overzicht

HS1CA-E is Hyman's Zigbee slimme koolmonoxide (CO)-alarm voor de EU-markt. Het maakt gebruik van een elektrochemische sensor en kan koolmonoxidegas met een concentratie van 0~1000 ppm nauwkeurig detecteren. Het is geslaagd voor de EN 50291-certificering.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Elektrochemisch |
| Detectiebereik | 0 ~ 1000 ppm |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | Gelijkstroom 3V (CR123A) |
| Levensduur sensor | Ongeveer 7 jaar |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RV |
| Installatiemethode | Wandmontage |

## Kenmerken

- Zeer nauwkeurige elektrochemische CO-sensor
- Concentratie-gegradueerd alarm (geluids- en lichtmelding)
- 85 dB alarmgeluid
- Herinnering batterij bijna leeg
- Herinnering voor het verstrijken van de levensduur van de sensor
- Zelftestfunctie
- OTA-firmware-upgrade

## Certificeringsnormen

- EN 50291 (Europese norm voor koolmonoxidemelders)
-CE/ROOD

## Toepasselijke accessoires

- CR123A-batterij
