# HS1DS - Magnetische deursensor (Zigbee)

## Overzicht

HS1DS is een magnetische Zigbee-deursensor gelanceerd door Heiman. Het wordt gebruikt om de openings- en sluitstatus van deuren en ramen te detecteren. Het is een basiscomponent van het intelligente beveiligingssysteem.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Reedschakelaar |
| Voedingsmethode | CR1632 knoopcelbatterij |
| Levensduur batterij | Ongeveer 2 ~ 3 jaar |
| Inbouwvrijheid | ≤10 mm |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RV |

## Kenmerken

- Realtime detectie van de status van openen/sluiten van deuren en ramen
- Sabotagealarm (sabotage)
- Herinnering batterij bijna leeg
-Ondersteunt koppelingen (deur openen, verlichting aandoen, inschakelen bij het verlaten van huis, enz.)
- Eenvoudig te installeren (3M plakband of schroefbevestiging)

## Toepasselijke accessoires

- CR1632 knoopcelbatterij
- 3M dubbelzijdige tape/schroeven
