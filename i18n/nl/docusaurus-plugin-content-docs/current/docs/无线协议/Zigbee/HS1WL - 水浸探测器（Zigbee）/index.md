# HS1WL - Waterlekdetector (Zigbee)

## Overzicht

HS1WL is de eerste generatie Zigbee-waterdompeldetector gelanceerd door Hyman. Het maakt gebruik van de sondedetectiemethode en kan waterlekkage of waterophoping effectief detecteren. Het is een basisbewakingsapparatuur voor onderdompeling in water.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Detectiemethode | Sondedetectie |
| Voedingsmethode | DC 3V-batterij |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RV |

## Kenmerken

- Realtime detectie van waterlekkage/waterophoping
- Zoemeralarm
- Herinnering batterij bijna leeg
- Sonde anti-corrosie ontwerp
