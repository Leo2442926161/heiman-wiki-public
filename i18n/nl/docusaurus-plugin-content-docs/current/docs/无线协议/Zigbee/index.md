#Zigbee

Zigbee is een draadloos mesh-netwerkprotocol met laag vermogen en lage gegevenssnelheid, gebaseerd op de IEEE 802.15.4-standaard. Het is een van de meest volwassen en populaire protocollen op het gebied van smart home.

## Hyman en Zigbee

Hyman is een van de eerste Chinese fabrikanten die het Zigbee-protocol adopteert en heeft een enorme Zigbee-productlijn. Hyman Zigbee-apparaten zijn compatibel met de **Zigbee 3.0**-standaard en kunnen samenwerken met reguliere Zigbee-gateways en -coördinatoren.

## Technische specificaties

| Kenmerken | Beschrijving |
|------|------|
| Frequentieband | 2,4 GHz (wereldwijd) |
| Overdrachtssnelheid | Tot 250 kbps |
| Netwerktopologie | Mesh zelfherstellend netwerk |
| Aantal knooppunten | Eén netwerk ondersteunt maximaal 65535 knooppunten |
| Communicatieafstand | Ongeveer 10-30 meter binnenshuis (zichtbare obstakels) |
| Stroomverbruik | Extreem laag, levensduur van de sensorbatterij tot 3 jaar |

## Compatibele platforms

Hyman Zigbee-apparaten kunnen worden aangesloten op de volgende platforms:

- **Zigbee2MQTT** — Open source Zigbee naar MQTT-bridge
- **ZHA** — Home Assistant met ingebouwde Zigbee-integratie
- **Hubitat** — Gelokaliseerde intelligente hub
- **Homey** — Slimme hub met meerdere protocollen
- **SmartThings Hub** — Samsung smarthome-hub
- **Tuya Zigbee Gateway** — Tuya-ecosysteem
- **Hyman HS6GW Gateway** — Officiële gateway
