# M1-Z - Infraroodsensor voor menselijk lichaam (Zigbee)

## Overzicht

M1-Z is een compacte Zigbee-infraroodsensor voor het menselijk lichaam, gelanceerd door Heiman. Het ondersteunt een detectiehoek van 110° of 360° (afhankelijk van de installatiemethode), heeft een ingebouwde verlichtingssensor en is geschikt voor slimme verlichtings- en beveiligingsscenario's.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Passief infrarood (PIR) + verlichting |
| Detectiehoek | 110° (wandmontage) / 360° (plafond) |
| Detectieafstand | ≤7m (25°C) |
| Verlichtingssterktebereik | 0 ~ 1200 lux |
| Voedingsmethode | DC 3V lithiumbatterij |
| Levensduur batterij | Ongeveer 3 jaar |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Installatiemethode | Wandmontage/plafond |

## Kenmerken

- Mobiele menselijke detectie
- Verlichtingsdetectie (0~1200 Lux)
- 110° / 360° schakelbaar
- Herinnering batterij bijna leeg
- Anti-sabotagealarm
- Compact ontwerp
