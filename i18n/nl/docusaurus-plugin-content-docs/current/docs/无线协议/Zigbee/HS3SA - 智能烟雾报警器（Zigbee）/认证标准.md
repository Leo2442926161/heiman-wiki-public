# Certificatienormen

HS3SA heeft de volgende certificeringen behaald:

- **EN 14604** — Europese norm voor rookmelders, zie [EN Europese norm](../../../../ApprovalStandards/EN European Standard.md)
- **CE/RED/EMC** — Richtlijn draadloze apparatuur van de Europese Unie en elektromagnetische compatibiliteit
- **Zigbee 3.0 / Q-Label** — Zigbee Alliance-certificering, zie [Overige certificeringen](../../../Certification Standards/Other Certifications.md)
