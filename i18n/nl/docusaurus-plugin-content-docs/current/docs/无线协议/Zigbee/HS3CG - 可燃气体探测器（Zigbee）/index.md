#HS3CG - Detector voor brandbaar gas (Zigbee)

## Overzicht

HS3CG is Hyman's derde generatie Zigbee-detector voor brandbaar gas. Het heeft een verbeterde detectienauwkeurigheid en reactiesnelheid op basis van de eerste generatie, en is geschikt voor de detectie van aardgas- en vloeibaar petroleumgaslekken.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Halfgeleider |
| Testgas | Aardgas (methaan)/vloeibaar petroleumgas (LPG) |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | AC 220V / DC 12V |
| Certificeringsnormen | GB 15322 |
| Bedrijfstemperatuur | -10°C ~ +50°C |

## Kenmerken

- Aardgas-/vloeibaar gaslekdetectie
- Concentratieclassificatie-alarm
- Automatische uitschakelkoppeling (met magneetventiel)
- 85 dB hoorbaar en visueel alarm
- Zelftestfunctie
- Herinnering voor het verstrijken van de levensduur van de sensor
- Verbeterde draadloze prestaties
