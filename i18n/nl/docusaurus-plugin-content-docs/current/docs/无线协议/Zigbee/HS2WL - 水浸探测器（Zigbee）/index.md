#HS2WL - Waterlekdetector (Zigbee)

## Overzicht

HS2WL is de tweede generatie Zigbee-waterdompeldetector, gelanceerd door Hyman. Het ondersteunt tweekanaals detectie van sondes en kabels en kan worden gebruikt om waterlekkage te detecteren, zoals het barsten van waterleidingen en het overlopen van gootstenen.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Detectiemethode | Sondedetectie + kabeldetectie (tweekanaals) |
| Voedingsmethode | DC 3V-batterij |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RV |

## Kenmerken

- Realtime detectie van waterlekkage/waterophoping
- Sonde + kabel tweekanaals detectie
- Zoemeralarm
- Herinnering batterij bijna leeg
- Sonde anti-corrosie ontwerp

## Toepasselijke accessoires
