#HS1CG - Detector voor brandbaar gas (Zigbee)

## Overzicht

HS1CG is een Zigbee-detector voor brandbaar gas, gelanceerd door Hyman. Het maakt gebruik van halfgeleidersensoren en kan lekkages van aardgas (methaan) en vloeibaar petroleumgas (LPG) detecteren. Het is geslaagd voor de GB 15322-certificering en ondersteunt het automatisch sluiten van de klep van de elektromagnetische klep.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Halfgeleider |
| Testgas | Aardgas (methaan)/vloeibaar petroleumgas (LPG) |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | AC 220V / DC 12V |
| Certificeringsnormen | GB 15322 |
| Bedrijfstemperatuur | -10°C ~ +50°C |

## Kenmerken

- Aardgas-/vloeibaar gaslekdetectie
- Concentratieclassificatie-alarm
- Automatische uitschakelkoppeling (met magneetventiel)
- 85 dB hoorbaar en visueel alarm
- Zelftestfunctie
- Herinnering voor het verstrijken van de levensduur van de sensor

## Certificeringsnormen

- GB 15322 (Chinese norm voor brandbare gasdetectoren)

## Toepasselijke accessoires

- Gasnoodafsluiter uit de JT-serie
