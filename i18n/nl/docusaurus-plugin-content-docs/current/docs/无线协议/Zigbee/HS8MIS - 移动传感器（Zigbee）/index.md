#HS8MIS - Mobiele sensor (Zigbee)

## Overzicht

HS8MIS is een multifunctionele Zigbee-bewegingssensor gelanceerd door Heiman. Het integreert PIR-functies voor menselijke detectie en verlichtingsdetectie en is geschikt voor slimme scènes met dubbele bewaking van licht en beweging.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Passief infrarood (PIR) + verlichting |
| Detectiehoek | 110° |
| Detectieafstand | ≤7 m |
| Voedingsmethode | DC 3V-batterij |

## Kenmerken

- Mobiele menselijke detectie
- Verlichtingsdetectie
- Ontwerp met laag stroomverbruik
- Ondersteuning van scènekoppeling
