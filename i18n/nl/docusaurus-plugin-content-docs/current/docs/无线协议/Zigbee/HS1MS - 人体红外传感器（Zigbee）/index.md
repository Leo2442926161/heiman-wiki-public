#HS1MS - Infraroodsensor voor het menselijk lichaam (Zigbee)

## Overzicht

HS1MS is een standaard Zigbee infraroodsensor voor het menselijk lichaam (PIR), gelanceerd door Hyman. Het maakt gebruik van passieve infrarooddetectietechnologie en kan infrarode warmtestraling waarnemen die wordt uitgezonden door een bewegend menselijk lichaam. Het wordt veel gebruikt in slimme verlichtings- en beveiligingsscenario's.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Passief infrarood (PIR) |
| Detectiehoek | 110° |
| Detectieafstand | ≤7m (25°C) |
| Voedingsmethode | DC 3V lithiumbatterij |
| Levensduur batterij | Ongeveer 3 jaar |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Installatiemethode | Wandmontage |

## Kenmerken

- Mobiele menselijke detectie
- 110° groothoekdetectie
- Herinnering batterij bijna leeg
- Anti-sabotagealarm
- Ondersteuning van scènekoppeling
