# HS8MLS - Micro-bewegingssensor (Zigbee)

## Overzicht

HS8MLS is een langeafstands-Zigbee-infraroodsensor voor het menselijk lichaam (PIR), gelanceerd door Heiman. Het is speciaal ontworpen voor scènes waarbij detectie op grote afstand vereist is en is geschikt voor open ruimtes zoals gangen en hallen.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Zigbee 3.0 |
| Sensortype | Passief infrarood (PIR) |
| Detectieafstand | Detectie op lange afstand |
| Voedingsmethode | DC 3V-batterij |

## Kenmerken

- Bewegingsdetectie over lange afstanden
- Ontwerp met laag stroomverbruik
- Ondersteuning van scènekoppeling
