# Technische parameters

Hieronder volgen algemene technische parameters voor Heiman Zigbee-apparaten.

## Draadloze parameters

| Parameter | Waarde |
|------|-----|
| Protocolstandaarden | Zigbee 3.0 (IEEE 802.15.4) |
| Werkfrequentieband | 2,4 GHz (wereldwijde gemeenschappelijke ISM-frequentieband) |
| Overdrachtssnelheid | Tot 250 kbps |
| Netwerktopologie | Mesh zelfherstellend netwerk |
| Knooppuntcapaciteit | Maximaal 65535 knooppunten in één netwerk |
| Communicatieafstand | Ongeveer 10~30 meter binnenshuis (obstakels aan de muur zien) |
| Versleutelingsmethode | AES-128-CCM |

## Compatibiliteit

| Project | Beschrijving |
|------|------|
| Zigbee-coördinator | Compatibel met Zigbee 3.0 standaardcoördinator |
| Ondersteunde platforms | Zigbee2MQTT, ZHA (Home Assistant), Hubitat, Homey, SmartThings, Tuya Zigbee gateway, Haiman HS6GW gateway |
| OTA-upgrade | Ondersteuning van draadloze firmware-upgrade (sommige modellen) |

## Elektriciteits- en stroomverbruik

| Parameter | Waarde |
|------|-----|
| Voedingsmethode | Werkt op batterijen (CR123A / CR2450 / AA) of USB / netstroom (stopcontacttype) |
| Levensduur batterij | Ongeveer 2-3 jaar voor sensoren (afhankelijk van de rapportagefrequentie) |
| Stroomverbruik in stand-by | Microampèreniveau, ondersteunt slaaponderzoeksmechanisme |

## Omgevingsparameters

| Parameter | Waarde |
|------|-----|
| Bedrijfstemperatuur | -10°C ~ +55°C |
| Bedrijfsvochtigheid | ≤ 95% RH (niet-condensatie) |
| IP-beschermingsniveau | Afhankelijk van het specifieke model (sensortype meestal IP30) |
