# HS1DS-M - Intelligente deursensor (materie over draad)

## Overzicht

HS1DS-M is de Matter over Thread-versie van de Heiman slimme deursensor. Het ondersteunt native toegang tot alle Matter-compatibele platforms zoals Apple Home, Google Home, Amazon Alexa, Home Assistant, enz. zonder dat een eigen gateway nodig is.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | Kwestie boven draad |
| Voedingsmethode | CR1632 knoopcelbatterij |
| Levensduur batterij | Ongeveer 2-3 jaar |
| Installatieafstand | ≤10 mm |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RV |

## Kenmerken

- Realtime detectie van de status van openen/sluiten van deuren en ramen
- Sabotagealarm (sabotage)
- Herinnering batterij bijna leeg
- Platformonafhankelijk compatibel: Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Thread Mesh-netwerk, lokale controle met lage latentie
- Geen speciale gateway vereist, Thread-borderrouter vereist
