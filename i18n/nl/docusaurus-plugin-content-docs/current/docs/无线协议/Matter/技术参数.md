# Technische parameters

Hieronder volgen algemene specificaties voor Hyman Matter-apparatuur.

## Draadloze parameters

| Parameter | Waarde |
|------|-----|
| Onderliggend protocol | Draad (IEEE 802.15.4) |
| Bedrijfsfrequentieband | 2,4 GHz (ISM-band) |
| Standaarden voor applicatielagen | Kwestie 1.x |
| Netwerktopologie | Mesh zelfherstellend netwerk |
| Versleutelingsmethode | AES-128-CCM |

## Hardwarevereisten

| Project | Beschrijving |
|------|------|
| Grensrouter | Vereist, zoals Apple HomePod mini, Apple TV 4K, Google Nest Hub 2e generatie, etc. |
| Ondersteuning mobiele terminal | iOS / Android (netwerkdistributie via verschillende ecologische apps) |
| Cross-ecologische interoperabiliteit | Apple HomeKit, Google Home, Amazon Alexa, Home Assistant, SmartThings |

## Elektriciteits- en stroomverbruik

| Parameter | Waarde |
|------|-----|
| Voedingsmethode | Werkt op batterijen (CR123A / CR2450, enz.) |
| Levensduur batterij | Ongeveer 1~3 jaar (afhankelijk van het apparaattype en de gebruiksfrequentie) |
| Stroomverbruik in stand-by | Extreem laag, ondersteunt het slaap-waakmechanisme |

## Omgevingsparameters

| Parameter | Waarde |
|------|-----|
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤ 95% RH (niet-condensatie) |
