# M1-M Infraroodsensor voor menselijk lichaam (materie)

M1-M is een **Matter over Thread** infraroodsensor voor het menselijk lichaam, gelanceerd door Hyman. Het integreert PIR-detectie- en verlichtingsdetectiefuncties en is geschikt voor slimme domotica-scenario's.

## Productoverzicht

- **Producttype**: infraroodsensor van het menselijk lichaam
- **Communicatieprotocol**: kwestie boven draad
- **Detectiemethode**: Passief infrarood (PIR) + verlichtingsdetectie
- **Typisch scenario**: lichten gaan aan als er mensen komen, beveiligingskoppeling, omgevingsautomatisering

## Technische parameters

| Parameter | Waarde |
|------|-----|
| Detectiehoek | 110° (wandmontage) / 360° (plafond) |
| Detectieafstand | ≤7m (25°C) |
| Verlichtingsbereik | 0~1200Lux |
| Communicatieprotocol | Kwestie over draad (IEEE 802.15.4) |
| Bedrijfsfrequentieband | 2,4 GHz |
| Voedingsmethode | DC 3V lithiumbatterij |
| Levensduur batterij | Ongeveer 1 jaar (materiemodus) |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RH (geen condensatie) |

## Kenmerken

- Mobiele menselijke detectie
- Detectie van omgevingsverlichting
- Detectieafstand ≤7m
- Ondersteuning van wand- of plafondmontage
- Multi-ecologische platformonafhankelijke interoperabiliteit
