# HS1SA-M Intelligente rookmelder (materie)

HS1SA-M is een slimme rookmelder **Matter over Thread**, gelanceerd door Hyman. Het maakt gebruik van foto-elektrische rookdetectietechnologie en ondersteunt cross-ecologische smart home-platforminterconnectie.

## Productoverzicht

- **Producttype**: Slimme rookmelder
- **Communicatieprotocol**: kwestie boven draad
- **Certificeringsnorm**: EN 14604
- **Typische scenario's**: Brandwaarschuwing in huizen, appartementen, hotels, enz.

## Technische parameters

| Parameter | Waarde |
|------|-----|
| Sensortype | Infrarood foto-elektrisch |
| Communicatieprotocol | Kwestie over draad (IEEE 802.15.4) |
| Bedrijfsfrequentieband | 2,4 GHz |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | DC 3V (CR123A-lithiumbatterij) |
| Levensduur batterij | Ongeveer 3 jaar |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RH (geen condensatie) |

## Kenmerken

- Foto-elektrische rookdetectie om langzame verbranding en open vuurrook te detecteren
- 85 dB hoog geluids- en lichtalarm
- Herinnering batterij bijna leeg
- Zelftestfunctie (test met één klik)
- Anti-sabotagealarm
- Dempen op afstand
- Multi-ecologische platformonafhankelijke interoperabiliteit

## Certificering

- EN 14604-certificering
- Matter-certificering

## Compatibele platforms

| Platform | Ondersteuning |
|------|----------|
| Apple-huis | ✅ Draadrandfrees vereist |

Er is een Thread-borderrouter vereist, zoals Apple HomePod mini, Google Nest Hub 2e generatie, Amazon Echo 4e generatie, etc.
