# Certificatienormen

HS1SA-M heeft de volgende certificeringen behaald:

- **EN 14604** — Europese norm voor rookmelders, zie [EN Europese norm](../../../../ApprovalStandards/EN European Standard.md)
- **CE-markering** — EU-conformiteitsverklaring, zie [CE EU Conformity](../../../../certification-standards/CE EU-Conformity.md)
- **Matter-certificering** — Geslaagd voor de CSA Alliance Matter-certificering om platformonafhankelijke interconnectie van slimme huizen te garanderen
- **Red Dot Design Award** — Red Dot Product Design Award
