#zaak

Matter (voorheen bekend als Project CHIP) is een smart home-interconnectiestandaard gelanceerd door de CSA Connection Standards Alliance, met als doel merkbarrières te slechten en inter-ecologische interoperabiliteit te bereiken.

## Heyman en materie

Hyman was een van de eersten die het Matter-protocol toepaste en lanceerde **de eerste Matter-gecertificeerde koolmonoxidemelders ter wereld**. Hyman's Matter-apparaten maken allemaal gebruik van de **Matter over Thread**-oplossing.

## Technische kenmerken

| Kenmerken | Beschrijving |
|------|------|
| Onderliggend protocol | Thread (mesh-netwerk met laag vermogen) |
| Netwerktopologie | Mesh zelfherstellend netwerk, hoe meer apparaten, hoe breder de dekking |
| Stroomverbruik | Extreem laag, kan 1-3 jaar werken op batterijvoeding |
| Beveiliging | Ondersteunt AES-128-CCM-codering, apparaatverificatie vereist |
| Interoperabiliteit | Over Apple HomeKit, Google Home, Amazon Alexa, Home Assistant en meer |

## Vereisten voor grensrouters

Voor het gebruik van Matter over Thread-apparaten is een Thread-borderrouter vereist:

- Apple HomePod mini / HomePod 2e generatie / Apple TV 4K
- Google Nest Hub 2e generatie / Nest Hub Max
-Amazon Echo 4e generatie
- Home Assistant Groen + Connect ZBT-2
- Huiselijke Pro
