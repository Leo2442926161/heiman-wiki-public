# H1-M temperatuur- en vochtigheidssensor (materie)

H1-M is een **Matter over Thread** temperatuur- en vochtigheidssensor gelanceerd door Hyman, die de omgevingstemperatuur en relatieve vochtigheid in realtime kan bewaken en de koppeling van slimme huisautomatisering ondersteunt.

## Productoverzicht

- **Producttype**: temperatuur- en vochtigheidssensor
- **Communicatieprotocol**: kwestie boven draad
- **Detectieparameters**: temperatuur, relatieve vochtigheid
- **Typische scenario's**: omgevingsmonitoring in woningen, kantoren, wijnkelders, enz.

## Technische parameters

| Parameter | Waarde |
|------|-----|
| Temperatuurnauwkeurigheid | ±0,3°C |
| Nauwkeurigheid vochtigheid | ±3% RV |
| Communicatieprotocol | Kwestie over draad (IEEE 802.15.4) |
| Bedrijfsfrequentieband | 2,4 GHz |
| Voedingsmethode | CR2-lithiumbatterij |
| Levensduur batterij | Ongeveer 2~3 jaar |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RH (geen condensatie) |

## Kenmerken

- Realtime temperatuur- en vochtigheidsmonitoring
- Zeer nauwkeurige detectie (temperatuur ±0,3°C, vochtigheid ±3% RH)
- Herinnering batterij bijna leeg
- Multi-ecologische platformonafhankelijke interoperabiliteit
