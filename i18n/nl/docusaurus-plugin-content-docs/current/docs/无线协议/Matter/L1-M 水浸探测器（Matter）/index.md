# L1-M Waterdompeldetector (materie)

L1-M is een **Matter over Thread** wateronderdompelingsdetector gelanceerd door Hyman. Het detecteert waterlekkage of opgehoopt water via een sonde, waardoor eigendomsverlies als gevolg van barsten van de waterleiding effectief kan worden voorkomen.

## Productoverzicht

- **Producttype**: Detectiesensor voor onderdompeling in water
- **Communicatieprotocol**: kwestie boven draad
- **Detectiemethode**: detectie van contact met sonde
- **Typische scenario's**: Keuken, badkamer, wasruimte, detectie van waterlekkage in de kelder

## Technische parameters

| Parameter | Waarde |
|------|-----|
| Detectiemethode | Type roestvrijstalen sondecontact |
| Communicatieprotocol | Kwestie over draad (IEEE 802.15.4) |
| Bedrijfsfrequentieband | 2,4 GHz |
| Voedingsmethode | Werkt op batterijen |
| Alarmmethode | Zoemer + app-push |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RH (geen condensatie) |

## Kenmerken

- Realtime detectie van waterlekkage/stilstaand water
- Zoemer ter plaatse alarm
- App op afstand alarm duwen
- Herinnering batterij bijna leeg
- Sonde anti-corrosie ontwerp
- Multi-ecologische platformonafhankelijke interoperabiliteit
