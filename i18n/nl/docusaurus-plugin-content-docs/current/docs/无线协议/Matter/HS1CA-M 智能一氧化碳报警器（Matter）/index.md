# HS1CA-M Intelligente koolmonoxidemelder (materie)

HS1CA-M is 's werelds eerste Matter-gecertificeerde koolmonoxidemelder**, die gebruik maakt van elektrochemische sensortechnologie en de cross-ecologische interconnectie van Matter over Thread ondersteunt.

## Productoverzicht

- **Producttype**: Intelligent koolmonoxidealarm
- **Communicatieprotocol**: kwestie boven draad
- **Certificeringsnorm**: EN 50291
- **Typisch scenario**: waarschuwing voor CO-lekkage in huizen, appartementen en andere plaatsen

## Technische parameters

| Parameter | Waarde |
|------|-----|
| Sensortype | Elektrochemische CO-sensor |
| Communicatieprotocol | Kwestie over draad (IEEE 802.15.4) |
| Bedrijfsfrequentieband | 2,4 GHz |
| Detectiebereik | 0~1000 ppm |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | DC 3V (CR123A-lithiumbatterij) |
| Levensduur batterij | Ongeveer 3 jaar |
| Levensduur sensor | Typisch 7 jaar |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤95% RH (geen condensatie) |

## Kenmerken

- Zeer nauwkeurige elektrochemische CO-sensor
- Concentratie-gegradueerd alarm (geluids- en lichtmelding)
- 85 dB hoog geluids- en lichtalarm
- Herinnering batterij bijna leeg
- Herinnering voor het verstrijken van de levensduur van de sensor
- Zelftestfunctie
- Multi-ecologische platformonafhankelijke interoperabiliteit

## Certificering

- EN 50291 gecertificeerd
- Matter-certificering
- Werkt met Home Assistant-certificering
