# Gascollector - draadloze gegevensverzamelaar voor gasmeters

## Overzicht

De gascollector is een wM-Bus draadloos gegevensverzamelingsapparaat dat door Hyman is gelanceerd voor het slimme gasveld. Het werkt in de 868 MHz-frequentieband (norm EN 13757) en wordt gebruikt om de meetgegevens van gasmeters op afstand uit te lezen en te uploaden naar het gecentraliseerde meteruitleessysteem. Aangedreven door lithiumbatterijen van industriële kwaliteit met een batterijlevensduur van 6 tot 10 jaar, is hij geschikt voor gecentraliseerde meteruitlezingsscenario's zoals appartementen, woonwijken en commerciële gebouwen.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | wM-Bus (EN 13757) |
| Bedrijfsfrequentieband | 868 MHz (Europese SRD-band) |
| Overdrachtssnelheid | 4,8 kbps / 16,384 kbps / 100 kbps (T/S/R-modus) |
| Communicatieafstand | Ongeveer 100~300 meter binnenshuis, ongeveer 500~1000 meter in de open ruimte |
| Voedingsmethode | Industriële lithiumbatterij (ER18505 / ER26500) |
| Levensduur batterij | 6~10 jaar (doorgaans 1 keer per dag gerapporteerd) |
| Bedrijfstemperatuur | -20°C ~ +60°C |
| Bedrijfsvochtigheid | ≤ 95% RV |
| Beschermingsniveau | IP65 ~ IP68 (afhankelijk van de installatieomgeving) |
| Gegevensversleuteling | AES-128 (optioneel) |

## Kenmerken

-Ondersteuning wM-Bus S/T/R drie werkmodi
- Verzamel automatisch gebruiksgegevens van gasmeters en rapporteer deze regelmatig
- Laag stroomverbruik, diepe slaap, lange batterijduur
- AES-128-gegevensversleuteling om de communicatiebeveiliging te garanderen
-Ondersteuning voor externe firmware-upgrade
- Mechanisme voor het opnieuw verbinden van de verbinding en het opnieuw verzenden van gegevens
- Ondersteunt EN 13757-3 applicatielaagprotocol
- Kan worden gebruikt met HS1-CM-collector / HS1-GW-gateway voor gecentraliseerd uploaden

## Certificeringsnormen

- EN 13757-4 (Draadloze M-Bus fysieke laag)
- EN 13757-3 (M-Bus applicatielaag)
-CE/ROOD
