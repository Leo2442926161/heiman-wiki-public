# Technische parameters

Hieronder volgen de algemene technische parameters van Heimann Wireless M-Bus (wM-Bus)-apparatuur.

## Draadloze parameters

| Parameter | Waarde |
|------|-----|
| Protocolstandaarden | EN 13757-4 (Draadloze M-Bus fysieke laag) |
| Werkfrequentieband | 868 MHz (Europese SRD-frequentieband, 868,0~868,6 MHz) |
| Modulatiemethode | FSK |
| Overdrachtssnelheid | 4,8 kbps / 16,384 kbps / 100 kbps (T-modus / S-modus / R-modus) |
| Netwerktopologie | Ster (instrument → verzamelaar/gateway) |
| Communicatieafstand | Ongeveer 100~300 meter binnenshuis, ongeveer 500~1000 meter in de open ruimte |
| Zendvermogen | ≤ 25 mW (+14 dBm) |

## Werkmodus

| Modus | Beoordeel | Beschrijving |
|------|------|------|
| S-modus | 16.384 kbps | Standaardmodus, laag stroomverbruik, geschikt voor instrumenten op batterijen |
| T-modus | 100 kbps | Hogesnelheidsmodus, geschikt voor scenario's met grote verkeersgegevens |
| R-modus | 4,8 kbps | Smalbandmodus, langeafstandstransmissie |

## Elektriciteits- en stroomverbruik

| Parameter | Waarde |
|------|-----|
| Voedingsmethode | Voeding lithiumbatterij (ER18505 / ER26500 en andere industriële lithiumbatterijen) |
| Levensduur batterij | 6~10 jaar (afhankelijk van de rapportagefrequentie is de typische waarde 1 keer/dag) |
| Stroomverbruik in stand-by | Microampèreniveau, diepe slaap |
| Gevoeligheid ontvangen | Ongeveer. -105 dBm (S-modus) |

## Omgevingsparameters

| Parameter | Waarde |
|------|-----|
| Bedrijfstemperatuur | -20°C ~ +60°C (industriële kwaliteit) |
| Bedrijfsvochtigheid | ≤ 95% RH (niet-condensatie) |
| IP-beschermingsniveau | IP65 ~ IP68 (afhankelijk van het specifieke model, van toepassing op instrumenten voor buitengebruik) |

## Gegevensprotocol

| Laag | Standaard |
|----|------|
| Fysieke laag | EN 13757-4 |
| Datalinklaag | EN 13757-4 (Draadloos M-Bus-frameformaat) |
| Applicatielaag | EN 13757-3 (M-Bus-applicatielaag) |
| Versleutelingsmethode | AES-128 (optionele codering) |
