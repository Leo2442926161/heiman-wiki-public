# Sub-1G (communicatie over lange afstand)

Sub-1G is een draadloze communicatietechnologie die werkt in de frequentieband lager dan 1 GHz. Het heeft de kenmerken van een ultralange transmissieafstand en een extreem laag stroomverbruik. Het is zeer geschikt voor IoT-scenario’s die grootschalige dekking vereisen.

## Kenmerken

| Kenmerken | Beschrijving |
|------|------|
| Bedrijfsfrequentieband | 433 MHz / 868 MHz (EU) / 915 MHz (VS) |
| Transmissieafstand | Tot 1-3 kilometer in open omgeving |
| Penetratievermogen | Beter dan 2,4 GHz, sterke penetratie in muurobstakels |
| Stroomverbruik | Extreem laag, levensduur van de batterij kan 5-10 jaar bedragen |
| Overdrachtssnelheid | Laag (geschikt voor scenario's met kleine datavolumes) |

## Toepassingsscenario's

- **Onafhankelijk rook-/CO-alarm**: onderling verbonden over verdiepingen in het gebouw
- **Brandbeveiligingssysteem voor grote gebouwen**: winkelcentra, hotels, scholen
- **Smart Senior Living Community**: noodoproep tussen units
- **Sub-1G + Wi-Fi Gateway**: het subapparaat communiceert via Sub-1G en de gateway maakt verbinding met de cloud

## Hyman Sub-1G-producten

Hyman heeft vele jaren ervaring met Sub-1G draadloze brandbeveiligingsproducten, waaronder de S1-serie stand-alone rookmelders. Deze apparaten kunnen onderlinge verbindingen tussen apparaten realiseren zonder internet.

> **Opmerking**: Sub-1G wordt meestal gebruikt in de brand- en beveiligingsproductlijnen van Heiman, en de smart home-productlijn is voornamelijk Zigbee/Matter.
