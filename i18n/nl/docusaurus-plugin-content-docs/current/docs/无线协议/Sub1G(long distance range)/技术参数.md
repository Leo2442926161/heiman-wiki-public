# Technische parameters

Hieronder volgen de algemene technische parameters van Hyman Sub-1G-producten uit de langeafstandsserie.

## Draadloze parameters

| Parameter | Waarde |
|------|-----|
| Bedrijfsfrequentieband | 433 MHz / 868 MHz / 915 MHz (afhankelijk van marktgebied) |
| Modulatiemethode | FSK / OOK |
| Overdrachtssnelheid | Ongeveer 10~100 kbps (videosegment en configuratie) |
| Netwerktopologie | Ster / point-to-point / internetnetwerken |
| Communicatieafstand | Ongeveer 1~3 kilometer in open omgeving (videosegment en stroom) |
| Netwerkcapaciteit | Eén groep ondersteunt maximaal 50+ knooppunten |

## Regionale toewijzing van frequentiebanden

| Regio | Gemeenschappelijke frequentiebanden |
|------|----------|
| China | 433 MHz |
| Europa | 868 MHz |
| Noord-Amerika | 915 MHz |

## Elektriciteits- en stroomverbruik

| Parameter | Waarde |
|------|-----|
| Voedingsmethode | Werkt op batterijen (CR123A / AA-lithiumbatterij) |
| Levensduur batterij | Ongeveer 3 tot 5 jaar (afhankelijk van de rapportagefrequentie) |
| Zendvermogen | ≤ 10 mW (+10 dBm), conform SRD-regelgeving |
| Stroomverbruik in stand-by | Microampèreniveau, extreem laag |
| Ontvangstgevoeligheid | Ongeveer -110 dBm ~ -120 dBm |

## Omgevingsparameters

| Parameter | Waarde |
|------|-----|
| Bedrijfstemperatuur | -10°C ~ +55°C |
| Bedrijfsvochtigheid | ≤ 95% RH (niet-condensatie) |
