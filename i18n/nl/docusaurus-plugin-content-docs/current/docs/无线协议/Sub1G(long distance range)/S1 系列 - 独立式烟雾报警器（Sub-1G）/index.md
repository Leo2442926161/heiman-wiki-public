# S1-serie - Standalone rookmelder (Sub-1G)

## Overzicht

De S1-serie is een stand-alone Sub-1G-rookmelder, gelanceerd door Hyman. Het werkt in de frequentieband 433 MHz/868 MHz, maakt gebruik van foto-elektrische rookdetectietechnologie en is geslaagd voor de EN 14604-certificering. Er is geen gateway of internet vereist. Apparaten zijn rechtstreeks met elkaar verbonden via Sub-1G-radiofrequentie. Wanneer een apparaat een alarm activeert, geven alle apparaten in dezelfde groep tegelijkertijd een alarm af. Het is geschikt voor grootschalige brandscenario's zoals gebouwen, winkelcentra en hotels.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | 433 MHz / 868 MHz Sub-1G (afhankelijk van marktgebied) |
| Sensortype | Infrarood foto-elektrisch |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | Werkt op batterijen (CR123A / AA-lithiumbatterij) |
| Levensduur batterij | ≥ 5 jaar |
| Communicatieafstand | Ongeveer 1~3 kilometer in open omgeving |
| Netwerkcapaciteit | Eén groep ondersteunt maximaal 50+ knooppunten |
| Interconnectiemethode | Draadloze radiofrequentie-interconnectie (geen gateway vereist) |
| Bedrijfstemperatuur | -10°C ~ +55°C |
| Bedrijfsvochtigheid | ≤ 95% RV |
| Installatiemethode | Plafondmontage |

## Kenmerken

- Foto-elektrische rookdetectie om rook van langzame brandwonden en open vuur te detecteren
- Draadloos verbindingsalarm: geactiveerd door elk apparaat in hetzelfde netwerk, zullen alle apparaten tegelijkertijd een alarm laten horen en afgaan
- Onafhankelijke werkmodus: geen gateway, geen internet vereist, directe communicatie tussen apparaten
- 85 dB hoog geluids- en lichtalarm
- Herinnering batterij bijna leeg
- Zelftestfunctie (test met één klik)
- Anti-sabotagealarm
-Ondersteuning op afstand dempen
- Ontwerp met ultralaag stroomverbruik, batterijduur van meer dan 5 jaar

## Certificeringsnormen

- EN 14604 (Europese norm voor rookmelders)
-CE/ROOD
- CPR (EU-verordening bouwproducten)

## Toepasselijke accessoires

- S1-RM draadloze repeater (grotere communicatieafstand)
- S1-RC afstandsbediening (test/mute/reset)
