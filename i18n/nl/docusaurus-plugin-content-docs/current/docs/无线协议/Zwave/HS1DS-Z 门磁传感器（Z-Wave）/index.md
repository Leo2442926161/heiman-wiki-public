# HS1DS-Z deursensor (Z-Wave)

HS1DS-Z is een **Z-Wave** deursensorsensor gelanceerd door Hyman. Het detecteert de openings- en sluitstatus van deuren en ramen via magnetische reedschakelaars en is compatibel met de Z-Wave Plus-standaard.

## Productoverzicht

- **Producttype**: Deursensor
- **Communicatieprotocol**: Z-Wave Plus
- **Detectiemethode**: Reed-schakelaar (reed-schakelaar)
- **Typische scenario's**: bewaking van deur- en raamstatus, beveiligingsimplementatie en intelligente koppeling

## Technische parameters

| Parameter | Waarde |
|------|-----|
| Detectiemethode | Reed-schakelaar (reed-schakelaar) |
| Communicatieprotocol | Z-Wave Plus (ITU-T G.9959) |
| Bedrijfsfrequentieband | 868,42 MHz (EU) / 908,42 MHz (VS) |
| Voedingsmethode | CR1632 knoopcelbatterij |
| Levensduur batterij | Ongeveer 2~3 jaar |
| Inbouwvrijheid | ≤10 mm |
| Versleutelingsmethode | S2 geverifieerd / S2 niet-geverifieerd |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤90% RH (geen condensatie) |

## Kenmerken

- Realtime detectie van de status van openen/sluiten van deuren en ramen
- Sabotagealarm (sabotage)
- Herinnering batterij bijna leeg
-Ondersteunt scènekoppeling (deur openen, verlichting aandoen, inschakelen bij het verlaten van huis)
- Eenvoudig te installeren (3M plakband of schroefbevestiging)
- Z-Wave S2 veilige codering
- Ondersteuning van OTA-firmware-upgrade
