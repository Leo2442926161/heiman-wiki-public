# Technische parameters

Hieronder volgen algemene technische specificaties voor Hyman Z-Wave-apparaten.

## Draadloze parameters

| Parameter | Waarde |
|------|-----|
| Protocolstandaarden | Z-Wave Plus (ITU-T G.9959) |
| Werkfrequentieband | 800~900 MHz (afhankelijk van de regio: Europese regelgeving 868,42 MHz, Amerikaanse regelgeving 908,42 MHz) |
| Overdrachtssnelheid | Tot 100 kbps |
| Netwerktopologie | Zelfherstellend mesh-netwerk (bronroutering) |
| Knooppuntcapaciteit | Maximaal 232 knooppunten in één netwerk |
| Communicatieafstand | Ongeveer 30 meter binnenshuis (tussen knooppunten), hele huis Mesh-dekking |
| Versleutelingsmethode | AES-128 (S2 geverifieerd / S2 niet-geverifieerd) |

## Regionale frequentieband

| Regio | Middenfrequentie |
|------|----------|
| Europa | 868,42 MHz |
| Noord-Amerika | 908,42 MHz |
| Andere gebieden | Onderworpen aan lokale regelgeving |

## Elektriciteits- en stroomverbruik

| Parameter | Waarde |
|------|-----|
| Voedingsmethode | Batterijvoeding (CR123A/AA) of netvoeding |
| Levensduur batterij | Ongeveer 2~3 jaar voor sensoren |
| Zendvermogen | ≤ 1 mW (conform SRD-regelgeving) |
| Gevoeligheid ontvangen | Ongeveer -100 dBm |

## Omgevingsparameters

| Parameter | Waarde |
|------|-----|
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤ 90% RH (niet-condensatie) |

## Netwerkkenmerken

| Kenmerken | Beschrijving |
|------|------|
| SmartStart | Ondersteuning (700-serie), scan de QR-code om lid te worden van het netwerk |
| S2-beveiliging | Ondersteunt drie beveiligingsniveaus: niet-versleuteld, S2 niet-geverifieerd, S2-geverifieerd |
| Bevat CC | Beveiliging CC, Beveiliging 2 CC, Firmware-update CC, Basis CC, Batterij CC, Notificatie CC en meer |
