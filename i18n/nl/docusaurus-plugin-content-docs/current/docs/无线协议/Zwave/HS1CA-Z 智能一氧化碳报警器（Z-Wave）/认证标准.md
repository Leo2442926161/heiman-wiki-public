# Certificatienormen

HS1CA-Z heeft de volgende certificeringen behaald:

- **EN 50291** — Europese norm voor koolmonoxidemelders, zie [EN Europese norm](../../../../Certification Standards/EN European Standard.md)
- **Z-Wave S2-certificering** — Ondersteunt het S2-beveiligingsframework, zie [Overige certificeringen](../../../Certification Standards/Other Certifications.md)
- **CE** — Conformiteitsverklaring van de Europese Unie
