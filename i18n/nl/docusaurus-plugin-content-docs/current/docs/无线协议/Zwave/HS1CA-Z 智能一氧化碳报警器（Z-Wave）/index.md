# HS1CA-Z Intelligente koolmonoxidemelder (Z-Wave)

HS1CA-Z is een **Z-Wave** slimme koolmonoxidemelder gelanceerd door Hyman. Het maakt gebruik van elektrochemische sensortechnologie en is compatibel met de Z-Wave Plus-standaard.

## Productoverzicht

- **Producttype**: Intelligent koolmonoxidealarm
- **Communicatieprotocol**: Z-Wave Plus
- **Certificeringsnorm**: EN 50291
- **Typisch scenario**: waarschuwing voor CO-lekkage in huizen, appartementen en andere plaatsen

## Technische parameters

| Parameter | Waarde |
|------|-----|
| Sensortype | Elektrochemische CO-sensor |
| Communicatieprotocol | Z-Wave Plus (ITU-T G.9959) |
| Bedrijfsfrequentieband | 868,42 MHz (EU) / 908,42 MHz (VS) |
| Detectiebereik | 0~1000 ppm |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | Werkt op batterijen |
| Levensduur sensor | Typisch 7 jaar |
| Versleutelingsmethode | S2 geverifieerd / S2 niet-geverifieerd |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤90% RH (geen condensatie) |

## Kenmerken

- Zeer nauwkeurige elektrochemische CO-sensor
- Concentratieclassificatie-alarm
- 85 dB hoog geluids- en lichtalarm
- Herinnering batterij bijna leeg
- Herinnering voor het verstrijken van de levensduur van de sensor
- Zelftestfunctie
- Z-Wave S2 veilige codering
- Ondersteuning van OTA-firmware-upgrade

## Certificering

- EN 50291 gecertificeerd
- Z-Wave Plus-gecertificeerd
- S2-veiligheidscertificering
