#Ondersteunde modellijst

Hyman Z-Wave-producten zijn voornamelijk gericht op de Europese en Amerikaanse smart home-markten en zijn compatibel met de Z-Wave Plus-standaard.

## Beveiligingsalarmcategorie

| Model | Productnaam | Beschrijving |
|------|----------|------|
| HS1SA-Z | Slimme rookmelder (Z-Wave) | Z-Wave versie rookmelder |
| HS1CA-Z | Intelligent koolmonoxidealarm (Z-Wave) | Z-Wave versie CO-alarm |
| HS1CG-Z | Brandbaar gasdetector (Z-Wave) | Z-Wave-versie gasdetector |

## Sensorklasse

| Model | Productnaam | Beschrijving |
|------|----------|------|
| HS1DS-Z | Deursensor (Z-Wave) | Z-Wave versie deur- en raamsensor |
| HS1MS-Z | Infraroodsensor van het menselijk lichaam (Z-Wave) | Z-Wave-versie van detectie van het menselijk lichaam |
| HS1WL-Z | Waterdompeldetector (Z-Wave) | Z-Wave-versie waterlekdetectie |

## Compatibiliteit

| Project | Beschrijving |
|------|------|
| Standaard | Z-Wave Plus (chips uit de 500/700-serie) |
| Compatibele hub | Homey, Hubitat, SmartThings, Vera, Fibaro, Home Assistant (Z-Wave JS) |
| S2-beveiliging | Ondersteuning van het S2-beveiligingsprotocol (niet-versleuteld / S2 niet-geverifieerd / S2-geverifieerd) |
| SmartStart | Ondersteuning (chipmodellen uit de 700-serie) |
| OTA-upgrade | Ondersteuning FW-update CC |
