# HS1SA-Z Slimme Rookmelder (Z-Wave)

HS1SA-Z is een **Z-Wave** slimme rookmelder gelanceerd door Hyman. Het maakt gebruik van foto-elektrische rookdetectietechnologie, is compatibel met de Z-Wave Plus-standaard en is geschikt voor de Europese en Amerikaanse smart home-markten.

## Productoverzicht

- **Producttype**: Slimme rookmelder
- **Communicatieprotocol**: Z-Wave Plus
- **Certificeringsnorm**: EN 14604
- **Typisch scenario**: Brandwaarschuwing in huizen, appartementen en andere plaatsen

## Technische parameters

| Parameter | Waarde |
|------|-----|
| Sensortype | Infrarood foto-elektrisch |
| Communicatieprotocol | Z-Wave Plus (ITU-T G.9959) |
| Bedrijfsfrequentieband | 868,42 MHz (EU) / 908,42 MHz (VS) |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | CR123A lithiumbatterij |
| Levensduur batterij | Ongeveer 2~3 jaar |
| Versleutelingsmethode | S2 geverifieerd / S2 niet-geverifieerd |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤90% RH (geen condensatie) |

## Kenmerken

- Foto-elektrische rookdetectie om langzame verbranding en open vuurrook te detecteren
- 85 dB hoog geluids- en lichtalarm
- Herinnering batterij bijna leeg
- Zelftestfunctie
- Anti-sabotagealarm
- Z-Wave S2 veilige codering
- Ondersteuning van OTA-firmware-upgrade

## Certificering

- EN 14604-certificering
- Z-Wave Plus-gecertificeerd
- S2-veiligheidscertificering
