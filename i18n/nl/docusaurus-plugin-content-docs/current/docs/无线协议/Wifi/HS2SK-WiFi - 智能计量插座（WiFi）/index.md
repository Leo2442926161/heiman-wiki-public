#HS2SK-WiFi - Slimme meteraansluiting (WiFi)

## Overzicht

HS2SK-WiFi is een slimme WiFi-meteraansluiting gelanceerd door Heiman. Het ondersteunt een directe 2,4GHz 802.11 b/g/n-verbinding met thuisrouters en kan monitoring op afstand en energieverbruik realiseren zonder dat er extra gateways nodig zijn. Geschikt voor intelligent beheer van elektrische apparaten in woningen, kantoren en andere scenario's.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | IEEE 802.11 b/g/n (2,4 GHz) |
| Maximale belasting | 16A |
| Maximaal vermogen | 3680W (230V) / 1800W (110V) |
| Meetnauwkeurigheid | ±2% |
| Voedingsmethode | Netvoeding |
| Stroomverbruik in stand-by | ≤ 0,5W |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤ 90% RV |
| Materiaal schaal | PC V0 vlamvertragend |
| Jack-specificaties | Europese norm / Amerikaanse norm (afhankelijk van marktversie) |
