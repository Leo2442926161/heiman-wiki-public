# Technische parameters

Hieronder volgen algemene technische specificaties voor Heimann WiFi-apparaten.

## Draadloze parameters

| Parameter | Waarde |
|------|-----|
| Protocolstandaarden | IEEE 802.11 b/g/n |
| Bedrijfsfrequentieband | 2,4 GHz (niet compatibel met 5 GHz) |
| Overdrachtssnelheid | Tot 150 Mbps (802.11n) |
| Netwerktopologie | Stertopologie (rechtstreeks verbonden met de router) |
| Communicatieafstand | Ongeveer 30~50 meter binnenshuis (afhankelijk van de prestaties van de router) |
| Versleutelingsmethode | WPA2-PSK / WPA3 |

## Netwerkdistributiemethode

| Werkwijze | Beschrijving |
|------|------|
| SmartConfig | Mobiele app Netwerkconfiguratie met één klik (traditionele WiFi-apparatuur) |
| Bluetooth-ondersteunde inrichting | WiFi-inloggegevens configureren via tijdelijke Bluetooth-kanalen (sommige nieuwe modellen) |
| AP-modus | Schakel de hotspot op het apparaat in en configureer deze nadat de telefoon is aangesloten |

## Elektriciteits- en stroomverbruik

| Parameter | Waarde |
|------|-----|
| Voedingsmethode | Netvoeding (type stopcontact/gatewaytype) of batterijvoeding (type alarm) |
| Stroomverbruikniveau | Hoger dan Zigbee/Thread vanwege de noodzaak om de TCP/IP-verbinding te behouden |
| Stroomverbruik in stand-by | Type stopcontact ongeveer 0,5W ~ 1W |

## Omgevingsparameters

| Parameter | Waarde |
|------|-----|
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤ 90% RH (niet-condensatie) |
