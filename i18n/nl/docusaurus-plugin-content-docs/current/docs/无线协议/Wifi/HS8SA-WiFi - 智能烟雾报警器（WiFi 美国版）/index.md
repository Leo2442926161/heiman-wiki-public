# HS8SA-WiFi - Slimme rookmelder (WiFi Amerikaanse versie)

## Overzicht

HS8SA-WiFi is een slimme WiFi-rookmelder die door Hyman voor de Amerikaanse markt is gelanceerd. Het maakt gebruik van foto-elektrische rookdetectietechnologie, ondersteunt WiFi-netwerken en onafhankelijk werkende dubbele modi, en is geslaagd voor de UL 217-certificering. Geschikt voor brandveiligheid in huizen, appartementen en andere woonwijken.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Communicatieprotocol | IEEE 802.11 b/g/n (2,4 GHz) |
| Sensortype | Infrarood foto-elektrisch |
| Alarmgeluidsdruk | ≥85 dB (3 meter) |
| Voedingsmethode | Werkt op batterijen |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤ 90% RV |
| Installatiemethode | Plafondmontage |

## Kenmerken

- Foto-elektrische rookdetectie om rook van langzame brandwonden en open vuur te detecteren
- WiFi-netwerkalarm op afstand
- Onafhankelijke bedrijfsmodus (lokaal geluids- en lichtalarm kan nog steeds optreden wanneer het netwerk is losgekoppeld)
- 85 dB hoog geluids- en lichtalarm
- Herinnering batterij bijna leeg
- Zelftestfunctie (test met één klik)
- Anti-sabotagealarm
-Ondersteuning op afstand dempen

## Certificeringsnormen

- UL 217 (Amerikaanse norm voor rookmelders)
-FCC
- NFPA 72-naleving
