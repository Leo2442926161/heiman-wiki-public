# Certificatienormen

HS8SA-WiFi heeft de volgende certificeringen doorstaan:

- **UL 217** — Amerikaanse veiligheidsnorm voor rookmelders, zie [UL Underwriters Laboratories](../../../../certification standards/ULUnderwriters Laboratories.md)
- **FCC** — Certificering door de Federal Communications Commission
- **CE** — Conformiteitsverklaring van de Europese Unie
