#HS6GW - Smart Home-gateway

## Overzicht

HS6GW is een slimme thuisgateway gelanceerd door Heiman, ondersteunt WiFi + Ethernet-uplink, ingebouwde Zigbee-coördinator en kan maximaal 128 Zigbee-subapparaten beheren. Het realiseren van externe toegang en clouddiensten door verbinding te maken met internet is de kern van het Haiman smart home-systeem.

## Technische specificaties

| Parameter | Waarde |
|------|-----|
| Uplink-communicatie | IEEE 802.11 b/g/n (2,4 GHz WiFi) + 10/100M Ethernet |
| Downlink-communicatie | Zigbee 3.0 (coördinatormodus) |
| Maximaal aantal subapparaten | 128 |
| Voedingsmethode | DC 5V / 1A (Micro-USB) |
| Stroomverbruik in stand-by | ≤ 1W |
| Bedrijfstemperatuur | -10°C ~ +50°C |
| Bedrijfsvochtigheid | ≤ 90% RV |
| Indicatielampje | WiFi-status/voeding/Zigbee-netwerkstatus |
| Methode opnieuw instellen | Knop reset + lang indrukken om fabrieksinstellingen te herstellen |

## Kenmerken

- Dubbele uplinks: bekabelde WiFi- of Ethernet-verbinding, flexibele keuze afhankelijk van de netwerkomgeving
-Ondersteuning voor toegang op afstand: controleer de status van het apparaat altijd en overal via de cloud-app
- Lokale scènekoppeling: ondersteunt geautomatiseerde bediening van apparatuur binnen het LAN (nog steeds beschikbaar wanneer de verbinding is verbroken)
- OTA-firmware-upgrade
- Ondersteuning van het Tuya / Smart Life-platform
- Beheer tot 128 Zigbee-subapparaten
-Ondersteuning van zelfherstel van het Zigbee-netwerk en routeringsoptimalisatie
