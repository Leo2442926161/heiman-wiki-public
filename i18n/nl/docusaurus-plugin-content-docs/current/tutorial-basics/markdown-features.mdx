# Markdown-functies

Heiman technische documentatie ondersteunt volledig Markdown-syntaxis en uitgebreide functies voor het schrijven van gestructureerde producttechnische documenten.

## Codevoorbeelden

API-aanroepvoorbeeld tijdens productontwikkeling en integratie:

```jsx title="Koppeling Rookmelder API"
const heimanClient = new HeimanClient({
  apiKey: 'your-api-key',
  deviceId: 'HS1SA-xxxxx',
});

// Apparaatstatus ophalen
const status = await heimanClient.getDeviceStatus();
console.log(status);
```

## Waarschuwingskaders

Gebruik waarschuwingskaders om belangrijke informatie te benadrukken:

:::tip

Heiman rookmelder HS1SA is EN14604-gecertificeerd

:::

:::caution

Zorg bij installatie van de gasmelder dat de bedrading correct is volgens de installatiehandleiding

:::

:::danger

Bij een gaslek mag u geen elektrische apparaten inschakelen. Open onmiddellijk ramen, ventileer en evacueer

:::
