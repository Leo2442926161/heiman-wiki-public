# Implantação de Soluções

A Heiman Technology fornece orientação completa para implantação de soluções de segurança inteligente.

## Processo de Integração

Os produtos Heiman suportam múltiplos métodos de implantação, incluindo implantação local independente e integração com plataforma em nuvem.

### Implantação Local

```bash
# Configurar Gateway de Incêndio Inteligente Heiman
heiman-cli configure --gateway HS1GW --mode local

# Registrar dispositivo detector
heiman-cli register --device HS1SA --gateway HS1GW
```

### Integração com Plataforma em Nuvem

Os produtos Heiman são integrados com as principais plataformas IoT:

- China Telecom Tianyi
- China Mobile OneNET
- Huawei HiLink
- Alibaba Cloud IoT
- Tuya Smart

## Suporte ao Protocolo Matter

Os produtos da série Matter Heiman são compatíveis com:

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

Alcançando interconectividade entre marcas e ecossistemas.

## Verificação de Implantação

Após a implantação, use o APP Heiman Smart Fire ou a plataforma de gerenciamento para verificação funcional e monitoramento em tempo real.
