# Sensor de temperatura de incêndio

O sensor de temperatura de incêndio (detector de calor) emite um alarme de incêndio ao detectar aumentos anormais na temperatura ambiente. É adequado para ambientes como cozinhas e garagens onde os sensores de fumaça são propensos a alarmes falsos.

## Princípio de funcionamento

O sensor de temperatura usa um elemento térmico para detectar mudanças de temperatura:

- **Alarme de temperatura fixa**: dispara um alarme quando a temperatura excede um limite definido (geralmente 57°C / 135°F)
- **Alarme de diferença de temperatura**: aciona um alarme quando a taxa de aumento de temperatura excede o valor definido (como 8,3°C/min)

## Escopo de aplicação

- Cozinha (sensores de fumaça são propensos a alarmes falsos devido ao cozimento nesta cena)
- Garagem/oficina (gases de escape e poeira podem interferir nos sensores de fumaça)
- Sala das caldeiras/sala dos equipamentos de aquecimento
- Banheiro/Sauna

## Funções principais

- Detecção de aumento anormal de temperatura
- Alarme anti-adulteração
- Lembrete de bateria fraca

> **Nota**: O sensor de temperatura e o sensor de fumaça são utilizados de forma complementar. O sensor de fumaça responde mais rápido e o sensor de temperatura tem capacidade anti-interferência mais forte.
