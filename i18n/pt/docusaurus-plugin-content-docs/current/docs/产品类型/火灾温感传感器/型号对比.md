# Comparação de modelos

## Comparação de vários modelos

| Modelo | Protocolo | Princípio de detecção | Fonte de alimentação | Recursos | Sugestões de jogos |
|------|------|----------|------|------|----------|
| Sensor de temperatura de incêndio | Zigbee/Sub-1G | Termistor (temperatura fixa + temperatura diferencial) | Bateria | Interferência antifumaça | Especial para cozinha/garagem, alarme para aumento anormal de temperatura, válvula solenóide de ligação para desligar o gás |

## Jogabilidade de cena recomendada

| Cenários | Modelos recomendados | Soluções de ligação |
|------|----------|----------|
| Segurança na cozinha | Sensor de temperatura | Aumento anormal de temperatura → Desligue a válvula de gás → Alarme sonoro e luminoso → Notificação push do celular para evitar alarmes falsos de fumaça de cozinha |
| Garagem/caldeira | Sensor de temperatura | Monitorar a temperatura na sala de equipamentos para evitar combustão espontânea e ligar o exaustor para resfriamento |
| Complementar com sensor de fumo | Sensor de temperatura + fumaça | O sensor de fumo cobre toda a casa, o sensor de temperatura cobre a cozinha/garagem, o alarme complementar é mais fiável |

## Sugestões de seleção

- **Apenas cozinha**: Sensor de temperatura (para evitar falsos alarmes de fumaça de cozimento)
**Funciona com sensor de fumaça**: sensor de fumaça cobre quarto/sala, sensor de temperatura cobre cozinha/garagem
