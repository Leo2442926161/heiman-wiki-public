# Guia de instalação

## Requisitos de local de instalação

- **Suporte de teto**, localizado no centro da sala
- **Cenários de melhor uso**: cozinhas, garagens, salas de caldeiras, banheiros - locais onde sensores de fumaça são propensos a alarmes falsos
- Pelo menos **30 cm** da parede
- Evite instalar perto de aberturas de ventilação e saídas de ar condicionado

## Etapas de instalação

1. Selecione o local de instalação (cozinha recomendada ou teto de garagem)
2. Fixe a base de montagem no teto
3. Insira a bateria e o sensor ligará automaticamente.
4. Gire o corpo principal e encaixe-o na base
5. Autoteste: Pressione o botão de teste, a luz indicadora acenderá e um som será emitido.
6. Emparelhamento via gateway (versão Zigbee)

## Notas

- O sensor de temperatura** não substitui o sensor de fumaça**, mas sim um complemento
- O sensor de fumaça responde mais rápido e o sensor de temperatura possui anti-interferência mais forte
- Recomenda-se instalar sensores de fumaça e de temperatura na mesma sala
- Teste uma vez por mês e substitua a bateria a tempo quando a bateria estiver fraca
