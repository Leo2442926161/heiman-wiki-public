# Comparação de modelos

## Comparação de vários modelos

| Modelo | Protocolo | Vídeo | Fonte de alimentação | Recursos | Sugestões de jogabilidade |
|------|------|------|------|------|----------|
| Campainha inteligente | Zigbee + Wi-Fi | 1080P opcional | Bateria/rede elétrica | Campainha + segurança dois em um | Notificação de toque de visitante no celular, intercomunicador visual remoto, gravação de detecção de movimento |

## Jogabilidade de cena recomendada

| Cenários | Modelos recomendados | Soluções de ligação |
|------|----------|----------|
| Gestão de entregas expressas/takeaway | Campainha inteligente | Tocar a campainha → notificação do celular → confirmação de intercomunicação remota → desbloqueio remoto (bloqueio inteligente vinculado) |
| Monitoramento de Segurança | Campainha inteligente | Detecção de presença de porta → Vídeo + alarme de pressão, sensor de porta vinculado/iluminação |
| Articulação de visitantes | Campainha inteligente | Toque a campainha → Tela interna/alto-falante → Acenda as luzes → Desbloqueie a ligação |
