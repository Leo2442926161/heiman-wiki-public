# Guia de instalação

## Requisitos de local de instalação

- **Instalado na altura da respiração** - aprox. **1,5 m** do solo
- Perto da área do quarto para garantir que o alarme possa ser ouvido durante o sono
- **Não instale em cantos**, mantenha a circulação de ar
- Mantenha longe de aberturas de ventilação, janelas, ventiladores e outros locais que afetem o fluxo de ar

## Etapas de instalação

1. Determine o local de instalação (corredor do quarto ou sala de estar recomendado)
2. Fixe a base de montagem na parede (parafusos ou fita adesiva 3M)
3. Insira a bateria e o sensor ligará automaticamente.
4. Instale o corpo principal na base
5. Autoteste: Pressione o botão de teste, a luz do alarme piscará e o alarme soará.
6. Emparelhamento via gateway

## Notas

- A vida útil típica do sensor é de **7 anos**, após os quais todo o dispositivo precisa ser substituído
- Teste a funcionalidade uma vez por mês
- Não bloqueie a ventilação do sensor
- Se houver alarme, abra primeiro as janelas para ventilação e depois verifique se há vazamento de fontes de CO.
