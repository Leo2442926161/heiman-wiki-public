# Guia de instalação

## Requisitos de local de instalação

- **instalação no teto**, sensor verticalmente para baixo
- Altura de instalação **2,5 ~ 3,5 m**
- O raio de detecção é de cerca de **3 ~ 5 m**
- **Evitar**: Existem obstáculos metálicos na frente do sensor (o metal bloqueará as ondas milimétricas)
- Manter afastado de grandes tubos metálicos, unidades internas de ar condicionado, etc.

## Etapas de instalação

1. Selecione o local de instalação no teto (o centro da sala é melhor)
2. Faça furos no teto e fixe a base com parafusos de expansão
3. Conecte a fiação (se for necessária fonte de alimentação) ou instale a bateria
4. Gire o corpo do sensor e encaixe-o na base
5. Depois que a energia é ligada, a luz indicadora acende e o sensor inicia o autoteste.
6. Entre no emparelhamento do APP e adicione o dispositivo
7. Teste: Entre na área de detecção e sente-se em silêncio. O sensor deve permanecer no estado “ocupado”.

## Notas

- Ondas milimétricas podem penetrar placas finas (gesso, madeira), mas serão bloqueadas por metal
-Paredes grossas e tetos metálicos reduzirão o alcance de detecção
- Recomendado para uso com sensor PIR para obter cobertura total da cena
