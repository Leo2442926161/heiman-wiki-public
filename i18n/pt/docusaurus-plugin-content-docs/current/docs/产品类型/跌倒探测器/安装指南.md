# Guia de instalação

## Método de uso

**Prendedor de cinto**: prendedor no cinto ou na cintura
- **Lanyard**: pendure no pescoço e pendure no peito

## Instruções de uso

1. Instale a bateria
2. Escolha o método de uso (clipe para cinto ou cordão)
3. Use-o em seu corpo
4. Emparelhamento via gateway
5. Teste de simulação:
   - Fique de pé e agache-se rapidamente ou deite-se no chão
   - O sensor deve detectar uma queda e disparar o alarme em segundos
   - Pressione o botão de socorro para testar o alarme manual

## Notas

- Recomendado para ser usado o dia todo (pode ser removido para carregar/substituir a bateria ao tomar banho)
- O algoritmo de IA é usado para distinguir quedas de atividades diárias (sentar, abaixar-se), com baixa taxa de alarmes falsos
- Alarme automático após detectar uma queda, ou você pode pressionar o botão manualmente para pedir ajuda
- Pode ser vinculado ao botão de emergência para obter dupla proteção
