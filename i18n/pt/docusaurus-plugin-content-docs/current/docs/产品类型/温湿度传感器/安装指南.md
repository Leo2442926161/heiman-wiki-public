# Guia de instalação

## Requisitos de local de instalação

- Instalado a uma altura de **1,2 ~ 1,5 m** (representando o nível médio do ambiente)
- **Manter afastado**: luz solar direta, portas e janelas, saídas de ar condicionado, radiadores
- **Manter afastado**: umidificadores, fogões de cozinha e outras fontes locais de mudanças de temperatura e umidade
- Escolha ambientes de uso frequente (quarto, sala, quarto de bebê)

## Etapas de instalação

1. Selecione o local de instalação (parede ou mesa)
2. Montagem na parede: fixe a base com parafusos ou cola 3M
3. Instale a bateria CR2 (observe a polaridade)
4. O sensor liga automaticamente e a tela (se houver) exibe a temperatura e a umidade.
5. Emparelhamento via gateway
6. Verifique os dados do APP para confirmar se estão normais.

## Notas

- Ao instalar contra a parede, certifique-se de que a ventilação do sensor não esteja bloqueada
-A vida útil da bateria é de cerca de 2 a 3 anos
- Os dados de temperatura e umidade em diferentes salas podem ser coletados através de vários sensores
- Pode ser usado para condicionadores de ar, desumidificadores, umidificadores e outros equipamentos vinculados
