# Guia de instalação

## Requisitos de local de instalação

- **Instale no centro da casa** para garantir que possa ser ouvido em todos os cômodos
- Corredor, teto da sala ou parede alta
- **2m ou mais** do chão (evite ser tocado por pessoas)
- Quando usado em conjunto com sensores de fumaça/CO, instale nos corredores dos quartos

## Etapas de instalação

1. Selecione o local de instalação (recomenda-se corredor ou sala de estar)
2. Use parafusos para fixar a base de montagem na parede ou teto
3. Conecte a fonte de alimentação (versão de rede) ou insira a bateria (versão de bateria)
4. Instale o corpo do alarme na base
5. Emparelhamento via gateway
6. Teste de ligação: acione qualquer sensor, o alarme sonoro e luminoso deve responder

## Notas

- Instale em uma altura fora do alcance das crianças
- Teste regularmente se o volume e a iluminação do alarme estão normais
- Para a versão com bateria, tome cuidado para substituí-la quando a bateria estiver fraca.
- Pode definir regras de ligação com outros sensores
