# Guia de instalação

## Requisitos de local de instalação

- **Instalação de montagem na parede**: Banheiro, cabeceira no quarto, próximo ao sofá na sala - em qualquer lugar ao seu alcance
- **Use-o em qualquer lugar**: coloque-o no bolso ou pendure-o no chaveiro
- **Para idosos**: Instalado na cabeceira do banheiro e do quarto, a altura é de cerca de 1,2 m

## Etapas de instalação

1. Determine o local de instalação ou se deseja carregá-lo com você
2. Montagem na parede: Use parafusos ou cola 3M para fixar a base de montagem
3. Instale a bateria
4. Coloque o botão de emergência na base ou leve-o consigo
5. Emparelhamento via gateway
6. Teste: pressione longamente o botão para acionar o alarme e confirme se o APP recebe a notificação

## Notas

Adote o design de toque anti-acidental de toque longo **, que requer pressão contínua por 1-2 segundos para acionar
- Recomenda-se vincular ao alarme sonoro e luminoso
- Teste a funcionalidade regularmente
- Para idosos, pode ser utilizado com detector de quedas para dupla proteção.
