# Comparação de modelos

## Comparação de vários modelos

| Modelo | Protocolo | Método de gatilho | Fonte de alimentação | Recursos | Sugestões de jogos |
|------|------|----------|------|------|----------|
| Botão de emergência | Zigbee | Gatilho de pressão longa | Bateria | Portátil + montado na parede de dupla finalidade | Os idosos podem carregá-lo consigo, ajuda com um clique em emergências, vinculado ao push do celular + alarme sonoro e luminoso |

## Jogabilidade de cena recomendada

| Cenários | Modelos recomendados | Soluções de ligação |
|------|----------|----------|
| Assistência domiciliar para idosos | Botão de emergência | Desconforto repentino para idosos → Toque longo → Pressione para os celulares de todos os familiares → Alarme sonoro e luminoso vinculado → Notificação de propriedade |
| Segurança do quarto infantil | Botão de emergência | Se uma criança enfrentar uma emergência sozinha em casa → Pressione o botão para acionar o alarme no celular dos pais |
| Balcão de loja/checkout | Botão de pânico | Instalação oculta, acionando alarme silencioso em caso de roubo, vinculado a vídeo de vigilância + notificação silenciosa de segurança |
| WC/banheiro | Botão de emergência | Se o idoso escorregar e cair no chuveiro, o botão de emergência à prova d'água está à mão para facilitar o socorro |
