# Guia de instalação

## Requisitos de local de instalação

Altura de montagem na parede: **1,8 ~ 2,4 m**
- Instalação no teto: Instalado no teto no centro da sala
- **Evitar**: Luz solar direta, saídas de aquecimento/ar condicionado, perto de fontes de calor
- Evite obstruções (móveis, divisórias de vidro) dentro do alcance de detecção

## Etapas de instalação

1. Selecione o local de instalação (montagem na parede ou no teto)
2. Prenda a base de montagem com parafusos
3. Insira a bateria, a luz indicadora pisca
4. Instale o corpo do sensor na base
5. Ajuste o ângulo de detecção (dentro da faixa horizontal de 110°)
6. Teste de caminhada: Depois de caminhar pela área de detecção, a luz indicadora deve acender
7. Emparelhamento via gateway

## Notas

- Os sensores infravermelhos são sensíveis às mudanças de temperatura e a distância de detecção pode ser reduzida em ambientes de alta temperatura no verão.
- Evite ficar de frente para as janelas (mudanças na temperatura externa causam alarmes falsos)
- Vida útil da bateria aproximadamente 3 anos (Zigbee) / 1 ano (Matter)
