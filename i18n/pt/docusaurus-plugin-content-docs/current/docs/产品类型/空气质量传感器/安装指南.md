# Guia de instalação

## Requisitos de local de instalação

- Instalado a **1,2 ~ 1,8 m** de altura (zona de respiração)
- Escolha um local que represente a qualidade média do ar na sala
- **Mantenha distância**: janelas, portas, aberturas de ventilação, saídas de ar condicionado
- **Fique longe**: fumaça de cozinha, umidificadores, aromaterapia e outras fontes locais de poluição

## Etapas de instalação

1. Selecione o local de instalação (sala, quarto ou parede do escritório)
2. Prenda a base de montagem com parafusos
3. Conecte à alimentação (alimentada por USB ou adaptador)
4. O sensor está ligado e o display (se houver) exibe dados após o autoteste.
5. Emparelhamento via gateway
6. Confirme se os dados estão normais no APP

## Notas

- Certifique-se de que a entrada e saída de ar do sensor não estejam bloqueadas
- Limpe a caixa do sensor regularmente para evitar entupimento de poeira
- Coloque-o no quarto das crianças para monitorar o nível de CO₂ e solicitar que você abra a janela para ventilação
- Recomendado para ser combinado com sistema de ar fresco ou purificador de ar
