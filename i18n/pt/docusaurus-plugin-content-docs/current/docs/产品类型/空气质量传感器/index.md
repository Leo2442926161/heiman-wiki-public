# sensor de qualidade do ar

Sensores de qualidade do ar (Monitor de Qualidade do Ar) são usados para detectar vários poluentes no ar interno para ajudar a manter um ambiente de vida saudável.

## Parâmetros de detecção

Os sensores de qualidade do ar Hyman detectam os seguintes indicadores:

| Parâmetros | Descrição | Padrões de Saúde |
|------|------|----------|
| PM2,5 | Partículas finas | ≤ 35 µg/m³ (média de 24 horas) |
| PM10 | Partículas respiráveis ​​| ≤ 50 µg/m³ (média de 24 horas) |
| CO₂ | Dióxido de carbono | ≤ 1000 ppm |
| TVOC | Compostos orgânicos voláteis totais | ≤ 0,6 mg/m³ |
| Temperatura | Temperatura ambiente | 18-26°C |
| Umidade | Umidade relativa | 40-60% UR |

## Funções principais

- Detecção multiparâmetro da qualidade do ar
- Visor colorido (dependendo do modelo)
- Alarme sonoro e luminoso por ultrapassagem do padrão
- Registros de dados históricos
- Purificador de ar / ar fresco vinculado

## Cenários aplicáveis

- Quarto: monitore os níveis de CO₂ e solicite a abertura das janelas para ventilação
- Quarto infantil: proteja a saúde respiratória das crianças
- Escritório: Aumente a produtividade e o conforto
- Casas recentemente reformadas: monitore a liberação de formaldeído/TVOC
