# Guia de instalação

## Requisitos de local de instalação

- **Localização central em toda a casa** garante cobertura de sinal Zigbee para todos os dispositivos
- Tente evitar colocá-lo em um gabinete todo de metal (o metal bloqueará os sinais sem fio)
- **Ethernet primeiro**: Recomenda-se usar uma conexão de rede com fio, que é mais estável
- Não mais que 20-30 m de distância de outros dispositivos Zigbee (dentro de casa)

## Etapas de instalação

1. Selecione o local de instalação (armário de TV na sala, próximo à caixa de corrente fraca, etc.)
2. Conecte o adaptador de energia
3. Conecte um cabo Ethernet (recomendado) ou conecte-se ao Wi-Fi
4. Aguarde até que a luz indicadora fique verde/acesa
5. Abra o APP Haiman e escaneie o código QR na parte inferior do gateway para adicionar
6. Após a conclusão da adição, você pode emparelhar o subdispositivo

## Notas

- O gateway é o núcleo da casa inteligente. Recomenda-se usar a fonte de alimentação reserva do UPS.
- A conexão Ethernet é mais estável que o Wi-Fi e é recomendável usá-la primeiro
- Se o sinal do dispositivo estiver fraco, adicione um dispositivo de relé Zigbee (como uma tomada inteligente)
- O número máximo de subdispositivos suportados por diferentes modelos é diferente e deve ser selecionado com base na escala.
