# Portal Inteligente

Smart Gateway é o centro central do sistema doméstico inteligente Haiman, responsável por conectar vários sensores Zigbee e realizar o controle de ligação entre dispositivos.

## Série de produtos

| Série | Métodos de comunicação | Recursos |
|------|----------|------|
| Série HS1GW | Zigbee + Wi-Fi/Ethernet | Gateway Básico |
| Série HS3GW | Zigbee + Wi-Fi/Ethernet | Aprimorado |
| Série HS5GW | Zigbee + Wi-Fi/Ethernet | Gateway de alto desempenho |
| Série HS6GW | Zigbee + Wi-Fi / Ethernet + Thread | O modelo mais recente suporta Zigbee e Matter over Thread |

## Funções principais

- Coordenador de Rede Zigbee
- Suporte Wi-Fi / Ethernet para conectar-se à Internet
- Descoberta e emparelhamento automático de dispositivos
- Ligação de cena local (alguns modelos)
- Acesso remoto (via Heiman Cloud Platform)
- Suporta até 128 subdispositivos (dependendo do modelo)

## Cenários aplicáveis

- O principal ponto de conexão da inteligência de toda a casa Heiman
- Coopere com o Haiman APP para realizar o monitoramento remoto
- Gestão local centralizada de pequenos e médios projetos comerciais

## Diferenças de outros gateways

Haiman Gateway é a entrada para o ecossistema oficial. Para usuários que desejam utilizar uma plataforma de terceiros, é recomendado utilizar o coordenador Zigbee universal com Zigbee2MQTT/ZHA.
