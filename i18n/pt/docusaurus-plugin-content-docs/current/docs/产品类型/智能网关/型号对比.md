# Comparação de modelos

## Comparação de vários modelos

| Modelo | Limite máximo do subdispositivo Zigbee | Método de rede | Recursos | Sugestões de jogos |
|------|------------------|----------|------|----------|
| [HS6GW](../../Protocolo sem fio/Zigbee/HS6GW - Gateway inteligente (Zigbee)) | 128 | Wi-Fi + Ethernet | O modelo mais recente e suporta futuras atualizações de Thread | O núcleo ecológico próprio da Haiman, combinado com o gerenciamento remoto do Haiman APP, adequado para usuários que não se preocupam com isso |
| HS5GW | 64 | Wi-Fi + Ethernet | Alto desempenho | Projetos de médio e grande porte, estáveis ​​e confiáveis ​​|
| HS3GW | 32 | Wi-Fi | Aprimorado | Suficiente para residências de pequeno e médio porte |
| HS1GW | 16 | Wi-Fi | Básico | Nível básico, adequado para um pequeno número de dispositivos |

## Jogabilidade de cena recomendada

| Cenário | Modelo recomendado | Descrição |
|------|----------|------|
| Ecologia Heiman para toda a casa | HS6GW | Usado com sensores Heiman Zigbee e gerenciamento remoto com Heiman APP, não há necessidade de construir seu próprio servidor |
| Usuários de plataformas de terceiros | — | Recomenda-se adquirir um coordenador Zigbee (como SLZB-06) diretamente para usar com Zigbee2MQTT/ZHA para maior flexibilidade |
| Apartamentos pequenos e médios | HS3GW | Limite de 32 dispositivos para atender às necessidades básicas |

## Sugestões de seleção

- **Ecologia própria de Haiman**: [HS6GW](../../Wireless Protocol/Zigbee/HS6GW - Smart Gateway (Zigbee)) (128 dispositivos, WiFi + Ethernet)
- **Plataforma de terceiros**: Coordenador Zigbee universal recomendado + Zigbee2MQTT/ZHA
