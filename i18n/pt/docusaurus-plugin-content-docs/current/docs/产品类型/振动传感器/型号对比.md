# Comparação de modelos

## Comparação de vários modelos

| Modelo | Protocolo | Sensor | Fonte de alimentação | Recursos | Sugestões de jogos |
|------|------|--------|------|------|----------|
| Sensor de vibração | Zigbee | Acelerômetro de 3 eixos | CR2450 | Detecção de vibração + inclinação | Anti-alavanca de portas e janelas, alarme de vibração de abertura ilegal de cofre/gaveta, monitoramento de vibração anormal de tubos |

## Jogabilidade de cena recomendada

| Cenários | Modelos recomendados | Soluções de ligação |
|------|----------|----------|
| Anti-intrusão de portas e janelas | Sensor de vibração | Detecção de impacto violento e vibração de porta/janela → alarme de pressão + alarme sonoro e luminoso + captura de câmera |
| Monitoramento seguro | Sensor de vibração | O cofre é movido ou vibrado → Notificação de alarme imediata |
| Monitoramento de dutos | Sensor de vibração | Monitorizar a vibração anormal das tubagens de água (falha da bomba de água/golpe de aríete), fornecer aviso prévio para manutenção |
| Cerca/Parede | Sensor de vibração | Detecção de escalada em parede/cerca, adequada para segurança de quintal |
