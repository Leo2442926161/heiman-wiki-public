# Guia de instalação

## Requisitos de local de instalação

- Instalado na superfície de objetos que precisam monitorar vibrações
- Locais de instalação comuns: **Dentro de portas e janelas**, **cofres**, **tubos**, **paredes**, **cercas**
- Certifique-se de que a superfície de instalação esteja plana e limpa

## Etapas de instalação

1. Selecione o objeto a ser monitorado (como o interior da porta do cofre)
2. Limpe a superfície de montagem
3. Use fita adesiva 3M ou parafusos para fixar o sensor
4. Instale a bateria CR2450
5. Emparelhamento via gateway
6. Teste de simulação: toque na superfície de instalação, o sensor deve disparar um alarme

## Notas

- Para cenários anti-arrombamento de portas e janelas, é mais eficaz instalá-lo na parte interna da porta/janela perto da fechadura.
- Para monitoramento de dutos, cole diretamente na superfície do duto
- Vida útil da bateria aproximadamente 2 anos
- Sensibilidade ajustável para atender diferentes cenários
