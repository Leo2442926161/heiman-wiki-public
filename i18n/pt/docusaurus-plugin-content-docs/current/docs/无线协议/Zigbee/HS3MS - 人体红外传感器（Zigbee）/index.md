#HS3MS - Sensor infravermelho do corpo humano (Zigbee)

## Visão geral

HS3MS é o sensor infravermelho de corpo humano Zigbee de terceira geração da Hyman. Ele adota um design de montagem no teto de 360° e sua sensibilidade de detecção e cobertura são significativamente melhoradas em comparação com a geração anterior.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Infravermelho Passivo (PIR) |
| Ângulo de detecção | 360° (instalação no teto) |
| Distância de detecção | ≤7m (25°C) |
| Método de fornecimento de energia | Bateria de lítio DC 3V |
| Vida útil da bateria | Aproximadamente 3 anos |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Método de instalação | Instalação de teto |

## Recursos

- Detecção humana móvel
- Cobertura omnidirecional de 360°
- Lembrete de bateria fraca
- Alarme anti-adulteração
- Sensibilidade de detecção aprimorada
- Suporte para ligação de cena
