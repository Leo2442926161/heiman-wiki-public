# HS1WL – Detector de Vazamento de Água (Zigbee)

## Visão geral

HS1WL é o detector de imersão em água Zigbee de primeira geração lançado pela Hyman. Ele adota o método de detecção de sonda e pode detectar efetivamente vazamento ou acúmulo de água. É um equipamento básico de monitoramento de imersão em água.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Método de detecção | Detecção de sonda |
| Método de fornecimento de energia | Bateria CC 3V |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤95% UR |

## Recursos

- Detecção em tempo real de vazamento/acúmulo de água
- Alarme sonoro
- Lembrete de bateria fraca
- Design anticorrosivo da sonda
