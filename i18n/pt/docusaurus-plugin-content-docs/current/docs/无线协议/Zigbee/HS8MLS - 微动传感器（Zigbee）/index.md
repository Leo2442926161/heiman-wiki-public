# HS8MLS - Micro Sensor de Movimento (Zigbee)

## Visão geral

HS8MLS é um sensor infravermelho de corpo humano (PIR) Zigbee de longo alcance lançado pela Heiman. Foi especialmente projetado para cenas que exigem detecção de longa distância e é adequado para espaços abertos, como corredores e halls.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Infravermelho Passivo (PIR) |
| Distância de detecção | Detecção de longa distância |
| Método de fornecimento de energia | Bateria CC 3V |

## Recursos

- Detecção de movimento de longa distância
- Design de baixo consumo de energia
- Suporte para ligação de cena
