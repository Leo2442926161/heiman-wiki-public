#HS8MIS - Sensor Móvel (Zigbee)

## Visão geral

HS8MIS é um sensor de movimento Zigbee multifuncional lançado pela Heiman. Ele integra funções de detecção humana PIR e detecção de iluminação e é adequado para cenas inteligentes com monitoramento duplo de luz e movimento.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Infravermelho Passivo (PIR) + Iluminação |
| Ângulo de detecção | 110° |
| Distância de detecção | ≤7m |
| Método de fornecimento de energia | Bateria CC 3V |

## Recursos

- Detecção humana móvel
- Detecção de iluminação
- Design de baixo consumo de energia
- Suporte para ligação de cena
