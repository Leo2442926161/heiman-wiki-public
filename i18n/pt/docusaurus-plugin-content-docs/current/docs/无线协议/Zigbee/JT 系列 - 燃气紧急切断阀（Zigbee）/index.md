# Série JT - Válvula de corte de emergência de gás (Zigbee)

## Visão geral

A válvula de corte de emergência de gás da série JT é um dispositivo de execução usado em conjunto com o detector de gás combustível Hyman Zigbee. Quando um vazamento de gás é detectado, ele pode fechar remota ou automaticamente a válvula do gasoduto para evitar acidentes.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo | Ligação autônoma + Zigbee |
| Gás de teste | Gás natural / Gás liquefeito de petróleo |
| Padrões de certificação | GB 15322 |
| Temperatura de funcionamento | -10°C ~ +50°C |

## Recursos

- Vinculado com HS1CG / HS3CG para fechar automaticamente a válvula
-Suporte para fechamento manual remoto da válvula
- Alarme de vazamento
- Controlador de válvula solenóide

## Padrões de certificação

- GB 15322

## Acessórios aplicáveis

- Detector de gás combustível HS1CG / HS3CG
