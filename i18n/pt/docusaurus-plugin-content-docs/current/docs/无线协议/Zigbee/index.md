#Zigbee

Zigbee é um protocolo de rede mesh sem fio de baixo consumo de energia e baixa taxa de dados baseado no padrão IEEE 802.15.4. É um dos protocolos mais maduros e populares no campo da casa inteligente.

##Hyman e Zigbee

Hyman é um dos primeiros fabricantes chineses a adotar o protocolo Zigbee e possui uma enorme linha de produtos Zigbee. Os dispositivos Hyman Zigbee são compatíveis com o padrão **Zigbee 3.0** e podem interoperar com gateways e coordenadores Zigbee convencionais.

## Especificações técnicas

| Recursos | Descrição |
|------|------|
| Banda de frequência | 2,4 GHz (em todo o mundo) |
| Taxa de transmissão | Até 250kbps |
| Topologia de rede | Rede de autocura em malha |
| Número de nós | A rede única suporta até 65.535 nós |
| Distância de comunicação | Cerca de 10-30 metros no interior (obstáculos visíveis) |
| Consumo de energia | Extremamente baixa, vida útil da bateria do sensor de até 3 anos |

## Plataformas compatíveis

Os dispositivos Hyman Zigbee podem ser conectados às seguintes plataformas:

- **Zigbee2MQTT** — Zigbee de código aberto para ponte MQTT
- **ZHA** — Home Assistant com integração Zigbee integrada
- **Hubitat** — Hub inteligente localizado
- **Casa** — Hub inteligente multiprotocolo
- **SmartThings Hub** — hub de casa inteligente Samsung
- **Tuya Zigbee Gateway** — Ecossistema Tuya
- **Gateway Hyman HS6GW** — Gateway Oficial
