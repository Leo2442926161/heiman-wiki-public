# HS3WL – Detector de Vazamento de Água (Zigbee)

## Visão geral

HS3WL é o detector de imersão em água Zigbee de terceira geração da Hyman. Possui sensibilidade de detecção aprimorada e desempenho sem fio com base na geração anterior e é adequado para cenários de monitoramento de vazamento de água.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Método de detecção | Detecção de sonda |
| Método de fornecimento de energia | Bateria CC 3V |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤95% UR |

## Recursos

- Detecção em tempo real de vazamento/acúmulo de água
- Sensibilidade de detecção aprimorada
- Alarme sonoro
- Lembrete de bateria fraca
- Design anticorrosivo da sonda
