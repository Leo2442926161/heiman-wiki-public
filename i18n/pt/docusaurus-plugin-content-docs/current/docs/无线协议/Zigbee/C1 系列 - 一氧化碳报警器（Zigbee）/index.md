# Série C1 - Alarme de Monóxido de Carbono (Zigbee)

## Visão geral

A série C1 é um alarme básico de monóxido de carbono Zigbee lançado pela Heiman. Ele usa sensores eletroquímicos e é adequado para cenários de detecção de vazamento de monóxido de carbono. Foi aprovado na certificação EN 50291.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Eletroquímica |
| Alcance de detecção | 0 ~ 1000 ppm |
| Pressão sonora de alarme | ≥85 dB (3 metros) |
| Método de fornecimento de energia | Alimentado por bateria |
| Vida útil do sensor | Aproximadamente 7 anos |
| Temperatura de funcionamento | -10°C ~ +50°C |

## Recursos

- Detecção eletroquímica de CO
- Alarme de classificação de concentração
- Alarme sonoro e visual de 85 dB
- Lembrete de bateria fraca
- Função de autoteste

## Padrões de certificação

-EN 50291
