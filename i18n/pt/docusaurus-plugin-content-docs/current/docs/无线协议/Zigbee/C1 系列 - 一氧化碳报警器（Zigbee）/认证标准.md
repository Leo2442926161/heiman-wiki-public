# Padrões de Certificação

A Série C1 passou pelas seguintes certificações:

- **EN 50291** — Norma europeia para alarmes de monóxido de carbono, consulte [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **CE/RED/EMC** — Diretiva de equipamentos sem fio da União Europeia e compatibilidade eletromagnética
