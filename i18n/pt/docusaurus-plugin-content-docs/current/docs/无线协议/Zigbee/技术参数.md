# Parâmetros técnicos

A seguir estão os parâmetros técnicos gerais para dispositivos Heiman Zigbee.

## Parâmetros sem fio

| Parâmetro | Valor |
|------|-----|
| Padrões de protocolo | Zigbee 3.0 (IEEE 802.15.4) |
| Banda de frequência de trabalho | 2,4 GHz (faixa de frequências ISM comum global) |
| Taxa de transmissão | Até 250kbps |
| Topologia de rede | Rede de autocura em malha |
| Capacidade dos nós | Máximo de 65.535 nós em uma única rede |
| Distância de comunicação | Cerca de 10~30 metros em ambientes fechados (vendo obstáculos na parede) |
| Método de criptografia | AES-128-CCM |

## Compatibilidade

| Projeto | Descrição |
|------|------|
| Coordenador Zigbee | Compatível com coordenador padrão Zigbee 3.0 |
| Plataformas suportadas | Zigbee2MQTT, ZHA (Home Assistant), Hubitat, Homey, SmartThings, gateway Tuya Zigbee, gateway Haiman HS6GW |
| Atualização OTA | Suporte para atualização de firmware over-the-air (alguns modelos) |

## Consumo elétrico e de energia

| Parâmetro | Valor |
|------|-----|
| Método de fornecimento de energia | Alimentado por bateria (CR123A / CR2450 / AA) ou USB / rede elétrica (tipo tomada) |
| Vida útil da bateria | Cerca de 2 a 3 anos para sensores (dependendo da frequência dos relatórios) |
| Consumo de energia em espera | Nível microamp, suporta mecanismo de pesquisa de sono |

## Parâmetros de ambiente

| Parâmetro | Valor |
|------|-----|
| Temperatura de funcionamento | -10°C ~ +55°C |
| Humidade operacional | ≤ 95% HR (sem condensação) |
| Nível de proteção IP | Depende do modelo específico (tipo de sensor geralmente IP30) |
