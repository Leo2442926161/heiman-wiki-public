# HS1DS - Sensor Magnético de Porta (Zigbee)

## Visão geral

HS1DS é um sensor magnético de porta Zigbee lançado pela Heiman. É usado para detectar o estado de abertura e fechamento de portas e janelas. É um componente básico do sistema de segurança inteligente.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Interruptor Reed |
| Método de fornecimento de energia | Bateria botão CR1632 |
| Vida útil da bateria | Cerca de 2 a 3 anos |
| Liberação de instalação | ≤10mm |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤95% UR |

## Recursos

- Detecção em tempo real do status de abertura/fechamento de portas e janelas
- Alarme de violação (Tamper)
- Lembrete de bateria fraca
-Suporta ligação (abrir porta, acender luzes, armar ao sair de casa, etc.)
- Fácil de instalar (fita adesiva 3M ou fixação com parafuso)

## Acessórios aplicáveis

Bateria botão CR1632
- Fita dupla face/parafusos 3M
