## Instruções de nomenclatura de modelo

| Regras de nomenclatura | Significado |
|----------|------|
| Sufixo -E | Versão UE (padrão 868MHz / EN) |
| Sufixo -Z | Versão chinesa (2,4 GHz / padrão chinês) |
| HS1/2/3/8 | Número de geração do produto, quanto maior o número, mais nova é a geração |
