#HS3SA - Alarme de fumaça inteligente (Zigbee)

## Visão geral

HS3SA é o alarme de fumaça inteligente Zigbee de terceira geração da Hyman. Possui desempenho de detecção aprimorado e capacidades de ligação com base na primeira geração e é adequado para o mercado da UE.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Fotoelétrico infravermelho |
| Pressão sonora de alarme | ≥85 dB (3 metros) |
| Método de fornecimento de energia | CC 3V (CR123A) |
| Vida útil da bateria | Aproximadamente 3 anos |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤95% UR |
| Método de instalação | Instalação de teto |

## Recursos

- Detecção fotoelétrica de fumaça
- Alarme sonoro e luminoso de 85 dB de altura
- Lembrete de bateria fraca
- Função de autoteste
- Alarme anti-adulteração
-Suporte para mudo remoto
- Atualização de firmware OTA
- Desempenho sem fio aprimorado
