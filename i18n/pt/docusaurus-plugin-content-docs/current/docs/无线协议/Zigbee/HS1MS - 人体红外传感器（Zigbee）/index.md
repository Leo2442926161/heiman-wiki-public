#HS1MS - Sensor infravermelho do corpo humano (Zigbee)

## Visão geral

HS1MS é um sensor infravermelho de corpo humano (PIR) Zigbee padrão lançado pela Hyman. Ele usa tecnologia de detecção infravermelha passiva e pode detectar a radiação térmica infravermelha emitida por um corpo humano em movimento. É amplamente utilizado em cenários inteligentes de iluminação e segurança.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Infravermelho Passivo (PIR) |
| Ângulo de detecção | 110° |
| Distância de detecção | ≤7m (25°C) |
| Método de fornecimento de energia | Bateria de lítio DC 3V |
| Vida útil da bateria | Aproximadamente 3 anos |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Método de instalação | Montagem na parede |

## Recursos

- Detecção humana móvel
- Detecção de grande angular de 110°
- Lembrete de bateria fraca
- Alarme anti-adulteração
- Suporte para ligação de cena
