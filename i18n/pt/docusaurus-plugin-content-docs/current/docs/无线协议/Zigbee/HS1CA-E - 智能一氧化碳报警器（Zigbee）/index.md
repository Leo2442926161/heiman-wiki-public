# HS1CA-E - Alarme Inteligente de Monóxido de Carbono (Zigbee)

## Visão geral

HS1CA-E é o alarme inteligente de monóxido de carbono (CO) Zigbee da Hyman para o mercado da UE. Ele usa um sensor eletroquímico e pode detectar com precisão o gás monóxido de carbono com uma concentração de 0 ~ 1000 ppm. Foi aprovado na certificação EN 50291.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Eletroquímica |
| Alcance de detecção | 0 ~ 1000 ppm |
| Pressão sonora de alarme | ≥85 dB (3 metros) |
| Método de fornecimento de energia | CC 3V (CR123A) |
| Vida útil do sensor | Aproximadamente 7 anos |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤95% UR |
| Método de instalação | Montagem na parede |

## Recursos

- Sensor eletroquímico de CO de alta precisão
- Alarme graduado de concentração (aviso sonoro e luminoso)
- Som de alarme de 85 dB
- Lembrete de bateria fraca
- Lembrete de expiração da vida útil do sensor
- Função de autoteste
- Atualização de firmware OTA

## Padrões de certificação

- EN 50291 (norma europeia para alarmes de monóxido de carbono)
-CE/VERMELHO

## Acessórios aplicáveis

- Bateria CR123A
