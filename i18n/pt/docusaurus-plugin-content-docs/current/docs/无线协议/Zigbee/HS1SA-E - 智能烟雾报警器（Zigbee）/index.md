# HS1SA-E - Alarme de fumaça inteligente (Zigbee)

## Visão geral

HS1SA-E é um alarme de fumaça inteligente Zigbee padrão lançado pela Hyman para o mercado da UE. Adota tecnologia de detecção fotoelétrica de fumaça e passou a certificação EN 14604 e ganhou o Red Dot Design Award.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Fotoelétrico infravermelho |
| Pressão sonora de alarme | ≥85 dB (3 metros) |
| Método de fornecimento de energia | CC 3V (CR123A) |
| Vida útil da bateria | Aproximadamente 3 anos |
| Distância de comunicação | ≤70 m (área aberta) |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤95% UR |
| Método de instalação | Instalação de teto |

## Recursos

- Detecção fotoelétrica de fumaça para detectar fumaça de queimaduras lentas e chamas abertas
- Alarme sonoro e luminoso de 85 dB de altura
- Lembrete de bateria fraca
- Função de autoteste (teste com um clique)
- Alarme anti-adulteração
-Suporte para mudo remoto
- Atualização de firmware OTA

## Padrões de certificação

- EN 14604 (norma europeia para alarmes de fumo)
- Prêmio Red Dot Design
-CE/VERMELHO

## Acessórios aplicáveis

- Bateria CR123A
- Parafusos de montagem/tubo de expansão
