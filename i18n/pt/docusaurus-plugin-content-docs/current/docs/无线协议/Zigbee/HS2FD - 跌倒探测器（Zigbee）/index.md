#HS2FD - Detector de queda (Zigbee)

## Visão geral

HS2FD é um detector de queda inteligente Zigbee lançado pela Hyman. É especialmente projetado para idosos. Possui um acelerômetro e giroscópio integrados. Ele usa um algoritmo de IA para identificar a postura de queda e pode enviar automaticamente um sinal de socorro.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Acelerômetro + Giroscópio |
| Tecnologia de Detecção | Algoritmo de detecção de queda AI |
| Método de fornecimento de energia | Bateria CC 3V |
| Método de uso | Clipe/cordão para cinto |

## Recursos

- Detecção automática de queda
- Peça ajuda manualmente com um clique
- Algoritmo de IA distingue quedas de atividades normais (sentar, curvar-se) para reduzir alarmes falsos
- Design vestível (clipe para cinto ou cordão)
- Lembrete de bateria fraca

## Cenários aplicáveis

- Atendimento domiciliar para idosos: atendimento 24 horas por dia para idosos que moram sozinhos
- Lares de idosos: custódia e gestão centralizada
- Centro de reabilitação: Monitoramento do risco de quedas em pacientes pós-operatórios
