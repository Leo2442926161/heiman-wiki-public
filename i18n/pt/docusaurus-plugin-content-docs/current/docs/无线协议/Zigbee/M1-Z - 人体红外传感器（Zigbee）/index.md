# M1-Z - Sensor infravermelho do corpo humano (Zigbee)

## Visão geral

M1-Z é um sensor infravermelho Zigbee compacto para corpo humano lançado pela Heiman. Ele suporta ângulo de detecção de 110° ou 360° (dependendo do método de instalação), possui um sensor de iluminação integrado e é adequado para cenários inteligentes de iluminação e segurança.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 |
| Tipo de sensor | Infravermelho Passivo (PIR) + Iluminação |
| Ângulo de detecção | 110° (montagem na parede) / 360° (teto) |
| Distância de detecção | ≤7m (25°C) |
| Faixa de iluminância | 0 ~ 1200 Lux |
| Método de fornecimento de energia | Bateria de lítio DC 3V |
| Vida útil da bateria | Aproximadamente 3 anos |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Método de instalação | De parede/teto |

## Recursos

- Detecção humana móvel
- Detecção de iluminação (0~1200 Lux)
- 110° / 360° comutável
- Lembrete de bateria fraca
- Alarme anti-adulteração
- Design compacto
