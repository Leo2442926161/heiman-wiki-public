#HS8SA - Alarme de fumaça inteligente (Zigbee)

## Visão geral

HS8SA é o alarme de fumaça inteligente Zigbee + Wi-Fi da Hyman para o mercado dos EUA. Possui função de ligação de alta temperatura e foi aprovado na certificação UL 217.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 / Wi-Fi |
| Tipo de sensor | Fotoelétrico infravermelho |
| Pressão sonora de alarme | ≥85 dB (3 metros) |
| Método de fornecimento de energia | CC 3V (CR123A) |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Padrões de certificação | UL 217 |
| Mercados aplicáveis ​​| Estados Unidos |

## Recursos

- Detecção fotoelétrica de fumaça
- Alarme de ligação de alta temperatura
- Alarme sonoro e luminoso de 85 dB de altura
- Lembrete de bateria fraca
- Função de autoteste
- Alarme anti-adulteração
-Suporte para mudo remoto
- Comunicação de protocolo duplo (Zigbee + Wi-Fi)

## Padrões de certificação

- UL 217 (padrão americano de alarme de fumaça)
