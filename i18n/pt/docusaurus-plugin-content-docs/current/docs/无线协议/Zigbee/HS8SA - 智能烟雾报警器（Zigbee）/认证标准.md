# Padrões de Certificação

HS8SA passou nas seguintes certificações:

- **EN 14604** — Norma europeia para alarmes de fumaça (tipo UE), consulte [EN European Standard](../../../../ApprovalStandards/EN European Standard.md)
- **UL 217** — Padrão de segurança de alarme de fumaça dos EUA (modelo dos EUA), consulte [UL Underwriters Laboratories](../../../../certification-standards/ULUnderwriters Laboratories.md)
- **CE/RED/EMC** — Diretiva de equipamentos sem fio da União Europeia e compatibilidade eletromagnética
- **Zigbee 3.0 / Q-Label** — Certificação Zigbee Alliance, consulte [Outras Certificações](../../../Certification Standards/Other Certifications.md)
