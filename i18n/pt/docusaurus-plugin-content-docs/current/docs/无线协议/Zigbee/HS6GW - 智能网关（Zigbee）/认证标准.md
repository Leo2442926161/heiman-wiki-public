# Padrões de Certificação

HS6GW (versão Zigbee) passou nas seguintes certificações:

- **CE/RED** — Diretiva de Equipamentos Sem Fio da União Europeia
- **CCC** — Certificação obrigatória de produto da China, consulte [CCC China Compulsory Certification](../../../../Certification Standards/CCC China Compulsory Certification.md)
