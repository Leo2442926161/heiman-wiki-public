# HS6GW - Gateway Inteligente (Zigbee)

## Visão geral

HS6GW é um gateway inteligente Zigbee lançado pela Heiman. Como coordenador da rede Zigbee da Heiman, ele se conecta à Internet por meio de uplink Wi-Fi ou Ethernet e suporta o acesso de até 128 subdispositivos.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Zigbee 3.0 (Coordenador) |
| Conexão de uplink | WiFi/Ethernet |
| Máximo de nós | 128 subdispositivos |
| Método de fornecimento de energia | Fonte de alimentação USB DC 5V |

## Recursos

- Coordenador de Rede Zigbee
- Conexão uplink Wi-Fi/Ethernet
- Descoberta e emparelhamento automático de dispositivos
- Ligação de cena local
- Acesso remoto (via Heiman Cloud Platform)
- Suporta até 128 subdispositivos
- Atualização de firmware OTA

## Cenários aplicáveis

- O principal ponto de conexão da inteligência de toda a casa Heiman
- Coopere com o Haiman APP para realizar o monitoramento remoto
