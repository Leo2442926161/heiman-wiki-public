# Alarme de fumaça inteligente HS1SA-M (matéria)

HS1SA-M é um alarme de fumaça inteligente **Matter over Thread** lançado pela Hyman. Ele usa tecnologia de detecção de fumaça fotoelétrica e suporta interconexão de plataforma doméstica inteligente ecológica.

## Visão geral do produto

- **Tipo de produto**: Alarme de fumaça inteligente
- **Protocolo de comunicação**: assunto em discussão
- **Padrão de Certificação**: EN 14604
- **Cenários típicos**: Alerta de incêndio em residências, apartamentos, hotéis, etc.

## Parâmetros técnicos

| Parâmetro | Valor |
|------|-----|
| Tipo de sensor | Fotoelétrico infravermelho |
| Protocolo de Comunicação | Matéria sobre Thread (IEEE 802.15.4) |
| Banda de frequência operacional | 2,4GHz |
| Pressão sonora de alarme | ≥85 dB (3 metros) |
| Método de fornecimento de energia | DC 3V (bateria de lítio CR123A) |
| Vida útil da bateria | Aproximadamente 3 anos |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤95% UR (sem condensação) |

## Recursos

- Detecção fotoelétrica de fumaça para detectar queimaduras lentas e fumaça de chama aberta
- Alarme sonoro e luminoso de 85 dB de altura
- Lembrete de bateria fraca
- Função de autoteste (teste com um clique)
- Alarme anti-adulteração
- Mudo remoto
- Importa interoperabilidade multi-ecológica entre plataformas

## Certificação

- Certificação EN 14604
- Certificação de matéria

## Plataformas compatíveis

| Plataforma | Suporte |
|------|----------|
| Página inicial da Apple | ✅ Necessário roteador de borda de linha |
| Página inicial do Google | ✅ Necessário roteador de borda de linha |
| AmazonAlexa | ✅ Necessário roteador de borda de linha |
| Assistente Doméstico | ✅ Matéria Certificada |
| Coisas inteligentes | ✅ Requer roteador de borda de rosca |

## Requisitos de roteamento de fronteira

É necessário um roteador de borda Thread, como Apple HomePod mini, Google Nest Hub 2ª geração, Amazon Echo 4ª geração, etc.
