# Sensor de temperatura e umidade H1-M (matéria)

H1-M é um sensor de temperatura e umidade **Matter over Thread** lançado pela Hyman, que pode monitorar a temperatura ambiente e a umidade relativa em tempo real e oferece suporte à ligação de automação residencial inteligente.

## Visão geral do produto

**Tipo de produto**: Sensor de temperatura e umidade
- **Protocolo de comunicação**: assunto em discussão
- **Parâmetros de detecção**: temperatura, umidade relativa
- **Cenários Típicos**: Monitoramento ambiental em residências, escritórios, adegas, etc.

## Parâmetros técnicos

| Parâmetro | Valor |
|------|-----|
| Precisão de temperatura | ±0,3°C |
| Precisão de umidade | ±3% UR |
| Protocolo de Comunicação | Matéria sobre Thread (IEEE 802.15.4) |
| Banda de frequência operacional | 2,4GHz |
| Método de fornecimento de energia | Bateria de lítio CR2 |
| Vida útil da bateria | Cerca de 2 a 3 anos |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤95% UR (sem condensação) |

## Recursos

- Monitoramento de temperatura e umidade em tempo real
- Detecção de alta precisão (temperatura ±0,3°C, umidade ±3% UR)
- Lembrete de bateria fraca
- Importa interoperabilidade multi-ecológica entre plataformas

## Plataformas compatíveis

| Plataforma | Suporte |
|------|----------|
| Página inicial da Apple | ✅ Necessário roteador de borda de linha |
| Página inicial do Google | ✅ Necessário roteador de borda de rosca |
| AmazonAlexa | ✅ Necessário roteador de borda de linha |
| Assistente Doméstico | ✅ |
| Coisas inteligentes | ✅ Requer roteador de borda de rosca |

## Requisitos de roteamento de fronteira

É necessário um roteador de borda Thread, como Apple HomePod mini, Google Nest Hub 2ª geração, Amazon Echo 4ª geração, etc.
