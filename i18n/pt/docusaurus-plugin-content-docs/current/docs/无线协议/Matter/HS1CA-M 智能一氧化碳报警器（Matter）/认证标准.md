# Padrões de Certificação

HS1CA-M passou nas seguintes certificações:

- **EN 50291** — Norma europeia para alarmes de monóxido de carbono, consulte [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **Marca CE** — Declaração de conformidade da UE, consulte [Conformidade CE UE](../../../../certification-standards/CE EU-Conformity.md)
- **Certificação Matter** — Aprovado na certificação CSA Alliance Matter para garantir a interconexão entre plataformas de casas inteligentes
