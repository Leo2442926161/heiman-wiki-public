# HS1DS-M - Sensor de porta inteligente (Matéria sobre Rosca)

## Visão geral

HS1DS-M é a versão Matter over Thread do sensor de porta inteligente Heiman. Ele suporta acesso nativo a todas as plataformas compatíveis com Matter, como Apple Home, Google Home, Amazon Alexa, Home Assistant, etc., sem a necessidade de um gateway proprietário.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | Matéria sobre Tópico |
| Método de fornecimento de energia | Bateria botão CR1632 |
| Vida útil da bateria | Cerca de 2-3 anos |
| Distância de instalação | ≤10mm |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤95% UR |

## Recursos

- Detecção em tempo real do status de abertura/fechamento de portas e janelas
- Alarme de violação (Tamper)
- Lembrete de bateria fraca
Compatível com várias plataformas: Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Rede Thread Mesh, controle local de baixa latência
- Não é necessário gateway dedicado, é necessário roteador de borda de thread
