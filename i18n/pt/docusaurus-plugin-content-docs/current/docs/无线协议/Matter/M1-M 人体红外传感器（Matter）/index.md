# Sensor infravermelho de corpo humano M1-M (matéria)

M1-M é um sensor infravermelho de corpo humano **Matter over Thread** lançado pela Hyman. Ele integra funções de detecção PIR e detecção de iluminação e é adequado para cenários de automação residencial inteligente.

## Visão geral do produto

** Tipo de produto **: Sensor infravermelho do corpo humano
- **Protocolo de comunicação**: assunto em discussão
**Método de detecção**: infravermelho passivo (PIR) + detecção de iluminação
- **Cenário Típico**: As luzes acendem quando as pessoas chegam, ligação de segurança, automação ambiental

## Parâmetros técnicos

| Parâmetro | Valor |
|------|-----|
| Ângulo de detecção | 110° (montagem na parede) / 360° (teto) |
| Distância de detecção | ≤7m (25°C) |
| Faixa de iluminação | 0~1200lux |
| Protocolo de Comunicação | Matéria sobre Thread (IEEE 802.15.4) |
| Banda de frequência operacional | 2,4GHz |
| Método de fornecimento de energia | Bateria de lítio DC 3V |
| Vida útil da bateria | Aproximadamente 1 ano (modo Matéria) |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤95% UR (sem condensação) |

## Recursos

- Detecção humana móvel
- Detecção de iluminação ambiente
- Distância de detecção ≤7m
- Suporte para instalação em parede ou teto
- Importa interoperabilidade multi-ecológica entre plataformas
