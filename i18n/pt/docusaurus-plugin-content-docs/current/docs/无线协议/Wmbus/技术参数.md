# Parâmetros técnicos

A seguir estão os parâmetros técnicos gerais do equipamento Heimann Wireless M-Bus (wM-Bus).

## Parâmetros sem fio

| Parâmetro | Valor |
|------|-----|
| Padrões de protocolo | EN 13757-4 (Camada física M-Bus sem fio) |
| Banda de frequência de trabalho | 868 MHz (faixa de frequências SRD europeia, 868,0~868,6 MHz) |
| Método de modulação | FSK |
| Taxa de transmissão | 4,8 kbps / 16,384 kbps / 100 kbps (modo T / modo S / modo R) |
| Topologia de rede | Estrela (instrumento → coletor/gateway) |
| Distância de comunicação | Cerca de 100~300 metros em ambientes fechados, cerca de 500~1000 metros em espaços abertos |
| Transmitir potência | ≤ 25 mW (+14 dBm) |

## Modo de trabalho

| Modo | Taxa | Descrição |
|------|------|------|
| Modo S | 16,384kbps | Modo padrão, baixo consumo de energia, adequado para instrumentos alimentados por bateria |
| Modo T | 100kbps | Modo de alta velocidade, adequado para cenários de grandes volumes de dados de tráfego |
| Modo R | 4,8kbps | Modo de banda estreita, transmissão de longa distância |

## Consumo elétrico e de energia

| Parâmetro | Valor |
|------|-----|
| Método de fornecimento de energia | Fonte de alimentação de bateria de lítio (ER18505 / ER26500 e outras baterias de lítio industriais) |
| Vida útil da bateria | 6~10 anos (dependendo da frequência de notificação, o valor típico é 1 vez/dia) |
| Consumo de energia em espera | Nível de microamp, sono profundo |
| Receber sensibilidade | Aprox. -105 dBm (modo S) |

## Parâmetros de ambiente

| Parâmetro | Valor |
|------|-----|
| Temperatura de funcionamento | -20°C ~ +60°C (grau industrial) |
| Humidade operacional | ≤ 95% HR (sem condensação) |
| Nível de proteção IP | IP65 ~ IP68 (dependendo do modelo específico, aplicável a instrumentos externos) |

## Protocolo de dados

| Camada | Padrão |
|----|------|
| Camada física | EN 13757-4 |
| Camada de enlace de dados | EN 13757-4 (Formato de quadro M-Bus sem fio) |
| Camada de aplicação | EN 13757-3 (camada de aplicação M-Bus) |
| Método de criptografia | AES-128 (criptografia opcional) |
