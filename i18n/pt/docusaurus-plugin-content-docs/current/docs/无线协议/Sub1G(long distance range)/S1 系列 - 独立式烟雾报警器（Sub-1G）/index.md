# Série S1 - Alarme de fumaça autônomo (Sub-1G)

## Visão geral

A série S1 é um alarme de fumaça Sub-1G independente lançado pela Hyman. Opera na faixa de frequência 433 MHz / 868 MHz, utiliza tecnologia de detecção fotoelétrica de fumaça e foi aprovado na certificação EN 14604. Nenhum gateway ou Internet é necessário. Os dispositivos são interligados diretamente por meio de radiofrequência Sub-1G. Quando qualquer dispositivo dispara um alarme, todos os dispositivos do mesmo grupo soarão o alarme simultaneamente. É adequado para cenários de ligação contra incêndio em grande escala, como edifícios, shopping centers e hotéis.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | 433 MHz / 868 MHz Sub-1G (dependendo da área de mercado) |
| Tipo de sensor | Fotoelétrico infravermelho |
| Pressão sonora de alarme | ≥85 dB (3 metros) |
| Método de fornecimento de energia | Alimentado por bateria (bateria de lítio CR123A / AA) |
| Vida útil da bateria | ≥ 5 anos |
| Distância de comunicação | Cerca de 1~3 quilómetros em ambiente aberto |
| Capacidade da rede | Um único grupo suporta até mais de 50 nós |
| Método de interligação | Interligação por radiofrequência sem fios (sem necessidade de gateway) |
| Temperatura de funcionamento | -10°C ~ +55°C |
| Umidade de trabalho | ≤ 95% UR |
| Método de instalação | Instalação de teto |

## Recursos

- Detecção fotoelétrica de fumaça para detectar fumaça de queimaduras lentas e chamas abertas
- Alarme de interconexão sem fio: acionado por qualquer dispositivo na mesma rede, todos os dispositivos soarão e acenderão o alarme simultaneamente
- Modo de trabalho independente: sem gateway, sem necessidade de Internet, comunicação direta entre dispositivos
- Alarme sonoro e luminoso de 85 dB de altura
- Lembrete de bateria fraca
- Função de autoteste (teste com um clique)
- Alarme anti-adulteração
-Suporte para mudo remoto
- Design de consumo de energia ultrabaixo, vida útil da bateria de mais de 5 anos

## Padrões de certificação

- EN 14604 (norma europeia para alarmes de fumo)
-CE/VERMELHO
- CPR (Regulamento Europeu sobre Produtos de Construção)

## Acessórios aplicáveis

- Repetidor sem fio S1-RM (distância de comunicação estendida)
- Controle remoto S1-RC (teste/mudo/reset)
