#Lista de modelos suportados

Os produtos Hyman Z-Wave destinam-se principalmente aos mercados domésticos inteligentes europeus e americanos e são compatíveis com o padrão Z-Wave Plus.

## Categoria de alarme de segurança

| Modelo | Nome do produto | Descrição |
|------|----------|------|
| HS1SA-Z | Alarme de fumaça inteligente (Z-Wave) | Alarme de fumaça versão Z-Wave |
| HS1CA-Z | Alarme inteligente de monóxido de carbono (Z-Wave) | Alarme de CO versão Z-Wave |
| HS1CG-Z | Detector de Gás Combustível (Z-Wave) | Detector de gás versão Z-Wave |

## Classe de sensor

| Modelo | Nome do produto | Descrição |
|------|----------|------|
| HS1DS-Z | Sensor de porta (Z-Wave) | Sensor de porta e janela versão Z-Wave |
| HS1MS-Z | Sensor infravermelho do corpo humano (Z-Wave) | Versão Z-Wave de detecção do corpo humano |
| HS1WL-Z | Detector de imersão em água (Z-Wave) | Detecção de vazamento de água na versão Z-Wave |

## Compatibilidade

| Projeto | Descrição |
|------|------|
| Padrão | Z-Wave Plus (chips da série 500/700) |
| Hub compatível | Homey, Hubitat, SmartThings, Vera, Fibaro, Home Assistant (Z-Wave JS) |
| Segurança S2 | Suporte ao protocolo de segurança S2 (não criptografado / S2 não autenticado / S2 autenticado) |
| SmartStart | Suporte (modelos de chips da série 700) |
| Atualização OTA | Suporte para atualização de firmware CC |
