#Onda Z

Z-Wave é um protocolo de rede Mesh sem fio projetado para casas inteligentes. É gerido pela Z-Wave Alliance e tem ampla penetração nos mercados europeu e americano.

## Especificações técnicas

| Recursos | Descrição |
|------|------|
| Banda de frequência operacional | 868,42 MHz (UE) / 908,42 MHz (EUA) |
| Taxa de transferência | Até 100 kbps (Série Z-Wave 800) |
| Topologia de rede | Rede mesh, até 232 nós |
| Compatibilidade | Compatível com versões anteriores de todos os dispositivos Z-Wave |
| Segurança | Criptografia Z-Wave S2 |

##Hyman e Z-Wave

Hyman oferece uma variedade de sensores compatíveis com Z-Wave, vendidos principalmente nos mercados europeu e americano. As versões Z-Wave dos dispositivos Heyman incluem:

- Sensor de monóxido de carbono HS1CA-Z
- Sensor de fumaça HS1SA-Z
- Sensor de porta HS1DS-Z

## Plataformas compatíveis

- Hub SmartThings
- Elevação Hubitat
- Caseiro Pro
- Assistente Doméstico (via Z-Wave JS)
-OpenHAB

> **Nota**: Z-Wave tem baixa popularidade no mercado chinês e os produtos Haiman Z-Wave são direcionados principalmente aos mercados de exportação europeus e americanos.
