# Sensor de porta HS1DS-Z (Z-Wave)

HS1DS-Z é um sensor de porta **Z-Wave** lançado pela Hyman. Ele detecta o status de abertura e fechamento de portas e janelas por meio de interruptores magnéticos e é compatível com o padrão Z-Wave Plus.

## Visão geral do produto

**Tipo de produto**: Sensor de porta
- **Protocolo de comunicação**: Z Wave Plus
- **Método de detecção**: Reed switch (reed switch)
- **Cenários típicos**: monitoramento de status de portas e janelas, implantação de segurança e ligação inteligente

## Parâmetros técnicos

| Parâmetro | Valor |
|------|-----|
| Método de detecção | Interruptor reed (interruptor reed) |
| Protocolo de Comunicação | Z-Wave Plus (ITU-T G.9959) |
| Banda de frequência operacional | 868,42 MHz (UE) / 908,42 MHz (EUA) |
| Método de fornecimento de energia | Bateria botão CR1632 |
| Vida útil da bateria | Cerca de 2 a 3 anos |
| Liberação de instalação | ≤10mm |
| Método de criptografia | S2 autenticado/S2 não autenticado |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤90% UR (sem condensação) |

## Recursos

- Detecção em tempo real do status de abertura/fechamento de portas e janelas
- Alarme de violação (Tamper)
- Lembrete de bateria fraca
-Suporta ligação de cena (abrir porta, acender luzes, armar ao sair de casa)
- Fácil de instalar (fita adesiva 3M ou fixação com parafuso)
- Criptografia segura Z Wave S2
- Suporte para atualização de firmware OTA
