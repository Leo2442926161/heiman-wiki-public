# Padrões de Certificação

HS1CA-Z passou nas seguintes certificações:

- **EN 50291** — Norma europeia para alarmes de monóxido de carbono, consulte [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **Certificação Z-Wave S2** — Suporta estrutura de segurança S2, consulte [Outras certificações](../../../Certification Standards/Other Certifications.md)
- **CE** — Declaração de Conformidade da União Europeia
