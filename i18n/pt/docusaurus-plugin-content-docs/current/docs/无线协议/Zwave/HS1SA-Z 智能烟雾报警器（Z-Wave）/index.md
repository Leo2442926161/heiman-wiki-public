# Alarme de fumaça inteligente HS1SA-Z (Z-Wave)

HS1SA-Z é um alarme de fumaça inteligente **Z-Wave** lançado pela Hyman. Ele usa tecnologia de detecção fotoelétrica de fumaça, é compatível com o padrão Z-Wave Plus e é adequado para os mercados domésticos inteligentes europeus e americanos.

## Visão geral do produto

- **Tipo de produto**: Alarme de fumaça inteligente
- **Protocolo de comunicação**: Z Wave Plus
- **Padrão de Certificação**: EN 14604
- **Cenário Típico**: Alerta de incêndio em residências, apartamentos e outros locais

## Parâmetros técnicos

| Parâmetro | Valor |
|------|-----|
| Tipo de sensor | Fotoelétrico infravermelho |
| Protocolo de Comunicação | Z-Wave Plus (ITU-T G.9959) |
| Banda de frequência operacional | 868,42 MHz (UE) / 908,42 MHz (EUA) |
| Pressão sonora de alarme | ≥85 dB (3 metros) |
| Método de fornecimento de energia | Bateria de lítio CR123A |
| Vida útil da bateria | Cerca de 2 a 3 anos |
| Método de criptografia | S2 autenticado/S2 não autenticado |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤90% UR (sem condensação) |

## Recursos

- Detecção fotoelétrica de fumaça para detectar queimaduras lentas e fumaça de chama aberta
- Alarme sonoro e luminoso de 85 dB de altura
- Lembrete de bateria fraca
- Função de autoteste
- Alarme anti-adulteração
- Criptografia segura Z Wave S2
- Suporte para atualização de firmware OTA

## Certificação

Certificado Z Wave Plus
- Certificação de segurança S2
