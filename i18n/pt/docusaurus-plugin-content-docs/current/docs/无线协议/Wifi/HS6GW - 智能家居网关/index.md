#HS6GW - Gateway de casa inteligente

## Visão geral

HS6GW é um gateway doméstico inteligente lançado pela Heiman, com suporte para uplink WiFi + Ethernet, coordenador Zigbee integrado e pode gerenciar até 128 subdispositivos Zigbee. Realizar acesso remoto e serviços em nuvem conectando-se à Internet é o centro do sistema doméstico inteligente Haiman.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Comunicação de ligação ascendente | IEEE 802.11 b/g/n (2,4 GHz WiFi) + Ethernet 10/100M |
| Comunicação de downlink | Zigbee 3.0 (modo coordenador) |
| Número máximo de subdispositivos | 128 |
| Método de fornecimento de energia | CC 5V/1A (Micro USB) |
| Consumo de energia em espera | ≤ 1W |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤ 90% UR |
| Luz Indicadora | Status do WiFi / Energia / Status da rede Zigbee |
| Método de redefinição | Reinicialização do botão + toque longo para restaurar as configurações de fábrica |

## Recursos

Uplinks duplos: conexão com fio WiFi ou Ethernet, escolha flexível de acordo com o ambiente de rede
-Suporte para acesso remoto: verifique o status do dispositivo a qualquer hora e em qualquer lugar através do aplicativo na nuvem
- Ligação de cena local: suporta operação automatizada de equipamentos dentro da LAN (ainda disponível quando desconectado)
- Atualização de firmware OTA
- Suporte à plataforma Tuya / Smart Life
- Gerencie até 128 subdispositivos Zigbee
-Suporte para autocura da rede Zigbee e otimização de roteamento
