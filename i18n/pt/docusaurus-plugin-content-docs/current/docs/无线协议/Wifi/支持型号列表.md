#Lista de modelos suportados

O equipamento Heiman WiFi é projetado principalmente para cenários de conexão direta e pode ser conectado à rede doméstica sem um gateway dedicado.

## Tomada inteligente

| Modelo | Nome do produto | Descrição |
|------|----------|------|
| HS2SK-WiFi | Tomada inteligente (WiFi) | Conectado diretamente ao WiFi de 2,4 GHz, suporta controle remoto e temporização |

## Categoria de alarme de segurança

| Modelo | Nome do produto | Descrição |
|------|----------|------|
| HS8SA-WiFi | Alarme de fumaça inteligente (versão WiFi dos EUA) | Alarme de fumaça versão WiFi para o mercado dos EUA |

## Classe de gateway

| Modelo | Nome do produto | Descrição |
|------|----------|------|
| HS6GW | Gateway doméstico inteligente (WiFi + Zigbee) | Conecte-se à Internet via WiFi e gerencie subdispositivos Zigbee |
| HS6GW-2 | Gateway doméstico inteligente (WiFi + Zigbee) | Gateway de segunda geração, atualização de desempenho |

## Notas

- Todos os dispositivos WiFi suportam apenas a banda **2,4 GHz** e não são compatíveis com 5 GHz
- Recomenda-se usar rede criptografada WPA2-PSK
- A rede requer um roteador para se conectar corretamente à Internet
