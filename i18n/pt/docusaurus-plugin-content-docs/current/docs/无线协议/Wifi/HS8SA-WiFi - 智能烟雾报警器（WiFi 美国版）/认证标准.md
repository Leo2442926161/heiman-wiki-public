# Padrões de Certificação

HS8SA-WiFi passou nas seguintes certificações:

- **UL 217** — Padrão americano de segurança para alarmes de fumaça, consulte [UL Underwriters Laboratories](../../../../certification standards/ULUnderwriters Laboratories.md)
- **FCC** — Certificação da Comissão Federal de Comunicações
- **CE** — Declaração de Conformidade da União Europeia
