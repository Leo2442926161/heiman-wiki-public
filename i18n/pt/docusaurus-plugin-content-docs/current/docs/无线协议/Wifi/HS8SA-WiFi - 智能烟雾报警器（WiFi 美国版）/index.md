# HS8SA-WiFi - Alarme de fumaça inteligente (versão WiFi dos EUA)

## Visão geral

HS8SA-WiFi é um alarme de fumaça inteligente WiFi lançado pela Hyman para o mercado dos EUA. Ele usa tecnologia fotoelétrica de detecção de fumaça, suporta rede WiFi e modos duplos de trabalho independentes e foi aprovado na certificação UL 217. Adequado para proteção contra incêndio em residências, apartamentos e outros locais residenciais.

## Especificações técnicas

| Parâmetro | Valor |
|------|-----|
| Protocolo de Comunicação | IEEE 802.11 b/g/n (2,4 GHz) |
| Tipo de sensor | Fotoelétrico infravermelho |
| Pressão sonora de alarme | ≥85 dB (3 metros) |
| Método de fornecimento de energia | Alimentado por bateria |
| Temperatura de funcionamento | -10°C ~ +50°C |
| Umidade de trabalho | ≤ 90% UR |
| Método de instalação | Instalação de teto |

## Recursos

- Detecção fotoelétrica de fumaça para detectar fumaça de queimaduras lentas e chamas abertas
- Alarme remoto de rede WiFi
- Modo de operação independente (alarme sonoro e luminoso local ainda pode ocorrer quando a rede estiver desconectada)
- Alarme sonoro e luminoso de 85 dB de altura
- Lembrete de bateria fraca
- Função de autoteste (teste com um clique)
- Alarme anti-adulteração
-Suporte para mudo remoto

## Padrões de certificação

- UL 217 (padrão americano para alarmes de fumaça)
-FCC
- Conformidade com a NFPA 72
