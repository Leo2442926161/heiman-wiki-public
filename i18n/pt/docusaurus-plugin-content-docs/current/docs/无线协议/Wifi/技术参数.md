# Parâmetros técnicos

A seguir estão as especificações técnicas gerais dos dispositivos Heimann WiFi.

## Parâmetros sem fio

| Parâmetro | Valor |
|------|-----|
| Padrões de protocolo | IEEE 802.11b/g/n |
| Banda de frequência operacional | 2,4 GHz (não compatível com 5 GHz) |
| Taxa de transmissão | Até 150 Mbps (802.11n) |
| Topologia de rede | Topologia em estrela (conectada diretamente ao roteador) |
| Distância de comunicação | Cerca de 30~50 metros em ambientes internos (dependendo do desempenho do roteador) |
| Método de criptografia | WPA2-PSK/WPA3 |

## Método de distribuição de rede

| Método | Descrição |
|------|------|
| SmartConfig | Mobile App Configuração de rede com um clique (equipamento WiFi tradicional) |
| Provisionamento assistido por Bluetooth | Configurando credenciais WiFi através de canais temporários Bluetooth (alguns modelos novos) |
| Modo AP | Ligue o hotspot no dispositivo e configure-o após conectar o telefone |

## Consumo elétrico e de energia

| Parâmetro | Valor |
|------|-----|
| Método de fornecimento de energia | Fonte de alimentação principal (tipo tomada/tipo gateway) ou fonte de alimentação por bateria (tipo alarme) |
| Nível de consumo de energia | Maior que Zigbee / Thread devido à necessidade de manter conexão TCP/IP |
| Consumo de energia em espera | Tipo de soquete cerca de 0,5W ~ 1W |

## Parâmetros de ambiente

| Parâmetro | Valor |
|------|-----|
| Temperatura de funcionamento | -10°C ~ +50°C |
| Humidade operacional | ≤ 90% HR (sem condensação) |
