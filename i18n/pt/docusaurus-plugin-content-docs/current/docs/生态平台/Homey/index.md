#Casa

Homey é um hub doméstico inteligente lançado pela empresa holandesa Athom, que suporta Zigbee, Z-Wave, Wi-Fi, Matter, Thread e outros protocolos.

## Método de integração

Os dispositivos Homey Zigbee e Z-Wave emparelham diretamente através do chip sem fio integrado do Homey, eliminando a necessidade de um gateway adicional. Os dispositivos Matter podem ser acessados ​​​​por meio do roteador Thread border integrado do Homey Pro.

## Vantagens

**Vários protocolos em um**: Homey Pro suporta Zigbee, Z-Wave, Thread e Wi-Fi ao mesmo tempo
- **Executar localmente**: toda a automação é executada localmente
- **Automação de fluxo**: programação de arrastar e soltar para criar cenários de ligação complexos
