# Dicas de uso

## Interferência do canal Zigbee

O chip Zigbee integrado do Homey usa canais específicos. Se o dispositivo Heiman ficar off-line com frequência:

1. Veja o canal atual em Homey Settings → Zigbee
2. Use o aplicativo Homey Insight para verificar interferências ambientais
3. Mude para um canal com menos interferência (longe do WiFi 06/01/11)

## Rede de distribuição de equipamentos

- Como entrar no modo de emparelhamento para dispositivos Heiman: pressione e segure o botão por 5 segundos (alguns modelos precisam pressionar 3 vezes rapidamente)
- Ao adicionar um dispositivo no Homey App, selecione "Adicionar dispositivo → Zigbee → Pesquisar"
- Quando o emparelhamento falhar, aproxime o dispositivo do Homey (num raio de 3 metros) e tente novamente

## Sugestões de otimização de fluxo

- Adicione a condição de "atraso" no Flow (por exemplo, 2 segundos) para evitar falsos positivos transitórios
- Use a função "tag" para classificar e gerenciar equipamentos Heiman

## Extensão do alcance do sinal

Se o dispositivo Heiman estiver longe de Homey, um dispositivo de retransmissão Zigbee (como um soquete inteligente) pode ser adicionado para estender a cobertura da rede.
