# Exemplo de automação

## Alarme de fumaça - pisca e notifica

**Configuração de fluxo:**

1. **Acionador:** Alarme de fumaça Hyman → O status muda para `alerta`
2. **Condições:** (Nenhuma)
3. **Ação:**
   - Luz da sala → piscando (vermelho, uma vez por segundo)
   - Notificação push → Celular: "⚠️ Alarme de fumaça acionado!"
   - Anúncio de voz → Alto-falante caseiro: "Fumaça detectada, preste atenção à segurança!"

## Alarme CO - Desligar a caldeira

**Configuração de fluxo:**

1. **Acionador:** Alarme Hyman CO → O status muda para `alerta`
2. **Condições:** (Nenhuma)
3. **Ação:**
   - Interruptor da caldeira → DESLIGADO
   - Tomada inteligente (sistema de ar fresco) → Ligado
   - Notificação push → "Alarme de monóxido de carbono, caldeira foi desligada"

## Alarme de limite excessivo de temperatura e umidade

1. **Gatilho:** Sensor de temperatura e umidade Hyman → Temperatura acima de 40°C
2. **Ação:**
   - Ar condicionado → Ativar modo de resfriamento
   - Notificação push → "A temperatura ambiente está muito alta, o ar condicionado foi ligado automaticamente"
