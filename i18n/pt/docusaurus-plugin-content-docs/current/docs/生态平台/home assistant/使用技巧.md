# Dicas de uso

## Configuração do dispositivo

- Ao emparelhar seu dispositivo Heiman Matter pela primeira vez, certifique-se de que seu roteador de borda (por exemplo, HomePod, Nest Hub, Home Assistant Green) esteja na mesma rede Thread
- Ao integrar dispositivos Zigbee via ZHA ou Zigbee2MQTT, é recomendado primeiro usar o firmware recomendado do coordenador USB (como Sonoff ZBDongle-E, Conbee II)

## Otimização automatizada

- Adicionado parâmetro `for:` (por exemplo, 3 segundos) à automação do alarme de fumaça para evitar alarmes falsos instantâneos
- Use `condição: estado` para combinar com o modo de segurança para evitar notificações desnecessárias no estado armado
- Use a função `group` do Home Assistant para agrupar sensores Heiman por área para facilitar a operação em lote

## Solução de problemas

- Quando o dispositivo estiver offline, primeiro verifique se o canal da rede Zigbee está em conflito com WiFi 2,4 GHz
- Quando o dispositivo Matter não puder ser descoberto, reinicie o roteador de borda e adicione novamente o dispositivo
