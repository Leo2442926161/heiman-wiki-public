# Exemplo de automação

## Alarme de fumaça aciona luzes em toda a casa

```yaml
alias: "Alarme de fumaça - luzes e notificações em toda a casa"
gatilhos:
  - gatilho: estado
    entidade_id: binário_sensor.heiman_smoke_alarm
    para: "ligado"
ações:
  - ação: light.turn_on
    alvo:
      id_da_entidade: todos
  - ação: notify.mobile_app_phone
    dados:
      mensagem: "⚠️ Alarme de fumaça acionado! Verifique imediatamente!"
      título: "Alarme de fumaça"
```

## Ligação de alarme de CO

```yaml
alias: "Alarme de CO - desligue o gás e abra as janelas"
gatilhos:
  - gatilho: estado
    entidade_id: binário_sensor.heiman_co_alarm
    para: "ligado"
ações:
  - ação: switch.turn_off
    id_da_entidade: switch.gas_fireplace
  - ação: cover.open
    entidade_id: cover.living_room_window
  - ação: notify.persistent_notification
    dados:
      mensagem: "O alarme de monóxido de carbono disparou, a válvula de gás foi fechada e as janelas foram abertas"
```

## Ligação do detector de imersão em água

```yaml
alias: "Detecção de inundação - Fechar válvula de água"
gatilhos:
  - gatilho: estado
    entidade_id: binário_sensor.heiman_water_leak
    para: "ligado"
ações:
  - ação: switch.turn_off
    entidade_id: switch.main_water_valve
  - ação: notify.mobile_app_phone
    dados:
      mensagem: "💧 Vazamento de água detectado! A válvula principal de água foi fechada!"
```

## Armando fora de casa - o sensor do corpo humano aciona o alarme

```yaml
alias: "Sair do modo inicial - alarme do sensor do corpo humano"
gatilhos:
  - gatilho: estado
    id_da_entidade: sensor_binário.heiman_motion_sensor
    para: "ligado"
condições:
  - condição: estado
    entidade_id: alarm_control_panel.home_alarm
    estado: armado_away
ações:
  - ação: alarm_control_panel.trigger
    entidade_id: alarm_control_panel.home_alarm
```
