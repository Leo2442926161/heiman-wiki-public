#Configuração do painel

## Painel de visão geral de segurança

Use o layout da vista de seção para organizar os equipamentos Heimann em áreas:

```yaml
tipo: seção
título: Alarme de Fumaça e CO
cartões:
  - tipo: entidade
    entidade: binário_sensor.heiman_smoke_alarm
    nome: alarme de fumaça
  - tipo: entidade
    entidade: binário_sensor.heiman_co_alarm
    nome: alarme de CO
```

## Estilo de cartão de cogumelo

```yaml
tipo: personalizado: cartão do painel de controle de alarme de cogumelo
entidade: alarm_control_panel.home_alarm
nome: sistema de segurança

tipo: personalizado: cartão de entidade cogumelo
entidade: binário_sensor.heiman_smoke_alarm
nome: alarme de fumaça
ícone: mdi:detector de fumaça
informação_primária: estado
secundário_info: última alteração
fill_container: verdadeiro
```

## Painel de sensores ambientais

```yaml
tipo: seção
título: Monitoramento Ambiental
cartões:
  - tipo: medidor
    entidade: sensor.heiman_temperature
    nome: temperatura
    unidade: "°C"
    min: 0
    máximo: 50
    gravidade:
      verde: 0
      amarelo: 30
      Vermelho: 40
  - tipo: medidor
    entidade: sensor.heiman_humidity
    nome: umidade
    unidade: "%"
    min: 0
    máximo: 100
    gravidade:
      verde: 30
      Amarelo: 60
      Vermelho: 80
```

## Painel de histórico de alarmes

```yaml
tipo: gráfico histórico
título: História do Sensor
entidades:
  -binário_sensor.heiman_smoke_alarm
  -binário_sensor.heiman_co_alarm
  -binário_sensor.heiman_water_leak
horas_para_show: 24
```

## Layout sugerido

- Parte superior: status do sistema de segurança + cartão de status do dispositivo de alarme
- Central: Monitoramento ambiental (temperatura, umidade, qualidade da água)
- Parte inferior: gráfico de tendências históricas de alarme do sensor
- Operações de atalho na barra lateral: armar/desarmar, troca de cena
