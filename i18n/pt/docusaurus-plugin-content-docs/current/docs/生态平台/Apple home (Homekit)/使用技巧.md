# Dicas de uso

## Homebridge vs Matéria

Os dispositivos Heiman podem se conectar ao Apple Home de duas maneiras:

| Método | Vantagens | Desvantagens |
|------|------|------|
| **Assunto por Tópico** | Suporte nativo, sem necessidade de ponte, baixa latência | Requer roteador de borda de thread |
| **Plug-in Homebridge** | Compatível com todos os dispositivos Zigbee | Requer execução adicional do serviço Homebridge |

## Construção de rede de threads

1. É necessário pelo menos um roteador de borda Thread (Apple HomePod mini, Apple TV 4K 3ª geração)
2. Os dispositivos Hyman Matter ingressarão automaticamente na rede Thread em um raio de 10 metros
3. Use **Nam* ou **Eve App** para visualizar a topologia da rede Thread

## Otimização automatizada

- A automação da categoria "Segurança" do Apple Home pode ser diretamente vinculada ao status de segurança
- Use o comando "Transform" para fazer as luzes mudarem suavemente em vez de mudarem instantaneamente
- Crie ligação entre plataformas através do aplicativo Shorcuts (como alarme de fumaça → enviar SMS)

## Perguntas frequentes

- Quando o dispositivo Matter exibir "Não está respondendo", reinicie o Apple TV/HomePod para recuperar.
