# Exemplo de automação

## Alarme de segurança de detecção de fumaça

**Configuração automatizada do HomeKit:**

1. **Condição de disparo:** Alarme de fumaça Hyman → fumaça detectada
2. **Configurações de segurança:** Mude o status do sistema de segurança para "Alarme"
3. **Executar ação:**
   - HomePod → Reproduzir áudio personalizado "Fumaça detectada, preste atenção à segurança"
   - Todas as luzes inteligentes → acendem e piscam
   - Notificações push para todos os membros da família

## Em casa/fora de casa

**Comando de atalho de atalhos: "Leave Home Security Check"**

```
quando a última pessoa sai de casa
  Configurar sistema de segurança → armar ao sair de casa
  Configurar o sensor de porta Heiman → Ativar notificações
  Configure o sensor corporal Heiman → ligue o alarme
  Desligue todas as luzes
  Configurações Termostato → Modo de economia de energia
  Enviar notificação "🏠 Modo fora de casa ativado"
```

## Lembrete de anormalidade de temperatura

**Condição de disparo:** Sensor de temperatura e umidade Hyman → Temperatura > 35°C

**Executar ação:**
- HomePod → A temperatura ambiente da transmissão de voz excede o limite
- Notificações push para iPhone
- Ligar automaticamente o ar condicionado (se o HomeBridge estiver configurado)
