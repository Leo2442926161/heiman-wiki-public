# Dicas de uso

## Driver personalizado

Os dispositivos Heimann são otimizados para compatibilidade no Hubitat com os seguintes drivers:

| Tipo de dispositivo | Driver recomendado |
|----------|----------|
| Alarme de fumaça/CO | Sensor genérico de fumaça/CO Zigbee |
| Sensor infravermelho do corpo humano | Sensor de movimento Heiman (terceiros) |
| Sensor Magnético de Porta | Sensor de contato Zigbee genérico |
| Sensor de vazamento de água | Sensor genérico de vazamento de água Zigbee |
| Sensor de temperatura e umidade | Sensor de temperatura/umidade Heiman |

## Otimização Zigbee

- Os modelos Hubitat C-8 e superiores usam chip EFR32, que tem a melhor compatibilidade
- Visualize e selecione o canal com menos interferência em Configurações → Zigbee
- Use ferramentas de detecção Zigbee, como Ubisys, para diagnosticar chamadas perdidas

## Solicitações de regras

- Recomenda-se adicionar a condição "Atraso anti-falso alarme" (0,5-2 segundos) no Rule Machine
- Ative notificações de "verificação de integridade" para regras importantes para garantir que elas continuem em execução
- Organize automação complexa no modo "sub-regra" para fácil depuração e manutenção

## Coexistência Z-Wave

Se você quiser usar dispositivos Z-Wave ao mesmo tempo, certifique-se de manter o espaçamento de frequência entre os canais Zigbee e Z-Wave.
