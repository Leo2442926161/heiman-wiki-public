# Exemplo de automação

## Alarme de fumaça de máquina de regras

**Nome da regra:** Ligação de segurança do alarme de fumaça

**Configuração:**
```
Gatilho: Hyman Smoke Alarm torna-se ativo
Condições: Modo! = Desarmado
Ação:
  - Acenda todas as luzes
  - Enviar notificação push "⚠️ Alarme de fumaça!"
  - Ligue a sirene (se disponível)
  - Defina o modo para "Emergência"
```

## Regra de monitoramento de temperatura da máquina

**Nome da regra:** Proteção contra altas temperaturas

**Configuração:**
```
Gatilho: Sensor de temperatura Heimann > 38°C
Condição: Status do ar condicionado = desligado
Ação:
  - Ligar o ar condicionado, modo de refrigeração, 24°C
  - Enviar notificação "A temperatura ambiente está muito alta e o ar condicionado foi ligado"
```

## Máquina de regras para detecção de vazamento de água

**Nome da regra:** Tratamento de emergência de vazamento de água

**Configuração:**
```
Gatilho: O detector de imersão em água Hyman fica molhado
Condição: Status da válvula de água = aberta
Ação:
  - Feche a válvula principal de água
  - Envie notificações push para todos os telefones celulares
  - Enviar notificação por SMS (opcional)
```
