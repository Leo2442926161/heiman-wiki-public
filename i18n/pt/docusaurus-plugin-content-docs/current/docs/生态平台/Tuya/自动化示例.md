# Exemplo de automação

## Cena de alarme de fumaça

**Nome da cena:** "Ligação do alarme de fumaça"

**Condição de disparo:** Alarme de fumaça Hyman → Fumaça detectada

**Executar ação:**
- Tomada inteligente → aberta (conectar exaustor)
- Luzes → acender e piscar
- Notificações push para celular
- Alto-falante → Reproduzir som de alarme

## Cena de saída de casa com um clique (Toque para Executar)

**Nome da cena:** "Arranjo depois de sair de casa"

**Execução manual:** Toque ou acionamento por voz

**Executar ação:**
- Sensor de porta Haiman → ligar o alarme
- Hyman Human Body Infrared → Ativar detecção
- Detector de imersão em água Hyman → ativar a detecção
- Todas as luzes → apagadas
- Bloqueio inteligente → Bloquear

## Controle automático de temperatura e umidade

**Tipo de automação:** Disparo de condição automático

**Condição de disparo:** Sensor de temperatura e umidade Hyman Umidade > 75%

**Executar ação:**
- Tomada inteligente (desumidificador) → Ligado
- Notificação push: "A umidade está muito alta, o desumidificador foi ligado"

**Condição de disparo:** Sensor de temperatura e umidade Hyman Umidade < 50%

**Executar ação:**
- Tomada inteligente (desumidificador) → Fechar
