# Exemplo de automação

## Ligação de alarme de fumaça

```yaml
alias: "Ligação de alarme de fumaça ZHA"
gatilhos:
  - gatilho: estado
    entidade_id: binário_sensor.heiman_smoke_zha
    para: "ligado"
ações:
  - ação: light.turn_on
    alvo:
      area_id: sala_de_estar
  - ação: tts.speak
    alvo:
      entidade_id: media_player.speaker
    dados:
      mensagem: "Alarme de fumaça acionado!"
```

## Automação de emparelhamento de dispositivos

```yaml
alias: "Lembrete de emparelhamento de novo dispositivo"
gatilhos:
  - gatilho: zha.event
    subtipo: heiman
    comando: device_joined
ações:
  - ação: notify.persistent_notification
    dados:
      mensagem: "O novo dispositivo Haiman ingressou na rede ZHA, vá para o nome e área de configuração"
```

## Detecção de imersão em água

```yaml
alias: "Teste de inundação ZHA"
gatilhos:
  - gatilho: estado
    entidade_id: binário_sensor.heiman_water_leak_zha
    para: "ligado"
ações:
  - ação: switch.turn_off
    entidade_id: switch.main_water_valve
  - ação: notify.mobile_app_phone
    dados:
      mensagem: "💧 Inundação detectada!"
```
