# Dicas de uso

## Emparelhamento de dispositivos

1. Digite "Configurações → Dispositivos e Serviços → ZHA → Configuração" no Home Assistant
2. Clique em "Adicionar dispositivo" e inicie o modo de emparelhamento do dispositivo Heiman
3. Se o emparelhamento falhar, aproxime o dispositivo do coordenador USB e tente novamente dentro de 3 metros

## Coordenador USB preferido

| Modelo | Chip | Índice de recomendação |
|------|------|----------|
| Sonoff ZBDongle-E | EFR32MG21 | ⭐⭐⭐⭐⭐ |
| Conbee II | 2652 | ⭐⭐⭐⭐ |
| Elelabs Zigbee USB | 2652P | ⭐⭐⭐⭐ |

## Coordenação de canais

O canal padrão do ZHA é 15. Se houver interferência grave com WiFi 2,4 GHz:
- Use o `Silicon Labs Flasher Utility` para modificar o canal
- Canais comumente recomendados: 15, 20, 25 (evite WiFi 1/6/11)

## Otimização de rede

- Adicione um dispositivo de roteamento Zigbee (como um soquete inteligente) a cada 3-4 dispositivos Heiman para expandir a rede
- Verifique a qualidade da conexão do dispositivo com a ferramenta de visualização de rede ZHA
- Verifique regularmente se há atualizações de firmware do dispositivo para manter a compatibilidade ideal
