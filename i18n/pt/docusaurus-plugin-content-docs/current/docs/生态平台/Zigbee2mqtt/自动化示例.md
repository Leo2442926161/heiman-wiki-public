# Exemplo de automação

## Status do dispositivo de assinatura MQTT

```bash
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
```

Exemplo de saída:
```
zigbee2mqtt/heiman_smoke {"smoke":true,"bateria":100,"qualidade do link":120}
```
