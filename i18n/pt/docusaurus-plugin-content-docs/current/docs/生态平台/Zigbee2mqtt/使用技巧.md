# Dicas de uso

## Seleção de canal

Os dispositivos Hyman Zigbee usam o canal 15 por padrão. É recomendado especificar manualmente em `configuration.yaml`:

```yaml
avançado:
  canal: 15
```

Evite a sobreposição com o canal WiFi 2,4 GHz 1/6/11. Use um analisador WiFi para escanear e selecionar o canal Zigbee com menos interferência.

## Backup de chave de rede

**Certifique-se de fazer backup de `coordinator_backup.txt` e `secret.yaml`: **

```bash
dados cp/coordinator_backup.txt dados/coordinator_backup.bak
dados cp/secret.yaml dados/secret.yaml.bak
```

Todos os dispositivos emparelhados precisam se conectar novamente após perderem a chave.

## Otimizações específicas do dispositivo

- Os dispositivos Heiman relatam a carga da bateria a cada 60 minutos por padrão, e o intervalo pode ser reduzido por meio da configuração de "relatórios"
- Adicionada configuração `homeassistant: true` para gerar entidades automaticamente sem escrever manualmente a configuração de descoberta MQTT
- Use `friendly_name` para definir um nome legível para o dispositivo Heiman

## Atualizar firmware

Alguns dispositivos Heiman Zigbee suportam atualizações OTA, e a opção `ota` precisa ser ativada em `configuration.yaml`.
