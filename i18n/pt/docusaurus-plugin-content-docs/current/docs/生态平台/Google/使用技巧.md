# Dicas de uso

## Sincronização de dispositivos

- Depois que o dispositivo Heiman for conectado ao Google Home por meio do Matter, ele será sincronizado automaticamente com o aplicativo Google Home
- Se o dispositivo não for exibido, clique em "Configurações" → "Dispositivos" → "Adicionar dispositivo" → "Dispositivos compatíveis com Matter" no aplicativo Google Home
- Se o código QR do Matter for perdido, ele poderá ser adicionado manualmente usando o código de configuração de 11 dígitos no seu dispositivo

## Otimização de rotina

- Adicionado "Anunciar o clima e confirmar o status de armamento" à rotina "Sair de casa"
- Use a função "Voice Broadcast" para notificar toda a casa quando ocorrer um alarme de fumaça
- Use a ação "Iniciar/Parar" para controlar a ativação e desativação do status de detecção do sensor Heiman

## Gerenciamento de localização de dispositivos

- Atribua salas específicas aos sensores Heimann (como "Alarme de fumaça na cozinha") para oferecer suporte a um controle de voz mais preciso
- Incorpore vários dispositivos Heiman no "Grupo de dispositivos domésticos" para ativar e desativar totalmente com um clique

## Requisitos de rede

- Dispositivos Matter over Thread exigem Google Nest Hub (2ª geração) ou Nest Hub Max como roteador de borda
- Dispositivos Zigbee se conectam via Google Nest Connect ou Hub compatível
