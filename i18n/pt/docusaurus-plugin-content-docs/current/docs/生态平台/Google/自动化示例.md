# Exemplo de automação

## Rotina "estou fora"

**Acionador:** diga "Estou fora" no Google Home

**Executar ação:**
- Sensor magnético da porta Heiman → braço
- Sensor infravermelho de corpo humano Heiman → Ativar detecção de partida de casa
- Desligue todas as luzes e eletrodomésticos
- Termostato → Definir para temperatura de economia de energia
- Google Nest Hub → Mostrar cartão de confirmação de saída de casa
- Resposta de voz: "O modo sair de casa foi ativado, tenha uma viagem segura"

## Transmissão "Detecção de fumaça"

**Acionador:** Alarme de fumaça Hyman → Fumaça detectada

**Executar ação:**
- Alto-falante Google Nest → Transmissão para toda a casa "⚠️ Fumaça detectada! Verifique imediatamente!"
- Todas as luzes inteligentes → Ligadas (por meio do conjunto de luzes inteligentes do Google Home)
- Notificações push para celulares de membros da família
- Câmera → Iniciar gravação

## Modo hora de dormir

**Acionador:** diga "Boa noite" ao Google Home

**Executar ação:**
- Sensor de porta Heiman → Ligue o alarme de porta aberta
- Infravermelho do corpo humano Heiman → ativar a detecção noturna
- Luzes → apagam gradualmente
- Resposta de voz: “Boa noite, sistema de segurança está ligado”
