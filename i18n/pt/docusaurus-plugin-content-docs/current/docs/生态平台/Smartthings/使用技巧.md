# Dicas de uso

## Compatibilidade de hub

Os dispositivos Hyman Zigbee são compatíveis com **SmartThings Hub v2** e **Hub v3 (Aeotec)**.

| Modelo Hub | Compatibilidade | Descrição |
|--------|--------|------|
| Hub v2 (STH-ETH-250) | ✅ | Requer atualização de firmware para o mais recente |
| Hub v3 (Aeotec) | ✅ | Suporte nativo, recomendado |
| Estação Samsung SmartThings | ⚠️ | Suporta apenas alguns dispositivos Zigbee |

## Solicitações de emparelhamento de dispositivos

- Clique em "+" → "Dispositivo" → "Marca" → "Hyman" no aplicativo SmartThings
- Dispositivos que não são reconhecidos automaticamente podem ser pesquisados manualmente através do "Scan Zigbee"
- Recomenda-se manter a distância entre o dispositivo e o Hub dentro de 5 metros para o primeiro emparelhamento

## Otimização da bateria

O intervalo de relatório padrão do sensor Heiman é maior. Se for necessário desempenho em tempo real, o “intervalo de detecção” pode ser ajustado para 5 minutos na página do dispositivo.

## Práticas recomendadas de automação

- Use o recurso "Modos" do SmartThings para diferenciar entre estados "em casa/fora de casa"
- Crie **dois conjuntos** de automação para sensores principais (fumaça, CO) para redundância
