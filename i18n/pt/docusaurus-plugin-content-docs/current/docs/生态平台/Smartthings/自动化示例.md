# Exemplo de automação

## Modo de dormir "Boa noite"

**Condição de disparo:** Sensor magnético da porta Haiman → Fechar (indicando que a porta do quarto foi fechada)

**Executar ação:**
- Apague todas as luzes da sala
- Bloqueio inteligente → Bloquear
- Sistema de segurança → armar à noite
- Notificação push: “Boa noite, o sistema está armado”

## Modo "Ausente"

**Condição de gatilho:** Sensor infravermelho do corpo humano Hyman → 15 minutos sem movimento

**Executar ação:**
- Todas as luzes apagadas
- Termostato → Modo de economia de energia
- Sistema de segurança → armar fora de casa
- Sensor de porta → ligar o alarme
- Notificação push: "O modo fora de casa está ativado, os sensores de portas e janelas estão ativados"

## Modo de emergência de alarme de fumaça

**Condição de disparo:** Alarme de fumaça Hyman → Fumaça detectada

**Executar ação:**
- Alto-falante inteligente → reproduzir som de alarme
- Todas as luzes inteligentes → piscando em vermelho
- Notificações push para todos os membros da família
- Se conectado ao SmartThings Hub → acionar a sirene
