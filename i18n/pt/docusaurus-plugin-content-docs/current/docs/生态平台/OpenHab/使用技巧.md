# Dicas de uso

## Configuração da coisa

Adicione o dispositivo Heiman por meio da ligação MQTT:

```java
Coisa mqtt:topic:heiman_smoke "Alarme de fumaça Heiman" @ "sala de estar" {
    Canais:
        Digite binário: smoke "estado de fumaça" [ stateTopic="zigbee2mqtt/heiman_smoke", transformaçãoPattern="JSONPATH:$.smoke" ]
        Digite o número: bateria "Energia da bateria" [ stateTopic="zigbee2mqtt/heiman_smoke", transformPattern="JSONPATH:$.battery" ]
}
```

## Definição do item

```java
Entre em contato com Smoke_Alarm "Alarme de fumaça [MAP(smoke.map):%s]" { channel="mqtt:topic:heiman_smoke:smoke" }
Número Smoke_Battery "Bateria do alarme de fumaça [%d%%]" { channel="mqtt:topic:heiman_smoke:battery" }
```

## Seleção de coordenadas

Recomenda-se usar um coordenador USB com chip **CC2531** ou **CC2652** e garantir que a versão do firmware seja a mais recente.

## Dicas de depuração

- Visualize o status `online/offline` do Thing no Paper UI
- Use `openhab-cli info` para verificar se as ligações estão carregadas corretamente
- Pesquise a palavra-chave `heiman` no log para solucionar problemas de comunicação do dispositivo
