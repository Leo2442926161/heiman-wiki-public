# Exemplo de automação

## Regras de alarme de fumaça

```javascript
regra "Alarme de fumaça acionado"
quando
    Item heiman_smoke_alarm alterado para ON
então
    sendCommand(Livingroom_Light,ON)
    sendNotification("admin", "⚠️ Alarme de fumaça acionado!")
    postUpdate(Alarm_State, "smoke_alert")
fim
```

## Regras do sensor de temperatura

```javascript
regra "Relatório de temperatura e umidade"
quando
    Item heiman_temperature recebeu atualização
então
    val temp = (heiman_temperature.state como número).floatValue
    val hum = (heiman_humidity.state como número).floatValue
    if (temperatura> 40) {
        sendCommand(Ar Condicionado, LIGADO)
        sendNotification("admin", "A temperatura está muito alta (" + temp + "°C), o ar condicionado foi ligado")
    }
    se (hum > 80) {
        sendNotification("admin", "A umidade está muito alta (" + hum + "%), preste atenção à desumidificação")
    }
fim
```

## Regras de detecção de inundação

```javascript
regra "Ligação de detecção de imersão em água"
quando
    Item heiman_water_leak alterado para ON
então
    sendCommand(Água_Válvula, DESLIGADO)
    sendBroadcastNotification("💧 Vazamento de água detectado! A válvula de água foi fechada!")
fim
```
