# Exemplo de automação

## Integração Alexa Guard (EUA)

**Pré-requisito:** Assine o Alexa Guard Plus (disponível nos EUA)

**Processo de configuração:**
1. Ligue o Guard Plus no aplicativo Alexa
2. Adicione o alarme Hyman Smoke/CO como um "dispositivo doméstico inteligente"
3. Ative a resposta inteligente "Detecção de alarme de fumaça"

**Resposta automática:**
- Alarme de fumaça acionado → Alexa disca automaticamente para contatos de emergência
- Gatilho de alarme de CO → Aviso de voz Alexa para toda a casa, ventilação automática
- Portas e janelas abrem de forma anormal → Notificação de alarme push

## Rotina de detecção de fumaça

**Acionador:** Alarme de fumaça Hyman → Fumaça detectada

**Executar ação:**
- Anúncio de voz Alexa: "⚠️ Fumaça detectada, preste atenção à segurança"
- Todos os dispositivos Echo → Reproduzir tom de alarme
- Conjunto de luz inteligente → tudo aceso
- Notificações push para celular

## Rotina "Saindo de Casa"

**Gatilho:** Comando de voz "Alexa, estou saindo"

**Executar ação:**
- Sensor magnético da porta Heiman → ativar o armar
- Heiman Body Sensor → Ativar detecção de movimento
- Detector de imersão em água Hyman → ativar a detecção
- Todas as luzes → apagadas
- Alexa respondeu: “O sistema de segurança está ligado, tenha uma boa viagem”
