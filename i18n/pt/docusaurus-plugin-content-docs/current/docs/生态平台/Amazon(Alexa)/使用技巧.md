# Dicas de uso

## Descoberta de dispositivos

- Assim que o dispositivo Heiman Matter for ligado, ele será descoberto automaticamente pelo aplicativo Alexa, normalmente em 30 segundos
- Se não for descoberto automaticamente: Entre no Alexa App → Dispositivo → "+" → "Adicionar dispositivo" → "Matéria"
- Use o código de configuração do Matter ou o código QR na parte inferior do dispositivo para concluir o emparelhamento

## Acesso ao dispositivo Zigbee (Echo Plus / Echo Show)

- Dispositivos Echo com Zigbee Hub podem emparelhar diretamente com sensores Heiman Zigbee
- Modo de emparelhamento: Alexa App → Dispositivo → "+" → "Zigbee" → "Pesquisar dispositivo"
- Os dispositivos Heimann são compatíveis com Alexa com coordenador Zigbee integrado

## Dicas de rotina

- Use "ações personalizadas" para transmitir comandos de voz mais detalhados
- Adicione "aguarde 5 segundos" à rotina de emergência antes de executar ações subsequentes para evitar sobreposição de voz
- Aproveite o recurso "Detecção de som" para dar ao Alexa respostas adicionais aos alarmes agudos do Heyman Alarm

## Alexa Guard NOTA

Alexa Guard Plus está atualmente disponível apenas nos Estados Unidos. Os usuários na China podem simular algumas funções através da Rotina.
