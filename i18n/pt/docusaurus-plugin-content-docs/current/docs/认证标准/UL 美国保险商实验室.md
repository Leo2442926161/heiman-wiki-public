# Laboratórios de Subscritores UL

UL (Underwriters Laboratories) é a organização de desenvolvimento e certificação de padrões de segurança de produtos com maior autoridade nos Estados Unidos. Os principais padrões UL envolvidos nos produtos Heiman são os seguintes:

## UL 217 — Padrão de alarme de fumaça (EUA)

| Projeto | Descrição |
|------|------|
| Âmbito de aplicação | Alarmes de fumaça residenciais (mercado dos EUA) |
| Requisitos de detecção | Detecção de fumaça fotoelétrica/ionização, alarme sonoro e luminoso, acima de 85 dB |
| Mercados aplicáveis ​​| Estados Unidos, Canadá |
| Produtos Heiman | HS8SA (versão dos EUA) |
| Significado da certificação | Atender às regulamentações contra incêndio de vários estados dos Estados Unidos é uma condição necessária para o acesso ao mercado na América do Norte |

**Diferenças de outras normas**: Os cenários de teste da UL 217 incluem cenários típicos de incêndio doméstico americano, como incêndios repentinos e incêndios de queima lenta, e são ligeiramente diferentes dos métodos de teste da EN 14604.

## UL 2034 — Padrão de alarme de monóxido de carbono (EUA)

| Projeto | Descrição |
|------|------|
| Âmbito de aplicação | Alarme doméstico de CO (mercado dos EUA) |
| Requisitos de detecção | Sensor eletroquímico, algoritmo de integração concentração × tempo |
| Mercados aplicáveis ​​| Estados Unidos, Canadá |
| Produtos Hayman | HS-720ES-M (versão dos EUA) |
| Significado da certificação | A maioria dos estados dos Estados Unidos exige a instalação de alarmes de CO |

## As principais diferenças entre a certificação UL e a certificação EN

| Aspectos | UL | PT |
|------|-----|-----|
| Padrões de alarme de fumaça | UL 217 | EN 14604 |
| Padrões de alarme de CO | UL 2034 | EN 50291 |
| Método de detecção | Ensaio específico da curva de incêndio | Diferentes testes de tipo de fogo |
| Tensão aplicável | 120V / bateria | 230V / bateria |
