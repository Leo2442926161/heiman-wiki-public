# EN Norma Europeia

EN (Norma Europeia, Norma Europeia) é uma norma técnica desenvolvida pelo Comité Europeu de Normalização (CEN) e pelo Comité Europeu de Normalização Eletrotécnica (CENELEC). As principais normas EN envolvidas nos produtos Heiman são as seguintes:

## EN 14604 — Norma para alarmes de fumaça

| Projeto | Descrição |
|------|------|
| Âmbito de aplicação | Alarme de fumaça independente |
| Requisitos de detecção | Detecção fotoelétrica de fumaça, alarme sonoro e luminoso, alerta de bateria fraca |
| Mercados aplicáveis ​​| Estados-Membros da UE, Reino Unido, Suíça, Noruega, etc. |
| Produtos Heiman | HS1SA-E, HS3SA, HS1SA-M, série S1, etc. |
| Significado da certificação | Cumprir os requisitos da regulamentação da UE contra incêndios em edifícios é uma condição necessária para o acesso ao mercado europeu |

**Itens de teste**: tempo de resposta, nível de pressão sonora (≥85 dB/3m), função mudo, duração da bateria, ciclo de temperatura, durabilidade de umidade, resistência à interferência de radiofrequência, etc.

## EN 50291 — Padrão de alarme de monóxido de carbono

| Projeto | Descrição |
|------|------|
| Âmbito de aplicação | Alarme doméstico de monóxido de carbono (CO) |
| Requisitos de teste | Sensor eletroquímico de CO, alarme de classificação de concentração |
| Mercados aplicáveis ​​| Estados-membros da UE, Reino Unido |
| Produtos Heiman | HS1CA-E, série C1, HS1CA-M, etc. |
| Significado da certificação | Para garantir a fiabilidade da deteção de fugas de CO, a instalação é obrigatória em alguns países |
