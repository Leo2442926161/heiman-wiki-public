# Gerenciamento de Versões de Documentos

A Wiki da Heiman Technology suporta gerenciamento de documentação em múltiplas versões, garantindo acesso à documentação técnica compatível com a versão do firmware do seu produto.

## Comparação de Versões

| Versão do Documento | Firmware Compatível | Data de Lançamento |
|--------------------|--------------------|-------------------|
| 2.0                | Série Matter        | 2025-06           |
| 1.0                | Série Zigbee        | 2024-01           |

## Visualizando Diferentes Versões

Use o menu suspenso de versão na barra de navegação superior para alternar entre diferentes versões de documentos.

## Atualizações de Versão

Atualizamos continuamente a documentação para acompanhar as iterações dos produtos. Verifique regularmente a versão mais recente.
