# Gestión de Versiones de Documentos

La Wiki de Heiman Technology admite la gestión de documentación multiversión, garantizando el acceso a la documentación técnica que coincida con la versión del firmware de su producto.

## Comparación de Versiones

| Versión de Documento | Firmware Compatible | Fecha de Publicación |
|------------|-------------------|-------------|
| 2.0        | Serie Matter      | 2025-06     |
| 1.0        | Serie Zigbee      | 2024-01     |

## Visualización de Diferentes Versiones

Use el menú desplegable de versiones en la barra de navegación superior para cambiar entre diferentes versiones de documentos.

## Actualizaciones de Versiones

Actualizamos continuamente la documentación para que coincida con las iteraciones de los productos. Verifique regularmente para obtener la versión más reciente.
