# Soporte Multilingüe

La Wiki de Heiman Technology admite documentación multilingüe para socios y clientes globales.

Los productos Heiman admiten interfaces de aplicación multilingüe y avisos de voz en dispositivos, que incluyen chino, inglés, japonés, coreano, español y más.
