# Comparación de modelos

## Comparación de varios modelos.

| Modelo | Protocolo | Vídeo | Fuente de alimentación | Características | Sugerencias de juego |
|------|------|------|------|------|----------|
| Timbre inteligente | Zigbee + Wi-Fi | 1080P opcional | Batería/red | Timbre + seguridad dos en uno | Notificación de timbre de visitante en el teléfono móvil, intercomunicador visual remoto, grabación de detección de movimiento |

## Jugabilidad de escena recomendada

| Escenarios | Modelos recomendados | Soluciones de enlace |
|------|----------|----------|
| Gestión de entrega urgente/para llevar | Timbre inteligente | Tocar el timbre → notificación del teléfono móvil → confirmación remota del intercomunicador → desbloqueo remoto (cerradura inteligente vinculada) |
| Monitoreo de seguridad | Timbre inteligente | Detección de puerta fija → Vídeo + alarma de empuje, sensor/iluminación de puerta vinculados |
| Vinculación de visitantes | Timbre inteligente | Toca el timbre → Pantalla interior/pantalla de altavoz → Enciende las luces → Desbloquea enlace |
