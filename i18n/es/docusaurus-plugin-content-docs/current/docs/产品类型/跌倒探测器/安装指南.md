# Guía de instalación

## Método de uso

- **Clip para cinturón**: Clip en el cinturón o en la cintura
- **Lanyard**: colgar alrededor del cuello y colgar en el pecho

## Instrucciones de uso

1. Instale la batería
2. Elija el método de uso (clip para cinturón o cordón)
3. Úsalo en tu cuerpo
4. Emparejamiento a través de puerta de enlace
5. Prueba de simulación:
   - Párate y luego rápidamente agáchate o acuéstate en el suelo.
   - El sensor debería detectar una caída y activar la alarma en cuestión de segundos.
   - Presione el botón de socorro para probar la alarma manual.

## Notas

- Se recomienda usarlo durante todo el día (se puede quitar para cargar/reemplazar la batería cuando se ducha)
- El algoritmo de IA se utiliza para distinguir las caídas de las actividades diarias (sentarse, agacharse), con una baja tasa de falsas alarmas.
- Alarma automática después de detectar una caída, o puedes presionar el botón manualmente para pedir ayuda
- Se puede vincular con el botón de emergencia para lograr una doble protección.
