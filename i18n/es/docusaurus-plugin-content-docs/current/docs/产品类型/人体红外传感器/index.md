# Sensor infrarrojo del cuerpo humano

El sensor de infrarrojos del cuerpo humano (PIR) detecta el movimiento detectando la radiación de calor infrarroja emitida por el cuerpo humano y es la solución de detección de presencia más utilizada en hogares inteligentes.

## Serie de productos

| Serie | Métodos de comunicación | Características |
|------|----------|------|
| HS1MS | Zigbee | Modelo básico, ángulo de detección de 110° |
| HS3MS | Zigbee | Modelo mejorado, instalación en techo de 360° |
| Serie M1 | Materia / Zigbee | Admite materia sobre hilo, opcional 110° o 360° |
| HS8MLS | Zigbee | Modelo de larga distancia |
| HS8MIS | Zigbee | Cuerpo humano + sensor compuesto de iluminación |

## Funciones principales

- Detección humana móvil
- Detección de iluminación (serie M1/HS8MIS)
- Distancia de detección ≤7m (25°C)
- Método de instalación: montado en la pared o en el techo.
- Duración de la batería: 3 años (Zigbee) / 1 año (Matter)

## Parámetros técnicos (serie M1)

- Ángulo de detección: 110° / 360°
- Distancia de detección: ≤7m
- Rango de iluminancia: 0~1200 Lux
- Fuente de alimentación: DC 3V (batería de litio)
- Duración de la batería: 3 años (Zigbee) / 1 año (Matter)

## Ejemplo de escenario

- La luz se enciende cuando alguien entra y se apaga cuando alguien se va
- Enlace de seguridad: presione la alarma cuando se detecte un movimiento anormal
- Armado al salir de casa: apaga automáticamente el aire acondicionado/calefacción para ahorrar energía cuando no hay nadie cerca
