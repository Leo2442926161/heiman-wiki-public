# Comparación de modelos

## Comparación de varios modelos.

| Modelo | Límite máximo del subdispositivo Zigbee | Método de conexión en red | Características | Sugerencias de juegos |
|------|------------------|----------|------|----------|
| [HS6GW](../../Protocolo inalámbrico/Zigbee/HS6GW - Puerta de enlace inteligente (Zigbee)) | 128 | Wi-Fi + Ethernet | El último modelo y admite futuras actualizaciones de Thread | El propio núcleo ecológico de Haiman, combinado con la gestión remota de la aplicación Haiman, adecuado para usuarios que no se molestan en utilizarlo |
| HS5GW | 64 | Wi-Fi + Ethernet | Alto rendimiento | Proyectos medianos y grandes, estables y confiables |
| HS3GW | 32 | Wi-Fi | Mejorado | Suficiente para viviendas pequeñas y medianas |
| HS1GW | 16 | Wi-Fi | Básico | Básico, adecuado para un pequeño número de dispositivos |

## Jugabilidad de escena recomendada

| Escenario | Modelo recomendado | Descripción |
|------|----------|------|
| Ecología Heiman para toda la casa | HS6GW | Utilizado con sensores Heiman Zigbee y gestión remota con la aplicación Heiman, no es necesario construir su propio servidor |
| Usuarios de plataformas de terceros | — | Se recomienda comprar un coordinador Zigbee (como SLZB-06) directamente para usarlo con Zigbee2MQTT/ZHA para una mayor flexibilidad.
| Apartamentos pequeños y medianos | HS3GW | Límite de 32 dispositivos para satisfacer las necesidades básicas |

## Sugerencias de selección

- **Ecología propia de Haiman**: [HS6GW](../../Wireless Protocol/Zigbee/HS6GW - Smart Gateway (Zigbee)) (128 dispositivos, WiFi + Ethernet)
- **Plataforma de terceros**: Coordinador Zigbee universal recomendado + Zigbee2MQTT/ZHA
