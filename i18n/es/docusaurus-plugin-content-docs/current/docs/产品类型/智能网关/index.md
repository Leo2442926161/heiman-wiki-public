# Puerta de enlace inteligente

Smart Gateway es el centro central del sistema de hogar inteligente Haiman, responsable de conectar varios sensores Zigbee y realizar el control de enlace entre dispositivos.

## Serie de productos

| Serie | Métodos de comunicación | Características |
|------|----------|------|
| Serie HS1GW | Zigbee + Wi-Fi/Ethernet | Puerta de enlace básica |
| Serie HS3GW | Zigbee + Wi-Fi/Ethernet | Mejorado |
| Serie HS5GW | Zigbee + Wi-Fi/Ethernet | Puerta de enlace de alto rendimiento |
| Serie HS6GW | Zigbee + Wi-Fi / Ethernet + Hilo | El último modelo es compatible con Zigbee y Matter over Thread |

## Funciones principales

- Coordinador de la Red Zigbee
- Admite Wi-Fi / Ethernet para conectarse a Internet
- Descubrimiento y emparejamiento automático de dispositivos
- Enlace de escena local (algunos modelos)
- Acceso remoto (a través de Heiman Cloud Platform)
- Admite hasta 128 subdispositivos (según el modelo)

## Escenarios aplicables

- El punto de conexión central de la inteligencia integral de Heiman
- Cooperar con la aplicación Haiman para realizar monitoreo remoto
- Gestión local centralizada de pequeños y medianos proyectos comerciales.

## Diferencias con otras puertas de enlace

Haiman Gateway es la entrada al ecosistema oficial. Para los usuarios que deseen utilizar una plataforma de terceros, se recomienda utilizar el coordinador universal Zigbee con Zigbee2MQTT/ZHA.
