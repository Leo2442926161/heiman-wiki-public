# Comparación de modelos

## Comparación de varios modelos.

| Modelo | Protocolo | Sensores | Fuente de alimentación | Características | Sugerencias de juegos |
|------|------|--------|------|------|----------|
| Sensor de vibración | Zigbee | Acelerómetro de 3 ejes | CR2450 | Detección de vibración + inclinación | Antipalanca para puertas y ventanas, alarma de vibración de apertura ilegal de caja fuerte/cajón, monitoreo anormal de vibración de tuberías |

## Jugabilidad de escena recomendada

| Escenarios | Modelos recomendados | Soluciones de enlace |
|------|----------|----------|
| Antipalanca de puertas y ventanas | Sensor de vibración | Detección de impacto violento y vibración de puerta/ventana → pulsar alarma + alarma sonora y luminosa + captura de cámara |
| Monitoreo seguro | Sensor de vibración | La caja fuerte se mueve o vibra → Notificación de alarma inmediata |
| Monitoreo de tuberías | Sensor de vibración | Monitorear la vibración anormal de las tuberías de agua (falla de la bomba de agua/golpe de ariete), proporcionar alerta temprana para el mantenimiento |
| Valla/Muro | Sensor de vibración | Detección de escalada de paredes/cercas, adecuada para seguridad de patios |
