# sensor de calidad del aire

Los sensores de calidad del aire (Air Quality Monitor) se utilizan para detectar diversos contaminantes en el aire interior para ayudar a mantener un entorno de vida saludable.

## Parámetros de detección

Los sensores de calidad del aire Hyman detectan los siguientes indicadores:

| Parámetros | Descripción | Estándares de Salud |
|------|------|----------|
| PM2.5 | Partículas finas | ≤ 35 µg/m³ (promedio de 24 horas) |
| PM10 | Partículas respirables | ≤ 50 µg/m³ (promedio de 24 horas) |
| CO₂ | Dióxido de carbono | ≤ 1000 ppm |
| TVOC | Compuestos orgánicos volátiles totales | ≤ 0,6 mg/m³ |
| Temperatura | Temperatura ambiente | 18-26°C |
| Humedad | Humedad relativa | 40-60 % de humedad relativa |

## Funciones principales

- Detección multiparamétrica de la calidad del aire.
- Pantalla a color (según modelo)
- Alarma sonora y luminosa por exceder el estándar.
- Registros de datos históricos.
- Purificador de aire/aire fresco vinculado

## Escenarios aplicables

- Dormitorio: controle los niveles de CO₂ y solicite abrir las ventanas para ventilar
- Habitación infantil: protege la salud respiratoria de los niños
- Oficina: Aumente la productividad y la comodidad
- Casas recientemente renovadas: monitorear la liberación de formaldehído/TVOC
