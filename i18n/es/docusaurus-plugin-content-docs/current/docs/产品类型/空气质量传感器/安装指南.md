# Guía de instalación

## Requisitos de ubicación de instalación

- Instalado a **1,2 ~ 1,8 m** de altura (zona de respiración)
- Elija una ubicación que represente la calidad promedio del aire en la habitación
- **Mantener alejado**: ventanas, puertas, rejillas de ventilación, salidas de aire acondicionado.
- **Manténgase alejado**: humos de cocina, humidificadores, aromaterapia y otras fuentes de contaminación locales

## Pasos de instalación

1. Seleccione la ubicación de instalación (sala de estar, dormitorio o pared de la oficina)
2. Asegure la base de montaje con tornillos.
3. Conéctese a la alimentación (USB o adaptador)
4. El sensor se enciende y la pantalla (si corresponde) muestra datos después de la autoprueba.
5. Emparejamiento a través de puerta de enlace
6. Confirma que los datos son normales en la APP.

## Notas

- Asegúrese de que la entrada y salida de aire del sensor no estén bloqueadas.
- Limpie la carcasa del sensor con regularidad para evitar que se obstruya con polvo.
- Colóquelo en la habitación de los niños para controlar el nivel de CO₂ y solicitarle que abra la ventana para ventilar.
- Recomendado para combinar con sistema de aire fresco o purificador de aire.
