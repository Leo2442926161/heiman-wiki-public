# Comparación de modelos

## Comparación de varios modelos.

| Modelo | Protocolo | Parámetros de detección | Fuente de alimentación | Características | Sugerencias de juego |
|------|------|----------|------|------|----------|
| Sensor de calidad del aire | Zigbee | PM2.5 / PM10 / CO₂ / TVOC / Temperatura y humedad | Red eléctrica | Multifunción en uno | Monitoree la calidad del aire interior, vincule automáticamente aire fresco/purificador + recordatorio push si excede el estándar |

## Jugabilidad de escena recomendada

| Escenarios | Modelos recomendados | Soluciones de enlace |
|------|----------|----------|
| Monitoreo de CO₂ en el dormitorio | Sensor de calidad del aire | CO₂ > 1000ppm → Recordatorio de apertura de ventana/sistema de aire fresco vinculado para ventilación, mejor calidad del sueño |
| Casa recién reformada | Sensor de calidad del aire | Monitoreo continuo de TVOC/formaldehído, extractor de aire vinculado cuando se excede el estándar, y se enviará una notificación tranquilizadora una vez que se cumpla el estándar |
| Habitación infantil | Sensor de calidad del aire | Monitoreo de PM2.5, encendido automático del purificador de aire cuando se excede el estándar para garantizar la salud respiratoria |
| Oficina | Sensor de calidad del aire | CO₂ + aire acondicionado vinculado a la temperatura y la humedad, aire fresco para mejorar la eficiencia en el trabajo |
