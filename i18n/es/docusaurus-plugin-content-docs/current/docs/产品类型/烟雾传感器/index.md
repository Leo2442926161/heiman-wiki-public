# sensor de humo

El sensor de humo Hyman utiliza **tecnología fotoeléctrica de detección de humo**, que puede detectar con sensibilidad partículas de humo visibles generadas en las primeras etapas de un incendio y lograr una alerta de incendio muy temprana.

## Funciones principales

- Detección fotoeléctrica de humo para detectar humo de quemaduras lentas y llamas abiertas.
- Alarma luminosa y sonora de 85 dB de alto nivel
- Recordatorio de batería baja
- Función de autoprueba (prueba con un clic)
- Alarma antisabotaje
-Admite silencio remoto

## Parámetros técnicos (tomando HS1SA-M como ejemplo)

- Tipo de sensor: fotoeléctrico infrarrojo
- Presión sonora de alarma: ≥85 dB (3 metros)
- Fuente de alimentación: batería CR123A
- Duración de la batería: aprox. 3 años
- Temperatura de funcionamiento: -10°C ~ +50°C
- Humedad de trabajo: ≤95% RH
