# Sensor de temperatura y humedad

Los sensores de temperatura y humedad se utilizan para monitorear la temperatura ambiente y la humedad relativa en tiempo real y son equipos básicos para el monitoreo inteligente del entorno doméstico.

## Serie de productos

| Serie | Métodos de comunicación | Características |
|------|----------|------|
| Serie H1 | Materia / Zigbee | La versión Matter over Thread tiene certificación WWHA |

## Funciones principales

- Monitoreo de temperatura y humedad en tiempo real.
- Recordatorio de batería baja
-Control de enlace de soporte (encienda el aire acondicionado cuando la temperatura sea alta, encienda el deshumidificador cuando la humedad sea alta, etc.)

## Parámetros técnicos

- Precisión de temperatura: ±0,3°C
- Precisión de la humedad: ±3% RH
- Fuente de alimentación: batería CR2
- Duración de la batería: alrededor de 2-3 años

## Ejemplo de escenario

- Aire acondicionado/calefacción vinculados para ajustar automáticamente la temperatura ambiente
- Deshumidificador/humidificador vinculado para mantener una humedad confortable
- Monitorear el ambiente de la bodega/purero.
- Control automático de extractores de cocina/baño.
- Seguimiento ambiental de invernaderos de plantación.
