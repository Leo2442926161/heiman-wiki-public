# Comparación de modelos

## Comparación de varios modelos.

| Modelo | Protocolo | Precisión | Fuente de alimentación | Características | Sugerencias de juego |
|------|------|------|------|------|----------|
| [H1-M](../../Protocolo inalámbrico/Materia/H1-M Sensor de temperatura y humedad (Materia)) | Asunto sobre Hilo | ±0,3°C/±3% HR | CR2 | Versión Matter, multiplataforma | Native Apple Home, vinculado al aire acondicionado/calefacción por suelo radiante para ajustar automáticamente la temperatura ambiente y encender automáticamente el extractor de aire cuando la humedad es alta |
| H1-Z | Zigbee | ±0,3°C/±3% HR | CR2 | Versión Zigbee | Conéctese a HA/Z2M, alarma de anomalía de temperatura (notificación de temperatura demasiado baja/alta para personas mayores) |

## Jugabilidad de escena recomendada

| Escenarios | Modelos recomendados | Soluciones de enlace |
|------|----------|----------|
| Ecología del hogar de Apple | H1-M | Acceso Native Matter, automatización HomeKit: temperatura > 28°C, aire acondicionado encendido, humedad > 70%, ventilación por extracción |
| Usuario en profundidad de HA | H1-Z o H1-M | Automatización HA: vinculación de temperatura y humedad calefacción por suelo radiante/humidificador/deshumidificador, datos históricos InfluxDB + Grafana |
| Bodega/vitrina para puros | H1-Z | Alarma de exceso de temperatura y humedad + notificación push en el teléfono móvil para proteger las colecciones |
| Vigilancia de la habitación del bebé | H1-M | Notificaciones push cuando la temperatura es demasiado alta o baja y vincula el aire acondicionado para ajustarlo a la zona de confort |
| Planta de invernadero | H1-Z | La temperatura y la humedad activan riego/ventilación/sombrilla automática |

## Sugerencias de selección

- **Materia multiplataforma**: [H1-M](../../Protocolo inalámbrico/Materia/Sensor de temperatura y humedad H1-M (Materia))
- **Sistema Zigbee existente**: H1-Z
