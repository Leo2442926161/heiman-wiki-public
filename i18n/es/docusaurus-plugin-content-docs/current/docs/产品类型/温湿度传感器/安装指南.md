# Guía de instalación

## Requisitos de ubicación de instalación

- Instalado a una altura de **1,2 ~ 1,5 m** (que representa el nivel medio del entorno)
- **Mantener alejado**: luz solar directa, puertas y ventanas, salidas de aire acondicionado, radiadores
- **Mantener alejado**: humidificadores, cocinas y otras fuentes locales de cambios de temperatura y humedad.
- Elija habitaciones que se utilicen con frecuencia (dormitorio, sala de estar, habitación del bebé)

## Pasos de instalación

1. Seleccione la ubicación de instalación (ya sea pared o escritorio)
2. Montaje en pared: fije la base con tornillos o pegamento 3M
3. Instale la batería CR2 (tenga en cuenta la polaridad)
4. El sensor se enciende automáticamente y la pantalla (si la hay) muestra la temperatura y la humedad.
5. Emparejamiento a través de puerta de enlace
6. Verifique los datos en la APLICACIÓN para confirmar que son normales.

## Notas

- Al instalarlo contra la pared, asegúrese de que la ventilación del sensor no esté bloqueada.
-La duración de la batería es de unos 2-3 años.
- Los datos de temperatura y humedad en diferentes habitaciones se pueden recopilar a través de múltiples sensores
- Se puede utilizar para acondicionadores de aire, deshumidificadores, humidificadores y otros equipos conectados.
