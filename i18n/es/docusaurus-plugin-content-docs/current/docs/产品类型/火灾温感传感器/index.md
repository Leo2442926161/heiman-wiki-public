# Sensor de temperatura de fuego

El sensor de temperatura de incendio (Detector de Calor) emite una alarma de incendio al detectar aumentos anormales en la temperatura ambiente. Es adecuado para escenas como cocinas y garajes donde los sensores de humo son propensos a generar falsas alarmas.

## Principio de funcionamiento

El sensor de temperatura utiliza un elemento térmico para detectar cambios de temperatura:

- **Alarma de temperatura fija**: activa una alarma cuando la temperatura excede un umbral establecido (generalmente 57 °C / 135 °F)
- **Alarma de diferencia de temperatura**: activa una alarma cuando la tasa de aumento de temperatura excede el valor establecido (como 8,3°C/min)

## Ámbito de aplicación

- Cocina (los sensores de humo son propensos a generar falsas alarmas debido a la cocción en esta escena)
- Garaje/taller (los gases de escape y el polvo pueden interferir con los sensores de humo)
- Sala de calderas/sala de equipos de calefacción.
- Baño/Sauna

## Funciones principales

- Detección de aumento anormal de temperatura.
- Alarma antisabotaje
- Recordatorio de batería baja

> **Nota**: El sensor de temperatura y el sensor de humo se utilizan de forma complementaria. El sensor de humo responde más rápido y el sensor de temperatura tiene una mayor capacidad antiinterferente.
