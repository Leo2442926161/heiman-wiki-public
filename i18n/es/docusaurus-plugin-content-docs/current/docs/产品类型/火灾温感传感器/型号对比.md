# Comparación de modelos

## Comparación de varios modelos.

| Modelo | Protocolo | Principio de detección | Fuente de alimentación | Características | Sugerencias de juego |
|------|------|----------|------|------|----------|
| Sensor de temperatura de incendio | Zigbee/Sub-1G | Termistor (temperatura fija + temperatura diferencial) | Batería | Interferencia antihumo | Especial para cocina/garaje, alarma por aumento anormal de temperatura, electroválvula de conexión para corte de gas |

## Jugabilidad de escena recomendada

| Escenarios | Modelos recomendados | Soluciones de enlace |
|------|----------|----------|
| Seguridad en la cocina | Sensor de temperatura | Aumento anormal de temperatura → Cierre la válvula de gas → Alarma sonora y luminosa → Notificación push en el teléfono móvil para evitar falsas alarmas de humo de cocina |
| Garaje/sala de calderas | Sensor de temperatura | Controle la temperatura en la sala de equipos para evitar la combustión espontánea y conecte el extractor de aire para que se enfríe |
| Complementario con sensor de humo | Sensor de temperatura + humo | El sensor de humo cubre toda la casa, el sensor de temperatura cubre la cocina/garaje, la alarma complementaria es más fiable |

## Sugerencias de selección

- **Solo cocina**: Sensor de temperatura (para evitar falsas alarmas de humo de cocina)
- **Funciona con sensor de humo**: el sensor de humo cubre el dormitorio/sala de estar, el sensor de temperatura cubre la cocina/garaje
