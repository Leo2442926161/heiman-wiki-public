# Comparación de modelos

## Comparación de varios modelos.

| Modelo | Protocolo | Potencia nominal | Precisión de medición | Características | Sugerencias de juegos |
|------|------|----------|----------|------|----------|
| [HS2SK](../../Protocolo inalámbrico/Zigbee/HS2SK - Enchufe inteligente (Zigbee)) | Zigbee | 16A/3680W | ±2% | Tipo de medición | Monitoreo del consumo de energía en tiempo real + interruptor remoto, apagado con un clic en modo de conexión fuera del hogar, estadísticas de consumo de energía del aire acondicionado/calentador de agua |
| [HS2SK-WiFi](../../Protocolo Inalámbrico/Wifi/HS2SK-WiFi - Toma Medidor Inteligente (WiFi)) | Wi-Fi | 16A/3680W | ±2% | Conexión directa sin puerta de enlace | No se requiere puerta de enlace adicional, control directo de la aplicación Smart Life, adecuado para usuarios que no desean comprar una puerta de enlace |

## Jugabilidad de escena recomendada

| Escenarios | Modelos recomendados | Soluciones de enlace |
|------|----------|----------|
| Estadísticas de consumo de energía | HS2SK | Zigbee está conectado a HA y vinculado con InfluxDB + Grafana para mostrar el consumo de energía diario/mensual y descubrir dispositivos de alto consumo de energía |
| Apagar al salir de casa | HS2SK | Al salir de casa, apague los electrodomésticos no esenciales como las luces del televisor, el aire acondicionado o la pecera con un solo clic para ahorrar energía de forma segura |
| Control de tiempo | HS2SK / HS2SK-WiFi | Calentador de agua/humidificador/pila de carga con interruptor de sincronización, combinado con precios de electricidad pico y valle para ahorrar facturas de electricidad |
| Usuarios sin puerta de enlace | HS2SK-WiFi | Conectado directamente al enrutador, controlado por la aplicación Smart Life, entrada de umbral cero |

## Sugerencias de selección

- **Puerta de enlace Zigbee existente**: [HS2SK](../../WirelessProtocol/Zigbee/HS2SK - Smart Socket (Zigbee))
- **No quiero comprar una puerta de enlace**: [HS2SK-WiFi](../../Wireless Protocol/Wifi/HS2SK-WiFi - Smart Metering Socket (WiFi))
