# Guía de instalación

## Requisitos de ubicación de instalación

- Enchufe directamente al tomacorriente de pared, **No conecte en serie** (No conecte a una regleta o cable de extensión)
- Asegúrese de que la corriente nominal del enchufe sea ≥ 16A
- No instalar en ambientes húmedos o de alta temperatura.

## Pasos de instalación

1. Conecte el enchufe inteligente directamente a un enchufe de pared de 220 V (o 110 V).
2. La luz indicadora se enciende después de suministrar energía.
3. Mantenga presionado el botón de cambio para ingresar al modo de emparejamiento.
4. Agregar dispositivo en la APLICACIÓN Heiman
5. Después de un emparejamiento exitoso, puedes controlar remotamente el interruptor.

## Precauciones de seguridad

- **Carga máxima**: 16A/3680W (estándar europeo), 15A/1800W (estándar estadounidense)
- **No sobrecargar**: Preste atención al límite de potencia cuando utilice aparatos eléctricos de alta potencia (aires acondicionados, calentadores)
- **NO CONECTAR EN SERIE**: No inserte enchufes inteligentes en regletas
- No cubra ni oscurezca la carcasa del enchufe.
- Compruebe el enchufe periódicamente para ver si se sobrecalienta.

## Aplicaciones típicas

- Control de sincronización de equipos de bajo consumo, como calentadores de agua, humidificadores y luces de peceras.
- Cooperar con la vinculación del sensor: la luz se enciende cuando alguien viene y se apaga cuando alguien se va.
- Estadísticas de energía: controle el consumo de energía de los electrodomésticos.
