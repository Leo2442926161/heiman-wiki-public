# Guía de instalación

## Requisitos de ubicación de instalación

- **instalación en techo**, sensor verticalmente hacia abajo
- Altura de instalación **2,5 ~ 3,5 m**
- El radio de detección es de aproximadamente **3 ~ 5 m**
- **Evitar**: Hay obstáculos metálicos delante del sensor (el metal bloqueará las ondas milimétricas)
- Mantener alejado de grandes tuberías metálicas, unidades internas de aire acondicionado, etc.

## Pasos de instalación

1. Seleccione la ubicación de instalación en el techo (lo mejor es el centro de la habitación)
2. Taladre agujeros en el techo y fije la base con tornillos de expansión.
3. Conecte el cableado (si se requiere fuente de alimentación principal) o instale la batería.
4. Gire el cuerpo del sensor y encájelo en la base.
5. Después de encender, la luz indicadora se enciende y el sensor comienza la autoprueba.
6. Ingrese al emparejamiento de la aplicación y agregue el dispositivo
7. Prueba: Ingrese al área de detección y siéntese en silencio. El sensor debe permanecer en el estado "ocupado".

## Notas

- Las ondas milimétricas pueden atravesar placas delgadas (placas de yeso, madera), pero serán bloqueadas por el metal.
-Las paredes gruesas y los techos metálicos reducirán el rango de detección.
- Se recomienda utilizar con sensor PIR para lograr una cobertura completa de la escena
