# Comparación de modelos

## Comparación de varios modelos.

| Modelo | Protocolo | Tecnología de detección | Instalación | Características | Sugerencias de juegos |
|------|------|----------|------|------|----------|
| [HS8OS](../../Protocolo inalámbrico/Zigbee/HS8OS - Sensor de presencia humana (Zigbee)) | Zigbee | Radar de ondas milimétricas | Techo | Detección humana estacionaria | Más preciso que PIR, el baño/dormitorio permanece estacionario durante mucho tiempo, resolviendo el problema de que PIR apague las luces sin que nadie |

## Jugabilidad de escena recomendada

| Escenarios | Modelos recomendados | Soluciones de enlace |
|------|----------|----------|
| Mantenga las luces encendidas en el baño | HS8OS | Las luces se encienden cuando hay gente y se apagan cuando la gente se va (detecta microrrespiración para resolver el problema del apagado de las luces PIR estacionarias) |
| Monitoreo del sueño en el dormitorio | HS8OS | Detecta si hay alguien en la cama y conecta el aire acondicionado para ajustar automáticamente la temperatura |
| Cuidado inteligente de personas mayores | HS8OS | Tiempo de espera en el baño para personas mayores → Sistema de alerta notifica a los familiares para prevenir accidentes |
| Ahorro de energía en salas de conferencias | HS8OS | Nadie llega a N minutos → Apaga luces y aire acondicionado automáticamente para ahorrar energía y reducir consumo |

## Sugerencias de selección

- **Solución para PIR ilumina los puntos débiles**: [HS8OS](../../Protocolo inalámbrico/Zigbee/HS8OS - Sensor de presencia del cuerpo humano (Zigbee)) (onda milimétrica, detección de quietud)
- **Emparejado con PIR**: PIR detecta movimiento + HS8OS detecta presencia estacionaria, uso complementario para obtener mejores resultados
