# Alarma sonora y luminosa (señal de advertencia)

La sirena inteligente se utiliza para proporcionar alarmas luminosas y con sonido fuerte. Se utiliza con varios sensores y puede utilizarse como actuador de salida del sistema de seguridad.

## Funciones principales

- Sonido de alarma de alto decibelio (≥110 dB)
- Luces estroboscópicas rojas/de color.
-Admite múltiples tonos de alarma
- Detección de manipulación
-Soporte de armado y desarmado remoto

## Escenarios aplicables

- Vinculado con sensor de humo/CO: suena en toda la casa cuando se detecta peligro
- Alarma de intrusión de seguridad: se activa cuando se abren puertas y ventanas ilegalmente
- Evacuación de emergencia: guiar a las personas para evacuar en caso de incendio.

## Enlace de escena

- **Alarma de humo + Sirena de luz y sonido**: Toda la casa hará sonar la alarma al mismo tiempo, alertando a todas las personas en la habitación.
- **Alarma con sensor de puerta + alarma sonora y luminosa**: disuade a los intrusos ilegales
- **Cooperar con la puerta de enlace**: enlace con otros dispositivos a través de la red Zigbee/Matter
