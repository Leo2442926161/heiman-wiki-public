# Guía de instalación

## Requisitos de ubicación de instalación

- **Instalado a la altura de la respiración** - aprox. **1,5 m** del suelo
- Cerca del área del dormitorio para garantizar que se pueda escuchar la alarma mientras duerme.
- **No instalar en esquinas**, mantener la circulación de aire.
- Mantener alejado de rejillas de ventilación, ventanas, ventiladores y otros lugares que afecten el flujo de aire.

## Pasos de instalación

1. Determine la ubicación de instalación (pasillo de dormitorio o sala de estar recomendados)
2. Fije la base de montaje en la pared (tornillos o cinta adhesiva 3M)
3. Inserte la batería y el sensor se encenderá automáticamente.
4. Instale el cuerpo principal en la base.
5. Autoprueba: Presione el botón de prueba, la luz de la alarma parpadeará y sonará la alarma.
6. Emparejamiento a través de puerta de enlace

## Notas

- La vida útil típica del sensor es de **7 años**, después de los cuales es necesario reemplazar todo el dispositivo
- Pruebe la funcionalidad una vez al mes.
- No bloquear la ventilación del sensor.
- Si hay una alarma, primero abra las ventanas para ventilar y luego verifique si hay alguna fuga de fuentes de CO.
