# Estándares de certificación

HS1CA-Z ha superado las siguientes certificaciones:

- **EN 50291** — Norma europea para alarmas de monóxido de carbono, consulte [Norma europea EN](../../../../Normas de certificación/Norma europea EN.md)
- **Certificación Z-Wave S2**: admite el marco de seguridad S2, consulte [Otras certificaciones](../../../Certification Standards/Other Certifications.md)
- **CE** — Declaración de conformidad de la Unión Europea
