# HS1CA-Z Alarma inteligente de monóxido de carbono (Z-Wave)

HS1CA-Z es una alarma inteligente de monóxido de carbono **Z-Wave** lanzada por Hyman. Utiliza tecnología de sensores electroquímicos y es compatible con el estándar Z-Wave Plus.

## Descripción general del producto

- **Tipo de producto**: Alarma inteligente de monóxido de carbono
- **Protocolo de comunicación**: Z-Wave Plus
- **Estándar de Certificación**: EN 50291
- **Escenario típico**: Advertencia de fuga de CO en casas, apartamentos y otros lugares

## Parámetros técnicos

| Parámetro | Valor |
|------|-----|
| Tipo de sensor | Sensor electroquímico de CO |
| Protocolo de comunicación | Z-Wave Plus (UIT-T G.9959) |
| Banda de frecuencia operativa | 868,42 MHz (UE) / 908,42 MHz (EE. UU.) |
| Rango de detección | 0~1000 ppm |
| Presión sonora de alarma | ≥85 dB (3 metros) |
| Método de suministro de energía | Funciona con pilas |
| Vida del sensor | Típico 7 años |
| Método de cifrado | S2 autenticado/S2 no autenticado |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤90% HR (sin condensación) |

## Características

- Sensor de CO electroquímico de alta precisión
- Alarma de clasificación de concentración.
- Alarma luminosa y sonora de 85 dB de alto nivel
- Recordatorio de batería baja
- Recordatorio de vencimiento de la vida útil del sensor
- Función de autoprueba
- Cifrado seguro Z-Wave S2
- Admite actualización de firmware OTA

## Certificación

- Certificado EN 50291
- Certificación Z-Wave Plus
- Certificación de seguridad S2
