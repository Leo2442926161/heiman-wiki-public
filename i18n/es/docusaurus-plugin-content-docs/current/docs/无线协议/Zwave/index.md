# Onda Z

Z-Wave es un protocolo de red Mesh inalámbrico diseñado para hogares inteligentes. Está gestionado por Z-Wave Alliance y tiene amplia penetración en los mercados europeo y americano.

## Especificaciones técnicas

| Características | Descripción |
|------|------|
| Banda de frecuencia operativa | 868,42 MHz (UE) / 908,42 MHz (EE. UU.) |
| Tasa de transferencia | Hasta 100 kbps (Serie Z-Wave 800) |
| Topología de red | Red mallada, hasta 232 nodos |
| Compatibilidad | Compatible con versiones anteriores de todos los dispositivos Z-Wave |
| Seguridad | Cifrado Z-Wave S2 |

## Hyman y Z-Wave

Hyman ofrece una variedad de sensores compatibles con Z-Wave, vendidos principalmente en los mercados europeo y americano. Las versiones Z-Wave de los dispositivos Heyman incluyen:

- Sensor de monóxido de carbono HS1CA-Z
- Sensor de humo HS1SA-Z
- Sensor de puerta HS1DS-Z

## Plataformas compatibles

- Centro de cosas inteligentes
- Elevación de Hubitat
- Hogareño Pro
- Asistente de inicio (a través de Z-Wave JS)
-OpenHAB

> **Nota**: Z-Wave tiene poca popularidad en el mercado chino y los productos Haiman Z-Wave están dirigidos principalmente a los mercados de exportación europeos y americanos.
