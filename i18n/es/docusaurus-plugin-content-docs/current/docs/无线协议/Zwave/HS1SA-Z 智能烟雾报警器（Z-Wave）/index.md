# HS1SA-Z Alarma de humo inteligente (Z-Wave)

HS1SA-Z es una alarma de humo inteligente **Z-Wave** lanzada por Hyman. Utiliza tecnología fotoeléctrica de detección de humo, es compatible con el estándar Z-Wave Plus y es adecuado para los mercados de hogares inteligentes europeos y americanos.

## Descripción general del producto

- **Tipo de producto**: Alarma de humo inteligente
- **Protocolo de comunicación**: Z-Wave Plus
- **Estándar de Certificación**: EN 14604
- **Escenario típico**: Aviso de incendio en casas, apartamentos y otros lugares

## Parámetros técnicos

| Parámetro | Valor |
|------|-----|
| Tipo de sensor | Fotoeléctrico infrarrojo |
| Protocolo de comunicación | Z-Wave Plus (UIT-T G.9959) |
| Banda de frecuencia operativa | 868,42 MHz (UE) / 908,42 MHz (EE. UU.) |
| Presión sonora de alarma | ≥85 dB (3 metros) |
| Método de suministro de energía | Batería de litio CR123A |
| Duración de la batería | Aproximadamente 2 ~ 3 años |
| Método de cifrado | S2 autenticado/S2 no autenticado |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤90% HR (sin condensación) |

## Características

- Detección fotoeléctrica de humo para detectar humo de combustión lenta y llama abierta.
- Alarma luminosa y sonora de 85 dB de alto nivel
- Recordatorio de batería baja
- Función de autoprueba
- Alarma antisabotaje
- Cifrado seguro Z-Wave S2
- Admite actualización de firmware OTA

## Certificación

- Certificación EN 14604
- Certificación Z-Wave Plus
- Certificación de seguridad S2
