# Sensor de puerta HS1DS-Z (Z-Wave)

HS1DS-Z es un sensor de puerta **Z-Wave** lanzado por Hyman. Detecta el estado de apertura y cierre de puertas y ventanas mediante interruptores de láminas magnéticos y es compatible con el estándar Z-Wave Plus.

## Descripción general del producto

- **Tipo de producto**: Sensor de puerta
- **Protocolo de comunicación**: Z-Wave Plus
- **Método de detección**: Interruptor de láminas (interruptor de láminas)
- **Escenarios típicos**: monitoreo del estado de puertas y ventanas, implementación de seguridad y vinculación inteligente

## Parámetros técnicos

| Parámetro | Valor |
|------|-----|
| Método de detección | Interruptor de láminas (interruptor de láminas) |
| Protocolo de comunicación | Z-Wave Plus (UIT-T G.9959) |
| Banda de frecuencia operativa | 868,42 MHz (UE) / 908,42 MHz (EE. UU.) |
| Método de suministro de energía | Pila de botón CR1632 |
| Duración de la batería | Aproximadamente 2 ~ 3 años |
| Espacio libre de instalación | ≤10mm |
| Método de cifrado | S2 autenticado/S2 no autenticado |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤90% HR (sin condensación) |

## Características

- Detección en tiempo real del estado de apertura/cierre de puertas y ventanas
- Alarma de manipulación (Tamper)
- Recordatorio de batería baja
-Admite vinculación de escenas (abrir puerta, encender luces, armar al salir de casa)
- Fácil de instalar (cinta adhesiva 3M o fijación con tornillos)
- Cifrado seguro Z-Wave S2
- Admite actualización de firmware OTA
