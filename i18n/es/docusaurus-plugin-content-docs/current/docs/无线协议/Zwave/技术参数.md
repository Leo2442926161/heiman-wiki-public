# Parámetros técnicos

Las siguientes son especificaciones técnicas generales para los dispositivos Hyman Z-Wave.

## Parámetros inalámbricos

| Parámetro | Valor |
|------|-----|
| Estándares de protocolo | Z-Wave Plus (UIT-T G.9959) |
| Banda de frecuencia de trabajo | 800~900 MHz (dependiendo de la región: regulación europea 868,42 MHz, regulación estadounidense 908,42 MHz) |
| Tasa de transmisión | Hasta 100 kbps |
| Topología de red | Red de malla de autorreparación (enrutamiento de origen) |
| Capacidad de nodos | Máximo 232 nodos en una sola red |
| Distancia de comunicación | Unos 30 metros en interiores (entre nodos), toda la casa Cobertura de malla |
| Método de cifrado | AES-128 (S2 autenticado/S2 no autenticado) |

## Banda de frecuencia regional

| Región | Frecuencia central |
|------|----------|
| Europa | 868,42 MHz |
| América del Norte | 908,42 MHz |
| Otras zonas | Sujeto a regulaciones locales |

## Consumo eléctrico y de energía

| Parámetro | Valor |
|------|-----|
| Método de suministro de energía | Alimentación por batería (CR123A/AA) o alimentación por red eléctrica |
| Duración de la batería | Alrededor de 2 a 3 años para sensores |
| Transmitir potencia | ≤ 1 mW (cumple con la normativa SRD) |
| Recibir sensibilidad | Aproximadamente -100 dBm |

## Parámetros ambientales

| Parámetro | Valor |
|------|-----|
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de funcionamiento | ≤ 90% HR (sin condensación) |

## Características de la red

| Características | Descripción |
|------|------|
| Inicio inteligente | Soporte (serie 700), escanea el código QR para unirte a la red |
| Seguridad S2 | Admite tres niveles de seguridad: sin cifrar, S2 no autenticado, S2 autenticado |
| Contiene CC | Security CC, Security 2 CC, Actualización de firmware CC, CC básico, CC de batería, CC de notificación y más |
