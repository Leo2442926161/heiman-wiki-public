#HS8MIS - Sensor Móvil (Zigbee)

## Descripción general

HS8MIS es un sensor de movimiento Zigbee multifuncional lanzado por Heiman. Integra funciones de detección humana PIR y detección de iluminación, y es adecuado para escenas inteligentes con monitoreo dual de luz y movimiento.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Infrarrojo Pasivo (PIR) + Iluminación |
| Ángulo de detección | 110° |
| Distancia de detección | ≤7m |
| Método de suministro de energía | Batería CC de 3 V |

## Características

- Detección humana móvil
- Detección de iluminación
- Diseño de bajo consumo de energía
- Enlace de escena de soporte
