# Estándares de certificación

HS6GW (versión Zigbee) ha pasado las siguientes certificaciones:

- **CE/RED** — Directiva sobre equipos inalámbricos de la Unión Europea
- **CCC**: Certificación obligatoria de productos de China, consulte [Certificación obligatoria de China CCC](../../../../Certification Standards/CCC China Compulsory Certification.md)
