# HS6GW - Puerta de enlace inteligente (Zigbee)

## Descripción general

HS6GW es una puerta de enlace inteligente Zigbee lanzada por Heiman. Como coordinador de la red Zigbee de Heiman, se conecta a Internet a través de un enlace ascendente Wi-Fi o Ethernet y admite el acceso de hasta 128 subdispositivos.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 (Coordinador) |
| Conexión de enlace ascendente | Wi-Fi/Ethernet |
| Nodos máximos | 128 subdispositivos |
| Método de suministro de energía | Fuente de alimentación USB de 5 V CC |

## Características

- Coordinador de la Red Zigbee
- Conexión de enlace ascendente Wi-Fi/Ethernet
- Descubrimiento y emparejamiento automático de dispositivos
- Vinculación de escena local.
- Acceso remoto (a través de Heiman Cloud Platform)
- Admite hasta 128 subdispositivos
- Actualización de firmware OTA

## Escenarios aplicables

- El punto de conexión central de la inteligencia integral de Heiman
- Cooperar con la aplicación Haiman para realizar monitoreo remoto
