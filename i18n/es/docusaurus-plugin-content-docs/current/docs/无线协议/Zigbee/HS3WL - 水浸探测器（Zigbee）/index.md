# HS3WL - Detector de fugas de agua (Zigbee)

## Descripción general

HS3WL es el detector de inmersión en agua Zigbee de tercera generación de Hyman. Tiene una sensibilidad de detección mejorada y un rendimiento inalámbrico basado en la generación anterior, y es adecuado para escenarios de monitoreo de fugas de agua.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Método de detección | Detección de sonda |
| Método de suministro de energía | Batería CC de 3 V |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤95% de humedad relativa |

## Características

- Detección en tiempo real de fugas de agua/acumulación de agua.
- Sensibilidad de detección mejorada
- Alarma sonora
- Recordatorio de batería baja
- Diseño de sonda anticorrosión.
