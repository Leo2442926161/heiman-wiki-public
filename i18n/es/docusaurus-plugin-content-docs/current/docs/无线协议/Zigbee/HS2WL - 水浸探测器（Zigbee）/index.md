#HS2WL - Detector de fugas de agua (Zigbee)

## Descripción general

HS2WL es el detector de inmersión en agua Zigbee de segunda generación lanzado por Hyman. Admite la detección de doble canal de sondas y cables, y se puede utilizar para detectar eventos de fugas de agua, como roturas de tuberías de agua y desbordamientos de fregaderos.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Método de detección | Detección de sonda + detección de cable (doble canal) |
| Método de suministro de energía | Batería CC de 3 V |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤95% de humedad relativa |

## Características

- Detección en tiempo real de fugas de agua/acumulación de agua.
- Sonda + cable de detección bicanal
- Alarma sonora
- Recordatorio de batería baja
- Diseño de sonda anticorrosión.

## Accesorios aplicables

- Extender el cable de detección.
