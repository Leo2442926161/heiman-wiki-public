#HS8OS - Sensor de Presencia Humana (Zigbee)

## Descripción general

HS8OS es un sensor de presencia humana Zigbee lanzado por Hyman. Utiliza tecnología de radar de ondas milimétricas y puede detectar la presencia del cuerpo humano en un estado estático (como acostado, sentado en silencio), compensando las deficiencias de los sensores PIR tradicionales que solo pueden detectar movimiento.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Radar de onda milimétrica (mmWave) |
| Rango de detección | Radio 3 ~ 5m |
| Capacidad de detección | Presencia de cuerpo humano estacionario (incluida la detección de microrrespiración) |
| Método de suministro de energía | Fuente de alimentación de CA |
| Método de instalación | Instalación en techo |

## Características

- Detección de cuerpo humano estacionario (micromovimientos respiratorios)
- Radio de cobertura 3~5 metros
- Instalación en techo, ángulo de detección óptimo
- Se puede utilizar de forma complementaria con sensores PIR.

## Escenarios aplicables

- Iluminación inteligente: siempre encendida cuando hay alguien allí, apagándose automáticamente cuando no hay nadie cerca
- Ahorro de energía del aire acondicionado: apaga automáticamente el aire acondicionado cuando la gente camina
- Atención inteligente a las personas mayores: alerta temprana para las personas mayores si permanecen inmóviles durante mucho tiempo
