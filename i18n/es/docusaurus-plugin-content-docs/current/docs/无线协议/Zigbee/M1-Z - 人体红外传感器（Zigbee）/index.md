# M1-Z - Sensor infrarrojo del cuerpo humano (Zigbee)

## Descripción general

M1-Z es un sensor infrarrojo compacto Zigbee del cuerpo humano lanzado por Heiman. Admite un ángulo de detección de 110° o 360° (según el método de instalación), tiene un sensor de iluminación incorporado y es adecuado para escenarios de seguridad e iluminación inteligente.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Infrarrojo Pasivo (PIR) + Iluminación |
| Ángulo de detección | 110° (montaje en pared) / 360° (techo) |
| Distancia de detección | ≤7m (25°C) |
| Rango de iluminancia | 0 ~ 1200 lux |
| Método de suministro de energía | Batería de litio CC de 3 V |
| Duración de la batería | Aproximadamente 3 años |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Método de instalación | Montado en pared/techo |

## Características

- Detección humana móvil
- Detección de iluminación (0~1200 Lux)
- 110° / 360° conmutable
- Recordatorio de batería baja
- Alarma antisabotaje
- Diseño compacto
