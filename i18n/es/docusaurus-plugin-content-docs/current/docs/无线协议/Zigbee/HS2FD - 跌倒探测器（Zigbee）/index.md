#HS2FD - Detector de Caídas (Zigbee)

## Descripción general

HS2FD es un detector de caídas inteligente Zigbee lanzado por Hyman. Está especialmente diseñado para personas mayores. Tiene acelerómetro y giroscopio incorporados. Utiliza un algoritmo de inteligencia artificial para identificar la postura de caída y puede enviar automáticamente una señal de socorro.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Acelerómetro + Giroscopio |
| Tecnología de detección | Algoritmo de detección de caídas por IA |
| Método de suministro de energía | Batería CC de 3 V |
| Método de uso | Clip/cordón para cinturón |

## Características

- Detección automática de caídas
- Solicite ayuda manualmente con un clic
- El algoritmo de IA distingue las caídas de las actividades normales (sentarse, agacharse) para reducir las falsas alarmas
- Diseño portátil (clip para cinturón o cordón)
- Recordatorio de batería baja

## Escenarios aplicables

- Atención domiciliaria a personas mayores: atención las 24 horas del día a personas mayores que viven solas
- Residencias de ancianos: custodia y gestión centralizada
- Centro de rehabilitación: Seguimiento del riesgo de caídas en pacientes postoperatorios
