# HS1CA-E - Alarma Inteligente de Monóxido de Carbono (Zigbee)

## Descripción general

HS1CA-E es la alarma inteligente de monóxido de carbono (CO) Zigbee de Hyman para el mercado de la UE. Utiliza un sensor electroquímico y puede detectar con precisión gas de monóxido de carbono con una concentración de 0~1000 ppm. Ha pasado la certificación EN 50291.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Electroquímica |
| Rango de detección | 0 ~ 1000 ppm |
| Presión sonora de alarma | ≥85 dB (3 metros) |
| Método de suministro de energía | CC 3 V (CR123A) |
| Vida del sensor | Aproximadamente 7 años |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤95% de humedad relativa |
| Método de instalación | Montaje en pared |

## Características

- Sensor de CO electroquímico de alta precisión
- Alarma graduada de concentración (aviso sonoro y luminoso)
- Sonido de alarma de 85 dB
- Recordatorio de batería baja
- Recordatorio de vencimiento de la vida útil del sensor
- Función de autoprueba
- Actualización de firmware OTA

## Estándares de certificación

- EN 50291 (norma europea para alarmas de monóxido de carbono)
-CE/ROJO

## Accesorios aplicables

- Batería CR123A
