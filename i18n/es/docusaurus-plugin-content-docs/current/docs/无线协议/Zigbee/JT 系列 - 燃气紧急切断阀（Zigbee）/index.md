# Serie JT - Válvula de cierre de emergencia de gas (Zigbee)

## Descripción general

La válvula de cierre de emergencia de gas de la serie JT es un dispositivo de ejecución que se utiliza junto con el detector de gas combustible Hyman Zigbee. Cuando se detecta una fuga de gas, puede cerrar de forma remota o automática la válvula del gasoducto para evitar accidentes.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo | Autónomo + enlace Zigbee |
| Gas de prueba | Gas natural / Gas licuado de petróleo |
| Normas de certificación | ES 15322 |
| Temperatura de funcionamiento | -10°C ~ +50°C |

## Características

- Vinculado con HS1CG / HS3CG para cerrar automáticamente la válvula
-Soporta cierre manual remoto de válvula.
- Pulsador de alarma de fuga
- Controlador de válvula solenoide

## Estándares de certificación

- ES 15322

## Accesorios aplicables

- Detector de gases combustibles HS1CG / HS3CG
