# Parámetros técnicos

Los siguientes son parámetros técnicos generales para los dispositivos Heiman Zigbee.

## Parámetros inalámbricos

| Parámetro | Valor |
|------|-----|
| Estándares de protocolo | Zigbee 3.0 (IEEE 802.15.4) |
| Banda de frecuencia de trabajo | 2,4 GHz (banda de frecuencia ISM común mundial) |
| Tasa de transmisión | Hasta 250 kbps |
| Topología de red | Red de malla de autorreparación |
| Capacidad de nodos | Máximo 65535 nodos en una sola red |
| Distancia de comunicación | Aproximadamente 10~30 metros en interiores (viendo obstáculos en la pared) |
| Método de cifrado | AES-128-CCM |

## Compatibilidad

| Proyecto | Descripción |
|------|------|
| Coordinador Zigbee | Compatible con el coordinador estándar Zigbee 3.0 |
| Plataformas compatibles | Zigbee2MQTT, ZHA (Home Assistant), Hubitat, Homey, SmartThings, puerta de enlace Tuya Zigbee, puerta de enlace Haiman HS6GW |
| Actualización OTA | Admite actualización de firmware inalámbrica (algunos modelos) |

## Consumo eléctrico y de energía

| Parámetro | Valor |
|------|-----|
| Método de suministro de energía | Funciona con pilas (CR123A / CR2450 / AA) o USB / red (tipo enchufe) |
| Duración de la batería | Aproximadamente 2 o 3 años para sensores (dependiendo de la frecuencia de los informes) |
| Consumo de energía en espera | Nivel de microamperios, compatible con el mecanismo de sondeo del sueño |

## Parámetros ambientales

| Parámetro | Valor |
|------|-----|
| Temperatura de funcionamiento | -10°C ~ +55°C |
| Humedad de funcionamiento | ≤ 95% HR (sin condensación) |
| Nivel de protección IP | Depende del modelo específico (el tipo de sensor suele ser IP30) |
