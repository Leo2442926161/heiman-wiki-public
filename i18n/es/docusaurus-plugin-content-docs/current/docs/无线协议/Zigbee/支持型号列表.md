## Instrucciones para nombrar el modelo

| Reglas de nomenclatura | Significado |
|----------|------|
| Sufijo -E | Versión UE (868 MHz/estándar EN) |
| Sufijo -Z | Versión China (2,4 GHz/estándar chino) |
| SA1/2/3/8 | Número de generación del producto, cuanto mayor sea el número, más nueva será la generación |
