#HS1MS - Sensor infrarrojo del cuerpo humano (Zigbee)

## Descripción general

HS1MS es un sensor infrarrojo del cuerpo humano (PIR) Zigbee estándar lanzado por Hyman. Utiliza tecnología de detección de infrarrojos pasivos y puede detectar la radiación de calor infrarroja emitida por un cuerpo humano en movimiento. Se utiliza ampliamente en escenarios de seguridad e iluminación inteligente.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Infrarrojo pasivo (PIR) |
| Ángulo de detección | 110° |
| Distancia de detección | ≤7m (25°C) |
| Método de suministro de energía | Batería de litio CC de 3 V |
| Duración de la batería | Aproximadamente 3 años |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Método de instalación | Montaje en pared |

## Características

- Detección humana móvil
- Detección de gran angular de 110°
- Recordatorio de batería baja
- Alarma antisabotaje
- Enlace de escena de soporte
