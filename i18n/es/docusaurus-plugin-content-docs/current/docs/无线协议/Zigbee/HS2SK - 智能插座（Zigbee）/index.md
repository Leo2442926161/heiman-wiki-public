#HS2SK - Enchufe Inteligente (Zigbee)

## Descripción general

HS2SK es un enchufe de medición inteligente Zigbee lanzado por Heiman. Admite control remoto de interruptores y medición del consumo de energía en tiempo real. Es adecuado para la gestión inteligente de electrodomésticos y tiene una precisión de ±2%.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tensión nominal | CA 220 V (estándar chino)/CA 110 V (estándar americano) |
| Carga máxima | 16A/3680W |
| Precisión de medición | ±2% |
| Método de suministro de energía | Alimentación eléctrica |

## Características

- Control remoto del interruptor
- Medición de energía/combustible en tiempo real
- Protección contra sobrecarga
- Interruptor temporizador
- Estadísticas de energía
- Enlace de escena de soporte

## Escenarios aplicables

- Apagar remotamente los electrodomésticos olvidados
- Estadísticas de consumo de energía del aire acondicionado/calentador de agua.
- Sal del modo hogar para apagar electrodomésticos no esenciales con un clic
