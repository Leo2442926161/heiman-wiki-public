#HS3MS - Sensor infrarrojo del cuerpo humano (Zigbee)

## Descripción general

HS3MS es el sensor infrarrojo del cuerpo humano Zigbee de tercera generación de Hyman. Adopta un diseño de montaje en techo de 360° y su sensibilidad de detección y cobertura mejoran significativamente en comparación con la generación anterior.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Infrarrojo pasivo (PIR) |
| Ángulo de detección | 360° (instalación en techo) |
| Distancia de detección | ≤7m (25°C) |
| Método de suministro de energía | Batería de litio CC de 3 V |
| Duración de la batería | Aproximadamente 3 años |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Método de instalación | Instalación en techo |

## Características

- Detección humana móvil
- Cobertura omnidireccional de 360°
- Recordatorio de batería baja
- Alarma antisabotaje
- Sensibilidad de detección mejorada
- Enlace de escena de soporte
