#zigbee

Zigbee es un protocolo de red en malla inalámbrica de bajo consumo y baja velocidad de datos basado en el estándar IEEE 802.15.4. Es uno de los protocolos más maduros y populares en el campo del hogar inteligente.

## Hyman y Zigbee

Hyman es uno de los primeros fabricantes chinos en adoptar el protocolo Zigbee y tiene una enorme línea de productos Zigbee. Los dispositivos Hyman Zigbee son compatibles con el estándar **Zigbee 3.0** y pueden interoperar con los coordinadores y puertas de enlace Zigbee convencionales.

## Especificaciones técnicas

| Características | Descripción |
|------|------|
| Banda de frecuencia | 2,4 GHz (mundial) |
| Tasa de transmisión | Hasta 250 kbps |
| Topología de red | Red de malla de autorreparación |
| Número de nodos | La red única admite hasta 65535 nodos |
| Distancia de comunicación | Unos 10-30 metros en interiores (obstáculos visibles) |
| Consumo de energía | Extremadamente baja, duración de la batería del sensor de hasta 3 años |

## Plataformas compatibles

Los dispositivos Hyman Zigbee se pueden conectar a las siguientes plataformas:

- **Zigbee2MQTT** — Puente Zigbee a MQTT de código abierto
- **ZHA** — Home Assistant con integración Zigbee incorporada
- **Hubitat** — Centro inteligente localizado
- **Hogareño**: centro inteligente multiprotocolo
- **SmartThings Hub**: centro de hogar inteligente de Samsung
- **Tuya Zigbee Gateway** — Ecosistema Tuya
- **Puerta de enlace Hyman HS6GW** — Portal oficial
