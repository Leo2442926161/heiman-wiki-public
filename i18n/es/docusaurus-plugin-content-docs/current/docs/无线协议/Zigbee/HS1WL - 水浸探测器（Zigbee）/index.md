# HS1WL - Detector de fugas de agua (Zigbee)

## Descripción general

HS1WL es el detector de inmersión en agua Zigbee de primera generación lanzado por Hyman. Adopta un método de detección de sonda y puede detectar eficazmente fugas o acumulación de agua. Es un equipo básico de monitoreo de inmersión en agua.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Método de detección | Detección de sonda |
| Método de suministro de energía | Batería CC de 3 V |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤95% de humedad relativa |

## Características

- Detección en tiempo real de fugas de agua/acumulación de agua.
- Alarma sonora
- Recordatorio de batería baja
- Diseño de sonda anticorrosión.
