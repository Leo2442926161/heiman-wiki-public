# HS1SA-E - Alarma de humo inteligente (Zigbee)

## Descripción general

HS1SA-E es una alarma de humo inteligente Zigbee estándar lanzada por Hyman para el mercado de la UE. Adopta tecnología fotoeléctrica de detección de humo, pasó la certificación EN 14604 y ganó el premio Red Dot Design.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Fotoeléctrico infrarrojo |
| Presión sonora de alarma | ≥85 dB (3 metros) |
| Método de suministro de energía | CC 3 V (CR123A) |
| Duración de la batería | Aproximadamente 3 años |
| Distancia de comunicación | ≤70 m (área abierta) |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤95% de humedad relativa |
| Método de instalación | Instalación en techo |

## Características

- Detección fotoeléctrica de humo para detectar humo de quemaduras lentas y llamas abiertas.
- Alarma luminosa y sonora de 85 dB de alto nivel
- Recordatorio de batería baja
- Función de autoprueba (prueba con un clic)
- Alarma antisabotaje
-Admite silencio remoto
- Actualización de firmware OTA

## Estándares de certificación

- EN 14604 (norma europea para alarmas de humo)
- Premio de diseño Red Dot
-CE/ROJO

## Accesorios aplicables

- Batería CR123A
- Tornillos de montaje/tubo de expansión.
