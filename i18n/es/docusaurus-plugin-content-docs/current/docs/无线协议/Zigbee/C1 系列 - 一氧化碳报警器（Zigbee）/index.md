# Serie C1 - Alarma de Monóxido de Carbono (Zigbee)

## Descripción general

La serie C1 es una alarma básica de monóxido de carbono Zigbee lanzada por Heiman. Utiliza sensores electroquímicos y es adecuado para escenarios de detección de fugas de monóxido de carbono. Ha pasado la certificación EN 50291.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Electroquímica |
| Rango de detección | 0 ~ 1000 ppm |
| Presión sonora de alarma | ≥85 dB (3 metros) |
| Método de suministro de energía | Funciona con pilas |
| Vida del sensor | Aproximadamente 7 años |
| Temperatura de funcionamiento | -10°C ~ +50°C |

## Características

- Detección electroquímica de CO
- Alarma de clasificación de concentración.
- Alarma sonora y visual de 85 dB.
- Recordatorio de batería baja
- Función de autoprueba

## Estándares de certificación

-EN 50291
