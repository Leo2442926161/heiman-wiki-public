# Estándares de certificación

HS8SA ha superado las siguientes certificaciones:

- **EN 14604** — Norma europea para alarmas de humo (tipo UE), consulte [Norma europea EN](../../../../ApprovalStandards/EN European Standard.md)
- **UL 217**: norma de seguridad de alarmas de humo de EE. UU. (modelo de EE. UU.), consulte [UL Underwriters Laboratories](../../../../certification-standards/ULUnderwriters Laboratories.md)
- **CE/RED/EMC** — Directiva de equipos inalámbricos de la Unión Europea y compatibilidad electromagnética
- **Zigbee 3.0 / Q-Label** — Certificación Zigbee Alliance, consulte [Otras certificaciones](../../../Certification Standards/Other Certifications.md)
