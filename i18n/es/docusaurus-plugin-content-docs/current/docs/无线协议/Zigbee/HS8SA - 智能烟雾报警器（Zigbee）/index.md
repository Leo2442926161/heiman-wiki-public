#HS8SA - Alarma de humo inteligente (Zigbee)

## Descripción general

HS8SA es la alarma de humo inteligente Zigbee + Wi-Fi de Hyman para el mercado estadounidense. Tiene función de enlace de alta temperatura y ha pasado la certificación UL 217.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 / Wi-Fi |
| Tipo de sensor | Fotoeléctrico infrarrojo |
| Presión sonora de alarma | ≥85 dB (3 metros) |
| Método de suministro de energía | CC 3 V (CR123A) |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Normas de certificación | UL 217 |
| Mercados aplicables | Estados Unidos |

## Características

- Detección de humo fotoeléctrica
- Alarma de varillaje de alta temperatura
- Alarma luminosa y sonora de 85 dB de alto nivel
- Recordatorio de batería baja
- Función de autoprueba
- Alarma antisabotaje
-Admite silencio remoto
- Comunicación con doble protocolo (Zigbee + Wi-Fi)

## Estándares de certificación

- UL 217 (norma americana de alarmas de humo)
