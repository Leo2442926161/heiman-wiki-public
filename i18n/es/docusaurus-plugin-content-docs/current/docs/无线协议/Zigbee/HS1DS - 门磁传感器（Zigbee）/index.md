# HS1DS - Sensor magnético de puerta (Zigbee)

## Descripción general

HS1DS es un sensor magnético de puerta Zigbee lanzado por Heiman. Se utiliza para detectar el estado de apertura y cierre de puertas y ventanas. Es un componente básico del sistema de seguridad inteligente.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Interruptor de láminas |
| Método de suministro de energía | Pila de botón CR1632 |
| Duración de la batería | Alrededor de 2 ~ 3 años |
| Espacio libre de instalación | ≤10mm |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤95% de humedad relativa |

## Características

- Detección en tiempo real del estado de apertura/cierre de puertas y ventanas
- Alarma de manipulación (Tamper)
- Recordatorio de batería baja
-Soporta vinculación (apertura de puerta, encendido de luces, armado al salir de casa, etc.)
- Fácil de instalar (cinta adhesiva 3M o fijación con tornillos)

## Accesorios aplicables

- Pila de botón CR1632
- Cinta/tornillos de doble cara 3M
