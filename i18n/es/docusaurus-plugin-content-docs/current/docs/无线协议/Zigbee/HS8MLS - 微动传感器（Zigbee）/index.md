# HS8MLS - Microsensor de movimiento (Zigbee)

## Descripción general

HS8MLS es un sensor infrarrojo del cuerpo humano (PIR) Zigbee de largo alcance lanzado por Heiman. Está especialmente diseñado para escenas que requieren detección a larga distancia y es adecuado para espacios abiertos como pasillos y vestíbulos.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Zigbee 3.0 |
| Tipo de sensor | Infrarrojo pasivo (PIR) |
| Distancia de detección | Detección de larga distancia |
| Método de suministro de energía | Batería CC de 3 V |

## Características

- Detección de movimiento a larga distancia
- Diseño de bajo consumo de energía
- Enlace de escena de soporte
