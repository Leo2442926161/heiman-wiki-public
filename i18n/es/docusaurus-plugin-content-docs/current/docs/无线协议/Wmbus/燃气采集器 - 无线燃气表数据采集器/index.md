# Colector de gas: recolector de datos de medidores de gas inalámbricos

## Descripción general

El recolector de gas es un dispositivo inalámbrico de recolección de datos wM-Bus lanzado por Hyman para el campo de gas inteligente. Opera en la banda de frecuencia de 868 MHz (norma EN 13757) y se utiliza para leer de forma remota los datos de medición de los medidores de gas y cargarlos en el sistema centralizado de lectura de medidores. Alimentado por baterías de litio de grado industrial con una duración de batería de 6 a 10 años, es adecuado para escenarios de lectura de medidores centralizados como apartamentos, áreas residenciales y edificios comerciales.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | wM-Bus (EN 13757) |
| Banda de frecuencia operativa | 868 MHz (banda SRD europea) |
| Tasa de transmisión | 4,8 kbps / 16,384 kbps / 100 kbps (modo T/S/R) |
| Distancia de comunicación | Aproximadamente 100~300 metros en interiores, aproximadamente 500~1000 metros en espacios abiertos |
| Método de suministro de energía | Batería de litio industrial (ER18505 / ER26500) |
| Duración de la batería | 6~10 años (normalmente reportado 1 vez/día) |
| Temperatura de funcionamiento | -20°C ~ +60°C |
| Humedad de trabajo | ≤ 95% HR |
| Nivel de protección | IP65 ~ IP68 (dependiendo del entorno de instalación) |
| Cifrado de datos | AES-128 (opcional) |

## Características

-Soporta tres modos de trabajo wM-Bus S/T/R
- Recopile automáticamente datos de uso del medidor de gas e infórmelo periódicamente
- Bajo consumo de energía, sueño profundo, batería de larga duración
- Cifrado de datos AES-128 para garantizar la seguridad de las comunicaciones.
-Admite actualización remota de firmware
- Mecanismo de desconexión, reconexión y retransmisión de datos.
- Admite el protocolo de capa de aplicación EN 13757-3
- Se puede utilizar con el recopilador HS1-CM / puerta de enlace HS1-GW para carga centralizada

## Estándares de certificación

- EN 13757-4 (Capa física M-Bus inalámbrico)
- EN 13757-3 (capa de aplicación M-Bus)
