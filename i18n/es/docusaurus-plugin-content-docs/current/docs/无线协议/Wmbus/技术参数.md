# Parámetros técnicos

Los siguientes son los parámetros técnicos generales del equipo Heimann Wireless M-Bus (wM-Bus).

## Parámetros inalámbricos

| Parámetro | Valor |
|------|-----|
| Estándares de protocolo | EN 13757-4 (Capa física M-Bus inalámbrico) |
| Banda de frecuencia de trabajo | 868 MHz (banda de frecuencia SRD europea, 868,0 ~ 868,6 MHz) |
| Método de modulación | FSK |
| Tasa de transmisión | 4,8 kbps / 16,384 kbps / 100 kbps (modo T / modo S / modo R) |
| Topología de red | Estrella (instrumento → colector/puerta de enlace) |
| Distancia de comunicación | Aproximadamente 100~300 metros en interiores, aproximadamente 500~1000 metros en espacios abiertos |
| Transmitir potencia | ≤ 25 mW (+14 dBm) |

## Modo de trabajo

| Modo | Tarifa | Descripción |
|------|------|------|
| Modo S | 16,384 kbps | Modo estándar, bajo consumo de energía, adecuado para instrumentos que funcionan con baterías |
| Modo T | 100 kbps | Modo de alta velocidad, adecuado para escenarios de datos de gran tráfico |
| Modo R | 4,8 kbps | Modo de banda estrecha, transmisión de larga distancia |

## Consumo eléctrico y de energía

| Parámetro | Valor |
|------|-----|
| Método de suministro de energía | Fuente de alimentación de batería de litio (ER18505/ER26500 y otras baterías de litio industriales) |
| Duración de la batería | 6~10 años (dependiendo de la frecuencia de los informes, el valor típico es 1 vez/día) |
| Consumo de energía en espera | Nivel de microamperios, sueño profundo |
| Recibir sensibilidad | Aprox. -105 dBm (modo S) |

## Parámetros ambientales

| Parámetro | Valor |
|------|-----|
| Temperatura de funcionamiento | -20°C ~ +60°C (grado industrial) |
| Humedad de funcionamiento | ≤ 95% HR (sin condensación) |
| Nivel de protección IP | IP65 ~ IP68 (según modelo específico, aplicable a instrumentos de exterior) |

## Protocolo de datos

| Capa | Estándar |
|----|------|
| Capa física | EN 13757-4 |
| Capa de enlace de datos | EN 13757-4 (Formato de trama M-Bus inalámbrico) |
| Capa de aplicación | EN 13757-3 (capa de aplicación M-Bus) |
| Método de cifrado | AES-128 (cifrado opcional) |
