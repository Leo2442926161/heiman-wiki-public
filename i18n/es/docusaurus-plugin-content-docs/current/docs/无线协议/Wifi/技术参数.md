# Parámetros técnicos

Las siguientes son especificaciones técnicas generales para los dispositivos WiFi de Heimann.

## Parámetros inalámbricos

| Parámetro | Valor |
|------|-----|
| Estándares de protocolo | IEEE 802.11 b/g/n |
| Banda de frecuencia operativa | 2,4 GHz (no compatible con 5 GHz) |
| Tasa de transmisión | Hasta 150 Mbps (802.11n) |
| Topología de red | Topología en estrella (conectado directamente al enrutador) |
| Distancia de comunicación | Aproximadamente 30 ~ 50 metros en interiores (dependiendo del rendimiento del enrutador) |
| Método de cifrado | WPA2-PSK/WPA3 |

## Método de distribución de red

| Método | Descripción |
|------|------|
| Configuración inteligente | Aplicación móvil Configuración de red con un clic (equipo WiFi tradicional) |
| Aprovisionamiento asistido por Bluetooth | Configuración de credenciales WiFi a través de canales temporales Bluetooth (algunos modelos nuevos) |
| Modo AP | Encienda el punto de acceso en el dispositivo y configúrelo después de conectar el teléfono |

## Consumo eléctrico y de energía

| Parámetro | Valor |
|------|-----|
| Método de suministro de energía | Fuente de alimentación de red (tipo enchufe/tipo puerta de enlace) o fuente de alimentación de batería (tipo alarma) |
| Nivel de consumo de energía | Superior a Zigbee/Thread debido a la necesidad de mantener la conexión TCP/IP |
| Consumo de energía en espera | Tipo de enchufe aproximadamente 0,5 W ~ 1 W |

## Parámetros ambientales

| Parámetro | Valor |
|------|-----|
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de funcionamiento | ≤ 90% HR (sin condensación) |
