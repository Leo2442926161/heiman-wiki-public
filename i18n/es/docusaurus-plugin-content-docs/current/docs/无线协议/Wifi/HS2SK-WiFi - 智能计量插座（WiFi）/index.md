#HS2SK-WiFi - Toma con contador inteligente (WiFi)

## Descripción general

HS2SK-WiFi es un enchufe de medición inteligente WiFi lanzado por Heiman. Admite conexión directa de 2,4 GHz 802.11 b/g/n a enrutadores domésticos y puede lograr control remoto y monitoreo del consumo de energía sin necesidad de puertas de enlace adicionales. Adecuado para la gestión inteligente de electrodomésticos en hogares, oficinas y otros escenarios.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | IEEE 802.11 b/g/n (2,4 GHz) |
| Carga máxima | 16A |
| Potencia máxima | 3680W (230V) / 1800W (110V) |
| Precisión de medición | ±2% |
| Método de suministro de energía | Alimentación eléctrica |
| Consumo de energía en espera | ≤ 0,5 W |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤ 90% HR |
| Material de la carcasa | PC V0 retardante de llama |
| Especificaciones del gato | Estándar europeo/estándar americano (según versión de mercado) |

## Características

-Admite control remoto de aplicación inteligente Smart Life/Tuya
- Monitoreo de energía en tiempo real y estadísticas de consumo de energía.
- Control remoto del interruptor
- Interruptor de tiempo y cuenta regresiva
- Vinculación y automatización de escenas.
- Control por voz (compatible con Alexa, Google Home)
