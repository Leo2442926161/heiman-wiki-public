#HS6GW - Puerta de enlace de hogar inteligente

## Descripción general

HS6GW es una puerta de enlace doméstica inteligente lanzada por Heiman, que admite enlace ascendente WiFi + Ethernet, coordinador Zigbee incorporado y puede administrar hasta 128 subdispositivos Zigbee. Realizar acceso remoto y servicios en la nube mediante la conexión a Internet es el centro central del sistema de hogar inteligente Haiman.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Comunicación de enlace ascendente | IEEE 802.11 b/g/n (WiFi de 2,4 GHz) + Ethernet 10/100 M |
| Comunicación de enlace descendente | Zigbee 3.0 (modo coordinador) |
| Número máximo de subdispositivos | 128 |
| Método de suministro de energía | CC 5 V/1 A (Micro USB) |
| Consumo de energía en espera | ≤ 1W |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤ 90% HR |
| Luz indicadora | Estado de WiFi/Encendido/Estado de red Zigbee |
| Método de reinicio | Botón de reinicio + pulsación larga para restaurar la configuración de fábrica |

## Características

- Enlaces ascendentes duales: conexión por cable WiFi o Ethernet, elección flexible según el entorno de red
-Admite acceso remoto: verifique el estado del dispositivo en cualquier momento y en cualquier lugar a través de la aplicación en la nube
- Enlace de escena local: admite la operación automatizada de equipos dentro de la LAN (aún disponible cuando está desconectado)
- Actualización de firmware OTA
- Soporte Tuya / plataforma Smart Life
- Administre hasta 128 subdispositivos Zigbee
-Admite la optimización de enrutamiento y autorreparación de la red Zigbee
