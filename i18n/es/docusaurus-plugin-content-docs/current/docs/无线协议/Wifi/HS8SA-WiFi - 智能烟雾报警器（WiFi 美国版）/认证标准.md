# Estándares de certificación

HS8SA-WiFi ha superado las siguientes certificaciones:

- **UL 217**: norma estadounidense de seguridad de alarmas de humo, consulte [UL Underwriters Laboratories](../../../../certificationstandards/ULUnderwriters Laboratories.md)
- **FCC** — Certificación de la Comisión Federal de Comunicaciones
- **CE** — Declaración de conformidad de la Unión Europea
