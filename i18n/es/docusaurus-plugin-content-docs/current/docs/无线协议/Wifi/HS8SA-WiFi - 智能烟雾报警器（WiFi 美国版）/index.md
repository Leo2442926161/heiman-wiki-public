# HS8SA-WiFi - Alarma de humo inteligente (versión WiFi para EE. UU.)

## Descripción general

HS8SA-WiFi es una alarma de humo inteligente WiFi lanzada por Hyman para el mercado estadounidense. Utiliza tecnología fotoeléctrica de detección de humo, admite redes WiFi y modos duales de trabajo independiente y ha pasado la certificación UL 217. Adecuado para protección contra incendios en hogares, apartamentos y otros lugares residenciales.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | IEEE 802.11 b/g/n (2,4 GHz) |
| Tipo de sensor | Fotoeléctrico infrarrojo |
| Presión sonora de alarma | ≥85 dB (3 metros) |
| Método de suministro de energía | Funciona con pilas |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤ 90% HR |
| Método de instalación | Instalación en techo |

## Características

- Detección fotoeléctrica de humo para detectar humo de quemaduras lentas y llamas abiertas.
- Pulsador de alarma remota de red WiFi
- Modo de operación independiente (la alarma local de sonido y luz aún puede ocurrir cuando la red está desconectada)
- Alarma luminosa y sonora de 85 dB de alto nivel
- Recordatorio de batería baja
- Función de autoprueba (prueba con un clic)
- Alarma antisabotaje
-Admite silencio remoto

## Estándares de certificación

- UL 217 (Norma americana para alarmas de humo)
-FCC
- Cumplimiento de NFPA 72
