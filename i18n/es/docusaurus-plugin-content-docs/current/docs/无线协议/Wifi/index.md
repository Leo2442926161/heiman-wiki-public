#wifi

Wi-Fi es el protocolo de red inalámbrica más popular y se basa en el estándar IEEE 802.11. Algunos dispositivos inteligentes de Heiman admiten conexión Wi-Fi y se pueden conectar directamente a un enrutador sin necesidad de una puerta de enlace adicional.

## Escenarios aplicables

Los dispositivos Wi-Fi son adecuados para las siguientes situaciones:

- Infraestructura de red completa, sin necesidad de comprar puertas de enlace adicionales
- Dispositivos que no son sensibles al consumo de energía (como enchufes inteligentes)
- Escenarios que requieren acceso remoto y servicios en la nube.
- Viviendas ya cubiertas por redes Wi-Fi Mesh maduras

## Productos Wi-Fi de Hyman

Heimann ofrece productos compatibles con la conectividad Wi-Fi:

- **Smart Socket HS2SK-WiFi** — Medición del consumo de energía, interruptor remoto
- **Smart Gateway**: se conecta al enrutador mediante Wi-Fi

## Notas

- La batería del sensor Wi-Fi se agota más rápido que Zigbee/Thread
- Una gran cantidad de dispositivos Wi-Fi puede afectar la estabilidad de la red
- Los principales productos de Haiman son Zigbee/Matter over Thread, y la línea de productos Wi-Fi está relativamente optimizada.
