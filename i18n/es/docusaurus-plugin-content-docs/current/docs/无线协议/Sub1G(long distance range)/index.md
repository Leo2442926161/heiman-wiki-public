# Sub-1G (comunicación de larga distancia)

Sub-1G es una tecnología de comunicación inalámbrica que opera en la banda de frecuencia por debajo de 1 GHz. Tiene las características de una distancia de transmisión ultralarga y un consumo de energía extremadamente bajo. Es muy adecuado para escenarios de IoT que requieren cobertura a gran escala.

## Características

| Características | Descripción |
|------|------|
| Banda de frecuencia operativa | 433 MHz / 868 MHz (UE) / 915 MHz (EE. UU.) |
| Distancia de transmisión | Hasta 1-3 kilómetros en ambiente abierto |
| Capacidad de penetración | Mejor que 2,4 GHz, fuerte penetración en obstáculos de pared |
| Consumo de energía | Extremadamente baja, la duración de la batería puede alcanzar entre 5 y 10 años |
| Tasa de transmisión | Bajo (adecuado para escenarios de volúmenes de datos pequeños) |

## Escenarios de aplicación

- **Alarma de humo/CO independiente**: interconectada entre plantas del edificio
- **Sistema de protección contra incendios de grandes edificios**: centros comerciales, hoteles, escuelas
- **Smart Senior Living Community**: llamada de emergencia entre unidades
- **Sub-1G + Wi-Fi Gateway**: el subdispositivo se comunica a través de Sub-1G y el gateway se conecta a la nube

## Productos Hyman Sub-1G

Hyman tiene muchos años de experiencia con productos inalámbricos de protección contra incendios Sub-1G, incluida la serie S1 de alarmas de humo independientes. Estos dispositivos pueden realizar interconexión entre dispositivos sin Internet.
