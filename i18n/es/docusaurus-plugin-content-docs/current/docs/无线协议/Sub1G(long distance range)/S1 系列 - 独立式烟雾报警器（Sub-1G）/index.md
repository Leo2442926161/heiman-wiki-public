# Serie S1: alarma de humo independiente (Sub-1G)

## Descripción general

La serie S1 es una alarma de humo Sub-1G independiente lanzada por Hyman. Opera en la banda de frecuencia de 433 MHz/868 MHz, utiliza tecnología fotoeléctrica de detección de humo y ha pasado la certificación EN 14604. No se requiere puerta de enlace ni Internet. Los dispositivos se interconectan directamente a través de radiofrecuencia Sub-1G. Cuando cualquier dispositivo activa una alarma, todos los dispositivos del mismo grupo emitirán una alarma simultáneamente. Es adecuado para escenarios de extinción de incendios a gran escala, como edificios, centros comerciales y hoteles.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | 433 MHz / 868 MHz Sub-1G (según zona de mercado) |
| Tipo de sensor | Fotoeléctrico infrarrojo |
| Presión sonora de alarma | ≥85 dB (3 metros) |
| Método de suministro de energía | Funciona con pilas (batería de litio CR123A/AA) |
| Duración de la batería | ≥ 5 años |
| Distancia de comunicación | Alrededor de 1 a 3 kilómetros en ambiente abierto |
| Capacidad de la red | Un solo grupo admite hasta más de 50 nodos |
| Método de interconexión | Interconexión inalámbrica por radiofrecuencia (no requiere puerta de enlace) |
| Temperatura de funcionamiento | -10°C ~ +55°C |
| Humedad de trabajo | ≤ 95% HR |
| Método de instalación | Instalación en techo |

## Modelo de serie

| Modelo | Descripción |
|------|------|
| S1-SA | Tipo básico, admite interconexión inalámbrica |
| S1-SA-P | Versión con función de cámara |
| S1-SA-T | Versión con varillaje de alta temperatura |
| S1-SA-RF | Tipo de interconexión RF |

## Características

- Detección fotoeléctrica de humo para detectar humo de quemaduras lentas y llamas abiertas.
- Alarma de interconexión inalámbrica: activada por cualquier dispositivo en la misma red, todos los dispositivos sonarán y encenderán la alarma simultáneamente
- Modo de trabajo independiente: sin puerta de enlace, no requiere Internet, comunicación directa entre dispositivos
- Alarma luminosa y sonora de 85 dB de alto nivel
- Recordatorio de batería baja
- Función de autoprueba (prueba con un clic)
- Alarma antisabotaje
-Admite silencio remoto
- Diseño de consumo de energía ultrabajo, duración de la batería de más de 5 años

## Estándares de certificación

- EN 14604 (norma europea para alarmas de humo)
-CE/ROJO
- CPR (Reglamento de Productos de Construcción de la UE)

## Accesorios aplicables

- Repetidor inalámbrico S1-RM (distancia de comunicación extendida)
- Control remoto S1-RC (prueba/silencio/reinicio)
