# Estándares de certificación

HS1CA-M ha superado las siguientes certificaciones:

- **EN 50291** — Norma europea para alarmas de monóxido de carbono, consulte [Norma europea EN](../../../../Normas de certificación/Norma europea EN.md)
- **Marca CE**: Declaración de conformidad de la UE, consulte [Conformidad CE UE](../../../../certification-standards/CE EU-Conformity.md)
- **Certificación Matter**: se aprobó la certificación CSA Alliance Matter para garantizar la interconexión multiplataforma de hogares inteligentes.
