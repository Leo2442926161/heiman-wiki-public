# Sensor de temperatura y humedad H1-M (Materia)

H1-M es un sensor de temperatura y humedad **Matter over Thread** lanzado por Hyman, que puede monitorear la temperatura ambiente y la humedad relativa en tiempo real y admite enlaces de automatización del hogar inteligente.

## Descripción general del producto

- **Tipo de producto**: Sensor de temperatura y humedad
- **Protocolo de comunicación**: Asunto sobre hilo
- **Parámetros de detección**: temperatura, humedad relativa
- **Escenarios Típicos**: Vigilancia ambiental en viviendas, oficinas, bodegas, etc.

## Parámetros técnicos

| Parámetro | Valor |
|------|-----|
| Precisión de temperatura | ±0,3 °C |
| Precisión de la humedad | ±3% HR |
| Protocolo de comunicación | Materia sobre hilo (IEEE 802.15.4) |
| Banda de frecuencia operativa | 2,4 GHz |
| Método de suministro de energía | Batería de litio CR2 |
| Duración de la batería | Aproximadamente 2 ~ 3 años |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤95% HR (sin condensación) |

## Características

- Monitoreo de temperatura y humedad en tiempo real.
- Detección de alta precisión (temperatura ±0,3°C, humedad ±3% RH)
- Recordatorio de batería baja
- Materia de interoperabilidad multiplataforma multiecológica

## Plataformas compatibles

| Plataforma | Soporte |
|------|----------|
| Inicio de Apple | ✅ Se requiere enrutador de borde de hilo |
| Inicio de Google | ✅ Se requiere enrutador de borde de hilo |
| AmazonAlexa | ✅ Se requiere enrutador de borde de hilo |
| Asistente de hogar | ✅ |

## Requisitos de ruta fronteriza

Se requiere un enrutador Thread Border, como Apple HomePod mini, Google Nest Hub de segunda generación, Amazon Echo de cuarta generación, etc.
