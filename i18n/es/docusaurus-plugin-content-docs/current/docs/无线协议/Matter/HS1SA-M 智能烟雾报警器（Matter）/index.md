# HS1SA-M Alarma de humo inteligente (Materia)

HS1SA-M es una alarma de humo inteligente **Matter over Thread** lanzada por Hyman. Utiliza tecnología fotoeléctrica de detección de humo y admite la interconexión de plataformas de hogar inteligente interecológicas.

## Descripción general del producto

- **Tipo de producto**: Alarma de humo inteligente
- **Protocolo de comunicación**: Asunto sobre hilo
- **Estándar de Certificación**: EN 14604
- **Escenarios típicos**: Aviso de incendio en viviendas, apartamentos, hoteles, etc.

## Parámetros técnicos

| Parámetro | Valor |
|------|-----|
| Tipo de sensor | Fotoeléctrico infrarrojo |
| Protocolo de comunicación | Materia sobre hilo (IEEE 802.15.4) |
| Banda de frecuencia operativa | 2,4 GHz |
| Presión sonora de alarma | ≥85 dB (3 metros) |
| Método de suministro de energía | CC 3 V (batería de litio CR123A) |
| Duración de la batería | Aproximadamente 3 años |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤95% HR (sin condensación) |

## Características

- Detección fotoeléctrica de humo para detectar humo de combustión lenta y llama abierta.
- Alarma luminosa y sonora de 85 dB de alto nivel
- Recordatorio de batería baja
- Función de autoprueba (prueba con un clic)
- Alarma antisabotaje
- Silencio remoto
- Materia de interoperabilidad multiplataforma multiecológica

## Certificación

- Certificación EN 14604
- Certificación de materia
