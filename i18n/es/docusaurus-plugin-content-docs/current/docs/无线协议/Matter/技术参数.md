# Parámetros técnicos

Las siguientes son especificaciones generales para los equipos Hyman Matter.

## Parámetros inalámbricos

| Parámetro | Valor |
|------|-----|
| Protocolo subyacente | Hilo (IEEE 802.15.4) |
| Banda de frecuencia operativa | 2,4 GHz (banda ISM) |
| Estándares de la capa de aplicación | Materia 1.x |
| Topología de red | Red de malla de autorreparación |
| Método de cifrado | AES-128-CCM |

## Requisitos de hardware

| Proyecto | Descripción |
|------|------|
| Enrutador fronterizo | Requerido, como Apple HomePod mini, Apple TV 4K, Google Nest Hub de segunda generación, etc. |
| Soporte para terminales móviles | iOS / Android (distribución en red a través de diversas aplicaciones ecológicas) |
| Interoperabilidad interecológica | Apple HomeKit, Google Home, Amazon Alexa, Asistente de hogar, SmartThings |

## Consumo eléctrico y de energía

| Parámetro | Valor |
|------|-----|
| Método de suministro de energía | Funciona con pilas (CR123A/CR2450, etc.) |
| Duración de la batería | Aproximadamente entre 1 y 3 años (según el tipo de dispositivo y la frecuencia de uso) |
| Consumo de energía en espera | Extremadamente bajo, apoya el mecanismo de despertar del sueño |

## Parámetros ambientales

| Parámetro | Valor |
|------|-----|
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de funcionamiento | ≤ 95% HR (sin condensación) |
