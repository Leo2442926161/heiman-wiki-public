#Importante

Matter (anteriormente conocido como Proyecto CHIP) es un estándar de interconexión de hogares inteligentes lanzado por CSA Connection Standards Alliance, con el objetivo de romper las barreras de las marcas y lograr la interoperabilidad ecológica cruzada.

## Heyman y la Materia

Hyman fue uno de los primeros en adoptar el protocolo Matter y lanzó **las primeras alarmas de monóxido de carbono certificadas por Matter** del mundo. Todos los dispositivos Matter de Hyman utilizan la solución **Matter over Thread**.

## Características técnicas

| Características | Descripción |
|------|------|
| Protocolo subyacente | Thread (red Mesh de bajo consumo) |
| Topología de red | Red de malla con autorreparación, cuantos más dispositivos, más amplia será la cobertura |
| Consumo de energía | Extremadamente bajo, puede funcionar durante 1 a 3 años con batería |
| Seguridad | Admite cifrado AES-128-CCM, se requiere autenticación del dispositivo |
| Interoperabilidad | En Apple HomeKit, Google Home, Amazon Alexa, Home Assistant y más |

## Requisitos del enrutador fronterizo

El uso de dispositivos Matter over Thread requiere un enrutador de borde Thread:

- Apple HomePod mini / HomePod 2.ª generación / Apple TV 4K
- Google Nest Hub 2.ª generación/Nest Hub Max
-Amazon Echo 4ta generación
- Asistente de inicio verde + Conectar ZBT-2
- Hogareño Pro

## Productos certificados por Hyman Matter

- Alarma de humo inteligente (versión EE. UU./versión UE)
- Alarma inteligente de monóxido de carbono (versión EE. UU./versión UE)
- Sensor infrarrojo del cuerpo humano M1-M
- Detector de inmersión en agua L1-M
- Sensor de temperatura y humedad H1-M
