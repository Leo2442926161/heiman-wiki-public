# HS1DS-M - Sensor de puerta inteligente (Materia sobre Hilo)

## Descripción general

HS1DS-M es la versión Matter over Thread del sensor de puerta inteligente Heiman. Admite acceso nativo a todas las plataformas compatibles con Matter, como Apple Home, Google Home, Amazon Alexa, Home Assistant, etc., sin necesidad de una puerta de enlace patentada.

## Especificaciones técnicas

| Parámetro | Valor |
|------|-----|
| Protocolo de comunicación | Asunto sobre Hilo |
| Método de suministro de energía | Pila de botón CR1632 |
| Duración de la batería | Alrededor de 2-3 años |
| Distancia de instalación | ≤10mm |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤95% de humedad relativa |

## Características

- Detección en tiempo real del estado de apertura/cierre de puertas y ventanas
- Alarma de manipulación (Tamper)
- Recordatorio de batería baja
- Compatible con varias plataformas: Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Red Thread Mesh, control local de baja latencia
- No se requiere puerta de enlace dedicada, se requiere enrutador de borde de subprocesos
