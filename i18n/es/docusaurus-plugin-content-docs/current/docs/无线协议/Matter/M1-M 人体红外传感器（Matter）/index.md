# M1-M Sensor infrarrojo del cuerpo humano (Materia)

M1-M es un sensor infrarrojo del cuerpo humano **Matter over Thread** lanzado por Hyman. Integra funciones de detección PIR y detección de iluminación y es adecuado para escenarios de automatización del hogar inteligente.

## Descripción general del producto

- **Tipo de producto**: Sensor infrarrojo del cuerpo humano
- **Protocolo de comunicación**: Asunto sobre hilo
- **Método de detección**: Infrarrojo pasivo (PIR) + detección de iluminación
- **Escenario típico**: las luces se encienden cuando viene gente, enlace de seguridad, automatización ambiental

## Parámetros técnicos

| Parámetro | Valor |
|------|-----|
| Ángulo de detección | 110° (montaje en pared) / 360° (techo) |
| Distancia de detección | ≤7m (25°C) |
| Rango de iluminación | 0~1200 lux |
| Protocolo de comunicación | Materia sobre hilo (IEEE 802.15.4) |
| Banda de frecuencia operativa | 2,4 GHz |
| Método de suministro de energía | Batería de litio CC de 3 V |
| Duración de la batería | Aproximadamente 1 año (modo Materia) |
| Temperatura de funcionamiento | -10°C ~ +50°C |
| Humedad de trabajo | ≤95% HR (sin condensación) |

## Características

- Detección humana móvil
- Detección de iluminación ambiental
- Distancia de detección ≤7m
- Soporte para instalación en pared o techo
- Materia de interoperabilidad multiplataforma multiecológica
