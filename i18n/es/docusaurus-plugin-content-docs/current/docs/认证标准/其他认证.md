# Otras certificaciones

## Marca de calidad Q-Label

Q-Label es una marca de certificación de calidad para la industria europea de hogares inteligentes, lanzada por Q-Lab Alliance (ahora parte de Zigbee Alliance / CSA Connectivity Standards Alliance).

| Proyecto | Descripción |
|------|------|
| Ámbito de aplicación | Certificación de interoperabilidad de dispositivos Zigbee |
| Contenido de la certificación | Compatibilidad de dispositivos con otras marcas de puertas de enlace/controladores Zigbee |
| Importancia | Garantizar la interoperabilidad entre marcas y reducir el riesgo de "comprar una casa y no usarla" |

Los productos Hyman Zigbee cuentan con la certificación Q-Label, lo que garantiza la compatibilidad con las siguientes plataformas:
- Amazon Echo Plus / Echo Show (Zigbee integrado)
- Puente Philips Hue
- Centro de cosas inteligentes
- Otras puertas de enlace de autenticación Q-Label

## Premio de diseño Red Dot

La alarma de humo de la serie HS1SA de Hyman ganó el **Red Dot Design Award** (Premio Red Dot Design Award), que refleja la fortaleza de Hyman en el diseño de productos.

## Sistema de Gestión de Calidad ISO 9001

Haiman ha pasado la certificación del sistema de gestión de calidad ISO 9001:2015 para garantizar el control de calidad de todo el proceso, desde I+D, producción hasta posventa.
