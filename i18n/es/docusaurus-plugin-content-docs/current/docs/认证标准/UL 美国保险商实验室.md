# Laboratorios UL Underwriters

UL (Underwriters Laboratories) es la organización de certificación y desarrollo de normas de seguridad de productos con mayor autoridad en los Estados Unidos. Los principales estándares UL involucrados en los productos Heiman son los siguientes:

## UL 217: Estándar de alarmas de humo (EE. UU.)

| Proyecto | Descripción |
|------|------|
| Ámbito de aplicación | Alarmas de humo residenciales (mercado estadounidense) |
| Requisitos de detección | Detección de humo fotoeléctrica/ionización, alarma sonora y luminosa, superior a 85 dB |
| Mercados aplicables | Estados Unidos, Canadá |
| Productos Heiman | HS8SA (versión estadounidense) |
| Importancia de la certificación | Cumplir con las normas contra incendios de varios estados de Estados Unidos es condición necesaria para acceder al mercado de América del Norte |

**Diferencias con otros estándares**: Los escenarios de prueba de UL 217 incluyen escenarios típicos de incendios domésticos estadounidenses, como incendios repentinos e incendios de combustión lenta, y son ligeramente diferentes de los métodos de prueba de EN 14604.

## UL 2034: Norma de alarma de monóxido de carbono (EE. UU.)

| Proyecto | Descripción |
|------|------|
| Ámbito de aplicación | Alarma doméstica de CO (mercado estadounidense) |
| Requisitos de detección | Sensor electroquímico, algoritmo de integración de concentración × tiempo |
| Mercados aplicables | Estados Unidos, Canadá |
| Productos Hayman | HS-720ES-M (versión estadounidense) |
| Importancia de la certificación | La mayoría de los estados de Estados Unidos exigen la instalación de alarmas de CO |

## Las principales diferencias entre la certificación UL y la certificación EN

| Aspectos | UL | ES |
|------|-----|-----|
| Normas de alarmas de humo | UL 217 | EN 14604 |
| Normas de alarma de CO | UL 2034 | EN 50291 |
| Método de detección | Ensayo específico de curva de fuego | Diferentes ensayos de tipo de fuego |
| Tensión aplicable | 120V / batería | 230V / batería |
