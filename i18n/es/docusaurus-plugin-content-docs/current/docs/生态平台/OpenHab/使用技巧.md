# Consejos de uso

## Configuración de cosas

Agregue el dispositivo Heiman mediante el enlace MQTT:

```java
Cosa mqtt:topic:heiman_smoke "Alarma de humo Heiman" @ "sala de estar" {
    Canales:
        Escriba binario: humo "estado de humo" [ stateTopic="zigbee2mqtt/heiman_smoke", transformPattern="JSONPATH:$.smoke" ]
        Número de tipo: batería "Energía de la batería" [ stateTopic="zigbee2mqtt/heiman_smoke", transformPattern="JSONPATH:$.battery" ]
}
```

## Definición del artículo

```java
Contacte con Smoke_Alarm "Alarma de humo [MAP(smoke.map):%s]" {channel="mqtt:topic:heiman_smoke:smoke" }
Número Smoke_Battery "Batería de la alarma de humo [%d%%]" {channel="mqtt:topic:heiman_smoke:battery" }
```

## Selección de coordenadas

Se recomienda utilizar un coordinador USB con chip **CC2531** o **CC2652** y asegurarse de que la versión del firmware sea la más reciente.

## Consejos de depuración

- Ver el estado "en línea/fuera de línea" de la cosa en la interfaz de usuario de Paper
- Utilice `openhab-cli info` para verificar si los enlaces están cargados correctamente
- Busque la palabra clave "heiman" en el registro para solucionar problemas de comunicación del dispositivo
