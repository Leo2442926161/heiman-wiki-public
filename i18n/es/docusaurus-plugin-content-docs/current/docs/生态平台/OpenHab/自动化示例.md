# Ejemplo de automatización

## Reglas de alarma de humo

```javascript
regla "Alarma de humo activada"
cuando
    El elemento heiman_smoke_alarm cambió a ON
entonces
    enviarComando(Luz_Salón, ENCENDIDO)
    sendNotification("admin", "⚠️ ¡Se activó la alarma de humo!")
    postUpdate(Estado_alarma, "alerta_humo")
fin
```

## Reglas del sensor de temperatura

```javascript
regla "Informe de temperatura y humedad"
cuando
    Artículo heiman_temperature recibió actualización
entonces
    val temp = (heiman_temperature.state como Número).floatValue
    val hum = (heiman_humidity.state como Número).floatValue
    si (temperatura > 40) {
        enviarComando(Aire acondicionado, ENCENDIDO)
        sendNotification("admin", "La temperatura es demasiado alta (" + temp + "°C), el aire acondicionado se ha encendido")
    }
    si (tararear > 80) {
        sendNotification("admin", "La humedad es demasiado alta (" + hum + "%), preste atención a la deshumidificación")
    }
fin
```

## Reglas de detección de inundaciones

```javascript
regla "Vínculo de detección de inmersión en agua"
cuando
    El elemento heiman_water_leak cambió a ON
entonces
    enviarComando(Válvula_Agua, APAGADO)
    sendBroadcastNotification ("💧 ¡Se detectó una fuga de agua! ¡La válvula de agua se cerró!")
fin
```
