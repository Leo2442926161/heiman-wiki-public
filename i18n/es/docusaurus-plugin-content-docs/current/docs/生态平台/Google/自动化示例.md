# Ejemplo de automatización

## Rutina "Estoy fuera"

**Activador:** Di "Estoy fuera" en Google Home

**Ejecutar acción:**
- Sensor magnético de puerta Heiman → brazo
- Sensor infrarrojo del cuerpo humano Heiman → Activar la detección de salida de casa
- Apagar todas las luces y electrodomésticos.
- Termostato → Ajustar a temperatura de ahorro de energía
- Google Nest Hub → Mostrar tarjeta de confirmación de salida de casa
- Respuesta de voz: "El modo Salir de casa se ha activado, que tengas un buen viaje"

## Transmisión de "detección de humo"

**Activador:** Alarma de humo Hyman → Humo detectado

**Ejecutar acción:**
- Google Nest Speaker → Transmisión para toda la casa "⚠️ ¡Se detectó humo! ¡Compruébelo de inmediato!"
- Todas las luces inteligentes → Encendido (a través del conjunto de luces inteligentes de Google Home)
- Notificaciones push a los teléfonos móviles de los miembros de la familia.
- Cámara → Iniciar grabación

## Modo hora de dormir

**Activador:** Di "Buenas noches" a Google Home

**Ejecutar acción:**
- Sensor de puerta Heiman → Activa la alarma de puerta abierta
- Infrarrojos del cuerpo humano Heiman → activar la detección nocturna
- Luces → se apagan gradualmente
- Respuesta de voz: "Buenas noches, el sistema de seguridad está activado"
