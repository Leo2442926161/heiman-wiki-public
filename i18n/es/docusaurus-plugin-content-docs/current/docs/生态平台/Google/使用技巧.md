# Consejos de uso

## Sincronización de dispositivos

- Después de que el dispositivo Heiman esté conectado a Google Home a través de Matter, se sincronizará automáticamente con la aplicación Google Home.
- Si no se muestra el dispositivo, haga clic en "Configuración" → "Dispositivos" → "Agregar dispositivo" → "Dispositivos compatibles con Matter" en la aplicación Google Home
- Si se pierde el código QR de Matter, se puede agregar manualmente usando el código de configuración de 11 dígitos en su dispositivo

## Optimización de rutina

- Se agregó "Anunciar el clima y confirmar el estado de armado" a la rutina "Salir de casa"
- Utilice la función "Transmisión de voz" para notificar a toda la casa cuando se produzca una alarma de humo.
- Utilice la acción "Iniciar/Parar" para controlar la activación y desactivación del estado de detección del sensor Heiman.

## Gestión de ubicación del dispositivo

- Asigne habitaciones específicas a sensores Heimann (como "Alarma de humo en la cocina") para admitir un control por voz más preciso
- Incorpore múltiples dispositivos Heiman al "Grupo de dispositivos domésticos" para habilitar el encendido y apagado completo con un solo clic

## Requisitos de red

- Los dispositivos Matter over Thread requieren Google Nest Hub (2.ª generación) o Nest Hub Max como enrutador fronterizo
- Los dispositivos Zigbee se conectan a través de Google Nest Connect o Hub compatible
