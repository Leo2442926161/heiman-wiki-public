# Ejemplo de automatización

## Alarma de seguridad de detección de humo

**Configuración automatizada de HomeKit:**

1. **Condición de activación:** Alarma de humo Hyman → humo detectado
2. **Configuración de seguridad:** Cambie el estado del sistema de seguridad a "Alarma"
3. **Ejecutar acción:**
   - HomePod → Reproducir audio personalizado "Se detectó humo, preste atención a la seguridad"
   - Todas las luces inteligentes → se encienden y parpadean
   - Notificaciones push a todos los miembros de la familia.

## Escena en casa/fuera de casa

**Comando de acceso directo: "Salir del control de seguridad del hogar"**

```
cuando la última persona sale de casa
  Configurar sistema de seguridad → armar al salir de casa
  Configurar el sensor de puerta Heiman → Activar notificaciones
  Configurar el sensor corporal Heiman → activar la alarma
  Apaga todas las luces
  Configuración Termostato → Modo de ahorro de energía
  Enviar notificación "🏠 Modo fuera de casa activado"
```

## Recordatorio de anomalía de temperatura

**Condición de activación:** Sensor de temperatura y humedad Hyman → Temperatura > 35°C

**Ejecutar acción:**
- HomePod → La temperatura ambiente de transmisión de voz excede el límite
- Notificaciones push al iPhone
- Enciende automáticamente el aire acondicionado (si HomeBridge está configurado)
