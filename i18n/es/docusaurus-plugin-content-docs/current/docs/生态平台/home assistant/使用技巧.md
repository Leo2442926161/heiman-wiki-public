# Consejos de uso

## Configuración del dispositivo

- Al emparejar su dispositivo Heiman Matter por primera vez, asegúrese de que su enrutador fronterizo (por ejemplo, HomePod, Nest Hub, Home Assistant Green) esté en la misma red Thread
- Al integrar dispositivos Zigbee a través de ZHA o Zigbee2MQTT, se recomienda utilizar primero el firmware recomendado del coordinador USB (como Sonoff ZBDongle-E, Conbee II)

## Optimización automatizada

- Se agregó el parámetro `for:` (por ejemplo, 3 segundos) a la automatización de la alarma de humo para evitar falsas alarmas instantáneas.
- Utilice `condición: estado` para hacer coincidir el modo de seguridad para evitar notificaciones innecesarias en el estado armado
- Utilice la función "grupo" de Home Assistant para agrupar los sensores Heiman por área para una fácil operación por lotes

## Solución de problemas

- Cuando el dispositivo esté fuera de línea, primero verifique si el canal de red Zigbee entra en conflicto con WiFi 2.4GHz
- Cuando no se pueda descubrir el dispositivo Matter, reinicie el enrutador fronterizo y vuelva a agregar el dispositivo
