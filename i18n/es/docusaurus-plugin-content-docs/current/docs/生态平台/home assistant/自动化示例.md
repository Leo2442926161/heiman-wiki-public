# Ejemplo de automatización

## La alarma de humo activa luces en toda la casa

```yaml
alias: "Alarma de humo - luces y notificaciones en toda la casa"
desencadenantes:
  - disparador: estado
    id_entidad: sensor_binario.heiman_smoke_alarm
    a: "encendido"
acciones:
  - acción: light.turn_on
    objetivo:
      id_entidad: todos
  - acción: notificar.mobile_app_phone
    datos:
      mensaje: "⚠️ ¡Se activó la alarma de humo! ¡Compruébelo de inmediato!"
      título: "Alarma de humo"
```

## Enlace de alarma de CO

```yaml
alias: "Alarma de CO: cierre el gas y abra las ventanas"
desencadenantes:
  - disparador: estado
    id_entidad: sensor_binario.heiman_co_alarm
    a: "encendido"
acciones:
  - acción: switch.turn_off
    entidad_id: switch.gas_fireplace
  - acción: cubrir.abrir
    entidad_id: cubierta.living_room_window
  - acción: notificar.notificación_persistente
    datos:
      mensaje: "Se ha disparado la alarma de monóxido de carbono, se ha cerrado la válvula de gas y se han abierto las ventanillas"
```

## Conexión del detector de inmersión en agua

```yaml
alias: "Detección de inundaciones: cerrar la válvula de agua"
desencadenantes:
  - disparador: estado
    id_entidad: sensor_binario.heiman_water_leak
    a: "encendido"
acciones:
  - acción: switch.turn_off
    entidad_id: switch.main_water_valve
  - acción: notificar.mobile_app_phone
    datos:
      mensaje: "💧 ¡Se detectó una fuga de agua! ¡La válvula principal de agua se ha cerrado!"
```

## Armado fuera de casa: el sensor del cuerpo humano activa la alarma

```yaml
alias: "Modo Salir de casa - alarma del sensor del cuerpo humano"
desencadenantes:
  - disparador: estado
    id_entidad: sensor_binario.heiman_motion_sensor
    a: "encendido"
condiciones:
  - condición: estado
    entidad_id: alarm_control_panel.home_alarm
    estado: armado_fuera
acciones:
  - acción: alarm_control_panel.trigger
    entidad_id: alarm_control_panel.home_alarm
```
