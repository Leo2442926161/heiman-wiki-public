# Configuración del panel

## Panel de descripción general de seguridad

Utilice el diseño de vista en sección para organizar el equipo Heimann en áreas:

```yaml
tipo: sección
título: Alarma de humo y CO
tarjetas:
  - tipo: entidad
    entidad: binario_sensor.heiman_smoke_alarm
    nombre: alarma de humo
  - tipo: entidad
    entidad: binario_sensor.heiman_co_alarm
    nombre: alarma de CO
```

## Estilo de tarjeta de hongo

```yaml
tipo: personalizado: tarjeta-de-panel-de-control-de-alarma-de-hongo
entidad: alarm_control_panel.home_alarm
nombre: sistema de seguridad

tipo: personalizado: tarjeta de entidad de hongo
entidad: binario_sensor.heiman_smoke_alarm
nombre: alarma de humo
icono: mdi: detector de humo
información_primaria: estado
información_secundaria: último cambio
fill_container: verdadero
```

## Panel de sensores ambientales

```yaml
tipo: sección
título: Monitoreo Ambiental
tarjetas:
  - tipo: calibre
    entidad: sensor.heiman_temperature
    nombre: temperatura
    unidad: "°C"
    mínimo: 0
    máx: 50
    gravedad:
      verde: 0
      amarillo: 30
      Rojo: 40
  - tipo: calibre
    entidad: sensor.heiman_humidity
    nombre: humedad
    unidad: "%"
    mínimo: 0
    máx: 100
    gravedad:
      verde: 30
      Amarillo: 60
      Rojo: 80
```

## Panel de historial de alarmas

```yaml
tipo: gráfico histórico
título: Historia del sensor
entidades:
  - binario_sensor.heiman_smoke_alarm
  - binario_sensor.heiman_co_alarm
  - binario_sensor.heiman_water_leak
horas_para_mostrar: 24
```

## Diseño sugerido

- Arriba: Estado del sistema de seguridad + tarjeta de estado del dispositivo de alarma
- Central: Monitoreo ambiental (temperatura, humedad, calidad del agua)
- Abajo: gráfico de tendencias históricas de alarmas de sensores
- Operaciones de acceso directo de la barra lateral: armar/desarmar, cambio de escena
