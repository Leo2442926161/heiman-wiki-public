# Consejos de uso

## Compatibilidad del concentrador

Los dispositivos Hyman Zigbee son compatibles con **SmartThings Hub v2** y **Hub v3 (Aeotec)**.

| Modelo de cubo | Compatibilidad | Descripción |
|---------|--------|------|
| Centro v2 (STH-ETH-250) | ✅ | Requiere actualización de firmware a la última versión |
| Hub v3 (Aeotec) | ✅ | Soporte nativo, recomendado |
| Estación Samsung SmartThings | ⚠️ | Solo admite algunos dispositivos Zigbee |

## Indicaciones de emparejamiento de dispositivos

- Haga clic en "+" → "Dispositivo" → "Marca" → "Hyman" en la aplicación SmartThings
- Los dispositivos que no se reconocen automáticamente se pueden buscar manualmente a través de "Scan Zigbee"
- Se recomienda mantener la distancia entre el dispositivo y el Hub dentro de los 5 metros para el primer emparejamiento.

## Optimización de la batería

El intervalo de informe predeterminado del sensor Heiman es más largo. Si se requiere rendimiento en tiempo real, el "intervalo de detección" se puede ajustar a 5 minutos en la página del dispositivo.

## Mejores prácticas de automatización

- Utilice la función "Modos" de SmartThings para diferenciar entre estados "en casa/fuera de casa"
- Crear **dos conjuntos** de automatización para sensores clave (humo, CO) para redundancia
