# Ejemplo de automatización

## Modo de hora de dormir "Buenas noches"

**Condición de activación:** Sensor magnético de puerta Haiman → Cerrar (lo que indica que la puerta del dormitorio se ha cerrado)

**Ejecutar acción:**
- Apagar todas las luces del salón.
- Bloqueo inteligente → Bloquear
- Sistema de seguridad → armar por la noche
- Notificación push: "Buenas noches, el sistema está armado"

## Modo "Ausente"

**Condición de activación:** Sensor infrarrojo del cuerpo humano Hyman → 15 minutos sin movimiento

**Ejecutar acción:**
- Todas las luces apagadas
- Termostato → Modo de ahorro de energía
- Sistema de seguridad → armado fuera de casa
- Sensor de puerta → encender alarma
- Notificación push: "El modo fuera de casa está activado, los sensores de puertas y ventanas están habilitados"

## Modo de emergencia de alarma de humo

**Condición de activación:** Alarma de humo Hyman → Humo detectado

**Ejecutar acción:**
- Altavoz inteligente → reproducir sonido de alarma
- Todas las luces inteligentes → parpadeando en rojo
- Notificaciones push a todos los miembros de la familia.
- Si está conectado a SmartThings Hub → activar sirena
