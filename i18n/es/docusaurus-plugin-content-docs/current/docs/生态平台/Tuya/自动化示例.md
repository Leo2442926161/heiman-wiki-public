# Ejemplo de automatización

## Escena de alarma de humo

**Nombre de la escena:** "Conexión de alarma de humo"

**Condición de activación:** Alarma de humo Hyman → Humo detectado

**Ejecutar acción:**
- Enchufe inteligente → abierto (conectar el extractor de aire)
- Luces → se encienden y parpadean
- Notificaciones push al teléfono móvil.
- Altavoz → Reproducir sonido de alarma

## Escena con un clic al salir de casa (Tocar para ejecutar)

**Nombre de la escena:** "Arreglo después de salir de casa"

**Ejecución manual:** Toque o disparador de voz

**Ejecutar acción:**
- Sensor de puerta Haiman → encender alarma
- Hyman Human Body Infrarrojos → Activar detección
- Detector de inmersión en agua Hyman → activar la detección
- Todas las luces → apagadas
- Bloqueo inteligente → Bloquear

## Control automático de temperatura y humedad.

**Tipo de automatización:** Activador de condición automático

**Condición de activación:** Sensor de temperatura y humedad Hyman Humedad > 75%

**Ejecutar acción:**
- Enchufe inteligente (deshumidificador) → Encendido
- Notificación push: "La humedad es demasiado alta, el deshumidificador se ha encendido"

**Condición de activación:** Sensor de temperatura y humedad Hyman Humedad < 50%

**Ejecutar acción:**
- Enchufe inteligente (deshumidificador) → Cerrar
