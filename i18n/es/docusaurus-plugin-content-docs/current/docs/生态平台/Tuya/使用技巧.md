# Consejos de uso

## Tocar y ejecutar frente a automatización

| Tipo | Método de activación | Escenarios aplicables |
|------|----------|----------|
| **Tocar para ejecutar** | Clic manual/voz | Salir de casa, volver a casa, modo de visualización de películas, etc. |
| **Automatización** | Activación automática del sensor | Alarma de humo, detección de fugas de agua, etc. |

El equipo tipo sensor Heiman debe usar "Automatización" y los requisitos de tipo de escena deben usar "Tocar para ejecutar".

## Nube a local

- Active la "Comunicación LAN" en la aplicación Smart Life para reducir la dependencia de la nube
- Algunos dispositivos Heiman admiten la ejecución local y se pueden vincular incluso si la red está desconectada.
- Ruta de configuración: Escena → Configuración en la esquina superior derecha → "Habilitar ejecución local"

## Compartir dispositivo

- Comparta dispositivos Heiman con miembros de la familia en la aplicación Smart Life
- Los usuarios compartidos pueden recibir notificaciones de alarma pero no pueden modificar las configuraciones del dispositivo
- Compartir con hasta 20 miembros de la familia

## Acceso a plataformas de terceros

Las cuentas de Tuya se pueden vincular a Alexa y Google Home para lograr un control de voz multiplataforma. Ruta vinculante: Smart Life → Configuración → Control de terceros.
