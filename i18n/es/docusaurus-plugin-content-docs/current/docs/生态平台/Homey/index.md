#hogareño

Homey es un centro de hogar inteligente lanzado por la empresa holandesa Athom, que admite Zigbee, Z-Wave, Wi-Fi, Matter, Thread y otros protocolos.

## Método de integración

Los dispositivos Homey Zigbee y Z-Wave se emparejan directamente a través del chip inalámbrico integrado de Homey, eliminando la necesidad de una puerta de enlace adicional. Se puede acceder a los dispositivos Matter a través del enrutador de borde Thread integrado de Homey Pro.

## Ventajas

- **Múltiples protocolos en uno**: Homey Pro admite Zigbee, Z-Wave, Thread y Wi-Fi al mismo tiempo
- **Ejecutar localmente**: toda la automatización se ejecuta localmente
- **Automatización de flujo**: programación de arrastrar y soltar para crear escenarios de vinculación complejos

> **Referencia**: Las aplicaciones relacionadas con Heiman se pueden buscar en Homey App Store
