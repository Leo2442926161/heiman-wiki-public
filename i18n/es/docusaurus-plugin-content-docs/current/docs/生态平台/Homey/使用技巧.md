# Consejos de uso

## Interferencia del canal Zigbee

El chip Zigbee integrado de Homey utiliza canales específicos. Si el dispositivo Heiman se desconecta con frecuencia:

1. Vea el canal actual en Configuración hogareña → Zigbee
2. Utilice la aplicación Homey Insight para buscar interferencias ambientales
3. Cambia a un canal con menos interferencias (lejos del WiFi 6/01/11)

## Red de distribución de equipos

- Cómo ingresar al modo de emparejamiento para dispositivos Heiman: presione y mantenga presionado el botón durante 5 segundos (algunos modelos necesitan presionar 3 veces rápidamente)
- Al agregar un dispositivo en la aplicación Homey, seleccione "Agregar dispositivo → Zigbee → Buscar"
- Cuando el emparejamiento falla, acerque el dispositivo a Homey (a menos de 3 metros) e inténtelo nuevamente

## Sugerencias de optimización del flujo

- Agregue una condición de "retraso" en el flujo (por ejemplo, 2 segundos) para evitar falsos positivos transitorios
- Utilice la función "etiqueta" para clasificar y gestionar equipos Heiman

## Ampliación del rango de señal

Si el dispositivo Heiman está lejos de Homey, se puede agregar un dispositivo de retransmisión Zigbee (como un enchufe inteligente) para ampliar la cobertura de la red.
