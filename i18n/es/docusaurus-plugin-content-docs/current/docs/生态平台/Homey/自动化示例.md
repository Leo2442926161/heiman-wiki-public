# Ejemplo de automatización

## Alarma de humo: parpadea las luces y notifica

**Configuración de flujo:**

1. **Activador:** Alarma de humo Hyman → El estado cambia a "alerta"
2. **Condiciones:** (Ninguna)
3. **Acción:**
   - Luz del salón → parpadea (roja, una vez por segundo)
   - Notificación push → Móvil: "⚠️ ¡Se activó la alarma de humo!"
   - Anuncio de voz → Altavoz hogareño: "¡Se detectó humo, preste atención a la seguridad!"

## Alarma de CO - Apagar la caldera

**Configuración de flujo:**

1. **Activador:** Alarma de CO Hyman → El estado cambia a "alerta"
2. **Condiciones:** (Ninguna)
3. **Acción:**
   - Interruptor caldera → APAGADO
   - Toma inteligente (sistema de aire fresco) → Encendido
   - Notificación push → "Alarma de monóxido de carbono, la caldera se ha apagado"

## Alarma de exceso de límite de temperatura y humedad

1. **Activador:** Sensor de temperatura y humedad Hyman → Temperatura superior a 40 °C
2. **Acción:**
   - Aire acondicionado → Activar el modo de refrigeración
   - Notificación push → "La temperatura ambiente es demasiado alta, el aire acondicionado se ha encendido automáticamente"
