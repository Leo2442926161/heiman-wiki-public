# Consejos de uso

## Descubrimiento de dispositivos

- Una vez que el dispositivo Heiman Matter esté encendido, la aplicación Alexa lo descubrirá automáticamente en un plazo de 30 segundos.
- Si no se descubre automáticamente: Ingrese a la aplicación Alexa → Dispositivo → "+" → "Agregar dispositivo" → "Materia"
- Utilice el código de configuración de Matter o el código QR en la parte inferior del dispositivo para completar el emparejamiento

## Acceso al dispositivo Zigbee (Echo Plus / Echo Show)

- Los dispositivos Echo con Zigbee Hub pueden emparejarse directamente con sensores Heiman Zigbee
- Modo de emparejamiento: Aplicación Alexa → Dispositivo → "+" → "Zigbee" → "Buscar dispositivo"
- Los dispositivos Heimann son compatibles con Alexa con el coordinador Zigbee integrado

## Consejos de rutina

- Utilice "acciones personalizadas" para transmitir indicaciones de voz más detalladas
- Agregue "esperar 5 segundos" a la rutina de emergencia antes de realizar acciones posteriores para evitar la superposición de voces.
- Aproveche la función "Detección de sonido" para brindarle a Alexa respuestas adicionales a las alarmas agudas de Heyman Alarm

## Guardia Alexa NOTA

Actualmente, Alexa Guard Plus solo está disponible en los Estados Unidos. Los usuarios de China pueden simular algunas funciones mediante Rutina.
