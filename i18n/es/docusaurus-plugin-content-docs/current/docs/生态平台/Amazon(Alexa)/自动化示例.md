# Ejemplo de automatización

## Integración de Alexa Guard (EE. UU.)

**Requisito previo:** Suscríbete a Alexa Guard Plus (disponible en EE. UU.)

**Proceso de configuración:**
1. Active Guard Plus en la aplicación Alexa
2. Agregue la alarma de humo/CO Hyman como un "dispositivo doméstico inteligente"
3. Habilite la respuesta inteligente "Detección de alarma de humo"

**Respuesta automática:**
- Alarma de humo activada → Alexa marca automáticamente los contactos de emergencia
- Activador de alarma de CO → Aviso por voz de Alexa para toda la casa, ventilación automática
- Las puertas y ventanas se abren de forma anormal → Notificación de alarma push

## Rutina de detección de humo

**Activador:** Alarma de humo Hyman → Humo detectado

**Ejecutar acción:**
- Anuncio de voz de Alexa: "⚠️ Se detectó humo, preste atención a la seguridad"
- Todos los dispositivos Echo → Reproducir tono de alarma
- Conjunto de luces inteligentes → todo iluminado
- Notificaciones push al teléfono móvil.

## Rutina "Salir de casa"

**Activador:** Comando de voz "Alexa, me voy"

**Ejecutar acción:**
- Sensor magnético de puerta Heiman → activar armado
- Sensor corporal Heiman → Activar detección de movimiento
- Detector de inmersión en agua Hyman → activar la detección
- Todas las luces → apagadas
- Alexa respondió: "El sistema de seguridad está activado, que tengas un buen viaje"
