#AmazonAlexa

Amazon Alexa es una de las plataformas de asistente de voz inteligente más populares del mundo y admite el protocolo Matter y un rico ecosistema de Skills.

## Método de integración

### Equipo de materia

Se puede acceder a los dispositivos Hyman Matter over Thread a través de dispositivos Echo habilitados para Matter (Echo de cuarta generación, Echo Plus, Echo Show 10, etc.):

1. Asegúrese de que su dispositivo Echo esté actualizado con el firmware más reciente
2. Abra la aplicación Alexa → Dispositivos → Agregar dispositivo
3. Seleccione "Materia" y escanee el código de emparejamiento.

### Dispositivos Zigbee

Algunos Echo Plus y Echo Show tienen chips Zigbee integrados y se pueden emparejar directamente con sensores Heiman Zigbee.

## Ejemplo de comando de voz

- "Alexa, pregunta al detector de humo si todo está bien"
- "Alexa, ¿cuál es la temperatura en la sala?"

> **Referencia**: Los sensores Hyman Matter están disponibles en la tienda oficial de Amazon
