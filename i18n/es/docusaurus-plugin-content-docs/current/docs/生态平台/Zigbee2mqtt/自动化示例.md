# Ejemplo de automatización

## Estado del dispositivo de suscripción MQTT

```golpecito
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
```

Ejemplo de salida:
```
zigbee2mqtt/heiman_smoke {"humo":true,"batería":100,"linkquality":120}
```

## Flujo de alarma de humo Node-RED

```json
[{"id":"1","type":"mqtt in","topic":"zigbee2mqtt/heiman_smoke","name":"Alarma de humo","qos":1,"broker":"localhost","x":200, "y":200,"wires":[["2"]]},{"id":"2","type":"switch","name":"Detectar humo","property":"payload.smoke ","conditions":[{"t":"eq","v":"true"}],"x":350,"y":200,"wires":[["3"]]},{"id":"3","type":" notificación","name":"Notificación push","title":"Alarma de humo","message":"¡Se detectó humo ","x":500,"y":200,"wires":[]}]
```

## Monitoreo en línea de equipos

```yaml
# configuración.yaml
sensor_binario:
  - plataforma: mqtt
    nombre: "Alarma de humo en línea"
    state_topic: "zigbee2mqtt/heiman_smoke/disponibilidad"
    payload_on: "en línea"
    payload_off: "fuera de línea"
    clase_dispositivo: conectividad
```
