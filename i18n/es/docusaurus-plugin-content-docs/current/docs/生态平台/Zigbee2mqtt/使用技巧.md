# Consejos de uso

## Selección de canal

Los dispositivos Hyman Zigbee utilizan el canal 15 de forma predeterminada. Se recomienda especificar manualmente en `configuration.yaml`:

```yaml
avanzado:
  canal: 15
```

Evite la superposición con el canal WiFi 2.4GHz 1/6/11. Utilice un analizador WiFi para escanear y seleccionar el canal Zigbee con la menor interferencia.

## Copia de seguridad de la clave de red

**Asegúrese de hacer una copia de seguridad de `coordinator_backup.txt` y `secret.yaml`: **

```golpecito
cp datos/coordinator_backup.txt datos/coordinator_backup.bak
cp datos/secret.yaml datos/secret.yaml.bak
```

Todos los dispositivos emparejados deben volver a unirse después de perder la clave.

## Optimizaciones específicas del dispositivo

- Los dispositivos Heiman informan la carga de la batería cada 60 minutos de forma predeterminada y el intervalo se puede acortar mediante la configuración de "informes".
- Se agregó la configuración `homeassistant: true` para generar entidades automáticamente sin escribir manualmente la configuración de descubrimiento MQTT
- Utilice `friendly_name` para establecer un nombre legible por humanos para el dispositivo Heiman

## Actualizar firmware

Algunos dispositivos Heiman Zigbee admiten actualizaciones OTA y la opción "ota" debe estar activada en "configuration.yaml".
