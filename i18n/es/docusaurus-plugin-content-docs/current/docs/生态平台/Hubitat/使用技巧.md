# Consejos de uso

## Controlador personalizado

Los dispositivos Heimann están optimizados para ser compatibles en Hubitat con los siguientes controladores:

| Tipo de dispositivo | Controlador recomendado |
|----------|----------|
| Alarma de humo/CO | Sensor genérico de humo/CO Zigbee |
| Sensor infrarrojo del cuerpo humano | Sensor de movimiento Heiman (de terceros) |
| Sensor magnético de puerta | Sensor de contacto Zigbee genérico |
| Sensor de fuga de agua | Sensor de fugas de agua Zigbee genérico |
| Sensor de temperatura y humedad | Sensor de temperatura/humedad Heiman |

## Optimización de Zigbee

- Los modelos Hubitat C-8 y superiores utilizan el chip EFR32, que tiene la mejor compatibilidad
- Ver y seleccionar el canal con menor interferencia en Configuración → Zigbee
- Utilice herramientas de rastreo Zigbee como Ubisys para diagnosticar llamadas perdidas

## Indicaciones de reglas

- Se recomienda agregar la condición "Retraso contra falsas alarmas" (0,5-2 segundos) en Rule Machine
- Habilite notificaciones de "verificación de estado" para reglas clave para garantizar que continúen ejecutándose
- Organice una automatización compleja en modo "sub-regla" para facilitar la depuración y el mantenimiento

## Coexistencia Z-Wave

Si desea utilizar dispositivos Z-Wave al mismo tiempo, asegúrese de mantener el espacio de frecuencia entre los canales Zigbee y Z-Wave.
