# Ejemplo de automatización

## Alarma de humo de máquina de reglas

**Nombre de la regla:** Enlace de seguridad de la alarma de humo

**Configuración:**
```
Activador: La alarma de humo Hyman se activa
Condiciones: Modo != Desarmado
Acción:
  - Enciende todas las luces.
  - Enviar notificación push "⚠️ ¡Alarma de humo!"
  - Enciende la sirena (si está disponible)
  - Establecer el modo en "Emergencia"
```

## Monitoreo de temperatura de la máquina de reglas

**Nombre de la regla:** Protección contra altas temperaturas

**Configuración:**
```
Activador: Sensor de temperatura Heimann > 38°C
Condición: Estado del aire acondicionado = apagado
Acción:
  - Encender el aire acondicionado, modo refrigeración, 24°C.
  - Enviar notificación "La temperatura ambiente es demasiado alta y se ha encendido el aire acondicionado"
```

## Detección de fugas de agua en la máquina de reglas

**Nombre de la regla:** Tratamiento de emergencia de fugas de agua

**Configuración:**
```
Activador: El detector de inmersión en agua Hyman se moja
Condición: Estado de la válvula de agua = abierta
Acción:
  - Cerrar la válvula principal de agua.
  - Enviar notificaciones push a todos los teléfonos móviles
  - Enviar notificación por SMS (opcional)
```
