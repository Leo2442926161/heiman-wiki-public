# Ejemplo de automatización

## Conexión de alarma de humo

```yaml
alias: "Conexión de alarma de humo ZHA"
desencadenantes:
  - disparador: estado
    entidad_id: binario_sensor.heiman_smoke_zha
    a: "encendido"
acciones:
  - acción: light.turn_on
    objetivo:
      area_id: salón_sala
  - acción: tts.speak
    objetivo:
      entidad_id: media_player.speaker
    datos:
      mensaje: "¡Alarma de humo activada!"
```

## Automatización de emparejamiento de dispositivos

```yaml
alias: "Recordatorio de emparejamiento de nuevo dispositivo"
desencadenantes:
  - disparador: zha.event
    subtipo: heiman
    comando: dispositivo_unido
acciones:
  - acción: notificar.notificación_persistente
    datos:
      mensaje: "El nuevo dispositivo Haiman se ha unido a la red ZHA, vaya al nombre y área de configuración"
```

## Detección de inmersión en agua

```yaml
alias: "Prueba de inundación ZHA"
desencadenantes:
  - disparador: estado
    entidad_id: binario_sensor.heiman_water_leak_zha
    a: "encendido"
acciones:
  - acción: switch.turn_off
    entidad_id: switch.main_water_valve
  - acción: notificar.mobile_app_phone
    datos:
      mensaje: "💧 ¡Se detectó inundación!"
```
