# Consejos de uso

## Emparejamiento de dispositivos

1. Ingrese a "Configuración → Dispositivos y servicios → ZHA → Configuración" en Home Assistant
2. Haga clic en "Agregar dispositivo" y luego inicie el modo de emparejamiento del dispositivo Heiman.
3. Si el emparejamiento falla, acerque el dispositivo al coordinador USB e inténtelo nuevamente dentro de los 3 metros

## Coordinación de canales

El canal predeterminado de ZHA es 15. Si hay interferencias graves con WiFi de 2,4 GHz:
- Utilice `Silicon Labs Flasher Utility` para modificar el canal
- Canales comúnmente recomendados: 15, 20, 25 (evite WiFi 6/01/11)

## Optimización de la red

- Agregue un dispositivo de enrutamiento Zigbee (como un enchufe inteligente) cada 3 o 4 dispositivos Heiman para expandir la red
- Verifique la calidad de la conexión del dispositivo con la herramienta de visualización de red ZHA
- Verifique periódicamente si hay actualizaciones de firmware del dispositivo para mantener una compatibilidad óptima
