# Funciones de Markdown

La documentación técnica de Heiman es totalmente compatible con la sintaxis de Markdown y funciones extendidas, lo que facilita la redacción de documentación de productos estructurada.

## Ejemplo de Código

Ejemplo de integración de API para desarrollo de productos:

```jsx title="Integración de API de Detector de Humo"
const heimanClient = new HeimanClient({
  apiKey: 'su-api-key',
  deviceId: 'HS1SA-xxxxx',
});

// Obtener estado del dispositivo
const status = await heimanClient.getDeviceStatus();
console.log(status);
```

## Advertencias

Use advertencias para resaltar información importante:

:::tip

El detector de humo Heiman HS1SA tiene certificación EN14604

:::

:::caution

Al instalar alarmas de gas, siga la guía de instalación para un cableado adecuado

:::

:::danger

En caso de fuga de gas, no opere ningún interruptor eléctrico. Abra las ventanas inmediatamente y evacue el área

:::
