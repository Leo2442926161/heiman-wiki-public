# Productos

Heiman Technology ofrece cuatro líneas de productos principales que abarcan Seguridad contra Incendios Inteligente, Seguridad de Gas Inteligente, Cuidado de Ancianos Inteligente y Sensores Inteligentes.

## Seguridad contra Incendios Inteligente

Incluye detectores de humo, alarmas de monóxido de carbono, detectores de calor, etc., compatibles con conectividad de red celular (4G/NB-IoT) para facilitar la integración con plataformas IoT de terceros.

Características clave:

- Compatibilidad total de red, integración multiplataforma
- Cumple con EN54-7, UL 217 y otros estándares internacionales
- Compatible con el protocolo Matter, funciona con Alexa, Google Home, Apple HomeKit

## Seguridad de Gas Inteligente

Sistemas inteligentes de alarma de gas industriales y comerciales, que proporcionan soluciones integrales de seguridad de gas inteligente para diversos entornos.

Características clave:

- Sensores de detección de gas de alta precisión
- Monitoreo remoto en tiempo real y alarma
- Activación automática de válvulas de cierre de emergencia de gas

## Cuidado de Ancianos Inteligente

Incluye detectores de humo, alarmas de gas, detectores de fugas de agua, sensores de caídas, dispositivos de seguridad para puertas y alarmas de emergencia.

Características clave:

- Algoritmo de detección de caídas basado en IA
- Llamada de emergencia con un solo botón
- Notificación a múltiples dispositivos para familiares y agencias de servicio

## Sensores Inteligentes

Cubren sensores de escenario completo que incluyen detección infrarroja, sensores magnéticos de puerta/ventana, monitoreo de temperatura/humedad y detección de calidad del aire.

Características clave:

- Diseño de ultra bajo consumo
- Compatible con protocolos múltiples Zigbee / Matter / WiFi
- Compatible con las principales plataformas de hogar inteligente
