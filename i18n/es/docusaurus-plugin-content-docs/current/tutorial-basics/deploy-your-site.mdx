# Implementación de Soluciones

Heiman Technology proporciona una guía completa de implementación de soluciones de seguridad inteligente de extremo a extremo.

## Proceso de Integración

Los productos Heiman admiten múltiples métodos de implementación, incluida la implementación local independiente y la integración en la nube.

### Implementación Local

```bash
# Configurar Gateway de Seguridad contra Incendios Inteligente Heiman
heiman-cli configure --gateway HS1GW --mode local

# Registrar dispositivo detector
heiman-cli register --device HS1SA --gateway HS1GW
```

### Integración en la Nube

Los productos Heiman están integrados con las principales plataformas IoT:

- China Telecom Tianyi
- China Mobile OneNET
- Huawei HiLink
- Alibaba Cloud IoT
- Tuya Smart

## Soporte del Protocolo Matter

Los productos de la serie Matter de Heiman son compatibles con:

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

Logrando interconectividad entre marcas y ecosistemas.

## Verificación de Implementación

Después de la implementación, use la aplicación Heiman Smart Fire o la plataforma de gestión para la verificación funcional y el monitoreo en tiempo real.
