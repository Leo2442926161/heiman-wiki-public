# Другие сертификаты

## Знак качества Q-Label

Q-Label — это знак сертификации качества для европейской индустрии умного дома, запущенный альянсом Q-Lab Alliance (теперь он входит в состав Zigbee Alliance/CSA Connectivity Standards Alliance).

| Проект | Описание |
|------|------|
| Область применения | Сертификация совместимости устройств Zigbee |
| Содержание сертификации | Совместимость устройства со шлюзами/контроллерами Zigbee других марок |
| Значение | Обеспечить межбрендовую совместимость и снизить риск «купить домой и не использовать» |

Продукты Hyman Zigbee сертифицированы Q-Label, что гарантирует совместимость со следующими платформами:
- Amazon Echo Plus/Echo Show (встроенный Zigbee)
- Philips Hue Bridge
- Центр SmartThings
- Другие шлюзы аутентификации Q-Label.

Компания Haiman прошла сертификацию системы менеджмента качества ISO 9001:2015, обеспечивающую полный контроль качества: от исследований и разработок, производства до послепродажного обслуживания.
