# Европейский стандарт EN

EN (Европейская норма, Европейский стандарт) — это технический стандарт, разработанный Европейским комитетом по стандартизации (CEN) и Европейским комитетом по электротехнической стандартизации (CENELEC). Основные стандарты EN, используемые в продукции Heiman, следующие:

## EN 14604 — Стандарт для датчиков дыма

| Проект | Описание |
|------|------|
| Область применения | Независимая дымовая сигнализация |
| Требования к обнаружению | Фотоэлектрический детектор дыма, звуковая и световая сигнализация, предупреждение о низком заряде батареи |
| Применимые рынки | Государства-члены ЕС, Великобритания, Швейцария, Норвегия и др. |
| Хейман продукты | Серия HS1SA-E, HS3SA, HS1SA-M, S1 и т. д. |
| Значение сертификации | Соблюдение требований правил пожарной безопасности ЕС является необходимым условием доступа на европейский рынок |

**Тестовые параметры**: время отклика, уровень звукового давления (≥85 дБ/3 м), функция отключения звука, срок службы батареи, температурный цикл, устойчивость к влажности, устойчивость к радиочастотным помехам и т. д.

## EN 50291 — Стандарт сигнализации о угарном газе

| Проект | Описание |
|------|------|
| Область применения | Бытовая сигнализация угарного газа (CO) |
| Требования к тестированию | Электрохимический датчик CO, сигнализация классификации концентрации |
| Применимые рынки | Государства-члены ЕС, Великобритания |
| Хейман продукты | HS1CA-E, серия C1, HS1CA-M и т. д. |
| Значение сертификации | Чтобы обеспечить надежность обнаружения утечек CO, в некоторых странах установка является обязательной |

## Другие соответствующие стандарты EN

- **EN 301 489**: Электромагнитная совместимость (ЭМС).
- **EN 300 220**: Радиоспектр устройств малого радиуса действия (SRD).
- **EN 62479**: Оценка воздействия на человека низкочастотных электромагнитных полей.
