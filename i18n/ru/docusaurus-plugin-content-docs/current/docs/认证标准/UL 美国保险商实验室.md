# Лаборатории страховщиков UL

UL (Underwriters Laboratories) — наиболее авторитетная организация по разработке и сертификации стандартов безопасности продукции в США. Основные стандарты UL, используемые в продукции Heiman, следующие:

## UL 217 — Стандарт дымовой сигнализации (США)

| Проект | Описание |
|------|------|
| Область применения | Бытовые датчики дыма (рынок США) |
| Требования к обнаружению | Фотоэлектрический/ионизационный детектор дыма, звуковая и световая сигнализация, выше 85 дБ |
| Применимые рынки | США, Канада |
| Хейман Продукты | HS8SA (версия для США) |
| Значение сертификации | Соблюдение противопожарных правил различных штатов США является необходимым условием доступа на рынок Северной Америки |

**Отличия от других стандартов**: Сценарии испытаний UL 217 включают типичные американские сценарии бытовых пожаров, такие как вспышки и медленное возгорание, и немного отличаются от методов испытаний EN 14604.

## UL 2034 — Стандарт сигнализации о угарном газе (США)

| Проект | Описание |
|------|------|
| Область применения | Бытовая сигнализация CO (рынок США) |
| Требования к обнаружению | Электрохимический датчик, алгоритм интегрирования концентрации × времени |
| Применимые рынки | США, Канада |
| Продукты Хеймана | HS-720ES-M (версия для США) |
| Значение сертификации | В большинстве штатов США обязательна установка сигнализаторов CO |

## Основные различия между сертификацией UL и сертификацией EN

| Аспекты | УЛ | RU |
|------|-----|-----|
| Стандарты дымовой сигнализации | УЛ 217 | EN 14604 |
| Стандарты сигнализации CO | УЛ 2034 | EN 50291 |
| Метод обнаружения | Специальное испытание кривой пожароопасности | Различные испытания на огнестойкость |
| Применимое напряжение | 120 В / аккумулятор | 230 В / аккумулятор |
