# CE Соответствие ЕС

Знак CE (Conformité Européenne, EU Conformity) является обязательным знаком безопасности для продукции, поступающей на рынок ЕС.

## Значение знака CE

Знак CE указывает на то, что продукт соответствует основным требованиям соответствующих директив ЕС и может свободно обращаться в государствах-членах ЕС и Европейской экономической зоне (ЕЭЗ).

## Путь сертификации CE для продукции Haiman

Сигнализаторы дыма/CO компании Hyman соответствуют требованиям CE благодаря:

### Регламент CPR (ЕС № 305/2011)

Постановление о строительных изделиях требует, чтобы изделия пожарной сигнализации проходили оценку стабильности рабочих характеристик:
- Согласно стандартам: EN 14604 (дым), EN 50291 (CO).
- Агентство по оценке: TÜV Rheinland и другие уполномоченные агентства.
- Уровень сертификации: DoP (Декларация качества) + знак CE.

### RED Директива (2014/53/EU)

Директива по радиооборудованию требует, чтобы беспроводное оборудование:
- Охватывает Zigbee, Z-Wave, Wi-Fi, Thread и другие беспроводные модули.
- Объекты испытаний: электромагнитная совместимость (ЭМС), радиочастота (РЧ), SAR, безопасность.

### Директива по ЭМС (2014/30/ЕС)

Директива по электромагнитной совместимости гарантирует, что оборудование не будет создавать помех другому электронному оборудованию.

## Продукты с маркировкой CE

Вся продукция Heymann, продаваемая на рынок ЕС, имеет маркировку CE, в том числе:
- HS1SA-E / HS3SA (дымовая сигнализация Zigbee)
- Серия HS1CA-E/C1 (сигнализация Zigbee CO)
- HS1SA-M/HS1CA-M (Материя превыше нити)
- Полный спектр датчиков Zigbee, таких как HS1MS / HS1DS.
