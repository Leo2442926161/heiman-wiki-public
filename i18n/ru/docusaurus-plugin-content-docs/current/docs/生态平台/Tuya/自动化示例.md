# Пример автоматизации

## Сцена с дымовой сигнализацией

**Название сцены:** «Связь дымовой сигнализации»

**Условие срабатывания:** Датчик дыма Hyman → Обнаружен дым

**Выполнить действие:**
- Умная розетка → открыть (подключить вытяжной вентилятор)
- Свет → включить и мигать
- Push-уведомления на мобильный телефон
- Динамик → Воспроизвести звуковой сигнал

## Сцена выхода из дома в один клик (Tap-to-Run)

**Название сцены:** «Аранжировка после выхода из дома»

**Выполнение вручную:** Нажмите или включите голосовой триггер.

**Выполнить действие:**
- Датчик двери Haiman → включить сигнализацию
- Инфракрасное излучение человеческого тела Hyman → Включить обнаружение
- Детектор погружения в воду Hyman → включить обнаружение
- Все индикаторы → выключены
- Умный замок → Блокировка

## Автоматический контроль температуры и влажности.

**Тип автоматизации:** Автоматический запуск по условию.

**Условие срабатывания:** Датчик температуры и влажности Hyman. Влажность > 75 %.

**Выполнить действие:**
- Умная розетка (осушитель) → Вкл.
- Push-уведомление: «Влажность слишком высокая, включен осушитель»

**Условие срабатывания:** Датчик температуры и влажности Hyman. Влажность < 50 %.

**Выполнить действие:**
- Умная розетка (осушитель) → Закрыть
