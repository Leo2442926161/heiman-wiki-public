# Советы по использованию

## Выбор канала

Устройства Hyman Zigbee по умолчанию используют канал 15. Рекомендуется указать вручную в `configuration.yaml`:

```ямл
продвинутый:
  канал: 15
```

Избегайте перекрытия с каналом Wi-Fi 2,4 ГГц 1/6/11. Используйте анализатор Wi-Fi для сканирования и выбора канала Zigbee с наименьшими помехами.

## Резервное копирование сетевого ключа

**Обязательно сделайте резервные копии `coordinator_backup.txt` и `secret.yaml`: **

``` баш
данные cp/coordinator_backup.txt data/coordinator_backup.bak
данные cp/secret.yaml data/secret.yaml.bak
```

Все сопряженные устройства должны повторно подключиться после потери ключа.

## Оптимизация для конкретного устройства

- Устройства Heiman по умолчанию сообщают о заряде батареи каждые 60 минут, и этот интервал можно сократить с помощью конфигурации «отчетности».
— Добавлена конфигурация «homeassistant: true» для автоматического создания объектов без ручного написания конфигурации обнаружения MQTT.
- Используйте «Friendly_name», чтобы установить удобочитаемое имя для устройства Heiman.

## Обновить прошивку

Некоторые устройства Heiman Zigbee поддерживают обновления OTA, поэтому в файлеconfiguration.yaml необходимо включить опцию ota.
