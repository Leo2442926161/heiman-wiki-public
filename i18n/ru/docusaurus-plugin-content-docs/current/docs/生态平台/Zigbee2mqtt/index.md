#Zigbee2MQTT

Zigbee2MQTT — это программное обеспечение шлюза Zigbee с открытым исходным кодом, которое соединяет устройства Zigbee с протоколом MQTT, обеспечивая интеграцию с такими платформами, как Home Assistant.

## Совместимость

Датчики Heiman Zigbee имеют хорошую совместимость с Zigbee2MQTT, и большинство устройств могут автоматически распознаваться и поддерживать все ключевые функции:

- Сигнализация дыма/угарного газа: состояние тревоги, обнаружение неисправностей, отключение звука, заряд батареи
- Датчики: температуры, влажности, освещенности, обнаружения присутствия
- Датчик двери: состояние открытия/закрытия, обнаружение взлома.

## Рекомендуемое оборудование

- Координаторы серии CC2652P/CC1352P (такие как SLZB-06, TubeZB, ZigStar)
- Шлюзы Zigbee серии Hyman HS6GW также можно использовать вместе
