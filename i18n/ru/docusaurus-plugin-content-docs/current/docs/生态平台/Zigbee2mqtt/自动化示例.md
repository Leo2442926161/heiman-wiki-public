# Пример автоматизации

## Статус устройства подписки MQTT

``` баш
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
```

Пример вывода:
```
zigbee2mqtt/heiman_smoke {"дым": правда, "батарея": 100, "качество связи": 120}
```

## Онлайн-мониторинг оборудования

```ямл
# конфигурация.yaml
двоичный_сенсор:
  - платформа: mqtt
    name: "Дымовая сигнализация онлайн"
    state_topic: "zigbee2mqtt/heman_smoke/availability"
    payload_on: "онлайн"
    payload_off: "оффлайн"
    класс_устройства: возможность подключения
```
