# Пример автоматизации

## Охранная сигнализация обнаружения дыма

**Автоматическая настройка HomeKit:**

1. **Условие срабатывания:** Датчик дыма Hyman → обнаружен дым
2. **Настройки безопасности:** переключите статус системы безопасности на «Тревога».
3. **Выполнить действие:**
   - HomePod → Воспроизвести собственный звук «Обнаружен дым, обратите внимание на безопасность»
   - Все умные фонари → загораются и мигают
   - Push-уведомления всем членам семьи

## Сцена дома/вдали от дома

**Команда быстрого доступа: «Выйти из проверки безопасности дома»**

```
когда последний человек уходит из дома
  Настроить систему безопасности → ставить на охрану при выходе из дома
  Настроить датчик двери Heiman → Включить уведомления
  Настроить датчик тела Heiman → включить сигнализацию
  Выключите все освещение
  Настройки Термостат → Режим энергосбережения
  Отправить уведомление «🏠 Режим «Вдали от дома» активирован»
```

## Напоминание об отклонении температуры

**Условие срабатывания:** Датчик температуры и влажности Hyman → Температура > 35°C

**Выполнить действие:**
- HomePod → Температура в помещении голосовой трансляции превышает допустимый предел
- Push-уведомления на iPhone
- Автоматически включать кондиционер (если настроен HomeBridge)
