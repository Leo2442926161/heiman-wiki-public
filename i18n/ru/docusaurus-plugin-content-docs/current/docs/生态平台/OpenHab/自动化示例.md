# Пример автоматизации

## Правила дымовой сигнализации

```Javascript
правило «Сработала дымовая сигнализация»
когда
    Элемент heiman_smoke_alarm изменен на ON.
тогда
    sendCommand(Гостиная_Свет, ВКЛ)
    sendNotification("admin", "⚠️ Сработала дымовая сигнализация!")
    postUpdate(Alarm_State, "smoke_alert")
конец
```

## Правила датчика температуры

```Javascript
правило «Отчет о температуре и влажности»
когда
    Элемент heiman_temperature получил обновление
тогда
    val temp = (heiman_temperature.state как число).floatValue
    val hum = (heiman_humidity.state as Number).floatValue
    если (температура > 40) {
        sendCommand(Кондиционер, ВКЛ)
        sendNotification("admin", "Температура слишком высокая (" + temp + "°C), кондиционер включен")
    }
    если (гул > 80) {
        sendNotification("admin", "Влажность слишком высокая (" + гул + "%), обратите внимание на осушение")
    }
конец
```
