# Советы по использованию

## Конфигурация вещи

Добавьте устройство Heiman через привязку MQTT:

```Ява
Thing mqtt:topic:heiman_smoke "Дымовая сигнализация Heiman" @ "гостиная" {
    Каналы:
        Введите двоичный файл: Smoke "состояние дыма" [stateTopic="zigbee2mqtt/heiman_smoke", TransformPattern="JSONPATH:$.smoke" ]
        Номер типа: батарея «Питание от батареи» [stateTopic="zigbee2mqtt/heiman_smoke", TransformationPattern="JSONPATH:$.battery"]
}
```

## Определение элемента

```Ява
Свяжитесь с Smoke_Alarm "Дымовая сигнализация [MAP(smoke.map):%s]" {channel="mqtt:topic:heiman_smoke:smoke" }
Number Smoke_Battery «Батарея дымовой сигнализации [%d%%]» {channel="mqtt:topic:heiman_smoke:battery" }
```

## Выбор координат

Рекомендуется использовать USB-координатор с чипом **CC2531** или **CC2652** и убедиться, что версия прошивки является последней.

## Советы по отладке

- Просмотр статуса вещи «онлайн/оффлайн» в пользовательском интерфейсе Paper.
- Используйте `openhab-cli info`, чтобы проверить, правильно ли загружены привязки.
- Выполните поиск по ключевому слову `heman` в журнале, чтобы устранить проблемы со связью устройства.
