#OpenHAB

OpenHAB — это платформа домашней автоматизации с открытым исходным кодом, ориентированная на интеграцию, независимую от поставщиков и протоколов.

## Метод интеграции

### Привязка Zigbee

Благодаря привязке Zigbee от OpenHAB датчики Heymann Zigbee можно подключать напрямую. Необходимо использовать с координатором Zigbee (например, Conbee II, SLZB-06).

### Связывание материи

OpenHAB 4.x и выше поддерживают протокол Matter. Устройства Hyman Matter можно интегрировать через привязку Matter, для чего требуется граничный маршрутизатор Thread.

### MQTT-метод

Используется с Zigbee2MQTT для получения данных датчиков через привязку MQTT.

> **Ссылка**: [Обязательный документ OpenHAB Zigbee] (https://www.openhab.org/addons/bindings/zigbee/)
