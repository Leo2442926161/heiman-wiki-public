# Советы по использованию

## Совместимость с хабом

Устройства Hyman Zigbee совместимы с **SmartThings Hub v2** и **Hub v3 (Aeotec)**.

| Модель концентратора | Совместимость | Описание |
|---------|--------|------|
| Концентратор v2 (STH-ETH-250) | ✅ | Требуется обновление прошивки до последней |
| Хаб v3 (Aeotec) | ✅ | Встроенная поддержка, рекомендуется |
| Станция Samsung SmartThings | ⚠️ | Поддерживает только некоторые устройства Zigbee |

## Запросы на сопряжение устройств

- Нажмите «+» → «Устройство» → «Марка» → «Hyman» в приложении SmartThings.
- Устройства, которые не распознаются автоматически, можно найти вручную с помощью «Scan Zigbee».
- При первом сопряжении рекомендуется сохранять расстояние между устройством и хабом в пределах 5 метров.

## Оптимизация батареи

Интервал отчетов по умолчанию для датчика Heiman больше. Если требуется производительность в режиме реального времени, «интервал обнаружения» можно настроить на странице устройства до 5 минут.

## Лучшие практики автоматизации

- Используйте функцию «Режимы» SmartThings, чтобы различать состояния «дома» и «вдали от дома».
- Создать **два комплекта** автоматики для ключевых датчиков (дыма, CO) для резервирования.
