# Пример автоматизации

## Режим сна «Спокойной ночи»

**Условие срабатывания:** Магнитный датчик двери Haiman → Закрыт (означает, что дверь спальни закрыта)

**Выполнить действие:**
- Выключите весь свет в гостиной.
- Умный замок → Блокировка
- Система безопасности → постановка на охрану ночью
- Push-уведомление: «Спокойной ночи, система поставлена на охрану»

## Режим «В гостях»

**Условие срабатывания:** Инфракрасный датчик человеческого тела Hyman → 15 минут отсутствия движения.

**Выполнить действие:**
- Все огни выключены
- Термостат → Режим энергосбережения
- Система безопасности → постановка на охрану вдали от дома
- Датчик двери → включить сигнализацию
- Push-уведомление: «Активирован режим «Вдали от дома», включены датчики дверей и окон»

## Аварийный режим дымовой сигнализации

**Условие срабатывания:** Датчик дыма Hyman → Обнаружен дым

**Выполнить действие:**
- Умный динамик → воспроизвести звуковой сигнал
- Все умные индикаторы → мигают красным
- Push-уведомления всем членам семьи
- При подключении к SmartThings Hub → включить сирену
