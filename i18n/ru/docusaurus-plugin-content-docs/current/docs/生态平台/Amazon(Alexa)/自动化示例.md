# Пример автоматизации

## Интеграция Alexa Guard (США)

**Необходимое условие:** подпишитесь на Alexa Guard Plus (доступно в США).

**Процесс настройки:**
1. Включите Guard Plus в приложении Alexa.
2. Добавьте сигнализацию дыма/CO от Hyman в качестве «устройства умного дома».
3. Включите интеллектуальный ответ «Обнаружение дымовой сигнализации».

**Автоматический ответ:**
- Сработала дымовая сигнализация → Alexa автоматически набирает экстренные контакты
- Триггер сигнализации CO → Голосовое предупреждение Alexa для всего дома, автоматическая вентиляция
- Двери и окна открываются ненормально → Push-уведомление о тревоге

## Процедура обнаружения дыма

**Триггер:** Дымовая сигнализация Hyman → Обнаружен дым

**Выполнить действие:**
- Голосовое объявление Alexa: «⚠️ Обнаружен дым, обратите внимание на безопасность»
- Все устройства Echo → Воспроизвести сигнал будильника
- Умный свет → все горит
- Push-уведомления на мобильный телефон

## Процедура «Уход из дома»

**Триггер:** Голосовая команда «Алекса, я ухожу»

**Выполнить действие:**
- Магнитный датчик двери Heiman → включить охрану
- Датчик тела Heiman → Включить обнаружение движения
- Детектор погружения в воду Hyman → включить обнаружение
- Все индикаторы → выключены
- Алекса ответила: «Система безопасности включена, счастливого пути»
