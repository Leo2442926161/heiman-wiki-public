# Советы по использованию

## Обнаружение устройства

- После включения устройства Heiman Matter оно автоматически обнаруживается приложением Alexa, обычно в течение 30 секунд.
- Если оно не обнаружено автоматически: войдите в приложение Alexa → Устройство → «+» → «Добавить устройство» → «Материя».
- Используйте код настройки Matter или QR-код в нижней части устройства, чтобы завершить сопряжение.

## Доступ к устройству Zigbee (Echo Plus/Echo Show)

- Устройства Echo с Zigbee Hub могут напрямую соединяться с датчиками Heiman Zigbee.
- Режим сопряжения: приложение Alexa → Устройство → «+» → «Zigbee» → «Поиск устройства».
- Устройства Heimann совместимы с Alexa и встроенным координатором Zigbee.

## Повседневные советы

- Используйте «настраиваемые действия» для трансляции более подробных голосовых подсказок.
- Добавьте «подождите 5 секунд» в аварийную процедуру перед выполнением последующих действий, чтобы избежать дублирования голосов.
- Воспользуйтесь функцией «Обнаружение звука», чтобы дать Alexa дополнительные ответы на высокие сигналы тревоги Heyman Alarm.

## Alexa Guard ПРИМЕЧАНИЕ

Alexa Guard Plus в настоящее время доступен только в США. Пользователи в Китае могут моделировать некоторые функции с помощью Routine.
