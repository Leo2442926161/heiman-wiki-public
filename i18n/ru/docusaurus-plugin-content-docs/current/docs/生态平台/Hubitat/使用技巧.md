# Советы по использованию

## Пользовательский драйвер

Устройства Heimann оптимизированы для совместимости в Hubitat со следующими драйверами:

| Тип устройства | Рекомендуемый драйвер |
|----------|----------|
| Сигнализация дыма/CO | Универсальный датчик дыма/CO Zigbee |
| Инфракрасный датчик человеческого тела | Датчик движения Heiman (сторонний) |
| Дверной магнитный датчик | Универсальный контактный датчик Zigbee |
| Датчик утечки воды | Универсальный датчик утечки воды Zigbee |
| Датчик температуры и влажности | Датчик температуры/влажности Heiman |

## Оптимизация Zigbee

- В моделях Hubitat C-8 и выше используется чип EFR32, который имеет лучшую совместимость.
- Посмотреть и выбрать канал с наименьшими помехами в Настройках → Zigbee
- Используйте инструменты анализа Zigbee, такие как Ubisys, для диагностики пропущенных вызовов.

## Подсказки правил

- Рекомендуется добавить в Rule Machine условие "Задержка антиложной тревоги" (0,5-2 секунды).
- Включите уведомления «проверки работоспособности» для ключевых правил, чтобы гарантировать, что они продолжают работать.
- Организовать сложную автоматизацию в режиме «подправила» для удобства отладки и обслуживания.

## Сосуществование Z-Wave

Если вы хотите одновременно использовать устройства Z-Wave, обязательно сохраните частотный интервал между каналами Zigbee и Z-Wave.
