#hubitat

Hubitat — это центр умного дома, ориентированный на локализованное управление, поддерживающий устройства Zigbee и Z-Wave, при этом вся автоматизация выполняется локально без необходимости использования облачных сервисов.

## Метод интеграции

### Устройства Zigbee

Датчики Hyman Zigbee напрямую соединяются с концентраторами Hubitat Elevation:

1. Войдите в режим сопряжения Zigbee в Hubitat.
2. Переведите датчик Heiman в режим сопряжения.
3. Устройство будет автоматически обнаружено и добавлено.

### Z-Wave устройства

Устройства Hyman Z-Wave также поддерживают прямое сопряжение с Hubitat, используя встроенный чип Z-Wave Hubitat.

### Пользовательский драйвер

Некоторым устройствам Heiman могут потребоваться специальные драйверы, и пользователи сообщества написали совместимые драйверы для обычных датчиков.

## Преимущества

- **100% локализация**: вся логика автоматизации выполняется локально в Hubitat и не зависит от облака.
- **Низкая задержка**: локальная сеть Zigbee/Z-Wave Mesh, быстрый отклик.
- **Богатое сообщество**: сообщество Hubitat предоставляет возможность обмена конфигурациями большого количества пользователей Heiman.
