# Пример автоматизации

## Правильная машина дымовой сигнализации

**Название правила:** Защитная связь дымовой сигнализации

**Конфигурация:**
```
Триггер: активируется дымовая сигнализация Hyman.
Условия: Режим != Снято с охраны
Действие:
  - Включите все освещение
  - Отправьте push-уведомление «⚠️ Датчик дыма!»
  - Включите сирену (если имеется)
  - Установите режим «Аварийный».
```

## Правило контроля температуры машины

**Название правила:** Защита от высоких температур

**Конфигурация:**
```
Триггер: Датчик температуры Heimann > 38°C
Условие: Состояние кондиционера = выключено.
Действие:
  - Включите кондиционер, режим охлаждения, 24°C.
  - Отправить уведомление «Температура в помещении слишком высокая, и кондиционер включен»
```

## Правило обнаружения утечки воды в машине

**Название правила:** Неотложная помощь при утечке воды.

**Конфигурация:**
```
Триггер: погружной детектор Hyman намокает.
Условие: Состояние водяного клапана = открыто.
Действие:
  - Закройте главный водяной клапан.
  - Отправляйте push-уведомления на все мобильные телефоны.
  - Отправить SMS-уведомление (необязательно)
```
