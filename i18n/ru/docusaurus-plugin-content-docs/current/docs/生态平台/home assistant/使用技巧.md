# Советы по использованию

## Конфигурация устройства

- При первом подключении устройства Heiman Matter убедитесь, что ваш пограничный маршрутизатор (например, HomePod, Nest Hub, Home Assistant Green) находится в той же сети Thread.
- При интеграции устройств Zigbee через ZHA или Zigbee2MQTT рекомендуется сначала использовать рекомендуемую прошивку USB-координатора (например, Sonoff ZBDongle-E, Conbee II)

## Автоматическая оптимизация

- Добавлен параметр `for:` (например, 3 секунды) для автоматизации дымовой сигнализации, чтобы предотвратить мгновенные ложные срабатывания.
- Используйте `condition:state` для соответствия режиму безопасности, чтобы избежать ненужных уведомлений в состоянии охраны.
- Используйте функцию «группировать» Home Assistant для группировки датчиков Heiman по площади для упрощения пакетной обработки.

## Устранение неполадок

- Когда устройство находится в автономном режиме, сначала проверьте, не конфликтует ли сетевой канал Zigbee с Wi-Fi 2,4 ГГц.
- Если устройство Matter не может быть обнаружено, перезапустите пограничный маршрутизатор и повторно добавьте устройство.
