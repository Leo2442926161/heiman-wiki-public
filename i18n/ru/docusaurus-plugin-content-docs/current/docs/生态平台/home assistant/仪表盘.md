# Конфигурация приборной панели

## Панель обзора безопасности

Используйте макет вида «Разрез», чтобы распределить оборудование Heimann по областям:

```ямл
тип: раздел
Название: Сигнализация дыма и угарного газа
карты:
  - тип: сущность
    объект:binary_sensor.heman_smoke_alarm
    имя: пожарная сигнализация
  - тип: сущность
    объект:binary_sensor.heman_co_alarm
    имя: Сигнализация CO
```

## Стиль карты «Гриб»

```ямл
тип: заказ: грибная-сигнализация-панель управления-карта
объект: Alarm_control_panel.home_alarm
название: система безопасности

тип: custom:mushroom-entity-card
объект:binary_sensor.heman_smoke_alarm
имя: пожарная сигнализация
значок: mdi:детектор дыма
первичная_информация: состояние
Secondary_info: последнее изменение
fill_container: правда
```

## Панель датчиков окружающей среды

```ямл
тип: раздел
Название: Мониторинг окружающей среды
карты:
  - тип: манометр
    объект: Sensor.heiman_temperature
    название: температура
    единица измерения: «°С»
    мин: 0
    макс: 50
    серьезность:
      зеленый: 0
      желтый: 30
      Красный: 40
  - тип: манометр
    объект: Sensor.heiman_humidity
    название: влажность
    единица: "%"
    мин: 0
    макс: 100
    серьезность:
      зеленый: 30
      Желтый: 60
      Красный: 80
```

## Панель истории тревог

```ямл
тип: график истории
Название: История датчиков
субъекты:
  -binary_sensor.heman_smoke_alarm
  -binary_sensor.heman_co_alarm
  -binary_sensor.heiman_water_leak
часов_до_шоу: 24
```

## Предлагаемый макет

- Вверху: статус системы безопасности + карта состояния устройства сигнализации.
- Центральный: мониторинг окружающей среды (температура, влажность, качество воды)
- Внизу: диаграмма исторических тенденций сигналов тревоги датчиков.
- Операции быстрого доступа на боковой панели: постановка/снятие с охраны, переключение сцен.
