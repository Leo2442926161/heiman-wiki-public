# Пример автоматизации

## Датчик дыма включает освещение по всему дому

```ямл
псевдоним: «Дымовая сигнализация — освещение и уведомления по всему дому»
триггеры:
  - триггер: состояние
    entity_id:binary_sensor.heman_smoke_alarm
    к: "включено"
действия:
  - действие: Light.turn_on
    цель:
      entity_id: все
  - действие: notify.mobile_app_phone
    данные:
      сообщение: «⚠️ Сработала дымовая сигнализация! Пожалуйста, проверьте немедленно!»
      title: "Дымовая сигнализация"
```

## Связь с сигнализацией CO

```ямл
псевдоним: «Тревога CO — выключите газ и откройте окна»
триггеры:
  - триггер: состояние
    entity_id:binary_sensor.heman_co_alarm
    к: "включено"
действия:
  - действие: переключатель.turn_off
    entity_id: switch.gas_fireplace
  - действие: крышка.открыть
    entity_id: Cover.living_room_window
  - действие: notify.persistent_notification
    данные:
      сообщение: «Сработала сигнализация угарного газа, газовый клапан закрыт, окна открыты»
```

## Соединение с датчиком погружения в воду

```ямл
псевдоним: «Обнаружение наводнения — закрытие водяного клапана»
триггеры:
  - триггер: состояние
    entity_id:binary_sensor.heman_water_leak
    к: "включено"
действия:
  - действие: переключатель.turn_off
    entity_id: switch.main_water_valve
  - действие: notify.mobile_app_phone
    данные:
      сообщение: «💧 Обнаружена утечка воды! Главный водяной клапан закрыт!»
```

## Постановка на охрану вдали от дома — датчик человеческого тела активирует тревогу

```ямл
псевдоним: «Режим выхода из дома — сигнал тревоги датчика человеческого тела»
триггеры:
  - триггер: состояние
    entity_id:binary_sensor.heman_motion_sensor
    к: "включено"
условия:
  - состояние: состояние
    entity_id: Alarm_control_panel.home_alarm
    состояние: вооруженный_отсутствие
действия:
  - действие: Alarm_control_panel.trigger
    entity_id: Alarm_control_panel.home_alarm
```
