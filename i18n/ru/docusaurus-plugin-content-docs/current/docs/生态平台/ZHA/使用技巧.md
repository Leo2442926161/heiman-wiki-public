# Советы по использованию

## Сопряжение устройств

1. Войдите в «Настройки → Устройства и службы → ZHA → Конфигурация» в Home Assistant.
2. Нажмите «Добавить устройство», а затем запустите режим сопряжения устройства Heiman.
3. Если сопряжение не удалось, поднесите устройство к USB-координатору и повторите попытку в пределах 3 метров.

## Предпочитаемый USB-координатор

| Модель | Чип | Индекс рекомендаций |
|------|------|----------|
| Sonoff ZBDDongle-E | ЭФР32МГ21 | ⭐⭐⭐⭐⭐ |

| Elelabs Zigbee USB | 2652П | ⭐⭐⭐⭐ |

## Координация каналов

Канал ZHA по умолчанию — 15. Если есть серьезные помехи в работе WiFi 2,4 ГГц:
- Используйте `Silicon Labs Flasher Utility` для изменения канала.
- Обычно рекомендуемые каналы: 15, 20, 25 (избегайте Wi-Fi 1/6/11)

## Оптимизация сети

- Добавляйте устройство маршрутизации Zigbee (например, интеллектуальную розетку) на каждые 3-4 устройства Heiman для расширения сети.
- Проверьте качество подключения устройства с помощью инструмента визуализации сети ZHA.
- Регулярно проверяйте наличие обновлений прошивки устройства для обеспечения оптимальной совместимости.
