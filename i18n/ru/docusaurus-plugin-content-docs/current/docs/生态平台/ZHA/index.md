#ZHA (Домашняя автоматизация Zigbee)

ZHA — это встроенное в Home Assistant решение для интеграции Zigbee, которое позволяет напрямую управлять устройствами Zigbee без дополнительного программного обеспечения.

## Совместимость

Датчики Hyman Zigbee имеют хорошую совместимость с ZHA и поддерживают:

- **Тревога дыма/CO**: состояние тревоги, обнаружение неисправностей, уровень заряда батареи, отключение звука.
- **Дверной датчик**: вкл/выкл, тампер
- **Инфракрасное излучение человека**: обнаружение движения, освещение
- **Температура и влажность**: температура, влажность.
- **Наводнение**: сигнализация об утечке воды.
- **Умная розетка**: выключатель, измерение энергопотребления

## Требования к оборудованию

- Координатор Zigbee: HUSBZB-1, Conbee II, SkyConnect, SLZB-06 и т. д.
- Подключитесь к Home Assistant через USB или сеть.

## Конфигурация

ZHA легко настраивается. Просто выберите последовательный порт и область для автоматического сканирования и обнаружения устройства. Нет необходимости вручную писать файлы конфигурации.
