# Пример автоматизации

## Соединение с дымовой сигнализацией

```ямл
псевдоним: «Связь дымовой сигнализации ZHA»
триггеры:
  - триггер: состояние
    entity_id:binary_sensor.heiman_smoke_zha
    к: "включено"
действия:
  - действие: Light.turn_on
    цель:
      area_id: гостиная_комната
  - действие: tts.speak
    цель:
      entity_id: media_player.speaker
    данные:
      сообщение: «Сработала дымовая сигнализация!»
```

## Автоматизация сопряжения устройств

```ямл
alias: «Напоминание о сопряжении нового устройства»
триггеры:
  - триггер: zha.event
    подтип: хейман
    команда: device_joined
действия:
  - действие: notify.persistent_notification
    данные:
      сообщение: «Новое устройство Haiman присоединилось к сети ZHA, перейдите к имени и области конфигурации»
```

## Обнаружение погружения в воду

```ямл
псевдоним: «Испытание наводнением ZHA»
триггеры:
  - триггер: состояние
    entity_id:binary_sensor.heiman_water_leak_zha
    к: "включено"
действия:
  - действие: переключатель.turn_off
    entity_id: switch.main_water_valve
  - действие: notify.mobile_app_phone
    данные:
      сообщение: «💧 Обнаружено наводнение!»
```
