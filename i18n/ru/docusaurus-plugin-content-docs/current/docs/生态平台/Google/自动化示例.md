# Пример автоматизации

## Рутина «Я выхожу»

**Триггер:** Скажите Google Home «Меня нет»

**Выполнить действие:**
- Магнитный датчик двери Heiman → рычаг
- Инфракрасный датчик человеческого тела Heiman → Включить обнаружение ухода из дома
- Выключите весь свет и электроприборы.
- Термостат → Установите энергосберегающую температуру.
- Google Nest Hub → Показать карту подтверждения выхода из дома
- Голосовой ответ: «Режим выхода из дома включен, счастливого пути»

## Трансляция «Обнаружение дыма»

**Триггер:** Дымовая сигнализация Hyman → Обнаружен дым

**Выполнить действие:**
- Динамик Google Nest → Трансляция на весь дом «⚠️ Обнаружен дым! Пожалуйста, проверьте немедленно!»
- Все интеллектуальные источники света → Вкл. (через набор интеллектуальных источников света Google Home)
- Push-уведомления на мобильные телефоны членов семьи
- Камера → Начать запись

## Ночной режим

**Триггер:** Скажите Google Home «Спокойной ночи».

**Выполнить действие:**
- Датчик двери Heiman → Включить сигнализацию открытой двери
- Инфракрасное тело человека Heiman → включить ночное обнаружение
- Свет → постепенно выключается
- Голосовой ответ: «Спокойной ночи, охранная система включена»
