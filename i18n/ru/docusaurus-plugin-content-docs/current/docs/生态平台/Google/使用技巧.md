# Советы по использованию

## Синхронизация устройств

- После подключения устройства Heiman к Google Home через Matter оно автоматически синхронизируется с приложением Google Home.
- Если устройство не отображается, нажмите «Настройки» → «Устройства» → «Добавить устройство» → «Устройства, совместимые с Matter» в приложении Google Home.
- Если QR-код Matter утерян, его можно добавить вручную, используя 11-значный код настройки на вашем устройстве.

## Регулярная оптимизация

- Добавлено «Объявить погоду и подтвердить статус постановки на охрану» в процедуру «Выйти из дома».
- Используйте функцию «Голосовое вещание», чтобы оповестить весь дом о срабатывании пожарной сигнализации.
- Используйте действие «Старт/Стоп» для управления включением и отключением состояния обнаружения датчика Heiman.

## Управление местоположением устройства

- Назначьте определенные комнаты для датчиков Heimann (например, «Кухонная сигнализация дыма») для более точного голосового управления.
- Включите несколько устройств Heiman в «Группу домашних устройств», чтобы можно было полностью включать и выключать их одним щелчком мыши.

## Требования к сети

- Для устройств Matter over Thread требуется Google Nest Hub (2-го поколения) или Nest Hub Max в качестве пограничного маршрутизатора.
- Устройства Zigbee подключаются через Google Nest Connect или совместимый концентратор.
