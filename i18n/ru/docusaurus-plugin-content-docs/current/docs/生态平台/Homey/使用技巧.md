# Советы по использованию

## Помехи канала Zigbee

Встроенный чип Zigbee Homey использует определенные каналы. Если устройство Heiman часто отключается от сети:

1. Посмотреть текущий канал в Homey Settings → Zigbee.
2. Используйте приложение Homey Insight для поиска помех окружающей среды.
3. Переключиться на канал с меньшими помехами (подальше от WiFi 1/6/11)

## Сеть распространения оборудования

- Как войти в режим сопряжения для устройств Heiman: нажмите и удерживайте кнопку в течение 5 секунд (некоторые модели нужно быстро нажать 3 раза)
- При добавлении устройства в приложении Homey выберите «Добавить устройство → Zigbee → Поиск».
- Если сопряжение не удалось, поднесите устройство ближе к Homey (в пределах 3 метров) и повторите попытку.

## Рекомендации по оптимизации потока

- Добавьте условие «задержки» в Flow (например, 2 секунды), чтобы избежать временных ложных срабатываний.
- Используйте функцию «тег» для классификации и управления оборудованием Heiman.

## Расширение диапазона сигнала

Если устройство Heiman находится далеко от Homey, можно добавить ретрансляционное устройство Zigbee (например, интеллектуальную розетку) для расширения покрытия сети.
