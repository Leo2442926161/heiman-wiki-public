# Пример автоматизации

## Датчик дыма - мигает и уведомляет

**Конфигурация потока:**

1. **Триггер:** Дымовая сигнализация Hyman → Статус меняется на «Тревога».
2. **Условия:** (Нет)
3. **Действие:**
   - Свет в гостиной → мигает (красным, раз в секунду)
   - Push-уведомление → Мобильное устройство: «⚠️ Сработала дымовая сигнализация!»
   - Голосовое объявление → Домашний динамик: «Обнаружен дым, обратите внимание на безопасность!»

## Сигнализация CO — выключить котел

**Конфигурация потока:**

1. **Триггер:** Сигнализация Hyman CO → Статус меняется на «Предупреждение».
2. **Условия:** (Нет)
3. **Действие:**
   - Выключатель котла → ВЫКЛ.
   - Умная розетка (система подачи свежего воздуха) → Вкл.
   - Push-уведомление → «Тревога угарного газа, котел выключен»

## Сигнализация превышения температуры и влажности

1. **Триггер:** Датчик температуры и влажности Hyman → Температура выше 40 °C.
2. **Действие:**
   - Кондиционер → Включить режим охлаждения.
   - Push-уведомление → «Температура в помещении слишком высокая, кондиционер автоматически включился»
