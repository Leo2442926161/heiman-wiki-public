#Домашний

Homey — это центр умного дома, запущенный голландской компанией Athom и поддерживающий протоколы Zigbee, Z-Wave, Wi-Fi, Matter, Thread и другие.

## Метод интеграции

Устройства Homey Zigbee и Z-Wave соединяются напрямую через встроенный беспроводной чип Homey, что устраняет необходимость в дополнительном шлюзе. Доступ к устройствам Matter можно получить через встроенный пограничный маршрутизатор Thread в Homey Pro.

## Преимущества

- **Несколько протоколов в одном**: Homey Pro поддерживает Zigbee, Z-Wave, Thread и Wi-Fi одновременно.
- **Запускать локально**: вся автоматизация выполняется локально.
- **Автоматизация потока**: программирование с помощью перетаскивания для создания сложных сценариев связи.

> **Справка**: приложения, связанные с Heiman, можно найти в Homey App Store.
