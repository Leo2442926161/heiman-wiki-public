# Сравнение моделей

## Сравнение различных моделей

| Модель | Протокол | Номинальная мощность | Точность измерения | Особенности | Предложения по игре |
|------|------|----------|----------|------|----------|
| [HS2SK](../../Протокол беспроводной связи/Zigbee/HS2SK — интеллектуальная розетка (Zigbee)) | Зигби | 16А/3680Вт | ±2% | Тип учета | Мониторинг энергопотребления в режиме реального времени + удаленный переключатель, отключение питания одним щелчком мыши в режиме связи вдали от дома, статистика энергопотребления кондиционера/водонагревателя |
| [HS2SK-WiFi](../../Протокол беспроводной связи/Wi-Fi/HS2SK-WiFi — розетка для интеллектуального счетчика (WiFi)) | Wi-Fi | 16А/3680Вт | ±2% | Прямое соединение без шлюза | Дополнительный шлюз не требуется, прямое управление через приложение Smart Life, подходит для пользователей, которые не хотят покупать шлюз |

## Рекомендуемая сцена игрового процесса

| Сценарии | Рекомендуемые модели | Решения для связи |
|------|----------|----------|
| Статистика энергопотребления | ХС2СК | Zigbee подключен к HA и связан с InfluxDB + Grafana для отображения ежедневного/ежемесячного энергопотребления и обнаружения устройств с высоким энергопотреблением |
| Выключение при выходе из дома | ХС2СК | Выходя из дома, выключайте второстепенные приборы, такие как телевизор/кондиционер/освещение аквариума, одним щелчком мыши, чтобы безопасно экономить энергию |
| Контроль времени | HS2SK / HS2SK-WiFi | Водонагреватель/увлажнитель/зарядная батарея с таймером в сочетании с пиковыми и низкими ценами на электроэнергию для экономии счетов за электроэнергию |
| Пользователи без шлюза | HS2SK-WiFi | Прямое подключение к маршрутизатору, управление с помощью приложения Smart Life, нулевой порог входа |

## Рекомендации по выбору

- **Существующий шлюз Zigbee**: [HS2SK](../../WirelessProtocol/Zigbee/HS2SK - Smart Socket (Zigbee))
- **Не хочу покупать шлюз**: [HS2SK-WiFi](../../Wireless Protocol/Wifi/HS2SK-WiFi - Smart Metering Socket (WiFi))
