# умная розетка

Haiman Smart Metering Plug поддерживает дистанционное управление переключателями и измерение энергопотребления в реальном времени, помогая пользователям добиться интеллектуального управления бытовой техникой.

## Серия продуктов

| Серия | Методы связи | Особенности |
|------|----------|------|
| серия HS2SK | Зигби | Тип измерения, поддерживает мониторинг мощности |

## Основные функции

- Дистанционное управление переключателем
- Учет электроэнергии/газа в режиме реального времени
- Защита от перегрузки
- Таймер
- Статистика мощности

## Технические параметры

- Номинальное напряжение: 220 В переменного тока (китайский стандарт)/110 В переменного тока (американский стандарт).
- Максимальная нагрузка: 16А/3680Вт
-Точность измерения: ± 2%

## Пример сценария

- Дистанционно выключайте забытые приборы
- Статистика энергопотребления кондиционера/водонагревателя
- Связанный режим выхода из дома: отключите все ненужные приборы одним щелчком мыши.
- Контроль времени освещения аквариума, увлажнителей и т. д.
