# Руководство по установке

## Требования к месту установки

- Устанавливается на поверхности объектов, требующих контроля вибрации.
- Общие места установки: **Внутри дверей и окон**, **сейфы**, **трубы**, **стены**, **заборы**.
- Убедитесь, что установочная поверхность ровная и чистая.

## Этапы установки

1. Выберите объект для наблюдения (например, внутреннюю часть двери сейфа).
2. Очистите монтажную поверхность.
3. Используйте клейкую ленту 3М или винты, чтобы закрепить датчик.
4. Установите батарею CR2450.
5. Сопряжение через шлюз
6. Тест моделирования: коснитесь поверхности установки, датчик должен вызвать сигнал тревоги.

## Примечания

- Для защиты от взлома дверей и окон более эффективно установить его на внутренней стороне двери/окна рядом с замком.
- Для мониторинга трубопровода приклейте непосредственно на поверхность трубопровода.
- Срок службы батареи около 2 лет
- Регулируемая чувствительность для различных сценариев
