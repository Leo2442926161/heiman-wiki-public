# Звуковая и световая сигнализация (предупреждающий сигнал)

Умная сирена используется для обеспечения громкой звуковой и световой сигнализации. Используется с различными датчиками и может использоваться как выходной исполнительный элемент системы безопасности.

## Основные функции

- Звуковой сигнал высокого децибела (≥110 дБ)
- Красные/цветные стробоскопы
-Поддержка нескольких сигналов будильника
- Обнаружение тампера
-Поддержка удаленной постановки и снятия с охраны

## Применимые сценарии

- Связан с датчиком дыма/CO: звучит по всему дому при обнаружении опасности.
- Сигнализация взлома: срабатывает при незаконном открытии дверей и окон.
- Экстренная эвакуация: помогите людям эвакуироваться в случае пожара.

## Связь со сценой

- **Дымовая сигнализация + Звуковая и световая сирена**: весь дом включит сигнал тревоги одновременно, предупреждая всех людей в комнате.
- **Сигнализация датчика двери + звуковая и световая сигнализация**: отпугивают незаконных злоумышленников.
- **Сотрудничество со шлюзом**: связь с другими устройствами через сеть Zigbee/Matter.
