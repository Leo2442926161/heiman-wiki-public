# Сравнение моделей

## Сравнение различных моделей

| Модель | Протокол | Метод обнаружения | Источник питания | Особенности | Предложения по игре |
|------|------|----------|------|------|----------|
| [HS1WL](../../Протокол беспроводной связи/Zigbee/HS1WL - Датчик погружения в воду (Zigbee)) | Зигби | Зонд | Батарея | Базовая модель | Сигнализация при протечке воды, подходит для установки под раковиной в ванной |
| [HS2WL](../../Протокол беспроводной связи/Zigbee/HS2WL - Датчик погружения в воду (Zigbee)) | Зигби | Зонд + кабель | Батарея | Двухканальный | Точка обнаружения зонда + кабель покрывает большую площадь, лучше всего рядом с водонагревателем / стиральной машиной |
| [HS3WL](../../Протокол беспроводной связи/Zigbee/HS3WL - Детектор утечки воды (Zigbee)) | Зигби | Зонд | Батарея | Расширенная версия | Более высокая чувствительность, более громкий зуммер |
| [L1-M](../../Беспроводной протокол/Материя/L1-M Детектор погружения в воду (Материя)) | Материя важнее темы | Зонд | Батарея | Версия вопроса | Сертифицирован WWHA, встроенный доступ к Apple Home |

## Рекомендуемая сцена игрового процесса

| Сценарии | Рекомендуемые модели | Решения для связи |
|------|----------|----------|
| Мониторинг протечек воды во всем доме | HS2WL + L1-М | Покрытие нескольких точек + встроенное уведомление HomeKit, обнаружение утечки воды → выключение основного водяного клапана → push-уведомление |
| Специально для стиральных машин | HS2WL | Тросовый щуп проложен внизу стиральной машины. При утечке воды немедленно подается сигнал тревоги + отключается питание стиральной машины |
| Водонагреватель/подогрев пола | HS3WL | Усовершенствованная версия обладает высокой чувствительностью и позволяет обнаруживать утечки в трубах теплого пола на начальном этапе |
| Пользователи Matter | Л1-М | Выделенный шлюз не требуется, связь сигнализации с малой задержкой Thread Apple Домашняя автоматизация |

## Рекомендации по выбору

- **Обнаружение точки**: [HS1WL](../../Беспроводной протокол/Zigbee/HS1WL - Детектор утечки воды (Zigbee)) или [L1-M](../../Беспроводной протокол/Matter/L1-M Детектор утечки воды (Matter))
- **Большая зона покрытия**: [HS2WL](../../Беспроводной протокол/Zigbee/HS2WL — детектор утечки воды (Zigbee)) (зонд + кабель)
- **Требования к материи**: [L1-M](../../Беспроводной протокол/Материя/L1-M Детектор погружения в воду (Материя))
