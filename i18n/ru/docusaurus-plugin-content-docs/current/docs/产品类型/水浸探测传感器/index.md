# Датчик обнаружения погружения в воду

Датчик утечки воды используется для обнаружения утечки или скопления воды, что может эффективно предотвратить потери имущества, вызванные разрывами водопроводных труб и затоплением.

## Серия продуктов

| Серия | Методы связи | Особенности |
|------|----------|------|
| HS1WL | Зигби | Базовая модель, обнаружение зонда |
| Серия HS2WL | Зигби | Датчик + кабель обнаружения, двухканальный |
| Серия HS3WL | Зигби | Расширенная версия |
| Серия L1 | Материя / Зигби | Версия Matter over Thread, сертифицирована WWHA |

## Основные функции

- Обнаружение протечек/застойной воды в режиме реального времени
- Зуммеровая сигнализация
- Напоминание о низком заряде батареи
- Антикоррозионная конструкция зонда

## Место установки

- Под раковиной в ванной.
- Под кухонной раковиной
- Рядом со стиральной или посудомоечной машиной.
- Под водонагревателем
- Подвал/гараж
- Возле радиатора

## Связь со сценой

- **Обнаружение утечки воды + электромагнитный клапан**: автоматическое закрытие основного клапана при обнаружении утечки воды.
- **Сигнализация утечки воды + Push**: сообщите об этом администрации объекта или членам семьи на свой мобильный телефон, когда вас нет дома.
- **Затопление + интеллектуальное отключение**: подключенный газовый водонагреватель или интеллектуальный водяной клапан.
