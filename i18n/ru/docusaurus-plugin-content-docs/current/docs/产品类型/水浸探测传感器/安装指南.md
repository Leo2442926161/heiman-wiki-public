# Руководство по установке

## Требования к месту установки

- **Положите на землю**, убедитесь, что щуп касается земли.
- **Рядом с источником воды**: под раковиной в ванной, под кухонной раковиной, рядом со стиральной машиной, под водонагревателем.
- Самая нижняя точка, где необходимо обнаружить утечку воды.

## Этапы установки

1. Определите место установки (земля в зоне риска протечки воды)
2. Очистите поверхность пола.
3. Поместите датчик на землю так, чтобы зонд касался земли.
4. При использовании кабельной версии (HS2WL) проложите удлинительный кабель вдоль контролируемой зоны.
5. Вставьте батарею, и датчик автоматически включится.
6. Тест моделирования: коснитесь датчика мокрыми пальцами, загорится индикатор сигнализации.
7. Сопряжение через шлюз

## Примечания

- Зонд может подвергнуться коррозии, если он долгое время находится во влажной среде, поэтому регулярно проверяйте его.
- Рекомендуется использовать электромагнитный клапан для автоматического отключения.
-Срок службы батареи около 2-3 лет
- Несколько датчиков для покрытия разных областей
