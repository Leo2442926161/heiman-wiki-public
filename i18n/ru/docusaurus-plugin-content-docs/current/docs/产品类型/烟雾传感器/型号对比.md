# Сравнение моделей

|------|------|------|------|------|----------|
| [HS1SA-E](../../Беспроводной протокол/Zigbee/HS1SA-E - Интеллектуальная дымовая сигнализация (Zigbee)) | Зигби | EN 14604 | CR123A | ЕС | Подключитесь к Zigbee2MQTT/ZHA, подключитесь к Home Assistant, автоматически включайте свет при обнаружении дыма + push-уведомления на мобильные устройства |

## Рекомендуемая сцена игрового процесса

| Сценарии | Рекомендуемые модели | Решения для связи |
|------|----------|----------|
| Глубокая интеграция умного дома | HS1SA-E / HS1SA-M | Автоматизация HA: сигнализация дыма → включить свет во всем доме → разблокировать дверные замки → выключить газовый клапан → нажать на мобильный телефон |
| Пользователи Apple HomeKit | ХС1СА-М | Доступ к Native Matter, HomePod в качестве пограничного маршрутизатора потока, автоматический запуск сцен |
| Требования европейских и американских стандартов | HS1SA-E (CE) / HS8SA (UL) | Соответствует требованиям местной пожарной сертификации и страхованию |
| Не требуется шлюз/низкая стоимость | серия S1 | Готовые к использованию после покупки, несколько устройств соединены между собой через Sub-1G, активируя друг друга при возникновении тревоги |
| Существующая система Z-Wave | HS1SA-Z | Работает с существующими шлюзами Z-Wave, такими как Hubitat/SmartThings |

## Рекомендации по выбору

- **Рынок ЕС**: [HS1SA-E](../../Wireless Protocol/Zigbee/HS1SA-E - Интеллектуальная дымовая сигнализация (Zigbee)) (Zigbee) или [HS3SA](../../Wireless Protocol/Zigbee/HS3SA - Интеллектуальная дымовая сигнализация (Zigbee)) (Zigbee), если требуется выбор Z-Wave [HS1SA-Z](../../Беспроводной протокол/Zwave/HS1SA-Z Интеллектуальная дымовая сигнализация (Z-Wave))
- **Рынок США**: [HS8SA](../../Wireless Protocol/Zigbee/HS8SA - Интеллектуальная дымовая сигнализация (Zigbee)) (сертификат UL 217)
- **Кроссплатформенность**: [HS1SA-M](../../Wireless Protocol/Matter/HS1SA-M Smart Smoke Alarm (Matter)) (Matter over Thread)
- **Автономная сцена**: [Серия S1](../../Беспроводной протокол/Sub1G (большое расстояние)/Серия S1 - Автономная дымовая сигнализация (Sub-1G)) (шлюз не требуется)
