# датчик дыма

В датчике дыма Hyman используется **фотоэлектрическая технология обнаружения дыма**, которая может чувствительно обнаруживать видимые частицы дыма, образующиеся на ранних стадиях пожара, и обеспечивать очень раннее предупреждение о пожаре.

## Серия продуктов

| Серия | Методы связи | Стандарты сертификации | Применимые рынки |
|------|----------|----------|----------|
| Серия HS1SA | Зигби/Z-Wave | EN 14604 | Европейский Союз |
| Серия HS3SA | Зигби | EN 14604 | Европейский Союз |
| HS8SA | Зигби/Wi-Fi | УЛ 217 | Соединенные Штаты |
| Серия S1 | Отдельно стоящий Sub-1G | EN 14604 | Общего назначения |
| ХС1СА-М | Материя важнее темы | EN 14604 | Глобальный |

## Основные функции

- Фотоэлектрический детектор дыма для обнаружения дыма от медленного горения и открытого огня.
- Звуковая и световая сигнализация громкостью 85 дБ.
- Напоминание о низком заряде батареи
- Функция самотестирования (тест в один клик)
- Сигнализация против несанкционированного доступа
-Поддержка удаленного отключения звука

## Технические параметры (на примере HS1SA-M)

- Тип датчика: инфракрасный фотоэлектрический
- Звуковое давление сигнализации: ≥85 дБ (3 метра)
- Источник питания: батарея CR123A.
- Срок службы батареи: ок. 3 года
- Рабочая температура: -10°C ~ +50°C
- Рабочая влажность: ≤95% относительной влажности.
