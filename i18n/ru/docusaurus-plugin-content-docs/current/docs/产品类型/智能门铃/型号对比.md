# Сравнение моделей

## Сравнение различных моделей

| Модель | Протокол | Видео | Источник питания | Особенности | Предложения по игровому процессу |
|------|------|------|------|------|----------|
| Умный дверной звонок | Зигби + Wi-Fi | 1080P опционально | Аккумулятор/сеть | Дверной звонок + охрана два в одном | Уведомление о звонке посетителя на мобильный телефон, удаленный визуальный домофон, запись при обнаружении движения |

## Рекомендуемая сцена игрового процесса

| Сценарии | Рекомендуемые модели | Решения для связи |
|------|----------|----------|
| Экспресс-доставка/менеджмент на вынос | Умный дверной звонок | Позвонить → уведомление на мобильный телефон → удаленное подтверждение внутренней связи → удаленная разблокировка (связанный интеллектуальный замок) |
| Мониторинг безопасности | Умный дверной звонок | Обнаружение закрытия двери → Видео + push-сигнализация, связанный датчик двери/освещение |
| Связь с посетителями | Умный дверной звонок | Позвоните в звонок → Внутренний экран/дисплей динамика → Включите свет → Разблокируйте связь |
