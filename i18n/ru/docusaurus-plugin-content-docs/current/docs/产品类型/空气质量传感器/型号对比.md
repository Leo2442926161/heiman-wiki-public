# Сравнение моделей

## Сравнение различных моделей

| Модель | Протокол | Параметры обнаружения | Источник питания | Особенности | Предложения по игровому процессу |
|------|------|----------|------|------|----------|
| Датчик качества воздуха | Зигби | PM2,5 / PM10 / CO₂ / TVOC / Температура и влажность | Питание от сети | Многофункциональность в одном | Контролируйте качество воздуха в помещении, автоматически связывайте свежий воздух/очиститель + push-напоминание, если оно превышает стандарт |

## Рекомендуемая сцена игрового процесса

| Сценарии | Рекомендуемые модели | Решения для связи |
|------|----------|----------|
| Мониторинг CO₂ в спальне | Датчик качества воздуха | CO₂ > 1000ppm → Напоминание об открытии окна/связанная система подачи свежего воздуха для вентиляции, улучшения качества сна |
| Недавно отремонтированный дом | Датчик качества воздуха | Непрерывный мониторинг содержания летучих органических соединений/формальдегида, связанный вытяжной вентилятор при превышении стандарта, и обнадеживающее уведомление будет отправлено после достижения стандарта |
| Детская комната | Датчик качества воздуха | Мониторинг PM2,5, автоматическое включение очистителя воздуха при превышении стандарта для обеспечения здоровья органов дыхания |
| Офис | Датчик качества воздуха | CO₂ + кондиционирование воздуха с учетом температуры и влажности для повышения эффективности работы |
