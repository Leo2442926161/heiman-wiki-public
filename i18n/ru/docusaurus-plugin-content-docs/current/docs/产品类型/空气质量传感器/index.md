# датчик качества воздуха

Датчики качества воздуха (Air Quality Monitor) используются для обнаружения различных загрязнителей в воздухе помещений и помогают поддерживать здоровую среду обитания.

## Параметры обнаружения

Датчики качества воздуха Hyman фиксируют следующие показатели:

| Параметры | Описание | Стандарты здравоохранения |
|------|------|----------|
| ТЧ2,5 | Мелкие твердые частицы | ≤ 35 мкг/м³ (среднее за 24 часа) |
| ПМ10 | Вдыхаемые твердые частицы | ≤ 50 мкг/м³ (среднее за 24 часа) |
| CO₂ | Углекислый газ | ≤ 1000 частей на миллион |
| ТВОК | Всего летучих органических соединений | ≤ 0,6 мг/м³ |
| Температура | Температура окружающей среды | 18-26°С |
| Влажность | Относительная влажность | 40-60% относительной влажности |

## Основные функции

- Многопараметрическое обнаружение качества воздуха
- Цветной дисплей (в зависимости от модели)
- Звуковая и световая сигнализация превышения норматива
- Записи исторических данных
- Связанный очиститель свежего воздуха/воздуха

## Применимые сценарии

- Спальня: контролирует уровень CO₂ и предлагает открыть окна для проветривания.
- Детская комната: защитите здоровье органов дыхания детей.
- Офис: повышение производительности и комфорта
- Недавно отремонтированные дома: следить за выбросами формальдегида/TVOC.
