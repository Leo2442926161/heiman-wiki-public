# Руководство по установке

## Требования к месту установки

- Высота настенного монтажа: **1,8 ~ 2,4 м**
- Установка на потолке: устанавливается на потолке в центре комнаты.
- **Избегайте**: прямых солнечных лучей, вентиляционных отверстий системы отопления/кондиционирования, вблизи источников тепла.
- Избегайте препятствий (мебели, стеклянных перегородок) в зоне обнаружения.

## Этапы установки

1. Выберите место установки (на стене или на потолке).
2. Закрепите монтажное основание винтами.
3. Вставьте аккумулятор, индикатор начнет мигать.
4. Установите корпус датчика на основание.
5. Отрегулируйте угол обнаружения (в пределах 110° по горизонтали).
6. Тест ходьбы: после прохождения зоны обнаружения должен загореться индикатор.
7. Сопряжение через шлюз

## Примечания

- Инфракрасные датчики чувствительны к изменениям температуры, поэтому расстояние обнаружения летом может сократиться в условиях высокой температуры.
- Избегайте обращений к окнам (изменения наружной температуры вызывают ложные срабатывания сигнализации).
- Срок службы батареи примерно 3 года (Zigbee)/1 год (Matter)
