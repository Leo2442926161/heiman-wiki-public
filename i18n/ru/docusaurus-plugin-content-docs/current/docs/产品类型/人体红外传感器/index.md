# Инфракрасный датчик человеческого тела

Инфракрасный датчик тела человека (PIR) обнаруживает движение путем обнаружения инфракрасного теплового излучения, излучаемого телом человека, и является наиболее часто используемым решением для обнаружения присутствия в умных домах.

## Серия продуктов

| Серия | Методы связи | Особенности |
|------|----------|------|
| ХС1МС | Зигби | Базовая модель, угол обнаружения 110° |
| ХС3МС | Зигби | Модернизированная модель, установка на потолок на 360° |
| Серия М1 | Материя / Зигби | Поддержка Matter over Thread, опционально: 110° или 360° |
| HS8MLS | Зигби | Модель на большие расстояния |
| HS8MIS | Зигби | Тело человека + композитный датчик освещенности |

## Основные функции

- Мобильное обнаружение людей
- Обнаружение освещенности (серия M1/HS8MIS)
- Расстояние обнаружения ≤7 м (25°C)
- Способ установки: настенный или потолочный.
- Срок службы батареи: 3 года (Zigbee)/1 год (Matter)

## Технические параметры (серия М1)

- Угол обнаружения: 110°/360°
- Расстояние обнаружения: ≤7 м
- Диапазон освещенности: 0 ~ 1200 люкс
- Источник питания: 3 В постоянного тока (литиевая батарея)
- Срок службы батареи: 3 года (Zigbee)/1 год (Matter)

## Пример сценария

- Свет включается, когда кто-то приходит, и гаснет, когда кто-то уходит.
- Связь безопасности: активируйте сигнал тревоги при обнаружении ненормального движения.
- Постановка на охрану при выходе из дома: автоматическое отключение кондиционера/отопления для экономии энергии, когда рядом никого нет.
