## Рекомендуемая сцена игрового процесса

| Сценарии | Рекомендуемые модели | Решения для связи |
|------|----------|----------|
| Умная безопасность всего дома | HS1CA-М | Matter over Thread изначально подключается к Apple Home + Google + Alexa, будильник → выключить газ → открыть окна |
| Стандартные требования ЕС | HS1CA-E | Подключитесь к HA через Z2M/ZHA, и электромагнитный клапан тягового газа автоматически отключится |
| Требования стандартов США | HS-720ES-М | Сертифицирован UL 2034, соответствует требованиям страхования |
| Недорогой ремонт старых домов | серия С1 | Основные функции, используемые со шлюзом |

## Рекомендации по выбору

- **Кроссплатформенность Matter**: [HS1CA-M](../../Wireless Protocol/Matter/HS1CA-M Интеллектуальная сигнализация угарного газа (Matter)) (Глобальная версия, официальная сертификация HA)
- **Рынок ЕС**: [HS1CA-E](../../Wireless Protocol/Zigbee/HS1CA-E - Интеллектуальная сигнализация угарного газа (Zigbee)) (Zigbee)
- **Рынок США**: HS-720ES-M (UL 2034)
- **Основные требования**: [Серия C1](../../Беспроводной протокол/Zigbee/Серия C1 - Сигнализация окиси углерода (Zigbee)) (Zigbee)
