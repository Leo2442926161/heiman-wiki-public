# датчик угарного газа

Окись углерода (CO) — бесцветный, не имеющий запаха и не вызывающий раздражения токсичный газ, известный как «невидимый убийца». В датчиках угарного газа Hyman используется **электрохимическая сенсорная технология** для точного определения концентрации CO в воздухе.

## Сертифицированные достижения

Компания Hyman выпускает **первую в мире систему сигнализации угарного газа, сертифицированную Matter** и сертифицированную Works with Home Assistant.

## Основные функции

- Высокоточный электрохимический датчик CO
- Диапазон обнаружения: 0 ~ 1000 частей на миллион.
- Сигнализация по степени концентрации (звуковая и световая подсказка)
- Звуковой сигнал 85 дБ
- Напоминание о низком заряде батареи
- Напоминание об истечении срока службы датчика (обычно 7 лет)
- Функция самотестирования

## Советы по безопасности

Концентрация CO и реакция организма человека:

| Концентрация CO (ppm) | Реакция человеческого тела |
|---------------|----------|
| 50 | Здоровые взрослые могут переносить 8 часов |
| 200 | Легкая головная боль через 2-3 часа |
| 400 | Головная боль и тошнота через 1-2 часа |
| 800 | Головокружение и судороги в течение 45 минут |
| 1600+ | Фатально в течение 2 часов |
