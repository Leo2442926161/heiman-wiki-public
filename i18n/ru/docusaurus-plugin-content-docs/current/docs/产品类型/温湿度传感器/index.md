# Датчик температуры и влажности

Датчики температуры и влажности используются для мониторинга температуры окружающей среды и относительной влажности в режиме реального времени и являются основным оборудованием для мониторинга окружающей среды умного дома.

## Серия продуктов

| Серия | Методы связи | Особенности |
|------|----------|------|
| Серия H1 | Материя / Зигби | Версия Matter over Thread сертифицирована WWHA |

## Основные функции

- Мониторинг температуры и влажности в режиме реального времени
- Напоминание о низком заряде батареи
-Поддержка управления связью (включение кондиционера при высокой температуре, включение осушителя при высокой влажности и т. д.)

## Технические параметры

- Точность температуры: ±0,3°C
- Точность влажности: ± 3% относительной влажности.
- Источник питания: батарея CR2.
- Срок службы батареи: около 2-3 лет

## Пример сценария

- Связанное кондиционирование/отопление для автоматического регулирования температуры в помещении.
- Связанный осушитель/увлажнитель для поддержания комфортной влажности.
- Мониторинг окружающей среды в винном погребе/сигарном шкафу
- Автоматическое управление вытяжными вентиляторами кухни/ванной комнаты.
- Экологический мониторинг посадки теплиц
