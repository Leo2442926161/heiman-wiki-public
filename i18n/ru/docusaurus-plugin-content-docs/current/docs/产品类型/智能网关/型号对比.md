# Сравнение моделей

## Сравнение различных моделей

| Модель | Максимальный лимит дополнительного устройства Zigbee | Сетевой метод | Особенности | Предложения по игре |
|------|------------------|----------|------|----------|
| [HS6GW](../../Протокол беспроводной связи/Zigbee/HS6GW — интеллектуальный шлюз (Zigbee)) | 128 | Wi-Fi + Ethernet | Последняя модель и поддержка будущих обновлений Thread | Собственное экологическое ядро ​​Haiman в сочетании с удаленным управлением через приложение Haiman, подходящее для пользователей, которых это не беспокоит |
| ХС5ГВ | 64 | Wi-Fi + Ethernet | Высокая производительность | Средние и крупные проекты, стабильные и надежные |
| ХС3ГВ | 32 | Wi-Fi | Расширенный | Достаточно для домов малого и среднего размера |
| HS1GW | 16 | Wi-Fi | Базовый | Начальный уровень, подходит для небольшого количества устройств |

## Рекомендуемая сцена игрового процесса

| Сценарий | Рекомендуемая модель | Описание |
|------|----------|------|
| Весь дом Хейман Экология | ХС6ГВ | Используется с датчиками Heiman Zigbee и удаленным управлением с помощью приложения Heiman, нет необходимости создавать собственный сервер |
| Пользователи сторонних платформ | — | Рекомендуется приобрести координатор Zigbee (например, SLZB-06) непосредственно для использования с Zigbee2MQTT/ZHA для большей гибкости |
| Маленькие и средние квартиры | ХС3ГВ | 32 ограничения на устройства для удовлетворения основных потребностей |

## Рекомендации по выбору

- **Собственная экология Haiman**: [HS6GW](../../Wireless Protocol/Zigbee/HS6GW - Smart Gateway (Zigbee)) (128 устройств, WiFi + Ethernet)
- **Сторонняя платформа**: рекомендуемый универсальный координатор Zigbee + Zigbee2MQTT/ZHA.
