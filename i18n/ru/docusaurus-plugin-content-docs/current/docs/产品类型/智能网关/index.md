# Умный шлюз

Smart Gateway — это основной центр системы умного дома Haiman, отвечающий за подключение различных датчиков Zigbee и реализацию управления связью между устройствами.

## Серия продуктов

| Серия | Методы связи | Особенности |
|------|----------|------|
| Серия HS1GW | Zigbee + Wi-Fi/Ethernet | Базовый шлюз |
| Серия HS3GW | Zigbee + Wi-Fi/Ethernet | Расширенный |
| Серия HS5GW | Zigbee + Wi-Fi/Ethernet | Высокопроизводительный шлюз |
| Серия HS6GW | Zigbee + Wi-Fi/Ethernet + Thread | Последняя модель поддерживает как Zigbee, так и Matter over Thread |

## Основные функции

- Координатор сети Zigbee
- Поддержка Wi-Fi/Ethernet для подключения к Интернету
- Автоматическое обнаружение и сопряжение устройств
- Привязка к локальной сцене (некоторые модели)
- Удаленный доступ (через облачную платформу Heiman)
- Поддерживает до 128 субустройств (в зависимости от модели)

## Применимые сценарии

- Основная точка подключения всей системы сбора данных Heiman.
- Сотрудничать с приложением Haiman для реализации удаленного мониторинга.
- Локальное централизованное управление малыми и средними коммерческими проектами

## Отличия от других шлюзов

Haiman Gateway — это вход в официальную экосистему. Пользователям, желающим использовать стороннюю платформу, рекомендуется использовать универсальный координатор Zigbee с Zigbee2MQTT/ZHA.
