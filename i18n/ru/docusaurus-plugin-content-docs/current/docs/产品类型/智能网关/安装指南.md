# Руководство по установке

## Требования к месту установки

- **Центральное расположение по всему дому** обеспечивает покрытие сигнала Zigbee для всех устройств.
- Старайтесь не размещать его в цельнометаллическом шкафу (металл блокирует беспроводные сигналы).
- **Сначала Ethernet**: рекомендуется использовать проводное сетевое соединение, которое более стабильно.
- Не более 20-30 м от других устройств Zigbee (в помещении)

## Этапы установки

1. Выберите место установки (тумба под телевизор в гостиной, рядом со слаботочной коробкой и т.п.)
2. Подключите адаптер питания.
3. Подключите кабель Ethernet (рекомендуется) или подключитесь к Wi-Fi.
4. Подождите, пока индикатор не загорится зеленым/горит постоянно.
5. Откройте приложение Haiman и отсканируйте QR-код в нижней части шлюза, чтобы добавить
6. После завершения добавления вы можете выполнить сопряжение дополнительного устройства.

## Примечания

- Шлюз — это ядро умного дома. Рекомендуется использовать резервный источник питания UPS.
- Соединение Ethernet более стабильно, чем Wi-Fi, и его рекомендуется использовать в первую очередь.
- Если сигнал устройства слабый, добавьте релейное устройство Zigbee (например, интеллектуальную розетку).
- Максимальное количество субустройств, поддерживаемых разными моделями, различно и должно выбираться в зависимости от масштаба.
