# Сравнение моделей

## Сравнение различных моделей

| Модель | Протокол | Принцип обнаружения | Источник питания | Особенности | Предложения по игре |
|------|------|----------|------|------|----------|
| Датчик температуры пожара | Зигби/Суб-1G | Термистор (фиксированная температура + перепад температуры) | Батарея | Противодымные помехи | Специально для кухни/гаража, сигнализация аномального повышения температуры, электромагнитный клапан для отключения газа |

## Рекомендуемая сцена игрового процесса

| Сценарии | Рекомендуемые модели | Решения для связи |
|------|----------|----------|
| Безопасность кухни | Датчик температуры | Аномальное повышение температуры → Отключите газовый клапан → Звуковая и световая сигнализация → Push-уведомление на мобильный телефон, чтобы избежать ложных сигналов тревоги о дыме от приготовления пищи |
| Гараж/котельная | Датчик температуры | Контролируйте температуру в аппаратной, чтобы предотвратить самовозгорание, и подключите вытяжной вентилятор для охлаждения |
| Дополняется датчиком дыма | Датчик температуры + дым | Датчик дыма охватывает весь дом, датчик температуры - кухню/гараж, дополнительная сигнализация более надежна |

## Рекомендации по выбору

- **Только на кухне**: датчик температуры (во избежание ложных срабатываний дыма при приготовлении пищи)
- **Работает с датчиком дыма**: датчик дыма охватывает спальню/гостиную, датчик температуры охватывает кухню/гараж.
