# Сравнение моделей

## Сравнение различных моделей

| Модель | Протокол | Технология обнаружения | Установка | Особенности | Предложения по игре |
|------|------|----------|------|------|----------|
| [HS8OS](../../Протокол беспроводной связи/Zigbee/HS8OS - Датчик присутствия человека (Zigbee)) | Зигби | Радар миллиметрового диапазона | Потолок | Стационарный детектор людей | Точнее, чем PIR, ванная/спальня остается неподвижной в течение длительного времени, что решает проблему отключения света PIR без посторонней помощи |

## Рекомендуемая сцена игрового процесса

| Сценарии | Рекомендуемые модели | Решения для связи |
|------|----------|----------|
| Не выключайте свет в ванной | HS8OS | Свет горит, когда люди есть, и выключается, когда люди уходят (обнаружение микродыхания для решения проблемы, связанной с отключением стационарного ИК-света) |
| Мониторинг сна в спальне | HS8OS | Определите, есть ли кто-нибудь на кровати, и подключите кондиционер для автоматической регулировки температуры |
| Умный уход за пожилыми людьми | HS8OS | Тайм-аут туалета для пожилых людей → Системное предупреждение оповещает членов семьи, чтобы предотвратить несчастные случаи |
| Экономия энергии в конференц-зале | HS8OS | Никто не доходит до N минут → Автоматически выключайте свет и кондиционер для экономии энергии и снижения потребления |

## Рекомендации по выбору

- **Решение для PIR-подсветки болевых точек**: [HS8OS](../../Wireless Protocol/Zigbee/HS8OS - Датчик присутствия человеческого тела (Zigbee)) (миллиметровые волны, обнаружение неподвижности)
- **В сочетании с PIR**: PIR обнаруживает движение + HS8OS обнаруживает неподвижное присутствие, дополнительное использование для достижения наилучших результатов.
