# Руководство по установке

## Способ ношения

- **Зажим для ремня**: зажимается на ремне или поясе.
- **Ремешок**: вешается на шею и на грудь.

## Инструкция по применению

1. Установите аккумулятор
2. Выберите способ ношения (зажим для ремня или ремешок).
3. Носите его на своем теле
4. Сопряжение через шлюз
5. Тест моделирования:
   - Встаньте, а затем быстро присядьте или лягте на землю.
   - Датчик должен обнаружить падение и вызвать тревогу в течение нескольких секунд.
   - Нажмите кнопку бедствия, чтобы проверить ручную сигнализацию.

## Примечания

- Рекомендуется носить в течение всего дня (можно снять, чтобы зарядить/заменить батарею во время принятия душа)
- Алгоритм искусственного интеллекта используется для распознавания падений от повседневной деятельности (сидение, наклон) с низким уровнем ложных тревог.
- Автоматический сигнал тревоги после обнаружения падения, или вы можете нажать кнопку вручную, чтобы вызвать помощь.
- Может быть связан с аварийной кнопкой для достижения двойной защиты.
