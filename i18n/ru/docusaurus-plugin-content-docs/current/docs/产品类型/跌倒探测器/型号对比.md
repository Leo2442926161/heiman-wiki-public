# Сравнение моделей

## Сравнение различных моделей

| Модель | Протокол | Датчик | Источник питания | Особенности | Предложения по игре |
|------|------|--------|------|------|----------|
| [HS2FD](../../Протокол беспроводной связи/Zigbee/HS2FD - Детектор падения (Zigbee)) | Зигби | Акселерометр + гироскоп | Батарея | AI-алгоритм обнаружения падения | Носимый дизайн, автоматическое обнаружение падения + помощь одним щелчком мыши, подходит для одиноких пожилых людей и домов престарелых |

## Рекомендуемая сцена игрового процесса

| Сценарии | Рекомендуемые модели | Решения для связи |
|------|----------|----------|
| Мониторинг одиноких пожилых людей | ХС2ФД | Обнаружение падения → Автоматическое нажатие на мобильные телефоны детей → Автоматический звонок 120 после таймаута и отсутствия ответа |
| Учреждение по уходу за пожилыми людьми | ХС2ФД | Платформа централизованного управления + Звуковая и световая сигнализация пола + Синхронная сигнализация портативного устройства для медсестер |
| Послеоперационная реабилитация | ХС2ФД | Риск падения обнаруживается, когда пациент встает с постели → Раннее предупреждение на посту медсестры |
| Ванная/Ночь | ХС2ФД | Конструкция с защитой IP, можно носить в ванне + автоматический сигнал тревоги, если вы встаете и падаете ночью |

## Рекомендации по выбору

- **Уход на дому**: [HS2FD](../../Wireless Protocol/Zigbee/HS2FD - Детектор падения (Zigbee)) (носимое устройство, обнаружение AI)
- **В сочетании с аварийной кнопкой**: автоматическое обнаружение падения HS2FD + аварийная кнопка для ручной помощи, двойная защита.
