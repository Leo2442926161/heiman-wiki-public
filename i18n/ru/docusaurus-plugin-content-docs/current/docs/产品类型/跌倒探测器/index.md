# детектор падения

Детектор падения специально разработан для пожилых людей. Он может автоматически обнаруживать события падения и вовремя отправлять сигнал бедствия. Это ключевой элемент оборудования для интеллектуальных решений по уходу за пожилыми людьми.

## Серия продуктов

| Серия | Методы связи | Особенности |
|------|----------|------|
| ХС2ФД | Зигби | Интеллектуальное обнаружение падения, автоматическая сигнализация |

## Основная технология

Он использует **алгоритм объединения нескольких датчиков**, объединяет данные акселерометра и гироскопа и использует алгоритм искусственного интеллекта для определения положения при падении, эффективно отличает падения от обычных действий (таких как сидение, наклон) и снижает частоту ложных тревог.

## Основные функции

- Автоматическое обнаружение падения
- Вызвать помощь вручную одним щелчком мыши
- Носимый дизайн (зажим для ремня или ремешок)
- Напоминание о низком заряде батареи

## Применимые сценарии

- Уход за пожилыми людьми на дому: круглосуточный уход за одинокими пожилыми людьми
- Дома престарелых: централизованное содержание и управление
- Реабилитационный центр: Мониторинг риска падений у послеоперационных пациентов
