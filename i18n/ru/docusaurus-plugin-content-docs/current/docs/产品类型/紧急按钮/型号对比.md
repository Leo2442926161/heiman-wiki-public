# Сравнение моделей

## Сравнение различных моделей

| Модель | Протокол | Метод триггера | Источник питания | Особенности | Предложения по игре |
|------|------|----------|------|------|----------|
| Аварийная кнопка | Зигби | Длительное нажатие триггера | Батарея | Портативный + настенный двойного назначения | Пожилые люди могут носить его с собой, помощь в чрезвычайных ситуациях одним щелчком мыши, подключение к мобильному телефону + звуковая и световая сигнализация |

## Рекомендуемая сцена игрового процесса

| Сценарии | Рекомендуемые модели | Решения для связи |
|------|----------|----------|
| Уход за пожилыми людьми на дому | Аварийная кнопка | Внезапный дискомфорт для пожилых людей → Длительное нажатие → Нажмите на мобильные телефоны всех членов семьи → Связанная звуковая и световая сигнализация → Уведомление о собственности |
| Безопасность детской комнаты | Аварийная кнопка | Если ребенок попал в чрезвычайную ситуацию дома один → Нажмите кнопку, чтобы включить сигнал тревоги на мобильном телефоне родителя |
| Магазин/Касса | Тревожная кнопка | Скрытая установка, активирующая тихую сигнализацию в случае ограбления, связанную с видеонаблюдением + тихое уведомление службы безопасности |
| Туалет/ванная | Аварийная кнопка | Если пожилой человек поскользнется и упадет в душе, водонепроницаемая аварийная кнопка окажется под рукой |
