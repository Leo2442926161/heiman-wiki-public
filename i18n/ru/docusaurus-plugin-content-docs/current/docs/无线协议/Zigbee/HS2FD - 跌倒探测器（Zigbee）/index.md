#HS2FD — Детектор падения (Zigbee)

## Обзор

HS2FD — это интеллектуальный детектор падения Zigbee, выпущенный компанией Hyman. Он создан специально для пожилых людей. Имеет встроенный акселерометр и гироскоп. Он использует алгоритм искусственного интеллекта для определения положения при падении и может автоматически отправлять сигнал бедствия.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Акселерометр + гироскоп |
| Технология обнаружения | Алгоритм обнаружения падения AI |
| Метод питания | Батарея постоянного тока 3 В |
| Метод ношения | Зажим для ремня/шнур |

## Особенности

- Автоматическое обнаружение падения
- Вызвать помощь вручную одним щелчком мыши
- Алгоритм искусственного интеллекта отличает падения от обычных действий (сидение, наклон) для уменьшения ложных тревог.
- Носимый дизайн (зажим для ремня или ремешок)
- Напоминание о низком заряде батареи

## Применимые сценарии

- Уход за пожилыми людьми на дому: круглосуточный уход за одинокими пожилыми людьми
- Дома престарелых: централизованное содержание и управление
- Реабилитационный центр: Мониторинг риска падений у послеоперационных пациентов
