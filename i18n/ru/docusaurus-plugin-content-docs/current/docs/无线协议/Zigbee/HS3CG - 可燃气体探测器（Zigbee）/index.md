#HS3CG — Детектор горючих газов (Zigbee)

## Обзор

HS3CG — детектор горючих газов Zigbee третьего поколения от Hyman. Он имеет улучшенную точность обнаружения и скорость реагирования по сравнению с первым поколением и подходит для обнаружения утечек природного и сжиженного нефтяного газа.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Полупроводник |
| Тестовый газ | Природный газ (метан)/Сжиженный нефтяной газ (СНГ) |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | 220 В переменного тока / 12 В постоянного тока |
| Стандарты сертификации | ГБ 15322 |
| Рабочая температура | -10°С ~ +50°С |

## Особенности

- Обнаружение утечек природного газа/сжиженного газа
- Сигнализация классификации концентрации
- Автоматическое отключение (с электромагнитным клапаном)
- Звуковая и визуальная сигнализация 85 дБ
- Функция самотестирования
- Напоминание об истечении срока службы датчика
- Улучшенная производительность беспроводной связи

## Стандарты сертификации

- ГБ 15322

## Применимые аксессуары

- Газовый аварийный запорный клапан серии JT
