# Стандарты сертификации

HS1SA-E прошел следующие сертификации:

- **EN 14604** — Европейский стандарт для датчиков дыма, см. [Европейский стандарт EN](../../../../ApprovalStandards/EN European Standard.md)
- **CE/RED/EMC** — Директива Европейского Союза по беспроводному оборудованию и электромагнитной совместимости.
- **Zigbee 3.0 / Q-Label** — сертификация Zigbee Alliance, см. [Другие сертификаты](../../../Стандарты сертификации/Другие сертификаты.md)
- **Red Dot Design Award** — награда Red Dot Product Design Award.
