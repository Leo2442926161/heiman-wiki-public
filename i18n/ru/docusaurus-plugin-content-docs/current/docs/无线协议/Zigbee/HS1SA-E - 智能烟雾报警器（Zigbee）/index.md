# HS1SA-E — Умная дымовая сигнализация (Zigbee)

## Обзор

HS1SA-E — это стандартная интеллектуальная дымовая сигнализация Zigbee, выпущенная компанией Hyman для рынка ЕС. Он использует фотоэлектрическую технологию обнаружения дыма, прошел сертификацию EN 14604 и получил награду Red Dot Design Award.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Инфракрасный фотоэлектрический |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | 3 В постоянного тока (CR123A) |
| Срок службы батареи | Примерно 3 года |
| Расстояние связи | ≤70 м (открытая площадка) |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности |
| Способ установки | Монтаж потолка |

## Особенности

- Фотоэлектрический детектор дыма для обнаружения дыма от медленного горения и открытого огня.
- Звуковая и световая сигнализация громкостью 85 дБ.
- Напоминание о низком заряде батареи
- Функция самотестирования (тест в один клик)
- Сигнализация против несанкционированного доступа
-Поддержка удаленного отключения звука
- ОТА-обновление прошивки

## Стандарты сертификации

- EN 14604 (Европейский стандарт для датчиков дыма)
- Премия Red Dot Design
-СЕ/КРАСНЫЙ

## Применимые аксессуары

- Батарея CR123A
- Монтажные винты/расширительная трубка
