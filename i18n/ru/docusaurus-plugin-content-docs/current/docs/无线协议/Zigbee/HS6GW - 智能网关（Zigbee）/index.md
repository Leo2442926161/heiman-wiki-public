# HS6GW — интеллектуальный шлюз (Zigbee)

## Обзор

HS6GW — это интеллектуальный шлюз Zigbee, выпущенный Heiman. Будучи координатором сети Heiman Zigbee, он подключается к Интернету через восходящий канал Wi-Fi или Ethernet и поддерживает доступ до 128 субустройств.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Zigbee 3.0 (Координатор) |
| Восходящая связь | Wi-Fi/Ethernet |
| Макс. узлы | 128 субустройств |
| Метод питания | Источник питания USB 5 В постоянного тока |

## Особенности

- Координатор сети Zigbee
- Восходящее соединение Wi-Fi/Ethernet
- Автоматическое обнаружение и сопряжение устройств
- Связь с локальной сценой
- Удаленный доступ (через облачную платформу Heiman)
- Поддерживает до 128 суб-устройств
- ОТА-обновление прошивки

## Применимые сценарии

- Основная точка подключения всей системы сбора данных Heiman.
- Сотрудничать с приложением Haiman для реализации удаленного мониторинга.
