# Стандарты сертификации

HS6GW (версия Zigbee) прошел следующие сертификации:

- **CE/RED** — Директива Европейского Союза по беспроводному оборудованию.
- **CCC** — Обязательная сертификация продукции в Китае, см. [Обязательная сертификация CCC в Китае](../../../../Стандарты сертификации/CCC China Compulory Certification.md)
