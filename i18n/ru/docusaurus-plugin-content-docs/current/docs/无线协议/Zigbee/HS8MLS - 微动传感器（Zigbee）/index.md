# HS8MLS — Микродатчик движения (Zigbee)

## Обзор

HS8MLS — это инфракрасный датчик человеческого тела (PIR) Zigbee дальнего действия, выпущенный Heiman. Он специально разработан для сцен, требующих обнаружения на большом расстоянии, и подходит для открытых пространств, таких как коридоры и холлы.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Пассивный инфракрасный порт (PIR) |
| Расстояние обнаружения | Обнаружение на большом расстоянии |
| Метод питания | Батарея постоянного тока 3 В |

## Особенности

- Обнаружение движения на большом расстоянии
- Конструкция с низким энергопотреблением
- Поддержка привязки сцены
