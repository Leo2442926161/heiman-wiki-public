# M1-Z — Инфракрасный датчик тела человека (Zigbee)

## Обзор

M1-Z — это компактный инфракрасный датчик человеческого тела Zigbee, выпущенный Heiman. Он поддерживает угол обнаружения 110° или 360° (в зависимости от способа установки), имеет встроенный датчик освещенности и подходит для сценариев интеллектуального освещения и безопасности.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Пассивное инфракрасное (PIR) + освещение |
| Угол обнаружения | 110° (на стене) / 360° (на потолке) |
| Расстояние обнаружения | ≤7 м (25°C) |
| Диапазон освещенности | 0 ~ 1200 люкс |
| Метод питания | Литиевая батарея постоянного тока 3 В |
| Срок службы батареи | Примерно 3 года |
| Рабочая температура | -10°С ~ +50°С |
| Способ установки | Настенный/потолочный |

## Особенности

- Мобильное обнаружение людей
- Обнаружение освещенности (0 ~ 1200 люкс)
- Переключение 110°/360°
- Напоминание о низком заряде батареи
- Сигнализация против несанкционированного доступа
- Компактный дизайн
