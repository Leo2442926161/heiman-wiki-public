#Список поддерживаемых моделей

Heyman предлагает самую широкую линейку продуктов Zigbee в отрасли, и ниже представлены выпущенные поддерживаемые модели.

## Категория охранной сигнализации

| Модель | Название продукта | Описание |
|------|----------|------|
| HS1SA-E | Умная дымовая сигнализация (Zigbee) | Версия для ЕС |
| HS1SA-Z | Умная дымовая сигнализация (Zigbee) | Китайская версия |
| ХС3СА | Интеллектуальная дымовая сигнализация (Zigbee) | Дымовая сигнализация третьего поколения |
| HS8SA | Интеллектуальная дымовая сигнализация (Zigbee) | Высокотемпературный тип связи |
| HS1CA-E | Интеллектуальная сигнализация угарного газа (Zigbee) | Версия для ЕС |
| HS1CA-Z | Интеллектуальная сигнализация угарного газа (Zigbee) | Китайская версия |
| HS1CG | Детектор горючих газов (Zigbee) | Природный газ/сжиженный газ |
| HS3CG | Детектор горючих газов (Zigbee) | Детектор газа третьего поколения |

## Инструкции по названию модели

| Правила именования | Значение |
|----------|------|
| Суффикс -Е | Версия для ЕС (стандарт 868 МГц / EN) |
| Суффикс -Z | Версия для Китая (2,4 ГГц/китайский стандарт) |
| ХС1/2/3/8 | Номер поколения продукта: чем больше число, тем новее поколение |
