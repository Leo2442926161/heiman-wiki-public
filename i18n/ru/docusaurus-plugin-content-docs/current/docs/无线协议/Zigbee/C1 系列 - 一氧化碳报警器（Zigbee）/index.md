Серия C1 — сигнализация окиси углерода (Zigbee)

## Обзор

Серия C1 — это базовая сигнализация угарного газа Zigbee, выпущенная Heiman. Он использует электрохимические датчики и подходит для сценариев обнаружения утечек угарного газа. Он прошел сертификацию EN 50291.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Электрохимический |
| Диапазон обнаружения | 0 ~ 1000 частей на миллион |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | Питание от аккумулятора |
| Срок службы датчика | Примерно 7 лет |
| Рабочая температура | -10°С ~ +50°С |

## Особенности

- Электрохимическое обнаружение CO
- Сигнализация классификации концентрации
- Звуковая и визуальная сигнализация 85 дБ
- Напоминание о низком заряде батареи
- Функция самотестирования

## Стандарты сертификации

-ЕН 50291
