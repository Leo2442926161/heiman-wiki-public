#HS1MS — Инфракрасный датчик человеческого тела (Zigbee)

## Обзор

HS1MS — это стандартный инфракрасный датчик человеческого тела (PIR) Zigbee, выпущенный компанией Hyman. Он использует технологию пассивного инфракрасного обнаружения и может улавливать инфракрасное тепловое излучение, излучаемое движущимся человеческим телом. Он широко используется в сценариях интеллектуального освещения и безопасности.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Пассивный инфракрасный порт (PIR) |
| Угол обнаружения | 110° |
| Расстояние обнаружения | ≤7 м (25°C) |
| Метод питания | Литиевая батарея постоянного тока 3 В |
| Срок службы батареи | Примерно 3 года |
| Рабочая температура | -10°С ~ +50°С |
| Способ установки | Настенный монтаж |

## Особенности

- Мобильное обнаружение людей
- Широкоугольное обнаружение 110°
- Напоминание о низком заряде батареи
- Сигнализация против несанкционированного доступа
- Поддержка привязки сцены
