#HS3MS — Инфракрасный датчик человеческого тела (Zigbee)

## Обзор

HS3MS — это инфракрасный датчик человеческого тела Zigbee третьего поколения от Hyman. Он имеет конструкцию, монтируемую на потолке на 360°, а его чувствительность обнаружения и охват значительно улучшены по сравнению с предыдущим поколением.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Пассивный инфракрасный порт (PIR) |
| Угол обнаружения | 360° (потолочный монтаж) |
| Расстояние обнаружения | ≤7 м (25°C) |
| Метод питания | Литиевая батарея постоянного тока 3 В |
| Срок службы батареи | Примерно 3 года |
| Рабочая температура | -10°С ~ +50°С |
| Способ установки | Монтаж потолка |

## Особенности

- Мобильное обнаружение людей
- Всенаправленное покрытие 360°
- Напоминание о низком заряде батареи
- Сигнализация против несанкционированного доступа
- Улучшена чувствительность обнаружения.
- Поддержка привязки сцены
