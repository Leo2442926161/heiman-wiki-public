#HS3SA — Умная дымовая сигнализация (Zigbee)

## Обзор

HS3SA — это интеллектуальная дымовая сигнализация Zigbee третьего поколения от Hyman. Он имеет улучшенные характеристики обнаружения и возможности связи на основе первого поколения и подходит для рынка ЕС.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Инфракрасный фотоэлектрический |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | 3 В постоянного тока (CR123A) |
| Срок службы батареи | Примерно 3 года |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности |
| Способ установки | Монтаж потолка |

## Особенности

- Фотоэлектрический детектор дыма
- Звуковая и световая сигнализация громкостью 85 дБ.
- Напоминание о низком заряде батареи
- Функция самотестирования
- Сигнализация против несанкционированного доступа
-Поддержка удаленного отключения звука
- ОТА-обновление прошивки
- Улучшенная производительность беспроводной связи
