#Зигби

Zigbee — это протокол беспроводной ячеистой сети с низким энергопотреблением и низкой скоростью передачи данных, основанный на стандарте IEEE 802.15.4. Это один из самых зрелых и популярных протоколов в области умного дома.

## Хайман и Зигби

Hyman — один из первых китайских производителей, принявших протокол Zigbee, и имеет огромную линейку продуктов Zigbee. Устройства Hyman Zigbee совместимы со стандартом **Zigbee 3.0** и могут взаимодействовать с основными шлюзами и координаторами Zigbee.

## Технические характеристики

| Особенности | Описание |
|------|------|
| Диапазон частот | 2,4 ГГц (по всему миру) |
| Скорость передачи | До 250 кбит/с |
| Топология сети | Сетка самовосстанавливающаяся сеть |
| Количество узлов | Одна сеть поддерживает до 65535 узлов |
| Расстояние связи | Около 10-30 метров в помещении (видимые препятствия) |
| Потребляемая мощность | Чрезвычайно низкий срок службы батареи датчика до 3 лет |

## Совместимые платформы

Устройства Hyman Zigbee можно подключать к следующим платформам:

- **Zigbee2MQTT** — мост Zigbee с MQTT с открытым исходным кодом.
- **ZHA** — Домашний помощник со встроенной интеграцией Zigbee.
- **Hubitat** — Локализованный интеллектуальный хаб
- **Homey** — Многопротокольный интеллектуальный хаб
- **SmartThings Hub** — центр умного дома Samsung.
- **Шлюз Tuya Zigbee** — Экосистема Tuya
- **Шлюз Hyman HS6GW** — Официальный шлюз

## Линия продуктов Hyman Zigbee

- Датчики дыма серии HS1SA / HS3SA / HS8SA
- Сигнализация угарного газа серии HS1CA/C1
- Детекторы горючих газов серии HS1CG/HS3CG
- Дверной датчик серии HS1DS
- Инфракрасные датчики человеческого тела серии HS1MS/HS3MS/M1
- Датчики погружения воды серии HS2WL / HS3WL / HS1WL
- Умные розетки серии HS2SK
- Детектор падения HS2FD
