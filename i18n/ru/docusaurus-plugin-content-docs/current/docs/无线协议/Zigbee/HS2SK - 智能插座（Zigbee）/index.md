#HS2SK — Умная розетка (Zigbee)

## Обзор

HS2SK — это интеллектуальная измерительная розетка Zigbee, выпущенная Heiman. Он поддерживает дистанционное управление переключателем и измерение энергопотребления в режиме реального времени. Он подходит для интеллектуального управления бытовой техникой и имеет точность ±2%.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Номинальное напряжение | 220 В переменного тока (китайский стандарт) / 110 В переменного тока (американский стандарт) |
| Максимальная нагрузка | 16А/3680Вт |
| Точность измерения | ±2% |
| Метод питания | Сетевой источник питания |

## Особенности

- Дистанционное управление переключателем
- Учет мощности/топлива в режиме реального времени
- Защита от перегрузки
- Таймер
- Статистика мощности
- Поддержка привязки сцены

## Применимые сценарии

- Дистанционно выключайте забытые приборы
- Статистика энергопотребления кондиционера/водонагревателя
- Выйдите из домашнего режима, чтобы отключить ненужные приборы одним щелчком мыши.
