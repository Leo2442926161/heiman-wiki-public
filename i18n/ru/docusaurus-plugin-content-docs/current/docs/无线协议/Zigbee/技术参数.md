# Технические параметры

Ниже приведены общие технические параметры устройств Heiman Zigbee.

## Параметры беспроводной связи

| Параметр | Значение |
|------|-----|
| Стандарты протокола | Зигби 3.0 (IEEE 802.15.4) |
| Рабочий диапазон частот | 2,4 ГГц (общий глобальный диапазон частот ISM) |
| Скорость передачи | До 250 кбит/с |
| Топология сети | Сетка самовосстанавливающаяся сеть |
| Мощность узла | Максимум 65535 узлов в одной сети |
| Расстояние связи | Около 10–30 метров в помещении (видя препятствия на стене) |
| Метод шифрования | AES-128-CCM |

## Совместимость

| Проект | Описание |
|------|------|
| Координатор Зигби | Совместим со стандартным координатором Zigbee 3.0 |
| Поддерживаемые платформы | Zigbee2MQTT, ZHA (Домашний помощник), Hubitat, Homey, SmartThings, шлюз Tuya Zigbee, шлюз Haiman HS6GW |
| ОТА-обновление | Поддержка обновления прошивки по беспроводной сети (некоторые модели) |

## Электроэнергия и энергопотребление

| Параметр | Значение |
|------|-----|
| Метод питания | Питание от аккумулятора (CR123A/CR2450/AA) или от USB/сети (тип розетки) |
| Срок службы батареи | Около 2–3 лет для датчиков (в зависимости от частоты отчетности) |
| Энергопотребление в режиме ожидания | Уровень микроампер, поддерживает механизм опроса сна |

## Параметры среды

| Параметр | Значение |
|------|-----|
| Рабочая температура | -10°С ~ +55°С |
| Рабочая влажность | ≤ 95 % относительной влажности (без конденсации) |
| Уровень защиты IP | Зависит от конкретной модели (тип датчика обычно IP30) |
