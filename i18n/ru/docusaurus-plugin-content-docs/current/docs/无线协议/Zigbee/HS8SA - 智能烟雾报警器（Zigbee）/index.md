#HS8SA — Умная дымовая сигнализация (Zigbee)

## Обзор

HS8SA — это интеллектуальная дымовая сигнализация Zigbee + Wi-Fi от Hyman для рынка США. Он имеет функцию высокотемпературной связи и прошел сертификацию UL 217.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 / Wi-Fi |
| Тип датчика | Инфракрасный фотоэлектрический |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | 3 В постоянного тока (CR123A) |
| Рабочая температура | -10°С ~ +50°С |
| Стандарты сертификации | УЛ 217 |
| Применимые рынки | Соединенные Штаты |

## Особенности

- Фотоэлектрический детектор дыма
- Сигнализация высокой температуры
- Звуковая и световая сигнализация громкостью 85 дБ.
- Напоминание о низком заряде батареи
- Функция самотестирования
- Сигнализация против несанкционированного доступа
-Поддержка удаленного отключения звука
- Двойной протокол связи (Zigbee + Wi-Fi)

## Стандарты сертификации

- UL 217 (Американский стандарт дымовой сигнализации)
