# Стандарты сертификации

HS8SA прошла следующие сертификации:

- **EN 14604** — Европейский стандарт для датчиков дыма (тип ЕС), см. [Европейский стандарт EN](../../../../ApprovalStandards/EN European Standard.md)
- **UL 217** — стандарт безопасности дымовой сигнализации США (модель для США), см. [UL Underwriters Laboratories](../../../../certification-standards/ULUnderwriters Laboratories.md)
- **CE/RED/EMC** — Директива Европейского Союза по беспроводному оборудованию и электромагнитной совместимости.
- **Zigbee 3.0 / Q-Label** — сертификация Zigbee Alliance, см. [Другие сертификаты](../../../Стандарты сертификации/Другие сертификаты.md)
