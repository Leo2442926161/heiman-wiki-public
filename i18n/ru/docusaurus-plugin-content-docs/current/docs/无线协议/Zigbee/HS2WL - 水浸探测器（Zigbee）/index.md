#HS2WL — Детектор утечки воды (Zigbee)

## Обзор

HS2WL — это детектор погружения в воду Zigbee второго поколения, выпущенный компанией Hyman. Он поддерживает двухканальное обнаружение датчиков и кабелей и может использоваться для обнаружения событий утечки воды, таких как разрывы водопроводных труб и переливы раковины.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Метод обнаружения | Обнаружение зонда + обнаружение кабеля (двухканальный) |
| Метод питания | Батарея постоянного тока 3 В |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности |

## Особенности

- Обнаружение утечки/скопления воды в режиме реального времени
- Зонд + кабель двухканального обнаружения
- Зуммеровая сигнализация
- Напоминание о низком заряде батареи
- Антикоррозионная конструкция зонда

## Применимые аксессуары

- Удлинить кабель обнаружения
