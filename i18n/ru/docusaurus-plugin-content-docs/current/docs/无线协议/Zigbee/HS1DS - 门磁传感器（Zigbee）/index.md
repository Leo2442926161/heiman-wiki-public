# HS1DS — Магнитный датчик двери (Zigbee)

## Обзор

HS1DS — это дверной магнитный датчик Zigbee, выпущенный Heiman. Он используется для определения состояния открытия и закрытия дверей и окон. Это базовый компонент интеллектуальной системы безопасности.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Герконовый переключатель |
| Метод питания | CR1632 батарейка-таблетка |
| Срок службы батареи | Около 2 ~ 3 лет |
| Разрешение на установку | ≤10 мм |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности |

## Особенности

- Обнаружение состояния открытия/закрытия дверей и окон в режиме реального времени
- Тамперная сигнализация (Тампер)
- Напоминание о низком заряде батареи
-Поддержка связи (открытие двери, включение света, постановка на охрану при выходе из дома и т.д.)
- Простота установки (клейкая лента 3М или крепление винтами)

## Применимые аксессуары

- Батарейка CR1632.
- Двусторонний скотч 3М/шурупы.
