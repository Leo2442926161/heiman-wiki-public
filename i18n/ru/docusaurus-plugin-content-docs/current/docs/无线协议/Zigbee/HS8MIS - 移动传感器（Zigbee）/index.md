#HS8MIS — мобильный датчик (Zigbee)

## Обзор

HS8MIS — это многофункциональный датчик движения Zigbee, выпущенный Heiman. Он объединяет функции обнаружения людей PIR и обнаружения освещенности и подходит для интеллектуальных сцен с двойным мониторингом света и движения.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Пассивное инфракрасное (PIR) + освещение |
| Угол обнаружения | 110° |
| Расстояние обнаружения | ≤7 м |
| Метод питания | Батарея постоянного тока 3 В |

## Особенности

- Мобильное обнаружение людей
- Обнаружение освещенности
- Конструкция с низким энергопотреблением
- Поддержка привязки сцены
