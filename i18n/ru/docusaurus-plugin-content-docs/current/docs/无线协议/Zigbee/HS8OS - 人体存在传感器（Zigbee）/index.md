#HS8OS — Датчик присутствия человека (Zigbee)

## Обзор

HS8OS — это датчик присутствия человека Zigbee, выпущенный компанией Hyman. Он использует технологию радара миллиметрового диапазона и может обнаруживать присутствие человеческого тела в статическом состоянии (например, когда он лежит, спокойно сидит), компенсируя недостатки традиционных PIR-датчиков, которые могут обнаруживать только движение.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Радар миллиметрового диапазона (mmWave) |
| Диапазон обнаружения | Радиус 3 ~ 5 м |
| Возможность обнаружения | Наличие неподвижного тела человека (в том числе обнаружение микродыхания) |
| Метод питания | Источник питания переменного тока |
| Способ установки | Монтаж потолка |

## Особенности

- Стационарное обнаружение тела человека (микродвижения дыхания)
- Радиус покрытия 3~5 метров
- Потолочный монтаж, оптимальный угол обнаружения
- Может использоваться в дополнение к PIR-датчикам.

## Применимые сценарии

- Интеллектуальное освещение: всегда включено, когда рядом кто-то есть, автоматически выключается, когда никого нет рядом
- Экономия энергии при кондиционировании воздуха: автоматически выключает кондиционер, когда люди ходят.
- Умный уход за пожилыми людьми: раннее предупреждение пожилых людей, если они остаются неподвижными в течение длительного времени.
