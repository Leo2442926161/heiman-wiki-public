# HS1WL — Детектор утечки воды (Zigbee)

## Обзор

HS1WL — это датчик погружения в воду Zigbee первого поколения, выпущенный компанией Hyman. Он использует метод обнаружения зонда и может эффективно обнаруживать утечку или скопление воды. Это базовое оборудование для мониторинга погружения в воду.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Метод обнаружения | Обнаружение зонда |
| Метод питания | Батарея постоянного тока 3 В |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности |

## Особенности

- Обнаружение утечки/скопления воды в режиме реального времени
- Зуммеровая сигнализация
- Напоминание о низком заряде батареи
- Антикоррозионная конструкция зонда
