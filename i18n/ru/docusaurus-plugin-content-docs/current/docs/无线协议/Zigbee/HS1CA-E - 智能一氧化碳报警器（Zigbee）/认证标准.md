# Стандарты сертификации

HS1CA-E прошел следующие сертификации:

- **EN 50291** — Европейский стандарт для сигналов тревоги угарного газа, см. [Европейский стандарт EN](../../../../Стандарты сертификации/Европейский стандарт EN.md)
- **CE/RED/EMC** — Директива Европейского Союза по беспроводному оборудованию и электромагнитной совместимости.
