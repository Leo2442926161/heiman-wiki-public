# HS1CA-E — интеллектуальный сигнализатор угарного газа (Zigbee)

## Обзор

HS1CA-E — это интеллектуальная сигнализация угарного газа (CO) Zigbee от Hyman для рынка ЕС. Он использует электрохимический датчик и может точно обнаруживать угарный газ с концентрацией 0 ~ 1000 частей на миллион. Он прошел сертификацию EN 50291.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип датчика | Электрохимический |
| Диапазон обнаружения | 0 ~ 1000 частей на миллион |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | 3 В постоянного тока (CR123A) |
| Срок службы датчика | Примерно 7 лет |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности |
| Способ установки | Настенный монтаж |

## Особенности

- Высокоточный электрохимический датчик CO
- Сигнализация по степени концентрации (звуковая и световая подсказка)
- Звуковой сигнал 85 дБ
- Напоминание о низком заряде батареи
- Напоминание об истечении срока службы датчика
- Функция самотестирования
- ОТА-обновление прошивки

## Стандарты сертификации

- EN 50291 (европейский стандарт сигнализации угарного газа)
-СЕ/КРАСНЫЙ

## Применимые аксессуары

- Батарея CR123A
