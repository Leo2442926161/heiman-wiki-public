# Серия JT - Клапан аварийного отключения газа (Zigbee)

## Обзор

Клапан аварийного отключения газа серии JT представляет собой исполнительное устройство, используемое совместно с детектором горючих газов Hyman Zigbee. При обнаружении утечки газа он может дистанционно или автоматически закрыть клапан газопровода, чтобы предотвратить несчастные случаи.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Зигби 3.0 |
| Тип | Автономный + соединение Zigbee |
| Тестовый газ | Природный газ / Сжиженный нефтяной газ |
| Стандарты сертификации | ГБ 15322 |
| Рабочая температура | -10°С ~ +50°С |

## Особенности

- Связан с HS1CG/HS3CG для автоматического закрытия клапана.
-Поддержка дистанционного ручного закрытия клапана
- Нажатие сигнализации утечки
- Привод электромагнитного клапана

## Стандарты сертификации

- ГБ 15322

## Применимые аксессуары

- Детектор горючих газов HS1CG/HS3CG
