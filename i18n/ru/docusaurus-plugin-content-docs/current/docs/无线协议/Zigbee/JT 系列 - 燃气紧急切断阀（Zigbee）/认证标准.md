# Стандарты сертификации

Серия JT прошла следующие сертификации:

- **GB 15322/CCC** — Обязательная сертификация продукции в Китае, см. [Обязательная сертификация CCC в Китае](../../../../Certification Standards/CCC China Compulory Certification.md)
- **CE** — Декларация соответствия Европейского Союза.
