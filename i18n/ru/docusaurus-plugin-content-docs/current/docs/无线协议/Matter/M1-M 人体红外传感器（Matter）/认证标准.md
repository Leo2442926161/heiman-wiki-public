# Стандарты сертификации

M1-M прошел следующие сертификации:

- **Знак CE** — Декларация соответствия ЕС, см. [CE EU Conformity](../../../../certification-standards/CE EU-Conformity.md)
- **Сертификация Matter** — пройдена сертификация CSA Alliance Matter для обеспечения межплатформенного взаимодействия умных домов.
