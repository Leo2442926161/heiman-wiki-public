# Инфракрасный датчик человеческого тела М1-М (Материя)

M1-M — это инфракрасный датчик человеческого тела **Материя превыше нити**, выпущенный компанией Hyman. Он объединяет функции обнаружения PIR и обнаружения освещенности и подходит для сценариев автоматизации умного дома.

## Обзор продукта

- **Тип продукта**: Инфракрасный датчик человеческого тела.
- **Протокол связи**: материя превыше нити
- **Метод обнаружения**: пассивное инфракрасное излучение (PIR) + обнаружение освещенности.
- **Типичный сценарий**: свет включается, когда приходят люди, система безопасности, автоматизация окружающей среды.

## Технические параметры

| Параметр | Значение |
|------|-----|
| Угол обнаружения | 110° (на стене) / 360° (на потолке) |
| Расстояние обнаружения | ≤7 м (25°C) |
| Диапазон освещенности | 0~1200 люкс |
| Протокол связи | Материя важнее потока (IEEE 802.15.4) |
| Диапазон рабочих частот | 2,4 ГГц |
| Метод питания | Литиевая батарея постоянного тока 3 В |
| Срок службы батареи | Примерно 1 год (режим Материи) |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности (без конденсации) |

## Особенности

- Мобильное обнаружение людей
- Обнаружение окружающего освещения
- Расстояние обнаружения ≤7 м
- Поддержка установки на стене или потолке
- Важна мультиэкологическая межплатформенная совместимость.
