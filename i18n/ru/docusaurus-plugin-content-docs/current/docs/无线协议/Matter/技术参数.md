# Технические параметры

Ниже приведены общие характеристики оборудования Hyman Matter.

## Параметры беспроводной связи

| Параметр | Значение |
|------|-----|
| Базовый протокол | Поток (IEEE 802.15.4) |
| Диапазон рабочих частот | 2,4 ГГц (диапазон ISM) |
| Стандарты прикладного уровня | Материя 1.x |
| Топология сети | Сетка самовосстанавливающаяся сеть |
| Метод шифрования | AES-128-CCM |

## Требования к оборудованию

| Проект | Описание |
|------|------|
| Пограничный маршрутизатор | Требуется, например, Apple HomePod mini, Apple TV 4K, Google Nest Hub 2-го поколения и т. д. |
| Поддержка мобильных терминалов | iOS / Android (сетевое распространение через различные экологические приложения) |
| Межэкологическая совместимость | Apple HomeKit, Google Home, Amazon Alexa, Домашний помощник, SmartThings |

## Электроэнергия и энергопотребление

| Параметр | Значение |
|------|-----|
| Метод питания | Питание от аккумулятора (CR123A/CR2450 и т. д.) |
| Срок службы батареи | Около 1–3 лет (в зависимости от типа устройства и частоты использования) |
| Энергопотребление в режиме ожидания | Чрезвычайно низкий уровень, поддерживает механизм пробуждения во время сна |

## Параметры среды

| Параметр | Значение |
|------|-----|
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤ 95 % относительной влажности (без конденсации) |
