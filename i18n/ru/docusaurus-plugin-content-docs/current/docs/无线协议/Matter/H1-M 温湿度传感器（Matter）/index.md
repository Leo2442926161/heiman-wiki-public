# Датчик температуры и влажности H1-M (Материя)

H1-M — это датчик температуры и влажности **Matter over Thread**, выпущенный компанией Hyman, который может контролировать температуру окружающей среды и относительную влажность в режиме реального времени и поддерживает связь с системой умной домашней автоматизации.

## Обзор продукта

- **Тип продукта**: Датчик температуры и влажности.
- **Протокол связи**: материя превыше нити
- **Параметры обнаружения**: температура, относительная влажность.
- **Типичные сценарии**: Мониторинг окружающей среды в домах, офисах, винных погребах и т. д.

## Технические параметры

| Параметр | Значение |
|------|-----|
| Точность температуры | ±0,3°С |
| Точность влажности | ±3% относительной влажности |
| Протокол связи | Материя важнее потока (IEEE 802.15.4) |
| Диапазон рабочих частот | 2,4 ГГц |
| Метод питания | Литиевая батарея CR2 |
| Срок службы батареи | Около 2-3 лет |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности (без конденсации) |

## Особенности

- Мониторинг температуры и влажности в режиме реального времени
- Высокоточное измерение (температура ±0,3°C, влажность ±3% относительной влажности)
- Напоминание о низком заряде батареи
- Важна мультиэкологическая межплатформенная совместимость.
