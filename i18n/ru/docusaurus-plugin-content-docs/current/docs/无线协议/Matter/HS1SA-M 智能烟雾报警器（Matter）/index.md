# HS1SA-M Интеллектуальная дымовая сигнализация (Материя)

HS1SA-M — это интеллектуальная дымовая сигнализация **Matter over Thread**, выпущенная компанией Hyman. Он использует фотоэлектрическую технологию обнаружения дыма и поддерживает межэкологическое соединение платформ умного дома.

## Обзор продукта

- **Тип продукта**: Умная дымовая сигнализация
- **Протокол связи**: материя превыше нити
- **Стандарт сертификации**: EN 14604.
- **Типичные сценарии**: предупреждение о пожаре в домах, квартирах, гостиницах и т. д.

## Технические параметры

| Параметр | Значение |
|------|-----|
| Тип датчика | Инфракрасный фотоэлектрический |
| Протокол связи | Материя важнее потока (IEEE 802.15.4) |
| Диапазон рабочих частот | 2,4 ГГц |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | 3 В постоянного тока (литиевая батарея CR123A) |
| Срок службы батареи | Примерно 3 года |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности (без конденсации) |

## Особенности

- Фотоэлектрический детектор дыма для обнаружения медленного горения и дыма от открытого огня.
- Звуковая и световая сигнализация громкостью 85 дБ.
- Напоминание о низком заряде батареи
- Функция самотестирования (тест в один клик)
- Сигнализация против несанкционированного доступа
- Дистанционное отключение звука
- Важна мультиэкологическая межплатформенная совместимость.

## Сертификация

- Сертификация материи
