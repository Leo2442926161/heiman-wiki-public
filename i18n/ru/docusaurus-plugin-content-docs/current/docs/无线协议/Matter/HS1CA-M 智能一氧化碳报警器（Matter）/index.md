# HS1CA-M Интеллектуальная сигнализация окиси углерода (Matter)

HS1CA-M — это первый в мире сигнализатор угарного газа, сертифицированный Matter**, в котором используется технология электрохимических датчиков и поддерживается межэкологическое соединение Matter over Thread.

## Обзор продукта

- **Тип продукта**: Интеллектуальный сигнализатор угарного газа
- **Протокол связи**: материя превыше нити
- **Стандарт сертификации**: EN 50291.
- **Типичный сценарий**: предупреждение об утечке CO в домах, квартирах и других местах.

## Технические параметры

| Параметр | Значение |
|------|-----|
| Тип датчика | Электрохимический датчик CO |
| Протокол связи | Материя важнее потока (IEEE 802.15.4) |
| Диапазон рабочих частот | 2,4 ГГц |
| Диапазон обнаружения | 0~1000 частей на миллион |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | 3 В постоянного тока (литиевая батарея CR123A) |
| Срок службы батареи | Примерно 3 года |
| Срок службы датчика | Типичные 7 лет |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности (без конденсации) |

## Особенности

- Высокоточный электрохимический датчик CO
- Сигнализация по степени концентрации (звуковая и световая подсказка)
- Звуковая и световая сигнализация громкостью 85 дБ.
- Напоминание о низком заряде батареи
- Напоминание об истечении срока службы датчика
- Функция самотестирования
- Важна мультиэкологическая межплатформенная совместимость.

## Сертификация

- Сертификат EN 50291
- Сертификация материи
- Работает с сертификацией Home Assistant
