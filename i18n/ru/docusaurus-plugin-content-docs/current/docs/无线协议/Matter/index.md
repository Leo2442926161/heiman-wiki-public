#Материя

Matter (ранее известный как Project CHIP) — это стандарт межсетевых соединений умного дома, запущенный Альянсом стандартов соединений CSA с целью разрушить барьеры брендов и достичь межэкологической совместимости.

## Хейман и Материя

Хайман был одним из первых, кто внедрил протокол Matter, запустив **первые в мире сигнализаторы угарного газа, сертифицированные Matter**. Все устройства Hyman Matter используют решение **Matter over Thread**.

## Технические характеристики

| Особенности | Описание |
|------|------|
| Базовый протокол | Нить (ячеистая сеть с низким энергопотреблением) |
| Топология сети | Ячеистая самовосстанавливающаяся сеть: чем больше устройств, тем шире покрытие |
| Потребляемая мощность | Чрезвычайно низкий уровень, может работать от аккумулятора 1–3 года |
| Безопасность | Поддерживает шифрование AES-128-CCM, требуется аутентификация устройства |
| Совместимость | В Apple HomeKit, Google Home, Amazon Alexa, Home Assistant и других устройствах |

## Требования к пограничному маршрутизатору

Для использования устройств Matter over Thread требуется граничный маршрутизатор Thread:

- Apple HomePod mini/HomePod 2-го поколения/Apple TV 4K
- Google Nest Hub 2-го поколения/Nest Hub Max
-Amazon Echo 4-го поколения
- Домашний помощник Зеленый + Connect ZBT-2
- Домашний профи

## Сертифицированная продукция Hyman Matter

- Умная дымовая сигнализация (версия для США/версия для ЕС)
- Интеллектуальная сигнализация угарного газа (версия для США/версия для ЕС)
- Инфракрасный датчик тела человека М1-М
- Датчик погружения в воду Л1-М
- Датчик температуры и влажности Н1-М
