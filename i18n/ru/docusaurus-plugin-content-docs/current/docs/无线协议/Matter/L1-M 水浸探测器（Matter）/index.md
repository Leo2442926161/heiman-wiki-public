# Детектор погружения в воду Л1-М (Материя)

L1-M — это погружной детектор воды **Материя превыше нити**, выпущенный компанией Hyman. Он обнаруживает утечку воды или скопившуюся воду с помощью зонда, что может эффективно предотвратить потери имущества, вызванные разрывами водопроводных труб.

## Обзор продукта

- **Тип продукта**: Датчик обнаружения погружения в воду
- **Протокол связи**: материя превыше нити
- **Метод обнаружения**: обнаружение контакта зонда.
- **Типичные сценарии**: Кухня, ванная комната, прачечная, обнаружение протечки воды в подвале.

## Технические параметры

| Параметр | Значение |
|------|-----|
| Метод обнаружения | Тип контакта зонда из нержавеющей стали |
| Протокол связи | Материя важнее потока (IEEE 802.15.4) |
| Диапазон рабочих частот | 2,4 ГГц |
| Метод питания | Питание от аккумулятора |
| Метод сигнализации | Зуммер + нажатие приложения |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности (без конденсации) |

## Особенности

- Обнаружение протечек/застойной воды в режиме реального времени
- Зуммер на месте сигнализации
- Удаленное нажатие тревоги через приложение
- Напоминание о низком заряде батареи
- Антикоррозионная конструкция зонда
- Важна мультиэкологическая межплатформенная совместимость.
