# HS1DS-M — Интеллектуальный дверной датчик (Материя превыше нити)

## Обзор

HS1DS-M — это версия умного дверного датчика Heiman Matter over Thread. Он поддерживает собственный доступ ко всем платформам, совместимым с Matter, таким как Apple Home, Google Home, Amazon Alexa, Home Assistant и т. д. без необходимости использования собственного шлюза.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | Материя важнее темы |
| Метод питания | CR1632 батарейка-таблетка |
| Срок службы батареи | Около 2-3 лет |
| Расстояние установки | ≤10 мм |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤95% относительной влажности |

## Особенности

- Обнаружение состояния открытия/закрытия дверей и окон в режиме реального времени
- Тамперная сигнализация (Тампер)
- Напоминание о низком заряде батареи
- Кроссплатформенная совместимость: Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Сеть Thread Mesh, локальное управление с низкой задержкой
- Не требуется выделенный шлюз, требуется пограничный маршрутизатор потока
