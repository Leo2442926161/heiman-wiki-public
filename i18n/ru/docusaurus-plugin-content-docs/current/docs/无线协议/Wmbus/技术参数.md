# Технические параметры

Ниже приведены общие технические параметры оборудования Heimann Wireless M-Bus (wM-Bus).

## Параметры беспроводной связи

| Параметр | Значение |
|------|-----|
| Стандарты протокола | EN 13757-4 (физический уровень беспроводной M-Bus) |
| Рабочий диапазон частот | 868 МГц (европейский диапазон частот SRD, 868,0–868,6 МГц) |
| Метод модуляции | ФСК |
| Скорость передачи | 4,8 кбит/с/16,384 кбит/с/100 кбит/с (режим T/режим S/режим R) |
| Топология сети | Звезда (инструмент → коллектор/шлюз) |
| Расстояние связи | Около 100–300 метров в помещении, около 500–1000 метров на открытом пространстве |
| Передача мощности | ≤ 25 мВт (+14 дБм) |

## Режим работы

| Режим | Оценить | Описание |
|------|------|------|
| Режим S | 16,384 кбит/с | Стандартный режим, низкое энергопотребление, подходит для инструментов с батарейным питанием |
| Т-режим | 100 кбит/с | Высокоскоростной режим, подходящий для сценариев с большими объемами трафика |
| Режим R | 4,8 кбит/с | Узкополосный режим, передача на большие расстояния |

## Электроэнергия и энергопотребление

| Параметр | Значение |
|------|-----|
| Метод питания | Источник питания от литиевой батареи (ER18505/ER26500 и другие промышленные литиевые батареи) |
| Срок службы батареи | 6–10 лет (в зависимости от частоты отчетности, типичное значение — 1 раз в день) |
| Энергопотребление в режиме ожидания | Уровень микроампер, глубокий сон |
| Получить чувствительность | Прибл. -105 дБм (режим S) |

## Параметры среды

| Параметр | Значение |
|------|-----|
| Рабочая температура | -20°C ~ +60°C (промышленное исполнение) |
| Рабочая влажность | ≤ 95 % относительной влажности (без конденсации) |
| Уровень защиты IP | IP65 ~ IP68 (в зависимости от конкретной модели, применимо к наружным приборам) |

## Протокол данных

| Слой | Стандарт |
|----|------|
| Физический уровень | ЕН 13757-4 |
| Канальный уровень | EN 13757-4 (формат кадра беспроводной шины M-Bus) |
| Прикладной уровень | EN 13757-3 (прикладной уровень M-Bus) |
| Метод шифрования | AES-128 (дополнительное шифрование) |
