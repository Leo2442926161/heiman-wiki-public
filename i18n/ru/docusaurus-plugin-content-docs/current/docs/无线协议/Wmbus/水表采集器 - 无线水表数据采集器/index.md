# Коллектор счетчиков воды - беспроводной сборщик данных счетчиков воды

## Обзор

Коллектор счетчиков воды — это беспроводное устройство сбора данных с шиной wM-Bus, выпущенное компанией Hyman для интеллектуального водоснабжения. Он работает в диапазоне частот 868 МГц (стандарт EN 13757) и поддерживает подсчет импульсов и беспроводную передачу данных. Он используется для удаленного считывания данных измерений счетчиков воды и загрузки их в централизованную систему снятия показаний счетчиков. Он подходит для таких сценариев, как квартиры, жилые районы, коммерческие здания и мониторинг промышленных вод.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | wM-шина (EN 13757) |
| Диапазон рабочих частот | 868 МГц (европейский диапазон SRD) |
| Скорость передачи | 4,8 кбит/с/16,384 кбит/с/100 кбит/с (режим T/S/R) |
| Расстояние связи | Около 100–300 метров в помещении, около 500–1000 метров на открытом пространстве |
| Интерфейс датчика | Герконовый переключатель/импульсный вход датчика Холла |
| Метод питания | Промышленная литиевая батарея (ER18505/ER26500) |
| Срок службы батареи | 6–10 лет (обычно сообщается 1 раз в день) |
| Рабочая температура | -20°С ~ +60°С |
| Рабочая влажность | ≤ 95% относительной влажности |
| Уровень защиты | IP65 ~ IP68 (в зависимости от среды установки) |
| Шифрование данных | AES-128 (опционально) |

## Особенности

-Поддерживает геркон/сбор импульсов датчика Холла
- Подсчет импульсов и преобразование потока, автоматический отчет о данных об использовании
-Поддержка wM-Bus S/T/R трех режимов работы
- Низкое энергопотребление, глубокий сон, длительное время автономной работы
- Шифрование данных AES-128 для обеспечения безопасности связи.
-Поддержка удаленного обновления прошивки
- Механизм отключения, повторного подключения и повторной передачи данных
- Поддерживает протокол прикладного уровня EN 13757-3.
- Может использоваться с коллектором HS1-CM/шлюзом HS1-GW для централизованной загрузки.
