# wM-Bus (беспроводная шина учета)

wM-Bus (Wireless Meter-Bus) — это протокол беспроводного считывания показаний счетчиков, определенный европейским стандартом (EN 13757). Он широко используется для удаленного сбора данных с приборов учета энергии, таких как счетчики воды, газа, электроэнергии и тепла.

## Технические характеристики

| Особенности | Описание |
|------|------|
| Стандарты | ЕН 13757 |
| Рабочий диапазон частот | 868 МГц (стандарт ЕС) |
| Расстояние передачи | Открытая среда 100-500 метров |
| Потребляемая мощность | Крайне низкий срок службы батареи до 10 лет |
| Основные приложения | Измерение энергии, централизованное снятие показаний счетчиков |

## Хейманн и wM-Bus

Heimann использует технологию wM-Bus для предоставления решений для интеллектуальных газовых и водных месторождений. Продукты wM-Bus в основном используются в крупных и средних промышленных и коммерческих проектах и ​​используются в сочетании с интеллектуальной платформой противопожарной защиты Haiman.

## Применимые сценарии

- Централизованное квартирное/квартирное считывание показаний газового счетчика.
- Мониторинг энергопотребления коммерческих зданий
- Инфраструктура умного города

> **Примечание**: wM-Bus — это вертикальный отраслевой протокол, который в основном используется для решений проектов Heiman в области газовой и противопожарной защиты и не подходит для обычных домашних сценариев «умного дома».
