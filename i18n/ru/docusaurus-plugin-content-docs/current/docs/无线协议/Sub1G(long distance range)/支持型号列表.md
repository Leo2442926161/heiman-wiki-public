#Список поддерживаемых моделей

Продукты серии Haiman Sub-1G для дальней связи в основном предназначены для крупномасштабных независимых сетевых сценариев и могут обеспечивать соединение и взаимодействие между устройствами без шлюза.

## Автономная дымовая сигнализация (серия S1)

| Модель | Название продукта | Описание |
|------|----------|------|
| С1-СА | Автономная дымовая сигнализация (Sub-1G) | Базовый тип, поддерживает беспроводное соединение |
| С1-СА-П | Автономная дымовая сигнализация (Sub-1G) | Версия с функцией камеры |
| С1-СА-Т | Независимая дымовая сигнализация (Sub-1G) | Версия с высокотемпературным соединением |
| С1-СА-РФ | Независимая дымовая сигнализация (взаимодействующий тип RF) | Версия радиочастотной сети |

## Подключенные аксессуары

| Модель | Название продукта | Описание |
|------|----------|------|
| С1-РМ | Беспроводной ретранслятор (Sub-1G) | Расширенная дистанция связи |
| С1-РК | Пульт дистанционного управления (Sub-1G) | Для тестирования/отключения звука/сброса |

## Особенности

- **Независимая работа**: нет шлюза, не требуется Интернет, прямое беспроводное соединение между устройствами.
- **Взаимосвязанная сигнализация**: любое устройство активирует сигнал тревоги, и все устройства в одной группе подадут сигнал тревоги одновременно.
- **Сверхдальнее расстояние**: расстояние связи может достигать 1–3 километров на открытой местности.
- **Сверхнизкое энергопотребление**: питание от батареи, срок службы батареи до 3–5 лет.
