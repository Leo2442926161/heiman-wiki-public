# Sub-1G (связь на дальние расстояния)

Sub-1G — это технология беспроводной связи, работающая в диапазоне частот ниже 1 ГГц. Он обладает характеристиками сверхдальнего расстояния передачи и чрезвычайно низкого энергопотребления. Он очень подходит для сценариев Интернета вещей, требующих широкомасштабного покрытия.

## Особенности

| Особенности | Описание |
|------|------|
| Диапазон рабочих частот | 433 МГц/868 МГц (ЕС)/915 МГц (США) |
| Расстояние передачи | До 1-3 километров на открытой местности |
| Проникающая способность | Лучше, чем 2,4 ГГц, сильное проникновение в стенные препятствия |
| Потребляемая мощность | Чрезвычайно низкий срок службы батареи может достигать 5-10 лет |
| Скорость передачи | Низкий (подходит для сценариев с небольшим объемом данных) |

## Сценарии применения

- **Независимая сигнализация дыма/CO**: соединена между собой на разных этажах здания.
- **Система противопожарной защиты крупных зданий**: торговые центры, гостиницы, школы.
- **Сообщество умных пожилых людей**: вызов службы экстренной помощи между подразделениями
- **Sub-1G + шлюз Wi-Fi**: дополнительное устройство обменивается данными через Sub-1G, а шлюз подключается к облаку.
