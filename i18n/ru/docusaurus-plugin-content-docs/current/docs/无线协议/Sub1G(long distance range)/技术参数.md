# Технические параметры

Ниже приведены общие технические параметры продуктов серии Hyman Sub-1G для дальней связи.

## Параметры беспроводной связи

| Параметр | Значение |
|------|-----|
| Диапазон рабочих частот | 433 МГц/868 МГц/915 МГц (в зависимости от региона рынка) |
| Метод модуляции | ФСК/ООК |
| Скорость передачи | Около 10~100 кбит/с (видеосегмент и конфигурация) |
| Топология сети | Звезда / точка-точка / сеть Интернет |
| Расстояние связи | Около 1–3 километров на открытой местности (видеосегмент и мощность) |
| Емкость сети | Одна группа поддерживает до 50+ узлов |

## Региональное распределение полос частот

| Регион | Общие диапазоны частот |
|------|----------|
| Китай | 433 МГц |
| Европа | 868 МГц |
| Северная Америка | 915 МГц |

## Электроэнергия и энергопотребление

| Параметр | Значение |
|------|-----|
| Метод питания | Питание от аккумулятора (литиевая батарея CR123A/AA) |
| Срок службы батареи | Около 3–5 лет (в зависимости от частоты отчетности) |
| Передача мощности | ≤ 10 мВт (+10 дБм), соответствует нормам SRD |
| Энергопотребление в режиме ожидания | Уровень микроампер, чрезвычайно низкий |
| Получение чувствительности | Около -110 дБм ~ -120 дБм |

## Параметры среды

| Параметр | Значение |
|------|-----|
| Рабочая температура | -10°С ~ +55°С |
| Рабочая влажность | ≤ 95 % относительной влажности (без конденсации) |
