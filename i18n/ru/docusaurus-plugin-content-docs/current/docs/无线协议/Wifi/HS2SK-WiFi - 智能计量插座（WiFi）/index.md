#HS2SK-WiFi - Умная розетка со счетчиком (WiFi)

## Обзор

HS2SK-WiFi — это интеллектуальная розетка для измерения Wi-Fi, выпущенная Heiman. Он поддерживает прямое соединение 802.11 b/g/n на частоте 2,4 ГГц с домашними маршрутизаторами и обеспечивает удаленное управление и мониторинг энергопотребления без необходимости использования дополнительных шлюзов. Подходит для интеллектуального управления электроприборами в домах, офисах и других местах.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | IEEE 802.11 b/g/n (2,4 ГГц) |
| Максимальная нагрузка | 16А |
| Максимальная мощность | 3680 Вт (230 В) / 1 800 Вт (110 В) |
| Точность измерения | ±2% |
| Метод питания | Сетевой источник питания |
| Энергопотребление в режиме ожидания | ≤ 0,5 Вт |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤ 90% относительной влажности |
| Материал корпуса | PC V0 огнестойкий |
| Технические характеристики Джека | Европейский стандарт/Американский стандарт (в зависимости от версии для рынка) |

## Особенности

-Поддержка Smart Life/Tuya Smart App дистанционного управления
- Мониторинг мощности в режиме реального времени и статистика энергопотребления
- Дистанционное управление переключателем
- Переключатель времени и обратный отсчет
- Связывание сцен и автоматизация
- Голосовое управление (совместимо с Alexa, Google Home)

## Стандарты сертификации

-CE/FCC
- РоХС
- ETL/UL (в зависимости от версии рынка)
