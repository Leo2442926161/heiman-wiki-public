# HS8SA-WiFi — интеллектуальная дымовая сигнализация (версия Wi-Fi для США)

## Обзор

HS8SA-WiFi — это интеллектуальная дымовая сигнализация Wi-Fi, выпущенная компанией Hyman для рынка США. Он использует фотоэлектрическую технологию обнаружения дыма, поддерживает сеть Wi-Fi и независимые два режима работы и прошел сертификацию UL 217. Подходит для обеспечения пожарной безопасности домов, квартир и других жилых помещений.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Протокол связи | IEEE 802.11 b/g/n (2,4 ГГц) |
| Тип датчика | Инфракрасный фотоэлектрический |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | Питание от аккумулятора |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤ 90% относительной влажности |
| Способ установки | Монтаж потолка |

## Особенности

- Фотоэлектрический детектор дыма для обнаружения дыма от медленного горения и открытого огня.
- Удаленный сигнал тревоги по сети Wi-Fi
- Независимый режим работы (локальная звуковая и световая сигнализация все равно может срабатывать при отключении сети)
- Звуковая и световая сигнализация громкостью 85 дБ.
- Напоминание о низком заряде батареи
- Функция самотестирования (тест в один клик)
- Сигнализация против несанкционированного доступа
-Поддержка удаленного отключения звука

## Стандарты сертификации

- UL 217 (Американский стандарт для датчиков дыма)
-FCC
- Соответствие NFPA 72
