# Стандарты сертификации

HS8SA-WiFi прошел следующие сертификации:

- **UL 217** — американский стандарт безопасности дымовой сигнализации, см. [UL Underwriters Laboratories](../../../../certificationstandards/ULUnderwriters Laboratories.md)
- **FCC** — сертификация Федеральной комиссии по связи.
- **CE** — Декларация соответствия Европейского Союза.
