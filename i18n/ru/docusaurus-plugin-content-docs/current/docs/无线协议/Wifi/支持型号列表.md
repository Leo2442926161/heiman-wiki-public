#Список поддерживаемых моделей

Wi-Fi-оборудование Heiman в основном предназначено для сценариев прямого подключения и может быть подключено к домашней сети без выделенного шлюза.

## Умная розетка

| Модель | Название продукта | Описание |
|------|----------|------|
| HS2SK-WiFi | Умная розетка (WiFi) | Прямое подключение к Wi-Fi 2,4 ГГц, поддержка дистанционного управления и синхронизации |

## Категория охранной сигнализации

| Модель | Название продукта | Описание |
|------|----------|------|
| HS8SA-Wi-Fi | Умная дымовая сигнализация (версия Wi-Fi для США) | Версия WiFi-сигнализатора дыма для рынка США |

## Класс шлюза

| Модель | Название продукта | Описание |
|------|----------|------|
| ХС6ГВ | Умный домашний шлюз (WiFi + Zigbee) | Подключайтесь к Интернету через Wi-Fi и управляйте подустройствами Zigbee |
| HS6GW-2 | Умный домашний шлюз (WiFi + Zigbee) | Шлюз второго поколения, повышение производительности |

## Примечания

- Все устройства Wi-Fi поддерживают только диапазон **2,4 ГГц** и несовместимы с диапазоном 5 ГГц.
- Рекомендуется использовать сеть с шифрованием WPA2-PSK.
- Для работы сети требуется маршрутизатор для правильного подключения к Интернету.
