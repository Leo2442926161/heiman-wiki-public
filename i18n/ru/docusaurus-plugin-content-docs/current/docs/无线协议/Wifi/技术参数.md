# Технические параметры

Ниже приведены общие технические характеристики Wi-Fi-устройств Heimann.

## Параметры беспроводной связи

| Параметр | Значение |
|------|-----|
| Стандарты протокола | IEEE 802.11 б/г/н |
| Диапазон рабочих частот | 2,4 ГГц (не совместим с 5 ГГц) |
| Скорость передачи | До 150 Мбит/с (802.11n) |
| Топология сети | Топология звезда (подключается напрямую к маршрутизатору) |
| Расстояние связи | Около 30–50 метров в помещении (в зависимости от производительности маршрутизатора) |
| Метод шифрования | WPA2-PSK/WPA3 |

## Метод сетевого распространения

| Метод | Описание |
|------|------|
| СмартКонфиг | Мобильное приложение Настройка сети в один клик (традиционное оборудование Wi-Fi) |
| Настройка с помощью Bluetooth | Настройка учетных данных WiFi через временные каналы Bluetooth (некоторые новые модели) |
| режим точки доступа | Включите точку доступа на устройстве и настройте ее после подключения телефона |

## Электроэнергия и энергопотребление

| Параметр | Значение |
|------|-----|
| Метод питания | Сетевое питание (тип розетки/тип шлюза) или аккумуляторное питание (тип сигнализации) |
| Уровень энергопотребления | Выше, чем Zigbee/Thread, из-за необходимости поддерживать соединение TCP/IP |
| Энергопотребление в режиме ожидания | Тип розетки около 0,5 Вт ~ 1 Вт |

## Параметры среды

| Параметр | Значение |
|------|-----|
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤ 90 % относительной влажности (без конденсации) |
