#Wi-Fi

Wi-Fi — самый популярный протокол беспроводной сети, основанный на стандарте IEEE 802.11. Некоторые интеллектуальные устройства Heiman поддерживают соединение Wi-Fi и могут быть напрямую подключены к маршрутизатору без необходимости использования дополнительного шлюза.

## Применимые сценарии

Устройства Wi-Fi подходят для следующих ситуаций:

- Полная сетевая инфраструктура, нет необходимости приобретать дополнительные шлюзы
- Устройства, не чувствительные к энергопотреблению (например, умные розетки)
- Сценарии, требующие удаленного доступа и облачных сервисов.
- Дома уже покрыты развитыми сетями Wi-Fi Mesh.

## Продукты Hyman Wi-Fi

Heimann предлагает продукты, поддерживающие подключение Wi-Fi:

- **Smart Socket HS2SK-WiFi** — Измерение энергопотребления, дистанционное включение
- **Smart Gateway** — подключается к маршрутизатору через Wi-Fi.

## Примечания

- Батарея датчика Wi-Fi разряжается быстрее, чем Zigbee/Thread.
- Большое количество устройств Wi-Fi может повлиять на стабильность сети.
- Основными продуктами Haiman являются Zigbee/Matter over Thread, а линейка продуктов Wi-Fi относительно оптимизирована.
