#HS6GW — шлюз умного дома

## Обзор

HS6GW — это шлюз для умного дома, выпущенный Heiman, поддерживающий восходящий канал Wi-Fi + Ethernet, встроенный координатор Zigbee и способный управлять до 128 подустройствами Zigbee. Реализация удаленного доступа и облачных сервисов посредством подключения к Интернету является основным узлом системы умного дома Haiman.

## Технические характеристики

| Параметр | Значение |
|------|-----|
| Восходящая связь | IEEE 802.11 b/g/n (2,4 ГГц Wi-Fi) + 10/100M Ethernet |
| Нисходящая связь | Zigbee 3.0 (режим координатора) |
| Максимальное количество подустройств | 128 |
| Метод питания | 5 В постоянного тока / 1 А (Микро USB) |
| Энергопотребление в режиме ожидания | ≤ 1 Вт |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤ 90% относительной влажности |
| Световой индикатор | Статус Wi-Fi/Питание/Состояние сети Zigbee |
| Метод сброса | Кнопка сброса + длительное нажатие для восстановления заводских настроек |

## Особенности

- Двойные восходящие каналы: проводное соединение Wi-Fi или Ethernet, гибкий выбор в зависимости от сетевой среды.
-Поддержка удаленного доступа: проверяйте состояние устройства в любое время и в любом месте через облачное приложение.
- Связь с локальной сценой: поддерживает автоматическую работу оборудования в локальной сети (все еще доступно при отключении)
- ОТА-обновление прошивки
- Поддержка платформы Tuya/Smart Life
- Управление до 128 суб-устройствами Zigbee
-Поддержка самовосстановления сети Zigbee и оптимизации маршрутизации.

- РоХС
