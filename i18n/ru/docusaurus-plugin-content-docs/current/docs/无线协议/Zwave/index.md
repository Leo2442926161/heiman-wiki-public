#Z-Wave

Z-Wave — это протокол беспроводной Mesh-сети, разработанный для умных домов. Он управляется Z-Wave Alliance и имеет широкое проникновение на европейский и американский рынки.

## Технические характеристики

| Особенности | Описание |
|------|------|
| Диапазон рабочих частот | 868,42 МГц (ЕС) / 908,42 МГц (США) |
| Скорость передачи | До 100 кбит/с (серия Z-Wave 800) |
| Топология сети | Mesh-сеть, до 232 узлов |
| Совместимость | Обратная совместимость со всеми устройствами Z-Wave |
| Безопасность | Z-Wave S2 шифрование |

## Хайман и Z-Wave

Hyman предлагает различные датчики, совместимые с Z-Wave, которые в основном продаются на рынках Европы и Америки. Версии устройств Heyman Z-Wave включают:

- Датчик угарного газа HS1CA-Z
- Датчик дыма HS1SA-Z
- Датчик двери HS1DS-Z

## Совместимые платформы

- Центр SmartThings
- Высота Хубитат
- Домашний профи
- Домашний помощник (через Z-Wave JS)
-OpenHAB

> **Примечание**: Z-Wave пользуется низкой популярностью на китайском рынке, а продукция Haiman Z-Wave в основном ориентирована на экспортные рынки Европы и Америки.
