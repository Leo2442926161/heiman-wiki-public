# HS1SA-Z Интеллектуальная дымовая сигнализация (Z-Wave)

HS1SA-Z — это интеллектуальная дымовая сигнализация **Z-Wave**, выпущенная компанией Hyman. Он использует фотоэлектрическую технологию обнаружения дыма, совместим со стандартом Z-Wave Plus и подходит для европейского и американского рынков умного дома.

## Обзор продукта

- **Тип продукта**: Умная дымовая сигнализация
- **Протокол связи**: Z-Wave Plus
- **Стандарт сертификации**: EN 14604.
- **Типичный сценарий**: предупреждение о пожаре в домах, квартирах и других местах.

## Технические параметры

| Параметр | Значение |
|------|-----|
| Тип датчика | Инфракрасный фотоэлектрический |
| Протокол связи | Z-Wave Plus (ITU-T G.9959) |
| Диапазон рабочих частот | 868,42 МГц (ЕС) / 908,42 МГц (США) |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | Литиевая батарея CR123A |
| Срок службы батареи | Около 2-3 лет |
| Метод шифрования | S2 аутентифицирован / S2 не аутентифицирован |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤90% относительной влажности (без конденсации) |

## Особенности

- Фотоэлектрический детектор дыма для обнаружения медленного горения и дыма от открытого огня.
- Звуковая и световая сигнализация громкостью 85 дБ.
- Напоминание о низком заряде батареи
- Функция самотестирования
- Сигнализация против несанкционированного доступа
- Безопасное шифрование Z-Wave S2
- Поддержка обновления прошивки OTA

## Сертификация

- Сертификация EN 14604
- Сертифицирован Z-Wave Plus
- Сертификация безопасности S2
