#Список поддерживаемых моделей

Продукты Hyman Z-Wave в основном ориентированы на рынки умного дома Европы и Америки и совместимы со стандартом Z-Wave Plus.

## Категория охранной сигнализации

| Модель | Название продукта | Описание |
|------|----------|------|
| HS1SA-Z | Умная дымовая сигнализация (Z-Wave) | Версия Z-Wave дымовая сигнализация |
| HS1CA-Z | Интеллектуальная сигнализация угарного газа (Z-Wave) | Версия Z-Wave сигнализация CO |
| HS1CG-Z | Детектор горючих газов (Z-Wave) | Детектор газа версии Z-Wave |

## Класс датчика

| Модель | Название продукта | Описание |
|------|----------|------|
| HS1DS-Z | Дверной датчик (Z-Wave) | Датчик двери и окна версии Z-Wave |
| ХС1МС-З | Инфракрасный датчик человеческого тела (Z-Wave) | Z-Wave версия зондирования человеческого тела |
| HS1WL-Z | Детектор погружения в воду (Z-Wave) | Версия Z-Wave для обнаружения утечек воды |

## Совместимость

| Проект | Описание |
|------|------|
| Стандарт | Z-Wave Plus (чипы серии 500/700) |
| Совместимый концентратор | Homey, Hubitat, SmartThings, Vera, Fibaro, Домашний помощник (Z-Wave JS) |
| С2 Безопасность | Поддержка протокола безопасности S2 (без шифрования/без аутентификации S2/с аутентификацией S2) |
| СмартСтарт | Поддержка (модели чипов серии 700) |
| ОТА-обновление | Поддержка обновления прошивки CC |
