# HS1CA-Z Интеллектуальная сигнализация окиси углерода (Z-Wave)

HS1CA-Z — это интеллектуальный сигнализатор угарного газа **Z-Wave**, выпущенный компанией Hyman. Он использует технологию электрохимического датчика и совместим со стандартом Z-Wave Plus.

## Обзор продукта

- **Тип продукта**: Интеллектуальный сигнализатор угарного газа
- **Протокол связи**: Z-Wave Plus
- **Стандарт сертификации**: EN 50291.
- **Типичный сценарий**: предупреждение об утечке CO в домах, квартирах и других местах.

## Технические параметры

| Параметр | Значение |
|------|-----|
| Тип датчика | Электрохимический датчик CO |
| Протокол связи | Z-Wave Plus (ITU-T G.9959) |
| Диапазон рабочих частот | 868,42 МГц (ЕС) / 908,42 МГц (США) |
| Диапазон обнаружения | 0~1000 частей на миллион |
| Звуковое давление сигнализации | ≥85 дБ (3 метра) |
| Метод питания | Питание от аккумулятора |
| Срок службы датчика | Типичные 7 лет |
| Метод шифрования | S2 аутентифицирован / S2 не аутентифицирован |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤90% относительной влажности (без конденсации) |

## Особенности

- Высокоточный электрохимический датчик CO
- Сигнализация классификации концентрации
- Звуковая и световая сигнализация громкостью 85 дБ.
- Напоминание о низком заряде батареи
- Напоминание об истечении срока службы датчика
- Функция самотестирования
- Безопасное шифрование Z-Wave S2
- Поддержка обновления прошивки OTA

## Сертификация

- Сертификат EN 50291
- Сертифицирован Z-Wave Plus
- Сертификация безопасности S2
