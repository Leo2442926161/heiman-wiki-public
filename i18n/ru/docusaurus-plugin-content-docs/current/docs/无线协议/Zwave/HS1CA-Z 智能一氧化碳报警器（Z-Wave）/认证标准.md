# Стандарты сертификации

HS1CA-Z прошел следующие сертификации:

- **EN 50291** — Европейский стандарт для сигналов тревоги угарного газа, см. [Европейский стандарт EN](../../../../Стандарты сертификации/Европейский стандарт EN.md)
- **Сертификация Z-Wave S2** — поддерживает структуру безопасности S2, см. [Другие сертификаты](../../../Certification Standards/Other Certifications.md)
- **CE** — Декларация соответствия Европейского Союза.
