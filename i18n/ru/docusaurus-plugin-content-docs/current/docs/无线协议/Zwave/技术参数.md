# Технические параметры

Ниже приведены общие технические характеристики устройств Hyman Z-Wave.

## Параметры беспроводной связи

| Параметр | Значение |
|------|-----|
| Стандарты протокола | Z-Wave Plus (ITU-T G.9959) |
| Рабочий диапазон частот | 800~900 МГц (в зависимости от региона: европейское регулирование 868,42 МГц, законодательство США 908,42 МГц) |
| Скорость передачи | До 100 кбит/с |
| Топология сети | Mesh самовосстанавливающаяся сеть (маршрутизация от источника) |
| Мощность узла | Максимум 232 узла в одной сети |
| Расстояние связи | Около 30 метров в помещении (между узлами), покрытие всего дома сеткой |
| Метод шифрования | AES-128 (с аутентификацией S2/без аутентификации S2) |

## Региональный диапазон частот

| Регион | Центральная частота |
|------|----------|
| Европа | 868,42 МГц |
| Северная Америка | 908,42 МГц |
| Другие области | В соответствии с местным законодательством |

## Электроэнергия и энергопотребление

| Параметр | Значение |
|------|-----|
| Метод питания | Питание от аккумулятора (CR123A/AA) или от сети |
| Срок службы батареи | Около 2-3 лет на датчики |
| Передача мощности | ≤ 1 мВт (соответствует нормам SRD) |
| Получить чувствительность | Примерно -100 дБм |

## Параметры среды

| Параметр | Значение |
|------|-----|
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤ 90 % относительной влажности (без конденсации) |

## Характеристики сети

| Особенности | Описание |
|------|------|
| СмартСтарт | Поддержка (серия 700): отсканируйте QR-код, чтобы присоединиться к сети |
| С2 Безопасность | Поддерживает три уровня безопасности: без шифрования, без аутентификации S2, с аутентификацией S2 |
| Содержит CC | Security CC, Security 2 CC, обновление встроенного ПО, базовая CC, батарея CC, уведомление CC и многое другое |
