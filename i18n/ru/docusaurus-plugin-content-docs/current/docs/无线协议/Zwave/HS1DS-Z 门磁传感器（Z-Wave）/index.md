# Дверной датчик HS1DS-Z (Z-Wave)

HS1DS-Z — это дверной датчик **Z-Wave**, выпущенный компанией Hyman. Он определяет состояние открытия и закрытия дверей и окон с помощью магнитных герконов и совместим со стандартом Z-Wave Plus.

## Обзор продукта

- **Тип продукта**: Дверной датчик
- **Протокол связи**: Z-Wave Plus
- **Метод обнаружения**: Геркон (геркон)
- **Типичные сценарии**: мониторинг состояния дверей и окон, развертывание системы безопасности и интеллектуальная связь.

## Технические параметры

| Параметр | Значение |
|------|-----|
| Метод обнаружения | Геркон (геркон) |
| Протокол связи | Z-Wave Plus (ITU-T G.9959) |
| Диапазон рабочих частот | 868,42 МГц (ЕС) / 908,42 МГц (США) |
| Метод питания | CR1632 батарейка-таблетка |
| Срок службы батареи | Около 2-3 лет |
| Разрешение на установку | ≤10 мм |
| Метод шифрования | S2 аутентифицирован / S2 не аутентифицирован |
| Рабочая температура | -10°С ~ +50°С |
| Рабочая влажность | ≤90% относительной влажности (без конденсации) |

## Особенности

- Обнаружение состояния открытия/закрытия дверей и окон в режиме реального времени
- Тамперная сигнализация (Тампер)
- Напоминание о низком заряде батареи
-Поддержка привязки сцен (открытие двери, включение света, постановка на охрану при выходе из дома)
- Простота установки (клейкая лента 3М или крепление винтами)
- Безопасное шифрование Z-Wave S2
- Поддержка обновления прошивки OTA
