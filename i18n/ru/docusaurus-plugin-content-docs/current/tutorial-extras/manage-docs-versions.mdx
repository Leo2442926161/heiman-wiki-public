# Управление версиями документов

Вики Heiman Technology поддерживает многоплатформенное управление версиями документации, обеспечивая доступ к технической документации, соответствующей версии прошивки вашего продукта.

## Сравнение версий

| Версия документа | Совместимая прошивка | Дата выпуска |
|----------------|---------------------|-------------|
| 2.0            | Серия Matter        | 2025-06     |
| 1.0            | Серия Zigbee        | 2024-01     |

## Просмотр разных версий

Используйте выпадающее меню версий в верхней панели навигации для переключения между различными версиями документов.

## Обновления версий

Мы постоянно обновляем документацию в соответствии с итерациями продуктов. Пожалуйста, регулярно проверяйте наличие последней версии.
