# Продукция

Heiman Technology предлагает четыре основные линейки продуктов: пожарная безопасность, газовая безопасность, уход за пожилыми и интеллектуальные датчики.

## Пожарная безопасность

Включает дымовые извещатели, извещатели угарного газа, тепловые детекторы и т.д., поддерживает подключение к сотовой сети (4G/NB-IoT) для лёгкой интеграции со сторонними IoT-платформами.

Основные возможности:

- Полная сетевая совместимость, мультиплатформенная интеграция
- Соответствие стандартам EN54-7, UL 217 и другим международным нормам
- Поддержка протокола Matter, совместимость с Alexa, Google Home, Apple HomeKit

## Газовая безопасность

Промышленные и коммерческие интеллектуальные системы газовой сигнализации, обеспечивающие комплексные решения для различных сред.

Основные возможности:

- Высокоточные датчики обнаружения газа
- Удалённый мониторинг и оповещение в реальном времени
- Автоматическая связь с аварийными запорными клапанами

## Уход за пожилыми

Включает дымовые извещатели, газовые сигнализаторы, датчики протечки воды, датчики падения, охранные устройства для дверей и аварийные сигнализации.

Основные возможности:

- Алгоритм обнаружения падений на базе ИИ
- Экстренный вызов одной кнопкой
- Оповещение на несколько устройств для семьи и служб

## Интеллектуальные датчики

Охватывают датчики для всех сценариев, включая инфракрасное обнаружение, магнитные датчики открытия дверей/окон, контроль температуры/влажности и обнаружение качества воздуха.

Основные возможности:

- Сверхнизкое энергопотребление
- Поддержка Zigbee / Matter / WiFi — мультипротокольность
- Совместимость с основными платформами умного дома
