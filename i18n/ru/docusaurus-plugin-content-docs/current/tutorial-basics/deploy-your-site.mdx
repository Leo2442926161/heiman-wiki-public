# Развертывание решений

Heiman Technology предоставляет сквозные рекомендации по развертыванию интеллектуальных решений безопасности.

## Процесс интеграции

Продукты Heiman поддерживают несколько методов развертывания, включая локальное автономное развертывание и интеграцию с облачной платформой.

### Локальное развертывание

```bash
# Настройка шлюза Heiman Smart Fire
heiman-cli configure --gateway HS1GW --mode local

# Регистрация устройства-детектора
heiman-cli register --device HS1SA --gateway HS1GW
```

### Интеграция с облачной платформой

Продукты Heiman интегрированы с основными IoT-платформами:

- China Telecom Tianyi
- China Mobile OneNET
- Huawei HiLink
- Alibaba Cloud IoT
- Tuya Smart

## Поддержка протокола Matter

Продукты Heiman серии Matter совместимы с:

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

Обеспечивая межбрендовую и межэкосистемную совместимость.

## Проверка развертывания

После развертывания используйте приложение Heiman Smart Fire APP или платформу управления для функциональной проверки и мониторинга в реальном времени.
