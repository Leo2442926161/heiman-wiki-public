# Возможности Markdown

Техническая документация Heiman полностью поддерживает синтаксис Markdown и расширенные возможности, что упрощает создание структурированной документации.

## Пример кода

Пример интеграции API для разработки продукта:

```jsx title="Интеграция API дымового извещателя"
const heimanClient = new HeimanClient({
  apiKey: 'your-api-key',
  deviceId: 'HS1SA-xxxxx',
});

// Получение статуса устройства
const status = await heimanClient.getDeviceStatus();
console.log(status);
```

## Адмонишены

Используйте адмонишены для выделения важной информации:

:::tip

Дымовой извещатель Heiman HS1SA сертифицирован по стандарту EN14604

:::

:::caution

При установке газовых сигнализаторов следуйте руководству по установке для правильного подключения

:::

:::danger

В случае утечки газа не пользуйтесь электроприборами. Немедленно откройте окна и покиньте помещение

:::
