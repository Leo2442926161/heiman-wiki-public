# EN 유럽 표준

EN(유럽 표준, 유럽 표준)은 유럽 표준화 위원회(CEN)와 유럽 전기 기술 표준화 위원회(CENELEC)가 개발한 기술 표준입니다. Heiman 제품과 관련된 주요 EN 표준은 다음과 같습니다.

## EN 14604 — 연기 경보기 표준

| 프로젝트 | 설명 |
|------|------|
| 적용범위 | 독립 화재 경보기 |
| 탐지 요구사항 | 광전식 연기 감지, 소리 및 빛 경보, 배터리 부족 알림 |
| 해당 시장 | EU 회원국, 영국, 스위스, 노르웨이 등 |
| 하이만 제품 | HS1SA-E, HS3SA, HS1SA-M, S1 시리즈 등 |
| 인증의 의의 | EU 건물 화재 규정의 요구 사항을 충족하는 것은 유럽 시장 진출을 위한 필수 조건입니다 |

**테스트 항목**: 응답 시간, 음압 수준(≥85dB/3m), 음소거 기능, 배터리 수명, 온도 주기, 습도 내구성, 무선 주파수 간섭에 대한 내성 등

## EN 50291 — 일산화탄소 경보 표준

| 프로젝트 | 설명 |
|------|------|
| 적용범위 | 가정용 일산화탄소(CO) 경보기 |
| 테스트 요구사항 | 전기화학적 CO 센서, 농도 분류 경보 |
| 해당 시장 | EU 회원국, 영국 |
| 하이만 제품 | HS1CA-E, C1 시리즈, HS1CA-M 등 |
| 인증의 의의 | CO 누출 감지의 신뢰성을 보장하기 위해 일부 국가에서는 설치가 필수입니다 |

## 기타 관련 EN 표준

- **EN 62368-1**: 오디오 및 비디오/정보 기술 장비의 안전
- **EN 301 489**: 전자기 호환성(EMC)
- **EN 300 220**: 단거리 장치(SRD) 무선 스펙트럼
- **EN 62479**: 저주파 전자기장에 대한 인체 노출 평가
