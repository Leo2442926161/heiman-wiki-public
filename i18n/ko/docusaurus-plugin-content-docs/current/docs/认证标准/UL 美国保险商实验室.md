# UL 보험업자 연구소

UL(Underwriters Laboratories)은 미국에서 가장 권위 있는 제품 안전 표준 개발 및 인증 기관입니다. Heiman 제품과 관련된 주요 UL 표준은 다음과 같습니다.

## UL 217 — 화재 경보기 표준(미국)

| 프로젝트 | 설명 |
|------|------|
| 적용범위 | 주거용 화재 경보기(미국 시장) |
| 탐지 요구사항 | 광전/이온화 연기 감지, 소리 및 빛 경보, 85dB 이상 |
| 해당 시장 | 미국, 캐나다 |
| 하이만 제품 | HS8SA(미국판) |
| 인증의 의의 | 미국 내 여러 주의 화재 규정을 준수하는 것은 북미 시장 진출을 위한 필수 조건입니다 |

**다른 표준과의 차이점**: UL 217의 테스트 시나리오에는 돌발 화재 및 느린 연소 화재와 같은 일반적인 미국 가정 화재 시나리오가 포함되며 EN 14604의 테스트 방법과 약간 다릅니다.

## UL 2034 — 일산화탄소 경보 표준(미국)

| 프로젝트 | 설명 |
|------|------|
| 적용범위 | 가정용 CO 경보기(미국 시장) |
| 탐지 요구사항 | 전기화학 센서, 농도×시간 적분 알고리즘 |
| 해당 시장 | 미국, 캐나다 |
| 헤이먼 제품 | HS-720ES-M(미국판) |
| 인증의 의의 | 미국 대부분의 주에서는 CO 경보기 설치를 의무화하고 있습니다.

## UL 인증과 EN 인증의 주요 차이점

| 측면 | UL | KO |
|------|-----|------|
| 연기 경보 표준 | UL 217 | EN 14604 |
| CO 경보 표준 | UL 2034 | EN 50291 |
| 탐지 방법 | 특정 화재 곡선 테스트 | 다양한 화재 유형 테스트 |
| 적용전압 | 120V / 배터리 | 230V / 배터리 |
