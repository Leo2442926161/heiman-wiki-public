# GB 중국 국가 표준

GB(Guóbiao, 국가 표준)는 중화인민공화국 국가 표준의 약어이며 국무원 표준화 행정 부서에서 제정합니다.

## Haiman 제품에 관련된 주요 GB 표준

| 표준수 | 표준명 | 적용제품 |
|------------|------------|---------|
| GB 20517 | 독립적인 연기 감지 화재 감지 경보 | 연기 경보 |
| GB 15322.1 | 가연성 가스 감지기 1부: 산업용 및 상업용 | 산업용 가스 감지기 |
| GB 15322.2 | 가연성 가스 감지기 2부: 가정용 | 가정용 가스 감지기 |
| GB/T 31141 | 가정용 일산화탄소 경보기 | 일산화탄소 경보 |
| GB 4706.1 | 가정용 및 유사 전기제품의 안전 | 스마트 소켓 |

## CCC와 GB의 관계

CCC 인증(중국 강제 인증)은 GB 표준을 기술 기반으로 합니다. CCC 인증을 통과한 제품은 관련 GB 표준을 준수함을 입증합니다.
