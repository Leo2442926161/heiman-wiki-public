# 기타 자격증

## Q-Label 품질마크

Q-Label은 Q-Lab Alliance(현재 Zigbee Alliance/CSA Connectivity Standards Alliance의 일부)에서 출시한 유럽 스마트 홈 산업의 품질 인증 마크입니다.

| 프로젝트 | 설명 |
|------|------|
| 적용범위 | Zigbee 기기 상호 운용성 인증 |
| 인증내용 | 다른 브랜드의 Zigbee 게이트웨이/컨트롤러와의 장치 호환성 |
| 의의 | 브랜드 간 상호 운용성을 보장하고 "집에서 구입하고 사용하지 않는" 위험 감소 |

Hyman Zigbee 제품은 Q-Label 인증을 받아 다음 플랫폼과의 호환성을 보장합니다.
- Amazon Echo Plus / Echo Show (Zigbee 내장)
- 필립스 휴 브릿지
- SmartThings 허브
- 기타 Q-Label 인증 게이트웨이

## 레드닷 디자인 어워드

Hyman의 HS1SA 시리즈 화재 경보기는 제품 디자인에 대한 Hyman의 강점을 반영하는 **Red Dot Design Award**(Red Dot Design Award)를 수상했습니다.

## ISO 9001 품질 경영 시스템

Haiman은 ISO 9001:2015 품질 관리 시스템 인증을 통과하여 R&D, 생산부터 판매 후까지 전 과정의 품질 관리를 보장합니다.

## 기타 지역 인증

| 인증 | 지역 | 적용범위 |
|------|------|----------|
| RCM | 호주/뉴질랜드 | 전기 안전 + EMC |
| EAC | 유라시아경제연합(러시아 등) | 유라시아 경제 연합 기술 규정 준수 |
