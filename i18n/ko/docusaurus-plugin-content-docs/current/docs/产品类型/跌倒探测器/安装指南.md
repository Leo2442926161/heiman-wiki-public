# 설치 가이드

## 착용방법

- **벨트 클립**: 벨트 또는 허리밴드에 클립
- **끈**: 목에 걸고 가슴에 걸 수 있음

## 사용 지침

1. 배터리 장착
2. 착용 방법 선택(벨트 클립 또는 랜야드)
3. 몸에 착용하세요
4. 게이트웨이를 통한 페어링
5. 시뮬레이션 테스트:
   - 일어서서 빠르게 쪼그려 앉거나 바닥에 눕는다.
   - 센서는 넘어짐을 감지하고 몇 초 내에 경보를 울려야 합니다.
   - 수동 경보를 테스트하려면 조난 버튼을 누르십시오.

## 메모

- 하루종일 착용 권장 (샤워시 탈부착하여 배터리 충전/교체 가능)
- 인공지능 알고리즘을 사용해 낙상과 일상생활(앉기, 굽히기)을 구분해 오경보율이 낮습니다.
- 넘어짐 감지 후 자동 알람이 울리거나 수동으로 버튼을 눌러 도움을 요청할 수 있습니다.
- 비상버튼과 연동하여 이중 보호가 가능합니다.
