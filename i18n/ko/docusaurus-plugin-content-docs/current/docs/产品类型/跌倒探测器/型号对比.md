# 모델 비교

## 다양한 모델 비교

| 모델 | 프로토콜 | 센서 | 전원공급장치 | 특징 | 게임 제안 |
|------|------|---------|------|------|----------|
| [HS2FD](../../무선 프로토콜/Zigbee/HS2FD - 넘어짐 감지기(Zigbee)) | 지그비 | 가속도계 + 자이로스코프 | 배터리 | AI 알고리즘 넘어짐 감지 | 독거 노인 및 요양원에 적합한 웨어러블 디자인, 자동 넘어짐 감지 + 원클릭 도움말 |

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 독거노인 모니터링 | HS2FD | 넘어짐 감지 → 어린이 휴대폰으로 자동 푸시 → 타임아웃 후 무응답 시 120 자동호출 |
| 노인요양기관 | HS2FD | 중앙 집중식 관리 플랫폼 + 바닥 소리 및 조명 경보 + 간호용 휴대용 장치 동기 경보 |
| 수술 후 재활 | HS2FD | 환자가 침대에서 일어나면 낙상의 위험이 감지됨 → 간호실에 조기 경보 |
| 욕실/밤 | HS2FD | IP 보호 설계, 목욕 중에도 착용 가능 + 밤에 일어나거나 넘어지면 자동 알람 |

## 선택 제안

- **홈케어**: [HS2FD](../../Wireless Protocol/Zigbee/HS2FD - 넘어짐 감지기(Zigbee)) (웨어러블, AI 감지)
- **비상 버튼과 페어링**: HS2FD 자동 넘어짐 감지 + 수동 도움을 위한 비상 버튼, 이중 보호
