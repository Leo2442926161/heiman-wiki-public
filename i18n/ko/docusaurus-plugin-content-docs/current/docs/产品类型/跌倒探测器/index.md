# 넘어짐 감지기

넘어짐 감지기는 노인을 위해 특별히 설계되었습니다. 추락 사고를 자동으로 감지하고 적시에 조난 신호를 보낼 수 있습니다. 스마트 노인 돌봄 솔루션의 핵심 장비입니다.

## 제품 시리즈

| 시리즈 | 의사소통 방법 | 특징 |
|------|----------|------|
| HS2FD | 지그비 | 지능형 넘어짐 감지, 자동 경보 |

## 핵심기술

**다중 센서 융합 알고리즘**을 채택하고 가속도계와 자이로스코프 데이터를 결합하며 AI 알고리즘을 사용하여 낙상 자세를 결정하고 정상적인 활동(예: 앉기, 몸 굽히기)과 낙상을 효과적으로 구별하고 잘못된 경보 비율을 줄입니다.

## 핵심 기능

- 자동 넘어짐 감지
- 한 번의 클릭으로 수동으로 도움을 요청하세요
- 착용 가능한 디자인(벨트 클립 또는 랜야드)
- 배터리 부족 알림

## 적용 가능한 시나리오

- 노인홈케어 : 독거노인을 24시간 돌봄
- 요양원: 중앙 집중식 보호 및 관리
- 재활센터: 수술 후 환자의 낙상 위험 모니터링

## 연계계획

- 넘어짐 감지 → 가족의 휴대폰으로 자동 알림
- 비상버튼 연동 → 이중구조 보호
- 하이만 스마트 노인돌봄 플랫폼 연계 → 기관 중앙관리
