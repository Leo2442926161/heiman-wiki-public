# 화재 온도 센서

화재 온도 센서(열 감지기)는 주변 온도의 비정상적인 상승을 감지하여 화재 경보를 발령합니다. 연기 센서로 인해 오경보가 발생하기 쉬운 주방, 차고 등의 현장에 적합합니다.

## 작동 원리

온도 센서는 열 요소를 사용하여 온도 변화를 감지합니다.

- **고정 온도 경보**: 온도가 설정된 임계값(보통 57°C / 135°F)을 초과하면 경보를 트리거합니다.
- **온도차 경보**: 온도 상승률이 설정값(예: 8.3°C/min)을 초과하면 경보가 발생합니다.

## 적용범위

- 주방(이 장면에서는 연기 감지기가 요리로 인해 오경보가 발생하기 쉽습니다)
- 차고/작업장(배기가스와 먼지가 연기 센서를 방해할 수 있음)
- 보일러실/난방설비실
- 욕실/사우나

## 핵심 기능

- 비정상적인 온도 상승 감지
- 변조 방지 경보
- 배터리 부족 알림

> **참고**: 온도 센서와 연기 센서는 상호보완적으로 사용됩니다. 연기 센서는 더 빠르게 반응하고 온도 센서는 더 강력한 간섭 방지 기능을 갖추고 있습니다.
