# 설치 가이드

## 설치 위치 요구 사항

- 방 중앙에 위치한 **천장 마운트**
- **최상의 사용 시나리오**: 주방, 차고, 보일러실, 욕실 - 연기 센서가 잘못된 경보를 받기 쉬운 장소
- 벽에서 최소 **30cm**
- 통풍구 및 에어컨 배출구 근처에 설치하지 마십시오.

## 설치 단계

1. 설치 위치를 선택하세요. (주방이나 차고 천장 추천)
2. 마운팅 베이스를 천장에 고정하세요.
3. 배터리를 삽입하면 센서가 자동으로 켜집니다.
4. 본체를 회전시켜 베이스에 끼워 넣습니다.
5. 자가 테스트: 테스트 버튼을 누르면 표시등이 켜지고 소리가 납니다.
6. 게이트웨이를 통한 페어링(Zigbee 버전)

## 메모

- 온도 센서**는 연기 센서**를 대체하는 것이 아니라 보완하는 것입니다.
- 연기 센서는 더 빠르게 반응하고 온도 센서는 더 강력한 간섭 방지 기능을 갖추고 있습니다.
- 같은 방에 연기 및 온도 센서를 모두 설치하는 것이 좋습니다.
- 한 달에 한 번 테스트하고 배터리가 부족할 때 배터리를 교체하십시오.
