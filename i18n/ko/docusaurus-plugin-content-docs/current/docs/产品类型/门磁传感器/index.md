# 도어 센서

도어 센서는 문과 창문의 개폐 상태를 감지하는 데 사용되며 지능형 보안 시스템의 기본 구성 요소입니다.

## 제품 시리즈

| 시리즈 | 의사소통 방법 | 특징 |
|------|----------|------|
| HS1DS 시리즈 | 지그비/Z-웨이브 | 표준형, 배터리 수명 3년 |
| HS1DS-M | 스레드에 대한 문제 | 문제 버전 |

## 핵심 기능

- 문 및 창문 개폐 상태 실시간 감지
- 탬퍼 알람(Tamper)
- 배터리 부족 알림
- 연동 지원(문열기, 전등켜기, 외출시 무장 등)
- 설치가 용이함(3M 접착테이프 또는 나사고정)

## 기술적인 매개변수

- 전원공급방식 : CR1632 버튼전지
- 배터리 수명 : 약 2~3년
- 작동 온도: -10°C ~ +50°C
- 설치 간격: ≤10mm

## 시나리오 예시

- **집에서 팔을 떼세요**: 문 센서가 문이 열리는 것을 감지하고 경보를 울립니다.
- **집에 가는 장면**: 문을 열고 자동으로 조명이 켜집니다.
- **창문 모니터링**: 기상 관측소와 연동되어 비가 오면 자동으로 창문을 닫습니다.
