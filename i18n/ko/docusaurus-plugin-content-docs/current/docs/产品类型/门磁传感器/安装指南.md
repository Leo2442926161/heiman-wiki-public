# 설치 가이드

## 설치 위치 요구 사항

- 본체(송신기)는 **문/창틀**에 장착됩니다.
- **문/새시**에 자석 장착
- 본체와 자석 사이의 간격 **≤ 10 mm**
- 두 개를 정렬하고 마킹 라인을 정렬합니다.

## 설치 단계

1. 설치 표면을 청소하여 건조하고 먼지가 없는지 확인하십시오.
2. 3M 접착 보호필름을 떼어내고 본체를 문틀에 붙입니다.
3. 자석을 해당 위치에 붙입니다. (문을 닫을 때 본체와 정렬됨)
4. 배터리(CR1632)를 삽입하면 표시등이 깜박여 전원이 켜졌음을 나타냅니다.
5. 문과 창문 개폐 테스트: 문이 열리면 경보가 울리고 닫힌 후 복원됩니다.
6. 게이트웨이를 통한 페어링

## 메모

- 설치 간격이 너무 크면 감지 실패의 원인이 됩니다.
- 금속 문과 창문은 무선 신호에 영향을 줄 수 있습니다. 문제를 해결하기 전에 테스트해 보는 것이 좋습니다.
- 배터리 수명은 약 2~3년입니다. 배터리가 부족하면 앱에서 알려줍니다.
