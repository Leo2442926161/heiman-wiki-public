# 모델 비교

## 다양한 모델 비교

| 모델 | 프로토콜 | 전원공급장치 | 배터리 수명 | 특징 | 게임 제안 |
|------|------|------|----------|------|---------|
| [HS1DS](../../무선 프로토콜/Zigbee/HS1DS - 도어 자기 센서(Zigbee)) | 지그비/Z-웨이브 | CR1632 | 2~3년 | 표준형, 이중 프로토콜 옵션 | 문 열림 시 조명 켜기, 외출 시 팔걸이, 문 및 창문 불법 열림 푸시 알림 + 소리와 빛 알람 연동 |
| [HS1DS-M](../../무선 프로토콜/물질/HS1DS-M - 스마트 도어 자기 센서(실보다 물질)) | 스레드에 대한 문제 | CR1632 | 2~3년 | Matter 버전, 크로스 플랫폼 | Apple 홈에 대한 기본 액세스, 외출 모드에서 무장하기 전에 문과 창문을 자동으로 확인 |

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 집에 도착하면 자동으로 조명을 켜세요 | HS1DS | 문 센서 감지로 문 열기 → 거실 조명 켜기 → 에어컨 켜기 → 환영 인사 |
| 집에서 멀리 무장 | HS1DS | 문, 창문 비정상 열림 감지 → 휴대폰 푸시 + 소리 및 빛 알람 연동 |
| 창 방수 연결 | HS1DS | 창문이 열리고 레인 센서가 작동됨 → 창문을 닫으라는 TTS 방송 알림 |
| Apple 홈 사용자 | HS1DS-M | Native Matter, HomeKit 자동화 "마지막 사람이 집을 떠난 후 확인" |
| 보관실/서랍 모니터링 | HS1DS | 소형 설치, 캐비닛 도어/서랍의 불법 개방 감지, 사무실/호텔에 적합 |

## 선택 제안

- **기존 Zigbee 게이트웨이**: [HS1DS](../../Wireless Protocol/Zigbee/HS1DS - Door Magnetic Sensor (Zigbee)) (Zigbee 버전)
- **교차 플랫폼 호환성 필요**: [HS1DS-M](../../Wireless Protocol/Matter/HS1DS-M - 스마트 도어 자기 센서(스레드보다 물질))(스레드보다 물질)
- **Z-Wave 시스템**: [HS1DS](../../무선 프로토콜/Zigbee/HS1DS - 도어 자기 센서(Zigbee)) (Z-Wave 버전)
