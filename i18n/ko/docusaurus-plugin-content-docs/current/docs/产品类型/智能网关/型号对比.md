# 모델 비교

## 다양한 모델 비교

| 모델 | Zigbee 하위 장치의 최대 제한 | 네트워킹 방식 | 특징 | 게임 제안 |
|------|-------|----------|------|----------|
| [HS6GW](../../무선 프로토콜/Zigbee/HS6GW - 지능형 게이트웨이(Zigbee)) | 128 | WiFi + 이더넷 | 최신 모델 및 향후 Thread 업그레이드 지원 | Haiman 자체 생태 코어와 Haiman APP 원격 관리가 결합되어 신경 쓰지 않는 사용자에게 적합 |
| HS5GW | 64 | WiFi + 이더넷 | 고성능 | 중대형 프로젝트, 안정적이고 신뢰할 수 있음 |
| HS3GW | 32 | WiFi | 향상된 | 중소형 주택에 적합 |
| HS1GW | 16 | WiFi | 기본 | 소수의 장치에 적합한 보급형 |

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 설명 |
|------|----------|------|
| 집 전체 하이만 생태 | HS6GW | Heiman Zigbee 센서 및 Heiman APP를 통한 원격 관리와 함께 사용하면 자체 서버를 구축할 필요가 없습니다 |
| 타사 플랫폼 사용자 | — | 더 높은 유연성을 위해 Zigbee2MQTT/ZHA와 함께 사용하려면 Zigbee 코디네이터(예: SLZB-06)를 직접 구입하는 것이 좋습니다 |
| 중소형 아파트 | HS3GW | 기본 요구 사항을 충족하기 위한 32개의 장치 제한 |

## 선택 제안

- **Haiman 자체 생태**: [HS6GW](../../Wireless Protocol/Zigbee/HS6GW - 스마트 게이트웨이(Zigbee)) (128개 장치, WiFi + 이더넷)
- **타사 플랫폼**: 권장되는 범용 Zigbee 코디네이터 + Zigbee2MQTT/ZHA
