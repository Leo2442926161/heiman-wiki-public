# 스마트 게이트웨이

스마트 게이트웨이는 하이만 스마트홈 시스템의 핵심 허브로, 다양한 지그비 센서를 연결하고 기기 간 연동 제어를 구현하는 역할을 담당합니다.

## 제품 시리즈

| 시리즈 | 의사소통 방법 | 특징 |
|------|----------|------|
| HS1GW 시리즈 | 지그비 + Wi-Fi/이더넷 | 기본 게이트웨이 |
| HS3GW 시리즈 | 지그비 + Wi-Fi/이더넷 | 향상된 |
| HS5GW 시리즈 | 지그비 + Wi-Fi/이더넷 | 고성능 게이트웨이 |
| HS6GW 시리즈 | 지그비 + Wi-Fi / 이더넷 + 스레드 | 최신 모델은 Zigbee와 Matter over Thread를 모두 지원합니다 |

## 핵심 기능

- 지그비 네트워크 코디네이터
- 인터넷 연결을 위한 Wi-Fi/이더넷 지원
- 자동 장치 검색 및 페어링
- 로컬 장면 연결(일부 모델)
- 원격 접속(Heiman Cloud Platform을 통해)
- 최대 128개의 하위 장치 지원(모델에 따라 다름)

## 적용 가능한 시나리오

- 하이만 집 전체 인텔리전스의 핵심 연결점
- 원격 모니터링을 실현하기 위해 Haiman APP와 협력
- 중소 상업 프로젝트의 지역 중앙 집중식 관리

## 다른 게이트웨이와의 차이점

Haiman Gateway는 공식 생태계로 들어가는 입구입니다. 타사 플랫폼을 사용하려는 사용자의 경우 Zigbee2MQTT/ZHA와 함께 범용 Zigbee 코디네이터를 사용하는 것이 좋습니다.
