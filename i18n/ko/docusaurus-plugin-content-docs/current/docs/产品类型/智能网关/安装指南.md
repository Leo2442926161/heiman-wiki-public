# 설치 가이드

## 설치 위치 요구 사항

- **집 전체의 중앙 위치**는 모든 장치에 대한 Zigbee 신호 범위를 보장합니다.
- 전체가 금속으로 된 캐비닛에 넣지 마십시오. (금속은 무선 신호를 차단합니다.)
- **이더넷 우선**: 보다 안정적인 유선 네트워크 연결을 사용하는 것이 좋습니다.
- 다른 Zigbee 장치로부터 20~30m 이내(실내)

## 설치 단계

1. 설치 위치를 선택하세요. (거실 TV 캐비닛, 약전류 박스 옆 등)
2. 전원 어댑터를 연결하세요
3. 이더넷 케이블을 연결하거나(권장) Wi-Fi에 연결합니다.
4. 표시등이 녹색으로 켜질 때까지 기다립니다.
5. Haiman 앱을 열고 게이트웨이 하단의 QR 코드를 스캔하여 추가하세요.
6. 추가가 완료되면 하위 장치를 페어링할 수 있습니다.

## 메모

- 게이트웨이는 스마트홈의 핵심이다. UPS 백업 전원 공급 장치를 사용하는 것이 좋습니다.
- 이더넷 연결은 Wi-Fi보다 안정적이므로 먼저 사용하는 것이 좋습니다.
- 기기 신호가 약한 경우 지그비 중계 장치(예: 스마트 소켓)를 추가하세요.
- 모델별로 지원되는 최대 하위 장치 개수가 다르므로 규모에 따라 선택해야 합니다.
