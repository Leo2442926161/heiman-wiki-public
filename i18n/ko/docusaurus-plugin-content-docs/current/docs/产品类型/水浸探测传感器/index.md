# 침수 감지 센서

누수 센서는 누수 또는 물 축적을 감지하는 데 사용되며, 이는 수도관 파열 및 물 범람으로 인한 재산 손실을 효과적으로 방지할 수 있습니다.

## 제품 시리즈

| 시리즈 | 의사소통 방법 | 특징 |
|------|----------|------|
| HS1WL | 지그비 | 기본 모델, 프로브 감지 |
| HS2WL 시리즈 | 지그비 | 프로브 + 케이블 감지, 듀얼 채널 |
| HS3WL 시리즈 | 지그비 | 향상된 버전 |
| L1 시리즈 | 물질/지그비 | Thread 버전 문제, WWHA 인증 |

## 핵심 기능

- 누수/고인수 실시간 감지
- 부저 알람
- 배터리 부족 알림
- 프로브 부식 방지 설계

## 설치 위치

- 세면대 밑
- 주방 싱크대 아래
- 세탁기/식기세척기 옆
- 온수기 아래
- 지하실/차고
- 라디에이터 근처

## 장면 연결

- **누수 감지 + 솔레노이드 밸브** : 누수 감지 시 자동으로 메인 밸브를 닫습니다.
- **누수 경보 + 푸시**: 집에 없을 때 숙소나 가족에게 휴대폰으로 알립니다.
- **홍수 + 스마트 차단**: 가스 온수기 또는 스마트 수도 밸브 연결
