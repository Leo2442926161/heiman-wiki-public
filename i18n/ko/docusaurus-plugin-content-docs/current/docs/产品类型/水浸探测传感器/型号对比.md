# 모델 비교

## 다양한 모델 비교

| 모델 | 프로토콜 | 탐지 방법 | 전원공급장치 | 특징 | 게임 제안 |
|------|------|----------|------|------|---------|
| [HS1WL](../../무선 프로토콜/Zigbee/HS1WL - 침수 감지기(Zigbee)) | 지그비 | 프로브 | 배터리 | 기본 모델 | 물이 새는 경우 알람 푸시, 욕실 세면대 아래에 적합 |
| [HS2WL](../../무선 프로토콜/Zigbee/HS2WL - 침수 감지기(Zigbee)) | 지그비 | 프로브 + 케이블 | 배터리 | 듀얼 채널 | 프로브 감지 지점 + 케이블이 긴 영역을 커버하며 온수기/세탁기 옆에 가장 적합 |
| [HS3WL](../../무선 프로토콜/Zigbee/HS3WL - 누수 감지기(Zigbee)) | 지그비 | 프로브 | 배터리 | 향상된 버전 | 더 높은 감도, 더 큰 부저 |
| [L1-M](../../무선 프로토콜/물질/L1-M 침수 감지기(물질)) | 스레드에 대한 문제 | 프로브 | 배터리 | 문제 버전 | WWHA 인증, Apple Home에 대한 기본 액세스 |

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 집 전체 누수 모니터링 | HS2WL + L1-M | 다중 포인트 적용 + 기본 HomeKit 알림, 누수 감지 → 주 수도 밸브 끄기 → 푸시 알림 |
| 세탁기 특별 | HS2WL | 케이블 프로브는 세탁기 바닥에 놓입니다. 물이 새면 즉시 알람이 울리고 세탁기 전원이 꺼집니다 |
| 온수기/바닥난방 | HS3WL | 향상된 버전은 높은 감도를 가지며 초기 단계에서 바닥 난방 배관의 누수를 감지할 수 있습니다 |
| 중요한 사용자 | L1-M | 전용 게이트웨이가 필요하지 않으며 스레드 저지연 알람 연결 Apple 홈 오토메이션 |

## 선택 제안

- **점 감지**: [HS1WL](../../무선 프로토콜/Zigbee/HS1WL - 누수 감지기(Zigbee)) 또는 [L1-M](../../무선 프로토콜/Matter/L1-M 누수 감지기(Matter))
- **넓은 지역 적용**: [HS2WL](../../무선 프로토콜/Zigbee/HS2WL - 누수 감지기(Zigbee)) (프로브 + 케이블)
- **물질 요구사항**: [L1-M](../../무선 프로토콜/물질/L1-M 침수 감지기(물질))
