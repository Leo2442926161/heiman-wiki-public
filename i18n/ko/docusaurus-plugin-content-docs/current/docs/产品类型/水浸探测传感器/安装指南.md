# 설치 가이드

## 설치 위치 요구 사항

- **지면에 놓고** 프로브가 지면에 닿는지 확인하세요.
- **수원 근처**: 욕실 세면대 아래, 주방 싱크대 아래, 세탁기 옆, 온수기 아래
- 누수를 감지해야 하는 가장 낮은 지점

## 설치 단계

1. 설치위치 결정(누수위험지역의 지반)
2. 바닥면을 깨끗이 청소한다
3. 프로브가 지면에 닿은 상태로 센서를 지면에 놓습니다.
4. 케이블 버전(HS2WL)을 사용하는 경우 모니터링할 영역을 따라 연장 케이블을 배치합니다.
5. 배터리를 삽입하면 센서가 자동으로 켜집니다.
6. 시뮬레이션 테스트: 젖은 손가락으로 프로브를 터치하면 알람 표시등이 켜집니다.
7. 게이트웨이를 통한 페어링

## 메모

- 프로브가 습한 환경에 장기간 노출되면 부식될 수 있으므로 정기적으로 점검하십시오.
- 자동 차단을 달성하려면 솔레노이드 밸브와 협력하는 것이 좋습니다.
- 배터리 수명은 약 2~3년입니다.
- 다양한 영역을 포괄하는 다중 센서
