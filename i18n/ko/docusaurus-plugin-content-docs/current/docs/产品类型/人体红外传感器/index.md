# 인체 적외선 센서

인체 적외선 센서(PIR)는 인체에서 방출되는 적외선 열 복사를 감지하여 움직임을 감지하며 스마트 홈에서 가장 일반적으로 사용되는 존재 감지 솔루션입니다.

## 제품 시리즈

| 시리즈 | 의사소통 방법 | 특징 |
|------|----------|------|
| HS1MS | 지그비 | 기본 모델, 110° 감지 각도 |
| HS3MS | 지그비 | 업그레이드 모델, 360° 천장 설치 |
| M1 시리즈 | 물질/지그비 | Matter over Thread 지원(110° 또는 360° 옵션) |
| HS8MLS | 지그비 | 장거리 모델 |
| HS8MIS | 지그비 | 인체 + 조명 복합 센서 |

## 핵심 기능

- 모바일 인간 감지
- 조도 감지(M1/HS8MIS 시리즈)
- 감지거리 7m 이하 (25°C)
- 설치 방법 : 벽걸이형 또는 천장형
- 배터리 수명 : 3년(Zigbee) / 1년(Matter)

## 기술적인 매개변수 (M1 시리즈)

- 감지 각도: 110° / 360°
- 감지 거리: 7m 이하
- 조도 범위: 0~1200 Lux
- 전원 공급 장치: DC 3V(리튬 배터리)
- 배터리 수명 : 3년(Zigbee) / 1년(Matter)

## 시나리오 예시

- 누군가 오면 조명이 켜지고 누군가 나가면 꺼집니다.
- 보안연동 : 비정상적인 움직임 감지시 푸시 알람
- 외출 시 무장: 주변에 사람이 없을 때 자동으로 에어컨/난방을 차단하여 에너지를 절약합니다.
