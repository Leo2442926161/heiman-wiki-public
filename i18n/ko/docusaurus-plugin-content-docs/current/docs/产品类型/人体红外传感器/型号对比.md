# 모델 비교

## 다양한 모델 비교

| 모델 | 프로토콜 | 감지 각도 | 설치 | 특징 | 게임 제안 |
|------|------|----------|------|------|---------|
| [HS1MS](../../무선프로토콜/지그비/HS1MS - 인체적외선센서(지그비)) | 지그비 | 110° | 벽걸이형 | 경제적이고 실용적인 기본 모델 | 사람이 오면 불이 켜지고, 사람이 나가면 불이 꺼지고, 보안 연동 감지 모바일 푸시 알람 |
| [HS3MS](../../무선프로토콜/지그비/HS3MS - 인체적외선센서(지그비)) | 지그비 | 360° | 천장 | 전체 범위 | 사각지대 없이 천장 전체를 커버하며 거실/로비 공간에 적합 |
]
| [M1-M](../../무선 프로토콜/물질/M1-M 인체 적외선 센서(물질)) | 스레드에 대한 문제 | 110° | 벽걸이 | 매터 에디션 | Native HA + Apple Home, 조명 연동 커튼 자동 조정 |
| [HS8MLS](../../무선 프로토콜/지그비/HS8MLS - 마이크로 모션 센서(지그비)) | 지그비 | 110° | 벽걸이형 | 장거리 | 복도/차고의 긴 직선 영역, 더 긴 감지 거리 |
| [HS8MIS](../../무선 프로토콜/지그비/HS8MIS - 모션 센서(지그비)) | 지그비 | 110° | 벽걸이 | 복합 감지 | PIR + 조명 투인원, 연결된 스마트 조명은 빛에 따라 자동으로 밝기를 조정합니다 |

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 사람들이 오면 조명이 켜지고 에너지가 절약됩니다 | M1-Z / HS8MIS | PIR은 누군가를 감지 → 조명을 켜고 + 조명을 읽어 밝기를 결정 → 주변에 아무도 없을 때 조명 끄기 |
| 보안 무장 연계 | HS1MS | 외출 모드에서 움직임 감지 → 푸시 알람 + 음향 및 조명 사이렌 + 카메라 녹화 |
| 전체 천장 범위 | HS3MS | 실내 중앙 천장, 360° 막다른 곳 없음, 열린 공간에 적합 |
| 물질 교차 생태학 | M1-M | Apple 홈 오토메이션 + Google/Alexa 음성, 조명 연결 커튼 |
| 긴 복도/차고 | HS8MLS | 장거리 감지, 차량/사람 출입 자동조명 |

## 선택 제안

- **복도/방 벽걸이**: [HS1MS](../../무선 프로토콜/Zigbee/HS1MS - 인체 적외선 센서(Zigbee)) 또는 [HS8MLS](../../무선 프로토콜/Zigbee/HS8MLS - 마이크로 모션 센서(Zigbee))(장거리)
- **천장 전체 커버리지**: [HS3MS](../../무선 프로토콜/Zigbee/HS3MS - 인체 적외선 센서(Zigbee)) (360°)
- **조명 연동 필요**: [M1 시리즈](../../무선 프로토콜/Zigbee/M1-Z - 인체 적외선 센서(Zigbee)) 또는 [HS8MIS](../../무선 프로토콜/Zigbee/HS8MIS - 움직임 센서(Zigbee))
- **물질 플랫폼**: [M1-M](../../무선 프로토콜/물질/M1-M 인체 적외선 센서(물질))
