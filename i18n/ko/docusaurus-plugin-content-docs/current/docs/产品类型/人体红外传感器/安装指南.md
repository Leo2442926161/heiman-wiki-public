# 설치 가이드

## 설치 위치 요구 사항

- 벽 장착 높이: **1.8 ~ 2.4m**
- 천장 설치 : 방 중앙의 천장에 설치
- **피해야 할 것**: 직사광선, 난방/에어컨 통풍구, 열원 근처
- 감지 범위 내에 장애물(가구, 유리 칸막이 등)을 피하세요.

## 설치 단계

1. 설치 위치를 선택하세요(벽걸이형 또는 천장형)
2. 나사로 장착 베이스를 고정합니다.
3. 배터리를 삽입하면 표시등이 깜박입니다.
4. 센서 본체를 베이스에 설치합니다.
5. 감지 각도 조정(수평 범위 110° 이내)
6. 보행 테스트: 감지 영역을 통과한 후 표시등이 켜져야 합니다.
7. 게이트웨이를 통한 페어링

## 메모

- 적외선 센서는 온도 변화에 민감하므로 여름철 고온 환경에서는 감지 거리가 짧아질 수 있습니다.
- 창문을 향하지 마세요(실외 온도 변화로 인해 잘못된 경보가 발생함).
- 배터리 수명 약 3년(Zigbee) / 1년(Matter)
