# 소리와 빛 알람(경고 신호)

스마트 사이렌은 큰 소리와 가벼운 경보를 제공하는 데 사용됩니다. 다양한 센서와 함께 사용되며 보안 시스템의 출력 액츄에이터로 사용할 수 있습니다.

## 핵심 기능

- 고데시벨 경보음(≥110dB)
- 빨간색/컬러 스트로브 조명
-다중 알람 톤 지원
- 변조 감지
-원격 무장 및 무장 해제 지원

## 적용 가능한 시나리오

- 연기/CO 센서와 연동: 위험 감지 시 집안 전체에 소리가 울림
- 보안 침입 경보: 문과 창문이 불법적으로 열렸을 때 작동
- 긴급 대피 : 화재 발생 시 대피 안내

## 장면 연결

- **연기 경보 + 소리 및 빛 사이렌**: 집 전체가 동시에 경보음을 울려 방에 있는 모든 사람에게 알립니다.
- **도어 센서 알람 + 소리 및 빛 알람**: 불법 침입자 방지
- **게이트웨이와 협력**: Zigbee/Matter 네트워크를 통해 다른 장치와 연결
