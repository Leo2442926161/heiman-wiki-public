# 설치 가이드

## 설치 위치 요구 사항

- **집 중앙에 설치** 모든 방에서 들을 수 있도록 합니다.
- 복도, 거실 천장 또는 높은 벽
- 지상에서 **2m 이상** (사람의 손에 닿지 않도록 주의)
- 연기/CO 센서와 함께 사용할 경우 침실 복도에 설치하세요.

## 설치 단계

1. 설치 위치를 선택하세요. (복도나 거실 권장)
2. 나사를 사용하여 장착 베이스를 벽이나 천장에 고정합니다.
3. 전원 공급 장치(주 전원 버전)를 연결하거나 배터리(배터리 버전)를 삽입합니다.
4. 베이스에 알람 본체를 설치합니다.
5. 게이트웨이를 통한 페어링
6. 연결 테스트: 모든 센서를 트리거하면 소리와 빛 경보가 응답해야 합니다.

## 메모

- 어린이의 손이 닿지 않는 높이에 설치하세요.
- 알람 볼륨과 조명이 정상인지 정기적으로 테스트하세요.
- 배터리 버전의 경우, 배터리가 부족할 때 교체하시기 바랍니다.
- 다른 센서와의 연계 규칙 설정 가능
