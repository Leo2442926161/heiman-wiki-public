# 모델 비교

## 다양한 모델 비교

| 모델 | 프로토콜 | 알람 음압 | 전원공급장치 | 특징 | 게임 제안 |
|------|------|----------|------|------|---------|
| 소리와 빛 알람(사이렌) | 지그비 | ≥110dB | 주전원 | 고휘도 스트로브 + 높은 데시벨 | 연기/CO/도어 자석 트리거 연결 집 전체의 소리 및 빛 경보, 집을 떠날 때 침입자 방지 |

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 집 전체 화재 연계 | 소리와 빛 경보 | 연기/일산화탄소 센서 알람 → 집 전체에 소리와 빛 알람이 울려 자고 있는 가족을 깨울 수 있습니다 |
| 보안침입억제 | 소리와 빛 경보 | 문열림 센서 불법개방 + 퇴실모드 → 사이렌 소리 + 침입자 저지용 전등 |
| 긴급 대피 안내 | 소리와 빛 경보 | 화재경보 발동 → 사이렌이 피난방향을 안내하고 도어센서와 연동하여 피난여부를 감지 |
