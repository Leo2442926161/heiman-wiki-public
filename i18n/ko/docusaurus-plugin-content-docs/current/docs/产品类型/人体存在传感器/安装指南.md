# 설치 가이드

## 설치 위치 요구 사항

- **천장 설치**, 센서 수직 하향
- 설치 높이 **2.5 ~ 3.5m**
- 감지 반경은 약 **3~5m**입니다.
- **피하십시오**: 센서 앞에 금속 장애물이 있습니다(금속은 밀리미터파를 차단합니다).
- 대형 금속배관, 에어컨 내부기기 등으로부터 멀리 두십시오.

## 설치 단계

1. 천장 설치 위치를 선택하세요. (방의 중앙이 가장 좋습니다.)
2. 천장에 구멍을 뚫고 베이스를 확장나사로 고정합니다.
3. 배선을 연결하거나(주 전원 공급 장치가 필요한 경우) 배터리를 설치합니다.
4. 센서 본체를 회전시켜 베이스에 끼웁니다.
5. 전원을 켜면 표시등이 켜지고 센서 자체 테스트가 시작됩니다.
6. 앱 페어링 진입 및 기기 추가
7. 테스트: 감지 영역에 들어가서 조용히 앉으십시오. 센서는 "점유" 상태를 유지해야 합니다.

## 메모

- 밀리미터파는 얇은 판자(석고보드, 목재)를 투과할 수 있으나 금속에 의해 차단됩니다.
- 두꺼운 벽과 금속 천장으로 인해 감지 범위가 줄어듭니다.
- 전체 장면 범위를 달성하려면 PIR 센서와 함께 사용하는 것이 좋습니다.
