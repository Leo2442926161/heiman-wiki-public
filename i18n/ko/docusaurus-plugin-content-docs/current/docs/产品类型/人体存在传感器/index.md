# 인간 존재 센서

Occupancy/Presence 센서는 **밀리미터파 레이더 기술**을 사용하여 고정된 인체의 존재를 감지하고 기존 PIR 적외선 센서를 보완합니다.

## 제품 시리즈

| 시리즈 | 의사소통 방법 | 특징 |
|------|----------|------|
| HS8OS | 지그비 | 밀리미터파 레이더 인간 존재 감지 |

## 기술 원칙

움직임만 감지할 수 있는 PIR 센서와 달리 존재감 센서는 마이크로 도플러 효과를 사용하여 밀리미터파 신호를 송수신하여 사람이 있는지 확인함으로써 작은 신체 움직임(예: 호흡으로 인한 가슴 오르락내리락)을 감지합니다.

## 핵심 기능

- **정지된 인체 감지**: 누워있거나 가만히 앉아 있는 등 미세한 움직임을 감지합니다.
- **범위 감지**: 적용 범위 반경 3-5미터
- **천장 설치**: 최적의 감지 각도
- **PIR 보완**: PIR은 대규모 움직임 감지를 담당하고 존재 센서는 정밀한 감지를 담당합니다.

## 시나리오 예시

- **스마트 조명**: 누군가 주변에 있으면 조명이 항상 켜져 있고 주변에 사람이 없으면 자동으로 꺼집니다(PIR 정적 조명 꺼짐 문제 해결).
- **에어컨 제어**: 방에 사람이 있는지 여부에 따라 자동으로 켜지거나 꺼집니다.
- **스마트 노인 케어**: 노인이 욕실/침실에 오랫동안 움직이지 않으면 시스템이 조기 경고를 발령합니다.
- **사무실 에너지 절약**: 회의실/워크스테이션이 비어 있을 때 자동으로 에너지를 조정합니다.
