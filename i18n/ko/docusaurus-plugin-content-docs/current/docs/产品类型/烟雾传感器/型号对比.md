## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 스마트 홈의 긴밀한 통합 | HS1SA-E / HS1SA-M | HA 자동화: 연기 경보기 → 집 전체의 조명 켜기 → 도어 잠금 장치 잠금 해제 → 가스 밸브 끄기 → 휴대폰으로 푸시 |
| Apple HomeKit 사용자 | HS1SA-M | Native Matter 액세스, HomePod를 스레드 경계 라우터로 사용, 자동 장면 트리거 |
| 유럽 ​​및 미국 표준 요구 사항 | HS1SA-E(CE) / HS8SA(UL) | 지역 화재 인증, 보험 준수 요구 사항 충족 |
| 게이트웨이가 필요하지 않음/저렴한 비용 | S1 시리즈 | 구매 후 바로 사용 가능, 여러 장치가 Sub-1G를 통해 상호 연결되어 알람 발생 시 서로 트리거 |
| Z-Wave 기존 시스템 | HS1SA-Z | Hubitat/SmartThings |와 같은 기존 Z-Wave 게이트웨이와 함께 작동합니다.

## 선택 제안

- **EU 시장**: [HS1SA-E](../../Wireless Protocol/Zigbee/HS1SA-E - Smart Smoke Alarm (Zigbee)) (Zigbee) 또는 [HS3SA](../../Wireless Protocol/Zigbee/HS3SA - Smart Smoke Alarm (Zigbee)) (Zigbee) Z-Wave 선택이 필요한 경우 [HS1SA-Z](../../Wireless 프로토콜/Zwave/HS1SA-Z 지능형 연기 경보기(Z-Wave))
- **미국 시장**: [HS8SA](../../Wireless Protocol/Zigbee/HS8SA - 스마트 연기 경보기(Zigbee)) (UL 217 인증)
- **크로스 플랫폼 문제**: [HS1SA-M](../../Wireless Protocol/Matter/HS1SA-M 스마트 연기 경보기(문제))(스레드보다 문제)
- **독립형 장면**: [S1 시리즈](../../무선 프로토콜/Sub1G(장거리 범위)/S1 시리즈 - 독립형 화재 경보기(Sub-1G)) (게이트웨이 필요 없음)
