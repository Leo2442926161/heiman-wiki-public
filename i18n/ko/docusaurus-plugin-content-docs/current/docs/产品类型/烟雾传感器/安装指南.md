# 설치 가이드

## 설치 위치 요구 사항

- **천장 장착형**, 방 중앙에 위치
- 벽이나 조명 기구로부터 최소 **30cm**
- **공기가 없는 지역을 피하세요**(모서리, 들보 뒤 등)
- **설치를 피하세요**: 욕실(증기), 주방(그을음), 통풍구 근처

## 설치 단계

1. 설치 위치를 선택하고 천장이 평평한지 확인하세요.
2. 장착 베이스를 천장 방향으로 회전시킵니다(나사 또는 3M 접착제 사용).
3. 배터리를 삽입하면(극성에 주의하세요) 센서가 자동으로 켜집니다.
4. 센서 본체를 회전시켜 베이스에 끼웁니다.
5. 테스트 버튼을 길게 누릅니다. 알람 표시등이 깜박이고 경고음이 울리면서 정상임을 나타냅니다.
6. 게이트웨이를 통한 페어링: 페어링 모드로 들어가서 앱에 장치를 추가합니다.

## 메모

- 한 달에 한 번 기능 테스트
- 배터리 수명은 약 3년입니다. 배터리가 부족할 때 제때에 교체해 주세요.
- 10년마다 센서 전체 교체를 권장합니다.
- 설치 높이는 연기 감지에 영향을 미치지 않으며, 연기가 천장까지 올라갑니다.
