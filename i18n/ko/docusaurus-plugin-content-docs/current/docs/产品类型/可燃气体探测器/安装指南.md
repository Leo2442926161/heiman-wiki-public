# 설치 가이드

## 설치 위치 요구 사항

- **천연가스(메탄)** : 공기보다 가벼우며 천장에서 **30cm** 이내에 설치
- **액화석유가스(프로판)** : 공기보다 무거워 **지상에서 30cm 이내에 설치**
- 가스렌지, 온수기, 가스배관 연결부 등 누수 가능성이 있는 곳과 가까운 곳

## 설치 단계

1. 감지할 가스의 종류(천연가스 또는 액화가스)를 확인하세요.
2. 해당 설치 높이를 선택하십시오.
3. 장착 베이스를 나사로 벽에 고정하세요.
4. 전원 공급 장치를 켭니다(대부분의 모델은 주 전원으로 전원이 공급됩니다).
5. 센서를 켜고 약 3~5분 동안 예열하세요.
6. 자가 테스트: 테스트 버튼을 누르면 소리와 빛 경보가 정상이 됩니다.
7. 게이트웨이를 통한 페어링(Zigbee 버전)
8. 솔레노이드 밸브를 연결해야 할 경우 차단 밸브 제어 라인을 연결하십시오.

## 메모

- 테스트 가스를 사용하여 센서 감도를 정기적으로 확인하십시오.
- 센서 수명은 약 3~5년 정도이며 만료되면 교체해야 합니다.
- 가스 냄새가 날 때는 절대 전기기구를 켜거나 끄거나 화염을 사용하지 마세요.
- 가스 종류에 따라 설치 높이가 다름(상단은 천연가스, 하단은 액화가스)
