# 모델 비교

| 모델 | 프로토콜 | 가스 감지 | 인증 | 전원공급장치 | 특징 | 게임 제안 |
|------|------|----------|------|------|------|---------|
| JT-G1 | 독립 | 천연가스/액화가스 | GB 15322 | 주전원 | 독립적인 소리와 빛 경보 | 게이트웨이 불필요, 부저 내장, 기본 가스 누출 경보 |
| [JT-HS1CG](../../무선 프로토콜/Zigbee/HS1CG - 가연성 가스 감지기(Zigbee)) | 지그비 | 천연가스/액화가스 | GB 15322 | 주전원 | 지그비 네트워킹 | 누출 감지 → 지그비 알림 휴대폰 → 연결된 솔레노이드 밸브 자동 종료 |
| [JT-HS3CG](../../무선 프로토콜/Zigbee/HS3CG - 가연성 가스 감지기(Zigbee)) | 지그비 | 천연가스/액화가스 | GB 15322 | 메인 | 향상된 | 더 높은 감도, 원격 밸브 폐쇄 지원 + 소리 및 빛 경보 |

| [JT 시리즈](../../무선 프로토콜/지그비/JT 시리즈 - 가스 비상 차단 밸브(지그비)) | 지그비 | 천연가스/액화가스 | GB 15322 | 메인 | 일반 시리즈 | 다양한 장면 요구에 맞게 사용할 수 있는 다양한 모델 |

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 주방 안전 자동화 | HS1CG / HS3CG | 누수 감지 → 솔레노이드 밸브 자동 폐쇄 → 환풍기 켜기 → 휴대폰으로 푸시 알림 → 소리와 빛 연동 알람 |
| 임대 주택/아파트 | JT-G1 | 독립 경보는 네트워킹이 필요하지 않으며 비용이 저렴하며 화재 검사 요구 사항을 충족합니다 |
| 원격 모니터링 요구 사항 | HS8CG | 이중 통신 보장 + 클라우드 플랫폼 원격 보기 상태 |
| 상업용 주방 | HS3CG | 고감도 + 연기 배출 시스템 연계 + 부동산 플랫폼 중앙 관리 |

## 선택 제안

- **독립형 사용**: JT-G1
- **Zigbee 네트워킹**: [HS1CG](../../무선 프로토콜/Zigbee/HS1CG - 가연성 가스 감지기(Zigbee)) 또는 [HS3CG](../../무선 프로토콜/Zigbee/HS3CG - 가연성 가스 감지기(Zigbee))
- **이중 통신 보장**: JT-HS8CG
