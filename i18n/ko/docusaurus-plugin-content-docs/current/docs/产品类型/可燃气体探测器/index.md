#가연성가스감지기

가연성 가스 감지기는 천연가스(메탄), 액화석유가스(프로판) 등 가연성 가스의 누출을 감지하는 데 사용됩니다. 주방 안전을 위한 핵심 장비입니다.

## 제품 시리즈

| 시리즈 | 통신방식 | 가스 감지 | 인증기준 |
|------|----------|----------|----------|
| JT-G1 | 독립 | 천연가스/액화가스 | GB 15322 |
| JT-HS1CG 시리즈 | 지그비 | 천연가스/액화가스 | GB 15322 |
| JT-HS3CG | 지그비 | 천연가스/액화가스 | GB 15322 |
| JT-HS8CG | 지그비/와이파이 | 천연가스/액화가스 | GB 15322 |

## 핵심 기능

- 안정성이 높은 반도체 가스 센서
- 누출농도 분류 경보
- 자동 차단 연동(솔레노이드 밸브 포함)
- 소리 및 빛 경보(≥85dB)
- 셀프 테스트 기능
- 센서 수명 만료 알림

## 장면 연결

- **누출 감지 → 연동 차단 밸브**: 가스 파이프라인 밸브 자동 폐쇄
- **경보 + 배기**: 배기팬/레인지후드 자동으로 켜짐
- **푸시 알림**: 누수 경보 정보를 휴대폰으로 실시간 수신합니다.

## 안전 팁

가스 냄새가 나면 전기기구를 켜거나 끄지 말고 화기를 사용하지 마세요. 즉시 창문을 열어 환기를 시키고, 안전한 야외 공간으로 이동하여 경찰에 신고해야 합니다.
