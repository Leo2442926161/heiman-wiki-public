# 설치 가이드

## 설치 위치 요구 사항

- **1.2 ~ 1.5m** 높이에 설치 (평균 환경 수준을 나타냄)
- **떨어져 두십시오**: 직사광선, 문과 창문, 에어컨 콘센트, 라디에이터
- **떨어져 두십시오**: 가습기, 주방 스토브 및 기타 온도 및 습도 변화의 원인이 되는 지역
- 자주 사용하는 방을 선택하세요(침실, 거실, 아기방)

## 설치 단계

1. 설치 위치를 선택하세요. (벽면 또는 데스크탑)
2. 벽면 장착: 나사 또는 3M 접착제로 베이스를 고정합니다.
3. CR2 배터리를 설치합니다(극성에 주의하세요).
4. 센서가 자동으로 켜지고 화면(있는 경우)에 온도와 습도가 표시됩니다.
5. 게이트웨이를 통한 페어링
6. 앱의 데이터를 확인하여 정상인지 확인하세요.

## 메모

- 벽에 설치할 경우 센서 통풍구가 막히지 않도록 하세요.
- 배터리 수명은 약 2~3년입니다.
- 여러 센서를 통해 서로 다른 공간의 온도 및 습도 데이터를 수집할 수 있습니다.
- 연결된 에어컨, 제습기, 가습기 및 기타 장비에 사용할 수 있습니다.
