# 공기질 센서

공기질 센서(Air Quality Monitor)를 사용하여 실내공기 중의 각종 오염물질을 감지하여 건강한 생활환경을 유지하는데 도움을 줍니다.

## 탐지 매개변수

Hyman 공기질 센서는 다음 지표를 감지합니다.

| 매개변수 | 설명 | 건강 기준 |
|------|------|----------|
| PM2.5 | 미세먼지 | ≤ 35 µg/m³(24시간 평균) |
| PM10 | 호흡성 입자상 물질 | ≤ 50 µg/m³(24시간 평균) |
| CO2 | 이산화탄소 | ≤ 1000ppm |
| TVOC | 총 휘발성 유기화합물 | ≤ 0.6mg/m³ |
| 온도 | 주변 온도 | 18~26°C |
| 습도 | 상대습도 | 40-60% 상대습도 |

## 핵심 기능

- 다중 매개변수 공기질 감지
- 컬러 디스플레이(모델에 따라 다름)
- 기준 초과시 소리 및 빛 경보
- 과거 데이터 기록
- 신선한 공기/공기청정기 연동

## 적용 가능한 시나리오

- 침실: CO2 수준을 모니터링하고 환기를 위해 창문을 열도록 요청
- 어린이방 : 어린이의 호흡기 건강을 보호합니다.
- 사무실: 생산성과 편안함 향상
- 신규 개조 주택: 포름알데히드/TVOC 방출 모니터링
