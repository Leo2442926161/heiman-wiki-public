# 설치 가이드

## 설치 위치 요구 사항

- **1.2~1.8m** 높이(호흡구역)에 설치
- 방의 평균 공기질을 나타내는 위치를 선택하세요.
- **떨어져 두십시오**: 창문, 문, 통풍구, 에어컨 콘센트
- **멀리 떨어져**: 주방 연기, 가습기, 아로마테라피 및 기타 지역 오염원

## 설치 단계

1. 설치 위치를 선택하세요. (거실, 침실, 사무실 벽)
2. 나사로 장착 베이스를 고정합니다.
3. 전원에 연결합니다(USB 또는 어댑터 전원 공급).
4. 센서의 전원이 켜지고 자체 테스트 후 디스플레이(있는 경우)에 데이터가 표시됩니다.
5. 게이트웨이를 통한 페어링
6. 앱에서 데이터가 정상인지 확인하세요.

## 메모

- 센서 공기 흡입구와 배출구가 막히지 않았는지 확인하세요.
- 먼지가 막히지 않도록 센서 하우징을 정기적으로 청소하십시오.
- 아이들 방에 놓아서 CO2 농도를 모니터링하고 환기를 위해 창문을 열라는 메시지를 표시합니다.
- 신선한 공기 시스템이나 공기청정기와의 결합을 권장합니다.
