# 모델 비교

## 다양한 모델 비교

| 모델 | 프로토콜 | 탐지 매개변수 | 전원공급장치 | 특징 | 게임 플레이 제안 |
|------|------|----------|------|------|---------|

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 침실 CO₂ 모니터링 | 공기질 센서 | CO2 > 1000ppm → 창문 열림 알림/환기용 신선한 공기 연계 시스템, 수면의 질 향상 |
| 새로 개조된 집 | 공기질 센서 | TVOC/포름알데히드에 대한 지속적인 모니터링, 기준 초과 시 배기팬 연결, 기준 충족 후 안심 알림 푸시 제공 |
| 어린이방 | 공기질 센서 | PM2.5 모니터링, 호흡기 건강을 보장하기 위해 기준을 초과하면 공기청정기가 자동으로 켜짐 |
| 사무실 | 공기질 센서 | CO2 + 온도 및 습도와 연계된 공조 신선한 공기로 업무 효율성 향상 |
