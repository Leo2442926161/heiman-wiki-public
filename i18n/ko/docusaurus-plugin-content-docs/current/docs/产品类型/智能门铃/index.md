# 스마트 초인종

Haiman Smart Doorbell은 초인종 호출 및 보안 모니터링 기능을 통합하고 원격 비디오 인터콤을 지원합니다.

## 핵심 기능

- 초인종 버튼 트리거
- 원격 라이브 영상(Wi-Fi 버전)
- 양방향 음성통화
- 움직임 감지 영상
- 나이트 비전 기능

## 적용 가능한 시나리오

- 방문자 전화: 집에서나 밖에서 전화를 받을 수 있습니다.
- 택배/테이크아웃 알림 : 문 열림 원격 확인
- 보안 모니터링: 문 앞의 의심스러운 활동을 기록합니다.

> **참고**: Heiman 초인종 제품 라인은 주로 Zigbee/Wi-Fi 솔루션을 기반으로 합니다. 특정 모델 매개변수에 대해서는 공식 담당자에게 문의하십시오.
