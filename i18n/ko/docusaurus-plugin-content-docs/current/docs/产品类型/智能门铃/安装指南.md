# 설치 가이드

## 설치 위치 요구 사항

- 초인종은 약 **1.4~1.6m** 높이(눈높이 기준)에 설치하세요.
- 방문자의 얼굴을 포착할 수 있도록 출입구를 향하세요.
- 카메라에 직사광선을 피하세요.

## 설치 단계

1. 설치 위치를 선택하세요. (문 옆이나 문틀 위)
2. 나사로 장착 베이스를 고정합니다.
3. 전원 공급 장치를 연결합니다(유선 버전 연결, 배터리 버전에 배터리 설치).
4. 초인종 본체를 베이스에 설치하세요.
5. 앱을 통해 Wi-Fi 페어링 및 연결(Wi-Fi 버전)
6. 카메라 각도 조정
7. 초인종 버튼, 비디오 및 음성 기능 테스트

## 메모

- Wi-Fi 버전은 현관에서 Wi-Fi 신호 범위가 양호해야 합니다.
- 배터리 버전의 충전 주기에 주의하시고, 도어 센서(문 열림 후 감지 및 깨우기 위해)와 함께 사용하는 것을 권장합니다.
- 정기적으로 카메라 렌즈를 청소하십시오.
- 초인종이 유효한 Zigbee 신호 범위 내에 있는지 확인하십시오.
