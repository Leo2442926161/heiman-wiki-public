# 설치 가이드

## 설치 위치 요구 사항

- **벽걸이 설치**: 욕실, 침실의 침대 옆, 거실의 소파 옆 - 손이 닿는 곳 어디든 가능
- **어디서나 착용 가능**: 주머니에 넣거나 열쇠고리에 걸어두세요
- **노인용**: 욕실 및 침실 침대 옆에 설치, 높이는 약 1.2m

## 설치 단계

1. 설치 위치나 착용 여부를 결정하세요.
2. 벽면 장착: 나사 또는 3M 접착제를 사용하여 장착 베이스를 고정합니다.
3. 배터리 설치
4. 비상 버튼을 받침대에 놓거나 휴대하십시오.
5. 게이트웨이를 통한 페어링
6. 테스트: 버튼을 길게 눌러 알람을 실행하고 앱이 알림을 수신하는지 확인하세요.

## 메모

- **길게 누르는 사고 방지 터치** 디자인을 채택하여 트리거하려면 1~2초 동안 계속 눌러야 합니다.
- 소리와 빛 알람과의 연동을 권장합니다.
- 정기적으로 기능 테스트
- 노인의 경우 이중 보호를 위해 추락 감지기와 함께 사용할 수 있습니다.
