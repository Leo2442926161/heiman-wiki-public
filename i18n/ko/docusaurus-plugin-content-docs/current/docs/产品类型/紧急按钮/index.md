# 패닉 버튼

비상버튼(긴급버튼/패닉버튼)은 위급상황 발생 시 신속하게 조난 신호를 발생시키는 데 사용되는 스마트 노인 돌봄 및 보안 시스템의 중요한 부분입니다.

## 제품 시리즈

| 시리즈 | 의사소통 방법 | 특징 |
|------|----------|------|
| 비상 버튼 | 지그비 | 휴대용 원버튼 호출 |

## 핵심 기능

- 한 번의 클릭으로 긴급 경보 발동
- 휴대용 디자인(휴대용 또는 벽걸이형 가능)
- 배터리 부족 알림
- 실수 방지 터치 디자인(트리거 길게 누르기)

## 적용 가능한 시나리오

- **스마트 노인 케어**: 노인이 갑자기 아프거나 쓰러졌을 때 클릭 한 번으로 도움을 요청할 수 있습니다.
- **어린이방** : 집에 아이들이 혼자 있을 때 긴급상황 발생 시 사용
- **의료**: 병동/요양원 침대 호출
- **상점/프론트 데스크**: 강도 또는 긴급 상황 발생 시 비밀 경보 발동

## 트리거 연결

- 가족/보호자에게 휴대폰 푸시 알림
- 소리와 빛 알람 연동
- 숙소에 자동으로 전화하거나 120번으로 전화하세요.
