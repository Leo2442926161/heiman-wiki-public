# 모델 비교

## 다양한 모델 비교

| 모델 | 프로토콜 | 트리거 방식 | 전원공급장치 | 특징 | 게임 플레이 제안 |
|------|------|----------|------|------|---------|
| 비상 버튼 | 지그비 | 길게 누르기 트리거 | 배터리 | 휴대용 + 벽걸이 겸용 | 노인들이 휴대할 수 있으며, 긴급 상황 시 원클릭 도움, 휴대폰 푸시 + 소리 및 빛 알람과 연결 |

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 노인 홈케어 | 비상 버튼 | 갑작스런 노약자 불편 → 길게 누르기 → 가족 전체의 휴대폰으로 푸시 → 소리와 빛 연동 알람 → 재산알림 |
| 어린이방 안전 | 비상 버튼 | 아이가 집에 혼자 긴급상황에 처했을 경우 → 버튼을 누르면 부모의 휴대폰으로 알람이 울립니다 |
| 상점/계산대 | 패닉 버튼 | 은폐설치, 도난시 무음경보, 감시영상 연동 + 보안에 조용히 알림 |
| 화장실/욕실 | 비상 버튼 | 노인이 샤워 중에 미끄러지거나 넘어지는 경우 방수 비상 버튼이 있어 쉽게 도움을 받을 수 있습니다 |
