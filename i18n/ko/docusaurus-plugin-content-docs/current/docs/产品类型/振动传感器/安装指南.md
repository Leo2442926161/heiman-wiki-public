# 설치 가이드

## 설치 위치 요구 사항

- 진동 감시가 필요한 물체의 표면에 설치
- 일반적인 설치 위치: **문과 창문 내부**, **금고**, **파이프**, **벽**, **울타리**
- 설치면이 평평하고 깨끗한지 확인하세요.

## 설치 단계

1. 모니터링할 객체를 선택하세요. (예: 금고문 내부)
2. 장착면을 청소하십시오.
3. 3M 접착 테이프나 나사를 사용하여 센서를 고정합니다.
4. CR2450 배터리를 설치하세요
5. 게이트웨이를 통한 페어링
6. 시뮬레이션 테스트: 설치 표면을 탭하면 센서가 알람을 트리거해야 합니다.

## 메모

- 문과 창문이 들림 방지되는 시나리오의 경우 자물쇠 근처의 문/창 안쪽에 설치하는 것이 더 효과적입니다.
- 파이프라인 모니터링을 위해 파이프라인 표면에 직접 붙입니다.
- 배터리 수명 약 2년
- 다양한 시나리오에 맞게 조정 가능한 감도
