# 모델 비교

## 다양한 모델 비교

| 모델 | 프로토콜 | 센서 | 전원공급장치 | 특징 | 게임 제안 |
|------|------|---------|------|------|----------|
| 진동센서 | 지그비 | 3축 가속도계 | CR2450 | 진동 + 기울기 감지 | 문 및 창문 열림 방지, 금고/서랍 불법 열림 진동 경보, 배관 이상 진동 모니터링 |

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 문과 창문의 비뚤어짐 방지 | 진동센서 | 문/창문의 격렬한 충격 및 진동 감지 → 푸시 알람 + 소리 및 빛 알람 + 카메라 캡처 |
| 안전한 모니터링 | 진동센서 | 금고가 움직이거나 진동하는 경우 → 즉시 경보 알림 |
| 파이프라인 모니터링 | 진동센서 | 수도관 이상진동(워터펌프고장/수격현상)을 모니터링, 유지보수 조기경보 |
| 울타리/벽 | 진동 센서 | 벽/담장 등반 감지, 마당 보안에 적합 |
