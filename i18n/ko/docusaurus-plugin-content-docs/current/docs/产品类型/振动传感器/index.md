# 진동센서

진동 센서는 물체의 진동이나 기울기를 감지하여 경보를 발생시킵니다. 문, 창문, 금고, 파이프 등 다양한 시나리오에 적합합니다.

## 핵심 기능

- 3차원 진동 감지
- 기울기 각도 감지
- 배터리 부족 알림

## 적용 가능한 시나리오

- **문 및 창문 열림 방지**: 폭력적인 침입으로 인한 진동 감지
- **금고/서랍 모니터**: 비정상적인 움직임 감지
- **배관 모니터링**: 송수관 진동 감지(워터펌프 이상 또는 수격 현상 표시)
- **벽/울타리 모니터링**: 올라가거나 충격을 감지합니다.

## 기술적인 매개변수

- 센서 유형: 3축 가속도계
- 감지크기 : 진동 + 기울기
- 전원공급방식 : CR2450 버튼전지
- 배터리 수명: 약. 2년
