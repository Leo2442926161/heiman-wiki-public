# 모델 비교

## 다양한 모델 비교

| 모델 | 프로토콜 | 정격 출력 | 측정 정확도 | 특징 | 게임 제안 |
|------|------|----------|----------|------|---------|
| [HS2SK](../../무선 프로토콜/Zigbee/HS2SK - 스마트 소켓(Zigbee)) | 지그비 | 16A / 3680W | ±2% | 계량 유형 | 실시간 소비전력 모니터링 + 원격스위치, 재택모드 연동 원클릭 전원차단, 에어컨/온수기 소비전력 통계 |
| [HS2SK-WiFi](../../무선 프로토콜/Wifi/HS2SK-WiFi - 스마트 미터 소켓(WiFi)) | WiFi | 16A / 3680W | ±2% | 게이트웨이 없이 직접 연결 | 추가 게이트웨이가 필요하지 않으며 게이트웨이 구매를 원하지 않는 사용자에게 적합한 Smart Life 앱 직접 제어 |

## 추천 장면 게임 플레이

| 시나리오 | 추천 모델 | 연계 솔루션 |
|------|----------|----------|
| 에너지 소비 통계 | HS2SK | Zigbee는 HA에 연결되고 InfluxDB + Grafana와 연결되어 일별/월별 전력 소비량을 표시하고 에너지를 많이 소비하는 기기를 발견합니다 |
| 집을 나갈 때 전원 끄기 | HS2SK | 집을 나갈 때 TV/에어컨/어항조명 등 불필요한 가전제품을 원클릭으로 꺼서 안전하게 에너지 절약 |
| 타이밍 제어 | HS2SK / HS2SK-WiFi | 타이밍 스위치 온수기/가습기/충전 파일, 피크 및 밸리 전기 가격과 결합하여 전기 요금 절감 |
| 게이트웨이가 없는 사용자 | HS2SK-WiFi | 라우터에 직접 연결되고 Smart Life 앱으로 제어되며 임계값 항목이 0입니다.

## 선택 제안

- **기존 Zigbee 게이트웨이**: [HS2SK](../../WirelessProtocol/Zigbee/HS2SK - 스마트 소켓(Zigbee))
- **게이트웨이 구매를 원하지 않음**: [HS2SK-WiFi](../../Wireless Protocol/Wifi/HS2SK-WiFi - 스마트 미터링 소켓(WiFi))
