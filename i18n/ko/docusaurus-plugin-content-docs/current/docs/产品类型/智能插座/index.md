# 스마트 소켓

Haiman 스마트 미터링 플러그는 원격 스위치 제어 및 실시간 전력 소비 측정을 지원하여 사용자가 가전 제품을 지능적으로 관리할 수 있도록 도와줍니다.

## 제품 시리즈

| 시리즈 | 의사소통 방법 | 특징 |
|------|----------|------|
| HS2SK 시리즈 | 지그비 | 계량 유형, 전력 모니터링 지원 |

## 핵심 기능

- 원격 스위치 제어
- 실시간 전력/가스 계량
- 과부하 보호
- 타이머 스위치
- 전력 통계

## 기술적인 매개변수

- 정격전압 : AC 220V(중국 규격) / AC 110V(미국 규격)
- 최대 부하: 16A/3680W
-측정 정확도: ±2%

## 시나리오 예시

- 잊어버린 가전제품을 원격으로 끄기
- 에어컨/온수기 에너지 소비 통계
- 연결된 퇴거 모드: 한 번의 클릭으로 불필요한 모든 가전제품을 끄십시오.
- 수조 조명, 가습기 등의 타이밍 제어
