# 설치 가이드

## 설치 위치 요구 사항

- 벽면 콘센트에 직접 연결하세요. **직렬로 연결하지 마세요**(스트립이나 연장 코드에 연결하지 마세요)
- 소켓의 정격 전류가 ≥ 16A인지 확인하십시오.
- 습기가 많거나 온도가 높은 환경에는 설치하지 마십시오.

## 설치 단계

1. 스마트 소켓을 220V(또는 110V) 벽면 콘센트에 직접 연결하세요.
2. 전원이 공급되면 표시등이 켜집니다.
3. 스위치 버튼을 길게 눌러 페어링 모드로 들어갑니다.
4. 하이만 앱에서 기기 추가
5. 성공적으로 페어링되면 스위치를 원격으로 제어할 수 있습니다.

## 안전 예방 조치

- **최대 부하**: 16A/3680W(유럽 표준), 15A/1800W(미국 표준)
- **과부하 금지**: 고출력 가전제품(에어컨, 히터) 사용 시 전력 제한에 주의하세요.
- **직렬로 연결하지 마세요**: 스마트 소켓을 멀티탭에 삽입하지 마세요.
- 소켓 하우징을 덮거나 가리지 마십시오.
- 플러그가 과열되지 않았는지 정기적으로 점검하십시오.

## 일반적인 애플리케이션

- 온수기, 가습기, 어항 조명 등 저전력 장비의 타이밍 제어
- 센서 연동에 협조 : 누군가 오면 조명이 켜지고 누군가 나가면 꺼집니다.
- 전력 통계: 가전제품의 에너지 소비량 모니터링
