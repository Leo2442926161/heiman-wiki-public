# 일산화탄소 센서

일산화탄소(CO)는 무색, 무취, 무자극의 독성 가스로 '보이지 않는 살인자'로 알려져 있습니다. Hyman 일산화탄소 센서는 **전기화학 센서 기술**을 사용하여 공기 중 CO 농도를 정확하게 감지합니다.

## 인증 업적

Hyman은 **세계 최초의 물질 인증 일산화탄소 경보기**를 출시하고 Works with Home Assistant의 인증을 받았습니다.

## 제품 시리즈

| 시리즈 | 의사소통 방법 | 인증기준 | 해당 시장 |
|------|----------|----------|----------|
| HS1CA 시리즈 | 지그비/Z-웨이브 | EN 50291 | 유럽연합 |
| C1 시리즈 | 지그비 | EN 50291 | 유럽연합 |
| HS1CA-M | 스레드에 대한 문제 | EN 50291 | 글로벌 |
| HS-720ES-M | 스레드에 대한 문제 | UL 2034 | 미국 |

## 핵심 기능

- 고정밀 전기화학 CO 센서
- 검출범위 : 0~1000ppm
- 농도 등급 경보(소리 및 빛 프롬프트)
- 85dB 알람음
- 배터리 부족 알림
- 센서 수명 만료 알림(일반적으로 7년)
- 셀프 테스트 기능

## 안전 팁

CO 농도와 인체 반응:

| CO 농도(ppm) | 인체 반응 |
|---------------|----------|
| 50 | 건강한 성인은 8시간을 견딜 수 있다 |
| 200 | 2-3시간 후 가벼운 두통 |
| 400 | 1~2시간 후 두통과 메스꺼움 |
| 800 | 45분 이내에 현기증과 경련이 발생함 |
| 1600+ | 2시간 이내에 치명적 |
