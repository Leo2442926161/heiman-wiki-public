# 설치 가이드

## 설치 위치 요구 사항

- **호흡 높이에 설치** - 약. **지상에서 1.5m**
- 자는 동안 알람을 들을 수 있도록 침실 공간과 가깝습니다.
- **모퉁이에 설치하지 마십시오**, 공기 순환을 유지하십시오.
- 통풍구, 창문, 팬 등 공기 흐름에 영향을 미치는 장소에서 멀리 두십시오.

## 설치 단계

1. 설치 위치 결정 (권장 침실 복도 또는 거실)
2. 장착 베이스를 벽에 고정합니다. (나사 또는 3M 접착 테이프)
3. 배터리를 삽입하면 센서가 자동으로 켜집니다.
4. 베이스에 본체를 설치합니다.
5. 자가 테스트: 테스트 버튼을 누르면 알람 표시등이 깜박이고 알람이 울립니다.
6. 게이트웨이를 통한 페어링

## 메모

- 일반적인 센서 수명은 **7년**이며, 그 이후에는 전체 장치를 교체해야 합니다.
- 한 달에 한 번 기능 테스트
- 센서 통풍구를 막지 마십시오.
- 경보가 발생하면 먼저 창문을 열어 환기를 시킨 후, CO 발생원의 누출 여부를 확인하세요.
