# 자동화 예시

## 연기 경보 장면

**장면 이름:** "연기 경보기 연결"

**발동 조건:** Hyman 연기 경보기 → 연기 감지

**작업 실행:**
- 스마트 소켓 → 열림(배기팬 연결)
- 조명 → 켜지고 깜박임
- 휴대폰으로 푸시 알림
- 스피커 → 알람 소리 재생

## 원클릭 집을 나서는 장면(Tap-to-Run)

**장면명:** "퇴거 후 정리"

**수동 실행:** 탭 또는 음성 트리거

**작업 실행:**
- 하이만 도어 센서 → 알람 켜기
- 하이만인체적외선 → 감지 켜기
- 하이만 침수 감지기 → 감지 켜기
- 모든 조명 → 꺼짐
- 스마트락 → 잠금

## 자동 온도 및 습도 조절

**자동화 유형:** 자동 조건 트리거

**트리거 조건:** Hyman 온도 및 습도 센서 습도 > 75%

**작업 실행:**
- 스마트 소켓(제습기) → On
- 푸시 알림: "습도가 너무 높아 제습기가 켜졌습니다."

**트리거 조건:** Hyman 온도 및 습도 센서 습도 < 50%

**작업 실행:**
- 스마트소켓(제습기) → 닫기
