#허비타트

Hubitat은 지역화된 제어에 초점을 맞춘 스마트 홈 허브로 Zigbee 및 Z-Wave 장치를 지원하며 모든 자동화는 클라우드 서비스 없이 로컬에서 수행됩니다.

## 통합 방법

### 지그비 장치

Hyman Zigbee 센서는 Hubitat Elevation 허브와 직접 페어링됩니다.

1. Hubitat에서 Zigbee 페어링 모드로 들어갑니다.
2. 하이만 센서를 페어링 모드로 전환하세요.
3. 장치가 자동으로 검색되어 추가됩니다.

### Z-Wave 장치

Hyman Z-Wave 장치는 Hubitat에 내장된 Z-Wave 칩을 활용하여 Hubitat과의 직접 페어링도 지원합니다.

### 사용자 정의 드라이버

일부 Heiman 장치에는 맞춤형 드라이버가 필요할 수 있으며 커뮤니티 사용자는 일반 센서에 대해 호환 가능한 드라이버를 작성했습니다.

## 장점

- **100% 현지화**: 모든 자동화 로직은 Hubitat에서 로컬로 실행되며 클라우드에 의존하지 않습니다.
- **낮은 대기 시간**: 로컬 Zigbee/Z-Wave 메시 네트워크, 빠른 응답
- **리치 커뮤니티**: Hubitat 커뮤니티에는 수많은 Heiman 사용자의 구성 공유가 있습니다.

> **참고**: [Hubitat 커뮤니티 - Heiman 장비 토론](https://community.hubitat.com/)
