# 자동화 예시

## 규칙 기계 화재 경보기

**규칙명:** 연기 경보기 안전 연동

**구성:**
````
트리거: Hyman 연기 경보가 활성화됩니다.
조건: 모드 != 무장해제
조치:
  - 모든 조명을 켜세요
  - "⚠️ 연기 경보!" 푸시 알림 보내기
  - 사이렌을 켜세요(가능한 경우)
  - 모드를 "긴급"으로 설정하세요.
````

## 규칙 기계 온도 모니터링

**규칙 이름:** 고온 보호

**구성:**
````
트리거: Heimann 온도 센서 > 38°C
조건: 에어컨 상태 = 꺼짐
조치:
  - 에어컨 켜기, 냉방 모드, 24°C
  - "실내 온도가 너무 높아 에어컨이 가동되었습니다" 알림 보내기
````
