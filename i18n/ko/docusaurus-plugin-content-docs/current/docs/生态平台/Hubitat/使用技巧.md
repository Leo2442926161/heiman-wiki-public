# 사용팁

## 커스텀 드라이버

Heimann 장치는 Hubitat에서 다음 드라이버와 호환되도록 최적화되었습니다.

| 장치 유형 | 추천 드라이버 |
|------------|----------|
| 연기/일산화탄소 경보 | 일반 Zigbee 연기/CO 센서 |
| 인체 적외선 센서 | Heiman 모션 센서(타사) |
| 도어 자기 센서 | 일반 Zigbee 접촉 센서 |
| 누수 센서 | 일반 지그비 누수 센서 |
| 온도 및 습도 센서 | Heiman 온도/습도 센서 |

## 지그비 최적화

- Hubitat C-8 이상 모델은 호환성이 가장 좋은 EFR32 칩을 사용합니다.
- 설정 → 지그비에서 간섭이 가장 적은 채널을 확인하고 선택하세요.
- Ubisys와 같은 Zigbee 스니핑 도구를 사용하여 통화 끊김 진단

## 규칙 프롬프트

- Rule Machine에 "Anti-false Alarm Delay" 조건(0.5~2초)을 추가하는 것이 좋습니다.
- 주요 규칙에 대한 "상태 확인" 알림을 활성화하여 계속 실행되도록 합니다.
- 손쉬운 디버깅 및 유지 관리를 위해 "하위 규칙" 모드에서 복잡한 자동화를 구성합니다.

## Z-Wave 공존

Z-Wave 장치를 동시에 사용하려면 Zigbee와 Z-Wave 채널 간의 주파수 간격을 유지하십시오.
