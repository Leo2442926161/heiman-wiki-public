# 대시보드 구성

## 보안 개요 패널

섹션 보기 레이아웃을 사용하여 Heimann 장비를 영역으로 구성합니다.

``yaml
유형: 섹션
제목: 연기 및 일산화탄소 경보기
카드:
  - 유형: 엔터티
    엔터티: Binary_sensor.heiman_smoke_alarm
    이름: 연기 경보기
  - 유형: 엔터티
    엔터티: Binary_sensor.heiman_co_alarm
    이름: CO 경보
````

## 버섯 카드 스타일

``yaml
유형: 관습:버섯-알람-제어판-카드
엔터티: Alarm_control_panel.home_alarm
이름: 보안 시스템

유형: 사용자 정의:버섯 엔터티 카드
엔터티: Binary_sensor.heiman_smoke_alarm
이름: 연기 경보기
아이콘: mdi:연기 감지기
Primary_info: 상태
second_info: 마지막으로 변경됨
fill_container: 사실
````

## 환경 센서 패널

``yaml
유형: 섹션
제목: 환경 모니터링
카드:
  - 유형: 게이지
    엔터티: 센서.하이만_온도
    이름: 온도
    단위: "°C"
    최소: 0
    최대: 50
    심각도:
      녹색: 0
      노란색: 30
      빨간색: 40
  - 유형: 게이지
    엔터티: 센서.하이만_습도
    이름: 습도
    단위: "%"
    최소: 0
    최대: 100
    심각도:
      녹색: 30
      노란색: 60
      빨간색: 80
````

## 알람 이력 패널

``yaml
유형: 기록 그래프
제목: 센서 역사
엔터티:
  -binary_sensor.heiman_smoke_alarm
  -binary_sensor.heiman_co_alarm
  -binary_sensor.heiman_water_leak
시간_to_show: 24
````

## 추천 레이아웃

- 상단 : 보안시스템 상태 + 경보장치 상태카드
- 중앙 : 환경 모니터링(온도, 습도, 수질)
- 하단 : 센서 알람 이력 트렌드 차트
- 사이드바 바로가기 작업: 무장/해제, 장면 전환
