# 사용팁

## 장치 구성

- Heiman Matter 장치를 처음으로 페어링할 때 경계 라우터(예: HomePod, Nest Hub, Home Assistant Green)가 동일한 Thread 네트워크에 있는지 확인하세요.
- ZHA 또는 Zigbee2MQTT를 통해 Zigbee 장치를 통합하는 경우 USB 코디네이터의 권장 펌웨어(예: Sonoff ZBDongle-E, Conbee II 등)를 먼저 사용하는 것이 좋습니다.

## 자동화된 최적화

- 즉각적인 허위 경보를 방지하기 위해 연기 경보 자동화에 'for:' 매개변수(예: 3초)를 추가했습니다.
- 무장 상태에서 불필요한 알림을 피하기 위해 '조건:상태'를 사용하여 보안 모드를 일치시킵니다.
- 홈어시스턴트의 `그룹` 기능을 사용하여 하이만 센서를 영역별로 그룹화하여 일괄 작업이 용이합니다.

## 문제 해결

- 기기가 오프라인일 때 먼저 Zigbee 네트워크 채널이 WiFi 2.4GHz와 충돌하는지 확인하세요.
- Matter 장치를 검색할 수 없는 경우 보더 라우터를 다시 시작하고 장치를 다시 추가하세요.
