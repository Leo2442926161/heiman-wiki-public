# 자동화 예시

## 화재 경보기는 집 전체에 조명을 작동시킵니다.

``yaml
별칭: "연기 경보기 - 집 전체의 조명 및 알림"
트리거:
  - 트리거: 상태
    엔터티_ID: Binary_sensor.heiman_smoke_alarm
    받는 사람: "켜기"
작업:
  - 액션: light.turn_on
    목표:
      엔터티_ID: 모두
  - 액션: inform.mobile_app_phone
    데이터:
      message: "⚠️ 연기 경보가 작동되었습니다! 즉시 확인하세요!"
      제목: "연기 경보기"
````

## CO 알람 연동

``yaml
별칭: "CO 경보 - 가스를 끄고 창문을 열어주세요"
트리거:
  - 트리거: 상태
    엔터티_ID: Binary_sensor.heiman_co_alarm
    받는 사람: "켜기"
작업:
  - 액션: switch.turn_off
    엔터티_ID: switch.gas_fireplace
  - 액션: 커버.오픈
    엔터티_ID:cover.living_room_window
  - 조치: inform.pertant_notification
    데이터:
      메시지: "일산화탄소 경보가 작동되었으며, 가스 밸브가 닫히고 창문이 열렸습니다."
````

## 침수 감지기 연동

``yaml
별칭: "홍수 감지 - 물 밸브 닫기"
트리거:
  - 트리거: 상태
    엔터티_ID: Binary_sensor.heiman_water_leak
    받는 사람: "켜기"
작업:
  - 액션: switch.turn_off
    엔터티_ID: switch.main_water_valve
  - 액션: inform.mobile_app_phone
    데이터:
      메시지: "💧 누수 감지! 주 급수 밸브가 닫혔습니다!"
````

## 집 밖에서 무장 - 인체 센서가 경보를 울립니다.

``yaml
별칭: "집에서 나가기 모드 - 인체 센서 알람"
트리거:
  - 트리거: 상태
    엔터티_ID: Binary_sensor.heiman_motion_sensor
    받는 사람: "켜기"
조건:
  - 조건 : 상태
    엔터티_ID: Alarm_control_panel.home_alarm
    상태: arm_away
작업:
  - 액션: Alarm_control_panel.trigger
    엔터티_ID: Alarm_control_panel.home_alarm
````
