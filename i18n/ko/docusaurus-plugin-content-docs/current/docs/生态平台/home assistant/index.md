#홈어시스턴트

Haiman Technology는 2026년 2월에 **Works with Home Assistant** 인증 프로그램에 공식적으로 참여하여 이 프로그램의 첫 번째 중국 파트너 중 하나가 되었습니다.

## 인증기기

다음 Heiman 장치는 홈어시스턴트와의 현지화 및 안정적인 호환성을 보장하기 위해 공식 인증되었습니다.

| 장비 | 지역 | 연결 방법 |
|------|------|----------|
| 스마트 연기 경보 | 미국 | 스레드에 대한 문제 |
| 스마트 연기 경보 | 유럽연합/중국 | 스레드에 대한 문제 |
| 스마트 일산화탄소 경보기 | 미국 | 스레드에 대한 문제 |
| 스마트 일산화탄소 경보기 | 유럽연합/중국 | 스레드에 대한 문제 |
| 인체 적외선 센서 M1 시리즈 | 글로벌 | 물질/지그비 |
| 침수 감지기 L1 시리즈 | 글로벌 | 물질/지그비 |
| 온도 및 습도 센서 H1 시리즈 | 글로벌 | 물질/지그비 |

## 통합 방법

### 작성자: Matter

1. Matter over Thread 경계 라우터(예: Home Assistant Green + Connect ZBT-2, Apple HomePod, Google Nest Hub 등)가 있는지 확인하세요.
2. 홈어시스턴트에서 Matter 통합 구성
3. 자동으로 검색할 장치를 추가하세요.

### Zigbee2MQTT/ZHA를 통해

Hyman Zigbee 장치는 추가 구성 없이 Zigbee2MQTT 및 ZHA에서 기본적으로 지원됩니다.

## 현지화 제어

인증된 모든 장치는 클라우드 서비스에 의존하지 않고 **현지화된 제어**를 지원합니다. 인터넷이 다운되더라도 센서 알람과 자동화 연동은 정상적으로 작동할 수 있습니다.

## 커뮤니티 참여

Heyman은 홈어시스턴트 커뮤니티의 토론에 적극적으로 참여하고 사용자 피드백을 기반으로 제품 경험을 지속적으로 최적화합니다.
