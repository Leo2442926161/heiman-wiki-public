# 사용팁

## 사물 구성

MQTT 바인딩을 통해 Heiman 장치를 추가합니다.

``자바
Thing mqtt:topic:heiman_smoke "하이만 연기 경보기" @ "거실" {
    채널:
        바이너리 유형: 연기 "연기 상태" [ stateTopic="zigbee2mqtt/heiman_smoke", 변환Pattern="JSONPATH:$.smoke" ]
        유형 번호: 배터리 "배터리 전원" [ stateTopic="zigbee2mqtt/heiman_smoke", 변환Pattern="JSONPATH:$.battery" ]
}
````

## 아이템 정의

``자바
Smoke_Alarm에 문의하세요 "연기 경보기 [MAP(smoke.map):%s]" { 채널="mqtt:topic:heiman_smoke:smoke" }
Number Smoke_Battery "연기 경보기 배터리 [%d%%]" { 채널="mqtt:topic:heiman_smoke:battery" }
````

## 좌표 선택

**CC2531** 또는 **CC2652** 칩이 있는 USB 코디네이터를 사용하고 펌웨어 버전이 최신인지 확인하는 것이 좋습니다.

## 디버깅 팁

- Paper UI에서 Thing의 '온라인/오프라인' 상태 보기
- 'openhab-cli info'를 사용하여 바인딩이 올바르게 로드되었는지 확인하세요.
- 로그에서 '하이만' 키워드를 검색하여 기기 통신 문제를 해결하세요.
