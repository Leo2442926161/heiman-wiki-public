# 자동화 예시

## 연기 경보 규칙

``자바스크립트
규칙 "연기 경보기 작동"
언제
    아이템 heiman_smoke_alarm이 ON으로 변경되었습니다.
그럼
    sendCommand(거실_조명, ON)
    sendNotification("admin", "⚠️ 연기 경보가 작동되었습니다!")
    postUpdate(Alarm_State, "smoke_alert")
끝
````
