# 자동화 예시

## MQTT 구독 장치 상태

``배쉬
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
````

출력 예:
````
zigbee2mqtt/heiman_smoke {"연기":true,"배터리":100,"linkquality":120}
````

``yaml

    payload_off: "오프라인"
