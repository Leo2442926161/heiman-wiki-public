# 사용팁

## 채널 선택

Hyman Zigbee 장치는 기본적으로 채널 15를 사용합니다. `configuration.yaml`에서 수동으로 지정하는 것이 좋습니다.

``yaml
고급:
  채널: 15
````

WiFi 2.4GHz 채널 1/6/11과 중복을 피하세요. WiFi 분석기를 사용하여 간섭이 가장 적은 Zigbee 채널을 스캔하고 선택하세요.

## 네트워크 키 백업

**`coordinator_backup.txt` 및 `secret.yaml`을 반드시 백업하세요. **

``배쉬
cp 데이터/coordinator_backup.txt 데이터/coordinator_backup.bak
cp 데이터/secret.yaml 데이터/secret.yaml.bak
````

키를 분실한 후에는 페어링된 모든 장치를 다시 연결해야 합니다.

## 기기별 최적화

- Heiman 기기는 기본적으로 60분마다 배터리 전원을 보고하며, '보고' 구성을 통해 간격을 단축할 수 있습니다.
- MQTT 검색 구성을 수동으로 작성하지 않고도 자동으로 엔터티를 생성하는 'homeassistant: true' 구성이 추가되었습니다.
- Heiman 장치에 대해 사람이 읽을 수 있는 이름을 설정하려면 `Friend_name`을 사용하십시오.

## 펌웨어 업데이트

일부 Heiman Zigbee 장치는 OTA 업데이트를 지원하며 `configuration.yaml`에서 `ota` 옵션을 켜야 합니다.
