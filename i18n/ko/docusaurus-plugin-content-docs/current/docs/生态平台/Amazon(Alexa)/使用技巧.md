# 사용팁

## 장치 검색

- Heiman Matter 장치의 전원이 켜지면 일반적으로 30초 이내에 Alexa 앱이 자동으로 검색합니다.
- 자동으로 검색되지 않는 경우: Alexa App → Device → "+" → "Add Device" → "Matter" 입력
- Matter 설정 코드 또는 기기 하단의 QR 코드를 이용하여 페어링을 완료하세요.

## Zigbee 장치 접근 (Echo Plus / Echo Show)

- Zigbee Hub가 있는 에코 장치는 Heiman Zigbee 센서와 직접 페어링할 수 있습니다.
- 페어링 모드: Alexa App → Device → "+" → "Zigbee" → "Search Device"
- Heimann 장치는 내장된 Zigbee 코디네이터와 Alexa와 호환됩니다.

## 일상적인 팁

- "사용자 정의 작업"을 사용하여 보다 자세한 음성 안내를 방송하세요.
- 음성 중복을 피하기 위해 후속 작업을 수행하기 전에 긴급 루틴에 "5초 대기"를 추가합니다.
- "소리 감지" 기능을 활용하여 Alexa가 Heyman 알람의 고음 알람에 추가 응답을 제공할 수 있습니다.

## 알렉사 가드 참고

Alexa Guard Plus는 현재 미국에서만 사용할 수 있습니다. 중국 사용자는 루틴을 통해 일부 기능을 시뮬레이션할 수 있습니다.
