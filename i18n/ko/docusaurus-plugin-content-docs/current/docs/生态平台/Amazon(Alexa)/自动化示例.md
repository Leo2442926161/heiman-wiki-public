# 자동화 예시

## Alexa Guard 통합(미국)

**전제 조건:** Alexa Guard Plus 구독(미국에서 사용 가능)

**구성 프로세스:**
1. Alexa 앱에서 Guard Plus를 켜세요
2. Hyman 연기/일산화탄소 경보기를 "스마트 홈 장치"로 추가
3. "연기 경보 감지" 스마트 응답 활성화

**자동 응답:**
- 연기 경보 작동 → Alexa가 자동으로 비상 연락처에 전화를 겁니다.
- CO 경보 작동 → 집 전체에 Alexa 음성 경고, 자동 환기
- 문과 창문이 비정상적으로 열림 → 푸시 알람 알림

## 연기 감지 루틴

**트리거:** Hyman 연기 경보기 → 연기 감지됨

**작업 실행:**
- 알렉사 음성 안내: "⚠️ 연기가 감지되었습니다. 안전에 유의해 주세요."
- 모든 에코 장치 → 알람 소리 재생
- 스마트 조명 설정 → 전체 조명 켜짐
- 휴대폰으로 푸시 알림

## "집 떠나기" 루틴

**트리거:** 음성 명령 "Alexa, 나 떠난다"

**작업 실행:**
- 하이만 도어 마그네틱 센서 → 무장 ON
- 하이만 바디센서 → 모션감지 켜기
- 하이만 침수 감지기 → 감지 켜기
- 모든 조명 → 꺼짐
- Alexa는 "보안 시스템이 켜져 있습니다. 안전한 여행 되세요"라고 대답했습니다.
