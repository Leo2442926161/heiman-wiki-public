# 사용팁

## 장치 페어링

1. 홈어시스턴트에서 "설정 → 장치 및 서비스 → ZHA → 구성"으로 들어갑니다.
2. "장치 추가"를 클릭한 후 Heiman 장치의 페어링 모드를 시작합니다.
3. 페어링에 실패할 경우 장치를 USB 코디네이터에 가까이 가져간 후 3미터 이내에 다시 시도하세요.

## 선호하는 USB 코디네이터

| 모델 | 칩 | 추천지수 |
|------|------|----------|
| 소노프 ZB동글-E | EFR32MG21 | ⭐⭐⭐⭐⭐ |
| 콘비 II | 2652 | ⭐⭐⭐⭐ |
| 엘레랩스 지그비 USB | 2652P | ⭐⭐⭐⭐ |

## 채널 조정

ZHA 기본 채널은 15입니다. WiFi 2.4GHz에 심각한 간섭이 있는 경우:
- 'Silicon Labs Flasher Utility'를 사용하여 채널을 수정하세요.
- 일반적으로 권장되는 채널: 15, 20, 25 (WiFi 1/6/11은 피하세요)

## 네트워크 최적화

- Heiman 장치 3~4개마다 Zigbee 라우팅 장치(예: 스마트 소켓)를 추가하여 네트워크 확장
- ZHA 네트워크 시각화 도구를 사용하여 장치 연결 품질을 확인하세요.
- 최적의 호환성을 유지하기 위해 장치 펌웨어 업데이트를 정기적으로 확인하십시오.
