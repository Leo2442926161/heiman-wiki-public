# 자동화 예시

## 연기 경보기 연동

``yaml
별칭: "ZHA 화재 경보기 연결"
트리거:
  - 트리거: 상태
    엔터티_ID: Binary_sensor.heiman_smoke_zha
    받는 사람: "켜기"
작업:
  - 액션: light.turn_on
    목표:
      Area_id: 거실
  - 동작: tts.speak
    목표:
      엔터티_ID: media_player.speaker
    데이터:
      메시지: "연기 경보가 작동되었습니다!"
````

## 기기 페어링 자동화

``yaml
별칭: "새 장치 페어링 알림"
트리거:
  - 트리거: zha.event
    하위유형: 하이만
    명령: device_joined
작업:
  - 조치: inform.pertant_notification
    데이터:
      메시지: "새로운 Haiman 장치가 ZHA 네트워크에 연결되었습니다. 구성 이름과 영역으로 이동하세요."
````

## 물 침수 감지

``yaml
별칭: "ZHA 홍수 테스트"
트리거:
  - 트리거: 상태
    엔터티_ID: Binary_sensor.heiman_water_leak_zha
    받는 사람: "켜기"
작업:
  - 액션: switch.turn_off
    엔터티_ID: switch.main_water_valve
  - 액션: inform.mobile_app_phone
    데이터:
      메시지: "💧 홍수가 감지되었습니다!"
````
