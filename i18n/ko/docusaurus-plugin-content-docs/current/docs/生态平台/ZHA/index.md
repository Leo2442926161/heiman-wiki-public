#ZHA(지그비 홈 오토메이션)

ZHA는 별도의 소프트웨어 없이 Zigbee 기기를 직접 관리할 수 있는 Home Assistant 내장 Zigbee 통합 솔루션입니다.

## 호환성

Hyman Zigbee 센서는 ZHA에서 우수한 호환성을 가지며 다음을 지원합니다.

- **연기/CO 경보**: 경보 상태, 오류 감지, 배터리 잔량, 음소거
- **도어 센서**: 켜기/끄기, 조작
- **인간 적외선**: 동작 감지, 조명
- **온도 및 습도**: 온도, 습도
- **홍수**: 누수 경보
- **스마트 소켓**: 스위치, 소비전력 측정

## 하드웨어 요구사항

- 지그비 코디네이터: HUSBZB-1, Conbee II, SkyConnect, SLZB-06 등
- USB 또는 네트워크를 통해 홈어시스턴트에 연결

## 구성

ZHA는 구성이 쉽습니다. 장치를 자동으로 검색하고 검색하려면 직렬 포트와 영역을 선택하기만 하면 됩니다. 구성 파일을 수동으로 작성할 필요가 없습니다.

> **참고**: [홈어시스턴트 ZHA 문서](https://www.home-assistant.io/integrations/zha/)
