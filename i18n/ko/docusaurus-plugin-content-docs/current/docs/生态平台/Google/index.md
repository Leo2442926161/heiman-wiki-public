#구글홈

Google Home은 Matter 프로토콜과 다양한 기기 생태계를 지원하는 Google의 스마트 홈 플랫폼입니다.

## 통합 방법

### 물질 장비

Thread 장치를 통한 Hyman Matter는 Google Home 앱에 직접 추가할 수 있습니다.

1. Thread Border 라우터(Google Nest Hub Max, Nest Hub 2세대 등)가 있는지 확인하세요.
2. Google Home 앱을 엽니다.
3. '추가' → 'Matter 호환 장치'를 클릭하세요.
4. 기기에서 Matter 페어링 코드를 스캔하세요.

### 음성 제어

연결되면 Google 어시스턴트 음성을 통해 제어할 수 있습니다.

- "Hey Google, 연기 경보기 상태는 어때?"
- "Hey Google, 거실 온도는 몇 도야?"

> **참고**: Hyman Matter 기기는 Amazon에서 Google Home을 지원하는 것으로 표시되어 있습니다.
