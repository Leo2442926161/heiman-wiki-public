# 자동화 예시

## "나 외출" 루틴

**트리거:** Google Home에 "나 나갔어"라고 말하세요.

**작업 실행:**
- 하이만 도어 마그네틱 센서 → 암
- 하이만 인체적외선센서 → 집이탈감지 켜기
- 모든 조명과 가전제품을 끄세요.
- 온도 조절기 → 에너지 절약 온도로 설정
- Google Nest Hub → 출국 확인 카드 제시
- 음성 응답: "퇴근 모드가 켜져 있습니다. 안전 여행 되세요"

## "연기감지" 방송

**트리거:** Hyman 연기 경보기 → 연기 감지됨

**작업 실행:**
- 구글 네스트 스피커 → 집 전체 방송 "⚠️ 연기 감지! 즉시 확인해주세요!"
- 모든 스마트 조명 → 켜짐(Google Home 스마트 조명 세트를 통해)
- 가족의 휴대폰으로 푸시 알림
- 카메라 → 녹화 시작

## 취침 모드

**트리거:** Google Home에 "잘 자"라고 말하세요.

**작업 실행:**
- 하이만 도어센서 → 문 열림 알람 켜기
- 하이만 인체적외선 → 야간감지 켜기
- 조명 → 점차 꺼짐
- 음성 응답: "잘 자요. 보안 시스템이 켜져 있습니다."
