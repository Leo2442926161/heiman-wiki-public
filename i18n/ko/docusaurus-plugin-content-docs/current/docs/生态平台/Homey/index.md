#집처럼

Homey는 네덜란드 회사 Athom이 출시한 스마트 홈 허브로 Zigbee, Z-Wave, Wi-Fi, Matter, Thread 및 기타 프로토콜을 지원합니다.

## 통합 방법

Homey Zigbee 및 Z-Wave 장치는 Homey의 내장 무선 칩을 통해 직접 페어링되므로 추가 게이트웨이가 필요하지 않습니다. Matter 장치는 Homey Pro에 내장된 Thread 경계 라우터를 통해 액세스할 수 있습니다.

## 장점

- **하나의 여러 프로토콜**: Homey Pro는 Zigbee, Z-Wave, Thread 및 Wi-Fi를 동시에 지원합니다.
- **로컬 실행**: 모든 자동화가 로컬에서 실행됩니다.
- **흐름 자동화**: 복잡한 연결 시나리오를 생성하기 위한 드래그 앤 드롭 프로그래밍

> **참고** : 홈미 앱스토어에서 하이만 관련 앱 검색 가능
