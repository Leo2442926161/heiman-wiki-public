# 자동화 예시

## 연기 경보 - 플래시 조명 및 알림

**흐름 구성:**

1. **트리거:** 하이만 연기 경보 → 상태가 `경보`로 변경됩니다.
2. **조건:** (없음)
3. **작업:**
   - 거실 조명 → 깜박임(빨간색, 1초에 한 번)
   - 푸시 알림 → 모바일: "⚠️ 연기 경보가 작동되었습니다!"
   - 음성 안내 → 홈미 스피커 : "연기가 감지되었습니다. 안전에 유의해주세요!"

## CO 경보 - 보일러를 끄세요

**흐름 구성:**

1. **트리거:** Hyman CO Alarm → 상태가 'alert'로 변경됩니다.
2. **조건:** (없음)
3. **작업:**
   - 보일러 스위치 → OFF
   - 스마트 소켓(외기시스템) → On
   - 푸시 알림 → "일산화탄소 경보, 보일러가 정지되었습니다"

## 온도 및 습도 한계 초과 경보

1. **트리거:** Hyman 온습도 센서 → 온도 40°C 이상
2. **작업:**
   - 에어컨 → 냉방 모드 켜기
   - 푸시 알림 → "실내 온도가 너무 높아 에어컨이 자동으로 켜졌습니다"
