# 사용팁

## Zigbee 채널 간섭

Homey에 내장된 Zigbee 칩은 특정 채널을 사용합니다. Heiman 장치가 자주 오프라인 상태가 되는 경우:

1. 홈미 설정 → 지그비에서 현재 채널을 확인하세요.
2. Homey Insight 앱을 사용하여 환경 간섭을 검색하세요.
3. 간섭이 적은 채널로 전환합니다(Wi-Fi 1/6/11에서 멀리 떨어져 있음).

## 장비 유통망

- 하이만 기기의 페어링 모드 진입 방법: 버튼을 5초간 길게 누르세요. (일부 모델은 3번 빠르게 눌러야 합니다.)
- 홈미 앱에서 기기 추가 시 "기기 추가 → 지그비 → 검색"을 선택하세요.
- 페어링 실패 시, 기기를 호미에게 더 가까이(3m 이내) 가져간 후 다시 시도해 보세요.

## 흐름 최적화 제안

- 일시적인 거짓 긍정을 방지하기 위해 흐름에 "지연" 조건(예: 2초)을 추가합니다.
- "태그" 기능을 활용하여 하이만 장비를 분류하고 관리하세요.

## 신호 범위 확장

Heiman 장치가 Homey에서 멀리 떨어져 있는 경우 Zigbee 중계 장치(예: 스마트 소켓)를 추가하여 네트워크 범위를 확장할 수 있습니다.
