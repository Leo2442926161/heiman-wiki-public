# 사용팁

## 홈브리지 대 물질

Heiman 장치는 두 가지 방법으로 Apple Home에 연결할 수 있습니다.

| 방법 | 장점 | 단점 |
|------|------|------|
| **스레드보다 중요한 문제** | 기본 지원, 브리지 필요 없음, 짧은 대기 시간 | 스레드 경계 라우터 필요 |
| **홈브리지 플러그인** | 모든 Zigbee 장치와 호환 가능 | 홈브리지 서비스 추가 실행 필요 |

## 스레드 네트워크 구축

1. 하나 이상의 Thread Border 라우터가 필요합니다(Apple HomePod mini, Apple TV 4K 3세대).
2. Hyman Matter 장치는 10미터 이내에 자동으로 Thread 네트워크에 연결됩니다.
3. **Nam* 또는 **Eve App**을 사용하여 스레드 네트워크 토폴로지를 봅니다.

## 자동화된 최적화

- Apple Home의 "보안" 카테고리 자동화는 보안 상태와 직접 연결 가능
- 조명이 즉시 전환되는 대신 부드럽게 변경되도록 하려면 "변형" 명령을 사용하세요.
- Shorters App을 통해 플랫폼 간 연결 생성(연기 경보기 → SMS 전송 등)

## FAQ

- Matter 장치에 "응답 없음"이 표시되면 Apple TV/HomePod를 다시 시작하여 복구하세요.
