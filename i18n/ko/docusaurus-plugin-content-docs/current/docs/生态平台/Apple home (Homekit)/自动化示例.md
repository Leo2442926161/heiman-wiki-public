# 자동화 예시

## 연기 감지 보안 경보

**HomeKit 자동 구성:**

1. **발동 조건:** 하이만 연기 경보기 → 연기 감지됨
2. **보안 설정:** 보안 시스템 상태를 "알람"으로 전환합니다.
3. **작업 실행:**
   - HomePod → 맞춤 오디오 재생 "연기가 감지되었습니다. 안전에 유의해주세요"
   - 모든 스마트 조명 → 점등 및 플래시
   - 모든 가족 구성원에게 푸시 알림

## 집에서/외출 장면

**단축키 단축키 명령: "집 보안 점검 나가기"**

````
마지막 사람이 집을 떠날 때
  보안시스템 설치 → 외출 시 무장 착용
  하이만 도어센서 설정 → 알림 켜기
  하이만 바디센서 설정 → 알람 켜기
  모든 조명을 끄십시오
  설정 온도 조절기 → 에너지 절약 모드
  "🏠 재택 모드 활성화됨" 알림 보내기
````

## 온도 이상 알림

**트리거 조건:** Hyman 온습도 센서 → 온도 > 35°C

**작업 실행:**
- HomePod → 음성방송 실내 온도가 한도를 초과했습니다.
- iPhone으로 알림 푸시
- 자동으로 에어컨 켜기(HomeBridge가 구성된 경우)
