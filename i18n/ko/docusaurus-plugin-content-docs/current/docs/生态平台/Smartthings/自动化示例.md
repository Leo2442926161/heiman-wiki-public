# 자동화 예시

## "Good Night" 취침 모드

**발동 조건:** 하이만 문 자기 센서 → 닫힘(침실 문이 닫혀 있음을 나타냄)

**작업 실행:**
- 거실의 모든 조명을 꺼주세요
- 스마트락 → 잠금
- 보안 시스템 → 야간에 무장
- 푸시 알림: "잘 자요. 시스템이 작동 중입니다."

## "외출" 모드

**발동 조건:** 하이만 인체 적외선 센서 → 15분간 움직임 없음

**작업 실행:**
- 모든 조명이 꺼졌습니다.
- 온도 조절기 → 에너지 절약 모드
- 보안 시스템 → 집 밖에서 무장
- 문 센서 → 알람 켜기
- 푸시 알림: "외출 모드가 활성화되었으며 문 및 창문 센서가 활성화되었습니다."

## 연기 경보 비상 모드

**발동 조건:** Hyman 연기 경보기 → 연기 감지

**작업 실행:**
- 스마트 스피커 → 알람 소리 재생
- 모든 스마트 조명 → 빨간색으로 깜박임
- 모든 가족 구성원에게 푸시 알림
- SmartThings Hub에 연결된 경우 → 사이렌 작동
