# 사용팁

## 허브 호환성

Hyman Zigbee 장치는 **SmartThings Hub v2** 및 **Hub v3(Aeotec)**과 호환됩니다.

| 허브 모델 | 호환성 | 설명 |
|---------|---------|------|
| 허브 v2(STH-ETH-250) | ✅ | 최신 펌웨어로 업데이트해야 |
| 허브 v3(에오텍) | ✅ | 기본 지원, 권장 |
| 삼성 SmartThings 스테이션 | ⚠️ | 일부 Zigbee 장치만 지원 |

## 장치 페어링 프롬프트

- SmartThings 앱에서 "+" → "기기" → "브랜드" → "하이만"을 클릭하세요.
- 자동으로 인식되지 않는 장치는 "지그비 스캔"을 통해 수동으로 검색할 수 있습니다.
- 최초 페어링 시 기기와 허브 사이의 거리를 5m 이내로 유지하는 것을 권장합니다.

## 배터리 최적화

Heiman 센서의 기본 보고 간격이 더 깁니다. 실시간 성능이 필요한 경우 장치 페이지에서 "감지 간격"을 5분으로 조정할 수 있습니다.

## 자동화 모범 사례

- SmartThings의 "모드" 기능을 사용하여 "재택/외출" 상태를 구분합니다.
- 중복성을 위해 주요 센서(연기, CO)에 대한 자동화 **2세트** 생성
