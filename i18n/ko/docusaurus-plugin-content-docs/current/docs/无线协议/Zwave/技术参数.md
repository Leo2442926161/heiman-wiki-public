# 기술적인 매개변수

다음은 Hyman Z-Wave 장치의 일반적인 기술 사양입니다.

## 무선 매개변수

| 매개변수 | 가치 |
|------|-----|
| 프로토콜 표준 | Z-Wave 플러스(ITU-T G.9959) |
| 작동 주파수 대역 | 800~900MHz(지역에 따라 다름: 유럽 규정 868.42MHz, 미국 규정 908.42MHz) |
| 전송률 | 최대 100kbps |
| 네트워크 토폴로지 | 메시 자가 치유 네트워크(소스 라우팅) |
| 노드 용량 | 단일 네트워크의 최대 232개 노드 |
| 통신거리 | 실내(노드 간) 약 30미터, 집 전체 메쉬 커버리지 |
| 암호화 방법 | AES-128(S2 인증됨 / S2 인증되지 않음) |

## 지역 주파수 대역

| 지역 | 중심 주파수 |
|------|----------|
| 유럽 ​​| 868.42MHz |
| 북미 | 908.42MHz |
| 기타 지역 | 현지 규정에 따름 |

## 전기 및 전력 소비

| 매개변수 | 가치 |
|------|-----|
| 전원 공급 방식 | 배터리 전원 공급 장치(CR123A/AA) 또는 주 전원 공급 장치 |
| 배터리 수명 | 센서의 경우 약 2~3년 |
| 전력 전송 | ≤ 1mW(SRD 규정 준수) |
| 수신감도 | 약 -100dBm |

## 환경 매개변수

| 매개변수 | 가치 |
|------|-----|
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤ 90% RH(비응결) |

## 네트워크 특성

| 특징 | 설명 |
|------|------|
| 스마트스타트 | 지원(700 시리즈), QR 코드를 스캔하여 네트워크에 연결 |
| S2 보안 | 세 가지 보안 수준 지원: 암호화되지 않음, S2 인증되지 않음, S2 인증됨 |
| CC가 포함되어 있습니다 | 보안 CC, 보안 2 CC, 펌웨어 업데이트 CC, 기본 CC, 배터리 CC, 알림 CC 등 |
