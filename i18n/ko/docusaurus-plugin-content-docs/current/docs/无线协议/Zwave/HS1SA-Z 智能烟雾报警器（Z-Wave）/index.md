# HS1SA-Z 스마트 연기 경보기(Z-Wave)

HS1SA-Z는 하이만에서 출시한 **Z-Wave** 스마트 연기 경보기입니다. 광전 연기 감지 기술을 사용하고 Z-Wave Plus 표준과 호환되며 유럽 및 미국 스마트 홈 시장에 적합합니다.

## 제품 개요

- **제품 유형**: 스마트 연기 경보기
- **통신 프로토콜**: Z-Wave Plus
- **인증기준**: EN 14604
- **일반적인 시나리오**: 주택, 아파트 및 기타 장소에서의 화재 경고

## 기술적인 매개변수

| 매개변수 | 가치 |
|------|-----|
| 센서 유형 | 적외선 광전 |
| 통신 프로토콜 | Z-Wave 플러스(ITU-T G.9959) |
| 작동 주파수 대역 | 868.42MHz(EU)/908.42MHz(미국) |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | CR123A 리튬 배터리 |
| 배터리 수명 | 약 2~3년 |
| 암호화 방법 | S2 인증됨 / S2 인증되지 않음 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤90% RH(응결 없음) |

## 기능

- 느린 화상 및 화염 연기를 감지하는 광전식 연기 감지기
- 85dB의 높은 소리와 빛 알람
- 배터리 부족 알림
- 셀프 테스트 기능
- 변조 방지 경보
- Z-Wave S2 보안 암호화
- OTA 펌웨어 업그레이드 지원

## 인증

- EN 14604 인증
- Z-Wave Plus 인증
- S2 보안 인증
