#Z-웨이브

Z-Wave는 스마트 홈용으로 설계된 무선 메시 네트워크 프로토콜입니다. Z-Wave Alliance에서 관리하며 유럽과 미국 시장에 널리 보급되어 있습니다.

## 기술 사양

| 특징 | 설명 |
|------|------|
| 작동 주파수 대역 | 868.42MHz(EU)/908.42MHz(미국) |
| 전송률 | 최대 100kbps(Z-Wave 800 시리즈) |
| 네트워크 토폴로지 | 메시 네트워크, 최대 232개 노드 |
| 호환성 | 모든 Z-Wave 장치와 역호환 가능 |
| 보안 | Z-Wave S2 암호화 |

## 하이만과 Z-Wave

Hyman은 주로 유럽과 미국 시장에서 판매되는 다양한 Z-Wave 호환 센서를 제공합니다. Heyman 장치의 Z-Wave 버전에는 다음이 포함됩니다.

- 일산화탄소 센서 HS1CA-Z
- 연기 감지기 HS1SA-Z
- 도어센서 HS1DS-Z

## 호환 플랫폼

- SmartThings 허브
- 휴비타트 엘리베이션
- 호미 프로
- 홈 어시스턴트(Z-Wave JS를 통해)
-오픈HAB

> **참고**: Z-Wave는 중국 시장에서 인기가 낮으며 Haiman Z-Wave 제품은 주로 유럽과 미국 수출 시장을 대상으로 합니다.
