# HS1CA-Z 지능형 일산화탄소 경보기(Z-Wave)

HS1CA-Z는 하이만에서 출시한 스마트 일산화탄소 경보기 **Z-Wave**입니다. 전기화학 센서 기술을 사용하며 Z-Wave Plus 표준과 호환됩니다.

## 제품 개요

- **제품 유형**: 지능형 일산화탄소 경보기
- **통신 프로토콜**: Z-Wave Plus
- **인증기준**: EN 50291
- **일반적인 시나리오**: 주택, 아파트 및 기타 장소에서의 CO 누출 경고

## 기술적인 매개변수

| 매개변수 | 가치 |
|------|-----|
| 센서 유형 | 전기화학 CO 센서 |
| 통신 프로토콜 | Z-Wave 플러스(ITU-T G.9959) |
| 작동 주파수 대역 | 868.42MHz(EU)/908.42MHz(미국) |
| 감지 범위 | 0~1000ppm |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | 배터리 구동 |
| 센서 수명 | 보통 7년 |
| 암호화 방법 | S2 인증됨 / S2 인증되지 않음 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤90% RH(응결 없음) |

## 기능

- 고정밀 전기화학 CO 센서
- 농도 분류 알람
- 85dB의 높은 소리와 빛 알람
- 배터리 부족 알림
- 센서 수명 만료 알림
- 셀프 테스트 기능
- Z-Wave S2 보안 암호화
- OTA 펌웨어 업그레이드 지원

## 인증

- EN 50291 인증
- Z-Wave Plus 인증
- S2 보안 인증
