# HS1DS-Z 도어센서(Z-Wave)

HS1DS-Z는 하이만에서 출시한 **Z-Wave** 도어 센서 센서입니다. 자기 리드 스위치를 통해 문과 창문의 개폐 상태를 감지하며 Z-Wave Plus 표준과 호환됩니다.

## 제품 개요

- **제품 유형**: 도어 센서
- **통신 프로토콜**: Z-Wave Plus
- **감지 방법**: 리드 스위치(reed switch)
- **일반적인 시나리오**: 문 및 창 상태 모니터링, 보안 배포 및 지능형 연결

## 기술적인 매개변수

| 매개변수 | 가치 |
|------|-----|
| 탐지 방법 | 리드 스위치(리드 스위치) |
| 통신 프로토콜 | Z-Wave 플러스(ITU-T G.9959) |
| 작동 주파수 대역 | 868.42MHz(EU)/908.42MHz(미국) |
| 전원 공급 방식 | CR1632 버튼 배터리 |
| 배터리 수명 | 약 2~3년 |
| 설치 여유 | 10mm 이하 |
| 암호화 방법 | S2 인증됨 / S2 인증되지 않음 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤90% RH(응결 없음) |

## 기능

- 문 및 창문 개폐 상태 실시간 감지
- 탬퍼 알람(Tamper)
- 배터리 부족 알림
- 장면 연동 지원(문열기, 조명켜기, 외출시 무장)
- 설치가 용이함(3M 접착테이프 또는 나사고정)
- Z-Wave S2 보안 암호화
- OTA 펌웨어 업그레이드 지원
