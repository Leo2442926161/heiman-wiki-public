#지원 모델 목록

Hyman Z-Wave 제품은 주로 유럽과 미국의 스마트 홈 시장을 대상으로 하며 Z-Wave Plus 표준과 호환됩니다.

## 보안알람 카테고리

| 모델 | 제품명 | 설명 |
|------|----------|------|
| HS1SA-Z | 스마트 연기 경보기(Z-Wave) | Z-Wave 버전 연기 경보기 |
| HS1CA-Z | 지능형 일산화탄소 경보기(Z-Wave) | Z-Wave 버전 CO 경보 |
| HS1CG-Z | 가연성 가스 감지기(Z-Wave) | Z-Wave 버전 가스 감지기 |

## 센서 클래스

| 모델 | 제품명 | 설명 |
|------|----------|------|
| HS1DS-Z | 도어센서(Z-Wave) | Z-Wave 버전 도어 및 창 센서 |
| HS1MS-Z | 인체적외선센서(Z-Wave) | 인체 감지의 Z-Wave 버전 |
| HS1WL-Z | 침수 감지기(Z-Wave) | Z-Wave 버전 누수 감지 |

## 호환성

| 프로젝트 | 설명 |
|------|------|
| 표준 | Z-Wave Plus(500/700 시리즈 칩) |
| 호환 허브 | Homey, Hubitat, SmartThings, Vera, Fibaro, 홈 어시스턴트(Z-Wave JS) |
| S2 보안 | S2 보안 프로토콜 지원(암호화되지 않음 / S2 인증되지 않음 / S2 인증됨) |
| 스마트스타트 | 지원(700 시리즈 칩 모델) |
| 오타 업그레이드 | FW 업데이트 CC 지원 |
