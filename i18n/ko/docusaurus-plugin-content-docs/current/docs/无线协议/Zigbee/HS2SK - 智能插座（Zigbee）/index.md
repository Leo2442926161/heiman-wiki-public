#HS2SK - 스마트 소켓(지그비)

## 개요

HS2SK는 하이만(Heiman)이 출시한 지그비(Zigbee) 스마트 미터링 소켓이다. 원격 스위치 제어 및 실시간 전력 소비 측정을 지원합니다. 가전제품의 지능형 관리에 적합하며 정확도는 ±2%입니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 정격전압 | AC 220V(중국 규격) / AC 110V(미국 규격) |
| 최대 부하 | 16A / 3680W |
| 측정 정확도 | ±2% |
| 전원 공급 방식 | 주 전원 공급 장치 |

## 기능

- 원격 스위치 제어
- 실시간 전력/연료 계량
- 과부하 보호
- 타이머 스위치
- 전력 통계
- 장면 연계 지원

## 적용 가능한 시나리오

- 잊어버린 가전제품을 원격으로 끄기
- 에어컨/온수기 에너지 소비 통계
- 한 번의 클릭으로 홈 모드에서 나가서 필요하지 않은 가전제품을 끄세요
