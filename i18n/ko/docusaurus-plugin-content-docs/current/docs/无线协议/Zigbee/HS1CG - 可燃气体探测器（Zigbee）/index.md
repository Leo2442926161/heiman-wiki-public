#HS1CG - 가연성 가스 감지기(Zigbee)

## 개요

HS1CG는 하이만에서 출시한 지그비 가연성 가스 감지기입니다. 반도체 센서를 사용해 천연가스(메탄)와 액화석유가스(LPG) 누출을 감지할 수 있다. GB 15322 인증을 통과했으며 연결 솔레노이드 밸브의 자동 밸브 폐쇄를 지원합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 반도체 |
| 테스트 가스 | 천연가스(메탄)/액화석유가스(LPG) |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | AC 220V / DC 12V |
| 인증기준 | GB 15322 |
| 작동 온도 | -10°C ~ +50°C |

## 기능

- 천연가스/액화가스 누출 감지
- 농도 분류 알람
- 자동 차단 연동(솔레노이드 밸브 포함)
- 85dB 청각 및 시각 경보
- 셀프 테스트 기능
- 센서 수명 만료 알림

## 인증기준

- GB 15322 (가연성 가스 감지기에 대한 중국 표준)

## 적용 가능한 액세서리

- JT 시리즈 가스 비상 차단 밸브
