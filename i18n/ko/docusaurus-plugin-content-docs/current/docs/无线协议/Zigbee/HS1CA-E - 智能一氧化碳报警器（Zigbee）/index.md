# HS1CA-E - 스마트 일산화탄소 경보기(지그비)

## 개요

HS1CA-E는 EU 시장을 위한 Hyman의 Zigbee 스마트 일산화탄소(CO) 경보기입니다. 전기화학 센서를 사용하여 0~1000ppm 농도의 일산화탄소 가스를 정확하게 감지할 수 있습니다. EN 50291 인증을 통과했습니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 전기화학 |
| 감지 범위 | 0 ~ 1000ppm |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | DC 3V(CR123A) |
| 센서 수명 | 약 7년 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% 상대습도 |
| 설치방법 | 벽걸이 |

## 기능

- 고정밀 전기화학 CO 센서
- 농도 등급 경보(소리 및 빛 프롬프트)
- 85dB 알람음
- 배터리 부족 알림
- 센서 수명 만료 알림
- 셀프 테스트 기능
-OTA 펌웨어 업그레이드

## 인증기준

- EN 50291(일산화탄소 경보기에 대한 유럽 표준)
-CE/레드

## 적용 가능한 액세서리

- CR123A 배터리
