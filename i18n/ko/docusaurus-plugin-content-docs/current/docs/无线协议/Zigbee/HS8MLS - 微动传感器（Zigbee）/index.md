# HS8MLS - 마이크로 모션 센서(지그비)

## 개요

HS8MLS는 하이만이 출시한 장거리 지그비 인체적외선센서(PIR)다. 장거리 감지가 필요한 장면을 위해 특별히 설계되었으며 복도, 홀 등 개방된 공간에 적합합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 수동적외선(PIR) |
| 감지 거리 | 장거리 감지 |
| 전원 공급 방식 | DC 3V 배터리 |

## 기능

- 장거리 움직임 감지
- 저전력 소모 설계
- 장면 연계 지원
