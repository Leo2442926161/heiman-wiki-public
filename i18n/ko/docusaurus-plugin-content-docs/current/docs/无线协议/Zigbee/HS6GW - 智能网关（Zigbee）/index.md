# HS6GW - 스마트 게이트웨이(지그비)

## 개요

HS6GW는 하이만이 출시한 지그비 스마트 게이트웨이이다. Heiman의 Zigbee 네트워크의 코디네이터로서 Wi-Fi 또는 이더넷 업링크를 통해 인터넷에 연결하고 최대 128개의 하위 장치에 대한 액세스를 지원합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0(코디네이터) |
| 업링크 연결 | Wi-Fi/이더넷 |
| 최대 노드 | 128개의 하위 장치 |
| 전원 공급 방식 | DC 5V USB 전원 |

## 기능

- 지그비 네트워크 코디네이터
- Wi-Fi/이더넷 업링크 연결
- 자동 장치 검색 및 페어링
- 로컬 씬 연계
- 원격 접속(Heiman Cloud Platform을 통해)
- 최대 128개의 하위 장치 지원
-OTA 펌웨어 업그레이드

## 적용 가능한 시나리오

- 하이만 집 전체 인텔리전스의 핵심 연결점
- 원격 모니터링을 실현하기 위해 Haiman APP와 협력
