#지그비

Zigbee는 IEEE 802.15.4 표준을 기반으로 하는 저전력, 낮은 데이터 속도의 무선 메시 네트워크 프로토콜입니다. 스마트 홈 분야에서 가장 성숙하고 널리 사용되는 프로토콜 중 하나입니다.

## 하이만과 지그비

Hyman은 Zigbee 프로토콜을 채택한 최초의 중국 제조업체 중 하나이며 거대한 Zigbee 제품 라인을 보유하고 있습니다. Hyman Zigbee 장치는 **Zigbee 3.0** 표준과 호환되며 주류 Zigbee 게이트웨이 및 코디네이터와 상호 운용될 수 있습니다.

## 기술 사양

| 특징 | 설명 |
|------|------|
| 주파수 대역 | 2.4GHz(전세계) |
| 전송률 | 최대 250kbps |
| 네트워크 토폴로지 | 메시 자가 치유 네트워크 |
| 노드 수 | 단일 네트워크는 최대 65535개의 노드를 지원합니다 |
| 통신거리 | 실내 약 10~30미터(눈에 보이는 장애물) |
| 전력 소비 | 매우 낮음, 센서 배터리 수명 최대 3년 |

## 호환 플랫폼

Hyman Zigbee 장치는 다음 플랫폼에 연결할 수 있습니다.

- **Zigbee2MQTT** — 오픈 소스 Zigbee-MQTT 브리지
- **ZHA** — Zigbee 통합이 내장된 홈 어시스턴트
- **Hubitat** — 현지화된 지능형 허브
- **Homey** — 다중 프로토콜 스마트 허브
- **SmartThings 허브** — 삼성 스마트 홈 허브
- **Tuya Zigbee 게이트웨이** — Tuya 생태계
- **Hyman HS6GW 게이트웨이** — 공식 게이트웨이

## Hyman Zigbee 제품 라인

- HS1SA / HS3SA / HS8SA 시리즈 연기 경보기
- HS1CA/C1 시리즈 일산화탄소 경보기
- HS1CG / HS3CG 시리즈 가연성 가스 감지기
- HS1DS 시리즈 도어 센서
- HS1MS/HS3MS/M1 시리즈 인체 적외선 센서
- HS2WL / HS3WL / HS1WL 시리즈 침수 감지기
- HS2SK 시리즈 스마트 소켓
- HS2FD 낙하 감지기
