#HS2FD - 넘어짐 감지기(지그비)

## 개요

HS2FD는 하이만에서 출시한 지그비 스마트 낙상 감지기입니다. 노인들을 위해 특별히 설계되었습니다. 가속도계와 자이로스코프가 내장되어 있습니다. AI 알고리즘을 이용해 넘어지는 자세를 파악해 자동으로 조난 신호를 보낼 수 있다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 가속도계 + 자이로스코프 |
| 탐지 기술 | AI 넘어짐 감지 알고리즘 |
| 전원 공급 방식 | DC 3V 배터리 |
| 착용방법 | 벨트 클립/끈 |

## 기능

- 자동 넘어짐 감지
- 한 번의 클릭으로 수동으로 도움을 요청하세요
- AI 알고리즘은 낙상과 일반적인 활동(앉기, 구부리기)을 구별하여 오경보를 줄입니다.
- 착용 가능한 디자인(벨트 클립 또는 랜야드)
- 배터리 부족 알림

## 적용 가능한 시나리오

- 노인홈케어 : 독거노인을 24시간 돌봄
- 요양원: 중앙 집중식 보호 및 관리
- 재활센터: 수술 후 환자의 낙상 위험 모니터링
