#HS8SA - 스마트 연기 경보기(지그비)

## 개요

HS8SA는 미국 시장을 위한 Hyman의 Zigbee + Wi-Fi 스마트 연기 경보기입니다. 고온 연결 기능을 갖추고 있으며 UL 217 인증을 통과했습니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 / Wi-Fi |
| 센서 유형 | 적외선 광전 |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | DC 3V(CR123A) |
| 작동 온도 | -10°C ~ +50°C |
| 인증기준 | UL 217 |
| 해당 시장 | 미국 |

## 기능

- 광전식 연기 감지
- 고온연동 경보
- 85dB의 높은 소리와 빛 알람
- 배터리 부족 알림
- 셀프 테스트 기능
- 변조 방지 경보
-원격 음소거 지원
- 듀얼 프로토콜 통신(Zigbee + Wi-Fi)

## 인증기준

- UL 217(미국 연기 경보기 표준)
