# JT 시리즈 - 가스비상차단밸브(Zigbee)

## 개요

JT 시리즈 가스 비상 차단 밸브는 Hyman Zigbee 가연성 가스 감지기와 함께 사용되는 실행 장치입니다. 가스 누출이 감지되면 가스 파이프라인 밸브를 원격 또는 자동으로 닫아 사고를 예방할 수 있습니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 유형 | 독립형 + Zigbee 연동 |
| 테스트 가스 | 천연가스/액화석유가스 |
| 인증기준 | GB 15322 |
| 작동 온도 | -10°C ~ +50°C |

## 기능

- HS1CG/HS3CG와 연동하여 밸브를 자동으로 닫음
- 원격 수동 밸브 폐쇄 지원
- 누출 경보 푸시
- 솔레노이드 밸브 드라이버

## 인증기준

- GB 15322

## 적용 가능한 액세서리

- HS1CG / HS3CG 가연성 가스 감지기
