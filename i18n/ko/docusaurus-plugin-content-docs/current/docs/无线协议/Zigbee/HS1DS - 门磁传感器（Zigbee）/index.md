# HS1DS - 도어 자기 센서(지그비)

## 개요

HS1DS는 하이만에서 출시한 지그비 도어 자기 센서입니다. 문과 창문의 개폐 상태를 감지하는 데 사용됩니다. 지능형 보안 시스템의 기본 구성 요소입니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 리드 스위치 |
| 전원 공급 방식 | CR1632 버튼 배터리 |
| 배터리 수명 | 약 2~3년 |
| 설치 여유 | 10mm 이하 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% 상대습도 |

## 기능

- 문 및 창문 개폐 상태 실시간 감지
- 탬퍼 알람(Tamper)
- 배터리 부족 알림
- 연동 지원(문열기, 전등켜기, 외출시 무장 등)
- 설치가 용이함(3M 접착테이프 또는 나사고정)

## 적용 가능한 액세서리

- CR1632 버튼 배터리
- 3M 양면테이프/나사
