# HS1WL - 누수 감지기(지그비)

## 개요

HS1WL은 하이만에서 출시한 1세대 지그비 침수형 감지기입니다. 프로브 감지 방법을 채택하여 누수 또는 물 축적을 효과적으로 감지할 수 있습니다. 기본적인 침수 모니터링 장비입니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 탐지 방법 | 프로브 감지 |
| 전원 공급 방식 | DC 3V 배터리 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% 상대습도 |

## 기능

- 누수/물누수 실시간 감지
- 부저 알람
- 배터리 부족 알림
- 프로브 부식 방지 설계
