#HS3MS - 인체 적외선 센서(지그비)

## 개요

HS3MS는 Hyman의 3세대 Zigbee 인체 적외선 센서입니다. 360° 천장 장착형 디자인을 채택했으며 감지 감도와 범위가 이전 세대에 비해 크게 향상되었습니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 수동적외선(PIR) |
| 감지 각도 | 360°(천장 설치) |
| 감지 거리 | 7m 이하(25°C) |
| 전원 공급 방식 | DC 3V 리튬 배터리 |
| 배터리 수명 | 약 3년 |
| 작동 온도 | -10°C ~ +50°C |
| 설치방법 | 천장 설치 |

## 기능

- 모바일 인간 감지
- 360° 전방향 커버리지
- 배터리 부족 알림
- 변조 방지 경보
- 검출 감도 향상
- 장면 연계 지원
