#HS1MS - 인체 적외선 센서(지그비)

## 개요

HS1MS는 하이만(Hyman)이 출시한 표준 지그비 인체적외선센서(PIR)다. 수동 적외선 감지 기술을 사용하며 움직이는 인체에서 방출되는 적외선 열 복사를 감지할 수 있습니다. 스마트 조명 및 보안 시나리오에 널리 사용됩니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 수동적외선(PIR) |
| 감지 각도 | 110° |
| 감지 거리 | 7m 이하(25°C) |
| 전원 공급 방식 | DC 3V 리튬 배터리 |
| 배터리 수명 | 약 3년 |
| 작동 온도 | -10°C ~ +50°C |
| 설치방법 | 벽걸이 |

## 기능

- 모바일 인간 감지
- 110° 광각 감지
- 배터리 부족 알림
- 변조 방지 경보
- 장면 연계 지원
