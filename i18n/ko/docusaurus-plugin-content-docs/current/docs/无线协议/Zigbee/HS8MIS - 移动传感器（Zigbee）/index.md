#HS8MIS - 모바일 센서(지그비)

## 개요

HS8MIS는 하이만에서 출시한 다기능 지그비 모션 센서입니다. PIR 인체 감지 및 조명 감지 기능이 통합되어 있으며 빛과 움직임을 이중으로 모니터링하는 스마트 장면에 적합합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 수동적외선(PIR) + 조명 |
| 감지 각도 | 110° |
| 감지 거리 | 7m 이하 |
| 전원 공급 방식 | DC 3V 배터리 |

## 기능

- 모바일 인간 감지
- 조명 감지
- 저전력 소모 설계
- 장면 연계 지원
