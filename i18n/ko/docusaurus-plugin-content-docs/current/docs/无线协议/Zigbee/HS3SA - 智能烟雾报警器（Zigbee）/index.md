#HS3SA - 스마트 연기 경보기(Zigbee)

## 개요

HS3SA는 Hyman의 3세대 Zigbee 스마트 연기 경보기입니다. 1세대 기반으로 탐지 성능과 연계 능력을 강화해 EU 시장에 적합하다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 적외선 광전 |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | DC 3V(CR123A) |
| 배터리 수명 | 약 3년 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% 상대습도 |
| 설치방법 | 천장 설치 |

## 기능

- 광전식 연기 감지
- 85dB의 높은 소리와 빛 알람
- 배터리 부족 알림
- 셀프 테스트 기능
- 변조 방지 경보
-원격 음소거 지원
-OTA 펌웨어 업그레이드
- 향상된 무선 성능

## 인증기준

-EN 14604
-CE/레드

## 적용 가능한 액세서리

- CR123A 배터리
