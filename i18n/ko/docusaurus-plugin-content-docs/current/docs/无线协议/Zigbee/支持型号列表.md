#지원 모델 목록

Heyman은 업계에서 가장 광범위한 Zigbee 제품 라인을 보유하고 있으며, 출시된 지원 모델은 다음과 같습니다.

## 보안알람 카테고리

| 모델 | 제품명 | 설명 |
|------|----------|------|
| HS1SA-E | 스마트 연기 경보기(Zigbee) | EU 버전 |
| HS1SA-Z | 스마트 연기 경보기(Zigbee) | 중국 버전 |
| HS3SA | 지능형 연기 경보기(Zigbee) | 3세대 연기 경보기 |
| HS8SA | 지능형 연기 경보기(Zigbee) | 고온 연결형 |
| HS1CA-E | 지능형 일산화탄소 경보기(Zigbee) | EU 버전 |
| HS1CA-Z | 지능형 일산화탄소 경보기(Zigbee) | 중국어 버전 |
| HS1CG | 가연성 가스 감지기(Zigbee) | 천연가스/액화가스 |
| HS3CG | 가연성가스 감지기(Zigbee) | 3세대 가스감지기 |

## 센서 클래스

| 모델 | 제품명 | 설명 |
|------|----------|------|
| HS1DS | 도어 자기 센서(Zigbee) | 문과 창문 개폐 감지 |
| HS1MS | 인체적외선센서(Zigbee) | 표준형 |
| HS3MS | 인체적외선센서(Zigbee) | 3세대, 향상된 검출 감도 |
| M1-Z | 인체적외선센서(Zigbee) | 컴팩트한 디자인 |
| HS2WL | 침수 감지기(Zigbee) | 2세대 |
| HS3WL | 침수 감지기(Zigbee) | 3세대 |
| HS1WL | 침수 감지기(Zigbee) | 1세대 |
| HS8MLS | 마이크로 모션 센서(Zigbee) | 모션 감지 |
| HS8MIS | 모션 센서(지그비) | 모션 감지 |
| HS8OS | 광센서(지그비) | 빛 감지 |

## 스마트 소켓 및 패널

| 모델 | 제품명 | 설명 |
|------|----------|------|
| HS2SK | 스마트 소켓(지그비) | 10A/16A 버전 |

## 특수 함수 클래스

| 모델 | 제품명 | 설명 |
|------|----------|------|
| HS2FD | 넘어짐 감지기(Zigbee) | 노인 넘어짐 감지 알람 |
| HS2FD | 비상버튼/긴급팔찌 | 도움 요청 알람 |
| JT 시리즈 | 가스비상차단밸브(Zigbee) | HS1CG와 연동하여 자동 밸브 폐쇄 |

## 모델 명명 지침

| 명명 규칙 | 의미 |
|------------|------|
| 접미사 -E | EU 버전(868MHz / EN 표준) |
| 접미사 -Z | 중국 버전(2.4GHz / 중국 표준) |
| HS1/2/3/8 | 제품 세대 번호, 숫자가 클수록 최신 세대 |
