# HS3WL - 누수 감지기(지그비)

## 개요

HS3WL은 Hyman의 3세대 Zigbee 침수 감지기입니다. 이전 세대를 기반으로 향상된 감지 감도와 무선 성능을 갖추고 있으며 누수 모니터링 시나리오에 적합합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 탐지 방법 | 프로브 감지 |
| 전원 공급 방식 | DC 3V 배터리 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% 상대습도 |

## 기능

- 누수/물누수 실시간 감지
- 향상된 감지 감도
- 부저 알람
- 배터리 부족 알림
- 프로브 부식 방지 설계
