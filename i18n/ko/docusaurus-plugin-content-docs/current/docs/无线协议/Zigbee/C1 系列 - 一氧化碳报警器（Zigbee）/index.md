# C1 시리즈 - 일산화탄소 경보기(지그비)

## 개요

C1 시리즈는 Heiman이 출시한 기본 Zigbee 일산화탄소 경보기입니다. 전기화학 센서를 사용하며 일산화탄소 누출 감지 시나리오에 적합합니다. EN 50291 인증을 통과했습니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 전기화학 |
| 감지 범위 | 0 ~ 1000ppm |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | 배터리 구동 |
| 센서 수명 | 약 7년 |
| 작동 온도 | -10°C ~ +50°C |

## 기능

- 전기화학적 CO 검출
- 농도 분류 알람
- 85dB 청각 및 시각 경보
- 배터리 부족 알림
- 셀프 테스트 기능

## 인증기준

-EN 50291
