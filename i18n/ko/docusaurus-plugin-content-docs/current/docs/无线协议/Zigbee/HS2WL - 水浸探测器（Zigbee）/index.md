#HS2WL - 누수 감지기(지그비)

## 개요

HS2WL은 하이만에서 출시한 2세대 Zigbee 침수 감지기입니다. 프로브 및 케이블의 이중 채널 감지를 지원하며 수도관 파열, 싱크대 넘침 등 누수 이벤트를 감지하는 데 사용할 수 있습니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 탐지 방법 | 프로브 감지 + 케이블 감지(듀얼 채널) |
| 전원 공급 방식 | DC 3V 배터리 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% 상대습도 |

## 기능

- 누수/물누수 실시간 감지
- 프로브 + 케이블 이중 채널 감지
- 부저 알람
- 배터리 부족 알림
- 프로브 부식 방지 설계

## 적용 가능한 액세서리

- 감지 케이블을 연장하세요.
