# 기술적인 매개변수

다음은 Heiman Zigbee 장치의 일반적인 기술 매개변수입니다.

## 무선 매개변수

| 매개변수 | 가치 |
|------|-----|
| 프로토콜 표준 | 지그비 3.0(IEEE 802.15.4) |
| 작동 주파수 대역 | 2.4GHz(글로벌 공통 ISM 주파수 대역) |
| 전송률 | 최대 250kbps |
| 네트워크 토폴로지 | 메시 자가 치유 네트워크 |
| 노드 용량 | 단일 네트워크의 최대 65535개 노드 |
| 통신거리 | 실내 약 10~30m (벽 장애물 보기) |
| 암호화 방법 | AES-128-CCM |

## 호환성

| 프로젝트 | 설명 |
|------|------|
| 지그비 코디네이터 | Zigbee 3.0 표준 코디네이터와 호환 |
| 지원되는 플랫폼 | Zigbee2MQTT, ZHA(홈 어시스턴트), Hubitat, Homey, SmartThings, Tuya Zigbee 게이트웨이, Haiman HS6GW 게이트웨이 |
| 오타 업그레이드 | 무선 펌웨어 업그레이드 지원(일부 모델) |

## 전기 및 전력 소비

| 매개변수 | 가치 |
|------|-----|
| 전원 공급 방식 | 배터리 전원(CR123A/CR2450/AA) 또는 USB/주 전원(소켓 유형) |
| 배터리 수명 | 센서의 경우 약 2~3년(보고 빈도에 따라 다름) |
| 대기전력 소모 | 마이크로암페어 수준, 수면 폴링 메커니즘 지원 |

## 환경 매개변수

| 매개변수 | 가치 |
|------|-----|
| 작동 온도 | -10°C ~ +55°C |
| 작동 습도 | ≤ 95% RH(비응결) |
| IP 보호 수준 | 특정 모델에 따라 다름(센서 유형은 일반적으로 IP30) |
