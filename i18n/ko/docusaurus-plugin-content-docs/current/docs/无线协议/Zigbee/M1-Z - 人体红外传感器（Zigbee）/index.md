# M1-Z - 인체적외선센서(지그비)

## 개요

M1-Z는 하이만에서 출시한 소형 지그비 인체 적외선 센서이다. 110° 또는 360° 감지 각도(설치 방법에 따라 다름)를 지원하고 조도 센서가 내장되어 있으며 스마트 조명 및 보안 시나리오에 적합합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 수동적외선(PIR) + 조명 |
| 감지 각도 | 110°(벽걸이) / 360°(천장) |
| 감지 거리 | 7m 이하(25°C) |
| 조도 범위 | 0 ~ 1200럭스 |
| 전원 공급 방식 | DC 3V 리튬 배터리 |
| 배터리 수명 | 약 3년 |
| 작동 온도 | -10°C ~ +50°C |
| 설치방법 | 벽걸이/천장 |

## 기능

- 모바일 인간 감지
- 조도감지(0~1200 Lux)
- 110° / 360° 전환 가능
- 배터리 부족 알림
- 변조 방지 경보
- 컴팩트한 디자인
