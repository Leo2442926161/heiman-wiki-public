# HS1SA-E - 스마트 연기 경보기(Zigbee)

## 개요

HS1SA-E는 Hyman이 EU 시장을 위해 출시한 표준 Zigbee 스마트 연기 경보기입니다. 광전 연기 감지 기술을 채택하고 EN 14604 인증을 통과했으며 Red Dot Design Award를 수상했습니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 적외선 광전 |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | DC 3V(CR123A) |
| 배터리 수명 | 약 3년 |
| 통신거리 | 70m 이하(개방 공간) |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% 상대습도 |
| 설치방법 | 천장 설치 |

## 기능

- 느린 화상 및 개방형 화염에서 발생하는 연기를 감지하는 광전식 연기 감지기
- 85dB의 높은 소리와 빛 알람
- 배터리 부족 알림
- 셀프 테스트 기능(원클릭 테스트)
- 변조 방지 경보
-원격 음소거 지원
-OTA 펌웨어 업그레이드

## 인증기준

- EN 14604(화재 경보기에 대한 유럽 표준)
- 레드닷 디자인 어워드
-CE/레드

## 적용 가능한 액세서리

- CR123A 배터리
- 장착 나사/확장 튜브
