#HS8OS - 인간 존재 센서(지그비)

## 개요

HS8OS는 하이만(Hyman)이 출시한 지그비 인간 존재 센서이다. 밀리미터파 레이더 기술을 사용하여 정적 상태(눕거나 조용히 앉아 있는 등)에서 인체의 존재를 감지할 수 있어 움직임만 감지할 수 있는 기존 PIR 센서의 단점을 보완합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 지그비 3.0 |
| 센서 유형 | 밀리미터파 레이더(mmWave) |
| 감지 범위 | 반경 3~5m |
| 탐지 능력 | 정지된 인체 존재 여부(미세호흡 감지 포함) |
| 전원 공급 방식 | AC 전원 |
| 설치방법 | 천장 설치 |

## 기능

- 정지된 인체 감지(호흡의 미세한 움직임)
- 적용 반경 3~5미터
- 천장 설치, 최적의 감지 각도
- PIR 센서와 보완적으로 사용할 수 있습니다.

## 적용 가능한 시나리오

- 지능형 조명: 사람이 있으면 항상 켜져 있고, 주변에 사람이 없으면 자동으로 꺼집니다.
- 에어컨 에너지 절약 : 사람이 걸을 때 자동으로 에어컨이 꺼짐
- 스마트 노인 돌봄: 노인이 장시간 움직이지 않을 경우 조기 경고
