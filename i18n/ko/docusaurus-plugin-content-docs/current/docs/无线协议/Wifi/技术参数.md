# 기술적인 매개변수

다음은 Heimann WiFi 장치의 일반 기술 사양입니다.

## 무선 매개변수

| 매개변수 | 가치 |
|------|-----|
| 프로토콜 표준 | IEEE 802.11b/g/n |
| 작동 주파수 대역 | 2.4GHz(5GHz와 호환되지 않음) |
| 전송률 | 최대 150Mbps(802.11n) |
| 네트워크 토폴로지 | 스타 토폴로지(라우터에 직접 연결) |
| 통신거리 | 실내 약 30~50m (라우터 성능에 따라 다름) |
| 암호화 방법 | WPA2-PSK / WPA3 |

## 네트워크 배포 방법

| 방법 | 설명 |
|------|------|
| 스마트 구성 | 모바일 앱 원클릭 네트워크 구성(기존 WiFi 장비) |
| Bluetooth 지원 프로비저닝 | Bluetooth 임시 채널을 통해 WiFi 자격 증명 구성(일부 새 모델) |
| AP 모드 | 장치에서 핫스팟을 켜고 전화가 연결된 후 구성 |

## 전기 및 전력 소비

| 매개변수 | 가치 |
|------|-----|
| 전원 공급 방식 | 주 전원 공급 장치(소켓 유형/게이트웨이 유형) 또는 배터리 전원 공급 장치(알람 유형) |
| 소비전력 수준 | TCP/IP 연결을 유지해야 하기 때문에 Zigbee/Thread보다 높음 |
| 대기전력 소모 | 소켓타입 약 0.5W ~ 1W |

## 환경 매개변수

| 매개변수 | 가치 |
|------|-----|
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤ 90% RH(비응결) |
