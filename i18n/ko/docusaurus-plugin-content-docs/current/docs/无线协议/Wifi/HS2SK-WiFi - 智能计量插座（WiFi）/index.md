#HS2SK-WiFi - 스마트 계량 소켓(WiFi)

## 개요

HS2SK-WiFi는 하이만에서 출시한 WiFi 스마트 미터링 소켓입니다. 홈 라우터에 대한 2.4GHz 802.11 b/g/n 직접 연결을 지원하며 추가 게이트웨이 없이도 원격 제어 및 에너지 소비 모니터링이 가능합니다. 가정, 사무실 및 기타 시나리오에서 전기 제품의 지능형 관리에 적합합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | IEEE 802.11 b/g/n(2.4GHz) |
| 최대 부하 | 16A |
| 최대 전력 | 3680W(230V) / 1800W(110V) |
| 측정 정확도 | ±2% |
| 전원 공급 방식 | 주 전원 공급 장치 |
| 대기전력 소모 | ≤ 0.5W |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤ 90% 상대습도 |
| 쉘 소재 | PC V0 난연제 |
| 잭 사양 | 유럽 ​​표준/미국 표준(시장 버전에 따라 다름) |

## 기능

- 스마트 라이프 지원 / Tuya 스마트 앱 원격 제어
- 실시간 전력 모니터링 및 에너지 소비 통계
- 원격 스위치 제어
- 타이밍 스위치 및 카운트다운
- 장면 연계 및 자동화
- 음성 제어(Alexa, Google Home과 호환 가능)

## 인증기준

-CE/FCC
- RoHS
- ETL/UL(시장 버전에 따라 다름)
