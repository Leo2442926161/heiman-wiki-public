#와이파이

Wi-Fi는 가장 널리 사용되는 무선 네트워크 프로토콜이며 IEEE 802.11 표준을 기반으로 합니다. 일부 Heiman 스마트 기기는 Wi-Fi 연결을 지원하며 추가 게이트웨이 없이 라우터에 직접 연결할 수 있습니다.

## 적용 가능한 시나리오

Wi-Fi 장치는 다음과 같은 상황에 적합합니다.

- 완벽한 네트워크 인프라로 추가 게이트웨이 구매가 필요하지 않습니다.
- 전력 소비에 민감하지 않은 장치(예: 스마트 소켓)
- 원격 접속 및 클라우드 서비스가 필요한 시나리오
- 이미 성숙한 Wi-Fi 메시 네트워크가 적용되는 주택

## 하이만 Wi-Fi 제품

Heimann은 Wi-Fi 연결을 지원하는 제품을 제공합니다.

- **스마트 소켓 HS2SK-WiFi** — 소비전력 측정, 원격 스위치
- **스마트 게이트웨이** — Wi-Fi를 통해 라우터에 연결

## 메모

- Wi-Fi 센서 배터리는 Zigbee/Thread보다 빠르게 소모됩니다.
- Wi-Fi 기기 수가 많으면 네트워크 안정성에 영향을 줄 수 있습니다.
- Haiman의 주요 제품은 Zigbee/Matter over Thread이며, Wi-Fi 제품군은 비교적 간소화되어 있습니다.
