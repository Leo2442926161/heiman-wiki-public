# HS8SA-WiFi - 스마트 연기 경보기(WiFi 미국 버전)

## 개요

HS8SA-WiFi는 Hyman이 미국 시장을 위해 출시한 WiFi 스마트 연기 경보기입니다. 광전식 연기 감지 기술을 사용하고 WiFi 네트워킹 및 독립적인 작동 듀얼 모드를 지원하며 UL 217 인증을 통과했습니다. 주택, 아파트 및 기타 주거 장소의 화재 안전 보호에 적합합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | IEEE 802.11 b/g/n(2.4GHz) |
| 센서 유형 | 적외선 광전 |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | 배터리 구동 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤ 90% 상대습도 |
| 설치방법 | 천장 설치 |

## 기능

- 느린 화상 및 개방형 화염에서 발생하는 연기를 감지하는 광전식 연기 감지기
- WiFi 네트워킹 원격 알람 푸시
- 독립 작동 모드(네트워크 연결이 끊어져도 로컬 사운드 및 조명 알람이 계속 발생할 수 있음)
- 85dB의 높은 소리와 빛 알람
- 배터리 부족 알림
- 셀프 테스트 기능(원클릭 테스트)
- 변조 방지 경보
-원격 음소거 지원

## 인증기준

- UL 217(화재 경보기에 대한 미국 표준)
-FCC
- NFPA 72 준수
