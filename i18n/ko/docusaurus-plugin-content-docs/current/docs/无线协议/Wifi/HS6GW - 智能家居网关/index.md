#HS6GW - 스마트 홈 게이트웨이

## 개요

HS6GW는 Heiman이 출시한 스마트 홈 게이트웨이로 WiFi + 이더넷 업링크, 내장 Zigbee 코디네이터를 지원하고 최대 128개의 Zigbee 하위 장치를 관리할 수 있습니다. 인터넷에 연결하여 원격접속과 클라우드 서비스를 구현하는 것이 하이만 스마트홈 시스템의 핵심 허브입니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 업링크 통신 | IEEE 802.11 b/g/n(2.4GHz WiFi) + 10/100M 이더넷 |
| 다운링크 통신 | Zigbee 3.0(코디네이터 모드) |
| 최대 하위 장치 수 | 128 |
| 전원 공급 방식 | DC 5V / 1A (마이크로 USB) |
| 대기전력 소모 | 1W 이하 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤ 90% 상대습도 |
| 표시 등 | WiFi 상태/전원/지그비 네트워크 상태 |
| 재설정 방법 | 버튼 재설정 + 길게 눌러 공장 설정 복원 |

## 기능

- 듀얼 업링크: WiFi 또는 이더넷 유선 연결, 네트워크 환경에 따른 유연한 선택
- 원격 접속 지원: 클라우드 앱을 통해 언제 어디서나 기기 상태 확인
- 로컬 장면 연결: LAN 내 장비의 자동 작동 지원(연결이 끊어진 경우에도 계속 사용 가능)
-OTA 펌웨어 업그레이드
- Tuya / Smart Life 플랫폼 지원
- 최대 128개의 Zigbee 하위 장치 관리
-Zigbee 네트워크 자가 치유 및 라우팅 최적화 지원

## 인증기준

-CE/FCC
- RoHS
