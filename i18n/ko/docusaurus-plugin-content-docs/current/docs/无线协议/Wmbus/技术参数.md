# 기술적인 매개변수

다음은 Heimann Wireless M-Bus(wM-Bus) 장비의 일반적인 기술 매개변수입니다.

## 무선 매개변수

| 매개변수 | 가치 |
|------|-----|
| 프로토콜 표준 | EN 13757-4(무선 M-Bus 물리 계층) |
| 작동 주파수 대역 | 868MHz(유럽 SRD 주파수 대역, 868.0~868.6MHz) |
| 변조 방식 | FSK |
| 전송률 | 4.8kbps / 16.384kbps / 100kbps (T 모드 / S 모드 / R 모드) |
| 네트워크 토폴로지 | 스타(기기 → 컬렉터/게이트웨이) |
| 통신거리 | 실내 약 100~300m, 개방형 약 500~1000m |
| 전력 전송 | ≤ 25mW(+14dBm) |

## 작업 모드

| 모드 | 평가 | 설명 |
|------|------|------|
| S 모드 | 16.384kbps | 표준 모드, 저전력 소비, 배터리 구동 기기에 적합 |
| T 모드 | 100kbps | 대규모 트래픽 데이터 시나리오에 적합한 고속 모드 |
| R 모드 | 4.8kbps | 협대역 모드, 장거리 전송 |

## 전기 및 전력 소비

| 매개변수 | 가치 |
|------|-----|
| 전원 공급 방식 | 리튬 배터리 전원 공급 장치(ER18505/ER26500 및 기타 산업용 리튬 배터리) |
| 배터리 수명 | 6~10년(보고빈도에 따라 대표값은 1회/일) |
| 대기전력 소모 | 마이크로암페어 수준, 깊은 잠 |
| 수신감도 | 대략. -105dBm(S 모드) |

## 환경 매개변수

| 매개변수 | 가치 |
|------|-----|
| 작동 온도 | -20°C ~ +60°C(산업용 등급) |
| 작동 습도 | ≤ 95% RH(비응결) |
| IP 보호 수준 | IP65 ~ IP68 (특정 모델에 따라, 실외 장비에 적용 가능) |

| 레이어 | 표준 |
|----|------|
| 물리층 | EN 13757-4 |
| 데이터링크 계층 | EN 13757-4(무선 M-Bus 프레임 형식) |
| 애플리케이션 계층 | EN 13757-3(M-버스 애플리케이션 계층) |
| 암호화 방법 | AES-128(선택적 암호화) |
