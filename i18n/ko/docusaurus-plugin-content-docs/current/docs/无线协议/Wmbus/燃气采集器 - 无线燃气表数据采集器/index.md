# 가스 수집기 - 무선 가스 계량기 데이터 수집기

## 개요

가스 수집기는 하이만이 스마트 가스전용으로 출시한 wM-Bus 무선 데이터 수집 장치입니다. 868MHz 주파수 대역(EN 13757 표준)에서 작동하며 가스 계량기 측정 데이터를 원격으로 읽고 중앙 검침 시스템에 업로드하는 데 사용됩니다. 배터리 수명이 6~10년인 산업용 등급 리튬 배터리로 구동되는 이 제품은 아파트, 주거 지역, 상업용 건물과 같은 중앙 집중식 검침 시나리오에 적합합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | wM-버스(EN 13757) |
| 작동 주파수 대역 | 868MHz(유럽 SRD 대역) |
| 전송률 | 4.8kbps/16.384kbps/100kbps(T/S/R 모드) |
| 통신거리 | 실내 약 100~300m, 개방형 약 500~1000m |
| 전원 공급 방식 | 산업용 리튬전지(ER18505 / ER26500) |
| 배터리 수명 | 6~10년(보통 1일 1회 보고) |
| 작동 온도 | -20°C ~ +60°C |
| 작동 습도 | ≤ 95% 상대습도 |
| 보호 수준 | IP65 ~ IP68 (설치 환경에 따라 다름) |
| 데이터 암호화 | AES-128(선택 사항) |

## 기능

-wM-Bus S/T/R 세 가지 작동 모드 지원
- 가스 계량기 사용 데이터를 자동으로 수집하고 정기적으로 보고합니다.
- 낮은 전력 소비, 깊은 수면, 긴 배터리 수명
- 통신 보안을 보장하는 AES-128 데이터 암호화
-원격 펌웨어 업그레이드 지원
- 단절 재접속 및 데이터 재전송 메커니즘
- EN 13757-3 애플리케이션 계층 프로토콜 지원
- 중앙 집중식 업로드를 위해 HS1-CM 컬렉터/HS1-GW 게이트웨이와 함께 사용할 수 있습니다.

## 인증기준

- EN 13757-3 (M-Bus 애플리케이션 레이어)
-CE/레드
