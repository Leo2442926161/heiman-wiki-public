# wM-Bus(무선 계량 버스)

wM-Bus(Wireless Meter-Bus)는 유럽 표준(EN 13757)에서 정의한 무선 검침 프로토콜입니다. 수도계량기, 가스계량기, 전기계량기, 열량계 등 에너지 계량 장비의 원격 데이터 수집에 널리 사용됩니다.

## 기술 사양

| 특징 | 설명 |
|------|------|
| 표준 | EN 13757 |
| 작동 주파수 대역 | 868MHz(EU 표준) |
| 전송거리 | 개방형 환경 100-500미터 |
| 전력 소비 | 매우 낮은 배터리 수명, 최대 10년 |
| 주요 용도 | 에너지 측정, 중앙 집중식 검침 |

## 하이만과 wM-Bus

Heimann은 wM-Bus 기술을 활용하여 스마트 가스 및 스마트 수자원 분야에 대한 솔루션을 제공합니다. wM-Bus 제품은 주로 중대형 산업 및 상업 프로젝트에 사용되며 Haiman 스마트 소방 플랫폼과 함께 사용됩니다.

## 적용 가능한 시나리오

- 중앙집중식 아파트/주거용 가스검침
- 상업용 건물 에너지 소비 모니터링
- 스마트시티 인프라

> **참고**: wM-Bus는 주로 Heiman의 가스 및 화재 보호 프로젝트 솔루션에 사용되는 수직 산업 프로토콜이며 일반 홈 스마트 홈 시나리오에는 적합하지 않습니다.
