# L1-M 침수 감지기(물질)

L1-M은 하이만에서 출시한 **Mater over Thread** 침수 감지기입니다. 누수나 누수를 프로브를 통해 감지하여 수도관 파열로 인한 재산 손실을 효과적으로 방지할 수 있습니다.

## 제품 개요

- **제품 유형**: 침수 감지 센서
- **통신 프로토콜**: 스레드보다 중요한 문제
- **감지 방법**: 프로브 접촉 감지
- **일반적인 시나리오**: 주방, 욕실, 세탁실, 지하 누수 감지

## 기술적인 매개변수

| 매개변수 | 가치 |
|------|-----|
| 탐지 방법 | 스테인레스 스틸 프로브 접촉 유형 |
| 통신 프로토콜 | 스레드보다 중요한 문제(IEEE 802.15.4) |
| 작동 주파수 대역 | 2.4GHz |
| 전원 공급 방식 | 배터리 구동 |
| 경보방식 | 부저 + 앱 푸시 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% RH(응결 없음) |

## 기능

- 누수/고인수 실시간 감지
- 부저 현장 알람
- 앱 원격 알람 푸시
- 배터리 부족 알림
- 프로브 부식 방지 설계
- 다중 생태학적 크로스 플랫폼 상호 운용성 문제
