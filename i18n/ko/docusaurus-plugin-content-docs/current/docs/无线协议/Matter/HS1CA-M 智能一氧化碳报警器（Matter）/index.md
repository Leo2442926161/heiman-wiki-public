# HS1CA-M 지능형 일산화탄소 경보기(물질)

HS1CA-M은 전기화학 센서 기술을 사용하고 Matter over Thread 교차 생태학적 상호 연결을 지원하는 세계 최초의 Matter 인증 일산화탄소 경보기**입니다.

## 제품 개요

- **제품 유형**: 지능형 일산화탄소 경보기
- **통신 프로토콜**: 스레드보다 중요한 문제
- **인증기준**: EN 50291
- **일반적인 시나리오**: 주택, 아파트 및 기타 장소에서의 CO 누출 경고

## 기술적인 매개변수

| 매개변수 | 가치 |
|------|-----|
| 센서 유형 | 전기화학 CO 센서 |
| 통신 프로토콜 | 스레드보다 중요한 문제(IEEE 802.15.4) |
| 작동 주파수 대역 | 2.4GHz |
| 감지 범위 | 0~1000ppm |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | DC 3V (CR123A 리튬 배터리) |
| 배터리 수명 | 약 3년 |
| 센서 수명 | 보통 7년 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% RH(응결 없음) |

## 기능

- 고정밀 전기화학 CO 센서
- 농도 등급 경보(소리 및 빛 프롬프트)
- 85dB의 높은 소리와 빛 알람
- 배터리 부족 알림
- 센서 수명 만료 알림
- 셀프 테스트 기능
- 다중 생태학적 크로스 플랫폼 상호 운용성 문제

## 인증

- EN 50291 인증
- 사항인증
- 홈어시스턴트 인증과 함께 작동

Apple HomePod mini, Google Nest Hub 2세대, Amazon Echo 4세대 등과 같은 스레드 경계 라우터가 필요합니다.
