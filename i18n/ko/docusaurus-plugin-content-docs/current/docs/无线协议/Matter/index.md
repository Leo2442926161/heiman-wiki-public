#물질

Matter(이전의 Project CHIP)는 브랜드 장벽을 허물고 생태계 간 상호 운용성을 달성하는 것을 목표로 CSA 연결 표준 연합(CSA Connection Standards Alliance)에서 출시한 스마트 홈 상호 연결 표준입니다.

## 헤이먼 앤 매터

Hyman은 **세계 최초의 Matter 인증 일산화탄소 경보기**를 출시하는 Matter 프로토콜의 얼리 어답터였습니다. Hyman's Matter 장치는 모두 **Matter over Thread** 솔루션을 사용합니다.

## 기술적 특징

| 특징 | 설명 |
|------|------|
| 기본 프로토콜 | 스레드(저전력 메시 네트워크) |
| 네트워크 토폴로지 | 메시 자가 치유 네트워크, 장치가 많을수록 적용 범위가 넓어집니다 |
| 전력 소비 | 매우 낮음, 배터리 전원으로 1~3년 동안 작동 가능 |
| 보안 | AES-128-CCM 암호화 지원, 장치 인증 필요 |
| 상호 운용성 | Apple HomeKit, Google Home, Amazon Alexa, Home Assistant 등 |

## 보더 라우터 요구 사항

Thread 장치를 통한 Matter를 사용하려면 Thread 경계 라우터가 필요합니다.

- 애플 홈팟 미니 / 홈팟 2세대 / 애플 TV 4K
- Google Nest Hub 2세대 / Nest Hub Max
-아마존 에코 4세대
- 홈어시스턴트 그린 + ZBT-2 연결
- 호미 프로

## Hyman Matter 인증 제품

- 스마트 연기 경보기(미국 버전/EU 버전)
- 지능형 일산화탄소 경보기(미국 버전/EU 버전)
- 인체 적외선 센서 M1-M
- 침수 감지기 L1-M
- 온도 및 습도 센서 H1-M
