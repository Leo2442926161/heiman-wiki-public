# HS1DS-M - 지능형 도어 센서(스레드보다 물질)

## 개요

HS1DS-M은 Heiman 스마트 도어 센서의 Matter over Thread 버전입니다. 독점 게이트웨이 없이도 Apple Home, Google Home, Amazon Alexa, Home Assistant 등과 같은 모든 Matter 호환 플랫폼에 대한 기본 액세스를 지원합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 스레드에 대한 문제 |
| 전원 공급 방식 | CR1632 버튼 배터리 |
| 배터리 수명 | 약 2~3년 |
| 설치거리 | 10mm 이하 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% 상대습도 |

## 기능

- 문 및 창문 개폐 상태 실시간 감지
- 탬퍼 알람(Tamper)
- 배터리 부족 알림
- 크로스 플랫폼 호환: Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Thread Mesh 네트워크, 낮은 대기 시간의 로컬 제어
- 전용 게이트웨이가 필요하지 않으며 스레드 경계 라우터가 필요합니다.
