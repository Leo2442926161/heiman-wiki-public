# H1-M 온도 및 습도 센서(물질)

H1-M은 하이만에서 출시한 **Mater over Thread** 온습도 센서로, 주변 온도와 상대습도를 실시간으로 모니터링하고 스마트 홈 자동화 연동을 지원합니다.

## 제품 개요

- **제품 유형**: 온도 및 습도 센서
- **통신 프로토콜**: 스레드보다 중요한 문제
- **감지 매개변수**: 온도, 상대습도
- **일반적인 시나리오**: 가정, 사무실, 와인 저장고 등의 환경 모니터링

## 기술적인 매개변수

| 매개변수 | 가치 |
|------|-----|
| 온도 정확도 | ±0.3°C |
| 습도 정확도 | ±3% 상대습도 |
| 통신 프로토콜 | 스레드보다 중요한 문제(IEEE 802.15.4) |
| 작동 주파수 대역 | 2.4GHz |
| 전원 공급 방식 | CR2 리튬 배터리 |
| 배터리 수명 | 약 2~3년 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% RH(응결 없음) |

## 기능

- 실시간 온도 및 습도 모니터링
- 고정밀 센싱(온도 ±0.3°C, 습도 ±3% RH)
- 배터리 부족 알림
- 다중 생태학적 크로스 플랫폼 상호 운용성 문제
