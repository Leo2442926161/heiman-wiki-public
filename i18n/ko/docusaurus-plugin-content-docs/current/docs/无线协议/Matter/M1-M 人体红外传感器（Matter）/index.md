# M1-M 인체 적외선 센서(물질)

M1-M은 하이만에서 출시한 **Mater over Thread** 인체 적외선 센서입니다. PIR 감지 및 조명 감지 기능을 통합하며 스마트 홈 자동화 시나리오에 적합합니다.

## 제품 개요

- **제품 유형**: 인체 적외선 센서
- **통신 프로토콜**: 스레드보다 중요한 문제
- **감지 방법**: 수동 적외선(PIR) + 조명 감지
- **전형적인 시나리오** : 사람이 오면 조명이 켜짐, 보안연동, 환경자동화

## 기술적인 매개변수

| 매개변수 | 가치 |
|------|-----|
| 감지 각도 | 110°(벽걸이) / 360°(천장) |
| 감지 거리 | 7m 이하(25°C) |
| 조명 범위 | 0~1200 럭스 |
| 통신 프로토콜 | 스레드보다 중요한 문제(IEEE 802.15.4) |
| 작동 주파수 대역 | 2.4GHz |
| 전원 공급 방식 | DC 3V 리튬 배터리 |
| 배터리 수명 | 약 1년(Matter 모드) |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% RH(응결 없음) |

## 기능

- 모바일 인간 감지
- 주변 조명 감지
- 감지 거리 7m 이하
- 벽이나 천장 설치 지원
- 다중 생태학적 크로스 플랫폼 상호 운용성 문제
