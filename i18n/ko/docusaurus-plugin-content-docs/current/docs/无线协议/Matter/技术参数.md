# 기술적인 매개변수

Hyman Matter 장비의 일반 사양은 다음과 같습니다.

## 무선 매개변수

| 매개변수 | 가치 |
|------|-----|
| 기본 프로토콜 | 스레드(IEEE 802.15.4) |
| 작동 주파수 대역 | 2.4GHz(ISM 대역) |
| 애플리케이션 계층 표준 | 물질 1.x |
| 네트워크 토폴로지 | 메시 자가 치유 네트워크 |
| 암호화 방법 | AES-128-CCM |

## 하드웨어 요구사항

| 프로젝트 | 설명 |
|------|------|
| 보더 라우터 | Apple HomePod mini, Apple TV 4K, Google Nest Hub 2세대 등 필수 |
| 모바일 단말기 지원 | iOS/Android(다양한 생태앱을 통한 네트워크 배포) |
| 생태간 상호 운용성 | Apple HomeKit, Google Home, Amazon Alexa, 홈 어시스턴트, SmartThings |

## 전기 및 전력 소비

| 매개변수 | 가치 |
|------|-----|
| 전원 공급 방식 | 배터리 구동(CR123A / CR2450 등) |
| 배터리 수명 | 약 1~3년(기기 종류 및 사용 빈도에 따라 다름) |
| 대기전력 소모 | 매우 낮음, 수면 깨우기 메커니즘 지원 |

## 환경 매개변수

| 매개변수 | 가치 |
|------|-----|
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤ 95% RH(비응결) |
