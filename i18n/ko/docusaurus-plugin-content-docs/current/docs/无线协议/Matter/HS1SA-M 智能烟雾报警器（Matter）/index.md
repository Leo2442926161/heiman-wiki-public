# HS1SA-M 지능형 연기 경보기(물질)

HS1SA-M은 하이만(Hyman)이 출시한 **Mater over Thread** 스마트 연기 경보기입니다. 광전 연기 감지 기술을 사용하고 생태 간 스마트 홈 플랫폼 상호 연결을 지원합니다.

## 제품 개요

- **제품 유형**: 스마트 연기 경보기
- **통신 프로토콜**: 스레드보다 중요한 문제
- **인증기준**: EN 14604
- **일반적인 시나리오**: 주택, 아파트, 호텔 등의 화재 경고

## 기술적인 매개변수

| 매개변수 | 가치 |
|------|-----|
| 센서 유형 | 적외선 광전 |
| 통신 프로토콜 | 스레드보다 중요한 문제(IEEE 802.15.4) |
| 작동 주파수 대역 | 2.4GHz |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | DC 3V (CR123A 리튬 배터리) |
| 배터리 수명 | 약 3년 |
| 작동 온도 | -10°C ~ +50°C |
| 작동 습도 | ≤95% RH(응결 없음) |

## 기능

- 느린 화상 및 화염 연기를 감지하는 광전식 연기 감지기
- 85dB의 높은 소리와 빛 알람
- 배터리 부족 알림
- 셀프 테스트 기능(원클릭 테스트)
- 변조 방지 경보
- 원격 음소거
- 다중 생태학적 크로스 플랫폼 상호 운용성 문제

## 인증

- EN 14604 인증
- 사항인증
