# 인증기준

HS1SA-M은 다음 인증을 통과했습니다.

- **EN 14604** — 연기 경보기에 대한 유럽 표준, [EN 유럽 표준](../../../../ApprovalStandards/EN European Standard.md) 참조
- **CE 마크** — EU 적합성 선언, [CE EU 적합성](../../../../certification-standards/CE EU-Conformity.md) 참조
- **Matter 인증** — 스마트 홈의 크로스 플랫폼 상호 연결을 보장하기 위해 CSA Alliance Matter 인증을 통과했습니다.
- **레드닷 디자인 어워드** — 레드닷 제품 디자인 어워드
