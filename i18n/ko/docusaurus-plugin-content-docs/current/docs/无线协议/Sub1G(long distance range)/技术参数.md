# 기술적인 매개변수

다음은 Hyman Sub-1G 장거리 시리즈 제품의 일반적인 기술 매개 변수입니다.

## 무선 매개변수

| 매개변수 | 가치 |
|------|-----|
| 작동 주파수 대역 | 433MHz/868MHz/915MHz(시장 지역에 따라 다름) |
| 변조 방식 | FSK / OOK |
| 전송률 | 약 10~100kbps(비디오 세그먼트 및 구성) |
| 네트워크 토폴로지 | 스타/지점간/인터넷 네트워킹 |
| 통신거리 | 개방형 환경에서 약 1~3km(영상부분 및 전원) |
| 네트워크 용량 | 단일 그룹은 최대 50개 이상의 노드를 지원합니다 |

## 지역별 주파수 대역 할당

| 지역 | 공통 주파수 대역 |
|------|----------|
| 중국 | 433MHz |
| 유럽 ​​| 868MHz |
| 북미 | 915MHz |

## 전기 및 전력 소비

| 매개변수 | 가치 |
|------|-----|
| 전원 공급 방식 | 배터리 구동(CR123A / AA 리튬 배터리) |
| 배터리 수명 | 약 3~5년 (보고빈도에 따라 다름) |
| 전력 전송 | ≤ 10mW(+10dBm), SRD 규정 준수 |
| 대기전력 소모 | 마이크로암페어 수준, 매우 낮음 |
| 수신감도 | 약 -110dBm ~ -120dBm |

## 환경 매개변수

| 매개변수 | 가치 |
|------|-----|
| 작동 온도 | -10°C ~ +55°C |
| 작동 습도 | ≤ 95% RH(비응결) |
