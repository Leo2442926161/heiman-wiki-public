# S1 시리즈 - 독립형 연기 경보기(Sub-1G)

## 개요

S1 시리즈는 Hyman이 출시한 독립형 Sub-1G 연기 경보기입니다. 433MHz/868MHz 주파수 대역에서 작동하고 광전 연기 감지 기술을 사용하며 EN 14604 인증을 통과했습니다. 게이트웨이나 인터넷이 필요하지 않습니다. 장치는 Sub-1G 무선 주파수를 통해 직접 상호 연결됩니다. 어떤 장치가 알람을 트리거하면 동일한 그룹의 모든 장치가 동시에 알람을 발생시킵니다. 건물, 쇼핑몰, 호텔 등 대규모 화재 연계 시나리오에 적합합니다.

## 기술 사양

| 매개변수 | 가치 |
|------|-----|
| 통신 프로토콜 | 433MHz/868MHz Sub-1G(시장 지역에 따라 다름) |
| 센서 유형 | 적외선 광전 |
| 알람 음압 | ≥85dB(3미터) |
| 전원 공급 방식 | 배터리 구동(CR123A / AA 리튬 배터리) |
| 배터리 수명 | ≥ 5년 |
| 통신거리 | 개방된 환경에서 약 1~3km |
| 네트워크 용량 | 단일 그룹은 최대 50개 이상의 노드를 지원합니다 |
| 상호접속방식 | 무선 무선 주파수 상호 연결(게이트웨이 필요 없음) |
| 작동 온도 | -10°C ~ +55°C |
| 작동 습도 | ≤ 95% 상대습도 |
| 설치방법 | 천장 설치 |

## 시리즈 모델

| 모델 | 설명 |
|------|------|
| S1-SA | 기본형, 무선연결 지원 |
| S1-SA-P | 카메라 기능이 있는 버전 |
| S1-SA-T | 고온 연결 버전 |
| S1-SA-RF | RF 상호 연결 유형 |

## 기능

- 느린 화상 및 개방형 화염에서 발생하는 연기를 감지하는 광전식 연기 감지기
- 무선 상호 연결 알람: 동일한 네트워크에 있는 모든 장치에 의해 트리거되며 모든 장치에서 동시에 알람이 울리고 켜집니다.
- 독립적인 작업 모드: 게이트웨이 없음, 인터넷 필요 없음, 장치 간 직접 통신
- 85dB의 높은 소리와 빛 알람
- 배터리 부족 알림
- 셀프 테스트 기능(원클릭 테스트)
- 변조 방지 경보
-원격 음소거 지원
- 초저전력 소모 설계, 배터리 수명 5년 이상

## 인증기준

- EN 14604(화재 경보기에 대한 유럽 표준)
-CE/레드
- CPR(EU 건설제품 규정)

## 적용 가능한 액세서리

- S1-RM 무선중계기(통신거리 연장)
- S1-RC 원격 제어(테스트/음소거/리셋)
