# Sub-1G(장거리 통신)

Sub-1G는 1GHz 이하 주파수 대역에서 동작하는 무선 통신 기술이다. 전송 거리가 매우 길고 전력 소비가 매우 낮은 특성을 가지고 있습니다. 대규모 커버리지가 필요한 IoT 시나리오에 매우 적합합니다.

## 기능

| 특징 | 설명 |
|------|------|
| 작동 주파수 대역 | 433MHz/868MHz(EU)/915MHz(미국) |
| 전송거리 | 개방형 환경에서 최대 1-3km |
| 침투능력 | 2.4GHz 이상, 벽 장애물까지 강력하게 침투 |
| 전력 소비 | 매우 낮은 배터리 수명은 5~10년에 달할 수 있습니다 |
| 전송률 | 낮음(소규모 데이터 볼륨 시나리오에 적합) |

## 애플리케이션 시나리오

- **독립적인 연기/일산화탄소 경보**: 건물 내 여러 층에 걸쳐 상호 연결됨
- **대형 건물 화재 예방 시스템**: 쇼핑몰, 호텔, 학교
- **스마트 시니어 생활 커뮤니티**: 단위 간 긴급전화
- **Sub-1G + Wi-Fi Gateway**: Sub-1G를 통해 서브디바이스와 통신하며, 게이트웨이는 클라우드에 연결됩니다.

## 하이만 Sub-1G 제품

Hyman은 S1 시리즈 독립형 화재 경보기를 포함하여 Sub-1G 무선 화재 방지 제품에 대한 다년간의 경험을 보유하고 있습니다. 이러한 장치는 인터넷 없이도 장치 간 상호 연결을 실현할 수 있습니다.

> **참고**: Sub-1G는 Heiman의 화재 및 보안 제품 라인에 주로 사용되며, 스마트 홈 제품 라인은 Zigbee/Matter가 주로 사용됩니다.
