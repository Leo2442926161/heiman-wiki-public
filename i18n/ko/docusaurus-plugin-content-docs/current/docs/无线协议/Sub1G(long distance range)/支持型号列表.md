#지원 모델 목록

Haiman Sub-1G 장거리 시리즈 제품은 주로 대규모 독립 네트워킹 시나리오를 위해 설계되었으며 게이트웨이 없이 장치 간의 상호 연결 및 상호 운용성을 실현할 수 있습니다.

## 독립형 연기 경보기(S1 시리즈)

| 모델 | 제품명 | 설명 |
|------|----------|------|
| S1-SA | 독립형 화재경보기(Sub-1G) | 기본형, 무선연결 지원 |
| S1-SA-P | 독립형 화재경보기(Sub-1G) | 카메라 기능이 있는 버전 |
| S1-SA-T | 독립형 연기경보기(Sub-1G) | 고온 연결 버전 |
| S1-SA-RF | 독립형 화재경보기(RF 상호연결형) | RF 네트워킹 버전 |

## 연결된 액세서리

| 모델 | 제품명 | 설명 |
|------|----------|------|
| S1-RM | 무선중계기(Sub-1G) | 통신거리 연장 |
| S1-RC | 원격제어(Sub-1G) | 테스트/음소거/재설정 |

## 기능

- **독립적인 작업**: 게이트웨이 없음, 인터넷 필요 없음, 장치 간 직접 무선 연결
- **상호 연결된 알람**: 모든 장치에서 알람이 발생하며, 동일한 그룹에 있는 모든 장치에 동시에 알람이 발생합니다.
- **초장거리**: 개방된 환경에서 통신 거리는 1~3km에 달할 수 있습니다.
- **초저전력 소비**: 배터리로 구동되며 배터리 수명은 최대 3~5년입니다.
