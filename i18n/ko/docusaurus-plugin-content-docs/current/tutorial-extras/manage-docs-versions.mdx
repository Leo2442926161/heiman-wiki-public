# 문서 버전 관리

하이만 테크놀로지 Wiki는 다중 버전 문서 관리를 지원하여 제품 펌웨어 버전에 맞는 기술 문서에 접근할 수 있도록 보장합니다.

## 버전 비교

|------------|-------------------|-------------|
| 2.0        | Matter 시리즈      | 2025-06     |
| 1.0        | Zigbee 시리즈      | 2024-01     |

## 다른 버전 보기

상단 탐색 모음의 버전 드롭다운 메뉴를 사용하여 다른 문서 버전 간에 전환할 수 있습니다.

## 버전 업데이트

제품 업데이트에 맞춰 문서를 지속적으로 업데이트하고 있습니다. 최신 버전을 확인하려면 정기적으로 방문해 주십시오.
