# 솔루션 배포

하이만 테크놀로지는 엔드투엔드 스마트 보안 솔루션 배포 가이드를 제공합니다.

## 통합 프로세스

하이만 제품은 로컬 단독 배포 및 클라우드 플랫폼 통합을 포함한 여러 배포 방식을 지원합니다.

### 로컬 배포

```bash
# 하이만 스마트 화재 게이트웨이 구성
heiman-cli configure --gateway HS1GW --mode local

# 감지기 기기 등록
heiman-cli register --device HS1SA --gateway HS1GW
```

### 클라우드 플랫폼 통합

하이만 제품은 주요 IoT 플랫폼과 통합됩니다:

- China Telecom Tianyi
- China Mobile OneNET
- Huawei HiLink
- Alibaba Cloud IoT
- Tuya Smart

## Matter 프로토콜 지원

하이만 Matter 시리즈 제품은 다음와 호환됩니다:

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

크로스 브랜드, 크로스 에코시스템 상호 연결을 실현합니다.

## 배포 검증

배포 후 하이만 스마트 화재 APP 또는 관리 플랫폼을 사용하여 기능 검증 및 실시간 모니터링을 수행하십시오.
