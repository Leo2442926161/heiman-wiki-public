## 이 사이트 소개

하이만 테크놀로지 Wiki는 파트너, 개발자 및 고객에게 원스톱 기술 리소스를 제공하기 위해 마련되었습니다:

- **제품** — 스마트 화재 안전, 스마트 가스 안전, 스마트 노인 케어, 스마트 센서
- **솔루션** — 업계 수준의 스마트 보안 솔루션
- **기술 문서** — API 통합 가이드, SDK 사용법, 설치 설명서
- **업계 인사이트** — 스마트 보안 동향 및 정책 해설
