## Bu Site Hakkında

Heiman Teknoloji Wiki, iş ortaklarına, geliştiricilere ve müşterilere tek duraktan teknik kaynak sağlamaya adanmıştır:

- **Ürünler** — Akıllı Yangın Güvenliği, Akıllı Gaz Güvenliği, Akıllı Yaşlı Bakımı, Akıllı Sensörler
- **Çözümler** — Sektör düzeyinde akıllı güvenlik çözümleri
- **Teknik Dokümanlar** — API entegrasyon kılavuzları, SDK kullanımı, kurulum kılavuzları
- **Sektör İçgörüleri** — Akıllı güvenlik trendleri ve politika yorumları
