# HS1DS-Z kapı sensörü (Z-Wave)

HS1DS-Z, Hyman tarafından piyasaya sürülen bir **Z-Wave** kapı sensörü sensörüdür. Manyetik dilli anahtarlar aracılığıyla kapı ve pencerelerin açılma ve kapanma durumunu algılar ve Z-Wave Plus standardıyla uyumludur.

## Ürüne Genel Bakış

- **Ürün tipi**: Kapı sensörü
- **İletişim Protokolü**: Z-Wave Plus
- **Algılama yöntemi**: Manyetik anahtar (küçük anahtar)
- **Tipik senaryolar**: Kapı ve pencere durum izleme, güvenlik dağıtımı ve akıllı bağlantı

## Teknik parametreler

| Parametre | Değer |
|----------|-----|
| Algılama yöntemi | Küçük indükleme anahtarı (küçük indükleme anahtarı) |
| İletişim Protokolü | Z-Wave Plus (ITU-T G.9959) |
| Çalışma frekansı bandı | 868,42 MHz (AB) / 908,42 MHz (ABD) |
| Güç kaynağı yöntemi | CR1632 düğme pil |
| Pil ömrü | Yaklaşık 2~3 yıl |
| Kurulum izni | ≤10 mm |
| Şifreleme yöntemi | S2 Kimliği Doğrulanmış / S2 Kimliği Doğrulanmamış |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤%90 bağıl nem (yoğuşmasız) |

## Özellikler

- Kapı ve pencere açılma/kapanma durumunun gerçek zamanlı tespiti
- Sabotaj alarmı (Sabotaj)
- Düşük pil hatırlatıcısı
-Sahne bağlantısını destekler (kapıyı açma, ışıkları açma, evden çıkarken devreye alma)
- Kurulumu kolay (3M yapışkan bant veya vidayla sabitleme)
- Z-Wave S2 güvenli şifreleme
- OTA ürün yazılımı yükseltmesini destekleyin
