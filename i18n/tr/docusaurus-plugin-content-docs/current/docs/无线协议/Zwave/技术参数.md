# Teknik parametreler

Aşağıda Hyman Z-Wave cihazlarının genel teknik özellikleri yer almaktadır.

## Kablosuz parametreler

| Parametre | Değer |
|----------|-----|
| Protokol standartları | Z-Wave Plus (ITU-T G.9959) |
| Çalışma frekans bandı | 800~900 MHz (bölgeye bağlı olarak: Avrupa düzenlemesi 868,42 MHz, ABD düzenlemesi 908,42 MHz) |
| İletim hızı | 100 kbps'ye kadar |
| Ağ topolojisi | Kendi kendini onaran ağ ağı (kaynak yönlendirme) |
| Düğüm kapasitesi | Tek bir ağda maksimum 232 düğüm |
| İletişim mesafesi | Yaklaşık 30 metre kapalı alanda (düğümler arasında), tüm ev Mesh kapsama alanı |
| Şifreleme yöntemi | AES-128 (S2 Kimliği Doğrulanmış / S2 Kimliği Doğrulanmamış) |

## Bölgesel frekans bandı

| Bölge | Merkez Frekansı |
|----------|----------|
| Avrupa | 868,42 MHz |
| Kuzey Amerika | 908,42 MHz |
| Diğer alanlar | Yerel düzenlemelere tabidir |

## Elektrik ve Güç Tüketimi

| Parametre | Değer |
|----------|-----|
| Güç kaynağı yöntemi | Pil güç kaynağı (CR123A/AA) veya şebeke güç kaynağı |
| Pil ömrü | Sensörler için yaklaşık 2~3 yıl |
| İletim gücü | ≤ 1 mW (SRD yönetmeliklerine uygun) |
| Hassasiyet alma | Yaklaşık -100 dBm |

## Ortam parametreleri

| Parametre | Değer |
|----------|-----|
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤ %90 bağıl nem (yoğuşmasız) |

## Ağ özellikleri

| Özellikler | Açıklama |
|------|------|
| SmartStart | Destek (700 serisi), ağa katılmak için QR kodunu tarayın |
| S2 Güvenliği | Üç güvenlik düzeyini destekler: Şifrelenmemiş, S2 Kimliği Doğrulanmamış, S2 Kimliği Doğrulanmış |
| CC'yi içerir | Güvenlik CC, Güvenlik 2 CC, Firmware Güncellemesi CC, Temel CC, Pil CC, Bildirim CC ve daha fazlası |
