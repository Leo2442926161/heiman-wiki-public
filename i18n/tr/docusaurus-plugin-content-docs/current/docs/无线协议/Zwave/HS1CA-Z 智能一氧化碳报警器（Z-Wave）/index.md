# HS1CA-Z Akıllı Karbon Monoksit Alarmı (Z-Wave)

HS1CA-Z, Hyman tarafından başlatılan bir **Z-Wave** akıllı karbon monoksit alarmıdır. Elektrokimyasal sensör teknolojisini kullanır ve Z-Wave Plus standardıyla uyumludur.

## Ürüne Genel Bakış

- **Ürün Tipi**: Akıllı Karbonmonoksit Alarmı
- **İletişim Protokolü**: Z-Wave Plus
- **Sertifika Standardı**: EN 50291
- **Tipik Senaryo**: Evlerde, apartmanlarda ve diğer yerlerde CO kaçağı uyarısı

## Teknik parametreler

| Parametre | Değer |
|----------|-----|
| Sensör Tipi | Elektrokimyasal CO Sensörü |
| İletişim Protokolü | Z-Wave Plus (ITU-T G.9959) |
| Çalışma frekansı bandı | 868,42 MHz (AB) / 908,42 MHz (ABD) |
| Algılama aralığı | 0~1000 sayfa/dakika |
| Alarm ses basıncı | ≥85 dB (3 metre) |
| Güç kaynağı yöntemi | Pille çalışan |
| Sensör ömrü | Tipik 7 yıl |
| Şifreleme yöntemi | S2 Kimliği Doğrulanmış / S2 Kimliği Doğrulanmamış |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤%90 bağıl nem (yoğuşmasız) |

## Özellikler

- Yüksek hassasiyetli elektrokimyasal CO sensörü
- Konsantrasyon sınıflandırma alarmı
- 85 dB yüksek sesli ve ışıklı alarm
- Düşük pil hatırlatıcısı
- Sensör ömrünün sona ermesi hatırlatıcısı
- Kendi kendine test fonksiyonu
- Z-Wave S2 güvenli şifreleme
- OTA ürün yazılımı yükseltmesini destekleyin

## Sertifika

- EN 50291 sertifikalı
- Z-Wave Plus sertifikalı
- S2 güvenlik sertifikası
