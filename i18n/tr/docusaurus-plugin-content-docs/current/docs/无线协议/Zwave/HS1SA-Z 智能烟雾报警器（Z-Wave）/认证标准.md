# Sertifikasyon Standartları

HS1SA-Z aşağıdaki sertifikaları geçmiştir:

- **EN 14604** — Duman alarmlarına yönelik Avrupa standardı, bkz. [EN Avrupa Standardı](../../../../ApprovalStandards/EN Avrupa Standardı.md)
- **Z-Wave S2 Sertifikasyonu** — S2 güvenlik çerçevesini destekler, bkz. [Diğer Sertifikalar](../../../Certification Standards/Other Certifications.md)
- **CE** — Avrupa Birliği Uygunluk Beyanı
