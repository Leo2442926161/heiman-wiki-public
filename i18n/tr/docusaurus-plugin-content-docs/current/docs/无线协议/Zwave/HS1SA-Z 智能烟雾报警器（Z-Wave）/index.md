# HS1SA-Z Akıllı Duman Alarmı (Z-Wave)

HS1SA-Z, Hyman tarafından piyasaya sürülen bir **Z-Wave** akıllı duman alarmıdır. Fotoelektrik duman algılama teknolojisini kullanır, Z-Wave Plus standardıyla uyumludur ve Avrupa ve Amerika akıllı ev pazarlarına uygundur.

## Ürüne Genel Bakış

- **Ürün Tipi**: Akıllı Duman Alarmı
- **İletişim Protokolü**: Z-Wave Plus
- **Sertifika Standardı**: EN 14604
- **Tipik Senaryo**: Evlerde, apartmanlarda ve diğer yerlerde yangın uyarısı

## Teknik parametreler

| Parametre | Değer |
|----------|-----|
| Sensör tipi | Kızılötesi fotoelektrik |
| İletişim Protokolü | Z-Wave Plus (ITU-T G.9959) |
| Çalışma frekansı bandı | 868,42 MHz (AB) / 908,42 MHz (ABD) |
| Alarm ses basıncı | ≥85 dB (3 metre) |
| Güç kaynağı yöntemi | CR123A lityum pil |
| Pil ömrü | Yaklaşık 2~3 yıl |
| Şifreleme yöntemi | S2 Kimliği Doğrulanmış / S2 Kimliği Doğrulanmamış |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤%90 bağıl nem (yoğuşmasız) |

## Özellikler

- Yavaş yanma ve açık alev dumanını tespit etmek için fotoelektrik duman algılama
- 85 dB yüksek sesli ve ışıklı alarm
- Düşük pil hatırlatıcısı
- Kendi kendine test fonksiyonu
- Kurcalamaya karşı alarm
- Z-Wave S2 güvenli şifreleme
- OTA ürün yazılımı yükseltmesini destekleyin

## Sertifika

- EN 14604 sertifikası
- Z-Wave Plus sertifikalı
- S2 güvenlik sertifikası
