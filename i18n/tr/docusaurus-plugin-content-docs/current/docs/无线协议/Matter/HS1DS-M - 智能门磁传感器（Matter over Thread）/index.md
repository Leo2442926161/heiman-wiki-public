# HS1DS-M - Akıllı kapı sensörü (Madde Üzerinden İplik)

## Genel Bakış

HS1DS-M, Heiman akıllı kapı sensörünün Matter Over Thread versiyonudur. Özel bir ağ geçidine ihtiyaç duymadan Apple Home, Google Home, Amazon Alexa, Home Assistant vb. gibi tüm Matter uyumlu platformlara yerel erişimi destekler.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Konu Üzerindeki Madde |
| Güç kaynağı yöntemi | CR1632 düğme pil |
| Pil ömrü | Yaklaşık 2-3 yıl |
| Kurulum mesafesi | ≤10 mm |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤%95 bağıl nem |

## Özellikler

- Kapı ve pencere açılma/kapanma durumunun gerçek zamanlı tespiti
- Sabotaj alarmı (Sabotaj)
- Düşük pil hatırlatıcısı
- Platformlar arası uyumlu: Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Thread Mesh ağı, düşük gecikmeli yerel kontrol
- Özel bir ağ geçidi gerekmez, İş parçacığı sınır yönlendiricisi gerekir
