# HS1CA-M Akıllı Karbon Monoksit Alarmı (Madde)

HS1CA-M, elektrokimyasal sensör teknolojisini kullanan ve Matter over Thread çapraz ekolojik ara bağlantısını destekleyen dünyanın ilk Matter sertifikalı karbon monoksit alarmıdır**.

## Ürüne Genel Bakış

- **Ürün Tipi**: Akıllı Karbonmonoksit Alarmı
- **İletişim Protokolü**: Konu Üzerinden Madde
- **Sertifika Standardı**: EN 50291
- **Tipik Senaryo**: Evlerde, apartmanlarda ve diğer yerlerde CO kaçağı uyarısı

## Teknik parametreler

| Parametre | Değer |
|----------|-----|
| Sensör Tipi | Elektrokimyasal CO Sensörü |
| İletişim Protokolü | Konu Üzerindeki Madde (IEEE 802.15.4) |
| Çalışma frekansı bandı | 2,4 GHz |
| Algılama aralığı | 0~1000 sayfa/dakika |
| Alarm ses basıncı | ≥85 dB (3 metre) |
| Güç kaynağı yöntemi | DC 3V (CR123A lityum pil) |
| Pil ömrü | Yaklaşık 3 yıl |
| Sensör ömrü | Tipik 7 yıl |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤%95 bağıl nem (yoğuşmasız) |

## Özellikler

- Yüksek hassasiyetli elektrokimyasal CO sensörü
- Konsantrasyon dereceli alarm (sesli ve ışıklı uyarı)
- 85 dB yüksek sesli ve ışıklı alarm
- Düşük pil hatırlatıcısı
- Sensör ömrünün sona ermesi hatırlatıcısı
- Kendi kendine test fonksiyonu
- Matter'ın çoklu-ekolojik platformlar arası birlikte çalışabilirliği

## Sertifika

- EN 50291 sertifikalı
- Madde sertifikası
- Ev Asistanı sertifikasıyla çalışır
