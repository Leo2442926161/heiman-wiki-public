# Sertifikasyon Standartları

HS1SA-M aşağıdaki sertifikaları geçmiştir:

- **EN 14604** — Duman alarmlarına yönelik Avrupa standardı, bkz. [EN Avrupa Standardı](../../../../ApprovalStandards/EN Avrupa Standardı.md)
- **CE İşareti** — AB Uygunluk Beyanı, bkz. [CE AB Uygunluğu](../../../../certification-standards/CE EU-Conformity.md)
- **Matter Sertifikasyonu** — Akıllı evlerin platformlar arası ara bağlantısını sağlamak için CSA Alliance Matter sertifikasını geçti
- **Red Dot Tasarım Ödülü** — Red Dot Ürün Tasarım Ödülü
