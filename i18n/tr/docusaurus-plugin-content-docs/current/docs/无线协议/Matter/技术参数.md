# 技术参数

以下为海曼 Matter 设备的通用技术参数。

## 无线参数

| 参数 | 值 |
|------|-----|
| 底层协议 | Thread（IEEE 802.15.4） |
| 工作频段 | 2.4 GHz（ISM 频段） |
| 应用层标准 | Matter 1.x |
| 网络拓扑 | Mesh 自愈网络 |
| 加密方式 | AES-128-CCM |

## 硬件要求

| 项目 | 说明 |
|------|------|
| 边界路由器 | 必需，如 Apple HomePod mini、Apple TV 4K、Google Nest Hub 2代 等 |
| 移动端支持 | iOS / Android（通过各生态 App 配网） |
| 跨生态互操作 | Apple HomeKit、Google Home、Amazon Alexa、Home Assistant、SmartThings |

## 电气与功耗

| 参数 | 值 |
|------|-----|
| 供电方式 | 电池供电（CR123A / CR2450 等） |
| 电池续航 | 约 1~3 年（视设备类型与使用频率） |
| 待机功耗 | 极低，支持睡眠唤醒机制 |

## 环境参数

| 参数 | 值 |
|------|-----|
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤ 95% RH（无凝结） |
