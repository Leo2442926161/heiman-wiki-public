# Teknik parametreler

Aşağıda Hyman Matter ekipmanının genel özellikleri yer almaktadır.

## Kablosuz parametreler

| Parametre | Değer |
|----------|-----|
| Temel protokol | Konu (IEEE 802.15.4) |
| Çalışma frekansı bandı | 2,4 GHz (ISM bandı) |
| Uygulama katmanı standartları | Madde 1.x |
| Ağ topolojisi | Kendi kendini onaran ağ ağı |
| Şifreleme yöntemi | AES-128-CCM |

## Donanım gereksinimleri

| Proje | Açıklama |
|------|------|
| Sınır yönlendiricisi | Apple HomePod mini, Apple TV 4K, Google Nest Hub 2. nesil vb. gibi gereklidir. |
| Mobil terminal desteği | iOS / Android (çeşitli ekolojik uygulamalar aracılığıyla ağ dağıtımı) |
| Ekolojiler arası birlikte çalışabilirlik | Apple HomeKit, Google Home, Amazon Alexa, Ev Asistanı, SmartThings |

## Elektrik ve Güç Tüketimi

| Parametre | Değer |
|----------|-----|
| Güç kaynağı yöntemi | Pille çalışan (CR123A / CR2450, vb.) |
| Pil ömrü | Yaklaşık 1~3 yıl (cihaz tipine ve kullanım sıklığına bağlı olarak) |
| Bekleme modunda güç tüketimi | Son derece düşük, uykudan uyandırma mekanizmasını destekler |

## Ortam parametreleri

| Parametre | Değer |
|----------|-----|
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤ %95 bağıl nem (yoğuşmasız) |
