# L1-M Suya Daldırma Dedektörü (Madde)

L1-M, Hyman tarafından piyasaya sürülen bir **Matter Over Thread** suya daldırma dedektörüdür. Su borusu patlamalarından kaynaklanan mal kayıplarını etkili bir şekilde önleyebilen bir sonda aracılığıyla su sızıntısını veya birikmiş suyu tespit eder.

## Ürüne Genel Bakış

- **Ürün Tipi**: Suya dalma algılama sensörü
- **İletişim Protokolü**: Konu Üzerinden Madde
- **Algılama yöntemi**: Prob teması algılama
- **Tipik Senaryolar**: Mutfak, banyo, çamaşır odası, bodrum katında su kaçağı tespiti

## Teknik parametreler

| Parametre | Değer |
|----------|-----|
| Algılama yöntemi | Paslanmaz çelik prob kontak tipi |
| İletişim Protokolü | Konu Üzerindeki Madde (IEEE 802.15.4) |
| Çalışma frekansı bandı | 2,4 GHz |
| Güç kaynağı yöntemi | Pille çalışan |
| Alarm yöntemi | Buzzer + Uygulama itme |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤%95 bağıl nem (yoğuşmasız) |

## Özellikler

- Su sızıntısının/durgun suyun gerçek zamanlı tespiti
- Yerinde sesli alarm
- Uygulama uzaktan alarmı itme
- Düşük pil hatırlatıcısı
- Korozyon önleyici prob tasarımı
- Matter'ın çoklu-ekolojik platformlar arası birlikte çalışabilirliği
