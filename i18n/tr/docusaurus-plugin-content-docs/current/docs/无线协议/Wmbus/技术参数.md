# Teknik parametreler

Aşağıdakiler Heimann Kablosuz M-Bus (wM-Bus) ekipmanının genel teknik parametreleridir.

## Kablosuz parametreler

| Parametre | Değer |
|----------|-----|
| Protokol standartları | EN 13757-4 (Kablosuz M-Bus fiziksel katmanı) |
| Çalışma frekans bandı | 868 MHz (Avrupa SRD frekans bandı, 868,0~868,6 MHz) |
| Modülasyon yöntemi | FSK |
| İletim hızı | 4,8 kbps / 16,384 kbps / 100 kbps (T modu / S modu / R modu) |
| Ağ topolojisi | Yıldız (enstrüman → toplayıcı/ağ geçidi) |
| İletişim mesafesi | Kapalı alanda yaklaşık 100~300 metre, açık alanda yaklaşık 500~1000 metre |
| İletim gücü | ≤ 25 mW (+14 dBm) |

## Çalışma modu

| Modu | Oranı | Açıklama |
|------|------|------|
| S modu | 16.384kbps | Standart mod, düşük güç tüketimi, pille çalışan cihazlara uygun |
| T modu | 100 kbps | Büyük trafik verileri senaryolarına uygun yüksek hız modu |
| R modu | 4,8 kbps | Dar bant modu, uzun mesafe iletimi |

## Elektrik ve Güç Tüketimi

| Parametre | Değer |
|----------|-----|
| Güç kaynağı yöntemi | Lityum pil güç kaynağı (ER18505 / ER26500 ve diğer endüstriyel lityum piller) |
| Pil ömrü | 6~10 yıl (raporlama sıklığına bağlı olarak tipik değer 1 saat/gündür) |
| Bekleme modunda güç tüketimi | Mikroamper seviyesi, derin uyku |
| Hassasiyet alma | Yaklaşık. -105 dBm (S modu) |

## Ortam parametreleri

| Parametre | Değer |
|----------|-----|
| Çalışma sıcaklığı | -20°C ~ +60°C (endüstriyel sınıf) |
| Çalışma nemi | ≤ %95 bağıl nem (yoğuşmasız) |
| IP koruma düzeyi | IP65 ~ IP68 (özel modele bağlı olarak, dış mekan enstrümanlarına uygulanabilir) |

## Veri protokolü

| Katman | Standart |
|----|------|
| Fiziksel katman | EN 13757-4 |
| Veri Bağlantı Katmanı | EN 13757-4 (Kablosuz M-Bus Çerçeve Formatı) |
| Uygulama katmanı | EN 13757-3 (M-Bus uygulama katmanı) |
| Şifreleme yöntemi | AES-128 (isteğe bağlı şifreleme) |
