# 技术参数

以下为海曼无线 M-Bus（wM-Bus）设备的通用技术参数。

## 无线参数

| 参数 | 值 |
|------|-----|
| 协议标准 | EN 13757-4（无线 M-Bus 物理层） |
| 工作频段 | 868 MHz（欧洲 SRD 频段，868.0~868.6 MHz） |
| 调制方式 | FSK |
| 传输速率 | 4.8 kbps / 16.384 kbps / 100 kbps（T 模式 / S 模式 / R 模式） |
| 网络拓扑 | 星型（仪表 → 采集器 / 网关） |
| 通信距离 | 室内约 100~300 米，空旷约 500~1000 米 |
| 发射功率 | ≤ 25 mW（+14 dBm） |

## 工作模式

| 模式 | 速率 | 说明 |
|------|------|------|
| S 模式 | 16.384 kbps | 标准模式，低功耗，适合电池供电仪表 |
| T 模式 | 100 kbps | 高速模式，适合大流量数据场景 |
| R 模式 | 4.8 kbps | 窄带模式，长距离传输 |

## 电气与功耗

| 参数 | 值 |
|------|-----|
| 供电方式 | 锂电池供电（ER18505 / ER26500 等工业锂电池） |
| 电池续航 | 6~10 年（视上报频率，典型值 1 次/天） |
| 待机功耗 | 微安级，深度休眠 |
| 接收灵敏度 | 约 -105 dBm（S 模式） |

## 环境参数

| 参数 | 值 |
|------|-----|
| 工作温度 | -20°C ~ +60°C（工业级） |
| 工作湿度 | ≤ 95% RH（无凝结） |
| IP 防护等级 | IP65 ~ IP68（视具体型号，户外仪表适用） |

## 数据协议

| 层 | 标准 |
|----|------|
| 物理层 | EN 13757-4 |
| 数据链路层 | EN 13757-4（无线 M-Bus 帧格式） |
| 应用层 | EN 13757-3（M-Bus 应用层） |
| 加密方式 | AES-128（可选加密） |
