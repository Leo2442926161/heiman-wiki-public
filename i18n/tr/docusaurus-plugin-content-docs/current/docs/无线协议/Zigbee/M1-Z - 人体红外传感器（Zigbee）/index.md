# M1-Z - İnsan Vücudu Kızılötesi Sensör (Zigbee)

## Genel Bakış

M1-Z, Heiman tarafından piyasaya sürülen kompakt bir Zigbee insan vücudu kızılötesi sensörüdür. 110° veya 360° algılama açısını destekler (kurulum yöntemine bağlı olarak), yerleşik aydınlatma sensörüne sahiptir ve akıllı aydınlatma ve güvenlik senaryolarına uygundur.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Zigbee 3.0 |
| Sensör Tipi | Pasif Kızılötesi (PIR) + Aydınlatma |
| Algılama açısı | 110° (duvara monte) / 360° (tavana) |
| Algılama mesafesi | ≤7 m (25°C) |
| Aydınlık aralığı | 0 ~ 1200 Lüks |
| Güç kaynağı yöntemi | DC 3V lityum pil |
| Pil ömrü | Yaklaşık 3 yıl |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Kurulum yöntemi | Duvara monte/tavan |

## Özellikler

- Mobil insan tespiti
- Aydınlatma algılama (0~1200 Lux)
- 110° / 360° değiştirilebilir
- Düşük pil hatırlatıcısı
- Kurcalamaya karşı alarm
- Kompakt tasarım
