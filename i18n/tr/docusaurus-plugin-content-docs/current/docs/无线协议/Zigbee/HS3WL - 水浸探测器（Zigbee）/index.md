# HS3WL - Su Kaçak Dedektörü (Zigbee)

## Genel Bakış

HS3WL, Hyman'ın üçüncü nesil Zigbee suya daldırma dedektörüdür. Önceki nesle göre gelişmiş algılama hassasiyeti ve kablosuz performansa sahiptir ve su kaçağı izleme senaryoları için uygundur.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Zigbee 3.0 |
| Algılama yöntemi | Prob tespiti |
| Güç kaynağı yöntemi | DC 3V pil |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤%95 bağıl nem |

## Özellikler

- Su kaçağı/su birikiminin gerçek zamanlı tespiti
- Geliştirilmiş algılama hassasiyeti
- Zil alarmı
- Düşük pil hatırlatıcısı
- Korozyon önleyici prob tasarımı
