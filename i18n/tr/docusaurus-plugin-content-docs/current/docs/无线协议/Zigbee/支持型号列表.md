#Desteklenen model listesi

Heyman, sektördeki en geniş Zigbee ürün yelpazesine sahiptir ve piyasaya sürülen desteklenen modeller aşağıdadır.

## Güvenlik alarmı kategorisi

| Modeli | Ürün Adı | Açıklama |
|------|----------|------|
| HS1SA-E | Akıllı duman alarmı (Zigbee) | AB versiyonu |
| HS1SA-Z | Akıllı Duman Alarmı (Zigbee) | Çin Versiyonu |
| HS3SA | Akıllı duman alarmı (Zigbee) | Üçüncü nesil duman alarmı |
| HS8SA | Akıllı duman alarmı (Zigbee) | Yüksek sıcaklık bağlantı tipi |
| HS1CA-E | Akıllı karbon monoksit alarmı (Zigbee) | AB versiyonu |
| HS1CA-Z | Akıllı karbon monoksit alarmı (Zigbee) | Çince versiyonu |
| HS1CG | Yanıcı Gaz Dedektörü (Zigbee) | Doğal Gaz / Sıvılaştırılmış Gaz |
| HS3CG | Yanıcı gaz dedektörü (Zigbee) | Üçüncü nesil gaz dedektörü |

## Sensör sınıfı

| Modeli | Ürün Adı | Açıklama |
|------|----------|------|
| HS1DS | Kapı manyetik sensörü (Zigbee) | Kapı ve pencere açılma ve kapanma tespiti |
| HS1MS | İnsan vücudu kızılötesi sensörü (Zigbee) | Standart tip |
| HS3MS | İnsan vücudu kızılötesi sensörü (Zigbee) | Üçüncü nesil, geliştirilmiş algılama hassasiyeti |
| M1-Z | İnsan vücudu kızılötesi sensörü (Zigbee) | Kompakt tasarım |
| HS2WL | Suya daldırma dedektörü (Zigbee) | İkinci nesil |
| HS3WL | Suya daldırma dedektörü (Zigbee) | Üçüncü nesil |
| HS1WL | Suya daldırma dedektörü (Zigbee) | Birinci nesil |
| HS8MLS | Mikro hareket sensörü (Zigbee) | Hareket algılama |
| HS8MIS | Hareket Sensörü (Zigbee) | Hareket Algılama |
| HS8OS | Işık sensörü (Zigbee) | Işık algılama |

|------|----------|------|

| Modeli | Ürün Adı | Açıklama |
|------|----------|------|
