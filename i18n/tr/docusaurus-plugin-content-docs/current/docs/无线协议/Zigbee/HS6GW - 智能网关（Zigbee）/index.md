# HS6GW - Akıllı Ağ Geçidi (Zigbee)

## Genel Bakış

HS6GW, Heiman tarafından başlatılan bir Zigbee akıllı ağ geçididir. Heiman'ın Zigbee ağının koordinatörü olarak Wi-Fi veya Ethernet uplink yoluyla internete bağlanır ve 128'e kadar alt cihazın erişimini destekler.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Zigbee 3.0 (Koordinatör) |
| Yukarı bağlantı bağlantısı | Wi-Fi / Ethernet |
| Maksimum Düğüm | 128 alt cihaz |
| Güç kaynağı yöntemi | DC 5V USB güç kaynağı |

## Özellikler

- Zigbee Ağ Koordinatörü
- Wi-Fi / Ethernet uplink bağlantısı
- Otomatik cihaz keşfi ve eşleştirme
- Yerel sahne bağlantısı
- Uzaktan erişim (Heiman Bulut Platformu aracılığıyla)
- 128'e kadar alt cihazı destekler
-OTA ürün yazılımı yükseltmesi

## Uygulanabilir senaryolar

- Heiman'ın tüm kuruma yönelik istihbaratının temel bağlantı noktası
- Uzaktan izlemeyi gerçekleştirmek için Haiman APP ile işbirliği yapın
