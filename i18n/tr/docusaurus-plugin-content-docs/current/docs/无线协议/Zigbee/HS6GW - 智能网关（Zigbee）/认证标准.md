# Sertifikasyon Standartları

HS6GW (Zigbee versiyonu) aşağıdaki sertifikaları geçmiştir:

- **CE/RED** — Avrupa Birliği Kablosuz Ekipman Direktifi
- **CCC** — Çin Zorunlu Ürün Sertifikasyonu, bkz. [CCC Çin Zorunlu Sertifikasyonu](../../../../Certification Standards/CCC China Zorunlu Sertifikasyonu.md)
