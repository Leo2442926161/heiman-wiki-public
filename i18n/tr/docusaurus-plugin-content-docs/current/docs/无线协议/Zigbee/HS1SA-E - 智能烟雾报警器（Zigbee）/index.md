# HS1SA-E - Akıllı Duman Alarmı (Zigbee)

## Genel Bakış

HS1SA-E, Hyman tarafından AB pazarı için piyasaya sürülen standart bir Zigbee akıllı duman alarmıdır. Fotoelektrik duman algılama teknolojisini benimsemiş olup EN 14604 sertifikasını geçmiş ve Red Dot Tasarım Ödülünü kazanmıştır.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Zigbee 3.0 |
| Sensör tipi | Kızılötesi fotoelektrik |
| Alarm ses basıncı | ≥85 dB (3 metre) |
| Güç kaynağı yöntemi | DC 3V (CR123A) |
| Pil ömrü | Yaklaşık 3 yıl |
| İletişim mesafesi | ≤70 m (açık alan) |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤%95 bağıl nem |
| Kurulum yöntemi | Tavan kurulumu |

## Özellikler

- Yavaş yanıklardan ve açık alevlerden kaynaklanan dumanı tespit etmek için fotoelektrik duman algılama
- 85 dB yüksek sesli ve ışıklı alarm
- Düşük pil hatırlatıcısı
- Kendi kendine test fonksiyonu (tek tıkla test)
- Kurcalamaya karşı alarm
-Uzaktan sessize alma desteği
-OTA ürün yazılımı yükseltmesi

## Sertifikasyon standartları

- EN 14604 (Duman alarmları için Avrupa standardı)
- Red Dot Tasarım Ödülü
-CE/KIRMIZI

## Uygulanabilir aksesuarlar

-CR123A pil
- Montaj vidaları/genleşme tüpü
