#HS3MS - İnsan Vücudu Kızılötesi Sensör (Zigbee)

## Genel Bakış

HS3MS, Hyman'ın üçüncü nesil Zigbee insan vücudu kızılötesi sensörüdür. 360° tavana monteli bir tasarımı benimser ve algılama hassasiyeti ve kapsama alanı önceki nesle kıyasla önemli ölçüde iyileştirilmiştir.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Zigbee 3.0 |
| Sensör Tipi | Pasif Kızılötesi (PIR) |
| Algılama açısı | 360° (tavan montajı) |
| Algılama mesafesi | ≤7 m (25°C) |
| Güç kaynağı yöntemi | DC 3V lityum pil |
| Pil ömrü | Yaklaşık 3 yıl |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Kurulum yöntemi | Tavan kurulumu |

## Özellikler

- Mobil insan tespiti
- 360° çok yönlü kapsama alanı
- Düşük pil hatırlatıcısı
- Kurcalamaya karşı alarm
- Geliştirilmiş algılama hassasiyeti
- Sahne bağlantısını destekleyin
