# JT Serisi - Gaz acil durum kapatma vanası (Zigbee)

## Genel Bakış

JT serisi gaz acil kapatma vanası, Hyman Zigbee yanıcı gaz dedektörü ile birlikte kullanılan bir uygulama cihazıdır. Bir gaz sızıntısı tespit edildiğinde, kazaları önlemek için gaz boru hattı vanasını uzaktan veya otomatik olarak kapatabilir.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Zigbee 3.0 |
| Tür | Bağımsız + Zigbee bağlantısı |
| Test gazı | Doğal gaz / Sıvılaştırılmış petrol gazı |
| Sertifikasyon standartları | GB 15322 |
| Çalışma sıcaklığı | -10°C ~ +50°C |

## Özellikler

- Vanayı otomatik olarak kapatmak için HS1CG / HS3CG ile bağlantılıdır
-Uzaktan manuel valf kapatmayı destekleyin
- Sızıntı alarmı itme
- Solenoid valf sürücüsü

## Sertifikasyon standartları

- GB 15322

## Uygulanabilir aksesuarlar

- HS1CG / HS3CG yanıcı gaz dedektörü
