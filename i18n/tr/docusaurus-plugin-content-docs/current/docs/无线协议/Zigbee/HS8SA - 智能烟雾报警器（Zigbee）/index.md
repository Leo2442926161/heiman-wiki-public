#HS8SA - Akıllı Duman Alarmı (Zigbee)

## Genel Bakış

HS8SA, Hyman'ın ABD pazarına yönelik Zigbee + Wi-Fi akıllı duman alarmıdır. Yüksek sıcaklık bağlantı fonksiyonuna sahiptir ve UL 217 sertifikasını geçmiştir.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Zigbee 3.0 / Wi-Fi |
| Sensör tipi | Kızılötesi fotoelektrik |
| Alarm ses basıncı | ≥85 dB (3 metre) |
| Güç kaynağı yöntemi | DC 3V (CR123A) |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Sertifikasyon standartları | UL217 |
| Uygulanabilir pazarlar | Amerika Birleşik Devletleri |

## Özellikler

- Fotoelektrik duman algılama
- Yüksek sıcaklık bağlantı alarmı
- 85 dB yüksek sesli ve ışıklı alarm
- Düşük pil hatırlatıcısı
- Kendi kendine test fonksiyonu
- Kurcalamaya karşı alarm
-Uzaktan sessize alma desteği
- Çift protokollü iletişim (Zigbee + Wi-Fi)

## Sertifikasyon standartları

- UL 217 (Amerikan duman alarmı standardı)
