# Sertifikasyon Standartları

HS8SA aşağıdaki sertifikaları geçti:

- **EN 14604** — Duman alarmları için Avrupa standardı (AB tipi), bkz. [EN Avrupa Standardı](../../../../ApprovalStandards/EN Avrupa Standardı.md)
- **UL 217** — ABD duman alarmı güvenlik standardı (ABD modeli), bkz. [UL Underwriters Laboratories](../../../../certification-standards/ULUnderwriters Laboratories.md)
- **CE/RED/EMC** — Avrupa Birliği Kablosuz Ekipman Yönergesi ve Elektromanyetik Uyumluluk
- **Zigbee 3.0 / Q-Label** — Zigbee Alliance sertifikası, bkz. [Diğer Sertifikalar](../../../Certification Standards/Other Certifications.md)
