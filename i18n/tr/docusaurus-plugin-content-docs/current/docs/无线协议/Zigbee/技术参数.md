# Teknik parametreler

Aşağıdakiler Heiman Zigbee cihazlarının genel teknik parametreleridir.

## Kablosuz parametreler

| Parametre | Değer |
|----------|-----|
| Protokol standartları | Zigbee 3.0 (IEEE 802.15.4) |
| Çalışma frekans bandı | 2,4 GHz (küresel ortak ISM frekans bandı) |
| İletim hızı | 250 kbps'ye kadar |
| Ağ topolojisi | Kendi kendini onaran ağ ağı |
| Düğüm kapasitesi | Tek bir ağda maksimum 65535 düğüm |
| İletişim mesafesi | Yaklaşık 10~30 metre iç mekan (duvar engellerini görme) |
| Şifreleme yöntemi | AES-128-CCM |

## Uyumluluk

| Proje | Açıklama |
|------|------|
| Zigbee Koordinatörü | Zigbee 3.0 standart koordinatörüyle uyumlu |
| Desteklenen platformlar | Zigbee2MQTT, ZHA (Ev Asistanı), Hubitat, Ev Gibi, SmartThings, Tuya Zigbee ağ geçidi, Haiman HS6GW ağ geçidi |
| OTA yükseltmesi | Kablosuz aygıt yazılımı yükseltmesini destekleyin (bazı modellerde) |

## Elektrik ve Güç Tüketimi

| Parametre | Değer |
|----------|-----|
| Güç kaynağı yöntemi | Pille çalışan (CR123A / CR2450 / AA) veya USB / şebeke (soket tipi) |
| Pil ömrü | Sensörler için yaklaşık 2~3 yıl (raporlama sıklığına bağlı olarak) |
| Bekleme modunda güç tüketimi | Microamp seviyesi, uyku yoklama mekanizmasını destekler |

## Ortam parametreleri

| Parametre | Değer |
|----------|-----|
| Çalışma sıcaklığı | -10°C ~ +55°C |
| Çalışma nemi | ≤ %95 bağıl nem (yoğuşmasız) |
| IP koruma düzeyi | Belirli modele bağlıdır (sensör tipi genellikle IP30) |
