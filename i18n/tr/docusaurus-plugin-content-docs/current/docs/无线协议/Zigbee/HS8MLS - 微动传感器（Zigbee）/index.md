# HS8MLS - Mikro Hareket Sensörü (Zigbee)

## Genel Bakış

HS8MLS, Heiman tarafından piyasaya sürülen uzun menzilli bir Zigbee insan vücudu kızılötesi sensörüdür (PIR). Uzak mesafeden algılama gerektiren sahneler için özel olarak tasarlanmış olup koridor, salon gibi açık alanlar için uygundur.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Zigbee 3.0 |
| Sensör Tipi | Pasif Kızılötesi (PIR) |
| Algılama mesafesi | Uzun mesafe algılama |
| Güç kaynağı yöntemi | DC 3V pil |

## Özellikler

- Uzun mesafeli hareket algılama
- Düşük güç tüketimi tasarımı
- Sahne bağlantısını destekleyin
