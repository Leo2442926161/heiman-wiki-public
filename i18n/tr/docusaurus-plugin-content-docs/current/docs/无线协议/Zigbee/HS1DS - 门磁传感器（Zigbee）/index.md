# HS1DS - Kapı Manyetik Sensörü (Zigbee)

## Genel Bakış

HS1DS, Heiman tarafından piyasaya sürülen bir Zigbee kapı manyetik sensörüdür. Kapı ve pencerelerin açılma ve kapanma durumlarını tespit etmek için kullanılır. Akıllı güvenlik sisteminin temel bir bileşenidir.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Zigbee 3.0 |
| Sensör Tipi | Kamış Anahtarı |
| Güç kaynağı yöntemi | CR1632 düğme pil |
| Pil ömrü | Yaklaşık 2 ~ 3 yıl |
| Kurulum izni | ≤10 mm |
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤%95 bağıl nem |

## Özellikler

- Kapı ve pencere açılma/kapanma durumunun gerçek zamanlı tespiti
- Sabotaj alarmı (Sabotaj)
- Düşük pil hatırlatıcısı
-Bağlantıyı destekler (kapıyı açma, ışıkları açma, evden çıkarken devreye alma vb.)
- Kurulumu kolay (3M yapışkan bant veya vidayla sabitleme)

## Uygulanabilir aksesuarlar

- CR1632 düğme pil
- 3M çift taraflı bant/vidalar
