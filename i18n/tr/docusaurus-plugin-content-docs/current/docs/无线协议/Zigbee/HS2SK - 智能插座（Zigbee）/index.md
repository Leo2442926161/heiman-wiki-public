#HS2SK - Akıllı Soket (Zigbee)

## Genel Bakış

HS2SK, Heiman tarafından piyasaya sürülen bir Zigbee akıllı ölçüm soketidir. Uzaktan anahtar kontrolünü ve gerçek zamanlı güç tüketimi ölçümünü destekler. Ev aletlerinin akıllı yönetimi için uygundur ve ±%2 doğruluğa sahiptir.

## Teknik özellikler

| Parametre | Değer |
|----------|-----|
| İletişim Protokolü | Zigbee 3.0 |
| Nominal gerilim | AC 220V (Çin standardı) / AC 110V (Amerikan standardı) |
| Maksimum yük | 16A / 3680W |
| Ölçüm doğruluğu | ±%2 |
| Güç kaynağı yöntemi | Şebeke güç kaynağı |

## Özellikler

- Uzaktan anahtar kontrolü
- Gerçek zamanlı güç/yakıt ölçümü
- Aşırı yük koruması
- Zamanlayıcı anahtarı
- Güç istatistikleri
- Sahne bağlantısını destekleyin

## Uygulanabilir senaryolar

- Unutulan cihazları uzaktan kapatın
- Klima/su ısıtıcısı enerji tüketimi istatistikleri
- Gerekli olmayan cihazları tek tıklamayla kapatmak için ev modundan çıkın
