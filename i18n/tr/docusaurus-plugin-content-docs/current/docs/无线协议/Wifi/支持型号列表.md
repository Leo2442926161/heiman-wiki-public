# 支持型号列表

海曼 WiFi 设备主要面向直连场景，无需专用网关即可接入家庭网络。

## 智能插座

| 型号 | 产品名称 | 说明 |
|------|----------|------|
| HS2SK-WiFi | 智能插座（WiFi） | 直连 2.4GHz WiFi，支持远程控制与定时 |

## 安防报警类

| 型号 | 产品名称 | 说明 |
|------|----------|------|
| HS8SA-WiFi | 智能烟雾报警器（WiFi 美国版） | 美国市场 WiFi 版烟雾报警器 |

## 网关类

| 型号 | 产品名称 | 说明 |
|------|----------|------|
| HS6GW | 智能家居网关（WiFi + Zigbee） | 通过 WiFi 连接互联网，管理 Zigbee 子设备 |
| HS6GW-2 | 智能家居网关（WiFi + Zigbee） | 第二代网关，性能升级 |

## 注意事项

- 所有 WiFi 设备仅支持 **2.4 GHz** 频段，不兼容 5 GHz
- 建议使用 WPA2-PSK 加密的网络
- 联网需要路由器正常连接互联网
