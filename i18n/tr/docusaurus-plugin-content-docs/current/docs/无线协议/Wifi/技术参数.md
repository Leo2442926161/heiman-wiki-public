# Teknik parametreler

Heimann WiFi cihazlarının genel teknik özellikleri aşağıdadır.

## Kablosuz parametreler

| Parametre | Değer |
|----------|-----|
| Protokol standartları | IEEE 802.11 b/g/n |
| Çalışma frekansı bandı | 2,4 GHz (5 GHz ile uyumlu değildir) |
| İletim hızı | 150 Mbps'ye (802.11n) kadar |
| Ağ topolojisi | Yıldız topolojisi (doğrudan yönlendiriciye bağlı) |
| İletişim mesafesi | Yaklaşık 30~50 metre iç mekan (yönlendirici performansına bağlı olarak) |
| Şifreleme yöntemi | WPA2-PSK / WPA3 |

## Ağ dağıtım yöntemi

| Yöntem | Açıklama |
|------|------|
| SmartConfig | Mobil Uygulama Tek tıklamayla ağ yapılandırması (geleneksel WiFi ekipmanı) |
| Bluetooth destekli provizyon | WiFi kimlik bilgilerini Bluetooth geçici kanalları aracılığıyla yapılandırma (bazı yeni modeller) |
| AP modu | Cihazdaki erişim noktasını açın ve telefon bağlandıktan sonra yapılandırın |

## Elektrik ve Güç Tüketimi

| Parametre | Değer |
|----------|-----|
| Güç kaynağı yöntemi | Şebeke güç kaynağı (soket tipi/ağ geçidi tipi) veya pil güç kaynağı (alarm tipi) |
| Güç tüketimi düzeyi | TCP/IP bağlantısını sürdürme ihtiyacı nedeniyle Zigbee / Thread'den daha yüksek |
| Bekleme modunda güç tüketimi | Soket tipi yaklaşık 0,5W ~ 1W |

## Ortam parametreleri

| Parametre | Değer |
|----------|-----|
| Çalışma sıcaklığı | -10°C ~ +50°C |
| Çalışma nemi | ≤ %90 bağıl nem (yoğuşmasız) |
