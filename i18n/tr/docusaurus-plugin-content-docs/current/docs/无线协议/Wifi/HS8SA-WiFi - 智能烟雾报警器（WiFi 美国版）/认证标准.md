# Sertifikasyon Standartları

HS8SA-WiFi aşağıdaki sertifikaları geçmiştir:

- **UL 217** — Amerikan duman alarmı güvenlik standardı, bkz. [UL Underwriters Laboratories](../../../../certificationstandards/ULUnderwriters Laboratories.md)
- **FCC** — Federal İletişim Komisyonu sertifikası
- **CE** — Avrupa Birliği Uygunluk Beyanı
