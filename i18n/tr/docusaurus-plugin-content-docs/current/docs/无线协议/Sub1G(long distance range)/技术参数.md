# Teknik parametreler

Hyman Sub-1G uzun mesafe serisi ürünlerin genel teknik parametreleri aşağıdadır.

## Kablosuz parametreler

| Parametre | Değer |
|----------|-----|
| Çalışma frekansı bandı | 433 MHz / 868 MHz / 915 MHz (pazar alanına bağlı olarak) |
| Modülasyon yöntemi | FSK / OOK |
| İletim hızı | Yaklaşık 10~100 kbps (video segmenti ve konfigürasyonu) |
| Ağ topolojisi | Yıldız / noktadan noktaya / İnternet ağı |
| İletişim mesafesi | Açık ortamda yaklaşık 1~3 kilometre (video bölümü ve güç) |
| Ağ kapasitesi | Tek bir grup 50'den fazla düğümü destekler |

## Bölgesel frekans bandı tahsisi

| Bölge | Ortak Frekans Bantları |
|----------|----------|
| Çin | 433 MHz |
| Avrupa | 868 MHz |
| Kuzey Amerika | 915MHz |

## Elektrik ve Güç Tüketimi

| Parametre | Değer |
|----------|-----|
| Güç kaynağı yöntemi | Pille çalışan (CR123A / AA lityum pil) |
| Pil ömrü | Yaklaşık 3~5 yıl (raporlama sıklığına bağlı olarak) |
| İletim gücü | ≤ 10 mW (+10 dBm), SRD düzenlemeleriyle uyumlu |
| Bekleme modunda güç tüketimi | Mikroamper seviyesi, son derece düşük |
| Alma hassasiyeti | Yaklaşık -110 dBm ~ -120 dBm |

## Ortam parametreleri

| Parametre | Değer |
|----------|-----|
| Çalışma sıcaklığı | -10°C ~ +55°C |
| Çalışma nemi | ≤ %95 bağıl nem (yoğuşmasız) |
