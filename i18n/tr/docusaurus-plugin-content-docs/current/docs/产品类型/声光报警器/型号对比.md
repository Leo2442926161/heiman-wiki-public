# 型号对比

## 各型号对比

| 型号 | 协议 | 报警声压 | 供电 | 特点 | 玩法建议 |
|------|------|----------|------|------|----------|
| 声光报警器（警号） | Zigbee | ≥110 dB | 市电 | 高亮频闪 + 高分贝 | 烟雾/CO/门磁触发联动全屋声光报警，离家布防时震慑入侵者 |

## 场景玩法推荐

| 场景 | 推荐型号 | 联动方案 |
|------|----------|----------|
| 全屋消防联动 | 声光报警器 | 任一烟雾/CO 传感器报警 → 全屋声光警号齐鸣，唤醒熟睡家人 |
| 安防入侵威慑 | 声光报警器 | 门磁非法打开 + 离家模式 → 警号响 + 灯光全亮，震慑入侵者 |
| 紧急疏散引导 | 声光报警器 | 火警触发 → 警号指引疏散方向，配合门磁检测是否已撤离 |
