# 安装指南

## 安装位置要求

- **安装在全屋中央位置**，确保所有房间都能听到
- 走廊、客厅天花板或高处墙壁
- 距地面 **2 m 以上**（避免被人触碰）
- 与烟雾/CO 传感器配合时，安装在卧室走廊

## 安装步骤

1. 选定安装位置（建议走廊或客厅）
2. 用螺丝将安装底座固定在墙面或天花板
3. 连接电源（市电版）或装入电池（电池版）
4. 将报警器主体安装到底座上
5. 通过网关配对
6. 联动测试：触发任意传感器，声光报警器应响应

## 注意事项

- 安装在儿童无法触及的高度
- 定期测试报警音量和灯光是否正常
- 电池版注意低电量更换
- 可与其他传感器设置联动规则
