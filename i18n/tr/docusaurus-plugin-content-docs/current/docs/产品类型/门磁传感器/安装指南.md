# 安装指南

## 安装位置要求

- 主体（发射器）安装在 **门/窗框** 上
- 磁铁安装在 **门/窗扇** 上
- 主体与磁铁的间隙 **≤ 10 mm**
- 两者对齐，标记线对齐

## 安装步骤

1. 清洁安装表面，确保干燥无尘
2. 撕下 3M 胶贴保护膜，将主体粘贴在门框上
3. 将磁铁粘贴在对应位置（关门时与主体对齐）
4. 装入电池（CR1632），指示灯闪烁表示开机
5. 开合门窗测试：开门时触发报警，关闭后恢复
6. 通过网关配对

## 注意事项

- 安装间隙过大将导致检测失效
- 金属门窗可能影响无线信号，建议测试后再固定
- 电池寿命约 2-3 年，低电量时 APP 会提醒
