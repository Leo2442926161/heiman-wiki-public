## 场景玩法推荐

| 场景 | 推荐型号 | 联动方案 |
|------|----------|----------|
| 厨房安全自动化 | HS1CG / HS3CG | 检测泄漏 → 自动关电磁阀 → 开排风扇 → 推送手机通知 → 联动声光报警器 |
| 出租房/公寓 | JT-G1 | 独立报警无需联网，成本低，满足消防检查要求 |
| 远程监控需求 | HS8CG | 双通信保障 + 云平台远程查看状态 |
| 商业厨房 | HS3CG | 高灵敏度 + 联动排烟系统 + 物业平台集中管理 |

## 选型建议

- **独立使用**：JT-G1
- **Zigbee 联网**：[HS1CG](../../无线协议/Zigbee/HS1CG - 可燃气体探测器（Zigbee）) 或 [HS3CG](../../无线协议/Zigbee/HS3CG - 可燃气体探测器（Zigbee）)
- **双通信保障**：JT-HS8CG
