## Önerilen sahne oynanışı

| Senaryolar | Önerilen modeller | Bağlantı çözümleri |
|------|----------|----------|
| Mutfak güvenlik otomasyonu | HS1CG / HS3CG | Sızıntıları tespit edin → Solenoid valfi otomatik olarak kapatın → Egzoz fanını açın → Cep telefonuna anlık bildirimler → Bağlantılı sesli ve ışıklı alarm |
| Kiralık ev/daire | JT-G1 | Bağımsız alarm ağ bağlantısı gerektirmez, düşük maliyetlidir, yangın denetimi gereksinimlerini karşılar |
| Uzaktan izleme gereksinimleri | HS8CG | Çift iletişim garantisi + bulut platformu uzaktan görüntüleme durumu |
| Ticari mutfak | HS3CG | Yüksek hassasiyet + bağlantılı duman tahliye sistemi + mülk platformunun merkezi yönetimi |

## Seçim önerileri

- **Bağımsız kullanım**: JT-G1
- **Zigbee Ağı**: [HS1CG](../../Kablosuz Protokol/Zigbee/HS1CG - Yanıcı Gaz Dedektörü (Zigbee)) veya [HS3CG](../../Kablosuz Protokol/Zigbee/HS3CG - Yanıcı Gaz Dedektörü (Zigbee))
- **İkili iletişim garantisi**: JT-HS8CG
