# 安装指南

## 安装位置要求

- **安装在天花板**，位于房间中央位置
- 距离墙壁或灯具至少 **30 cm**
- **避开死角空气区**（如角落、梁后）
- **避免安装在**：浴室（蒸汽）、厨房（油烟）、通风口附近

## 安装步骤

1. 选定安装位置，确保天花板平整
2. 将安装底座旋转固定在天花板上（用螺丝或 3M 胶贴）
3. 装入电池（注意极性），传感器自动开机
4. 将传感器主体旋转卡入底座
5. 长按测试键，报警灯闪烁并发出报警声表示正常
6. 通过网关配对：进入配对模式，在 APP 中添加设备

## 注意事项

- 每月测试一次功能
- 电池寿命约 3 年，低电量时请及时更换
- 建议每 10 年更换整个传感器
- 安装高度不影响烟雾检测，烟雾上升至天花板
