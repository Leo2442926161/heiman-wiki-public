# 安装指南

## 安装位置要求

- **放置在地面**，确保探针接触地面
- **靠近水源**：洗手间台盆下方、厨房水槽下方、洗衣机旁、热水器下方
- 需要检测漏水的最低点位置

## 安装步骤

1. 确定安装位置（漏水风险区域的地面）
2. 清洁地面表面
3. 将传感器放置在地面，探针接触地面
4. 如使用线缆版（HS2WL），将延长线缆沿需要监测的区域铺设
5. 装入电池，传感器自动开机
6. 模拟测试：用湿手指触碰探针，报警灯亮起
7. 通过网关配对

## 注意事项

- 探针长期接触潮湿环境可能腐蚀，定期检查
- 建议配合电磁阀实现自动关断
- 电池寿命约 2-3 年
- 多个传感器可覆盖不同区域
