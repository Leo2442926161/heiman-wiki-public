# 型号对比

## 各型号对比

| 型号 | 协议 | 检测方式 | 供电 | 特点 | 玩法建议 |
|------|------|----------|------|------|----------|
| [HS1WL](../../无线协议/Zigbee/HS1WL - 水浸探测器（Zigbee）) | Zigbee | 探针 | 电池 | 基础款 | 漏水即报警推送，适合洗手间台盆下 |
| [HS2WL](../../无线协议/Zigbee/HS2WL - 水浸探测器（Zigbee）) | Zigbee | 探针+线缆 | 电池 | 双通道 | 探针检测点位 + 线缆覆盖长区域，热水器/洗衣机旁最佳 |
| [HS3WL](../../无线协议/Zigbee/HS3WL - 水浸探测器（Zigbee）) | Zigbee | 探针 | 电池 | 增强版 | 灵敏度更高，蜂鸣器更响 |
| [L1-M](../../无线协议/Matter/L1-M 水浸探测器（Matter）) | Matter over Thread | 探针 | 电池 | Matter 版 | WWHA 认证，原生接入 Apple Home |

## 场景玩法推荐

| 场景 | 推荐型号 | 联动方案 |
|------|----------|----------|
| 全屋漏水监控 | HS2WL + L1-M | 多个点位覆盖 + 原生 HomeKit 通知，检测漏水 → 关总水阀 → 推送通知 |
| 洗衣机专用 | HS2WL | 线缆探针铺设在洗衣机底部，一旦漏水立即报警 + 关洗衣机电源 |
| 热水器/地暖 | HS3WL | 增强版灵敏度高，地暖管漏水初期即可检测 |
| Matter 用户 | L1-M | 无需专用网关，Thread 低延迟报警联动 Apple Home 自动化 |

## 选型建议

- **点位检测**：[HS1WL](../../无线协议/Zigbee/HS1WL - 水浸探测器（Zigbee）) 或 [L1-M](../../无线协议/Matter/L1-M 水浸探测器（Matter）)
- **大面积覆盖**：[HS2WL](../../无线协议/Zigbee/HS2WL - 水浸探测器（Zigbee）)（探针+线缆）
- **Matter 需求**：[L1-M](../../无线协议/Matter/L1-M 水浸探测器（Matter）)
