# 温湿度传感器

温湿度传感器用于实时监测环境温度和相对湿度，是智能家居环境监控的基础设备。

## 产品系列

| 系列 | 通信方式 | 特点 |
|------|----------|------|
| H1 系列 | Matter / Zigbee | Matter over Thread 版已获 WWHA 认证 |

## 核心功能

- 实时温湿度监测
- 低电量提醒
- 支持联动控制（温度高时开空调、湿度高时开除湿机等）

## 技术参数

- 温度精度：±0.3°C
- 湿度精度：±3% RH
- 供电方式：CR2 电池
- 电池寿命：约 2-3 年

## 场景示例

- 联动空调/暖气自动调节室温
- 联动除湿机/加湿器维持舒适湿度
- 监测酒窖/雪茄柜环境
- 厨房/卫生间排气扇自动控制
- 种植大棚环境监测
