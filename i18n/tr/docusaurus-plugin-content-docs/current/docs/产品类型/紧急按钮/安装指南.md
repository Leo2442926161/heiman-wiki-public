# 安装指南

## 安装位置要求

- **壁挂安装**：卫生间、卧室床头、客厅沙发旁——触手可及的位置
- **随身佩戴**：可放入口袋或挂在钥匙扣上
- **针对老人**：安装在卫生间和卧室床头，高度 1.2 m 左右

## 安装步骤

1. 确定安装位置或是否随身佩戴
2. 壁挂：用螺丝或 3M 胶固定安装底座
3. 装入电池
4. 将紧急按钮放在底座上或随身携带
5. 通过网关配对
6. 测试：长按按钮触发报警，确认 APP 收到通知

## 注意事项

- 采用**长按防误触**设计，需要持续按压 1-2 秒触发
- 建议与声光报警器联动
- 定期测试功能
- 老人使用可配合跌倒探测器双重保障
