# Kurulum Kılavuzu

## Kurulum yeri gereksinimleri

- Duvara montaj yüksekliği: **1,8 ~ 2,4 m**
- Tavan montajı: Odanın ortasındaki tavana monte edilir
- ** Kaçının**: Doğrudan güneş ışığı, ısıtma/klima delikleri, ısı kaynaklarının yakını
- Algılama menzilindeki engellerden (mobilya, cam bölmeler) kaçının

## Kurulum adımları

1. Kurulum yerini seçin (duvara monte veya tavana monte)
2. Montaj tabanını vidalarla sabitleyin
3. Pili takın, gösterge ışığı yanıp söner
4. Sensör gövdesini tabana takın
5. Algılama açısını ayarlayın (110° yatay aralık dahilinde)
6. Yürüme testi: Algılama alanından geçtikten sonra gösterge ışığı yanmalıdır.
7. Ağ geçidi aracılığıyla eşleştirme

## Notlar

- Kızılötesi sensörler sıcaklık değişimlerine karşı hassastır ve yaz aylarında yüksek sıcaklıktaki ortamlarda algılama mesafesi kısalabilir.
- Pencerelere bakmaktan kaçının (dış sıcaklıktaki değişiklikler yanlış alarmlara neden olur)
- Pil ömrü yaklaşık 3 yıl (Zigbee) / 1 yıl (Madde)
