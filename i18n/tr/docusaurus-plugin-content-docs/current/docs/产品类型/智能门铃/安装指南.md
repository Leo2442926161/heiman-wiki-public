# 安装指南

## 安装位置要求

- 安装在门铃高度约 **1.4 ~ 1.6 m**（与视线平齐）
- 朝向门口通道，确保能拍摄到访客面部
- 避免阳光直射摄像头

## 安装步骤

1. 选定安装位置（大门旁或门框上）
2. 用螺丝固定安装底座
3. 连接电源（有线版接线，电池版装入电池）
4. 将门铃主体安装到底座上
5. 通过 APP 配对并连接 Wi-Fi（Wi-Fi 版）
6. 调整摄像头角度
7. 测试门铃按键、视频和语音功能

## 注意事项

- Wi-Fi 版需要门口有良好的 Wi-Fi 信号覆盖
- 电池版注意充电周期，建议配合门磁使用（检测开门后唤醒）
- 定期清理摄像头镜头
- 确保门铃在有效 Zigbee 信号范围内
