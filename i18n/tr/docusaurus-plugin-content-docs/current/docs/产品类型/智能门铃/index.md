# akıllı kapı zili

Haiman Akıllı Kapı Zili, kapı zili arama ve güvenlik izleme işlevlerini entegre eder ve uzaktan görüntülü interkomu destekler.

## Temel işlevler

- Kapı zili düğmesi tetikleyicisi
- Uzaktan canlı video (Wi-Fi versiyonu)
- İki yönlü sesli arama
- Hareket algılama videosu
- Gece görüş fonksiyonu

## Uygulanabilir senaryolar

- Ziyaretçi çağrıları: bunları evde veya dışarıda yanıtlayabilirsiniz
- Ekspres teslimat/paket bildirimi: kapının açılmasının uzaktan onaylanması
- Güvenlik izleme: kapının önündeki şüpheli etkinlikleri kaydedin

> **Not**: Heiman kapı zili ürün grubu temel olarak Zigbee/Wi-Fi çözümlerine dayanmaktadır. Belirli model parametreleri için lütfen yetkiliye danışın.
