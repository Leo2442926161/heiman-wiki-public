# 型号对比

## 各型号对比

| 型号 | 协议 | 视频 | 供电 | 特点 | 玩法建议 |
|------|------|------|------|------|----------|
| 智能门铃 | Zigbee + Wi-Fi | 1080P 可选 | 电池/市电 | 门铃+安防二合一 | 访客按铃手机通知，远程可视对讲，移动侦测录像 |

## 场景玩法推荐

| 场景 | 推荐型号 | 联动方案 |
|------|----------|----------|
| 快递/外卖管理 | 智能门铃 | 按铃 → 手机通知 → 远程对讲确认 → 远程开锁（联动智能锁） |
| 安防监控 | 智能门铃 | 门前逗留检测 → 录像 + 推送报警，联动门磁/灯光 |
| 访客联动 | 智能门铃 | 按铃 → 室内屏/音箱显示画面 → 开灯 → 开锁联动 |
