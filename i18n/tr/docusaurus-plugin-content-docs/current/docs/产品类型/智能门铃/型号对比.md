# Model karşılaştırması

## Çeşitli modellerin karşılaştırılması

| Modeli | Protokol | Videosu | Güç Kaynağı | Özellikler | Oynanış Önerileri |
|------|------|------|------|------|----------|
| Akıllı kapı zili | Zigbee + Wi-Fi | 1080P isteğe bağlı | Pil/şebeke | Kapı zili + güvenlik ikisi bir arada | Ziyaretçi zil sesi cep telefonu bildirimi, uzaktan görsel interkom, hareket algılama kaydı |

## Önerilen sahne oynanışı

| Senaryolar | Önerilen modeller | Bağlantı çözümleri |
|------|----------|----------|
| Ekspres teslimat/paket servisi yönetimi | Akıllı kapı zili | Zili çalma → cep telefonu bildirimi → uzaktan interkom onayı → uzaktan kilit açma (bağlantılı akıllı kilit) |
| Güvenlik İzleme | Akıllı Kapı Zili | Kapı Desteği Algılama → Video + Push Alarmı, Bağlantılı Kapı Sensörü/Aydınlatma |
| Ziyaretçi bağlantısı | Akıllı kapı zili | Zili çalın → İç mekan ekranı/hoparlör ekranı → Işıkları açın → Bağlantının kilidini açın |
