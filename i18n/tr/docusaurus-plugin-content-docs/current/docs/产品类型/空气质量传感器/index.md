# 空气质量传感器

空气质量传感器（Air Quality Monitor）用于检测室内空气中的多种污染物，帮助维持健康的居住环境。

## 检测参数

海曼空气质量传感器可检测以下指标：

| 参数 | 说明 | 健康标准 |
|------|------|----------|
| PM2.5 | 细颗粒物 | ≤ 35 µg/m³（24小时平均） |
| PM10 | 可吸入颗粒物 | ≤ 50 µg/m³（24小时平均） |
| CO₂ | 二氧化碳 | ≤ 1000 ppm |
| TVOC | 总挥发性有机物 | ≤ 0.6 mg/m³ |
| 温度 | 环境温度 | 18-26°C |
| 湿度 | 相对湿度 | 40-60% RH |

## 核心功能

- 多参数空气质量检测
- 彩色显示屏（视型号）
- 超标声光报警
- 历史数据记录
- 联动新风/空气净化器

## 适用场景

- 卧室：监测 CO₂ 水平，提示开窗通风
- 儿童房：保障儿童呼吸健康
- 办公室：提高工作效率和舒适度
- 新装修房：监测甲醛/TVOC 释放情况
