# 安装指南

## 安装位置要求

- **吸顶安装**，传感器垂直向下
- 安装高度 **2.5 ~ 3.5 m**
- 检测半径约 **3 ~ 5 m**
- **避免**：传感器前方有金属障碍物（金属会遮挡毫米波）
- 远离大型金属管道、空调内机等

## 安装步骤

1. 选定天花板安装位置（房间中央最佳）
2. 在天花板上打孔，用膨胀螺丝固定底座
3. 接线（如需市电供电）或装入电池
4. 将传感器主体旋转卡入底座
5. 通电后指示灯亮起，传感器开始自检
6. 进入 APP 配对，添加设备
7. 测试：进入检测区域静坐，传感器应保持"有人"状态

## 注意事项

- 毫米波可穿透薄板（石膏板、木板），但金属会遮挡
- 厚墙壁、金属天花板会降低检测范围
- 建议与 PIR 传感器配合使用，实现全场景覆盖
