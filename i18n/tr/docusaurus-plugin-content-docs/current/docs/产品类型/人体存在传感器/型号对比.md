# 型号对比

## 各型号对比

| 型号 | 协议 | 检测技术 | 安装 | 特点 | 玩法建议 |
|------|------|----------|------|------|----------|
| [HS8OS](../../无线协议/Zigbee/HS8OS - 人体存在传感器（Zigbee）) | Zigbee | 毫米波雷达 | 吸顶 | 静止人体探测 | 比 PIR 更精准，卫生间/卧室长时间静止有人场景，解决 PIR 无人灭灯问题 |

## 场景玩法推荐

| 场景 | 推荐型号 | 联动方案 |
|------|----------|----------|
| 卫生间长开灯 | HS8OS | 人在灯亮，人走灯灭（检测微动呼吸，解决 PIR 静止熄灯痛点） |
| 卧室睡眠监测 | HS8OS | 检测床上是否有人，联动空调自动调节温度 |
| 智慧养老 | HS8OS | 老人卫生间静止超时 → 系统预警通知家人，防止意外 |
| 会议室节能 | HS8OS | 无人达到 N 分钟 → 自动关灯关空调，节能降耗 |

## 选型建议

- **解决 PIR 熄灯痛点**：[HS8OS](../../无线协议/Zigbee/HS8OS - 人体存在传感器（Zigbee）)（毫米波，静止检测）
- **与 PIR 搭配**：PIR 检测移动 + HS8OS 检测静止存在，互补使用效果最佳
