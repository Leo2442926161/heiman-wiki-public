# 型号对比

## 各型号对比

| 型号 | 协议 | 额定功率 | 计量精度 | 特点 | 玩法建议 |
|------|------|----------|----------|------|----------|
| [HS2SK](../../无线协议/Zigbee/HS2SK - 智能插座（Zigbee）) | Zigbee | 16A / 3680W | ±2% | 计量型 | 实时功耗监测 + 远程开关，联动离家模式一键断电，统计空调/热水器用电量 |
| [HS2SK-WiFi](../../无线协议/Wifi/HS2SK-WiFi - 智能计量插座（WiFi）) | WiFi | 16A / 3680W | ±2% | 直连无需网关 | 无需额外网关，Smart Life App 直接控制，适合不想买网关的用户 |

## 场景玩法推荐

| 场景 | 推荐型号 | 联动方案 |
|------|----------|----------|
| 能耗统计 | HS2SK | Zigbee 接入 HA，联动 InfluxDB + Grafana 展示每日/月用电量，发现高能耗电器 |
| 离家断电 | HS2SK | 离家模式一键关闭电视/空调/鱼缸灯等非必要电器，安全省电 |
| 定时控制 | HS2SK / HS2SK-WiFi | 定时开关热水器/加湿器/充电桩，结合峰谷电价节省电费 |
| 无网关用户 | HS2SK-WiFi | 直连路由器，Smart Life App 控制，零门槛入门 |

## 选型建议

- **已有 Zigbee 网关**：[HS2SK](../../无线协议/Zigbee/HS2SK - 智能插座（Zigbee）)
- **不想买网关**：[HS2SK-WiFi](../../无线协议/Wifi/HS2SK-WiFi - 智能计量插座（WiFi）)
