# 安装指南

## 安装位置要求

- 直接插入墙壁插座，**请勿串联**（不要插入插排或延长线）
- 确保插座额定电流 ≥ 16A
- 不要安装在潮湿或高温环境

## 安装步骤

1. 将智能插座直接插入 220V（或 110V）墙壁插座
2. 通电后指示灯亮起
3. 长按开关按钮进入配对模式
4. 在海曼 APP 中添加设备
5. 配对成功后，即可远程控制开关

## 安全注意事项

- **最大负载**：16A / 3680W（欧标），15A / 1800W（美标）
- **请勿过载**：使用大功率电器（空调、取暖器）时注意功率限制
- **不要串联**：禁止将智能插座插入插排
- 请勿覆盖或遮挡插座外壳
- 定期检查插头是否过热

## 典型应用

- 热水器、加湿器、鱼缸灯等低功率设备定时控制
- 配合传感器联动：人来灯亮、人走灯灭
- 电量统计：监测家电能耗
