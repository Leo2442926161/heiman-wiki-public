# Kontrol paneli yapılandırması

## Güvenliğe Genel Bakış Paneli

Heimann ekipmanlarını alanlara göre düzenlemek için Kesit görünümü düzenini kullanın:

```yaml
tür: bölüm
başlık: Duman ve CO Alarmı
kartlar:
  - tür: varlık
    varlık: Binary_sensor.heiman_smoke_alarm
    adı: duman alarmı
  - tür: varlık
    varlık: Binary_sensor.heiman_co_alarm
    isim: CO alarmı
''''

## Mantar Kartı Stili

```yaml
tür: özel:mantar-alarm-kontrol paneli-kartı
varlık: alarm_control_panel.home_alarm
adı: güvenlik sistemi

tür: özel:mantar varlık kartı
varlık: Binary_sensor.heiman_smoke_alarm
adı: duman alarmı
simge: mdi:duman dedektörü
birincil_bilgi: durum
Secondary_info: son değişiklik
fill_container: doğru
''''

## Çevresel sensör paneli

```yaml
tür: bölüm
başlık: Çevresel İzleme
kartlar:
  - tip: gösterge
    varlık: sensör.heiman_temperature
    adı: sıcaklık
    birim: "°C"
    dk: 0
    maksimum: 50
    ciddiyet:
      yeşil: 0
      sarı: 30
      Kırmızı: 40
  - tip: gösterge
    varlık: sensör.heiman_humidity
    adı: nem
    birim: "%"
    dk: 0
    maksimum: 100
    ciddiyet:
      yeşil: 30
      Sarı: 60
      Kırmızı: 80
''''

## Alarm geçmişi paneli

```yaml
tür: geçmiş grafiği
başlık: Sensör Geçmişi
varlıklar:
  - ikili_sensor.heiman_smoke_alarm
  - ikili_sensor.heiman_co_alarm
  - ikili_sensor.heiman_water_leak
saat_to_show: 24
''''

## Önerilen düzen

- Üst: Güvenlik sistemi durumu + alarm cihazı durum kartı
- Merkezi: Çevresel izleme (sıcaklık, nem, su kalitesi)
- Alt: Sensör alarmı geçmiş trend tablosu
- Kenar çubuğu kısayol işlemleri: etkinleştirme/devre dışı bırakma, sahne değiştirme
