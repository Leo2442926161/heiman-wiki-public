# Kullanım ipuçları

## Cihaz yapılandırması

- Heiman Matter cihazınızı ilk kez eşleştirirken sınır yönlendiricinizin (örn. HomePod, Nest Hub, Home Assistant Green) aynı Thread ağında olduğundan emin olun
- Zigbee cihazlarını ZHA veya Zigbee2MQTT aracılığıyla entegre ederken, öncelikle USB koordinatörünün önerilen donanım yazılımını (Sonoff ZBDongle-E, Conbee II gibi) kullanmanız önerilir.

## Otomatik optimizasyon

- Anlık yanlış alarmları önlemek amacıyla duman alarmı otomasyonuna `for:` parametresi (örn. 3 saniye) eklendi
- Silahlı durumda gereksiz bildirimleri önlemek amacıyla güvenlik modunu eşleştirmek için "koşul: durum"u kullanın
- Kolay toplu işlem için Heiman sensörlerini alana göre gruplandırmak için Home Assistant'ın "grup" işlevini kullanın

## Sorun Giderme

- Cihaz çevrimdışıyken öncelikle Zigbee ağ kanalının WiFi 2,4GHz ile çakışıp çakışmadığını kontrol edin
- Matter cihazı bulunamadığında sınır yönlendiricisini yeniden başlatın ve cihazı yeniden ekleyin
