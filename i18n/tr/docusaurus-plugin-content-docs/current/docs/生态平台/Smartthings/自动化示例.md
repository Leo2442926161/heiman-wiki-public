# Otomasyon örneği

## "İyi Geceler" yatma vakti modu

**Tetikleme koşulu:** Haiman kapı manyetik sensörü → Kapat (yatak odası kapısının kapatıldığını gösterir)

**Eylemi yürütün:**
- Oturma odasındaki tüm ışıkları kapatın
- Akıllı kilit → Kilitle
- Güvenlik sistemi → geceleri devreye alma
- Anında bildirim: "İyi geceler, sistem devreye alındı"

## "Dışarıda" modu

**Tetikleme koşulu:** Hyman insan vücudu kızılötesi sensörü → 15 dakika hareketsizlik

**Eylemi yürütün:**
- Bütün ışıklar kapatıldı
- Termostat → Enerji tasarrufu modu
- Güvenlik sistemi → evden uzakta silahlanma
- Kapı sensörü → alarmı aç
- Anında bildirim: "Evden uzakta modu etkinleştirildi, kapı ve pencere sensörleri etkinleştirildi"

## Duman alarmı acil durum modu

**Tetikleme Durumu:** Hyman Duman Alarmı → Duman Algılandı

**Eylemi yürütün:**
- Akıllı hoparlör → alarm sesini çal
- Tüm akıllı ışıklar → yanıp sönen kırmızı
- Tüm aile üyelerine anlık bildirimler
- SmartThings Hub'a bağlıysa → sireni tetikleyin
