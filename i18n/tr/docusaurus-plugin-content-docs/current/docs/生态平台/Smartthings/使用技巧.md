# Kullanım ipuçları

## Hub Uyumluluğu

Hyman Zigbee cihazları **SmartThings Hub v2** ve **Hub v3 (Aeotec)** ile uyumludur.

| Merkez Modeli | Uyumluluk | Açıklama |
|-----------|-----------|------|
| Hub v2 (STH-ETH-250) | ✅ | En son sürüme donanım yazılımı güncellemesi gerektirir |
| Hub v3 (Aeotec) | ✅ | Yerel destek, önerilir |
| Samsung SmartThings İstasyonu | ⚠️ | Yalnızca bazı Zigbee cihazlarını destekler |

## Cihaz eşleştirme istemleri

- SmartThings Uygulamasında "+" → "Cihaz" → "Marka" → "Hyman" seçeneğine tıklayın
- Otomatik olarak tanınmayan cihazlar "Zigbee'yi Tara" aracılığıyla manuel olarak aranabilir
- İlk eşleştirmede cihaz ile Hub arasındaki mesafenin 5 metre içinde tutulması önerilir.

## Pil optimizasyonu

Heiman sensörünün varsayılan raporlama aralığı daha uzundur. Gerçek zamanlı performans isteniyorsa cihaz sayfasında "algılama aralığı" 5 dakikaya ayarlanabilmektedir.

## En İyi Otomasyon Uygulamaları

- "Evde/evden uzakta" durumları arasında ayrım yapmak için SmartThings'in "Modlar" özelliğini kullanın
- Yedeklilik amacıyla temel sensörler (duman, CO) için **iki set** otomasyon oluşturun
