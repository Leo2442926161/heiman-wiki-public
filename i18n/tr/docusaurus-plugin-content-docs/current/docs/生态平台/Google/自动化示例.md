# Otomasyon örneği

## "Dışarıdayım" rutini

**Tetikleyici:** Google Home'a "Dışarıdayım" deyin

**Eylemi yürütün:**
- Heiman kapı manyetik sensörü → kol
- Heiman insan vücudu kızılötesi sensörü → Evden ayrılma algılamayı açın
- Tüm ışıkları ve aletleri kapatın
- Termostat → Enerji tasarrufu sıcaklığına ayarlayın
- Google Nest Hub → Evden ayrılma onay kartını göster
- Sesli yanıt: "Evden ayrılma modu açıldı, iyi yolculuklar"

## "Duman Algılama" Yayını

**Tetikleyici:** Hyman Duman Alarmı → Duman Algılandı

**Eylemi yürütün:**
- Google Nest Hoparlör → Tüm Ev Yayını "⚠️ Duman algılandı! Lütfen hemen kontrol edin!"
- Tüm akıllı ışıklar → Açık (Google Home akıllı ışık seti aracılığıyla)
- Aile üyelerinin cep telefonlarına anlık bildirimler
- Kamera → Kaydı başlat

## Uyku Vakti modu

**Tetikleyici:** Google Home'a "İyi Geceler" deyin

**Eylemi yürütün:**
- Heiman kapı sensörü → Kapı açık alarmını açın
- Heiman insan vücudu kızılötesi → gece algılamayı açın
- Işıklar → yavaş yavaş kapanıyor
- Sesli yanıt: "İyi geceler, güvenlik sistemi açık"
