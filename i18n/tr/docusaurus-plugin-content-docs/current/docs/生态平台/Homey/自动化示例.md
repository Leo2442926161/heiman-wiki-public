# Otomasyon örneği

## Duman Alarmı - flaş ışıkları ve bildirim

**Akış yapılandırması:**

1. **Tetikleyici:** Hyman Duman Alarmı → Durum "uyarı" olarak değişir
2. **Koşullar:** (Yok)
3. **Eylem:**
   - Oturma odası ışığı → yanıp sönüyor (kırmızı, saniyede bir)
   - Anında Bildirim → Mobil: "⚠️ Duman alarmı tetiklendi!"
   - Sesli duyuru → Ev Gibi Konuşmacı: "Duman algılandı, lütfen güvenliğe dikkat edin!"

## CO Alarmı - Kazanı kapatın

**Akış yapılandırması:**

1. **Tetikleyici:** Hyman CO Alarmı → Durum "uyarı" olarak değişir
2. **Koşullar:** (Yok)
3. **Eylem:**
   - Kazan anahtarı → KAPALI
   - Akıllı priz (temiz hava sistemi) → Açık
   - Anında bildirim → "Karbon monoksit alarmı, kazan kapatıldı"

## Sıcaklık ve nem limit aşımı alarmı

1. **Tetikleyici:** Hyman Sıcaklık ve Nem Sensörü → 40°C'nin üzerindeki sıcaklık
2. **Eylem:**
   - Klima → Soğutma modunu açın
   - Anında bildirim → "Oda sıcaklığı çok yüksek, klima otomatik olarak açıldı"
