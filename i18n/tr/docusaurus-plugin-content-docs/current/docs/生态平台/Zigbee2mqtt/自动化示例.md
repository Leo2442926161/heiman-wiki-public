# Otomasyon örneği

## MQTT abonelik cihazı durumu

``` bash
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
''''

Çıkış örneği:
''''
zigbee2mqtt/heiman_smoke {"duman":true,"pil":100,"bağlantı kalitesi":120}
''''
