# Otomasyon örneği

## Duman alarmı bağlantısı

```yaml
takma ad: "ZHA duman alarmı bağlantısı"
tetikleyiciler:
  - tetikleyici: durum
    varlık_kimliği: ikili_sensor.heiman_smoke_zha
    için: "açık"
eylemler:
  - eylem: light.turn_on
    hedef:
      alan_kimliği: oturma_odası
  - eylem: tts.speak
    hedef:
      varlık_id: media_player.speaker
    veriler:
      mesaj: "Duman alarmı tetiklendi!"
''''

## Cihaz eşleştirme otomasyonu

```yaml
takma ad: "Yeni cihaz eşleştirme hatırlatıcısı"
tetikleyiciler:
  - tetikleyici: zha.event
    alt tür: Heiman
    komut: Device_joined
eylemler:
  - eylem: notify.persistent_notification
    veriler:
      mesaj: "Yeni Haiman cihazı ZHA ağına katıldı, lütfen konfigürasyon adına ve alanına gidin"
''''

```yaml
takma ad: "ZHA Su Baskını Testi"
tetikleyiciler:
  - tetikleyici: durum
    varlık_kimliği: ikili_sensor.heiman_water_leak_zha

  - eylem: notify.mobile_app_phone

      mesaj: "💧 Su baskını algılandı!"
''''
