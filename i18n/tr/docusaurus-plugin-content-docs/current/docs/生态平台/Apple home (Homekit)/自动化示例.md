# Otomasyon örneği

## Duman algılama güvenlik alarmı

**HomeKit Otomatik Yapılandırması:**

1. **Tetikleme durumu:** Hyman duman alarmı → duman algılandı
2. **Güvenlik Ayarları:** Güvenlik sistemi durumunu "Alarm" olarak değiştirin
3. **Eylemi yürütün:**
   - HomePod → Özel ses çalın "Duman algılandı, lütfen güvenliğe dikkat edin"
   - Tüm akıllı ışıklar → yanar ve yanıp söner
   - Tüm aile üyelerine anlık bildirimler

## Evde/evden uzakta sahne

**Kısayol komutu: "Ev Güvenlik Kontrolünden Ayrıl"**

''''
son kişi evden ayrıldığında
  Güvenlik sistemini kurun → evden çıkarken devreye alın
  Heiman Kapı Sensörünü kurun → Bildirimleri açın
  Heiman vücut sensörünü kurun → alarmı açın
  Tüm ışıkları kapat
  Ayarlar Termostat → Enerji Tasarruf Modu
  "🏠 Evden uzakta modu etkinleştirildi" bildirimini gönder
''''

## Sıcaklık anormalliği hatırlatıcısı

**Tetikleme koşulu:** Hyman sıcaklık ve nem sensörü → Sıcaklık > 35°C

**Eylemi yürütün:**
- HomePod → Ses yayını odası sıcaklığı limiti aşıyor
- Bildirimleri iPhone'a itin
- Klimayı otomatik olarak açın (HomeBridge yapılandırılmışsa)
