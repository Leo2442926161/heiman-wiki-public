# Kullanım ipuçları

## Cihaz keşfi

- Heiman Matter cihazı açıldığında, genellikle 30 saniye içinde Alexa Uygulaması tarafından otomatik olarak keşfedilecektir.
- Otomatik olarak keşfedilmezse: Alexa Uygulaması → Cihaz → "+" → "Cihaz Ekle" → "Madde" girin
- Eşleştirmeyi tamamlamak için cihazın altındaki Matter kurulum kodunu veya QR kodunu kullanın

## Zigbee cihaz erişimi (Echo Plus / Echo Show)

- Zigbee Hub'lı Echo cihazları doğrudan Heiman Zigbee sensörleriyle eşleşebilir
- Eşleştirme modu: Alexa Uygulaması → Cihaz → "+" → "Zigbee" → "Cihaz Ara"
- Heimann cihazları yerleşik Zigbee koordinatörüyle Alexa uyumludur

## Rutin İpuçları

- Daha ayrıntılı sesli komutlar yayınlamak için "özel eylemler"i kullanın
- Ses çakışmasını önlemek için sonraki eylemleri gerçekleştirmeden önce acil durum rutinine "5 saniye bekleyin" seçeneğini ekleyin
- Heyman Alarm'ın tiz alarmlarına Alexa'ya ek yanıtlar vermek için "Ses Algılama" özelliğinden yararlanın

## Alexa Muhafız NOTU

Alexa Guard Plus şu anda yalnızca Amerika Birleşik Devletleri'nde mevcuttur. Çin'deki kullanıcılar Rutin aracılığıyla bazı işlevleri simüle edebilir.
