# Otomasyon örneği

## Alexa Guard entegrasyonu (ABD)

**Önkoşul:** Alexa Guard Plus'a abone olun (ABD'de mevcuttur)

**Yapılandırma süreci:**
1. Alexa Uygulamasında Guard Plus'ı açın
2. Hyman Duman/CO Alarmını "Akıllı Ev Cihazı" olarak ekleyin
3. "Duman Alarmı Algılama" akıllı yanıtını etkinleştirin

**Otomatik yanıt:**
- Duman alarmı tetiklendi → Alexa acil durumda iletişime geçilecek kişileri otomatik olarak arar
- CO alarm tetikleyici → Tüm ev için Alexa sesli uyarı, otomatik havalandırma
- Kapılar ve pencereler anormal şekilde açılıyor → Push alarm bildirimi

## Duman algılama rutini

**Tetikleyici:** Hyman Duman Alarmı → Duman Algılandı

**Eylemi yürütün:**
- Alexa sesli anonsu: "⚠️ Duman algılandı, lütfen güvenliğe dikkat edin"
- Tüm Yankı cihazları → Alarm sesini çal
- Akıllı ışık seti → hepsi yanıyor
- Cep telefonuna bildirimleri itin

## "Evden Ayrılma" rutini

**Tetikleyici:** Sesli komut "Alexa, gidiyorum"

**Eylemi yürütün:**
- Heiman kapı manyetik sensörü → devreye almayı açın
- Heiman Vücut Sensörü → Hareket algılamayı açın
- Hyman suya daldırma dedektörü → algılamayı aç
- Tüm ışıklar → kapalı
- Alexa cevapladı: "Güvenlik sistemi açık, iyi yolculuklar"
