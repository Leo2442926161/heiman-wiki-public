# Otomasyon örneği

## Duman alarmı kuralları

```javascript
"Duman alarmı tetiklendi" kuralı
ne zaman
    Heiman_smoke_alarm öğesi AÇIK olarak değiştirildi
o zaman
    sendCommand(Livingroom_Light, ON)
    sendNotification("admin", "⚠️ Duman alarmı tetiklendi!")
    postUpdate(Alarm_State, "duman_alert")
son
''''
