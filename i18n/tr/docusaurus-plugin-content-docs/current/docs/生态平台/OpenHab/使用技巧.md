# Kullanım ipuçları

## Nesne yapılandırması

Heiman cihazını MQTT bağlama yoluyla ekleyin:

```java
Şey mqtt:topic:heiman_smoke "Heiman duman alarmı" @ "oturma odası" {
    Kanallar:
        İkili yazın: duman "duman durumu" [ stateTopic = "zigbee2mqtt/heiman_smoke", transformasyonPattern = "JSONPATH: $.smoke" ]
        Numarayı yazın: pil "Pil gücü" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.battery" ]
}
''''

## Öğe tanımı

```java
Smoke_Alarm ile iletişime geçin "Duman alarmı [MAP(smoke.map):%s]" {channel="mqtt:topic:heiman_smoke:smoke" }
Number Smoke_Battery "Duman alarmı pili [%d%%]" {channel="mqtt:topic:heiman_smoke:battery" }
''''

## Koordinat seçimi

**CC2531** veya **CC2652** çipli bir USB koordinatörü kullanılması ve donanım yazılımı sürümünün en son olduğundan emin olunması önerilir.

## Hata Ayıklama İpuçları

- Paper UI'da Thing'in "çevrimiçi/çevrimdışı" durumunu görüntüleyin
- Bağlamaların doğru yüklenip yüklenmediğini kontrol etmek için 'openhab-cli bilgisi'ni kullanın
- Cihaz iletişim sorunlarını gidermek için günlükte `heiman' anahtar kelimesini arayın
