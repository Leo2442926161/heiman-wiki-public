# UL Underwriters Laboratuvarları

UL (Underwriters Laboratories), Amerika Birleşik Devletleri'ndeki en yetkili ürün güvenliği standardı geliştirme ve sertifikalandırma kuruluşudur. Heiman ürünlerinde yer alan ana UL standartları aşağıdaki gibidir:

## UL 217 — Duman Alarmı Standardı (ABD)

| Proje | Açıklama |
|------|------|
| Uygulama kapsamı | Konut duman alarmları (ABD pazarı) |
| Algılama gereksinimleri | Fotoelektrik/iyonizasyon duman algılama, sesli ve ışıklı alarm, 85 dB'nin üzerinde |
| Uygulanabilir pazarlar | Amerika Birleşik Devletleri, Kanada |
| Heiman Ürünler | HS8SA (ABD versiyonu) |
| Sertifikasyonun Önemi | Amerika Birleşik Devletleri'ndeki çeşitli eyaletlerin yangın yönetmeliklerine uymak, Kuzey Amerika'da pazara erişim için gerekli bir koşuldur |

**Diğer standartlardan farklılıklar**: UL 217'nin test senaryoları, ani yangınlar ve yavaş yanan yangınlar gibi tipik Amerikan ev yangını senaryolarını içerir ve EN 14604'ün test yöntemlerinden biraz farklıdır.

## UL 2034 — Karbon Monoksit Alarm Standardı (ABD)

| Proje | Açıklama |
|------|------|
| Uygulama kapsamı | Ev tipi CO alarmı (ABD pazarı) |
| Algılama gereksinimleri | Elektrokimyasal sensör, konsantrasyon × zaman entegrasyon algoritması |
| Uygulanabilir pazarlar | Amerika Birleşik Devletleri, Kanada |
| Hayman Ürünler | HS-720ES-M (ABD versiyonu) |
| Sertifikasyonun Önemi | Amerika Birleşik Devletleri'ndeki çoğu eyalet CO alarmlarının kurulumunu zorunlu kılmaktadır |

## UL sertifikası ile EN sertifikası arasındaki temel farklar

| Yönleri | UL | TR |
|------|-----|-----|
| Duman alarmı standartları | UL217 | EN 14604 |
| CO alarm standartları | UL 2034 | EN 50291 |
| Algılama yöntemi | Özel yangın eğrisi testi | Farklı yangın tipi testleri |
| Uygulanabilir voltaj | 120V / pil | 230V / pil |
