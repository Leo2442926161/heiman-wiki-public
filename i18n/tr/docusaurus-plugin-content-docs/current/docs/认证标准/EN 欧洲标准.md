# EN Avrupa Standardı

EN (Avrupa Normu, Avrupa Standardı), Avrupa Standardizasyon Komitesi (CEN) ve Avrupa Elektroteknik Standardizasyon Komitesi (CENELEC) tarafından geliştirilen teknik bir standarttır. Heiman ürünlerinde yer alan başlıca EN standartları aşağıdaki gibidir:

## EN 14604 — Duman alarmları standardı

| Proje | Açıklama |
|------|------|
| Uygulama kapsamı | Bağımsız duman alarmı |
| Algılama gereksinimleri | Fotoelektrik duman algılama, sesli ve ışıklı alarm, düşük pil uyarısı |
| Uygulanabilir pazarlar | AB üye ülkeleri, Birleşik Krallık, İsviçre, Norveç vb. |
| Heiman ürünleri | HS1SA-E, HS3SA, HS1SA-M, S1 serisi, vb. |
| Sertifikasyonun Önemi | AB bina yangını düzenlemelerinin gerekliliklerini karşılamak, Avrupa pazarına erişim için gerekli bir koşuldur |

**Test öğeleri**: tepki süresi, ses basıncı seviyesi (≥85 dB/3m), sessize alma işlevi, pil ömrü, sıcaklık döngüsü, nem dayanıklılığı, radyo frekansı girişimine karşı direnç vb.

## EN 50291 — Karbon monoksit alarm standardı

| Proje | Açıklama |
|------|------|
| Uygulama kapsamı | Ev karbon monoksit (CO) alarmı |
| Test gereksinimleri | Elektrokimyasal CO sensörü, konsantrasyon sınıflandırma alarmı |
| Uygulanabilir pazarlar | AB üye ülkeleri, Birleşik Krallık |
| Heiman ürünleri | HS1CA-E, C1 serisi, HS1CA-M, vb. |
| Sertifikasyonun Önemi | CO sızıntısı tespitinin güvenilirliğini sağlamak için bazı ülkelerde kurulum zorunludur |
