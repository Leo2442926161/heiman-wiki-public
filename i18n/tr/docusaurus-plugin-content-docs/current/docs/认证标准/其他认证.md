# Diğer sertifikalar

## Q-Label kalite işareti

Q-Label, Q-Lab Alliance (şu anda Zigbee Alliance / CSA Connectivity Standards Alliance'ın bir parçası) tarafından başlatılan, Avrupa akıllı ev endüstrisine yönelik bir kalite sertifikasyon markasıdır.

| Proje | Açıklama |
|------|------|
| Uygulama kapsamı | Zigbee cihazı birlikte çalışabilirlik sertifikası |
| Sertifika içeriği | Diğer Zigbee ağ geçitleri/kontrolör markalarıyla cihaz uyumluluğu |
| Önemi | Markalar arası birlikte çalışabilirliği sağlayın ve "evden satın alıp kullanmama" riskini azaltın |

Hyman Zigbee ürünleri Q-Label sertifikalıdır ve aşağıdaki platformlarla uyumluluğu garanti eder:
- Amazon Echo Plus / Echo Gösterisi (Zigbee yerleşik)
- Philips Hue Köprüsü
- SmartThings Merkezi
- Diğer Q-Label kimlik doğrulama ağ geçitleri

## Red Dot Tasarım Ödülü

Hyman'ın HS1SA serisi duman alarmı, Hyman'ın ürün tasarımındaki gücünü yansıtan **Red Dot Tasarım Ödülü**'nü (Red Dot Tasarım Ödülü) kazandı.

## ISO 9001 Kalite Yönetim Sistemi

Haiman, Ar-Ge'den üretime ve satış sonrasına kadar tam süreç kalite kontrolünü sağlamak için ISO 9001:2015 kalite yönetim sistemi sertifikasını geçti.
