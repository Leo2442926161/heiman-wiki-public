# Çözüm Dağıtımı

Heiman Teknoloji, uçtan uca akıllı güvenlik çözümü dağıtım rehberliği sağlar.

## Entegrasyon Süreci

Heiman ürünleri, yerel bağımsız dağıtım ve bulut platformu entegrasyonu dahil olmak üzere birden çok dağıtım yöntemini destekler.

### Yerel Dağıtım

```bash
# Heiman Akıllı Yangın Ağ Geçidini Yapılandır
heiman-cli configure --gateway HS1GW --mode local

# Dedektör cihazını kaydet
heiman-cli register --device HS1SA --gateway HS1GW
```

### Bulut Platformu Entegrasyonu

Heiman ürünleri başlıca IoT platformlarıyla entegredir:

- China Telecom Tianyi
- China Mobile OneNET
- Huawei HiLink
- Alibaba Cloud IoT
- Tuya Smart

## Matter Protokol Desteği

Heiman Matter serisi ürünler şunlarla uyumludur:

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

Çapraz marka, çapraz ekosistem birlikte çalışabilirliği sağlar.

## Dağıtım Doğrulaması

Dağıtımdan sonra, işlevsel doğrulama ve gerçek zamanlı izleme için Heiman Akıllı Yangın Uygulamasını veya yönetim platformunu kullanın.
