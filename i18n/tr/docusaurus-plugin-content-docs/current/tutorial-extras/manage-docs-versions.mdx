# Doküman Versiyon Yönetimi

Heiman Teknoloji Wiki, çoklu versiyon dokümantasyon yönetimini destekleyerek ürün yazılım sürümünüzle eşleşen teknik dokümantasyona erişim sağlar.

## Versiyon Karşılaştırması

| Doküman Versiyonu | Uyumlu Yazılım | Yayın Tarihi |
|------------|-------------------|-------------|
| 2.0        | Matter Serisi      | 2025-06     |
| 1.0        | Zigbee Serisi      | 2024-01     |

## Farklı Versiyonları Görüntüleme

Farklı doküman versiyonları arasında geçiş yapmak için üst gezinme çubuğundaki versiyon açılır menüsünü kullanın.

## Versiyon Güncellemeleri

Ürün yinelemelerine uyum sağlamak için dokümantasyonu sürekli güncelliyoruz. En son sürüm için lütfen düzenli olarak kontrol edin.
