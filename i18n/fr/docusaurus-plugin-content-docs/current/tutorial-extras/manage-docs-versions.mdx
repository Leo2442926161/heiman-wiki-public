# Gestion des Versions de Documents

Heiman Technology maintient plusieurs versions de documentation pour correspondre aux différentes versions de produits et micrologiciels.

## Versions Disponibles

| Version | Description |
| --- | --- |
| 2.0 | Dernière version recommandée avec fonctionnalités Matter |
| 1.5 | Version stable pour les systèmes existants |
| 1.0 | Version initiale avec fonctionnalités de base |

## Politique de Version

- **Version Majeure** — Changements importants de produit ou de protocole qui peuvent affecter la compatibilité
- **Version Mineure** — Nouvelles fonctionnalités et améliorations avec compatibilité ascendante
- **Correctif** — Corrections de bugs et mises à jour de sécurité

## Sélectionner une Version

Utilisez le menu déroulant de sélection de version en haut de la page pour passer à une version différente de la documentation. Par défaut, la dernière version stable sera affichée.
