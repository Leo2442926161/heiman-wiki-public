# Support Multilingue

Le Wiki Heiman Technology prend en charge plusieurs langues pour servir notre base d'utilisateurs mondiale.

## Langues Actuellement Disponibles

- **English** (Anglais)
- **简体中文** (Chinois Simplifié)
- **Français** (Français)

## Comment Changer de Langue

Utilisez le sélecteur de langue dans la barre de navigation en haut de la page pour passer à votre langue préférée. Le contenu sera affiché dans la langue sélectionnée si la traduction est disponible.

## Contribution aux Traductions

Nous accueillons les contributions de la communauté pour les traductions ! Si vous souhaitez aider à traduire la documentation dans d'autres langues, veuillez [nous contacter](https://www.heiman.com.cn) pour plus d'informations.

Les fichiers de traduction sont situés dans le répertoire `i18n/<code-de-langue>/` et utilisent le format JSON pour les chaînes de l'interface utilisateur et les fichiers MDX pour le contenu de la documentation.
