#Liste des modèles pris en charge

Les produits de la série longue distance Haiman Sub-1G sont principalement conçus pour des scénarios de réseau indépendants à grande échelle et peuvent réaliser l'interconnexion et l'interopérabilité entre les appareils sans passerelle.

## Détecteur de fumée autonome (série S1)

| Modèle | Nom du produit | Descriptif |
|------|----------|------|
| S1-SA | Avertisseur de fumée autonome (Sub-1G) | Type de base, prend en charge l'interconnexion sans fil |
| S1-SA-P | Avertisseur de fumée autonome (Sub-1G) | Version avec fonction caméra |
| S1-SA-T | Détecteur de fumée indépendant (Sub-1G) | Version liaison haute température |
| S1-SA-RF | Détecteur de fumée indépendant (type interconnecté RF) | Version réseau RF |

## Accessoires connectés

| Modèle | Nom du produit | Descriptif |
|------|----------|------|
| S1-RM | Répéteur sans fil (Sub-1G) | Distance de communication étendue |
| S1-RC | Télécommande (Sub-1G) | Pour tester/Muet/Réinitialiser |

## Fonctionnalités

- **Travail indépendant** : Pas de passerelle, pas d'Internet requis, interconnexion sans fil directe entre les appareils
- **Alarme interconnectée** : n'importe quel appareil déclenche une alarme et tous les appareils du même groupe déclencheront une alarme simultanément.
- **Ultra-longue distance** : la distance de communication peut atteindre 1 à 3 kilomètres dans un environnement ouvert
- **Consommation d'énergie ultra faible** : alimenté par batterie, autonomie jusqu'à 3 à 5 ans
