Série # S1 - Détecteur de fumée autonome (Sub-1G)

## Aperçu

La série S1 est un détecteur de fumée Sub-1G autonome lancé par Hyman. Il fonctionne dans la bande de fréquences 433 MHz/868 MHz, utilise une technologie de détection de fumée photoélectrique et a passé la certification EN 14604. Aucune passerelle ou Internet n'est requis. Les appareils sont directement interconnectés via la fréquence radio Sub-1G. Lorsqu'un appareil déclenche une alarme, tous les appareils du même groupe déclenchent une alarme simultanément. Il convient aux scénarios d’incendie à grande échelle tels que les bâtiments, les centres commerciaux et les hôtels.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | 433 MHz / 868 MHz Sub-1G (selon la zone de marché) |
| Type de capteur | Photoélectrique infrarouge |
| Pression sonore de l'alarme | ≥85 dB (3 mètres) |
| Méthode d'alimentation | Alimenté par batterie (pile au lithium CR123A / AA) |
| Autonomie de la batterie | ≥ 5 ans |
| Distance de communication | Environ 1 à 3 kilomètres en environnement ouvert |
| Capacité du réseau | Un seul groupe prend en charge jusqu'à 50+ nœuds |
| Méthode d'interconnexion | Interconnexion radiofréquence sans fil (aucune passerelle requise) |
| Température de fonctionnement | -10°C ~ +55°C |
| Humidité de travail | ≤ 95 % HR |
| Méthode d'installation | Pose au plafond |

## Fonctionnalités

- Détection de fumée photoélectrique pour détecter la fumée provenant de brûlures lentes et de flammes nues
- Alarme d'interconnexion sans fil : déclenchée par n'importe quel appareil du même réseau, tous les appareils émettent une alarme sonore et lumineuse simultanément.
- Mode de travail indépendant : pas de passerelle, pas d'Internet requis, communication directe entre appareils
- Alarme sonore et lumineuse élevée de 85 dB
- Rappel de batterie faible
- Fonction d'autotest (test en un clic)
- Alarme anti-effraction
-Supporte la sourdine à distance
- Conception à très faible consommation d'énergie, autonomie de la batterie de plus de 5 ans

## Normes de certification

- EN 14604 (norme européenne pour les détecteurs de fumée)
-CE/ROUGE
- CPR (Règlement UE Produits de Construction)

## Accessoires applicables

- Répéteur sans fil S1-RM (distance de communication étendue)
- Télécommande S1-RC (test/muet/réinitialisation)
