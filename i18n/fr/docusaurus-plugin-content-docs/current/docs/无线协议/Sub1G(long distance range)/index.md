# Sub-1G (communication longue distance)

Sub-1G est une technologie de communication sans fil fonctionnant dans la bande de fréquences inférieure à 1 GHz. Il présente les caractéristiques d’une distance de transmission ultra longue et d’une consommation d’énergie extrêmement faible. Il est très adapté aux scénarios IoT qui nécessitent une couverture à grande échelle.

## Fonctionnalités

| Caractéristiques | Descriptif |
|------|------|
| Bande de fréquence de fonctionnement | 433 MHz / 868 MHz (UE) / 915 MHz (États-Unis) |
| Distance de transmission | Jusqu'à 1-3 kilomètres en environnement ouvert |
| Capacité de pénétration | Mieux que 2,4 GHz, forte pénétration dans les obstacles muraux |
| Consommation d'énergie | Extrêmement faible, la durée de vie de la batterie peut atteindre 5 à 10 ans |
| Taux de transmission | Faible (adapté aux scénarios de petits volumes de données) |

## Scénarios d'application

- **Détecteur de fumée/CO indépendant** : interconnecté entre les étages du bâtiment
- **Système de protection incendie des grands bâtiments** : centres commerciaux, hôtels, écoles
- **Smart Senior Living Community** : appel d'urgence inter-unités
- **Sub-1G + Wi-Fi Gateway** : le sous-appareil communique via Sub-1G et la passerelle se connecte au cloud
