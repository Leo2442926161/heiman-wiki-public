# Paramètres techniques

Voici les spécifications techniques générales des appareils Hyman Z-Wave.

## Paramètres sans fil

| Paramètre | Valeur |
|------|-----|
| Normes de protocole | Z-Wave Plus (ITU-T G.9959) |
| Bande de fréquence de travail | 800~900 MHz (selon la région : réglementation européenne 868,42 MHz, réglementation américaine 908,42 MHz) |
| Taux de transmission | Jusqu'à 100 kbit/s |
| Topologie du réseau | Réseau maillé d'auto-réparation (routage source) |
| Capacité du nœud | Maximum 232 nœuds dans un seul réseau |
| Distance de communication | Environ 30 mètres à l'intérieur (entre les nœuds), couverture maillée dans toute la maison |
| Méthode de cryptage | AES-128 (authentifié S2 / non authentifié S2) |

## Bande de fréquence régionale

| Région | Fréquence centrale |
|------|----------|
| Europe | 868,42 MHz |
| Amérique du Nord | 908,42 MHz |
| Autres domaines | Soumis aux réglementations locales |

## Consommation électrique et électrique

| Paramètre | Valeur |
|------|-----|
| Méthode d'alimentation | Alimentation par pile (CR123A/AA) ou secteur |
| Autonomie de la batterie | Environ 2 à 3 ans pour les capteurs |
| Puissance de transmission | ≤ 1 mW (conforme à la réglementation SRD) |
| Recevez la sensibilité | Environ -100 dBm |

## Paramètres d'environnement

| Paramètre | Valeur |
|------|-----|
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de fonctionnement | ≤ 90 % RH (sans condensation) |

## Caractéristiques du réseau

| Caractéristiques | Descriptif |
|------|------|
| Démarrage intelligent | Support (série 700), scannez le code QR pour rejoindre le réseau |
| Sécurité S2 | Prend en charge trois niveaux de sécurité : non chiffré, S2 non authentifié, S2 authentifié |
| Contient CC | Security CC, Security 2 CC, Firmware Update CC, Basic CC, Battery CC, Notification CC et plus encore |
