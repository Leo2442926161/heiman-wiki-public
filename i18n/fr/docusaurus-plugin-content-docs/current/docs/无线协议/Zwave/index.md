#Z-Wave

Z-Wave est un protocole réseau maillé sans fil conçu pour les maisons intelligentes. Il est géré par la Z-Wave Alliance et bénéficie d'une large pénétration sur les marchés européens et américains.

## Spécifications techniques

| Caractéristiques | Descriptif |
|------|------|
| Bande de fréquence de fonctionnement | 868,42 MHz (UE) / 908,42 MHz (États-Unis) |
| Taux de transfert | Jusqu'à 100 kbps (série Z-Wave 800) |
| Topologie du réseau | Réseau maillé, jusqu'à 232 nœuds |
| Compatibilité | Rétrocompatible avec tous les appareils Z-Wave |
| Sécurité | Cryptage Z-Wave S2 |

## Hyman et Z-Wave

Hyman propose une variété de capteurs compatibles Z-Wave, principalement vendus sur les marchés européens et américains. Les versions Z-Wave des appareils Heyman incluent :

- Capteur de monoxyde de carbone HS1CA-Z
- Capteur de fumée HS1SA-Z
- Capteur de porte HS1DS-Z

## Plateformes compatibles

- Centre SmartThings
- Altitude Hubitat
- Homey Pro
- Assistant à domicile (via Z-Wave JS)
-OuvertHAB

> **Remarque** : Z-Wave a une faible popularité sur le marché chinois et les produits Haiman Z-Wave sont principalement destinés aux marchés d'exportation européens et américains.
