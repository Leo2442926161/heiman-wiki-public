# HS1CA-Z Alarme intelligente de monoxyde de carbone (Z-Wave)

HS1CA-Z est un avertisseur intelligent de monoxyde de carbone **Z-Wave** lancé par Hyman. Il utilise la technologie des capteurs électrochimiques et est compatible avec la norme Z-Wave Plus.

## Présentation du produit

- **Type de produit** : Alarme intelligente de monoxyde de carbone
- **Protocole de communication** : Z-Wave Plus
- **Norme de certification** : EN 50291
- **Scénario typique** : avertissement de fuite de CO dans les maisons, les appartements et autres lieux

## Paramètres techniques

| Paramètre | Valeur |
|------|-----|
| Type de capteur | Capteur de CO électrochimique |
| Protocole de communication | Z-Wave Plus (ITU-T G.9959) |
| Bande de fréquence de fonctionnement | 868,42 MHz (UE) / 908,42 MHz (États-Unis) |
| Plage de détection | 0 ~ 1 000 ppm |
| Pression sonore de l'alarme | ≥85 dB (3 mètres) |
| Méthode d'alimentation | Alimenté par batterie |
| Durée de vie du capteur | Typique 7 ans |
| Méthode de cryptage | S2 authentifié / S2 non authentifié |
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de travail | ≤90 % RH (sans condensation) |

## Fonctionnalités

- Capteur de CO électrochimique de haute précision
- Alarme de classification de concentration
- Alarme sonore et lumineuse élevée de 85 dB
- Rappel de batterie faible
- Rappel d'expiration de la durée de vie du capteur
- Fonction d'autotest
- Cryptage sécurisé Z-Wave S2
- Prise en charge de la mise à niveau du micrologiciel OTA

## Certifications

- Certifié EN 50291
- Certifié Z-Wave Plus
-Certification de sécurité S2
