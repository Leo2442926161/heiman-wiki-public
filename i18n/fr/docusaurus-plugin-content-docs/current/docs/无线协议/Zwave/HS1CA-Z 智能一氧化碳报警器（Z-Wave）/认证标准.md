# Normes de certification

HS1CA-Z a passé les certifications suivantes :

- **EN 50291** — Norme européenne pour les avertisseurs de monoxyde de carbone, voir [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **Certification Z-Wave S2** — Prend en charge le cadre de sécurité S2, voir [Autres certifications](../../../Certification Standards/Other Certifications.md)
- **CE** — Déclaration de conformité de l'Union européenne
