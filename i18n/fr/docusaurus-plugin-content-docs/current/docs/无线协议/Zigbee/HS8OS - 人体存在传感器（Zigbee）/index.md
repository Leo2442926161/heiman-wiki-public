#HS8OS - Capteur de présence humaine (Zigbee)

## Aperçu

HS8OS est un capteur de présence humaine Zigbee lancé par Hyman. Il utilise la technologie radar à ondes millimétriques et peut détecter la présence d'un corps humain dans un état statique (comme s'allonger, s'asseoir tranquillement), compensant ainsi les défauts des capteurs PIR traditionnels qui ne peuvent détecter que des mouvements.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Type de capteur | Radar à ondes millimétriques (mmWave) |
| Plage de détection | Rayon 3 ~ 5 m |
| Capacité de détection | Présence d'un corps humain stationnaire (y compris détection de micro-respiration) |
| Méthode d'alimentation | Alimentation CA |
| Méthode d'installation | Pose au plafond |

## Fonctionnalités

- Détection du corps humain stationnaire (micro-mouvements respiratoires)
- Rayon de couverture 3 ~ 5 mètres
- Installation au plafond, angle de détection optimal
- Peut être utilisé en complément des capteurs PIR

## Scénarios applicables

- Éclairage intelligent : toujours allumé lorsqu'il y a quelqu'un, s'éteignant automatiquement lorsqu'il n'y a personne
- Économie d'énergie de la climatisation : éteint automatiquement le climatiseur lorsque les gens marchent
- Soins intelligents aux personnes âgées : alerte précoce pour les personnes âgées si elles restent stationnaires pendant une longue période
