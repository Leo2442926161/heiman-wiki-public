## Instructions de dénomination du modèle

| Règles de dénomination | Signification |
|--------------|------|
| Suffixe -E | Version UE (868 MHz / norme EN) |
| Suffixe -Z | Version chinoise (2,4 GHz / norme chinoise) |
| HS1/2/3/8 | Numéro de génération de produit, plus le nombre est grand, plus la génération est récente |
