# HS8MLS - Micro capteur de mouvement (Zigbee)

## Aperçu

HS8MLS est un capteur infrarouge (PIR) du corps humain Zigbee à longue portée lancé par Heiman. Il est spécialement conçu pour les scènes nécessitant une détection longue distance et convient aux espaces ouverts tels que les couloirs et les halls.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Type de capteur | Infrarouge passif (PIR) |
| Distance de détection | Détection longue distance |
| Méthode d'alimentation | Batterie DC 3V |

## Fonctionnalités

- Détection de mouvement longue distance
- Conception à faible consommation d'énergie
- Prise en charge de la liaison de scène
