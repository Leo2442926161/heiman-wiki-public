# HS1CA-E - Avertisseur intelligent de monoxyde de carbone (Zigbee)

## Aperçu

HS1CA-E est l'alarme intelligente de monoxyde de carbone (CO) Zigbee de Hyman destinée au marché européen. Il utilise un capteur électrochimique et peut détecter avec précision le monoxyde de carbone avec une concentration de 0 à 1 000 ppm. Il a passé la certification EN 50291.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Type de capteur | Électrochimique |
| Plage de détection | 0 ~ 1 000 ppm |
| Pression sonore de l'alarme | ≥85 dB (3 mètres) |
| Méthode d'alimentation | C.C 3 V (CR123A) |
| Durée de vie du capteur | Environ 7 ans |
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de travail | ≤95 % HR |
| Méthode d'installation | Montage mural |

## Fonctionnalités

- Capteur de CO électrochimique de haute précision
- Alarme de concentration graduée (invite sonore et lumineuse)
- Son d'alarme de 85 dB
- Rappel de batterie faible
- Rappel d'expiration de la durée de vie du capteur
- Fonction d'autotest
-Mise à jour du micrologiciel OTA

## Normes de certification

- EN 50291 (norme européenne pour les avertisseurs de monoxyde de carbone)
-CE/ROUGE

## Accessoires applicables

- Pile CR123A
