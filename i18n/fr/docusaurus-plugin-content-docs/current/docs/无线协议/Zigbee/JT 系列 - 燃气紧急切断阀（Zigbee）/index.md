# Série JT - Vanne d'arrêt d'urgence gaz (Zigbee)

## Aperçu

La vanne d'arrêt d'urgence au gaz de la série JT est un dispositif d'exécution utilisé conjointement avec le détecteur de gaz combustible Hyman Zigbee. Lorsqu'une fuite de gaz est détectée, il peut fermer à distance ou automatiquement la vanne du gazoduc pour éviter les accidents.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Tapez | Liaison autonome + Zigbee |
| Gaz d'essai | Gaz naturel / Gaz de pétrole liquéfié |
| Normes de certification | GB 15322 |
| Température de fonctionnement | -10°C ~ +50°C |

## Fonctionnalités

- Lié à HS1CG / HS3CG pour fermer automatiquement la vanne
-Supporte la fermeture manuelle de la vanne à distance
- Poussée d'alarme de fuite
- Pilote d'électrovanne

## Normes de certification

-GB 15322

## Accessoires applicables

- Détecteur de gaz combustible HS1CG / HS3CG
