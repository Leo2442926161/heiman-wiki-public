#HS1MS - Capteur infrarouge du corps humain (Zigbee)

## Aperçu

HS1MS est un capteur infrarouge (PIR) du corps humain Zigbee standard lancé par Hyman. Il utilise une technologie de détection infrarouge passive et peut détecter le rayonnement thermique infrarouge émis par un corps humain en mouvement. Il est largement utilisé dans les scénarios d’éclairage intelligent et de sécurité.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Type de capteur | Infrarouge passif (PIR) |
| Angle de détection | 110° |
| Distance de détection | ≤7 m (25°C) |
| Méthode d'alimentation | Batterie au lithium CC 3 V |
| Autonomie de la batterie | Environ 3 ans |
| Température de fonctionnement | -10°C ~ +50°C |
| Méthode d'installation | Montage mural |

## Fonctionnalités

- Détection humaine mobile
- Détection grand angle 110°
- Rappel de batterie faible
- Alarme anti-effraction
- Prise en charge de la liaison de scène
