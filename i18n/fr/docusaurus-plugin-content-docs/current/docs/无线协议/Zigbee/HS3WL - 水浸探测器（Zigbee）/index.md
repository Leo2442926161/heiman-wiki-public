# HS3WL - Détecteur de fuite d'eau (Zigbee)

## Aperçu

HS3WL est le détecteur d'immersion dans l'eau Zigbee de troisième génération d'Hyman. Il présente une sensibilité de détection et des performances sans fil améliorées basées sur la génération précédente et convient aux scénarios de surveillance des fuites d'eau.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Méthode de détection | Détection de sonde |
| Méthode d'alimentation | Batterie DC 3V |
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de travail | ≤95 % HR |

## Fonctionnalités

- Détection en temps réel des fuites d'eau/accumulation d'eau
- Sensibilité de détection améliorée
- Alarme sonore
- Rappel de batterie faible
- Conception anticorrosion de la sonde
