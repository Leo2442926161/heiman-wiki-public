# HS6GW - Passerelle intelligente (Zigbee)

## Aperçu

HS6GW est une passerelle intelligente Zigbee lancée par Heiman. En tant que coordinateur du réseau Zigbee de Heiman, il se connecte à Internet via une liaison montante Wi-Fi ou Ethernet et prend en charge l'accès à jusqu'à 128 sous-appareils.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 (Coordinateur) |
| Connexion de liaison montante | Wi-Fi/Ethernet |
| Nombre maximum de nœuds | 128 sous-appareils |
| Méthode d'alimentation | Alimentation USB CC 5 V |

## Fonctionnalités

- Coordinateur du réseau Zigbee
- Connexion liaison montante Wi-Fi/Ethernet
- Découverte et couplage automatiques des appareils
- Lien avec la scène locale
- Accès à distance (via Heiman Cloud Platform)
- Prend en charge jusqu'à 128 sous-appareils
-Mise à jour du micrologiciel OTA

## Scénarios applicables

- Le point de connexion central de l'intelligence Heiman pour toute la maison
- Coopérer avec Haiman APP pour réaliser une surveillance à distance
