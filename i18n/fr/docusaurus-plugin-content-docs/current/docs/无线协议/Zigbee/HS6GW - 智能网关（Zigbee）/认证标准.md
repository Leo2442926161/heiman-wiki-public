# Normes de certification

HS6GW (version Zigbee) a passé les certifications suivantes :

- **CE/RED** — Directive de l'Union européenne sur les équipements sans fil
- **CCC** — Certification obligatoire des produits en Chine, voir [Certification obligatoire CCC Chine](../../../../Certification Standards/CCC China Compulsory Certification.md)
