# HS1WL - Détecteur de fuite d'eau (Zigbee)

## Aperçu

HS1WL est le détecteur d'immersion dans l'eau Zigbee de première génération lancé par Hyman. Il adopte une méthode de détection de sonde et peut détecter efficacement les fuites d'eau ou l'accumulation d'eau. Il s'agit d'un équipement de surveillance de base de l'immersion dans l'eau.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Méthode de détection | Détection de sonde |
| Méthode d'alimentation | Batterie DC 3V |
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de travail | ≤95 % HR |

## Fonctionnalités

- Détection en temps réel des fuites d'eau/accumulation d'eau
- Alarme sonore
- Rappel de batterie faible
- Conception anticorrosion de la sonde
