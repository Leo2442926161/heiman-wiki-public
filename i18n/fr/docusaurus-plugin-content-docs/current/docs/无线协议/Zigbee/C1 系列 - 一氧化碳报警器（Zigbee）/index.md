# Série C1 - Avertisseur de monoxyde de carbone (Zigbee)

## Aperçu

La série C1 est un avertisseur de monoxyde de carbone Zigbee de base lancé par Heiman. Il utilise des capteurs électrochimiques et convient aux scénarios de détection de fuites de monoxyde de carbone. Il a passé la certification EN 50291.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Type de capteur | Électrochimique |
| Plage de détection | 0 ~ 1 000 ppm |
| Pression sonore de l'alarme | ≥85 dB (3 mètres) |
| Méthode d'alimentation | Alimenté par batterie |
| Durée de vie du capteur | Environ 7 ans |
| Température de fonctionnement | -10°C ~ +50°C |

## Fonctionnalités

- Détection électrochimique du CO
- Alarme de classification de concentration
- Alarme sonore et visuelle de 85 dB
- Rappel de batterie faible
- Fonction d'autotest

## Normes de certification

-EN 50291
