#HS8SA - Détecteur de fumée intelligent (Zigbee)

## Aperçu

HS8SA est le détecteur de fumée intelligent Zigbee + Wi-Fi de Hyman pour le marché américain. Il dispose d'une fonction de liaison à haute température et a passé la certification UL 217.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 / Wi-Fi |
| Type de capteur | Photoélectrique infrarouge |
| Pression sonore de l'alarme | ≥85 dB (3 mètres) |
| Méthode d'alimentation | C.C 3 V (CR123A) |
| Température de fonctionnement | -10°C ~ +50°C |
| Normes de certification | UL217 |
| Marchés applicables | États-Unis |

## Fonctionnalités

- Détection de fumée photoélectrique
-Alarme de liaison haute température
- Alarme sonore et lumineuse élevée de 85 dB
- Rappel de batterie faible
- Fonction d'autotest
- Alarme anti-effraction
-Supporte la sourdine à distance
- Communication double protocole (Zigbee + Wi-Fi)

## Normes de certification

- UL 217 (norme américaine sur les détecteurs de fumée)
