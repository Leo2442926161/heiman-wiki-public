# Normes de certification

HS8SA a passé les certifications suivantes :

- **EN 14604** — Norme européenne pour les détecteurs de fumée (type UE), voir [EN European Standard](../../../../ApprovalStandards/EN European Standard.md)
- **UL 217** — Norme américaine de sécurité des détecteurs de fumée (modèle américain), voir [UL Underwriters Laboratories](../../../../certification-standards/ULUnderwriters Laboratories.md)
- **CE/RED/EMC** — Directive sur les équipements sans fil de l'Union européenne et compatibilité électromagnétique
- **Zigbee 3.0 / Q-Label** — Certification Zigbee Alliance, voir [Autres certifications](../../../Certification Standards/Autres certifications.md)
