#HS2FD - Détecteur de chute (Zigbee)

## Aperçu

HS2FD est un détecteur de chute intelligent Zigbee lancé par Hyman. Il est spécialement conçu pour les personnes âgées. Il dispose d'un accéléromètre et d'un gyroscope intégrés. Il utilise un algorithme d'IA pour identifier la posture en cas de chute et peut envoyer automatiquement un signal de détresse.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Type de capteur | Accéléromètre + Gyroscope |
| Technologie de détection | Algorithme de détection de chute AI |
| Méthode d'alimentation | Batterie DC 3V |
| Méthode de port | Clip de ceinture/longe |

## Fonctionnalités

- Détection automatique des chutes
- Appeler à l'aide manuellement en un seul clic
- L'algorithme d'IA distingue les chutes des activités normales (s'asseoir, se pencher) pour réduire les fausses alarmes
- Conception portable (clip de ceinture ou cordon)
- Rappel de batterie faible

## Scénarios applicables

- Soins à domicile pour personnes âgées : soins 24 heures sur 24 pour les personnes âgées vivant seules
- Maisons de retraite : garde et gestion centralisées
- Centre de rééducation : Surveillance du risque de chute chez les patients postopératoires
