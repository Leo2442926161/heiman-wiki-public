# HS1SA-E - Détecteur de fumée intelligent (Zigbee)

## Aperçu

HS1SA-E est un détecteur de fumée intelligent Zigbee standard lancé par Hyman pour le marché européen. Il adopte une technologie de détection de fumée photoélectrique et a passé la certification EN 14604 et a remporté le Red Dot Design Award.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Type de capteur | Photoélectrique infrarouge |
| Pression sonore de l'alarme | ≥85 dB (3 mètres) |
| Méthode d'alimentation | C.C 3 V (CR123A) |
| Autonomie de la batterie | Environ 3 ans |
| Distance de communication | ≤70 m (aire ouverte) |
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de travail | ≤95 % HR |
| Méthode d'installation | Pose au plafond |

## Fonctionnalités

- Détection de fumée photoélectrique pour détecter la fumée provenant de brûlures lentes et de flammes nues
- Alarme sonore et lumineuse élevée de 85 dB
- Rappel de batterie faible
- Fonction d'autotest (test en un clic)
- Alarme anti-effraction
-Supporte la sourdine à distance
-Mise à jour du micrologiciel OTA

## Normes de certification

- EN 14604 (norme européenne pour les détecteurs de fumée)
- Prix Red Dot Design
-CE/ROUGE

## Accessoires applicables

- Pile CR123A
- Vis de montage/tube d'expansion
