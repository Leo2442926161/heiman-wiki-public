# M1-Z - Capteur infrarouge du corps humain (Zigbee)

## Aperçu

M1-Z est un capteur infrarouge compact Zigbee pour le corps humain lancé par Heiman. Il prend en charge un angle de détection de 110° ou 360° (selon la méthode d'installation), dispose d'un capteur d'éclairage intégré et convient aux scénarios d'éclairage et de sécurité intelligents.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | Zigbee 3.0 |
| Type de capteur | Infrarouge passif (PIR) + éclairage |
| Angle de détection | 110° (mural) / 360° (plafond) |
| Distance de détection | ≤7 m (25°C) |
| Plage d'éclairement | 0 ~ 1200 Lux |
| Méthode d'alimentation | Batterie au lithium CC 3 V |
| Autonomie de la batterie | Environ 3 ans |
| Température de fonctionnement | -10°C ~ +50°C |
| Méthode d'installation | Mural/plafond |

## Fonctionnalités

- Détection humaine mobile
- Détection d'éclairage (0 ~ 1200 Lux)
- Commutable 110° / 360°
- Rappel de batterie faible
- Alarme anti-effraction
- Conception compacte
