# Capteur infrarouge du corps humain M1-M (Matière)

M1-M est un capteur infrarouge de corps humain **Matter over Thread** lancé par Hyman. Il intègre des fonctions de détection PIR et de détection d'éclairage et convient aux scénarios de domotique intelligente.

## Présentation du produit

- **Type de produit** : Capteur infrarouge du corps humain
- **Protocole de communication** : l'affaire passe avant le fil de discussion
- **Méthode de détection** : Infrarouge passif (PIR) + détection d'éclairage
- **Scénario typique** : Les lumières s'allument lorsque des personnes arrivent, lien de sécurité, automatisation environnementale

## Paramètres techniques

| Paramètre | Valeur |
|------|-----|
| Angle de détection | 110° (mural) / 360° (plafond) |
| Distance de détection | ≤7 m (25°C) |
| Plage d'éclairage | 0~1200 Lux |
| Protocole de communication | La question avant le fil (IEEE 802.15.4) |
| Bande de fréquence de fonctionnement | 2,4 GHz |
| Méthode d'alimentation | Batterie au lithium CC 3 V |
| Autonomie de la batterie | Environ 1 an (mode Matter) |
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de travail | ≤95 % RH (sans condensation) |

## Fonctionnalités

- Détection humaine mobile
- Détection de l'éclairage ambiant
- Distance de détection ≤7m
- Support d'installation au mur ou au plafond
- Interopérabilité multi-plateforme multi-écologique
