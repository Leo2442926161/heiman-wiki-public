#Matière

Matter (anciennement connu sous le nom de Project CHIP) est une norme d'interconnexion pour les maisons intelligentes lancée par la CSA Connection Standards Alliance, visant à éliminer les barrières des marques et à parvenir à une interopérabilité interécologique.

## Heyman et la matière

Hyman a été l'un des premiers à adopter le protocole Matter, en lançant **les premiers avertisseurs de monoxyde de carbone certifiés Matter** au monde. Les appareils Matter de Hyman utilisent tous la solution **Matter over Thread**.

## Caractéristiques techniques

| Caractéristiques | Descriptif |
|------|------|
| Protocole sous-jacent | Thread (réseau maillé basse consommation) |
| Topologie du réseau | Réseau maillé d'auto-réparation, plus il y a d'appareils, plus la couverture est large |
| Consommation d'énergie | Extrêmement faible, peut fonctionner pendant 1 à 3 ans sur batterie |
| Sécurité | Prend en charge le cryptage AES-128-CCM, authentification de l'appareil requise |
| Interopérabilité | Sur Apple HomeKit, Google Home, Amazon Alexa, Home Assistant et plus encore |

## Exigences du routeur frontalier

L’utilisation d’appareils Matter over Thread nécessite un routeur de bordure Thread :

- Apple HomePod mini / HomePod 2e génération / Apple TV 4K
- Google Nest Hub 2e génération / Nest Hub Max
-Amazon Echo 4ème génération
- Home Assistant Vert + Connect ZBT-2
- Homey Pro
