# Normes de certification

HS1SA-M a passé les certifications suivantes :

- **EN 14604** — Norme européenne pour les détecteurs de fumée, voir [EN European Standard](../../../../ApprovalStandards/EN European Standard.md)
- **Marquage CE** — Déclaration de conformité UE, voir [Conformité CE UE](../../../../certification-standards/CE EU-Conformity.md)
- **Certification Matter** — A obtenu la certification CSA Alliance Matter pour garantir l'interconnexion multiplateforme des maisons intelligentes
- **Prix Red Dot Design** — Prix Red Dot Product Design
