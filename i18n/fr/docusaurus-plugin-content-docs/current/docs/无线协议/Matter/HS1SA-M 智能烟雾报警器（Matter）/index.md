# HS1SA-M Détecteur de fumée intelligent (Matter)

HS1SA-M est un détecteur de fumée intelligent **Matter over Thread** lancé par Hyman. Il utilise une technologie de détection de fumée photoélectrique et prend en charge l’interconnexion de plates-formes de maison intelligente écologiques.

## Présentation du produit

- **Type de produit** : Détecteur de fumée intelligent
- **Protocole de communication** : l'affaire passe avant le fil de discussion
- **Norme de certification** : EN 14604
- **Scénarios typiques** : alerte incendie dans les maisons, appartements, hôtels, etc.

## Paramètres techniques

| Paramètre | Valeur |
|------|-----|
| Type de capteur | Photoélectrique infrarouge |
| Protocole de communication | La question avant le fil (IEEE 802.15.4) |
| Bande de fréquence de fonctionnement | 2,4 GHz |
| Pression sonore de l'alarme | ≥85 dB (3 mètres) |
| Méthode d'alimentation | DC 3 V (pile au lithium CR123A) |
| Autonomie de la batterie | Environ 3 ans |
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de travail | ≤95 % RH (sans condensation) |

## Fonctionnalités

- Détection de fumée photoélectrique pour détecter la fumée à combustion lente et à flamme nue
- Alarme sonore et lumineuse élevée de 85 dB
- Rappel de batterie faible
- Fonction d'autotest (test en un clic)
- Alarme anti-effraction
- Muet à distance
- Interopérabilité multi-plateforme multi-écologique

## Certifications

- Certification EN 14604
- Certification de matière
