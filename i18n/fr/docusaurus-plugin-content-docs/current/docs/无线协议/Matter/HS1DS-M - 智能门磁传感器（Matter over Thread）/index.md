# HS1DS-M - Capteur de porte intelligent (Matter over Thread)

## Aperçu

HS1DS-M est la version Matter over Thread du capteur de porte intelligent Heiman. Il prend en charge l'accès natif à toutes les plateformes compatibles Matter telles que Apple Home, Google Home, Amazon Alexa, Home Assistant, etc. sans avoir besoin d'une passerelle propriétaire.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | La question avant le fil |
| Méthode d'alimentation | Pile bouton CR1632 |
| Autonomie de la batterie | Environ 2-3 ans |
| Distance d'installation | ≤10 mm |
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de travail | ≤95 % HR |

## Fonctionnalités

- Détection en temps réel de l'état d'ouverture/fermeture des portes et fenêtres
- Alarme anti-sabotage (Tamper)
- Rappel de batterie faible
- Compatible multiplateforme : Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Réseau Thread Mesh, contrôle local à faible latence
- Aucune passerelle dédiée requise, routeur de bordure Thread requis
