# Détecteur d'immersion d'eau L1-M (Matière)

L1-M est un détecteur d'immersion dans l'eau **Matter over Thread** lancé par Hyman. Il détecte les fuites d'eau ou l'eau accumulée grâce à une sonde, ce qui peut prévenir efficacement les pertes de propriété causées par l'éclatement des conduites d'eau.

## Présentation du produit

- **Type de produit** : Capteur de détection d'immersion dans l'eau
- **Protocole de communication** : l'affaire passe avant le fil de discussion
- **Méthode de détection** : Détection de contact de sonde
- **Scénarios typiques** : Détection de fuite d'eau dans la cuisine, la salle de bain, la buanderie et le sous-sol

## Paramètres techniques

| Paramètre | Valeur |
|------|-----|
| Méthode de détection | Type de contact de sonde en acier inoxydable |
| Protocole de communication | La question avant le fil (IEEE 802.15.4) |
| Bande de fréquence de fonctionnement | 2,4 GHz |
| Méthode d'alimentation | Alimenté par batterie |
| Méthode d'alarme | Buzzer + App push |
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de travail | ≤95 % RH (sans condensation) |

## Fonctionnalités

- Détection en temps réel des fuites d'eau/eau stagnante
- Alarme sonore sur site
- Poussée d'alarme à distance via l'application
- Rappel de batterie faible
- Conception anticorrosion de la sonde
- Interopérabilité multi-plateforme multi-écologique
