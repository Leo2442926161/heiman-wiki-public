# Paramètres techniques

Voici les spécifications générales de l’équipement Hyman Matter.

## Paramètres sans fil

| Paramètre | Valeur |
|------|-----|
| Protocole sous-jacent | Sujet (IEEE 802.15.4) |
| Bande de fréquence de fonctionnement | 2,4 GHz (bande ISM) |
| Normes de couche d'application | Matière 1.x |
| Topologie du réseau | Réseau maillé d’auto-guérison |
| Méthode de cryptage | AES-128-CCM |

## Configuration matérielle requise

| Projet | Descriptif |
|------|------|
| Routeur frontalier | Requis, comme Apple HomePod mini, Apple TV 4K, Google Nest Hub 2e génération, etc. |
| Prise en charge des terminaux mobiles | iOS / Android (distribution en réseau via diverses applications écologiques) |
| Interopérabilité interécologique | Apple HomeKit, Google Home, Amazon Alexa, Assistant domestique, SmartThings |

## Consommation électrique et électrique

| Paramètre | Valeur |
|------|-----|
| Méthode d'alimentation | Alimenté par batterie (CR123A / CR2450, etc.) |
| Autonomie de la batterie | Environ 1 à 3 ans (selon le type d'appareil et la fréquence d'utilisation) |
| Consommation d'énergie en veille | Extrêmement faible, prend en charge le mécanisme de réveil |

## Paramètres d'environnement

| Paramètre | Valeur |
|------|-----|
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de fonctionnement | ≤ 95 % HR (sans condensation) |
