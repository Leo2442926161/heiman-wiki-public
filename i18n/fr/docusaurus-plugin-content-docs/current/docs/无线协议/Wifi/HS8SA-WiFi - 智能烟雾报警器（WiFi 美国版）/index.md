# HS8SA-WiFi - Détecteur de fumée intelligent (Version WiFi US)

## Aperçu

HS8SA-WiFi est un détecteur de fumée intelligent WiFi lancé par Hyman pour le marché américain. Il utilise une technologie de détection de fumée photoélectrique, prend en charge le réseau WiFi et les modes de fonctionnement indépendants, et a passé la certification UL 217. Convient pour la protection contre l'incendie dans les maisons, appartements et autres lieux d'habitation.

## Spécifications techniques

| Paramètre | Valeur |
|------|-----|
| Protocole de communication | IEEE 802.11 b/g/n (2,4 GHz) |
| Type de capteur | Photoélectrique infrarouge |
| Pression sonore de l'alarme | ≥85 dB (3 mètres) |
| Méthode d'alimentation | Alimenté par batterie |
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de travail | ≤ 90 % HR |
| Méthode d'installation | Pose au plafond |

## Fonctionnalités

- Détection de fumée photoélectrique pour détecter la fumée provenant de brûlures lentes et de flammes nues
- Poussée d'alarme à distance du réseau WiFi
- Mode de fonctionnement indépendant (une alarme sonore et lumineuse locale peut toujours se produire lorsque le réseau est déconnecté)
- Alarme sonore et lumineuse élevée de 85 dB
- Rappel de batterie faible
- Fonction d'autotest (test en un clic)
- Alarme anti-effraction
-Supporte la sourdine à distance

## Normes de certification

- UL 217 (norme américaine pour les détecteurs de fumée)
-FCC
- Conformité NFPA 72
