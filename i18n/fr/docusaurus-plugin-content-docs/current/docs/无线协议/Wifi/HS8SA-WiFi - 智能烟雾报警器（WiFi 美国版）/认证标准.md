# Normes de certification

HS8SA-WiFi a passé les certifications suivantes :

- **UL 217** — Norme américaine de sécurité des détecteurs de fumée, voir [UL Underwriters Laboratories](../../../../certification standards/ULUnderwriters Laboratories.md)
- **FCC** — Certification de la Commission fédérale des communications
- **CE** — Déclaration de conformité de l'Union européenne
