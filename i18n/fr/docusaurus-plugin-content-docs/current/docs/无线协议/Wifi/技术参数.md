# Paramètres techniques

Voici les spécifications techniques générales des appareils WiFi Heimann.

## Paramètres sans fil

| Paramètre | Valeur |
|------|-----|
| Normes de protocole | IEEE 802.11b/g/n |
| Bande de fréquence de fonctionnement | 2,4 GHz (non compatible avec 5 GHz) |
| Taux de transmission | Jusqu'à 150 Mbit/s (802.11n) |
| Topologie du réseau | Topologie en étoile (directement connectée au routeur) |
| Distance de communication | Environ 30 à 50 mètres en intérieur (selon les performances du routeur) |
| Méthode de cryptage | WPA2-PSK/WPA3 |

## Méthode de distribution réseau

| Méthode | Descriptif |
|------|------|
| SmartConfig | Application mobile Configuration réseau en un clic (équipement WiFi traditionnel) |
| Approvisionnement assisté par Bluetooth | Configuration des informations d'identification WiFi via les canaux temporaires Bluetooth (certains nouveaux modèles) |
| Mode AP | Activez le point d'accès sur l'appareil et configurez-le une fois le téléphone connecté |

## Consommation électrique et électrique

| Paramètre | Valeur |
|------|-----|
| Méthode d'alimentation | Alimentation secteur (type prise/type passerelle) ou alimentation par batterie (type alarme) |
| Niveau de consommation électrique | Supérieur à Zigbee/Thread en raison de la nécessité de maintenir une connexion TCP/IP |
| Consommation d'énergie en veille | Type de prise environ 0,5 W ~ 1 W |

## Paramètres d'environnement

| Paramètre | Valeur |
|------|-----|
| Température de fonctionnement | -10°C ~ +50°C |
| Humidité de fonctionnement | ≤ 90 % RH (sans condensation) |
