# Paramètres techniques

Voici les paramètres techniques généraux de l’équipement Heimann Wireless M-Bus (wM-Bus).

## Paramètres sans fil

| Paramètre | Valeur |
|------|-----|
| Normes de protocole | EN 13757-4 (Couche physique M-Bus sans fil) |
| Bande de fréquence de travail | 868 MHz (bande de fréquence européenne SRD, 868,0 ~ 868,6 MHz) |
| Méthode de modulation | FSK |
| Taux de transmission | 4,8 kbps / 16,384 kbps / 100 kbps (mode T / mode S / mode R) |
| Topologie du réseau | Étoile (instrument → collecteur/passerelle) |
| Distance de communication | Environ 100 à 300 mètres en intérieur, environ 500 à 1 000 mètres en espace ouvert |
| Puissance de transmission | ≤ 25 mW (+14 dBm) |

## Mode de travail

| Mode | Tarif | Descriptif |
|------|------|------|
| Mode S | 16,384 kbit/s | Mode standard, faible consommation d'énergie, adapté aux instruments alimentés par batterie |
| Mode T | 100 kbit/s | Mode haute vitesse, adapté aux scénarios de données de trafic volumineux |
| Mode R | 4,8 kbit/s | Mode bande étroite, transmission longue distance |

## Consommation électrique et électrique

| Paramètre | Valeur |
|------|-----|
| Méthode d'alimentation | Alimentation par batterie au lithium (ER18505 / ER26500 et autres batteries au lithium industrielles) |
| Autonomie de la batterie | 6 à 10 ans (selon la fréquence des rapports, la valeur typique est de 1 heure/jour) |
| Consommation d'énergie en veille | Niveau microampère, sommeil profond |
| Recevez la sensibilité | Env. -105 dBm (mode S) |

## Paramètres d'environnement

| Paramètre | Valeur |
|------|-----|
| Température de fonctionnement | -20°C ~ +60°C (qualité industrielle) |
| Humidité de fonctionnement | ≤ 95 % HR (sans condensation) |
| Niveau de protection IP | IP65 ~ IP68 (selon le modèle spécifique, applicable aux instruments extérieurs) |

## Protocole de données

| Couche | Norme |
|----|------|
| Couche physique | EN 13757-4 |
| Couche de liaison de données | EN 13757-4 (Format de trame M-Bus sans fil) |
| Couche d'application | EN 13757-3 (couche d'application M-Bus) |
| Méthode de cryptage | AES-128 (cryptage en option) |
