# Norme européenne EN

EN (norme européenne, norme européenne) est une norme technique élaborée par le Comité européen de normalisation (CEN) et le Comité européen de normalisation électrotechnique (CENELEC). Les principales normes EN impliquées dans les produits Heiman sont les suivantes :

## EN 14604 — Norme pour les détecteurs de fumée

| Projet | Descriptif |
|------|------|
| Champ d'application | Détecteur de fumée indépendant |
| Exigences de détection | Détection de fumée photoélectrique, alarme sonore et lumineuse, invite de batterie faible |
| Marchés applicables | États membres de l'UE, Royaume-Uni, Suisse, Norvège, etc. |
| Produits Heiman | Séries HS1SA-E, HS3SA, HS1SA-M, S1, etc. |
| Importance de la certification | Le respect des exigences de la réglementation européenne relative aux incendies dans les bâtiments est une condition nécessaire pour accéder au marché européen |

**Éléments de test** : temps de réponse, niveau de pression acoustique (≥85 dB/3 m), fonction muet, durée de vie de la batterie, cycle de température, durabilité de l'humidité, résistance aux interférences radiofréquences, etc.

## EN 50291 — Norme sur les alarmes de monoxyde de carbone

| Projet | Descriptif |
|------|------|
| Champ d'application | Alarme domestique de monoxyde de carbone (CO) |
| Exigences de test | Capteur de CO électrochimique, alarme de classification de concentration |
| Marchés applicables | États membres de l'UE, Royaume-Uni |
| Produits Heiman | HS1CA-E, série C1, HS1CA-M, etc. |
| Importance de la certification | Pour garantir la fiabilité de la détection des fuites de CO, l'installation est obligatoire dans certains pays |
