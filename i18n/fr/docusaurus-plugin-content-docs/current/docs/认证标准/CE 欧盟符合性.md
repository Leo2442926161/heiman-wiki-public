# Conformité CE UE

La marque CE (Conformité Européenne, EU Conformity) est une marque de sécurité obligatoire pour les produits entrant sur le marché de l'UE.

## Signification du marquage CE

Le marquage CE indique que le produit est conforme aux exigences fondamentales des directives européennes pertinentes et peut circuler librement dans les États membres de l'UE et dans l'Espace économique européen (EEE).

## Chemin de certification CE pour les produits Haiman

Les détecteurs de fumée/CO Hyman répondent aux exigences CE grâce à :

### Règlement CPR (UE n° 305/2011)

Le Règlement sur les Produits de Construction exige que les produits d’alarme incendie soient soumis à une évaluation de la stabilité de leurs performances :
- Conforme aux normes : EN 14604 (fumée), EN 50291 (CO)
- Agence d'évaluation : TÜV Rheinland et autres agences notifiées
- Niveau de certification : DoP (Déclaration de Performance) + marquage CE

### Directive RED (2014/53/UE)

La directive sur les équipements radio exige que les équipements sans fil :
- Couvre Zigbee, Z-Wave, Wi-Fi, Thread et autres modules sans fil
- Éléments de test : compatibilité électromagnétique (EMC), radiofréquence (RF), SAR, sécurité

### Directive CEM (2014/30/UE)

La directive sur la compatibilité électromagnétique garantit que l'équipement ne provoque pas d'interférences avec d'autres équipements électroniques.

## Produits marqués CE

Tous les produits Heymann vendus sur le marché de l'UE portent le marquage CE, notamment :
- HS1SA-E / HS3SA (détecteur de fumée Zigbee)
- Série HS1CA-E/C1 (alarme CO Zigbee)
- HS1SA-M/HS1CA-M (La question sur le fil)
- Gamme complète de capteurs Zigbee tels que HS1MS / HS1DS
