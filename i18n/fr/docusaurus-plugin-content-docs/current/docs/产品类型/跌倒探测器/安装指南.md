# Guide d'installation

## Méthode de port

- **Clip de ceinture** : Clip sur la ceinture ou la taille
- **Lanyard** : à accrocher autour du cou et à accrocher sur la poitrine

## Mode d'emploi

1. Installez la batterie
2. Choisissez la méthode de port (clip ceinture ou cordon)
3. Portez-le sur votre corps
4. Couplage via passerelle
5. Essai de simulation :
   - Se lever puis rapidement s'accroupir ou s'allonger sur le sol
   - Le capteur doit détecter une chute et déclencher l'alarme en quelques secondes
   - Appuyez sur le bouton de détresse pour tester l'alarme manuelle

## Remarques

- Il est recommandé de le porter toute la journée (peut être retiré pour charger/remplacer la batterie lorsque vous prenez une douche)
- L'algorithme d'IA est utilisé pour distinguer les chutes des activités quotidiennes (s'asseoir, se pencher), avec un faible taux de fausses alarmes
- Alarme automatique après détection d'une chute, ou vous pouvez appuyer manuellement sur le bouton pour appeler à l'aide
- Peut être lié au bouton d'urgence pour obtenir une double protection
