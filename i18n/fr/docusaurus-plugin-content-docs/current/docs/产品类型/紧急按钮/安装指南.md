# Guide d'installation

## Exigences relatives à l'emplacement d'installation

- **Installation murale** : salle de bain, chevet dans la chambre, à côté du canapé dans le salon - n'importe où à portée de main.
- **Portez-le partout** : mettez-le dans votre poche ou accrochez-le à votre porte-clés
- **Pour les personnes âgées** : Installé dans la salle de bain et au chevet de la chambre, la hauteur est d'environ 1,2 m

## Étapes d'installation

1. Déterminez l'emplacement d'installation ou si vous devez le porter avec vous
2. Montage mural : utilisez des vis ou de la colle 3M pour fixer la base de montage
3. Installez la batterie
4. Placez le bouton d'urgence sur la base ou emportez-le avec vous
5. Couplage via passerelle
6. Test : appuyez longuement sur le bouton pour déclencher l'alarme et confirmer que l'application reçoit la notification.

## Remarques

- Adoptez la conception **toucher anti-accidentel à pression longue**, qui nécessite une pression continue pendant 1 à 2 secondes pour déclencher
- Il est recommandé de faire le lien avec l'alarme sonore et lumineuse
- Tester régulièrement les fonctionnalités
- Pour les personnes âgées, il peut être utilisé avec un détecteur de chute pour une double protection.
