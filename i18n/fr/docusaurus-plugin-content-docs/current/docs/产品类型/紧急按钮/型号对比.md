# Comparaison de modèles

## Comparaison de différents modèles

| Modèle | Protocole | Méthode de déclenchement | Alimentation | Caractéristiques | Suggestions de jeu |
|------|------|----------|------|------|----------|
| Bouton d'urgence | Zigbee | Appuyez longuement sur la gâchette | Batterie | Portable + mural à double usage | Les personnes âgées peuvent l'emporter avec elles, aide en un clic en cas d'urgence, liée au téléphone portable + alarme sonore et lumineuse |

## Gameplay de scène recommandé

| Scénarios | Modèles recommandés | Solutions de liaison |
|------|----------|--------------|
| Soins à domicile pour personnes âgées | Bouton d'urgence | Inconfort soudain pour les personnes âgées → Appui long → Push sur les téléphones portables de tous les membres de la famille → Alarme sonore et lumineuse liée → Notification de propriété |
| Sécurité des chambres d'enfants | Bouton d'urgence | Si un enfant rencontre une urgence seul à la maison → Appuyez sur le bouton pour déclencher l'alarme sur le téléphone portable du parent |
| Boutique/Caisse | Bouton panique | Installation dissimulée, déclenchant une alarme silencieuse en cas de vol, liée à une vidéo de surveillance + avertissant silencieusement la sécurité |
| Toilettes/salle de bains | Bouton d'urgence | Si les personnes âgées glissent et tombent sous la douche, le bouton d'urgence étanche est à portée de main pour une aide facile |
