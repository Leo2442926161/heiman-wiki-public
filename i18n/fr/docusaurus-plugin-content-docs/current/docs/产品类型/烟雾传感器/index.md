# capteur de fumée

Le capteur de fumée Hyman utilise une **technologie de détection de fumée photoélectrique**, qui peut détecter avec sensibilité les particules de fumée visibles générées dès les premiers stades d'un incendie et permettre une alerte incendie très précoce.

## Série de produits

| Série | Méthodes de communication | Normes de certification | Marchés applicables |
|------|----------|----------|--------------|
| Série HS1SA | Zigbee / Z-Wave | EN 14604 | Union européenne |
| Série HS3SA | Zigbee | EN 14604 | Union européenne |
| HS8SA | Zigbee / Wi-Fi | UL217 | États-Unis |
| Série S1 | Sub-1G Autonome | EN 14604 | Usage général |
| HS1SA-M | La question avant le fil | EN 14604 | Mondial |

## Fonctions de base

- Détection de fumée photoélectrique pour détecter la fumée provenant de brûlures lentes et de flammes nues
- Alarme sonore et lumineuse élevée de 85 dB
- Rappel de batterie faible
- Fonction d'autotest (test en un clic)
- Alarme anti-effraction
-Supporte la sourdine à distance

## Paramètres techniques (en prenant HS1SA-M comme exemple)

- Type de capteur : photoélectrique infrarouge
- Pression sonore de l'alarme : ≥85 dB (3 mètres)
- Alimentation : pile CR123A
- Autonomie de la batterie : env. 3 ans
- Température de fonctionnement : -10°C ~ +50°C
- Humidité de travail : ≤95% RH
