# Passerelle intelligente

Smart Gateway est la plaque tournante centrale du système de maison intelligente Haiman, responsable de la connexion de divers capteurs Zigbee et de la réalisation du contrôle de liaison entre les appareils.

## Série de produits

| Série | Méthodes de communication | Caractéristiques |
|------|----------|------|
| Série HS1GW | Zigbee + Wi-Fi/Ethernet | Passerelle de base |
| Série HS3GW | Zigbee + Wi-Fi/Ethernet | Amélioré |
| Série HS5GW | Zigbee + Wi-Fi/Ethernet | Passerelle haute performance |
| Série HS6GW | Zigbee + Wi-Fi / Ethernet + Fil | Le dernier modèle prend en charge à la fois Zigbee et Matter over Thread |

## Fonctions de base

- Coordinateur du réseau Zigbee
- Prise en charge Wi-Fi / Ethernet pour se connecter à Internet
- Découverte et couplage automatiques des appareils
- Liaison de scène locale (certains modèles)
- Accès à distance (via Heiman Cloud Platform)
- Prend en charge jusqu'à 128 sous-appareils (selon le modèle)

## Scénarios applicables

- Le point de connexion central de l'intelligence Heiman pour toute la maison
- Coopérer avec Haiman APP pour réaliser une surveillance à distance
- Gestion centralisée locale des petits et moyens projets commerciaux

## Différences par rapport aux autres passerelles

Haiman Gateway est l'entrée de l'écosystème officiel. Pour les utilisateurs qui souhaitent utiliser une plateforme tierce, il est recommandé d'utiliser le coordinateur universel Zigbee avec Zigbee2MQTT/ZHA.
