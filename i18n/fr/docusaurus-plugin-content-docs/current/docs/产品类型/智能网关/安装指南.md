# Guide d'installation

## Exigences relatives à l'emplacement d'installation

- **L'emplacement central dans toute la maison** assure une couverture du signal Zigbee pour tous les appareils
- Essayez d'éviter de le placer dans une armoire entièrement métallique (le métal bloquera les signaux sans fil)
- **Ethernet d'abord** : Il est recommandé d'utiliser une connexion réseau filaire, plus stable
- Pas à plus de 20-30 m des autres appareils Zigbee (à l'intérieur)

## Étapes d'installation

1. Sélectionnez l'emplacement d'installation (meuble TV dans le salon, à côté du boîtier de courant faible, etc.)
2. Connectez l'adaptateur secteur
3. Connectez un câble Ethernet (recommandé) ou connectez-vous au Wi-Fi
4. Attendez que le voyant s'allume en vert/fixe
5. Ouvrez l'application Haiman et scannez le code QR au bas de la passerelle pour ajouter
6. Une fois l'ajout terminé, vous pouvez coupler le sous-appareil

## Remarques

- La passerelle est le cœur de la maison intelligente. Il est recommandé d'utiliser une alimentation de secours UPS.
- La connexion Ethernet est plus stable que le Wi-Fi et il est recommandé de l'utiliser en premier
- Si le signal de l'appareil est faible, ajoutez un dispositif relais Zigbee (comme une prise intelligente)
- Le nombre maximum de sous-appareils pris en charge par différents modèles est différent et doit être sélectionné en fonction de l'échelle.
