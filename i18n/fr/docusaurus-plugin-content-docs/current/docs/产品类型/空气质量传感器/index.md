# capteur de qualité de l'air

Des capteurs de qualité de l’air (Air Quality Monitor) sont utilisés pour détecter divers polluants dans l’air intérieur afin de contribuer au maintien d’un milieu de vie sain.

## Paramètres de détection

Les capteurs de qualité de l'air Hyman détectent les indicateurs suivants :

| Paramètres | Descriptif | Normes de santé |
|------|------|----------|
| PM2,5 | Particules fines | ≤ 35 µg/m³ (moyenne sur 24 heures) |
| PM10 | Particules respirables | ≤ 50 µg/m³ (moyenne sur 24 heures) |
| CO₂ | Dioxyde de carbone | ≤ 1 000 ppm |
| COVT | Composés organiques volatils totaux | ≤ 0,6 mg/m³ |
| Température | Température ambiante | 18-26°C |
| Humidité | Humidité relative | 40-60 % d'humidité relative |

## Fonctions de base

- Détection multi-paramètres de la qualité de l'air
- Écran couleur (selon modèle)
- Alarme sonore et lumineuse pour dépassement de la norme
- Enregistrements de données historiques
- Purificateur d'air/air frais lié

## Scénarios applicables

- Chambre : surveillez les niveaux de CO₂ et invitez à ouvrir les fenêtres pour aérer
- Chambre d'enfant : protéger la santé respiratoire des enfants
- Bureau : Augmentez la productivité et le confort
- Maisons récemment rénovées : surveiller les rejets de formaldéhyde/COVT
