# Sonde de température incendie

Le capteur de température incendie (Heat Detector) émet une alarme incendie en détectant des augmentations anormales de la température ambiante. Il convient aux scènes telles que les cuisines et les garages où les capteurs de fumée sont sujets à de fausses alarmes.

## Principe de fonctionnement

Le capteur de température utilise un élément thermique pour détecter les changements de température :

- **Alarme de température fixe** : Déclenche une alarme lorsque la température dépasse un seuil défini (généralement 57°C / 135°F)
- **Alarme de différence de température** : déclenche une alarme lorsque le taux d'augmentation de la température dépasse la valeur définie (par exemple 8,3°C/min)

## Champ d'application

- Cuisine (les détecteurs de fumée sont sujets aux fausses alarmes dues à la cuisson dans cette scène)
- Garage/atelier (les gaz d'échappement et la poussière peuvent interférer avec les capteurs de fumée)
- Chaufferie/local matériel de chauffage
- Salle de bain/Sauna

## Fonctions de base

- Détection d'échauffement anormal
- Alarme anti-effraction
- Rappel de batterie faible

> **Remarque** : Le capteur de température et le capteur de fumée sont utilisés de manière complémentaire. Le capteur de fumée répond plus rapidement et le capteur de température a une capacité anti-interférence plus forte.
