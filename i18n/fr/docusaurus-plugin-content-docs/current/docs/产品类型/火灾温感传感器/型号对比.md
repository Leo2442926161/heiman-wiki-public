# Comparaison de modèles

## Comparaison de différents modèles

| Modèle | Protocole | Principe de détection | Alimentation | Caractéristiques | Suggestions de jeu |
|------|------|----------|------|------|----------|
| Capteur de température d'incendie | Zigbee / Sous-1G | Thermistance (température fixe + température différentielle) | Batterie | Interférence anti-fumée | Spécial pour cuisine/garage, alarme pour élévation anormale de température, électrovanne de liaison pour couper le gaz |

## Gameplay de scène recommandé

| Scénarios | Modèles recommandés | Solutions de liaison |
|------|----------|--------------|
| Sécurité de la cuisine | Capteur de température | Élévation anormale de la température → Fermez le robinet de gaz → Alarme sonore et lumineuse → Notification push sur téléphone portable pour éviter les fausses alarmes de fumée de cuisson |
| Garage/chaufferie | Capteur de température | Surveiller la température dans la salle d'équipement pour éviter une combustion spontanée et relier le ventilateur d'extraction pour refroidir |
| Complémentaire avec capteur de fumée | Sonde de température + fumée | Le capteur de fumée couvre toute la maison, le capteur de température couvre la cuisine/garage, l'alarme complémentaire est plus fiable |

## Suggestions de sélection

- **Cuisine uniquement** : Sonde de température (pour éviter les fausses alarmes de fumée de cuisson)
- **Fonctionne avec un capteur de fumée** : le capteur de fumée couvre la chambre/le salon, le capteur de température couvre la cuisine/le garage.
