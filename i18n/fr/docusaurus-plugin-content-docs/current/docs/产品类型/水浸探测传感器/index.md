# Capteur de détection d'immersion dans l'eau

Le capteur de fuite d'eau est utilisé pour détecter les fuites d'eau ou l'accumulation d'eau, ce qui peut prévenir efficacement les pertes de propriété causées par l'éclatement des conduites d'eau et les inondations.

## Série de produits

| Série | Méthodes de communication | Caractéristiques |
|------|----------|------|
| HS1WL | Zigbee | Modèle de base, détection par sonde |
| Série HS2WL | Zigbee | Détection sonde + câble, double canal |
| Série HS3WL | Zigbee | Version améliorée |
| Série L1 | Matière / Zigbee | Version Matter over Thread, certifiée WWHA |

## Fonctions de base

- Détection en temps réel des fuites d'eau/eau stagnante
- Alarme sonore
- Rappel de batterie faible
- Conception anticorrosion de la sonde

## Emplacement d'installation

- Sous le lavabo de la salle de bain
- Sous l'évier de la cuisine
- A côté du lave-linge/lave-vaisselle
- Sous le chauffe-eau
- Sous-sol/garage
- Près du radiateur

## Lien de scène

- **Détection de fuite d'eau + électrovanne** : ferme automatiquement la vanne principale lorsqu'une fuite d'eau est détectée
- **Alarme de fuite d'eau + Push** : avertissez la propriété ou les membres de votre famille sur votre téléphone portable lorsque vous n'êtes pas à la maison
- **Flood + Smart Shutoff** : chauffe-eau à gaz connecté ou vanne d'eau intelligente
