# Guide d'installation

## Exigences relatives à l'emplacement d'installation

- **Placer au sol**, assurez-vous que la sonde touche le sol
- **Près du point d'eau** : sous le lavabo de la salle de bain, sous l'évier de la cuisine, à côté du lave-linge, sous le chauffe-eau
- Le point le plus bas où les fuites d'eau doivent être détectées

## Étapes d'installation

1. Déterminez l'emplacement d'installation (le sol dans la zone à risque de fuite d'eau)
2. Nettoyez la surface du sol
3. Placez le capteur au sol avec la sonde touchant le sol
4. Si vous utilisez la version câble (HS2WL), posez le câble d'extension le long de la zone à surveiller.
5. Insérez la batterie et le capteur s'allumera automatiquement.
6. Test de simulation : touchez la sonde avec les doigts mouillés, le voyant d'alarme s'allumera
7. Couplage via passerelle

## Remarques

- La sonde peut se corroder si elle est exposée longtemps à un environnement humide, vérifiez-la régulièrement
- Il est recommandé de coopérer avec l'électrovanne pour obtenir un arrêt automatique
-La durée de vie de la batterie est d'environ 2-3 ans
- Plusieurs capteurs pour couvrir différentes zones
