# Capteur infrarouge du corps humain

Le capteur infrarouge du corps humain (PIR) détecte les mouvements en détectant le rayonnement thermique infrarouge émis par le corps humain et constitue la solution de détection de présence la plus couramment utilisée dans les maisons intelligentes.

## Série de produits

| Série | Méthodes de communication | Caractéristiques |
|------|----------|------|
| HS1MS | Zigbee | Modèle de base, angle de détection 110° |
| HS3MS | Zigbee | Modèle amélioré, installation au plafond à 360° |
| Série M1 | Matière / Zigbee | Prend en charge Matter over Thread, en option 110° ou 360° |
| HS8MLS | Zigbee | Modèle longue distance |
| HS8MIS | Zigbee | Corps humain + capteur composite d'éclairage |

## Fonctions de base

- Détection humaine mobile
- Détection d'éclairage (série M1/HS8MIS)
- Distance de détection ≤7m (25°C)
- Méthode d'installation : mural ou au plafond
- Autonomie de la batterie : 3 ans (Zigbee) / 1 an (Matter)

## Paramètres techniques (série M1)

- Angle de détection : 110° / 360°
- Distance de détection : ≤7m
- Plage d'éclairement : 0 ~ 1200 Lux
- Alimentation : DC 3V (batterie au lithium)
- Autonomie de la batterie : 3 ans (Zigbee) / 1 an (Matter)

## Exemple de scénario

- La lumière s'allume quand quelqu'un arrive et s'éteint quand quelqu'un part
- Liaison de sécurité : appuyez sur l'alarme lorsqu'un mouvement anormal est détecté
- Armement en quittant la maison : éteint automatiquement la climatisation/le chauffage pour économiser de l'énergie lorsque personne n'est présent
