# Guide d'installation

## Exigences relatives à l'emplacement d'installation

- Hauteur de montage mural : **1,8 ~ 2,4 m**
- Installation au plafond : Installé au plafond au centre de la pièce
- **Éviter** : Lumière directe du soleil, bouches de chauffage/climatisation, à proximité de sources de chaleur
- Eviter les obstructions (meubles, cloisons vitrées) dans la plage de détection

## Étapes d'installation

1. Sélectionnez l'emplacement d'installation (mur ou plafond)
2. Fixez la base de montage avec des vis
3. Insérez la batterie, le voyant clignote
4. Installez le corps du capteur sur la base
5. Ajustez l'angle de détection (dans une plage horizontale de 110 °)
6. Test de marche : après avoir traversé la zone de détection, le voyant doit s'allumer
7. Couplage via passerelle

## Remarques

- Les capteurs infrarouges sont sensibles aux changements de température et la distance de détection peut être raccourcie dans les environnements à haute température en été.
- Évitez de faire face aux fenêtres (les changements de température extérieure provoquent de fausses alarmes)
- Autonomie de la batterie environ 3 ans (Zigbee) / 1 an (Matter)
