# Guide d'installation

## Exigences relatives à l'emplacement d'installation

- **installation au plafond**, capteur verticalement vers le bas
- Hauteur d'installation **2,5 ~ 3,5 m**
- Le rayon de détection est d'environ **3 ~ 5 m**
- **Éviter** : Il y a des obstacles métalliques devant le capteur (le métal bloquera les ondes millimétriques)
- Tenir à l'écart des gros tuyaux métalliques, des unités internes du climatiseur, etc.

## Étapes d'installation

1. Sélectionnez l'emplacement d'installation au plafond (le centre de la pièce est le meilleur)
2. Percez des trous dans le plafond et fixez la base avec des vis d'expansion
3. Connectez le câblage (si l'alimentation secteur est requise) ou installez la batterie
4. Faites pivoter le corps du capteur et enclenchez-le dans la base
5. Après la mise sous tension, le voyant s'allume et le capteur commence l'auto-test.
6. Entrez le couplage de l'application et ajoutez un appareil
7. Test : entrez dans la zone de détection et asseyez-vous tranquillement. Le capteur doit rester dans l'état « occupé ».

## Remarques

- Les ondes millimétriques peuvent pénétrer dans les panneaux minces (plaques de plâtre, bois), mais seront bloquées par le métal
-Les murs épais et les plafonds métalliques réduiront la portée de détection
- Il est recommandé de l'utiliser avec un capteur PIR pour obtenir une couverture complète de la scène
