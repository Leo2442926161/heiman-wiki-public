# Exemple d'automatisation

## Alarme de sécurité de détection de fumée

**Configuration automatisée HomeKit :**

1. **Condition de déclenchement :** Alarme de fumée Hyman → fumée détectée
2. **Paramètres de sécurité :** Basculez l'état du système de sécurité sur « Alarme »
3. **Exécuter l'action :**
   - HomePod → Lire un son personnalisé "Fumée détectée, veuillez faire attention à la sécurité"
   - Toutes les lumières intelligentes → s'allument et clignotent
   - Notifications push à tous les membres de la famille

## Scène à la maison/hors de la maison

**Commande de raccourci des raccourcis : "Quitter le contrôle de sécurité du domicile"**

```
quand la dernière personne quitte la maison
  Configurer le système de sécurité → armer lorsque vous quittez la maison
  Configurer le capteur de porte Heiman → Activer les notifications
  Configurer le capteur corporel Heiman → activer l'alarme
  Éteignez toutes les lumières
  Paramètres Thermostat → Mode économie d'énergie
  Envoyer une notification "🏠 Mode hors domicile activé"
```

## Rappel d'anomalie de température

**Condition de déclenchement :** Capteur de température et d'humidité Hyman → Température > 35°C

**Exécuter l'action :**
- HomePod → La température ambiante de la diffusion vocale dépasse la limite
- Notifications push sur iPhone
- Allumer automatiquement la climatisation (si HomeBridge est configuré)
