# Conseils d'utilisation

## Configuration des objets

Ajoutez un appareil Heiman via la liaison MQTT :

```java
Chose mqtt:topic:heiman_smoke "Détecteur de fumée Heiman" @ "salon" {
    Canaux :
        Tapez binaire : smoke "smoke state" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.smoke" ]
        Numéro de type : batterie "Alimentation par batterie" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.battery" ]
}
```

## Définition de l'élément

```java
Contacter Smoke_Alarm "Détecteur de fumée [MAP(smoke.map):%s]" { canal="mqtt:topic:heiman_smoke:smoke" }
Numéro Smoke_Battery "Batterie du détecteur de fumée [%d%%]" { canal="mqtt:topic:heiman_smoke:battery" }
```

## Sélection des coordonnées

Il est recommandé d'utiliser un coordinateur USB avec une puce **CC2531** ou **CC2652** et de s'assurer que la version du firmware est la plus récente.

## Conseils de débogage

- Afficher le statut « en ligne/hors ligne » de Thing dans Paper UI
- Utilisez `openhab-cli info` pour vérifier si les liaisons sont chargées correctement
- Recherchez le mot-clé « heiman » dans le journal pour résoudre les problèmes de communication de l'appareil
