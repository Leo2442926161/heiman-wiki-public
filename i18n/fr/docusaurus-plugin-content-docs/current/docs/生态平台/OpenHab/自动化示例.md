# Exemple d'automatisation

## Règles relatives aux avertisseurs de fumée

```javascript
règle « Alarme de fumée déclenchée »
quand
    L'élément heiman_smoke_alarm a été modifié en ON
alors
    sendCommand (Livingroom_Light, ON)
    sendNotification("admin", "⚠️ Alarme de fumée déclenchée !")
    postUpdate (Alarm_State, "smoke_alert")
fin
```

## Règles du capteur de température

```javascript
règle "Rapport de température et d'humidité"
quand
    L'article heiman_temperature a reçu une mise à jour
alors
    val temp = (heiman_temperature.state comme numéro).floatValue
    val hum = (heiman_humidity.state comme numéro).floatValue
    si (température > 40) {
        sendCommand (climatiseur, ON)
        sendNotification("admin", "La température est trop élevée (" + temp + "°C), le climatiseur a été allumé")
    }
    si (bourdonnement > 80) {
        sendNotification("admin", "L'humidité est trop élevée (" + hum + "%), veuillez faire attention à la déshumidification")
    }
fin
```

## Règles de détection des inondations

```javascript
règle "Liaison de détection d'immersion dans l'eau"
quand
    L'élément heiman_water_leak a été modifié en ON
alors
    sendCommand (Water_Valve, OFF)
    sendBroadcastNotification("💧 Fuite d'eau détectée ! La vanne d'eau a été fermée !")
fin
```
