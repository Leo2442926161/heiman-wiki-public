# Conseils d'utilisation

## Interférence du canal Zigbee

La puce Zigbee intégrée de Homey utilise des canaux spécifiques. Si l'appareil Heiman se déconnecte fréquemment :

1. Affichez la chaîne actuelle dans Paramètres Homey → Zigbee
2. Utilisez l'application Homey Insight pour rechercher les interférences environnementales
3. Passez à un canal avec moins d'interférences (loin du WiFi 1/6/11)

## Réseau de distribution d'équipements

- Comment accéder au mode d'appairage pour les appareils Heiman : appuyez et maintenez le bouton enfoncé pendant 5 secondes (certains modèles doivent appuyer 3 fois rapidement)
- Lors de l'ajout d'un appareil dans l'application Homey, sélectionnez "Ajouter un appareil → Zigbee → Rechercher"
- Lorsque l'appairage échoue, rapprochez l'appareil de Homey (à moins de 3 mètres) et réessayez

## Suggestions d'optimisation des flux

- Ajouter une condition de "délai" dans Flow (par exemple 2 secondes) pour éviter les faux positifs transitoires
- Utilisez la fonction "tag" pour classer et gérer les équipements Heiman

## Extension de la portée du signal

Si l'appareil Heiman est éloigné de Homey, un dispositif relais Zigbee (comme une prise intelligente) peut être ajouté pour étendre la couverture réseau.
