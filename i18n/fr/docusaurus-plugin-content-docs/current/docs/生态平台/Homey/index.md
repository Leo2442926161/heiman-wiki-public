#Homey

Homey est un hub pour maison intelligente lancé par la société néerlandaise Athom, qui prend en charge Zigbee, Z-Wave, Wi-Fi, Matter, Thread et d'autres protocoles.

## Méthode d'intégration

Les appareils Homey Zigbee et Z-Wave se couplent directement via la puce sans fil intégrée de Homey, éliminant ainsi le besoin d'une passerelle supplémentaire. Les appareils Matter sont accessibles via le routeur de bordure Thread intégré de Homey Pro.

## Avantages

- **Plusieurs protocoles en un** : Homey Pro prend en charge Zigbee, Z-Wave, Thread et Wi-Fi en même temps
- **Exécuter localement** : toute l'automatisation est exécutée localement
- **Flow Automation** : programmation par glisser-déposer pour créer des scénarios de liaison complexes

> **Référence** : les applications liées à Heiman peuvent être recherchées dans Homey App Store
