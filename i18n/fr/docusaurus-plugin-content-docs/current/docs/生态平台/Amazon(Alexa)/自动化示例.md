# Exemple d'automatisation

## Intégration d'Alexa Guard (États-Unis)

**Prérequis :** Abonnez-vous à Alexa Guard Plus (disponible aux États-Unis)

**Processus de configuration :**
1. Activez Guard Plus dans l'application Alexa
2. Ajoutez le détecteur de fumée/CO Hyman en tant que « appareil domestique intelligent »
3. Activer la réponse intelligente « Détection d'alarme de fumée »

**Réponse automatique :**
- Alarme de fumée déclenchée → Alexa compose automatiquement les contacts d'urgence
- Déclencheur d'alarme CO → Alerte vocale Alexa pour toute la maison, ventilation automatique
- Les portes et fenêtres s'ouvrent anormalement → Notification d'alarme push

## Routine de détection de fumée

**Déclencheur :** Alarme de fumée Hyman → Fumée détectée

**Exécuter l'action :**
- Annonce vocale Alexa : "⚠️ Fumée détectée, veuillez faire attention à la sécurité"
- Tous les appareils Echo → Jouer une tonalité d'alarme
- Ensemble de lumières intelligentes → tout est allumé
- Notifications push sur téléphone mobile

## Routine « Quitter la maison »

**Déclencheur :** Commande vocale "Alexa, je pars"

**Exécuter l'action :**
- Capteur magnétique de porte Heiman → activer l'armement
- Capteur corporel Heiman → Activer la détection de mouvement
- Détecteur d'immersion dans l'eau Hyman → activer la détection
- Toutes les lumières → éteintes
- Alexa a répondu : "Le système de sécurité est activé, bon voyage"
