#AmazonAlexa

Amazon Alexa est l'une des plateformes d'assistant vocal intelligent les plus populaires au monde, prenant en charge le protocole Matter et un riche écosystème de compétences.

## Méthode d'intégration

### Matériel de matière

Les appareils Hyman Matter over Thread sont accessibles via les appareils Echo compatibles Matter (Echo 4e génération, Echo Plus, Echo Show 10, etc.) :

1. Assurez-vous que votre appareil Echo est mis à jour avec le dernier firmware
2. Ouvrez l'application Alexa → Appareils → Ajouter un appareil
3. Sélectionnez « Matter » et scannez le code d'appairage

### Appareils Zigbee

Certains Echo Plus et Echo Show sont dotés de puces Zigbee intégrées et peuvent être directement associés aux capteurs Heiman Zigbee.

## Exemple de commande vocale

- "Alexa, demande au détecteur de fumée si tout va bien"
- "Alexa, quelle est la température dans le salon ?"

> **Référence** : Les capteurs Hyman Matter sont disponibles sur la boutique officielle d'Amazon
