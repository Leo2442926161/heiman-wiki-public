# Conseils d'utilisation

## Découverte d'appareils

- Une fois l'appareil Heiman Matter allumé, il sera automatiquement découvert par l'application Alexa, généralement dans les 30 secondes.
- S'il n'est pas découvert automatiquement : entrez dans l'application Alexa → Appareil → "+" → "Ajouter un appareil" → "Matter"
- Utilisez le code de configuration Matter ou le code QR situé au bas de l'appareil pour terminer le couplage

## Accès aux appareils Zigbee (Echo Plus / Echo Show)

- Les appareils Echo avec Zigbee Hub peuvent s'associer directement aux capteurs Heiman Zigbee
- Mode d'appairage : Application Alexa → Appareil → "+" → "Zigbee" → "Rechercher un appareil"
- Les appareils Heimann sont compatibles Alexa avec le coordinateur Zigbee intégré

## Conseils de routine

- Utilisez des "actions personnalisées" pour diffuser des invites vocales plus détaillées
- Ajoutez "attendre 5 secondes" à la routine d'urgence avant d'effectuer les actions suivantes pour éviter le chevauchement des voix
- Profitez de la fonction « Détection sonore » pour donner à Alexa des réponses supplémentaires aux alarmes aiguës de l'alarme Heyman

## Alexa Garde REMARQUE

Alexa Guard Plus n'est actuellement disponible qu'aux États-Unis. Les utilisateurs en Chine peuvent simuler certaines fonctions via Routine.
