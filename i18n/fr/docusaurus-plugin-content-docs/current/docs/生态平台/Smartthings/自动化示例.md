# Exemple d'automatisation

## Mode coucher « Bonne nuit »

**Condition de déclenchement :** Capteur magnétique de porte Haiman → Fermer (indiquant que la porte de la chambre a été fermée)

**Exécuter l'action :**
- Éteignez toutes les lumières du salon
- Verrouillage intelligent → Verrouillage
- Système de sécurité → armer la nuit
- Notification Push : "Bonne nuit, le système est armé"

## Mode "Absent"

**Condition de déclenchement :** Capteur infrarouge du corps humain Hyman → 15 minutes sans mouvement

**Exécuter l'action :**
- Toutes les lumières éteintes
- Thermostat → Mode économie d'énergie
- Système de sécurité → armement hors domicile
- Capteur de porte → activer l'alarme
- Notification push : "Le mode Hors domicile est activé, les capteurs de porte et de fenêtre sont activés"

## Mode d'urgence du détecteur de fumée

**Condition de déclenchement :** Alarme de fumée Hyman → Fumée détectée

**Exécuter l'action :**
- Haut-parleur intelligent → jouer le son de l'alarme
- Toutes les lumières intelligentes → rouge clignotant
- Notifications push à tous les membres de la famille
- Si connecté à SmartThings Hub → déclencher la sirène
