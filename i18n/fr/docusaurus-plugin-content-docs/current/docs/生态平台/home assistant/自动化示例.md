# Exemple d'automatisation

## Un détecteur de fumée déclenche des lumières dans toute la maison

```yaml
Alias : "Détecteur de fumée - lumières et notifications dans toute la maison"
déclencheurs :
  - déclencheur : état
    entité_id : binaire_sensor.heiman_smoke_alarm
    à : "on"
actions :
  - action : light.turn_on
    cible :
      entité_id : tout
  - action : notify.mobile_app_phone
    données :
      message : "⚠️ Alarme de fumée déclenchée ! Veuillez vérifier immédiatement !"
      titre : "Détecteur de fumée"
```

## Liaison d'alarme CO

```yaml
Alias : "Alarme CO - coupez le gaz et ouvrez les fenêtres"
déclencheurs :
  - déclencheur : état
    entité_id : binaire_sensor.heiman_co_alarm
    à : "on"
actions :
  - action : switch.turn_off
    entité_id : switch.gas_fireplace
  - action : couvrir.ouvrir
    entité_id : cover.living_room_window
  - action : notify.persistent_notification
    données :
      message : "L'alarme de monoxyde de carbone s'est déclenchée, le robinet de gaz a été fermé et les fenêtres ont été ouvertes"
```

## Liaison du détecteur d'immersion dans l'eau

```yaml
Alias : "Détection d'inondation - Fermer la vanne d'eau"
déclencheurs :
  - déclencheur : état
    entité_id : binaire_sensor.heiman_water_leak
    à : "on"
actions :
  - action : switch.turn_off
    entité_id : switch.main_water_valve
  - action : notify.mobile_app_phone
    données :
      message : "💧 Fuite d'eau détectée ! La vanne d'eau principale a été fermée !"
```

## Armement hors de chez soi : le capteur du corps humain déclenche l'alarme

```yaml
Alias : "Mode Quitter la maison - Alarme du capteur du corps humain"
déclencheurs :
  - déclencheur : état
    entité_id : binaire_sensor.heiman_motion_sensor
    à : "on"
conditions :
  - état : état
    entité_id : alarm_control_panel.home_alarm
    état : armé_away
actions :
  - action : alarm_control_panel.trigger
    entité_id : alarm_control_panel.home_alarm
```
