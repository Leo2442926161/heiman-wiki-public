# Conseils d'utilisation

## Configuration de l'appareil

- Lorsque vous associez votre appareil Heiman Matter pour la première fois, assurez-vous que votre routeur frontalier (par exemple HomePod, Nest Hub, Home Assistant Green) est sur le même réseau Thread.
- Lors de l'intégration d'appareils Zigbee via ZHA ou Zigbee2MQTT, il est recommandé d'utiliser d'abord le firmware recommandé du coordinateur USB (tel que Sonoff ZBDongle-E, Conbee II)

## Optimisation automatisée

- Ajout du paramètre `for:` (par exemple 3 secondes) à l'automatisation des détecteurs de fumée pour éviter les fausses alarmes instantanées
- Utilisez « condition : état » pour correspondre au mode de sécurité afin d'éviter les notifications inutiles dans l'état armé
- Utilisez la fonction « groupe » de Home Assistant pour regrouper les capteurs Heiman par zone pour faciliter l'opération par lots

## Dépannage

- Lorsque l'appareil est hors ligne, vérifiez d'abord si le canal réseau Zigbee est en conflit avec le WiFi 2,4 GHz.
- Lorsque l'appareil Matter ne peut pas être découvert, redémarrez le routeur frontalier et rajoutez l'appareil
