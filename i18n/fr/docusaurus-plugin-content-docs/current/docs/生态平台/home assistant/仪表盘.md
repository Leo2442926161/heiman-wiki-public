# Configuration du tableau de bord

## Panneau de présentation de la sécurité

Utilisez la présentation de la vue en coupe pour organiser l'équipement Heimann en zones :

```yaml
type : section
titre : Détecteur de fumée et de CO
cartes :
  - type : entité
    entité : binaire_sensor.heiman_smoke_alarm
    nom : détecteur de fumée
  - type : entité
    entité : binaire_sensor.heiman_co_alarm
    nom : alarme CO
```

## Style de carte champignon

```yaml
type : personnalisé : carte-panneau-de-contrôle-d'alarme-champignon
entité : alarm_control_panel.home_alarm
nom : système de sécurité

type : personnalisé : carte d'entité champignon
entité : binaire_sensor.heiman_smoke_alarm
nom : détecteur de fumée
icône : mdi: détecteur de fumée
Primary_info : état
secondaire_info : dernière modification
fill_container : vrai
```

## Panneau de capteurs environnementaux

```yaml
type : section
titre : Surveillance environnementale
cartes :
  - type : jauge
    entité : sensor.heiman_temperature
    nom : température
    unité : "°C"
    min : 0
    maximum : 50
    gravité :
      vert : 0
      jaune : 30
      Rouge : 40
  - type : jauge
    entité : sensor.heiman_humidity
    nom : humidité
    unité : "%"
    min : 0
    maximum : 100
    gravité :
      vert : 30
      Jaune : 60
      Rouge : 80
```

## Panneau d'historique des alarmes

```yaml
type : historique-graphique
titre : Historique des capteurs
entités :
  - binaire_sensor.heiman_smoke_alarm
  - binaire_sensor.heiman_co_alarm
  - binaire_sensor.heiman_water_leak
heures_to_show : 24
```

## Disposition suggérée

- En haut : état du système de sécurité + carte d'état du dispositif d'alarme
- Central : Surveillance environnementale (température, humidité, qualité de l'eau)
- En bas : graphique de tendance historique des alarmes du capteur
- Opérations de raccourci de la barre latérale : armer/désarmer, changement de scène
