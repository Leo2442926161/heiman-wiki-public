# Conseils d'utilisation

## Couplage d'appareils

1. Entrez « Paramètres → Appareils et services → ZHA → Configuration » dans Home Assistant
2. Cliquez sur "Ajouter un appareil", puis démarrez le mode d'appairage de l'appareil Heiman
3. Si le couplage échoue, rapprochez l'appareil du coordinateur USB et réessayez dans un rayon de 3 mètres.

## Coordinateur USB préféré

| Modèle | Puce | Indice de recommandations |
|------|------|----------|
| Sonoff ZBDongle-E | EFR32MG21 | ⭐⭐⭐⭐⭐ |
| Conbée II | 2652 | ⭐⭐⭐⭐ |
| Elelabs Zigbee USB | 2652P | ⭐⭐⭐⭐ |

## Coordination des canaux

Le canal par défaut de ZHA est 15. En cas d'interférences graves avec le WiFi 2,4 GHz :
- Utilisez `Silicon Labs Flasher Utility` pour modifier le canal
- Canaux couramment recommandés : 15, 20, 25 (éviter le WiFi 1/6/11)

## Optimisation du réseau

- Ajoutez un périphérique de routage Zigbee (tel qu'une prise intelligente) tous les 3 à 4 appareils Heiman pour étendre le réseau
- Vérifiez la qualité de la connexion des appareils avec l'outil de visualisation du réseau ZHA
- Vérifiez régulièrement les mises à jour du micrologiciel de l'appareil pour maintenir une compatibilité optimale
