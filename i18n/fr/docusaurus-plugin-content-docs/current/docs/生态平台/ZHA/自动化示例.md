# Exemple d'automatisation

## Raccordement du détecteur de fumée

```yaml
alias : « Connexion du détecteur de fumée ZHA »
déclencheurs :
  - déclencheur : état
    entité_id : binaire_sensor.heiman_smoke_zha
    à : "on"
actions :
  - action : light.turn_on
    cible :
      zone_id : salon_chambre
  - action : tts.speak
    cible :
      entité_id : media_player.speaker
    données :
      message : « Alarme de fumée déclenchée !
```

## Automatisation du couplage des appareils

```yaml
Alias : "Rappel de couplage d'un nouveau périphérique"
déclencheurs :
  - déclencheur : zha.event
    sous-type : Heiman
    commande : device_joined
actions :
  - action : notify.persistent_notification
    données :
      message : "Le nouvel appareil Haiman a rejoint le réseau ZHA, veuillez accéder au nom et à la zone de configuration"
```

## Détection d'immersion dans l'eau

```yaml
pseudonyme : "Test d'inondation ZHA"
déclencheurs :
  - déclencheur : état
    entité_id : binaire_sensor.heiman_water_leak_zha
    à : "on"
actions :
  - action : switch.turn_off
    entité_id : switch.main_water_valve
  - action : notify.mobile_app_phone
    données :
      message : "💧 Inondation détectée !"
```
