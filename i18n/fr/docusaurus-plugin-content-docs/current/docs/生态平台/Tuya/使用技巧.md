# Conseils d'utilisation

## Tap-to-Run vs automatisation

| Tapez | Méthode de déclenchement | Scénarios applicables |
|------|----------|--------------|
| **Appuyez pour exécuter** | Clic/voix manuel | Quitter la maison, rentrer à la maison, mode visionnage de films, etc. |
| **Automatisation** | Déclenchement automatique du capteur | Détecteur de fumée, détection de fuite d'eau, etc. |

Les équipements de type capteur Heiman doivent utiliser « Automation » et les exigences de type de scène doivent utiliser « Tap-to-Run ».

## Cloud vers local

- Activez « Communication LAN » dans l'application Smart Life pour réduire la dépendance au cloud
- Certains appareils Heiman prennent en charge l'exécution locale et peuvent être liés même si le réseau est déconnecté.
- Chemin de configuration : Scène → Paramètres dans le coin supérieur droit → "Activer l'exécution locale"

## Partage d'appareils

- Partagez des appareils Heiman avec les membres de votre famille dans l'application Smart Life
- Les utilisateurs partagés peuvent recevoir des notifications d'alarme mais ne peuvent pas modifier les configurations des appareils
- Partagez avec jusqu'à 20 membres de la famille

## Accès à la plateforme tierce

Les comptes Tuya peuvent être liés à Alexa et Google Home pour obtenir un contrôle vocal multiplateforme. Chemin de liaison : Smart Life → Paramètres → Contrôle tiers.
