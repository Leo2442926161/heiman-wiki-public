#GoogleAccueil

Google Home est la plate-forme de maison intelligente de Google qui prend en charge le protocole Matter et un large éventail d'écosystèmes d'appareils.

## Méthode d'intégration

### Matériel de matière

Les appareils Hyman Matter over Thread peuvent être ajoutés directement à l'application Google Home :

1. Assurez-vous de disposer d'un routeur de bordure Thread (Google Nest Hub Max, Nest Hub 2e génération, etc.)
2. Ouvrez l'application Google Home
3. Cliquez sur "Ajouter" → "Appareils compatibles Matter"
4. Scannez le code de couplage Matter sur votre appareil

### Commande vocale

Une fois connecté, vous pouvez le contrôler via la voix de Google Assistant :

- "Ok Google, quel est l'état du détecteur de fumée ?"
- "Hey Google, quelle est la température dans le salon ?"

> **Référence** : les appareils Hyman Matter sont marqués sur Amazon comme prenant en charge Google Home
