# Exemple d'automatisation

## Routine "Je sors"

**Déclencheur :** Dites "Je suis absent" sur Google Home

**Exécuter l'action :**
- Capteur magnétique de porte Heiman → bras
- Capteur infrarouge du corps humain Heiman → Activer la détection du départ à domicile
- Éteignez toutes les lumières et tous les appareils
- Thermostat → Régler sur une température d'économie d'énergie
- Google Nest Hub → Afficher la carte de confirmation de départ
- Réponse vocale : "Le mode Quitter la maison a été activé, bon voyage"

## Diffusion "Détection de fumée"

**Déclencheur :** Alarme de fumée Hyman → Fumée détectée

**Exécuter l'action :**
- Google Nest Speaker → Whole House Broadcast "⚠️ Fumée détectée ! Veuillez vérifier immédiatement !"
- Toutes les lumières intelligentes → Allumées (via le jeu d'éclairage intelligent Google Home)
- Notifications push sur les téléphones mobiles des membres de la famille
- Appareil photo → Démarrer l'enregistrement

## Mode heure du coucher

**Déclencheur :** Dites "Bonne nuit" à Google Home

**Exécuter l'action :**
- Capteur de porte Heiman → Activer l'alarme d'ouverture de porte
- Infrarouge du corps humain Heiman → activer la détection nocturne
- Lumières → s'éteignent progressivement
- Réponse vocale : "Bonne nuit, le système de sécurité est activé"
