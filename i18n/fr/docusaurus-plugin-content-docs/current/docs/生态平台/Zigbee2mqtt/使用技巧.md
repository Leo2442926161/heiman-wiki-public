# Conseils d'utilisation

## Sélection des chaînes

Les appareils Hyman Zigbee utilisent le canal 15 par défaut. Il est recommandé de spécifier manuellement dans `configuration.yaml` :

```yaml
avancé :
  canal : 15
```

Évitez les chevauchements avec le canal WiFi 2,4 GHz 1/6/11. Utilisez un analyseur WiFi pour scanner et sélectionner le canal Zigbee avec le moins d'interférences.

## Sauvegarde de la clé réseau

**Assurez-vous de sauvegarder `coordinator_backup.txt` et `secret.yaml` : **

```bash
cp data/coordinator_backup.txt data/coordinator_backup.bak
cp data/secret.yaml data/secret.yaml.bak
```

Tous les appareils couplés doivent se reconnecter après avoir perdu la clé.

## Optimisations spécifiques aux appareils

- Les appareils Heiman signalent par défaut la puissance de la batterie toutes les 60 minutes, et l'intervalle peut être raccourci grâce à la configuration de « rapport »
- Ajout de la configuration `homeassistant : true` pour générer automatiquement des entités sans écrire manuellement la configuration de découverte MQTT
- Utilisez `friendly_name` pour définir un nom lisible par l'homme pour l'appareil Heiman

## Mettre à jour le micrologiciel

Certains appareils Heiman Zigbee prennent en charge les mises à jour OTA et l'option « ota » doit être activée dans « configuration.yaml ».
