# Exemple d'automatisation

## Règle Machine Détecteur de Fumée

**Nom de la règle :** Liaison de sécurité du détecteur de fumée

**Configuration :**
```
Déclencheur : le détecteur de fumée Hyman devient actif
Conditions : Mode != Désarmé
Action :
  - Allumez toutes les lumières
  - Envoyer une notification push "⚠️ Alarme fumée !"
  - Allumez la sirène (si disponible)
  - Régler le mode sur "Urgence"
```

## Règle Surveillance de la température de la machine

**Nom de la règle :** Protection contre les hautes températures

**Configuration :**
```
Déclencheur : capteur de température Heimann > 38°C
Condition : État du climatiseur = éteint
Action :
  - Allumer la climatisation, mode refroidissement, 24°C
  - Envoyer une notification "La température ambiante est trop élevée et le climatiseur a été démarré"
```

## Règle Détection des fuites d'eau de la machine

**Nom de la règle :** Traitement d'urgence des fuites d'eau

**Configuration :**
```
Déclencheur : le détecteur d'immersion dans l'eau Hyman devient mouillé
Condition : État de la vanne d'eau = ouverte
Action :
  - Fermez le robinet d'eau principal
  - Envoyez des notifications push à tous les téléphones mobiles
  - Envoyer une notification par SMS (facultatif)
```
