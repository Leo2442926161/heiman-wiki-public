# Conseils d'utilisation

## Pilote personnalisé

Les appareils Heimann sont optimisés pour la compatibilité dans Hubitat avec les pilotes suivants :

| Type d'appareil | Pilote recommandé |
|----------|----------|
| Avertisseur de fumée/CO | Capteur de fumée/CO Zigbee générique |
| Capteur infrarouge du corps humain | Capteur de mouvement Heiman (tiers) |
| Capteur magnétique de porte | Capteur de contact Zigbee générique |
| Capteur de fuite d'eau | Capteur de fuite d'eau Zigbee générique |
| Capteur de température et d'humidité | Capteur de température/humidité Heiman |

## Optimisation Zigbee

- Les modèles Hubitat C-8 et supérieurs utilisent la puce EFR32, qui offre la meilleure compatibilité
- Affichez et sélectionnez la chaîne avec le moins d'interférences dans Paramètres → Zigbee
- Utilisez les outils de détection Zigbee tels que Ubisys pour diagnostiquer les appels interrompus

## Invites de règles

- Il est recommandé d'ajouter la condition "Délai anti-fausse alarme" (0,5-2 secondes) dans Rule Machine
- Activer les notifications de « bilan de santé » pour les règles clés afin de garantir qu'elles continuent à s'exécuter
- Organisez l'automatisation complexe en mode "sous-règles" pour un débogage et une maintenance faciles

## Coexistence Z-Wave

Si vous souhaitez utiliser des appareils Z-Wave en même temps, veillez à maintenir l'espacement de fréquence entre les canaux Zigbee et Z-Wave.
