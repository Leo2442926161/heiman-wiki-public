# Déploiement de Solutions

Les solutions Heiman Technology peuvent être déployées dans divers environnements. Voici un guide pour vous aider à démarrer.

## Déploiement Cloud

Nos solutions sont compatibles avec les principales plateformes cloud :

- AWS IoT Core
- Microsoft Azure IoT Hub
- Alibaba Cloud IoT
- Google Cloud IoT Core

## Déploiement Sur Site

Pour les clients nécessitant un déploiement local, nous fournissons des solutions complètes sur site avec :

- Passerelle de données locale
- Serveur de gestion en périphérie
- Application mobile pour la gestion sur site

## Étapes de Déploiement

1. Évaluation du site et planification
2. Installation et configuration des dispositifs
3. Configuration réseau et tests de connectivité
4. Mise en service du système et calibrage
5. Formation des utilisateurs et documentation

## Commandes

```bash
./configure --enable-iot --platform=linux
make && make install
systemctl start heiman-service
```

Pour plus d'informations, veuillez vous référer au guide d'installation du produit spécifique.
