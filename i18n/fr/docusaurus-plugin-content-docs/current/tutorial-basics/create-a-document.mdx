# Centre de Documentation

Bienvenue au centre de documentation de Heiman Technology. Ici, vous trouverez des informations détaillées sur les produits, des guides d'installation, des documents de développement et bien plus encore.

## Types de Documents

### Spécifications Produit
Spécifications techniques détaillées, indicateurs de performance et caractéristiques pour tous les produits Heiman.

### Guides d'Installation
Instructions étape par étape pour l'installation et la configuration des dispositifs, y compris les schémas de câblage et les exigences d'alimentation.

### Documentation Développeur
Documents d'intégration d'API, protocoles de communication (y compris Matter, Zigbee, NB-IoT) et guides de connexion à des plateformes tierces.

### FAQ
Questions fréquemment posées sur l'installation des produits, le dépannage et la maintenance.

## Comment Naviguer

Utilisez la barre latérale de gauche pour parcourir les catégories de documents. Vous pouvez également utiliser la barre de recherche en haut pour trouver rapidement des informations spécifiques.

## Support

Si vous ne trouvez pas ce que vous cherchez, veuillez [nous contacter](https://www.heiman.com.cn) pour obtenir de l'aide.
