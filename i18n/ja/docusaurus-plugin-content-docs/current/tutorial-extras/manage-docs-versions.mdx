# ドキュメントバージョン管理

海曼科技 Wiki はマルチバージョンのドキュメント管理をサポートしており、製品ファームウェアバージョンに対応した技術ドキュメントにアクセスできます。

## バージョン対応表

| ドキュメントバージョン | 対応製品ファームウェア | リリース日 |
|---------|------------|---------|
| 2.0    | Matter シリーズ  | 2025-06 |
| 1.0    | Zigbee シリーズ  | 2024-01 |

## 異なるバージョンの表示

ページ上部のナビゲーションバーにあるバージョンドロップダウンメニューから、異なるバージョンのドキュメントを切り替えて表示できます。

## バージョン更新について

製品の反復に合わせてドキュメントを継続的に更新しています。最新バージョンを定期的にご確認ください。
