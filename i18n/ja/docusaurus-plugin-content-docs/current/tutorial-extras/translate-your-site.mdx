# 多言語対応

海曼科技 Wiki は多言語ドキュメントに対応しており、世界中のパートナーやお客様がご利用いただけます。

## 現在対応している言語

- **簡体字中文 (zh-Hans)** — デフォルト
- **日本語 (ja)** — 追加予定

## 国際化設定

海曼製品は多言語対応の APP インターフェースとデバイス音声ガイダンスをサポートしており、中文、英語、日本語、韓国語、スペイン語などに対応しています。

## 翻訳への貢献

海曼技術ドキュメントの翻訳にご協力いただける場合は、公式サイトからお問い合わせください。
