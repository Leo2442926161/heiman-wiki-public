# 製品センター

海曼科技は、スマート消防、スマートガス、スマート介護、スマートセンシングの4つの主要製品ラインを提供しています。

## スマート消防

煙警報器、一酸化炭素警報器、温度検知器などの製品を含み、セルラーネットワーク（4G/NB-IoT）接続をサポートし、サードパーティのIoTプラットフォームと容易に連携できます。

主な特長：

- 全ネットワーク対応、マルチプラットフォーム連携
- EN54-7、UL 217 などの国際規格に準拠
- Matter プロトコルに対応、Alexa、Google Home、Apple HomeKit と互換性あり

## スマートガス

産業用及び商業用のスマートガス警報システム。多様な環境に全方位的なスマートガス安全ソリューションを提供します。

主な特長：

- 高精度ガス検知センサー
- リアルタイム遠隔監視と警報
- ガス緊急遮断弁との自動連動

## スマート介護

煙警報器、ガス警報器、水浸検知器、人体転倒センサー、玄関防犯機器及び緊急警報器を含みます。

主な特長：

- 転倒検知 AI アルゴリズム
- ワンタッチ緊急通報
- マルチ端末連動で家族と介護機関に通知

## スマートセンシング

赤外線検知、ドア/窓磁気センシング、温湿度モニタリング、空気品質検知など全シーンのセンサーをカバーします。

主な特長：

- 超低消費電力設計
- Zigbee / Matter / WiFi マルチプロトコル対応
- 主流スマートホームプラットフォームに接続可能
