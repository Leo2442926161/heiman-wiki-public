# ソリューション導入

海曼科技は、エンドツーエンドのスマートセキュリティソリューション導入ガイドを提供します。

## ソリューション統合プロセス

海曼製品は、ローカル独立導入とクラウドプラットフォーム連携を含む複数の導入方式をサポートしています。

### ローカル導入

```bash
# 海曼スマート消防ゲートウェイの設定
heiman-cli configure --gateway HS1GW --mode local

# 検知器デバイスの登録
heiman-cli register --device HS1SA --gateway HS1GW
```

### クラウドプラットフォーム連携

海曼製品は主要なIoTプラットフォームに接続済みです：

- 中国電信天翼
- 中国移動 OneNET
- ファーウェイ HiLink
- アリババクラウド IoT
- 塗鴉智能

## Matter プロトコル対応

海曼 Matter シリーズ製品は以下と互換性があります：

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

クロスブランド、クロスエコシステムの相互接続を実現します。

## 導入検証

導入完了後は、海曼スマート消防 APP または管理プラットフォームから機能検証とリアルタイム監視を行うことができます。
