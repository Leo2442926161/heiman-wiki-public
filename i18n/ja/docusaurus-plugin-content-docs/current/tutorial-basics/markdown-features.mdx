# Markdown 機能

海曼の技術ドキュメントは Markdown 構文と拡張機能を全面的にサポートしており、構造化された製品技術文書の作成に便利です。

## コード例

製品開発連携における API 呼び出しの例：

```jsx title="煙警報器 API 連携"
const heimanClient = new HeimanClient({
  apiKey: 'your-api-key',
  deviceId: 'HS1SA-xxxxx',
});

// デバイス状態の取得
const status = await heimanClient.getDeviceStatus();
console.log(status);
```

## 注意喚起ボックス

重要な情報を強調するために注意喚起ボックスを使用します：

:::tip

海曼煙警報器 HS1SA は EN14604 認証を取得しています

:::

:::caution

ガス警報器を設置する際は、設置ガイドに従って正しく配線してください

:::

:::danger

ガス漏れ時は電気スイッチに触れず、すぐに窓を開けて換気し避難してください

:::
