# GB 中国国家規格

GB（Guóbiāo、National Standard）は、中華人民共和国国家標準の略称で、国務院標準化行政部門によって制定されます。

## Haiman 製品に関係する主な GB 規格

|規格番号 |規格名 |対象製品 |
|----------|----------|----------|
| GB 20517 |独立した煙感知式火災検知警報器 |煙警報器 |
| GB 15322.1 |可燃性ガス検知器 その1：産業用・商業用 |工業用ガス検知器 |
| GB 15322.2 |可燃性ガス検知器その２：家庭用 |家庭用ガス検知器 |
| GB/T 31141 |家庭用一酸化炭素警報器 |一酸化炭素警報 |
| GB 4706.1 |家庭用および同様の電気製品の安全性 |スマートソケット |

## CCCとGBの関係

CCC認証（中国強制認証）はGB規格を技術基盤としています。 CCC 認証に合格した製品は、関連する GB 規格に準拠していることを証明します。
