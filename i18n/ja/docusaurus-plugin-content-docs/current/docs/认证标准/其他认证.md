# その他の認定

## Q-Label 品質マーク

Q-Label は、Q-Lab Alliance (現在は Zigbee Alliance / CSA Connectivity Standards Alliance の一部) によって立ち上げられた、ヨーロッパのスマート ホーム業界の品質認証マークです。

|プロジェクト |説明 |
|------|------|
|適用範囲 | Zigbee デバイスの相互運用性認証 |
|認定内容 |他のブランドの Zigbee ゲートウェイ/コントローラーとのデバイス互換性 |
|重要性 |ブランド間の相互運用性を確保し、「家で買っても使わない」リスクを軽減 |

Hyman Zigbee 製品は Q-Label 認定を受けており、次のプラットフォームとの互換性が保証されています。
- Amazon Echo Plus / Echo Show (Zigbee内蔵)
- フィリップス ヒュー ブリッジ
- スマートシングスハブ
- その他の Q-Label 認証ゲートウェイ

## レッド・ドット・デザイン賞

ハイマンのHS1SAシリーズ煙警報器は**レッド・ドット・デザイン賞**(レッド・ドット・デザイン賞)を受賞しており、これはハイマンの製品デザインにおける強みを反映しています。

## ISO 9001 品質マネジメントシステム

Haiman は ISO 9001:2015 品質管理システム認証を取得し、研究開発、生産から販売までの全プロセスの品質管理を保証します。
