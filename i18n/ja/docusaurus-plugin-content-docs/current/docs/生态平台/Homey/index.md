#アットホーム

Homey はオランダの Athom 社が発売したスマート ホーム ハブで、Zigbee、Z-Wave、Wi-Fi、Matter、Thread などのプロトコルをサポートしています。

## 統合方法

Homey Zigbee と Z-Wave デバイスは、Homey の内蔵ワイヤレス チップを介して直接ペアリングできるため、追加のゲートウェイは必要ありません。 Matter デバイスには、Homey Pro の内蔵 Thread ボーダー ルーターを通じてアクセスできます。

## 利点

- **複数のプロトコルを 1 つに**: Homey Pro は、Zigbee、Z-Wave、Thread、Wi-Fi を同時にサポートします。
- **ローカルで実行**: すべてのオートメーションはローカルで実行されます
- **フロー オートメーション**: 複雑なリンク シナリオを作成するためのドラッグ アンド ドロップ プログラミング
