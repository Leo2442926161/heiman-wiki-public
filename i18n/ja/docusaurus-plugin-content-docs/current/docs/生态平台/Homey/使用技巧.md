# 使い方のヒント

## Zigbee チャネル干渉

Homey の内蔵 Zigbee チップは特定のチャネルを使用します。 Heiman デバイスが頻繁にオフラインになる場合:

1. Homey設定→Zigbeeで現在のチャンネルを表示します。
2. Homey Insight アプリを使用して環境干渉をスキャンします
3. 干渉の少ないチャンネルに切り替えます (WiFi 1/6/11 から離れた場所)

## 機器流通ネットワーク

- Heiman デバイスのペアリング モードに入る方法: ボタンを 5 秒間押し続けます (一部のモデルはすばやく 3 回押す必要があります)
- Homeyアプリでデバイスを追加する場合、「デバイスの追加→Zigbee→検索」を選択します
- ペアリングに失敗した場合は、デバイスをHomeyに近づけて(3メートル以内)、もう一度試してください。

## フロー最適化の提案

- 一時的な誤検知を回避するために、フローに「遅延」条件を追加します (例: 2 秒)。
- 「タグ」機能を使用してハイマン機器を分類して管理します

## 信号範囲の拡張

Heiman デバイスが Homey から遠く離れている場合は、Zigbee リレー デバイス (スマート ソケットなど) を追加して、ネットワーク カバレッジを拡張できます。
