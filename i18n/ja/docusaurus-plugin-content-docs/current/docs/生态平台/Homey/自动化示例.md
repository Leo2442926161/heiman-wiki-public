# 自動化の例

## 煙警報器 - 点滅して通知します

**フロー構成:**

1. **トリガー:** ハイマン煙警報器 → ステータスが「警報」に変わります
2. **条件:** (なし)
3. **アクション:**
   - リビングルームの照明 → 点滅（赤、1秒に1回）
   - プッシュ通知 → モバイル: 「⚠️ 煙警報器が作動しました!」
   - 音声アナウンス → ホーミースピーカー：「煙を検知しました。安全にご注意ください！」

## CO アラーム - ボイラーを停止します

**フロー構成:**

1. **トリガー:** ハイマン CO アラーム → ステータスが「アラート」に変わります
2. **条件:** (なし)
3. **アクション:**
   ・ボイラースイッチ→OFF
   - スマートソケット（フレッシュエアシステム）→オン
   ・プッシュ通知→「一酸化炭素警報器、ボイラー停止しました」

## 温湿度オーバーリミットアラーム

1. **トリガー:** ハイマン温湿度センサー → 温度が 40°C 以上
2. **アクション:**
   - エアコン → 冷房モードをオンにする
   ・プッシュ通知→「室温が高すぎるため、エアコンが自動でONになりました」
