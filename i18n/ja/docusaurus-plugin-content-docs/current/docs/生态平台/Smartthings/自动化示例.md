# 自動化の例

## 「おやすみ」就寝モード

**トリガー条件:** Haiman ドア磁気センサー → 閉じる (寝室のドアが閉まっていることを示します)

**アクションの実行:**
- リビングルームの照明をすべて消してください
- スマートロック→ロック
- セキュリティシステム → 夜間の警戒
- プッシュ通知: 「おやすみ、システムは準備完了です」

## 「外出中」モード

**トリガー条件:** Hyman 人体赤外線センサー → 15 分間動きがない

**アクションの実行:**
- すべての照明が消えた
- サーモスタット → 省エネモード
- セキュリティシステム → 外出禁止
- ドアセンサー → アラームをオンにする
- プッシュ通知: 「外出モードが有効になり、ドアと窓のセンサーが有効になりました」

## 煙警報器緊急モード

**トリガー条件:** ハイマン煙警報器 → 煙を検出

**アクションの実行:**
- スマートスピーカー → アラーム音を再生
- すべてのスマートライト → 赤で点滅
- 家族全員にプッシュ通知
- SmartThings Hubに接続されている場合 → サイレンをトリガー
