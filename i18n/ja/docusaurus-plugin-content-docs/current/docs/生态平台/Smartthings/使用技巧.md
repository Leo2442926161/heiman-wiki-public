# 使い方のヒント

## ハブの互換性

Hyman Zigbee デバイスは **SmartThings Hub v2** および **Hub v3 (Aeotec)** と互換性があります。

|ハブモデル |互換性 |説明 |
|----------|----------|------|
|ハブ v2 (STH-ETH-250) | ✅ |ファームウェアを最新のものにアップデートする必要があります |
|ハブ v3 (Aeotec) | ✅ |ネイティブ サポート、推奨 |
|サムスン スマートシングス ステーション | ⚠️ |一部の Zigbee デバイスのみをサポートします。

## デバイスのペアリングのプロンプト

- SmartThings アプリで「+」→「デバイス」→「ブランド」→「Hyman」をクリックします。
- 自動的に認識されないデバイスは、「Scan Zigbee」を通じて手動で検索できます
- 最初のペアリングでは、デバイスとハブの間の距離を 5 メートル以内に保つことをお勧めします

## バッテリーの最適化

Heiman センサーのデフォルトのレポート間隔はより長くなります。リアルタイムのパフォーマンスが必要な場合は、デバイス ページで「検出間隔」を 5 分に調整できます。

## 自動化のベスト プラクティス

- SmartThings の「モード」機能を使用して、「在宅/外出中」状態を区別します
- 冗長性を確保するために、主要センサー (煙、一酸化炭素) の自動化を **2 セット**作成します
