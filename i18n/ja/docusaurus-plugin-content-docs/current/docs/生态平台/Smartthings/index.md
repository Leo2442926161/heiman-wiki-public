#スマートシングス

SmartThings は、Samsung が発売したスマート ホーム プラットフォームで、Zigbee、Z-Wave、Matter プロトコルをサポートしています。

## 統合方法

Heiman デバイスは、次の方法で SmartThings に接続できます。

1. **Matter デバイス**: SmartThings Hub または Matter 対応の Samsung デバイス (例: Family Hub 冷蔵庫、SmartThings Station) 経由で直接追加します。
2. **Zigbee デバイス**: SmartThings Hub v2/v3 には Zigbee チップが組み込まれており、Heiman Zigbee センサーに直接接続できます。
3. **Z-Wave デバイス**: SmartThings Hub は Z-Wave をサポートしており、Heiman Z-Wave デバイスは直接ペアリングできます

## 認証ステータス

Hyman Matter 人体赤外線センサーは、Amazon などのチャネルで SmartThings をサポートしているとマークされており、ユーザーから良い評価を受けています。
