# 自動化の例

## Alexa Guard の統合 (米国)

**前提条件:** Alexa Guard Plus に登録する (米国で利用可能)

**構成プロセス:**
1.AlexaアプリでGuard Plusをオンにする
2. Hyman 煙/一酸化炭素警報器を「スマート ホーム デバイス」として追加します。
3.「煙警報検知」スマート応答を有効にする

**自動応答:**
- 煙警報器が作動 → Alexa が自動的に緊急連絡先にダイヤル
- COアラームトリガー→Alexa音声で家全体に警告、自動換気
- ドアや窓が異常に開く → プッシュアラーム通知

## 煙検知ルーチン

**トリガー:** ハイマン煙警報 → 煙を検出

**アクションの実行:**
- Alexa音声アナウンス：「⚠️煙を検知しました。安全にご注意ください」
- すべての Echo デバイス → アラーム音を再生する
- スマートライト設定 → 全点灯
- 携帯電話へのプッシュ通知

## 「家を出る」ルーチン

**トリガー:** 音声コマンド「アレクサ、出発します」

**アクションの実行:**
- ハイマンドア磁気センサー → アーミングをオンにします
- Heiman Body Sensor → モーション検知をオンにします
- ハイマン浸水検知器 → 検知をオンにする
- 全灯 → 消灯
- Alexa は「セキュリティ システムがオンになっています。安全にご旅行ください」と答えました。
