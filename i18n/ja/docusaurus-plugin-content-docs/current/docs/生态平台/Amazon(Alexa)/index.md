#アマゾンアレクサ

Amazon Alexa は、世界で最も人気のあるスマート音声アシスタント プラットフォームの 1 つであり、Matter プロトコルと豊富なスキル エコシステムをサポートしています。

## 統合方法

### 重要な装備

Hyman Matter over Thread デバイスには、Matter 対応の Echo デバイス (Echo 第 4 世代、Echo Plus、Echo Show 10 など) を通じてアクセスできます。

1. Echo デバイスが最新のファームウェアに更新されていることを確認します
2. Alexaアプリ→デバイス→デバイスの追加を開きます
3.「Matter」を選択し、ペアリングコードをスキャンします

### Zigbee デバイス

一部の Echo Plus および Echo Show には Zigbee チップが組み込まれており、Heiman Zigbee センサーと直接ペアリングできます。

## 音声コマンドの例

- 「アレクサ、煙感知器に何も問題ないか聞いて」
- 「アレクサ、リビングルームの温度は何度ですか?」
