# 使い方のヒント

## デバイスの検出

- Heiman Matter デバイスの電源がオンになると、通常 30 秒以内に Alexa アプリによって自動的に検出されます。
- 自動的に検出されない場合：Alexaアプリ→デバイス→「+」→「デバイスを追加」→「件」と入力します。
- デバイスの底面にあるセットアップ コードまたは QR コードを使用してペアリングを完了します

## Zigbee デバイスアクセス (Echo Plus / Echo Show)

- Zigbee Hub を備えた Echo デバイスは、Heiman Zigbee センサーと直接ペアリングできます
- ペアリングモード：Alexaアプリ→デバイス→「+」→「Zigbee」→「デバイスを検索」
- Heimann デバイスは Alexa と互換性があり、Zigbee コーディネーターが組み込まれています

## 日常的なヒント

- 「カスタム アクション」を使用して、より詳細な音声プロンプトをブロードキャストします
- 音声の重複を避けるために、後続のアクションを実行する前に緊急ルーチンに「5秒待つ」を追加します。
- 「サウンド検出」機能を利用して、Heyman Alarm の高音アラームに対して Alexa に追加の応答を与えます

## アレクサガードメモ

Alexa Guard Plus は現在米国でのみ利用可能です。中国のユーザーは、ルーチンを通じて一部の機能をシミュレートできます。
