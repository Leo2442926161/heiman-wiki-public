# 使い方のヒント

## モノの構成

MQTT バインディング経由で Heiman デバイスを追加します。

```ジャワ
Thing mqtt:topic:heiman_smoke "ヘイマン煙警報器" @ "リビング ルーム" {
    チャンネル:
        バイナリ型: スモーク "スモーク状態" [ stateTopic="zigbee2mqtt/heiman_smoke"、transformationPattern="JSONPATH:$.smoke" ]
        タイプ番号: バッテリー "バッテリー電源" [stateTopic="zigbee2mqtt/heiman_smoke",transformationPattern="JSONPATH:$.battery" ]
}
「」

## 項目定義

```ジャバ
Smoke_Alarm に連絡する "煙警報 [MAP(smoke.map):%s]" { channel="mqtt:topic:heiman_smoke:smoke" }
Number Smoke_Battery "煙警報器バッテリー [%d%%]" { channel="mqtt:topic:heiman_smoke:battery" }
「」

## 座標の選択

**CC2531** または **CC2652** チップを搭載した USB コーディネーターを使用し、ファームウェアのバージョンが最新であることを確認することをお勧めします。

## デバッグのヒント

- Paper UI で Thing の「オンライン/オフライン」ステータスを表示
- `openhab-cli info` を使用して、バインディングが正しくロードされているかどうかを確認します
- ログ内で「heiman」キーワードを検索して、デバイスの通信の問題をトラブルシューティングします
