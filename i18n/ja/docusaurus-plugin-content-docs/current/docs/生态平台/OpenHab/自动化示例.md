# 自動化の例

## 煙警報器のルール

```JavaScript
ルール「煙警報器が作動しました」
いつ
    アイテム heiman_smoke_alarm が ON に変更されました

    sendCommand(リビングルーム_ライト、オン)
    sendNotification("管理者", "⚠️ 火災警報器が作動しました!")
    postUpdate(警報状態, "煙警報")
終わり
