# 使い方のヒント

## タップツーラン vs 自動化

|タイプ |トリガー方法 |該当するシナリオ |
|------|----------|----------|
| **タップトゥラン** |手動クリック/音声 |外出・帰宅・動画視聴モードなど |
| **自動化** |自動センサートリガー |煙警報器、漏水検知など |

Heiman センサー タイプの機器では「オートメーション」を使用し、シーン タイプの要件では「Tap-to-Run」を使用する必要があります。

## クラウドからローカルへ

- スマートライフアプリの「LAN通信」をONにしてクラウド依存度を軽減
- 一部の Heiman デバイスはローカル実行をサポートしており、ネットワークが切断されていてもリンクできます。
- 設定パス：シーン→右上の設定→「ローカル実行を有効にする」

## デバイスの共有

- Smart Life アプリで Heiman デバイスを家族と共有します
- 共有ユーザーはアラーム通知を受信できますが、デバイス構成を変更することはできません
- 最大20人の家族と共有

## サードパーティのプラットフォームへのアクセス

Tuya アカウントを Alexa および Google Home にバインドして、クロスプラットフォームの音声制御を実現できます。バインド パス: Smart Life → 設定 → サードパーティ コントロール。
