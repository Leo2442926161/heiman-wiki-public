#トゥヤ（トゥヤ）

Tuya (Tuya Smart) は世界をリードする IoT クラウド プラットフォームであり、Heiman はその重要なエコロジカル パートナーの 1 つです。 Tuya エコシステムを通じて、Heiman デバイスは 1,000 以上のブランドのスマート ホーム システムに迅速に接続できます。

## 統合方法

### スマートライフアプリ

Hyman Zigbee デバイスは、Tuya の Smart Life アプリ経由で設定および制御できます。

1. Tuya 対応 Zigbee ゲートウェイ (Heiman HS6GW シリーズなど) を準備します。
2. スマートライフアプリを開く → デバイスを追加
3. Zigbeeゲートウェイ→サブデバイスの検索を選択します
4. センサーをペアリングモードにして自動的に検出します

### クロスプラットフォームのサポート

Tuya エコシステムを通じて、Heiman の機器は次のプラットフォームにリンクできます。

-アマゾンアレクサ
- Google アシスタント
- Apple Siri ショートカット
- イフト

### ローカライズされた Tuya ゲートウェイ

Home Assistant の Tuya Local 統合と同様に、Tuya Local を使用すると、Heiman デバイスをローカルで直接制御できるため、クラウドでの遅延が軽減されます。

## 適用範囲

Tuya の互換性のある Heiman 製品ラインには、煙センサー、一酸化炭素センサー、ドアセンサー、人体赤外線、温度と湿度、浸水、ソケット、その他の Zigbee デバイスが含まれます。
