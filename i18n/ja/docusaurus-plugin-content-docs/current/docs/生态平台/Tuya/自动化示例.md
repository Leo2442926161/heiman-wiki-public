# 自動化の例

## 煙警報器のシーン

**シーン名:** 「煙警報器連携」

**トリガー条件:** ハイマン煙警報器 → 煙を検出

**アクションの実行:**
- スマートソケット→オープン（排気ファンを接続）
- ライト → 点灯して点滅します
- 携帯電話へのプッシュ通知
- スピーカー → アラーム音を再生

## ワンクリックで家を出るシーン (Tap-to-Run)

**シーン名：** 「家出後の手配」

**手動実行:** タップまたは音声トリガー

**アクションの実行:**
- ハイマンドアセンサー → アラームをオンにする
- ハイマン人体赤外線 → 検出をオンにします
- ハイマン浸水検知器 → 検知をオンにする
- 全灯 → 消灯
- スマートロック→ロック

## 温度と湿度の自動制御

**自動化タイプ:** 自動条件トリガー

**トリガー条件:** Hyman 温度および湿度センサー 湿度 > 75%

**アクションの実行:**
- スマートソケット（除湿器）→オン
- プッシュ通知: 「湿度が高すぎるため、除湿器がオンになりました」

**トリガー条件:** Hyman 温湿度センサー 湿度 < 50%

**アクションの実行:**
- スマートソケット（除湿機）→閉じる
