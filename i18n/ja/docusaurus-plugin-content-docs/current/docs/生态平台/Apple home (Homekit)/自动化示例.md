# 自動化の例

## 煙検知セキュリティアラーム

**HomeKit 自動構成:**

1. **トリガー条件:** ハイマン煙警報 → 煙を検知
2. **セキュリティ設定:** セキュリティ システムのステータスを「アラーム」に切り替えます。
3. **アクションの実行:**
   - HomePod → カスタムオーディオを再生「煙が検出されました。安全にご注意ください」
   - すべてのスマートライト → 点灯および点滅
   - 家族全員にプッシュ通知

## 自宅/外出先のシーン

**ショートカット ショートカット コマンド:「ホーム セキュリティ チェックを離れる」**

  セキュリティシステムを設定→外出時に腕を立てる
  Heiman Door Sensor を設定 → 通知をオンにする
  Heiman ボディセンサーを設定 → アラームをオンにする
  すべての照明を消してください
  設定 サーモスタット → 省エネモード
  「🏠外出中モードが有効になりました」という通知を送信します

## 温度異常リマインダー

**トリガー条件:** ハイマン温湿度センサー → 温度 > 35°C

**アクションの実行:**
- HomePod → 音声ブロードキャストの室温が制限を超えています
- iPhoneへのプッシュ通知
- エアコンを自動的にオンにします (HomeBridge が構成されている場合)
