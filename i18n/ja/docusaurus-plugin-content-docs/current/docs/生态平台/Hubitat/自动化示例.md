# 自動化の例

## ルールマシン煙警報器

**ルール名:** 煙警報器の安全リンク

**構成:**
「」
トリガー: ハイマン煙警報器が作動する
条件: モード != 武装解除
アクション:
  - すべての照明をオンにしてください
  - プッシュ通知「⚠️煙警報器！」を送信
  - サイレンをオンにします (可能な場合)
  - モードを「緊急」に設定します
「」

## ルールマシンの温度監視

**ルール名:** 高温保護

**構成:**
「」
トリガー: ハイマン温度センサー > 38°C
条件: エアコンの状態 = オフ
アクション:
  - エアコンをオン、冷房モード、24°C
  - 「室温が高すぎるためエアコンを開始しました」を通知する
「」

## ルールマシンの水漏れ検知

**ルール名:** 水漏れの応急処置

**構成:**
「」
トリガー: Hyman 浸水検知器が濡れた
条件: 給水バルブの状態 = 開いている
アクション:
  - 元栓を閉めてください
  - すべての携帯電話にプッシュ通知を送信します
  - SMS 通知の送信 (オプション)
「」
