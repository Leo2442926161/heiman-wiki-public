# 使い方のヒント

## カスタムドライバー

Heimann デバイスは、Hubitat で次のドライバーとの互換性が得られるように最適化されています。

|デバイスの種類 |推奨ドライバー |
|----------|----------|
|煙/一酸化炭素警報器 |汎用 Zigbee 煙/一酸化炭素センサー |
|人体赤外線センサー | Heiman モーション センサー (サードパーティ) |
|ドア磁気センサー |汎用 Zigbee 接触センサー |
|漏水センサー |汎用 Zigbee 漏水センサー |
|温湿度センサー | Heiman 温度/湿度センサー |

## Zigbee の最適化

- Hubitat C-8以降のモデルはEFR32チップを使用しており、互換性が最も優れています。
- [設定] → [Zigbee] で干渉が最も少ないチャンネルを表示して選択します。
- Ubisys などの Zigbee スニッフィング ツールを使用して切断された通話を診断する

## ルールのプロンプト

- ルール マシンに「誤報防止遅延」条件 (0.5 ～ 2 秒) を追加することをお勧めします。
- 主要なルールの「ヘルスチェック」通知を有効にして、ルールが継続的に実行されることを確認します
- 複雑な自動化を「サブルール」モードで整理し、デバッグとメンテナンスを容易にします

## Z-Wave の共存

Z-Wave デバイスを同時に使用する場合は、Zigbee チャネルと Z-Wave チャネル間の周波数間隔を必ず維持してください。
