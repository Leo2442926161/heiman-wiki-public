# 自動化の例

## 煙警報器連動

```ヤムル
別名：「ZHA煙警報器連動」
トリガー:
  - トリガー: 状態
    エンティティ ID: binary_sensor.heiman_smoke_zha
    に:「オン」
アクション:
  - アクション: light.turn_on
    ターゲット:
      エリアID: リビングルーム
  - アクション: tts.speak
    ターゲット:
      エンティティ ID: media_player.speaker
    データ:
      メッセージ: 「煙警報器が作動しました!」
「」

## デバイスのペアリングの自動化

```ヤムル
別名: 「新しいデバイスのペアリングリマインダー」
トリガー:
  - トリガー: zha.event
    サブタイプ: ヘイマン
    コマンド: device_joined
アクション:
  - アクション：notify.persistent_notification
    データ:
      メッセージ: 「新しい Haiman デバイスが ZHA ネットワークに参加しました。構成名とエリアに移動してください」
「」

## 水没検知

```ヤムル
別名：「ZHA フラッドテスト」
トリガー:
  - トリガー: 状態
    エンティティ ID: binary_sensor.heiman_water_leak_zha
    に:「オン」
アクション:
  - アクション: switch.turn_off
    エンティティID: switch.main_water_valve
  - アクション：notify.mobile_app_phone
    データ:
      メッセージ: 「💧 洪水が検出されました!」
「」
