# 使い方のヒント

## デバイスのペアリング

1.ホームアシスタントで「設定→デバイスとサービス→ZHA→構成」と入力します。
2.「デバイスの追加」をクリックして、Heiman デバイスのペアリングモードを開始します。
3. ペアリングに失敗した場合は、デバイスを USB コーディネーターに近づけて、3 メートル以内でもう一度試してください。

## チャネル調整

ZHA のデフォルト チャネルは 15 です。WiFi 2.4GHz に重大な干渉がある場合:
- 「Silicon Labs Flasher Utility」を使用してチャンネルを変更します
- 一般的に推奨されるチャンネル: 15、20、25 (WiFi 1/6/11 は避けてください)

## ネットワークの最適化

- ネットワークを拡張するには、3 ～ 4 台の Heiman デバイスごとに Zigbee ルーティング デバイス (スマート ソケットなど) を追加します。
- ZHAネットワーク可視化ツールでデバイスの接続品質を確認
- 最適な互換性を維持するために、デバイスのファームウェアのアップデートを定期的に確認します。
