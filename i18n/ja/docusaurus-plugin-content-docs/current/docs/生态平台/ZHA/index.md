#ZHA (Zigbee ホーム オートメーション)

ZHA は、Home Assistant に組み込まれた Zigbee 統合ソリューションで、追加のソフトウェアなしで Zigbee デバイスを直接管理できます。

## 互換性

Hyman Zigbee センサーは ZHA との互換性が高く、以下をサポートします。

- **煙/一酸化炭素警報**: 警報ステータス、故障検出、バッテリーレベル、ミュート
- **ドアセンサー**: オン/オフ、タンパー
- **人体赤外線**: 動き検出、照明
- **温度と湿度**: 温度、湿度
- **洪水**: 水漏れ警報
- **スマートソケット**: スイッチ、消費電力測定

## ハードウェア要件

- Zigbee コーディネーター: HUSBZB-1、Conbee II、SkyConnect、SLZB-06 など。
- USB またはネットワーク経由でホーム アシスタントに接続します

## 構成

ZHA は設定が簡単です。シリアル ポートとエリアを選択するだけで、デバイスが自動的にスキャンされて検出されます。設定ファイルを手動で記述する必要はありません。
