#GoogleHome

Google Home は、Matter プロトコルと幅広いデバイス エコシステムをサポートする Google のスマート ホーム プラットフォームです。

## 統合方法

### 重要な装備

Hyman Matter over Thread デバイスは、Google Home アプリに直接追加できます。

1. スレッド境界ルーター（Google Nest Hub Max、Nest Hub 第 2 世代など）があることを確認します。
2. Google Homeアプリを開きます
3.「追加」→「対応デバイス」をクリックします。
4. デバイス上の Matter ペアリング コードをスキャンします

### 音声コントロール

接続すると、Google アシスタントの音声で制御できます。

- 「OK Google、火災警報器の状態はどうですか？」
- 「OK Google、リビングルームの温度は何度ですか?」

> **参考**: Hyman Matter デバイスは Amazon で Google Home をサポートしているとマークされています
