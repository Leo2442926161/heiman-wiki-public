# 自動化の例

## 「外出中」ルーチン

**トリガー:** Google Home に「外出中」と言う

**アクションの実行:**
- ハイマンドア磁気センサー→アーム
- ハイマン人体赤外線センサー → 帰宅検知ON
- すべての照明と電化製品を消してください
- サーモスタット → 省エネ温度に設定
- Google Nest Hub → 帰宅確認カードを表示
- 音声応答: 「外出モードがオンになりました。安全にご旅行ください。」

## 「煙探知」放送

**トリガー:** ハイマン煙警報 → 煙を検出

**アクションの実行:**
- Google Nest Speaker → 家全体にブロードキャスト「⚠️煙を検知しました！すぐにご確認ください！」
- すべてのスマート ライト → オン (Google Home スマート ライト セット経由)
- 家族の携帯電話にプッシュ通知
- カメラ → 録画開始

## 就寝モード

**トリガー:** Google Home に「おやすみ」と言う

**アクションの実行:**
- ハイマンドアセンサー → ドアオープンアラームをオンにする
- ハイマン人体赤外線 → 夜間検知をオンにする
- ライト→徐々に消灯
- 音声応答：「おやすみ、セキュリティシステムが作動しています」
