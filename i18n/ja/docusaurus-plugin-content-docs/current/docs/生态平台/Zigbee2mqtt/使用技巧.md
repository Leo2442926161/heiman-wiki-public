# 使い方のヒント

## チャンネルの選択

Hyman Zigbee デバイスはデフォルトでチャネル 15 を使用します。 `configuration.yaml` で手動で指定することをお勧めします。

```ヤムル
高度な:
  チャンネル: 15
「」

WiFi 2.4GHz チャネル 1/6/11 との重複を避けてください。 WiFi アナライザーを使用してスキャンし、干渉が最も少ない Zigbee チャネルを選択します。

## ネットワークキーのバックアップ

**必ず「coordinator_backup.txt」と「secret.yaml」をバックアップしてください: **

「」バッシュ
cp データ/coordinator_backup.txt データ/coordinator_backup.bak
cp データ/secret.yaml データ/secret.yaml.bak
「」

キーを紛失した後は、ペアリングされたすべてのデバイスが再接続する必要があります。

## デバイス固有の最適化

- Heiman デバイスはデフォルトで 60 分ごとにバッテリー残量を報告しますが、「レポート」設定によって間隔を短縮できます。
- MQTT 検出設定を手動で記述せずにエンティティを自動的に生成するための「homeassistant: true」設定を追加しました
- `friend_name` を使用して、Heiman デバイスに人間が読める名前を設定します

## ファームウェアをアップデートする

一部の Heiman Zigbee デバイスは OTA アップデートをサポートしており、「configuration.yaml」で「ota」オプションをオンにする必要があります。
