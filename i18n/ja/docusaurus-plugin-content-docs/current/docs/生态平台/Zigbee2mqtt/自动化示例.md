# 自動化の例

## MQTT サブスクリプションデバイスのステータス

「」バッシュ
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
「」

出力例:
「」
zigbee2mqtt/heiman_smoke {"煙":true,"バッテリー":100,"リンク品質":120}
「」
