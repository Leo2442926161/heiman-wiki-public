# 使い方のヒント

## デバイス構成

- Heiman Matter デバイスを初めてペアリングするときは、境界ルーター (HomePod、Nest Hub、Home Assistant Green など) が同じ Thread ネットワーク上にあることを確認してください。
- ZHA または Zigbee2MQTT 経由で Zigbee デバイスを統合する場合、最初に USB コーディネーターの推奨ファームウェア (Sonoff ZBDongle-E、Conbee II など) を使用することをお勧めします。

## 自動最適化

- 瞬間的な誤警報を防ぐため、煙警報自動化に「for:」パラメータ（例：3秒）を追加しました。
- `condition: state` を使用してセキュリティ モードに一致させ、武装状態での不必要な通知を回避します
- ホームアシスタントの「グループ」機能を使用して、Heiman センサーをエリアごとにグループ化し、バッチ操作を簡単にします

## トラブルシューティング

- デバイスがオフラインの場合、まず Zigbee ネットワーク チャネルが WiFi 2.4GHz と競合していないかどうかを確認します。
- 案件デバイスが検出できない場合は、境界ルータを再起動し、デバイスを再追加します
