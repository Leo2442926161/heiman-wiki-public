# 自動化の例

## 煙警報器が家全体の照明を作動させます

```ヤムル
別名: 「煙警報器 - 家中のライトと通知」
トリガー:
  - トリガー: 状態
    エンティティ ID: binary_sensor.heiman_smoke_alarm
    に:「オン」
アクション:
  - アクション: light.turn_on
    ターゲット:
      エンティティ ID: すべて
  - アクション：notify.mobile_app_phone
    データ:
      メッセージ: 「⚠️ 煙警報器が作動しました! すぐに確認してください!」
      タイトル：「煙警報器」
「」

## CO警報連動

```ヤムル
別名: 「一酸化炭素警報 - ガスを止めて窓を開けてください」
トリガー:
  - トリガー: 状態
    エンティティ ID: binary_sensor.heiman_co_alarm
    に:「オン」
アクション:
  - アクション: switch.turn_off
    エンティティ ID: switch.gas_fireplace
  - アクション: カバー.オープン
    エンティティ ID: cover.living_room_window
  - アクション：notify.persistent_notification
    データ:
      メッセージ: 「一酸化炭素警報が作動しました。ガスのバルブが閉じられ、窓が開きました。」
「」

## 水没検知器連携

```ヤムル
別名: 「洪水検出 - 給水バルブを閉めてください」
トリガー:
  - トリガー: 状態
    エンティティ ID: binary_sensor.heiman_water_leak
    に:「オン」
アクション:
  - アクション: switch.turn_off
    エンティティID: switch.main_water_valve
  - アクション：notify.mobile_app_phone
    データ:
      メッセージ: 「💧 水漏れを検出しました! 元栓が閉まっています!」
「」

  - アクション:alarm_control_panel.trigger
