#ホームアシスタント

Haiman Technology は、2026 年 2 月に **Works with Home Assistant** 認定プログラムに正式に参加し、このプログラムの最初の中国パートナーの 1 つとなりました。

## 認証デバイス

次の Heiman デバイスは、ローカリゼーションと Home Assistant との安定した互換性を保証するために正式に認定されています。

|設備 |地域 |接続方法 |
|------|------|----------|
|スマート煙警報器 |米国 |スレッドよりも重要 |
|スマート煙警報器 | EU / 中国 |スレッドよりも重要 |
|スマート一酸化炭素警報器 |米国 |スレッドよりも重要 |
|スマート一酸化炭素警報器 | EU / 中国 |スレッドよりも重要 |
|人体赤外線センサー M1シリーズ |グローバル |マター / ジグビー |
|浸水検知器 L1シリーズ |グローバル |マター / ジグビー |
|温湿度センサー H1シリーズ |グローバル |マター / ジグビー |

## 統合方法

### 件により

1. Matter over Thread 境界ルーター (Home Assistant Green + Connect ZBT-2、Apple HomePod、Google Nest Hub など) があることを確認します。
2. ホーム アシスタントで Matter 統合を構成する
3. デバイスを追加して自動的に検出します

### Zigbee2MQTT/ZHA経由

Hyman Zigbee デバイスは、Zigbee2MQTT および ZHA によってネイティブにサポートされており、追加の構成は必要ありません。

## ローカリゼーション制御

すべての認定デバイスは、クラウド サービスに依存せずに **ローカル制御** をサポートします。インターネットがダウンしても、センサーアラームやオートメーション連携は正常に動作します。

## コミュニティへの参加

ヘイマンは、ホーム アシスタント コミュニティのディスカッションに積極的に参加し、ユーザーのフィードバックに基づいて製品エクスペリエンスの最適化を続けています。
