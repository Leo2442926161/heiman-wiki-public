# ダッシュボードの設定

## セキュリティ概要パネル

断面図レイアウトを使用して、Heimann 機器をエリアに整理します。

```ヤムル
タイプ: セクション
タイトル: 煙および一酸化炭素警報器
カード:
  - タイプ: エンティティ
    エンティティ: binary_sensor.heiman_smoke_alarm
    名前: 煙警報器
  - タイプ: エンティティ
    エンティティ: binary_sensor.heiman_co_alarm
    名前: COアラーム
「」

## キノコカードスタイル

```ヤムル
タイプ: カスタム:キノコアラームコントロールパネルカード
エンティティ:alarm_control_panel.home_alarm
名前: セキュリティシステム

タイプ: カスタム:マッシュルームエンティティカード
エンティティ: binary_sensor.heiman_smoke_alarm
名前: 煙警報器
アイコン: mdi:煙探知器
プライマリ情報: 状態
セカンダリ情報: 最後に変更された情報
fill_container: true
「」

## 環境センサーパネル

```ヤムル
タイプ: セクション
タイトル: 環境モニタリング
カード:
  - タイプ: ゲージ
    エンティティ: sensor.heiman_temperature
    名前：温度
    単位：「℃」
    分: 0
    最大: 50
    重大度:
      緑: 0
      黄色: 30
      赤：40
  - タイプ: ゲージ
    エンティティ: sensor.heiman_humidity
    名前：湿度
    単位：「%」
    分: 0
    最大: 100
    重大度:
      緑: 30
      黄：60
      赤：80
「」

## アラーム履歴パネル

```ヤムル
タイプ: 履歴グラフ
タイトル: センサーの歴史
エンティティ:
  - binary_sensor.heiman_smoke_alarm
  - binary_sensor.heiman_co_alarm
  - binary_sensor.heiman_water_leak
上映時間: 24
「」

## 推奨されるレイアウト

- 上部: セキュリティ システム ステータス + 警報装置ステータス カード
- 中央: 環境モニタリング (温度、湿度、水質)
- 下: センサーアラーム履歴トレンドチャート
- サイドバーのショートカット操作：アーム/アーム解除、シーン切り替え
