#対応機種一覧

Heiman WiFi 機器は主に直接接続シナリオ向けに設計されており、専用のゲートウェイなしでホーム ネットワークに接続できます。

## スマートソケット

|モデル |製品名 |説明 |
|------|----------|------|
| HS2SK-WiFi |スマートソケット(WiFi) | 2.4GHz WiFiに直接接続し、リモートコントロールとタイミングをサポート |

## セキュリティアラームカテゴリ

|モデル |製品名 |説明 |
|------|----------|------|
| HS8SA-WiFi |スマート煙警報器 (WiFi US バージョン) |米国市場向け WiFi バージョン煙警報器 |

## ゲートウェイクラス

|モデル |製品名 |説明 |
|------|----------|------|
| HS6GW |スマート ホーム ゲートウェイ (WiFi + Zigbee) | WiFi 経由でインターネットに接続し、Zigbee サブデバイスを管理する |
| HS6GW-2 |スマート ホーム ゲートウェイ (WiFi + Zigbee) |第 2 世代ゲートウェイ、パフォーマンスのアップグレード |

## 注意事項

- すべての WiFi デバイスは **2.4 GHz** 帯域のみをサポートし、5 GHz とは互換性がありません
- WPA2-PSK暗号化ネットワークの使用を推奨します
- ネットワークには、インターネットに正しく接続するためのルーターが必要です
