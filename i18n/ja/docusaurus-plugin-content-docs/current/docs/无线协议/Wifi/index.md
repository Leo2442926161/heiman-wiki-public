#Wi-Fi

Wi-Fi は最も一般的なワイヤレス ネットワーク プロトコルで、IEEE 802.11 標準に基づいています。一部の Heiman スマート デバイスは Wi-Fi 接続をサポートしており、追加のゲートウェイを必要とせずにルーターに直接接続できます。

## 該当するシナリオ

Wi-Fi デバイスは次の状況に適しています。

- 完全なネットワーク インフラストラクチャ。追加のゲートウェイを購入する必要はありません。
- 消費電力の影響を受けないデバイス (スマート ソケットなど)
- リモート アクセスとクラウド サービスが必要なシナリオ
- すでに成熟した Wi-Fi メッシュ ネットワークでカバーされている家庭

## ハイマン Wi-Fi 製品

ハイマンは、Wi-Fi 接続をサポートする製品を提供しています。

- **Smart Socket HS2SK-WiFi** — 消費電力測定、リモートスイッチ
- **スマート ゲートウェイ** — Wi-Fi 経由でルーターに接続します

## 注意事項

- Wi-Fi センサーのバッテリーの消耗が Zigbee/Thread よりも早くなります
- 多数の Wi-Fi デバイスがネットワークの安定性に影響を与える可能性があります
- Haiman の主力製品は Zigbee/Matter over Thread であり、Wi-Fi 製品ラインは比較的合理化されています
