#HS6GW - スマート ホーム ゲートウェイ

## 概要

HS6GW は Heiman が発売したスマート ホーム ゲートウェイで、WiFi + イーサネット アップリンク、内蔵 Zigbee コーディネーターをサポートし、最大 128 台の Zigbee サブデバイスを管理できます。インターネットに接続してリモートアクセスとクラウドサービスを実現することが、Haimanスマートホームシステムの中核ハブとなります。

## 技術仕様

|パラメータ |値 |
|------|-----|
|アップリンク通信 | IEEE 802.11 b/g/n (2.4 GHz WiFi) + 10/100M イーサネット |
|ダウンリンク通信 | Zigbee 3.0 (コーディネーターモード) |
|サブデバイスの最大数 | 128 |
|電源供給方式 | DC 5V / 1A (マイクロUSB) |
|待機時消費電力 | ≤ 1W |
|動作温度 | -10℃～+50℃ |
|使用湿度 |相対湿度 90% 以下 |
|インジケーターライト | WiFi ステータス/電源/Zigbee ネットワーク ステータス |
|リセット方法 |ボタンをリセット + 長押しして工場出荷時の設定に戻す |

## 特徴

- デュアルアップリンク: WiFi またはイーサネット有線接続、ネットワーク環境に応じた柔軟な選択
- リモートアクセスをサポート：クラウドアプリを通じていつでもどこでもデバイスのステータスを確認
- ローカルシーン連携：LAN内の機器の自動操作をサポート（切断時も利用可能）
- OTAファームウェアのアップグレード
- Tuya / スマートライフプラットフォームをサポート
- 最大 128 個の Zigbee サブデバイスを管理
-Zigbeeネットワークの自己修復とルーティングの最適化をサポート

## 認証基準

-CE/FCC
- RoHS
