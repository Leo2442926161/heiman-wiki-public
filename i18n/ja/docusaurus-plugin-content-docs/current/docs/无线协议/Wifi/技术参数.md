# 技術パラメータ

以下は、Heimann WiFi デバイスの一般的な技術仕様です。

## 無線パラメータ

|パラメータ |値 |
|------|-----|
|プロトコル標準 | IEEE 802.11 b/g/n |
|動作周波数帯域 | 2.4 GHz (5 GHz には非対応) |
|伝送速度 |最大 150 Mbps (802.11n) |
|ネットワークトポロジ |スター型トポロジー（ルータ直結） |
|通信距離 |屋内で約30～50メートル（ルーターの性能による） |
|暗号化方式 | WPA2-PSK / WPA3 |

## ネットワーク配信方法

|方法 |説明 |
|------|------|
|スマートコンフィグ |モバイル アプリ ワンクリック ネットワーク設定 (従来の WiFi 機器) |
| Bluetooth 支援プロビジョニング | Bluetooth 一時チャネルを介した WiFi 認証情報の構成 (一部の新しいモデル) |
| APモード |デバイスのホットスポットをオンにし、電話が接続された後に設定します。

## 電気と電力の消費量

|パラメータ |値 |
|------|-----|
|電源供給方式 |主電源（ソケットタイプ/ゲートウェイタイプ）またはバッテリー電源（アラームタイプ）|
|消費電力レベル | TCP/IP 接続を維持する必要があるため、Zigbee / Thread よりも高い |
|待機時消費電力 |ソケットタイプ 約0.5W～1W |

## 環境パラメータ

|パラメータ |値 |
|------|-----|
|動作温度 | -10℃～+50℃ |
|動作湿度 | ≤ 90% RH (結露なきこと) |
