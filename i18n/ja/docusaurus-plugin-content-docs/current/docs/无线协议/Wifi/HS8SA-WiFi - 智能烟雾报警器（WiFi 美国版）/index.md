# HS8SA-WiFi - スマート煙警報器 (WiFi US バージョン)

## 概要

HS8SA-WiFi は、Hyman が米国市場向けに発売した WiFi スマート煙警報器です。光電煙感知技術を使用し、WiFi ネットワーキングと独立動作デュアル モードをサポートし、UL 217 認証に合格しています。住宅、アパート、その他の住宅地の防火保護に適しています。

## 技術仕様

|パラメータ |値 |
|------|-----|
|通信プロトコル | IEEE 802.11 b/g/n (2.4 GHz) |
|センサーの種類 |赤外線光電 |
|警報音圧 | ≥85 dB (3 メートル) |
|電源供給方式 |電池式 |
|動作温度 | -10℃～+50℃ |
|使用湿度 |相対湿度 90% 以下 |
|インストール方法 |天井設置 |

## 特徴

- ゆっくりとした燃焼や裸火からの煙を検出する光電式煙検出器
- WiFiネットワークリモートアラームプッシュ
- 独立動作モード（ネットワークが切断されても、ローカルの音と光のアラームが発生する可能性があります）
- 85 dB の高音と光アラーム
- バッテリー残量低下リマインダー
- セルフテスト機能（ワンクリックテスト）
- 不正行為防止アラーム
-サポートリモートミュート

## 認証基準

- UL 217 (煙警報器に関する米国規格)
-FCC
- NFPA 72 準拠
