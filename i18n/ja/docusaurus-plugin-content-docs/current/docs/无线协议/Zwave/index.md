#Z-ウェーブ

Z-Wave は、スマート ホーム向けに設計されたワイヤレス メッシュ ネットワーク プロトコルです。 Z-Wave Alliance によって管理されており、ヨーロッパとアメリカの市場に広く浸透しています。

## 技術仕様

|特長 |説明 |
|------|------|
|動作周波数帯域 | 868.42 MHz (EU) / 908.42 MHz (米国) |
|転送速度 |最大 100 kbps (Z-Wave 800 シリーズ) |
|ネットワークトポロジ |メッシュ ネットワーク、最大 232 ノード |
|互換性 |すべての Z-Wave デバイスとの下位互換性 |
|セキュリティ | Z-Wave S2 暗号化 |

## ハイマンと Z-Wave

Hyman は、主にヨーロッパとアメリカの市場で販売されているさまざまな Z-Wave 互換センサーを提供しています。 Heyman デバイスの Z-Wave バージョンには次のものがあります。

・一酸化炭素センサー HS1CA-Z
・煙感知器 HS1SA-Z
・ドアセンサー HS1DS-Z

## 互換性のあるプラットフォーム

- スマートシングスハブ
- フビタット標高
- ホーミープロ
- ホームアシスタント (Z-Wave JS 経由)
-OpenHAB

> **注意**: Z-Wave は中国市場では人気が低く、Haiman Z-Wave 製品は主にヨーロッパとアメリカの輸出市場をターゲットとしています。
