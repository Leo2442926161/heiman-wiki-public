#対応機種一覧

Hyman Z-Wave 製品は主にヨーロッパとアメリカのスマート ホーム市場をターゲットにしており、Z-Wave Plus 標準と互換性があります。

## セキュリティアラームカテゴリ

|モデル |製品名 |説明 |
|------|----------|------|
| HS1SA-Z |スマート煙警報器（Z-Wave） | Z-Waveバージョン煙警報器 |
| HS1CA-Z |インテリジェント一酸化炭素警報器 (Z-Wave) | Z-Wave版COアラーム |
| HS1CG-Z |可燃性ガス検知器（Z-Wave） | Z-Waveバージョンガス検知器 |

## センサークラス

|モデル |製品名 |説明 |
|------|----------|------|
| HS1DS-Z |ドアセンサー（Z-Wave） | Z-Wave バージョンのドアおよび窓センサー |
| HS1MS-Z |人体赤外線センサー（Z-Wave） |人体センシングのZ-Waveバージョン |
| HS1WL-Z |浸水検知器（Z-Wave） | Z-Wave版漏水検知 |

## 互換性

|プロジェクト |説明 |
|------|------|
|標準 | Z-Wave Plus (500 / 700 シリーズチップ) |
|互換性のあるハブ | Homey、Hubitat、SmartThings、Vera、Fibaro、ホーム アシスタント (Z-Wave JS) |
| S2セキュリティ | S2 セキュリティ プロトコルをサポート (非暗号化 / S2 非認証 / S2 認証) |
|スマートスタート |サポート (700 シリーズチップモデル) |
| OTA アップグレード | FWアップデートCCをサポート |
