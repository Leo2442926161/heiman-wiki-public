#対応機種一覧

Hyman Matter デバイスはすべて、Matter over Thread ソリューションを採用しています。リリース済みの対応機種は以下の通りです。

## セキュリティアラームカテゴリ

|モデル |製品名 |説明 |
|------|----------|------|
| HS1SA-M |インテリジェント煙警報器（件） |米国版 / EU 版、世界初の Matter 認定煙警報器 |
| HS1CA-M |インテリジェント一酸化炭素警報器（件） |米国版 / EU 版、世界初の Matter 認定 CO 警報器 |

## センサークラス

|モデル |製品名 |説明 |
|------|----------|------|
| M1-M |人体赤外線センサー（物質） |パッシブ赤外線検出、Apple HomeKit、Google Home などをサポート |
| L1-M |浸水検知器（物質） |漏水検知、リアルタイムアラームプッシュ |
| H1-M |温湿度センサー（物質） |温度、湿度の検出、環境モニタリング |
| HS1DS-M |ドアセンサー（件） | ドアセンサードアと窓の開閉検知、不正行為防止アラーム |

## 注意事項

- Matter デバイスでは、Apple HomePod mini、Google Nest Hub 第 2 世代など、**スレッド ボーダー ルーター** (ボーダー ルーター) を使用する必要があります。
- 現在のHyman Matter製品ラインは現在も拡張されており、さらに多くのモデルが続々とリリースされる予定です。
