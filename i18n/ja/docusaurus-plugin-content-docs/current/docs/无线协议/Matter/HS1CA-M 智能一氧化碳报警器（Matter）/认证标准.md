# 認証基準

HS1CA-M は次の認証に合格しています。

- **EN 50291** — 一酸化炭素警報器の欧州規格。[EN 欧州規格](../../../../認証規格/EN 欧州規格.md) を参照。
- **CE マーク** — EU 適合宣言。[CE EU 適合性](../../../../certification-standards/CE EU-Conformity.md) を参照してください。
- **Matter 認証** - スマート ホームのクロスプラットフォーム相互接続を保証する CSA Alliance Matter 認証に合格
