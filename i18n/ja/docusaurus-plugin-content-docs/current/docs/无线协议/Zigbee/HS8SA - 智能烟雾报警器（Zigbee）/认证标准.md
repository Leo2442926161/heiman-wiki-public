# 認証基準

HS8SA は次の認証に合格しています。

- **EN 14604** — 煙警報器 (EU タイプ) の欧州規格、[EN 欧州規格](../../../../ApprovalStandards/EN European Standard.md) を参照
- **UL 217** — 米国煙警報器安全規格 (US モデル)、[UL Underwriters Laboratories](../../../../certification-standards/ULUnderwriters Laboratories.md) を参照
- **CE/RED/EMC** — 欧州連合無線機器指令と電磁両立性
- **Zigbee 3.0 / Q-Label** — Zigbee Alliance 認定、[その他の認定](../../../認定規格/その他の認定.md) を参照
