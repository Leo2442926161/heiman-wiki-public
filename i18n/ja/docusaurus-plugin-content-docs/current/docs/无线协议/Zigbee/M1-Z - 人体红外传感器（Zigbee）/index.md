# M1-Z - 人体赤外線センサー (Zigbee)

## 概要

M1-Z は、Heiman が発売したコンパクトな Zigbee 人体赤外線センサーです。 110° または 360° の検出角度 (設置方法に応じて) をサポートし、照度センサーが内蔵されており、スマート照明とセキュリティのシナリオに適しています。

## 技術仕様

|パラメータ |値 |
|------|-----|
|通信プロトコル |ジグビー3.0 |
|センサーの種類 |パッシブ赤外線 (PIR) + 照明 |
|検出角度 | 110° (壁掛け) / 360° (天井) |
|検出距離 | ≤7 m (25°C) |
|照度範囲 | 0～1200ルクス |
|電源供給方式 | DC 3V リチウム電池 |
|バッテリー寿命 |約3年 |
|動作温度 | -10℃～+50℃ |
|インストール方法 |壁掛け/天井 |

## 特徴

- モバイル人間検出
- 照度検出 (0~1200 ルクス)
- 110°/360°切り替え可能
- バッテリー残量低下リマインダー
- 不正行為防止アラーム
- コンパクトなデザイン
