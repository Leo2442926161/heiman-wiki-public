# 技術パラメータ

以下は、Heiman Zigbee デバイスの一般的な技術パラメータです。

## 無線パラメータ

|パラメータ |値 |
|------|-----|
|プロトコル標準 | Zigbee 3.0 (IEEE 802.15.4) |
|使用周波数帯域 | 2.4 GHz (世界共通 ISM 周波数帯域) |
|伝送速度 |最大250kbps |
|ネットワークトポロジ |メッシュ自己修復ネットワーク |
|ノード容量 |単一ネットワーク内の最大ノード数は 65535 |
|通信距離 |屋内約10～30メートル（壁障害物が見える） |
|暗号化方式 | AES-128-CCM |

## 互換性

|プロジェクト |説明 |
|------|------|
| Zigbee コーディネーター | Zigbee 3.0標準コーディネーターと互換性 |
|サポートされているプラ​​ットフォーム | Zigbee2MQTT、ZHA (ホーム アシスタント)、Hubitat、Homey、SmartThings、Tuya Zigbee ゲートウェイ、Haiman HS6GW ゲートウェイ |
| OTA アップグレード |無線ファームウェア アップグレードをサポート (一部のモデル) |

## 電気と電力の消費量

|パラメータ |値 |
|------|-----|
|電源供給方式 |電池駆動 (CR123A / CR2450 / AA) または USB / 電源 (ソケットタイプ) |
|バッテリー寿命 |センサーは約2～3年（報告頻度により異なります） |
|待機時消費電力 |マイクロアンペアレベル、スリープポーリングメカニズムをサポート |

## 環境パラメータ

|パラメータ |値 |
|------|-----|
|動作温度 | -10℃～+55℃ |
|動作湿度 | ≤ 95% RH (結露なきこと) |
| IP保護レベル |特定のモデルに依存します (センサータイプは通常 IP30) |
