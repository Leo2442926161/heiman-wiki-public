# HS1WL - 漏水検知器 (Zigbee)

## 概要

HS1WL は、Hyman が発売した第一世代の Zigbee 水没検知器です。プローブ検出方式を採用しており、漏水や水の滞留を効果的に検出できます。ベーシックな浸水監視装置です。

## 技術仕様

|パラメータ |値 |
|------|-----|
|通信プロトコル |ジグビー3.0 |
|検出方法 |プローブ検出 |
|電源供給方式 | DC 3V バッテリー |
|動作温度 | -10℃～+50℃ |
|使用湿度 |相対湿度 95% 以下 |

## 特徴

- 漏水・水溜まりをリアルタイムに検知
- ブザーアラーム
- バッテリー残量低下リマインダー
- プローブの耐腐食設計
