# 認証基準

HS1SA-E は次の認定に合格しています。

- **EN 14604** — 煙警報器の欧州規格。[EN 欧州規格](../../../../ApprovalStandards/EN European Standard.md) を参照。
- **CE/RED/EMC** — 欧州連合無線機器指令と電磁両立性
- **Zigbee 3.0 / Q-Label** — Zigbee Alliance 認定、[その他の認定](../../../認定規格/その他の認定.md) を参照
- **レッド ドット デザイン賞** — レッド ドット プロダクト デザイン賞
