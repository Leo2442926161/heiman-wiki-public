# JTシリーズ - ガス緊急遮断弁（Zigbee）

## 概要

JT シリーズガス緊急遮断弁は、Hyman Zigbee 可燃性ガス検知器と組み合わせて使用される実行装置です。ガス漏れが検出されると、ガスパイプラインのバルブを遠隔または自動的に閉じて事故を防止します。

## 技術仕様

|パラメータ |値 |
|------|-----|
|通信プロトコル |ジグビー3.0 |
|タイプ |スタンドアロン + Zigbee 連携 |
|テストガス |天然ガス・液化石油ガス |
|認証基準 | GB15322 |
|動作温度 | -10℃～+50℃ |

## 特徴

・HS1CG/HS3CGと連動し、バルブを自動閉止します。
-リモート手動バルブ閉鎖をサポート
- 漏れ警報プッシュ
- 電磁弁ドライバー

## 認証基準

- GB 15322

## 適用可能なアクセサリ

- HS1CG / HS3CG 可燃性ガス検知器
