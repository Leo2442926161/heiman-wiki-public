# HS6GW - スマート ゲートウェイ (Zigbee)

## 概要

HS6GW は、Heiman によって発売された Zigbee スマート ゲートウェイです。 Heiman の Zigbee ネットワークのコーディネーターとして、Wi-Fi またはイーサネット アップリンクを通じてインターネットに接続し、最大 128 個のサブデバイスのアクセスをサポートします。

## 技術仕様

|パラメータ |値 |
|------|-----|
|通信プロトコル | Zigbee 3.0 (コーディネーター) |
|アップリンク接続 | Wi-Fi / イーサネット |
|最大ノード数 | 128 サブデバイス |
|電源供給方式 | DC 5V USB電源 |

## 特徴

- Zigbee ネットワーク コーディネーター
- Wi-Fi / イーサネットアップリンク接続
- 自動デバイス検出とペアリング
- ローカルシーン連携
- リモートアクセス (Heiman Cloud Platform 経由)
- 最大 128 個のサブデバイスをサポート
- OTAファームウェアのアップグレード

## 該当するシナリオ

- Heiman の全社インテリジェンスのコア接続ポイント
- Haiman APPと連携して遠隔監視を実現します
