# C1 シリーズ - 一酸化炭素警報器 (Zigbee)

## 概要

C1 シリーズは、Heiman が発売した基本的な Zigbee 一酸化炭素警報器です。電気化学センサーを使用しており、一酸化炭素漏れ検出シナリオに適しています。 EN 50291 認証に合格しています。

## 技術仕様

|パラメータ |値 |
|------|-----|
|通信プロトコル |ジグビー3.0 |
|センサーの種類 |電気化学 |
|検知範囲 | 0～1000ppm |
|警報音圧 | ≥85 dB (3 メートル) |
|電源供給方式 |電池式 |
|センサーの寿命 |約7年 |
|動作温度 | -10℃～+50℃ |

## 特徴

- 電気化学的 CO 検出
- 濃度分類アラーム
- 85 dB の可聴および視覚アラーム
- バッテリー残量低下リマインダー
- セルフテスト機能

## 認証基準

-EN 50291
