# HS3WL - Water Leak Detector (Zigbee)

## 概要

HS3WL は、Hyman の第 3 世代 Zigbee 浸水検知器です。前世代に基づいて検出感度と無線性能が強化されており、漏水監視シナリオに適しています。

## 技術仕様

|パラメータ |値 |
|------|-----|
|通信プロトコル |ジグビー3.0 |
|検出方法 |プローブ検出 |
|電源供給方式 | DC 3V バッテリー |
|動作温度 | -10℃～+50℃ |
|使用湿度 |相対湿度 95% 以下 |

## 特徴

- 漏水・水溜まりをリアルタイムに検知
- Enhanced detection sensitivity
- ブザーアラーム
- バッテリー残量低下リマインダー
- プローブの耐腐食設計
