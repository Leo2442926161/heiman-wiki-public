# HS8MLS - マイクロモーションセンサー (Zigbee)

## 概要

HS8MLS は、Heiman が発売した長距離 Zigbee 人体赤外線センサー (PIR) です。 It is specially designed for scenes requiring long-distance detection and is suitable for open spaces such as corridors and halls.

## 技術仕様

|パラメータ |値 |
|------|-----|
|通信プロトコル |ジグビー3.0 |
|センサーの種類 |パッシブ赤外線 (PIR) |
|検出距離 |長距離検出 |
|電源供給方式 | DC 3V バッテリー |

## 特徴

- 長距離移動検知
- 低消費電力設計
- シーン連携をサポート
