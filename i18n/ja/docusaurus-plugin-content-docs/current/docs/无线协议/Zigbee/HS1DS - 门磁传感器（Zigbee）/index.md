# HS1DS - ドア磁気センサー (Zigbee)

## 概要

HS1DS は、Heiman が発売した Zigbee ドア磁気センサーです。ドアや窓の開閉状態を検知するために使用されます。これは、インテリジェント セキュリティ システムの基本コンポーネントです。

## 技術仕様

|パラメータ |値 |
|------|-----|
|通信プロトコル |ジグビー3.0 |
|センサーの種類 |リードスイッチ |
|電源供給方式 | CR1632ボタン電池 |
|バッテリー寿命 |約2～3年 |
|取り付けクリアランス | ≤10mm |
|動作温度 | -10℃～+50℃ |
|使用湿度 |相対湿度 95% 以下 |

## 特徴

- ドアと窓の開閉状態をリアルタイムに検出
- タンパーアラーム（タンパー）
- バッテリー残量低下リマインダー
・連動対応（ドアを開ける、照明をつける、外出時の警戒など）
- 取り付けが簡単 (3M 粘着テープまたはネジ固定)

## 適用可能なアクセサリ

- CR1632ボタン電池
- 3M両面テープ/ネジ
