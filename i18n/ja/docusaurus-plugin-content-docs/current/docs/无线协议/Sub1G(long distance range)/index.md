# Sub-1G（長距離通信）

Sub-1G は、1 GHz 以下の周波数帯域で動作する無線通信技術です。超長距離伝送と極めて低い消費電力が特徴です。大規模なカバレッジを必要とする IoT シナリオに非常に適しています。

## 特徴

|特長 |説明 |
|------|------|
|動作周波数帯域 | 433 MHz / 868 MHz (EU) / 915 MHz (米国) |
|伝送距離 |オープン環境では最大 1 ～ 3 キロメートル |
|浸透力 | 2.4 GHz よりも優れており、壁の障害物への強力な浸透 |
|消費電力 |非常に低いバッテリー寿命は 5 ～ 10 年に達する可能性があります。
|伝送速度 |低 (データ量が少ないシナリオに適しています) |

## アプリケーションのシナリオ

- **独立した煙/一酸化炭素警報器**: 建物内のフロア全体で相互接続
- **大規模な建物の防火システム**: ショッピング モール、ホテル、学校
- **スマート シニア リビング コミュニティ**: ユニット間の緊急通報
- **Sub-1G + Wi-Fi ゲートウェイ**: サブデバイスは Sub-1G 経由で通信し、ゲートウェイはクラウドに接続します。

## ハイマン サブ 1G 製品

ハイマンは、スタンドアロン型煙警報器の S1 シリーズなど、Sub-1G 無線防火製品に関して長年の経験を持っています。これらのデバイスは、インターネットを介さずにデバイス間の相互接続を実現できます。

> **注意**: Sub-1G は主に Heiman の火災およびセキュリティ製品ラインで使用されており、スマート ホーム製品ラインは主に Zigbee/Matter です。
