#対応機種一覧

Haiman Sub-1G 長距離シリーズ製品は、主に大規模な独立ネットワーキング シナリオ向けに設計されており、ゲートウェイなしでデバイス間の相互接続と相互運用性を実現できます。

## スタンドアロン煙警報器 (S1 シリーズ)

|モデル |製品名 |説明 |
|------|----------|------|
| S1-SA |スタンドアロン型煙警報器 (Sub-1G) |ベーシックタイプ、無線接続対応 |
| S1-SA-P |スタンドアロン型煙警報器 (Sub-1G) |カメラ機能付きバージョン |
| S1-SA-T |独立型煙警報器 (Sub-1G) |高温連動バージョン |
| S1-SA-RF |独立型煙警報器（RF連動型） | RF ネットワーキングのバージョン |

## 接続されたアクセサリ

|モデル |製品名 |説明 |
|------|----------|------|
| S1-RM |無線中継器（Sub-1G） |通信距離の延長 |
| S1-RC |リモコン (Sub-1G) |テスト/ミュート/リセット用 |

## 特徴

- **独立した作業**: ゲートウェイもインターネットも必要なく、デバイス間の直接ワイヤレス相互接続
- **相互接続されたアラーム**: どのデバイスもアラームをトリガーし、同じグループ内のすべてのデバイスが同時にアラームを鳴らします。
- **超長距離**: 通信距離はオープン環境で1~3kmに達します。
- **超低消費電力**: バッテリー駆動、バッテリー寿命は最大 3 ～ 5 年
