# S1 シリーズ - スタンドアロン煙警報器 (Sub-1G)

## 概要

S1 シリーズは、Hyman が発売したスタンドアロン型の Sub-1G 煙警報器です。 433 MHz / 868 MHz の周波数帯域で動作し、光電煙感知技術を使用し、EN 14604 認証に合格しています。ゲートウェイやインターネットは必要ありません。デバイスは、Sub-1G 無線周波数を通じて直接相互接続されます。いずれかのデバイスがアラームをトリガーすると、同じグループ内のすべてのデバイスが同時にアラームを鳴らします。建物、ショッピングモール、ホテルなどの大規模な火災連鎖シナリオに適しています。

|動作温度 | -10℃～+55℃ |

|インストール方法 |天井設置 |

## 特徴

- ゆっくりとした燃焼や裸火からの煙を検出する光電式煙検出器
- ワイヤレス相互接続アラーム: 同じネットワーク内の任意のデバイスによってトリガーされ、すべてのデバイスが同時にアラームを鳴らして点灯します。
- 独立した動作モード: ゲートウェイなし、インターネット不要、デバイス間の直接通信
- 85 dB の高音と光アラーム
- バッテリー残量低下リマインダー
- セルフテスト機能（ワンクリックテスト）
- 不正行為防止アラーム
-サポートリモートミュート
- 超低消費電力設計、5年以上の電池寿命

## 認証基準

-CE/レッド
- CPR (EU 建設製品規制)
