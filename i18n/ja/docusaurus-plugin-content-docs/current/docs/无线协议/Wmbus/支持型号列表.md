#対応機種一覧

Hyman Wireless M-Bus (wM-Bus) 製品は主に公共料金計量の分野で使用され、遠隔検針とエネルギー消費監視をサポートします。

## ガスメーターカテゴリ

|モデル |製品名 |説明 |
|------|----------|------|
| HS1-GM |ワイヤレスガスメーター（wM-Bus） | EN 13757規格をサポート |
| HS1-GM-T |ワイヤレスガスメーター（wM-Bus） |温度補償付きバージョン |

## 水道メーター測定クラス

|モデル |製品名 |説明 |
|------|----------|------|
| HS1-WM |ワイヤレス水道メーター (wM-Bus) |リードスイッチ/ホールセンサー |
| HS1-WM-R |ワイヤレス水道メーター（wM-Bus） |遠隔送信タイプ |

## 熱計インターフェース

|モデル |製品名 |説明 |
|------|----------|------|
| HS1-HM |熱量計無線モジュール（wM-Bus） |標準熱量計に取り付け |

## データの収集と中継

|モデル |製品名 |説明 |
|------|----------|------|
| HS1-CM |ワイヤレスコレクタ(wM-Bus) |周辺機器データの一元収集 |
| HS1-GW | wM-バスゲートウェイ | wM-Bus データを TCP/IP アップロードに変換 |

## 説明

- Hyman wM-Bus 製品は主に **公共事業** および **産業用計量** シナリオを対象としています
- 特定のモデルと仕様はプロジェクトのニーズに基づいてカスタマイズできます
- ワイヤレス M-Bus は **868 MHz (ヨーロッパの標準帯域)** を使用し、EN 13757 シリーズの規格に準拠しています
