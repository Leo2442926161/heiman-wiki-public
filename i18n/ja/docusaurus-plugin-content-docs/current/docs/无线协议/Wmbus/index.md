# wM-Bus (無線メーターバス)

wM-Bus (Wireless Meter-Bus) は、欧州規格 (EN 13757) で定義された無線検針プロトコルです。水道メーター、ガスメーター、電気メーター、熱メーターなどのエネルギー計測機器の遠隔データ収集に広く使用されています。

## 技術仕様

|特長 |説明 |
|------|------|
|規格 | EN 13757 |
|使用周波数帯域 | 868 MHz (EU 標準) |
|伝送距離 |オープン環境 100 ～ 500 メートル |
|消費電力 |非常に低いバッテリー寿命で最長 10 年 |
|主な用途 |エネルギー測定、集中検針 |

## ハイマンと wM-Bus

ハイマンは、wM-Bus テクノロジーを利用して、スマートガスおよびスマート水分野向けのソリューションを提供しています。 wM-Bus 製品は主に大規模および中規模の産業および商業プロジェクトで使用され、Haiman スマート防火プラットフォームと組み合わせて使用​​されます。

## 該当するシナリオ

- アパート/住宅のガスメーターの集中検針
- 商業ビルのエネルギー消費量の監視
- スマートシティインフラストラクチャ

> **注意**: wM-Bus は垂直産業プロトコルであり、主に Heiman のガスおよび防火プロジェクト ソリューションに使用されており、通常の家庭のスマート ホーム シナリオには適していません。
