# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |検出方法 |電源 |特長 |ゲームの提案 |
|------|------|----------|------|------|----------|
| [HS1WL](../../無線プロトコル/Zigbee/HS1WL - 浸水検知器(Zigbee)) |ジグビー |プローブ |バッテリー |ベーシックモデル｜水漏れ時にプッシュアラーム、浴室のシンク下に最適 |
| [HS2WL](../../無線プロトコル/Zigbee/HS2WL - 浸水検知器(Zigbee)) |ジグビー |プローブ+ケーブル |バッテリー |デュアルチャンネル |プローブ検出ポイント + ケーブルで長いエリアをカバー、給湯器/洗濯機の隣に最適 |
| [HS3WL](../../無線プロトコル/Zigbee/HS3WL - 漏水検知器 (Zigbee)) |ジグビー |プローブ |バッテリー |強化版 |より高感度、より大きなブザー |
| [L1-M](../../無線プロトコル/物質/L1-M 浸水検知器(物質)) |スレッドよりも重要 |プローブ |バッテリー |事項バージョン | WWHA 認定、Apple Home へのネイティブ アクセス |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|家全体の水漏れ監視 | HS2WL+L1-M |複数地点のカバー + ネイティブ HomeKit 通知、水漏れを検出 → 水道の元栓を止める → プッシュ通知 |
|洗濯機専用 | HS2WL |ケーブルプローブは洗濯機の底に敷かれています。水漏れが発生すると、すぐに警報+洗濯機の電源をオフにします。
|給湯器・床暖房 | HS3WL |強化版は感度が高く、床暖房配管の漏れを初期段階で検知可能 |
|重要なユーザー | L1-M |専用ゲートウェイ不要、スレッド低遅延アラーム連携 Apple Home Automation |

## 選択の提案

- **ポイント検出**: [HS1WL](../../無線プロトコル/Zigbee/HS1WL - 漏水検知器 (Zigbee)) または [L1-M](../../無線プロトコル/物質/L1-M 漏水検知器 (物質))
- **広いエリアをカバー**: [HS2WL](../../ワイヤレスプロトコル/Zigbee/HS2WL - 漏水検知器 (Zigbee)) (プローブ + ケーブル)
- **物質要件**: [L1-M](../../無線プロトコル/物質/L1-M 浸水検知器 (物質))
