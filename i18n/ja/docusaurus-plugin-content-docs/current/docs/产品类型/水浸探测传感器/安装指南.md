# インストールガイド

## 設置場所の要件

- **地面に置きます**、プローブが地面に触れていることを確認してください
- **水源の近く**: 浴室の洗面台の下、キッチンのシンクの下、洗濯機の横、給湯器の下
- 水漏れを検出する必要がある最低点

## インストール手順

1. 設置場所の決定（漏水危険箇所の地面）
2. 床面を掃除します
3. プローブを地面に接触させてセンサーを地面に置きます。
4. ケーブル バージョン (HS2WL) を使用する場合は、監視するエリアに沿って延長ケーブルを配線します。
5. バッテリーを挿入すると、センサーが自動的にオンになります。
6. シミュレーションテスト: 濡れた指でプローブに触れると、アラームライトが点灯します。
7. ゲートウェイ経由でペアリングする

## 注意事項

- プローブは湿気の多い環境に長時間さらされると腐食する可能性がありますので、定期的に点検してください。
- 自動シャットダウンを実現するには、ソレノイドバルブと連携することをお勧めします。
・電池寿命は約2～3年
- さまざまなエリアをカバーする複数のセンサー
