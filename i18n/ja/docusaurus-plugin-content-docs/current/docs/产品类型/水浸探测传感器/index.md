# 浸水検知センサー

漏水センサーは水漏れや水の蓄積を検出するために使用され、水道管の破裂や水の浸水による財産の損失を効果的に防ぐことができます。

## 製品シリーズ

|シリーズ |コミュニケーション方法 |特長 |
|------|----------|------|
| HS1WL |ジグビー |基本モデル、プローブ検出 |
| HS2WLシリーズ |ジグビー |プローブ + ケーブル検出、デュアルチャネル |
| HS3WLシリーズ |ジグビー |強化版 |
| L1シリーズ |マター / ジグビー | Matter over Thread バージョン、WWHA 認定 |

## コア機能

- 漏水・滞水をリアルタイムに検知
- ブザーアラーム
- バッテリー残量低下リマインダー
- プローブの耐腐食設計

## 設置場所

- バスルームのシンクの下
- キッチンのシンクの下
- 洗濯機/食器洗い機の隣
- 給湯器の下
- 地下室/ガレージ
- ラジエターの近く

## シーン連携

- **水漏れ検知 + 電磁弁**: 水漏れを検知すると自動的に元弁を閉じます。
- **水漏れアラーム + プッシュ**: 不在時に携帯電話で施設や家族に通知します。
- **洪水 + スマート遮断**: リンクされたガス給湯器またはスマート給水バルブ
