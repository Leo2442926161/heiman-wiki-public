# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |ガスを検知 |認証 |電源 |特長 |ゲームの提案 |
|------|------|----------|------|------|------|----------|
| JT-G1 |独立 |天然ガス・液化ガス | GB15322 |主電源 |独立した音と光のアラーム |ゲートウェイ不要、ブザー内蔵、ベーシックなガス漏れ警報器 |
| [JT-HS1CG](../../無線プロトコル/Zigbee/HS1CG - 可燃性ガス検知器(Zigbee)) |ジグビー |天然ガス/液化ガス | GB15322 |主電源 | Zigbee ネットワーキング |漏れを検出 → Zigbee 通知携帯電話 → リンクされた電磁弁が自動的にシャットダウン |
| [JT-HS3CG](../../無線プロトコル/Zigbee/HS3CG - 可燃性ガス検知器(Zigbee)) |ジグビー |天然ガス/液化ガス | GB15322 |電源 |強化された |より高い感度、リモートバルブ閉鎖 + 音と光のアラームをサポート |
| JT-HS8CG | Zigbee / Wi-Fi |天然ガス・液化ガス | GB15322 |主電源 |ネットワークデュアルモード | WiFi 直接接続または Zigbee ゲートウェイ、デュアル通信保証 |
| [JTシリーズ](../../無線プロトコル/Zigbee/JTシリーズ - ガス緊急遮断弁(Zigbee)) |ジグビー |天然ガス/液化ガス | GB15322 |電源 |一般シリーズ |さまざまなシーンのニーズに合わせて複数のモデルを用意 |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|キッチン安全オートメーション | HS1CG / HS3CG |漏水検知 → 電磁弁自動閉 → 排気ファンON → 携帯電話へプッシュ通知 → 音と光で連動警報 |
|賃貸住宅・アパート | JT-G1 |独立した警報器はネットワークを必要とせず、低コストで、消防検査要件を満たします。
|リモート監視の要件 | HS8CG |デュアル通信保証+クラウドプラットフォーム遠隔視聴状況 |
|業務用厨房 | HS3CG |高感度 + 連動排煙システム + 物件プラットフォームの一元管理 |

## 選択の提案

- **スタンドアロン使用**: JT-G1
- **Zigbee ネットワーキング**: [HS1CG](../../ワイヤレス プロトコル/Zigbee/HS1CG - 可燃性ガス検知器 (Zigbee)) または [HS3CG](../../ワイヤレス プロトコル/Zigbee/HS3CG - 可燃性ガス検知器 (Zigbee))
- **デュアル通信保証**：JT-HS8CG
