# インストールガイド

## 設置場所の要件

- **天然ガス (メタン)**: 空気より軽く、天井から **30 cm** 以内に設置
- **液化石油ガス (プロパン)**: 空気より重く、**地面から 30 cm 以内に設置**
- ガスコンロ、給湯器、ガス管接続部などの漏れの可能性のある場所の近く

## インストール手順

1. 検知するガスの種類を確認（天然ガスまたは液化ガス）
2. 対応する設置高さを選択します
3. 取り付けベースを壁にネジで固定します
4. 電源をオンにします (ほとんどのモデルは主電源から電力を供給します)
5. センサーの電源を入れ、約 3 ～ 5 分間ウォームアップします。
6. セルフテスト: テストボタンを押すと、音と光のアラームが正常になります。
7. ゲートウェイ経由のペアリング（Zigbee 版）
8. 電磁弁をリンクする必要がある場合は、遮断弁制御ラインを接続します。

## 注意事項

- テストガスを使用してセンサーの感度を定期的にチェックします
- センサーの寿命は約3〜5年であり、期限切れになったら交換する必要があります。
- ガスの臭いがするときは、電化製品のスイッチをオンまたはオフにしたり、直火を使用したりしないでください。
- 設置高さはガスの種類によって異なります（上部が天然ガス、下部が液化ガス）
