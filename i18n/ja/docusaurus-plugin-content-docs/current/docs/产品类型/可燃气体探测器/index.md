#可燃性ガス検知器

可燃性ガス検知器は、天然ガス（メタン）や液化石油ガス（プロパン）などの可燃性ガスの漏れを検知するために使用されます。厨房の安全を守る中核となる機器です。

## 製品シリーズ

|シリーズ |通信方法 |ガス検知 |認証基準 |
|------|----------|----------|----------|
| JT-G1 |独立 |天然ガス・液化ガス | GB15322 |
| JT-HS1CGシリーズ |ジグビー |天然ガス/液化ガス | GB15322 |
| JT-HS3CG |ジグビー |天然ガス・液化ガス | GB15322 |
| JT-HS8CG | Zigbee / Wi-Fi |天然ガス・液化ガス | GB15322 |

## コア機能

- 高安定性半導体ガスセンサー
- リーク濃度分類警報
・自動遮断連動（電磁弁付）
- 音と光のアラーム (≥85 dB)
- セルフテスト機能
- センサー寿命期限切れリマインダー

## シーン連携

- **漏れを検出 → 連動遮断弁**: ガスパイプラインのバルブを自動的に閉じます。
- **アラーム + 排気**: 排気ファン/レンジフードを自動的にオンにします。
- **プッシュ通知**: 携帯電話はリアルタイムで漏洩警報情報を受信します。

## 安全に関するヒント

ガスの臭いがした場合は、電化製品のスイッチを入れたり切ったり、裸火を使用したりしないでください。すぐに窓を開けて換気し、屋外の安全な場所に行って警察に通報してください。
