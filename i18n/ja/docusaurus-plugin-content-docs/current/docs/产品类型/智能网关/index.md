# スマートゲートウェイ

スマートゲートウェイは、Haimanスマートホームシステムの中核ハブであり、Zigbeeの各種センサーを接続し、デバイス間の連携制御を実現します。

## 製品シリーズ

|シリーズ |コミュニケーション方法 |特長 |
|------|----------|------|
| HS1GWシリーズ | Zigbee + Wi-Fi / イーサネット |基本ゲートウェイ |
| HS3GWシリーズ | Zigbee + Wi-Fi / イーサネット |強化された |
| HS5GWシリーズ | Zigbee + Wi-Fi / イーサネット |高性能ゲートウェイ |
| HS6GWシリーズ | Zigbee + Wi-Fi / イーサネット + スレッド |最新モデルは、Zigbee と Matter over Thread の両方をサポート |

## コア機能

- Zigbee ネットワーク コーディネーター
- インターネットに接続するための Wi-Fi / イーサネットをサポート
- 自動デバイス検出とペアリング
・ローカルシーン連携（一部機種）
- リモートアクセス (Heiman Cloud Platform 経由)
- 最大 128 個のサブデバイスをサポート (モデルに応じて)

## 該当するシナリオ

- Heiman の全社インテリジェンスのコア接続ポイント
- Haiman APPと連携して遠隔監視を実現します
- 中小規模の商業プロジェクトの地域集中管理

## 他のゲートウェイとの違い

Haiman Gateway は公式エコシステムへの入り口です。サードパーティのプラットフォームを使用したいユーザーには、Zigbee2MQTT/ZHA を備えたユニバーサル Zigbee コーディネーターを使用することをお勧めします。
