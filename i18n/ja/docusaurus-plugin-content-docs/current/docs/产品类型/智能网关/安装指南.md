# インストールガイド

## 設置場所の要件

- **家全体の中心にある**により、すべてのデバイスの Zigbee 信号のカバー範囲が確保されます。
- 全金属製のキャビネット内に置かないでください (金属は無線信号をブロックします)。
- **イーサネットを優先**: より安定した有線ネットワーク接続を使用することをお勧めします。
- 他の Zigbee デバイスから 20 ～ 30 m 以内の距離 (屋内)

## インストール手順

1. 設置場所を選択します（リビングルームのテレビキャビネット、弱電ボックスの隣など）
2. 電源アダプターを接続します
3. Ethernet ケーブルを接続する (推奨) か、Wi-Fi に接続します
4. インジケータライトが緑色/点灯するまで待ちます。
5. Haiman APP を開き、ゲートウェイの下部にある QR コードをスキャンして追加します
6.追加が完了したら、サブデバイスをペアリングできます

## 注意事項

- ゲートウェイはスマートホームの中核です。 UPS バックアップ電源の使用をお勧めします。
- イーサネット接続は Wi-Fi よりも安定しているため、最初に使用することをお勧めします
- デバイスの信号が弱い場合は、Zigbee 中継デバイス (スマート ソケットなど) を追加します。
- モデルごとにサポートされるサブデバイスの最大数は異なるため、規模に基づいて選択する必要があります。
