# モデルの比較

## 各種モデルの比較

|モデル | Zigbee サブデバイスの最大制限 |ネットワーク方式 |特長 |ゲームの提案 |
|------|------|----------|------|----------|
| [HS6GW](../../ワイヤレス プロトコル/Zigbee/HS6GW - インテリジェント ゲートウェイ (Zigbee)) | 128 | WiFi + イーサネット |最新モデルで、将来のスレッドのアップグレードもサポート | Haiman 独自のエコロジーコアと Haiman APP リモート管理を組み合わせた、それを気にしないユーザーに適しています |
| HS5GW | 64 | WiFi + イーサネット |高性能 |中規模および大規模プロジェクト、安定性と信頼性 |
| HS3GW | 32 | WiFi |強化された |中小規模の住宅には十分 |
| HS1GW | 16 | WiFi |基本 |エントリーレベル、少数のデバイスに適しています |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル｜説明 |
|------|----------|------|
|全館ハイマンエコロジー | HS6GW | Heiman Zigbee センサーと Heiman APP によるリモート管理を併用すると、独自のサーバーを構築する必要がなくなります。
|サードパーティプラットフォームのユーザー | — |柔軟性を高めるために、Zigbee コーディネーター (SLZB-06 など) を直接購入して Zigbee2MQTT/ZHA とともに使用することをお勧めします。
|中小規模マンション | HS3GW |基本的なニーズを満たすための 32 台のデバイス制限 |

## 選択の提案

- **Haiman 独自のエコロジー**: [HS6GW](../../ワイヤレス プロトコル/Zigbee/HS6GW - スマート ゲートウェイ (Zigbee)) (128 デバイス、WiFi + イーサネット)
- **サードパーティ プラットフォーム**: 推奨されるユニバーサル Zigbee コーディネーター + Zigbee2MQTT/ZHA
