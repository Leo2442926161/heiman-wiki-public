# インストールガイド

## 設置場所の要件

- **天井設置**、センサーは垂直下向き
・設置高さ**2.5～3.5m**
- 検知半径は約 **3 ～ 5 m**
- **避けてください**: センサーの前に金属製の障害物があります (金属はミリ波を遮断します)
・大きな金属パイプやエアコン内部ユニットなどから遠ざけてください。

## インストール手順

1.天井の設置場所を選択します（部屋の中央が最適です）
2. 天井に穴を開け、拡張ネジでベースを固定します。
3. 配線を接続するか (主電源が必要な場合)、バッテリーを取り付けます。
4. センサー本体を回転させ、ベースにはめ込みます。
5. 電源を入れると、インジケーターライトが点灯し、センサーの自己テストが開始されます。
6. APPペアリングを入力し、デバイスを追加します
7. テスト: 検出エリアに入り、静かに座ります。センサーは「占有」状態のままでなければなりません。

## 注意事項

- ミリ波は薄い板（石膏ボード、木材）を透過しますが、金属には遮断されます。
- 厚い壁や金属製の天井では、検出範囲が狭くなります。
- シーン全体をカバーするには、PIR センサーと併用することをお勧めします。
