# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |検出技術 |インストール |特長 |ゲームの提案 |
|------|------|----------|------|------|----------|
| [HS8OS](../../ワイヤレスプロトコル/Zigbee/HS8OS - 人感センサー (Zigbee)) |ジグビー |ミリ波レーダー |天井 |静止人検知 | PIR よりも正確で、バスルーム/寝室は長時間静止しており、誰もいない PIR で照明が消えるという問題を解決します。

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|バスルームの電気はつけておきます | HS8OS |人がいるときは照明が点灯し、人が離れると消灯します (微呼吸を検出して、PIR 定常照明消灯の問題点を解決します) |
|寝室の睡眠モニタリング | HS8OS |ベッド上の人の在否を検知し、エアコンと連動して温度を自動調節 |
|スマート高齢者ケア | HS8OS |高齢者のトイレタイムアウト → システム警告で家族に事故防止 |
|会議室の省エネ | HS8OS | N 分間に誰も到達しない → 照明と空調を自動的にオフにしてエネルギーを節約し、消費量を削減 |

## 選択の提案

- **PIR の問題点を解決するソリューション**: [HS8OS](../../ワイヤレス プロトコル/Zigbee/HS8OS - 人体存在センサー (Zigbee)) (ミリ波、静止検出)
- **PIR との組み合わせ**: PIR は動きを検出 + HS8OS は静止した存在を検出し、最良の結果を得るために補完的に使用します。
