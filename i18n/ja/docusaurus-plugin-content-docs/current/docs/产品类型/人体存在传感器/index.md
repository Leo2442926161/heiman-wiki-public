# 人感センサー

占有/存在センサーは、**ミリ波レーダー技術**を使用して静止した人体の存在を検出し、従来の PIR 赤外線センサーを補完します。

## 製品シリーズ

|シリーズ |コミュニケーション方法 |特長 |
|------|----------|------|
| HS8OS |ジグビー |ミリ波レーダーによる人感検知 |

## 技術原則

動きのみを検出できる PIR センサーとは異なり、プレゼンス センサーはマイクロ ドップラー効果を利用して、ミリ波信号を送受信することで小さな体の動き (呼吸による胸の上下など) を検出し、人の存在の有無を判断します。

## コア機能

- **静止人体検出**: 横になっている、静止しているなどの微動を検出します。
- **範囲検出**: カバー範囲半径 3 ～ 5 メートル
- **天井設置**: 最適な検出角度
- **PIR との補完**: PIR は大規模な動きの検出を担当し、存在センサーは詳細な検出を担当します。

## シナリオの例

- **スマート照明**: 誰かが近くにいるときはライトが常に点灯し、誰もいないときは自動的にオフになります (PIR 静的ライトオフの問題を解決します)
- **空調制御**: 部屋に人の有無に応じて自動的にオンまたはオフになります。
- **スマート高齢者ケア**: 高齢者が浴室/寝室で長時間静止している場合、システムは早期警告を発します。
- **オフィスの省エネ**: 会議室/ワークステーションに人がいないときにエネルギーを自動的に調整します。
