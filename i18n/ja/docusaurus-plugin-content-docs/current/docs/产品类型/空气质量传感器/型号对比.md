# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |検出パラメータ |電源 |特長 |ゲームプレイに関する提案 |
|------|------|----------|------|------|----------|
|空気質センサー |ジグビー | PM2.5 / PM10 / CO₂ / TVOC / 温度と湿度 |主電源 | 1台で多機能 |室内の空気の質を監視し、新鮮な空気/清浄機を自動的にリンクし、基準を超えた場合はリマインダーをプッシュします |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|寝室の CO₂ モニタリング |空気質センサー | CO₂ > 1000ppm → 窓開けリマインダー/換気のためのリンクされた新鮮な空気システム、睡眠の質の向上 |
|新しく改装された家 |空気質センサー | TVOC・ホルムアルデヒドを継続監視、基準値超過時は換気扇連動、基準値達成後はプッシュ通知で安心 |
|子供部屋 |空気質センサー | PM2.5を監視し、基準を超えると空気清浄機を自動的にオンにして呼吸器の健康を確保 |
|オフィス |空気質センサー | CO₂＋温湿度連動空調新鮮空気で作業効率UP |
