# 空気質センサー

空気質センサー（空気質モニター）は、室内空気中のさまざまな汚染物質を検出し、健康的な生活環境の維持に役立ちます。

## 検出パラメータ

Hyman の大気質センサーは次の指標を検出します。

|パラメータ |説明 |健康基準 |
|------|------|----------|
| PM2.5 |微小粒子状物質 | ≤ 35 µg/m3 (24 時間平均) |
| PM10 |吸入性粒子状物質 | ≤ 50 µg/m3 (24 時間平均) |
| CO₂ |二酸化炭素 | ≤ 1000 ppm |
| TVOC |総揮発性有機化合物 | ≤ 0.6 mg/m3 |
|温度 |周囲温度 | 18～26℃ |
|湿度 |相対湿度 | 40-60%相対湿度 |

## コア機能

- マルチパラメータ空気品質検出
- カラー表示（モデルによって異なります）
- 基準値超過を音と光で警報
- 履歴データ記録
- リンクされた新鮮な空気/空気清浄機

## 該当するシナリオ

- 寝室: CO₂ レベルを監視し、換気のために窓を開けるよう促します。
- 子供部屋: 子供の呼吸器の健康を守ります。
- オフィス: 生産性と快適性の向上
- 新しく改装された住宅: ホルムアルデヒド/TVOC 放出を監視します。
