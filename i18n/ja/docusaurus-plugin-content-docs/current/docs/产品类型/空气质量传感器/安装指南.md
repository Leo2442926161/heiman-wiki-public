# インストールガイド

## 設置場所の要件

- **1.2 ～ 1.8 m** の高さに設置 (呼吸ゾーン)
- 室内の平均的な空気の質を表す場所を選択してください
- **遠ざけてください**: 窓、ドア、通気口、エアコンの吹き出し口
- **近寄らない**: キッチンの煙、加湿器、アロマセラピー、その他の地域の汚染源

## インストール手順

1. 設置場所を選択します (リビングルーム、ベッドルーム、またはオフィスの壁)
2. 取り付けベースをネジで固定します
3. 電源に接続します (USB またはアダプター電源)
4. センサーの電源がオンになり、セルフテスト後にディスプレイ (存在する場合) にデータが表示されます。
5. ゲートウェイ経由でペアリングする
6. APPでデータが正常であることを確認します

## 注意事項

- センサーの空気入口と出口が塞がれていないことを確認してください。
- ほこりの詰まりを防ぐために、センサーハウジングを定期的に掃除してください。
- 子供部屋に設置すると、CO₂ レベルを監視し、換気のために窓を開けるように促します。
- フレッシュエアシステムや空気清浄機との併用をお勧めします。
