# 火災温度センサー

火災温度センサー（熱感知器）は、周囲温度の異常な上昇を検知して火災警報を発します。キッチンやガレージなど煙センサーが誤報しやすいシーンに最適です。

## 動作原理

温度センサーは、熱素子を使用して温度変化を検出します。

- **固定温度アラーム**: 温度が設定されたしきい値 (通常は 57°C / 135°F) を超えるとアラームがトリガーされます。
- **温度差アラーム**: 温度上昇率が設定値 (8.3°C/min など) を超えたときにアラームをトリガーします。

## 適用範囲

- キッチン (このシーンでは調理が行われるため、煙センサーが誤警報を起こしやすい)
- ガレージ/作業場 (排気ガスや粉塵が煙センサーに干渉する可能性があります)
・ボイラー室・暖房機器室
- バスルーム/サウナ

## コア機能

・異常温度上昇の検知
- 不正行為防止アラーム
- バッテリー残量低下リマインダー

> **注意**: 温度センサーと煙センサーは補完的に使用されます。煙センサーはより速く反応し、温度センサーはより強力な抗干渉能力を備えています。
