# インストールガイド

## 設置場所の要件

- **天井マウント**、部屋の中央に設置
- **最適な使用シナリオ**: キッチン、ガレージ、ボイラー室、バスルーム - 煙センサーが誤報を起こしやすい場所
- 壁から少なくとも **30 cm**
- 通気口やエアコンの吹き出し口の近くへの設置は避けてください。

## インストール手順

1. 設置場所を選択します（推奨のキッチンまたはガレージの天井）
2. 取付ベースを天井に固定します
3. バッテリーを挿入すると、センサーが自動的にオンになります。
4. 本体を回転させてベースにはめ込みます
5. セルフテスト: テストボタンを押すと、インジケーターライトが点灯し、音が鳴ります。
6. ゲートウェイ経由のペアリング（Zigbee 版）

## 注意事項

- 温度センサー**は煙センサー**の代替品ではなく、補完するものです。
- 煙センサーはより速く反応し、温度センサーはより強力な耐干渉性を備えています。
- 煙センサーと温度センサーの両方を同じ部屋に設置することをお勧めします。
- 月に一度テストし、バッテリーが少なくなったら適時にバッテリーを交換してください。
