# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |検出原理 |電源 |特長 |ゲームプレイの提案 |
|------|------|----------|------|------|----------|
|火災温度センサー | Zigbee / サブ 1G |サーミスタ（定温＋差温） |バッテリー |煙干渉防止 |キッチン・ガレージ専用、異常温度上昇警報、ガス遮断連動電磁弁 |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|キッチンの安全 |温度センサー |異常温度上昇 → ガス栓を閉める → 音と光で警報 → 携帯電話プッシュ通知で調理煙誤報を回避 |
|ガレージ/ボイラー室 |温度センサー |機器室の温度を監視して自然発火を防止し、排気ファンを連動させて冷却 |
|煙センサーと補完 |温度センサー+スモーク |煙センサーは家全体をカバーし、温度センサーはキッチン/ガレージをカバーし、補助警報器はより信頼性が高くなります。

## 選択の提案

- **キッチンのみ**: 温度センサー (料理の煙による誤報を避けるため)
- **煙センサーと連動**: 煙センサーは寝室/リビングルームをカバーし、温度センサーはキッチン/ガレージをカバーします。
