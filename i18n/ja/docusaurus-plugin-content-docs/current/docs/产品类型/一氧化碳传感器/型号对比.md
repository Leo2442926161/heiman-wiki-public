## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|家全体のスマートセキュリティ | HS1CA-M | Matter over Thread は Apple Home + Google + Alexa にネイティブに接続し、アラーム→ガスを止める→窓を開ける |
| EU 規格要件 | HS1CA-E | Z2M/ZHAを介してHAに接続すると、リンケージガス電磁弁が自動的に遮断されます。
|米国規格要件 | HS-720ES-M | UL 2034 認定、保険要件を満たしています |
|古民家を低コストで改修 | C1シリーズ |ゲートウェイで使用される基本機能 |

- **米国市場**: HS-720ES-M (UL 2034)
