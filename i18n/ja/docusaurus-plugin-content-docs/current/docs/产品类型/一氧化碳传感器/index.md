#一酸化炭素センサー

一酸化炭素 (CO) は、「目に見えない殺人者」として知られる、無色、無臭、非刺激性の有毒ガスです。 Hyman 一酸化炭素センサーは **電気化学センサー技術**を使用して、空気中の CO 濃度を正確に検出します。

## 認定された実績

ハイマンは **世界初の Matter 認定一酸化炭素警報器**を発売し、Works with Home Assistant の認定を受けています。

## コア機能

- 高精度電気化学的 CO センサー
- 検出範囲: 0~1000 ppm
- 濃度段階アラーム（音と光のプロンプト）
- 85dBのアラーム音
- バッテリー残量低下リマインダー
- センサーの寿命期限切れリマインダー (通常 7 年)
- セルフテスト機能

## 安全に関するヒント

CO濃度と人体の反応：

| CO濃度(ppm) |人体の反応 |
|---------------|----------|
| 50 |健康な成人は 8 時間まで耐えられます。
| 200 | 2 ～ 3 時間後に軽い頭痛 |
| 400 | 1 ～ 2 時間後に頭痛と吐き気 |
| 800 | 45 分以内のめまいとけいれん |
| 1600+ | 2時間以内に死亡 |
