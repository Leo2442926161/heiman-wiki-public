# インストールガイド

## 設置場所の要件

- **呼吸する高さに設置** - 約。 **地上から1.5m**
- 就寝中にアラームが聞こえるように寝室エリアの近くに設置
- **コーナーには設置しないでください**、空気の循環を保ちます
- 通気口、窓、ファン、その他空気の流れに影響を与える場所から遠ざけてください。

## インストール手順

1. 設置場所を決定します（寝室の廊下またはリビングルームを推奨）
2. 取り付けベースを壁に固定します (ネジまたは 3M 粘着テープ)。
3. バッテリーを挿入すると、センサーが自動的にオンになります。
4. 本体をベースに取り付けます
5. セルフテスト: テストボタンを押すと、アラームライトが点滅し、アラームが鳴ります。
6. ゲートウェイ経由でペアリングする

## 注意事項

- 一般的なセンサーの寿命は **7 年**で、その後はデバイス全体を交換する必要があります。
- 月に 1 回機能をテストします
- センサーの通気口をふさがないでください。
・警報が発報した場合は、まず窓を開けて換気し、CO発生源の漏洩がないか確認してください。
