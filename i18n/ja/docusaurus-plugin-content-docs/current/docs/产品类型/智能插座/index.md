# スマートソケット

Haiman スマート メータリング プラグは、リモート スイッチ制御とリアルタイムの消費電力測定をサポートし、ユーザーが家電製品のインテリジェントな管理を実現できるように支援します。

## 製品シリーズ

|シリーズ |コミュニケーション方法 |特長 |
|------|----------|------|
| HS2SKシリーズ |ジグビー |メータリングタイプ、電力監視をサポート |

## コア機能

- リモートスイッチ制御
- リアルタイムの電力/ガス計測
- 過負荷保護
- タイマースイッチ
- 電力統計

## 技術パラメータ

- 定格電圧: AC 220V (中国規格) / AC 110V (アメリカ規格)
- 最大負荷: 16A/3680W
-測定精度: ±2%

## シナリオの例

- 忘れた家電製品の電源をリモートでオフにする
・エアコン・給湯器のエネルギー消費量統計
- 連動した外出モード: ワンクリックで不要な家電製品をすべてオフにします
・水槽照明、加湿器等のタイミング制御
