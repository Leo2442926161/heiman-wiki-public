# インストールガイド

## 設置場所の要件

- 壁コンセントに直接差し込みます。**直列に接続しないでください** (ストリップや延長コードに差し込まないでください)。
- ソケットの定格電流が 16A 以上であることを確認してください。
- 湿気の多い環境や高温の環境には設置しないでください。

## インストール手順

1. スマートソケットを 220V (または 110V) の壁コンセントに直接差し込みます。
2. 電源投入後、表示灯が点灯します。
3. スイッチボタンを長押ししてペアリングモードに入ります。
4. Heiman APP にデバイスを追加します
5. ペアリングが成功すると、スイッチをリモート制御できます。

## 安全上の注意事項

- **最大負荷**: 16A/3680W (ヨーロッパ標準)、15A/1800W (米国標準)
- **過負荷に注意してください**: 高出力の電気製品 (エアコン、ヒーター) を使用する場合は、電力制限に注意してください。
- **直列に接続しないでください**: スマート ソケットを電源タップに挿入しないでください。
- ソケットハウジングを覆ったり隠したりしないでください。
- プラグが過熱していないか定期的にチェックしてください。

## 一般的なアプリケーション

- 給湯器、加湿器、水槽照明などの低電力機器のタイミング制御
・センサー連動で人が来ると点灯、離れると消灯します。
- 電力統計: 家電製品のエネルギー消費量を監視します
