# 落下検出器

転倒検知器は高齢者向けに特別に設計されています。落下事象を自動的に検出し、適時に救難信号を送信できます。これは、スマートな高齢者介護ソリューションの重要な機器です。

## 製品シリーズ

|シリーズ |コミュニケーション方法 |特長 |
|------|----------|------|
| HS2FD |ジグビー |インテリジェントな転倒検出、自動アラーム |

## コアテクノロジー

**マルチセンサー フュージョン アルゴリズム**を採用し、加速度計とジャイロスコープのデータを組み合わせ、AI アルゴリズムを使用して転倒姿勢を判定し、転倒と通常の活動 (座る、かがむなど) を効果的に区別し、誤警報率を削減します。

## コア機能

- 自動落下検知
- ワンクリックで手動で助けを求めます
- ウェアラブルなデザイン (ベルト クリップまたはストラップ)
- バッテリー残量低下リマインダー

## 該当するシナリオ

- 在宅高齢者介護：一人暮らし高齢者を24時間体制で介護します。
- 老人ホーム: 集中管理と管理
- リハビリテーションセンター: 術後患者の転倒リスクをモニタリング

## 連携プラン

- 転倒を検知 → 家族の携帯電話に自動通知
- 緊急ボタン連動 → 二重救助保護
- Haimanスマート高齢者介護プラットフォーム連携 → 施設の一元管理
