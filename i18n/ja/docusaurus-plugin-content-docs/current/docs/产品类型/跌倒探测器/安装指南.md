# インストールガイド

## 装着方法

- **ベルトクリップ**: ベルトまたはウエストバンドにクリップします。
- **ストラップ**: 首に掛けたり、胸に掛けたりします。

## 使用説明書

1. バッテリーを取り付けます
2. 着用方法を選択します (ベルトクリップまたはストラップ)
3. 体に装着する
4. ゲートウェイ経由でペアリングする
5. シミュレーションテスト:
   - 立ってすぐにしゃがむか、地面に横になります。
   - センサーが落下を検知し、数秒以内にアラームをトリガーします。
   - 救難ボタンを押して手動アラームをテストします

## 注意事項

- 一日中着用することをお勧めします（シャワーを浴びるときにバッテリーを充電/交換するために取り外し可能）
- AI アルゴリズムを使用して、誤警報率が低く、日常動作 (座る、かがむなど) による転倒を区別します。
- 転倒を検知すると自動警報、または手動でボタンを押して助けを求めることもできます
・緊急ボタンと連動して二重保護が可能
