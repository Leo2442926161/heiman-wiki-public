# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |センサー |電源 |特長 |ゲームの提案 |
|------|------|----------|------|------|----------|
| [HS2FD](../../ワイヤレスプロトコル/Zigbee/HS2FD - 落下検知器 (Zigbee)) |ジグビー |加速度計 + ジャイロスコープ |バッテリー | AIアルゴリズム転倒検知 |ウェアラブルデザイン、自動転倒検知 + ワンクリックヘルプ、一人暮らしの高齢者や介護施設に最適 |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|ひとり暮らし高齢者の見守り | HS2FD |転倒検知 → お子様の携帯電話に自動プッシュ → タイムアウト後無応答で自動120番通報 |
|老人介護施設 | HS2FD |集中管理プラットフォーム + フロア音と光アラーム + 看護ハンドヘルドデバイス同期アラーム |
|術後リハビリテーション | HS2FD |患者がベッドから起き上がると転倒の危険性を検知 → ナースステーションで早期警戒 |
|バスルーム/夜 | HS2FD | IP保護設計、お風呂で着用可能 + 夜間起きて転倒すると自動アラーム |

## 選択の提案

- **ホームケア**: [HS2FD](../../ワイヤレスプロトコル/Zigbee/HS2FD - 転倒検知器 (Zigbee)) (ウェアラブル、AI 検知)
- **緊急ボタンとペアリング**: HS2FD 自動落下検出 + 緊急ボタンによる手動支援、二重保護
