# スマートドアホン

Haiman スマート ドアベルは、ドアベル通話とセキュリティ監視機能を統合し、リモート ビデオ インターコムをサポートします。

## コア機能

- ドアベルボタントリガー
- リモートライブビデオ（Wi-Fi版）
- 双方向音声通話
- 動体検知ビデオ
- 暗視機能

## 該当するシナリオ

- 訪問者の電話: 自宅でも外出先でも応答できます
- 速達/持ち帰り通知: ドアの開閉を遠隔確認
- セキュリティ監視: ドアの前での不審な行動を記録します。

> **注意**: Heiman ドアベル製品ラインは主に Zigbee/Wi-Fi ソリューションに基づいています。特定のモデルパラメータについては公式を参照してください。
