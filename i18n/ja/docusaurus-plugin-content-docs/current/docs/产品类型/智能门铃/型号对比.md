# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |ビデオ |電源 |特長 |ゲームプレイに関する提案 |
|------|------|------|------|------|----------|
|スマートドアベル | Zigbee + Wi-Fi | 1080P オプション |バッテリー/電源 |ドアベル + セキュリティ ツーインワン |訪問者呼び出し携帯電話通知、リモートビジュアルインターホン、動体検知録画 |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|速達・持ち帰り管理 |スマートドアベル |ベルを鳴らす → 携帯電話通知 → 遠隔インターホン確認 → 遠隔解錠（スマートロック連動） |
|セキュリティ監視 |スマートドアベル |ドア滞在検知 → ビデオ + プッシュアラーム、連動ドアセンサー/照明 |
|来場者連携 |スマートドアベル |ベルを鳴らす → 室内画面・スピーカー表示 → 照明点灯 → 連携解除 |
