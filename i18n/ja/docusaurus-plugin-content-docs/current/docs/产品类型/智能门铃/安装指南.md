# インストールガイド

## 設置場所の要件

- ドアホンは約 **1.4 ～ 1.6 m** (目の高さ) の高さに設置してください。
- 訪問者の顔を確実に捉えることができるように、出入り口に面します。
- カメラに直射日光が当たらないようにしてください。

## インストール手順

1. 設置場所を選択します (ドアの横またはドア枠上)
2. 取り付けベースをネジで固定します
3. 電源を接続します（有線バージョンの場合は接続、バッテリーバージョンの場合はバッテリーを取り付けます）
4. ドアホン本体をベースに取り付けます
5. APP経由でペアリングしてWi-Fiに接続します（Wi-Fiバージョン）
6. カメラ角度を調整する
7. ドアホンのボタン、ビデオ、音声機能をテストする

## 注意事項

- Wi-Fi バージョンでは、ドアで Wi-Fi 信号が十分にカバーされている必要があります
- バッテリーバージョンの充電サイクルに注意し、ドアセンサー（ドアが開いた後に検出して起動する）と併用することをお勧めします。
- カメラのレンズを定期的に掃除してください
- ドアホンが有効な Zigbee 信号範囲内にあることを確認してください
