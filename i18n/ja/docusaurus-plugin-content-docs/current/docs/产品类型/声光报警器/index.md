# 音と光の警報（警告信号）

スマートサイレンを使用して、大音量と光のアラームを提供します。各種センサーと併用し、セキュリティシステムの出力アクチュエーターとして使用できます。

## コア機能

- 高デシベルのアラーム音 (≥110 dB)
- 赤/カラーストロボライト
-複数のアラーム音をサポート
- 改ざん検出
- リモートの武装解除をサポート

## 該当するシナリオ

- 煙/COセンサーと連動：危険を感知すると家中に音が響きます。
- セキュリティ侵入警報: ドアや窓が不法に開けられたときに作動します。
- 緊急避難: 火災が発生した場合に人々を避難させるよう誘導します。

## シーン連携

- **煙警報 + 音と光のサイレン**: 家全体が同時に警報を鳴らし、部屋にいるすべての人に警告します。
- **ドアセンサーアラーム + 音と光のアラーム**: 不法侵入者を阻止します。
- **ゲートウェイと連携**: Zigbee/Matter ネットワークを介して他のデバイスとリンク
