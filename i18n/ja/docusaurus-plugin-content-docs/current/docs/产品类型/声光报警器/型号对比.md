# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |警報音圧 |電源 |特長 |ゲームの提案 |
|------|------|----------|------|------|----------|
|音と光の警報器（サイレン） |ジグビー | ≧110dB |主電源 |高輝度ストロボ+高デシベル |煙/一酸化炭素/ドアマグネットトリガーリンケージ家全体の音と光アラーム、家を出るときの侵入者を阻止 |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|家全体の火災連動 |音と光のアラーム |煙/一酸化炭素センサー警報器 → 家中に音と光の警報器が鳴り、眠っている家族を起こします |
|セキュリティ侵入抑止 |音と光のアラーム |不法ドアセンサー+外出モード→サイレン音+全灯で侵入者を抑止 |
|緊急避難誘導 |音と光のアラーム |火災警報器→サイレンで避難方向を誘導し、ドアセンサーと連携して避難の有無を検知 |
