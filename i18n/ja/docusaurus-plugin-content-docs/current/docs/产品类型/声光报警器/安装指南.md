# インストールガイド

## 設置場所の要件

- **家の中心に設置**して、すべての部屋で聞こえるようにします。
- 廊下、リビングルームの天井または高い壁
- **地面から2m以上**（人が触れないように注意してください）
- 煙/一酸化炭素センサーと組み合わせて使用する場合は、寝室の廊下に設置してください。

## インストール手順

1. 設置場所を選択します（廊下またはリビングルームを推奨）
2. ネジを使用して取り付けベースを壁または天井に固定します
3. 電源を接続するか（主電源バージョン）、バッテリーを挿入します（バッテリーバージョン）。
4. 警報器本体をベースに取り付けます。
5. ゲートウェイ経由でペアリングする
6. リンケージテスト: センサーをトリガーすると、音と光のアラームが反応するはずです。

## 注意事項

・お子様の手の届かない高さに設置してください。
- アラームの音量と照明が正常かどうかを定期的にテストしてください
・電池式の場合、電池残量が少なくなったら交換してください。
- 他のセンサーとの連携ルールを設定可能
