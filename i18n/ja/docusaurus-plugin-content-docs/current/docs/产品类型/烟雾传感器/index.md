#煙センサー

ハイマン煙センサーは **光電煙感知技術**を使用しており、火災の初期段階で発生する目に見える煙の粒子を敏感に検出し、非常に早い段階で火災警報を発することができます。

## 製品シリーズ

|シリーズ |コミュニケーション方法 |認証基準 |該当する市場 |
|------|----------|----------|----------|
| HS1SAシリーズ |ジグビー / Z-Wave | EN 14604 |欧州連合 |
| HS3SAシリーズ |ジグビー | EN 14604 |欧州連合 |
| HS8SA | Zigbee / Wi-Fi | UL217 |米国 |

| HS1SA-M |スレッドよりも重要 | EN 14604 |グローバル |

## コア機能

- ゆっくりとした燃焼や裸火からの煙を検出する光電式煙検出器
- 85 dB の高音と光アラーム
- バッテリー残量低下リマインダー
- セルフテスト機能（ワンクリックテスト）
- 不正行為防止アラーム
-サポートリモートミュート

## 技術パラメータ (HS1SA-M を例にします)

- センサータイプ: 赤外線光電
- アラーム音圧: ≥85 dB (3 メートル)
- 電源：CR123A電池
- バッテリー寿命: 約。 3年
- 動作温度: -10°C ~ +50°C
- 動作湿度: ≤95% RH
