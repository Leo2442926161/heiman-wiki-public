# インストールガイド

## 設置場所の要件

- **天井取り付け**、部屋の中央に設置
- 壁または照明器具から少なくとも **30 cm**
- **デッドエアエリアを避ける** (コーナー、梁の後ろなど)
- **設置は避けてください**: バスルーム (蒸気)、キッチン (すす)、通気口の近く

## インストール手順

1. 設置場所を選択し、天井が平らであることを確認してください
2. 取り付けベースを天井に回転させます (ネジまたは 3M 接着剤を使用)
3. 電池を挿入すると (極性に注意してください)、センサーが自動的にオンになります。
4. センサー本体を回転させ、ベースにはめ込みます。
5. テストボタンを押し続けます。アラームランプが点滅し、アラーム音が鳴り、正常であることを示します。
6. ゲートウェイ経由のペアリング: ペアリングモードに入り、APP にデバイスを追加します。

## 注意事項

- 月に 1 回機能をテストします
- 電池の寿命は約3年です。電池が消耗したら早めに交換してください。
- 10年ごとにセンサー全体を交換することをお勧めします
- 設置高さは煙検知に影響しません。煙は天井まで上昇します。
