# インストールガイド

## 設置場所の要件

- **1.2 ～ 1.5 m** の高さに設置 (環境の平均レベルを表す)
- **遠ざけてください**: 直射日光、ドアや窓、エアコンの吹き出し口、ラジエーター
- **遠ざけてください**: 加湿器、キッチンストーブ、その他の温度や湿度の変化を引き起こす局所的な原因
- よく使用する部屋（寝室、リビングルーム、ベビールーム）を選択してください

## インストール手順

1. 設置場所を選択します (壁またはデスクトップ)
2. 壁取り付け: ベースをネジまたは 3M 接着剤で固定します。
3. CR2 電池を取り付けます (極性に注意してください)。
4. センサーが自動的にオンになり、画面 (ある場合) に温度と湿度が表示されます。
5. ゲートウェイ経由でペアリングする
6. APP 内のデータをチェックして、正常であることを確認します。

## 注意事項

- 壁に設置する場合は、センサーの通気口がふさがれていないことを確認してください。
・電池寿命は約2～3年
- 複数のセンサーを介して、さまざまな部屋の温度と湿度のデータを収集できます
- エアコン、除湿器、加湿器などの機器と連動して使用可能
