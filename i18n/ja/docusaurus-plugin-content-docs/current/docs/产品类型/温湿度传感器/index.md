# 温湿度センサー

温度センサーと湿度センサーは、周囲の温度と相対湿度をリアルタイムで監視するために使用され、スマートホーム環境監視の基本機器です。

## 製品シリーズ

|シリーズ |コミュニケーション方法 |特長 |
|------|----------|------|
| H1シリーズ |マター / ジグビー | Matter over Thread バージョンは WWHA 認定 |

## コア機能

- リアルタイムの温度と湿度の監視
- バッテリー残量低下リマインダー
・連動制御対応（気温が高い場合はエアコンON、湿度が高い場合は除湿機ONなど）

## 技術パラメータ

- 温度精度: ±0.3℃
- 湿度精度: ±3% RH
- 電源: CR2電池
- 電池寿命：約2～3年

## シナリオの例

- 冷暖房連動で室温を自動調整
・除湿・加湿を連動させて快適な湿度を保ちます。
- ワインセラー/シガーキャビネットの環境を監視
- キッチン/浴室の換気扇の自動制御
- 植栽温室の環境モニタリング
