# インストールガイド

## 設置場所の要件

- 壁取り付け高さ: **1.8 ～ 2.4 m**
・天井設置：部屋の中央の天井に設置します。
- **避けてください**: 直射日光、暖房/エアコンの吹き出し口、熱源の近く
- 検知範囲内に障害物（家具、ガラスの仕切り）がある場合は避けてください。

## インストール手順

1. 設置場所を選択します（壁掛けまたは天井取り付け）
2. 取り付けベースをネジで固定します
3. バッテリーを挿入すると、インジケーターライトが点滅します。
4. センサー本体をベースに取り付けます
5. 検出角度を調整します（水平110°の範囲内）
6. 歩行テスト: 検出エリアを歩くと、インジケーターライトが点灯します。
7. ゲートウェイ経由でペアリングする

## 注意事項

・赤外線センサーは温度変化に敏感であり、夏場の高温環境では検知距離が短くなる場合があります。
- 窓に面した場所を避ける（屋外温度の変化により誤警報が発生する）
- 電池寿命 約3年(Zigbee) / 1年(Matter)
