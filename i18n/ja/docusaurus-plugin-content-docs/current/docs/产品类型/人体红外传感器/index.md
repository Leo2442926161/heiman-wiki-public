# 人体赤外線センサー

人体赤外線センサー (PIR) は、人体から発せられる赤外線熱放射を検出することで動きを感知し、スマート ホームで最も一般的に使用されている存在検出ソリューションです。

## 製品シリーズ

|シリーズ |コミュニケーション方法 |特長 |
|------|----------|------|
| HS1MS |ジグビー |ベーシックモデル 検出角110° |
| HS3MS |ジグビー |アップグレードモデル、360°天井設置 |
| M1シリーズ |マター / ジグビー |マターオーバースレッドをサポート、オプションで 110° または 360° |
| HS8MLS |ジグビー |長距離モデル |
| HS8MIS |ジグビー |人体＋照明複合センサー |

## コア機能

- モバイル人間検出
・照度検出（M1/HS8MISシリーズ）
- 検出距離 ≤7m (25°C)
- 設置方法: 壁掛けまたは天井取り付け
- バッテリー寿命: 3 年 (Zigbee) / 1 年 (Matter)

## 技術パラメータ (M1 シリーズ)

- 検出角度: 110° / 360°
- 検出距離: ≤7m
- 照度範囲: 0~1200 ルクス
- 電源：DC 3V（リチウム電池）
- バッテリー寿命: 3 年 (Zigbee) / 1 年 (Matter)

## シナリオの例

- 人が来るとライトが点灯し、人が離れると消灯します。
- セキュリティ連携：異常な動きを検知するとプッシュアラーム
- 外出時の警戒：周りに人がいないときは、空調/暖房を自動的に停止してエネルギーを節約します
