## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|人が来ると照明が点灯して省エネ | M1-Z / HS8MIS | PIR で人を検知 → ライトを点灯 + 照度を読み取って明るさを判断 → 周囲に人がいないときはライトを消す |
|セキュリティアーミングリンケージ | HS1MS |外出モードで動きを検知 → プッシュアラーム + 音と光のサイレン + カメラ録画 |
|天井を完全にカバー | HS3MS |屋内中天井、360°行き止まりがなく、オープンスペースに最適 |
|物質クロスエコロジー | M1-M | Apple ホームオートメーション + Google/Alexa 音声、光連動カーテン |
|長い廊下/ガレージ | HS8MLS |長距離検知、車両・人の出入りを自動照明 |

## 選択の提案

- **廊下/部屋の壁取り付け**: [HS1MS](../../ワイヤレス プロトコル/Zigbee/HS1MS - 人体赤外線センサー (Zigbee)) または [HS8MLS](../../ワイヤレス プロトコル/Zigbee/HS8MLS - マイクロ モーション センサー (Zigbee)) (長距離)
- **家全体の天井カバー範囲**: [HS3MS](../../ワイヤレスプロトコル/Zigbee/HS3MS - 人体赤外線センサー (Zigbee)) (360°)
- **照明連携が必要です**: [M1 シリーズ](../../ワイヤレス プロトコル/Zigbee/M1-Z - 人体赤外線センサー (Zigbee)) または [HS8MIS](../../ワイヤレス プロトコル/Zigbee/HS8MIS - 動作センサー (Zigbee))
- **Matter プラットフォーム**: [M1-M](../../無線プロトコル/Matter/M1-M 人体赤外線センサー (Matter))
