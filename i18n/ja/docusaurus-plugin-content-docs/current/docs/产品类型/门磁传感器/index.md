# ドアセンサー

ドアセンサーはドアや窓の開閉状態を検出するために使用され、インテリジェントセキュリティシステムの基本コンポーネントです。

## 製品シリーズ

|シリーズ |コミュニケーション方法 |特長 |
|------|----------|------|
| HS1DSシリーズ |ジグビー / Z-Wave |スタンダードタイプ、電池寿命3年 |
| HS1DS-M |スレッドよりも重要 |事項バージョン |

## コア機能

- ドアと窓の開閉状態をリアルタイムに検出
- タンパーアラーム（タンパー）
- バッテリー残量低下リマインダー
・連動対応（ドアを開ける、照明をつける、外出時の警戒など）
- 取り付けが簡単 (3M 粘着テープまたはネジ固定)

## 技術パラメータ

電源方式：CR1632ボタン電池
- 電池寿命：約2～3年
- 動作温度: -10°C ~ +50°C
- 取り付けクリアランス: ≤10 mm

## シナリオの例

- **家から離れてください**: ドアセンサーがドアが開いたことを検知し、アラームを鳴らします。
- **帰宅シーン**: ドアを開けると自動的に照明が点灯します。
- **窓監視**: 気象観測所にリンクされており、雨が降ると自動的に窓を閉めます。
