# インストールガイド

## 設置場所の要件

・本体（送信機）は**ドア・窓枠**に取り付けます。
- **ドア/サッシ**に磁石が取り付けられています
- 本体と磁石の隙間 **≦ 10 mm**
- 2 つの位置を合わせて、マーキングラインを合わせます。

## インストール手順

1. 設置面を掃除して、乾燥していて埃がないことを確認します。
2. 3M粘着保護フィルムを剥がし、本体をドアフレームに貼り付けます。
3. 磁石を対応する位置に貼り付けます（ドアを閉めたときに本体に合わせます）。
4. バッテリー (CR1632) を挿入すると、インジケーターライトが点滅して電源が入っていることを示します。
5. ドアと窓の開閉テスト: ドアが開くと警報が作動し、閉じると元に戻ります。
6. ゲートウェイ経由でペアリングする

## 注意事項

- 設置隙間が大きすぎると検出不良の原因となります
- 金属製のドアや窓は無線信号に影響を与える可能性があります。修正する前にテストすることをお勧めします。
- バッテリー寿命は約 2 ～ 3 年で、バッテリーが少なくなるとアプリが通知します。
