# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |電源 |バッテリー寿命 |特長 |ゲームの提案 |
|------|------|------|----------|------|----------|
| [HS1DS](../../ワイヤレスプロトコル/Zigbee/HS1DS - ドア磁気センサー (Zigbee)) |ジグビー / Z-Wave | CR1632 | 2～3年 |標準タイプ、デュアルプロトコルはオプション |ドアオープン時に照明点灯、外出時にアーム、ドアや窓の不正開をプッシュ通知+音と光の警報連動 |
| [HS1DS-M](../../ワイヤレスプロトコル/マター/HS1DS-M - スマートドア磁気センサー (マターオーバースレッド)) |スレッドよりも重要 | CR1632 | 2～3年 | Matter バージョン、クロスプラットフォーム | Apple Home へのネイティブ アクセス、外出モードに入る前にドアと窓を自動的にチェック |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|帰宅すると自動的に照明が点灯します | HS1DS |ドアセンサー検知でドアを開ける → リビングの照明をつける → エアコンをつける → ようこそアナウンス |
|家から出て武装する | HS1DS |ドアや窓の異常な開きを検知 → 携帯電話プッシュ＋音光警報連動 |
|窓防雨リンケージ | HS1DS |窓が開いて雨センサーが作動 → TTS ブロードキャストで窓を閉めるようリマインダー |
| Apple Home ユーザー | HS1DS-M | Native Matter、HomeKit 自動化「最後の人が家を出た後にチェック」 |
|保管室/引き出しの監視 | HS1DS |小型設置、キャビネットのドア/引き出しの不正な開きを検出、オフィス/ホテルに最適 |

## 選択の提案

- **既存の Zigbee ゲートウェイ**: [HS1DS](../../ワイヤレス プロトコル/Zigbee/HS1DS - ドア磁気センサー (Zigbee)) (Zigbee バージョン)
- **クロスプラットフォーム互換性が必要です**: [HS1DS-M](../../ワイヤレス プロトコル/マター/HS1DS-M - スマート ドア磁気センサー (スレッド上のマター)) (スレッド上のマター)
- **Z-Wave システム**: [HS1DS](../../ワイヤレス プロトコル/Zigbee/HS1DS - ドア磁気センサー (Zigbee)) (Z-Wave バージョン)
