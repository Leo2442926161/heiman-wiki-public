# インストールガイド

## 設置場所の要件

- 振動を監視する必要がある物の表面に設置します
- 一般的な設置場所: **ドアと窓の内側**、**金庫**、**パイプ**、**壁**、**フェンス**
- 設置面が平らで清潔であることを確認してください。

## インストール手順

1. 監視対象（金庫のドアの内側など）を選択します。
2. 取り付け面をきれいにします
3. 3M粘着テープまたはネジを使用してセンサーを固定します
4. CR2450 電池を取り付けます。
5. ゲートウェイ経由でペアリングする
6. シミュレーションテスト: 設置面をタップすると、センサーがアラームをトリガーします。

## 注意事項

- ドアや窓のこじ開け防止の場合は、ドアや窓の内側の鍵付近に設置するとより効果的です。
- パイプライン監視の場合、パイプライン表面に直接貼り付けます。
- 電池寿命は約2年
- さまざまなシナリオに合わせて感度を調整可能
