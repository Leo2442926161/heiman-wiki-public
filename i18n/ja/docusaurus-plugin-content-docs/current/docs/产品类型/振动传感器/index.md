# 振動センサー

振動センサーは物体の振動や傾きを検知して警報を発します。ドア、窓、金庫、パイプなど様々なシーンに最適です。

## コア機能

- 3次元振動検出
- 傾斜角度検出
- バッテリー残量低下リマインダー

## 該当するシナリオ

- **ドアと窓のこじ開け防止**: 暴力的な侵入による振動を検出します。
- **金庫/引き出しモニター**: 異常な動きを検出します。
- **パイプライン監視**: 水道管の振動を検出します (給水ポンプの異常またはウォーターハンマーを示します)
- **壁/フェンスの監視**: 登りや衝撃を検出します。

## 技術パラメータ

- センサーの種類: 3 軸加速度センサー
- 検出寸法: 振動 + 傾き
電源方式：CR2450ボタン電池
- バッテリー寿命: 約。 2年
