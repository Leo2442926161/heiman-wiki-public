# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |センサー |電源 |特長 |ゲームの提案 |
|------|------|----------|------|------|----------|
|振動センサー |ジグビー | 3軸加速度計 | CR2450 |振動＋傾き検知 |ドアや窓のこじ開け防止、金庫・引き出しの不正開振動警報、パイプの異常振動監視など |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|ドアや窓のこじ開け防止 |振動センサー |ドア・窓などの激しい衝撃・振動を検知 → プッシュ警報＋音・光警報＋カメラ撮影 |
|安全な監視 |振動センサー |金庫が移動または振動した場合 → 即時警報通知 |
|パイプライン監視 |振動センサー |水道管の異常振動（ウォーターポンプ故障・ウォーターハンマー）を監視し、メンテナンスを早期に警告 |
|フェンス/壁 |振動センサー |壁/フェンスのよじ登り検知、ヤードセキュリティに最適 |
