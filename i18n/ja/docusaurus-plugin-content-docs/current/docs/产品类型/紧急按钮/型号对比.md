# モデルの比較

## 各種モデルの比較

|モデル |プロトコル |トリガー方法 |電源 |特長 |ゲームプレイの提案 |
|------|------|----------|------|------|----------|
|緊急ボタン |ジグビー |トリガーを長押し |バッテリー |ポータブル+壁掛け両用 |高齢者が持ち運べる、緊急時にワンクリックで助けられる、携帯電話のプッシュ+音と光のアラームと連動 |

## おすすめシーンのゲームプレイ

|シナリオ |おすすめモデル |連携ソリューション |
|------|----------|----------|
|高齢者の在宅介護 |緊急ボタン |高齢者の突然の不快感 → 長押し → 家族全員の携帯電話にプッシュ → 音と光で連動アラーム → 物件通知 |
|子供部屋の安全 |緊急ボタン |子供が一人で家にいて緊急事態に遭遇した場合 → ボタンを押して親の携帯電話のアラームを鳴らします |
|ショップ/レジカウンター |パニックボタン |隠蔽設置、盗難発生時にサイレントアラームを作動、監視ビデオと連動 + 静かにセキュリティに通知 |
|トイレ・浴室 |緊急ボタン |高齢者がシャワー中に滑って転んでも、手元にある防水緊急ボタンで簡単に助けられます |
