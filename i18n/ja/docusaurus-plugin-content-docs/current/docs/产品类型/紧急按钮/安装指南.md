# インストールガイド

## 設置場所の要件

- **壁掛け設置**: バスルーム、ベッドルームのベッドサイド、リビングルームのソファの横など、手の届く範囲ならどこにでも設置可能
- **どこにでも着用可能**: ポケットに入れたり、キーチェーンに掛けたりできます。
- **高齢者向け**: 浴室と寝室のベッドサイドに設置、高さ約1.2m

## インストール手順

1. 設置場所または着用するかどうかを決定します。
2. 壁取り付け: ネジまたは 3M 接着剤を使用して取り付けベースを固定します。
3. バッテリーを取り付けます
4. 緊急ボタンをベースに置くか、持ち歩きます。
5. ゲートウェイ経由でペアリングする
6. テスト: ボタンを長押ししてアラームをトリガーし、APP が通知を受信することを確認します。

## 注意事項

- **長押し防止タッチ** 設計を採用しており、トリガーするには 1 ～ 2 秒間押し続ける必要があります。
- 音と光のアラームとの連動がおすすめ
- 機能を定期的にテストします
●高齢者向けには転倒検知器と併用することで二重の保護が可能です。
