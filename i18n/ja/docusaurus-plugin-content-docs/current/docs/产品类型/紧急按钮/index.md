# パニックボタン

緊急ボタン (緊急ボタン/パニック ボタン) は、緊急時に救難信号を素早く発するために使用され、スマートな高齢者ケアおよびセキュリティ システムの重要な部分です。

## 製品シリーズ

|シリーズ |コミュニケーション方法 |特長 |
|------|----------|------|
|緊急ボタン |ジグビー |ポータブル、ボタン 1 つで助けを求める |

## コア機能

- ワンクリックで緊急警報をトリガー
- ポータブル設計 (持ち運びまたは壁掛け可能)
- バッテリー残量低下リマインダー
- 偶発的なタッチ防止設計 (トリガーの長押し)

## 該当するシナリオ

- **スマート高齢者ケア**: 高齢者は、突然病気になったり倒れたりしたときに、ワンクリックで助けを求めることができます。
- **子供部屋**: 子供が家に一人でいるときに緊急時に使用されます。
- **医療**: 病棟/老人ホームのベッドコール
- **ショップ/フロントデスク**: 強盗や緊急事態の場合に秘密のアラームを作動させます

## トリガー連動

- 家族/保護者への携帯電話のプッシュ通知
- 音と光のアラーム連動
- 宿泊施設または120に自動的に電話します
