# 海曼科技企業紹介

**深圳市海曼科技股份有限公司**（Shenzhen Heiman Technology Co., Ltd.）は 2005 年に設立され、自主開発、生産、販売、サービスを一体化した **国家ハイテク企業** 及び **国家専精特新「小巨人」企業** です。

## 海曼について

海曼科技はスマートセンシング製品の研究開発と革新に特化しており、コア製品は以下をカバーしています：

- **スマート消防** — 煙警報器、一酸化炭素警報器などの消防センシング製品
- **スマートガス** — ガス漏れ警報器、ガス緊急遮断弁など
- **スマート介護** — 緊急呼出装置、人体転倒検知器、水浸検知器など
- **スマートセンシング** — 赤外線検知器、ドア/窓磁気センサー、温湿度センサーなど
