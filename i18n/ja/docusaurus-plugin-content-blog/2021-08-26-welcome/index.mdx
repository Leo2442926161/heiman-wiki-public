## このサイトについて

海曼科技 Wiki は、パートナー、開発者、お客様にワンストップの技術リソースを提供し、以下の内容を迅速に理解するお手伝いをします：

- **製品センター** — スマート消防、スマートガス、スマート介護、スマートセンシングの全シリーズ製品
- **ソリューション** — 業界レベルのスマートセキュリティ全体ソリューション
- **技術ドキュメント** — API 連携ガイド、SDK 使用説明、インストール・導入マニュアル
- **業界インサイト** — スマートセキュリティ分野の最先端トレンドと政策解説
