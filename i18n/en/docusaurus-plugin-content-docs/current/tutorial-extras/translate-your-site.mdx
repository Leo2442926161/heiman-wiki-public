# Multi-Language Support

The Heiman Technology Wiki supports multi-language documentation for global partners and customers.

## Currently Supported Languages

- **简体中文 (zh-Hans)** — Default
- **English (en)**

## Product Internationalization

Heiman products support multi-language APP interfaces and device voice prompts, including Chinese, English, Japanese, Korean, Spanish, and more.

## Contributing Translations

If you would like to help translate Heiman technical documentation, please contact us through our official website.
