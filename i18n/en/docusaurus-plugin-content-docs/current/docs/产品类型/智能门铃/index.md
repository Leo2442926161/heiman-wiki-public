# Smart Doorbell

Heiman Smart Doorbell integrates doorbell calling and security monitoring functions, supporting remote video intercom.

## Core Features

- Doorbell button trigger
- Remote real-time video (Wi-Fi models)
- Two-way audio
- Motion detection recording
- Night vision

## Applicable Scenarios

- Visitor calling: Answer whether at home or away
- Delivery/food notification: Remotely confirm and unlock
- Security monitoring: Record suspicious activity at the door

> **Note**: Heiman doorbell product line is primarily based on Zigbee/Wi-Fi solutions. Please consult official sources for specific model specifications.
