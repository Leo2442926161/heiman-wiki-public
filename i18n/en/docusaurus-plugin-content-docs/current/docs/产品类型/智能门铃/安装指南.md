# Installation Guide

## Installation Location Requirements

- Install at a height of approximately **1.4 ~ 1.6 m** (at eye level)
- Face the doorway to capture visitors' faces
- Avoid direct sunlight on the camera

## Installation Steps

1. Choose an installation location (next to the door or on the door frame)
2. Secure the mounting base with screws
3. Connect power (wire model: connect wires; battery model: insert battery)
4. Attach the doorbell body to the mounting base
5. Pair via the app and connect to Wi-Fi (Wi-Fi models)
6. Adjust the camera angle
7. Test the doorbell button, video, and audio functions

## Precautions

- Wi-Fi models require good Wi-Fi signal coverage at the door
- For battery models, pay attention to charging cycles; consider pairing with a door sensor (wake on door open)
- Clean the camera lens periodically
- Ensure the doorbell is within effective Zigbee signal range
