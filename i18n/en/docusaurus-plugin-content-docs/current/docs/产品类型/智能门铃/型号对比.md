# Model Comparison

## Model Comparison

| Model | Protocol | Video | Power | Features | Recommendation |
|------|------|------|------|------|----------|
| Smart Doorbell | Zigbee + Wi-Fi | 1080P optional | Battery/Mains | 2-in-1 doorbell + security | Visitor press → phone notification, remote video intercom, motion detection recording |

## Recommended Use Cases

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Delivery/food management | Smart Doorbell | Press → phone notification → remote intercom confirm → remote unlock (联动 smart lock) |
| Security monitoring | Smart Doorbell | Loitering detection → recording + push alert, 联动 door sensor / lights |
| Visitor联动 | Smart Doorbell | Press → indoor screen/speaker shows video → turn on lights → unlock联动 |
