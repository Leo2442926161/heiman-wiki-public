# Installation Guide

## Installation Location Requirements

- **Natural gas (methane)**: Lighter than air, install **within 30 cm** of the ceiling
- **Liquefied petroleum gas (propane)**: Heavier than air, install **within 30 cm** of the floor
- Near potential leak points such as gas stoves, water heaters, and gas pipe connections

## Installation Steps

1. Confirm the type of gas to be detected (natural gas or LPG)
2. Select the appropriate installation height
3. Fix the mounting base to the wall with screws
4. Connect power (most models use mains power)
5. Power on the detector and allow approximately 3-5 minutes warm-up
6. Self-test: Press the test button, verify audible and visual alarms work
7. Pair with gateway (Zigbee models)
8. If integrating with a solenoid valve, connect the shut-off valve control wire

## Precautions

- Periodically test sensor sensitivity with test gas
- Sensor lifespan is approximately 3-5 years, replace when expired
- If you smell gas, do not operate electrical switches or use open flames
- Installation height depends on gas type (natural gas high, LPG low)
