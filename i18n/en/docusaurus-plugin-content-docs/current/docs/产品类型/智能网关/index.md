# Smart Gateway

The Smart Gateway is the central hub of the Heiman smart home system. It connects various Zigbee sensors and enables device-to-device automation.

## Product Series

| Series | Communication | Features |
|------|----------|------|
| HS1GW Series | Zigbee + Wi-Fi / Ethernet | Basic gateway |
| HS3GW Series | Zigbee + Wi-Fi / Ethernet | Enhanced |
| HS5GW Series | Zigbee + Wi-Fi / Ethernet | High-performance gateway |
| HS6GW Series | Zigbee + Wi-Fi / Ethernet + Thread | Latest model, supports both Zigbee and Matter over Thread |

## Core Features

- Zigbee network coordinator
- Wi-Fi / Ethernet internet connectivity
- Automatic device discovery and pairing
- Local scene automation (select models)
- Remote access (via Heiman cloud platform)
- Supports up to 128 child devices (varies by model)

## Applicable Scenarios

- Core connection point for whole-home Heiman smart system
- Remote monitoring via the Heiman app
- Local centralized management for small to medium commercial projects

## Difference from Other Gateways

The Heiman gateway is the entry point to the official ecosystem. For users who wish to use third-party platforms, a universal Zigbee coordinator with Zigbee2MQTT / ZHA is recommended.
