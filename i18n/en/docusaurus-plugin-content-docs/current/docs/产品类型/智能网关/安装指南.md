# Installation Guide

## Installation Location Requirements

- **Central location in the home**, ensuring Zigbee signal reaches all devices
- Avoid placing inside all-metal cabinets (metal shields wireless signals)
- **Ethernet preferred**: Use a wired network connection for better stability
- Keep within 20-30 m of other Zigbee devices (indoor)

## Installation Steps

1. Choose an installation location (living room TV cabinet, near the弱电箱, etc.)
2. Connect the power adapter
3. Connect the Ethernet cable (recommended) or connect to Wi-Fi
4. Wait for the indicator light to turn solid green
5. Open the Heiman app and scan the QR code on the bottom of the gateway to add it
6. Once added, start pairing child devices

## Precautions

- The gateway is the core of your smart home; consider using a UPS backup power supply
- Ethernet is more stable than Wi-Fi; wired connection is recommended
- If device signal is weak, add Zigbee range extenders (such as smart plugs)
- Different models support different maximum numbers of child devices; choose according to scale
