# Model Comparison

## Model Comparison

| Model | Max Zigbee Devices | Connectivity | Features | Recommendation |
|------|------------------|----------|------|----------|
| [HS6GW](../../无线协议/Zigbee/HS6GW.md) | 128 | WiFi + Ethernet | Latest model, supports future Thread upgrade | Core of Heiman ecosystem, remote management via Heiman app, ideal for users who want a hassle-free experience |
| HS5GW | 64 | WiFi + Ethernet | High-performance | Medium to large projects, stable and reliable |
| HS3GW | 32 | WiFi | Enhanced | Sufficient for small to medium homes |
| HS1GW | 16 | WiFi | Basic | Entry-level, suitable for a few devices |

## Recommended Use Cases

| Scenario | Recommended Model | Description |
|------|----------|------|
| Whole-home Heiman ecosystem | HS6GW | Use with Heiman Zigbee sensors, remote management via Heiman app, no need to set up your own server |
| Third-party platform users | — | Buy a Zigbee coordinator (e.g., SLZB-06) with Zigbee2MQTT / ZHA for more flexibility |
| Small to medium homes | HS3GW | 32 device limit, meets basic needs |

## Selection Guide

- **Heiman ecosystem**: [HS6GW](../../无线协议/Zigbee/HS6GW.md) (128 devices, WiFi + Ethernet)
- **Third-party platforms**: Recommended: universal Zigbee coordinator + Zigbee2MQTT / ZHA
