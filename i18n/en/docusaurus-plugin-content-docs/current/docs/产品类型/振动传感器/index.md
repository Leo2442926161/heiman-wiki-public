# Vibration Sensor

The Vibration Sensor detects vibration or tilt of objects to trigger alarms, suitable for doors, windows, safes, pipes, and more.

## Core Features

- 3-axis vibration detection
- Tilt angle detection
- Low battery reminder

## Applicable Scenarios

- **Door/window tamper detection**: Detects vibration from forced entry
- **Safe/drawer monitoring**: Detects abnormal movement
- **Pipe monitoring**: Detects pipe vibration (indicating pump issues or water hammer)
- **Wall/fence monitoring**: Detects climbing or impact

## Technical Specifications

- Sensor type: 3-axis accelerometer
- Detection dimensions: Vibration + Tilt
- Power: CR2450 coin cell battery
- Battery life: Approximately 2 years
