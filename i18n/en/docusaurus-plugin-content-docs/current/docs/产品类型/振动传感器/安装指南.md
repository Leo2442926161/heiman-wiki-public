# Installation Guide

## Installation Location Requirements

- Install on the surface of the object to be monitored for vibration
- Common installation locations: **inside of doors/windows**, **safes**, **pipes**, **walls**, **fences**
- Ensure the mounting surface is flat and clean

## Installation Steps

1. Select the object to monitor (e.g., inside of a safe door)
2. Clean the mounting surface
3. Secure the sensor with 3M adhesive or screws
4. Insert the CR2450 battery
5. Pair with the gateway
6. Test: Tap the mounting surface, the sensor should trigger an alarm

## Precautions

- For door/window tamper scenarios, install on the inside near the lock for best results
- For pipe monitoring, attach directly to the pipe surface
- Battery life is approximately 2 years
- Sensitivity can be adjusted for different scenarios
