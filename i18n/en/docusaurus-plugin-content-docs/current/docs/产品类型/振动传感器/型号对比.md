# Model Comparison

## Model Comparison

| Model | Protocol | Sensor | Power | Features | Recommendation |
|------|------|--------|------|------|----------|
| Vibration Sensor | Zigbee | 3-axis accelerometer | CR2450 | Vibration + tilt detection | Door/window tamper, safe/drawer intrusion vibration alarm, abnormal pipe vibration monitoring |

## Recommended Use Cases

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Door/window tamper | Vibration Sensor | Detect forced door/window vibration → push alert + siren + camera capture |
| Safe monitoring | Vibration Sensor | Safe moved or vibrated → immediate alert notification |
| Pipe monitoring | Vibration Sensor | Monitor abnormal pipe vibration (pump failure/water hammer), early maintenance warning |
| Fence/wall | Vibration Sensor | Fence/gate climbing detection, suitable for yard security |
