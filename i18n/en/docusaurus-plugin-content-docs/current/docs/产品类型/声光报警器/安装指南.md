# Installation Guide

## Installation Location Requirements

- **Install in a central location** so all rooms can hear the alarm
- Hallway, living room ceiling or high on a wall
- **At least 2 m** above the floor (to prevent tampering)
- When pairing with smoke/CO sensors, install near bedroom hallways

## Installation Steps

1. Choose an installation location (hallway or living room recommended)
2. Secure the mounting base to the wall or ceiling with screws
3. Connect power (mains model) or insert batteries (battery model)
4. Attach the siren body to the mounting base
5. Pair with the gateway
6. Test: Trigger any sensor and confirm the siren responds

## Precautions

- Install at a height out of reach of children
- Periodically test the alarm volume and lights
- For battery models, replace batteries when low
- Configure联动 rules with other sensors as needed
