# Siren

The Smart Siren provides loud audible and visual alarms. Used with various sensors, it serves as the output actuator of a security system.

## Core Features

- High-decibel alarm sound (≥110 dB)
- Red/color strobe light
- Multiple alarm tones supported
- Tamper detection
- Remote arm/disarm support

## Applicable Scenarios

- 联动 with smoke/CO sensors: sounds throughout the house when danger is detected
- Security intrusion alarm: triggers when doors/windows are illegally opened
- Emergency evacuation: guides occupants during a fire alarm

## Automation Scenarios

- **Smoke alarm + Siren**: All sirens sound simultaneously, alerting everyone in the house
- **Door sensor alarm + Siren**: Deters intruders
- **With gateway**: 联动 with other devices via Zigbee/Matter network
