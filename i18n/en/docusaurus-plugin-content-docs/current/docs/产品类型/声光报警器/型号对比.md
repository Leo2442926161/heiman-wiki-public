# Model Comparison

## Model Comparison

| Model | Protocol | Alarm Volume | Power | Features | Recommendation |
|------|------|----------|------|------|----------|
| Siren | Zigbee | ≥110 dB | Mains | Bright strobe + high volume | 联动 with smoke/CO/door sensors for whole-house alarm, deters intruders when armed away |

## Recommended Use Cases

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Whole-house fire联动 | Siren | Any smoke/CO sensor triggers → all sirens sound, waking sleeping occupants |
| Security intrusion deterrence | Siren | Door sensor illegally opened + away mode → siren sounds + all lights on, deterring intruders |
| Emergency evacuation | Siren | Fire trigger → siren guides evacuation direction, paired with door sensors to confirm egress |
