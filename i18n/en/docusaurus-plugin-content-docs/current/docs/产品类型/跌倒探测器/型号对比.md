# Model Comparison

## Model Comparison

| Model | Protocol | Sensors | Power | Features | Recommendation |
|------|------|--------|------|------|----------|
| [HS2FD](../../无线协议/Zigbee/HS2FD.md) | Zigbee | Accelerometer + Gyroscope | Battery | AI fall detection | Wearable design, automatic fall detection + one-touch求救, suitable for elderly living alone and nursing homes |

## Recommended Use Cases

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Elderly living alone | HS2FD | Fall detected → auto-push to children's phones → timeout no response auto-call 120 |
| Nursing homes | HS2FD | Centralized management platform + floor siren + caregiver handheld同步 alert |
| Post-surgery rehab | HS2FD | Patient gets up and fall risk detected → early warning to nurse station |
| Bathroom/nighttime | HS2FD | IP-rated design, wearable in shower + nighttime fall automatic alert |

## Selection Guide

- **Home care**: [HS2FD](../../无线协议/Zigbee/HS2FD.md) (wearable, AI detection)
- **Pair with emergency button**: HS2FD automatic fall detection + Emergency Button manual求救, dual protection
