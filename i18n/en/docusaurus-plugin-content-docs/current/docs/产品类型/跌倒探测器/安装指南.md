# Installation Guide

## Wearing Methods

- **Belt clip**: Clip onto a belt or waistband
- **Lanyard**: Hang around the neck, resting on the chest

## Usage Instructions

1. Insert battery
2. Choose wearing method (belt clip or lanyard)
3. Wear on the body
4. Pair with the gateway
5. Test simulation:
   - Quickly squat down or lie on the floor from a standing position
   - The sensor should detect the fall and trigger an alarm within seconds
   - Press the求救 button to test manual alarm

## Precautions

- It is recommended to **wear it all day** (can be removed to charge/replace batteries during shower)
- Uses AI algorithm to distinguish falls from daily activities (sitting, bending), low false alarm rate
- Automatically alarms when a fall is detected, manual button presses also work
- Can be used with the emergency button for dual protection
