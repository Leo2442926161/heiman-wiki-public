# 安装指南

## 安装位置要求

- 安装在 **1.2 ~ 1.8 m** 高度（呼吸区域）
- 选择能代表房间平均空气质量的位置
- **远离**：窗户、门窗、通风口、空调出风口
- **远离**：厨房油烟、加湿器、香薰等局部污染源

## 安装步骤

1. 选定安装位置（客厅、卧室或办公室墙壁）
2. 用螺丝固定安装底座
3. 连接电源（USB 或适配器供电）
4. 传感器开机，自检后显示屏（如有）显示数据
5. 通过网关配对
6. 在 APP 中确认数据正常

## 注意事项

- 确保传感器进气口和排气口不被遮挡
- 定期清洁传感器外壳，防止灰尘堵塞
- 放在儿童房可监测 CO₂ 水平，提示开窗换气
- 建议搭配新风系统或空气净化器联动
