# 安装指南

## 安装位置要求

- **安装在呼吸高度**——距离地面约 **1.5 m**
- 靠近卧室区域，确保睡眠时能听到报警
- **不要安装在角落**，保持空气流通
- 远离通风口、窗户、风扇等影响气流的位置

## 安装步骤

1. 确定安装位置（建议卧室走廊或客厅）
2. 将安装底座固定在墙上（螺丝或 3M 胶贴）
3. 装入电池，传感器自动开机
4. 将主体安装到底座上
5. 自检：按下测试键，报警灯闪烁并发出报警声
6. 通过网关配对

## 注意事项

- 传感器寿命典型 **7 年**，到期后需更换整机
- 每月测试一次功能
- 请勿遮挡传感器通气孔
- 如有报警先开窗通风，再检查是否有 CO 源泄漏
