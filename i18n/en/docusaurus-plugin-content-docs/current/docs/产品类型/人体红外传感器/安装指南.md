# 安装指南

## 安装位置要求

- 壁挂安装高度：**1.8 ~ 2.4 m**
- 吸顶安装：安装在房间中央天花板
- **避免**：阳光直射、暖气/空调出风口、热源附近
- 检测范围内避免有遮挡物（家具、玻璃隔断）

## 安装步骤

1. 选定安装位置（壁挂或吸顶）
2. 用螺丝固定安装底座
3. 装入电池，指示灯闪烁
4. 将传感器主体安装到底座上
5. 调整检测角度（水平 110° 范围内）
6. 步行测试：走过检测区域，指示灯应亮起
7. 通过网关配对

## 注意事项

- 红外传感器对温度变化敏感，夏季高温环境检测距离可能缩短
- 避免正对窗户（室外温度变化引起误报）
- 电池寿命约 3 年（Zigbee）/ 1 年（Matter）
