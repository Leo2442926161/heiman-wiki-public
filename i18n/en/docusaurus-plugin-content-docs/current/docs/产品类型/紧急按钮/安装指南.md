# Installation Guide

## Installation Location Requirements

- **Wall-mounted**: Bathroom, bedside, next to living room sofa — within easy reach
- **Wearable**: Can be kept in a pocket or attached to a keychain
- **For elderly**: Install in bathroom and beside bed at approximately 1.2 m height

## Installation Steps

1. Decide on installation location or whether to carry it
2. Wall-mount: Secure the mounting base with screws or 3M adhesive
3. Insert battery
4. Place the emergency button on the base or carry it
5. Pair with the gateway
6. Test: Long press to trigger alarm, confirm the app receives notification

## Precautions

- **Long press anti-accidental design**: Requires holding for 1-2 seconds to trigger
- It is recommended to 联动 with a siren
- Test functionality periodically
- For elderly users, can be used together with a fall detector for dual protection
