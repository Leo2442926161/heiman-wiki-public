# Model Comparison

## Model Comparison

| Model | Protocol | Rated Power | Metering Accuracy | Features | Recommendation |
|------|------|----------|----------|------|----------|
| [HS2SK](../../无线协议/Zigbee/HS2SK.md) | Zigbee | 16A / 3680W | ±2% | Metering type | Real-time power monitoring + remote switch, away mode一键断电, track AC/water heater energy usage |
| [HS2SK-WiFi](../../无线协议/Wifi/HS2SK-WiFi.md) | WiFi | 16A / 3680W | ±2% | Direct connect, no gateway needed | No extra gateway required, control via Smart Life App, ideal for users who don't want a gateway |

## Recommended Use Cases

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Energy monitoring | HS2SK | Zigbee接入 HA, 联动 InfluxDB + Grafana dashboard for daily/monthly power usage, identify high-consumption appliances |
| Away power-off | HS2SK | Away mode一键 turn off TV/AC/aquarium lights etc., safe and energy-saving |
| Timer control | HS2SK / HS2SK-WiFi | Schedule water heater/humidifier/charger, leverage off-peak electricity pricing |
| No gateway user | HS2SK-WiFi | Direct connect to router, Smart Life App control, zero门槛 setup |

## Selection Guide

- **Already have Zigbee gateway**: [HS2SK](../../无线协议/Zigbee/HS2SK.md)
- **Don't want a gateway**: [HS2SK-WiFi](../../无线协议/Wifi/HS2SK-WiFi.md)
