# Smart Plug

Heiman Smart Metering Plug supports remote on/off control and real-time power consumption metering, helping users achieve intelligent management of home appliances.

## Product Series

| Series | Communication | Features |
|------|----------|------|
| HS2SK Series | Zigbee | Metering type, supports power monitoring |

## Core Features

- Remote on/off control
- Real-time power/energy metering
- Overload protection
- Timer scheduling
- Energy statistics

## Technical Specifications

- Rated voltage: AC 220V (China standard) / AC 110V (US standard)
- Max load: 16A / 3680W
- Metering accuracy: ±2%

## Use Cases

- Remotely turn off appliances left on
- Track AC/water heater energy consumption
- Away mode: one-touch shut-off of all non-essential appliances
- Timer control for aquarium lights, humidifiers, etc.
