# HS1SA-Z 智能烟雾报警器（Z-Wave）

HS1SA-Z 是海曼推出的 **Z-Wave** 智能烟雾报警器，采用光电式感烟技术，兼容 Z-Wave Plus 标准，适用于欧美智能家居市场。

## 产品概述

- **产品类型**：智能烟雾报警器
- **通信协议**：Z-Wave Plus
- **认证标准**：EN 14604
- **典型场景**：家庭、公寓等场所的火灾预警

## 技术参数

| 参数 | 值 |
|------|-----|
| 传感器类型 | 红外光电式 |
| 通信协议 | Z-Wave Plus（ITU-T G.9959） |
| 工作频段 | 868.42 MHz（欧盟）/ 908.42 MHz（美国） |
| 报警声压 | ≥85 dB（3 米） |
| 供电方式 | CR123A 锂电池 |
| 电池寿命 | 约 2~3 年 |
| 加密方式 | S2 Authenticated / S2 Unauthenticated |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤90% RH（无凝结） |

## 功能特性

- 光电式烟雾检测，可探测慢燃和明火烟雾
- 85 dB 高声光报警
- 低电量提醒
- 自检功能
- 防拆报警
- Z-Wave S2 安全加密
- 支持 OTA 固件升级

## 认证

- EN 14604 认证
- Z-Wave Plus 认证
- S2 安全认证
