# 技术参数

以下为海曼 Z-Wave 设备的通用技术参数。

## 无线参数

| 参数 | 值 |
|------|-----|
| 协议标准 | Z-Wave Plus（ITU-T G.9959） |
| 工作频段 | 800~900 MHz（视区域：欧规 868.42 MHz，美规 908.42 MHz） |
| 传输速率 | 最高 100 kbps |
| 网络拓扑 | Mesh 自愈网络（源路由） |
| 节点容量 | 单网络最多 232 个节点 |
| 通信距离 | 室内约 30 米（节点间），全屋 Mesh 覆盖 |
| 加密方式 | AES-128（S2 Authenticated / S2 Unauthenticated） |

## 区域频段

| 区域 | 中心频率 |
|------|----------|
| 欧洲 | 868.42 MHz |
| 北美 | 908.42 MHz |
| 其他区域 | 视当地法规 |

## 电气与功耗

| 参数 | 值 |
|------|-----|
| 供电方式 | 电池供电（CR123A / AA）或市电供电 |
| 电池续航 | 传感器类约 2~3 年 |
| 发射功率 | ≤ 1 mW（符合 SRD 法规） |
| 接收灵敏度 | 约 -100 dBm |

## 环境参数

| 参数 | 值 |
|------|-----|
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤ 90% RH（无凝结） |

## 网络特性

| 特性 | 说明 |
|------|------|
| SmartStart | 支持（700 系列），扫码即加入网络 |
| S2 安全 | 支持三种安全等级：Unencrypted、S2 Unauthenticated、S2 Authenticated |
| 包含 CC | Security CC、Security 2 CC、Firmware Update CC、Basic CC、Battery CC、Notification CC 等 |
