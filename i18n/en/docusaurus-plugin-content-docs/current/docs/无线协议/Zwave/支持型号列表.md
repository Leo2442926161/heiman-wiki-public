# 支持型号列表

海曼 Z-Wave 产品主要面向欧美智能家居市场，兼容 Z-Wave Plus 标准。

## 安防报警类

| 型号 | 产品名称 | 说明 |
|------|----------|------|
| HS1SA-Z | 智能烟雾报警器（Z-Wave） | Z-Wave 版烟雾报警器 |
| HS1CA-Z | 智能一氧化碳报警器（Z-Wave） | Z-Wave 版 CO 报警器 |
| HS1CG-Z | 可燃气体探测器（Z-Wave） | Z-Wave 版燃气探测器 |
