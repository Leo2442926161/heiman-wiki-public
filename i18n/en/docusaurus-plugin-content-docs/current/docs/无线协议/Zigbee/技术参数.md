# 技术参数

以下为海曼 Zigbee 设备的通用技术参数。

## 无线参数

| 参数 | 值 |
|------|-----|
| 协议标准 | Zigbee 3.0（IEEE 802.15.4） |
| 工作频段 | 2.4 GHz（全球通用 ISM 频段） |
| 传输速率 | 最高 250 kbps |
| 网络拓扑 | Mesh 自愈网络 |
| 节点容量 | 单网络最多 65535 个节点 |
| 通信距离 | 室内约 10~30 米（视墙体障碍物） |
| 加密方式 | AES-128-CCM |

## 兼容性

| 项目 | 说明 |
|------|------|
| Zigbee 协调器 | 兼容 Zigbee 3.0 标准协调器 |
| 支持平台 | Zigbee2MQTT、ZHA（Home Assistant）、Hubitat、Homey、SmartThings、Tuya Zigbee 网关、海曼 HS6GW 网关 |
| OTA 升级 | 支持空中固件升级（部分型号） |

## 电气与功耗

| 参数 | 值 |
|------|-----|
| 供电方式 | 电池供电（CR123A / CR2450 / AA）或 USB / 市电（插座类） |
| 电池续航 | 传感器类约 2~3 年（视上报频率） |
| 待机功耗 | 微安级，支持休眠轮询机制 |

## 环境参数

| 参数 | 值 |
|------|-----|
| 工作温度 | -10°C ~ +55°C |
| 工作湿度 | ≤ 95% RH（无凝结） |
| IP 防护等级 | 视具体型号（传感器类通常 IP30） |
