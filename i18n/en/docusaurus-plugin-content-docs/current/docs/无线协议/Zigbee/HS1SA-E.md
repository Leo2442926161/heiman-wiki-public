# HS1SA-E - 智能烟雾报警器（Zigbee）

## 概述

HS1SA-E 是海曼面向欧盟市场推出的标准型 Zigbee 智能烟雾报警器，采用光电式感烟技术，已通过 EN 14604 认证并荣获红点设计大奖。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 传感器类型 | 红外光电式 |
| 报警声压 | ≥85 dB（3 米） |
| 供电方式 | DC 3V（CR123A） |
| 电池寿命 | 约 3 年 |
| 通信距离 | ≤70 m（开阔区域） |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH |
| 安装方式 | 吸顶安装 |

## 功能特性

- 光电式烟雾检测，可探测慢燃和明火产生的烟雾
- 85 dB 高声光报警
- 低电量提醒
- 自检功能（一键测试）
- 防拆报警
- 支持远程静音
- OTA 固件升级

## 认证标准

- EN 14604（烟雾报警器欧洲标准）
- Red Dot Design Award（红点设计大奖）
- CE / RED

## 适用配件

- CR123A 电池
- 安装螺丝 / 膨胀管
