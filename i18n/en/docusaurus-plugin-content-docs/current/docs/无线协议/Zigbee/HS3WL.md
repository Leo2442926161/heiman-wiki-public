# HS3WL - 水浸探测器（Zigbee）

## 概述

HS3WL 是海曼第三代 Zigbee 水浸探测器，在上一代基础上增强了检测灵敏度和无线性能，适用于漏水监测场景。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 检测方式 | 探针检测 |
| 供电方式 | DC 3V 电池 |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH |

## 功能特性

- 漏水 / 积水实时检测
- 增强型检测灵敏度
- 蜂鸣器报警
- 低电量提醒
- 探针防腐蚀设计
