# HS3SA - 智能烟雾报警器（Zigbee）

## 概述

HS3SA 是海曼第三代 Zigbee 智能烟雾报警器，在第一代基础上增强了检测性能和联动能力，适用于欧盟市场。

## 技术规格

| 参数 | 值 |
|------|-----|
| 通信协议 | Zigbee 3.0 |
| 传感器类型 | 红外光电式 |
| 报警声压 | ≥85 dB（3 米） |
| 供电方式 | DC 3V（CR123A） |
| 电池寿命 | 约 3 年 |
| 工作温度 | -10°C ~ +50°C |
| 工作湿度 | ≤95% RH |
| 安装方式 | 吸顶安装 |

## 功能特性

- 光电式烟雾检测
- 85 dB 高声光报警
- 低电量提醒
- 自检功能
- 防拆报警
- 支持远程静音
- OTA 固件升级
- 增强型无线性能

## 认证标准

- EN 14604
- CE / RED

## 适用配件

- CR123A 电池
