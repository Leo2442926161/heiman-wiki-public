# Dashboard Configuration

## Security Overview Panel

Use a Section view layout to organize Heiman devices by area:

```yaml
type: section
title: Smoke & CO Alarms
cards:
  - type: entity
    entity: binary_sensor.heiman_smoke_alarm
    name: Smoke Alarm
  - type: entity
    entity: binary_sensor.heiman_co_alarm
    name: CO Alarm
```

## Mushroom Card Styles

```yaml
type: custom:mushroom-alarm-control-panel-card
entity: alarm_control_panel.home_alarm
name: Security System

type: custom:mushroom-entity-card
entity: binary_sensor.heiman_smoke_alarm
name: Smoke Alarm
icon: mdi:smoke-detector
primary_info: state
secondary_info: last-changed
fill_container: true
```

## Environmental Sensors Panel

```yaml
type: section
title: Environment Monitoring
cards:
  - type: gauge
    entity: sensor.heiman_temperature
    name: Temperature
    unit: "°C"
    min: 0
    max: 50
    severity:
      green: 0
      yellow: 30
      red: 40
  - type: gauge
    entity: sensor.heiman_humidity
    name: Humidity
    unit: "%"
    min: 0
    max: 100
    severity:
      green: 30
      yellow: 60
      red: 80
```

## Alarm History Panel

```yaml
type: history-graph
title: Sensor History
entities:
  - binary_sensor.heiman_smoke_alarm
  - binary_sensor.heiman_co_alarm
  - binary_sensor.heiman_water_leak
hours_to_show: 24
```

## Recommended Layout

- Top: Security system status + alarm device status cards
- Middle: Environment monitoring (temperature, humidity, water)
- Bottom: Sensor alarm history trend graph
- Sidebar quick actions: Arm/disarm, scene switching
