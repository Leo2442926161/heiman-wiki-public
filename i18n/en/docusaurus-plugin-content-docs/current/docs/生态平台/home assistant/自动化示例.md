# Automation Examples

## Smoke Alarm Triggers Whole-House Lights

```yaml
alias: "Smoke Alarm - Light Up All Lights & Notify"
triggers:
  - trigger: state
    entity_id: binary_sensor.heiman_smoke_alarm
    to: "on"
actions:
  - action: light.turn_on
    target:
      entity_id: all
  - action: notify.mobile_app_phone
    data:
      message: "⚠️ Smoke alarm triggered! Please check immediately!"
      title: "Smoke Alarm"
```

## CO Alarm Automation

```yaml
alias: "CO Alarm - Shut Off Gas & Open Window"
triggers:
  - trigger: state
    entity_id: binary_sensor.heiman_co_alarm
    to: "on"
actions:
  - action: switch.turn_off
    entity_id: switch.gas_fireplace
  - action: cover.open
    entity_id: cover.living_room_window
  - action: notify.persistent_notification
    data:
      message: "Carbon monoxide alarm triggered, gas valve closed and window opened"
```

## Water Leak Detector Automation

```yaml
alias: "Water Leak - Shut Off Water Valve"
triggers:
  - trigger: state
    entity_id: binary_sensor.heiman_water_leak
    to: "on"
actions:
  - action: switch.turn_off
    entity_id: switch.main_water_valve
  - action: notify.mobile_app_phone
    data:
      message: "💧 Water leak detected! Main water valve closed!"
```

## Away Arm - Motion Sensor Triggered Alarm

```yaml
alias: "Away Mode - Motion Sensor Alarm"
triggers:
  - trigger: state
    entity_id: binary_sensor.heiman_motion_sensor
    to: "on"
conditions:
  - condition: state
    entity_id: alarm_control_panel.home_alarm
    state: armed_away
actions:
  - action: alarm_control_panel.trigger
    entity_id: alarm_control_panel.home_alarm
```
