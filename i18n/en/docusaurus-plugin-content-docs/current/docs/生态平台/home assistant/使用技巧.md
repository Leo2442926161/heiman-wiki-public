# Tips & Tricks

## Device Configuration

- When pairing Heiman Matter devices for the first time, ensure the border router (e.g., HomePod, Nest Hub, Home Assistant Green) is on the same Thread network
- For Zigbee devices integrated via ZHA or Zigbee2MQTT, it's recommended to use the latest firmware on your USB coordinator (e.g., Sonoff ZBDongle-E, Conbee II)

## Automation Optimization

- Add a `for:` parameter (e.g., 3 seconds) to smoke alarm automations to prevent transient false triggers
- Use `condition: state` with the alarm mode to avoid unnecessary notifications when armed
- Use Home Assistant's `group` feature to organize Heiman sensors by area for batch operations

## Troubleshooting

- If a device goes offline, first check whether the Zigbee channel conflicts with WiFi 2.4GHz
- If Matter devices are not discoverable, restart the border router and re-add the device
