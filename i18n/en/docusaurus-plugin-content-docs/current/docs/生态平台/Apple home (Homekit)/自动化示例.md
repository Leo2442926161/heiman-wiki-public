# Automation Examples

## Smoke Detection Security Alarm

**HomeKit Automation Configuration:**

1. **Trigger:** Heiman smoke alarm → smoke detected
2. **Security Setting:** Switch security system state to "Alarm"
3. **Actions:**
   - HomePod → play custom audio "Smoke detected, please stay safe"
   - All smart lights → turn on and flash
   - Push notification to all family members

## Home/Away Scenes

**Shortcut: "Away Security Check"**

```
When the last person leaves home
   Set security system → away arm
   Set Heiman door/window sensor → enable notifications
   Set Heiman motion sensor → enable alarm
   Turn off all lights
   Set thermostat → eco mode
   Send notification "🏠 Away mode activated"
```

## Temperature Anomaly Alert

**Trigger:** Heiman temperature/humidity sensor → temperature > 35°C

**Actions:**
- HomePod → voice announcement of high temperature
- Push notification to iPhone
- Automatically turn on AC (if HomeBridge is configured)
