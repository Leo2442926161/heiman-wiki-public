# Tips & Tricks

## Homebridge vs Matter

Heiman devices can be connected to Apple Home in two ways:

| Method | Advantages | Disadvantages |
|------|------|------|
| **Matter over Thread** | Native support, no bridge needed, low latency | Requires Thread border router |
| **Homebridge Plugin** | Compatible with all Zigbee devices | Requires running a separate Homebridge service |

## Thread Network Setup

1. At least one Thread border router is needed (Apple HomePod mini, Apple TV 4K 3rd gen)
2. Heiman Matter devices will automatically join the Thread network within 10 meters
3. Use **Nam** or **Eve App** to view the Thread network topology

## Automation Optimization

- Apple Home's "Security" category automations can directly link to security system status
- Use "Transition" commands for smooth light changes instead of instant switching
- Create cross-platform automations via Shortcuts (e.g., smoke alarm → send SMS)

## Common Issues

- If a Matter device shows "No Response," restart the Apple TV/HomePod to recover
