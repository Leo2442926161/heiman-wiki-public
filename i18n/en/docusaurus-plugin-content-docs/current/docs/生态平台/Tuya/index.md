# Tuya (Tuya Smart)

Tuya Smart is a leading global IoT cloud platform, and Heiman is one of its key ecosystem partners. Through the Tuya ecosystem, Heiman devices can quickly integrate with smart home systems from 1000+ brands.

## Integration Methods

### Smart Life App

Heiman Zigbee devices can be configured and controlled via Tuya's Smart Life App:

1. Prepare a Tuya-compatible Zigbee gateway (e.g., Heiman HS6GW series)
2. Open Smart Life App → Add Device
3. Select Zigbee gateway → Search for sub-devices
4. Put the sensor into pairing mode and it will be discovered automatically

### Cross-Platform Support

Through the Tuya ecosystem, Heiman devices can work with:

- Amazon Alexa
- Google Assistant
- Apple Siri Shortcuts
- IFTTT

### Local Tuya Gateway

Using Tuya Local (e.g., Home Assistant's Tuya Local integration), Heiman devices can be controlled locally, reducing cloud latency.

## Scope of Products

Tuya-compatible Heiman product lines include: smoke sensors, carbon monoxide sensors, door/window sensors, PIR motion sensors, temperature/humidity sensors, water leak detectors, smart plugs, and other Zigbee devices.
