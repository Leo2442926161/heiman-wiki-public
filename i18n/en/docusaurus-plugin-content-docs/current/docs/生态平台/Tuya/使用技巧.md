# Tips & Tricks

## Tap-to-Run vs Automation

| Type | Trigger Method | Use Case |
|------|----------|----------|
| **Tap-to-Run** | Manual tap / voice | Away, home, movie mode, etc. |
| **Automation** | Auto-triggered by sensors | Smoke alarm, water leak detection, etc. |

Heiman sensor-type devices should use "Automation," while scene-based needs should use "Tap-to-Run."

## Cloud to Local

- Enable "LAN Communication" in Smart Life App to reduce cloud dependency
- Some Heiman devices support local execution and can still run automations even when offline
- Settings path: Scene → top-right settings → "Enable Local Execution"

## Device Sharing

- Share Heiman devices with family members in Smart Life App
- Shared users can receive alarm notifications but cannot modify device settings
- Up to 20 family members can be shared

## Third-Party Platform Integration

Your Tuya account can be linked to Alexa and Google Home for cross-platform voice control. Link path: Smart Life → Settings → Third-Party Control.
