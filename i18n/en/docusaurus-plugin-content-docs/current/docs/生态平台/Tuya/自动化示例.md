# Automation Examples

## Smoke Alarm Scene

**Scene Name:** "Smoke Alarm Automation"

**Trigger:** Heiman smoke alarm → smoke detected

**Actions:**
- Smart plug → turn on (connected to exhaust fan)
- Lights → turn on and flash
- Push notification to phone
- Speaker → play alarm sound

## One-Tap Away Scene (Tap-to-Run)

**Scene Name:** "Away Arm"

**Manual Trigger:** Tap or voice command

**Actions:**
- Heiman door/window sensor → enable alarm
- Heiman PIR motion sensor → enable detection
- Heiman water leak detector → enable detection
- All lights → turn off
- Smart lock → lock

## Temperature & Humidity Auto Control

**Automation Type:** Automatic condition trigger

**Trigger:** Heiman temperature/humidity sensor humidity > 75%

**Actions:**
- Smart plug (dehumidifier) → turn on
- Push notification: "Humidity too high, dehumidifier turned on"

**Trigger:** Heiman temperature/humidity sensor humidity < 50%

**Actions:**
- Smart plug (dehumidifier) → turn off
