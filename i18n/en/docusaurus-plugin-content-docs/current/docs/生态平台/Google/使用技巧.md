# Tips & Tricks

## Device Sync

- After adding Heiman devices via Matter to Google Home, they will sync automatically to the Google Home App
- If a device does not appear, go to Google Home App → "Settings" → "Devices" → "Add Device" → "Compatible Matter device"
- If the Matter QR code is lost, use the 11-digit setup code on the device to add it manually

## Routine Optimization

- Add "broadcast weather and confirm arm status" to your "Away" routine
- Use the "voice broadcast" feature for whole-home notification during smoke alarms
- Use "start/stop" actions to enable or disable Heiman sensor detection states

## Device Location Management

- Assign Heiman sensors to specific rooms (e.g., "Kitchen Smoke Alarm") for more precise voice control
- Group multiple Heiman devices into "Home Device Groups" for one-touch control

## Network Requirements

- Matter over Thread devices require a Google Nest Hub (2nd gen) or Nest Hub Max as a border router
- Zigbee devices connect via Google Nest Connect or a compatible hub
