# Automation Examples

## "I'm Leaving" Routine

**Trigger:** Say "I'm leaving" to Google Home

**Actions:**
- Heiman door/window sensor → arm
- Heiman PIR motion sensor → enable away detection
- Turn off all lights and appliances
- Thermostat → set to eco temperature
- Google Nest Hub → show away confirmation card
- Voice reply: "Away mode activated, have a safe trip"

## "Smoke Detected" Broadcast

**Trigger:** Heiman smoke alarm → smoke detected

**Actions:**
- Google Nest speakers → whole-home broadcast "⚠️ Smoke detected! Please check immediately!"
- All smart lights → turn on (via Google Home smart light group)
- Push notification to family members' phones
- Cameras → start recording

## Bedtime Mode

**Trigger:** Say "Good night" to Google Home

**Actions:**
- Heiman door/window sensor → enable door-open alarm
- Heiman PIR motion sensor → enable night detection
- Lights → gradually fade off
- Voice reply: "Good night, security system is now active"
