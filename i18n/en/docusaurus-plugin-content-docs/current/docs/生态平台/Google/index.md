# Google Home

Google Home is Google's smart home platform, supporting the Matter protocol and a broad device ecosystem.

## Integration Methods

### Matter Devices

Heiman Matter over Thread devices can be added directly to the Google Home App:

1. Ensure you have a Thread border router (Google Nest Hub Max, Nest Hub 2nd gen, etc.)
2. Open the Google Home App
3. Tap "Add" → "Compatible Matter device"
4. Scan the Matter pairing code on the device

### Voice Control

Once connected, you can use Google Assistant voice commands:

- "Hey Google, what's the smoke alarm status?"
- "Hey Google, what's the temperature in the living room?"

> **Reference**: Heiman Matter devices are listed as supporting Google Home on Amazon
