# Tips & Tricks

## Device Discovery

- When a Heiman Matter device is powered on, the Alexa App typically discovers it automatically within 30 seconds
- If not auto-discovered: go to Alexa App → Devices → "+" → "Add Device" → "Matter"
- Use the Matter setup code or QR code on the bottom of the device to complete pairing

## Zigbee Device Integration (Echo Plus / Echo Show)

- Echo devices with a built-in Zigbee hub can pair directly with Heiman Zigbee sensors
- Pairing mode: Alexa App → Devices → "+" → "Zigbee" → "Search for devices"
- Heiman devices are compatible with Alexa's built-in Zigbee coordinator

## Routine Tips

- Use "Custom Action" for more detailed voice prompts
- Add a "Wait 5 seconds" step in emergency routines to avoid voice overlap
- Use the "Sound Detection" feature so Alexa can respond to Heiman alarm high-pitch alerts

## Alexa Guard Notes

Alexa Guard Plus is currently only available in the US region. Users in other regions can simulate some features through Routines.
