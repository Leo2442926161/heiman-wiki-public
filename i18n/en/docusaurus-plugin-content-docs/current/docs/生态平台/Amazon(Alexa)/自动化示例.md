# Automation Examples

## Alexa Guard Integration (US Region)

**Prerequisite:** Subscribe to Alexa Guard Plus (available in US region)

**Setup Process:**
1. Enable Guard Plus in the Alexa App
2. Add Heiman smoke/CO alarms as "Smart Home Devices"
3. Enable "Smoke Alarm Detection" smart response

**Automatic Response:**
- Smoke alarm triggered → Alexa automatically calls emergency contacts
- CO alarm triggered → Alexa broadcasts voice warning throughout the home, auto-ventilates
- Door/window abnormally opened → push alarm notification

## Smoke Detection Routine

**Trigger:** Heiman smoke alarm → smoke detected

**Actions:**
- Alexa voice announcement: "⚠️ Smoke detected, please stay safe"
- All Echo devices → play alarm sound
- Smart light group → turn all on
- Push notification to phone

## "Away" Routine

**Trigger:** Voice command "Alexa, I'm leaving"

**Actions:**
- Heiman door/window sensor → arm
- Heiman motion sensor → enable movement detection
- Heiman water leak detector → enable detection
- All lights → turn off
- Alexa replies: "Security system activated, have a safe trip"
