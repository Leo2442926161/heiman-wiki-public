# Amazon Alexa

Amazon Alexa is one of the world's most popular smart voice assistant platforms, supporting the Matter protocol and a rich Skills ecosystem.

## Integration Methods

### Matter Devices

Heiman Matter over Thread devices can be connected via Matter-compatible Echo devices (Echo 4th gen, Echo Plus, Echo Show 10, etc.):

1. Ensure your Echo device is updated to the latest firmware
2. Open the Alexa App → Devices → Add Device
3. Select "Matter" and scan the pairing code

### Zigbee Devices

Some Echo Plus and Echo Show devices have a built-in Zigbee chip and can pair directly with Heiman Zigbee sensors.

## Voice Command Examples

- "Alexa, ask the smoke detector if everything is OK"
- "Alexa, what's the temperature in the living room?"

> **Reference**: Heiman Matter sensors are available on the Amazon official store
