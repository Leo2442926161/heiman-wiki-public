# Homey

Homey is a smart home hub by Athom, a Dutch company, supporting Zigbee, Z-Wave, Wi-Fi, Matter, Thread, and more.

## Integration Methods

Heiman Zigbee and Z-Wave devices can pair directly with Homey's built-in wireless chip, no additional gateway required. Matter devices can be connected via Homey Pro's built-in Thread border router.

## Advantages

- **Multi-protocol all-in-one**: Homey Pro supports Zigbee, Z-Wave, Thread, and Wi-Fi simultaneously
- **Local execution**: All automations run locally
- **Flow automations**: Drag-and-drop programming for creating complex automation scenes

> **Reference**: Search for Heiman-related apps in the Homey App Store
