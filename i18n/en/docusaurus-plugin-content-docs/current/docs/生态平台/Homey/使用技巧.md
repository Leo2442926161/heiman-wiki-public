# Tips & Tricks

## Zigbee Channel Interference

Homey's built-in Zigbee chip uses a specific channel. If Heiman devices frequently drop off:

1. Check the current channel in Homey Settings → Zigbee
2. Use the Homey Insight App to scan for environmental interference
3. Switch to a less congested channel (away from WiFi 1/6/11)

## Device Pairing

- To put Heiman devices into pairing mode: press and hold the button for 5 seconds (some models require 3 quick presses)
- When adding a device in the Homey App, select "Add Device → Zigbee → Search"
- If pairing fails, move the device closer to Homey (within 3 meters) and try again

## Flow Optimization Tips

- Add a "delay" condition in your Flow (e.g., 2 seconds) to avoid transient false triggers
- Use the "tag" feature to categorize and manage Heiman devices

## Signal Range Extension

If Heiman devices are far from Homey, add Zigbee router devices (e.g., smart plugs) to extend network coverage.
