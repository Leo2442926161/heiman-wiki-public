# Automation Examples

## Smoke Alarm - Flash Lights & Notify

**Flow Configuration:**

1. **Trigger:** Heiman Smoke Alarm → state changes to `alert`
2. **Condition:** (none)
3. **Actions:**
   - Living room lights → flash (red, once per second)
   - Push notification → phone: "⚠️ Smoke alarm triggered!"
   - Voice announcement → Homey Speaker: "Smoke detected, please stay safe!"

## CO Alarm - Shut Off Boiler

**Flow Configuration:**

1. **Trigger:** Heiman CO Alarm → state changes to `alert`
2. **Condition:** (none)
3. **Actions:**
   - Boiler switch → turn off
   - Smart plug (ventilation system) → turn on
   - Push notification → "Carbon monoxide alarm, boiler has been shut off"

## Temperature & Humidity Alert

1. **Trigger:** Heiman temperature sensor → temperature above 40°C
2. **Actions:**
   - Air conditioner → turn on cooling mode
   - Push notification → "Room temperature too high, AC turned on automatically"
