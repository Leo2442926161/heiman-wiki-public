# Tips & Tricks

## Channel Selection

Heiman Zigbee devices use channel 15 by default. It is recommended to specify it manually in `configuration.yaml`:

```yaml
advanced:
  channel: 15
```

Avoid overlap with WiFi 2.4GHz channels 1/6/11. Use a WiFi analyzer to scan and select the Zigbee channel with the least interference.

## Network Key Backup

**Always back up `coordinator_backup.txt` and `secret.yaml`:**

```bash
cp data/coordinator_backup.txt data/coordinator_backup.bak
cp data/secret.yaml data/secret.yaml.bak
```

If the keys are lost, all paired devices will need to be re-joined.

## Device-Specific Optimization

- Heiman devices report battery level every 60 minutes by default; use the `reporting` configuration to shorten the interval
- Add `homeassistant: true` to automatically generate entities without manually writing MQTT discovery config
- Use `friendly_name` to set readable names for Heiman devices

## Firmware Updates

Some Heiman Zigbee devices support OTA updates. Enable the `ota` option in `configuration.yaml`.
