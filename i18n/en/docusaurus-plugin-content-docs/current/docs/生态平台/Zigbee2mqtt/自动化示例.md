# Automation Examples

## MQTT Subscribe to Device Status

```bash
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
```

Example output:
```
zigbee2mqtt/heiman_smoke  {"smoke":true,"battery":100,"linkquality":120}
```

## Device Online Monitoring

```yaml
# configuration.yaml
binary_sensor:
  - platform: mqtt
    name: "Smoke Alarm Online"
    state_topic: "zigbee2mqtt/heiman_smoke/availability"
    payload_on: "online"
    payload_off: "offline"
    device_class: connectivity
```
