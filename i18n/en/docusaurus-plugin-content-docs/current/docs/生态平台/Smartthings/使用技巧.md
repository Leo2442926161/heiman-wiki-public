# Tips & Tricks

## Hub Compatibility

Heiman Zigbee devices are compatible with **SmartThings Hub v2** and **Hub v3 (Aeotec)**.

| Hub Model | Compatibility | Notes |
|---------|--------|------|
| Hub v2 (STH-ETH-250) | ✅ | Requires latest firmware |
| Hub v3 (Aeotec) | ✅ | Natively supported, recommended |
| Samsung SmartThings Station | ⚠️ | Only supports select Zigbee devices |

## Device Pairing Tips

- In the SmartThings App, tap "+" → "Device" → "Brand" → "Heiman"
- For devices not automatically recognized, use "Scan Zigbee" for manual discovery
- Keep devices within 5 meters of the Hub during initial pairing

## Battery Optimization

Heiman sensors have a longer default reporting interval. For real-time monitoring, adjust the "Detection Interval" to 5 minutes in the device settings.

## Automation Best Practices

- Use SmartThings' "Mode" feature to distinguish between "Home/Away" states
- Create **two** automations for critical sensors (smoke, CO) as redundant safeguards
