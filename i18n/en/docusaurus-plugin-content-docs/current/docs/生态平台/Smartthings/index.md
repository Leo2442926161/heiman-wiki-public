# SmartThings

SmartThings is Samsung's smart home platform, supporting Zigbee, Z-Wave, and Matter protocols.

## Integration Methods

Heiman devices can be connected to SmartThings in the following ways:

1. **Matter devices**: Add directly via SmartThings Hub or Matter-compatible Samsung devices (e.g., Family Hub refrigerator, SmartThings Station)
2. **Zigbee devices**: SmartThings Hub v2/v3 has a built-in Zigbee chip for direct connection to Heiman Zigbee sensors
3. **Z-Wave devices**: SmartThings Hub supports Z-Wave; Heiman Z-Wave devices can pair directly

## Certification Status

Heiman Matter PIR motion sensors are already marked as supporting SmartThings on Amazon and other channels, with positive user reviews.

> **Reference**: [SmartThings Compatible Devices](https://www.smartthings.com/products)
