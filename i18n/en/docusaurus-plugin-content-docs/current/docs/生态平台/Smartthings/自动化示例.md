# Automation Examples

## "Good Night" Bedtime Mode

**Trigger:** Heiman door/window sensor → closed (bedroom door secured)

**Actions:**
- Turn off all living room lights
- Smart lock → lock
- Security system → night arm
- Push notification: "Good night, system is armed"

## "Away" Mode

**Trigger:** Heiman PIR motion sensor → no movement for 15 minutes

**Actions:**
- All lights off
- Thermostat → eco mode
- Security system → away arm
- Door/window sensor → enable alarm
- Push notification: "Away mode activated, door and window sensors enabled"

## Smoke Alarm Emergency Mode

**Trigger:** Heiman smoke alarm → smoke detected

**Actions:**
- Smart speaker → play alarm sound
- All smart lights → flash red
- Push notification to all family members
- If connected to SmartThings Hub → trigger siren
