# Automation Examples

## Smoke Alarm Rule

```javascript
rule "Smoke Alarm Triggered"
when
    Item heiman_smoke_alarm changed to ON
then
    sendCommand(Livingroom_Light, ON)
    sendNotification("admin", "⚠️ Smoke alarm triggered!")
    postUpdate(Alarm_State, "smoke_alert")
end
```

## Temperature Sensor Rule

```javascript
rule "Temperature & Humidity Report"
when
    Item heiman_temperature received update
then
    val temp = (heiman_temperature.state as Number).floatValue
    val hum = (heiman_humidity.state as Number).floatValue
    if (temp > 40) {
        sendCommand(AirConditioner, ON)
        sendNotification("admin", "Temperature too high (" + temp + "°C), AC turned on")
    }
    if (hum > 80) {
        sendNotification("admin", "Humidity too high (" + hum + "%), please dehumidify")
    }
end
```

## Water Leak Detection Rule

```javascript
rule "Water Leak Detection Automation"
when
    Item heiman_water_leak changed to ON
then
    sendCommand(Water_Valve, OFF)
    sendBroadcastNotification("💧 Water leak detected! Water valve closed!")
end
```
