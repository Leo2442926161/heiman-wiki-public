# Tips & Tricks

## Thing Configuration

Add Heiman devices via the MQTT binding:

```java
Thing mqtt:topic:heiman_smoke "Heiman Smoke Alarm" @ "Living Room" {
    Channels:
        Type binary : smoke "Smoke Status" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.smoke" ]
        Type number : battery "Battery Level" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.battery" ]
}
```

## Item Definition

```java
Contact Smoke_Alarm "Smoke Alarm [MAP(smoke.map):%s]" { channel="mqtt:topic:heiman_smoke:smoke" }
Number Smoke_Battery "Smoke Alarm Battery [%d%%]" { channel="mqtt:topic:heiman_smoke:battery" }
```

## Coordinator Selection

It is recommended to use USB coordinators with **CC2531** or **CC2652** chips, ensuring the firmware is up to date.

## Debugging Tips

- Check the Thing's `online/offline` status in Paper UI
- Use `openhab-cli info` to verify bindings are loaded correctly
- Search for `heiman` in the logs to troubleshoot device communication issues
