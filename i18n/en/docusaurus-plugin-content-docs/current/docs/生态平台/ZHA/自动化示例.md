# Automation Examples

## Smoke Alarm Automation

```yaml
alias: "ZHA Smoke Alarm Automation"
triggers:
  - trigger: state
    entity_id: binary_sensor.heiman_smoke_zha
    to: "on"
actions:
  - action: light.turn_on
    target:
      area_id: living_room
  - action: tts.speak
    target:
      entity_id: media_player.speaker
    data:
      message: "Smoke alarm triggered!"
```

## Device Pairing Automation

```yaml
alias: "New Device Pairing Notification"
triggers:
  - trigger: zha.event
    subtype: heiman
    command: device_joined
actions:
  - action: notify.persistent_notification
    data:
      message: "New Heiman device has joined the ZHA network. Go configure its name and area."
```

## Water Leak Detection

```yaml
alias: "ZHA Water Leak Detection"
triggers:
  - trigger: state
    entity_id: binary_sensor.heiman_water_leak_zha
    to: "on"
actions:
  - action: switch.turn_off
    entity_id: switch.main_water_valve
  - action: notify.mobile_app_phone
    data:
      message: "💧 Water leak detected!"
```
