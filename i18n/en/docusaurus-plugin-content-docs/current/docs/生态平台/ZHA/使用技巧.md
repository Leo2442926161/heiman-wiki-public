# Tips & Tricks

## Device Pairing

1. In Home Assistant, go to "Settings → Devices & Services → ZHA → Configure"
2. Click "Add Device," then activate pairing mode on your Heiman device
3. If pairing fails, move the device closer to the USB coordinator (within 3 meters) and retry

## Channel Coordination

ZHA uses channel 15 by default. If there is severe interference with WiFi 2.4GHz:
- Use `Silicon Labs Flasher Utility` to change the channel
- Recommended channels: 15, 20, 25 (avoid WiFi 1/6/11)

## Network Optimization

- Add a Zigbee router device (e.g., a smart plug) every 3-4 Heiman devices to extend the network
- Use ZHA's network visualization tool to check device connection quality
- Regularly check for device firmware updates to maintain optimal compatibility
