# Automation Examples

## Rule Machine Smoke Alarm

**Rule Name:** Smoke Alarm Safety Automation

**Configuration:**
```
Trigger: Heiman smoke alarm becomes active
Condition: Mode != disarmed
Actions:
  - Turn on all lights
  - Send push notification "⚠️ Smoke alarm!"
  - Activate siren (if available)
  - Set mode to "Emergency"
```

## Rule Machine Temperature Monitoring

**Rule Name:** High Temperature Protection

**Configuration:**
```
Trigger: Heiman temperature sensor > 38°C
Condition: AC status = off
Actions:
  - Turn on AC, cooling mode, 24°C
  - Send notification "Room temperature too high, AC started"
```

## Rule Machine Water Leak Detection

**Rule Name:** Water Leak Emergency Response

**Configuration:**
```
Trigger: Heiman water leak detector becomes wet
Condition: Water valve status = open
Actions:
  - Close main water valve
  - Send push notification to all phones
  - Send SMS notification (optional)
```
