# Tips & Tricks

## Custom Drivers

For best compatibility with Heiman devices in Hubitat, use the following drivers:

| Device Type | Recommended Driver |
|----------|----------|
| Smoke/CO Alarm | Generic Zigbee Smoke/CO Sensor |
| PIR Motion Sensor | Heiman Motion Sensor (3rd party) |
| Door/Window Sensor | Generic Zigbee Contact Sensor |
| Water Leak Detector | Generic Zigbee Water Leak Sensor |
| Temperature/Humidity Sensor | Heiman Temperature/Humidity Sensor |

## Zigbee Optimization

- Hubitat C-8 and above use the EFR32 chip for best compatibility
- Check and select the least congested channel in Settings → Zigbee
- Use Zigbee sniffer tools (e.g., Ubisys) to diagnose drop-off issues

## Rule Tips

- Add a "false trigger delay" condition (0.5-2 seconds) in Rule Machine
- Enable "Health Check" notifications for critical rules to ensure they keep running
- Organize complex automations using "sub-rules" for easier debugging and maintenance

## Z-Wave Coexistence

If using Z-Wave devices simultaneously, maintain sufficient frequency spacing between Zigbee and Z-Wave channels.
