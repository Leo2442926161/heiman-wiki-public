# Hubitat

Hubitat is a smart home hub focused on local control, supporting Zigbee and Z-Wave devices. All automations run locally with no cloud dependency.

## Integration Methods

### Zigbee Devices

Heiman Zigbee sensors can pair directly with the Hubitat Elevation hub:

1. Put Hubitat into Zigbee pairing mode
2. Put the Heiman sensor into pairing mode
3. The device will be automatically discovered and added

### Z-Wave Devices

Heiman Z-Wave devices are also supported for direct pairing with Hubitat using its built-in Z-Wave chip.

### Custom Drivers

Some Heiman devices may require custom drivers. Community users have already written compatible drivers for common sensors.

## Advantages

- **100% Local**: All automation logic runs locally on Hubitat, no cloud dependency
- **Low Latency**: Local Zigbee/Z-Wave mesh network for fast response
- **Rich Community**: Hubitat Community has a wealth of configuration sharing from Heiman users

> **Reference**: [Hubitat Community - Heiman Device Discussion](https://community.hubitat.com/)
