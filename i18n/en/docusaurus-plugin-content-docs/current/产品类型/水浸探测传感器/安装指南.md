# Installation Guide

## Installation Location Requirements

- **Place on the floor**, ensure probes contact the ground
- **Near water sources**: under bathroom vanities, under kitchen sinks, next to washing machines, below water heaters
- Position at the lowest point where leaks need to be detected

## Installation Steps

1. Determine installation location (floor area with water leak risk)
2. Clean the floor surface
3. Place the sensor on the floor, probes contacting the ground
4. If using the cable version (HS2WL), lay the extension cable along the area to be monitored
5. Insert battery, sensor powers on automatically
6. Simulation test: touch the probes with a wet finger, alarm LED lights up
7. Pair with gateway

## Precautions

- Probes in prolonged contact with humid environments may corrode, inspect periodically
- Recommended for use with solenoid valves for automatic shutoff
- Battery life approximately 2-3 years
- Multiple sensors can cover different areas
