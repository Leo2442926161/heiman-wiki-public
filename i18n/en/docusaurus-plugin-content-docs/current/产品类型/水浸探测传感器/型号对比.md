# Model Comparison

## Model Comparison

| Model | Protocol | Detection Method | Power | Features | Recommended Use |
|-------|----------|----------------|-------|----------|----------------|
| [HS1WL](../../无线协议/Zigbee/HS1WL.md) | Zigbee | Probe | Battery | Basic model | Alarm push on leak detection, suitable under bathroom vanity |
| [HS2WL](../../无线协议/Zigbee/HS2WL.md) | Zigbee | Probe + Cable | Battery | Dual channel | Probe detects point leaks + cable covers long areas, ideal next to water heaters/washing machines |
| [HS3WL](../../无线协议/Zigbee/HS3WL.md) | Zigbee | Probe | Battery | Enhanced model | Higher sensitivity, louder buzzer |
| [L1-M](../../无线协议/Matter/L1-M.md) | Matter over Thread | Probe | Battery | Matter version | WWHA certified, native Apple Home integration |

## Usage Scenarios

| Scenario | Recommended Model | Automation Scheme |
|----------|------------------|-------------------|
| Whole-home leak monitoring | HS2WL + L1-M | Multiple point coverage + native HomeKit notification, detect leak → close main valve → push notification |
| Washing machine dedicated | HS2WL | Cable probe laid under washing machine, immediate alarm on leak + shut off washing machine power |
| Water heater/floor heating | HS3WL | Enhanced high sensitivity, detects underfloor heating pipe leaks at an early stage |
| Matter users | L1-M | No dedicated gateway needed, Thread low-latency alarm + Apple Home automation |

## Selection Guide

- **Point detection**: [HS1WL](../../无线协议/Zigbee/HS1WL.md) or [L1-M](../../无线协议/Matter/L1-M.md)
- **Large area coverage**: [HS2WL](../../无线协议/Zigbee/HS2WL.md) (probe + cable)
- **Matter required**: [L1-M](../../无线协议/Matter/L1-M.md)
