# Model Comparison

## Model Comparison

| Model | Protocol | Detection Angle | Mounting | Features | Recommended Use |
|-------|----------|---------------|----------|----------|----------------|
| [HS1MS](../../无线协议/Zigbee/HS1MS.md) | Zigbee | 110° | Wall | Basic, economical | Light on/off with motion, security automation push alert on movement |
| [HS3MS](../../无线协议/Zigbee/HS3MS.md) | Zigbee | 360° | Ceiling | Full coverage | Ceiling mount for full 360° coverage without死角, ideal for living rooms/halls |
| [M1-Z](../../无线协议/Zigbee/M1-Z.md) | Zigbee | 110° | Wall | Light level detection | PIR + light sensor: no light trigger during daytime, auto-light at night |
| [M1-M](../../无线协议/Matter/M1-M.md) | Matter over Thread | 110° | Wall | Matter version | Native HA + Apple Home, light level auto-adjusts curtains |
| [HS8MLS](../../无线协议/Zigbee/HS8MLS.md) | Zigbee | 110° | Wall | Long range | Corridors/garages with long straight areas, longer detection distance |
| [HS8MIS](../../无线协议/Zigbee/HS8MIS.md) | Zigbee | 110° | Wall | Composite sensor | PIR + light level 2-in-1, smart light auto-adjusts brightness based on ambient light |

## Usage Scenarios

| Scenario | Recommended Model | Automation Scheme |
|----------|------------------|-------------------|
| Energy-saving auto light | M1-Z / HS8MIS | PIR detects person → turn on light + read light level to determine brightness → auto off after timeout |
| Security arm automation | HS1MS | Away mode detects movement → push alert + siren + camera recording |
| 360° ceiling coverage | HS3MS | Center ceiling mount indoors, 360° no blind spots, suitable for open spaces |
| Matter cross-ecosystem | M1-M | Apple Home automation + Google/Alexa voice, light level curtain automation |
| Long hallway/garage | HS8MLS | Long-distance detection, automatic lighting for vehicle/personnel entry/exit |

## Selection Guide

- **Corridor/room wall mount**: [HS1MS](../../无线协议/Zigbee/HS1MS.md) or [HS8MLS](../../无线协议/Zigbee/HS8MLS.md) (long range)
- **Ceiling full-room coverage**: [HS3MS](../../无线协议/Zigbee/HS3MS.md) (360°)
- **Light level automation needed**: [M1 Series](../../无线协议/Zigbee/M1-Z.md) or [HS8MIS](../../无线协议/Zigbee/HS8MIS.md)
- **Matter platform**: [M1-M](../../无线协议/Matter/M1-M.md)
