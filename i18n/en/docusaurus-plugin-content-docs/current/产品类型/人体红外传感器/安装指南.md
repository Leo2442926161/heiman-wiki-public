# Installation Guide

## Installation Location Requirements

- Wall-mount installation height: **1.8 ~ 2.4 m**
- Ceiling installation: mount on the center of the room ceiling
- **Avoid**: direct sunlight, near heaters/AC vents, heat sources
- Ensure no obstructions (furniture, glass partitions) within the detection range

## Installation Steps

1. Select installation location (wall-mount or ceiling-mount)
2. Secure the mounting base with screws
3. Insert battery, LED flashes
4. Mount the sensor body onto the base
5. Adjust detection angle (within 110° horizontal range)
6. Walk test: walk through the detection area, the LED should light up
7. Pair with gateway

## Precautions

- PIR sensors are sensitive to temperature changes; detection distance may decrease in high-temperature summer environments
- Avoid facing windows directly (outdoor temperature changes may cause false triggers)
- Battery life approximately 3 years (Zigbee) / 1 year (Matter)
