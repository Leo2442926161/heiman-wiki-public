# PIR Motion Sensor

PIR (Passive Infrared) motion sensors detect movement by sensing infrared heat radiation emitted by the human body, making them the most commonly used presence detection solution in smart homes.

## Product Series

| Series | Communication | Features |
|--------|--------------|----------|
| HS1MS | Zigbee | Basic model, 110° detection angle |
| HS3MS | Zigbee | Upgraded model, 360° ceiling mount |
| M1 Series | Matter / Zigbee | Supports Matter over Thread, available in 110° or 360° |
| HS8MLS | Zigbee | Long-range model |
| HS8MIS | Zigbee | PIR + light level composite sensor |

## Core Features

- Moving human body detection
- Light level detection (M1/HS8MIS series)
- Detection distance ≤7m (25°C)
- Installation: wall-mount or ceiling-mount
- Battery life: 3 years (Zigbee) / 1 year (Matter)

## Technical Specifications (M1 Series)

- Detection Angle: 110° / 360°
- Detection Distance: ≤7m
- Light Level Range: 0~1200 Lux
- Power: DC 3V (Lithium battery)
- Battery Life: 3 years (Zigbee) / 1 year (Matter)

## Scenario Examples

- Light on when motion detected, off when no motion
- Security automation: push alarm on unusual movement
- Away arm: auto-turn off AC/heating when unoccupied to save energy
