# Installation Guide

## Installation Location Requirements

- Install at **1.2 ~ 1.5 m** height (represents average ambient conditions)
- **Keep away from**: direct sunlight, doors/windows, AC vents, radiators
- **Keep away from**: humidifiers, kitchen stoves, and other local temperature/humidity change sources
- Choose frequently used rooms (bedroom, living room, baby room)

## Installation Steps

1. Select installation location (wall or tabletop both acceptable)
2. Wall mounting: secure the base with screws or 3M adhesive
3. Insert CR2 battery (observe polarity)
4. Sensor powers on automatically, display (if equipped) shows temperature and humidity
5. Pair with gateway
6. Confirm data is normal in the app

## Precautions

- When mounting on a wall, ensure sensor ventilation holes are not blocked
- Battery life approximately 2-3 years
- Temperature and humidity data from different rooms can be collected using multiple sensors
- Can be used to automate AC, dehumidifier, humidifier, and other devices
