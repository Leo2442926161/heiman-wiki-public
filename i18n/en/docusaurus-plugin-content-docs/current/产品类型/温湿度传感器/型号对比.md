# Model Comparison

## Model Comparison

| Model | Protocol | Accuracy | Power | Features | Recommended Use |
|-------|----------|----------|-------|----------|----------------|
| [H1-M](../../无线协议/Matter/H1-M.md) | Matter over Thread | ±0.3°C / ±3% RH | CR2 | Matter version, cross-platform | Native Apple Home, auto-regulate room temperature with AC/floor heating, auto-turn on exhaust fan when humidity is high |
| H1-Z | Zigbee | ±0.3°C / ±3% RH | CR2 | Zigbee version | Connect to HA/Z2M, temperature anomaly alert (notify if elderly home gets too cold/hot) |

## Usage Scenarios

| Scenario | Recommended Model | Automation Scheme |
|----------|------------------|-------------------|
| Apple Home ecosystem | H1-M | Native Matter integration, HomeKit automation: temp > 28°C turn on AC, humidity > 70% turn on exhaust |
| HA power users | H1-Z or H1-M | HA automation: temp/humidity control floor heating/humidifier/dehumidifier, historical data with InfluxDB + Grafana |
| Wine cellar/cigar cabinet | H1-Z | Temp/humidity out-of-range alert + phone push, protect collection items |
| Baby room monitoring | H1-M | Push notification if temp too high/low, auto-adjust AC to comfortable range |
| Plant greenhouse | H1-Z | Temp/humidity triggers auto-watering/ventilation/shade curtain |

## Selection Guide

- **Matter cross-platform**: [H1-M](../../无线协议/Matter/H1-M.md)
- **Existing Zigbee system**: H1-Z
