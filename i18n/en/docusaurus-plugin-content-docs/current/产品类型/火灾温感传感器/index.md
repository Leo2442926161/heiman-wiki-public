# Heat Detector

Heat detectors (temperature sensors) detect abnormal temperature rises in the environment to trigger fire alarms, making them suitable for kitchens, garages, and other scenarios where smoke sensors are prone to false alarms.

## Operating Principle

Heat detectors use thermal sensing elements to detect temperature changes:

- **Fixed Temperature Alarm**: triggers when the temperature exceeds a set threshold (typically 57°C / 135°F)
- **Rate-of-Rise Alarm**: triggers when the temperature rise rate exceeds a set value (e.g., 8.3°C/min)

## Applicable Scenarios

- Kitchens (smoke sensors are prone to false alarms from cooking fumes)
- Garages/workshops (exhaust and dust may interfere with smoke sensors)
- Boiler rooms/heating equipment rooms
- Bathrooms/saunas

## Core Features

- Abnormal temperature rise detection
- Tamper alarm
- Low battery notification

> **Note**: Heat detectors complement smoke sensors — smoke sensors respond faster, while heat detectors are more resistant to interference.
