# Installation Guide

## Installation Location Requirements

- **Ceiling installation**, centered in the room
- **Best use scenarios**: kitchens, garages, boiler rooms, bathrooms — places where smoke sensors are prone to false alarms
- At least **30 cm** away from walls
- Avoid installing near ventilation openings and AC vents

## Installation Steps

1. Select installation location (recommended: kitchen or garage ceiling)
2. Secure the mounting base to the ceiling
3. Insert battery, sensor powers on automatically
4. Rotate and snap the body onto the base
5. Self-test: press the test button, LED lights up and sound is emitted
6. Pair with gateway (Zigbee version)

## Precautions

- Heat detectors are **not a replacement** for smoke sensors, but a complement
- Smoke sensors respond faster, heat detectors are more resistant to interference
- It is recommended to install both smoke and heat sensors in the same room
- Test once per month, replace battery promptly when low
