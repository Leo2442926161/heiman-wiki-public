# Installation Guide

## Installation Location Requirements

- **Ceiling mount**, sensor pointing vertically downward
- Installation height **2.5 ~ 3.5 m**
- Detection radius approximately **3 ~ 5 m**
- **Avoid**: metal obstacles in front of the sensor (metal blocks millimeter waves)
- Keep away from large metal pipes, AC indoor units, etc.

## Installation Steps

1. Select ceiling installation location (center of the room is best)
2. Drill holes in the ceiling, secure the base with expansion screws
3. Connect wiring (if mains-powered) or insert battery
4. Rotate and snap the sensor body onto the base
5. After power-on, the LED lights up and the sensor starts self-test
6. Enter the app to pair and add the device
7. Test: sit still in the detection area, the sensor should maintain "occupied" status

## Precautions

- Millimeter waves can penetrate thin panels (drywall, wood) but metal will block them
- Thick walls and metal ceilings will reduce detection range
- Recommended for use with PIR sensors for full-scenario coverage
