# Model Comparison

## Model Comparison

| Model | Protocol | Detection Technology | Mounting | Features | Recommended Use |
|-------|----------|---------------------|----------|----------|----------------|
| [HS8OS](../../无线协议/Zigbee/HS8OS.md) | Zigbee | Millimeter-wave radar | Ceiling | Stationary human detection | More precise than PIR, ideal for bathrooms/bedrooms where people stay still for long periods, solves PIR false-off issue |

## Usage Scenarios

| Scenario | Recommended Model | Automation Scheme |
|----------|------------------|-------------------|
| Bathroom always-on light | HS8OS | Lights on when occupied, off when vacant (detects subtle breathing movement, solves PIR stationary turn-off issue) |
| Bedroom sleep monitoring | HS8OS | Detect if someone is in bed, auto-adjust AC temperature |
| Elderly care | HS8OS | Elderly stationary in bathroom beyond timeout → system alert notifies family, preventing accidents |
| Meeting room energy saving | HS8OS | Unoccupied for N minutes → auto-turn off lights and AC, energy saving |

## Selection Guide

- **Solve PIR light-off issue**: [HS8OS](../../无线协议/Zigbee/HS8OS.md) (millimeter-wave, stationary detection)
- **Pair with PIR**: PIR for motion detection + HS8OS for stationary detection, best results when used together
