# Installation Guide

## Installation Location Requirements

- **Mount on ceiling**, centered in the room
- At least **30 cm** away from walls or light fixtures
- **Avoid dead air spaces** (e.g., corners, behind beams)
- **Do not install in**: bathrooms (steam), kitchens (cooking fumes), near ventilation openings

## Installation Steps

1. Select the installation location, ensure the ceiling is flat
2. Rotate and secure the mounting base to the ceiling (using screws or 3M adhesive)
3. Insert the battery (observe polarity), the sensor powers on automatically
4. Rotate and snap the sensor body onto the base
5. Long-press the test button — the alarm LED flashes and an alarm sound indicates normal operation
6. Pair with gateway: enter pairing mode, add the device in the app

## Precautions

- Test functionality once a month
- Battery life is approximately 3 years, replace promptly when low
- Replace the entire sensor every 10 years
- Installation height does not affect smoke detection — smoke rises to the ceiling
