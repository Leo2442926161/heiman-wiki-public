# Installation Guide

## Installation Location Requirements

- **Install at breathing height** — approximately **1.5 m** from the floor
- Near bedroom areas to ensure the alarm can be heard while sleeping
- **Do not install in corners**, ensure air circulation
- Keep away from ventilation openings, windows, fans, and other locations that affect airflow

## Installation Steps

1. Determine the installation location (recommended: bedroom hallway or living room)
2. Secure the mounting base to the wall (screws or 3M adhesive)
3. Insert the battery, the sensor powers on automatically
4. Mount the body onto the base
5. Self-test: press the test button, the alarm LED flashes and an alarm sounds
6. Pair with gateway

## Precautions

- Sensor lifespan is typically **7 years**, replace the entire unit when expired
- Test functionality once a month
- Do not block the sensor ventilation holes
- If the alarm sounds, first open windows for ventilation, then check for CO source leaks
