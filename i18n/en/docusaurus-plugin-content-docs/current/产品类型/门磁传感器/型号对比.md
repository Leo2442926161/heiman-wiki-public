# Model Comparison

## Model Comparison

| Model | Protocol | Power | Battery Life | Features | Recommended Use |
|-------|----------|-------|-------------|----------|----------------|
| [HS1DS](../../无线协议/Zigbee/HS1DS.md) | Zigbee / Z-Wave | CR1632 | 2-3 years | Standard, dual protocol options | Turn on light when door opens, away arm, push notification + siren on unauthorized door/window opening |
| [HS1DS-M](../../无线协议/Matter/HS1DS-M.md) | Matter over Thread | CR1632 | 2-3 years | Matter version, cross-platform | Native Apple Home integration, away mode auto-checks doors/windows before arming |

## Usage Scenarios

| Scenario | Recommended Model | Automation Scheme |
|----------|------------------|-------------------|
| Auto-light on arrival | HS1DS | Door opens → turn on living room lights → turn on AC → welcome announcement |
| Away arm | HS1DS | Unusual door/window opening detected → phone push + siren alarm |
| Window rain prevention | HS1DS | Window open + rain sensor triggered → TTS broadcast reminder to close window |
| Apple Home users | HS1DS-M | Native Matter, HomeKit automation "check doors closed when last person leaves" |
| Storage/cabinet monitoring | HS1DS | Compact installation, detect unauthorized cabinet/drawer opening, ideal for offices/hotels |

## Selection Guide

- **Existing Zigbee gateway**: [HS1DS](../../无线协议/Zigbee/HS1DS.md) (Zigbee version)
- **Cross-platform compatibility needed**: [HS1DS-M](../../无线协议/Matter/HS1DS-M.md) (Matter over Thread)
- **Z-Wave system**: [HS1DS](../../无线协议/Zigbee/HS1DS.md) (Z-Wave version)
