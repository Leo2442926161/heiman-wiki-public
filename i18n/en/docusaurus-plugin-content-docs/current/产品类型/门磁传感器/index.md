# Door/Window Sensor

Door/window sensors detect the open/close status of doors and windows, serving as a fundamental component of smart security systems.

## Product Series

| Series | Communication | Features |
|--------|--------------|----------|
| HS1DS Series | Zigbee / Z-Wave | Standard type, 3-year battery life |
| HS1DS-M | Matter over Thread | Matter version |

## Core Features

- Real-time door/window open/close status detection
- Tamper alarm
- Low battery notification
- Supports automation (open door turn on light, away arm, etc.)
- Easy installation (3M adhesive or screw fixing)

## Technical Specifications

- Power Supply: CR1632 coin cell battery
- Battery Life: Approximately 2-3 years
- Operating Temperature: -10°C ~ +50°C
- Installation Gap: ≤10 mm

## Scenario Examples

- **Away Arm**: Door sensor triggers alarm when door is opened while armed away
- **Welcome Home**: Open the door to automatically turn on lights
- **Window Monitoring**: Pair with weather station, auto-close window alert when it rains
