# Installation Guide

## Installation Location Requirements

- Main body (transmitter) installs on the **door/window frame**
- Magnet installs on the **door/window leaf**
- Gap between main body and magnet **≤ 10 mm**
- Align both parts, alignment marks should line up

## Installation Steps

1. Clean the installation surface, ensure dry and dust-free
2. Peel off the 3M adhesive protective film, attach the main body to the door frame
3. Attach the magnet to the corresponding position (aligns with the main body when closed)
4. Insert battery (CR1632), LED flashes indicating power-on
5. Test opening and closing: triggers alarm when opened, resets when closed
6. Pair with gateway

## Precautions

- Excessive installation gap will cause detection failure
- Metal doors/windows may affect wireless signal, test before finalizing
- Battery life approximately 2-3 years, app will notify when low
