# Solution Deployment

Heiman Technology provides end-to-end smart security solution deployment guidance.

## Integration Process

Heiman products support multiple deployment methods, including local standalone deployment and cloud platform integration.

### Local Deployment

```bash
# Configure Heiman Smart Fire Gateway
heiman-cli configure --gateway HS1GW --mode local

# Register detector device
heiman-cli register --device HS1SA --gateway HS1GW
```

### Cloud Platform Integration

Heiman products are integrated with major IoT platforms:

- China Telecom Tianyi
- China Mobile OneNET
- Huawei HiLink
- Alibaba Cloud IoT
- Tuya Smart

## Matter Protocol Support

## Deployment Verification

After deployment, use the Heiman Smart Fire APP or management platform for functional verification and real-time monitoring.
