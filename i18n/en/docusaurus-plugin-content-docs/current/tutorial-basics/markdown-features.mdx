# Markdown Features

Heiman technical documentation fully supports Markdown syntax and extended features, making it easy to write structured product documentation.

## Code Example

API integration example for product development:

```jsx title="Smoke Alarm API Integration"
const heimanClient = new HeimanClient({
  apiKey: 'your-api-key',
  deviceId: 'HS1SA-xxxxx',
});

// Get device status
const status = await heimanClient.getDeviceStatus();
console.log(status);
```
