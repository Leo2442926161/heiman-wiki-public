# Supported Models

Heiman Sub-1G long-distance series products are primarily designed for large-area standalone networking scenarios, enabling device-to-device intercommunication without a gateway.

## Standalone Smoke Alarms (S1 Series)

| Model | Product Name | Description |
|-------|--------------|-------------|
| S1-SA | Standalone Smoke Alarm (Sub-1G) | Basic type, supports wireless interlink |
| S1-SA-P | Standalone Smoke Alarm (Sub-1G) | With photo capture function |
| S1-SA-T | Standalone Smoke Alarm (Sub-1G) | High-temperature linked version |
| S1-SA-RF | Standalone Smoke Alarm (RF Interlink Type) | RF networking version |

## Interlink Accessories

| Model | Product Name | Description |
|-------|--------------|-------------|
| S1-RM | Wireless Repeater (Sub-1G) | Extends communication distance |
| S1-RC | Remote Control (Sub-1G) | For test / silence / reset |

## Features

- **Standalone Operation**: No gateway, no internet required, devices communicate wirelessly with each other directly
- **Interlinked Alarm**: When any device triggers an alarm, all devices in the same group alarm simultaneously
- **Ultra-long Distance**: Communication distance up to 1~3 km in open environments
- **Ultra-low Power**: Battery powered, battery life up to 3~5 years
