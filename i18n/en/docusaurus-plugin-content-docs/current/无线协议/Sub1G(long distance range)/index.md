# Sub-1G (Long Distance Communication)

Sub-1G is a wireless communication technology operating below 1 GHz, featuring ultra-long transmission distance and extremely low power consumption, making it ideal for IoT scenarios requiring wide area coverage.

## Features

| Feature | Description |
|---------|-------------|
| Operating Frequency | 433 MHz / 868 MHz (EU) / 915 MHz (US) |
| Transmission Distance | Up to 1-3 km in open environments |
| Penetration | Superior to 2.4 GHz, strong wall/obstacle penetration |
| Power Consumption | Extremely low, battery life up to 5-10 years |
| Data Rate | Low (suitable for small data volume scenarios) |

## Application Scenarios

- **Standalone Smoke/CO Alarms**: Cross-floor interlinking within buildings
- **Large Building Fire Systems**: Shopping malls, hotels, schools
- **Smart Senior Care Communities**: Cross-unit emergency calling
- **Sub-1G + Wi-Fi Gateway**: Sub-devices communicate via Sub-1G, gateway connects to cloud
