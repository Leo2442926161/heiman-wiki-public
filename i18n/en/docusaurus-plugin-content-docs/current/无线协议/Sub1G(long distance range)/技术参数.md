# Technical Specifications

The following are the general technical specifications for Heiman Sub-1G long-distance series products.

## Wireless Parameters

| Parameter | Value |
|-----------|-------|
| Operating Frequency | 433 MHz / 868 MHz / 915 MHz (depending on market region) |
| Modulation | FSK / OOK |
| Data Rate | Approx. 10~100 kbps (depending on frequency band and configuration) |
| Network Topology | Star / Point-to-point / Interlinked networking |
| Communication Distance | Open environment approx. 1~3 km (depending on frequency band and power) |
| Network Capacity | Up to 50+ nodes per group |

## Regional Frequency Allocation

| Region | Common Frequency |
|--------|-----------------|
| China | 433 MHz |
| Europe | 868 MHz |
| North America | 915 MHz |

## Electrical & Power

| Parameter | Value |
|-----------|-------|
| Power Supply | Battery (CR123A / AA Lithium) |
| Battery Life | Approx. 3~5 years (depending on reporting frequency) |
| Transmit Power | ≤ 10 mW (+10 dBm), compliant with SRD regulations |
| Standby Power | Microamp level, extremely low |
| Receive Sensitivity | Approx. -110 dBm ~ -120 dBm |

## Environmental Parameters

| Parameter | Value |
|-----------|-------|
| Operating Temperature | -10°C ~ +55°C |
| Operating Humidity | ≤ 95% RH (non-condensing) |
