# Certifications

## Standard Descriptions

- **EN 14604**: European standard for standalone smoke alarms. Covers photoelectric smoke detection, alarm sound level (≥85 dB at 3 m), battery low indication, temperature cycling, humidity endurance, and RF interference immunity.
- **CE**: Mandatory conformity marking under CPR (EU No. 305/2011) and RED (2014/53/EU) directives.
- **CPR**: Construction Products Regulation requires performance stability assessment for fire alarm products.

For more details, please refer to the respective certification standard pages.
