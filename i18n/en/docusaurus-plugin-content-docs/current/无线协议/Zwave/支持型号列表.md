# Supported Models

Heiman Z-Wave products are primarily targeted at the European and American smart home markets, compatible with the Z-Wave Plus standard.

## Security & Alarm

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS1SA-Z | Smart Smoke Alarm (Z-Wave) | Z-Wave version smoke alarm |
| HS1CA-Z | Smart Carbon Monoxide Alarm (Z-Wave) | Z-Wave version CO alarm |
| HS1CG-Z | Gas Detector (Z-Wave) | Z-Wave version gas detector |

## Sensors

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS1DS-Z | Door/Window Sensor (Z-Wave) | Z-Wave version door/window contact |
| HS1MS-Z | PIR Motion Sensor (Z-Wave) | Z-Wave version motion detector |
| HS1WL-Z | Water Leak Detector (Z-Wave) | Z-Wave version water leak detector |

## Compatibility

| Item | Description |
|------|-------------|
| Standard | Z-Wave Plus (500 / 700 series chips) |
| Compatible Hubs | Homey, Hubitat, SmartThings, Vera, Fibaro, Home Assistant (Z-Wave JS) |
| S2 Security | Supports S2 security protocol (Unencrypted / S2 Unauthenticated / S2 Authenticated) |
| SmartStart | Supported (700 series chip models) |
| OTA Upgrade | Supports FW Update CC |
