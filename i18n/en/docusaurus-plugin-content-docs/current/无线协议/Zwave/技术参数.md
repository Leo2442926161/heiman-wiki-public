# Technical Specifications

The following are the general technical specifications for Heiman Z-Wave devices.

## Wireless Parameters

| Parameter | Value |
|-----------|-------|
| Protocol Standard | Z-Wave Plus (ITU-T G.9959) |
| Operating Frequency | 800~900 MHz (by region: EU 868.42 MHz, US 908.42 MHz) |
| Data Rate | Up to 100 kbps |
| Network Topology | Self-healing Mesh (source routing) |
| Node Capacity | Up to 232 nodes per network |
| Communication Range | Indoor approx. 30 meters (node to node), whole-home Mesh coverage |
| Encryption | AES-128 (S2 Authenticated / S2 Unauthenticated) |

## Regional Frequency Bands

| Region | Center Frequency |
|--------|-----------------|
| Europe | 868.42 MHz |
| North America | 908.42 MHz |
| Other Regions | Subject to local regulations |

## Electrical & Power

| Parameter | Value |
|-----------|-------|
| Power Supply | Battery (CR123A / AA) or mains powered |
| Battery Life | Sensor types approx. 2~3 years |
| Transmit Power | ≤ 1 mW (compliant with SRD regulations) |
| Receive Sensitivity | Approx. -100 dBm |

## Environmental Parameters

| Parameter | Value |
|-----------|-------|
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤ 90% RH (non-condensing) |

## Network Features

| Feature | Description |
|---------|-------------|
| SmartStart | Supported (700 series), scan to join network |
| S2 Security | Supports three security levels: Unencrypted, S2 Unauthenticated, S2 Authenticated |
| Included CCs | Security CC, Security 2 CC, Firmware Update CC, Basic CC, Battery CC, Notification CC, etc. |
