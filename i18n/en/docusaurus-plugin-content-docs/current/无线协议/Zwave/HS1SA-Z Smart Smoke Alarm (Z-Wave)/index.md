# HS1SA-Z Smart Smoke Alarm (Z-Wave)

HS1SA-Z is a **Z-Wave** smart smoke alarm from Heiman, using photoelectric smoke detection technology, compatible with the Z-Wave Plus standard, designed for European and American smart home markets.

## Product Overview

- **Product Category**: Smart Smoke Alarm
- **Communication Protocol**: Z-Wave Plus
- **Certification Standard**: EN 14604
- **Typical Scenarios**: Fire warning in homes, apartments, etc.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Sensor Type | Infrared Photoelectric |
| Communication Protocol | Z-Wave Plus (ITU-T G.9959) |
| Operating Frequency | 868.42 MHz (EU) / 908.42 MHz (US) |
| Alarm Sound Level | ≥85 dB (at 3 meters) |
| Power Supply | CR123A Lithium Battery |
| Battery Life | Approx. 2~3 years |
| Encryption | S2 Authenticated / S2 Unauthenticated |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤90% RH (non-condensing) |

## Features

- Photoelectric smoke detection, detects smoldering and open flame smoke
- 85 dB loud sound and light alarm
- Low battery notification
- Self-test function
- Tamper alarm
- Z-Wave S2 security encryption
- Supports OTA firmware upgrade

## Certifications

- EN 14604 Certified
- Z-Wave Plus Certified
- S2 Security Certified
