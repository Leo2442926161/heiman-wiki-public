# HS1DS-Z Door/Window Sensor (Z-Wave)

HS1DS-Z is a **Z-Wave** door/window sensor from Heiman, using a reed switch to detect the open/close status of doors and windows, compatible with the Z-Wave Plus standard.

## Product Overview

- **Product Category**: Door/Window Sensor
- **Communication Protocol**: Z-Wave Plus
- **Detection Method**: Reed Switch (dry contact)
- **Typical Scenarios**: Door/window status monitoring, security arming, smart automation

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Detection Method | Reed Switch (dry contact) |
| Communication Protocol | Z-Wave Plus (ITU-T G.9959) |
| Operating Frequency | 868.42 MHz (EU) / 908.42 MHz (US) |
| Power Supply | CR1632 Coin Cell Battery |
| Battery Life | Approx. 2~3 years |
| Installation Gap | ≤10 mm |
| Encryption | S2 Authenticated / S2 Unauthenticated |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤90% RH (non-condensing) |

## Features

- Real-time door/window open/close status detection
- Tamper alarm
- Low battery notification
- Supports scene automation (open door turn on lights, away mode arm)
- Easy installation (3M adhesive or screw mounting)
- Z-Wave S2 security encryption
- Supports OTA firmware upgrade
