# Z-Wave

Z-Wave is a wireless Mesh network protocol designed specifically for smart homes, managed by the Z-Wave Alliance, with widespread adoption in European and American markets.

## Technical Specifications

| Feature | Description |
|---------|-------------|
| Operating Frequency | 868.42 MHz (EU) / 908.42 MHz (US) |
| Data Rate | Up to 100 kbps (Z-Wave 800 series) |
| Network Topology | Mesh network, up to 232 nodes |
| Compatibility | Backward compatible with all Z-Wave devices |
| Security | Z-Wave S2 encryption |

## Heiman and Z-Wave

Heiman offers multiple Z-Wave compatible sensors, primarily sold in European and American markets. Z-Wave version Heiman devices include:

- Carbon Monoxide Sensor HS1CA-Z
- Smoke Sensor HS1SA-Z
- Door/Window Sensor HS1DS-Z

## Compatible Platforms

- SmartThings Hub
- Hubitat Elevation
- Homey Pro
- Home Assistant (via Z-Wave JS)
- OpenHAB

> **Note**: Z-Wave has low adoption in the Chinese market. Heiman Z-Wave products are mainly targeted at European and American export markets.
