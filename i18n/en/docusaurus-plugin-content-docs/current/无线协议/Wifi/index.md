# Wi-Fi

Wi-Fi is the most widely adopted wireless network protocol, based on the IEEE 802.11 standard. Some Heiman smart devices support Wi-Fi connectivity, allowing direct connection to a router without requiring an additional gateway.

## Application Scenarios

Wi-Fi devices are suitable for the following situations:

- Network infrastructure is already in place, no need to purchase an additional gateway
- Devices that are not sensitive to power consumption (e.g., smart plugs)
- Scenarios requiring remote access and cloud services
- Homes with existing mature Wi-Fi Mesh network coverage

## Heiman Wi-Fi Products

Heiman offers products with Wi-Fi connectivity:

- **Smart Plug HS2SK-WiFi** — Power metering, remote on/off
- **Smart Gateway** — Connects to router via Wi-Fi

## Notes

- Battery consumption on Wi-Fi sensors is faster than Zigbee/Thread
- A large number of Wi-Fi devices may affect network stability
- Heiman's main products are primarily Zigbee / Matter over Thread; the Wi-Fi product line is relatively streamlined
