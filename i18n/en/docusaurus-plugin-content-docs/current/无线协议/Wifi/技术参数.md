# Technical Specifications

The following are the general technical specifications for Heiman WiFi devices.

## Wireless Parameters

| Parameter | Value |
|-----------|-------|
| Protocol Standard | IEEE 802.11 b/g/n |
| Operating Frequency | 2.4 GHz (not compatible with 5 GHz) |
| Data Rate | Up to 150 Mbps (802.11n) |
| Network Topology | Star topology (direct to router) |
| Communication Range | Indoor approx. 30~50 meters (depends on router performance) |
| Encryption | WPA2-PSK / WPA3 |

## Pairing Methods

| Method | Description |
|--------|-------------|
| SmartConfig | One-key App pairing (traditional WiFi devices) |
| Bluetooth-assisted pairing | Configure WiFi credentials via temporary Bluetooth channel (select newer models) |
| AP Mode | Device creates a hotspot, phone connects to configure |

## Electrical & Power

| Parameter | Value |
|-----------|-------|
| Power Supply | Mains powered (plug types / gateway types) or battery powered (alarm types) |
| Power Level | Relatively higher than Zigbee / Thread due to maintaining TCP/IP connection |
| Standby Power | Plug types approx. 0.5W ~ 1W |

## Environmental Parameters

| Parameter | Value |
|-----------|-------|
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤ 90% RH (non-condensing) |
