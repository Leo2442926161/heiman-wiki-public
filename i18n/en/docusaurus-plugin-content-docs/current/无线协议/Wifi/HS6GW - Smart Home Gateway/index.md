# HS6GW - Smart Home Gateway

## Overview

HS6GW is a smart home gateway from Heiman, supporting WiFi + Ethernet uplink with a built-in Zigbee coordinator, capable of managing up to 128 Zigbee sub-devices. By connecting to the internet, it enables remote access and cloud services, serving as the core hub of the Heiman smart home system.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Uplink Communication | IEEE 802.11 b/g/n (2.4 GHz WiFi) + 10/100M Ethernet |
| Downlink Communication | Zigbee 3.0 (Coordinator mode) |
| Maximum Sub-devices | 128 |
| Power Supply | DC 5V / 1A (Micro USB) |
| Standby Power | ≤ 1W |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤ 90% RH |
| Indicators | WiFi status / Power / Zigbee network status |
| Reset Method | Button reset + long press factory reset |

## Features

- Dual uplink: WiFi or Ethernet wired connection, flexible choice based on network environment
- Remote access: view device status anytime via cloud App
- Local scene automation: supports device automation within LAN (still works when offline)
- OTA firmware upgrade
- Supports Tuya / Smart Life platform
- Manages up to 128 Zigbee sub-devices
- Supports Zigbee network self-healing and route optimization
