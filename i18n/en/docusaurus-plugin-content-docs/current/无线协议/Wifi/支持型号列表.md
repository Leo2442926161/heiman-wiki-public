# Supported Models

Heiman WiFi devices are primarily designed for direct-connect scenarios, connecting to the home network without a dedicated gateway.

## Smart Plugs

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS2SK-WiFi | Smart Plug (WiFi) | Direct connect 2.4GHz WiFi, supports remote control and timer |

## Security & Alarm

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS8SA-WiFi | Smart Smoke Alarm (WiFi US Version) | US market WiFi version smoke alarm |

## Gateways

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS6GW | Smart Home Gateway (WiFi + Zigbee) | Connects to internet via WiFi, manages Zigbee sub-devices |
| HS6GW-2 | Smart Home Gateway (WiFi + Zigbee) | 2nd generation gateway, performance upgrade |

## Notes

- All WiFi devices only support **2.4 GHz** band, not compatible with 5 GHz
- WPA2-PSK encrypted network recommended
- Internet connectivity requires the router to have a working internet connection
