# HS8SA-WiFi - Smart Smoke Alarm (WiFi US Version)

## Overview

HS8SA-WiFi is a WiFi smart smoke alarm from Heiman for the US market, using photoelectric smoke detection technology, supporting both WiFi-connected and standalone operation modes, certified to UL 217. Suitable for fire safety protection in homes, apartments, and other residential spaces.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | IEEE 802.11 b/g/n (2.4 GHz) |
| Sensor Type | Infrared Photoelectric |
| Alarm Sound Level | ≥85 dB (at 3 meters) |
| Power Supply | Battery powered |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤ 90% RH |
| Installation Method | Ceiling mount |

## Features

- Photoelectric smoke detection, detects smoldering and open flame smoke
- WiFi-connected remote alarm push notification
- Standalone operation mode (local audible and visual alarm still works when offline)
- 85 dB loud sound and light alarm
- Low battery notification
- Self-test function (one-button test)
- Tamper alarm
- Supports remote silence

## Certification Standards

- UL 217 (US standard for smoke alarms)
- FCC
- NFPA 72 Compliant
