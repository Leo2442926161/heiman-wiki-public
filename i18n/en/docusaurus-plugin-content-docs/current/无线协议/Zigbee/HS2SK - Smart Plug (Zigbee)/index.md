# HS2SK - Smart Plug (Zigbee)

## Overview

HS2SK is a Zigbee smart metering plug from Heiman, supporting remote on/off control and real-time power consumption metering with ±2% accuracy, suitable for smart management of household appliances.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Rated Voltage | AC 220V (China standard) / AC 110V (US standard) |
| Maximum Load | 16A / 3680W |
| Metering Accuracy | ±2% |
| Power Supply | Mains powered |

## Features

- Remote on/off control
- Real-time power / energy metering
- Overload protection
- Timer on/off
- Energy statistics
- Supports scene automation

## Application Scenarios

- Remotely turn off forgotten appliances
- Monitor AC / water heater energy consumption
- One-touch shut-off of non-essential appliances in away mode
