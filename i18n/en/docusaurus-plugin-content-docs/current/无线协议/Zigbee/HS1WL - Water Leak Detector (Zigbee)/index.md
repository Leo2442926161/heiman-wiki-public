# HS1WL - Water Leak Detector (Zigbee)

## Overview

HS1WL is the first-generation Zigbee water leak detector from Heiman, using probe detection to effectively detect leaks or standing water. It is a basic water leak monitoring device.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Detection Method | Probe detection |
| Power Supply | DC 3V Battery |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |

## Features

- Real-time leak / standing water detection
- Buzzer alarm
- Low battery notification
- Probe anti-corrosion design
