# HS3CG - Gas Detector (Zigbee)

## Overview

HS3CG is Heiman's third-generation Zigbee gas detector, with improved detection accuracy and response speed over the first generation, suitable for natural gas and LPG leak detection.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Semiconductor |
| Detection Gas | Natural Gas (Methane) / Liquefied Petroleum Gas (LPG) |
| Alarm Sound Level | ≥85 dB (at 3 meters) |
| Power Supply | AC 220V / DC 12V |
| Certification Standard | GB 15322 |
| Operating Temperature | -10°C ~ +50°C |

## Features

- Natural gas / LPG leak detection
- Level-based graded alarm
- Automatic shut-off linkage (with solenoid valve)
- 85 dB audible and visual alarm
- Self-test function
- Sensor end-of-life notification
- Enhanced wireless performance

## Certification Standards

- GB 15322

## Applicable Accessories

- JT series gas emergency shut-off valve
