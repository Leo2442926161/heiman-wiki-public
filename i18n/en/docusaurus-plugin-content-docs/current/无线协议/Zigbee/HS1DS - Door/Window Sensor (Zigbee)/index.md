# HS1DS - Door/Window Sensor (Zigbee)

## Overview

HS1DS is a Zigbee door/window sensor from Heiman, used to detect the open/close status of doors and windows. It is a fundamental component of smart security systems.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Reed Switch |
| Power Supply | CR1632 Coin Cell Battery |
| Battery Life | Approx. 2 ~ 3 years |
| Installation Gap | ≤10 mm |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |

## Features

- Real-time door/window open/close status detection
- Tamper alarm
- Low battery notification
- Supports automation (open door turn on lights, away mode arm, etc.)
- Easy installation (3M adhesive or screw mounting)

## Applicable Accessories

- CR1632 coin cell battery
- 3M double-sided tape / screws
