# HS8MIS - Motion Sensor (Zigbee)

## Overview

HS8MIS is a multi-function Zigbee motion sensor from Heiman, integrating PIR human detection and illuminance sensing for smart scenarios requiring both light and motion monitoring.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Passive Infrared (PIR) + Illuminance |
| Detection Angle | 110° |
| Detection Distance | ≤7 m |
| Power Supply | DC 3V Battery |

## Features

- Human motion detection
- Illuminance detection
- Low power design
- Supports scene automation
