# HS2FD - Fall Detector (Zigbee)

## Overview

HS2FD is a Zigbee smart fall detector from Heiman, designed specifically for the elderly. It features a built-in accelerometer and gyroscope, uses AI algorithms to recognize fall postures, and can automatically send a distress signal.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Accelerometer + Gyroscope |
| Detection Technology | AI Fall Detection Algorithm |
| Power Supply | DC 3V Battery |
| Wearing Method | Belt clip / Lanyard |

## Features

- Automatic fall detection
- One-button manual SOS
- AI algorithm distinguishes falls from normal activities (sitting, bending) to reduce false alarms
- Wearable design (belt clip or lanyard)
- Low battery notification

## Application Scenarios

- Home care: 24/7 protection for seniors living alone
- Nursing homes: Centralized monitoring management
- Rehabilitation centers: Fall risk monitoring for post-surgery patients
