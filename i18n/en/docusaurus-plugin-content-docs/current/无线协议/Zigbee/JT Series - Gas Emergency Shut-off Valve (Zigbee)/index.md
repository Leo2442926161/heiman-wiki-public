# JT Series - Gas Emergency Shut-off Valve (Zigbee)

## Overview

The JT series gas emergency shut-off valve is an actuator device that works in conjunction with Heiman Zigbee gas detectors. When a gas leak is detected, it can remotely or automatically close the gas pipeline valve to prevent accidents.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Type | Standalone + Zigbee linkage |
| Detection Gas | Natural Gas / LPG |
| Certification Standard | GB 15322 |
| Operating Temperature | -10°C ~ +50°C |

## Features

- Automatic valve closure linked with HS1CG / HS3CG
- Supports remote manual valve closure
- Leak alarm push notification
- Solenoid valve drive

## Certification Standards

- GB 15322

## Applicable Accessories

- HS1CG / HS3CG Gas Detector
