# HS1CA-E - Smart Carbon Monoxide Alarm (Zigbee)

## Overview

HS1CA-E is a Zigbee smart carbon monoxide (CO) alarm from Heiman for the EU market, using an electrochemical sensor to accurately detect CO concentrations from 0 to 1000 ppm. It is EN 50291 certified.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Electrochemical |
| Detection Range | 0 ~ 1000 ppm |
| Alarm Sound Level | ≥85 dB (at 3 meters) |
| Power Supply | DC 3V (CR123A) |
| Sensor Life | Approx. 7 years |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |
| Installation Method | Wall mount |

## Features

- High-precision electrochemical CO sensor
- Level-based graded alarm (audible and visual)
- 85 dB alarm sound
- Low battery notification
- Sensor end-of-life notification
- Self-test function
- OTA firmware upgrade

## Certification Standards

- EN 50291 (European standard for carbon monoxide alarms)
- CE / RED

## Applicable Accessories

- CR123A battery
