# HS6GW - Smart Gateway (Zigbee)

## Overview

HS6GW is a Zigbee smart gateway from Heiman, serving as the coordinator for the Heiman Zigbee network. It connects to the internet via Wi-Fi or Ethernet and supports up to 128 sub-devices.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 (Coordinator) |
| Uplink Connection | Wi-Fi / Ethernet |
| Maximum Nodes | 128 sub-devices |
| Power Supply | DC 5V USB power |

## Features

- Zigbee network coordinator
- Wi-Fi / Ethernet uplink connection
- Automatic device discovery and pairing
- Local scene automation
- Remote access (via Heiman cloud platform)
- Supports up to 128 sub-devices
- OTA firmware upgrade

## Application Scenarios

- Central connectivity point for Heiman whole-home smart systems
- Remote monitoring via Heiman App
