# HS8MLS - Micro Motion Sensor (Zigbee)

## Overview

HS8MLS is a long-range Zigbee PIR motion sensor from Heiman, designed for scenarios requiring long-distance detection, suitable for corridors, halls, and other open spaces.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Passive Infrared (PIR) |
| Detection Distance | Long range |
| Power Supply | DC 3V Battery |

## Features

- Long-distance motion detection
- Low power design
- Supports scene automation
