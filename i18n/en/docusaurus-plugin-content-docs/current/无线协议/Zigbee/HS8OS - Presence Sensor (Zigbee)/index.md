# HS8OS - Presence Sensor (Zigbee)

## Overview

HS8OS is a Zigbee presence sensor from Heiman, using millimeter wave radar technology to detect human presence even when stationary (e.g., lying down, sitting still), compensating for the limitation of traditional PIR sensors that only detect movement.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Millimeter Wave Radar (mmWave) |
| Detection Range | Radius 3 ~ 5 m |
| Detection Capability | Stationary human presence (including micro-motion breathing detection) |
| Power Supply | AC powered |
| Installation Method | Ceiling mount |

## Features

- Stationary human detection (breathing micro-motion)
- Coverage radius 3~5 meters
- Ceiling mount for optimal detection angle
- Can be used as a complement to PIR sensors

## Application Scenarios

- Smart lighting: lights on when occupied, off when vacant
- HVAC energy saving: auto turn off AC when no one is present
- Smart senior care: alert for prolonged inactivity
