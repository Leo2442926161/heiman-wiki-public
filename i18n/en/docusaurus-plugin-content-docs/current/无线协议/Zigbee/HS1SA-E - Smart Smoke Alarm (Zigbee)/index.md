# HS1SA-E - Smart Smoke Alarm (Zigbee)

## Overview

HS1SA-E is a standard Zigbee smart smoke alarm from Heiman for the EU market, using photoelectric smoke detection technology. It is EN 14604 certified and has won the Red Dot Design Award.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Infrared Photoelectric |
| Alarm Sound Level | ≥85 dB (at 3 meters) |
| Power Supply | DC 3V (CR123A) |
| Battery Life | Approx. 3 years |
| Communication Range | ≤70 m (open area) |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |
| Installation Method | Ceiling mount |

## Features

- Photoelectric smoke detection, detects smoldering and open flame smoke
- 85 dB loud sound and light alarm
- Low battery notification
- Self-test function (one-button test)
- Tamper alarm
- Supports remote silence
- OTA firmware upgrade

## Certification Standards

- EN 14604 (European standard for smoke alarms)
- Red Dot Design Award
- CE / RED

## Applicable Accessories

- CR123A battery
- Mounting screws / wall plugs
