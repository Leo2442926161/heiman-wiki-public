## Model Naming Convention

| Naming Rule | Meaning |
|-------------|---------|
| Suffix -E | EU version (868MHz / EN standards) |
| Suffix -Z | China version (2.4GHz / Chinese standards) |
| HS1/2/3/8 | Product generation number, higher number = newer generation |
