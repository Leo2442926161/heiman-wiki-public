# HS3WL - Water Leak Detector (Zigbee)

## Overview

HS3WL is Heiman's third-generation Zigbee water leak detector, with enhanced detection sensitivity and wireless performance over the previous generation, suitable for water leak monitoring scenarios.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Detection Method | Probe detection |
| Power Supply | DC 3V Battery |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |

## Features

- Real-time leak / standing water detection
- Enhanced detection sensitivity
- Buzzer alarm
- Low battery notification
- Probe anti-corrosion design
