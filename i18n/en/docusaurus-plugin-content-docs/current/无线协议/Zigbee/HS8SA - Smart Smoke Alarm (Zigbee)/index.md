# HS8SA - Smart Smoke Alarm (Zigbee)

## Overview

HS8SA is a Zigbee + Wi-Fi smart smoke alarm from Heiman for the US market, featuring high-temperature linkage functionality and UL 217 certification.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 / Wi-Fi |
| Sensor Type | Infrared Photoelectric |
| Alarm Sound Level | ≥85 dB (at 3 meters) |
| Power Supply | DC 3V (CR123A) |
| Operating Temperature | -10°C ~ +50°C |
| Certification Standard | UL 217 |
| Target Market | United States |

## Features

- Photoelectric smoke detection
- High-temperature linked alarm
- 85 dB loud sound and light alarm
- Low battery notification
- Self-test function
- Tamper alarm
- Supports remote silence
- Dual protocol communication (Zigbee + Wi-Fi)

## Certification Standards

- UL 217 (US standard for smoke alarms)
