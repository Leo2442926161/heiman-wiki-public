# Zigbee

Zigbee is a low-power, low-data-rate wireless Mesh network protocol based on the IEEE 802.15.4 standard. It is one of the most mature and widely adopted protocols in the smart home domain.

## Heiman and Zigbee

Heiman is one of the earliest Chinese manufacturers to adopt the Zigbee protocol, with an extensive Zigbee product line. Heiman Zigbee devices are compatible with the **Zigbee 3.0** standard and can interoperate with mainstream Zigbee gateways and coordinators.

## Technical Specifications

| Feature | Description |
|---------|-------------|
| Frequency | 2.4 GHz (worldwide) |
| Data Rate | Up to 250 kbps |
| Network Topology | Self-healing Mesh |
| Node Count | Up to 65,535 nodes per network |
| Communication Range | Indoor approx. 10-30 meters (depending on obstacles) |
| Power Consumption | Extremely low, sensor battery life up to 3 years |

## Compatible Platforms

Heiman Zigbee devices can connect to the following platforms:

- **Zigbee2MQTT** — Open-source Zigbee to MQTT bridge
- **ZHA** — Home Assistant built-in Zigbee integration
- **Hubitat** — Local smart hub
- **Homey** — Multi-protocol smart hub
- **SmartThings Hub** — Samsung smart home hub
- **Tuya Zigbee Gateway** — Tuya ecosystem
- **Heiman HS6GW Gateway** — Official gateway
