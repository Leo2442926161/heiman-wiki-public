# HS3SA - Smart Smoke Alarm (Zigbee)

## Overview

HS3SA is Heiman's third-generation Zigbee smart smoke alarm, featuring enhanced detection performance and linkage capabilities over the first generation, designed for the EU market.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Infrared Photoelectric |
| Alarm Sound Level | ≥85 dB (at 3 meters) |
| Power Supply | DC 3V (CR123A) |
| Battery Life | Approx. 3 years |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |
| Installation Method | Ceiling mount |

## Features

- Photoelectric smoke detection
- 85 dB loud sound and light alarm
- Low battery notification
- Self-test function
- Tamper alarm
- Supports remote silence
- OTA firmware upgrade
- Enhanced wireless performance

## Certification Standards

- EN 14604
- CE / RED

## Applicable Accessories

- CR123A battery
