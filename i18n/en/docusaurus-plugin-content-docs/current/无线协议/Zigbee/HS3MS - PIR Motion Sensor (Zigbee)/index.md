# HS3MS - PIR Motion Sensor (Zigbee)

## Overview

HS3MS is Heiman's third-generation Zigbee PIR motion sensor, featuring a 360° ceiling-mount design with significantly improved detection sensitivity and coverage compared to the previous generation.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Passive Infrared (PIR) |
| Detection Angle | 360° (ceiling mount) |
| Detection Distance | ≤7 m (25°C) |
| Power Supply | DC 3V Lithium Battery |
| Battery Life | Approx. 3 years |
| Operating Temperature | -10°C ~ +50°C |
| Installation Method | Ceiling mount |

## Features

- Human motion detection
- 360° omnidirectional coverage
- Low battery notification
- Tamper alarm
- Enhanced detection sensitivity
- Supports scene automation
