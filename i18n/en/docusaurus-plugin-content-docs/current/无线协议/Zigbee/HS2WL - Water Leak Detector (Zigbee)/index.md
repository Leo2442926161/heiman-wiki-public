# HS2WL - Water Leak Detector (Zigbee)

## Overview

HS2WL is the second-generation Zigbee water leak detector from Heiman, supporting dual-channel detection via both probe and cable for detecting pipe bursts, sink overflows, and other water leak events.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Detection Method | Probe detection + Cable detection (dual-channel) |
| Power Supply | DC 3V Battery |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |

## Features

- Real-time leak / standing water detection
- Probe + cable dual-channel detection
- Buzzer alarm
- Low battery notification
- Probe anti-corrosion design

## Applicable Accessories

- Extension detection cable
