# C1 Series - Carbon Monoxide Alarm (Zigbee)

## Overview

The C1 series is a basic Zigbee carbon monoxide alarm from Heiman, using an electrochemical sensor for CO leak detection, certified to EN 50291.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Electrochemical |
| Detection Range | 0 ~ 1000 ppm |
| Alarm Sound Level | ≥85 dB (at 3 meters) |
| Power Supply | Battery powered |
| Sensor Life | Approx. 7 years |
| Operating Temperature | -10°C ~ +50°C |

## Features

- Electrochemical CO detection
- Level-based graded alarm
- 85 dB audible and visual alarm
- Low battery notification
- Self-test function

## Certification Standards

- EN 50291
