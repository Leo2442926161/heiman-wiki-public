# Technical Specifications

The following are the general technical specifications for Heiman Zigbee devices.

## Wireless Parameters

| Parameter | Value |
|-----------|-------|
| Protocol Standard | Zigbee 3.0 (IEEE 802.15.4) |
| Operating Frequency | 2.4 GHz (worldwide ISM band) |
| Data Rate | Up to 250 kbps |
| Network Topology | Self-healing Mesh |
| Node Capacity | Up to 65,535 nodes per network |
| Communication Range | Indoor approx. 10~30 meters (depending on wall obstacles) |
| Encryption | AES-128-CCM |

## Compatibility

| Item | Description |
|------|-------------|
| Zigbee Coordinator | Compatible with Zigbee 3.0 standard coordinators |
| Supported Platforms | Zigbee2MQTT, ZHA (Home Assistant), Hubitat, Homey, SmartThings, Tuya Zigbee Gateway, Heiman HS6GW Gateway |
| OTA Upgrade | Supports over-the-air firmware upgrade (select models) |

## Electrical & Power

| Parameter | Value |
|-----------|-------|
| Power Supply | Battery (CR123A / CR2450 / AA) or USB / mains (plug type) |
| Battery Life | Sensor types approx. 2~3 years (depending on reporting frequency) |
| Standby Power | Microamp level, supports sleep polling mechanism |

## Environmental Parameters

| Parameter | Value |
|-----------|-------|
| Operating Temperature | -10°C ~ +55°C |
| Operating Humidity | ≤ 95% RH (non-condensing) |
| IP Rating | Varies by model (sensor types typically IP30) |
