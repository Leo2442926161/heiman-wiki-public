# Supported Models

All Heiman Matter devices use the Matter over Thread solution. The following supported models have been released.

## Security & Alarm

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS1SA-M | Smart Smoke Alarm (Matter) | US/EU version, world's first Matter-certified smoke alarm |
| HS1CA-M | Smart Carbon Monoxide Alarm (Matter) | US/EU version, world's first Matter-certified CO alarm |

## Sensors

| Model | Product Name | Description |
|-------|--------------|-------------|
| M1-M | PIR Motion Sensor (Matter) | Passive infrared detection, supports Apple HomeKit, Google Home, etc. |
| L1-M | Water Leak Detector (Matter) | Water leak detection, real-time alert push |
| H1-M | Temperature & Humidity Sensor (Matter) | Temperature and humidity monitoring |
| HS1DS-M | Door/Window Sensor (Matter) | Door/window open/close detection, tamper alarm |

## Notes

- Matter devices require a **Thread Border Router** to operate, such as Apple HomePod mini, Google Nest Hub 2nd Gen, etc.
- The current Heiman Matter product line is still expanding; more models will be released gradually.
