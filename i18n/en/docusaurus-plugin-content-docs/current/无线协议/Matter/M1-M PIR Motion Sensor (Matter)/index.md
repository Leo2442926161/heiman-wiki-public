# M1-M PIR Motion Sensor (Matter)

M1-M is a **Matter over Thread** PIR motion sensor from Heiman, integrating PIR detection and illuminance sensing for smart home automation scenarios.

## Product Overview

- **Product Category**: PIR Motion Sensor
- **Communication Protocol**: Matter over Thread
- **Detection Method**: Passive Infrared (PIR) + Illuminance Detection
- **Typical Scenarios**: Light on when motion detected, security linkage, environment automation

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Detection Angle | 110° (wall-mount) / 360° (ceiling-mount) |
| Detection Distance | ≤7 m (25°C) |
| Illuminance Range | 0~1200 Lux |
| Communication Protocol | Matter over Thread (IEEE 802.15.4) |
| Operating Frequency | 2.4 GHz |
| Power Supply | DC 3V Lithium Battery |
| Battery Life | Approx. 1 year (Matter mode) |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH (non-condensing) |

## Features

- Motion detection
- Ambient illuminance detection
- Detection distance ≤7m
- Supports wall or ceiling mounting
- Matter multi-ecosystem cross-platform interoperability
