# HS1CA-M Smart Carbon Monoxide Alarm (Matter)

HS1CA-M is the world's **first Matter-certified carbon monoxide alarm**, featuring electrochemical sensor technology and supporting Matter over Thread cross-ecosystem interoperability.

## Product Overview

- **Product Category**: Smart Carbon Monoxide Alarm
- **Communication Protocol**: Matter over Thread
- **Certification Standard**: EN 50291
- **Typical Scenarios**: CO leak warning in homes, apartments, etc.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Sensor Type | Electrochemical CO Sensor |
| Communication Protocol | Matter over Thread (IEEE 802.15.4) |
| Operating Frequency | 2.4 GHz |
| Detection Range | 0~1000 ppm |
| Alarm Sound Level | ≥85 dB (at 3 meters) |
| Power Supply | DC 3V (CR123A Lithium Battery) |
| Battery Life | Approx. 3 years |
| Sensor Life | Typically 7 years |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH (non-condensing) |

## Features

- High-precision electrochemical CO sensor
- Level-based graded alarm (audible and visual)
- 85 dB loud sound and light alarm
- Low battery notification
- Sensor end-of-life notification
- Self-test function
- Matter multi-ecosystem cross-platform interoperability

## Certifications
