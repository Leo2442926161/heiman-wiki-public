# HS1SA-M Smart Smoke Alarm (Matter)

HS1SA-M is a **Matter over Thread** smart smoke alarm from Heiman, using photoelectric smoke detection technology and supporting cross-ecosystem smart home platform interoperability.

## Product Overview

- **Product Category**: Smart Smoke Alarm
- **Communication Protocol**: Matter over Thread
- **Certification Standard**: EN 14604
- **Typical Scenarios**: Fire warning in homes, apartments, hotels, etc.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Sensor Type | Infrared Photoelectric |
| Communication Protocol | Matter over Thread (IEEE 802.15.4) |
| Operating Frequency | 2.4 GHz |
| Alarm Sound Level | ≥85 dB (at 3 meters) |
| Power Supply | DC 3V (CR123A Lithium Battery) |
| Battery Life | Approx. 3 years |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH (non-condensing) |

## Features

- Photoelectric smoke detection, detects smoldering and open flame smoke
- 85 dB loud sound and light alarm
- Low battery notification
- Self-test function (one-button test)
- Tamper alarm
- Remote silence
- Matter multi-ecosystem cross-platform interoperability

## Certifications

- EN 14604 Certified
- Matter Certified
