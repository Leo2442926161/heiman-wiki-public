# L1-M Water Leak Detector (Matter)

L1-M is a **Matter over Thread** water leak detector from Heiman, using probe detection to detect leaks or standing water, helping prevent property damage from burst pipes.

## Product Overview

- **Product Category**: Water Leak Sensor
- **Communication Protocol**: Matter over Thread
- **Detection Method**: Probe contact detection
- **Typical Scenarios**: Leak detection in kitchens, bathrooms, laundry rooms, basements

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Detection Method | Stainless steel probe contact |
| Communication Protocol | Matter over Thread (IEEE 802.15.4) |
| Operating Frequency | 2.4 GHz |
| Power Supply | Battery powered |
| Alarm Method | Buzzer + App push notification |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH (non-condensing) |

## Features

- Real-time leak/standing water detection
- On-site buzzer alarm
- Remote App alert push
- Low battery notification
- Probe anti-corrosion design
- Matter multi-ecosystem cross-platform interoperability
