# Certifications

## Applicable Standards

- [EN European Standards](../../../../认证标准/EN%20欧洲标准.md) — EN 13757-3, EN 13757-4
- [CE EU Conformity](../../../../认证标准/CE%20欧盟符合性.md)

## Standard Descriptions

- **EN 13757-4**: European standard for Wireless M-Bus physical layer communication at 868 MHz.
- **EN 13757-3**: European standard for M-Bus application layer protocol for meter data exchange.
- **CE / RED**: Compliance with Radio Equipment Directive (2014/53/EU) for wireless devices.

For more details, please refer to the respective certification standard pages.
