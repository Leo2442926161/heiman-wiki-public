# Water Meter Collector - Wireless Water Meter Data Collector

## Overview

The Water Meter Collector is a wM-Bus wireless data collection device from Heiman for the smart water utility sector, operating in the 868 MHz band (EN 13757 standard). It supports pulse counting and wireless data transmission for remote reading of water meter metering data and uploading to a centralized meter reading system. Suitable for apartments, residential communities, commercial buildings, and industrial water monitoring.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | wM-Bus (EN 13757) |
| Operating Frequency | 868 MHz (European SRD band) |
| Data Rate | 4.8 kbps / 16.384 kbps / 100 kbps (T/S/R modes) |
| Communication Distance | Indoor approx. 100~300 m, open approx. 500~1000 m |
| Sensor Interface | Reed switch / Hall sensor pulse input |
| Power Supply | Industrial lithium battery (ER18505 / ER26500) |
| Battery Life | 6~10 years (typical reporting 1x/day) |
| Operating Temperature | -20°C ~ +60°C |
| Operating Humidity | ≤ 95% RH |
| Protection Rating | IP65 ~ IP68 (depending on installation environment) |
| Data Encryption | AES-128 (optional) |

## Features

- Supports reed switch / Hall sensor pulse collection
- Pulse counting and flow conversion, automatic usage data reporting
- Supports wM-Bus S/T/R three operating modes
- Low power deep sleep, ultra-long battery life
- AES-128 data encryption for communication security
- Supports remote firmware upgrade
- Disconnection reconnection and data re-transmission mechanism
- Supports EN 13757-3 application layer protocol
- Can work with HS1-CM collector / HS1-GW gateway for centralized upload

## Certification Standards
