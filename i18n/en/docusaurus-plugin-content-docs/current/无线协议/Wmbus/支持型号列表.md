# Supported Models

Heiman wireless M-Bus (wM-Bus) products are primarily used in the utility metering sector, supporting remote meter reading and energy monitoring.

## Gas Metering

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS1-GM | Wireless Gas Meter (wM-Bus) | Supports EN 13757 standard |
| HS1-GM-T | Wireless Gas Meter (wM-Bus) | With temperature compensation |

## Water Metering

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS1-WM | Wireless Water Meter (wM-Bus) | Reed switch / Hall sensor |
| HS1-WM-R | Wireless Water Meter (wM-Bus) | Remote transmission type |

## Heat Meter Interface

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS1-HM | Heat Meter Wireless Module (wM-Bus) | Retrofit for standard heat meters |

## Data Collection & Relay

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS1-CM | Wireless Collector (wM-Bus) | Centralized collection of surrounding meter data |
| HS1-GW | wM-Bus Gateway | Converts wM-Bus data to TCP/IP for upload |

## Notes

- Heiman wM-Bus products are primarily designed for **utility** and **industrial metering** scenarios
- Specific models and specifications may be customized based on project requirements
- Wireless M-Bus operates on **868 MHz (European standard band)**, compliant with EN 13757 series standards
