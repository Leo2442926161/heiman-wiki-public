# Sub-1G (Fernkommunikation)

Sub-1G ist eine drahtlose Kommunikationstechnologie, die im Frequenzband unter 1 GHz arbeitet. Es zeichnet sich durch eine extrem lange Übertragungsentfernung und einen extrem geringen Stromverbrauch aus. Es eignet sich sehr gut für IoT-Szenarien, die eine großflächige Abdeckung erfordern.

## Funktionen

| Funktionen | Beschreibung |
|------|------|
| Betriebsfrequenzband | 433 MHz / 868 MHz (EU) / 915 MHz (USA) |
| Übertragungsentfernung | Bis zu 1-3 Kilometer in offener Umgebung |
| Durchdringungsfähigkeit | Besser als 2,4 GHz, starkes Eindringen in Wandhindernisse |
| Stromverbrauch | Extrem niedrig, die Batterielebensdauer kann 5-10 Jahre erreichen |
| Übertragungsrate | Niedrig (geeignet für Szenarien mit kleinem Datenvolumen) |

## Anwendungsszenarien

- **Unabhängiger Rauch-/CO-Melder**: über die Etagen des Gebäudes hinweg miteinander verbunden
- **Brandschutzsystem für große Gebäude**: Einkaufszentren, Hotels, Schulen
- **Smart Senior Living Community**: Abteilungsübergreifender Notruf
- **Sub-1G + Wi-Fi Gateway**: Das Untergerät kommuniziert über Sub-1G und das Gateway stellt eine Verbindung zur Cloud her

## Hyman Sub-1G-Produkte

Hyman verfügt über langjährige Erfahrung mit drahtlosen Sub-1G-Brandschutzprodukten, einschließlich der eigenständigen Rauchmelder der S1-Serie. Diese Geräte können eine Verbindung zwischen Geräten ohne Internet realisieren.

> **Hinweis**: Sub-1G wird hauptsächlich in den Feuer- und Sicherheitsproduktlinien von Heiman verwendet, und die Smart-Home-Produktlinie besteht hauptsächlich aus Zigbee/Matter.
