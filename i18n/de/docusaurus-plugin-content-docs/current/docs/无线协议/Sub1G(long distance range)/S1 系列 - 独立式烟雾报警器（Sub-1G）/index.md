# S1-Serie – Eigenständiger Rauchmelder (Sub-1G)

## Übersicht

Die S1-Serie ist ein eigenständiger Sub-1G-Rauchmelder von Hyman. Es arbeitet im 433-MHz-/868-MHz-Frequenzband, nutzt fotoelektrische Rauchsensortechnologie und hat die EN 14604-Zertifizierung bestanden. Es ist kein Gateway oder Internet erforderlich. Geräte sind über die Sub-1G-Funkfrequenz direkt miteinander verbunden. Wenn ein Gerät einen Alarm auslöst, lösen alle Geräte derselben Gruppe gleichzeitig einen Alarm aus. Es eignet sich für groß angelegte Feuerverknüpfungsszenarien wie Gebäude, Einkaufszentren und Hotels.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | 433 MHz / 868 MHz Sub-1G (je nach Marktgebiet) |
| Sensortyp | Infrarot-Fotoelektrik |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | Batteriebetrieben (CR123A / AA-Lithiumbatterie) |
| Akkulaufzeit | ≥ 5 Jahre |
| Kommunikationsentfernung | Ungefähr 1 bis 3 Kilometer in offener Umgebung |
| Netzwerkkapazität | Eine einzelne Gruppe unterstützt bis zu 50+ Knoten |
| Verbindungsmethode | Drahtlose Hochfrequenzverbindung (kein Gateway erforderlich) |
| Betriebstemperatur | -10°C ~ +55°C |
| Arbeitsfeuchtigkeit | ≤ 95 % relative Luftfeuchtigkeit |
| Installationsmethode | Deckenmontage |

## Funktionen

- Photoelektrische Raucherkennung zur Erkennung von Rauch durch langsame Verbrennungen und offene Flammen
- Drahtloser Verbindungsalarm: Wird von jedem Gerät im selben Netzwerk ausgelöst, ertönen und leuchten alle Geräte gleichzeitig
- Unabhängiger Arbeitsmodus: kein Gateway, kein Internet erforderlich, direkte Kommunikation zwischen Geräten
- 85 dB hoher Ton- und Lichtalarm
- Erinnerung an niedrigen Batteriestand
- Selbsttestfunktion (One-Click-Test)
- Manipulationsalarm
-Unterstützt Remote-Stummschaltung
- Design mit extrem geringem Stromverbrauch, Batterielebensdauer von mehr als 5 Jahren

## Zertifizierungsstandards

- EN 14604 (Europäische Norm für Rauchmelder)
-CE/ROT
- CPR (EU-Bauproduktenverordnung)
