# Zertifizierungsstandards

Die S1-Serie hat die folgenden Zertifizierungen bestanden:

- **EN 14604** – Europäische Norm für Rauchmelder, siehe [EN European Standard](../../../../ApprovalStandards/EN European Standard.md)
- **CE** – Konformitätserklärung der Europäischen Union
