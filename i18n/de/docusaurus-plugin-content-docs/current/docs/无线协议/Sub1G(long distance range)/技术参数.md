# Technische Parameter

Im Folgenden sind die allgemeinen technischen Parameter der Produkte der Hyman Sub-1G-Langstreckenserie aufgeführt.

## Wireless-Parameter

| Parameter | Wert |
|------|-----|
| Betriebsfrequenzband | 433 MHz / 868 MHz / 915 MHz (abhängig vom Marktgebiet) |
| Modulationsmethode | FSK / OOK |
| Übertragungsrate | Ungefähr 10–100 kbps (Videosegment und Konfiguration) |
| Netzwerktopologie | Stern-/Punkt-zu-Punkt-/Internet-Vernetzung |
| Kommunikationsentfernung | Ungefähr 1 bis 3 Kilometer in offener Umgebung (Videosegment und Leistung) |
| Netzwerkkapazität | Eine einzelne Gruppe unterstützt bis zu 50+ Knoten |

## Regionale Frequenzbandzuteilung

| Region | Gemeinsame Frequenzbänder |
|------|----------|
| China | 433 MHz |
| Europa | 868 MHz |
| Nordamerika | 915 MHz |

## Strom- und Stromverbrauch

| Parameter | Wert |
|------|-----|
| Stromversorgungsmethode | Batteriebetrieben (CR123A / AA-Lithiumbatterie) |
| Akkulaufzeit | Etwa 3 bis 5 Jahre (abhängig von der Berichtshäufigkeit) |
| Sendeleistung | ≤ 10 mW (+10 dBm), konform mit den SRD-Vorschriften |
| Standby-Stromverbrauch | Mikroampere-Pegel, extrem niedrig |
| Empfangsempfindlichkeit | Ungefähr -110 dBm ~ -120 dBm |

## Umgebungsparameter

| Parameter | Wert |
|------|-----|
| Betriebstemperatur | -10°C ~ +55°C |
| Betriebsfeuchtigkeit | ≤ 95 % relative Luftfeuchtigkeit (nicht kondensierend) |
