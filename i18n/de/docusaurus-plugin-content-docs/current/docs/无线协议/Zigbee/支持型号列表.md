## Anweisungen zur Modellbenennung

| Benennungsregeln | Bedeutung |
|----------|------|
| Suffix -E | EU-Version (868 MHz / EN-Standard) |
| Suffix -Z | China-Version (2,4 GHz / chinesischer Standard) |
| HS1/2/3/8 | Produktgenerationsnummer, je größer die Zahl, desto neuer die Generation |
