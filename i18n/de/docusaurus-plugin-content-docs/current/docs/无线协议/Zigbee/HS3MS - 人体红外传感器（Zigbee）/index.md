#HS3MS – Infrarotsensor für den menschlichen Körper (Zigbee)

## Übersicht

HS3MS ist Hymans Zigbee-Infrarotsensor für den menschlichen Körper der dritten Generation. Es verfügt über ein 360°-Deckenmontagedesign und seine Erkennungsempfindlichkeit und Abdeckung sind im Vergleich zur Vorgängergeneration deutlich verbessert.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Passives Infrarot (PIR) |
| Erfassungswinkel | 360° (Deckenmontage) |
| Erkennungsentfernung | ≤7 m (25°C) |
| Stromversorgungsmethode | DC 3V Lithiumbatterie |
| Akkulaufzeit | Ungefähr 3 Jahre |
| Betriebstemperatur | -10°C ~ +50°C |
| Installationsmethode | Deckenmontage |

## Funktionen

- Mobile Personenerkennung
- 360° omnidirektionale Abdeckung
- Erinnerung an niedrigen Batteriestand
- Manipulationsalarm
- Verbesserte Erkennungsempfindlichkeit
- Unterstützung der Szenenverknüpfung
