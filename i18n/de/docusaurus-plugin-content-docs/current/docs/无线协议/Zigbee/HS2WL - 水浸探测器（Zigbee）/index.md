#HS2WL – Wasserleckdetektor (Zigbee)

## Übersicht

HS2WL ist der von Hyman auf den Markt gebrachte Zigbee-Wassertauchdetektor der zweiten Generation. Es unterstützt die Zweikanalerkennung von Sonden und Kabeln und kann zur Erkennung von Wasserlecks wie Wasserrohrbrüchen und Überläufen von Spülbecken verwendet werden.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Erkennungsmethode | Sondenerkennung + Kabelerkennung (Zweikanal) |
| Stromversorgungsmethode | DC 3V-Batterie |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % relative Luftfeuchtigkeit |

## Funktionen

- Echtzeiterkennung von Wasserlecks/Wasseransammlungen
- Zweikanalige Erkennung von Sonde und Kabel
- Summeralarm
- Erinnerung an niedrigen Batteriestand
- Korrosionsschutzdesign der Sonde

## Anwendbares Zubehör

- Detektionskabel verlängern
