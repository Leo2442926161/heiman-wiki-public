# HS8MLS – Mikrobewegungssensor (Zigbee)

## Übersicht

HS8MLS ist ein Zigbee-Infrarotsensor (PIR) für den menschlichen Körper mit großer Reichweite, der von Heiman auf den Markt gebracht wurde. Es wurde speziell für Szenen entwickelt, die eine Erkennung aus großer Entfernung erfordern, und eignet sich für offene Räume wie Korridore und Hallen.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Passives Infrarot (PIR) |
| Erkennungsentfernung | Fernerkennung |
| Stromversorgungsmethode | DC 3V-Batterie |

## Funktionen

- Bewegungserkennung über große Entfernungen
- Design mit geringem Stromverbrauch
- Unterstützung der Szenenverknüpfung
