# HS6GW – Smart Gateway (Zigbee)

## Übersicht

HS6GW ist ein von Heiman eingeführtes Zigbee-Smart-Gateway. Als Koordinator des Zigbee-Netzwerks von Heiman stellt es über WLAN oder Ethernet-Uplink eine Verbindung zum Internet her und unterstützt den Zugriff von bis zu 128 Untergeräten.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 (Koordinator) |
| Uplink-Verbindung | WLAN / Ethernet |
| Max. Knoten | 128 Untergeräte |
| Stromversorgungsmethode | DC 5V USB-Netzteil |

## Funktionen

- Koordinator des Zigbee-Netzwerks
- Wi-Fi/Ethernet-Uplink-Verbindung
- Automatische Geräteerkennung und -kopplung
- Lokale Szenenverknüpfung
- Fernzugriff (über Heiman Cloud Platform)
- Unterstützt bis zu 128 Untergeräte
- OTA-Firmware-Upgrade

## Anwendbare Szenarien

- Der zentrale Verbindungspunkt der Heiman-Geheimdienste für das ganze Haus
- Arbeiten Sie mit der Haiman APP zusammen, um eine Fernüberwachung zu realisieren
