# Zertifizierungsstandards

HS6GW (Zigbee-Version) hat die folgenden Zertifizierungen bestanden:

- **CE/RED** – Richtlinie der Europäischen Union für drahtlose Geräte
- **CCC** – China Compulsory Product Certification, siehe [CCC China Compulsory Certification](../../../../Certification Standards/CCC China Compulsory Certification.md)
