#HS8MIS - Mobiler Sensor (Zigbee)

## Übersicht

HS8MIS ist ein multifunktionaler Zigbee-Bewegungssensor von Heiman. Es integriert PIR-Funktionen zur Personenerkennung und Beleuchtungserkennung und eignet sich für intelligente Szenen mit doppelter Überwachung von Licht und Bewegung.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Passives Infrarot (PIR) + Beleuchtung |
| Erfassungswinkel | 110° |
| Erkennungsentfernung | ≤7 m |
| Stromversorgungsmethode | DC 3V-Batterie |

## Funktionen

- Mobile Personenerkennung
- Beleuchtungserkennung
- Design mit geringem Stromverbrauch
- Unterstützung der Szenenverknüpfung
