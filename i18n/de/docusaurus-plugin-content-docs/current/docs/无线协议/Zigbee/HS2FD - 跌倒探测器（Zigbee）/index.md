#HS2FD – Sturzdetektor (Zigbee)

## Übersicht

HS2FD ist ein intelligenter Zigbee-Sturzdetektor, der von Hyman eingeführt wurde. Es ist speziell für ältere Menschen konzipiert. Es verfügt über einen eingebauten Beschleunigungsmesser und ein Gyroskop. Mithilfe eines KI-Algorithmus erkennt es die Sturzhaltung und kann automatisch ein Notsignal aussenden.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Beschleunigungsmesser + Gyroskop |
| Erkennungstechnologie | KI-Algorithmus zur Sturzerkennung |
| Stromversorgungsmethode | DC 3V-Batterie |
| Trageweise | Gürtelclip/Schlüsselband |

## Funktionen

- Automatische Sturzerkennung
- Rufen Sie mit einem Klick manuell um Hilfe
- Der KI-Algorithmus unterscheidet Stürze von normalen Aktivitäten (Sitzen, Bücken), um Fehlalarme zu reduzieren
- Tragbares Design (Gürtelclip oder Lanyard)
- Erinnerung an niedrigen Batteriestand

## Anwendbare Szenarien

- Häusliche Altenpflege: Rund-um-die-Uhr-Betreuung allein lebender älterer Menschen
- Pflegeheime: zentrale Betreuung und Verwaltung
- Rehabilitationszentrum: Überwachung des Sturzrisikos bei postoperativen Patienten
