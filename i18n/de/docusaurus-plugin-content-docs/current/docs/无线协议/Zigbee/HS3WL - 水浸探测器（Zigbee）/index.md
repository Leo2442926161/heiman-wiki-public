# HS3WL – Wasserleckdetektor (Zigbee)

## Übersicht

HS3WL ist der Zigbee-Wassertauchdetektor der dritten Generation von Hyman. Es verfügt über eine verbesserte Erkennungsempfindlichkeit und drahtlose Leistung im Vergleich zur vorherigen Generation und eignet sich für Szenarien zur Überwachung von Wasserlecks.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Erkennungsmethode | Sondenerkennung |
| Stromversorgungsmethode | DC 3V-Batterie |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % relative Luftfeuchtigkeit |

## Funktionen

- Echtzeiterkennung von Wasserlecks/Wasseransammlungen
- Erhöhte Erkennungsempfindlichkeit
- Summeralarm
- Erinnerung an niedrigen Batteriestand
- Korrosionsschutzdesign der Sonde
