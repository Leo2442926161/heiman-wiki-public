# JT-Serie – Gas-Notabsperrventil (Zigbee)

## Übersicht

Das Gas-Notabsperrventil der JT-Serie ist ein Ausführungsgerät, das in Verbindung mit dem Hyman Zigbee-Detektor für brennbare Gase verwendet wird. Wenn ein Gasleck erkannt wird, kann das Ventil der Gasleitung aus der Ferne oder automatisch geschlossen werden, um Unfälle zu verhindern.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Geben Sie | ein Eigenständig + Zigbee-Verknüpfung |
| Prüfgas | Erdgas / Flüssiggas |
| Zertifizierungsstandards | GB 15322 |
| Betriebstemperatur | -10°C ~ +50°C |

## Funktionen

- Verbunden mit HS1CG / HS3CG zum automatischen Schließen des Ventils
-Unterstützt das manuelle Schließen des Ventils aus der Ferne
- Leckalarm-Taster
- Magnetventiltreiber

## Zertifizierungsstandards

- GB 15322

## Anwendbares Zubehör

- HS1CG / HS3CG Detektor für brennbare Gase
