# HS1CA-E – Intelligenter Kohlenmonoxidmelder (Zigbee)

## Übersicht

HS1CA-E ist der intelligente Zigbee-Kohlenmonoxidmelder (CO) von Hyman für den EU-Markt. Es verwendet einen elektrochemischen Sensor und kann Kohlenmonoxidgas mit einer Konzentration von 0 bis 1000 ppm genau erkennen. Es hat die EN 50291-Zertifizierung bestanden.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Elektrochemisch |
| Erfassungsbereich | 0 ~ 1000 ppm |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | DC 3V (CR123A) |
| Sensorlebensdauer | Ungefähr 7 Jahre |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % relative Luftfeuchtigkeit |
| Installationsmethode | Wandmontage |

## Funktionen

- Hochpräziser elektrochemischer CO-Sensor
- Konzentrationsabhängiger Alarm (Ton- und Lichtaufforderung)
- 85 dB Alarmton
- Erinnerung an niedrigen Batteriestand
- Erinnerung an den Ablauf der Sensorlebensdauer
- Selbsttestfunktion
- OTA-Firmware-Upgrade

## Zertifizierungsstandards

- EN 50291 (Europäische Norm für Kohlenmonoxidmelder)
-CE/ROT

## Anwendbares Zubehör

- CR123A-Batterie
