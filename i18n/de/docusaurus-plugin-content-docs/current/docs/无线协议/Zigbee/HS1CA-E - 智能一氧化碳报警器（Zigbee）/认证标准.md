# Zertifizierungsstandards

HS1CA-E hat die folgenden Zertifizierungen bestanden:

- **EN 50291** – Europäische Norm für Kohlenmonoxidmelder, siehe [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **CE/RED/EMC** – Richtlinie der Europäischen Union für drahtlose Geräte und elektromagnetische Verträglichkeit
