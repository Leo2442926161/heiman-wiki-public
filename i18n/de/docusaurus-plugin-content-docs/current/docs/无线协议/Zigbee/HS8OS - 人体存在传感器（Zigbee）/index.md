#HS8OS – Menschlicher Anwesenheitssensor (Zigbee)

## Übersicht

HS8OS ist ein von Hyman eingeführter Zigbee-Anwesenheitssensor. Es nutzt Millimeterwellen-Radartechnologie und kann die Anwesenheit des menschlichen Körpers in einem statischen Zustand (z. B. Liegen, ruhiges Sitzen) erkennen und so die Mängel herkömmlicher PIR-Sensoren ausgleichen, die nur Bewegungen erkennen können.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Millimeterwellenradar (mmWave) |
| Erfassungsbereich | Radius 3 ~ 5 m |
| Erkennungsfähigkeit | Vorhandensein eines stationären menschlichen Körpers (einschließlich Mikroatmungserkennung) |
| Stromversorgungsmethode | Wechselstromversorgung |
| Installationsmethode | Deckenmontage |

## Funktionen

- Erkennung stationärer menschlicher Körper (Mikrobewegungen der Atmung)
- Abdeckungsradius 3~5 Meter
- Deckenmontage, optimaler Erfassungswinkel
- Kann ergänzend zu PIR-Sensoren verwendet werden

## Anwendbare Szenarien

- Intelligente Beleuchtung: immer an, wenn jemand da ist, automatisch aus, wenn niemand da ist
- Energieeinsparung der Klimaanlage: Schaltet die Klimaanlage automatisch aus, wenn Personen gehen
- Intelligente Altenpflege: Frühwarnung für ältere Menschen, wenn sie längere Zeit stationär bleiben
