#HS3SA – Intelligenter Rauchmelder (Zigbee)

## Übersicht

HS3SA ist der intelligente Zigbee-Rauchmelder der dritten Generation von Hyman. Es verfügt über eine verbesserte Erkennungsleistung und Verknüpfungsfähigkeiten gegenüber der ersten Generation und ist für den EU-Markt geeignet.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Infrarot-Fotoelektrik |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | DC 3V (CR123A) |
| Akkulaufzeit | Ungefähr 3 Jahre |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % relative Luftfeuchtigkeit |
| Installationsmethode | Deckenmontage |

## Funktionen

- Photoelektrische Raucherkennung
- 85 dB hoher Ton- und Lichtalarm
- Erinnerung an niedrigen Batteriestand
- Selbsttestfunktion
- Manipulationsalarm
-Unterstützt Remote-Stummschaltung
- OTA-Firmware-Upgrade
- Verbesserte WLAN-Leistung

## Zertifizierungsstandards

-CE/ROT

## Anwendbares Zubehör

- CR123A-Batterie
