#Zigbee

Zigbee ist ein drahtloses Mesh-Netzwerkprotokoll mit geringem Stromverbrauch und niedriger Datenrate, das auf dem IEEE 802.15.4-Standard basiert. Es ist eines der ausgereiftesten und beliebtesten Protokolle im Smart-Home-Bereich.

## Hyman und Zigbee

Hyman ist einer der ersten chinesischen Hersteller, der das Zigbee-Protokoll übernommen hat, und verfügt über eine riesige Zigbee-Produktlinie. Hyman Zigbee-Geräte sind mit dem **Zigbee 3.0**-Standard kompatibel und können mit gängigen Zigbee-Gateways und -Koordinatoren zusammenarbeiten.

## Technische Spezifikationen

| Funktionen | Beschreibung |
|------|------|
| Frequenzband | 2,4 GHz (weltweit) |
| Übertragungsrate | Bis zu 250 kbit/s |
| Netzwerktopologie | Mesh-Selbstheilungsnetzwerk |
| Anzahl der Knoten | Ein einzelnes Netzwerk unterstützt bis zu 65535 Knoten |
| Kommunikationsentfernung | Etwa 10-30 Meter im Innenbereich (sichtbare Hindernisse) |
| Stromverbrauch | Extrem niedrige Sensorbatterielebensdauer bis zu 3 Jahre |

## Kompatible Plattformen

Hyman Zigbee-Geräte können mit den folgenden Plattformen verbunden werden:

- **Zigbee2MQTT** – Open-Source-Zigbee-zu-MQTT-Brücke
- **ZHA** – Home Assistant mit integrierter Zigbee-Integration
- **Hubitat** – Lokalisierter intelligenter Hub
- **Homey** – Multiprotokoll-Smart-Hub
- **SmartThings Hub** – Samsung Smart Home Hub
- **Tuya Zigbee Gateway** – Tuya-Ökosystem
- **Hyman HS6GW Gateway** – Offizielles Gateway

## Hyman Zigbee-Produktlinie

- Rauchmelder der Serien HS1SA / HS3SA / HS8SA
- Kohlenmonoxidmelder der Serie HS1CA/C1
- Detektoren für brennbare Gase der Serien HS1CG / HS3CG
- Türsensor der HS1DS-Serie
- Infrarotsensoren für den menschlichen Körper der Serien HS1MS/HS3MS/M1
- Wassertauchmelder der Serien HS2WL / HS3WL / HS1WL
- Intelligente Steckdosen der HS2SK-Serie
- HS2FD-Sturzdetektor
