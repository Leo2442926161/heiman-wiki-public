# M1-Z – Infrarotsensor für den menschlichen Körper (Zigbee)

## Übersicht

M1-Z ist ein kompakter Zigbee-Infrarotsensor für den menschlichen Körper, der von Heiman auf den Markt gebracht wurde. Es unterstützt einen Erfassungswinkel von 110° oder 360° (abhängig von der Installationsmethode), verfügt über einen integrierten Beleuchtungssensor und eignet sich für intelligente Beleuchtungs- und Sicherheitsszenarien.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Passives Infrarot (PIR) + Beleuchtung |
| Erfassungswinkel | 110° (Wandmontage) / 360° (Decke) |
| Erkennungsentfernung | ≤7 m (25°C) |
| Beleuchtungsstärkebereich | 0 ~ 1200 Lux |
| Stromversorgungsmethode | DC 3V Lithiumbatterie |
| Akkulaufzeit | Ungefähr 3 Jahre |
| Betriebstemperatur | -10°C ~ +50°C |
| Installationsmethode | Wandmontage/Decke |

## Funktionen

- Mobile Personenerkennung
- Beleuchtungserkennung (0~1200 Lux)
- 110° / 360° umschaltbar
- Erinnerung an niedrigen Batteriestand
- Manipulationsalarm
- Kompaktes Design
