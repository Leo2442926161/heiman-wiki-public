# C1-Serie – Kohlenmonoxidmelder (Zigbee)

## Übersicht

Die C1-Serie ist ein einfacher Zigbee-Kohlenmonoxidmelder von Heiman. Es verwendet elektrochemische Sensoren und eignet sich für Szenarien zur Erkennung von Kohlenmonoxidlecks. Es hat die EN 50291-Zertifizierung bestanden.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Elektrochemisch |
| Erfassungsbereich | 0 ~ 1000 ppm |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | Batteriebetrieben |
| Sensorlebensdauer | Ungefähr 7 Jahre |
| Betriebstemperatur | -10°C ~ +50°C |

## Funktionen

- Elektrochemischer CO-Nachweis
- Konzentrationsklassifizierungsalarm
- 85 dB akustischer und optischer Alarm
- Erinnerung an niedrigen Batteriestand
- Selbsttestfunktion

## Zertifizierungsstandards

-EN 50291
