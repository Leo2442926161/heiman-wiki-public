# HS1WL – Wasserleckdetektor (Zigbee)

## Übersicht

HS1WL ist der von Hyman auf den Markt gebrachte Zigbee-Wassertauchdetektor der ersten Generation. Es nutzt die Sondenerkennungsmethode und kann Wasserlecks oder Wasseransammlungen effektiv erkennen. Es handelt sich um ein grundlegendes Gerät zur Überwachung des Eintauchens in Wasser.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Erkennungsmethode | Sondenerkennung |
| Stromversorgungsmethode | DC 3V-Batterie |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % relative Luftfeuchtigkeit |

## Funktionen

- Echtzeiterkennung von Wasserlecks/Wasseransammlungen
- Summeralarm
- Erinnerung an niedrigen Batteriestand
- Korrosionsschutzdesign der Sonde
