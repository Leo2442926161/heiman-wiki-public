# HS1DS – Türmagnetsensor (Zigbee)

## Übersicht

HS1DS ist ein von Heiman eingeführter Zigbee-Türmagnetsensor. Es dient zur Erkennung des Öffnungs- und Schließzustands von Türen und Fenstern. Es ist ein Grundbestandteil des intelligenten Sicherheitssystems.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Reed-Schalter |
| Stromversorgungsmethode | CR1632-Knopfbatterie |
| Akkulaufzeit | Ungefähr 2 ~ 3 Jahre |
| Einbaufreiraum | ≤10 mm |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % relative Luftfeuchtigkeit |

## Funktionen

- Echtzeiterkennung des Öffnungs-/Schließstatus von Türen und Fenstern
- Manipulationsalarm (Tamper)
- Erinnerung an niedrigen Batteriestand
-Unterstützt die Verknüpfung (Öffnen der Tür, Einschalten des Lichts, Scharfschalten beim Verlassen des Hauses usw.)
- Einfache Montage (3M-Klebeband oder Schraubbefestigung)

## Anwendbares Zubehör

- CR1632-Knopfbatterie
- 3M doppelseitiges Klebeband/Schrauben
