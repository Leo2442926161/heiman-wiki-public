# HS1SA-E – Intelligenter Rauchmelder (Zigbee)

## Übersicht

HS1SA-E ist ein standardmäßiger intelligenter Zigbee-Rauchmelder, der von Hyman für den EU-Markt eingeführt wurde. Es verfügt über eine fotoelektrische Raucherkennungstechnologie, hat die EN 14604-Zertifizierung bestanden und den Red Dot Design Award gewonnen.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Infrarot-Fotoelektrik |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | DC 3V (CR123A) |
| Akkulaufzeit | Ungefähr 3 Jahre |
| Kommunikationsentfernung | ≤70 m (offene Fläche) |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % relative Luftfeuchtigkeit |
| Installationsmethode | Deckenmontage |

## Funktionen

- Photoelektrische Raucherkennung zur Erkennung von Rauch durch langsame Verbrennungen und offene Flammen
- 85 dB hoher Ton- und Lichtalarm
- Erinnerung an niedrigen Batteriestand
- Selbsttestfunktion (One-Click-Test)
- Manipulationsalarm
-Unterstützt Remote-Stummschaltung
- OTA-Firmware-Upgrade

## Zertifizierungsstandards

- EN 14604 (Europäische Norm für Rauchmelder)
- Red Dot Design Award
-CE/ROT

## Anwendbares Zubehör

- CR123A-Batterie
- Befestigungsschrauben/Expansionsrohr
