#HS2SK - Intelligente Steckdose (Zigbee)

## Übersicht

HS2SK ist eine von Heiman eingeführte Zigbee-Smart-Metering-Steckdose. Es unterstützt die Fernsteuerung von Schaltern und die Messung des Stromverbrauchs in Echtzeit. Es eignet sich zur intelligenten Steuerung von Haushaltsgeräten und hat eine Genauigkeit von ±2 %.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Nennspannung | AC 220 V (chinesischer Standard) / AC 110 V (amerikanischer Standard) |
| Maximale Belastung | 16A / 3680W |
| Messgenauigkeit | ±2% |
| Stromversorgungsmethode | Netzstromversorgung |

## Funktionen

- Fernschaltersteuerung
- Echtzeit-Leistungs-/Kraftstoffmessung
- Überlastschutz
- Zeitschaltuhr
- Leistungsstatistik
- Unterstützung der Szenenverknüpfung

## Anwendbare Szenarien

- Schalten Sie vergessene Geräte aus der Ferne aus
- Statistik des Energieverbrauchs der Klimaanlage/des Warmwasserbereiters
- Verlassen Sie den Home-Modus, um nicht unbedingt benötigte Geräte mit einem Klick auszuschalten
