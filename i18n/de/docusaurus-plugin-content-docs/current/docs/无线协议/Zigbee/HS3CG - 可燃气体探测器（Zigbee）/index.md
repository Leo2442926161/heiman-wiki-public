#HS3CG – Detektor für brennbare Gase (Zigbee)

## Übersicht

HS3CG ist Hymans Zigbee-Detektor für brennbare Gase der dritten Generation. Es verfügt über eine verbesserte Erkennungsgenauigkeit und Reaktionsgeschwindigkeit gegenüber der ersten Generation und eignet sich für die Lecksuche bei Erdgas und Flüssiggas.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Halbleiter |
| Prüfgas | Erdgas (Methan)/Flüssiggas (LPG) |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | AC 220V / DC 12V |
| Zertifizierungsstandards | GB 15322 |
| Betriebstemperatur | -10°C ~ +50°C |

## Funktionen

- Erdgas-/Flüssiggas-Leckerkennung
- Konzentrationsklassifizierungsalarm
- Automatisches Abschaltgestänge (mit Magnetventil)
- 85 dB akustischer und optischer Alarm
- Selbsttestfunktion
- Erinnerung an den Ablauf der Sensorlebensdauer
- Verbesserte WLAN-Leistung

## Zertifizierungsstandards

- GB 15322

## Anwendbares Zubehör

- Gas-Notabsperrventil der Serie JT
