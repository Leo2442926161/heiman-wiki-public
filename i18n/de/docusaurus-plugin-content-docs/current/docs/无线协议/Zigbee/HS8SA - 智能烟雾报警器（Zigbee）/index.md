#HS8SA – Intelligenter Rauchmelder (Zigbee)

## Übersicht

HS8SA ist der intelligente Zigbee + Wi-Fi-Rauchmelder von Hyman für den US-Markt. Es verfügt über eine Hochtemperatur-Verbindungsfunktion und hat die UL 217-Zertifizierung bestanden.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 / WLAN |
| Sensortyp | Infrarot-Fotoelektrik |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | DC 3V (CR123A) |
| Betriebstemperatur | -10°C ~ +50°C |
| Zertifizierungsstandards | UL 217 |
| Anwendbare Märkte | Vereinigte Staaten |

## Funktionen

- Photoelektrische Raucherkennung
- Gestängealarm bei hoher Temperatur
- 85 dB hoher Ton- und Lichtalarm
- Erinnerung an niedrigen Batteriestand
- Selbsttestfunktion
- Manipulationsalarm
-Unterstützt Remote-Stummschaltung
- Dual-Protokoll-Kommunikation (Zigbee + Wi-Fi)

## Zertifizierungsstandards

- UL 217 (amerikanischer Rauchmelderstandard)
