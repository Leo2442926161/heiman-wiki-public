# Zertifizierungsstandards

HS8SA hat die folgenden Zertifizierungen bestanden:

- **EN 14604** – Europäische Norm für Rauchmelder (EU-Typ), siehe [EN European Standard](../../../../ApprovalStandards/EN European Standard.md)
- **UL 217** – US-amerikanischer Sicherheitsstandard für Rauchmelder (US-Modell), siehe [UL Underwriters Laboratories](../../../../certification-standards/ULUnderwriters Laboratories.md)
- **CE/RED/EMC** – Richtlinie der Europäischen Union für drahtlose Geräte und elektromagnetische Verträglichkeit
- **Zigbee 3.0 / Q-Label** – Zigbee Alliance-Zertifizierung, siehe [Andere Zertifizierungen](../../../Certification Standards/Other Certifications.md)
