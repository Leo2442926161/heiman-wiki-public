#HS1MS – Infrarotsensor für den menschlichen Körper (Zigbee)

## Übersicht

HS1MS ist ein standardmäßiger Zigbee-Infrarotsensor (PIR) für den menschlichen Körper, der von Hyman auf den Markt gebracht wurde. Es nutzt passive Infrarot-Erkennungstechnologie und kann die von einem sich bewegenden menschlichen Körper abgegebene Infrarot-Wärmestrahlung erfassen. Es wird häufig in intelligenten Beleuchtungs- und Sicherheitsszenarien eingesetzt.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Zigbee 3.0 |
| Sensortyp | Passives Infrarot (PIR) |
| Erfassungswinkel | 110° |
| Erkennungsentfernung | ≤7 m (25°C) |
| Stromversorgungsmethode | DC 3V Lithiumbatterie |
| Akkulaufzeit | Ungefähr 3 Jahre |
| Betriebstemperatur | -10°C ~ +50°C |
| Installationsmethode | Wandmontage |

## Funktionen

- Mobile Personenerkennung
- 110° Weitwinkelerkennung
- Erinnerung an niedrigen Batteriestand
- Manipulationsalarm
- Unterstützung der Szenenverknüpfung
