# Technische Parameter

Im Folgenden sind allgemeine technische Parameter für Heiman Zigbee-Geräte aufgeführt.

## Wireless-Parameter

| Parameter | Wert |
|------|-----|
| Protokollstandards | Zigbee 3.0 (IEEE 802.15.4) |
| Arbeitsfrequenzband | 2,4 GHz (globales gemeinsames ISM-Frequenzband) |
| Übertragungsrate | Bis zu 250 kbit/s |
| Netzwerktopologie | Mesh-Selbstheilungsnetzwerk |
| Knotenkapazität | Maximal 65535 Knoten in einem einzelnen Netzwerk |
| Kommunikationsentfernung | Etwa 10 bis 30 Meter im Innenbereich (Wandhindernisse sehen) |
| Verschlüsselungsmethode | AES-128-CCM |

## Kompatibilität

| Projekt | Beschreibung |
|------|------|
| Zigbee-Koordinator | Kompatibel mit dem Zigbee 3.0-Standardkoordinator |
| Unterstützte Plattformen | Zigbee2MQTT, ZHA (Home Assistant), Hubitat, Homey, SmartThings, Tuya Zigbee Gateway, Haiman HS6GW Gateway |
| OTA-Upgrade | Unterstützt Over-the-Air-Firmware-Upgrade (einige Modelle) |

## Strom- und Stromverbrauch

| Parameter | Wert |
|------|-----|
| Stromversorgungsmethode | Batteriebetrieben (CR123A / CR2450 / AA) oder USB / Netz (Steckdosentyp) |
| Akkulaufzeit | Etwa 2–3 Jahre für Sensoren (abhängig von der Meldehäufigkeit) |
| Standby-Stromverbrauch | Mikroampere-Pegel, unterstützt Sleep-Polling-Mechanismus |

## Umgebungsparameter

| Parameter | Wert |
|------|-----|
| Betriebstemperatur | -10°C ~ +55°C |
| Betriebsfeuchtigkeit | ≤ 95 % relative Luftfeuchtigkeit (nicht kondensierend) |
| IP-Schutzstufe | Hängt vom jeweiligen Modell ab (Sensortyp normalerweise IP30) |
