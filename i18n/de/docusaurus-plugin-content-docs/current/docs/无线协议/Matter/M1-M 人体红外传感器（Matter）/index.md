# M1-M Infrarotsensor für den menschlichen Körper (Materie)

M1-M ist ein **Matter over Thread**-Infrarotsensor für den menschlichen Körper, der von Hyman auf den Markt gebracht wurde. Es integriert PIR-Erkennungs- und Beleuchtungserkennungsfunktionen und eignet sich für Smart-Home-Automatisierungsszenarien.

## Produktübersicht

- **Produkttyp**: Infrarotsensor für den menschlichen Körper
- **Kommunikationsprotokoll**: Materie über Thread
- **Erkennungsmethode**: Passives Infrarot (PIR) + Beleuchtungserkennung
- **Typisches Szenario**: Lichter gehen an, wenn Leute kommen, Sicherheitsverknüpfung, Umgebungsautomatisierung

## Technische Parameter

| Parameter | Wert |
|------|-----|
| Erfassungswinkel | 110° (Wandmontage) / 360° (Decke) |
| Erkennungsentfernung | ≤7 m (25°C) |
| Beleuchtungsbereich | 0~1200 Lux |
| Kommunikationsprotokoll | Materie über Thread (IEEE 802.15.4) |
| Betriebsfrequenzband | 2,4 GHz |
| Stromversorgungsmethode | DC 3V Lithiumbatterie |
| Akkulaufzeit | Ungefähr 1 Jahr (Matter-Modus) |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % RH (keine Kondensation) |

## Funktionen

- Mobile Personenerkennung
- Erkennung der Umgebungsbeleuchtung
- Erkennungsentfernung ≤7m
- Unterstützt Wand- oder Deckenmontage
- Materielle multiökologische plattformübergreifende Interoperabilität
