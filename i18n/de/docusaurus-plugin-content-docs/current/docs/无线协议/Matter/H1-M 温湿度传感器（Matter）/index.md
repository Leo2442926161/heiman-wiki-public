# H1-M Temperatur- und Feuchtigkeitssensor (Matter)

H1-M ist ein von Hyman eingeführter **Matter over Thread**-Temperatur- und Feuchtigkeitssensor, der die Umgebungstemperatur und die relative Luftfeuchtigkeit in Echtzeit überwachen kann und die Verknüpfung mit der Smart-Home-Automatisierung unterstützt.

## Produktübersicht

- **Produkttyp**: Temperatur- und Feuchtigkeitssensor
- **Kommunikationsprotokoll**: Materie über Thread
- **Erkennungsparameter**: Temperatur, relative Luftfeuchtigkeit
- **Typische Szenarien**: Umweltüberwachung in Häusern, Büros, Weinkellern usw.

## Technische Parameter

| Parameter | Wert |
|------|-----|
| Temperaturgenauigkeit | ±0,3°C |
| Luftfeuchtigkeitsgenauigkeit | ±3 % relative Luftfeuchtigkeit |
| Kommunikationsprotokoll | Materie über Thread (IEEE 802.15.4) |
| Betriebsfrequenzband | 2,4 GHz |
| Stromversorgungsmethode | CR2-Lithiumbatterie |
| Akkulaufzeit | Ungefähr 2 bis 3 Jahre |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % RH (keine Kondensation) |

## Funktionen

- Echtzeitüberwachung von Temperatur und Luftfeuchtigkeit
- Hochpräzise Erfassung (Temperatur ±0,3 °C, Luftfeuchtigkeit ±3 % RH)
- Erinnerung an niedrigen Batteriestand
- Materielle multiökologische plattformübergreifende Interoperabilität

## Kompatible Plattformen

| Plattform | Unterstützung |
|------|----------|
| Apple Home | ✅ Thread-Grenzfräser erforderlich |
| Google-Startseite | ✅ Thread-Grenzfräser erforderlich |
| Amazon Alexa | ✅ Thread-Grenzfräser erforderlich |
| Heimassistent | ✅ |
| SmartThings | ✅ Erfordert Thread-Grenzfräser |

## Anforderungen an die Grenzführung

Es ist ein Thread-Border-Router erforderlich, z. B. Apple HomePod mini, Google Nest Hub 2. Generation, Amazon Echo 4. Generation usw.
