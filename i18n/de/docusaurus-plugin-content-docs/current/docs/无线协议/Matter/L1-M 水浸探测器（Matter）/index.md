# L1-M Wasserimmersionsdetektor (Materie)

L1-M ist ein von Hyman eingeführter **Matter over Thread**-Wasserimmersionsdetektor. Über eine Sonde werden Wasserlecks oder angesammeltes Wasser erkannt, wodurch Sachschäden durch Wasserrohrbrüche wirksam verhindert werden können.

## Produktübersicht

- **Produkttyp**: Wassereintaucherkennungssensor
- **Kommunikationsprotokoll**: Materie über Thread
- **Erkennungsmethode**: Sondenkontakterkennung
- **Typische Szenarien**: Küche, Badezimmer, Waschküche, Erkennung von Wasserlecks im Keller

## Technische Parameter

| Parameter | Wert |
|------|-----|
| Erkennungsmethode | Sondenkontakttyp aus Edelstahl |
| Kommunikationsprotokoll | Materie über Thread (IEEE 802.15.4) |
| Betriebsfrequenzband | 2,4 GHz |
| Stromversorgungsmethode | Batteriebetrieben |
| Alarmmethode | Summer + App-Push |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % RH (keine Kondensation) |

## Funktionen

- Echtzeiterkennung von Wasserlecks/stehendem Wasser
- Summer-Alarm vor Ort
- App-Fernalarm-Push
- Erinnerung an niedrigen Batteriestand
- Korrosionsschutzdesign der Sonde
- Materielle multiökologische plattformübergreifende Interoperabilität

## Kompatible Plattformen

| Plattform | Unterstützung |
|------|----------|
| Apple Home | ✅ Thread-Grenzfräser erforderlich |
| Google-Startseite | ✅ Thread-Grenzfräser erforderlich |
| Amazon Alexa | ✅ Thread-Grenzfräser erforderlich |
| Heimassistent | ✅ |

## Anforderungen an die Grenzführung

Es ist ein Thread-Border-Router erforderlich, z. B. Apple HomePod mini, Google Nest Hub 2. Generation, Amazon Echo 4. Generation usw.
