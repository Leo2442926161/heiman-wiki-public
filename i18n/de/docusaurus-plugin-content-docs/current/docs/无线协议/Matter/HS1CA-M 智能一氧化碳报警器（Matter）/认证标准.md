# Zertifizierungsstandards

HS1CA-M hat die folgenden Zertifizierungen bestanden:

- **EN 50291** – Europäische Norm für Kohlenmonoxidmelder, siehe [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **CE-Kennzeichnung** – EU-Konformitätserklärung, siehe [CE EU-Konformität](../../../../certification-standards/CE EU-Conformity.md)
- **Matter-Zertifizierung** – Hat die Matter-Zertifizierung der CSA Alliance bestanden, um die plattformübergreifende Verbindung von Smart Homes sicherzustellen
