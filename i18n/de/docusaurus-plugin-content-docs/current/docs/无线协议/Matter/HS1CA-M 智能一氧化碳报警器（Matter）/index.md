# HS1CA-M Intelligenter Kohlenmonoxidmelder (Matter)

HS1CA-M ist der weltweit erste Matter-zertifizierte Kohlenmonoxidmelder**, der elektrochemische Sensortechnologie nutzt und die kreuzökologische Verbindung „Matter over Thread“ unterstützt.

## Produktübersicht

- **Produkttyp**: Intelligenter Kohlenmonoxidmelder
- **Kommunikationsprotokoll**: Materie über Thread
- **Zertifizierungsstandard**: EN 50291
- **Typisches Szenario**: CO-Leckagewarnung in Häusern, Wohnungen und anderen Orten

## Technische Parameter

| Parameter | Wert |
|------|-----|
| Sensortyp | Elektrochemischer CO-Sensor |
| Kommunikationsprotokoll | Materie über Thread (IEEE 802.15.4) |
| Betriebsfrequenzband | 2,4 GHz |
| Erfassungsbereich | 0~1000 ppm |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | DC 3V (CR123A Lithiumbatterie) |
| Akkulaufzeit | Ungefähr 3 Jahre |
| Sensorlebensdauer | Typischerweise 7 Jahre |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % RH (keine Kondensation) |

## Funktionen

- Hochpräziser elektrochemischer CO-Sensor
- Konzentrationsabhängiger Alarm (Ton- und Lichtaufforderung)
- 85 dB hoher Ton- und Lichtalarm
- Erinnerung an niedrigen Batteriestand
- Erinnerung an den Ablauf der Sensorlebensdauer
- Selbsttestfunktion
- Materielle multiökologische plattformübergreifende Interoperabilität

## Zertifizierung

- EN 50291 zertifiziert
- Materielle Zertifizierung
- Funktioniert mit der Home Assistant-Zertifizierung
