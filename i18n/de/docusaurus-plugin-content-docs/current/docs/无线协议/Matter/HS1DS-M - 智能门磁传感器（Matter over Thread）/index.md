# HS1DS-M – Intelligenter Türsensor (Matter over Thread)

## Übersicht

HS1DS-M ist die Matter-over-Thread-Version des Heiman-Smart-Türsensors. Es unterstützt den nativen Zugriff auf alle Matter-kompatiblen Plattformen wie Apple Home, Google Home, Amazon Alexa, Home Assistant usw., ohne dass ein proprietäres Gateway erforderlich ist.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | Materie über Faden |
| Stromversorgungsmethode | CR1632-Knopfbatterie |
| Akkulaufzeit | Ungefähr 2-3 Jahre |
| Installationsabstand | ≤10 mm |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % relative Luftfeuchtigkeit |

## Funktionen

- Echtzeiterkennung des Öffnungs-/Schließstatus von Türen und Fenstern
- Manipulationsalarm (Tamper)
- Erinnerung an niedrigen Batteriestand
- Plattformübergreifend kompatibel: Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Thread Mesh-Netzwerk, lokale Steuerung mit geringer Latenz
- Kein dediziertes Gateway erforderlich, Thread-Border-Router erforderlich
