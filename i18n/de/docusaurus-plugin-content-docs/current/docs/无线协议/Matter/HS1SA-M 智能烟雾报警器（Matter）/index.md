# HS1SA-M Intelligenter Rauchmelder (Matter)

HS1SA-M ist ein von Hyman auf den Markt gebrachter intelligenter Rauchmelder „Matter over Thread“. Es nutzt fotoelektrische Raucherkennungstechnologie und unterstützt die übergreifende Vernetzung von Smart-Home-Plattformen.

## Produktübersicht

- **Produkttyp**: Intelligenter Rauchmelder
- **Kommunikationsprotokoll**: Materie über Thread
- **Zertifizierungsstandard**: EN 14604
- **Typische Szenarien**: Brandwarnung in Häusern, Wohnungen, Hotels usw.

## Technische Parameter

| Parameter | Wert |
|------|-----|
| Sensortyp | Infrarot-Fotoelektrik |
| Kommunikationsprotokoll | Materie über Thread (IEEE 802.15.4) |
| Betriebsfrequenzband | 2,4 GHz |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | DC 3V (CR123A Lithiumbatterie) |
| Akkulaufzeit | Ungefähr 3 Jahre |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤95 % RH (keine Kondensation) |

## Funktionen

- Photoelektrische Raucherkennung zur Erkennung von langsam brennendem und offenem Flammenrauch
- 85 dB hoher Ton- und Lichtalarm
- Erinnerung an niedrigen Batteriestand
- Selbsttestfunktion (One-Click-Test)
- Manipulationsalarm
- Remote-Stummschaltung
- Materielle multiökologische plattformübergreifende Interoperabilität

## Zertifizierung

- EN 14604-Zertifizierung
- Materielle Zertifizierung

## Kompatible Plattformen

| Plattform | Unterstützung |
|------|----------|
| Apple Home | ✅ Thread-Grenzfräser erforderlich |
| Google-Startseite | ✅ Thread-Grenzfräser erforderlich |
| Amazon Alexa | ✅ Thread-Grenzfräser erforderlich |
| Heimassistent | ✅ Matter-zertifiziert |
| SmartThings | ✅ Erfordert Thread-Grenzfräser |

## Anforderungen an die Grenzführung

Es ist ein Thread-Border-Router erforderlich, z. B. Apple HomePod mini, Google Nest Hub 2. Generation, Amazon Echo 4. Generation usw.
