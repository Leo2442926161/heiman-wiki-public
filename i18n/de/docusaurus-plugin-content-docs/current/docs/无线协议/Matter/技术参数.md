# Technische Parameter

Im Folgenden finden Sie allgemeine Spezifikationen für Hyman Matter-Geräte.

## Wireless-Parameter

| Parameter | Wert |
|------|-----|
| Zugrundeliegendes Protokoll | Thread (IEEE 802.15.4) |
| Betriebsfrequenzband | 2,4 GHz (ISM-Band) |
| Standards der Anwendungsschicht | Materie 1.x |
| Netzwerktopologie | Mesh-Selbstheilungsnetzwerk |
| Verschlüsselungsmethode | AES-128-CCM |

## Hardwareanforderungen

| Projekt | Beschreibung |
|------|------|
| Grenzrouter | Erforderlich, z. B. Apple HomePod mini, Apple TV 4K, Google Nest Hub 2. Generation usw. |
| Unterstützung für mobile Endgeräte | iOS / Android (Netzwerkverteilung über verschiedene ökologische Apps) |
| Ökologische Interoperabilität | Apple HomeKit, Google Home, Amazon Alexa, Home Assistant, SmartThings |

## Strom- und Stromverbrauch

| Parameter | Wert |
|------|-----|
| Stromversorgungsmethode | Batteriebetrieben (CR123A / CR2450 usw.) |
| Akkulaufzeit | Ca. 1~3 Jahre (je nach Gerätetyp und Nutzungshäufigkeit) |
| Standby-Stromverbrauch | Extrem niedrig, unterstützt den Schlaf-Weck-Mechanismus |

## Umgebungsparameter

| Parameter | Wert |
|------|-----|
| Betriebstemperatur | -10°C ~ +50°C |
| Betriebsfeuchtigkeit | ≤ 95 % relative Luftfeuchtigkeit (nicht kondensierend) |
