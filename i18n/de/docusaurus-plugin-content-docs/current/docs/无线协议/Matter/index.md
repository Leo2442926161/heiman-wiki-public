#Materie

Matter (früher bekannt als Project CHIP) ist ein Smart-Home-Verbindungsstandard, der von der CSA Connection Standards Alliance ins Leben gerufen wurde und darauf abzielt, Markenbarrieren abzubauen und übergreifende ökologische Interoperabilität zu erreichen.

## Heyman und Matter

Hyman gehörte zu den ersten Anwendern des Matter-Protokolls und brachte **die weltweit ersten Matter-zertifizierten Kohlenmonoxidmelder** auf den Markt. Die Matter-Geräte von Hyman nutzen alle die **Matter over Thread**-Lösung.

## Technische Merkmale

| Funktionen | Beschreibung |
|------|------|
| Zugrundeliegendes Protokoll | Thread (Low-Power-Mesh-Netzwerk) |
| Netzwerktopologie | Mesh-Selbstheilungsnetzwerk: Je mehr Geräte, desto größer die Abdeckung |
| Stromverbrauch | Extrem niedrig, kann 1-3 Jahre lang mit Batteriestrom betrieben werden |
| Sicherheit | Unterstützt AES-128-CCM-Verschlüsselung, Geräteauthentifizierung erforderlich |
| Interoperabilität | Über Apple HomeKit, Google Home, Amazon Alexa, Home Assistant und mehr |

## Anforderungen an Grenzrouter

Für die Verwendung von Matter-over-Thread-Geräten ist ein Thread-Border-Router erforderlich:

- Apple HomePod mini / HomePod 2. Generation / Apple TV 4K
- Google Nest Hub 2. Generation / Nest Hub Max
-Amazon Echo 4. Generation
- Home Assistant Green + Connect ZBT-2
- Homey Pro
