# Gassammler – kabelloser Gaszähler-Datensammler

## Übersicht

Der Gaskollektor ist ein drahtloses wM-Bus-Datenerfassungsgerät, das von Hyman für den Smart-Gas-Bereich eingeführt wurde. Es arbeitet im 868-MHz-Frequenzband (Norm EN 13757) und dient der Fernauslesung der Messdaten von Gaszählern und deren Hochladen in das zentrale Zählerauslesesystem. Angetrieben durch Lithiumbatterien in Industriequalität mit einer Batterielebensdauer von 6 bis 10 Jahren eignet es sich für zentralisierte Zählerablesungsszenarien wie Wohnungen, Wohngebiete und Gewerbegebäude.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | wM-Bus (EN 13757) |
| Betriebsfrequenzband | 868 MHz (Europäisches SRD-Band) |
| Übertragungsrate | 4,8 kbps / 16,384 kbps / 100 kbps (T/S/R-Modus) |
| Kommunikationsentfernung | Etwa 100 bis 300 Meter im Innenbereich, etwa 500 bis 1000 Meter im Freien |
| Stromversorgungsmethode | Industrielle Lithiumbatterie (ER18505 / ER26500) |
| Akkulaufzeit | 6–10 Jahre (in der Regel einmal täglich gemeldet) |
| Betriebstemperatur | -20°C ~ +60°C |
| Arbeitsfeuchtigkeit | ≤ 95 % relative Luftfeuchtigkeit |
| Schutzstufe | IP65 ~ IP68 (abhängig von der Installationsumgebung) |
| Datenverschlüsselung | AES-128 (optional) |

## Funktionen

-Unterstützt drei Arbeitsmodi wM-Bus S/T/R
- Erfassen Sie automatisch die Nutzungsdaten von Gaszählern und melden Sie diese regelmäßig
- Geringer Stromverbrauch, tiefer Schlaf, lange Akkulaufzeit
- AES-128-Datenverschlüsselung zur Gewährleistung der Kommunikationssicherheit
-Unterstützt Remote-Firmware-Upgrade
- Mechanismus zur Trennung der Verbindung und zur Neuübertragung der Daten
- Unterstützt das Anwendungsschichtprotokoll EN 13757-3
- Kann mit dem HS1-CM-Kollektor / HS1-GW-Gateway für zentralisiertes Hochladen verwendet werden
