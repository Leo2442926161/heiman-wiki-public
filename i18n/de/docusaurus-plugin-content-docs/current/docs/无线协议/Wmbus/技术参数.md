# Technische Parameter

Im Folgenden sind die allgemeinen technischen Parameter der Heimann Wireless M-Bus (wM-Bus)-Geräte aufgeführt.

## Wireless-Parameter

| Parameter | Wert |
|------|-----|
| Protokollstandards | EN 13757-4 (Physische Schicht des drahtlosen M-Bus) |
| Arbeitsfrequenzband | 868 MHz (europäisches SRD-Frequenzband, 868,0–868,6 MHz) |
| Modulationsmethode | FSK |
| Übertragungsrate | 4,8 kbps / 16,384 kbps / 100 kbps (T-Modus / S-Modus / R-Modus) |
| Netzwerktopologie | Stern (Instrument → Kollektor/Gateway) |
| Kommunikationsentfernung | Etwa 100 bis 300 Meter im Innenbereich, etwa 500 bis 1000 Meter im Freien |
| Sendeleistung | ≤ 25 mW (+14 dBm) |

## Arbeitsmodus

| Modus | Bewerten | Beschreibung |
|------|------|------|
| S-Modus | 16,384 kbit/s | Standardmodus, geringer Stromverbrauch, geeignet für batteriebetriebene Instrumente |
| T-Modus | 100 kbit/s | Hochgeschwindigkeitsmodus, geeignet für große Verkehrsdatenszenarien |
| R-Modus | 4,8 kbit/s | Schmalbandmodus, Fernübertragung |

## Strom- und Stromverbrauch

| Parameter | Wert |
|------|-----|
| Stromversorgungsmethode | Lithiumbatterie-Stromversorgung (ER18505 / ER26500 und andere industrielle Lithiumbatterien) |
| Akkulaufzeit | 6–10 Jahre (abhängig von der Berichtshäufigkeit, typischer Wert ist 1 Mal/Tag) |
| Standby-Stromverbrauch | Mikroampere-Pegel, Tiefschlaf |
| Sensibilität erhalten | Ca. -105 dBm (S-Modus) |

## Umgebungsparameter

| Parameter | Wert |
|------|-----|
| Betriebstemperatur | -20°C ~ +60°C (Industriequalität) |
| Betriebsfeuchtigkeit | ≤ 95 % relative Luftfeuchtigkeit (nicht kondensierend) |
| IP-Schutzstufe | IP65 ~ IP68 (abhängig vom jeweiligen Modell, anwendbar für Outdoor-Instrumente) |

## Datenprotokoll

| Schicht | Standard |
|----|------|
| Physikalische Schicht | EN 13757-4 |
| Datenverbindungsschicht | EN 13757-4 (Wireless M-Bus Frame Format) |
| Anwendungsschicht | EN 13757-3 (M-Bus-Anwendungsschicht) |
| Verschlüsselungsmethode | AES-128 (optionale Verschlüsselung) |
