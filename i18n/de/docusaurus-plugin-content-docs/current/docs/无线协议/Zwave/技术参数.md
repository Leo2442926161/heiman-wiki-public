# Technische Parameter

Im Folgenden finden Sie allgemeine technische Spezifikationen für Hyman Z-Wave-Geräte.

## Wireless-Parameter

| Parameter | Wert |
|------|-----|
| Protokollstandards | Z-Wave Plus (ITU-T G.9959) |
| Arbeitsfrequenzband | 800~900 MHz (je nach Region: Europäische Regelung 868,42 MHz, US-amerikanische Regelung 908,42 MHz) |
| Übertragungsrate | Bis zu 100 kbit/s |
| Netzwerktopologie | Selbstheilendes Mesh-Netzwerk (Quellenrouting) |
| Knotenkapazität | Maximal 232 Knoten in einem einzigen Netzwerk |
| Kommunikationsentfernung | Ungefähr 30 Meter im Innenbereich (zwischen Knoten), ganzes Haus. Netzabdeckung |
| Verschlüsselungsmethode | AES-128 (S2 authentifiziert / S2 nicht authentifiziert) |

## Regionales Frequenzband

| Region | Mittenfrequenz |
|------|----------|
| Europa | 868,42 MHz |
| Nordamerika | 908,42 MHz |
| Andere Bereiche | Vorbehaltlich lokaler Vorschriften |

## Strom- und Stromverbrauch

| Parameter | Wert |
|------|-----|
| Stromversorgungsmethode | Batteriestromversorgung (CR123A/AA) oder Netzstromversorgung |
| Akkulaufzeit | Etwa 2 bis 3 Jahre für Sensoren |
| Sendeleistung | ≤ 1 mW (konform mit SRD-Vorschriften) |
| Sensibilität erhalten | Ungefähr -100 dBm |

## Umgebungsparameter

| Parameter | Wert |
|------|-----|
| Betriebstemperatur | -10°C ~ +50°C |
| Betriebsfeuchtigkeit | ≤ 90 % relative Luftfeuchtigkeit (nicht kondensierend) |

## Netzwerkeigenschaften

| Funktionen | Beschreibung |
|------|------|
| SmartStart | Support (Serie 700), scannen Sie den QR-Code, um dem Netzwerk beizutreten |
| S2-Sicherheit | Unterstützt drei Sicherheitsstufen: Unverschlüsselt, S2 nicht authentifiziert, S2 authentifiziert |
| Enthält CC | Security CC, Security 2 CC, Firmware Update CC, Basic CC, Battery CC, Notification CC und mehr |
