# HS1SA-Z Intelligenter Rauchmelder (Z-Wave)

HS1SA-Z ist ein intelligenter **Z-Wave**-Rauchmelder von Hyman. Es nutzt fotoelektrische Raucherkennungstechnologie, ist mit dem Z-Wave Plus-Standard kompatibel und eignet sich für den europäischen und amerikanischen Smart-Home-Markt.

## Produktübersicht

- **Produkttyp**: Intelligenter Rauchmelder
- **Kommunikationsprotokoll**: Z-Wave Plus
- **Zertifizierungsstandard**: EN 14604
- **Typisches Szenario**: Brandwarnung in Häusern, Wohnungen und anderen Orten

## Technische Parameter

| Parameter | Wert |
|------|-----|
| Sensortyp | Infrarot-Fotoelektrik |
| Kommunikationsprotokoll | Z-Wave Plus (ITU-T G.9959) |
| Betriebsfrequenzband | 868,42 MHz (EU) / 908,42 MHz (USA) |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | CR123A Lithiumbatterie |
| Akkulaufzeit | Ungefähr 2 bis 3 Jahre |
| Verschlüsselungsmethode | S2 authentifiziert / S2 nicht authentifiziert |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤90 % RH (keine Kondensation) |

## Funktionen

- Photoelektrische Raucherkennung zur Erkennung von langsam brennendem und offenem Flammenrauch
- 85 dB hoher Ton- und Lichtalarm
- Erinnerung an niedrigen Batteriestand
- Selbsttestfunktion
- Manipulationsalarm
- Sichere Z-Wave S2-Verschlüsselung
- Unterstützt OTA-Firmware-Upgrade

## Zertifizierung

- EN 14604-Zertifizierung
- Z-Wave Plus-zertifiziert
- S2-Sicherheitszertifizierung
