# Zertifizierungsstandards

HS1CA-Z hat die folgenden Zertifizierungen bestanden:

- **EN 50291** – Europäische Norm für Kohlenmonoxidmelder, siehe [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **Z-Wave S2-Zertifizierung** – Unterstützt das S2-Sicherheitsframework, siehe [Andere Zertifizierungen](../../../Certification Standards/Other Certifications.md)
- **CE** – Konformitätserklärung der Europäischen Union
