# HS1CA-Z Intelligenter Kohlenmonoxidmelder (Z-Wave)

HS1CA-Z ist ein intelligenter **Z-Wave**-Kohlenmonoxidmelder von Hyman. Es nutzt elektrochemische Sensortechnologie und ist mit dem Z-Wave Plus-Standard kompatibel.

## Produktübersicht

- **Produkttyp**: Intelligenter Kohlenmonoxidmelder
- **Kommunikationsprotokoll**: Z-Wave Plus
- **Zertifizierungsstandard**: EN 50291
- **Typisches Szenario**: CO-Leckagewarnung in Häusern, Wohnungen und anderen Orten

## Technische Parameter

| Parameter | Wert |
|------|-----|
| Sensortyp | Elektrochemischer CO-Sensor |
| Kommunikationsprotokoll | Z-Wave Plus (ITU-T G.9959) |
| Betriebsfrequenzband | 868,42 MHz (EU) / 908,42 MHz (USA) |
| Erfassungsbereich | 0~1000 ppm |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | Batteriebetrieben |
| Sensorlebensdauer | Typischerweise 7 Jahre |
| Verschlüsselungsmethode | S2 authentifiziert / S2 nicht authentifiziert |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤90 % RH (keine Kondensation) |

## Funktionen

- Hochpräziser elektrochemischer CO-Sensor
- Konzentrationsklassifizierungsalarm
- 85 dB hoher Ton- und Lichtalarm
- Erinnerung an niedrigen Batteriestand
- Erinnerung an den Ablauf der Sensorlebensdauer
- Selbsttestfunktion
- Sichere Z-Wave S2-Verschlüsselung
- Unterstützt OTA-Firmware-Upgrade

## Zertifizierung

- EN 50291 zertifiziert
- Z-Wave Plus-zertifiziert
- S2-Sicherheitszertifizierung
