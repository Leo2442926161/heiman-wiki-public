# HS1DS-Z Türsensor (Z-Wave)

HS1DS-Z ist ein von Hyman eingeführter **Z-Wave**-Türsensorsensor. Es erkennt den Öffnungs- und Schließstatus von Türen und Fenstern über magnetische Reed-Schalter und ist mit dem Z-Wave Plus-Standard kompatibel.

## Produktübersicht

- **Produkttyp**: Türsensor
- **Kommunikationsprotokoll**: Z-Wave Plus
- **Erkennungsmethode**: Reed-Schalter (Reed-Schalter)
- **Typische Szenarien**: Tür- und Fensterstatusüberwachung, Sicherheitseinsatz und intelligente Verknüpfung

## Technische Parameter

| Parameter | Wert |
|------|-----|
| Erkennungsmethode | Reed-Schalter (Reed-Schalter) |
| Kommunikationsprotokoll | Z-Wave Plus (ITU-T G.9959) |
| Betriebsfrequenzband | 868,42 MHz (EU) / 908,42 MHz (USA) |
| Stromversorgungsmethode | CR1632-Knopfbatterie |
| Akkulaufzeit | Ungefähr 2 bis 3 Jahre |
| Einbaufreiraum | ≤10 mm |
| Verschlüsselungsmethode | S2 authentifiziert / S2 nicht authentifiziert |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤90 % RH (keine Kondensation) |

## Funktionen

- Echtzeiterkennung des Öffnungs-/Schließstatus von Türen und Fenstern
- Manipulationsalarm (Tamper)
- Erinnerung an niedrigen Batteriestand
-Unterstützt Szenenverknüpfung (Tür öffnen, Licht einschalten, Scharfschalten beim Verlassen des Hauses)
- Einfache Montage (3M-Klebeband oder Schraubbefestigung)
- Sichere Z-Wave S2-Verschlüsselung
- Unterstützt OTA-Firmware-Upgrade
