#Z-Welle

Z-Wave ist ein drahtloses Mesh-Netzwerkprotokoll, das für Smart Homes entwickelt wurde. Es wird von der Z-Wave Alliance verwaltet und verfügt über eine breite Marktdurchdringung auf dem europäischen und amerikanischen Markt.

## Technische Spezifikationen

| Funktionen | Beschreibung |
|------|------|
| Betriebsfrequenzband | 868,42 MHz (EU) / 908,42 MHz (USA) |
| Übertragungsrate | Bis zu 100 kbit/s (Z-Wave 800-Serie) |
| Netzwerktopologie | Mesh-Netzwerk, bis zu 232 Knoten |
| Kompatibilität | Abwärtskompatibel mit allen Z-Wave-Geräten |
| Sicherheit | Z-Wave S2-Verschlüsselung |

## Hyman und Z-Wave

Hyman bietet eine Vielzahl von Z-Wave-kompatiblen Sensoren an, die hauptsächlich auf dem europäischen und amerikanischen Markt verkauft werden. Zu den Z-Wave-Versionen von Heyman-Geräten gehören:

- Kohlenmonoxidsensor HS1CA-Z
- Rauchsensor HS1SA-Z
- Türsensor HS1DS-Z

## Kompatible Plattformen

- SmartThings Hub
- Hubitat-Erhebung
- Homey Pro
- Home Assistant (über Z-Wave JS)
-OpenHAB

> **Hinweis**: Z-Wave erfreut sich auf dem chinesischen Markt einer geringen Beliebtheit und Haiman Z-Wave-Produkte richten sich hauptsächlich an europäische und amerikanische Exportmärkte.
