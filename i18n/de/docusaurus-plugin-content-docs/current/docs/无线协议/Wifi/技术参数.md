# Technische Parameter

Im Folgenden finden Sie allgemeine technische Spezifikationen für Heimann-WLAN-Geräte.

## Wireless-Parameter

| Parameter | Wert |
|------|-----|
| Protokollstandards | IEEE 802.11 b/g/n |
| Betriebsfrequenzband | 2,4 GHz (nicht kompatibel mit 5 GHz) |
| Übertragungsrate | Bis zu 150 Mbit/s (802.11n) |
| Netzwerktopologie | Sterntopologie (direkt mit dem Router verbunden) |
| Kommunikationsentfernung | Etwa 30–50 Meter im Innenbereich (abhängig von der Leistung des Routers) |
| Verschlüsselungsmethode | WPA2-PSK / WPA3 |

## Netzwerkverteilungsmethode

| Methode | Beschreibung |
|------|------|
| SmartConfig | Mobile App Netzwerkkonfiguration mit einem Klick (herkömmliche WLAN-Geräte) |
| Bluetooth-gestützte Bereitstellung | Konfigurieren von WLAN-Anmeldeinformationen über temporäre Bluetooth-Kanäle (einige neue Modelle) |
| AP-Modus | Schalten Sie den Hotspot auf dem Gerät ein und konfigurieren Sie ihn, nachdem das Telefon verbunden ist |

## Strom- und Stromverbrauch

| Parameter | Wert |
|------|-----|
| Stromversorgungsmethode | Netzstromversorgung (Steckdosentyp/Gateway-Typ) oder Batteriestromversorgung (Alarmtyp) |
| Stromverbrauchsniveau | Höher als Zigbee/Thread, da die TCP/IP-Verbindung aufrechterhalten werden muss |
| Standby-Stromverbrauch | Sockeltyp ca. 0,5W ~ 1W |

## Umgebungsparameter

| Parameter | Wert |
|------|-----|
| Betriebstemperatur | -10°C ~ +50°C |
| Betriebsfeuchtigkeit | ≤ 90 % relative Luftfeuchtigkeit (nicht kondensierend) |
