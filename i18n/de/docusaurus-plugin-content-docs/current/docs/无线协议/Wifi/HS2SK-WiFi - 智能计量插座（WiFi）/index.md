#HS2SK-WiFi - Intelligente Steckdose (WiFi)

## Übersicht

HS2SK-WiFi ist eine von Heiman eingeführte WLAN-Smart-Metering-Steckdose. Es unterstützt die direkte 2,4-GHz-802.11-b/g/n-Verbindung zu Heimroutern und ermöglicht die Fernsteuerung und Überwachung des Energieverbrauchs, ohne dass zusätzliche Gateways erforderlich sind. Geeignet für die intelligente Verwaltung von Elektrogeräten in Haushalten, Büros und anderen Szenarien.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | IEEE 802.11 b/g/n (2,4 GHz) |
| Maximale Belastung | 16A |
| Maximale Leistung | 3680W (230V) / 1800W (110V) |
| Messgenauigkeit | ±2% |
| Stromversorgungsmethode | Netzstromversorgung |
| Standby-Stromverbrauch | ≤ 0,5W |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤ 90 % relative Luftfeuchtigkeit |
| Schalenmaterial | PC V0 schwer entflammbar |
| Jack-Spezifikationen | Europäischer Standard / amerikanischer Standard (abhängig von der Marktversion) |

## Funktionen

-Unterstützt die Smart Life / Tuya Smart App-Fernbedienung
- Echtzeit-Stromüberwachung und Energieverbrauchsstatistiken
- Fernschaltersteuerung
- Zeitschalter und Countdown
- Szenenverknüpfung und Automatisierung
- Sprachsteuerung (kompatibel mit Alexa, Google Home)

## Zertifizierungsstandards

-CE/FCC
- RoHS
- ETL/UL (je nach Marktversion)
