# HS8SA-WiFi – Intelligenter Rauchmelder (WiFi US-Version)

## Übersicht

HS8SA-WiFi ist ein intelligenter WLAN-Rauchmelder, der von Hyman für den US-Markt eingeführt wurde. Es nutzt fotoelektrische Raucherkennungstechnologie, unterstützt WiFi-Netzwerke und unabhängige Arbeitsmodi und hat die UL 217-Zertifizierung bestanden. Geeignet für den Brandschutz in Häusern, Wohnungen und anderen Wohnräumen.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Kommunikationsprotokoll | IEEE 802.11 b/g/n (2,4 GHz) |
| Sensortyp | Infrarot-Fotoelektrik |
| Alarmschalldruck | ≥85 dB (3 Meter) |
| Stromversorgungsmethode | Batteriebetrieben |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤ 90 % relative Luftfeuchtigkeit |
| Installationsmethode | Deckenmontage |

## Funktionen

- Photoelektrische Raucherkennung zur Erkennung von Rauch durch langsame Verbrennungen und offene Flammen
- WiFi-Netzwerk-Fernalarm-Push
- Unabhängiger Betriebsmodus (lokaler Ton- und Lichtalarm kann weiterhin auftreten, wenn die Netzwerkverbindung getrennt ist)
- 85 dB hoher Ton- und Lichtalarm
- Erinnerung an niedrigen Batteriestand
- Selbsttestfunktion (One-Click-Test)
- Manipulationsalarm
-Unterstützt Remote-Stummschaltung

## Zertifizierungsstandards

- UL 217 (amerikanischer Standard für Rauchmelder)
-FCC
- NFPA 72-Konformität
