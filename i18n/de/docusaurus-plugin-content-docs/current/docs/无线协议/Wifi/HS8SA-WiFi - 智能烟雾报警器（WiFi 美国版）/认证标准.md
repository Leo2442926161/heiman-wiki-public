# Zertifizierungsstandards

HS8SA-WiFi hat die folgenden Zertifizierungen bestanden:

- **UL 217** – Amerikanischer Sicherheitsstandard für Rauchmelder, siehe [UL Underwriters Laboratories](../../../../certification standards/ULUnderwriters Laboratories.md)
- **FCC** – Zertifizierung der Federal Communications Commission
- **CE** – Konformitätserklärung der Europäischen Union
