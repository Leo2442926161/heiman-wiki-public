#HS6GW – Smart Home Gateway

## Übersicht

HS6GW ist ein von Heiman eingeführtes Smart-Home-Gateway, das WiFi + Ethernet-Uplink unterstützt, einen integrierten Zigbee-Koordinator hat und bis zu 128 Zigbee-Untergeräte verwalten kann. Die Realisierung von Fernzugriff und Cloud-Diensten durch Verbindung mit dem Internet ist der Kern des Haiman-Smart-Home-Systems.

## Technische Spezifikationen

| Parameter | Wert |
|------|-----|
| Uplink-Kommunikation | IEEE 802.11 b/g/n (2,4 GHz WLAN) + 10/100M Ethernet |
| Downlink-Kommunikation | Zigbee 3.0 (Koordinatormodus) |
| Maximale Anzahl an Untergeräten | 128 |
| Stromversorgungsmethode | DC 5V / 1A (Micro-USB) |
| Standby-Stromverbrauch | ≤ 1W |
| Betriebstemperatur | -10°C ~ +50°C |
| Arbeitsfeuchtigkeit | ≤ 90 % relative Luftfeuchtigkeit |
| Kontrollleuchte | WLAN-Status/Strom/Zigbee-Netzwerkstatus |
| Methode zurücksetzen | Zurücksetzen der Taste + langes Drücken, um die Werkseinstellungen wiederherzustellen |

## Funktionen

- Duale Uplinks: WLAN- oder Ethernet-Kabelverbindung, flexible Auswahl je nach Netzwerkumgebung
-Unterstützung des Fernzugriffs: Überprüfen Sie den Gerätestatus jederzeit und überall über die Cloud-App
- Lokale Szenenverknüpfung: Unterstützt den automatisierten Betrieb von Geräten im LAN (immer noch verfügbar, wenn die Verbindung getrennt ist)
- OTA-Firmware-Upgrade
- Unterstützt die Tuya / Smart Life-Plattform
- Verwalten Sie bis zu 128 Zigbee-Untergeräte
-Unterstützung der Selbstheilung und Routing-Optimierung des Zigbee-Netzwerks
