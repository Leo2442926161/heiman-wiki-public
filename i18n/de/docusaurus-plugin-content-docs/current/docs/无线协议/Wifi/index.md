#WLAN

Wi-Fi ist das beliebteste drahtlose Netzwerkprotokoll und basiert auf dem IEEE 802.11-Standard. Einige Smart-Geräte von Heiman unterstützen eine Wi-Fi-Verbindung und können direkt mit einem Router verbunden werden, ohne dass ein zusätzliches Gateway erforderlich ist.

## Anwendbare Szenarien

WLAN-Geräte eignen sich für folgende Situationen:

- Vollständige Netzwerkinfrastruktur, keine Notwendigkeit, zusätzliche Gateways zu kaufen
- Geräte, die nicht empfindlich auf den Stromverbrauch reagieren (z. B. intelligente Steckdosen)
- Szenarien, die Fernzugriff und Cloud-Dienste erfordern
- Häuser, die bereits über ausgereifte WLAN-Mesh-Netzwerke verfügen

## Hyman Wi-Fi-Produkte

Heimann bietet Produkte an, die Wi-Fi-Konnektivität unterstützen:

- **Smart Socket HS2SK-WiFi** – Stromverbrauchsmessung, Fernschalter
- **Smart Gateway** – verbindet sich über WLAN mit dem Router

## Notizen

- Der Akku des Wi-Fi-Sensors wird schneller entladen als bei Zigbee/Thread
- Eine große Anzahl von WLAN-Geräten kann die Netzwerkstabilität beeinträchtigen
- Die Hauptprodukte von Haiman sind Zigbee/Matter over Thread, und die Wi-Fi-Produktlinie ist relativ schlank
