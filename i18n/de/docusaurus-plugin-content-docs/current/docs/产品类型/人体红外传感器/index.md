# Infrarotsensor des menschlichen Körpers

Der Infrarotsensor für den menschlichen Körper (PIR) erkennt Bewegungen, indem er die vom menschlichen Körper abgegebene Infrarotwärmestrahlung erkennt und ist die am häufigsten verwendete Lösung zur Anwesenheitserkennung in Smart Homes.

## Produktserie

| Serie | Kommunikationsmethoden | Funktionen |
|------|----------|------|
| HS1MS | Zigbee | Basismodell, 110° Erfassungswinkel |
| HS3MS | Zigbee | Verbessertes Modell, 360°-Deckeninstallation |
| M1-Serie | Materie / Zigbee | Unterstützt Materie über Faden, optional 110° oder 360° |
| HS8MLS | Zigbee | Langstreckenmodell |
| HS8MIS | Zigbee | Menschlicher Körper + Beleuchtungs-Verbundsensor |

## Kernfunktionen

- Mobile Personenerkennung
- Beleuchtungserkennung (M1/HS8MIS-Serie)
- Erfassungsentfernung ≤7m (25°C)
- Installationsmethode: Wandmontage oder Deckenmontage
- Batterielebensdauer: 3 Jahre (Zigbee) / 1 Jahr (Matter)

## Technische Parameter (M1-Serie)

- Erfassungswinkel: 110° / 360°
- Erkennungsentfernung: ≤7m
- Beleuchtungsstärkebereich: 0~1200 Lux
- Stromversorgung: DC 3V (Lithiumbatterie)
- Batterielebensdauer: 3 Jahre (Zigbee) / 1 Jahr (Matter)

## Szenariobeispiel

- Das Licht geht an, wenn jemand kommt und geht aus, wenn jemand geht
- Sicherheitsverknüpfung: Alarm auslösen, wenn ungewöhnliche Bewegungen erkannt werden
- Scharfschaltung beim Verlassen des Hauses: Klimaanlage/Heizung automatisch abschalten, um Energie zu sparen, wenn niemand in der Nähe ist
