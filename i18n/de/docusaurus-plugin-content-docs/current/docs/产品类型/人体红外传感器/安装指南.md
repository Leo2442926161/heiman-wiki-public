# Installationsanleitung

## Anforderungen an den Installationsort

- Wandmontagehöhe: **1,8 ~ 2,4 m**
- Deckenmontage: Montage an der Decke in der Raummitte
- **Vermeiden**: Direkte Sonneneinstrahlung, Lüftungsschlitze der Heizung/Klimaanlage, in der Nähe von Wärmequellen
- Vermeiden Sie Hindernisse (Möbel, Glastrennwände) im Erfassungsbereich

## Installationsschritte

1. Wählen Sie den Installationsort (Wandmontage oder Deckenmontage).
2. Befestigen Sie den Montagesockel mit Schrauben
3. Legen Sie den Akku ein, die Kontrollleuchte blinkt
4. Installieren Sie das Sensorgehäuse auf der Basis
5. Passen Sie den Erfassungswinkel an (innerhalb des horizontalen Bereichs von 110°).
6. Gehtest: Nach dem Durchschreiten des Erfassungsbereichs sollte die Kontrollleuchte aufleuchten
7. Kopplung über Gateway

## Notizen

- Infrarotsensoren reagieren empfindlich auf Temperaturänderungen und der Erfassungsabstand kann sich in Umgebungen mit hohen Temperaturen im Sommer verkürzen.
- Vermeiden Sie den Blick auf Fenster (Änderungen der Außentemperatur führen zu Fehlalarmen)
- Batterielebensdauer ca. 3 Jahre (Zigbee) / 1 Jahr (Matter)
