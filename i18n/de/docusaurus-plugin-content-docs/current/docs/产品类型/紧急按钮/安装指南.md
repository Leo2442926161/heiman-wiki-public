# Installationsanleitung

## Anforderungen an den Installationsort

- **Wandmontage**: Badezimmer, Nachttisch im Schlafzimmer, neben dem Sofa im Wohnzimmer – überall in Reichweite
- **Tragen Sie es überall**: Stecken Sie es in Ihre Tasche oder hängen Sie es an Ihren Schlüsselbund
- **Für ältere Menschen**: Wird im Badezimmer und am Bett des Schlafzimmers installiert und beträgt ca. 1,2 m

## Installationsschritte

1. Bestimmen Sie den Installationsort oder ob Sie es bei sich tragen möchten
2. Wandmontage: Befestigen Sie die Montagebasis mit Schrauben oder 3M-Kleber
3. Installieren Sie die Batterie
4. Platzieren Sie den Notrufknopf auf der Basis oder tragen Sie ihn bei sich
5. Kopplung über Gateway
6. Test: Drücken Sie lange auf die Taste, um den Alarm auszulösen und zu bestätigen, dass die APP die Benachrichtigung empfängt

## Notizen

- Übernehmen Sie das Design **Langes Drücken gegen unbeabsichtigte Berührung**, das zum Auslösen ein kontinuierliches Drücken für 1–2 Sekunden erfordert
- Es wird empfohlen, eine Verbindung mit dem Ton- und Lichtalarm herzustellen
- Testen Sie regelmäßig die Funktionalität
- Für ältere Menschen kann es mit einem Sturzdetektor für doppelten Schutz verwendet werden.
