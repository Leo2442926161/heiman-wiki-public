# Installationsanleitung

## Anforderungen an den Installationsort

- Installiert in einer Höhe von **1,2 ~ 1,5 m** (entspricht dem durchschnittlichen Niveau der Umgebung)
- **Fernhalten**: direkte Sonneneinstrahlung, Türen und Fenster, Klimaanlagenauslässe, Heizkörper
- **Fernhalten**: Luftbefeuchter, Küchenherde und andere lokale Quellen von Temperatur- und Feuchtigkeitsschwankungen
- Wählen Sie Räume, die häufig genutzt werden (Schlafzimmer, Wohnzimmer, Babyzimmer)

## Installationsschritte

1. Wählen Sie den Installationsort (entweder Wand oder Schreibtisch)
2. Wandmontage: Befestigen Sie die Basis mit Schrauben oder 3M-Kleber
3. Legen Sie die CR2-Batterie ein (beachten Sie die Polarität).
4. Der Sensor schaltet sich automatisch ein und auf dem Bildschirm (falls vorhanden) werden Temperatur und Luftfeuchtigkeit angezeigt.
5. Kopplung über Gateway
6. Überprüfen Sie die Daten in der APP, um sicherzustellen, dass sie normal sind.

## Notizen

- Stellen Sie bei der Installation an der Wand sicher, dass die Sensoröffnung nicht blockiert ist
-Die Batterielebensdauer beträgt ca. 2-3 Jahre
- Temperatur- und Luftfeuchtigkeitsdaten in verschiedenen Räumen können über mehrere Sensoren erfasst werden
- Kann für angeschlossene Klimaanlagen, Luftentfeuchter, Luftbefeuchter und andere Geräte verwendet werden
