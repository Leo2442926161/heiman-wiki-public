# Installationsanleitung

## Anforderungen an den Installationsort

- **Auf Atemhöhe installiert** - ca. **1,5 m** über dem Boden
- In der Nähe des Schlafbereichs, um sicherzustellen, dass der Alarm während des Schlafens hörbar ist
- **Nicht in Ecken installieren**, Luftzirkulation aufrechterhalten
- Von Lüftungsschlitzen, Fenstern, Ventilatoren und anderen Orten fernhalten, die den Luftstrom beeinträchtigen

## Installationsschritte

1. Bestimmen Sie den Installationsort (empfohlener Schlafzimmerflur oder Wohnzimmer)
2. Befestigen Sie den Montagesockel an der Wand (Schrauben oder 3M-Klebeband).
3. Legen Sie die Batterie ein und der Sensor schaltet sich automatisch ein.
4. Installieren Sie den Hauptkörper auf der Basis
5. Selbsttest: Drücken Sie die Testtaste, die Alarmleuchte blinkt und der Alarm ertönt.
6. Kopplung über Gateway

## Notizen

- Die typische Lebensdauer des Sensors beträgt **7 Jahre**, danach muss das gesamte Gerät ausgetauscht werden
- Testen Sie die Funktionalität einmal im Monat
- Blockieren Sie nicht die Sensoröffnung
- Öffnen Sie im Alarmfall zunächst die Fenster zum Lüften und prüfen Sie dann, ob CO-Quellen austreten.
