# Kohlenmonoxidsensor

Kohlenmonoxid (CO) ist ein farbloses, geruchloses und nicht reizendes giftiges Gas, das als „unsichtbarer Killer“ bekannt ist. Kohlenmonoxidsensoren von Hyman nutzen **elektrochemische Sensortechnologie**, um die CO-Konzentration in der Luft genau zu erfassen.

## Zertifizierte Erfolge

Hyman bringt **den weltweit ersten Matter-zertifizierten Kohlenmonoxidmelder** auf den Markt und ist von Works with Home Assistant zertifiziert.

## Kernfunktionen

- Hochpräziser elektrochemischer CO-Sensor
- Erkennungsbereich: 0~1000 ppm
- Konzentrationsabhängiger Alarm (Ton- und Lichtaufforderung)
- 85 dB Alarmton
- Erinnerung an niedrigen Batteriestand
- Erinnerung an den Ablauf der Sensorlebensdauer (normalerweise 7 Jahre)
- Selbsttestfunktion

## Sicherheitstipps

CO-Konzentration und Reaktion des menschlichen Körpers:

| CO-Konzentration (ppm) | Reaktion des menschlichen Körpers |
|---------------|----------|
| 50 | Gesunde Erwachsene vertragen 8 Stunden |
| 200 | Leichte Kopfschmerzen nach 2-3 Stunden |
| 400 | Kopfschmerzen und Übelkeit nach 1-2 Stunden |
| 800 | Schwindel und Krämpfe innerhalb von 45 Minuten |
| 1600+ | Tödlich innerhalb von 2 Stunden |
