# Installationsanleitung

## Anforderungen an den Installationsort

- **Erdgas (Methan)**: leichter als Luft, installiert innerhalb von **30 cm** von der Decke
- **Flüssiggas (Propan)**: schwerer als Luft, installiert innerhalb von **30 cm über dem Boden**
- In der Nähe potenzieller Leckstellen wie Gasherden, Warmwasserbereitern und Gasrohranschlüssen

## Installationsschritte

1. Bestätigen Sie die Art des zu erkennenden Gases (Erdgas oder Flüssiggas).
2. Wählen Sie die entsprechende Einbauhöhe
3. Befestigen Sie den Montagesockel mit Schrauben an der Wand
4. Schalten Sie die Stromversorgung ein (die meisten Modelle werden über das Stromnetz betrieben).
5. Schalten Sie den Sensor ein und lassen Sie ihn etwa 3–5 Minuten lang aufwärmen.
6. Selbsttest: Drücken Sie die Testtaste, der Ton- und Lichtalarm ist normal.
7. Kopplung über Gateway (Zigbee-Version)
8. Wenn Sie das Magnetventil anschließen müssen, schließen Sie die Steuerleitung des Absperrventils an

## Notizen

- Sensorempfindlichkeit regelmäßig mit Prüfgas prüfen
- Die Lebensdauer des Sensors beträgt etwa 3–5 Jahre und muss nach Ablauf ersetzt werden
- Schalten Sie niemals Elektrogeräte ein oder aus und verwenden Sie niemals eine offene Flamme, wenn Sie Gas riechen
- Einbauhöhe abhängig von der Gasart (oben Erdgas, unten Flüssiggas)
