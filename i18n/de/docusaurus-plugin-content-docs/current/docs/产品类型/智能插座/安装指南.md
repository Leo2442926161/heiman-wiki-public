# Installationsanleitung

## Anforderungen an den Installationsort

- Direkt an die Wandsteckdose anschließen, **Nicht in Reihe schalten** (Nicht an eine Steckdose oder ein Verlängerungskabel anschließen)
- Stellen Sie sicher, dass der Nennstrom der Steckdose ≥ 16A beträgt
- Nicht in feuchten Umgebungen oder Umgebungen mit hohen Temperaturen installieren

## Installationsschritte

1. Stecken Sie die Smart-Steckdose direkt in eine 220-V- (oder 110-V-)Wandsteckdose
2. Die Kontrollleuchte leuchtet auf, nachdem Strom zugeführt wurde.
3. Halten Sie die Umschalttaste gedrückt, um in den Kopplungsmodus zu gelangen
4. Gerät in der Heiman-App hinzufügen
5. Nach erfolgreicher Kopplung können Sie den Switch fernsteuern

## Sicherheitsvorkehrungen

- **Maximale Belastung**: 16A/3680W (europäischer Standard), 15A/1800W (US-Standard)
- **Nicht überlasten**: Bei der Verwendung leistungsstarker Elektrogeräte (Klimaanlagen, Heizgeräte) auf die Leistungsgrenze achten
- **NICHT IN REIHE ANSCHLIESSEN**: Smart-Steckdosen nicht in Steckdosenleisten einstecken
- Decken Sie das Steckdosengehäuse nicht ab oder verdecken Sie es nicht
- Überprüfen Sie den Stecker regelmäßig auf Überhitzung

## Typische Anwendungen

- Zeitsteuerung von Geräten mit geringem Stromverbrauch wie Warmwasserbereitern, Luftbefeuchtern und Aquarienbeleuchtungen
- Kooperieren Sie mit der Sensorverknüpfung: Das Licht geht an, wenn jemand kommt, und aus, wenn jemand geht.
- Leistungsstatistik: Überwachen Sie den Energieverbrauch von Haushaltsgeräten
