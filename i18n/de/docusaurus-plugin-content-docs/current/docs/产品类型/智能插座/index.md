# Smart-Steckdose

Der Haiman Smart Metering Plug unterstützt die Fernsteuerung von Schaltern und die Messung des Stromverbrauchs in Echtzeit und hilft Benutzern so bei der intelligenten Verwaltung von Haushaltsgeräten.

## Produktserie

| Serie | Kommunikationsmethoden | Funktionen |
|------|----------|------|
| HS2SK-Serie | Zigbee | Messart, unterstützt Leistungsüberwachung |

## Kernfunktionen

- Fernschaltersteuerung
- Strom-/Gasmessung in Echtzeit
- Überlastschutz
- Zeitschaltuhr
- Leistungsstatistik

## Technische Parameter

- Nennspannung: AC 220 V (chinesischer Standard) / AC 110 V (amerikanischer Standard)
- Maximale Belastung: 16A/3680W
-Messgenauigkeit: ±2 %

## Szenariobeispiel

- Schalten Sie vergessene Geräte aus der Ferne aus
- Statistik des Energieverbrauchs der Klimaanlage/des Warmwasserbereiters
- Verknüpfter Leaving-Home-Modus: Schalten Sie alle nicht unbedingt benötigten Geräte mit einem Klick aus
- Zeitsteuerung von Aquarienbeleuchtung, Luftbefeuchtern usw.
