# intelligente Türklingel

Die Haiman Smart Doorbell integriert Türklingel- und Sicherheitsüberwachungsfunktionen und unterstützt die Remote-Video-Gegensprechanlage.

## Kernfunktionen

- Türklingelknopfauslöser
- Remote-Live-Video (Wi-Fi-Version)
- Zwei-Wege-Sprachanrufe
- Bewegungserkennungsvideo
- Nachtsichtfunktion

## Anwendbare Szenarien

- Besucheranrufe: Sie können sie zu Hause oder unterwegs beantworten
- Express-Liefer-/Abholbenachrichtigung: Fernbestätigung des Öffnens der Tür
- Sicherheitsüberwachung: Verdächtige Aktivitäten vor der Tür aufzeichnen

> **Hinweis**: Die Türklingel-Produktlinie von Heiman basiert hauptsächlich auf Zigbee/Wi-Fi-Lösungen. Für spezifische Modellparameter wenden Sie sich bitte an den Beamten.
