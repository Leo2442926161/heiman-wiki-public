# Installationsanleitung

## Anforderungen an den Installationsort

- Installieren Sie die Türklingel in einer Höhe von ca. **1,4 ~ 1,6 m** (auf Augenhöhe)
- Schauen Sie zur Tür, um sicherzustellen, dass Sie das Gesicht des Besuchers erfassen können
- Vermeiden Sie direkte Sonneneinstrahlung auf die Kamera

## Installationsschritte

1. Wählen Sie den Montageort (neben der Tür oder am Türrahmen)
2. Befestigen Sie den Montagesockel mit Schrauben
3. Schließen Sie die Stromversorgung an (schließen Sie die kabelgebundene Version an, installieren Sie die Batterie in der Batterieversion).
4. Montieren Sie das Türklingelgehäuse auf der Basis
5. Koppeln und verbinden Sie sich per APP mit dem WLAN (Wi-Fi-Version)
6. Passen Sie den Kamerawinkel an
7. Türklingeltasten, Video- und Sprachfunktionen testen

## Notizen

- Die WLAN-Version erfordert eine gute WLAN-Signalabdeckung an der Tür
- Achten Sie auf den Ladezyklus der Batterieversion. Es wird empfohlen, sie mit dem Türsensor zu verwenden (um nach dem Öffnen der Tür zu erkennen und aufzuwachen).
- Reinigen Sie das Kameraobjektiv regelmäßig
- Stellen Sie sicher, dass sich die Türklingel innerhalb der gültigen Zigbee-Signalreichweite befindet
