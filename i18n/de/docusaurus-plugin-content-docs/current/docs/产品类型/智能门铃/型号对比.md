# Modellvergleich

## Vergleich verschiedener Modelle

| Modell | Protokoll | Video | Stromversorgung | Funktionen | Gameplay-Vorschläge |
|------|------|------|------|------|----------|
| Intelligente Türklingel | Zigbee + WLAN | 1080P optional | Batterie/Netz | Türklingel + Sicherheit zwei in einem | Benachrichtigung über das Klingeln des Besuchers auf dem Mobiltelefon, visuelle Fernsprechanlage, Aufzeichnung der Bewegungserkennung |

## Empfohlenes Szenen-Gameplay

| Szenarien | Empfohlene Modelle | Verbindungslösungen |
|------|----------|----------|
| Express-Lieferung/Takeaway-Management | Intelligente Türklingel | Klingeln → Handy-Benachrichtigung → Fern-Intercom-Bestätigung → Fern-Entriegelung (verknüpftes Smart Lock) |
| Sicherheitsüberwachung | Intelligente Türklingel | Türstopper-Erkennung → Video + Push-Alarm, verbundener Türsensor/Beleuchtung |
| Besucherverknüpfung | Intelligente Türklingel | Klingeln → Indoor-Bildschirm/Lautsprecher-Display → Lichter einschalten → Verknüpfung entriegeln |
