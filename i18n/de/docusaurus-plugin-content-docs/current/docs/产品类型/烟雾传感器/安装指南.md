# Installationsanleitung

## Anforderungen an den Installationsort

- **Deckenmontiert**, in der Mitte des Raumes platziert
- Mindestens **30 cm** Abstand zu Wänden oder Leuchten
- **Tote Luftbereiche vermeiden** (z. B. Ecken, hinter Balken)
- **Vermeiden Sie die Installation in**: Badezimmer (Dampf), Küche (Ruß), in der Nähe von Lüftungsschlitzen

## Installationsschritte

1. Wählen Sie den Installationsort und stellen Sie sicher, dass die Decke flach ist
2. Drehen Sie den Montagesockel zur Decke (verwenden Sie Schrauben oder 3M-Kleber).
3. Legen Sie die Batterie ein (achten Sie auf die Polarität), und der Sensor schaltet sich automatisch ein.
4. Drehen Sie das Sensorgehäuse und lassen Sie es in der Basis einrasten
5. Drücken und halten Sie die Testtaste. Das Alarmlicht blinkt und ein Alarmton ertönt, was darauf hinweist, dass es normal ist.
6. Kopplung über Gateway: Wechseln Sie in den Kopplungsmodus und fügen Sie das Gerät zur APP hinzu

## Notizen

- Testen Sie die Funktionalität einmal im Monat
- Die Batterielebensdauer beträgt ca. 3 Jahre. Bitte ersetzen Sie sie rechtzeitig, wenn die Batterie schwach ist
- Es wird empfohlen, den gesamten Sensor alle 10 Jahre auszutauschen
- Die Montagehöhe hat keinen Einfluss auf die Raucherkennung, Rauch steigt bis zur Decke
