# Rauchsensor

Der Hyman-Rauchsensor nutzt die **photoelektrische Rauchsensortechnologie**, die sichtbare Rauchpartikel, die in den frühen Stadien eines Brandes entstehen, empfindlich erkennen und eine sehr frühe Brandwarnung ermöglichen kann.

## Kernfunktionen

- Photoelektrische Raucherkennung zur Erkennung von Rauch durch langsame Verbrennungen und offene Flammen
- 85 dB hoher Ton- und Lichtalarm
- Erinnerung an niedrigen Batteriestand
- Selbsttestfunktion (One-Click-Test)
- Manipulationsalarm
-Unterstützt Remote-Stummschaltung

## Technische Parameter (am Beispiel HS1SA-M)

- Sensortyp: Infrarot-Fotoelektrik
- Alarmschalldruck: ≥85 dB (3 Meter)
- Stromversorgung: CR123A-Batterie
- Batterielebensdauer: ca. 3 Jahre
- Betriebstemperatur: -10 °C ~ +50 °C
- Arbeitsfeuchtigkeit: ≤95 % RH
