# Modellvergleich

## Vergleich verschiedener Modelle

| Modell | Protokoll | Sensor | Stromversorgung | Funktionen | Spielvorschläge |
|------|------|--------|------|------|----------|
| [HS2FD](../../Wireless Protocol/Zigbee/HS2FD - Sturzdetektor (Zigbee)) | Zigbee | Beschleunigungsmesser + Gyroskop | Batterie | KI-Algorithmus zur Sturzerkennung | Tragbares Design, automatische Sturzerkennung + Ein-Klick-Hilfe, geeignet für allein lebende ältere Menschen und Pflegeheime |

## Empfohlenes Szenen-Gameplay

| Szenarien | Empfohlene Modelle | Verbindungslösungen |
|------|----------|----------|
| Überwachung allein lebender älterer Menschen | HS2FD | Sturzerkennung → Automatischer Push auf Kinderhandys → Automatischer Anruf 120 nach Zeitüberschreitung und ohne Antwort |
| Altenpflegeeinrichtung | HS2FD | Zentralisierte Verwaltungsplattform + Ton- und Lichtalarm auf dem Boden + synchroner Alarm für das Pflege-Handgerät |
| Postoperative Rehabilitation | HS2FD | Sturzgefahr wird beim Aufstehen des Patienten erkannt → Frühwarnung auf der Pflegestation |
| Badezimmer/Nacht | HS2FD | IP-geschütztes Design, in der Badewanne tragbar + automatischer Alarm, wenn Sie nachts aufstehen und fallen |

## Auswahlvorschläge

- **Häusliche Pflege**: [HS2FD](../../Wireless Protocol/Zigbee/HS2FD – Sturzdetektor (Zigbee)) (tragbar, KI-Erkennung)
- **Gepaart mit Notruftaste**: Automatische Sturzerkennung HS2FD + Notruftaste für manuelle Hilfe, doppelter Schutz
