# Installationsanleitung

## Trageweise

- **Gürtelclip**: Clip am Gürtel oder Hosenbund
- **Lanyard**: um den Hals hängen und an der Brust hängen

## Gebrauchsanweisung

1. Installieren Sie die Batterie
2. Trageart wählen (Gürtelclip oder Trageband)
3. Tragen Sie es am Körper
4. Kopplung über Gateway
5. Simulationstest:
   - Stehen Sie auf und gehen Sie dann schnell in die Hocke oder legen Sie sich auf den Boden
   - Der Sensor sollte einen Sturz erkennen und innerhalb von Sekunden den Alarm auslösen
   - Drücken Sie die Notruftaste, um den manuellen Alarm zu testen

## Notizen

- Empfohlen, den ganzen Tag getragen zu werden (kann zum Laden/Austauschen des Akkus beim Duschen abgenommen werden)
- Ein KI-Algorithmus wird verwendet, um Stürze von alltäglichen Aktivitäten (Sitzen, Bücken) zu unterscheiden, mit einer geringen Fehlalarmrate
- Automatischer Alarm nach Erkennung eines Sturzes, oder Sie können die Taste manuell drücken, um Hilfe zu rufen
- Kann mit der Notruftaste verknüpft werden, um einen doppelten Schutz zu erreichen
