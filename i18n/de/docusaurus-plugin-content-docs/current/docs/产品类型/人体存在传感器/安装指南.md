# Installationsanleitung

## Anforderungen an den Installationsort

- **Deckenmontage**, Sensor senkrecht nach unten
- Installationshöhe **2,5 ~ 3,5 m**
- Der Erfassungsradius beträgt ca. **3 ~ 5 m**
- **Vermeiden**: Vor dem Sensor befinden sich Metallhindernisse (Metall blockiert Millimeterwellen)
- Von großen Metallrohren, Inneneinheiten von Klimaanlagen usw. fernhalten.

## Installationsschritte

1. Wählen Sie den Installationsort an der Decke (am besten die Mitte des Raumes).
2. Bohren Sie Löcher in die Decke und befestigen Sie die Basis mit Dehnschrauben
3. Schließen Sie die Verkabelung an (falls eine Netzstromversorgung erforderlich ist) oder installieren Sie die Batterie
4. Drehen Sie das Sensorgehäuse und lassen Sie es in der Basis einrasten
5. Nach dem Einschalten leuchtet die Kontrollleuchte auf und der Sensor beginnt mit dem Selbsttest.
6. Geben Sie die APP-Kopplung ein und fügen Sie das Gerät hinzu
7. Test: Betreten Sie den Erfassungsbereich und sitzen Sie ruhig. Der Sensor soll im „Besetzt“-Zustand bleiben.

## Notizen

- Millimeterwellen können dünne Platten (Gipsplatten, Holz) durchdringen, werden aber von Metall blockiert
-Dicke Wände und Metalldecken verringern die Erfassungsreichweite
- Empfohlen für die Verwendung mit einem PIR-Sensor, um eine vollständige Szenenabdeckung zu erreichen
