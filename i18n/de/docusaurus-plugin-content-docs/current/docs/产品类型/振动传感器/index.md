# Vibrationssensor

Der Vibrationssensor löst einen Alarm aus, indem er die Vibration oder Neigung eines Objekts erkennt. Es eignet sich für verschiedene Szenarien wie Türen, Fenster, Tresore und Rohre.

## Kernfunktionen

- Dreidimensionale Vibrationserkennung
- Neigungswinkelerkennung
- Erinnerung an niedrigen Batteriestand

## Anwendbare Szenarien

- **Tür- und Fenster-Aufbruchschutz**: Erkennt Vibrationen, die durch gewaltsamen Einbruch verursacht werden
- **Safe/Schubladenmonitor**: Erkennt abnormale Bewegungen
- **Pipeline-Überwachung**: Erkennen Sie Vibrationen in Wasserleitungen (was auf Anomalien der Wasserpumpe oder einen Wasserschlag hinweist)
- **Wand-/Zaunüberwachung**: Erkennen Sie Klettern oder Stöße

## Technische Parameter

- Sensortyp: 3-Achsen-Beschleunigungsmesser
- Erfassungsmaße: Vibration + Neigung
- Stromversorgungsmethode: CR2450-Knopfbatterie
- Batterielebensdauer: ca. 2 Jahre
