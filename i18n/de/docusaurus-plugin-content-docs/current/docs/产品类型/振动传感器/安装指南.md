# Installationsanleitung

## Anforderungen an den Installationsort

- Installiert auf der Oberfläche von Objekten, die Vibrationen überwachen müssen
- Gängige Installationsorte: **In Türen und Fenstern**, **Tresore**, **Rohre**, **Wände**, **Zäune**
- Stellen Sie sicher, dass die Installationsoberfläche flach und sauber ist

## Installationsschritte

1. Wählen Sie das zu überwachende Objekt aus (z. B. die Innenseite der Tresortür).
2. Reinigen Sie die Montagefläche
3. Befestigen Sie den Sensor mit 3M-Klebeband oder Schrauben
4. Installieren Sie die CR2450-Batterie
5. Kopplung über Gateway
6. Simulationstest: Tippen Sie auf die Installationsfläche, der Sensor sollte einen Alarm auslösen

## Notizen

- Für Tür- und Fenster-Aufbruchschutzszenarien ist es effektiver, es an der Innenseite der Tür/des Fensters in der Nähe des Schlosses zu installieren.
- Zur Rohrleitungsüberwachung direkt auf die Rohrleitungsoberfläche kleben
- Batterielebensdauer ca. 2 Jahre
- Einstellbare Empfindlichkeit für verschiedene Szenarien
