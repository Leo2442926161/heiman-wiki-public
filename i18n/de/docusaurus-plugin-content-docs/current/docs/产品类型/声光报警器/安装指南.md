# Installationsanleitung

## Anforderungen an den Installationsort

- **In der Mitte des Hauses installieren**, um sicherzustellen, dass es in allen Räumen gehört werden kann
- Flur, Wohnzimmerdecke oder hohe Wand
- **2 m oder mehr** über dem Boden (Berührung durch Personen vermeiden)
- Bei Verwendung in Verbindung mit Rauch-/CO-Sensoren in Schlafzimmerfluren installieren

## Installationsschritte

1. Wählen Sie den Installationsort (Flur oder Wohnzimmer wird empfohlen)
2. Befestigen Sie den Montagesockel mit Schrauben an der Wand oder Decke
3. Schließen Sie das Netzteil an (Netzversion) oder legen Sie den Akku ein (Batterieversion).
4. Installieren Sie das Alarmgehäuse auf der Basis
5. Kopplung über Gateway
6. Verbindungstest: Lösen Sie einen beliebigen Sensor aus. Der Ton- und Lichtalarm sollte reagieren

## Notizen

- In einer für Kinder unzugänglichen Höhe installieren
- Testen Sie regelmäßig, ob die Alarmlautstärke und -beleuchtung normal ist
- Achten Sie bei der Batterieversion darauf, diese auszutauschen, wenn die Batterie schwach ist.
- Kann Verbindungsregeln mit anderen Sensoren festlegen
