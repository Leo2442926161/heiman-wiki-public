# Installationsanleitung

## Anforderungen an den Installationsort

- **Zentrale Lage im ganzen Haus** gewährleistet die Zigbee-Signalabdeckung für alle Geräte
- Vermeiden Sie es, das Gerät in einem Ganzmetallschrank aufzustellen (Metall blockiert Funksignale).
- **Ethernet zuerst**: Es wird empfohlen, eine kabelgebundene Netzwerkverbindung zu verwenden, da diese stabiler ist
- Nicht mehr als 20-30 m von anderen Zigbee-Geräten entfernt (in Innenräumen)

## Installationsschritte

1. Wählen Sie den Installationsort (TV-Schrank im Wohnzimmer, neben der Schwachstrombox usw.)
2. Schließen Sie das Netzteil an
3. Schließen Sie ein Ethernet-Kabel an (empfohlen) oder stellen Sie eine WLAN-Verbindung her
4. Warten Sie, bis die Anzeigeleuchte grün leuchtet bzw. dauerhaft leuchtet
5. Öffnen Sie die Haiman-App und scannen Sie zum Hinzufügen den QR-Code unten am Gateway
6. Nachdem das Hinzufügen abgeschlossen ist, können Sie das Untergerät koppeln

## Notizen

- Das Gateway ist das Herzstück des Smart Homes. Es wird empfohlen, eine USV-Notstromversorgung zu verwenden.
- Die Ethernet-Verbindung ist stabiler als WLAN und sollte zuerst verwendet werden
- Wenn das Gerätesignal schwach ist, fügen Sie ein Zigbee-Relaisgerät hinzu (z. B. eine Smart-Steckdose).
- Die maximale Anzahl der von verschiedenen Modellen unterstützten Untergeräte ist unterschiedlich und sollte je nach Maßstab ausgewählt werden.
