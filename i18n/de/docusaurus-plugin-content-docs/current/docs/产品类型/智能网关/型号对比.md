# Modellvergleich

## Vergleich verschiedener Modelle

| Modell | Maximale Grenze des Zigbee-Untergeräts | Vernetzungsmethode | Funktionen | Spielvorschläge |
|------|----|----------|------|----------|
| [HS6GW](../../Wireless Protocol/Zigbee/HS6GW – Intelligentes Gateway (Zigbee)) | 128 | WLAN + Ethernet | Das neueste Modell und unterstützt zukünftige Thread-Upgrades | Haimans eigener ökologischer Kern, kombiniert mit der Haiman APP-Fernverwaltung, geeignet für Benutzer, die sich nicht darum kümmern |
| HS5GW | 64 | WLAN + Ethernet | Hohe Leistung | Mittlere und große Projekte, stabil und zuverlässig |
| HS3GW | 32 | WLAN | Verbessert | Ausreichend für kleine und mittlere Häuser |
| HS1GW | 16 | WLAN | Grundlegend | Einstiegsmodell, geeignet für eine kleine Anzahl von Geräten |

## Empfohlenes Szenen-Gameplay

| Szenario | Empfohlenes Modell | Beschreibung |
|------|----------|------|
| Ganzhaus-Heiman-Ökologie | HS6GW | Wird mit Heiman Zigbee-Sensoren und Fernverwaltung mit der Heiman-App verwendet, Sie müssen keinen eigenen Server erstellen |
| Benutzer von Drittanbieterplattformen | — | Für eine höhere Flexibilität wird empfohlen, einen Zigbee-Koordinator (z. B. SLZB-06) direkt zu erwerben, um ihn mit Zigbee2MQTT/ZHA zu verwenden
| Kleine und mittlere Wohnungen | HS3GW | 32 Gerätelimit zur Deckung der Grundbedürfnisse |

## Auswahlvorschläge

- **Haimans eigene Ökologie**: [HS6GW](../../Wireless Protocol/Zigbee/HS6GW - Smart Gateway (Zigbee)) (128 Geräte, WiFi + Ethernet)
- **Plattform von Drittanbietern**: Empfohlener Universal Zigbee Coordinator + Zigbee2MQTT/ZHA
