# Smart Gateway

Smart Gateway ist der zentrale Knotenpunkt des Haiman-Smart-Home-Systems und verantwortlich für die Verbindung verschiedener Zigbee-Sensoren und die Realisierung der Verbindungssteuerung zwischen Geräten.

## Produktserie

| Serie | Kommunikationsmethoden | Funktionen |
|------|----------|------|
| HS1GW-Serie | Zigbee + WLAN / Ethernet | Basis-Gateway |
| HS3GW-Serie | Zigbee + WLAN / Ethernet | Verbessert |
| HS5GW-Serie | Zigbee + WLAN / Ethernet | Hochleistungs-Gateway |
| HS6GW-Serie | Zigbee + Wi-Fi / Ethernet + Thread | Das neueste Modell unterstützt sowohl Zigbee als auch Matter over Thread |

## Kernfunktionen

- Koordinator des Zigbee-Netzwerks
- Unterstützt Wi-Fi/Ethernet für die Verbindung mit dem Internet
- Automatische Geräteerkennung und -kopplung
- Lokale Szenenverknüpfung (einige Modelle)
- Fernzugriff (über Heiman Cloud Platform)
- Unterstützt bis zu 128 Untergeräte (je nach Modell)

## Anwendbare Szenarien

- Der zentrale Verbindungspunkt der Heiman-Geheimdienste für das ganze Haus
- Arbeiten Sie mit der Haiman APP zusammen, um eine Fernüberwachung zu realisieren
- Lokale zentrale Verwaltung kleiner und mittlerer kommerzieller Projekte

## Unterschiede zu anderen Gateways

Haiman Gateway ist der Eingang zum offiziellen Ökosystem. Für Benutzer, die eine Plattform eines Drittanbieters verwenden möchten, wird empfohlen, den universellen Zigbee-Koordinator mit Zigbee2MQTT/ZHA zu verwenden.
