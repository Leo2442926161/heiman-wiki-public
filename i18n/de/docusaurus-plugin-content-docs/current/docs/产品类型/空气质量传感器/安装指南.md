# Installationsanleitung

## Anforderungen an den Installationsort

- Installiert in **1,2 ~ 1,8 m** Höhe (Atmungszone)
- Wählen Sie einen Standort, der der durchschnittlichen Luftqualität im Raum entspricht
- **Fernhalten**: Fenster, Türen, Lüftungsschlitze, Klimaanlagenauslässe
- **Fernbleiben**: Küchendämpfe, Luftbefeuchter, Aromatherapie und andere lokale Schadstoffquellen

## Installationsschritte

1. Wählen Sie den Installationsort (Wohnzimmer, Schlafzimmer oder Bürowand)
2. Befestigen Sie den Montagesockel mit Schrauben
3. An die Stromversorgung anschließen (USB oder Adapter)
4. Der Sensor ist eingeschaltet und das Display (falls vorhanden) zeigt Daten nach dem Selbsttest an.
5. Kopplung über Gateway
6. Bestätigen Sie, dass die Daten in der APP normal sind

## Notizen

- Stellen Sie sicher, dass der Lufteinlass und -auslass des Sensors nicht blockiert ist
- Reinigen Sie das Sensorgehäuse regelmäßig, um eine Verstopfung durch Staub zu verhindern
- Platzieren Sie es im Kinderzimmer, um den CO₂-Gehalt zu überwachen und Sie aufzufordern, das Fenster zum Lüften zu öffnen
- Empfohlen zur Kombination mit Frischluftsystem oder Luftreiniger
