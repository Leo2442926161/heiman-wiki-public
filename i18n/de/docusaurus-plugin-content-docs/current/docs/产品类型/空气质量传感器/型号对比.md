# Modellvergleich

## Vergleich verschiedener Modelle

| Modell | Protokoll | Erkennungsparameter | Stromversorgung | Funktionen | Gameplay-Vorschläge |
|------|------|----------|------|------|----------|
| Luftqualitätssensor | Zigbee | PM2,5 / PM10 / CO₂ / TVOC / Temperatur und Luftfeuchtigkeit | Netzstrom | Multifunktion in einem | Überwachen Sie die Luftqualität in Innenräumen, verknüpfen Sie automatisch Frischluft/Luftreiniger und senden Sie eine Push-Erinnerung, wenn die Norm überschritten wird |

## Empfohlenes Szenen-Gameplay

| Szenarien | Empfohlene Modelle | Verbindungslösungen |
|------|----------|----------|
| CO₂-Überwachung im Schlafzimmer | Luftqualitätssensor | CO₂ > 1000 ppm → Fensteröffnungserinnerung/vernetztes Frischluftsystem für Belüftung, bessere Schlafqualität |
| Neu renoviertes Haus | Luftqualitätssensor | Kontinuierliche Überwachung von TVOC/Formaldehyd, angeschlossener Abluftventilator bei Überschreitung des Standards und eine beruhigende Benachrichtigung, sobald der Standard erreicht ist |
| Kinderzimmer | Luftqualitätssensor | PM2,5-Überwachung, automatisches Einschalten des Luftreinigers bei Überschreitung des Standards, um die Gesundheit der Atemwege sicherzustellen |
| Büro | Luftqualitätssensor | CO₂ + Temperatur und Luftfeuchtigkeit gekoppelte Klimaanlage, Frischluft zur Verbesserung der Arbeitseffizienz |
