# Luftqualitätssensor

Luftqualitätssensoren (Air Quality Monitor) werden eingesetzt, um verschiedene Schadstoffe in der Raumluft zu erkennen und so zur Aufrechterhaltung eines gesunden Wohnklimas beizutragen.

## Erkennungsparameter

Luftqualitätssensoren von Hyman erkennen die folgenden Indikatoren:

| Parameter | Beschreibung | Gesundheitsstandards |
|------|------|----------|
| PM2,5 | Feinstaub | ≤ 35 µg/m³ (24-Stunden-Durchschnitt) |
| PM10 | Feinstaub | ≤ 50 µg/m³ (24-Stunden-Durchschnitt) |
| CO₂ | Kohlendioxid | ≤ 1000 ppm |
| TVOC | Gesamtmenge flüchtiger organischer Verbindungen | ≤ 0,6 mg/m³ |
| Temperatur | Umgebungstemperatur | 18-26°C |
| Luftfeuchtigkeit | Relative Luftfeuchtigkeit | 40-60 % relative Luftfeuchtigkeit |

## Kernfunktionen

- Multiparameter-Luftqualitätserkennung
- Farbdisplay (je nach Modell)
- Ton- und Lichtalarm bei Überschreitung des Standards
- Historische Datensätze
- Verbundene Frischluft/Luftreiniger

## Anwendbare Szenarien

- Schlafzimmer: Überwachen Sie den CO₂-Gehalt und fordern Sie auf, die Fenster zum Lüften zu öffnen
- Kinderzimmer: Schützen Sie die Gesundheit der Atemwege von Kindern
- Büro: Produktivität und Komfort steigern
- Neu renovierte Häuser: Formaldehyd-/TVOC-Freisetzung überwachen
