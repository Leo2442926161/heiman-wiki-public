# Installationsanleitung

## Anforderungen an den Installationsort

- **Deckenmontage**, in der Mitte des Raumes angebracht
- **Beste Einsatzszenarien**: Küchen, Garagen, Heizräume, Badezimmer – Orte, an denen Rauchsensoren anfällig für Fehlalarme sind
- Mindestens **30 cm** von der Wand entfernt
- Vermeiden Sie die Installation in der Nähe von Lüftungsschlitzen und Klimaanlagenauslässen

## Installationsschritte

1. Wählen Sie den Installationsort (empfohlene Küchen- oder Garagendecke)
2. Befestigen Sie den Montagesockel an der Decke
3. Legen Sie die Batterie ein und der Sensor schaltet sich automatisch ein.
4. Drehen Sie den Hauptkörper und lassen Sie ihn in der Basis einrasten
5. Selbsttest: Drücken Sie die Testtaste, die Kontrollleuchte leuchtet auf und es ertönt ein Ton.
6. Kopplung über Gateway (Zigbee-Version)

## Notizen

- Der Temperatursensor** ist kein Ersatz für den Rauchsensor**, sondern eine Ergänzung
- Der Rauchsensor reagiert schneller und der Temperatursensor verfügt über eine stärkere Entstörung
- Es wird empfohlen, sowohl Rauch- als auch Temperatursensoren im selben Raum zu installieren
- Testen Sie einmal im Monat und tauschen Sie die Batterie rechtzeitig aus, wenn die Batterie schwach ist
