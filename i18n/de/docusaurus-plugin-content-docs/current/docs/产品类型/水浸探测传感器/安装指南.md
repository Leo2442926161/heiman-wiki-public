# Installationsanleitung

## Anforderungen an den Installationsort

- **Auf den Boden legen**, achten Sie darauf, dass die Sonde den Boden berührt
- **In der Nähe der Wasserquelle**: unter dem Waschbecken im Badezimmer, unter dem Spülbecken in der Küche, neben der Waschmaschine, unter dem Warmwasserbereiter
- Der tiefste Punkt, an dem Wasserlecks erkannt werden müssen

## Installationsschritte

1. Bestimmen Sie den Installationsort (den Boden im Risikobereich für Wasserlecks).
2. Reinigen Sie die Bodenoberfläche
3. Platzieren Sie den Sensor so auf dem Boden, dass die Sonde den Boden berührt
4. Bei Verwendung der Kabelvariante (HS2WL) das Verlängerungskabel entlang des zu überwachenden Bereichs verlegen
5. Legen Sie die Batterie ein und der Sensor schaltet sich automatisch ein.
6. Simulationstest: Berühren Sie die Sonde mit nassen Fingern, die Alarmleuchte leuchtet auf
7. Kopplung über Gateway

## Notizen

- Die Sonde kann korrodieren, wenn sie längere Zeit einer feuchten Umgebung ausgesetzt wird. Überprüfen Sie sie daher regelmäßig
- Es wird empfohlen, mit dem Magnetventil zusammenzuarbeiten, um eine automatische Abschaltung zu erreichen
-Die Batterielebensdauer beträgt ca. 2-3 Jahre
- Mehrere Sensoren zur Abdeckung verschiedener Bereiche
