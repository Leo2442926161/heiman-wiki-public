# Sensor zur Erkennung des Eintauchens in Wasser

Der Wasserlecksensor dient zur Erkennung von Wasserlecks oder Wasseransammlungen, wodurch Sachschäden durch Wasserrohrbrüche und Wasserüberschwemmungen wirksam verhindert werden können.

## Produktserie

| Serie | Kommunikationsmethoden | Funktionen |
|------|----------|------|
| HS1WL | Zigbee | Grundmodell, Sondenerkennung |
| HS2WL-Serie | Zigbee | Sonden- und Kabelerkennung, zweikanalig |
| HS3WL-Serie | Zigbee | Erweiterte Version |
| L1-Serie | Materie / Zigbee | Matter over Thread-Version, WWHA-zertifiziert |

## Kernfunktionen

- Echtzeiterkennung von Wasserlecks/stehendem Wasser
- Summeralarm
- Erinnerung an niedrigen Batteriestand
- Korrosionsschutzdesign der Sonde

## Installationsort

- Unter dem Waschbecken im Badezimmer
- Unter der Küchenspüle
- Neben der Waschmaschine/Spülmaschine
- Unter dem Warmwasserbereiter
- Keller/Garage
- In der Nähe des Kühlers

## Szenenverknüpfung

- **Wasserleckerkennung + Magnetventil**: Hauptventil automatisch schließen, wenn Wasserleckage erkannt wird
- **Wasserleckalarm + Push**: Benachrichtigen Sie die Unterkunft oder Familienmitglieder auf Ihrem Mobiltelefon, wenn Sie nicht zu Hause sind
- **Hochwasser + intelligente Abschaltung**: Verbundener Gaswarmwasserbereiter oder intelligentes Wasserventil
