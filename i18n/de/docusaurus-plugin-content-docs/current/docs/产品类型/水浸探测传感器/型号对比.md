# Modellvergleich

## Vergleich verschiedener Modelle

| Modell | Protokoll | Erkennungsmethode | Stromversorgung | Funktionen | Spielvorschläge |
|------|------|----------|------|------|----------|
| [HS1WL](../../Wireless Protocol/Zigbee/HS1WL – Wassertauchdetektor (Zigbee)) | Zigbee | Sonde | Batterie | Grundmodell | Alarmknopf bei austretendem Wasser, geeignet für unter dem Waschbecken im Badezimmer |
| [HS2WL](../../Wireless Protocol/Zigbee/HS2WL – Wassertauchdetektor (Zigbee)) | Zigbee | Sonde + Kabel | Batterie | Zweikanal | Sondenerkennungspunkt + Kabel decken einen großen Bereich ab, am besten neben einem Warmwasserbereiter/einer Waschmaschine |
| [HS3WL](../../Wireless Protocol/Zigbee/HS3WL - Wasserleckdetektor (Zigbee)) | Zigbee | Sonde | Batterie | Erweiterte Version | Höhere Empfindlichkeit, lauterer Summer |
| [L1-M](../../Wireless Protocol/Matter/L1-M Wasserimmersionsdetektor (Matter)) | Materie über Faden | Sonde | Batterie | Materieversion | WWHA-zertifizierter, nativer Zugriff auf Apple Home |

## Empfohlenes Szenen-Gameplay

| Szenarien | Empfohlene Modelle | Verbindungslösungen |
|------|----------|----------|
| Überwachung von Wasserlecks im ganzen Haus | HS2WL + L1-M | Abdeckung mehrerer Punkte + native HomeKit-Benachrichtigung, Wasserleck erkennen → Hauptwasserventil ausschalten → Push-Benachrichtigung |
| Speziell für Waschmaschinen | HS2WL | Die Kabelsonde wird unten in der Waschmaschine verlegt. Sobald Wasser austritt, wird sofort ein Alarm ausgelöst und die Waschmaschine ausgeschaltet
| Warmwasserbereiter/Fußbodenheizung | HS3WL | Die verbesserte Version verfügt über eine hohe Empfindlichkeit und kann Lecks in Fußbodenheizungsrohren bereits im Anfangsstadium erkennen |
| Matter-Benutzer | L1-M | Kein dediziertes Gateway erforderlich, Thread-Alarmverknüpfung mit geringer Latenz Apple Home Automation |

## Auswahlvorschläge

- **Punkterkennung**: [HS1WL](../../Wireless Protocol/Zigbee/HS1WL - Wasserleckdetektor (Zigbee)) oder [L1-M](../../Wireless Protocol/Matter/L1-M Wasserleckdetektor (Matter))
- **Großflächenabdeckung**: [HS2WL](../../Wireless Protocol/Zigbee/HS2WL - Wasserleckdetektor (Zigbee)) (Sonde + Kabel)
- **Matter-Anforderungen**: [L1-M](../../Wireless Protocol/Matter/L1-M Wasserimmersionsdetektor (Matter))
