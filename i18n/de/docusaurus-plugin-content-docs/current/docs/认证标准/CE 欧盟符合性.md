# CE EU-Konformität

Das CE-Zeichen (Conformité Européenne, EU-Konformität) ist ein obligatorisches Sicherheitszeichen für Produkte, die auf den EU-Markt gelangen.

## Bedeutung des CE-Zeichens

Das CE-Zeichen zeigt an, dass das Produkt den grundlegenden Anforderungen der relevanten EU-Richtlinien entspricht und in den EU-Mitgliedstaaten und im Europäischen Wirtschaftsraum (EWR) frei verkehren kann.

## CE-Zertifizierungspfad für Haiman-Produkte

Rauch-/CO-Melder von Hyman erfüllen die CE-Anforderungen durch:

### CPR-Verordnung (EU Nr. 305/2011)

Die Bauproduktenverordnung schreibt vor, dass Brandmeldeprodukte einer Leistungsstabilitätsbewertung unterzogen werden müssen:
- Gemäß Normen: EN 14604 (Rauch), EN 50291 (CO)
- Begutachtungsstelle: TÜV Rheinland und weitere benannte Stellen
- Zertifizierungsstufe: DoP (Leistungserklärung) + CE-Zeichen

### RED-Richtlinie (2014/53/EU)

Die Funkgeräterichtlinie schreibt vor, dass drahtlose Geräte:
- Deckt Zigbee, Z-Wave, Wi-Fi, Thread und andere drahtlose Module ab
- Prüfgegenstände: Elektromagnetische Verträglichkeit (EMV), Radiofrequenz (RF), SAR, Sicherheit

### EMV-Richtlinie (2014/30/EU)

Die Richtlinie zur elektromagnetischen Verträglichkeit stellt sicher, dass Geräte keine Interferenzen mit anderen elektronischen Geräten verursachen.

## CE-gekennzeichnete Produkte

Alle auf dem EU-Markt verkauften Heymann-Produkte tragen das CE-Zeichen, darunter:
- HS1SA-E / HS3SA (Zigbee-Rauchmelder)
- HS1CA-E/C1-Serie (Zigbee CO-Alarm)
- HS1SA-M/HS1CA-M (Materie über Faden)
- Komplettes Sortiment an Zigbee-Sensoren wie HS1MS / HS1DS
