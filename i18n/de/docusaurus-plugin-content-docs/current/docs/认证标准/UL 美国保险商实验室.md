# UL Underwriters Laboratories

UL (Underwriters Laboratories) ist die maßgeblichste Organisation für die Entwicklung und Zertifizierung von Produktsicherheitsstandards in den Vereinigten Staaten. Die wichtigsten UL-Standards für Heiman-Produkte sind folgende:

## UL 217 – Rauchmelderstandard (USA)

| Projekt | Beschreibung |
|------|------|
| Anwendungsbereich | Rauchmelder für Privathaushalte (US-Markt) |
| Nachweisanforderungen | Photoelektrische/ionisierte Raucherkennung, Ton- und Lichtalarm, über 85 dB |
| Anwendbare Märkte | Vereinigte Staaten, Kanada |
| Heiman-Produkte | HS8SA (US-Version) |
| Bedeutung der Zertifizierung | Die Einhaltung der Brandschutzbestimmungen verschiedener US-Bundesstaaten ist eine notwendige Voraussetzung für den Marktzugang in Nordamerika |

**Unterschiede zu anderen Normen**: Die Testszenarien von UL 217 umfassen typische amerikanische Hausbrandszenarien wie Blitzbrände und langsam brennende Brände und unterscheiden sich geringfügig von den Testmethoden von EN 14604.

## UL 2034 – Kohlenmonoxid-Alarmstandard (USA)

| Projekt | Beschreibung |
|------|------|
| Anwendungsbereich | CO-Alarm für Haushalte (US-Markt) |
| Nachweisanforderungen | Elektrochemischer Sensor, Konzentration × Zeit-Integrationsalgorithmus |
| Anwendbare Märkte | Vereinigte Staaten, Kanada |
| Hayman-Produkte | HS-720ES-M (US-Version) |
| Bedeutung der Zertifizierung | Die meisten Bundesstaaten der Vereinigten Staaten schreiben die Installation von CO-Meldern vor |

## Die Hauptunterschiede zwischen UL-Zertifizierung und EN-Zertifizierung

| Aspekte | UL | DE |
|------|-----|-----|
| Normen für Rauchmelder | UL 217 | EN 14604 |
| CO-Alarmstandards | UL 2034 | EN 50291 |
| Erkennungsmethode | Spezifischer Brandkurventest | Verschiedene Brandtypprüfungen |
| Anwendbare Spannung | 120V / Batterie | 230V / Batterie |
