# EN Europäische Norm

EN (Europäische Norm, Europäische Norm) ist eine technische Norm, die vom Europäischen Komitee für Normung (CEN) und dem Europäischen Komitee für elektrotechnische Normung (CENELEC) entwickelt wurde. Die wichtigsten EN-Normen für Heiman-Produkte sind folgende:

## EN 14604 – Norm für Rauchmelder

| Projekt | Beschreibung |
|------|------|
| Anwendungsbereich | Unabhängiger Rauchmelder |
| Nachweisanforderungen | Photoelektrische Raucherkennung, Ton- und Lichtalarm, Meldung bei niedrigem Batteriestand |
| Anwendbare Märkte | EU-Mitgliedstaaten, Großbritannien, Schweiz, Norwegen usw. |
| Heiman-Produkte | HS1SA-E, HS3SA, HS1SA-M, S1-Serie usw. |
| Bedeutung der Zertifizierung | Die Erfüllung der Anforderungen der EU-Brandschutzvorschriften für Gebäude ist eine notwendige Voraussetzung für den europäischen Marktzugang |

**Testpunkte**: Reaktionszeit, Schalldruckpegel (≥85 dB/3m), Stummschaltungsfunktion, Batterielebensdauer, Temperaturzyklus, Feuchtigkeitsbeständigkeit, Beständigkeit gegen Funkfrequenzstörungen usw.

## EN 50291 – Kohlenmonoxid-Alarmnorm

| Projekt | Beschreibung |
|------|------|
| Anwendungsbereich | Kohlenmonoxid (CO)-Alarm für den Haushalt |
| Prüfanforderungen | Elektrochemischer CO-Sensor, Konzentrationsklassifizierungsalarm |
| Anwendbare Märkte | EU-Mitgliedstaaten, Großbritannien |
| Heiman-Produkte | HS1CA-E, C1-Serie, HS1CA-M usw. |
| Bedeutung der Zertifizierung | Um die Zuverlässigkeit der CO-Leckerkennung zu gewährleisten, ist in einigen Ländern die Installation vorgeschrieben |
