# Andere Zertifizierungen

## Q-Label-Qualitätszeichen

Q-Label ist ein Qualitätszertifizierungszeichen für die europäische Smart-Home-Branche, das von der Q-Lab Alliance (jetzt Teil der Zigbee Alliance / CSA Connectivity Standards Alliance) ins Leben gerufen wurde.

| Projekt | Beschreibung |
|------|------|
| Anwendungsbereich | Zertifizierung der Zigbee-Geräteinteroperabilität |
| Zertifizierungsinhalt | Gerätekompatibilität mit Zigbee-Gateways/Controllern anderer Marken |
| Bedeutung | Stellen Sie markenübergreifende Interoperabilität sicher und verringern Sie das Risiko, dass Sie es „zu Hause kaufen und nicht nutzen“ |

Hyman Zigbee-Produkte sind Q-Label-zertifiziert und gewährleisten die Kompatibilität mit den folgenden Plattformen:
- Amazon Echo Plus / Echo Show (Zigbee integriert)
- Philips Hue Bridge
- SmartThings Hub
- Andere Q-Label-Authentifizierungs-Gateways

## Red Dot Design Award

Der Rauchmelder der HS1SA-Serie von Hyman gewann den **Red Dot Design Award** (Red Dot Design Award), was Hymans Stärke im Produktdesign widerspiegelt.

## ISO 9001-Qualitätsmanagementsystem

Haiman hat die Zertifizierung des Qualitätsmanagementsystems ISO 9001:2015 bestanden, um eine vollständige Prozessqualitätskontrolle von der Forschung und Entwicklung über die Produktion bis zum Kundendienst sicherzustellen.
