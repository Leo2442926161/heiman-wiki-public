# Anwendungstipps

## Gerätekopplung

1. Geben Sie im Home Assistant „Einstellungen → Geräte und Dienste → ZHA → Konfiguration“ ein
2. Klicken Sie auf „Gerät hinzufügen“ und starten Sie dann den Kopplungsmodus des Heiman-Geräts
3. Wenn die Kopplung fehlschlägt, bringen Sie das Gerät in die Nähe des USB-Koordinators und versuchen Sie es innerhalb von 3 Metern erneut

## Kanalkoordination

Der ZHA-Standardkanal ist 15. Bei schwerwiegenden Störungen des WLAN 2,4 GHz:
- Verwenden Sie das „Silicon Labs Flasher Utility“, um den Kanal zu ändern
- Häufig empfohlene Kanäle: 15, 20, 25 (WiFi 1/6/11 vermeiden)

## Netzwerkoptimierung

- Fügen Sie alle 3–4 Heiman-Geräte ein Zigbee-Routing-Gerät (z. B. einen Smart Socket) hinzu, um das Netzwerk zu erweitern
- Überprüfen Sie die Qualität der Geräteverbindung mit dem ZHA-Netzwerkvisualisierungstool
- Überprüfen Sie regelmäßig, ob Firmware-Updates für das Gerät verfügbar sind, um eine optimale Kompatibilität zu gewährleisten
