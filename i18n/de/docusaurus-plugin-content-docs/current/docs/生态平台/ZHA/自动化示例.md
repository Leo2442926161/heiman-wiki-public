# Automatisierungsbeispiel

## Rauchmelderverknüpfung

„yaml
Alias: „ZHA-Rauchmelder-Verknüpfung“
Auslöser:
  - Auslöser: Zustand
    Entitäts-ID: Binary_sensor.heiman_smoke_zha
    zu: „auf“
Aktionen:
  - Aktion: light.turn_on
    Ziel:
      Bereichs-ID: Wohnzimmer
  - Aktion: tts.speak
    Ziel:
      Entitäts-ID: media_player.speaker
    Daten:
      Meldung: „Rauchalarm ausgelöst!“
„

## Automatisierung der Gerätekopplung

„yaml
Alias: „Erinnerung an die Kopplung neuer Geräte“
Auslöser:
  - Auslöser: zha.event
    Untertyp: Heiman
    Befehl: device_joined
Aktionen:
  - Aktion: notify.persistent_notification
    Daten:
      Meldung: „Das neue Haiman-Gerät ist dem ZHA-Netzwerk beigetreten, bitte gehen Sie zum Konfigurationsnamen und -bereich.“
„

## Wassereintaucherkennung

„yaml
Alias: „ZHA Flood Test“
Auslöser:
  - Auslöser: Zustand
    Entitäts-ID: Binary_sensor.heiman_water_leak_zha
    zu: „auf“
Aktionen:
  - Aktion: switch.turn_off
    Entitäts-ID: switch.main_water_valve
  - Aktion: notify.mobile_app_phone
    Daten:
      Meldung: „💧 Überschwemmung erkannt!“
„
