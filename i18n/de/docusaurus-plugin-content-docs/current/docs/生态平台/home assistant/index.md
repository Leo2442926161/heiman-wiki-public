#HomeAssistant

Haiman Technology trat im Februar 2026 offiziell dem Zertifizierungsprogramm **Works with Home Assistant** bei und wurde einer der ersten chinesischen Partner des Programms.

## Authentifizierungsgerät

Die folgenden Heiman-Geräte wurden offiziell zertifiziert, um die Lokalisierung und stabile Kompatibilität mit Home Assistant zu gewährleisten:

| Ausrüstung | Region | Verbindungsmethode |
|------|------|----------|
| Intelligenter Rauchmelder | Vereinigte Staaten | Materie über Faden |
| Intelligenter Rauchmelder | EU / China | Materie über Faden |
| Intelligenter Kohlenmonoxidmelder | Vereinigte Staaten | Materie über Faden |
| Intelligenter Kohlenmonoxidmelder | EU / China | Materie über Faden |
| Infrarotsensor für den menschlichen Körper M1-Serie | Global | Materie / Zigbee |
| Wassertauchdetektor L1-Serie | Global | Materie / Zigbee |
| Temperatur- und Feuchtigkeitssensor H1-Serie | Global | Materie / Zigbee |

## Integrationsmethode

### von Matter

1. Stellen Sie sicher, dass Sie über einen Matter-over-Thread-Grenzrouter verfügen (z. B. Home Assistant Green + Connect ZBT-2, Apple HomePod, Google Nest Hub usw.).
2. Konfigurieren Sie die Matter-Integration in Home Assistant
3. Fügen Sie das Gerät hinzu, um es automatisch zu erkennen

### über Zigbee2MQTT/ZHA

Hyman Zigbee-Geräte werden nativ von Zigbee2MQTT und ZHA unterstützt, ohne dass eine zusätzliche Konfiguration erforderlich ist.

## Lokalisierungskontrolle

Alle zertifizierten Geräte unterstützen **lokale Steuerung**, ohne auf Cloud-Dienste angewiesen zu sein. Selbst wenn das Internet ausfällt, können Sensoralarme und Automatisierungsverbindungen weiterhin normal funktionieren.

## Community-Engagement

Heyman beteiligt sich aktiv an Diskussionen in der Home Assistant-Community und optimiert weiterhin das Produkterlebnis auf der Grundlage des Benutzerfeedbacks.
