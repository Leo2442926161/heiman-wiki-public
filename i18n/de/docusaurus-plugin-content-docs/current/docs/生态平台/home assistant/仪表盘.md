# Dashboard-Konfiguration

## Sicherheitsübersichtsbereich

Verwenden Sie das Layout der Abschnittsansicht, um Heimann-Geräte in Bereiche zu organisieren:

„yaml
Typ: Abschnitt
Titel: Rauch- und CO-Melder
Karten:
  - Typ: Entität
    Entität: binärer_sensor.heiman_smoke_alarm
    Name: Rauchmelder
  - Typ: Entität
    Entität: binärer_sensor.heiman_co_alarm
    Name: CO-Alarm
„

## Pilzkartenstil

„yaml
Typ: Benutzerdefiniert:Mushroom-Alarm-Control-Panel-Card
Entität: alarm_control_panel.home_alarm
Name: Sicherheitssystem

Typ: benutzerdefiniert:mushroom-entity-card
Entität: binärer_sensor.heiman_smoke_alarm
Name: Rauchmelder
Symbol: mdi:rauchmelder
primäre_info: Staat
sekundäre_info: zuletzt geändert
fill_container: true
„

## Umgebungssensorpanel

„yaml
Typ: Abschnitt
Titel: Umweltüberwachung
Karten:
  - Typ: Messgerät
    Entität: sensor.heiman_temperature
    Name: Temperatur
    Einheit: „°C“
    Min.: 0
    maximal: 50
    Schweregrad:
      grün: 0
      gelb: 30
      Rot: 40
  - Typ: Messgerät
    Entität: sensor.heiman_humidity
    Name: Luftfeuchtigkeit
    Einheit: „%“
    Min.: 0
    maximal: 100
    Schweregrad:
      grün: 30
      Gelb: 60
      Rot: 80
„

## Alarmverlaufsfenster

„yaml
Typ: Verlaufsdiagramm
Titel: Sensorgeschichte
Entitäten:
  - binärer_sensor.heiman_smoke_alarm
  - binärer_sensor.heiman_co_alarm
  - binärer_sensor.heiman_water_leak
Stunden_bis_Show: 24
„

## Vorgeschlagenes Layout

- Oben: Status des Sicherheitssystems + Statuskarte des Alarmgeräts
- Zentral: Umweltüberwachung (Temperatur, Luftfeuchtigkeit, Wasserqualität)
- Unten: Historisches Trenddiagramm des Sensoralarms
- Shortcut-Funktionen in der Seitenleiste: Scharf-/Unscharfschalten, Szenenwechsel
