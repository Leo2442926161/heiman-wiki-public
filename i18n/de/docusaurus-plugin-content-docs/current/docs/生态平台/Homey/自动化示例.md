# Automatisierungsbeispiel

## Rauchmelder – blinkt und benachrichtigt

**Flow-Konfiguration:**

1. **Auslöser:** Hyman-Rauchmelder → Status ändert sich in „Alarm“.
2. **Bedingungen:** (Keine)
3. **Aktion:**
   - Wohnzimmerlicht → blinkt (rot, einmal pro Sekunde)
   - Push-Benachrichtigung → Mobil: „⚠️ Rauchmelder ausgelöst!“
   - Sprachansage → Homey Speaker: „Rauch erkannt, bitte auf Sicherheit achten!“

## CO-Alarm – Kessel abschalten

**Flow-Konfiguration:**

1. **Auslöser:** Hyman CO-Alarm → Status ändert sich in „Alarm“.
2. **Bedingungen:** (Keine)
3. **Aktion:**
   - Kesselschalter → AUS
   - Intelligente Steckdose (Frischluftsystem) → Ein
   - Push-Benachrichtigung → „Kohlenmonoxid-Alarm, Kessel wurde abgeschaltet“

## Temperatur- und Feuchtigkeitsüberschreitungsalarm

1. **Auslöser:** Hyman-Temperatur- und Luftfeuchtigkeitssensor → Temperatur über 40 °C
2. **Aktion:**
   - Klimaanlage → Kühlmodus einschalten
   - Push-Benachrichtigung → „Die Raumtemperatur ist zu hoch, die Klimaanlage wurde automatisch eingeschaltet“
