# Anwendungstipps

## Störung des Zigbee-Kanals

Der integrierte Zigbee-Chip von Homey nutzt bestimmte Kanäle. Wenn das Heiman-Gerät häufig offline geht:

1. Sehen Sie sich den aktuellen Kanal in den Homey-Einstellungen → Zigbee an
2. Verwenden Sie die Homey Insight App, um nach Umgebungseinflüssen zu suchen
3. Wechseln Sie zu einem Kanal mit weniger Störungen (weg von WLAN 1/6/11)

## Gerätevertriebsnetz

- So gelangen Sie in den Kopplungsmodus für Heiman-Geräte: Halten Sie die Taste 5 Sekunden lang gedrückt (bei einigen Modellen muss dreimal schnell gedrückt werden).
- Wenn Sie ein Gerät in der Homey App hinzufügen, wählen Sie „Gerät hinzufügen → Zigbee → Suchen“
- Wenn die Kopplung fehlschlägt, bringen Sie das Gerät näher an Homey heran (innerhalb von 3 Metern) und versuchen Sie es erneut

## Vorschläge zur Flussoptimierung

- Fügen Sie im Flow eine „Verzögerungs“-Bedingung hinzu (z. B. 2 Sekunden), um vorübergehende Fehlalarme zu vermeiden
- Verwenden Sie die Funktion „Tag“, um Heiman-Geräte zu klassifizieren und zu verwalten

## Erweiterung der Signalreichweite

Wenn das Heiman-Gerät weit von Homey entfernt ist, kann ein Zigbee-Relaisgerät (z. B. eine Smart-Steckdose) hinzugefügt werden, um die Netzwerkabdeckung zu erweitern.
