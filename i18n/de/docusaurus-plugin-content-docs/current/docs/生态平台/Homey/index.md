#Heimlich

Homey ist ein Smart-Home-Hub des niederländischen Unternehmens Athom, der Zigbee, Z-Wave, Wi-Fi, Matter, Thread und andere Protokolle unterstützt.

## Integrationsmethode

Homey Zigbee- und Z-Wave-Geräte koppeln sich direkt über den integrierten Wireless-Chip von Homey, sodass kein zusätzliches Gateway erforderlich ist. Auf Matter-Geräte kann über den integrierten Thread-Border-Router von Homey Pro zugegriffen werden.

## Vorteile

- **Mehrere Protokolle in einem**: Homey Pro unterstützt Zigbee, Z-Wave, Thread und Wi-Fi gleichzeitig
- **Lokal ausführen**: Die gesamte Automatisierung wird lokal ausgeführt
- **Flow-Automatisierung**: Drag-and-Drop-Programmierung zur Erstellung komplexer Verknüpfungsszenarien

> **Referenz**: Heiman-bezogene Apps können im Homey App Store gesucht werden
