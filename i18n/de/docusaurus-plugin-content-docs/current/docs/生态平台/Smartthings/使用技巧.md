# Anwendungstipps

## Hub-Kompatibilität

Hyman Zigbee-Geräte sind mit **SmartThings Hub v2** und **Hub v3 (Aeotec)** kompatibel.

| Hub-Modell | Kompatibilität | Beschreibung |
|---------|--------|------|
| Hub v2 (STH-ETH-250) | ✅ | Erfordert ein Firmware-Update auf die neueste Version |
| Hub v3 (Aeotec) | ✅ | Native Unterstützung, empfohlen |
| Samsung SmartThings Station | ⚠️ | Unterstützt nur einige Zigbee-Geräte |

## Aufforderungen zur Gerätekopplung

- Klicken Sie in der SmartThings-App auf „+“ → „Gerät“ → „Marke“ → „Hyman“.
- Geräte, die nicht automatisch erkannt werden, können manuell über „Scan Zigbee“ gesucht werden.
- Es wird empfohlen, den Abstand zwischen dem Gerät und dem Hub beim ersten Pairing auf höchstens 5 Meter zu halten

## Batterieoptimierung

Das standardmäßige Meldeintervall des Heiman-Sensors ist länger. Wenn Echtzeitleistung erforderlich ist, kann das „Erkennungsintervall“ auf der Geräteseite auf 5 Minuten angepasst werden.

## Best Practices für die Automatisierung

- Verwenden Sie die Funktion „Modi“ von SmartThings, um zwischen den Zuständen „Zuhause“ und „Abwesend“ zu unterscheiden
- Erstellen Sie aus Redundanzgründen **zwei Automatisierungssätze** für wichtige Sensoren (Rauch, CO).
