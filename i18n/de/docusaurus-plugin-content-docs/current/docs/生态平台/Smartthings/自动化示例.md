# Automatisierungsbeispiel

## „Gute Nacht“-Schlafenszeitmodus

**Auslösebedingung:** Haiman-Türmagnetsensor → Schließen (zeigt an, dass die Schlafzimmertür geschlossen wurde)

**Aktion ausführen:**
- Schalten Sie alle Lichter im Wohnzimmer aus
- Smart Lock → Sperren
- Sicherheitssystem → Nachts aktivieren
- Push-Benachrichtigung: „Gute Nacht, das System ist scharf“

## „Abwesend“-Modus

**Auslösebedingung:** Hyman-Infrarotsensor für den menschlichen Körper → 15 Minuten lang keine Bewegung

**Aktion ausführen:**
- Alle Lichter ausgeschaltet
- Thermostat → Energiesparmodus
- Sicherheitssystem → Scharfschaltung außerhalb des Hauses
- Türsensor → Alarm einschalten
- Push-Benachrichtigung: „Away-from-Home-Modus ist aktiviert, Tür- und Fenstersensoren sind aktiviert“

## Rauchmelder-Notfallmodus

**Auslösebedingung:** Hyman-Rauchmelder → Rauch erkannt

**Aktion ausführen:**
- Intelligenter Lautsprecher → Alarmton abspielen
- Alle Smart Lights → blinken rot
- Push-Benachrichtigungen an alle Familienmitglieder
- Bei Verbindung mit SmartThings Hub → Sirene auslösen
