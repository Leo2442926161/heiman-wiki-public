#SmartThings

SmartThings ist eine von Samsung eingeführte Smart-Home-Plattform, die die Protokolle Zigbee, Z-Wave und Matter unterstützt.

## Integrationsmethode

Heiman-Geräte können auf folgende Weise eine Verbindung zu SmartThings herstellen:

1. **Matter-Geräte**: Direkt über SmartThings Hub oder Matter-fähige Samsung-Geräte hinzufügen (z. B. Family Hub-Kühlschrank, SmartThings Station)
2. **Zigbee-Gerät**: SmartThings Hub v2/v3 verfügt über einen integrierten Zigbee-Chip und kann direkt mit dem Heiman Zigbee-Sensor verbunden werden
3. **Z-Wave-Geräte**: SmartThings Hub unterstützt Z-Wave und Heiman Z-Wave-Geräte können direkt gekoppelt werden

## Authentifizierungsstatus

Der Hyman Matter-Infrarotsensor für den menschlichen Körper wurde auf Amazon und anderen Kanälen als SmartThings-unterstützend gekennzeichnet und hat von Benutzern gute Bewertungen erhalten.
