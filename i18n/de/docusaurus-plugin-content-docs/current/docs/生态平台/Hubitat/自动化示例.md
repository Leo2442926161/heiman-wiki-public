# Automatisierungsbeispiel

## Rule Machine Rauchmelder

**Name der Regel:** Sicherheitsverknüpfung für Rauchmelder

**Konfiguration:**
„
Auslöser: Hyman-Rauchmelder wird aktiv
Bedingungen: Modus != Unscharf
Aktion:
  - Schalten Sie alle Lichter ein
  - Push-Benachrichtigung „⚠️ Rauchmelder!“ senden
  - Schalten Sie die Sirene ein (falls verfügbar)
  - Modus auf „Notfall“ stellen
„

##Überwachung der Maschinentemperatur

**Regelname:** Hochtemperaturschutz

**Konfiguration:**
„
Auslöser: Heimann-Temperatursensor > 38°C
Bedingung: Klimaanlagenstatus = Aus
Aktion:
  - Klimaanlage einschalten, Kühlmodus, 24°C
  - Benachrichtigung senden „Die Raumtemperatur ist zu hoch und die Klimaanlage wurde gestartet“
„

## Rule Machine Wasserleckerkennung

**Name der Regel:** Notfallbehandlung von Wasserlecks

**Konfiguration:**
„
Auslöser: Hyman-Wassertauchmelder wird nass
Bedingung: Status des Wasserventils = offen
Aktion:
  - Hauptwasserventil schließen
  - Senden Sie Push-Benachrichtigungen an alle Mobiltelefone
  - SMS-Benachrichtigung senden (optional)
„
