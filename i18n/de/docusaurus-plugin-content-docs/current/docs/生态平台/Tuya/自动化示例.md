# Automatisierungsbeispiel

## Rauchmelder-Szene

**Szenenname:** „Rauchalarmverknüpfung“

**Trigger Condition:** Hyman Smoke Alarm → Smoke Detected

**Aktion ausführen:**
- Smart-Steckdose → öffnen (Abluftventilator anschließen)
- Lichter → schalten sich ein und blinken
- Push-Benachrichtigungen an das Mobiltelefon
- Lautsprecher → Alarmton abspielen

## One-Click-Szene beim Verlassen des Hauses (Tap-to-Run)

**Szenenname:** „Arrangement nach dem Verlassen des Hauses“

**Manuelle Ausführung:** Tippen oder Sprachauslösung

**Aktion ausführen:**
- Haiman-Türsensor → Alarm einschalten
- Hyman Human Body Infrared → Turn on detection
- Hyman-Wassertauchdetektor → Erkennung einschalten
- Alle Lichter → aus
- Smart Lock → Sperren

## Automatische Temperatur- und Luftfeuchtigkeitsregelung

**Automatisierungstyp:** Automatischer Bedingungsauslöser

**Trigger condition:** Hyman temperature and humidity sensor Humidity > 75%

**Aktion ausführen:**
- Intelligente Steckdose (Luftentfeuchter) → Ein
- Push notification: "Humidity is too high, dehumidifier has been turned on"

**Trigger condition:** Hyman temperature and humidity sensor Humidity < 50%

**Aktion ausführen:**
- Intelligente Steckdose (Luftentfeuchter) → Schließen
