#AmazonAlexa

Amazon Alexa ist eine der beliebtesten Plattformen für intelligente Sprachassistenten weltweit und unterstützt das Matter-Protokoll und ein umfangreiches Skills-Ökosystem.

## Integrationsmethode

### Matter-Ausrüstung

Auf Hyman Matter over Thread-Geräte kann über Matter-fähige Echo-Geräte (Echo 4. Generation, Echo Plus, Echo Show 10 usw.) zugegriffen werden:

1. Stellen Sie sicher, dass Ihr Echo-Gerät auf die neueste Firmware aktualisiert ist
2. Öffnen Sie die Alexa-App → Geräte → Gerät hinzufügen
3. Wählen Sie „Matter“ und scannen Sie den Pairing-Code

### Zigbee-Geräte

Einige Echo Plus und Echo Show verfügen über integrierte Zigbee-Chips und können direkt mit Heiman Zigbee-Sensoren gekoppelt werden.

## Beispiel für einen Sprachbefehl

- „Alexa, frage den Rauchmelder, ob alles in Ordnung ist“
- „Alexa, wie ist die Temperatur im Wohnzimmer?“

> **Referenz**: Hyman Matter-Sensoren sind im offiziellen Amazon-Shop erhältlich
