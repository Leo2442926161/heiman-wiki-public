# Automatisierungsbeispiel

## Alexa Guard-Integration (USA)

**Voraussetzung:** Alexa Guard Plus abonnieren (in den USA verfügbar)

**Konfigurationsprozess:**
1. Aktivieren Sie Guard Plus in der Alexa-App
2. Fügen Sie den Hyman Rauch-/CO-Melder als „Smart Home-Gerät“ hinzu.
3. Aktivieren Sie die intelligente Reaktion „Rauchalarmerkennung“.

**Automatische Antwort:**
- Rauchmelder ausgelöst → Alexa wählt automatisch Notfallkontakte
- CO-Alarmauslöser → Alexa-Sprachwarnung für das ganze Haus, automatische Belüftung
- Türen und Fenster öffnen sich ungewöhnlich → Alarmbenachrichtigung drücken

## Raucherkennungsroutine

**Auslöser:** Hyman-Rauchmelder → Rauch erkannt

**Aktion ausführen:**
- Alexa-Sprachansage: „⚠️ Rauch erkannt, bitte auf Sicherheit achten“
- Alle Echo-Geräte → Alarmton abspielen
- Intelligentes Lichtset → alles leuchtet
- Push-Benachrichtigungen an das Mobiltelefon

## „Leaving Home“-Routine

**Auslöser:** Sprachbefehl „Alexa, ich gehe“

**Aktion ausführen:**
- Magnetsensor der Heiman-Tür → Scharfschaltung einschalten
- Heiman-Körpersensor → Bewegungserkennung einschalten
- Hyman-Wassertauchdetektor → Erkennung einschalten
- Alle Lichter → aus
- Alexa antwortete: „Das Sicherheitssystem ist eingeschaltet, gute Fahrt.“
