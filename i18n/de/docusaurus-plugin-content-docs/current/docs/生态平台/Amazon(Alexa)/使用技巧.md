# Anwendungstipps

## Geräteerkennung

- Sobald das Heiman Matter-Gerät eingeschaltet ist, wird es normalerweise innerhalb von 30 Sekunden automatisch von der Alexa-App erkannt
- Wenn es nicht automatisch erkannt wird: Geben Sie Alexa App → Gerät → „+“ → „Gerät hinzufügen“ → „Matter“ ein.
- Verwenden Sie den Matter-Setup-Code oder QR-Code auf der Unterseite des Geräts, um die Kopplung abzuschließen

## Zigbee-Gerätezugriff (Echo Plus / Echo Show)

- Echo-Geräte mit Zigbee Hub können direkt mit Heiman Zigbee-Sensoren gekoppelt werden
- Kopplungsmodus: Alexa App → Gerät → „+“ → „Zigbee“ → „Gerät suchen“
- Heimann-Geräte sind Alexa-kompatibel mit integriertem Zigbee-Koordinator

## Routine-Tipps

- Verwenden Sie „benutzerdefinierte Aktionen“, um detailliertere Sprachansagen zu übertragen
- Fügen Sie „5 Sekunden warten“ zur Notfallroutine hinzu, bevor Sie weitere Aktionen ausführen, um Sprachüberschneidungen zu vermeiden
- Nutzen Sie die Funktion „Geräuscherkennung“, um Alexa zusätzliche Reaktionen auf die hohen Alarmtöne des Heyman-Alarms zu geben

## Alexa Guard HINWEIS

Alexa Guard Plus ist derzeit nur in den USA verfügbar. Benutzer in China können einige Funktionen über Routine simulieren.
