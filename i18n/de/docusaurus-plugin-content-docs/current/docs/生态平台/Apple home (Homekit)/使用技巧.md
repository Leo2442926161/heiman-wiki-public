# Anwendungstipps

## Homebridge gegen Materie

Heiman-Geräte können auf zwei Arten eine Verbindung zu Apple Home herstellen:

| Methode | Vorteile | Nachteile |
|------|------|------|
| **Materie über Faden** | Native Unterstützung, keine Bridge erforderlich, geringe Latenz | Erfordert Thread-Border-Router |
| **Homebridge-Plug-in** | Kompatibel mit allen Zigbee-Geräten | Erfordert zusätzliche Ausführung des Homebridge-Dienstes |

## Thread-Netzwerkaufbau

1. Mindestens ein Thread-Border-Router ist erforderlich (Apple HomePod mini, Apple TV 4K 3. Generation)
2. Hyman Matter-Geräte werden innerhalb von 10 Metern automatisch dem Thread-Netzwerk beitreten
3. Verwenden Sie **Nam* oder **Eve App**, um die Thread-Netzwerktopologie anzuzeigen

## Automatisierte Optimierung

- Die Automatisierung der Kategorie „Sicherheit“ von Apple Home kann direkt mit dem Sicherheitsstatus verknüpft werden
- Verwenden Sie den Befehl „Transformieren“, damit sich die Lichter sanft ändern, anstatt sofort umzuschalten
- Erstellen Sie eine plattformübergreifende Verknüpfung über die Shorcuts-App (z. B. Rauchmelder → SMS senden).

## FAQ

- Wenn das Matter-Gerät „Reagiert nicht“ anzeigt, starten Sie Apple TV/HomePod zur Wiederherstellung neu.
