# Automatisierungsbeispiel

## Sicherheitsalarm zur Raucherkennung

**Automatische HomeKit-Konfiguration:**

1. **Auslösebedingung:** Hyman-Rauchmelder → Rauch erkannt
2. **Sicherheitseinstellungen:** Schalten Sie den Status des Sicherheitssystems auf „Alarm“ um.
3. **Aktion ausführen:**
   - HomePod → Benutzerdefiniertes Audio abspielen „Rauch erkannt, bitte auf Sicherheit achten“
   - Alle intelligenten Lichter → leuchten und blinken
   - Push-Benachrichtigungen an alle Familienmitglieder

## Szene „Zu Hause/außerhalb der Heimat“.

**Kurzbefehle: „Sicherheitskontrolle zu Hause verlassen“**

„
wenn die letzte Person das Haus verlässt
  Sicherheitssystem einrichten → beim Verlassen des Hauses aktivieren
  Heiman-Türsensor einrichten → Benachrichtigungen aktivieren
  Heiman-Körpersensor einrichten → Alarm einschalten
  Schalten Sie alle Lichter aus
  Einstellungen Thermostat → Energiesparmodus
  Benachrichtigung senden „🏠 Abwesenheitsmodus aktiviert“
„

## Erinnerung an Temperaturanomalien

**Auslösebedingung:** Hyman Temperatur- und Feuchtigkeitssensor → Temperatur > 35°C

**Aktion ausführen:**
- HomePod → Die Raumtemperatur der Sprachübertragung überschreitet den Grenzwert
- Push-Benachrichtigungen an das iPhone
- Klimaanlage automatisch einschalten (wenn HomeBridge konfiguriert ist)
