#GoogleHome

Google Home ist die Smart-Home-Plattform von Google, die das Matter-Protokoll und eine Vielzahl von Geräte-Ökosystemen unterstützt.

## Integrationsmethode

### Matter-Ausrüstung

Hyman Matter over Thread-Geräte können direkt zur Google Home App hinzugefügt werden:

1. Stellen Sie sicher, dass Sie über einen Thread-Border-Router verfügen (Google Nest Hub Max, Nest Hub 2. Generation usw.).
2. Öffnen Sie die Google Home-App
3. Klicken Sie auf „Hinzufügen“ → „Matter-kompatible Geräte“
4. Scannen Sie den Matter-Kopplungscode auf Ihrem Gerät

### Sprachsteuerung

Sobald die Verbindung hergestellt ist, können Sie es über die Stimme von Google Assistant steuern:

- „Hey Google, wie ist der Status des Rauchmelders?“
- „Hey Google, wie hoch ist die Temperatur im Wohnzimmer?“

> **Referenz**: Hyman Matter-Geräte sind auf Amazon als Geräte gekennzeichnet, die Google Home unterstützen
