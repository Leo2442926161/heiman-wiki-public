# Automatisierungsbeispiel

## „Ich bin raus“-Routine

**Auslöser:** Sagen Sie „Ich bin draußen“ zu Google Home

**Aktion ausführen:**
- Heiman-Türmagnetsensor → Arm
- Heiman-Infrarotsensor für den menschlichen Körper → Aktivieren Sie die Erkennung des Verlassens des Hauses
- Schalten Sie alle Lichter und Geräte aus
- Thermostat → Auf Energiespartemperatur einstellen
- Google Nest Hub → Bestätigungskarte beim Verlassen des Hauses anzeigen
- Sprachantwort: „Der Verlassen-Zuhause-Modus wurde aktiviert, gute Reise.“

## Sendung „Raucherkennung“.

**Auslöser:** Hyman-Rauchmelder → Rauch erkannt

**Aktion ausführen:**
- Google Nest Speaker → Whole House Broadcast „⚠️ Rauch erkannt! Bitte sofort überprüfen!“
- Alle Smart Lights → Ein (über Google Home Smart Light Set)
- Push-Benachrichtigungen an die Mobiltelefone von Familienmitgliedern
- Kamera → Aufnahme starten

## Schlafenszeitmodus

**Auslöser:** Sagen Sie „Gute Nacht“ zu Google Home

**Aktion ausführen:**
- Heiman-Türsensor → Tür-Offen-Alarm einschalten
- Infrarot des menschlichen Körpers von Heiman → Nachterkennung einschalten
- Lichter → schalten sich allmählich aus
- Sprachantwort: „Gute Nacht, das Sicherheitssystem ist eingeschaltet“
