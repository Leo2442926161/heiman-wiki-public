# Automatisierungsbeispiel

## Status des MQTT-Abonnementgeräts

„Bash
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
„

Ausgabebeispiel:
„
zigbee2mqtt/heiman_smoke {"smoke":true,"battery":100,"linkquality":120}
„

## Node-RED Rauchmelder-Stream

„json
[{"id":1,"type":mqtt in,topic:zigbee2mqtt/heiman_smoke,name:Rauchmelder,qos:1,broker:localhost,x:200,y:200,wires:[[2"]]},{"id":2,type:switch,name:Rauch erkennen,property:payload.smoke ","conditions":[{"t":eq",v:true"}],"x":350,"y":200,"wires":[["3"]]},{"id":3,"type": Benachrichtigung, "name": "Push-Benachrichtigung", "title": "Rauchalarm", "message": "Rauch erkannt!", "x":500,"y":200,"wires":[]}]
„

## Online-Überwachung von Geräten

„yaml
#configuration.yaml
binärer_sensor:
  - Plattform: mqtt
    Name: „Rauchmelder online“
    state_topic: „zigbee2mqtt/heiman_smoke/availability“
    payload_on: „online“
    payload_off: „offline“
    Geräteklasse: Konnektivität
„
