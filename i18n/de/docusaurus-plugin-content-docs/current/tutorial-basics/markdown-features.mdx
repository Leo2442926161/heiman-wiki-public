# Markdown-Funktionen

Die Heiman Technische Dokumentation unterstützt umfassend die Markdown-Syntax und erweiterte Funktionen für die Erstellung strukturierter Produktdokumentationen.

## Codebeispiel

API-Aufrufbeispiel für die Produktentwicklungsanbindung:

```jsx title="Rauchmelder-API anbinden"
const heimanClient = new HeimanClient({
  apiKey: 'your-api-key',
  deviceId: 'HS1SA-xxxxx',
});

// Gerätestatus abrufen
const status = await heimanClient.getDeviceStatus();
console.log(status);
```

## Hinweisboxen

Verwenden Sie Hinweisboxen, um wichtige Informationen hervorzuheben:

:::tip

Der Heiman Rauchmelder HS1SA ist nach EN14604 zertifiziert

:::

:::caution

Stellen Sie bei der Installation von Gasmeldern sicher, dass die Verkabelung gemäß der Installationsanleitung korrekt erfolgt

:::

:::danger

Bei Gasleuch keinesfalls elektrische Schalter betätigen, sofort Fenster öffnen, lüften und den Bereich verlassen

:::
