# Lösungsbereitstellung

Heiman Technology bietet Ihnen eine umfassende Anleitung zur Bereitstellung intelligenter Sicherheitslösungen.

## Lösungsintegrationsprozess

Heiman-Produkte unterstützen verschiedene Bereitstellungsmethoden, einschließlich lokaler unabhängiger Bereitstellung und Cloud-Plattformanbindung.

### Lokale Bereitstellung

```bash
# Heiman Smart Fire Gateway konfigurieren
heiman-cli configure --gateway HS1GW --mode local

# Meldergerät registrieren
heiman-cli register --device HS1SA --gateway HS1GW
```

### Cloud-Plattformanbindung

Heiman-Produkte sind bereits mit den wichtigsten IoT-Plattformen verbunden:

- China Telecom Tianyi
- China Mobile OneNET
- Huawei HiLink
- Alibaba Cloud IoT
- Tuya Smart

## Matter-Protokollunterstützung

Heiman Matter-Serie Produkte sind kompatibel mit:

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

Ermöglicht geräte- und plattformübergreifende Interoperabilität.

## Bereitstellungsvalidierung

Nach Abschluss der Bereitstellung können Sie die Funktionen über die Heiman Smart Fire APP oder das Verwaltungsportal validieren und in Echtzeit überwachen.
