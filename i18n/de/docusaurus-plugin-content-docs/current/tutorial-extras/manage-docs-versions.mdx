# Dokumentenversionsverwaltung

Das Heiman Technology Wiki unterstützt die Verwaltung mehrerer Dokumentversionen, um sicherzustellen, dass Sie auf die technischen Dokumentationen zugreifen können, die Ihrer Produkt-Firmware-Version entsprechen.

## Versionsvergleich

| Dokumentversion | Zutreffende Produkt-Firmware | Veröffentlichungsdatum |
|---------|------------|---------|
| 2.0    | Matter Serie  | 2025-06 |
| 1.0    | Zigbee Serie  | 2024-01 |

## Verschiedene Versionen anzeigen

Im Version-Dropdown-Menü in der oberen Navigationsleiste können Sie zwischen verschiedenen Dokumentversionen wechseln.

## Versionsaktualisierungshinweise

Wir aktualisieren die Dokumentation kontinuierlich, um mit der Produktentwicklung Schritt zu halten. Es wird empfohlen, regelmäßig die neueste Version zu überprüfen.
