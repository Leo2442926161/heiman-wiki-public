# Gestione Versioni dei Documenti

Il Wiki di Heiman Technology supporta la gestione della documentazione multi-versione, garantendo l'accesso alla documentazione tecnica corrispondente alla versione del firmware del prodotto.

## Confronto Versioni

| Versione Doc | Firmware Compatibile | Data Rilascio |
|-------------|---------------------|--------------|
| 2.0         | Serie Matter         | 2025-06      |

## Visualizzare Versioni Diverse

Utilizza il menu a discesa delle versioni nella barra di navigazione superiore per passare da una versione all'altra dei documenti.

## Aggiornamenti delle Versioni

Aggiorniamo continuamente la documentazione per seguire l'evoluzione dei prodotti. Ti invitiamo a controllare regolarmente per la versione più recente.
