# Sub-1G (comunicazione a lunga distanza)

Sub-1G è una tecnologia di comunicazione wireless che opera nella banda di frequenza inferiore a 1 GHz. Ha le caratteristiche di una distanza di trasmissione ultra lunga e di un consumo energetico estremamente basso. È molto adatto per scenari IoT che richiedono una copertura su larga scala.

## Caratteristiche

| Caratteristiche | Descrizione |
|------|------|
| Banda di frequenza operativa | 433 MHz / 868 MHz (UE) / 915 MHz (USA) |
| Distanza di trasmissione | Fino a 1-3 chilometri in ambiente aperto |
| Capacità di penetrazione | Migliore di 2,4 GHz, forte penetrazione negli ostacoli sul muro |
| Consumo energetico | Estremamente bassa, la durata della batteria può raggiungere i 5-10 anni |
| Velocità di trasmissione | Basso (adatto a scenari con volumi di dati ridotti) |

## Scenari applicativi

- **Allarme fumo/CO indipendente**: interconnesso tra i piani dell'edificio
- **Impianti antincendio di grandi edifici**: centri commerciali, alberghi, scuole
- **Comunità Smart Senior Living**: chiamata di emergenza tra unità
- **Sub-1G + Gateway Wi-Fi**: il sotto-dispositivo comunica tramite Sub-1G e il gateway si connette al cloud
