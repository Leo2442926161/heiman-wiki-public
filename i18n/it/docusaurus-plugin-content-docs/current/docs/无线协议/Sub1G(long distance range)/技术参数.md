# Parametri tecnici

Di seguito sono riportati i parametri tecnici generali dei prodotti della serie a lunga distanza Hyman Sub-1G.

## Parametri wireless

| Parametro | Valore |
|------|-----|
| Banda di frequenza operativa | 433 MHz / 868 MHz / 915 MHz (a seconda dell'area di mercato) |
| Metodo di modulazione | FSK/OOK |
| Velocità di trasmissione | Circa 10~100 kbps (segmento video e configurazione) |
| Topologia di rete | Stella / punto a punto / rete Internet |
| Distanza di comunicazione | Circa 1~3 chilometri in ambiente aperto (segmento video e alimentazione) |
| Capacità di rete | Un singolo gruppo supporta fino a 50+ nodi |

## Assegnazione delle bande di frequenza regionali

| Regione | Bande di frequenza comuni |
|------|----------|
| Cina | 433 MHz |
| Europa | 868 MHz|
| Nord America | 915 MHz |

## Consumo elettrico ed energetico

| Parametro | Valore |
|------|-----|
| Metodo di alimentazione | Alimentato a batteria (batteria al litio CR123A / AA) |
| Durata della batteria | Circa 3~5 anni (a seconda della frequenza di segnalazione) |
| Trasmettere potenza | ≤ 10 mW (+10 dBm), conforme alla normativa SRD |
| Consumo energetico in standby | Livello microampere, estremamente basso |
| Sensibilità di ricezione | Circa -110 dBm ~ -120 dBm |

## Parametri ambientali

| Parametro | Valore |
|------|-----|
| Temperatura operativa | -10°C ~ +55°C|
| Umidità operativa | ≤ 95% UR (senza condensa) |
