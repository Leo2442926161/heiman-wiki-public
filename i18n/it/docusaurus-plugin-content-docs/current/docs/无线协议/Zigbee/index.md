#Zigbee

Zigbee è un protocollo di rete mesh wireless a basso consumo e bassa velocità di trasmissione dati basato sullo standard IEEE 802.15.4. È uno dei protocolli più maturi e apprezzati nel campo della casa intelligente.

##Hyman e Zigbee

Hyman è uno dei primi produttori cinesi ad adottare il protocollo Zigbee e dispone di un'enorme linea di prodotti Zigbee. I dispositivi Hyman Zigbee sono compatibili con lo standard **Zigbee 3.0** e possono interagire con i gateway e i coordinatori Zigbee tradizionali.

## Specifiche tecniche

| Caratteristiche | Descrizione |
|------|------|
| Banda di frequenza | 2,4 GHz (mondiale) |
| Velocità di trasmissione | Fino a 250 kbps |
| Topologia di rete | Rete mesh autorigenerante |
| Numero di nodi | La rete singola supporta fino a 65535 nodi |
| Distanza di comunicazione | Circa 10-30 metri in ambienti chiusi (ostacoli visibili) |
| Consumo energetico | Estremamente bassa, durata della batteria del sensore fino a 3 anni |

## Piattaforme compatibili

I dispositivi Hyman Zigbee possono essere collegati alle seguenti piattaforme:

- **Zigbee2MQTT**: bridge da Zigbee a MQTT open source
- **ZHA**: Assistente domestico con integrazione Zigbee integrata
- **Habitat**: hub intelligente localizzato
- **Homey**: hub intelligente multiprotocollo
- **SmartThings Hub**: hub Samsung per la casa intelligente
- **Tuya Zigbee Gateway** — Ecosistema Tuya
- **Hyman HS6GW Gateway** — Gateway ufficiale
