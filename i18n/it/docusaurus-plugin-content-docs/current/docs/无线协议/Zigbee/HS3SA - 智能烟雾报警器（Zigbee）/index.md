#HS3SA - Rilevatore di fumo intelligente (Zigbee)

## Panoramica

HS3SA è il rilevatore di fumo intelligente Zigbee di terza generazione di Hyman. Ha migliorato le prestazioni di rilevamento e le capacità di collegamento basate sulla prima generazione ed è adatto per il mercato dell'UE.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Fotoelettrico a infrarossi |
| Pressione sonora dell'allarme | ≥85 dB (3 metri) |
| Metodo di alimentazione | CC 3 V (CR123A) |
| Durata della batteria | Circa 3 anni |
| Temperatura operativa | -10°C ~ +50°C|
| Umidità di lavoro | ≤95%UR |
| Metodo di installazione | Installazione a soffitto |

## Caratteristiche

- Rilevazione fumi fotoelettrica
- Allarme sonoro e luminoso alto 85 dB
- Promemoria di batteria scarica
- Funzione di autotest
- Allarme antimanomissione
-Supporta l'audio remoto
- Aggiornamento del firmware OTA
- Prestazioni wireless migliorate
