# Serie C1 - Allarme monossido di carbonio (Zigbee)

## Panoramica

La serie C1 è un allarme di monossido di carbonio Zigbee di base lanciato da Heiman. Utilizza sensori elettrochimici ed è adatto per scenari di rilevamento di perdite di monossido di carbonio. Ha superato la certificazione EN 50291.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Elettrochimico |
| Campo di rilevamento | 0 ~ 1000 ppm |
| Pressione sonora dell'allarme | ≥85 dB (3 metri) |
| Metodo di alimentazione | Alimentato a batteria |
| Durata del sensore | Circa 7 anni |
| Temperatura operativa | -10°C ~ +50°C|

## Caratteristiche

- Rilevazione elettrochimica di CO
- Allarme classificazione concentrazione
- Allarme acustico e visivo da 85 dB
- Promemoria di batteria scarica
- Funzione di autotest

## Standard di certificazione

-EN 50291
