# Standard di certificazione

La Serie C1 ha superato le seguenti certificazioni:

- **EN 50291** — Standard europeo per gli allarmi di monossido di carbonio, vedere [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **CE/RED/EMC**: Direttiva dell'Unione Europea sulle apparecchiature wireless e compatibilità elettromagnetica
