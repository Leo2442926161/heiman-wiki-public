#HS8SA - Rilevatore di fumo intelligente (Zigbee)

## Panoramica

HS8SA è il rilevatore di fumo intelligente Zigbee + Wi-Fi di Hyman per il mercato statunitense. Ha una funzione di collegamento ad alta temperatura e ha superato la certificazione UL 217.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | Zigbee 3.0 / Wi-Fi |
| Tipo di sensore | Fotoelettrico a infrarossi |
| Pressione sonora dell'allarme | ≥85 dB (3 metri) |
| Metodo di alimentazione | CC 3 V (CR123A) |
| Temperatura operativa | -10°C ~ +50°C|
| Norme di certificazione | UL217 |
| Mercati applicabili | Stati Uniti |

## Caratteristiche

- Rilevazione fumi fotoelettrica
- Allarme collegamento alta temperatura
- Allarme sonoro e luminoso alto 85 dB
- Promemoria di batteria scarica
- Funzione di autotest
- Allarme antimanomissione
-Supporta l'audio remoto
- Comunicazione doppio protocollo (Zigbee + Wi-Fi)

## Standard di certificazione

- UL 217 (standard americano sui rilevatori di fumo)
