# Standard di certificazione

HS8SA ha superato le seguenti certificazioni:

- **EN 14604** — Norma europea per i rilevatori di fumo (tipo UE), vedere [Norma europea EN](../../../../ApprovalStandards/EN European Standard.md)
- **UL 217**: standard di sicurezza per i rilevatori di fumo statunitense (modello USA), vedere [UL Underwriters Laboratories](../../../../certification-standards/ULUnderwriters Laboratories.md)
- **CE/RED/EMC**: Direttiva dell'Unione Europea sulle apparecchiature wireless e compatibilità elettromagnetica
- **Zigbee 3.0 / Q-Label**: certificazione Zigbee Alliance, vedere [Altre certificazioni](../../../Certification Standards/Other Certifications.md)
