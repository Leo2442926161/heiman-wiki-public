# Standard di certificazione

HS1SA-E ha superato le seguenti certificazioni:

- **EN 14604** — Norma europea per i rilevatori di fumo, vedere [Norma europea EN](../../../../ApprovalStandards/EN European Standard.md)
- **CE/RED/EMC**: Direttiva dell'Unione Europea sulle apparecchiature wireless e compatibilità elettromagnetica
- **Zigbee 3.0 / Q-Label**: certificazione Zigbee Alliance, vedere [Altre certificazioni](../../../Certification Standards/Other Certifications.md)
- **Red Dot Design Award** — Premio Red Dot Product Design
