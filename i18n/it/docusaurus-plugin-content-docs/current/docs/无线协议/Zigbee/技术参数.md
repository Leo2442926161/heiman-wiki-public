# Parametri tecnici

Di seguito sono riportati i parametri tecnici generali per i dispositivi Heiman Zigbee.

## Parametri wireless

| Parametro | Valore |
|------|-----|
| Norme del protocollo | Zigbee 3.0 (IEEE 802.15.4) |
| Banda di frequenza operativa | 2,4 GHz (banda di frequenza ISM comune globale) |
| Velocità di trasmissione | Fino a 250 kbps |
| Topologia di rete | Rete mesh autorigenerante |
| Capacità del nodo | Massimo 65535 nodi in un'unica rete |
| Distanza di comunicazione | Circa 10~30 metri in interni (vista ostacoli sul muro) |
| Metodo di crittografia | AES-128-CCM |

## Compatibilità

| Progetto | Descrizione |
|------|------|
| Coordinatore Zigbee | Compatibile con il coordinatore standard Zigbee 3.0 |
| Piattaforme supportate | Zigbee2MQTT, ZHA (assistente domestico), Hubitat, Homey, SmartThings, gateway Tuya Zigbee, gateway Haiman HS6GW |
| Aggiornamento OTA | Supporta l'aggiornamento del firmware via etere (alcuni modelli) |

## Consumo elettrico ed energetico

| Parametro | Valore |
|------|-----|
| Metodo di alimentazione | Alimentazione a batteria (CR123A / CR2450 / AA) o USB/rete (tipo presa) |
| Durata della batteria | Circa 2~3 anni per i sensori (a seconda della frequenza di segnalazione) |
| Consumo energetico in standby | Livello microamp, supporta il meccanismo di polling del sonno |

## Parametri ambientali

| Parametro | Valore |
|------|-----|
| Temperatura operativa | -10°C ~ +55°C|
| Umidità operativa | ≤ 95% UR (senza condensa) |
| Livello di protezione IP | Dipende dal modello specifico (tipo di sensore solitamente IP30) |
