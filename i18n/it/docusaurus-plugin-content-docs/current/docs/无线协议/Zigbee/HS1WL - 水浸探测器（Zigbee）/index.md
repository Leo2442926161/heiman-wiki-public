# HS1WL - Rilevatore di perdite d'acqua (Zigbee)

## Panoramica

HS1WL è il rilevatore di immersione in acqua Zigbee di prima generazione lanciato da Hyman. Adotta il metodo di rilevamento della sonda e può rilevare efficacemente perdite d'acqua o accumulo di acqua. È un'apparecchiatura di base per il monitoraggio dell'immersione in acqua.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Metodo di rilevamento | Rilevamento sonda |
| Metodo di alimentazione | Batteria DC 3V |
| Temperatura operativa | -10°C ~ +50°C|
| Umidità di lavoro | ≤95%UR |

## Caratteristiche

- Rilevamento in tempo reale di perdite/accumuli d'acqua
- Allarme acustico
- Promemoria di batteria scarica
- Design anticorrosione della sonda
