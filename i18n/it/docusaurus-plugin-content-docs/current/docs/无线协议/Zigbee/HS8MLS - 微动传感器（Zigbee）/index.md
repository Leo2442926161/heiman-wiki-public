# HS8MLS - Sensore di micromovimento (Zigbee)

## Panoramica

HS8MLS è un sensore a infrarossi (PIR) per il corpo umano Zigbee a lungo raggio lanciato da Heiman. È appositamente progettato per scene che richiedono il rilevamento a lunga distanza ed è adatto per spazi aperti come corridoi e sale.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Infrarossi passivi (PIR) |
| Distanza di rilevamento | Rilevamento a lunga distanza |
| Metodo di alimentazione | Batteria DC 3V |

## Caratteristiche

- Rilevamento del movimento a lunga distanza
- Design a basso consumo energetico
- Supporta il collegamento delle scene
