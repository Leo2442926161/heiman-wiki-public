# Serie JT - Valvola di intercettazione di emergenza del gas (Zigbee)

## Panoramica

La valvola di intercettazione di emergenza del gas della serie JT è un dispositivo di esecuzione utilizzato insieme al rilevatore di gas combustibile Hyman Zigbee. Quando viene rilevata una perdita di gas, è possibile chiudere in remoto o automaticamente la valvola del gasdotto per evitare incidenti.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Digitare | Indipendente + collegamento Zigbee |
| Gas di prova | Gas naturale/Gas di petrolio liquefatto |
| Norme di certificazione | GB15322 |
| Temperatura operativa | -10°C ~ +50°C|

## Caratteristiche

- Collegato con HS1CG / HS3CG per chiudere automaticamente la valvola
-Supporta la chiusura manuale remota della valvola
- Pulsante allarme perdite
- Driver dell'elettrovalvola

## Standard di certificazione

-GB15322

## Accessori applicabili

- Rilevatore di gas combustibili HS1CG / HS3CG
