# HS1CA-E - Allarme intelligente per monossido di carbonio (Zigbee)

## Panoramica

HS1CA-E è il rilevatore intelligente di monossido di carbonio (CO) Zigbee di Hyman per il mercato dell'UE. Utilizza un sensore elettrochimico e può rilevare con precisione il gas monossido di carbonio con una concentrazione di 0~1000 ppm. Ha superato la certificazione EN 50291.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Elettrochimico |
| Campo di rilevamento | 0 ~ 1000 ppm |
| Pressione sonora dell'allarme | ≥85 dB (3 metri) |
| Metodo di alimentazione | CC 3 V (CR123A) |
| Durata del sensore | Circa 7 anni |
| Temperatura operativa | -10°C ~ +50°C|
| Umidità di lavoro | ≤95%UR |
| Metodo di installazione | Montaggio a parete |

## Caratteristiche

- Sensore elettrochimico di CO ad alta precisione
- Allarme graduato di concentrazione (avviso sonoro e luminoso)
- Suono di allarme da 85 dB
- Promemoria di batteria scarica
- Promemoria scadenza durata sensore
- Funzione di autotest
- Aggiornamento del firmware OTA

## Standard di certificazione

- EN 50291 (Norma europea per gli allarmi di monossido di carbonio)
-CE/ROSSO

## Accessori applicabili

- Batteria CR123A
