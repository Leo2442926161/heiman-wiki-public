#HS1MS - Sensore a infrarossi per il corpo umano (Zigbee)

## Panoramica

HS1MS è un sensore a infrarossi (PIR) per il corpo umano Zigbee standard lanciato da Hyman. Utilizza la tecnologia di rilevamento a infrarossi passivi e può rilevare la radiazione termica a infrarossi emessa da un corpo umano in movimento. È ampiamente utilizzato negli scenari di illuminazione e sicurezza intelligenti.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Infrarossi passivi (PIR) |
| Angolo di rilevamento | 110°|
| Distanza di rilevamento | ≤7 m (25°C) |
| Metodo di alimentazione | Batteria al litio DC 3V |
| Durata della batteria | Circa 3 anni |
| Temperatura operativa | -10°C ~ +50°C|
| Metodo di installazione | Montaggio a parete |

## Caratteristiche

- Rilevamento umano mobile
- Rilevamento grandangolare di 110°
- Promemoria di batteria scarica
- Allarme antimanomissione
- Supporta il collegamento delle scene
