#HS8MIS - Sensore mobile (Zigbee)

## Panoramica

HS8MIS è un sensore di movimento Zigbee multifunzionale lanciato da Heiman. Integra le funzioni di rilevamento umano PIR e di rilevamento dell'illuminazione ed è adatto per scene intelligenti con doppio monitoraggio di luce e movimento.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Infrarossi passivi (PIR) + Illuminazione |
| Angolo di rilevamento | 110°|
| Distanza di rilevamento | ≤7 m|
| Metodo di alimentazione | Batteria DC 3V |

## Caratteristiche

- Rilevamento umano mobile
- Rilevamento dell'illuminazione
- Design a basso consumo energetico
- Supporta il collegamento delle scene
