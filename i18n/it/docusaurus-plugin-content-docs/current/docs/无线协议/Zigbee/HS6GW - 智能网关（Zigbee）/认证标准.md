# Standard di certificazione

HS6GW (versione Zigbee) ha superato le seguenti certificazioni:

- **CE/RED**: Direttiva dell'Unione Europea sulle apparecchiature wireless
- **CCC** — Certificazione obbligatoria del prodotto Cina, vedere [Certificazione obbligatoria CCC Cina](../../../../Certification Standards/CCC China Compulsory Certification.md)
