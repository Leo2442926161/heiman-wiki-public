# HS6GW - Gateway intelligente (Zigbee)

## Panoramica

HS6GW è un gateway intelligente Zigbee lanciato da Heiman. In qualità di coordinatore della rete Zigbee di Heiman, si connette a Internet tramite Wi-Fi o uplink Ethernet e supporta l'accesso di un massimo di 128 sotto-dispositivi.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | Zigbee 3.0 (Coordinatore) |
| Connessione uplink | Wi-Fi/Ethernet |
| Nodi massimi | 128 sottodispositivi |
| Metodo di alimentazione | Alimentatore USB CC 5 V |

## Caratteristiche

- Coordinatore della rete Zigbee
- Connessione uplink Wi-Fi/Ethernet
- Rilevamento e associazione automatici del dispositivo
- Collegamento alla scena locale
- Accesso remoto (tramite Heiman Cloud Platform)
- Supporta fino a 128 sotto-dispositivi
- Aggiornamento del firmware OTA

## Scenari applicabili

- Il punto di connessione principale dell'intelligenza di tutta la casa Heiman
- Collabora con l'APP Haiman per realizzare il monitoraggio remoto
