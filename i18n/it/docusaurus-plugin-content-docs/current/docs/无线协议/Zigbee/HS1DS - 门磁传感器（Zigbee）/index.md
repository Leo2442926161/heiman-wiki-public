# HS1DS - Sensore magnetico per porta (Zigbee)

## Panoramica

HS1DS è un sensore magnetico per porte Zigbee lanciato da Heiman. Viene utilizzato per rilevare lo stato di apertura e chiusura di porte e finestre. È un componente base del sistema di sicurezza intelligente.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Interruttore a lamella |
| Metodo di alimentazione | Batteria a bottone CR1632 |
| Durata della batteria | Circa 2 ~ 3 anni |
| Spazio di installazione | ≤10mm|
| Temperatura operativa | -10°C ~ +50°C|
| Umidità di lavoro | ≤95%UR |

## Caratteristiche

- Rilevamento in tempo reale dello stato di apertura/chiusura di porte e finestre
- Allarme manomissione (Tamper)
- Promemoria di batteria scarica
-Supporta il collegamento (apertura della porta, accensione delle luci, inserimento quando si esce di casa, ecc.)
- Facile da installare (nastro adesivo 3M o fissaggio con viti)

## Accessori applicabili

- Batteria a bottone CR1632
- Nastro biadesivo/viti 3M
