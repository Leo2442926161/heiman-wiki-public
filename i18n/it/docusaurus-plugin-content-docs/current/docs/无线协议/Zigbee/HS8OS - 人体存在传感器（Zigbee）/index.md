#HS8OS - Sensore di presenza umana (Zigbee)

## Panoramica

HS8OS è un sensore di presenza umana Zigbee lanciato da Hyman. Utilizza la tecnologia radar a onde millimetriche ed è in grado di rilevare la presenza del corpo umano in uno stato statico (come sdraiato, seduto in silenzio), compensando le carenze dei tradizionali sensori PIR che possono rilevare solo il movimento.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Radar a onde millimetriche (mmWave) |
| Campo di rilevamento | Raggio 3 ~ 5 m |
| Capacità di rilevamento | Presenza di un corpo umano fermo (compreso il rilevamento della microrespirazione) |
| Metodo di alimentazione | Alimentazione CA |
| Metodo di installazione | Installazione a soffitto |

## Caratteristiche

- Rilevamento stazionario del corpo umano (micromovimenti respiratori)
- Raggio di copertura 3~5 metri
- Installazione a soffitto, angolo di rilevamento ottimale
- Può essere utilizzato complementare ai sensori PIR

## Scenari applicabili

- Illuminazione intelligente: sempre accesa quando c'è qualcuno, si spegne automaticamente quando non c'è nessuno
- Risparmio energetico dell'aria condizionata: spegne automaticamente l'aria condizionata quando le persone camminano
- Assistenza intelligente agli anziani: allarme tempestivo per gli anziani in caso di permanenza prolungata
