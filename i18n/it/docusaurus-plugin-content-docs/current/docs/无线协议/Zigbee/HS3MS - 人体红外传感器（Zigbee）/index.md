#HS3MS - Sensore a infrarossi per il corpo umano (Zigbee)

## Panoramica

HS3MS è il sensore a infrarossi per il corpo umano Zigbee di terza generazione di Hyman. Adotta un design con montaggio a soffitto a 360° e la sensibilità di rilevamento e la copertura sono notevolmente migliorate rispetto alla generazione precedente.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Infrarossi passivi (PIR) |
| Angolo di rilevamento | 360° (installazione a soffitto) |
| Distanza di rilevamento | ≤7 m (25°C) |
| Metodo di alimentazione | Batteria al litio DC 3V |
| Durata della batteria | Circa 3 anni |
| Temperatura operativa | -10°C ~ +50°C|
| Metodo di installazione | Installazione a soffitto |

## Caratteristiche

- Rilevamento umano mobile
- Copertura omnidirezionale a 360°
- Promemoria di batteria scarica
- Allarme antimanomissione
- Sensibilità di rilevamento migliorata
- Supporta il collegamento delle scene
