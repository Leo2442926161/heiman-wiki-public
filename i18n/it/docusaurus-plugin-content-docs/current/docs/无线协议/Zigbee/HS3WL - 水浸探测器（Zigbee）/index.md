# HS3WL - Rilevatore di perdite d'acqua (Zigbee)

## Panoramica

HS3WL è il rilevatore di immersione in acqua Zigbee di terza generazione di Hyman. Ha una sensibilità di rilevamento e prestazioni wireless migliorate rispetto alla generazione precedente ed è adatto per scenari di monitoraggio delle perdite d'acqua.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Metodo di rilevamento | Rilevamento sonda |
| Metodo di alimentazione | Batteria DC 3V |
| Temperatura operativa | -10°C ~ +50°C|
| Umidità di lavoro | ≤95%UR |

## Caratteristiche

- Rilevamento in tempo reale di perdite/accumuli d'acqua
- Sensibilità di rilevamento migliorata
- Allarme acustico
- Promemoria di batteria scarica
- Design anticorrosione della sonda
