#HS3CG - Rilevatore di gas combustibile (Zigbee)

## Panoramica

HS3CG è il rilevatore di gas combustibili Zigbee di terza generazione di Hyman. Presenta una precisione di rilevamento e una velocità di risposta migliorate rispetto alla prima generazione ed è adatto per il rilevamento di perdite di gas naturale e gas di petrolio liquefatto.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Semiconduttore |
| Gas di prova | Gas naturale (metano)/Gas di petrolio liquefatto (GPL) |
| Pressione sonora dell'allarme | ≥85 dB (3 metri) |
| Metodo di alimentazione | CA 220 V/CC 12 V |
| Norme di certificazione | GB15322 |
| Temperatura operativa | -10°C ~ +50°C|

## Caratteristiche

- Rilevamento fughe di gas naturale/gas liquefatto
- Allarme classificazione concentrazione
- Collegamento di spegnimento automatico (con elettrovalvola)
- Allarme acustico e visivo da 85 dB
- Funzione di autotest
- Promemoria scadenza durata sensore
- Prestazioni wireless migliorate

## Standard di certificazione
