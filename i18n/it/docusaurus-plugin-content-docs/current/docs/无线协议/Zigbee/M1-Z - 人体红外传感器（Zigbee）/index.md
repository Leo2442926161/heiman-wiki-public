# M1-Z - Sensore a infrarossi per il corpo umano (Zigbee)

## Panoramica

M1-Z è un sensore a infrarossi compatto per il corpo umano Zigbee lanciato da Heiman. Supporta un angolo di rilevamento di 110° o 360° (a seconda del metodo di installazione), dispone di un sensore di illuminazione integrato ed è adatto per scenari di illuminazione e sicurezza intelligenti.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | ZigBee 3.0 |
| Tipo di sensore | Infrarossi passivi (PIR) + Illuminazione |
| Angolo di rilevamento | 110° (a parete) / 360° (a soffitto) |
| Distanza di rilevamento | ≤7 m (25°C) |
| Intervallo di illuminamento | 0 ~ 1200 Lux |
| Metodo di alimentazione | Batteria al litio DC 3V |
| Durata della batteria | Circa 3 anni |
| Temperatura operativa | -10°C ~ +50°C|
| Metodo di installazione | Parete/soffitto |

## Caratteristiche

- Rilevamento umano mobile
- Rilevamento dell'illuminazione (0~1200 Lux)
- Commutabile 110°/360°
- Promemoria di batteria scarica
- Allarme antimanomissione
- Design compatto
