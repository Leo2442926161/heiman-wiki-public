# Sensore a infrarossi per il corpo umano M1-M (materia)

M1-M è un sensore a infrarossi per il corpo umano **Matter over Thread** lanciato da Hyman. Integra le funzioni di rilevamento PIR e rilevamento dell'illuminazione ed è adatto per scenari di automazione domestica intelligente.

## Panoramica del prodotto

- **Tipo di prodotto**: sensore a infrarossi per il corpo umano
- **Protocollo di comunicazione**: Matter over Thread
- **Metodo di rilevamento**: Infrarossi passivi (PIR) + rilevamento dell'illuminazione
- **Scenario tipico**: le luci si accendono quando arrivano le persone, collegamento di sicurezza, automazione ambientale

## Parametri tecnici

| Parametro | Valore |
|------|-----|
| Angolo di rilevamento | 110° (a parete) / 360° (a soffitto) |
| Distanza di rilevamento | ≤7 m (25°C) |
| Gamma di illuminazione | 0~1200 Lux |
| Protocollo di comunicazione | Questione sul thread (IEEE 802.15.4) |
| Banda di frequenza operativa | 2,4GHz |
| Metodo di alimentazione | Batteria al litio DC 3V |
| Durata della batteria | Circa 1 anno (modalità Matter) |
| Temperatura operativa | -10°C ~ +50°C|
| Umidità di lavoro | ≤95% UR (senza condensa) |

## Caratteristiche

- Rilevamento umano mobile
- Rilevamento dell'illuminazione ambientale
- Distanza di rilevamento ≤7m
- Supporta l'installazione a parete o soffitto
- Materia multi-ecologica interoperabilità multipiattaforma
