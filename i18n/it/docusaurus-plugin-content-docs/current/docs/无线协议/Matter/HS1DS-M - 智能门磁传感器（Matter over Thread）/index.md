# HS1DS-M - Sensore porta intelligente (Matter over Thread)

## Panoramica

HS1DS-M è la versione Matter over Thread del sensore porta intelligente Heiman. Supporta l'accesso nativo a tutte le piattaforme compatibili con Matter come Apple Home, Google Home, Amazon Alexa, Home Assistant, ecc. senza la necessità di un gateway proprietario.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Protocollo di comunicazione | Questione sul filo |
| Metodo di alimentazione | Batteria a bottone CR1632 |
| Durata della batteria | Circa 2-3 anni |
| Distanza di installazione | ≤10mm|
| Temperatura operativa | -10°C ~ +50°C|
| Umidità di lavoro | ≤95%UR |

## Caratteristiche

- Rilevamento in tempo reale dello stato di apertura/chiusura di porte e finestre
- Allarme manomissione (Tamper)
- Promemoria di batteria scarica
- Compatibile con più piattaforme: Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Rete Thread Mesh, controllo locale a bassa latenza
- Nessun gateway dedicato richiesto, è richiesto il router di confine Thread
