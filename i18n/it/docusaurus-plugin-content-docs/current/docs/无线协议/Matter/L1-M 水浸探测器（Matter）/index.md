# Rilevatore di immersione in acqua L1-M (materia)

L1-M è un rilevatore di immersione in acqua **Matter over Thread** lanciato da Hyman. Rileva perdite d'acqua o acqua accumulata attraverso una sonda, che può prevenire efficacemente perdite di proprietà causate da scoppi di tubi dell'acqua.

## Panoramica del prodotto

- **Tipo di prodotto**: sensore di rilevamento dell'immersione in acqua
- **Protocollo di comunicazione**: Matter over Thread
- **Metodo di rilevamento**: rilevamento del contatto della sonda
- **Scenari tipici**: rilevamento di perdite d'acqua in cucina, bagno, lavanderia, cantina

## Parametri tecnici

| Parametro | Valore |
|------|-----|
| Metodo di rilevamento | Tipo di contatto sonda in acciaio inossidabile |
| Protocollo di comunicazione | Questione sul thread (IEEE 802.15.4) |
| Banda di frequenza operativa | 2,4GHz |
| Metodo di alimentazione | Alimentato a batteria |
| Metodo di allarme | Cicalino + spinta dell'app |
| Temperatura operativa | -10°C ~ +50°C|
| Umidità di lavoro | ≤95% UR (senza condensa) |

## Caratteristiche

- Rilevamento in tempo reale di perdite d'acqua/acqua stagnante
- Allarme in loco con cicalino
- Push allarme remoto dell'app
- Promemoria di batteria scarica
- Design anticorrosione della sonda
- Materia multi-ecologica interoperabilità multipiattaforma
