# Standard di certificazione

HS1SA-M ha superato le seguenti certificazioni:

- **EN 14604** — Norma europea per i rilevatori di fumo, vedere [Norma europea EN](../../../../ApprovalStandards/EN European Standard.md)
- **Marchio CE** — Dichiarazione di conformità UE, vedere [Conformità CE UE](../../../../certification-standards/CE EU-Conformity.md)
- **Certificazione Matter**: ha superato la certificazione CSA Alliance Matter per garantire l'interconnessione multipiattaforma delle case intelligenti
- **Red Dot Design Award** — Premio Red Dot Product Design
