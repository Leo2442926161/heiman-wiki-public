# Parametri tecnici

Di seguito sono riportate le specifiche generali per le apparecchiature Hyman Matter.

## Parametri wireless

| Parametro | Valore |
|------|-----|
| Protocollo sottostante | Discussione (IEEE 802.15.4) |
| Banda di frequenza operativa | 2,4 GHz (banda ISM) |
| Standard del livello di applicazione | Materia 1.x |
| Topologia di rete | Rete mesh autorigenerante |
| Metodo di crittografia | AES-128-CCM |

## Requisiti hardware

| Progetto | Descrizione |
|------|------|
| Router di confine | Necessario, come Apple HomePod mini, Apple TV 4K, Google Nest Hub 2a generazione, ecc. |
| Supporto per terminali mobili | iOS/Android (distribuzione in rete tramite varie app ecologiche) |
| Interoperabilità interecologica | Apple HomeKit, Google Home, Amazon Alexa, Assistente domestico, SmartThings |

## Consumo elettrico ed energetico

| Parametro | Valore |
|------|-----|
| Metodo di alimentazione | Alimentato a batteria (CR123A / CR2450, ecc.) |
| Durata della batteria | Circa 1~3 anni (a seconda del tipo di dispositivo e della frequenza di utilizzo) |
| Consumo energetico in standby | Estremamente basso, supporta il meccanismo di risveglio dal sonno |

## Parametri ambientali

| Parametro | Valore |
|------|-----|
| Temperatura operativa | -10°C ~ +50°C|
| Umidità operativa | ≤ 95% UR (senza condensa) |
