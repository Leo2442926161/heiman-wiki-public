# Standard di certificazione

HS1CA-Z ha superato le seguenti certificazioni:

- **EN 50291** — Standard europeo per gli allarmi di monossido di carbonio, vedere [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **Certificazione Z-Wave S2**: supporta il framework di sicurezza S2, vedere [Altre certificazioni](../../../Certification Standards/Other Certifications.md)
- **CE** — Dichiarazione di conformità dell'Unione Europea
