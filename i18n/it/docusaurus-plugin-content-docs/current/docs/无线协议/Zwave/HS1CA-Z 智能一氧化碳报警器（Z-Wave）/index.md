# HS1CA-Z Allarme intelligente di monossido di carbonio (Z-Wave)

HS1CA-Z è un allarme intelligente per monossido di carbonio **Z-Wave** lanciato da Hyman. Utilizza la tecnologia dei sensori elettrochimici ed è compatibile con lo standard Z-Wave Plus.

## Panoramica del prodotto

- **Tipo di prodotto**: Allarme intelligente di monossido di carbonio
- **Protocollo di comunicazione**: Z-Wave Plus
- **Norma di certificazione**: EN 50291
- **Scenario tipico**: avviso di perdite di CO in case, appartamenti e altri luoghi

## Parametri tecnici

| Parametro | Valore |
|------|-----|
| Tipo di sensore | Sensore elettrochimico di CO |
| Protocollo di comunicazione | Z-Wave Plus (ITU-T G.9959) |
| Banda di frequenza operativa | 868,42 MHz (UE) / 908,42 MHz (USA) |
| Campo di rilevamento | 0~1000 ppm |
| Pressione sonora dell'allarme | ≥85 dB (3 metri) |
| Metodo di alimentazione | Alimentato a batteria |
| Durata del sensore | Tipico 7 anni |
| Metodo di crittografia | S2 autenticato/S2 non autenticato |
| Temperatura operativa | -10°C ~ +50°C|
| Umidità di lavoro | ≤90% UR (senza condensa) |

## Caratteristiche

- Sensore elettrochimico di CO ad alta precisione
- Allarme classificazione concentrazione
- Allarme sonoro e luminoso alto 85 dB
- Promemoria di batteria scarica
- Promemoria scadenza durata sensore
- Funzione di autotest
- Crittografia sicura Z-Wave S2
- Supporta l'aggiornamento del firmware OTA

##Certificazione

- Certificato EN 50291
- Certificato Z-Wave Plus
-Certificazione di sicurezza S2
