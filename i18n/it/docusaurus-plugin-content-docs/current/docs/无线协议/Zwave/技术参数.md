# Parametri tecnici

Di seguito sono riportate le specifiche tecniche generali dei dispositivi Hyman Z-Wave.

## Parametri wireless

| Parametro | Valore |
|------|-----|
| Norme del protocollo | Z-Wave Plus (ITU-T G.9959) |
| Banda di frequenza operativa | 800~900 MHz (a seconda della regione: normativa europea 868,42 MHz, normativa statunitense 908,42 MHz) |
| Velocità di trasmissione | Fino a 100 kbps |
| Topologia di rete | Rete mesh autorigenerante (routing sorgente) |
| Capacità del nodo | Massimo 232 nodi in un'unica rete |
| Distanza di comunicazione | Circa 30 metri all'interno (tra i nodi), copertura in mesh per tutta la casa |
| Metodo di crittografia | AES-128 (S2 autenticato/S2 non autenticato) |

## Banda di frequenza regionale

| Regione | Frequenza centrale |
|------|----------|
| Europa | 868,42 MHz|
| Nord America | 908,42 MHz |
| Altre zone | Soggetto alle normative locali |

## Consumo elettrico ed energetico

| Parametro | Valore |
|------|-----|
| Metodo di alimentazione | Alimentazione a batteria (CR123A/AA) o da rete |
| Durata della batteria | Circa 2~3 anni per i sensori |
| Trasmettere potenza | ≤ 1 mW (conforme alla normativa SRD) |
| Ricevi sensibilità | Circa -100 dBm |

## Parametri ambientali

| Parametro | Valore |
|------|-----|
| Temperatura operativa | -10°C ~ +50°C|
| Umidità operativa | ≤ 90% UR (senza condensa) |

## Caratteristiche della rete

| Caratteristiche | Descrizione |
|------|------|
| SmartStart | Supporto (serie 700), scansiona il codice QR per unirti alla rete |
| Sicurezza S2 | Supporta tre livelli di sicurezza: non crittografato, S2 non autenticato, S2 autenticato |
| Contiene CC | Security CC, Security 2 CC, Aggiornamento firmware CC, Basic CC, Batteria CC, Notifica CC e altro |
