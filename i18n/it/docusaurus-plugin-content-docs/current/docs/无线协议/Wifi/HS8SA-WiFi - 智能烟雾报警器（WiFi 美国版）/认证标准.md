# Standard di certificazione

HS8SA-WiFi ha superato le seguenti certificazioni:

- **UL 217**: standard americano di sicurezza per i rilevatori di fumo, vedere [UL Underwriters Laboratories](../../../../certificationstandards/ULUnderwriters Laboratories.md)
- **FCC**: certificazione della Federal Communications Commission
- **CE** — Dichiarazione di conformità dell'Unione Europea
