# Parametri tecnici

Di seguito sono riportate le specifiche tecniche generali dei dispositivi WiFi Heimann.

## Parametri wireless

| Parametro | Valore |
|------|-----|
| Norme del protocollo | IEEE 802.11 b/g/n |
| Banda di frequenza operativa | 2,4 GHz (non compatibile con 5 GHz) |
| Velocità di trasmissione | Fino a 150Mbps (802.11n) |
| Topologia di rete | Topologia a stella (direttamente collegata al router) |
| Distanza di comunicazione | Circa 30~50 metri in interni (a seconda delle prestazioni del router) |
| Metodo di crittografia | WPA2-PSK / WPA3 |

## Metodo di distribuzione in rete

| Metodo | Descrizione |
|------|------|
| SmartConfig | App mobile Configurazione di rete con un clic (apparecchiature WiFi tradizionali) |
| Provisioning assistito da Bluetooth | Configurazione delle credenziali WiFi tramite canali temporanei Bluetooth (alcuni nuovi modelli) |
| Modalità AP | Attiva l'hotspot sul dispositivo e configuralo dopo aver collegato il telefono |

## Consumo elettrico ed energetico

| Parametro | Valore |
|------|-----|
| Metodo di alimentazione | Alimentazione da rete (tipo presa/tipo gateway) o da batteria (tipo allarme) |
| Livello di consumo energetico | Superiore a Zigbee/Thread a causa della necessità di mantenere la connessione TCP/IP |
| Consumo energetico in standby | Tipo di presa circa 0,5 W ~ 1 W |

## Parametri ambientali

| Parametro | Valore |
|------|-----|
| Temperatura operativa | -10°C ~ +50°C|
| Umidità operativa | ≤ 90% UR (senza condensa) |
