#WiFi

Il Wi-Fi è il protocollo di rete wireless più diffuso e si basa sullo standard IEEE 802.11. Alcuni dispositivi intelligenti Heiman supportano la connessione Wi-Fi e possono essere collegati direttamente a un router senza la necessità di un gateway aggiuntivo.

## Scenari applicabili

I dispositivi Wi-Fi sono adatti per le seguenti situazioni:

- Infrastruttura di rete completa, non è necessario acquistare gateway aggiuntivi
- Dispositivi non sensibili al consumo energetico (come le prese intelligenti)
- Scenari che richiedono accesso remoto e servizi cloud
- Abitazioni già coperte da reti Wi-Fi Mesh mature

## Prodotti Wi-Fi Hyman

Heimann offre prodotti che supportano la connettività Wi-Fi:

- **Smart Socket HS2SK-WiFi** — Misurazione del consumo energetico, interruttore remoto
- **Smart Gateway**: si connette al router tramite Wi-Fi

## Note

- La batteria del sensore Wi-Fi si scarica più velocemente di Zigbee/Thread
- Un numero elevato di dispositivi Wi-Fi potrebbe influire sulla stabilità della rete
- I prodotti principali di Haiman sono Zigbee/Matter over Thread e la linea di prodotti Wi-Fi è relativamente snella
