#HS6GW - Gateway domestico intelligente

## Panoramica

HS6GW è un gateway domestico intelligente lanciato da Heiman, che supporta uplink WiFi + Ethernet, coordinatore Zigbee integrato e può gestire fino a 128 sotto-dispositivi Zigbee. Realizzare l'accesso remoto e i servizi cloud collegandosi a Internet è il fulcro del sistema di casa intelligente Haiman.

## Specifiche tecniche

| Parametro | Valore |
|------|-----|
| Comunicazione uplink | IEEE 802.11 b/g/n (WiFi 2,4 GHz) + Ethernet 10/100M |
| Comunicazione downlink | Zigbee 3.0 (modalità coordinatore) |
| Numero massimo di sottodispositivi | 128|
| Metodo di alimentazione | CC 5 V/1 A (Micro USB) |
| Consumo energetico in standby | ≤ 1W|
| Temperatura operativa | -10°C ~ +50°C|
| Umidità di lavoro | ≤ 90%UR |
| Indicatore luminoso | Stato WiFi/Alimentazione/Stato rete Zigbee |
| Metodo di ripristino | Pulsante reset + pressione prolungata per ripristinare le impostazioni di fabbrica |

## Caratteristiche

- Doppi uplink: connessione cablata WiFi o Ethernet, scelta flessibile in base all'ambiente di rete
-Supporta l'accesso remoto: controlla lo stato del dispositivo sempre e ovunque tramite l'app cloud
- Collegamento alla scena locale: supporta il funzionamento automatizzato delle apparecchiature all'interno della LAN (ancora disponibile quando disconnesso)
- Aggiornamento del firmware OTA
- Supporta la piattaforma Tuya/Smart Life
- Gestisci fino a 128 sotto-dispositivi Zigbee
-Supporta l'autoriparazione e l'ottimizzazione del routing della rete Zigbee
