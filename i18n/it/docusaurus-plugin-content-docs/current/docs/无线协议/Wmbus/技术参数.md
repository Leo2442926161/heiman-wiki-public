# Parametri tecnici

Di seguito sono riportati i parametri tecnici generali delle apparecchiature Heimann Wireless M-Bus (wM-Bus).

## Parametri wireless

| Parametro | Valore |
|------|-----|
| Norme del protocollo | EN 13757-4 (Strato fisico Wireless M-Bus) |
| Banda di frequenza operativa | 868 MHz (banda di frequenza europea SRD, 868,0~868,6 MHz) |
| Metodo di modulazione | FSK |
| Velocità di trasmissione | 4,8 kbps / 16,384 kbps / 100 kbps (modalità T / modalità S / modalità R) |
| Topologia di rete | Stella (strumento → collettore/gateway) |
| Distanza di comunicazione | Circa 100~300 metri all'interno, circa 500~1000 metri all'aperto |
| Trasmettere potenza | ≤ 25 mW (+14 dBm) |

## Modalità di lavoro

| Modalità | Vota | Descrizione |
|------|------|------|
| Modalità S | 16.384 kbps | Modalità standard, basso consumo energetico, adatta per strumenti alimentati a batteria |
| Modalità T | 100 kbps | Modalità ad alta velocità, adatta a scenari di dati di traffico elevati |
| Modalità R | 4,8 kbps | Modalità a banda stretta, trasmissione a lunga distanza |

## Consumo elettrico ed energetico

| Parametro | Valore |
|------|-----|
| Metodo di alimentazione | Alimentazione a batteria al litio (ER18505 / ER26500 e altre batterie al litio industriali) |
| Durata della batteria | 6~10 anni (a seconda della frequenza di reporting, il valore tipico è 1 volta/giorno) |
| Consumo energetico in standby | Livello microamp, sonno profondo |
| Ricevi sensibilità | ca. -105 dBm (modalità S) |

## Parametri ambientali

| Parametro | Valore |
|------|-----|
| Temperatura operativa | -20°C ~ +60°C (grado industriale) |
| Umidità operativa | ≤ 95% UR (senza condensa) |
| Livello di protezione IP | IP65 ~ IP68 (a seconda del modello specifico, applicabile a strumenti esterni) |

## Protocollo dati

| Strato | Norma |
|----|------|
| Strato fisico | EN 13757-4|
| Livello collegamento dati | EN 13757-4 (Formato frame M-Bus wireless) |
| Livello di applicazione | EN 13757-3 (livello di applicazione M-Bus) |
| Metodo di crittografia | AES-128 (crittografia opzionale) |
