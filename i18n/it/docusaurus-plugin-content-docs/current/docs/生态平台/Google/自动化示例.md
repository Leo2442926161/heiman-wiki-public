# Esempio di automazione

## Routine "Sono fuori".

**Trigger:** di' "Sono fuori" a Google Home

**Esegui azione:**
- Sensore magnetico porta Heiman → braccio
- Sensore a infrarossi del corpo umano Heiman → Attiva il rilevamento della partenza da casa
- Spegnere tutte le luci e gli elettrodomestici
- Termostato → Imposta sulla temperatura di risparmio energetico
- Google Nest Hub → Mostra la scheda di conferma dell'uscita di casa
- Risposta vocale: "La modalità Esci da casa è stata attivata, fai buon viaggio"

## Trasmissione "Rilevazione di fumo".

**Attivazione:** Allarme fumo Hyman → Fumo rilevato

**Esegui azione:**
- Google Nest Speaker → Trasmissione per tutta la casa "⚠️ Fumo rilevato! Controlla immediatamente!"
- Tutte le luci intelligenti → Accese (tramite il set di luci intelligenti di Google Home)
- Notifiche push sui telefoni cellulari dei membri della famiglia
- Fotocamera → Avvia la registrazione

## Modalità buonanotte

**Trigger:** di' "Buonanotte" a Google Home

**Esegui azione:**
- Sensore porta Heiman → Attiva l'allarme porta aperta
- Infrarossi del corpo umano Heiman → attiva il rilevamento notturno
- Luci → si spengono gradualmente
- Risposta vocale: "Buona notte, il sistema di sicurezza è attivo"
