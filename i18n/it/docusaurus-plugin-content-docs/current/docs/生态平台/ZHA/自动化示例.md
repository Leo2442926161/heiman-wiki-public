# Esempio di automazione

## Collegamento dell'allarme fumo

```yaml
alias: "Collegamento rilevatore di fumo ZHA"
attiva:
  - trigger: stato
    entità_id: binario_sensore.heiman_smoke_zha
    a: "su"
azioni:
  - azione: light.turn_on
    obiettivo:
      area_id: soggiorno_camera
  - azione: tts.speak
    obiettivo:
      entità_id: media_player.speaker
    dati:
      messaggio: "Allarme fumo attivato!"
```

## Automazione dell'associazione dei dispositivi

```yaml
alias: "Promemoria per l'associazione del nuovo dispositivo"
attiva:
  - trigger: zha.event
    sottotipo: heimman
    comando: dispositivo_unito
azioni:
  - azione: notify.persistent_notification
    dati:
      messaggio: "Il nuovo dispositivo Haiman si è unito alla rete ZHA, vai al nome e all'area della configurazione"
```

## Rilevamento dell'immersione in acqua

```yaml
alias: "ZHA Flood Test"
attiva:
  - trigger: stato
    entità_id: Binary_sensor.heiman_water_leak_zha
    a: "su"
azioni:
  - azione: switch.turn_off
    entità_id: switch.main_water_valve
  - azione: notify.mobile_app_phone
    dati:
      messaggio: "💧 Rilevata allagamento!"
```
