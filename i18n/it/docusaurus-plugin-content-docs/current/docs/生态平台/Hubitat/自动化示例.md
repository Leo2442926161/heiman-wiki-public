# Esempio di automazione

## Rilevatore di fumo Rule Machine

**Nome regola:** Collegamento di sicurezza dell'allarme fumo

**Configurazione:**
```
Trigger: l'allarme fumo Hyman diventa attivo
Condizioni: Modalità != Disinserito
Azione:
  - Accendi tutte le luci
  - Invia notifica push "⚠️ Allarme fumo!"
  - Accendi la sirena (se disponibile)
  - Imposta la modalità su "Emergenza"
```

## Monitoraggio della temperatura della macchina regola

**Nome regola:** Protezione dalle alte temperature

**Configurazione:**
```
Trigger: sensore di temperatura Heimann > 38°C
Condizione: Stato del condizionatore = spento
Azione:
  - Accendere l'aria condizionata, modalità raffreddamento, 24°C
  - Invia notifica "La temperatura ambiente è troppo alta ed è stato avviato il condizionatore"
```

## Rilevamento perdite d'acqua della Rule Machine

**Nome della regola:** Trattamento di emergenza delle perdite d'acqua

**Configurazione:**
```
Trigger: il rilevatore a immersione in acqua Hyman si bagna
Condizione: Stato valvola acqua = aperta
Azione:
  - Chiudere la valvola principale dell'acqua
  - Invia notifiche push a tutti i telefoni cellulari
  - Invia notifica via SMS (facoltativo)
```
