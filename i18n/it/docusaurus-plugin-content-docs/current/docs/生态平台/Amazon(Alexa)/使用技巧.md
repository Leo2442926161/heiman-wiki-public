# Suggerimenti per l'uso

## Rilevamento del dispositivo

- Una volta acceso, il dispositivo Heiman Matter verrà rilevato automaticamente dall'app Alexa, in genere entro 30 secondi
- Se non viene rilevato automaticamente: Accedi all'App Alexa → Dispositivo → "+" → "Aggiungi dispositivo" → "Obiettivo"
- Utilizza il codice di configurazione Matter o il codice QR nella parte inferiore del dispositivo per completare l'associazione

## Accesso al dispositivo Zigbee (Echo Plus / Echo Show)

- I dispositivi Echo con Zigbee Hub possono essere accoppiati direttamente con i sensori Heiman Zigbee
- Modalità di abbinamento: App Alexa → Dispositivo → "+" → "Zigbee" → "Cerca dispositivo"
- I dispositivi Heimann sono compatibili con Alexa con il coordinatore Zigbee integrato

## Suggerimenti di routine

- Utilizza "azioni personalizzate" per trasmettere istruzioni vocali più dettagliate
- Aggiungi "attendi 5 secondi" alla routine di emergenza prima di eseguire le azioni successive per evitare sovrapposizioni vocali
- Approfitta della funzione "Rilevamento del suono" per fornire ad Alexa risposte aggiuntive agli allarmi acuti dell'Heyman Alarm

## Guardia Alexa NOTA

Alexa Guard Plus è attualmente disponibile solo negli Stati Uniti. Gli utenti in Cina possono simulare alcune funzioni tramite Routine.
