# Esempio di automazione

## Integrazione di Alexa Guard (Stati Uniti)

**Prerequisito:** Abbonarsi ad Alexa Guard Plus (disponibile negli Stati Uniti)

**Processo di configurazione:**
1. Attiva Guard Plus nell'app Alexa
2. Aggiungi il rilevatore di fumo/CO Hyman come "Dispositivo domestico intelligente"
3. Abilita la risposta intelligente "Rilevamento allarme fumo".

**Risposta automatica:**
- Allarme fumo attivato → Alexa compone automaticamente i contatti di emergenza
- Attivazione allarme CO → Avviso vocale Alexa per tutta la casa, ventilazione automatica
- Porte e finestre si aprono in modo anomalo → Notifica allarme push

## Routine di rilevamento del fumo

**Attivazione:** Allarme fumo Hyman → Fumo rilevato

**Esegui azione:**
- Annuncio vocale Alexa: "⚠️ Rilevato fumo, prestare attenzione alla sicurezza"
- Tutti i dispositivi Echo → Riproduci il tono di allarme
- Set luci intelligenti → tutto illuminato
- Notifiche push sul cellulare

## Routine "Uscire di casa".

**Trigger:** Comando vocale "Alexa, me ne vado"

**Esegui azione:**
- Sensore magnetico porta Heiman → attiva l'inserimento
- Sensore corporeo Heiman → Attiva il rilevamento del movimento
- Rilevatore di immersione in acqua Hyman → attiva il rilevamento
- Tutte le luci → spente
- Alexa ha risposto: "Il sistema di sicurezza è acceso, fai buon viaggio"
