# Esempio di automazione

## Allarme di sicurezza per il rilevamento del fumo

**Configurazione automatizzata HomeKit:**

1. **Condizione di attivazione:** Allarme fumo Hyman → fumo rilevato
2. **Impostazioni di sicurezza:** cambia lo stato del sistema di sicurezza su "Allarme"
3. **Esegui azione:**
   - HomePod → Riproduci audio personalizzato "Fumo rilevato, prestare attenzione alla sicurezza"
   - Tutte le luci intelligenti → si accendono e lampeggiano
   - Notifiche push a tutti i membri della famiglia

## Scena a casa/fuori casa

**Comando di scelta rapida delle scorciatoie: "Esci da casa Controllo sicurezza"**

```
quando l'ultima persona esce di casa
  Configura il sistema di sicurezza → attiva quando esci di casa
  Configura il sensore porta Heiman → Attiva le notifiche
  Configura il sensore corporeo Heiman → attiva la sveglia
  Spegni tutte le luci
  Impostazioni Termostato → Modalità risparmio energetico
  Invia notifica "🏠 Modalità Fuori Casa attivata"
```

## Promemoria anomalie della temperatura

**Condizione di attivazione:** Sensore di temperatura e umidità Hyman → Temperatura > 35°C

**Esegui azione:**
- HomePod → La temperatura della stanza di trasmissione vocale supera il limite
- Notifiche push su iPhone
- Accendere automaticamente l'aria condizionata (se HomeBridge è configurato)
