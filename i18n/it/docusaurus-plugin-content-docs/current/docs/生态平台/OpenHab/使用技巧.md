# Suggerimenti per l'uso

## Configurazione della cosa

Aggiungi il dispositivo Heiman tramite associazione MQTT:

```Java
Thing mqtt:topic:heiman_smoke "Allarme fumo Heiman" @ "soggiorno" {
    Canali:
        Digitare binario: smoke "smoke state" [ stateTopic="zigbee2mqtt/heiman_smoke", trasformazionePattern="JSONPATH:$.smoke"]
        Numero tipo: batteria "Potenza batteria" [stateTopic="zigbee2mqtt/heiman_smoke", trasformazionePattern="JSONPATH:$.battery"]
}
```

## Definizione dell'articolo

```Java
Contatta Smoke_Alarm "Allarme fumo [MAP(smoke.map):%s]" { channel="mqtt:topic:heiman_smoke:smoke" }
Numero Smoke_Battery "Batteria rilevatore di fumo [%d%%]" { channel="mqtt:topic:heiman_smoke:battery" }
```

## Selezione delle coordinate

Si consiglia di utilizzare un coordinatore USB con chip **CC2531** o **CC2652** e assicurarsi che la versione del firmware sia la più recente.

## Suggerimenti per il debug

- Visualizza lo stato "online/offline" dell'oggetto nell'interfaccia utente di Paper
- Utilizzare `openhab-cli info` per verificare se i collegamenti sono caricati correttamente
- Cerca la parola chiave "heiman" nel registro per risolvere i problemi di comunicazione del dispositivo
