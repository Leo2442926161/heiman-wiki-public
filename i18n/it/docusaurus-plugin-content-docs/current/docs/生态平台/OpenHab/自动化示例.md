# Esempio di automazione

## Regole di allarme fumo

```javascript
regola "Allarme fumo attivato"
quando
    L'elemento heiman_smoke_alarm è stato modificato in ON
allora
    sendCommand(Luce_soggiorno, ON)
    sendNotification("admin", "⚠️ Allarme fumo attivato!")
    postUpdate(Stato_allarme, "smoke_alert")
fine
```

## Regole del sensore di temperatura

```javascript
regola "Rapporto temperatura e umidità"
quando
    Articolo heimman_temperature ricevuto aggiornamento
allora
    val temp = (heiman_temperature.state come numero).floatValue
    val hum = (heiman_humidity.state come numero).floatValue
    se (temperatura > 40) {
        sendCommand(Condizionatore, ON)
        sendNotification("admin", "La temperatura è troppo alta (" + temp + "°C), il condizionatore è stato acceso")
    }
    se (ronzio > 80) {
        sendNotification("admin", "L'umidità è troppo alta (" + ronzio + "%), prestare attenzione alla deumidificazione")
    }
fine
```

## Regole per il rilevamento delle inondazioni

```javascript
regola "Collegamento rilevamento immersione in acqua"
quando
    L'elemento heiman_water_leak è stato modificato in ON
allora
    sendCommand(Valvola_Acqua, OFF)
    sendBroadcastNotification("💧 Rilevata perdita d'acqua! La valvola dell'acqua è stata chiusa!")
fine
```
