# Esempio di automazione

## L'allarme fumo attiva le luci in tutta la casa

```yaml
alias: "Allarme fumo - luci e notifiche in tutta la casa"
attiva:
  - trigger: stato
    entità_id: binario_sensore.heiman_smoke_alarm
    a: "su"
azioni:
  - azione: light.turn_on
    obiettivo:
      entità_id: tutto
  - azione: notify.mobile_app_phone
    dati:
      messaggio: "⚠️ Allarme fumo attivato! Controlla immediatamente!"
      titolo: "Allarme fumo"
```

## Collegamento allarme CO

```yaml
alias: "Allarme CO - chiudi gas e apri finestre"
attiva:
  - trigger: stato
    entità_id: binario_sensore.heiman_co_alarm
    a: "su"
azioni:
  - azione: switch.turn_off
    entità_id: switch.gas_fireplace
  - azione: cover.open
    entità_id: cover.living_room_window
  - azione: notify.persistent_notification
    dati:
      messaggio: "È intervenuto l'allarme monossido di carbonio, è stata chiusa la valvola del gas e sono state aperte le finestre"
```

## Collegamento del rilevatore di immersione in acqua

```yaml
alias: "Rilevamento allagamento - Chiudi valvola acqua"
attiva:
  - trigger: stato
    entità_id: binario_sensore.heiman_water_leak
    a: "su"
azioni:
  - azione: switch.turn_off
    entità_id: switch.main_water_valve
  - azione: notify.mobile_app_phone
    dati:
      messaggio: "💧 Rilevata perdita d'acqua! La valvola principale dell'acqua è stata chiusa!"
```

## Inserimento lontano da casa: il sensore del corpo umano attiva l'allarme

```yaml
alias: "Modalità Leave Home - Allarme sensore corpo umano"
attiva:
  - trigger: stato
    entità_id: binario_sensore.heiman_motion_sensor
    a: "su"
condizioni:
  - condizione: stato
    entità_id: pannello_di_controllo_allarme.home_alarm
    stato: armato_lontano
azioni:
  - azione: allarme_controllo_panel.trigger
    entità_id: pannello_di_controllo_allarme.home_alarm
```
