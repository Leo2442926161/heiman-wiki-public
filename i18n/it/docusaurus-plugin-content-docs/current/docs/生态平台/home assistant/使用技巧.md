# Suggerimenti per l'uso

## Configurazione del dispositivo

- Quando accoppi il tuo dispositivo Heiman Matter per la prima volta, assicurati che il tuo router di confine (ad esempio HomePod, Nest Hub, Home Assistant Green) sia sulla stessa rete Thread
- Quando si integrano dispositivi Zigbee tramite ZHA o Zigbee2MQTT, si consiglia di utilizzare prima il firmware consigliato del coordinatore USB (come Sonoff ZBDongle-E, Conbee II)

## Ottimizzazione automatizzata

- Aggiunto il parametro "for:" (ad esempio 3 secondi) all'automazione dell'allarme fumo per prevenire falsi allarmi istantanei
- Utilizzare "condizione: stato" per abbinare la modalità di sicurezza per evitare notifiche non necessarie nello stato armato
- Utilizza la funzione "gruppo" di Home Assistant per raggruppare i sensori Heiman per area per un facile funzionamento in batch

## Risoluzione dei problemi

- Quando il dispositivo è offline, controlla innanzitutto se il canale della rete Zigbee è in conflitto con WiFi 2.4GHz
- Quando non è possibile rilevare il dispositivo Matter, riavviare il router di confine e aggiungere nuovamente il dispositivo
