# Configurazione della dashboard

## Pannello Panoramica della sicurezza

Utilizzare il layout della vista in sezione per organizzare l'attrezzatura Heimann in aree:

```yaml
tipologia: sezione
titolo: Allarme fumo e monossido di carbonio
carte:
  - tipo: entità
    entità: Binary_sensor.heiman_smoke_alarm
    nome: rilevatore di fumo
  - tipo: entità
    entità: Binary_sensor.heiman_co_alarm
    nome: allarme CO
```

## Stile carta fungo

```yaml
tipo: personalizzato:scheda-centrale-allarme-fungo
entità: pannello_di_controllo_allarme.home_alarm
nome: sistema di sicurezza

tipo: personalizzato:carta-entità-fungo
entità: Binary_sensor.heiman_smoke_alarm
nome: rilevatore di fumo
icona: mdi:rivelatore di fumo
informazioni_primarie: stato
secondaria_info: ultima modifica
fill_container: vero
```

## Pannello sensore ambientale

```yaml
tipologia: sezione
titolo: Monitoraggio ambientale
carte:
  - tipo: scartamento
    entità: sensor.heiman_temperature
    nome: temperatura
    unità: "°C"
    minimo: 0
    massimo: 50
    gravità:
      verde: 0
      giallo: 30
      Rosso: 40
  - tipo: scartamento
    entità: sensor.heiman_humidity
    nome: umidità
    unità: "%"
    minimo: 0
    massimo: 100
    gravità:
      verde: 30
      Giallo: 60
      Rosso: 80
```

## Pannello cronologia allarmi

```yaml
tipo: grafico storico
titolo: Storia del sensore
entità:
  -binary_sensor.heiman_smoke_alarm
  -binary_sensor.heiman_co_alarm
  -binary_sensor.heiman_water_leak
ore_di_spettacolo: 24
```

## Disposizione suggerita

- In alto: Stato sistema di sicurezza + scheda stato dispositivi di allarme
- Centrale: Monitoraggio ambientale (temperatura, umidità, qualità dell'acqua)
- In basso: grafico dell'andamento storico degli allarmi del sensore
- Operazioni di scelta rapida della barra laterale: inserimento/disinserimento, cambio scena
