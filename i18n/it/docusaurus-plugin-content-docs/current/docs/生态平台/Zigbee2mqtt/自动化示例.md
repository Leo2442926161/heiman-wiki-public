# Esempio di automazione

## Stato del dispositivo con abbonamento MQTT

"bash."
mosquitto_sub -h localhost -t "zigbee2mqtt/heiman_smoke/#" -v
```

Esempio di output:
```
zigbee2mqtt/heiman_smoke {"smoke":true,"battery":100,"linkquality":120}
```

## Monitoraggio online delle apparecchiature

```yaml
# configurazione.yaml
sensore_binario:
  - piattaforma: mqtt
    nome: "Rilevatore di fumo online"
    state_topic: "zigbee2mqtt/heiman_smoke/disponibilità"
    payload_on: "online"
    payload_off: "offline"
    device_class: connettività
