# Esempio di automazione

## Modalità buonanotte "Buonanotte".

**Condizione di attivazione:** Sensore magnetico porta Haiman → Chiudi (indica che la porta della camera è stata chiusa)

**Esegui azione:**
- Spegni tutte le luci del soggiorno
- Blocco intelligente → Blocca
- Sistema di sicurezza → inserimento notturno
- Notifica push: "Buona notte, il sistema è inserito"

## Modalità "Assente".

**Condizione di attivazione:** Sensore a infrarossi del corpo umano Hyman → 15 minuti di inattività

**Esegui azione:**
- Tutte le luci spente
- Termostato → Modalità risparmio energetico
- Sistema di sicurezza → inserimento fuori casa
- Sensore porta → attiva l'allarme
- Notifica push: "La modalità Fuori casa è attivata, i sensori di porte e finestre sono abilitati"

## Modalità di emergenza del rilevatore di fumo

**Condizione di attivazione:** Allarme fumo Hyman → Fumo rilevato

**Esegui azione:**
- Altoparlante intelligente → riproduci il suono della sveglia
- Tutte le luci intelligenti → lampeggiano in rosso
- Notifiche push a tutti i membri della famiglia
- Se connesso all'Hub SmartThings → attiva la sirena
