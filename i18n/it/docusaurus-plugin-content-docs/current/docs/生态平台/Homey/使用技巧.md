# Suggerimenti per l'uso

## Interferenza del canale Zigbee

Il chip Zigbee integrato di Homey utilizza canali specifici. Se il dispositivo Heiman va spesso offline:

1. Visualizza il canale corrente in Impostazioni Homey → Zigbee
2. Utilizza l'app Homey Insight per cercare eventuali interferenze ambientali
3. Passa a un canale con meno interferenze (lontano dal WiFi 6/1/11)

## Rete di distribuzione delle apparecchiature

- Come accedere alla modalità di accoppiamento per i dispositivi Heiman: tenere premuto il pulsante per 5 secondi (alcuni modelli devono premere 3 volte velocemente)
- Quando aggiungi un dispositivo nell'app Homey, seleziona "Aggiungi dispositivo → Zigbee → Cerca"
- Quando l'accoppiamento fallisce, avvicina il dispositivo a Homey (entro 3 metri) e riprova

## Suggerimenti per l'ottimizzazione del flusso

- Aggiungi la condizione "ritardo" in Flow (ad esempio 2 secondi) per evitare falsi positivi temporanei
- Utilizzare la funzione "tag" per classificare e gestire le apparecchiature Heiman

## Estensione della portata del segnale

Se il dispositivo Heiman è lontano da Homey, è possibile aggiungere un dispositivo relè Zigbee (come una presa intelligente) per estendere la copertura di rete.
