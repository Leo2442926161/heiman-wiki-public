#Accogliente

Homey è un hub di casa intelligente lanciato dalla società olandese Athom, che supporta Zigbee, Z-Wave, Wi-Fi, Matter, Thread e altri protocolli.

## Metodo di integrazione

I dispositivi Homey Zigbee e Z-Wave si accoppiano direttamente tramite il chip wireless integrato di Homey, eliminando la necessità di un gateway aggiuntivo. È possibile accedere ai dispositivi Matter tramite il router di confine Thread integrato di Homey Pro.

## Vantaggi

- **Più protocolli in uno**: Homey Pro supporta Zigbee, Z-Wave, Thread e Wi-Fi contemporaneamente
- **Esegui localmente**: tutta l'automazione viene eseguita localmente
- **Automazione del flusso**: programmazione drag-and-drop per creare scenari di collegamento complessi

> **Riferimento**: le app correlate a Heiman possono essere cercate nell'Homey App Store
