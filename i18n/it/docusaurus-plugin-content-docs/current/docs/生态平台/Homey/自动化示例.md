# Esempio di automazione

## Allarme fumo: luci flash e notifiche

**Configurazione flusso:**

1. **Attivazione:** Allarme fumo Hyman → Lo stato cambia in "avviso".
2. **Condizioni:** (Nessuna)
3. **Azione:**
   - Luce soggiorno → lampeggiante (rossa, una volta al secondo)
   - Notifica push → Mobile: "⚠️ Allarme fumo attivato!"
   - Annuncio vocale → Homey Speaker: "Rilevato fumo, prestare attenzione alla sicurezza!"

## Allarme CO - Spegne la caldaia

**Configurazione flusso:**

1. **Attivazione:** Allarme CO Hyman → Lo stato cambia in "avviso".
2. **Condizioni:** (Nessuna)
3. **Azione:**
   - Interruttore caldaia → OFF
   - Presa intelligente (sistema di aria fresca) → Accesa
   - Notifica push → "Allarme monossido di carbonio, caldaia è stata spenta"

## Allarme di superamento del limite di temperatura e umidità

1. **Attivazione:** Sensore di temperatura e umidità Hyman → Temperatura superiore a 40°C
2. **Azione:**
   - Condizionatore d'aria → Attiva la modalità di raffreddamento
   - Notifica push → "La temperatura della stanza è troppo alta, il condizionatore è stato acceso automaticamente"
