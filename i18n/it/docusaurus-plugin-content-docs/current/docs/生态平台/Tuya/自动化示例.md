# Esempio di automazione

## Scena dell'allarme fumo

**Nome della scena:** "Collegamento dell'allarme fumo"

**Condizione di attivazione:** Allarme fumo Hyman → Fumo rilevato

**Esegui azione:**
- Presa intelligente → aperta (collegare la ventola di scarico)
- Luci → si accendono e lampeggiano
- Notifiche push sul cellulare
- Altoparlante → Riproduci il suono della sveglia

## Scena in cui si esce di casa con un clic (Tap-to-Run)

**Nome della scena:** "Accordo dopo aver lasciato casa"

**Esecuzione manuale:** tocco o attivazione vocale

**Esegui azione:**
- Sensore porta Haiman → attiva l'allarme
- Infrarossi del corpo umano Hyman → Attiva il rilevamento
- Rilevatore di immersione in acqua Hyman → attiva il rilevamento
- Tutte le luci → spente
- Blocco intelligente → Blocca

## Controllo automatico della temperatura e dell'umidità

**Tipo di automazione:** Attivazione automatica delle condizioni

**Condizione di attivazione:** Sensore di temperatura e umidità Hyman Umidità > 75%

**Esegui azione:**
- Presa intelligente (deumidificatore) → Accesa
- Notifica push: "L'umidità è troppo alta, il deumidificatore è stato acceso"

**Condizione di attivazione:** Sensore di temperatura e umidità Hyman Umidità < 50%

**Esegui azione:**
- Presa Smart (deumidificatore) → Chiudi
