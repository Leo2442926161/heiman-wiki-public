# Altre certificazioni

## Marchio di qualità Q-Label

Q-Label è un marchio di certificazione di qualità per l'industria europea della casa intelligente, lanciato dalla Q-Lab Alliance (ora parte della Zigbee Alliance/CSA Connectivity Standards Alliance).

| Progetto | Descrizione |
|------|------|
| Ambito di applicazione | Certificazione di interoperabilità dei dispositivi Zigbee |
| Contenuto della certificazione | Compatibilità del dispositivo con altre marche di gateway/controller Zigbee |
| Significato | Garantire l'interoperabilità tra marchi e ridurre il rischio di "comprarlo a casa e non utilizzarlo" |

I prodotti Hyman Zigbee sono certificati Q-Label, garantendo la compatibilità con le seguenti piattaforme:
- Amazon Echo Plus/Echo Show (Zigbee integrato)
-Ponte Philips Hue
- Hub SmartThings
- Altri gateway di autenticazione Q-Label

## Sistema di gestione della qualità ISO 9001

Haiman ha superato la certificazione del sistema di gestione della qualità ISO 9001:2015 per garantire il controllo di qualità dell'intero processo dalla ricerca e sviluppo, dalla produzione al post-vendita.
