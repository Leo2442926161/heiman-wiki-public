# Confronto dei modelli

## Confronto tra vari modelli

| Modello | Protocollo | Certificazione | Alimentazione | Mercato | Suggerimenti di gioco |
|------|------|------|------|------|----------|
| [HS1SA-E](../../Protocollo wireless/Zigbee/HS1SA-E - Rilevatore di fumo intelligente (Zigbee)) | Zigbee | EN14604| CR123A | UE | Connettiti a Zigbee2MQTT/ZHA, collega con Home Assistant, accendi automaticamente le luci quando viene rilevato fumo + notifiche push mobile |
| [HS1SA-Z](../../Protocollo wireless/Zwave/HS1SA-Z Rilevatore di fumo intelligente (Z-Wave)) | Z-Wave | EN14604| CR123A | UE | Connettiti a SmartThings/Hubitat, collabora con l'attivazione magnetica della porta Z-Wave e avvisa automaticamente l'allarme fumo dopo essere uscito di casa |
| [HS1SA-M](../../Protocollo wireless/Matter/HS1SA-M Allarme fumo intelligente (Matter)) | Questione sul filo | EN14604| CR123A | Globale | Accesso nativo a Apple Home/Google Home, non è richiesto alcun gateway dedicato, collegamento di allarmi localizzati a bassa latenza Thread |
| [HS3SA](../../Protocollo wireless/Zigbee/HS3SA - Rilevatore di fumo intelligente (Zigbee)) | Zigbee | EN14604| CR123A | UE | Utilizzato con il gateway Haiman HS6GW, collegamento di scenari multi-dispositivo, adatto agli utenti ecologici di Haiman |
| [HS8SA](../../Protocollo wireless/Zigbee/HS8SA - Rilevatore di fumo intelligente (Zigbee)) | Zigbee / Wi-Fi | UL217 | Batteria/rete | Stati Uniti | Esclusivamente per il mercato statunitense, certificato UL, soddisfa le norme antincendio locali, supporta la connessione diretta WiFi o il gateway Zigbee |
| [Serie S1](../../Protocollo wireless/Sub1G (lunga portata)/Serie S1 - Rilevatore di fumo autonomo (Sub-1G)) | Sub-1G | EN14604| Batteria | Universale | Interconnessione indipendente, nessun gateway e Internet richiesti, interconnessione wireless Sub-1G tra più unità S1, adatta per semplici case in affitto |

## Scena di gioco consigliata

| Scenari | Modelli consigliati | Soluzioni di collegamento |
|------|----------|----------|
| Profonda integrazione della casa intelligente | HS1SA-E / HS1SA-M | Automazione HA: rilevatore di fumo → accendere le luci in tutta la casa → sbloccare le serrature delle porte → chiudere la valvola del gas → inviare al cellulare |
| Utenti Apple HomeKit | HS1SA-M | Accesso Native Matter, HomePod come router di confine Thread, attivazione automatica delle scene |
| Requisiti standard europei e americani | HS1SA-E (CE) / HS8SA (UL) | Soddisfare la certificazione antincendio locale, i requisiti di conformità assicurativa |
| Nessun gateway richiesto/costo basso | Serie S1 | Pronte all'uso dopo l'acquisto, più unità sono interconnesse tramite Sub-1G, attivandosi a vicenda quando si verifica un allarme |
| Sistema esistente Z-Wave | HS1SA-Z | Funziona con i gateway Z-Wave esistenti come Hubitat/SmartThings |

## Suggerimenti per la selezione

- **Mercato UE**: [HS1SA-E](../../Wireless Protocol/Zigbee/HS1SA-E - Allarme fumo intelligente (Zigbee)) (Zigbee) o [HS3SA](../../Wireless Protocol/Zigbee/HS3SA - Allarme fumo intelligente (Zigbee)) (Zigbee) se è richiesta la selezione Z-Wave [HS1SA-Z](../../Protocollo wireless/Zwave/HS1SA-Z Rilevatore di fumo intelligente (Z-Wave))
- **Mercato statunitense**: [HS8SA](../../Wireless Protocol/Zigbee/HS8SA - Rilevatore di fumo intelligente (Zigbee)) (certificato UL 217)
- **Questione multipiattaforma**: [HS1SA-M](../../Wireless Protocol/Matter/HS1SA-M Allarme fumo intelligente (Matter)) (Matter over Thread)
- **Scena autonoma**: [Serie S1](../../Protocollo wireless/Sub1G (lunga distanza)/Serie S1 - Allarme fumo autonomo (Sub-1G)) (non è richiesto alcun gateway)
