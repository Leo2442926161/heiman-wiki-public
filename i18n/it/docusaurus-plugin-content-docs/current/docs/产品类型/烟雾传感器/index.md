# sensore di fumo

Il sensore di fumo Hyman utilizza la **tecnologia di rilevamento del fumo fotoelettrico**, in grado di rilevare in modo sensibile le particelle di fumo visibili generate nelle prime fasi di un incendio e ottenere un allarme incendio molto precoce.

## Funzioni principali

- Rilevazione fumo fotoelettrica per rilevare il fumo proveniente da bruciature lente e fiamme libere
- Allarme sonoro e luminoso alto 85 dB
- Promemoria di batteria scarica
- Funzione di autotest (test con un clic)
- Allarme antimanomissione
-Supporta l'audio remoto

## Parametri tecnici (prendendo come esempio HS1SA-M)

- Tipo di sensore: fotoelettrico a infrarossi
- Pressione sonora dell'allarme: ≥85 dB (3 metri)
- Alimentazione: batteria CR123A
- Durata della batteria: ca. 3 anni
- Temperatura di funzionamento: -10°C ~ +50°C
- Umidità di lavoro: ≤95% RH
