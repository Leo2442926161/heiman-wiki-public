# Sensore a infrarossi del corpo umano

Il sensore a infrarossi del corpo umano (PIR) rileva il movimento rilevando la radiazione termica infrarossa emessa dal corpo umano ed è la soluzione di rilevamento della presenza più comunemente utilizzata nelle case intelligenti.

## Serie di prodotti

| Serie | Metodi di comunicazione | Caratteristiche |
|------|----------|------|
| HS1MS | Zigbee | Modello base, angolo di rilevamento 110° |
| HS3MS | Zigbee | Modello aggiornato, installazione a soffitto a 360° |
| Serie M1 | Materia / Zigbee | Supporta Matter over Thread, opzionale 110° o 360° |
| HS8MLS | Zigbee | Modello a lunga distanza |
| HS8MIS | Zigbee | Corpo umano + sensore composito di illuminazione |

## Funzioni principali

- Rilevamento umano mobile
- Rilevamento dell'illuminazione (serie M1/HS8MIS)
- Distanza di rilevamento ≤7m (25°C)
- Metodo di installazione: a parete o a soffitto
- Durata della batteria: 3 anni (Zigbee) / 1 anno (Matter)

## Parametri tecnici (serie M1)

- Angolo di rilevamento: 110° / 360°
- Distanza di rilevamento: ≤7m
- Intervallo di illuminamento: 0~1200 Lux
- Alimentazione: DC 3V (batteria al litio)
- Durata della batteria: 3 anni (Zigbee) / 1 anno (Matter)

## Esempio di scenario

- La luce si accende quando qualcuno arriva e si spegne quando qualcuno esce
- Collegamento di sicurezza: premere l'allarme quando viene rilevato un movimento anomalo
- Inserimento quando si esce di casa: spegne automaticamente l'aria condizionata/riscaldamento per risparmiare energia quando non c'è nessuno nelle vicinanze
