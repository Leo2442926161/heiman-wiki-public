# Guida all'installazione

## Requisiti della posizione di installazione

- Altezza di montaggio a parete: **1,8 ~ 2,4 m**
- Installazione a soffitto: Installata a soffitto al centro della stanza
- **Evitare**: Luce solare diretta, bocchette di riscaldamento/aria condizionata, vicino a fonti di calore
- Evitare ostacoli (mobili, pareti divisorie in vetro) all'interno del campo di rilevamento

## Passaggi di installazione

1. Selezionare il luogo di installazione (a parete o a soffitto)
2. Fissare la base di montaggio con le viti
3. Inserire la batteria, la spia lampeggia
4. Installare il corpo del sensore sulla base
5. Regolare l'angolo di rilevamento (entro l'intervallo orizzontale di 110°)
6. Test di camminata: dopo aver attraversato l'area di rilevamento, la spia dovrebbe accendersi
7. Associazione tramite gateway

## Note

- I sensori a infrarossi sono sensibili alle variazioni di temperatura e la distanza di rilevamento potrebbe ridursi in ambienti ad alta temperatura in estate.
- Evitare di affacciarsi sulle finestre (i cambiamenti della temperatura esterna causano falsi allarmi)
- Durata della batteria circa 3 anni (Zigbee) / 1 anno (Matter)
