# Sensore di vibrazione

Il sensore di vibrazione attiva un allarme rilevando la vibrazione o l'inclinazione di un oggetto. È adatto a vari scenari come porte, finestre, casseforti e tubi.

## Funzioni principali

- Rilevamento delle vibrazioni tridimensionali
- Rilevamento dell'angolo di inclinazione
- Promemoria di batteria scarica

## Scenari applicabili

- **Antiscasso per porte e finestre**: Rileva le vibrazioni causate da un'effrazione violenta
- **Monitor cassaforte/cassetto**: rileva movimenti anomali
- **Monitoraggio tubazioni**: rileva le vibrazioni delle tubazioni dell'acqua (indicando anomalie della pompa dell'acqua o colpi d'ariete)
- **Monitoraggio muro/recinzione**: rileva arrampicata o impatto

## Parametri tecnici

- Tipo di sensore: accelerometro a 3 assi
- Dimensioni di rilevamento: vibrazione + inclinazione
- Metodo di alimentazione: batteria a bottone CR2450
- Durata della batteria: ca. 2 anni
