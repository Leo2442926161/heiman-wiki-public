# Guida all'installazione

## Metodo di indossamento

- **Clip da cintura**: agganciarlo alla cintura o alla cintura
- **Cordino**: da appendere al collo e da appendere al petto

## Istruzioni per l'uso

1. Installare la batteria
2. Scegli il metodo di indossamento (clip da cintura o cordino)
3. Indossalo sul tuo corpo
4. Associazione tramite gateway
5. Prova di simulazione:
   - Stare in piedi e poi accovacciarsi rapidamente o sdraiarsi a terra
   - Il sensore dovrebbe rilevare una caduta e attivare l'allarme in pochi secondi
   - Premere il pulsante di soccorso per testare l'allarme manuale

## Note

- Consigliato per essere indossato tutto il giorno (può essere rimosso per caricare/sostituire la batteria quando si fa la doccia)
- L'algoritmo AI viene utilizzato per distinguere le cadute dalle attività quotidiane (sedersi, chinarsi), con un basso tasso di falsi allarmi
- Allarme automatico dopo aver rilevato una caduta oppure è possibile premere manualmente il pulsante per chiedere aiuto
- Può essere collegato al pulsante di emergenza per ottenere una doppia protezione
