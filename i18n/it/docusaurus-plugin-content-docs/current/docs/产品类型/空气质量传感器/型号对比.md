# Confronto dei modelli

## Confronto tra vari modelli

| Modello | Protocollo | Parametri di rilevamento | Alimentazione | Caratteristiche | Suggerimenti di gioco |
|------|------|----------|------|------|----------|
| Sensore di qualità dell'aria | Zigbee | PM2,5 / PM10 / CO₂ / TVOC / Temperatura e umidità | Alimentazione di rete | Multifunzione in uno | Monitora la qualità dell'aria interna, collega automaticamente aria fresca/purificatore + promemoria push se supera lo standard |

## Scena di gioco consigliata

| Scenari | Modelli consigliati | Soluzioni di collegamento |
|------|----------|----------|
| Monitoraggio della CO₂ nella camera da letto | Sensore di qualità dell'aria | CO₂ > 1000 ppm → Promemoria per l'apertura delle finestre/sistema di aria fresca collegato per la ventilazione, migliore qualità del sonno |
| Casa recentemente ristrutturata | Sensore di qualità dell'aria | Monitoraggio continuo di TVOC/formaldeide, ventola di scarico collegata quando viene superato lo standard e verrà inviata una notifica rassicurante una volta raggiunto lo standard |
| Cameretta per bambini | Sensore di qualità dell'aria | Monitoraggio PM2.5, accensione automatica del purificatore d'aria quando si supera lo standard per garantire la salute respiratoria |
| Ufficio | Sensore di qualità dell'aria | CO₂ + aria condizionata collegata a temperatura e umidità aria fresca per migliorare l'efficienza del lavoro |
