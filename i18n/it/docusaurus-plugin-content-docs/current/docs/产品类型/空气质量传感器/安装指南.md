# Guida all'installazione

## Requisiti della posizione di installazione

- Installato a **1,2 ~ 1,8 m** di altezza (zona di respirazione)
- Scegli una posizione che rappresenti la qualità media dell'aria nella stanza
- **Tenere lontano**: finestre, porte, prese d'aria, prese dell'aria condizionata
- **Stare lontano**: fumi di cucina, umidificatori, aromaterapia e altre fonti di inquinamento locale

## Passaggi di installazione

1. Selezionare il luogo di installazione (soggiorno, camera da letto o parete dell'ufficio)
2. Fissare la base di montaggio con le viti
3. Collegare all'alimentazione (alimentato tramite USB o adattatore)
4. Il sensore è acceso e il display (se presente) visualizza i dati dopo l'autotest.
5. Associazione tramite gateway
6. Confermare che i dati siano normali nell'APP

## Note

- Assicurarsi che l'ingresso e l'uscita dell'aria del sensore non siano ostruiti
- Pulire regolarmente l'alloggiamento del sensore per evitare l'intasamento della polvere
- Posizionalo nella stanza dei bambini per monitorare il livello di CO₂ e chiederti di aprire la finestra per la ventilazione
- Consigliato per essere combinato con un sistema di aria fresca o un purificatore d'aria
