# Confronto dei modelli

## Scena di gioco consigliata

| Scenari | Modelli consigliati | Soluzioni di collegamento |
|------|----------|----------|
| Sicurezza intelligente per tutta la casa | HS1CA-M | Matter over Thread si connette nativamente ad Apple Home + Google + Alexa, allarme → spegni gas → apri finestre |
| Requisiti standard UE | HS1CA-E | Collegarsi a HA tramite Z2M/ZHA e l'elettrovalvola del gas di collegamento si spegne automaticamente |
| Requisiti degli standard statunitensi | HS-720ES-M | Certificato UL 2034, soddisfa i requisiti assicurativi |
| Riqualificazione a basso costo di vecchie case | Serie C1 | Funzioni di base, utilizzate con gateway |

## Suggerimenti per la selezione

- **Multipiattaforma Matter**: [HS1CA-M](../../Wireless Protocol/Matter/HS1CA-M Allarme intelligente di monossido di carbonio (Matter)) (Versione globale, certificazione ufficiale HA)
- **Mercato UE**: [HS1CA-E](../../Wireless Protocol/Zigbee/HS1CA-E - Allarme intelligente di monossido di carbonio (Zigbee)) (Zigbee)
- **Mercato USA**: HS-720ES-M (UL 2034)
- **Requisiti di base**: [Serie C1](../../Protocollo wireless/Zigbee/Serie C1 - Allarme monossido di carbonio (Zigbee)) (Zigbee)
