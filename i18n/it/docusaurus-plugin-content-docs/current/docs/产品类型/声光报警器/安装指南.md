# Guida all'installazione

## Requisiti della posizione di installazione

- **Installa al centro della casa** per garantire che possa essere ascoltato in tutte le stanze
- Corridoio, soffitto del soggiorno o parete alta
- **2m o più** da terra (evitare di essere toccati dalle persone)
- Se utilizzato insieme ai sensori di fumo/CO, installare nei corridoi delle camere da letto

## Passaggi di installazione

1. Selezionare il luogo di installazione (si consiglia corridoio o soggiorno)
2. Utilizzare le viti per fissare la base di montaggio alla parete o al soffitto
3. Collegare l'alimentazione (versione a rete) o inserire la batteria (versione a batteria)
4. Installare il corpo dell'allarme sulla base
5. Associazione tramite gateway
6. Test di collegamento: attiva qualsiasi sensore, l'allarme sonoro e luminoso dovrebbe rispondere

## Note

- Installare ad un'altezza fuori dalla portata dei bambini
- Verificare regolarmente se il volume dell'allarme e l'illuminazione sono normali
- Per la versione a batteria, fare attenzione a sostituirla quando la batteria è scarica.
- Può impostare regole di collegamento con altri sensori
