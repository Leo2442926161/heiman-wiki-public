# Guida all'installazione

## Requisiti della posizione di installazione

- **La posizione centrale in tutta la casa** garantisce la copertura del segnale Zigbee per tutti i dispositivi
- Cerca di evitare di posizionarlo in un armadio interamente in metallo (il metallo bloccherà i segnali wireless)
- **Prima Ethernet**: si consiglia di utilizzare una connessione di rete cablata, che è più stabile
- Non più di 20-30 m di distanza da altri dispositivi Zigbee (in ambienti chiusi)

## Passaggi di installazione

1. Selezionare il luogo di installazione (mobile TV in soggiorno, accanto alla scatola della corrente debole, ecc.)
2. Collegare l'alimentatore
3. Collega un cavo Ethernet (consigliato) o connettiti al Wi-Fi
4. Attendere che la spia diventi verde/accesa fissa
5. Apri l'APP Haiman e scansiona il codice QR nella parte inferiore del gateway per aggiungerlo
6. Una volta completata l'aggiunta, è possibile associare il dispositivo secondario

## Note

- Il gateway è il cuore della casa intelligente. Si consiglia di utilizzare l'alimentatore di backup UPS.
- La connessione Ethernet è più stabile del Wi-Fi e si consiglia di utilizzarla per prima
- Se il segnale del dispositivo è debole, aggiungi un dispositivo relè Zigbee (come una presa intelligente)
- Il numero massimo di sottodispositivi supportati da diversi modelli è diverso e deve essere selezionato in base alla scala.
