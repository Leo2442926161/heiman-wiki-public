# Gateway intelligente

Smart Gateway è l'hub principale del sistema di casa intelligente Haiman, responsabile della connessione di vari sensori Zigbee e della realizzazione del controllo del collegamento tra i dispositivi.

## Serie di prodotti

| Serie | Metodi di comunicazione | Caratteristiche |
|------|----------|------|
| Serie HS1GW | Zigbee + Wi-Fi/Ethernet | Gateway di base |
| Serie HS3GW | Zigbee + Wi-Fi/Ethernet | Migliorato |
| Serie HS5GW | Zigbee + Wi-Fi/Ethernet | Gateway ad alte prestazioni |
| Serie HS6GW | Zigbee + Wi-Fi/Ethernet + Filo | L'ultimo modello supporta sia Zigbee che Matter over Thread |

## Funzioni principali

- Coordinatore della rete Zigbee
- Supporta Wi-Fi/Ethernet per la connessione a Internet
- Rilevamento e associazione automatici del dispositivo
- Collegamento alla scena locale (alcuni modelli)
- Accesso remoto (tramite Heiman Cloud Platform)
- Supporta fino a 128 sotto-dispositivi (a seconda del modello)

## Scenari applicabili

- Il punto di connessione principale dell'intelligenza di tutta la casa Heiman
- Collabora con l'APP Haiman per realizzare il monitoraggio remoto
- Gestione centralizzata locale di progetti commerciali di piccole e medie dimensioni

## Differenze rispetto ad altri gateway

Haiman Gateway è l'ingresso all'ecosistema ufficiale. Per gli utenti che desiderano utilizzare una piattaforma di terze parti, si consiglia di utilizzare il coordinatore universale Zigbee con Zigbee2MQTT/ZHA.
