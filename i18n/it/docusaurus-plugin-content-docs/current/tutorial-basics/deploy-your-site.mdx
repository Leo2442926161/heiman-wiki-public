# Distribuzione delle Soluzioni

Heiman Technology fornisce guide end-to-end per la distribuzione di soluzioni di sicurezza intelligente.

## Processo di Integrazione

I prodotti Heiman supportano molteplici metodi di distribuzione, inclusa la distribuzione locale autonoma e l'integrazione con piattaforme cloud.

### Distribuzione Locale

```bash
# Configura gateway antincendio intelligente Heiman
heiman-cli configure --gateway HS1GW --mode local

# Registra dispositivo rivelatore
heiman-cli register --device HS1SA --gateway HS1GW
```

### Integrazione Piattaforma Cloud

I prodotti Heiman sono integrati con le principali piattaforme IoT:

- China Telecom Tianyi
- China Mobile OneNET
- Huawei HiLink
- Alibaba Cloud IoT
- Tuya Smart

## Supporto Protocollo Matter

I prodotti Heiman serie Matter sono compatibili con:

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

Garantendo interconnettività cross-brand e cross-ecosistema.

## Verifica della Distribuzione

Dopo la distribuzione, utilizza l'APP Heiman Smart Fire o la piattaforma di gestione per la verifica funzionale e il monitoraggio in tempo reale.
