# Funzionalità Markdown

La documentazione tecnica Heiman supporta completamente la sintassi Markdown e le funzionalità estese, semplificando la scrittura di documentazione strutturata sui prodotti.

## Esempio di Codice

Esempio di integrazione API per lo sviluppo prodotto:

```jsx title="Integrazione API Rivelatore Fumo"
const heimanClient = new HeimanClient({
  apiKey: 'your-api-key',
  deviceId: 'HS1SA-xxxxx',
});

// Ottieni stato dispositivo
const status = await heimanClient.getDeviceStatus();
console.log(status);
```

## Avvisi

Usa gli avvisi per evidenziare informazioni importanti:

:::tip

Il rivelatore di fumo Heiman HS1SA è certificato EN14604

:::

:::caution

Durante l'installazione dei rivelatori di gas, segui la guida all'installazione per un cablaggio corretto

:::

:::danger

In caso di perdita di gas, non azionare interruttori elettrici. Aprire immediatamente le finestre ed evacuare l'area

:::
