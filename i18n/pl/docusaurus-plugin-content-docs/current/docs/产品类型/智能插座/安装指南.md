# Przewodnik instalacji

## Wymagania dotyczące lokalizacji instalacji

- Podłącz bezpośrednio do gniazdka ściennego, **Nie podłączaj szeregowo** (Nie podłączaj do listwy lub przedłużacza)
- Upewnij się, że prąd znamionowy gniazda wynosi ≥ 16A
- Nie instaluj w środowiskach wilgotnych lub o wysokiej temperaturze

## Kroki instalacji

1. Podłącz inteligentne gniazdo bezpośrednio do gniazdka ściennego 220 V (lub 110 V).
2. Lampka kontrolna zapala się po podłączeniu zasilania.
3. Naciśnij i przytrzymaj przycisk przełącznika, aby przejść do trybu parowania
4. Dodaj urządzenie w aplikacji Heiman
5. Po pomyślnym sparowaniu możesz zdalnie sterować przełącznikiem

## Środki ostrożności

- **Maksymalne obciążenie**: 16A/3680W (norma europejska), 15A/1800W (norma amerykańska)
- **Nie przeciążaj**: Zwróć uwagę na ograniczenie mocy podczas korzystania z urządzeń elektrycznych dużej mocy (klimatyzatory, grzejniki)
- **NIE PODŁĄCZAJ SZEREGOWO**: Nie wkładaj inteligentnych gniazdek do listew zasilających
- Nie zakrywaj ani nie zasłaniaj obudowy gniazda
- Regularnie sprawdzaj wtyczkę, aby sprawdzić, czy się nie przegrzewa

## Typowe zastosowania

- Sterowanie czasowe urządzeń małej mocy, takich jak podgrzewacze wody, nawilżacze i oświetlenie akwariów
- Współpracuj z czujnikiem: światło włącza się, gdy ktoś przychodzi i gaśnie, gdy ktoś wychodzi.
- Statystyki mocy: monitoruj zużycie energii przez urządzenia gospodarstwa domowego
