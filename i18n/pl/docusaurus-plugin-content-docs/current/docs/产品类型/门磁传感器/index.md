# Czujnik drzwi

Czujnik drzwiowy służy do wykrywania stanu otwarcia i zamknięcia drzwi i okien i jest podstawowym elementem inteligentnego systemu bezpieczeństwa.

## Seria produktów

| Seria | Metody komunikacji | Funkcje |
|------|----------|------|
| Seria HS1DS | Zigbee / Z-Wave | Typ standardowy, 3-letnia żywotność baterii |
| HS1DS-M | Materia nad wątkiem | Wersja materii |

## Podstawowe funkcje

- Wykrywanie w czasie rzeczywistym stanu otwarcia/zamknięcia drzwi i okien
- Alarm sabotażowy (Tamper)
- Przypomnienie o niskim poziomie baterii
-Obsługuje łączenie (otwieranie drzwi, włączanie świateł, uzbrajanie przy wyjściu z domu itp.)
- Łatwy w montażu (taśma klejąca 3M lub mocowanie śrubowe)

## Parametry techniczne

- Sposób zasilania: bateria guzikowa CR1632
- Żywotność baterii: około 2-3 lata
- Temperatura pracy: -10°C ~ +50°C
- Luz montażowy: ≤10 mm

## Przykład scenariusza

- **Uzbroić poza domem**: Czujnik drzwi wykrywa otwarcie drzwi i uruchamia alarm.
- **Idź do domu**: Otwórz drzwi i automatycznie włącz światła
- **Monitorowanie okna**: Połączony ze stacją pogodową, automatycznie zamknie okno w przypadku deszczu.
