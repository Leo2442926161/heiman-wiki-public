# Czujnik podczerwieni ludzkiego ciała

Czujnik podczerwieni ciała ludzkiego (PIR) wykrywa ruch poprzez wykrywanie promieniowania cieplnego podczerwonego emitowanego przez ludzkie ciało i jest najczęściej stosowanym rozwiązaniem do wykrywania obecności w inteligentnych domach.

## Seria produktów

| Seria | Metody komunikacji | Funkcje |
|------|----------|------|
| HS1MS | Zigbee | Model podstawowy, kąt detekcji 110° |
| HS3MS | Zigbee | Ulepszony model, montaż sufitowy 360° |
| Seria M1 | Materia / Zigbee | Obsługuje materię nad nicią, opcjonalnie 110° lub 360° |
| HS8MLS | Zigbee | Model długodystansowy |
| HS8MIS | Zigbee | Ciało ludzkie + czujnik kompozytowy oświetlenia |

## Podstawowe funkcje

- Mobilne wykrywanie ludzi
- Wykrywanie oświetlenia (seria M1/HS8MIS)
- Odległość wykrywania ≤7m (25°C)
- Sposób montażu: montaż na ścianie lub na suficie
- Żywotność baterii: 3 lata (Zigbee) / 1 rok (Materia)

## Parametry techniczne (seria M1)

- Kąt detekcji: 110° / 360°
- Odległość wykrywania: ≤7m
- Zakres natężenia oświetlenia: 0 ~ 1200 luksów
- Zasilanie: DC 3V (bateria litowa)
- Żywotność baterii: 3 lata (Zigbee) / 1 rok (Materia)

## Przykład scenariusza

- Światło włącza się, gdy ktoś przychodzi i gaśnie, gdy ktoś wychodzi
- Połączenie bezpieczeństwa: alarm push w przypadku wykrycia nieprawidłowego ruchu
- Uzbrajanie po wyjściu z domu: automatycznie wyłącza klimatyzację/ogrzewanie, aby oszczędzać energię, gdy nikogo nie ma w pobliżu
