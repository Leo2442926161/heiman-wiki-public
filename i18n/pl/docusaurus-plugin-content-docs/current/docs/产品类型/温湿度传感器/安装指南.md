# 安装指南

## 安装位置要求

- 安装在 **1.2 ~ 1.5 m** 高度（代表环境平均水平）
- **远离**：阳光直射、门窗、空调出风口、暖气片
- **远离**：加湿器、厨房灶台等局部温湿度变化源
- 选择经常活动的房间（卧室、客厅、婴儿房）

## 安装步骤

1. 选定安装位置（墙壁或桌面均可）
2. 壁挂安装：用螺丝或 3M 胶固定底座
3. 装入 CR2 电池（注意极性）
4. 传感器自动开机，屏幕（如有）显示温湿度
5. 通过网关配对
6. 在 APP 中查看数据确认正常

## 注意事项

- 贴墙安装时确保传感器通气孔不被遮挡
- 电池寿命约 2-3 年
- 不同房间的温湿度数据可通过多个传感器采集
- 可用于联动空调、除湿机、加湿器等设备
