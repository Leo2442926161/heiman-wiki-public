# czujnik dymu

Czujnik dymu Hyman wykorzystuje **fotoelektryczną technologię wykrywania dymu**, która może z dużą czułością wykrywać widoczne cząsteczki dymu powstające we wczesnych stadiach pożaru i zapewniać bardzo wczesne ostrzeżenie o pożarze.

## Seria produktów

| Seria | Metody komunikacji | Standardy certyfikacji | Obowiązujące rynki |
|------|----------|----------|--------------|
| Seria HS1SA | Zigbee / Z-Wave | EN 14604 | Unia Europejska |
| Seria HS3SA | Zigbee | EN 14604 | Unia Europejska |
| HS8SA | Zigbee / Wi-Fi | UL217 | Stany Zjednoczone |
| Seria S1 | Wolnostojący Sub-1G | EN 14604 | Cel ogólny |
| HS1SA-M | Materia nad wątkiem | EN 14604 | Globalny |

## Podstawowe funkcje

- Fotoelektryczna detekcja dymu do wykrywania dymu z powolnego spalania i otwartego płomienia
- Alarm dźwiękowy i świetlny o głośności 85 dB
- Przypomnienie o niskim poziomie baterii
- Funkcja autotestu (test jednym kliknięciem)
- Alarm antysabotażowy
-Wsparcie zdalnego wyciszenia

## Parametry techniczne (na przykładzie HS1SA-M)

- Typ czujnika: fotoelektryczny na podczerwień
- Ciśnienie akustyczne alarmu: ≥85 dB (3 metry)
- Zasilanie: bateria CR123A
- Żywotność baterii: ok. 3 lata
- Temperatura pracy: -10°C ~ +50°C
- Wilgotność robocza: ≤95% RH
