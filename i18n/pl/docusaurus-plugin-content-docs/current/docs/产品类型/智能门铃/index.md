# inteligentny dzwonek do drzwi

Inteligentny dzwonek do drzwi Haiman integruje funkcje dzwonienia do drzwi i monitorowania bezpieczeństwa oraz obsługuje zdalny wideodomofon.

## Podstawowe funkcje

- Spust przycisku dzwonka
- Zdalne wideo na żywo (wersja Wi-Fi)
- Dwukierunkowe połączenia głosowe
- Film z detekcją ruchu
- Funkcja noktowizora

## Obowiązujące scenariusze

- Połączenia gości: możesz je odbierać w domu lub poza nim
- Powiadomienie o dostawie ekspresowej/na wynos: zdalne potwierdzenie otwarcia drzwi
- Monitoring bezpieczeństwa: rejestruj podejrzane działania przed drzwiami

> **Uwaga**: Linia dzwonków do drzwi Heiman opiera się głównie na rozwiązaniach Zigbee/Wi-Fi. Aby uzyskać szczegółowe informacje na temat parametrów modelu, skonsultuj się z urzędnikiem.
