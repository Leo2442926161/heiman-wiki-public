# Porównanie modeli

## Porównanie różnych modeli

| Modelka | Protokół | Wideo | Zasilanie | Funkcje | Sugestie dotyczące rozgrywki |
|------|------|------|------|------|---------|
| Inteligentny dzwonek do drzwi | Zigbee + Wi-Fi | 1080P opcjonalnie | Bateria/sieć | Dzwonek + zabezpieczenie dwa w jednym | Powiadomienie o dzwonieniu gościa na telefon komórkowy, zdalny domofon wizualny, nagrywanie po wykryciu ruchu |

## Zalecana scena rozgrywki

| Scenariusze | Polecane modele | Rozwiązania łączące |
|------|----------|---------|
| Zarządzanie dostawami ekspresowymi/na wynos | Inteligentny dzwonek do drzwi | Zadzwoń dzwonkiem → powiadomienie z telefonu komórkowego → zdalne potwierdzenie domofonem → zdalne odblokowanie (połączony inteligentny zamek) |
| Monitorowanie bezpieczeństwa | Inteligentny dzwonek do drzwi | Wykrywanie otwarcia drzwi → Wideo + Alarm Push, połączony czujnik drzwi/oświetlenie |
| Powiązanie gości | Inteligentny dzwonek do drzwi | Zadzwoń → Ekran wewnętrzny/wyświetlacz głośnika → Włącz światła → Odblokuj połączenie |
