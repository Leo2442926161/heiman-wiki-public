# Przewodnik instalacji

## Wymagania dotyczące lokalizacji instalacji

- Instalowany na powierzchni obiektów wymagających monitorowania wibracji
- Typowe miejsca instalacji: **wewnętrzne drzwi i okna**, **sejfy**, **rury**, **ściany**, **ogrodzenia**
- Upewnij się, że powierzchnia instalacji jest płaska i czysta

## Kroki instalacji

1. Wybierz obiekt, który chcesz monitorować (np. wnętrze drzwi sejfu)
2. Oczyść powierzchnię montażową
3. Do zamocowania czujnika użyj taśmy samoprzylepnej lub śrub 3M
4. Zainstaluj baterię CR2450
5. Parowanie przez bramkę
6. Test symulacyjny: dotknij powierzchni montażowej, czujnik powinien wywołać alarm

## Notatki

- W przypadku zabezpieczeń przed wyważeniem drzwi i okien skuteczniej jest zainstalować je po wewnętrznej stronie drzwi/okna, w pobliżu zamka.
- W celu monitorowania rurociągu przykleić bezpośrednio na powierzchnię rurociągu
- Żywotność baterii około 2 lat
- Regulowana czułość dostosowana do różnych scenariuszy
