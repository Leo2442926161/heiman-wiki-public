# Czujnik wibracji

Czujnik wibracji uruchamia alarm po wykryciu wibracji lub przechylenia obiektu. Nadaje się do różnych scenariuszy, takich jak drzwi, okna, sejfy i rury.

## Podstawowe funkcje

- Trójwymiarowe wykrywanie wibracji
- Wykrywanie kąta pochylenia
- Przypomnienie o niskim poziomie baterii

## Obowiązujące scenariusze

- **Zabezpieczenie przed podważeniem drzwi i okien**: Wykrywa wibracje spowodowane gwałtownym włamaniem
- **Monitor sejfu/szuflady**: Wykrywa nieprawidłowy ruch
- **Monitorowanie rurociągu**: Wykrywa wibracje rury wodnej (wskazujące nieprawidłowości w pracy pompy wodnej lub uderzenia wodne)
- **Monitorowanie ścian/płotów**: wykrywanie wspinaczki lub uderzenia

## Parametry techniczne

- Typ czujnika: 3-osiowy akcelerometr
- Wymiary detekcji: wibracje + przechylenie
- Sposób zasilania: bateria guzikowa CR2450
- Żywotność baterii: ok. 2 lata
