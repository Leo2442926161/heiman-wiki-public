# Porównanie modeli

## Porównanie różnych modeli

| Modelka | Protokół | Czujnik | Zasilanie | Funkcje | Propozycje gier |
|------|------|-------|------|------|-------------|
| Czujnik wibracji | Zigbee | Akcelerometr 3-osiowy | CR2450 | Wykrywanie wibracji i pochylenia | Zabezpieczenie przed podważeniem drzwi i okien, alarm wibracyjny przed nielegalnym otwarciem sejfu/szuflady, monitorowanie nieprawidłowych wibracji rur |

## Zalecana scena rozgrywki

| Scenariusze | Polecane modele | Rozwiązania łączące |
|------|----------|---------|
| Zabezpieczenie przed podważeniem drzwi i okien | Czujnik wibracji | Wykrywanie gwałtownego uderzenia i wibracji drzwi/okna → alarm push + alarm dźwiękowy i świetlny + przechwytywanie kamery |
| Bezpieczne monitorowanie | Czujnik wibracji | Sejf zostanie poruszony lub wibrowany → Natychmiastowe powiadomienie o alarmie |
| Monitorowanie rurociągu | Czujnik wibracji | Monitoruj nieprawidłowe wibracje rur wodociągowych (awaria pompy wodnej/uderzenie wodne), zapewniaj wczesne ostrzeganie w celu konserwacji |
| Płot/Ściana | Czujnik wibracji | Wykrywanie wspinania się po ścianach/ogrodzeniu, odpowiednie do zabezpieczenia podwórza |
