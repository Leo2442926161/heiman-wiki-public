# czujnik jakości powietrza

Czujniki jakości powietrza (monitor jakości powietrza) służą do wykrywania różnych substancji zanieczyszczających powietrze w pomieszczeniach, aby pomóc w utrzymaniu zdrowego środowiska życia.

## Parametry wykrywania

Czujniki jakości powietrza Hyman wykrywają następujące wskaźniki:

| Parametry | Opis | Standardy zdrowotne |
|------|------|--------------|
| PM2,5 | Drobne cząstki stałe | ≤ 35 µg/m3 (średnia 24-godzinna) |
| PM10 | Wdychane cząstki stałe | ≤ 50 µg/m3 (średnia 24-godzinna) |
| CO₂ | Dwutlenek węgla | ≤ 1000 ppm |
| TVOC | Ogółem lotne związki organiczne | ≤ 0,6 mg/m3 |
| Temperatura | Temperatura otoczenia | 18-26°C |
| Wilgotność | Wilgotność względna | 40-60% wilgotności względnej |

## Podstawowe funkcje

- Wieloparametrowa detekcja jakości powietrza
- Kolorowy wyświetlacz (w zależności od modelu)
- Alarm dźwiękowy i świetlny w przypadku przekroczenia normy
- Zapisy danych historycznych
- Połączony oczyszczacz świeżego powietrza/powietrza

## Obowiązujące scenariusze

- Sypialnia: Monitoruj poziom CO₂ i monituj o otwarcie okien w celu wentylacji
- Pokój dziecięcy: chroń zdrowie układu oddechowego dzieci
- Biuro: Zwiększ produktywność i komfort
- Nowo wyremontowane domy: monitoruj uwalnianie formaldehydu/TVOC
