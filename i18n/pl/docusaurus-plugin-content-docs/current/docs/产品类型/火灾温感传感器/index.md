# Czujnik temperatury ognia

Czujnik temperatury pożaru (czujnik ciepła) generuje alarm pożarowy po wykryciu nieprawidłowego wzrostu temperatury otoczenia. Nadaje się do scen takich jak kuchnie i garaże, gdzie czujniki dymu są podatne na fałszywe alarmy.

## Zasada działania

Czujnik temperatury wykorzystuje element termiczny do wykrywania zmian temperatury:

- **Alarm stałej temperatury**: uruchamia alarm, gdy temperatura przekroczy ustawiony próg (zwykle 57°C / 135°F)
- **Alarm różnicy temperatur**: wyzwala alarm, gdy tempo wzrostu temperatury przekracza ustawioną wartość (np. 8,3°C/min)

## Zakres zastosowania

- Kuchnia (czujniki dymu są podatne na fałszywe alarmy ze względu na gotowanie w tej scenie)
- Garaż/warsztat (spaliny i kurz mogą zakłócać działanie czujników dymu)
- Kotłownia/pomieszczenie urządzeń grzewczych
- Łazienka/sauna

## Podstawowe funkcje

- Wykrywanie nieprawidłowego wzrostu temperatury
- Alarm antysabotażowy
- Przypomnienie o niskim poziomie baterii

> **Uwaga**: Czujnik temperatury i czujnik dymu są stosowane komplementarnie. Czujnik dymu reaguje szybciej, a czujnik temperatury ma silniejsze właściwości przeciwzakłóceniowe.
