# Czujnik wykrywania zanurzenia w wodzie

Czujnik wycieku wody służy do wykrywania wycieków lub gromadzenia się wody, co może skutecznie zapobiegać stratom majątkowym spowodowanym pęknięciem rur wodociągowych i zalaniem wodą.

## Seria produktów

| Seria | Metody komunikacji | Funkcje |
|------|----------|------|
| HS1WL | Zigbee | Model podstawowy, wykrywanie sondy |
| Seria HS2WL | Zigbee | Sonda + wykrywanie kabla, dwukanałowy |
| Seria HS3WL | Zigbee | Wersja ulepszona |
| Seria L1 | Materia / Zigbee | Wersja Matter over Thread, z certyfikatem WWHA |

## Podstawowe funkcje

- Wykrywanie w czasie rzeczywistym wycieków wody/stojącej wody
- Alarm dźwiękowy
- Przypomnienie o niskim poziomie baterii
- Konstrukcja antykorozyjna sondy

## Miejsce instalacji

- Pod zlewem w łazience
- Pod zlewem w kuchni
- Obok pralki/zmywarki
- Pod podgrzewaczem wody
- Piwnica/garaż
- W pobliżu grzejnika

## Połączenie scen

- **Wykrywanie wycieków wody + zawór elektromagnetyczny**: Automatycznie zamyka główny zawór w przypadku wykrycia wycieku wody
- **Alarm wycieku wody + Push**: Powiadom nieruchomość lub członków rodziny na telefonie komórkowym, gdy nie ma Cię w domu
- **Zalanie + Inteligentne odcięcie**: Połączony gazowy podgrzewacz wody lub inteligentny zawór wody
