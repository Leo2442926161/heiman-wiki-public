# Inteligentna brama

Smart Gateway to rdzeń systemu inteligentnego domu Haiman, odpowiedzialny za łączenie różnych czujników Zigbee i realizację kontroli połączeń między urządzeniami.

## Seria produktów

| Seria | Metody komunikacji | Funkcje |
|------|----------|------|
| Seria HS1GW | Zigbee + Wi-Fi / Ethernet | Podstawowa brama |
| Seria HS3GW | Zigbee + Wi-Fi / Ethernet | Ulepszone |
| Seria HS5GW | Zigbee + Wi-Fi / Ethernet | Brama o wysokiej wydajności |
| Seria HS6GW | Zigbee + Wi-Fi / Ethernet + wątek | Najnowszy model obsługuje zarówno Zigbee, jak i Matter over Thread |

## Podstawowe funkcje

- Koordynator sieci Zigbee
- Obsługa Wi-Fi / Ethernet, aby połączyć się z Internetem
- Automatyczne wykrywanie i parowanie urządzeń
- Połączenie scen lokalnych (niektóre modele)
- Dostęp zdalny (poprzez platformę chmurową Heiman)
- Obsługuje do 128 urządzeń podrzędnych (w zależności od modelu)

## Obowiązujące scenariusze

- Podstawowy punkt połączenia inteligencji całego domu Heiman
- Współpracuj z aplikacją Haiman, aby zrealizować zdalne monitorowanie
- Lokalne, scentralizowane zarządzanie małymi i średnimi projektami komercyjnymi

## Różnice w stosunku do innych bramek

Haiman Gateway to wejście do oficjalnego ekosystemu. Użytkownikom, którzy chcą korzystać z platformy innej firmy, zaleca się użycie uniwersalnego koordynatora Zigbee z Zigbee2MQTT/ZHA.
