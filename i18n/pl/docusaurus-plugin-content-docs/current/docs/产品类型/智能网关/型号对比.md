# Porównanie modeli

## Porównanie różnych modeli

| Modelka | Maksymalny limit urządzenia podrzędnego Zigbee | Metoda sieciowa | Funkcje | Propozycje gier |
|------|--------|----------|------|--------------|
| [HS6GW](../../Protokół bezprzewodowy/Zigbee/HS6GW - Inteligentna bramka (Zigbee)) | 128 | Wi-Fi + Ethernet | Najnowszy model i obsługuje przyszłe aktualizacje wątków | Własny rdzeń ekologiczny Haiman w połączeniu ze zdalnym zarządzaniem aplikacją Haiman, odpowiedni dla użytkowników, którym to nie przeszkadza |
| HS5GW | 64 | Wi-Fi + Ethernet | Wysoka wydajność | Średnie i duże projekty, stabilne i niezawodne |
| HS3GW | 32 | Wi-Fi | Ulepszone | Wystarczający dla małych i średnich domów |
| HS1GW | 16 | Wi-Fi | Podstawowy | Poziom podstawowy, odpowiedni dla niewielkiej liczby urządzeń |

## Zalecana scena rozgrywki

| Scenariusz | Polecany model | Opis |
|------|----------|------|
| Ekologia Heimana w całym domu | HS6GW | Używany z czujnikami Heiman Zigbee i zdalnym zarządzaniem za pomocą aplikacji Heiman, nie ma potrzeby budowania własnego serwera |
| Użytkownicy platform zewnętrznych | — | Zaleca się zakup koordynatora Zigbee (takiego jak SLZB-06) bezpośrednio do użytku z Zigbee2MQTT/ZHA w celu uzyskania większej elastyczności |
| Małe i średnie mieszkania | HS3GW | Limit 32 urządzeń zaspokajający podstawowe potrzeby |

## Sugestie dotyczące wyboru

- **Własna ekologia Haimana**: [HS6GW](../../Wireless Protocol/Zigbee/HS6GW - Smart Gateway (Zigbee)) (128 urządzeń, WiFi + Ethernet)
- **Platforma strony trzeciej**: Zalecany uniwersalny koordynator Zigbee + Zigbee2MQTT/ZHA
