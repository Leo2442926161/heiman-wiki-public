# Inne certyfikaty

## Znak jakości Q-Label

Q-Label to znak certyfikacji jakości dla europejskiego przemysłu inteligentnych domów, wprowadzony przez Q-Lab Alliance (obecnie część Zigbee Alliance / CSA Connectivity Standards Alliance).

| Projekt | Opis |
|------|------|
| Zakres zastosowania | Certyfikacja interoperacyjności urządzeń Zigbee |
| Treść certyfikatu | Kompatybilność urządzenia z bramkami/kontrolerami Zigbee innych marek |
| Znaczenie | Zapewnij interoperacyjność między markami i zmniejsz ryzyko „kupowania czegoś w domu i nieużywania go” |

Produkty Hyman Zigbee posiadają certyfikat Q-Label, zapewniający kompatybilność z następującymi platformami:
- Amazon Echo Plus / Echo Show (wbudowany Zigbee)
- Mostek Philips Hue
- Centrum SmartThings
- Inne bramy uwierzytelniające Q-Label

## Nagroda Red Dot za projekt

Czujnik dymu serii HS1SA firmy Hyman zdobył nagrodę **Red Dot Design Award** (Red Dot Design Award), która odzwierciedla siłę firmy Hyman w projektowaniu produktów.

## System zarządzania jakością ISO 9001

Haiman przeszedł certyfikację systemu zarządzania jakością ISO 9001:2015, aby zapewnić pełną kontrolę jakości procesu, od badań i rozwoju, produkcji po sprzedaż.
