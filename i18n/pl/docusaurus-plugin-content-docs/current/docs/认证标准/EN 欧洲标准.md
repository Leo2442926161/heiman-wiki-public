# Norma europejska EN

EN (Norma Europejska, Norma Europejska) to norma techniczna opracowana przez Europejski Komitet Normalizacyjny (CEN) i Europejski Komitet Normalizacyjny Elektrotechniki (CENELEC). Główne normy EN dotyczące produktów Heiman są następujące:

## EN 14604 — Norma dotycząca czujników dymu

| Projekt | Opis |
|------|------|
| Zakres zastosowania | Niezależny czujnik dymu |
| Wymagania dotyczące wykrywania | Fotoelektryczna detekcja dymu, alarm dźwiękowy i świetlny, komunikat o niskim stanie baterii |
| Obowiązujące rynki | Państwa członkowskie UE, Wielka Brytania, Szwajcaria, Norwegia itp. |
| Produkty Heimana | Seria HS1SA-E, HS3SA, HS1SA-M, S1 itp. |
| Znaczenie certyfikacji | Spełnienie wymogów unijnych przepisów przeciwpożarowych budynków jest warunkiem koniecznym dostępu do rynku europejskiego |

**Elementy testowane**: czas reakcji, poziom ciśnienia akustycznego (≥85 dB/3m), funkcja wyciszenia, żywotność baterii, cykl temperaturowy, trwałość wilgotności, odporność na zakłócenia o częstotliwości radiowej itp.

## EN 50291 — Norma dotycząca alarmu tlenku węgla

| Projekt | Opis |
|------|------|
| Zakres zastosowania | Alarm dotyczący tlenku węgla (CO) w gospodarstwie domowym |
| Wymagania testowe | Elektrochemiczny czujnik CO, alarm klasyfikacji stężeń |
| Obowiązujące rynki | Państwa członkowskie UE, Wielka Brytania |
| Produkty Heimana | HS1CA-E, seria C1, HS1CA-M itp. |
| Znaczenie certyfikacji | Aby zapewnić niezawodność wykrywania wycieków CO, w niektórych krajach instalacja jest obowiązkowa
