# Laboratoria Underwriters UL

UL (Underwriters Laboratories) to najbardziej autorytatywna organizacja opracowująca i certyfikująca standardy bezpieczeństwa produktów w Stanach Zjednoczonych. Główne standardy UL dotyczące produktów Heiman są następujące:

## UL 217 — Standard czujnika dymu (USA)

| Projekt | Opis |
|------|------|
| Zakres zastosowania | Mieszkaniowe czujniki dymu (rynek amerykański) |
| Wymagania dotyczące wykrywania | Fotoelektryczna/jonizacyjna detekcja dymu, alarm dźwiękowy i świetlny, powyżej 85 dB |
| Obowiązujące rynki | Stany Zjednoczone, Kanada |
| Produkty Heimana | HS8SA (wersja amerykańska) |
| Znaczenie certyfikacji | Spełnienie przepisów przeciwpożarowych różnych stanów USA jest warunkiem koniecznym dostępu do rynku w Ameryce Północnej |

**Różnice w stosunku do innych norm**: Scenariusze testowe normy UL 217 obejmują typowe amerykańskie scenariusze pożarów domowych, takie jak pożary błyskawiczne i pożary wolno palące się, i nieznacznie różnią się od metod testowych normy EN 14604.

## UL 2034 — Norma alarmowa dotycząca tlenku węgla (USA)

| Projekt | Opis |
|------|------|
| Zakres zastosowania | Alarm CO dla gospodarstw domowych (rynek amerykański) |
| Wymagania dotyczące wykrywania | Czujnik elektrochemiczny, algorytm całkowania stężenia × czasu |
| Obowiązujące rynki | Stany Zjednoczone, Kanada |
| Produkty Haymana | HS-720ES-M (wersja amerykańska) |
| Znaczenie certyfikacji | Większość stanów w Stanach Zjednoczonych nakazuje instalację alarmów CO |

## Główne różnice między certyfikatem UL a certyfikatem EN

| Aspekty | UŁ | PL |
|------|-----|-----|
| Normy dotyczące czujników dymu | UL217 | EN 14604 |
| Normy alarmów CO | UL 2034 | EN 50291 |
| Metoda wykrywania | Specyficzny test krzywej pożaru | Różne badania typu ogniowego |
| Obowiązujące napięcie | 120V / akumulator | 230V / akumulator |
