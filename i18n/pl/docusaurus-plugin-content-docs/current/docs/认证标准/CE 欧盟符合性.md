# Zgodność CE UE

Znak CE (Conformité Européenne, EU Conformity) jest obowiązkowym znakiem bezpieczeństwa produktów wprowadzanych na rynek UE.

## Znaczenie znaku CE

Znak CE wskazuje, że produkt spełnia podstawowe wymagania odpowiednich dyrektyw UE i może być swobodnie rozprowadzany na terenie krajów członkowskich UE oraz Europejskiego Obszaru Gospodarczego (EOG).

## Ścieżka certyfikacji CE dla produktów Haiman

Czujniki dymu/CO firmy Hyman spełniają wymagania CE dzięki:

### Rozporządzenie CPR (UE nr 305/2011)

Rozporządzenie w sprawie wyrobów budowlanych wymaga, aby produkty sygnalizacji pożaru poddano ocenie stabilności działania:
- Według norm: EN 14604 (dym), EN 50291 (CO)
- Agencja oceniająca: TÜV Rheinland i inne agencje notyfikowane
- Poziom certyfikacji: DoP (Deklaracja Właściwości Użytkowych) + znak CE

### Dyrektywa RED (2014/53/UE)

Dyrektywa w sprawie sprzętu radiowego wymaga, aby sprzęt bezprzewodowy:
- Obejmuje Zigbee, Z-Wave, Wi-Fi, Thread i inne moduły bezprzewodowe
- Elementy testowe: kompatybilność elektromagnetyczna (EMC), częstotliwość radiowa (RF), SAR, bezpieczeństwo

### Dyrektywa EMC (2014/30/UE)

Dyrektywa w sprawie kompatybilności elektromagnetycznej zapewnia, że sprzęt nie powoduje zakłóceń w działaniu innego sprzętu elektronicznego.

## Produkty oznaczone znakiem CE

Wszystkie produkty firmy Heymann sprzedawane na rynek UE posiadają znak CE, m.in.:
- HS1SA-E / HS3SA (czujnik dymu Zigbee)
- Seria HS1CA-E/C1 (alarm Zigbee CO)
- HS1SA-M/HS1CA-M (Materia nad gwintem)
- Pełna gama czujników Zigbee, takich jak HS1MS / HS1DS
