# Wskazówki dotyczące użytkowania

## Konfiguracja rzeczy

Dodaj urządzenie Heiman poprzez powiązanie MQTT:

```Java
Thing mqtt:topic:heiman_smoke "Heiman smoke alarm" @ "living room" {
    Kanały:
        Type binary: smoke "smoke state" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.smoke" ]
        Type number : battery "Battery power" [ stateTopic="zigbee2mqtt/heiman_smoke", transformationPattern="JSONPATH:$.battery" ]
}
```

## Definicja elementu

```Java
Contact Smoke_Alarm "Smoke alarm [MAP(smoke.map):%s]" { channel="mqtt:topic:heiman_smoke:smoke" }
Number Smoke_Battery "Smoke alarm battery [%d%%]" { channel="mqtt:topic:heiman_smoke:battery" }
```

## Wybór współrzędnych

It is recommended to use a USB coordinator with **CC2531** or **CC2652** chip and ensure that the firmware version is the latest.

## Wskazówki dotyczące debugowania

- View Thing’s `online/offline` status in Paper UI
- Use `openhab-cli info` to check if the bindings are loaded correctly
- Search the `heiman` keyword in the log to troubleshoot device communication issues
