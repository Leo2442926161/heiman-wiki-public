# Przykład automatyzacji

## Zasady czujnika dymu

```javascript
reguła „Włączył się alarm dymowy”
kiedy
    Pozycja heiman_smoke_alarm zmieniona na WŁ
wtedy
    sendCommand(Livingroom_Light, WŁ.)
    sendNotification("admin", "⚠️Włączył się czujnik dymu!")
    postUpdate(Alarm_State, "alarm_dymu")
koniec
```

## Zasady dotyczące czujników temperatury

```javascript
reguła „Raport temperatury i wilgotności”
kiedy
    Otrzymano aktualizację pozycji heiman_temperature
wtedy
    val temp = (heiman_temperature.state jako liczba).floatValue
    val hum = (heiman_humidity.state jako liczba).floatValue
    if (temp > 40) {
        sendCommand(Klimatyzator, WŁ.)
        sendNotification("admin", "Temperatura jest za wysoka (" + temp + "°C), klimatyzator został włączony")
    }
    if (buczenie > 80) {
        sendNotification("admin", "Wilgotność jest zbyt wysoka (" + buczenie + "%), proszę zwrócić uwagę na osuszanie")
    }
koniec
```
