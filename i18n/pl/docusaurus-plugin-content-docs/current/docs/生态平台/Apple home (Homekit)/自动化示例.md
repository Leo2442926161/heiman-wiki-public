# Przykład automatyzacji

## Alarm bezpieczeństwa wykrywający dym

**Automatyczna konfiguracja HomeKit:**

1. **Warunek zadziałania:** Alarm dymu Hyman → wykryto dym
2. **Ustawienia zabezpieczeń:** Przełącz status systemu bezpieczeństwa na „Alarm”
3. **Wykonaj akcję:**
   - HomePod → Odtwórz niestandardowy dźwięk „Wykryto dym, proszę zwrócić uwagę na bezpieczeństwo”
   - Wszystkie inteligentne światła → świecą i migają
   - Powiadomienia push do wszystkich członków rodziny

## Scena w domu/poza domem

**Polecenie skrótu: „Wyjdź z domu, kontrola bezpieczeństwa”**

```
kiedy ostatnia osoba wychodzi z domu
  Skonfiguruj system bezpieczeństwa → uzbroić przy wychodzeniu z domu
  Skonfiguruj czujnik drzwi Heiman → Włącz powiadomienia
  Skonfiguruj czujnik ciała Heimana → włącz alarm
  Wyłącz wszystkie światła
  Ustawienia Termostat → Tryb oszczędzania energii
  Wyślij powiadomienie „🏠 Włączono tryb poza domem”
```

## Przypomnienie o nieprawidłowościach w temperaturze

**Warunek zadziałania:** Czujnik temperatury i wilgotności Hyman → Temperatura > 35°C

**Wykonaj akcję:**
- HomePod → Temperatura w pomieszczeniu do transmisji głosowej przekracza limit
- Powiadomienia push na iPhone'a
- Automatycznie włącz klimatyzację (jeśli skonfigurowano HomeBridge)
