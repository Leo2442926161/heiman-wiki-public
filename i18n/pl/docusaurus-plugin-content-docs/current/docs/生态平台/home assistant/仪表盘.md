# Konfiguracja panelu

## Panel przeglądu zabezpieczeń

Użyj układu widoku przekroju, aby uporządkować sprzęt Heimann w obszarach:

```yaml
typ: sekcja
tytuł: Alarm dymu i CO
karty:
  - typ: podmiot
    jednostka: binary_sensor.heiman_smoke_alarm
    nazwa: czujnik dymu
  - typ: podmiot
    podmiot: binary_sensor.heiman_co_alarm
    nazwa: Alarm CO
```

## Styl karty grzybowej

```yaml
typ: niestandardowy: karta-panel-sterowania-alarm-grzyb
jednostka: panel_kontroli_alarmów.alarm_home
nazwa: system bezpieczeństwa

typ: niestandardowy:karta-obiektugrzybowego
jednostka: binary_sensor.heiman_smoke_alarm
nazwa: czujnik dymu
ikona: mdi:detektor dymu
podstawowe_info: stan
drugorzędne_info: ostatnia zmiana
fill_container: prawda
```

## Panel czujników środowiskowych

```yaml
typ: sekcja
tytuł: Monitoring Środowiska
karty:
  - typ: miernik
    obiekt: czujnik.heiman_temperatura
    nazwa: temperatura
    jednostka: „°C”
    min: 0
    maks.: 50
    dotkliwość:
      zielony: 0
      żółty: 30
      Czerwony: 40
  - typ: miernik
    obiekt: sensor.heiman_humidity
    nazwa: wilgotność
    jednostka: „%”
    min: 0
    maks.: 100
    dotkliwość:
      zielony: 30
      Żółty: 60
      Czerwony: 80
```

## Panel historii alarmów

```yaml
typ: wykres historii
tytuł: Historia czujnika
podmioty:
  - binary_sensor.heiman_smoke_alarm
  - binary_sensor.heiman_co_alarm
  - binary_sensor.heiman_water_leak
godziny_do_pokazu: 24
```

## Sugerowany układ

- Góra: Stan systemu alarmowego + karta stanu urządzenia alarmowego
- Centralny: Monitoring środowiska (temperatura, wilgotność, jakość wody)
- Dół: Wykres trendu historycznego alarmów czujnika
- Operacje skrótów na pasku bocznym: uzbrajanie/rozbrajanie, przełączanie scen
