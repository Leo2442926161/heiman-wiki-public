# Wskazówki dotyczące użytkowania

## Konfiguracja urządzenia

- Podczas pierwszego parowania urządzenia Heiman Matter upewnij się, że router graniczny (np. HomePod, Nest Hub, Home Assistant Green) znajduje się w tej samej sieci Thread
- W przypadku integracji urządzeń Zigbee poprzez ZHA lub Zigbee2MQTT zaleca się w pierwszej kolejności skorzystać z zalecanego oprogramowania koordynatora USB (takiego jak Sonoff ZBDongle-E, Conbee II)

## Automatyczna optymalizacja

- Dodano parametr `for:` (np. 3 sekundy) do automatyki czujników dymu, aby zapobiec natychmiastowym fałszywym alarmom
- Użyj „warunku: stan”, aby dopasować tryb bezpieczeństwa, aby uniknąć niepotrzebnych powiadomień w stanie uzbrojonym
- Użyj funkcji „grupowania” w Home Assistant, aby pogrupować czujniki Heimana według obszaru, aby ułatwić obsługę wsadową

## Rozwiązywanie problemów

- Gdy urządzenie jest w trybie offline, najpierw sprawdź, czy kanał sieciowy Zigbee nie koliduje z Wi-Fi 2,4 GHz
- Jeśli nie można wykryć urządzenia Matter, zrestartuj router graniczny i ponownie dodaj urządzenie
