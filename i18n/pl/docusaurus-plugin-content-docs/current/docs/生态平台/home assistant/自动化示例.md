# Przykład automatyzacji

## Czujnik dymu włącza światła w całym domu

```yaml
alias: "Czujnik dymu - światła i powiadomienia w całym domu"
wyzwalacze:
  - wyzwalacz: stan
    Entity_id: binary_sensor.heiman_smoke_alarm
    do: „włączony”
działania:
  - akcja: światło.włącz_włącz
    cel:
      identyfikator_encji: wszystko
  - akcja: notify.mobile_app_phone
    dane:
      wiadomość: „⚠️ Uruchomił się czujnik dymu! Sprawdź natychmiast!”
      tytuł: "Czujnik dymu"
```

## Połączenie alarmu CO

```yaml
alias: "Alarm CO - zakręć gaz i otwórz okna"
wyzwalacze:
  - wyzwalacz: stan
    Entity_id: binary_sensor.heiman_co_alarm
    do: „włączony”
działania:
  - akcja: switch.turn_off
    identyfikator_encji: przełącznik.kominek_gazowy
  - akcja: pokrywa.otwórz
    Entity_id: cover.living_room_window
  - akcja: notify.persistent_notification
    dane:
      komunikat: „Włączył się alarm czadu, zawór gazowy został zamknięty, a okna zostały otwarte”
```

## Łącznik czujnika zanurzenia w wodzie

```yaml
alias: „Wykrywanie zalania – zamknij zawór wody”
wyzwalacze:
  - wyzwalacz: stan
    Entity_id: binary_sensor.heiman_water_leak
    do: „włączony”
działania:
  - akcja: switch.turn_off
    identyfikator_encji: przełącznik.main_water_valve
  - akcja: notify.mobile_app_phone
    dane:
      komunikat: „💧 Wykryto wyciek wody! Główny zawór wody został zamknięty!”
```

## Uzbrojenie poza domem - czujnik obecności człowieka uruchamia alarm

```yaml
alias: „Tryb wyjścia z domu – alarm z czujnika obecności człowieka”
wyzwalacze:
  - wyzwalacz: stan
    Entity_id: binary_sensor.heiman_motion_sensor
    do: „włączony”
warunki:
  - stan: stan
    Entity_id: alarm_control_panel.home_alarm
    stan: uzbrojony
działania:
  - akcja: panel_kontroli_alarmów.trigger
    Entity_id: alarm_control_panel.home_alarm
```
