# Przykład automatyzacji

## Reguła Alarmu dymu maszyny

**Nazwa reguły:** Połączenie bezpieczeństwa czujnika dymu

**Konfiguracja:**
```
Wyzwalacz: Alarm dymu Hyman staje się aktywny
Warunki: Tryb != Rozbrojony
Działanie:
  - Włącz wszystkie światła
  - Wyślij powiadomienie push „⚠️ Czujnik dymu!”
  - Włącz syrenę (jeśli jest dostępna)
  - Ustaw tryb na „Awaryjny”
```

## Reguła Monitorowanie temperatury maszyny

**Nazwa reguły:** Ochrona przed wysoką temperaturą

**Konfiguracja:**
```
Wyzwalacz: czujnik temperatury Heimanna > 38°C
Warunek: Stan klimatyzatora = wyłączony
Działanie:
  - Włącz klimatyzację, tryb chłodzenia, 24°C
  - Wyślij powiadomienie „Temperatura w pomieszczeniu jest za wysoka i klimatyzator został uruchomiony”
```

## Reguła Wykrywanie wycieków wody z urządzenia

**Nazwa reguły:** Doraźne leczenie wycieku wody

**Konfiguracja:**
```
Wyzwalacz: Detektor zanurzeniowy w wodzie Hyman zostaje zamoczony
Stan: Stan zaworu wody = otwarty
Działanie:
  - Zamknąć główny zawór wody
  - Wysyłaj powiadomienia push na wszystkie telefony komórkowe
  - Wyślij powiadomienie SMS (opcjonalnie)
```
