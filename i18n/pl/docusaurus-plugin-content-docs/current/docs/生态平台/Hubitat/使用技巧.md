# Wskazówki dotyczące użytkowania

## Niestandardowy sterownik

Urządzenia Heimann są zoptymalizowane pod kątem kompatybilności w Hubitacie z następującymi sterownikami:

| Typ urządzenia | Zalecany sterownik |
|---------|----------|
| Alarm dymu/CO | Ogólny czujnik dymu/CO Zigbee |
| Czujnik podczerwieni ludzkiego ciała | Czujnik ruchu Heiman (firma trzecia) |
| Czujnik magnetyczny drzwi | Ogólny czujnik kontaktowy Zigbee |
| Czujnik wycieku wody | Ogólny czujnik wycieku wody Zigbee |
| Czujnik temperatury i wilgotności | Czujnik temperatury/wilgotności Heimana |

## Optymalizacja Zigbee

- Modele Hubitat C-8 i nowsze wykorzystują układ EFR32, który ma najlepszą kompatybilność
- Wyświetl i wybierz kanał z najmniejszymi zakłóceniami w Ustawieniach → Zigbee
- Użyj narzędzi do wykrywania zerwanych połączeń Zigbee, takich jak Ubisys

## Monit o regułę

- Zaleca się dodanie warunku „Opóźnienie fałszywego alarmu” (0,5-2 sekundy) w Rule Machine
- Włącz powiadomienia „kontroli stanu” dla kluczowych reguł, aby mieć pewność, że będą nadal działać
- Zorganizuj złożoną automatyzację w trybie „reguł podrzędnych”, aby ułatwić debugowanie i konserwację

## Współistnienie Z-Wave

Jeśli chcesz jednocześnie korzystać z urządzeń Z-Wave, pamiętaj o zachowaniu odstępu częstotliwości pomiędzy kanałami Zigbee i Z-Wave.
