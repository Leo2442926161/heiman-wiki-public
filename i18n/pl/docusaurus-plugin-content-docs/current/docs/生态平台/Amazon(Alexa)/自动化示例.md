# Przykład automatyzacji

## Integracja z Alexa Guard (USA)

**Warunek wstępny:** Subskrybuj usługę Alexa Guard Plus (dostępną w USA)

**Proces konfiguracji:**
1. Włącz Guard Plus w aplikacji Alexa
2. Dodaj czujnik dymu/CO firmy Hyman jako „inteligentne urządzenie domowe”
3. Włącz inteligentną reakcję „Wykrywanie alarmu dymu”.

**Automatyczna odpowiedź:**
- Uruchomił się czujnik dymu → Alexa automatycznie wybiera kontakty alarmowe
- Wyzwolenie alarmu CO → Ostrzeżenie głosowe Alexa dla całego domu, automatyczna wentylacja
- Drzwi i okna otwierają się nieprawidłowo → Powiadomienie o alarmie push

## Procedura wykrywania dymu

**Wyzwalacz:** Alarm dymu Hyman → Wykryto dym

**Wykonaj akcję:**
- Komunikat głosowy Alexa: „⚠️ Wykryto dym, proszę zwrócić uwagę na bezpieczeństwo”
- Wszystkie urządzenia Echo → Odtwórz dźwięk alarmu
- Inteligentny zestaw świateł → wszystko świeci
- Powiadomienia push na telefon komórkowy

## Procedura „wychodzenia z domu”.

**Wyzwalacz:** Polecenie głosowe „Alexa, wychodzę”

**Wykonaj akcję:**
- Czujnik magnetyczny drzwi Heiman → włącz uzbrojenie
- Czujnik ciała Heimana → Włącz wykrywanie ruchu
- Detektor zanurzenia w wodzie Hyman → włącz detekcję
- Wszystkie światła → wyłączone
- Alexa odpowiedziała: „System bezpieczeństwa jest włączony, bezpiecznej podróży”
