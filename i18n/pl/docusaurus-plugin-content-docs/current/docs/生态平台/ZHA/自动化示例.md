# Przykład automatyzacji

## Połączenie czujnika dymu

```yaml
alias: „Połączenie czujnika dymu ZHA”
wyzwalacze:
  - wyzwalacz: stan
    Entity_id: binary_sensor.heiman_smoke_zha
    do: „włączony”
działania:
  - akcja: światło.włącz_włącz
    cel:
      area_id: salon
  - akcja: tts.speak
    cel:
      Entity_id: media_player.speaker
    dane:
      komunikat: „Włączył się alarm dymowy!”
```

## Automatyzacja parowania urządzeń

```yaml
alias: "Przypomnienie o sparowaniu nowego urządzenia"
wyzwalacze:
  - wyzwalacz: zha.event
    podtyp: heiman
    polecenie: urządzenie_dołączone
działania:
  - akcja: notify.persistent_notification
    dane:
      komunikat: „Nowe urządzenie Haiman dołączyło do sieci ZHA, przejdź do nazwy konfiguracji i obszaru”
```

## Wykrywanie zanurzenia w wodzie

```yaml
alias: „Test powodziowy ZHA”
wyzwalacze:
  - wyzwalacz: stan
    Entity_id: binary_sensor.heiman_water_leak_zha
    do: „włączony”
działania:
  - akcja: switch.turn_off
    identyfikator_encji: przełącznik.main_water_valve
  - akcja: notify.mobile_app_phone
    dane:
      wiadomość: „💧 Wykryto powódź!”
```
