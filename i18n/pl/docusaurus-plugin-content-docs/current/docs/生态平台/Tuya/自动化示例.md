# Przykład automatyzacji

## Scena z alarmem dymnym

**Nazwa sceny:** „Połączenie z czujnikiem dymu”

**Warunek wyzwalania:** Alarm dymu Hyman → Wykryto dym

**Wykonaj akcję:**
- Inteligentne gniazdo → otwarte (podłącz wentylator wyciągowy)
- Światła → włączają się i migają
- Powiadomienia push na telefon komórkowy
- Głośnik → Odtwórz dźwięk alarmu

## Scena wychodzenia z domu jednym kliknięciem (Tap-to-Run)

**Nazwa sceny:** „Aranżacja po wyjściu z domu”

**Wykonywanie ręczne:** Dotknij lub aktywuj głosem

**Wykonaj akcję:**
- Czujnik drzwi Haiman → włącz alarm
- Hyman Human Body Infrared → Włącz wykrywanie
- Detektor zanurzenia w wodzie Hyman → włącz detekcję
- Wszystkie światła → wyłączone
- Inteligentny zamek → Blokada

## Automatyczna kontrola temperatury i wilgotności

**Typ automatyzacji:** Automatyczne wyzwalanie warunku

**Warunek wyzwalania:** Czujnik temperatury i wilgotności Hyman Wilgotność > 75%

**Wykonaj akcję:**
- Inteligentne gniazdo (osuszacz) → Włączone
- Powiadomienie push: „Za wysoka wilgotność, włączył się osuszacz”

**Stan zadziałania:** Czujnik temperatury i wilgotności Hyman Wilgotność < 50%

**Wykonaj akcję:**
- Inteligentne gniazdo (osuszacz) → Zamknij
