# Przykład automatyzacji

## Rutyna „Nie ma mnie”.

**Wyzwalacz:** Powiedz Google Home „Nie ma mnie”.

**Wykonaj akcję:**
- Czujnik magnetyczny drzwi Heiman → ramię
- Czujnik podczerwieni Heiman ludzkiego ciała → Włącz wykrywanie wyjścia z domu
- Wyłącz wszystkie światła i urządzenia
- Termostat → Ustawić temperaturę energooszczędną
- Google Nest Hub → Pokaż kartę potwierdzenia opuszczenia domu
- Odpowiedź głosowa: „Tryb wychodzenia z domu został włączony, bezpiecznej podróży”

## Transmisja „Wykrywanie dymu”.

**Wyzwalacz:** Alarm dymu Hyman → Wykryto dym

**Wykonaj akcję:**
- Głośnik Google Nest → Transmisja w całym domu „⚠️ Wykryto dym! Sprawdź natychmiast!”
- Wszystkie inteligentne światła → Włączone (poprzez zestaw inteligentnych świateł Google Home)
- Powiadomienia push na telefony komórkowe członków rodziny
- Kamera → Rozpocznij nagrywanie

## Tryb przed snem

**Wyzwalacz:** Powiedz „Dobranoc” Google Home

**Wykonaj akcję:**
- Czujnik drzwi Heiman → Włącz alarm otwartych drzwi
- Podczerwień ludzkiego ciała Heimana → włącz wykrywanie w nocy
- Światła → stopniowo wyłączają się
- Odpowiedź głosowa: „Dobranoc, system bezpieczeństwa jest włączony”
