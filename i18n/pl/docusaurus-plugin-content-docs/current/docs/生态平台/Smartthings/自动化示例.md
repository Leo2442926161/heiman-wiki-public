# Przykład automatyzacji

## Tryb „Dobranoc” na dobranoc

**Warunek wyzwalania:** Czujnik magnetyczny drzwi Haimana → Zamknij (wskazując, że drzwi sypialni zostały zamknięte)

**Wykonaj akcję:**
- Wyłącz wszystkie światła w salonie
- Inteligentny zamek → Blokada
- System bezpieczeństwa → uzbroić w nocy
- Powiadomienie push: „Dobranoc, system uzbrojony”

## Tryb „Poza domem”.

**Warunek wyzwalania:** Czujnik podczerwieni Hyman na ciało ludzkie → 15 minut bezruchu

**Wykonaj akcję:**
- Wszystkie światła wyłączone
- Termostat → Tryb oszczędzania energii
- System bezpieczeństwa → uzbrojenie poza domem
- Czujnik drzwi → włącz alarm
- Powiadomienie push: „Tryb poza domem jest aktywny, czujniki drzwi i okien są włączone”

## Tryb awaryjny czujnika dymu

**Warunek wyzwalania:** Alarm dymu Hyman → Wykryto dym

**Wykonaj akcję:**
- Inteligentny głośnik → odtwarzaj dźwięk alarmu
- Wszystkie inteligentne światła → migają na czerwono
- Powiadomienia push do wszystkich członków rodziny
- Jeśli podłączony do SmartThings Hub → włącz syrenę
