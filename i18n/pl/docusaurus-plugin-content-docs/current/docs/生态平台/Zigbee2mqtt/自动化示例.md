# Przykład automatyzacji

## Stan urządzenia z subskrypcją MQTT

,,bicie
subskrypcja_komara -h host lokalny -t "zigbee2mqtt/heiman_smoke/#" -v
```

Przykład wyjścia:
```
zigbee2mqtt/heiman_smoke {"dym": prawda, "bateria": 100, "jakość łącza": 120}
```

## Monitorowanie sprzętu online

```yaml
# konfiguracja.yaml
czujnik_binarny:
  - platforma: mqtt
    nazwa: "Czujnik dymu online"
    state_topic: "zigbee2mqtt/heiman_smoke/dostępność"
    payload_on: "online"
    payload_off: „offline”
    klasa_urządzenia: łączność
```
