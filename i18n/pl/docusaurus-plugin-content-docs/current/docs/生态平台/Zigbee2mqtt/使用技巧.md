# Wskazówki dotyczące użytkowania

## Wybór kanału

Urządzenia Hyman Zigbee domyślnie korzystają z kanału 15. Zalecane jest ręczne określenie w pliku `configuration.yaml`:

```yaml
zaawansowane:
  kanał: 15
```

Unikaj nakładania się kanałów Wi-Fi 2,4 GHz 1/6/11. Użyj analizatora Wi-Fi, aby przeskanować i wybrać kanał Zigbee z najmniejszymi zakłóceniami.

## Kopia zapasowa klucza sieciowego

**Pamiętaj o utworzeniu kopii zapasowej plików `coordinator_backup.txt` i `secret.yaml`: **

,,bicie
cp data/coordinator_backup.txt data/coordinator_backup.bak
cp dane/secret.yaml dane/secret.yaml.bak
```

Wszystkie sparowane urządzenia muszą się ponownie połączyć po utracie klucza.

## Optymalizacje specyficzne dla urządzenia

- Urządzenia Heiman domyślnie raportują stan baterii co 60 minut, a interwał można skrócić poprzez konfigurację `raportowania`
- Dodano konfigurację „homeassistant: true” w celu automatycznego generowania jednostek bez ręcznego zapisywania konfiguracji wykrywania MQTT
- Użyj `przyjaznej_nazwy`, aby ustawić czytelną dla człowieka nazwę dla urządzenia Heiman

## Zaktualizuj oprogramowanie sprzętowe

Niektóre urządzenia Heiman Zigbee obsługują aktualizacje OTA i opcja `ota` musi być włączona w `configuration.yaml`.
