#HS6GW - Inteligentna bramka domowa

## Przegląd

HS6GW to inteligentna bramka domowa firmy Heiman, obsługująca łącze WiFi + Ethernet, wbudowany koordynator Zigbee i mogąca zarządzać maksymalnie 128 urządzeniami podrzędnymi Zigbee. Realizacja zdalnego dostępu i usług w chmurze poprzez połączenie z Internetem jest głównym elementem systemu inteligentnego domu Haiman.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Komunikacja w górę | IEEE 802.11 b/g/n (WiFi 2,4 GHz) + Ethernet 10/100M |
| Komunikacja w dół | Zigbee 3.0 (tryb koordynatora) |
| Maksymalna liczba podurządzeń | 128 |
| Sposób zasilania | DC 5 V / 1 A (MicroUSB) |
| Pobór mocy w trybie gotowości | ≤ 1W |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤ 90% wilgotności względnej |
| Lampka kontrolna | Stan Wi-Fi/Zasilanie/Stan sieci Zigbee |
| Metoda resetowania | Przycisk reset + długie naciśnięcie przywraca ustawienia fabryczne |

## Funkcje

- Podwójne łącza nadrzędne: połączenie przewodowe WiFi lub Ethernet, elastyczny wybór w zależności od środowiska sieciowego
-Wsparcie zdalnego dostępu: sprawdzaj stan urządzenia w dowolnym miejscu i czasie za pośrednictwem aplikacji w chmurze
- Łączenie scen lokalnych: obsługuje zautomatyzowane działanie sprzętu w sieci LAN (nadal dostępne po odłączeniu)
- Aktualizacja oprogramowania sprzętowego OTA
- Obsługa platformy Tuya / Smart Life
- Zarządzaj maksymalnie 128 urządzeniami podrzędnymi Zigbee
-Wsparcie samonaprawy sieci Zigbee i optymalizacji routingu
