# HS8SA-WiFi – Inteligentny czujnik dymu (wersja WiFi US)

## Przegląd

HS8SA-WiFi to inteligentny czujnik dymu Wi-Fi wprowadzony przez firmę Hyman na rynek amerykański. Wykorzystuje technologię fotoelektrycznego wykrywania dymu, obsługuje sieci Wi-Fi i niezależne podwójne tryby pracy oraz przeszedł certyfikat UL 217. Nadaje się do ochrony przeciwpożarowej w domach, mieszkaniach i innych miejscach mieszkalnych.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | IEEE 802.11 b/g/n (2,4 GHz) |
| Typ czujnika | Fotoelektryczny na podczerwień |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | Zasilanie bateryjne |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤ 90% wilgotności względnej |
| Metoda instalacji | Instalacja sufitowa |

## Funkcje

- Fotoelektryczna detekcja dymu do wykrywania dymu z powolnego spalania i otwartego płomienia
- Zdalne naciśnięcie alarmu w sieci Wi-Fi
- Niezależny tryb pracy (lokalny alarm dźwiękowy i świetlny może nadal występować, gdy sieć jest odłączona)
- Alarm dźwiękowy i świetlny o głośności 85 dB
- Przypomnienie o niskim poziomie baterii
- Funkcja autotestu (test jednym kliknięciem)
- Alarm antysabotażowy
-Wsparcie zdalnego wyciszenia

## Standardy certyfikacji

- UL 217 (amerykańska norma dotycząca czujników dymu)
-FCC
- Zgodność z NFPA 72
