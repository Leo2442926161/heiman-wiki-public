# Standardy certyfikacji

HS8SA-WiFi przeszło następujące certyfikaty:

- **UL 217** — amerykańska norma bezpieczeństwa czujników dymu, patrz [UL Underwriters Laboratories](../../../../certification standards/ULUnderwriters Laboratories.md)
- **FCC** — certyfikat Federalnej Komisji Łączności
- **CE** — Deklaracja zgodności Unii Europejskiej
