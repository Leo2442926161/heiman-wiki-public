# Parametry techniczne

Poniżej znajdują się ogólne dane techniczne urządzeń WiFi firmy Heimann.

## Parametry bezprzewodowe

| Parametr | Wartość |
|------|-----|
| Standardy protokołu | IEEE 802.11 b/g/n |
| Pasmo częstotliwości roboczej | 2,4 GHz (niekompatybilny z 5 GHz) |
| Szybkość transmisji | Do 150 Mb/s (802.11n) |
| Topologia sieci | Topologia gwiazdy (bezpośrednio podłączona do routera) |
| Odległość komunikacyjna | Około 30 ~ 50 metrów w pomieszczeniu (w zależności od wydajności routera) |
| Metoda szyfrowania | WPA2-PSK / WPA3 |

## Metoda dystrybucji sieciowej

| Metoda | Opis |
|------|------|
| Inteligentna konfiguracja | Aplikacja mobilna Konfiguracja sieci jednym kliknięciem (tradycyjny sprzęt WiFi) |
| Udostępnianie wspomagane przez Bluetooth | Konfigurowanie poświadczeń Wi-Fi za pośrednictwem tymczasowych kanałów Bluetooth (niektóre nowe modele) |
| Tryb AP | Włącz hotspot na urządzeniu i skonfiguruj go po podłączeniu telefonu |

## Zużycie energii elektrycznej i mocy

| Parametr | Wartość |
|------|-----|
| Sposób zasilania | Zasilanie sieciowe (typ gniazda/bramka) lub zasilanie akumulatorowe (typ alarmowy) |
| Poziom zużycia energii | Wyższa niż Zigbee/Thread ze względu na konieczność utrzymywania połączenia TCP/IP |
| Pobór mocy w trybie gotowości | Typ gniazda około 0,5W ~ 1W |

## Parametry środowiska

| Parametr | Wartość |
|------|-----|
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤ 90% RH (bez kondensacji) |
