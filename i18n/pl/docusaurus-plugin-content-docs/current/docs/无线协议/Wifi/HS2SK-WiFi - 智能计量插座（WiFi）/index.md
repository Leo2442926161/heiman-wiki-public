#HS2SK-WiFi - Inteligentne gniazdo pomiarowe (WiFi)

## Przegląd

HS2SK-WiFi to inteligentne gniazdo pomiarowe WiFi wprowadzone na rynek przez firmę Heiman. Obsługuje bezpośrednie połączenie 2,4 GHz 802.11 b/g/n z routerami domowymi i umożliwia zdalne sterowanie oraz monitorowanie zużycia energii bez potrzeby stosowania dodatkowych bramek. Nadaje się do inteligentnego zarządzania urządzeniami elektrycznymi w domach, biurach i innych scenariuszach.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | IEEE 802.11 b/g/n (2,4 GHz) |
| Maksymalne obciążenie | 16A |
| Maksymalna moc | 3680 W (230 V) / 1800 W (110 V) |
| Dokładność pomiaru | ±2% |
| Sposób zasilania | Zasilanie sieciowe |
| Pobór mocy w trybie gotowości | ≤ 0,5 W |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤ 90% wilgotności względnej |
| Materiał powłoki | PC V0 trudnopalny |
| Specyfikacja Jacka | Norma europejska / norma amerykańska (w zależności od wersji rynkowej) |

## Funkcje

-Wsparcie Smart Life/inteligentna aplikacja Tuya do zdalnego sterowania
- Monitorowanie mocy w czasie rzeczywistym i statystyki zużycia energii
- Zdalne sterowanie przełącznikiem
- Przełącznik czasowy i odliczanie
- Łączenie scen i automatyzacja
- Sterowanie głosowe (kompatybilne z Alexą, Google Home)
