# Parametry techniczne

Poniżej przedstawiono ogólne parametry techniczne sprzętu Heimann Wireless M-Bus (wM-Bus).

## Parametry bezprzewodowe

| Parametr | Wartość |
|------|-----|
| Standardy protokołu | EN 13757-4 (Warstwa fizyczna bezprzewodowej magistrali M-Bus) |
| Robocze pasmo częstotliwości | 868 MHz (europejskie pasmo częstotliwości SRD, 868,0~868,6 MHz) |
| Metoda modulacji | FSK |
| Szybkość transmisji | 4,8 kbps / 16,384 kbps / 100 kbps (tryb T / tryb S / tryb R) |
| Topologia sieci | Gwiazda (instrument → kolektor/bramka) |
| Odległość komunikacyjna | Około 100~300 metrów w pomieszczeniu, około 500~1000 metrów na otwartej przestrzeni |
| Moc nadawania | ≤ 25 mW (+14 dBm) |

## Tryb pracy

| Tryb | Oceń | Opis |
|------|------|------|
| Tryb S | 16,384 kb/s | Tryb standardowy, niski pobór mocy, odpowiedni do instrumentów zasilanych bateryjnie |
| Tryb T | 100 kb/s | Tryb dużej prędkości, odpowiedni dla scenariuszy danych o dużym ruchu |
| Tryb R | 4,8 kb/s | Tryb wąskopasmowy, transmisja na duże odległości |

## Zużycie energii elektrycznej i mocy

| Parametr | Wartość |
|------|-----|
| Sposób zasilania | Zasilacz z baterii litowej (ER18505 / ER26500 i inne przemysłowe baterie litowe) |
| Żywotność baterii | 6~10 lat (w zależności od częstotliwości raportowania, typowa wartość to 1 raz dziennie) |
| Pobór mocy w trybie gotowości | Poziom mikroampera, głęboki sen |
| Otrzymuj czułość | Około. -105 dBm (tryb S) |

## Parametry środowiska

| Parametr | Wartość |
|------|-----|
| Temperatura pracy | -20°C ~ +60°C (klasa przemysłowa) |
| Wilgotność robocza | ≤ 95% RH (bez kondensacji) |
| Poziom ochrony IP | IP65 ~ IP68 (w zależności od konkretnego modelu, dotyczy instrumentów zewnętrznych) |

## Protokół danych

| Warstwa | Standardowe |
|----|------|
| Warstwa fizyczna | EN 13757-4 |
| Warstwa łącza danych | EN 13757-4 (Format ramki bezprzewodowej magistrali M-Bus) |
| Warstwa aplikacji | EN 13757-3 (warstwa aplikacji M-Bus) |
| Metoda szyfrowania | AES-128 (szyfrowanie opcjonalne) |
