# M1-M Czujnik podczerwieni ludzkiego ciała (materia)

M1-M to czujnik podczerwieni ciała ludzkiego **Materia nad nicią** wprowadzony na rynek przez firmę Hyman. Integruje funkcje wykrywania PIR i wykrywania oświetlenia i nadaje się do scenariuszy inteligentnej automatyki domowej.

## Przegląd produktu

- **Rodzaj produktu**: Czujnik podczerwieni ciała ludzkiego
- **Protokół komunikacyjny**: Materia ponad wątkiem
- **Metoda detekcji**: Pasywna podczerwień (PIR) + detekcja oświetlenia
- **Typowy scenariusz**: Światła włączają się, gdy przychodzą ludzie, połączenie bezpieczeństwa, automatyzacja środowiska

## Parametry techniczne

| Parametr | Wartość |
|------|-----|
| Kąt detekcji | 110° (montaż na ścianie) / 360° (sufit) |
| Odległość wykrywania | ≤7 m (25°C) |
| Zasięg oświetlenia | 0~1200 luksów |
| Protokół komunikacyjny | Materia ponad wątkiem (IEEE 802.15.4) |
| Pasmo częstotliwości roboczej | 2,4 GHz |
| Sposób zasilania | Bateria litowa DC 3V |
| Żywotność baterii | Około 1 rok (tryb materii) |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% RH (bez kondensacji) |

## Funkcje

- Mobilne wykrywanie ludzi
- Wykrywanie oświetlenia otoczenia
- Odległość wykrywania ≤7m
- Wsparcie montażu na ścianie lub suficie
- Ma znaczenie wieloekologiczna interoperacyjność między platformami
