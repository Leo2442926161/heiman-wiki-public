# Inteligentny czujnik dymu HS1SA-M (materia)

HS1SA-M to inteligentny czujnik dymu **Materia ponad nicią** wprowadzony na rynek przez firmę Hyman. Wykorzystuje technologię fotoelektrycznego wykrywania dymu i obsługuje międzyekologiczne połączenia platform inteligentnego domu.

## Przegląd produktu

- **Typ produktu**: Inteligentny czujnik dymu
- **Protokół komunikacyjny**: Materia ponad wątkiem
- **Norma certyfikacyjna**: EN 14604
- **Typowe scenariusze**: Ostrzeżenie o pożarze w domach, mieszkaniach, hotelach itp.

## Parametry techniczne

| Parametr | Wartość |
|------|-----|
| Typ czujnika | Fotoelektryczny na podczerwień |
| Protokół komunikacyjny | Materia ponad wątkiem (IEEE 802.15.4) |
| Pasmo częstotliwości roboczej | 2,4 GHz |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | DC 3V (bateria litowa CR123A) |
| Żywotność baterii | Około 3 lata |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% RH (bez kondensacji) |

## Funkcje

- Fotoelektryczna detekcja dymu w celu wykrycia dymu wolno palącego się i otwartego płomienia
- Alarm dźwiękowy i świetlny o głośności 85 dB
- Przypomnienie o niskim poziomie baterii
- Funkcja autotestu (test jednym kliknięciem)
- Alarm antysabotażowy
- Zdalne wyciszenie
- Ma znaczenie wieloekologiczna interoperacyjność między platformami

## Certyfikacja

- Certyfikat EN 14604
- Certyfikacja sprawy
