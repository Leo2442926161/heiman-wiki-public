# H1-M Czujnik temperatury i wilgotności (materii)

H1-M to czujnik temperatury i wilgotności **Matter over Thread** wprowadzony na rynek przez firmę Hyman, który może monitorować temperaturę otoczenia i wilgotność względną w czasie rzeczywistym oraz obsługuje połączenie inteligentnej automatyki domowej.

## Przegląd produktu

- **Typ produktu**: Czujnik temperatury i wilgotności
- **Protokół komunikacyjny**: Materia ponad wątkiem
- **Parametry detekcji**: temperatura, wilgotność względna
- **Typowe scenariusze**: Monitoring środowiska w domach, biurach, piwnicach z winami itp.

## Parametry techniczne

| Parametr | Wartość |
|------|-----|
| Dokładność temperatury | ±0,3°C |
| Dokładność wilgotności | ±3% wilgotności względnej |
| Protokół komunikacyjny | Materia ponad wątkiem (IEEE 802.15.4) |
| Pasmo częstotliwości roboczej | 2,4 GHz |
| Sposób zasilania | Bateria litowa CR2 |
| Żywotność baterii | Około 2 ~ 3 lata |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% RH (bez kondensacji) |

## Funkcje

- Monitorowanie temperatury i wilgotności w czasie rzeczywistym
- Wysoka precyzja wykrywania (temperatura ±0,3°C, wilgotność ±3% RH)
- Przypomnienie o niskim poziomie baterii
- Ma znaczenie wieloekologiczna interoperacyjność między platformami
