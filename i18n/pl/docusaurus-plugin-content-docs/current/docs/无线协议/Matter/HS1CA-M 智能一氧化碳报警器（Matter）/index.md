# HS1CA-M Inteligentny alarm dotyczący tlenku węgla (materii)

HS1CA-M to pierwszy na świecie alarm wykrywający tlenek węgla z certyfikatem Matter**, wykorzystujący technologię czujników elektrochemicznych i obsługujący ekologiczne połączenia Matter over Thread.

## Przegląd produktu

- **Typ produktu**: Inteligentny alarm czadu
- **Protokół komunikacyjny**: Materia ponad wątkiem
- **Norma certyfikacyjna**: EN 50291
- **Typowy scenariusz**: Ostrzeżenie o wycieku CO w domach, mieszkaniach i innych miejscach

## Parametry techniczne

| Parametr | Wartość |
|------|-----|
| Typ czujnika | Elektrochemiczny czujnik CO |
| Protokół komunikacyjny | Materia ponad wątkiem (IEEE 802.15.4) |
| Pasmo częstotliwości roboczej | 2,4 GHz |
| Zasięg detekcji | 0 ~ 1000 ppm |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | DC 3V (bateria litowa CR123A) |
| Żywotność baterii | Około 3 lata |
| Żywotność czujnika | Typowe 7 lat |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% RH (bez kondensacji) |

## Funkcje

- Bardzo precyzyjny elektrochemiczny czujnik CO
- Alarm stopniowania stężenia (komunikat dźwiękowy i świetlny)
- Alarm dźwiękowy i świetlny o głośności 85 dB
- Przypomnienie o niskim poziomie baterii
- Przypomnienie o upływie żywotności czujnika
- Funkcja autotestu
- Ma znaczenie wieloekologiczna interoperacyjność między platformami

## Certyfikacja
