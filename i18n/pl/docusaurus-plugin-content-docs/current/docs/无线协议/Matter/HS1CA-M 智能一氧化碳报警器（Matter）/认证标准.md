# Standardy certyfikacji

HS1CA-M przeszedł następujące certyfikaty:

- **EN 50291** — Europejska norma dotycząca czujników czadu, patrz [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **Znak CE** — Deklaracja zgodności UE, patrz [Zgodność CE UE](../../../../certification-standards/CE EU-Conformity.md)
- **Certyfikat Matter** — Uzyskał certyfikat CSA Alliance Matter zapewniający wzajemne połączenie inteligentnych domów między platformami
