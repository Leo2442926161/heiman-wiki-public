# Detektor zanurzenia w wodzie L1-M (materia)

L1-M to detektor zanurzeniowy w wodzie **Materia nad nicią** wprowadzony na rynek przez firmę Hyman. Wykrywa wyciek wody lub nagromadzoną wodę za pomocą sondy, co może skutecznie zapobiegać stratom majątkowym spowodowanym pęknięciem rury wodociągowej.

## Przegląd produktu

- **Typ produktu**: Czujnik wykrywania zanurzenia w wodzie
- **Protokół komunikacyjny**: Materia ponad wątkiem
- **Metoda wykrywania**: Wykrywanie kontaktu sondy
- **Typowe scenariusze**: Kuchnia, łazienka, pralnia, wykrywanie wycieków wody w piwnicy

## Parametry techniczne

| Parametr | Wartość |
|------|-----|
| Metoda wykrywania | Typ styku sondy ze stali nierdzewnej |
| Protokół komunikacyjny | Materia ponad wątkiem (IEEE 802.15.4) |
| Pasmo częstotliwości roboczej | 2,4 GHz |
| Sposób zasilania | Zasilanie bateryjne |
| Metoda alarmowania | Brzęczyk + Push aplikacji |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% RH (bez kondensacji) |

## Funkcje

- Wykrywanie w czasie rzeczywistym wycieków wody/stojącej wody
- Brzęczyk alarmu na miejscu
- Zdalne naciśnięcie alarmu aplikacji
- Przypomnienie o niskim poziomie baterii
- Konstrukcja antykorozyjna sondy
- Ma znaczenie wieloekologiczna interoperacyjność między platformami
