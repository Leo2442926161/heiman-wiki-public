# HS1DS-M - Inteligentny czujnik drzwi (Materia nad gwintem)

## Przegląd

HS1DS-M to wersja inteligentnego czujnika drzwi Heiman w wersji Matter over Thread. Obsługuje natywny dostęp do wszystkich platform kompatybilnych z Matter, takich jak Apple Home, Google Home, Amazon Alexa, Home Assistant itp. bez potrzeby posiadania własnej bramy.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | Materia nad wątkiem |
| Sposób zasilania | Bateria guzikowa CR1632 |
| Żywotność baterii | Około 2-3 lata |
| Odległość montażowa | ≤10 mm |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% wilgotności względnej |

## Funkcje

- Wykrywanie w czasie rzeczywistym stanu otwarcia/zamknięcia drzwi i okien
- Alarm sabotażowy (Tamper)
- Przypomnienie o niskim poziomie baterii
- Kompatybilny z wieloma platformami: Apple Home/Google Home/Alexa/SmartThings/Home Assistant
- Sieć Thread Mesh, sterowanie lokalne o niskim opóźnieniu
- Nie jest wymagana żadna dedykowana brama, wymagany jest router graniczny wątku
