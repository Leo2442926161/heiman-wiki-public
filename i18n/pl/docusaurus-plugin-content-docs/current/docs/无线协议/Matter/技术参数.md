# Parametry techniczne

Poniżej znajdują się ogólne specyfikacje sprzętu Hyman Matter.

## Parametry bezprzewodowe

| Parametr | Wartość |
|------|-----|
| Podstawowy protokół | Wątek (IEEE 802.15.4) |
| Pasmo częstotliwości roboczej | 2,4 GHz (pasmo ISM) |
| Standardy warstwy aplikacji | Materia 1.x |
| Topologia sieci | Siatka samonaprawiająca się sieć |
| Metoda szyfrowania | AES-128-CCM |

## Wymagania sprzętowe

| Projekt | Opis |
|------|------|
| Router graniczny | Wymagane, takie jak Apple HomePod mini, Apple TV 4K, Google Nest Hub 2. generacji itp. |
| Obsługa terminali mobilnych | iOS / Android (dystrybucja sieciowa za pośrednictwem różnych aplikacji ekologicznych) |
| Interoperacyjność międzyekologiczna | Apple HomeKit, Google Home, Amazon Alexa, Asystent domowy, SmartThings |

## Zużycie energii elektrycznej i mocy

| Parametr | Wartość |
|------|-----|
| Sposób zasilania | Zasilanie bateryjne (CR123A / CR2450 itp.) |
| Żywotność baterii | Około 1~3 lat (w zależności od rodzaju urządzenia i częstotliwości użytkowania) |
| Pobór mocy w trybie gotowości | Wyjątkowo niski, wspiera mechanizm budzenia się podczas snu |

## Parametry środowiska

| Parametr | Wartość |
|------|-----|
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤ 95% RH (bez kondensacji) |
