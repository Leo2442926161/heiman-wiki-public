#Materia

Matter (wcześniej znany jako Project CHIP) to standard połączeń inteligentnych domów wprowadzony przez stowarzyszenie CSA Connection Standards Alliance, którego celem jest przełamanie barier marki i osiągnięcie międzyekologicznej interoperacyjności.

## Heyman i materia

Hyman jako pierwszy przyjął protokół Matter, wprowadzając **pierwsze na świecie alarmy tlenku węgla z certyfikatem Matter**. Wszystkie urządzenia Hyman's Matter korzystają z rozwiązania **Matter over Thread**.

## Właściwości techniczne

| Funkcje | Opis |
|------|------|
| Podstawowy protokół | Wątek (sieć Mesh małej mocy) |
| Topologia sieci | Siatka samonaprawiająca się, im więcej urządzeń, tym szerszy zasięg |
| Zużycie energii | Niezwykle niski, może działać przez 1-3 lata na zasilaniu bateryjnym |
| Bezpieczeństwo | Obsługuje szyfrowanie AES-128-CCM, wymagane uwierzytelnienie urządzenia |
| Interoperacyjność | W Apple HomeKit, Google Home, Amazon Alexa, Home Assistant i nie tylko |

## Wymagania routera granicznego

Korzystanie z urządzeń Matter over Thread wymaga routera granicznego Thread:

- Apple HomePod mini / HomePod drugiej generacji / Apple TV 4K
- Google Nest Hub drugiej generacji / Nest Hub Max
-Amazon Echo 4. generacji
- Asystent Domowy Zielony + Connect ZBT-2
- Domowy profesjonalista
