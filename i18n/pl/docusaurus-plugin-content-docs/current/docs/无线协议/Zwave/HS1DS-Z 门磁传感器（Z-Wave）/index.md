# Czujnik drzwi HS1DS-Z (Z-Wave)

HS1DS-Z to czujnik drzwiowy **Z-Wave** wprowadzony na rynek przez firmę Hyman. Wykrywa stan otwarcia i zamknięcia drzwi i okien za pomocą magnetycznych kontaktronów i jest kompatybilny ze standardem Z-Wave Plus.

## Przegląd produktu

- **Typ produktu**: Czujnik drzwi
- **Protokół komunikacyjny**: Z-Wave Plus
- **Metoda wykrywania**: Kontaktron (kontaktron)
- **Typowe scenariusze**: Monitorowanie stanu drzwi i okien, wdrażanie zabezpieczeń i inteligentne połączenia

## Parametry techniczne

| Parametr | Wartość |
|------|-----|
| Metoda wykrywania | Kontaktron (kontaktron) |
| Protokół komunikacyjny | Z-Wave Plus (ITU-T G.9959) |
| Pasmo częstotliwości roboczej | 868,42 MHz (UE) / 908,42 MHz (USA) |
| Sposób zasilania | Bateria guzikowa CR1632 |
| Żywotność baterii | Około 2 ~ 3 lata |
| Luz instalacyjny | ≤10 mm |
| Metoda szyfrowania | S2 Uwierzytelnione / S2 Nieuwierzytelnione |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤90% RH (bez kondensacji) |

## Funkcje

- Wykrywanie w czasie rzeczywistym stanu otwarcia/zamknięcia drzwi i okien
- Alarm sabotażowy (Tamper)
- Przypomnienie o niskim poziomie baterii
-Obsługuje łączenie scen (otwieranie drzwi, włączanie świateł, uzbrajanie przy wyjściu z domu)
- Łatwy w montażu (taśma klejąca 3M lub mocowanie śrubowe)
- Bezpieczne szyfrowanie Z-Wave S2
— Obsługa aktualizacji oprogramowania sprzętowego OTA
