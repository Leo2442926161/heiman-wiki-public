#Z-Wave

Z-Wave to bezprzewodowy protokół sieciowy Mesh przeznaczony dla inteligentnych domów. Jest zarządzany przez Z-Wave Alliance i ma szeroką penetrację na rynku europejskim i amerykańskim.

## Specyfikacje techniczne

| Funkcje | Opis |
|------|------|
| Pasmo częstotliwości roboczej | 868,42 MHz (UE) / 908,42 MHz (USA) |
| Szybkość transferu | Do 100 kbps (seria Z-Wave 800) |
| Topologia sieci | Sieć mesh, do 232 węzłów |
| Kompatybilność | Wstecznie kompatybilny ze wszystkimi urządzeniami Z-Wave |
| Bezpieczeństwo | Szyfrowanie Z-Wave S2 |

## Hyman i Z-Wave

Hyman oferuje szeroką gamę czujników kompatybilnych z Z-Wave, sprzedawanych głównie na rynku europejskim i amerykańskim. Wersje Z-Wave urządzeń Heyman obejmują:

- Czujnik czadu HS1CA-Z
- Czujnik dymu HS1SA-Z
- Czujnik drzwi HS1DS-Z

## Zgodne platformy

- Centrum SmartThings
- Wzniesienie Hubitatu
- Domowy profesjonalista
- Asystent domowy (przez Z-Wave JS)
