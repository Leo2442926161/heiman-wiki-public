# HS1CA-Z Inteligentny alarm czadu (Z-Wave)

HS1CA-Z to inteligentny alarm czadu **Z-Wave** wprowadzony na rynek przez firmę Hyman. Wykorzystuje technologię czujników elektrochemicznych i jest kompatybilny ze standardem Z-Wave Plus.

## Przegląd produktu

- **Typ produktu**: Inteligentny alarm czadu
- **Protokół komunikacyjny**: Z-Wave Plus
- **Norma certyfikacyjna**: EN 50291
- **Typowy scenariusz**: Ostrzeżenie o wycieku CO w domach, mieszkaniach i innych miejscach

## Parametry techniczne

| Parametr | Wartość |
|------|-----|
| Typ czujnika | Elektrochemiczny czujnik CO |
| Protokół komunikacyjny | Z-Wave Plus (ITU-T G.9959) |
| Pasmo częstotliwości roboczej | 868,42 MHz (UE) / 908,42 MHz (USA) |
| Zasięg detekcji | 0 ~ 1000 ppm |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | Zasilanie bateryjne |
| Żywotność czujnika | Typowe 7 lat |
| Metoda szyfrowania | S2 Uwierzytelnione / S2 Nieuwierzytelnione |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤90% RH (bez kondensacji) |

## Funkcje

- Bardzo precyzyjny elektrochemiczny czujnik CO
- Alarm klasyfikacji stężeń
- Alarm dźwiękowy i świetlny o głośności 85 dB
- Przypomnienie o niskim poziomie baterii
- Przypomnienie o upływie żywotności czujnika
- Funkcja autotestu
- Bezpieczne szyfrowanie Z-Wave S2
— Obsługa aktualizacji oprogramowania sprzętowego OTA

## Certyfikacja

- Certyfikat EN 50291
- Certyfikat Z-Wave Plus
- Certyfikat bezpieczeństwa S2
