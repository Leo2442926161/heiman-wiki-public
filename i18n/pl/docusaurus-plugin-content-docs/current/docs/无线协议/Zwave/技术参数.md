# Parametry techniczne

Poniżej znajdują się ogólne dane techniczne urządzeń Hyman Z-Wave.

## Parametry bezprzewodowe

| Parametr | Wartość |
|------|-----|
| Standardy protokołu | Z-Wave Plus (ITU-T G.9959) |
| Robocze pasmo częstotliwości | 800~900 MHz (w zależności od regionu: przepisy europejskie 868,42 MHz, przepisy amerykańskie 908,42 MHz) |
| Szybkość transmisji | Do 100 kb/s |
| Topologia sieci | Siatka samonaprawiająca się (routing źródłowy) |
| Pojemność węzła | Maksymalnie 232 węzły w jednej sieci |
| Odległość komunikacyjna | Około 30 metrów w pomieszczeniu (między węzłami), cały dom Zasięg siatki |
| Metoda szyfrowania | AES-128 (uwierzytelniony S2 / nieuwierzytelniony S2) |

## Regionalne pasmo częstotliwości

| Region | Częstotliwość środkowa |
|------|--------------|
| Europa | 868,42 MHz |
| Ameryka Północna | 908,42 MHz |
| Inne obszary | Z zastrzeżeniem lokalnych przepisów |

## Zużycie energii elektrycznej i mocy

| Parametr | Wartość |
|------|-----|
| Sposób zasilania | Zasilanie bateryjne (CR123A/AA) lub sieciowe |
| Żywotność baterii | Około 2 ~ 3 lata na czujniki |
| Moc nadawania | ≤ 1 mW (zgodnie z przepisami SRD) |
| Otrzymuj czułość | Około -100 dBm |

## Parametry środowiska

| Parametr | Wartość |
|------|-----|
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤ 90% RH (bez kondensacji) |

## Charakterystyka sieci

| Funkcje | Opis |
|------|------|
| Inteligentny start | Wsparcie (seria 700), zeskanuj kod QR, aby dołączyć do sieci |
| Bezpieczeństwo S2 | Obsługuje trzy poziomy bezpieczeństwa: nieszyfrowany, nieuwierzytelniony S2, uwierzytelniony S2 |
| Zawiera CC | Bezpieczeństwo CC, Bezpieczeństwo 2 CC, Aktualizacja oprogramowania sprzętowego CC, Podstawowe CC, Bateria CC, Powiadomienia CC i więcej |
