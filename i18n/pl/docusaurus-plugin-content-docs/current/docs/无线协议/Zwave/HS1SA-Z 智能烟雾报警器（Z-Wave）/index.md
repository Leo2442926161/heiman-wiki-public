# Inteligentny czujnik dymu HS1SA-Z (Z-Wave)

HS1SA-Z to inteligentny czujnik dymu **Z-Wave** wprowadzony na rynek przez firmę Hyman. Wykorzystuje technologię fotoelektrycznego wykrywania dymu, jest kompatybilny ze standardem Z-Wave Plus i nadaje się na europejski i amerykański rynek inteligentnych domów.

## Przegląd produktu

- **Typ produktu**: Inteligentny czujnik dymu
- **Protokół komunikacyjny**: Z-Wave Plus
- **Norma certyfikacyjna**: EN 14604
- **Typowy scenariusz**: Ostrzeżenie o pożarze w domach, mieszkaniach i innych miejscach

## Parametry techniczne

| Parametr | Wartość |
|------|-----|
| Typ czujnika | Fotoelektryczny na podczerwień |
| Protokół komunikacyjny | Z-Wave Plus (ITU-T G.9959) |
| Pasmo częstotliwości roboczej | 868,42 MHz (UE) / 908,42 MHz (USA) |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | Bateria litowa CR123A |
| Żywotność baterii | Około 2 ~ 3 lata |
| Metoda szyfrowania | S2 Uwierzytelnione / S2 Nieuwierzytelnione |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤90% RH (bez kondensacji) |

## Funkcje

- Fotoelektryczna detekcja dymu w celu wykrycia dymu wolno palącego się i otwartego płomienia
- Alarm dźwiękowy i świetlny o głośności 85 dB
- Przypomnienie o niskim poziomie baterii
- Funkcja autotestu
- Alarm antysabotażowy
- Bezpieczne szyfrowanie Z-Wave S2
— Obsługa aktualizacji oprogramowania sprzętowego OTA

## Certyfikacja

- Certyfikat EN 14604
- Certyfikat Z-Wave Plus
- Certyfikat bezpieczeństwa S2
