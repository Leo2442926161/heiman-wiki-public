# Standardy certyfikacji

HS1SA-Z posiada następujące certyfikaty:

- **EN 14604** — Europejska norma dotycząca czujników dymu, patrz [EN European Standard](../../../../ApprovalStandards/EN European Standard.md)
- **Certyfikat Z-Wave S2** — Obsługuje platformę bezpieczeństwa S2, zobacz [Inne certyfikaty](../../../Standardy certyfikacji/Inne certyfikaty.md)
- **CE** — Deklaracja zgodności Unii Europejskiej
