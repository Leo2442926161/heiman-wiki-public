# Seria S1 — samodzielny czujnik dymu (Sub-1G)

## Przegląd

Seria S1 to samodzielny czujnik dymu Sub-1G wprowadzony na rynek przez firmę Hyman. Działa w paśmie częstotliwości 433 MHz / 868 MHz, wykorzystuje technologię fotoelektrycznego wykrywania dymu i uzyskał certyfikat EN 14604. Nie jest wymagana brama ani Internet. Urządzenia są ze sobą bezpośrednio połączone za pomocą częstotliwości radiowej Sub-1G. Gdy którekolwiek urządzenie uruchomi alarm, wszystkie urządzenia w tej samej grupie uruchomią alarm jednocześnie. Nadaje się do scenariuszy połączeń pożarowych na dużą skalę, takich jak budynki, centra handlowe i hotele.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | 433 MHz / 868 MHz Sub-1G (w zależności od obszaru rynku) |
| Typ czujnika | Fotoelektryczny na podczerwień |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | Zasilanie bateryjne (bateria litowa CR123A / AA) |
| Żywotność baterii | ≥ 5 lat |
| Odległość komunikacyjna | Około 1~3 kilometrów w otwartym terenie |
| Pojemność sieci | Pojedyncza grupa obsługuje do 50+ węzłów |
| Metoda połączenia | Bezprzewodowe połączenie radiowe (nie wymaga bramy) |
| Temperatura pracy | -10°C ~ +55°C |
| Wilgotność robocza | ≤ 95% wilgotności względnej |
| Metoda instalacji | Instalacja sufitowa |

## Funkcje

- Fotoelektryczna detekcja dymu do wykrywania dymu z powolnego spalania i otwartego płomienia
- Alarm połączenia bezprzewodowego: wywołany przez dowolne urządzenie w tej samej sieci, wszystkie urządzenia jednocześnie włączą alarm dźwiękowy i świetlny
- Niezależny tryb pracy: brak bramy, brak konieczności korzystania z Internetu, bezpośrednia komunikacja pomiędzy urządzeniami
- Alarm dźwiękowy i świetlny o głośności 85 dB
- Przypomnienie o niskim poziomie baterii
- Funkcja autotestu (test jednym kliknięciem)
- Alarm antysabotażowy
-Wsparcie zdalnego wyciszenia
- Konstrukcja o bardzo niskim zużyciu energii, żywotność baterii ponad 5 lat

## Standardy certyfikacji

- EN 14604 (europejska norma dotycząca czujników dymu)
-CE/CZERWONY
- CPR (rozporządzenie UE dotyczące wyrobów budowlanych)

## Odpowiednie akcesoria

- Repeater bezprzewodowy S1-RM (zwiększony zasięg komunikacji)
- Pilot S1-RC (test/wyciszenie/reset)
