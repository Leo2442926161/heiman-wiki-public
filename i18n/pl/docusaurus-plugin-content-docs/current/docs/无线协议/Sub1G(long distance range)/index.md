# Sub-1G (komunikacja na duże odległości)

Sub-1G to technologia komunikacji bezprzewodowej działająca w paśmie częstotliwości poniżej 1 GHz. Charakteryzuje się bardzo dużą odległością transmisji i wyjątkowo niskim zużyciem energii. Jest bardzo odpowiedni w scenariuszach IoT, które wymagają zasięgu na dużą skalę.

## Funkcje

| Funkcje | Opis |
|------|------|
| Pasmo częstotliwości roboczej | 433 MHz / 868 MHz (UE) / 915 MHz (USA) |
| Odległość transmisji | Do 1-3 kilometrów w terenie otwartym |
| Zdolność penetracji | Lepsza niż 2,4 GHz, silna penetracja przeszkód ściennych |
| Zużycie energii | Niezwykle niski, żywotność baterii może sięgać 5-10 lat |
| Szybkość transmisji | Niski (odpowiedni dla scenariuszy z małą ilością danych) |

## Scenariusze zastosowań

- **Niezależny czujnik dymu/CO**: połączony na piętrach budynku
- **System przeciwpożarowy dużych budynków**: galerie handlowe, hotele, szkoły
- **Smart Senior Living Community**: Połączenie alarmowe między jednostkami
- **Sub-1G + bramka Wi-Fi**: urządzenie podrzędne komunikuje się poprzez Sub-1G, a brama łączy się z chmurą
