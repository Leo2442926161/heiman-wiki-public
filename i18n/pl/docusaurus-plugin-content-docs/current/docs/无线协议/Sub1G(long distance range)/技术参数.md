# Parametry techniczne

Poniżej przedstawiono ogólne parametry techniczne produktów serii dalekobieżnej Hyman Sub-1G.

## Parametry bezprzewodowe

| Parametr | Wartość |
|------|-----|
| Pasmo częstotliwości roboczej | 433 MHz / 868 MHz / 915 MHz (w zależności od obszaru rynku) |
| Metoda modulacji | FSK / OOK |
| Szybkość transmisji | Około 10 ~ 100 kbps (segment wideo i konfiguracja) |
| Topologia sieci | Sieć gwiazdowa / punkt-punkt / sieć internetowa |
| Odległość komunikacyjna | Około 1~3 kilometrów w otwartym środowisku (segment wideo i zasilanie) |
| Pojemność sieci | Pojedyncza grupa obsługuje do 50+ węzłów |

## Regionalny przydział pasma częstotliwości

| Region | Wspólne pasma częstotliwości |
|------|--------------|
| Chiny | 433 MHz |
| Europa | 868 MHz |
| Ameryka Północna | 915 MHz |

## Zużycie energii elektrycznej i mocy

| Parametr | Wartość |
|------|-----|
| Sposób zasilania | Zasilanie bateryjne (bateria litowa CR123A / AA) |
| Żywotność baterii | Około 3 ~ 5 lat (w zależności od częstotliwości raportowania) |
| Moc nadawania | ≤ 10 mW (+10 dBm), zgodnie z przepisami SRD |
| Pobór mocy w trybie gotowości | Poziom mikroampera, wyjątkowo niski |
| Czułość odbioru | Około -110 dBm ~ -120 dBm |

## Parametry środowiska

| Parametr | Wartość |
|------|-----|
| Temperatura pracy | -10°C ~ +55°C |
| Wilgotność robocza | ≤ 95% RH (bez kondensacji) |
