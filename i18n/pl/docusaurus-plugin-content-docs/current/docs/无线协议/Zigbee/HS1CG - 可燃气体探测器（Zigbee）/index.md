#HS1CG – Detektor gazów palnych (Zigbee)

## Przegląd

HS1CG to detektor gazów palnych Zigbee wprowadzony na rynek przez firmę Hyman. Wykorzystuje czujniki półprzewodnikowe i może wykrywać wycieki gazu ziemnego (metanu) i gazu płynnego (LPG). Przeszedł certyfikację GB 15322 i obsługuje automatyczne zamykanie zaworu elektromagnetycznego mechanizmu łączącego.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Półprzewodnik |
| Gaz testowy | Gaz ziemny (metan)/gaz płynny (LPG) |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | AC 220V / DC 12V |
| Standardy certyfikacji | GB 15322 |
| Temperatura pracy | -10°C ~ +50°C |

## Funkcje

- Wykrywanie wycieków gazu ziemnego / skroplonego
- Alarm klasyfikacji stężeń
- Automatyczny podnośnik odcinający (z zaworem elektromagnetycznym)
- Alarm dźwiękowy i wizualny 85 dB
- Funkcja autotestu
- Przypomnienie o upływie żywotności czujnika

## Standardy certyfikacji

- GB 15322 (chińska norma dotycząca detektorów gazów palnych)

## Odpowiednie akcesoria

- Awaryjny zawór odcinający gaz serii JT
