#Lista obsługiwanych modeli

Heyman ma najszerszą w branży linię produktów Zigbee, a poniżej znajdują się obsługiwane modele.

## Kategoria alarmu bezpieczeństwa

| Modelka | Nazwa produktu | Opis |
|------|----------|------|
| HS1SA-E | Inteligentny czujnik dymu (Zigbee) | Wersja UE |
| HS1SA-Z | Inteligentny czujnik dymu (Zigbee) | Wersja chińska |
| HS3SA | Inteligentny czujnik dymu (Zigbee) | Czujnik dymu trzeciej generacji |
| HS8SA | Inteligentny czujnik dymu (Zigbee) | Typ połączenia wysokotemperaturowego |
| HS1CA-E | Inteligentny alarm czadu (Zigbee) | Wersja UE |
| HS1CA-Z | Inteligentny alarm czadu (Zigbee) | Wersja chińska |
| HS1CG | Detektor gazów palnych (Zigbee) | Gaz ziemny / gaz skroplony |
| HS3CG | Detektor gazów palnych (Zigbee) | Detektor gazu trzeciej generacji |

| Modelka | Nazwa produktu | Opis |
|------|----------|------|
| HS1DS | Czujnik magnetyczny drzwi (Zigbee) | Detekcja otwarcia i zamknięcia drzwi i okien |
| HS1MS | Czujnik podczerwieni ludzkiego ciała (Zigbee) | Typ standardowy |
| HS3MS | Czujnik podczerwieni ludzkiego ciała (Zigbee) | Trzecia generacja, poprawiona czułość wykrywania |
| M1-Z | Czujnik podczerwieni ludzkiego ciała (Zigbee) | Kompaktowa konstrukcja |
| HS2WL | Detektor zanurzenia w wodzie (Zigbee) | Druga generacja |
| HS3WL | Detektor zanurzenia w wodzie (Zigbee) | Trzecia generacja |
| HS1WL | Detektor zanurzenia w wodzie (Zigbee) | Pierwsza generacja |
| HS8MLS | Czujnik mikroruchu (Zigbee) | Detekcja ruchu |

| HS8OS | Czujnik światła (Zigbee) | Wykrywanie światła |

| Modelka | Nazwa produktu | Opis |
|------|----------|------|
| HS2FD | Detektor upadku (Zigbee) | Alarm wykrywania upadku osób starszych |

## Instrukcje dotyczące nazewnictwa modeli

| Zasady nazewnictwa | Znaczenie |
|-------------|------|
| Przyrostek -E | Wersja UE (norma 868 MHz / EN) |
| Przyrostek -Z | Wersja chińska (2,4 GHz / standard chiński) |
| HS1/2/3/8 | Numer generacji produktu, im większy numer, tym nowsza generacja |
