#HS8MIS – Czujnik mobilny (Zigbee)

## Przegląd

HS8MIS to wielofunkcyjny czujnik ruchu Zigbee wprowadzony na rynek przez firmę Heiman. Integruje funkcje wykrywania ludzi PIR i wykrywania oświetlenia i nadaje się do inteligentnych scen z podwójnym monitorowaniem światła i ruchu.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Pasywna podczerwień (PIR) + oświetlenie |
| Kąt detekcji | 110° |
| Odległość wykrywania | ≤7 m |
| Sposób zasilania | Bateria DC 3V |

## Funkcje

- Mobilne wykrywanie ludzi
- Wykrywanie oświetlenia
- Konstrukcja o niskim zużyciu energii
- Wsparcie dla łączenia scen
