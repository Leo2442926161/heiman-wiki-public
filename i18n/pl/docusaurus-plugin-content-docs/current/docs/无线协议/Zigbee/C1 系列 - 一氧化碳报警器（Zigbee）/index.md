# Seria C1 – Alarm tlenku węgla (Zigbee)

## Przegląd

Seria C1 to podstawowy alarm czadu Zigbee firmy Heiman. Wykorzystuje czujniki elektrochemiczne i nadaje się do scenariuszy wykrywania wycieków tlenku węgla. Przeszedł certyfikat EN 50291.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Elektrochemiczny |
| Zasięg detekcji | 0 ~ 1000 ppm |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | Zasilanie bateryjne |
| Żywotność czujnika | Około 7 lat |
| Temperatura pracy | -10°C ~ +50°C |

## Funkcje

- Elektrochemiczna detekcja CO
- Alarm klasyfikacji stężeń
- Alarm dźwiękowy i wizualny 85 dB
- Przypomnienie o niskim poziomie baterii
- Funkcja autotestu

## Standardy certyfikacji

-EN 50291
