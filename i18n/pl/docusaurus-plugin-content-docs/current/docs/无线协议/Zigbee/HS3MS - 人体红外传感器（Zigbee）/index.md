#HS3MS – Czujnik podczerwieni ludzkiego ciała (Zigbee)

## Przegląd

HS3MS to trzecia generacja czujnika podczerwieni Zigbee firmy Hyman, przeznaczonego do pomiaru ludzkiego ciała. Przyjmuje konstrukcję 360° montowaną na suficie, a jej czułość wykrywania i zasięg są znacznie ulepszone w porównaniu z poprzednią generacją.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Pasywna podczerwień (PIR) |
| Kąt detekcji | 360° (montaż sufitowy) |
| Odległość wykrywania | ≤7 m (25°C) |
| Sposób zasilania | Bateria litowa DC 3V |
| Żywotność baterii | Około 3 lata |
| Temperatura pracy | -10°C ~ +50°C |
| Metoda instalacji | Instalacja sufitowa |

## Funkcje

- Mobilne wykrywanie ludzi
- Zasięg dookólny 360°
- Przypomnienie o niskim poziomie baterii
- Alarm antysabotażowy
- Poprawiona czułość wykrywania
- Wsparcie dla łączenia scen
