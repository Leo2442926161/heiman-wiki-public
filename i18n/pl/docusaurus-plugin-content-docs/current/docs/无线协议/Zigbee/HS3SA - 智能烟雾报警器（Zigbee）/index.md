#HS3SA – Inteligentny czujnik dymu (Zigbee)

## Przegląd

HS3SA to inteligentny czujnik dymu Zigbee trzeciej generacji firmy Hyman. Ma zwiększoną wydajność wykrywania i możliwości łączenia w oparciu o pierwszą generację i jest odpowiedni na rynek UE.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Fotoelektryczny na podczerwień |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | DC 3 V (CR123A) |
| Żywotność baterii | Około 3 lata |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% wilgotności względnej |
| Metoda instalacji | Instalacja sufitowa |

## Funkcje

- Fotoelektryczna detekcja dymu
- Alarm dźwiękowy i świetlny o głośności 85 dB
- Przypomnienie o niskim poziomie baterii
- Funkcja autotestu
- Alarm antysabotażowy
-Wsparcie zdalnego wyciszenia
- Aktualizacja oprogramowania sprzętowego OTA
— Zwiększona wydajność sieci bezprzewodowej

## Standardy certyfikacji

-CE/CZERWONY
