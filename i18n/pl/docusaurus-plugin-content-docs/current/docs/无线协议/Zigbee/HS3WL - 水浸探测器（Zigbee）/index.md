# HS3WL – Detektor wycieków wody (Zigbee)

## Przegląd

HS3WL to trzecia generacja zanurzeniowego detektora Zigbee firmy Hyman. Ma zwiększoną czułość wykrywania i wydajność sieci bezprzewodowej w porównaniu z poprzednią generacją i nadaje się do scenariuszy monitorowania wycieków wody.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Metoda wykrywania | Wykrywanie sondy |
| Sposób zasilania | Bateria DC 3V |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% wilgotności względnej |

## Funkcje

- Wykrywanie w czasie rzeczywistym wycieków/akumulacji wody
— Zwiększona czułość wykrywania
- Alarm dźwiękowy
- Przypomnienie o niskim poziomie baterii
- Konstrukcja antykorozyjna sondy
