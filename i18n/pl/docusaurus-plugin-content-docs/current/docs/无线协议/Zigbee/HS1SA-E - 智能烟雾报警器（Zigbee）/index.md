# HS1SA-E – Inteligentny czujnik dymu (Zigbee)

## Przegląd

HS1SA-E to standardowy inteligentny czujnik dymu Zigbee wprowadzony przez firmę Hyman na rynek UE. Wykorzystuje technologię fotoelektrycznego wykrywania dymu, przeszedł certyfikację EN 14604 i zdobył nagrodę Red Dot Design Award.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Fotoelektryczny na podczerwień |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | DC 3 V (CR123A) |
| Żywotność baterii | Około 3 lata |
| Odległość komunikacyjna | ≤70 m (otwarta przestrzeń) |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% wilgotności względnej |
| Metoda instalacji | Instalacja sufitowa |

## Funkcje

- Fotoelektryczna detekcja dymu do wykrywania dymu z powolnego spalania i otwartego płomienia
- Alarm dźwiękowy i świetlny o głośności 85 dB
- Przypomnienie o niskim poziomie baterii
- Funkcja autotestu (test jednym kliknięciem)
- Alarm antysabotażowy
-Wsparcie zdalnego wyciszenia
- Aktualizacja oprogramowania sprzętowego OTA

## Standardy certyfikacji

- EN 14604 (europejska norma dotycząca czujników dymu)
- Nagroda Red Dot za projekt
-CE/CZERWONY

## Odpowiednie akcesoria

- Bateria CR123A
- Śruby montażowe/rura rozprężna
