# HS1DS – Czujnik magnetyczny drzwi (Zigbee)

## Przegląd

HS1DS to magnetyczny czujnik drzwi Zigbee wprowadzony na rynek przez firmę Heiman. Służy do wykrywania stanu otwarcia i zamknięcia drzwi i okien. Jest podstawowym elementem inteligentnego systemu bezpieczeństwa.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Kontaktron |
| Sposób zasilania | Bateria guzikowa CR1632 |
| Żywotność baterii | Około 2 ~ 3 lata |
| Luz instalacyjny | ≤10 mm |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% wilgotności względnej |

## Funkcje

- Wykrywanie w czasie rzeczywistym stanu otwarcia/zamknięcia drzwi i okien
- Alarm sabotażowy (Tamper)
- Przypomnienie o niskim poziomie baterii
-Obsługuje łączenie (otwieranie drzwi, włączanie świateł, uzbrajanie przy wyjściu z domu itp.)
- Łatwy w montażu (taśma klejąca 3M lub mocowanie śrubowe)

## Odpowiednie akcesoria

- Bateria guzikowa CR1632
- Taśma dwustronna/wkręty 3M
