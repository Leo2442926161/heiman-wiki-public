# Standardy certyfikacji

HS1DS przeszedł następujące certyfikaty:

- **CE/RED/EMC** — Dyrektywa Unii Europejskiej dotycząca sprzętu bezprzewodowego i kompatybilności elektromagnetycznej
- **Zigbee 3.0 / Q-Label** — Certyfikat Zigbee Alliance, patrz [Inne certyfikaty](../../../Certification Standards/Other Certifications.md)
