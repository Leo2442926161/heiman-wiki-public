# Parametry techniczne

Poniżej przedstawiono ogólne parametry techniczne urządzeń Heiman Zigbee.

## Parametry bezprzewodowe

| Parametr | Wartość |
|------|-----|
| Standardy protokołu | Zigbee 3.0 (IEEE 802.15.4) |
| Robocze pasmo częstotliwości | 2,4 GHz (globalne wspólne pasmo częstotliwości ISM) |
| Szybkość transmisji | Do 250 kb/s |
| Topologia sieci | Siatka samonaprawiająca się sieć |
| Pojemność węzła | Maksymalnie 65535 węzłów w jednej sieci |
| Odległość komunikacyjna | Około 10~30 metrów w pomieszczeniu (widoczne przeszkody na ścianie) |
| Metoda szyfrowania | AES-128-CCM |

## Zgodność

| Projekt | Opis |
|------|------|
| Koordynator Zigbee | Kompatybilny ze standardowym koordynatorem Zigbee 3.0 |
| Obsługiwane platformy | Zigbee2MQTT, ZHA (asystent domowy), Hubitat, Homey, SmartThings, bramka Tuya Zigbee, bramka Haiman HS6GW |
| Aktualizacja OTA | Obsługa bezprzewodowej aktualizacji oprogramowania sprzętowego (niektóre modele) |

## Zużycie energii elektrycznej i mocy

| Parametr | Wartość |
|------|-----|
| Sposób zasilania | Zasilanie bateryjne (CR123A / CR2450 / AA) lub USB / sieciowe (typ gniazdka) |
| Żywotność baterii | Około 2–3 lata na czujniki (w zależności od częstotliwości raportowania) |
| Pobór mocy w trybie gotowości | Poziom mikroampera, obsługuje mechanizm odpytywania snu |

## Parametry środowiska

| Parametr | Wartość |
|------|-----|
| Temperatura pracy | -10°C ~ +55°C |
| Wilgotność robocza | ≤ 95% RH (bez kondensacji) |
| Poziom ochrony IP | Zależy od konkretnego modelu (typ czujnika zwykle IP30) |
