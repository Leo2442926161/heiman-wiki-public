# Standardy certyfikacji

HS6GW (wersja Zigbee) posiada następujące certyfikaty:

- **CE/RED** — Dyrektywa Unii Europejskiej dotycząca sprzętu bezprzewodowego
- **CCC** — Obowiązkowa certyfikacja produktu w Chinach, patrz [CCC China Obowiązkowa certyfikacja](../../../../Certification Standards/CCC China Compulsory Certification.md)
