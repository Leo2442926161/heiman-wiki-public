# HS6GW - Inteligentna bramka (Zigbee)

## Przegląd

HS6GW to inteligentna bramka Zigbee wprowadzona na rynek przez firmę Heiman. Jako koordynator sieci Zigbee firmy Heiman łączy się z Internetem poprzez Wi-Fi lub Ethernet uplink i obsługuje dostęp aż do 128 urządzeń podrzędnych.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | Zigbee 3.0 (Koordynator) |
| Połączenie w górę | Wi-Fi/Ethernet |
| Maksymalna liczba węzłów | 128 podurządzeń |
| Sposób zasilania | Zasilacz USB DC 5V |

## Funkcje

- Koordynator sieci Zigbee
- Połączenie Wi-Fi/Ethernet typu uplink
- Automatyczne wykrywanie i parowanie urządzeń
- Powiązanie scen lokalnych
- Dostęp zdalny (poprzez platformę chmurową Heiman)
- Obsługuje do 128 urządzeń podrzędnych
- Aktualizacja oprogramowania sprzętowego OTA

## Obowiązujące scenariusze

- Podstawowy punkt połączenia inteligencji całego domu Heiman
- Współpracuj z aplikacją Haiman, aby zrealizować zdalne monitorowanie
