# HS8MLS – Mikroczujnik ruchu (Zigbee)

## Przegląd

HS8MLS to czujnik podczerwieni Zigbee (PIR) ludzkiego ciała dalekiego zasięgu wprowadzony na rynek przez firmę Heiman. Został specjalnie zaprojektowany do scen wymagających wykrywania na duże odległości i nadaje się do otwartych przestrzeni, takich jak korytarze i hole.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Pasywna podczerwień (PIR) |
| Odległość wykrywania | Wykrywanie na dużą odległość |
| Sposób zasilania | Bateria DC 3V |

## Funkcje

- Wykrywanie ruchu na dużą odległość
- Konstrukcja o niskim zużyciu energii
- Wsparcie dla łączenia scen
