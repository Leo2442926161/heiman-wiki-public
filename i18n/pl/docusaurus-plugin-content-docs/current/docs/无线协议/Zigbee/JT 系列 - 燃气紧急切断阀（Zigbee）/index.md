# Seria JT – Awaryjny zawór odcinający gaz (Zigbee)

## Przegląd

Awaryjny zawór odcinający gaz serii JT jest urządzeniem wykonawczym stosowanym w połączeniu z detektorem gazów palnych Hyman Zigbee. Po wykryciu wycieku gazu może zdalnie lub automatycznie zamknąć zawór gazociągu, aby zapobiec wypadkom.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Wpisz | Samodzielny + połączenie Zigbee |
| Gaz testowy | Gaz ziemny / Gaz płynny |
| Standardy certyfikacji | GB 15322 |
| Temperatura pracy | -10°C ~ +50°C |

## Funkcje

- Połączony z HS1CG / HS3CG, aby automatycznie zamknąć zawór
-Wsparcie zdalnego ręcznego zamykania zaworu
- Naciśnięcie alarmu wycieku
- Sterownik zaworu elektromagnetycznego

## Standardy certyfikacji

- GB 15322

## Odpowiednie akcesoria

- Detektor gazów palnych HS1CG / HS3CG
