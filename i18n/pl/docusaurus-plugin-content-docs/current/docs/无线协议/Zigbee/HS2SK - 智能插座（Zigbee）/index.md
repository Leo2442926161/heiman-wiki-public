#HS2SK - Inteligentne gniazdo (Zigbee)

## Przegląd

HS2SK to inteligentne gniazdo pomiarowe Zigbee wprowadzone na rynek przez firmę Heiman. Obsługuje zdalne sterowanie przełącznikiem i pomiar zużycia energii w czasie rzeczywistym. Nadaje się do inteligentnego zarządzania urządzeniami gospodarstwa domowego i ma dokładność ±2%.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Napięcie znamionowe | AC 220 V (norma chińska) / AC 110 V (norma amerykańska) |
| Maksymalne obciążenie | 16A / 3680W |
| Dokładność pomiaru | ±2% |
| Sposób zasilania | Zasilanie sieciowe |

## Funkcje

- Zdalne sterowanie przełącznikiem
- Pomiar mocy/paliwa w czasie rzeczywistym
- Zabezpieczenie przed przeciążeniem
- Wyłącznik czasowy
- Statystyki mocy
- Wsparcie dla łączenia scen

## Obowiązujące scenariusze

- Zdalne wyłączanie zapomnianych urządzeń
- Statystyki zużycia energii przez klimatyzator / podgrzewacz wody
- Wyjdź z trybu domowego, aby jednym kliknięciem wyłączyć niepotrzebne urządzenia
