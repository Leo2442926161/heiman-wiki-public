# HS1WL – Detektor wycieków wody (Zigbee)

## Przegląd

HS1WL to detektor zanurzeniowy Zigbee pierwszej generacji wprowadzony na rynek przez firmę Hyman. Przyjmuje metodę wykrywania sondy i może skutecznie wykryć wyciek wody lub gromadzenie się wody. Jest to podstawowy sprzęt do monitorowania zanurzenia w wodzie.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Metoda wykrywania | Wykrywanie sondy |
| Sposób zasilania | Bateria DC 3V |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% wilgotności względnej |

## Funkcje

- Wykrywanie w czasie rzeczywistym wycieków/akumulacji wody
- Alarm dźwiękowy
- Przypomnienie o niskim poziomie baterii
- Konstrukcja antykorozyjna sondy
