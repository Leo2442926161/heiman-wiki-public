#HS1MS – Czujnik podczerwieni ludzkiego ciała (Zigbee)

## Przegląd

HS1MS to standardowy czujnik podczerwieni Zigbee (PIR) ludzkiego ciała wprowadzony na rynek przez firmę Hyman. Wykorzystuje technologię pasywnej detekcji podczerwieni i potrafi wykryć promieniowanie cieplne w podczerwieni emitowane przez poruszające się ludzkie ciało. Jest szeroko stosowany w scenariuszach inteligentnego oświetlenia i bezpieczeństwa.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Pasywna podczerwień (PIR) |
| Kąt detekcji | 110° |
| Odległość wykrywania | ≤7 m (25°C) |
| Sposób zasilania | Bateria litowa DC 3V |
| Żywotność baterii | Około 3 lata |
| Temperatura pracy | -10°C ~ +50°C |
| Metoda instalacji | Montaż naścienny |

## Funkcje

- Mobilne wykrywanie ludzi
- Wykrywanie szerokiego kąta 110°
- Przypomnienie o niskim poziomie baterii
- Alarm antysabotażowy
- Wsparcie dla łączenia scen
