# HS1CA-E – Inteligentny alarm tlenku węgla (Zigbee)

## Przegląd

HS1CA-E to inteligentny czujnik czadu Zigbee firmy Hyman przeznaczony na rynek UE. Wykorzystuje czujnik elektrochemiczny i może dokładnie wykryć tlenek węgla o stężeniu 0 ~ 1000 ppm. Przeszedł certyfikat EN 50291.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Elektrochemiczny |
| Zasięg detekcji | 0 ~ 1000 ppm |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | DC 3 V (CR123A) |
| Żywotność czujnika | Około 7 lat |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% wilgotności względnej |
| Metoda instalacji | Montaż naścienny |

## Funkcje

- Bardzo precyzyjny elektrochemiczny czujnik CO
- Alarm stopniowania stężenia (komunikat dźwiękowy i świetlny)
- Dźwięk alarmu 85 dB
- Przypomnienie o niskim poziomie baterii
- Przypomnienie o upływie żywotności czujnika
- Funkcja autotestu
- Aktualizacja oprogramowania sprzętowego OTA

## Standardy certyfikacji

- EN 50291 (europejska norma dotycząca czujników czadu)
-CE/CZERWONY

## Odpowiednie akcesoria

- Bateria CR123A
