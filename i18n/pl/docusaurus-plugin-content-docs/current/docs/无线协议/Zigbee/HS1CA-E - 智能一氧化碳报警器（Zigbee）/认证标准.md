# Standardy certyfikacji

HS1CA-E posiada następujące certyfikaty:

- **EN 50291** — Europejska norma dotycząca czujników czadu, patrz [EN European Standard](../../../../Certification Standards/EN European Standard.md)
- **CE/RED/EMC** — Dyrektywa Unii Europejskiej dotycząca sprzętu bezprzewodowego i kompatybilności elektromagnetycznej
