#HS2WL – Detektor wycieków wody (Zigbee)

## Przegląd

HS2WL to czujka zanurzeniowa Zigbee drugiej generacji wprowadzona na rynek przez firmę Hyman. Obsługuje dwukanałowe wykrywanie sond i kabli i może być wykorzystywane do wykrywania zdarzeń wycieków wody, takich jak pęknięcie rury wodociągowej i przepełnienie zlewu.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Metoda wykrywania | Wykrywanie sondy + wykrywanie kabla (dwukanałowy) |
| Sposób zasilania | Bateria DC 3V |
| Temperatura pracy | -10°C ~ +50°C |
| Wilgotność robocza | ≤95% wilgotności względnej |

## Funkcje

- Wykrywanie w czasie rzeczywistym wycieków/akumulacji wody
- Sonda + kabel do wykrywania dwukanałowego
- Alarm dźwiękowy
- Przypomnienie o niskim poziomie baterii
- Konstrukcja antykorozyjna sondy

## Odpowiednie akcesoria

- Przedłuż kabel detekcyjny
