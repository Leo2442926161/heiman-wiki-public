#HS8SA – Inteligentny czujnik dymu (Zigbee)

## Przegląd

HS8SA to inteligentny czujnik dymu Zigbee + Wi-Fi firmy Hyman przeznaczony na rynek amerykański. Posiada funkcję łączenia w wysokiej temperaturze i przeszedł certyfikat UL 217.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | Zigbee 3.0 / Wi-Fi |
| Typ czujnika | Fotoelektryczny na podczerwień |
| Alarmowe ciśnienie akustyczne | ≥85 dB (3 metry) |
| Sposób zasilania | DC 3 V (CR123A) |
| Temperatura pracy | -10°C ~ +50°C |
| Standardy certyfikacji | UL217 |
| Obowiązujące rynki | Stany Zjednoczone |

## Funkcje

- Fotoelektryczna detekcja dymu
- Alarm wysokiej temperatury połączenia
- Alarm dźwiękowy i świetlny o głośności 85 dB
- Przypomnienie o niskim poziomie baterii
- Funkcja autotestu
- Alarm antysabotażowy
-Wsparcie zdalnego wyciszenia
- Komunikacja dwuprotokołowa (Zigbee + Wi-Fi)

## Standardy certyfikacji

- UL 217 (amerykańska norma dotycząca czujników dymu)
