# Standardy certyfikacji

HS8SA przeszła następujące certyfikaty:

- **EN 14604** — Europejska norma dotycząca czujników dymu (typ UE), patrz [EN European Standard](../../../../ApprovalStandards/EN European Standard.md)
- **UL 217** — amerykańska norma bezpieczeństwa czujników dymu (model amerykański), patrz [UL Underwriters Laboratories](../../../../certification-standards/ULUnderwriters Laboratories.md)
- **CE/RED/EMC** — Dyrektywa Unii Europejskiej dotycząca sprzętu bezprzewodowego i kompatybilności elektromagnetycznej
- **Zigbee 3.0 / Q-Label** — Certyfikat Zigbee Alliance, patrz [Inne certyfikaty](../../../Certification Standards/Other Certifications.md)
