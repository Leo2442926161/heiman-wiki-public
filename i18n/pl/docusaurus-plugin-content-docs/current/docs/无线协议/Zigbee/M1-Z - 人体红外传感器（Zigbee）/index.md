# M1-Z – Czujnik podczerwieni ciała ludzkiego (Zigbee)

## Przegląd

M1-Z to kompaktowy czujnik podczerwieni Zigbee do ciała ludzkiego wprowadzony na rynek przez firmę Heiman. Obsługuje kąt detekcji 110° lub 360° (w zależności od metody instalacji), ma wbudowany czujnik oświetlenia i nadaje się do inteligentnych scenariuszy oświetleniowych i bezpieczeństwa.

## Specyfikacje techniczne

| Parametr | Wartość |
|------|-----|
| Protokół komunikacyjny | ZigBee 3.0 |
| Typ czujnika | Pasywna podczerwień (PIR) + oświetlenie |
| Kąt detekcji | 110° (montaż na ścianie) / 360° (sufit) |
| Odległość wykrywania | ≤7 m (25°C) |
| Zakres oświetlenia | 0 ~ 1200 luksów |
| Sposób zasilania | Bateria litowa DC 3V |
| Żywotność baterii | Około 3 lata |
| Temperatura pracy | -10°C ~ +50°C |
| Metoda instalacji | Naścienny/sufitowy |

## Funkcje

- Mobilne wykrywanie ludzi
- Wykrywanie oświetlenia (0 ~ 1200 luksów)
- Możliwość przełączania 110° / 360°
- Przypomnienie o niskim poziomie baterii
- Alarm antysabotażowy
- Kompaktowa konstrukcja
