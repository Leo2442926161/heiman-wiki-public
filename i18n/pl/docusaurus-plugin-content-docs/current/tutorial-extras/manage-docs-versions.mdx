# Zarządzanie wersjami dokumentacji

Wiki Heiman Technology obsługuje zarządzanie dokumentacją w wielu wersjach, zapewniając dostęp do dokumentacji technicznej odpowiadającej wersji oprogramowania sprzętowego produktu.

## Tabela zgodności wersji

| Wersja dokumentacji | Odpowiednie oprogramowanie sprzętowe | Data wydania |
|-------------------|--------------------------------------|--------------|
| 2.0               | Seria Matter                         | 2025-06      |
| 1.0               | Seria Zigbee                         | 2024-01      |

## Przeglądanie różnych wersji

W rozwijanym menu wersji na górnym pasku nawigacyjnym możesz przełączać się między różnymi wersjami dokumentacji.

## Informacje o aktualizacjach wersji

Stale aktualizujemy dokumentację, aby odpowiadała iteracjom produktów. Zalecamy regularne sprawdzanie najnowszej wersji.
