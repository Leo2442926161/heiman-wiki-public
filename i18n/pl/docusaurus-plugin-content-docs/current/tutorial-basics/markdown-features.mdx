# Funkcje Markdown

Dokumentacja techniczna Heiman w pełni obsługuje składnię Markdown i funkcje rozszerzone, ułatwiając tworzenie strukturalnej dokumentacji technicznej produktów.

## Przykład kodu

Przykład wywołania API podczas integracji produktu:

```jsx title="Integracja API czujnika dymu"
const heimanClient = new HeimanClient({
  apiKey: 'your-api-key',
  deviceId: 'HS1SA-xxxxx',
});

// Pobierz status urządzenia
const status = await heimanClient.getDeviceStatus();
console.log(status);
```

## Ramki ostrzegawcze

Używaj ramek ostrzegawczych, aby wyróżnić ważne informacje:

:::tip

Czujnik dymu Heiman HS1SA posiada certyfikat EN14604

:::

:::caution

Podczas instalacji czujnika gazu upewnij się, że okablowanie jest wykonane zgodnie z instrukcją instalacji

:::

:::danger

W przypadku wycieku gazu nie włączaj żadnych urządzeń elektrycznych, natychmiast otwórz okna i ewakuuj się

:::
