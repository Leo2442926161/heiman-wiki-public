# Wdrażanie rozwiązań

Heiman Technology oferuje kompleksowe wskazówki dotyczące wdrażania inteligentnych rozwiązań bezpieczeństwa.

## Proces integracji rozwiązań

Produkty Heiman obsługują wiele metod wdrażania, w tym lokalne wdrożenie niezależne i integrację z platformą chmurową.

### Wdrożenie lokalne

```bash
# Konfiguracja inteligentnej bramki przeciwpożarowej Heiman
heiman-cli configure --gateway HS1GW --mode local

# Rejestracja urządzenia czujnika
heiman-cli register --device HS1SA --gateway HS1GW
```

### Integracja z platformą chmurową

Produkty Heiman są zintegrowane z głównymi platformami IoT:

- China Telecom Tianyi
- China Mobile OneNET
- Huawei HiLink
- Alibaba Cloud IoT
- Tuya Smart

## Obsługa protokołu Matter

Produkty Heiman z serii Matter są kompatybilne z:

- Amazon Alexa
- Google Home
- Apple HomeKit
- Samsung SmartThings

Umożliwiając interoperacyjność między markami i ekosystemami.

## Weryfikacja wdrożenia

Po zakończeniu wdrożenia można przeprowadzić weryfikację funkcji i monitorowanie w czasie rzeczywistym za pomocą aplikacji Heiman Smart Fire lub platformy zarządzania.
