Matter

Matter（原名 Project CHIP）是 CSA 连接标准联盟推出的智能家居互联标准，旨在打破品牌壁垒，实现跨生态的互操作性。

海曼与 Matter​

海曼是 Matter 协议的早期采用者，推出了全球首批 Matter 认证的一氧化碳报警器。海曼的 Matter 设备均采用 Matter over Thread 方案。

技术特点​

特性说明
底层协议Thread（低功耗 Mesh 网络）
网络拓扑Mesh 自愈网络，设备越多覆盖越广
功耗极低，电池供电可运行 1-3 年
安全性支持 AES-128-CCM 加密，设备需要认证
互操作性跨 Apple HomeKit、Google Home、Amazon Alexa、Home Assistant 等

边界路由器要求​

使用 Matter over Thread 设备需要 Thread 边界路由器（Border Router）：

  • Apple HomePod mini / HomePod 2代 / Apple TV 4K
  • Google Nest Hub 2代 / Nest Hub Max
  • Amazon Echo 4代
  • Home Assistant Green + Connect ZBT-2
  • Homey Pro

海曼 Matter 认证产品​

  • 人体红外传感器 M1-M
